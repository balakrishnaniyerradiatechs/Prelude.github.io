Viewing Dead Stock
The TPCx Dead Stock Report/Update screen sorts by vendor and price.
Field/Function Key Descriptions
 
Field/Function Key
Description
Product
The item available for dead stock update.
Description
The description of the item.
Vendor
The TPCx vendor for the item.
Description
The description of the vendor.
Quantity
The quantity available of the item.
Price
The price of the item for the specified customer.
Extension
The extension of the item (Quantity x Price).
Update
Updates the item as dead stock.
Dead
An asterisk (*) appears if the item is already dead stock on the hub
F2-Upload
Updates the items selected as dead stock to the hub.
F5-Select All
For all products on the screen, sets Update to Y.
F6-Deselect All
For all products on the screen, sets Update to N.
F9-Goto Line
Enter a line number to go to.
 
If a quantity changes for a dead stock item such as a sales order or inventory adjustment, Prelude notifies the hub to change the quantity. When you sell and invoice dead stock, Nightly Processing sends a billing request to TPCx with information about the sale to bill the trading partner.
