Dead Stock Search
The Dead Stock Search page allows you to search the dead stock database for anyone selling discounted, slow-moving material. You can also see the items you posted as dead stock, the quantity, your price, and the date you posted them. You cannot purchase dead stock via the B2B Marketplace; you must submit a P/O normally through Prelude. To access the Dead Stock Search, click Dead Stock Search in the B2B Marketplace navigation pane.
Selection Criteria
•
Keyword Search on Item Code and Descriptions - use keywords from the item code and description.
•
UPC - matching UPC.
•
Manufacturer Part # - matching manufacturer part numbers.
•
Include in Search - check to include your dead stock.
•
Sort Order - indicate how the report should sort.
•
Trading Partners - choose trading partners you want to search for dead stock. Includes all trading partners. If you do not have a relationship with a trading partner you want to buy from, you must initiate a relationship.
•
Manufacturers - choose manufacturers you want to search.
When you click Search. The Dead Stock Search Results page appears with the results. Click a column header to resort the rows by that column.
Column Descriptions
Column
Description
Item Code
Prelude part number. Click to view the Search Results Select page.
Manufacturer
The manufacturer for the item.
Item Description
A description of the item.
Price
The price of the item.
Quantity
The quantity of the item that is available for sale.
Date Posted
The date the stock was first posted to the dead stock database.
Trading Partner
The trading partner who is selling the material.
UPC
The UPC of the item, from Trading Partner Connect's database.
Print
Allows you to print the search results.
Email
E-mail the results to the default e-mail address for your login ID.
Click an item code to open the Search Results Select page, where you can see if you have a relationship with the trading partner that carries the item.
The Search Results Select page contains all the information for the line item you selected from the Dead Stock Search Results page except for Date Posted and UPC. Additionally, a message appears in red if you have a trading relationship with the Trading Partner Connect member selling this material. If you do not, then you can click Request Relationship to initiate a relationship. If you have questions about how to initiate a trading partner relationship, see the B2B Marketplace online help.
