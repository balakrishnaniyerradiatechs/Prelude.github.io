Company Maintenance
Once the TPCx module is activated, go to System Parameter > Maintenance > Company Maintenance > F6-Modules and press ENTER over Y to view the TPCx Information screen.
Field/Function Buttons
 
Field/Function Key
Description
TPCx System Number
Your TPCx system ID. You many have multiple TPCx system. Activant provides this number.
TPCx System Path
The system path to the spoke.
TPCx External ID
The external id, which is the interface system’s customer code. Activant provides this ID.
TPCx Type
The TPCx type (typically CCODE).
TPCx Version
The Trading Partner Connect version number.
Dead Stock Customer
The dead stock customer to default with the Dead Stock Identification procedure.
Administrator Email
The email address used to notify the TPCx administrator that a product’s TPCx data could not be updated.
F2-Save
Saves the information.
