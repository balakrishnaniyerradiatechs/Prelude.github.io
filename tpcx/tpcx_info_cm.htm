Customer Maintenance
To see if a customer is rationalized as a trading partner, go to Order Processing > Maintenance > Customer Maintenance > Action Bar > Add’l Opts > TPCx Information. The TPCx Information screen appears.
Field/Function Key Descriptions
 
Field/Function Key
Description
TPCx Rationalized
Indicates if you have a trading partner relationship with this customer:
•
Null - no relationship.
•
Y - current relationship.
•
N - cancelled relationship.
OK
Closes the screen.
