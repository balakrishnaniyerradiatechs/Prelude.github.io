Sending and Receiving TPCx Data
Once the EDI inbound and outbound documents are setup, run Flat File Processing to send and receive data. To send and receive transactions:
1
Choose E-Commerce > Flat Files > Processes > Flat File Processing. The Flat File Processing screen appears.
2
Choose from the following selection criteria:
•
Transfer Type - The transfer type to be updated. Enter TPCX for trading partner connect interface.
•
Transaction Type - Enter the transaction type to send or receive. For example, if you’re sending an outbound P/O to the hub, enter OPO.
•
Auto Create Flag - For outbound transactions, enter N. For inbound transactions, enter Y.
•
Import/Export Data - Indicate which kind of transactions you want:
-
E - Outbound.
-
I - Inbound.
3
Press F2-Execute to send or receive data from TPCx.
