Unpost Dead Stock
 
The Unpost Dead Stock page allows you change an item marked as dead to a regular inventory item using search criteria.
Selection Criteria
•
Keyword Search on Item Code and Descriptions - use keywords from the item code and description.
•
UPC - matching UPC.
•
Manufacturer Part # - matching manufacturer part numbers.
•
Include in Search - check to include your dead stock.
•
Sort Order - indicate how the report should sort.
•
Manufacturers - choose manufacturers you want to search.
Click Search to view the Un-Post Dead Stock Search Results page. Some of the column headers are links you can click to resort the rows according to that column. If the rows already sort by that column, click to invert the sort.
Column Descriptions
 
Column
Description
Un-Post
Check to remove the item from its dead stock listings. You may view all pages of data before you press Save. You do not have to press Save for each page..
Item Code
The item code by which the item is listed in Prelude.
Manufacturer
The manufacturer for the item.
Item Description
A description of the item.
Price
The price of the item.
Quantity
The quantity of item posted as dead stock.
Date Posted
The date on which the stock was first posted to the dead stock database.
UPC
The UPC of the item, from Trading Partner Connect's database.
Select All on Current Page
Checks Un-Post for all rows on the current page of results only.
Deselect All on Current Page
Clears Un-Post for all rows on the current page of results only.
Select All on Each Page
Checks Un-Post for all returned results on all pages.
Deselect All on Each Page
Clears Un-Post for all returned results on all pages
Save
Remove all items with the Un-Post checkbox enabled from the dead stock database and send a document to Prelude that clears the dead stock setting.
