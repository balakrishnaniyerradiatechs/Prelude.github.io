Vendor Maintenance
To see if a vendor is rationalized as a trading partner, go to Purchasing > Maintenance > Vendor Maintenance > Action Bar > Add’l Opts > TPCx Information. The TPCx Information screen appears.
Field/Function Key Descriptions
 
Field/Function Key
Description
TPCx Rationalized
Indicates if you have a trading partner relationship with this vendor:
•
Null - no relationship.
•
Y - current relationship.
•
N - cancelled relationship.
TPCx Line
Indicates if this vendor has rationalized products:
•
Null - never had rationalized products.
•
Y - current has at least one rationalized products.
•
N - previously had rationalized products, but does not now.
OK
Closes the screen.
