Data Mining Report
The Data Mining Report allows you to receive a report that searches the dead stock database and matches dead stock for sale with items for which you have normal usage. The report also matches other trading partners' usage to your dead stock, and provides you with a report of potential buyers and their contact information. You can send the report manually to yourself. Also, Activant sends a monthly Data Mining Report to the contacts you specify. Each trading partner you select also receives a dead stock report similar to yours, but focused on their business instead of yours.
Selection Criteria
•
Trading Partners - determines which trading partners will be included in the report. You can select one or multiple trading partners.
•
Manufacturers - determines which manufacturers will be included in the report. You can select one or multiple trading partners.
•
Find Items for Me to Buy Where My Months Supply Is Less Than - determines which of your items the system will try to match against the dead stock database.
•
Sort By - choose item or manufacturer.
•
Recipient Email - the e-mail to which TPCx will send the report.
•
Report Format - choose PDF attachment or tab-delimited text file.
Click Submit to send an acknowledgement and the report to the Recipient Email only.
If you chose tab-delimited text format (.txt), to import it into Microsoft Excel:
1
Open Excel.
2
Select Data | Get External Data | Import Text File. The Import Text File dialog appears.
3
Select the text file you want to import.
4
Ensure that the file type is set to Delimited, and if you are importing a Data Mining Report, set starting import row to 2.
5
Click Next.
6
Check Tab and verify that all other delimiters are unchecked.
7
Click Next.
8
Select all the columns in the Data Preview field.
9
Set the Column Data Format to Text.
10
Click Finish. The data from the tab-delimited text file will display in Excel.
These instructions are based on Excel 2000 and may vary by version. If you have a different version, refer to Excel documentation for more information.
