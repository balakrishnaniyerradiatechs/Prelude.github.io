Shop Job Code Product screen
Enter products and labor charges to apply every time a job is used in the Shop Job Code Product Defaults screen. These charges are then included on the order each time this job code is entered. The Repair Amount for this job code is the same regardless of the additional cost of these items.
Menu Path: Shop Repair > Maintenance > Job Code Maintenance > F7-Products
Field/Function Key Descriptions
Field/Function Key
Description
Product
The part number and description that is associated with the job.
Code
Indicates if the product is:
•
R - Required.
•
O - Optional.
Price
The price of the product. This price is used to calculate the cost of the job unless a Repair Amount is entered for the job code. Refer to Setting Up Job Codes for more information.
Quantity
The quantity needed to complete the repair.
F2-Save
Saves the product information.
