Quotation Header screen
The Quotation Header screen allows you to define quote-level information, such as follow-up date, written by, and salesrep information. Some of this information can be overridden in the Line Item Detail screen.
Menu Path: Add. Modules > Shop Repair > Shop Repair Quote Subsystem > Shop Repair Quote Entry > F5-Hdr
Field/Function Key Descriptions
 
 
Field/Function Key
Description
Quote Type
The type of quote. Choose from:
•
01 – Regular.
•
02 – Plain.
•
03 – Repair.
•
04 – Standing order that can be converted to a regular shop repair order on a regular basis.
Purchasing Contact
The purchasing agent from the company that is placing the quote. Only available if your company uses CRM.
Ship-To Number
The ship-to address for this quote. Choose from:
•
D - Use default address from Customer Maintenance.
•
P - Customer will pick up at your warehouse.
•
N - Create new ship-to address. The address is stored in Customer Ship-To Maintenance and may be used on another quote.
•
M - Enter a new ship-to address for this quote only.
•
C - Use the Purchasing Contact email.
•
<#> - Use existing ship-to address.
Written By
The user ID of the person taking the quote.
Sell Warehouse
The warehouse to receive credit for the sale.
Ship Warehouse
The warehouse to ship the products.
Ship Via Code
The code that represents how the quote will be shipped (e.g., 01 - UPS, 02 - FedEx).
Contact
The customer’s A/R contact person.
Email Address
The email address for the customer’s A/R contact person.
Customer P/O#
The customer’s purchase quote number.
Cust Release#
The release number the customer provides in addition to or instead of a P/O number.
Job Number
The customer’s job number.
Quote Date
The date the quote should be recorded. Defaults to current date.
Required Date
The date the customer requires the products.
Expiration Date
The date the quote is no longer valid.
Follow Up Date
The date the person who entered the quote should follow-up with the customer.
Instructions
Any special instructions.
F2-Save
Saves the information.
F5-Add Info
Opens the Order Additional Information screen.
F6-Ship-To
Opens the Ship-To Address screen
F7-Tax Jur
Opens the Tax Jurisdiction screen.
F8-Credit Info
Opens the Credit Information screen.
Action Bar
Repair Type
Not applicable for shop repair quotes.
Currency
Allows you to change the exchange rate.
Future Commit
Allows you to override future commitment for products on this quote.
