Setting Shop Repair Parameters
To set system options to determine how shop repair orders are entered and updated:
1
Select Option Parameters > Shop Repair Parameters to open the System Options For Shop screen.
2
For Identify CN by Model, indicate whether you want to start a shop repair order by identifying the customer or the part number (refer to Entering a Shop Repair Order). Choose from the following:
•
Y - Enter the model and serial number first to identify the customer.
•
N - Enter the customer number first, then the model and serial number.
3
For Require Tech on Job, determine if a technician must be assigned to a job (refer to Creating the Job). Choose from the following:
•
Y - You must assign the technician before he/she can clock into a job.
•
N - You do not have to assign a technician. Any technicians that clock into the job are automatically added to the list of assigned technicians.
4
For Require Problem Notes, determine if problem notes are required for each job. Problem notes give a more complete description of what needs to be repaired (refer to Creating the Job). Choose from the following:
•
Y - Require problem notes for each job on a shop repair order.
•
N - Do not require problems notes.
5
For Enter Hours Only, determine if access to the Hours field is allowed in Technician Billing Entry (refer to Billing Jobs for Technicians). Choose from the following:
•
Y - Allow access to Hours in Technician Billing Entry to edit start and stop time.
•
N - Do not allow access Hours in Technician Billing Entry.
6
For Update Asset For Capital Repair, determine if a sub-asset or a main asset is created for a shop repair order. Choose from the following:
•
Y - Update the main asset for the shop/repair order with the dollars for add-on capital updated in Asset Maintenance > F7-Books > Asset Additions.
•
N - Do not update the main asset. The system creates a sub-asset for the add-on capital.
7
For Continuous Shop Invoice Print, if continuous printing is turned on for invoicing, indicate if shop invoices should print continuously also. Choose from the following:
•
Y - Print shop invoices continuously.
•
N - Do not continuously print shop invoices.
8
For Repair Group Optional, indicate if the Repair Group is required in the Shop Repair Job Entry screen (refer to Identifying Repair Groups). Choose from the following:
•
Y - A Repair Group is optional but not required. The system does not check the technician’s repair skill set.
•
N - A Repair Group is required. The repair group entered must be established for the technician’s skill set in Technician Maintenance (refer to Setting up Technicians).
9
For Auto Shop Charges, indicate if a separate shop charge is automatically created when you complete the shop order (refer to Completing the Order):
•
Y - Automatically create a shop charge for each job on the order, without prompting. A separate charge generates for each job and is determined by the pay type assigned to the job. Refer to Creating Pay Types.
•
N - Do not automatically create shop charges. The Shop Repair Job Entry screens appears when you press F6-Complete from the Shop Repair Recap Information screen, so that you can manually add each shop charge to each job.
10
If you use shop repair orders to repair rental equipment; Update Equip Expense With ‘List’, determines if the equipment expense is updated with the list price or the cost of the shop/repair order. Choose from the following:
•
Y - Update the equipment expense with the list price from the shop repair order.
•
N - Update the equipment expense with the cost from the shop repair order.
11
For Prorate Labor Cost, determine how labor cost is calculated. Choose from the following:
•
Y - Pro-rate labor cost based on actual time posted to the job.
•
N - Do not pro-rate labor cost. Calculate labor cost based on billable time.
Labor cost is only pro-rated for the first service labor product for each shop/repair job on the order.
12
For Use Pline for Capital Postings, determine whether to use the G/L account from the product line or the A/R offset account for capital postings:
•
Y - Use the product line.
•
N - Use the A/R offset account.
This flag only applies to shop repair orders with a pay type that have Capitalize set to Y. Refer to Creating Pay Types.
If the unit is still on hand, the posting goes to the inventory account. If the inventory is sold, the posting goes to the cost of goods account.
13
For Whse, enter the warehouse(s) authorized to set up new equipment. Refer to Entering Standard Repair and Setup Time for Products.
14
For Process Inventory as WIP, indicate if you will use the extra work-in-process step to account for inventory on a shop repair order:
•
Y - Process inventory on a shop repair order as work-in-process (WIP). You must use Release Confirmation Entry to relieve inventory from the on-hand balance and move it into the WIP balance. When the order in invoiced, the quantities move out of WIP. Refer to the Shop Inventory Work In Process Report to list products in the Work in Process Account.
•
N - Do not process inventory as work-in-process. When you print the shop repair ticket, inventory balances are not affected until the order is invoiced. Release Confirmation Entry is not used.
15
For Convert Estimated Labor, indicate if estimated labor on a shop repair quote is converted to the shop repair order:
•
Y - Convert estimated labor from a quote.
•
N - Do not convert estimated labor from a quote.
16
For Create Multiple Setups, indicate if multiple shop repair set up orders can be generated from one sales order:
•
Y - When you enter a line item on a sales order that requires a shop repair set up order, the number of shop repair set up orders generated will equal the quantity of the main item on the sales order.
•
N - Only one shop repair set up order is generated for each item on a sales order that requires a set up, even if the quantity is more than one.
17
Press F2-Save to save any changes to the system options.
