Setting up Technicians
Before you can use Technician Job Billing Entry to record time clocked for a job (refer to Billing Jobs for Technicians) you must establish the following information for each of your technicians.
•
Labor rates to automatically calculate and add labor costs to the job based on the shop labor type chosen when the clocked hours are entered (refer to Identifying Shop Labor Types).
•
Repair groups that the technician is authorized to work on (refer to Identifying Repair Groups). You cannot assign a technician to work on a job with a repair group for which he/she is not authorized. You must specify all possible repair groups. For example, if you want the technician to be able to work on all repair groups, you must enter each repair group. You can also identify the technician’s skill level in the repair group.
To set up a shop repair technician:
1
Select Shop Repair > Maintenance > Technician Maintenance to open the Technician Maintenance screen.
2
For Technician, enter the code for the technician.
3
For Description, enter the technician’s name.
4
Enter the address and phone number information for the technician.
5
Enter the hourly pay rate that should be charged for the technician’s clocked time for each of the following:
•
Regular Time Rate. Used for Type R in Shop Labor Type Maintenance.
•
Overtime Rate. Used for Type O in Shop Labor Type Maintenance.
•
Premium Rate. Used for Type P in Shop Labor Type Maintenance.
6
Press F5-Repair Skill Set to open the Shop Repair Skill Set screen.
7
For RGroup, enter the codes for all repair groups that this technician is authorized to work on (refer to Identifying Repair Groups for information on Repair Group Maintenance).
8
If applicable, enter the skill Level that this type of repair group represents.
9
Press F2-Save to save the repair group information.
10
Press F2-Save to save the technician information.
