Setting Up Inspection Codes
Inspection codes identify the type of inspection required for a serial number. Once you set up the codes in Shop Repair Inspection Code Maintenance, you can apply them to serial numbers (refer to Setting up Sub-Serial Numbers). This information appears on the inspection ticket when you print a shop repair order.
To set up inspection codes:
1
Select Shop Repair > Maintenance > Inspection Code Maintenance to open the Shop Repair Inspection Code screen.
2
Enter an Inspection Code to identify the type of inspection. These could be a series of inspection levels or standardized inspection codes.
3
Enter a Description of the inspection code.
4
For Serviceable, choose from the following:
•
Y - The inspection code indicates that the product requires servicing.
•
N - The inspection code is not for a serviceable item.
5
Press F2-Save to save the inspection code information.
