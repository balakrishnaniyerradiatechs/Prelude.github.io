Generating Purchase Orders for a Shop Repair Order
If a product on a shop repair order is backordered, you can generate a purchase order for the product if:
•
The Sc (supply code) is B, E, or D.
•
Generate P/O from S/O is set to Y in your user ID.
•
S/O-P/O, N/S Items Only is set to Y in Purchasing Option Parameters. You can only generate purchase orders for non-stock items.
•
Your company’s settings allow purchase orders to generate from Shop Repair Order Entry.
If the Automate P/O Generation flag in Purchasing Option Parameters is set to Y, the system automatically generates the purchase orders without the following prompts.
To generate purchase orders for line items:
1
If the above requirements are met and you can enter costs, the Cost Prompt screen appears.
2
Enter a cost and press ENTER. A dialog box with the following message appears:
Do You Wish To Automatically Generate A Purchase Order For This Product?
3
Choose one of the following:
•
Yes - Generate the P/O at the end of the order.
•
No - Allow the P/O to generate as part of Nightly Processing.
4
If you select Yes, the Automatic Purchase Order Vendor screen appears.
5
The primary vendor for the product displays, but you can enter a different vendor number if necessary.
6
Press F2-Save.
7
After you finish entering the rest of the products/jobs on the order, select No Action or Print (refer to Saving the Shop Repair Order) to generate the purchase order. If the above requirements are met, a dialog box with the following message appears:
Do You Wish To Proceed WIth Purchase Order Update?
8
Choose one of the following:
•
Yes - Immediately generate the purchase orders.
•
No - Allow the purchase orders to generate as part of Nightly Processing.
9
If you select Yes, the Generate P/Os for Backorders screen lists the purchase orders for each vendor you chose.
10
Choose from the following:
•
F2-Generate All: Generates purchase orders for the vendors listed.
•
F5-Edit: Allows you to edit and the selected purchase order.
•
F6-Generate: Generates one purchase order at a time for the highlighted vendor.
11
When the orders are generated, press F9-Continue. The Output dialog box appears for each vendor.
12
Select the appropriate method for delivering the purchase order to the vendor.
