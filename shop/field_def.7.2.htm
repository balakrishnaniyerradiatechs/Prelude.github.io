Daily Resource Schedule screen
The Daily Resource Schedule screen allows you to view the technicians you have available and to assign them to a job.
Menu Path: Add. Modules > Shop Repair >> Shop Repair Order Entry > Shop Repair Job Entry > F9-Schedule > F5-Review
Field/Function Key Descriptions
 
Field/Function Key
Description
Resources
The technicians that are set up to work on repair jobs.
Hours of the day
Asterisks appear under the hours that each technician is already assigned to jobs.
F2-Save
Saves the resource information.
F5-Review
Opens the Scheduling Documents screen to allow you to review the time, customer, and job information assigned to the highlighted technician.
F6-Schedule
Opens the Schedule Resource screen to allow you to schedule the highlighted technician to the job you are in.
