Open Quote Repair Inquiry
The Open Quote Repair Inquiry screen lists open shop repair quotes for a customer.
Viewing Open Quotes
To list open quotes for a customer:
1
Select Shop Repair > Shop Quote Subsystem > Open Repair Quote Inquiry. The Open Quote Repair Inquiry screen appears.
2
Enter the Customer or keyword.
3
Choose a Sort option:
•
Order number.
•
Order date.
•
Required date.
•
Serial number.
•
Model number.
•
Sales order dollars.
4
Choose a Filter option:
•
No.
•
Yes - use the following selection criteria:
-
Model number.
-
Serial number.
-
Required date range.
-
Quote number.
-
Cust release#.
-
Customer P/O#.
-
Repair groups.
Open shop repair quotes for the customer appear.
Field/Function Key Descriptions
Field/Function Key
Description
Quote#
The shop repair quote number.
Notes
An asterisk (*) appears if notes were entered for the order.
Hold
Indicates if the order is on quote hold:
•
H - On quote hold.
•
R - Released from quote hold.
•
Blank - Not on hold.
Disp
Disposition analysis. If the quote was deleted, a deletion reason code appears.
Qte-Date
The date on which the quote was entered.
Model Number
The main part that was quoted.
Serial Number
The serial number of the part that was quoted.
Quote Amount
The total price that the customer was quoted.
Conv
Indicates if the quote was converted into an order:
•
Y - The quote was converted.
•
N - The quote was not converted.
F2-Finish
Close this screen.
F5-Review
Opens the Shop Repair Quote Display screen to display detailed information for the highlighted order.
F6-Quote#
Allows you to enter a quote number to review.
F8-Log
Opens the Quote Log Inquiry screen to display any transaction that affected the order.
F9-Goto Ln
Allows you to enter an order number, page number, line number, B (bottom), or T (top) to go directly to.
