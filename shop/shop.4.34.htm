Previewing Invoices
To preview invoices:
1
Choose Order Processing > Invoice Preview Printing > Invoice Preview Report. The Invoice Preview Report screen appears.
2
Enter selection criteria and press F2-Save. A dialog box appears.
3
If you select Execute, the output appears.
4
Select an output. A sample of the report is shown on the next page.
