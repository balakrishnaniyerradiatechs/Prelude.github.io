Field Definitions
Calendar Header screen
The Calendar Header screen allows you to enter the header information (date, warehouse, type) for the scheduling Calendar.
Refer to Scheduling Resources for Shop Repair Jobs.
Menu Path: Shop Repair > Processes > Calendar Scheduling > Scheduling Calendar Entry
Menu Path: Shop Repair > Shop Repair Order Entry > Shop Repair Recap Information screen > F8-Schedule.
Field/Function Key Descriptions
Field/Function Key
Description
Date
The date to schedule.
Whse
The warehouse the schedule is for.
Hours
The number of hours to schedule.
Type
The type of calendar that you are scheduling/reviewing.
•
D - Delivery.
•
R - Repair.
•
W - Work Orders.
View
The type of calendar you want to schedule/view:
•
D - Detail Information. Opens the Repair Detail screen so you can enter information for the job and schedule appointments. Refer to Scheduling Appointments by Day.
•
W - Weekly calendar. Opens the Shop/Repair Week View screen so you can schedule resources for a week. Refer to Scheduling Appointments by Week.
•
M - Monthly calendar. Opens the Scheduling Calendar screen to allow you to view/enter detail repair information for a day of the month. Refer to Scheduling Appointments by Month.
F2-Save
Saves the header information and opens the appropriate screen.
