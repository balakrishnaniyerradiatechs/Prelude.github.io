Changing Time Clock Entries
If a technician makes a mistake in Shop Repair Timeclock Entry, you can use Shop/Repair Timeclock Maintenance to correct the error. If the error was made in Technician Job Billing Entry, you can correct the error either in Technician Job Billing Entry or Shop Repair Time Clock Maintenance.
If you correct a Technician Job Billing Entry in Shop Repair Time Clock Maintenance, the correction is not reflected on the order. You must manually correct the order.
To correct a time clock entry:
1
Select Shop Repair > Processes > Shop Time Clock > Shop Repair Timeclock Maintenance to open the Shop/Repair Timeclock Maintenance screen.
2
For Technician#, enter the Technician number whose time should be corrected.
3
Enter the Date for which you need to adjust time worked. The default is the current date.
4
For Job/Code, edit, delete, or add the job or code:
•
To enter a code (e.g., SW - start work, BT - break time), enter the code or press F3 to search.
•
To enter a job, enter the order number and job number in the following pattern <order>!<job>. For example, to clock into job B for order 98650, enter 98650!B.
5
Change the Start and End time as necessary.
6
Repeat steps 4-5 for each entry to correct.
7
Press F2-Finish.
