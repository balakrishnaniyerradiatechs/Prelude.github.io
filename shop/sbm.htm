Service Bulletin Maintenance screen
When a vendor sends you a service bulletin for product(s) that need to be repaired, you can use Service Bulletin Maintenance to enter information about the repair and indicate which products and serial numbers are affected. When you enter the affected product/serial number combination on a shop repair order, you are prompted to add the service bulletin as a job on the order.
For more information, refer to Entering Service Bulletins.
Menu Path: Shop Repair > Maintenance > Service Bulletin Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Bulletin Number
The bulletin number from the vendor.
Entry Date
The date the bulletin was entered.
Description
A brief description of the bulletin.
Issue Date
The date the bulletin goes into effect.
Completion Date
The date the service bulletin needs to be completed.
Internal Date
The internal date that your company wants to complete all service work on this bulletin.
Alert Level
Indicates the type of bulletin:
•
1 - Safety rework (the repair is critical before someone gets hurt).
•
2 - Repair.
•
3 - Repair bulletin.
•
4 - Information bulletin.
Pay Type
The pay type code that indicates how the job will be billed (e.g., CB, WTY). For more information on pay types, refer to Creating Pay Types.
Repair Group
The repair group code to identify the type of repair job. For more information, refer to Identifying Repair Groups.
Repair Time
The published or expected amount of time it should take to perform the repair.
Location Filed
Where the bulletin is filed, so that all of the bulletin information does not have to be typed into the system.
F2-Save
Saves the service bulletin information.
F4-Delete
Deletes the bulletin.
F5-Products
Opens the Service Bulletin Product Coverage screen, so you can assign model and serial numbers to this bulletin. When you enter an order for one of these assigned products, you are prompted to review the service bulletin.
F6-Situation/Correction
Allows you to enter information about the problem and the solution.
