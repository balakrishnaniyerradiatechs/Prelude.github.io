Override Product Serial Information screen
The Override Product Serial Information screen in Shop Repair Job Entry, allows you to enter or override the serial number set up for a product. If you change a serial number for a product that contains sub-serialized products, the sub-serialized products are updated also. Refer to Setting up Sub-Serial Numbers.
Menu Path: Shop Repair > Shop Repair Order Entry > Action Bar > Ovr Serial Num
Field/Function Key Descriptions
Field/Function Key
Description
Model Number
The model number to enter a serial number for.
Serial Number
The serial number associated with the model.
F2-Save
Saves the serial number information.
