Generating Transfers for a Shop Repair Order
If a product on a shop repair order is backordered, you can automatically generate a transfer if:
•
T/R From S/O in your user ID is set to Y.
•
Auto Transfer Generation in Inventory Parameters > F5-Add Info is set to Y or W.
•
At least one line item has a supply code (Sc) of T or R. Sc is automatically set to T for non-stock products with a Replenishment Path of T. For stock products, you must manually set Sc to T or R.
If the Automate Transfer Generation flag in Purchasing Option Parameters is set to Y, the system automatically generates the transfers without the following prompts.
If the above requirements are met, a dialog box with the following message appears when a shop repair item is backordered:
Do You Wish To Automatically Generate
A Transfer For This Product?
1
Choose one of the following:
•
Yes - Generate the transfer at the end of the order.
•
No - Allow the transfer to generate as part of Nightly Processing.
2
If you select Yes, the Auto Transfer Generation screen appears. You can view the inventory status of the product in all warehouses.
3
Enter the Shipping Warehouse and press ENTER.
4
Press F2-Save.
5
After you finish entering the rest of the products on the order, select No Action or Print (refer to Saving the Shop Repair Order) to generate the transfer. A dialog box with the following message appears:
Do You Wish To Proceed WIth Transfer Update?
6
Choose one of the following:
•
Yes - Immediately generate the transfer.
•
No - Allow the transfer to generate as part of Nightly Processing.
7
If you select Yes, the Generate Stock Transfers screen appears.
8
Choose from the following:
•
F2-Generate All: Generates transfers for the warehouses listed. Transfer number(s) appear in T/R#.
•
F5-Edit: Allows you to edit and the transfer that is selected.
•
F6-Generate: Generates one transfer at a time for the selected warehouse.
9
When the transfers are generated, press F9-Continue. A dialog box with a message similar to the following appears:
Current Warehouse is from 70 to 10. Do You Wish To...
10
Choose one of the following:
•
Print to print the transfer pick ticket.
•
Suspend to complete the order later. The transfer will not print and you must go into the transfer again to release it.
