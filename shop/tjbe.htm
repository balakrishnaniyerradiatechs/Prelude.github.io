Technician Job Billing Entry screen
In Technician Job Billing Entry, the technician or a supervisor can enter all time worked on a job or jobs for a single day. Because all the time worked during the day is entered at once, you cannot invoice orders until Technician Job Billing Entry is completed for all technicians.
If you use this method, labor costs are added to the order when Technician Job Billing Entry is complete. The charges are based on Technician Shop Rate Maintenance. To use this method, you must also set up shop labor types and technician shop rates.
For more information, refer to Billing Jobs for Technicians.
Menu Path: Shop Repair > Shop Repair Time Clock > Technician Job Billing Entry
Field/Function Key Descriptions
Field/Function Key
Description
Technician#
The code for the technician clocking into the job. For more information, refer to Setting up Technicians.
Date
The date the labor was performed.
Time (HH:MM)
The total number of hours the technician was clocked into jobs for the day.
Document/Job Code
The shop/repair order that was updated with labor hours.
Job
The job number that was updated with labor hours.
Description
The description of the customer or job code.
Product Number
The labor codes set up as service items in Shop Labor Type Maintenance. For more information, refer to Identifying Shop Labor Types.
Type
The type of shop/repair labor hours:
•
R - Regular time.
•
O - Overtime.
•
P - Premium.
Hours
The total time worked. If you enter a value here, Start Time and End Time are calculated automatically.
Start Time
The time the technician started work on the job.
End Time
The time the technician ended work on the job.
Time
The total time the technician worked on the job.
Total Time
The total number of hours for all jobs.
F2-Save
Saves the timeclock entries.
