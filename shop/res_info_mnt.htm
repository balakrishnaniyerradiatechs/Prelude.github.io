Resource Information Maintenance screen
The Resource Information screen allows you to override and start and stop times for any individual resource.
Refer to Identifying Available Hours for Resources.
Menu Path: Shop Repair > Processes > Scheduling Calendar Resource Maintenance > F5-Resource Info
Field/Function Key Descriptions
Field/Function Key
Description
Resource Type
The resource type (refer to Defining Available Resource Types) that identifies this group of resources.
Warehouse
The warehouse where the resources exist.
Resource
The resource that you are entering working hours for.
Start/End
Days of the week
The start and end time for each day of the week that this resource is available.
Lunch
The default start and end time that this resource will be on lunch break.
AM Break
The default start and end time that this resource will be on morning break.
PM Break
The default start and end time that this resource will be on afternoon break.
F2-Save
Saves the operating hour information for the resource.
F4-Del
Deletes the resource information.
