Saving a Quote
To save a quote:
1
Press F2-Save. The Shop Repair Recap Information screen appears. The screen lists each job on the quote.
2
To edit a job, press F5-Edit. Refer to Entering a Job.
3
To add shop charges to the quote, from the Action Bar, choose Calc. Charges (for more information on shop charges, refer to Creating Shop Charges) and the following message appears:
Additional shop charges calculated. Do you wish to generate a job for charges?
4
Choose Yes. The Shop Repair Job Entry screen appears. Refer to Entering a Job.
If Auto Shop Charges in Shop Repair Options is set to Y (refer to Setting Shop Repair Parameters), the above message does not appear. The shop charge is automatically created for each job on the order and is determined by the pay type assigned to the job. Refer to Creating Pay Types.
5
If you want to review the price, cost, profit, and GP% for labor, parts, and miscellaneous charges; from the Action Bar, select Recap Rpt to run the Shop Job Recap Report.
6
When the job information is correct, press F2-Save. A prompt appears with the following choices:
•
No Action - Saves the changes without printing any documents.
•
Print - Allows you to choose which documents to generate (refer to Printing Quote Tickets for a description of each document):
-
Pickup Document - Enter Y to print a pick up document. You can also enter pickup instructions. Refer to Printing Pickup Tickets.
-
Pickup Date - Enter the date on which the item(s) should be picked up. Defaults to the date entered in Pick Up in the Shop Repair Information screen.
-
Delivery Document - Enter Y to print a delivery document. You can also enter delivery instructions. Refer to Printing Delivery Tickets.
-
Delivery Date - Enter the date on which the item(s) should be delivered to the customer.
-
Print Quote - Enter Y to print the quote along with the pickup and delivery documents.
•
Suspend - Saves the quote but does not flag it to be printed.
•
Order - Converts the quote to a shop repair order. The Shop Repair Information screen appears. Refer to Starting or Editing a Shop Repair Order.
•
Complete - Finishes the quote and adds fees and taxes (refer to Adding Fees and Taxes).
•
Return - Goes back to the Quote Repair Order Entry screen.
You may receive warning messages that the order was placed on Admin Quote Hold.
