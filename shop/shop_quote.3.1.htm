Shop Quote Subsystem
The Shop Quote Subsystem allows you to quote repair costs to a customer for a product model and serial number without having to create a shop repair order. Refer to the following sections for information on quotes.
•
Entering a Shop Repair Quote.
•
Printing Quote Tickets.
•
Shop Repair Quote Inquiries.
•
Open Repair Quote Report.
If you only want to view information about a shop repair quote, use Shop Repair Quote Inquiry. If you use Shop Repair Quote Entry, you must save the quote, even if you do not change anything. These extra saves create extremely long log files and make tracking activity on the quote difficult. Refer to Shop Repair Quote Inquiries for more information on inquiries.
