Service Bulletin Inquiry
The Service Bulletin Inquiry displays product alert and repair information published by a manufacturer for a product. Refer to Entering Service Bulletins for information on Service Bulletin Maintenance.
Viewing Service Bulletins
To view published Service Bulletin Information:
1
Select Shop Repair > Inquiries > Service Bulletin Inquiry to open the Service Bulletin Inquiry screen.
2
Enter the Bulletin Number.
The service bulletin information appears.
Field/Function Key Descriptions
Field/Function Key
Description
Bulletin Number
The bulletin number from the vendor.
Entry Date
The date the bulletin was entered.
Description
A brief description of the bulletin.
Issue Date
The date the bulletin goes into effect.
Completion Date
The date the service bulletin should be completed.
Internal Date
The internal date that your company wants to complete all service work on this bulletin.
Alert Level
Indicates the type of bulletin:
•
1 - Safety re-work (the repair is critical before someone gets hurt).
•
2 - Repair.
•
3 - Repair bulletin.
•
4 - Information bulletin.
Pay Type
The pay type code that indicates how the job will be billed (e.g., CB, WTY). For more information on pay types, refer to Creating Pay Types.
Repair Group
The repair group code to identify the type of repair job. Refer to Identifying Repair Groups.
Repair Time
The published or expected amount of time it should take to complete the repair.
Location Filed
The physical location where the bulletin is filed. You may want to enter this so that you do not have to type all of the bulletin information into the system.
OK
Closes the screen.
F5-Products
Opens the Service Bulletin Product Information screen which lists model and serial numbers for this bulletin. When you enter an order for one of these assigned products, you have the option to review the service bulletin.
F6-Situation/
Correction
Allows you to view any information entered about the problem and the solution.
