Adding Comments
Always insert comments after the line item they are related to. Comments inserted after a line item are always linked to that line item and print on documents according to their comment code.
To add comments to a line item:
1
Move the cursor to a blank line after the related line item.
If the item is not the last line on the order, move the cursor to the line below where you want the comment to appear and press Cntrl and Insert at the same time to create a blank line.
2
Enter the appropriate Comment Code (refer the Comment Code Definitions).
3
Press ENTER. The Comment Description Entry screen appears.
4
Enter a Description.
5
Press F2-Save. Continue entering line items and jobs as necessary.

If a standard description exists, enter the standard description number/name in <std desc>. For example, if RGA is the name of your standard message and you want the comment to print on all documents (C comment code), enter $C.RGA to pull in the standard description of RGA and print it on all documents.
