Printing Pick Tickets
A pick ticket lists each product on the order that is available to pick.
If you continuously print pick tickets, you do not need to use this procedure.
To batch print pick tickets:
1
Select Order Processing > Batch Ticket Printing > Pick Ticket Printing to open the Pick Ticket Printing screen.
2
Enter the Warehouse for which you want to print pick tickets.
After you enter a Warehouse Number, you can press F5 to reprint a pick ticket. Enter the Release Number and Printer Form Queue and press F2.
3
For Order Number, choose one of the following:
•
To print a single pick ticket, enter the shop repair order number.
•
To print all pick tickets, leave this field blank.
4
For Customer, choose one of the following:
•
To print pick tickets for a customer, enter the customer number.
•
To print all pick tickets, leave this field blank.
5
For Printer Form Queue, enter your UNIX printer number.
6
Press ENTER. The Load Pick Ticket Forms dialog box appears.
7
Choose one of the following:
•
Print Pick Tickets - Prints the pick tickets and creates releases.
•
Mask Print Test - Tests the alignment of the pick ticket forms in the printer. Not necessary if you are using a laser printer. After the forms are correctly aligned, select Print Pick Tickets.
•
Abort Printing - Cancels. No releases generate.
8
After you select Print Pick Tickets, the Print Confirmation box appears.
9
Choose one of the following:
•
Continue - All pick tickets printed successfully.
•
Reprint All - No pick tickets printed successfully. This option is also useful if only a couple of pick tickets printed successfully from a large print job.
•
Single Reprint - Reprints pick tickets individually. This option is useful if a couple of pick tickets jammed in the printer.
The most common reasons that a pick ticket does not print is if:
•
The order is on hold.
•
The order is suspended (refer to Saving the Shop Repair Order).
•
The products are not available.
•
The order has a future pick date.
