History Shop Repair Inquiry
The History Shop Repair Inquiry screen lists invoiced orders for a customer.
Viewing Invoiced Shop Repair Orders
To list historical shop repair orders:
1
Select Shop Repair > Inquiries > History Shop Repair Inquiry to open the History Shop Repair Inquiry screen.
2
Enter the Customer Number or keyword.
3
Choose a Sort option:
•
Order number.
•
Order date.
•
Required date.
•
Serial number.
•
Model number.
•
Sales order number.
4
Choose a Filter option:
•
No.
•
Yes - use the following selection criteria:
-
Model number.
-
Serial number.
-
Required date range.
-
Order number.
-
Cust release#.
-
Customer P/O#.
-
Repair groups.
Invoiced shop repair orders for the customer appear.
Field/Function Key Descriptions
Field/Function Key
Description
Order Number
The shop repair order number.
Notes
An asterisk (*) appears if notes were entered for the order.
Invoice Number
The invoice number for the order.
Invoice Date
The date the order was invoiced.
Model Number
The main part that was repaired on the order.
Serial Number
The serial number of the part that was repaired (if one was entered).
Invoice Amount
The total price that the customer paid for the order.
F2-Finish
Close this screen.
F5-Review
Open the Shop Repair Order Display screen to allow you to view detail information for the order.
F8-Log
Open the Order Log Inquiry screen for the order.
F9-Goto Line
Identify a reference number on the screen that you want to jump to. Valid codes are order number, page number, line number, B (bottom), or T (top).
