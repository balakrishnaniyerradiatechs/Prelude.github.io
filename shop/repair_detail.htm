Repair Detail screen
The Repair Detail screen allows you to view detail schedule information for each day and to enter or edit appointments for the resources available.
Refer to Scheduling Appointments by Day.
Menu Path: Shop Repair > Processes > Scheduling Calendar Entry > View = D
Field/Function Key Descriptions
Field/Function Key
Description
From
The beginning time frame of scheduled appointments.
To
The ending time frame of scheduled appointments.
Tech
The technician that is scheduled.
Document
The document number that the technician is assigned to.
Date
The date the technician is scheduled.
Time
The time the technician is scheduled.
PUD
The pick-up date. The date the product must be picked up from the customer.
Prom
The date the repair was promised to be completed.
LCD
The last contact date with the customer.
W/B
The employee that entered the order (written by).
T
Type of billing
•
C - Customer will be billed.
•
W - The product is under warranty.
•
I - Internal billing, no charge to the customer.
S
The status of the appointment:
•
O - Open.
•
C - Closed.
W
An asterisk (*) indicates that work has begun on the job.
Description
The customer description
Model
The product being repaired
F2-Save
Closes the screen.
F5-New Appt
Opens the Shop/Repair Appointment Entry screen to allow you to enter a new appointment for the technician.
F6-Edit Appt
Opens the Shop/Repair Appointment Entry screen to allow you to edit an appointment for the technician.
F9-Goto Line
Allows you to enter a line number, page number, reference number to go directly to.
