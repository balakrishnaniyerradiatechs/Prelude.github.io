Scheduling Conflict Report
The Scheduling Conflict report lists schedule information for any situation where a resource is scheduled for more than one job/appointment at a time.
Menu Path: Shop Repair > Processes > Calendar Scheduling > Scheduling Conflict Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where the resources are scheduled.
•
Resource Type. The resource types to include. Choose from the following:
-
D - Delivery.
-
O - Outside source.
-
P - Pickup.
-
R - Repair.
-
S - Setup.
-
U - Order pulling.
-
W - Work Center.
•
Resource. The resources within the resource type to include. For example, if Resource Type is Repair, enter the technicians to include in the report.
•
Date Range. The schedule date range to include.
