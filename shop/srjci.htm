Shop Job Code Inquiry
The Shop Job Code Inquiry displays information for job codes set up in Shop Job Code Maintenance. Refer to Setting Up Job Codes.
Viewing Job Codes
To view job code information:
1
Select Shop Repair > Inquiries > Job Code Inquiry to open the Shop Job Code Inquiry screen.
2
Enter the Job Code (e.g., OC).
3
Enter the Model Number, if applicable
Information about the job code/model number appears.
Field/Function Key Descriptions
Field/Function Key
Description
Job Code
The code for the job (e.g., OC). Up to 10 alphanumeric characters.
Model Number
If this job applies to a single model number, the applicable model number.
Job Description
A description of the repair job.
Pay Type
The pay type code that indicates how the job will be billed (e.g., CB, WTY). For more information on pay types, refer to Creating Pay Types.
Repair Group
The repair group code to identify the type of repair job. For more information, refer to Identifying Repair Groups.
Repair Amount
If applicable, the amount that should be charged regardless of the price of parts and labor. If this field is blank the price is the total price of parts and labor.
Repair Amount Type
Indicates what costs are included in the Repair Amount:
•
L - Labor.
•
P - Parts.
•
B or null - Both labor and parts (default).
Est. Repair Time
The total estimated repair time for the job.
Tech
If a technician should be assigned to this job, the code for that technician. The technician must be authorized to work on the type of repair indicated in Repair Group. For more information on technicians, refer to Setting up Technicians.
Est. Hrs
If a technician is assigned to the job, the estimated time the technician should spend working on the job.
F2-Save
Closes the screen.
F5-Problem
Displays the problems notes screen.
F6-Solution
Displays the solutions notes screen.
F7-Products
Opens the Shop Job Code Product screen. Displays products entered for the job.
F8-Comments
Opens the comments screen.
Action Bar
Copy
Not applicable.
