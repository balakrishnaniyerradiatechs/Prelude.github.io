Sales Order Header screen
The Sales Order Header screen allows you to define order-level information, such as required date, written by, and salesrep information. Some of this information can be overridden in the Line Item Detail screen.
Menu Path: Shop Repair > Shop Repair Order Entry > F5-Hdr
Field/Function Key Descriptions
Field/Function Key
Description
Order Type
The type of order. Choose from:
•
01 – Regular.
•
02 – Ship complete.
•
03 – Tag and hold.
•
04 – Direct order.
•
05 – Over the counter.
•
06 – Confirmation.
•
07 – Consignment.
•
08 – Confirmation consignment.
•
09 – RGA.
•
10 – Repair.
•
11 – Lease.
•
12 – Invoice when complete.
•
13 – Ship backorder complete.
•
14 – Ship line item complete.
•
15 – Show.
•
16 – Order projection.
•
17 – Manufacturer rep.
•
18 – Crib replenishment.
For more information on order types, refer to the order processing documentation.
Entry
The date the order was entered.
Purchasing Contact
The purchasing agent from the company that is placing the order. Only available if your company uses CRM.
Ship-To Number
The ship-to address for this order. Choose from:
•
D - Use default address from Customer Maintenance.
•
P - Customer will pick up at your warehouse.
•
N - Create new ship-to address. The address is stored in Customer Ship-To Maintenance and may be used on another order.
•
M - Enter a new ship-to address for this order only.
•
C - Use the purchasing contact address
•
<#> - Use existing ship-to address.
Written By
The user ID of the person taking the order.
Sell Warehouse
The warehouse that will receive credit for the sale.
Ship Warehouse
The warehouse that will ship the products.
Ship Via Code
The code that represents how the order will be shipped (e.g., 01 - UPS, 02 - FedEx).
Contact
The customer’s A/R contact person.
Email Address
The email address for the customer’s A/R contact person.
Customer P/O#
The customer’s purchase order number.
Cust Release#
The release number the customer provides in addition to or instead of a P/O number.
Job Number
The customer’s job number.
Order Date
The date the order should is entered. Defaults to current date.
Required Date
The date the customer requires the products.
Pick Ticket Date
The date the products should be picked.
Instructions
Any special instructions to appear on the pick ticket and invoice.
F2-Save
Saves the information.
F5-Add Info
Opens the Order Additional Information screen.
F6-Ship-To
Opens the Ship-To Address screen
F7-Tax Jur
Opens the Tax Jurisdiction screen.
F8-Credit Info
Opens the Credit Information screen.
Action Bar
Repair Type
Not applicable for shop repair quotes.
Consign Type
Not applicable for shop repair quotes.
Fut Commit
Allows you to override future commitment for products on this order. Required products are taken out of current stock.
Rent Info
Opens the Rental Information screen. Only applicable if you are using the Rental Module and you generated a shop repair directly from a rental order.
Currency
Allows you to change the exchange rate.
