Shop Quote History Inquiry screen
The Shop Quote History Inquiry screen displays information about the quote including line item quantities, prices, and costs for the quote.
Menu Path: Shop Repair > Shop Quote Subsystem > Quote Repair History Inquiry > F5-Review
Field/Function Key Descriptions
Field/Function Key
Description
Quote Number
The number of the shop repair quote.
Note (not labelled)
An asterisk (*) indicates if header notes were entered for the quote. To view or enter header notes, choose Action Bar > Notes > Header Notes.
Customer
The number of the customer who is having a product repaired.
Ln#
The line number.
Note (not labelled)
An asterisk (*) indicates if notes were entered for the line. To view or enter line notes, choose Action Bar > Notes > Line Notes.
Product Number
The number of the product or the pay type for a job. For more information on entering products, refer to Adding or Editing Line Items. For more information on entering jobs, refer to Creating the Job.
Quantity
The quantity of the product required to complete the job.
U/M
The product’s unit of measure (e.g., EA, DZ).
Gross Price
The gross price for the product.
Disc%
The discount percentage.
Net Price
The price of one unit of measure of the product, less the discount.
Description
A description of the product or job.
Conv
Indicates if the quote was converted to a sales order:
•
Y - The quote was converted.
•
N - The quote was not converted.
Whse
The selling warehouse.
Customer Product#
The customer’s number for this product.
Job
The number of the job (e.g., A, B, C). If the line is a product, indicates the job associated with the product.
Tx
Indicates if the product is taxable:
•
Y - Product is taxable.
•
N - Product is not taxable.
AQH
If the line is on admin quote hold, the code that indicates the reason for the hold appears.
Src
The source of the price.
Profit
The profit for the product.
GP%
The gross profit percentage for the product.
Extension
The total price for the product.
Totals (not labelled)
Profit
The total profit for the quote.
Gp%
The gross profit percent for the quote.
Q/O Amount
The total price of the quote.
F2-Finish
Closes the screen.
F5-Hdr Scr
Opens the Shop Repair Information screen. For more information on entering header information for an quote, refer to Starting or Editing a Shop Repair Quote.
F6-Comp Scr
Opens the Sales Order Completion screen. For more information, refer to Adding Fees and Taxes.
F7-Job
Opens the Shop Repair Recap Information screen to allow you to view a summary of the job.
F8-Log
Opens the Quote Log Inquiry screen.
F9-Goto Line
Allows you to identify a reference number on the screen that you want to jump to. Valid codes are page number, line number, B (bottom), or T (top).
Action Bar
Examine
Displays detailed information for the highlighted line item.
Notes - Header Notes
Displays header notes, if available.
Notes - Line Notes
Displays line notes, if available.
Notes - Service Problem Notes
Displays notes about the problem with the part being repaired, if available.
Notes - Service Solution Notes
Displays notes about how the repair was completed, if available.
