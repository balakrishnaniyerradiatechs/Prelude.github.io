Scheduling Calendar Resource Type Maintenance screen
Before you can schedule technicians and delivery trucks for shop repair jobs, you must use the Scheduling Calendar Resource Type Maintenance screen to identify the types of resources you want to schedule (e.g., delivery trucks, repair technicians). You can set up a different resource type for each supervisor so he/she can schedule an independent group of resources.
For more information, refer to Defining Available Resource Types.
Menu Path: Shop Repair > Processes > Scheduling Calendar Resource Type Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Resource Type
The 1 or 2 alpha/numeric code that identifies the type of resource you will schedule (e.g., R for repair technicians, D for delivery trucks). You can set up multiple groups of resource types so that different supervisors can maintain schedules for different groups of technicians (e.g., R1, R2, R3).
Description
The description of the resource (e.g., delivery truck).
Type
The type of schedule that this resource is for. Press F3 to view the available types. These are set up by Prelude and determine what the resource is validated against when you schedule resources (e.g., the Shop/Repair type is validated against Shop Repair Technicians).
F2-Save
Saves the resource type information.
F4-Del
Deletes the resource type.
