Copying a Job Code
If you need to create new job code that is similar to an existing job code, you can copy the existing job code and then edit it.
To copy a job code:
1
Select Shop Repair > Maintenance > Job Code Maintenance to open the Shop Job Code Maintenance screen.
2
From the Action Bar, choose Copy to open the Shop Job Copy screen.
3
For From Job Code, enter the code for the existing job that you want to copy.
4
For To Job Code, enter the code for the new job code that you want to create.
5
If the new job code is associated with a product, enter the product number in Model Number.
6
Enter a brief description of the new job code in Job Description.
7
Press F2-Save.
