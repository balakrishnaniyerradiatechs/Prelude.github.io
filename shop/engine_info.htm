Engine/Transmission Information screen
The Engine/Transmission Information screen allows you to enter detailed information about a model/serial number combination. This data includes the make and model of the engine and/or transmission and a list of parts that are attached to the main part.
Menu Path: Shop Repair > Maintenance > Model Information Maintenance > Serial
Field/Function Key Descriptions
Field/Function Key
Description
Model
The part number and description for which you are entering data.
Serial
The serial number for which you are entering data.
Engine Info, Transmission Info, Other Info
Make
The name of the make of the engine, transmission, or other part on the model/serial number combination.
Model
The model number of the engine, transmission, or other part on the model/serial number combination.
Spec
Engine specifications.
Serial
The serial number of the engine, transmission, or other part on the model/serial number combination.
Attachment Model
The part number of the model that is attached to the main model/serial number combination.
Description
The description of the attachment model.
Serial Number
The serial number of the model that is attached to the main model/serial number combination.
Quantity
The quantity of items that are attached.
F2-Save
Saves the damage code information.
F5-Edit
Allows you to edit the information.
