Inspections screen
The Inspections screen within Sub-Serial Number Maintenance, allows you to enter inspection codes required for the serialized item. The inspection codes are set up in Inspection Code Maintenance.
For more information, refer to Setting Up Inspection Codes.
Menu Path: Shop Repair > Maintenance > Inspection Code Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Codes
The inspection code to assign to the sub-serialized product number.
Description
The description of the inspection code.
Srv
Indicates if the inspection code is for a serviceable item:
•
Y - The inspection code is for a serviceable item.
•
N - The inspection code is not for a serviceable item.
Ref#
Any reference number, if applicable.
F2-Save
Saves the inspection information for the sub-serialized product.
