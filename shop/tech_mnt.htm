Technician Maintenance screen
If you use Technician Job Billing Entry to record time clocked for a job, you must use Technician Maintenance to set up labor costs and authorized repair groups for each technician. The labor costs are automatically calculated and added to the job based on the shop labor type chosen when the clocked hours are entered.
Menu Path: Shop Repair > Maintenance > Technician Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Technician
The code for the technician for which you want to establish labor rates and repair groups.
Description
The technician’s name.
Address One
The first line of the technician’s home address.
Address Two
The second line of the technician’s home address.
Address Three
The third line of the technician’s home address.
City
The technician’s home city.
State
The technician’s home state.
Zip
The technician’s zip code.
Phone
The technician’s home phone number.
Fax
The technician’s fax number.
Pager
The technician’s pager number.
Cellular
The technician’s cell phone number.
Email Address
The technician’s email address.
Regular Time Rate
The hourly labor cost for regular time in Technician Job Billing Entry.
Overtime Rate
The hourly labor cost for overtime in Technician Job Billing Entry.
Premium Rate
The hourly labor cost for premium time in Technician Job Billing Entry.
F2-Save
Saves the information for this technician.
F4-Del
Deletes the technician record.
F5-Repair Skill Set
Opens the Shop Repair Skill Set screen to allow you to enter the repair groups that the technician is authorized to work on.
