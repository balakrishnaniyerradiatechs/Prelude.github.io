Shop Job Copy screen
If you need to make a new job code that is similar to an existing job code, you can copy the existing job code and edit it. The Shop Job Copy screen allows you to specify which code you want to copy and the name of the new code.
Menu Path: Shop Repair > Maintenance > Job Code Maintenance > Copy
Field/Function Key Descriptions
Field/Function Key
Description
From Job Code
The code for the job to copy (e.g., OC). Up to 10 alphanumeric characters.
Model Number
If this job applies to a single model number, the applicable model number.
Job Description
A description of the repair job.
To Job Code
The code for the new job (e.g., OC). Up to 10 alphanumeric characters.
Model Number
If this job applies to a single model number, the applicable model number.
Job Description
A description of the repair job.
F2-Save
Copies the job code information.
