If you only want to view information about a shop repair order, use Shop Repair Order Inquiry. If you use Shop Repair Order Entry, you must save the order, even if you do not change anything. These extra saves create extremely long log files and make tracking activity on the order difficult. Refer to Shop Repair Inquiries for more information on inquiries.
