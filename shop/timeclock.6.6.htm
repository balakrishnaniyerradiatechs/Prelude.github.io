Identifying Shop Labor Types
Before you can use Technician Job Billing Entry to enter time clocked into a job, you must set up shop labor types (e.g., shop labor, field labor). When a technician enters time against a job, a shop labor type must be specified (each shop labor type must be set up as a product in Product Maintenance). Labor charges are automatically entered on the shop repair order based on the shop labor type entered and the technician’s rate (refer to Setting up Technicians for more information on shop rates).
To set up shop labor types:
1
Select Shop Repair > Maintenance > Shop Labor Type Maintenance to open the Shop Labor Type Maintenance screen.
2
For Default Product, enter the product number that represents the shop labor type that you would like to default into Technician Job Billing Entry.
This product must be already be set up in Product Maintenance as a service product.
3
For Product, enter a product number to use as a shop labor type in Technician Job Billing Entry.
4
Determine which Type of pay rate should calculate labor cost (refer to Setting up Technicians for more information on rates):
•
R - Regular.
•
O - Overtime.
•
P - Premium.
5
Repeat steps 3-4 for each product number that can be used as a shop labor type.
6
Press F2-Save.
