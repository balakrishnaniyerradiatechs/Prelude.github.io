Identifying Repair Groups
A repair group identifies the type of repair that a part requires (e.g., maintenance, steering). Repair groups are used in the following five places:
•
Sales Rep Maintenance - Identifies which types of repairs a technician is allowed to work on. Refer to Setting up Technicians.
•
Product Maintenance - For products with warranty work, specifies what type of repair the product needs. Refer to Entering Standard Repair and Setup Time for Products.
•
Service Bulletin Maintenance - For a service bulletin, establishes the type of repair that the service bulletin requires. Refer to Entering Service Bulletins.
•
Shop Repair Order Entry - In the Shop Repair Job Entry screen, identifies which type of repair will be completed for the job. Refer to Creating the Job.
•
Shop Repair Job Code Maintenance - In the Shop Job Code Maintenance screen, defines the type of repair that will be completed for a job. Refer to Setting Up Job Codes.
To create a repair group:
1
Select Shop Repair > Maintenance > Repair Group Maintenance to open the Repair Group Maintenance screen.
2
For Repair Group, enter a new code (up to 6 alphanumeric characters) to represent a repair group.
3
Enter a brief Description of the repair group (e.g., maintenance, steering).
4
Press F2-Save.
