Quote Repair Order Entry screen
The Quote Repair Order Entry screen is used when:
•
Starting or Editing a Shop Repair Quote.
•
Entering a Job.
•
Adding or Editing Line Items.
Menu Path: Shop Repair > Shop Repair Quote Subsystem > Shop Repair Quote Entry
Field/Function Key Descriptions
Field/Function Key
Description
Quote Number
The number of the shop repair quote.
Note (not labelled)
An asterisk (*) indicates if header notes were entered for the order. To view or enter header notes, choose Action Bar > Notes > Header Notes.
Customer
The number of the customer who is having a product repaired.
Product
The number of the product or job. For more information on entering products, refer to Adding or Editing Line Items. For more information on entering jobs, refer to Creating the Job.
Description
A description of the product or job.
Notes
An asterisk (*) indicates if notes were entered for the line. To view or enter line notes, choose Action Bar > Notes > Line Notes.
RFQ
An asterisk (*) indicates if the quote generated a vendor request for quote.
AQI
If the line is on admin quote hold, the code that indicates the reason for the hold appears.
Conv
Indicates if the quote was converted to a sales order:
•
Y - The quote was converted.
•
N - The quote was not converted.
Job
The number of the job (e.g., A, B, C). If the line is a product, indicates the job associated with the product.
Quantity
The quantity of the product required to complete the job.
U/M
The product’s unit of measure (e.g., EA, DZ).
Whse
The warehouse where the product is located.
Price
The price of one unit of measure of the product.
Totals
Quantity
The total quantity of products on the quote.
Weight
The total weight of the products on the quote (if weight information is available).
Profit
The total profit for the quote.
Gp%
The gross profit percent for the quote.
Dollars
The total price of the quote.
Function Buttons Before an Order Number is Chosen
F5-Startup
Opens the Startup Parameters screen.
Function Buttons and Action Bar After an Order Number is Chosen but Before a Customer is Chosen
F7-Conv Prospect
Opens the Prospect Conversion screen so you can convert a prospect to a customer and then use that customer for the order.
F8-Enter New Customer
Opens the Temporary Customer Maintenance screen so you can enter information about a new customer.
F9-Conv EDI
Opens the EDI Conversion screen. Not applicable for shop repair orders.
Action Bar
Dup Quote
Opens the Quote Duplication Entry screen so you can copy an open order for the same customer or a different customer.
Dup History Quote
Opens the Quote History Duplication screen so you can copy an invoiced order for the same customer or a different customer.
Function Buttons and Action Bar After an Order Number and Customer are Chosen
F2-Save
Saves the order.
F4-Del
Deletes the order.
F5-Hdr Scr
Opens the Shop Repair Information screen. For more information on entering header information for an order, refer to Starting or Editing a Shop Repair Quote.
F6-Totals
Opens a screen that allows you to edit the totals and set the order up for lot billing.
F7-Job
Opens the Shop Repair Job Entry screen and allows you to create a new job or edit an existing job. For more information on entering jobs, refer to Entering a Job.
F9-Goto Line
Allows you to identify a reference number on the screen that you want to jump to. Valid codes are page number, line number, B (bottom), or T (top).
Action Bar
Examine
Opens a screen that allows you to see detail information for the product that your cursor is on.
Review
Opens the Shop/Repair Quote Review screen, which allows you to see detailed information for each line on the order.
ProdFunc - Add Comment Lines
Opens the Comments Description Entry screen, which allows you to enter comments. For more information on entering comments, refer to Adding Comments.
ProdFunc - Line Item Split
Opens the Order Line Split screen, which allows you to separate an existing line into different required dates.
ProdFunc - Rental Unit Sold
Not applicable for shop repair quote.
ProdFunc - Buyer Message
If the product is linked to a purchase order, allows you to enter a message for the buyer.
OrderFunc - Reprice Order Options
Allows you to choose a method to reprice each line on the quote:
•
Customers Regular Price - the customer’s usual price column, matrix, or contract.
•
Gross Margin Percentage - the gross margin percent for the order.
•
Less Discount - the percent discount off of suggested retail, the customer’s default price column, or the current price.
•
Cost Plus - the amount you want to add to cost.
•
Minimum Gross Margin - recalculate all prices to the product line minimum.
•
Column Pricing - a new price column for all items.
•
Sum and Reprice - re-evaluate all products for quantity breaks.
•
Pricing By Square Footage - not applicable for shop repair orders.
OrderFunc - Manual Administrative Order Hold
Puts the quote on administrative order hold.
OrderFunc - Manual Administrative Invoice Hold
Puts the quote on administrative invoice hold.
OrderFunc - Rate Shop
Not applicable for shop repair quotes.
OrderFunc - Campaign Priority Code
Not applicable for shop repair quotes.
OrderFunc - Allocated Inv Review
If any products are available for reallocation, opens the Backorder Reallocation Availability screen. You can use this screen to determine if you can reallocate stock from one quote to another.
OrderFunc - Order Line Sort Options
Reorders lines using one of the following methods:
•
Customer product number.
•
Product description.
•
Product number.
OrderFunc - Import RF Batch Order Lines
Allow you to import radio frequency line items for 05 & 06 order types.
JobFunc - Sort Lines By Job Number
Regroups lines so that all products associated with a job appear with that job.
JobFunc - Change Job Number
Moves a product from one job to another.
JobFunc - Move To Another Job
Moves a product to a job on a different quote.
Notes - Header Notes
Allows you to enter notes for the order.
Notes - Line Notes
Allows you to enter notes for the line that your cursor is on.
Notes - Conversion Notes
Not applicable for shop repair quotes.
Notes - EDI Line item Notes
Not applicable for shop repair quotes.
Notes - EDI Header Notes
Not applicable for shop repair quotes.
