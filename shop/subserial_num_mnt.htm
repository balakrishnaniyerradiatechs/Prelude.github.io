Sub-Serial Number Maintenance Screen
The Sub-Serial Number Maintenance screen allows you to assign sub-serial numbers to parts that make up a main serialized product. For example, the main product might be a car and the sub-products are the parts being repaired. When you use the Override Serial Number function in Shop Repair Job Entry to update the sub-serial number (refer to Creating the Job), the system updates the sub-serial number along with the main serial number for the order.
For more information, refer to Setting up Sub-Serial Numbers.
Menu Path: Shop Repair > Maintenance > Sub-Serial Number Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Product
Line 1: The main product number for the serialized being repaired.
Line 2: The description of the main product number.
Line 3: The extended description of the main product number if applicable.
Serial Number
The serial number of the item being repaired.
Warehouse
The warehouse where the repair is taking place.
Customer
The customer set up to use the main product number.
Ship-To Number
The ship-to number for the customer.
Product
The serialized item considered a sub item of the main product number.
Description
A description of the product number.
Serial Number
The serial number for the sub-product.
Reference #
A free form reference number for serialized items.
M
Indicates if multi-level inspections are required.
F2-Save
Saves the sub-serial number information.
F5-Inspections
Open the Inspections screen to allow you to enter inspection codes for the main product number.
F6-Notes
Opens the Notes screen to allow you to enter notes for the main product number.
F7-Rev Serial
Opens the Sub-Serial Review screen to display detail information for the sub-serialized product.
