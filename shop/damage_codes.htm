Damage Code Maintenance screen
The Damage Code Maintenance screen allows you to setup a code to describe how a product may be damaged. The damage code is used to enter warranty information for a job.
Menu Path: Shop Repair > Maintenance > Damage Code Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Code
A code (up to 4 alphanumeric digits) that represents a type of damage (e.g., 01, 02). Used with warranty jobs.
Description
A description of the damage code (e.g., Bent, Crushed).
F2-Save
Saves the damage code information.
F4-Del
Deletes the damage code.
