Setting Up Damage Codes
When you enter warranty information for a job (refer to Entering Warranty Information for a Job), you must enter a damage code for the job. The damage code indicates how the part was damaged (e.g., bent, crushed).
To set up a damage code:
1
Select Shop Repair > Maintenance > Damage Code Maintenance to open the Damage Code Maintenance screen.
2
Enter a Code to represent the type of damage (up to 4 alphanumeric characters).
3
Enter a Description of the type of damage.
4
Repeat steps 2-3 for each code to enter or edit.
5
Press F2-Save.
