Shop/Repair Week View screen
The Shop/Repair Week View screen allows you to edit the existing appointments scheduled for the 5 days or enter new appointments for a job.
Menu Path: Shop Repair > Processes > Calendar Scheduling > Scheduling Calendar Entry > View = W
Field/Function Key Descriptions
Field/Function Key
Description
Tech
The ID of the technician whose time is scheduled.
Document
The document or schedule type assigned to the technician.
Prom
The date the repair work was promised to be completed.
T
Type of billing
•
C - Customer will be billed.
•
W - The product is under warranty.
•
I - Internal billing, no charge to the customer.
S
The status of the appointment:
•
O - Open.
•
C - Closed.
W
An asterisk (*) indicates that work has begun on the job.
Dates
The hours scheduled to this technician for each date for the next 5 days.
F2-Save
Saves the schedule information.
F5-New Appt
Opens the Shop/Repair Appointment Entry screen to allow you to enter a new appointment for a technician.
F6-Edit Appt
Opens the Shop/Repair Appointment Entry screen to allow you to edit an appointment for a technician.
F7-Prev Wk
Displays the schedule for the previous week.
F8-Next Wk
Displays the schedule for the next week.
