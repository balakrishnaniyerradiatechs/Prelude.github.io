Shop Repair Order Display screen
The Shop Repair Order Display screen appears when you choose to review a shop repair order from an inquiry screen. You can view information about the order including line item quantities, pricing, and cost and totals.
Field/Function Key Descriptions
Field/Function Key
Description
Order Number
The number of the shop repair order.
Note (not labelled)
An asterisk (*) indicates if header notes were entered for the order. To view or enter header notes, choose Action Bar > Notes > Header Notes.
Customer
The number of the customer who is having a product repaired.
Ln#
The line number.
Note (not labelled)
An asterisk (*) indicates if notes were entered for the line. To view or enter line notes, choose Action Bar > Notes > Line Notes.
Product
The number of the product or job. For more information on entering products, refer to Adding or Editing Line Items. For more information on entering jobs, refer to Creating the Job.
Description
A description of the product or job.
Job
The number of the job (e.g., A, B, C). If the line is a product, indicates the job associated with the product.
Quantity
The quantity of the product required to complete the job.
U/M
The product’s unit of measure (e.g., EA, DZ).
Qty-Pck
The quantity that is available to be picked.
Qty-Shp
The quantity shipped.
Qty-BO
The quantity back ordered.
S Whse
The selling warehouse.
Linkage
If the line is back ordered and linked to another transaction, the document number appears. The prefix indicates what type of document the line is linked to:
•
P - Purchase order.
•
T - Transfer.
•
W - Work order.
Tx
Indicates if the product is taxable:
•
Y - Product is taxable.
•
N - Product is not taxable.
Cde
An F indicates that the order has a future pick ticket date.
Scr
The source of the price.
AIH
If the line is on admin invoice hold, the code that indicates the reason for the hold appears.
AOH
If the line is on admin order hold, the code that indicates the reason for the hold appears.
Gross-Price
The gross price for the product.
Disc%
The discount percentage.
Net Price
The price of one unit of measure of the product, less the discount.
Extension
The total price for the product.
Totals
Quantity
The total quantity of products on the order.
Weight
The total weight of the products on the order (if weight information is available).
Profit
The total profit for the order.
Gp%
The gross profit percent for the order.
S/O Amount
The total price of the order.
F2-Finish
Closes the screen.
F5-Hdr Scr
Opens the Shop Repair Information screen. For more information on entering header information for an order, refer to Starting or Editing a Shop Repair Order.
F6-Cpl Scr
Opens the Sales Order Completion screen. For more information, refer to Adding Fees and Taxes.
F7-Linkage
Displays any linked documents (e.g., transfers, work orders, purchase orders).
F8-Job Info
Opens the Shop Repair Job Entry screen and allows you to view job information. For more information on entering jobs, refer to Creating the Job.
F9-Goto Line
Allows you to identify a reference number on the screen that you want to jump to. Valid codes are page number, line number, B (bottom), or T (top).
Action Bar
Examine
Displays detail information for the product that your cursor is on.
Notes
Allows you to view or edit notes for the following:
•
Header Notes. The shop repair order header.
•
Line Notes. The line the cursor is on.
The remaining notes are not applicable for shop/repair orders.
Images
Allow you to print images of documents/information, if you have the imaging software.
