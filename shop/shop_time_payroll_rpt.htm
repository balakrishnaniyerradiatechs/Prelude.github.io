Shop Timekeeping Payroll Report
The Shop Timekeeping Payroll Report uses time clock entries to show the time spent on each job, including miscellaneous and unpaid hours. Refer to Time Clock Entry & Billing.
Menu Path: Shop Repair > Reports > Shop Timekeeping Payroll Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Technician# range. The technician(s) for which you want to view productivity.
•
Date range. The beginning and ending time clock entry dates to include. These dates are entered automatically when a technician clocks in and out of a job but may be changed by a manager.
You can generate this report with totals only or with data for each day worked.
Column Descriptions
Column Heading
Description
Technician
The technician’s number and name.
Job/Code
Either the time clock code for non-job related time (e.g,. SW for start work) or the order number and job being worked on. Order number and jobs are entered using the pattern <order>!<job>. For example, job B for order 98650 would be 98650!B.
Description
A description of the time clock code (e.g., start work) or the customer number and name if an order and job number was entered.
Start
The time that the technician clocked into the job.
End
The time that the technician clocked into another job or code.
Length
The amount of time between the start and end time.
Job.Hrs.
Total time clocked into a job.
Misc.Pd.Hrs.
Time not clocked into a job that is defined as paid in Shop Timeclock Code Maintenance. Refer to Setting up Time Clock Codes.
Unpaid.Hrs.
Time not clocked into a job that is defined as unpaid in Shop Timeclock Code Maintenance. Refer toSetting up Time Clock Codes.
