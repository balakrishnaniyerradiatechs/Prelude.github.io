Scheduling Calendar Resource Maintenance screen
Once you set up your resource types (refer to Defining Available Resource Types), you can use the Scheduling Calendar Resource Maintenance screen to assign resources to each type and set up a schedule of operating hours to identify when each resource is available.
For more information, refer to Identifying Available Hours for Resources.
Menu Path: Shop Repair > Processes > Scheduling Calendar Resource Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Resource Type
The resource type (refer to Defining Available Resource Types) that identifies this group of resources.
Warehouse
The warehouse where the resources exist.
Resource
Depending on the Resource Type, the resources to assign to this type. For example, if Resource Type is Repair, enter the technician numbers (refer to Setting up Technicians). If Resource Type is Delivery, enter the truck numbers or names.
Description
The description of the resource. If you entered a technician number for Resource, the technicianâ€™s name will appear. If you entered any other type of resource, enter a description of the resource. For example, you can enter a description or designation for the truck.
Operating Hours: Start/End
These are the default operating hours for the resources for your company. You can override the hours for each resource using F5-Resource Info
Days of the week
The start and end time for each day of the week that your company is open for business
Lunch
The default start and end time that your resources will be on lunch break
AM Break
The default start and end time that your resources will be on morning break.
PM Break
The default start and end time that your resources will be on afternoon break.
F2-Save
Saves the operating hour information for your resources.
F4-Del
Deletes the resource information.
F5-Resource Info
Opens the Resource Information Maintenance screen to allow you to override the working hours for each resource.
