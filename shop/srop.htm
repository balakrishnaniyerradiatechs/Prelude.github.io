Printing Repair Tickets
The repair ticket prints information about each job and lists the products required. Extra space is provided so you can write in any missing data. External repair tickets only provide pricing for jobs with a Pay Type that has Charge Customer set to Y. Internal repair tickets provide pricing for all jobs, regardless of Pay Type.
If you are continuously printing repair tickets, do not use this process. They print immediately after you flag them to print. Refer to Saving the Shop Repair Order.
To batch print repair tickets (internal or external):
1
Select Shop Repair > Batch Printing > Shop Repair Ticket Printing to open the Shop Repair Order Printing screen.
2
Enter the following selection criteria as necessary:
•
Shop Order Number - The number of the order.
•
Division - The division number for the order(s).
•
Region -The region number for the order(s).
•
Selling Warehouse - The warehouse to receive credit for the repair.
•
Department - The department number for the order(s).
•
Cost Center - The cost center number for the order(s).
•
Customer - The customer number(s) for the orders.
•
Printer Form Queue - The printer where the repair tickets should print.
3
Press F2-Save and choose Execute. A dialog box with a message similar to the following appears:
2 Records Selected For Shop Repair Printing! Load Shop Ticket Forms In Printer & Align Visually.
4
Choose one of the following:
•
Print Tickets - Continue with printing.
•
Abort Print - Stop printing. No repair tickets print.
5
After you select Print Tickets, the Print Confirmation dialog box appears. Choose one of the following:
•
Process - All repair tickets printed successfully.
•
Reprint All - None of the repair tickets printed successfully. This option is also useful if only a couple of repair tickets printed successfully from a large print job.
•
Single Reprint - Reprints repair tickets individually. This option is useful if a couple of repair tickets jammed in the printer.
•
Abort - Cancels all repair ticket printing.
6
If you selected Process, the Process Verification dialog box appears. Choose one of the following:
•
Reprompt - Keep this dialog box open.
•
Yes - Complete repair ticket processing.
•
No - Return to step 5.
