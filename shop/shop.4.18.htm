Saving the Shop Repair Order
After you finish entering products and jobs (refer to Entering a Shop Repair Order), you must save the order.
To save a shop repair order:
1
Press F2-Save. The Shop Repair Recap Information screen lists each job on the order.
2
Choose from the following:
•
To edit a job, press F5-Edit. Refer to Creating the Job for more information.
•
To schedule a job, press F8-Schedule. Refer to the Scheduling Resources for Shop Repair Jobs for information on scheduling technicians for repair jobs.
•
To review price, cost, profit, and GP% for labor, parts, and miscellaneous charges; from the Action Bar, select Recap Rpt to run the Shop Job Recap Report.
•
To add shop charges, from the Action Bar, choose Calc. Charges (for more information on shop charges, refer to Creating Shop Charges). The following message appears:
Choose Yes. The Shop Repair Job Entry screen appears. Refer to Creating the Job.
Additional shop charges calculated. Do you wish to generate job for charges?
If Auto Shop Charges in Shop Repair Options is set to Y (refer to Setting Shop Repair Parameters), the above message does not appear. The shop charge is automatically created for each job on the order and is determined by the pay type assigned to the job. Refer to Creating Pay Types.
3
When the shop repair information is correct, press F2-Save. The following question appears:
Do You Wish To.....
4
Choose from the following:
•
No Action - Saves the changes without printing any documents.
•
Print - Opens the Shop Document Selection screen to allow you to choose which documents to generate (refer to Printing Shop Repair Tickets for a description of each document):
-
Internal Shop Ticket - Enter Y to print an internal shop ticket. Refer to Printing Repair Tickets.
-
External Shop Ticket - Enter Y to print an external shop ticket. The external shop ticket is the same as an internal shop ticket, but only includes pricing for jobs with a Pay Type for which Charge Customer is set to Y. Refer to Printing Repair Tickets.
-
Pickup Document - Enter Y to print a pick up document. You can also enter pickup instructions. Refer to Printing Pickup Tickets.
-
Pickup Date - Enter the date on which the item(s) should be picked up. Defaults to the date entered in Pick Up in the Shop Repair Information screen.
-
Delivery Document - Enter Y to print a delivery document. You can also enter delivery instructions. Refer to Printing Delivery Tickets.
-
Delivery Date - Enter the date on which the item(s) should be delivered to the customer.
-
Invoice Preview - Enter Y to print an preview of the invoice.
-
Pick Ticket - Enter Y to print a pick ticket. Refer to Printing Pick Tickets.
-
Inspection Ticket - Enter Y to print an inspection ticket if you use sub serial numbers for equipment that requires inspection. Refer to Printing Inspection Tickets.
•
Complete - Finishes the order and adds fees and taxes (refer to Completing the Order).
•
Suspend - Saves the order but does not flag it to be picked. If the product becomes available and/or the pick date passes, the pick ticket will not print because the order is suspended. Also, based on your system settings, any backordered products on a suspended shop repair order may not be eligible to be placed on a P/O.
•
Return - Goes back to the Shop Repair Order Entry screen.
If any items require immediate transfers, refer to Generating Transfers for a Shop Repair Order.
If any items were backordered, refer to Generating Purchase Orders for a Shop Repair Order.
Process Inventory as WIP, in Shop Repair Parameters (refer to Setting Shop Repair Parameters) indicates if you will use the extra work-in-process step to account for inventory on a shop repair order:
•
Y - Process inventory on a shop repair order as work-in-process (WIP). You must use Release Confirmation Entry to relieve inventory from the on-hand balance and move it into the WIP balance. When the order in invoiced, the quantities move out of WIP. Refer to the Shop Inventory Work In Process Report to list products in the Work in Process Account.
•
N - Do not process inventory as work-in-process. When you print the shop repair ticket, inventory balances are not affected until the order is invoiced. Release Confirmation Entry is not used.
