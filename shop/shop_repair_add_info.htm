Shop/Repair Additional Information screen
The Shop/Repair Additional Information screen allows you to view detailed information about a model/serial number combination. This data includes the make and model of the engine and/or transmission and a list of parts that are attached to the main part.
Menu Path: Shop Repair > Inquiries > Model/Serial Repair Inquiry > F6-Equipment Info > F5-Add Info
Field/Function Key Descriptions
Field/Function Key
Description
Order#
The number of the open shop/repair order.
Bulletin#
The service bulletin that is on the order, if any.
Job
The job number.
History#
The number of the invoiced shop/repair order.
Bulletin#
The service bulletin from the order, if any.
Job
The job number.
Hrs In
The number of hours on the product’s meter when the order was created.
Hrs Out
The number of hours on the product’s meter when the order was invoiced.
F2-Save
Closes the screen.
F5-Review
Displays the related transaction.
