Model/Serial Repair Inquiry
The Model/Serial Repair Inquiry lists orders for a product by serial number. You can view detail product and repair information for each order.
Viewing Repair Orders by Model/Serial Number
To view Model/Serial Repair Information:
1
Select Shop Repair > Inquiries > Model/Serial Repair Inquiry to open the Model/Serial Repair Inquiry screen.
2
Enter the Product to view information.
3
Enter the Serial Number of the model.
Information about that model/serial number combination appears.
Field/Function Key Descriptions
Field/Function Key
Description
Order Number
The shop repair order number.
Order Date
The date the order was entered.
Invoice Date
The date the order was invoiced.
Invoice Amount
The dollar amount on the invoice.
Hours In
If hours were recorded for the repair, the number on the hourly meter when the product was brought in.
Hours Out
If hours were recorded for the repair, the number on the hourly meter when the product was repaired.
Repair Groups
The number of the repair groups for each job on the order (refer to Creating the Job or Identifying Repair Groups for more information).
F2-Finish
Closes this screen.
F5-Review
Opens the Shop Repair Order Display screen for the order.
F6-Equipment Info
Opens the Shop/Repair Information screen for the model/serial number combination.
F7-SN History
Opens the Serial Number History Inquiry.
F9-Goto Line
Identifies a reference number on the screen that you want to jump to. Valid codes are order number, page number, line number, B (bottom), or T (top).
