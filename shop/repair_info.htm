Shop Repair Information screen
The Shop Repair Information screen gathers header-level information about the order, including the customer and the model/serial# being repaired. You can also review information about previous repairs to the product and any applicable service bulletins.
For more information, refer to Starting or Editing a Shop Repair Order.
Menu Path: Shop Repair > Shop Repair Order Entry
Field/Function Key Descriptions
Field/Function Key
Description
Model
The part number of the product to be repaired.
Serial
The serial number associated with the model.
Customer
The number of the customer for whom the product is being repaired.
Ship-To
The location to ship the product after it is repaired:
•
D - Duplicates the customer’s address (default).
•
Number - the number of an existing ship-to address.
•
P - Pickup order.
•
M - Manually enter a ship-to address.
•
N - Create a new ship-to address.
Contact
The name of the customer contact. Defaulted from Customer Maintenance.
Phone
The customer’s phone number. Defaulted from Customer Maintenance.
Fax
The customer’s fax number. Defaulted from Customer Maintenance.
Customer P/O#
The customer’s purchase order number.
Instructions
Special instructions about the order to print on the pick ticket. Press F3 to select a standard instruction code (set up in Instruction Code Maintenance /IMM).
Hours In
If the product has a meter, the total hours of use from the meter.
Hrs Out
If Hours In was entered, the total hours of use from the meter after the repair is completed.
Lst Hrs
The meter reading the last time the product was repaired.
Loaner
Indicates if a replacement product is being loaned to the customer:
•
Y - A replacement product is being loaned. The Loaner Information screen appears so that you can enter information about the loaned product.
•
N - A replacement product is not being loaned.
Setup
Indicates if the order is a setup for a product sold on a sales order:
•
Y - This is a setup order.
•
N - This is not a setup order.
Purchased
The date this model was purchased.
Failed
The date the model failed. May be used for warranty repairs.
Promised
The date the repair is expected or required to be completed.
Pick Up
The date the product must be picked up at the customer’s site.
Original Est.
The original estimate for the price of the repair.
Approved By
The name of the person who approved the original estimate.
Date
The date of the original approval.
Current Est.
The new estimate for the price of the repair.
Approved By
The name of the person who approved the new estimate.
Date
The date of the new approval.
Payment Terms
The payments terms assigned to the order.
Last Contact
The date of the last contact with the customer. This is for information only and displays in the Scheduling Calendar (refer to Scheduling Resources for Shop Repair Jobs).
F2-Save
Saves all information on the Shop Repair Information screen. If a service bulletin exists for this model and serial number, a message notifies you of the bulletins. You can accept the bulletin and have it automatically placed on the order as a job.
F5-Hdr
Opens the Sales Order Header screen.
F6-Model Info
Opens the Model Information Maintenance screen.
F7-Repair History
Opens the Model/Serial Repair Inquiry screen.
F8-Bulletins
Displays the Shop Repair Service Bulletin Entry screen, which provides information on any bulletins related to the model and serial number indicated.
Action Bar
Notes - Header Notes
Allows you to enter header notes for the shop repair order.
Notes - Pickup Instructions
Allows you to enter pickup instructions for the order. You can also enter these instructions after entering a date in Pick Up or choosing to print a pickup ticket from the Shop Document Selection screen.
Rental
Indicates the rental status of the products:
•
R - Available to rent.
•
D - Not available to rent while this order is open.
Frt Dtl
Opens the Shop/Repair Order Shipment Data screen to display freight information updated from the manifest system interface.
