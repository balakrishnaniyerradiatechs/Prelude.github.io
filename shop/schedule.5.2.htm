Defining Available Resource Types
Before you can schedule technicians and delivery trucks for shop repair jobs, you must identify the types of resources you want to schedule (e.g., delivery trucks, repair technicians). You can set up a different resource type for each supervisor so he/she can schedule an independent group of resources.
You can then identify the hours that each resource is available (refer to Identifying Available Hours for Resources) and schedule your technicians for shop repair orders (refer to Scheduling Resources for Shop Repair Jobs).
To set up a resource type:
1
Select Shop Repair > Processes > Calendar Scheduling > Scheduling Calendar Resource Type Maintenance to open the Scheduling Calendar Resource Type Maintenance screen.
2
For Resource Type, enter a 1 or 2 alpha/numeric code that identifies the type of resource you will schedule (e.g., R for repair technicians, D for delivery trucks). You can set up multiple groups of resource types so that different supervisors can maintain schedules for different groups of technicians (e.g., R1, R2, R3).
3
Enter the Description of the resource (e.g., delivery truck).
4
Enter the Type of schedule that this resource is for. Press F3 to view the available types. These are set up by Prelude and determine what the resource is validated against when you schedule resources (e.g., the Shop/Repair type is validated against Shop Repair Technicians).
5
Press F2-Save to save the resource type information.
