Loaner Information screen
The Loaner Information screen appears from the Shop Repair Information screen when you indicate that the customer will receive a replacement. You can enter data about a replacement part that is being loaned to a customer.
Menu Path: Shop Repair > Shop Repair Order Entry > enter a part number
Field/Function Key Descriptions
Field/Function Key
Description
Model Number
The part number for the model you are loaning to the customer.
Serial Number
The serial number for the model that you are loaning to the customer.
Order Number
The order number for the loan transaction.
Hours Out
If the product has an hourly meter, the number of hours on the meter when the product was loaned to the customer.
Hours In
If the product has an hourly meter, the number of hours on the meter when the product was returned.
F2-Save
Saves the loaner information.
