Adding Fees and Taxes
After you complete the jobs (refer to Completing Jobs), you can add freight charges, handling fees, taxes, etc.
To add fees and taxes:
1
When all jobs for the order are complete, from the Shop Repair Recap Information screen, press F2.
2
On the dialog box, choose Complete. The Sales Order Completion screen appears.
3
Enter any extra charges or discounts (e.g., freight).
4
If a deposit is required for a backorder, enter the amount in Deposit Required.
5
To change the Tax? flag for any charges, press F5-Tax Jur, change the data, and press F2-Complete.
6
On the Sales Order Completion screen, press F2-Complete.
7
One of the following occurs:
•
If this is an 05 order with cash terms, take payment for the order immediately (refer to Taking Payment for an Order).
•
If this is an 05 order without cash terms, the Print Invoice dialog box appears. Select Yes to print an invoice. If you are using continuous printing, the invoice prints immediately. If you are using batch printing.
•
If the order is a confirmation (e.g., 04, 06, or 08), the dialog box with the following message appears:
Choose from the following:
Are You Ready To Release The Order For Invoicing. Choose one of the following:
-
Yes - Flag the order for invoicing.
-
No - The order is not ready for invoicing.
