Shop Repair Efficiency Report
The Shop Repair Efficiency Report compares the billed time for a job to the time that a technician was clocked into the job.
Menu Path: Shop Repair > Reports > Shop Repair Efficiency Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Invoice Date range. The beginning and ending invoice dates to include.
Column Descriptions
Column Heading
Description
Tech
The technician’s number. For more information on technicians, refer to Setting up Technicians.
Description
The technician’s name.
Job
The job number (e.g., A, B, C) and description. Orders can have multiple jobs. Refer to Creating the Job.
Description
The description of the job.
Billed Time
The amount of time that was billed to the customer or vendor.
TCE Time
The actual time clocked in.
Efficiency
Compares the TCE Time to the Billed Time to measure how efficient the technician was for the job. For example, if TCE Time is 1 hour and Billed Time is 1 hour, the technician is 100% efficient. If TCE Time is 2 hours and Billed Time is 1 hour, the technician is 50% efficient.
Order#
The shop repair order number.
