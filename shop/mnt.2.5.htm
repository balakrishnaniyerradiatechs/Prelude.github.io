Creating Pay Types
Each job on a shop repair order requires a pay type to specify a vendor or customer, provide warranty information, and to indicate how to bill the job. The information that you enter for each pay type applies to the job in the Shop Repair Job Entry screen for the shop repair order.
To enter a pay type:
1
Select Shop Repair > Maintenance > Pay Type Maintenance to open the Pay Type Maintenance screen.
2
For Pay Type, enter a code to represent a pay type (up to 6 alphanumeric characters).
3
Enter a brief Description of the pay type.
4
If you want to use a customer’s pricing structure to price labor and parts, enter the customer’s number in Customer.
5
If prices for this pay type should be based on a customer price group, enter the code for the customer price group in Price Group.
6
For Charge Customer?, indicate if the customer should pay for the parts and labor for this pay type. Choose from the following:
•
Y - The customer pays for parts and labor.
•
N - The customer does not pay for jobs using this pay type.
7
For Capitalize, indicate if the dollars for this pay type are capitalized. Choose from the following:
•
Y - If Charge Customer? is set to N, the system creates a general ledger entry when the shop repair order is invoiced. The dollars for this pay type are considered no-charge and posted to Add-On Capital for the equipment ID number (refer to the Update Asset for Capital Repair field in Setting Shop Repair Parameters).
•
N - The dollars for the shop repair order are not capitalized.
8
For Warranty Flag, indicate if the pay type is for warranty work. Choose from the following:
•
Y - This pay type is for warranty work. A vendor claim automatically generates when a job with this pay type is invoiced.
•
N - The pay type is not for warranty work.
9
For Submit Electronic, indicate if vendor claims are submitted electronically to the vendor for payment. Choose from the following:
•
Y - Vendor claims are submitted electronically to this vendor.
•
N - Vendor claims are submitted using the regular vendor claim process.
10
For Require Main SRT?, indicate if the vendor requires a main SRT code for the warranty work. Choose from the following:
•
Y - A main SRT code is required to complete a job with this pay type (refer to Completing the Order for more information).
•
N - A main SRT code is not required.
11
For Require Failed PN?, indicate if the vendor needs to know the part number that failed on the model being repaired. Choose from the following:
•
Y - The failed part number is required to complete a job with this pay type (refer to Completing the Order).
•
N - The failed part number is not required.
12
For Price Labor at Cost, determine how Technician Job Billing Entry should price labor line items. Choose from the following:
•
Y - Price labor line items at the labor rate set up for the technician in Technician Maintenance (refer to Setting up Technicians).
•
N - Price labor line items at the retail price for the labor type.
(refer to Entering Billable Time for Technicians).
13
If this is a rental item, for Operating Expense, indicate if this pay type updates non-operating expenses for the equipment:
•
Y - Post to Non-Operating Expenses for the (Rental > Equipment Subsystem > Maintenance > Equipment Maintenance > F8-Fin).
•
N - Do not post to Non-Operating Expenses.
14
For Rework?, indicate if this pay type is for a job that is to re-do a previous job:
•
Y - This is a rework job that is back for repair.
•
N - This is not a rework job.
This flag determines the default for Rework in the Shop Repair Job Entry screen.
15
If this is a warranty pay type, for Warranty Vendor#, enter the number for the vendor that will pay the warranty claim.
16
If any fees should be automatically added to a job with this pay type, press F6-Shop Chgs to open the Shop Charges screen. You can enter shop charge codes associated with this pay type (refer to Creating Shop Charges).
Auto Shop Charges in Shop Option Parameters (refer to Setting Shop Repair Parameters) indicates if a separate shop charge is automatically created when you complete the shop order (refer to Completing the Order):
•
Y - Automatically create a shop charge for each job on the order, without prompting. A separate charge generates for each job and is determined by the pay type assigned to the job. Refer to Creating Pay Types.
•
N - Do not automatically create shop charges. The Shop Repair Job Entry screens appears when you press F6-Complete from the Shop Repair Recap Information screen, so that you can manually add each shop charge to each job.
17
If you need to override company-level G/L accounts for this pay type, press F5-G/L Overrides and enter the Currency Indicator to open the Shop/Repair Payment Type G/L Overrides screen. After you enter any overrides, press F2-Save.
18
When the pay type is complete, press F2-Save.
