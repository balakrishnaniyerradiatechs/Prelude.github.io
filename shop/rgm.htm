Repair Group Maintenance screen
A repair group identifies the type of repair that a part requires (e.g., maintenance, steering). Repair groups are used in five places:
•
Sales Rep Maintenance - Identifies which types of repairs a technician is allowed to work on. For more information, refer to Setting up Technicians.
•
Product Maintenance - For products with warranty work, specifies what type of repair the product needs. For more information, refer to Entering Standard Repair and Setup Time for Products.
•
Service Bulletin Maintenance - For a service bulletin, establishes the type of repair that the service bulletin requires. For more information, refer to Entering Service Bulletins.
•
Shop Repair Order Entry - In the Shop Repair Job Entry screen, identifies which type of repair will be completed for the job. For more information, refer to Creating the Job.
•
Shop Repair Job Code Maintenance - the Shop Job Code Maintenance screen, defines the type of repair that will be completed for a job. For more information, refer to Setting Up Job Codes.
Menu Path: Shop Repair > Maintenance > Repair Group Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Repair Group
A code that represents a type of repair (e.g., 01, 02). Up to 6 alphanumeric digits.
Description
A description of the repair group (e.g., Maintenance, Steering).
F2-Save
Saves the repair group information.
F4-Del
Deletes the repair group.
