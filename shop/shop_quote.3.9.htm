Entering a Kit Parent
If assembled kits are available, a dialog box appears with the following message after you press F2-Save on the Quote Entry Detail screen:
Pre-Assembled KIT Units Available - Sell Pre-Assembled Units?
Choose from the following:
•
YES - The Quote Repair Order Entry screen returns to allow you to continue with the order.
•
NO. The kit parent behaves differently based on its type. Refer to the following sections for information on kit types:
