Shop Repair Skill Set screen
The Shop Repair Skill Set screen allows you to identify which repair groups that a technician is authorized to work on.
Menu Path: Shop Repair > Maintenance > Technician Maintenance > F5-Repair Skill Set
Field/Function Key Descriptions
Field/Function Key
Description
RGroup
The repair groups that the technician is authorized to work on. These codes are set up in Repair Group Maintenance (refer to Identifying Repair Groups).
Description
The description of the repair group.
Level
The internal code that represents the level of skill for this type of repair group. These are established by your company and not validated in the system.
F-Save
Saves the repair skill set information and returns to the Technician Maintenance screen.
