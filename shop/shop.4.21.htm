Generating Work Orders for Shop Repair Orders
If you enter a line item that is a Bill of Material (BOM) parent and no finished goods exist, you may be prompted to enter a work order if:
•
Sc is W.
•
Generate W/O from S/O in your user ID is set to Y or R.
•
Your company settings allow work orders to generate from Shop Repair Order Entry.
If the Auto Build-To-Order W/O flag in BMW/Work Order Parameters is set to W, the system automatically generates the work orders without the following prompts.
To enter a work order from Shop Repair Order Entry:
1
Press F2-Save on the Line Item Detail screen (refer to Adding or Editing Line Items). A dialog box with the following message appears:
Do You Wish To Continue With Work Order Entry?
2
If you select Yes or Repair, the Work Order Entry screen appears. Enter the work order and continue entering line items (refer to Adding or Editing Line Items) and jobs (refer to Creating the Job) as necessary.
