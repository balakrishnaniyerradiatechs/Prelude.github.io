Shop Repair Service Bulletin Entry screen
The Shop Repair Service Bulletin Entry screen lists any applicable service bulletins for this model/serial number combination. You can access this screen by pressing F8 on the Shop Repair Information screen or it appears automatically when you press F2 on the Shop Repair Information screen if any service bulletins exist.
Menu Path: Shop Repair > Shop Repair Order Entry > F8 Bulletins
Field/Function Key Descriptions
Field/Function Key
Description
Bulletin#
The number of the service bulletin that is available. For more information on service bulletins, refer to Entering Service Bulletins.
Description
The description of the service bulletin.
Cpl-Date
The vendor’s target completion date for the service bulletin.
Int-Date
Your company’s target completion date for the service bulletin.
Alert
The level of the service bulletin:
•
1 - Safety rework.
•
2 - Repair.
•
3 - Repair bulletin.
•
4 - Information bulletin.
Order#
If the service bulletin is a job on an open order, the order number appears.
Accept
Indicates if you would like to create a job for the service bulletin:
•
B - The repair indicated in the bulletin was already completed.
•
N - Do not add the bulletin to this order. A warning appears when you save the data in the Shop Repair Information screen.
•
Y - Add the bulletin as a job to the order.
F2-Save
Saves the Accept status of the bulletin.
F5-Review
Opens the Service Bulletin Inquiry.
