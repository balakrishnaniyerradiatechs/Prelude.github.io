Quote Repair History Inquiry
The History Shop Repair Inquiry screen lists expired quotes for a customer.
Viewing Historical Quotes
To list expired quotes for a customer:
1
Select Shop Repair > Shop Quote Subsystem > Quote Repair History Inquiry. The Quote Repair History Inquiry screen appears.
2
Enter the Customer or keyword.
3
Choose a Sort option:
•
Order number.
•
Order date.
•
Required date.
•
Serial number.
•
Model number.
•
Sales Order dollars.
4
Choose a Filter option:
•
No.
•
Yes - use the following selection criteria:
-
Model number.
-
Serial number.
-
Required date range.
-
Quote number.
-
Cust release#.
-
Customer P/O#.
-
Repair groups.
Expired shop repair quotes for the customer appear.
Field/Function Key Descriptions
Field/Function Key
Description
Quote#
The shop repair quote number.
Type
The type of quotation.
N
An asterisk (*) appears if notes were entered for the quote.
Qte Date
The date the quote was entered.
Exp-Date
The date the quote expired.
DA
Disposition analysis. If the quote was deleted, a deletion reason code appears.
Merchandise
The total price that the customer was quoted.
Conv
Indicates if the quote was converted to a sales order:
•
Y - The quote was converted.
•
N - The quote was not converted.
F2-Finish
Closes this screen.
F5-Review
Opens the Shop Quote History Inquiry screen to display detail information for the order.
F6-Enter Quote#
Allows you to enter an quote number to review.
F8-Log
Opens the Quote Log Inquiry screen to display any transaction that affected the order.
F9-Goto Line
Identify a reference number on the screen that you want to jump to. Valid codes are order number, page number, line number, B (bottom), or T (top).
