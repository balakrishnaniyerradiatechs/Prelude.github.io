Shop Repair Recap Information screen
The Shop Repair Recap Information screen lists information for each job on an order, including time spent on the repair and charges for parts and labor. From this screen you can schedule technicians, enter or complete a job, or view billing information for the job.
For more information, refer to Completing Jobs.
Menu Path: Shop Repair > Shop Repair Order Entry > F2-Save
Field/Function Key Descriptions
Field/Function Key
Description
Job
The number of the job (e.g., A, B, C).
Description
The description of the job.
PType
The pay type for the job. For more information on pay types, refer to Creating Pay Types.
Status
The status of the job. The system updates this field as the order changes. Options are:
•
Complete.
•
Technician clocked in.
•
Open.
•
Quotation.
•
Pending warranty response.
•
Pending warranty submittal.
•
Technician clocked in.
•
Waiting on parts.
Actual
The actual time that technician(s) were clocked into the job.
Billed
The time that will be billed to the customer or the vendor.
Effic.
The efficiency of the time spent on the job. For example, if Actual is 5 and Billed is 5, the efficiency is 100. If Actual is 10 and Billed is 5, efficiency is 50.
Labor
The price of labor for this job.
Parts
The price of the parts for the job.
Misc.
The price of any shop charges (refer to Creating Shop Charges for more information).
NoCharge
The amount of the job that will not be charged to the customer.
Charge
The amount of the job that will be charged to the customer.
Totals
Lists totals for the order for the following fields:
•
Actual.
•
Billed.
•
Effic.
•
Labor.
•
Parts.
•
Misc.
•
NoCharge.
•
Charge.
F2-Save
Saves the job and order information.
F5-Edit
Opens the Shop Repair Job Entry screen.
F6-Complete Job
Attempts to complete the job. If the system cannot complete the job, an error message appears with a reason (e.g., no solution text).
F7-Pre-Scan Edit
Reviews the job to see what remaining tasks are required for the job before it can be completed (e.g., enter solution text, main SRT, or failed part number).
F8-Schedule
Opens the Calendar Header screen to allow you to schedule repair technicians to this shop repair order. Refer to Scheduling Resources for Shop Repair Jobs.
Action Bar
Calc. Charges
Allows you to calculate any required shop charges for each job. For more information on shop charges, refer to Creating Shop Charges.
Recap Rpt
Opens the Shop Job Recap Report which compares actual and quoted labor, parts, and miscellaneous fees for an order.
