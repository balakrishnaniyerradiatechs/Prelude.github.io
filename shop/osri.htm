Open Shop Repair Inquiry
The Open Shop Repair Inquiry screen lists open (not invoiced) orders for a customer.
Viewing Open Shop Repair Orders
To list and review open shop repair orders for a customer:
1
Select Shop Repair > Inquiries > Open Shop Repair Inquiry to open the Open Shop Repair Inquiry screen.
2
Enter the Customer Number or keyword.
3
Choose a Sort option:
•
Order number.
•
Order date.
•
Required date.
•
Serial number.
•
Model number.
•
Sales order number.
4
Choose a Filter option:
•
No.
•
Yes - use the following selection criteria:
-
Model number.
-
Serial number.
-
Required date range.
-
Order number.
-
Cust release#.
-
Customer P/O#.
-
Repair groups.
Open shop repair orders for the customer appear.
Field/Function Key Descriptions
Field/Function Key
Description
Order#
The shop repair order number.
N
An asterisk (*) appears if notes were entered for the order.
I
Indicates the status of the order’s invoice:
•
Y - Flagged for invoicing.
•
H - On hold.
•
Blank - Not on hold or flagged for invoicing.
CH
Indicates if the order is on credit hold:
•
H - On credit hold.
•
R - Released from credit hold.
•
Blank - Not on credit hold.
Model
The main part that was repaired on the order.
Serial Number
The serial number of the part that was repaired (if one was entered).
Req-Date
The date the customer needs the repair completed.
S/O Amount
The total price that the customer will pay for the order.
F2-Finish
Close this screen.
F5-Review
Opens the Shop Repair Order Display screen to allow you to view detail information for the order.
F6-Enter Order #
Allows you to enter an order number and go directly to the review screen for the order.
F8-Log
Opens the Order Log Inquiry screen to allow you to view every transaction that affected the order.
F9-Goto Line
Identify a reference number on the screen that you want to jump to. Valid codes are order number, page number, line number, B (bottom), or T (top).
