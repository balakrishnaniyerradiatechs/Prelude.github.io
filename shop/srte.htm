Shop Repair Time Clock Entry screen
A technician uses the Shop Repair Time Clock Entry screen to clock into a time clock code or a job.
Menu Path: Shop Repair > Shop Repair Time Clock > Shop Time Clock Entry
Field/Function Key Descriptions
Field/Function Key
Description
Technician
The code for the technician clocking into the job.
Date
The current date and day of week.
Job/Code
Either the time clock code or the order number:
•
To clock into a code (e.g., SW - start work, BT - break time), enter the code.
•
To clock into a job, enter the order number and job number in the pattern <order>!<job>. For example, to clock into job B for order 98650, enter 98650!B.
•
To clock into a work order, enter W, then enter the work order number. In this case, you must enter a Burden code to post a labor burden to the work order.
Description
A description of the time clock code (e.g., start work) or the customer number and name if an order and job number were entered.
Repair
If an order and job number were entered, the repair number (e.g., 98650).
Job Number
If an order and job number were entered, the job number (e.g., B).
Start Time
The current time. The stop time is calculated when the technician clocks into the next job or code.
Comment
Comments regarding the work being done.
F2-Save
Saves the time clock input. If the entry is for a repair job, this information automatically updates the shop repair order and job on the Shop Repair Recap Information screen. You must manually add the cost of the labor to the order before you complete it.
