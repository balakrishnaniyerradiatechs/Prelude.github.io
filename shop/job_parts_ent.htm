Parts Entry screen
The Parts Entry screen allows you to view or enter products that are part of a job code and are required to complete a job.
Menu Path: Shop Repair > Shop Repair Job Entry > Action Bar > Parts Entry
Field/Function Key Descriptions
Field/Function Key
Description
Product
The part number associated with the job.
Description
The description of the part.
Code
Indicates if the product is:
•
R - Required.
•
O - Optional.
Original-Quantity
The quantity associated with the job code, if the product is part of a job code. Refer to Setting Up Job Codes for more information.
Quantity
The quantity that is needed to complete the repair.
F2-Save
Saves the product information.
