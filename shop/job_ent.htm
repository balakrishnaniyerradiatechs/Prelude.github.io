Shop Repair Job Entry screen
Each shop repair order may have one or multiple jobs. A job is a task to be completed for the model on the order and is assigned a job number (e.g., A, B, C, D). The Shop Repair Job Entry screen allows you to enter problem information and indicate which parts are needed to complete each job.
For more information, refer to Entering a Job.
Menu Path: Shop Repair > Shop Repair Order Entry > F7-Job
Field/Function Key Descriptions
Field/Function Key
Description
Order Number
The order number for the job.
Job Number
The job number (e.g., A, B, C). Each job consists of labor charges and any parts or items that must be purchased to complete the job.
Description
A description of the repair job a job code (for more information, refer to Setting Up Job Codes).
Pay Type
The code for the pay type that indicates how the job will be billed (e.g., CB, WTY). For more information, refer to Creating Pay Types.
Repair Group
The code for the repair group for this job (e.g., 01, 02). For more information, refer to Identifying Repair Groups.
Status of Job
The status of the job. The system updates this field as the order changes. Options are:
•
C - Complete.
•
N - Technician clocked in.
•
O - Open.
•
Q - Quotation.
•
R - Pending warranty response.
•
S - Pending warranty submittal.
•
T - Technician clocked in.
•
W - Waiting on parts.
Repair Amount
Determines how the total repair amount is calculated:
•
If you will charge the customer a set fee for this job no matter what parts are needed or what labor is required, enter the total Repair Amount. The prices are recalculated to total this amount.
•
If you want the price of the job to depend on the products used and the labor required, leave this field blank. The price of the job is the price of the products and labor used to complete the job.
Repair Amount Type.
If you entered a Repair Amount, you can indicate what costs are included:
•
L - Labor.
•
P - Parts.
•
B or null - Both labor and parts (default).
Est. Repair Time
The estimated time to repair the product. This information is used to calculate job efficiency. The default comes from the time entered for a setup job or service bulletin.
Schedule
Indicates if you need to schedule this job to a repair technician:
•
Y - This job needs to be scheduled (refer to Scheduling Resources for Shop Repair Jobs).
•
N - This job does not need to be scheduled.
Up Sale?
Indicates if the job is a sale in addition to the primary job the customer came in for:
•
Y - This job is an additional sale.
•
N - This job is not an additional sale.
Rework
Indicates if this job is to re-do a previous job:
•
Y - This is a rework job that is back for repair. In the following field, you must enter the number for the Tech that worked on the original repair that caused the need for the rework job.
•
N - This is not a rework job.
The default for Rework comes from the pay type (refer to Creating Pay Types).
Tech
If Rework is set to Y, the technician that worked on the repair job the first time and caused the need for the job to be re-done.
Tech (table)
The code of the technician who will repair the product. This technician must be authorized to do the type of repair work as indicated in Repair Group. For more information on technicians, refer to Setting up Technicians.
Description
The description of the technician.
Est. Hrs.
If a technician is assigned to the job, the estimated time the technician should spend working on the job.
Act. Hrs.
The actual hours that the technician clocked into the job. This is updated as technicians clock in and out of the job.
Sch-date
The date work is scheduled for the job, if you scheduled a technician for the job (F9-Schedule > F5-Review > F6-Schedule).
Time
The time the technician is scheduled to start the work (F9-Schedule > F5-Review > F6-Schedule).
F2-Save
Saves the job information.
F5-Problem
Opens a problems notes screen so that you can enter a more complete description of what needs to be repaired. If Require Problem Notes in Shop Repair Option Parameters is set to Y (refer to Setting Shop Repair Parameters), notes are required. Text entered here defaults into the problem text for the job.
F6-Solution
Opens a solutions notes screen. When you complete the job, solution test is required. Text entered here defaults into the solution text for the job.
F7-Warranty
Opens the Warranty Information screen. For more information, refer to Entering Warranty Information for a Job.
F8-Time Clock
Opens the Time Clock Display screen, which shows all time clocked into the job. For more information, refer to Time Clock Entry & Billing.
Action Bar
Parts Entry
Opens the Parts Entry screen, which displays any parts entered for the job if a job code was used. You can also enter parts for the job. Refer to Entering Parts for a Job.
Ovr Serial Num
Opens the Override Product Serial Information screen to allow you to override the serial number set up for a model. If sub-serial numbers are set up for the product, the sub serial numbers are updated also. Refer to Setting up Sub-Serial Numbers.
Rent Info
Opens the Rental Information screen to allow you to view or edit the Maintenance Code if this repair is the result of a rental item returned for scheduled maintenance.
