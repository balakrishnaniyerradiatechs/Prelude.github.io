Setting up Time Clock Codes
Before you can use Shop Timeclock Entry to record your technicians’ time during the day, you must set up timeclock codes to record time that is not spent working on a job (e.g., cleaning the shop, break time).
To create a timeclock code:
1
Select Shop Repair > Maintenance > Shop Time Clock Code Maintenance to open the Shop Time Clock Code Maintenance screen.
2
For Code, enter a code (up to 4 alphanumeric characters) for the task.
3
Enter a Description of the task (e.g., Break Time, Meeting, Parts lookup).
4
For Paid, choose one of the following:
•
Y - Time spent clocked into this code is paid.
•
N - Time spent clocked into this code is not paid.
Paid and unpaid time appears on the Shop Timekeeping Payroll Report.
5
For PTime, choose from the following:
•
Y - Time spent clocked into this code is considered as productive time and accumulates under Actual Hours on the Productivity By Technician Report.
•
N - Time spent clocked into this code is not considered as productive time.
6
Repeat steps 2-5 for each code to add or edit.
7
Press F2-Save.
