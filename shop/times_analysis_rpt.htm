Times Analysis Report
The Times Analysis Report lists time clock code entries (e.g., SW - start work, BT - break time) and the time that the technician clocked against the code. This report only lists time recorded in Shop Time Clock Entry, For more information on technicians, refer to Time Clock Entry & Billing.
Menu Path: Shop Repair > Reports > Times Analysis Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Technician#. The technician(s) for which you want to view productivity. Refer to Setting up Technicians.
•
Entry Date range. A range of time clock entry dates. These dates are entered automatically when a technician clocks in and out of a job but may be changed by a manager.
•
Totals Only?. Indicates if you want to view the time clocked into each code or the total time clocked into all codes.
Column Descriptions
Column Heading
Description
Tech
The technician’s number.
Tech Name (not labelled)
The technician’s name.
Date
The entry date for the clocked hours.
Time Code
The time code the technician clocked into. For more information on time codes, refer to Setting up Time Clock Codes.
Time
The time that the technician spent clocked into the time code.
Cost Ext
The extended cost for the time code. Calculated by multiplying the technician’s hourly rate by the hours he/she worked on the corresponding time code.
