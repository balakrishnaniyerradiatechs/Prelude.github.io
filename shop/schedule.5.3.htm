Identifying Available Hours for Resources
Once you set up your resource types (refer to Defining Available Resource Types), you can assign resources to each type and set up a schedule of operating hours to identify when each resource is available.
To schedule available hours for your resources:
1
Select Shop Repair > Processes > Calendar Scheduling > Scheduling Calendar Resource Maintenance to open the Scheduling Calendar Resource Maintenance screen.
2
Enter the Resource Type (refer to Defining Available Resource Types) that identifies this group of resources.
3
Enter the Warehouse where the resources exist.
4
For Resource, depending on the Resource Type enter the resources to assign to this type. For example, if Resource Type is Repair, enter the technician numbers (refer to Setting up Technicians). If Resource Type is Delivery, enter the truck numbers or names.
5
For Description, if you entered a technician number for Resource, the technicianâ€™s name will appear. If you entered any other type of resource, enter a description of the resource. For example, you can enter a description or designation for the truck.
6
In the Operating Hours table, enter the default Start and End time for each day of the week that your company is open for business.
7
If you want to make sure that technicians or resources are not scheduled during lunch and/or break times, enter a company level default Start and End time for Lunch, AM Break, and PM Break.
8
If you want to override and start and stop times for any individual resource, highlight the Resource and press F5-Resource Info to open the Resource Information Maintenance screen.
9
Enter the Start and End times for the resource for each day of the week and for Lunch and Break times.
10
Press F2-Save to return to the Scheduling Calendar Resource Maintenance screen.
11
Press F2-Save to save the resource schedule information.
