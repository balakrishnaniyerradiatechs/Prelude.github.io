Shop Repair Order Entry screen
The Shop Repair Order Entry screen allows you to enter the information necessary to begin a repair ticket for a product/model number. From this screen you can access, enter, and view all the information necessary to repair a product.
For more information, refer to Entering a Shop Repair Order.
Menu Path: Shop Repair > Shop Repair Order Entry
Field/Function Key Descriptions
Field/Function Key
Description
Order Number
The number of the shop repair order.
Note (not labelled)
An asterisk (*) indicates if header notes were entered for the order. To view or enter header notes, choose Action Bar > Notes > Header Notes.
Customer
The number of the customer who is having a product repaired.
Product
The number of the product or job. For more information on entering products, refer to Adding or Editing Line Items. For more information on entering jobs, refer to Creating the Job.
Description
A description of the product or job.
Notes
An asterisk (*) indicates if notes were entered for the line. To view or enter line notes, choose Action Bar > Notes > Line Notes.
AOI
If the line is on admin order hold, the code that indicates the reason for the hold appears.
AII
If the line is on admin invoice hold, the code that indicates the reason for the hold appears.
Linkage
If the line is back ordered and linked to another transaction, the document number appears. The prefix indicates what type of document the line is linked to:
•
P - Purchase order.
•
T - Transfer.
•
W - Work order.
Job
The number of the job (e.g., A, B, C). If the line is a product, indicates the job that the product is associated with.
Quantity
The quantity of the product that is required to complete the job.
U/M
The product’s unit of measure (e.g., EA, DZ).
Whse
The shipping warehouse in which the product is located.
Price
The price of one unit of measure of the product.
Totals
Quantity
The total quantity of products on the order.
Weight
The total weight of the products on the order (if weight information is available).
Profit
The total profit for the order.
Gp%
The gross profit percent for the order.
Amount
The total price of the order.
Function Buttons Before an Order Number is Chosen
F2-Save
Saves the order.
F5-Startup
Opens the Startup Parameters screen.
F8-Last Order #
Opens the last order you saved in this session.
Function Buttons and Action Bar After an Order Number is Chosen but Before a Customer is Chosen
F5-Conv Quote
Opens the Quote Conversion screen. You can choose to convert an entire quote or individual lines on the quote.
F6-Conv EDI
Opens the EDI Conversion screen. Not applicable for shop repair orders.
F7-Conv Prospect
Opens the Prospect Conversion screen. You can use this screen to convert a prospect to a customer and then use that customer for the order.
F8-Enter New Customer
Opens the Temporary Customer Maintenance screen. You can use this screen to enter information about a new customer.
Action Bar
User Fields
See your company procedures or system administrator.
Dup Order
Opens the Order Duplication Entry screen to allow you to copy an open order for the same customer or a different customer.
Dup Order Hist
Opens the Order Duplication Entry screen to allow you to copy an invoiced order for the same customer or a different customer.
Auto Reverse
Opens the Auto Reverse Order screen to allow you to create a credit memo for an entire invoiced order.
Function Buttons and Action Bar After an Order Number and Customer are Chosen
F2-Save
Saves the order.
F4-Del
Deletes the order.
F5-Hdr Scr
Opens the Shop Repair Information screen. For more information on entering header information for an order, refer to Starting or Editing a Shop Repair Order.
F6-Totals
Allows you to edit the totals and set the order up for lot billing.
F7-Job
Opens the Shop Repair Job Entry screen and allows you to create a new job or edit an existing job. For more information on entering jobs, refer to Creating the Job.
F9-Goto Line
Allows you to identify a reference number on the screen that you want to jump to. Valid codes are page number, line number, B (bottom), or T (top).
Action Bar
Examine
Displays detail information for the product that your cursor is on.
Review
Opens the Shop/Repair Order Review screen to display detailed information for each line on the order.
ProdFunc - Add Comment Lines
Opens the Comments Description Entry screen, which allows you to enter comments. For more information on entering comments, refer to Adding Comments.
ProdFunc - Line Item Split
Opens the Order Line Split screen, which allows you to separate an existing line into different required dates.
ProdFunc - Rental Unit Sold
Not applicable for shop repair orders.
ProdFunc - Buyer Message
If the product is linked to a purchase order, allows you to enter a message for the buyer.
OrderFunc - Reprice Order Options
Allows you to choose a method to reprice each line on the order:
•
Customers Regular Price - The customer’s usual price column, matrix, or contract.
•
Gross Margin Percentage - The gross margin percent for the order.
•
Less Discount - The percent discount off of suggested retail, the customer’s default price column, or the current price.
•
Cost Plus - The amount you want to add to cost.
•
Minimum Gross Margin - Recalculate all prices to the product line minimum.
•
Column Pricing - A new price column for all items.
•
Sum and Reprice - Reevaluate all products for quantity breaks.
•
Pricing By Square Footage - Not applicable for shop repair orders.
OrderFunc - Manual Order Administrative Hold
Puts the order on Administrative Order Hold.
OrderFunc - Manual Administrative Invoice Hold
Puts the order on Administrative Invoice Hold.
 
OrderFunc - Rate Shop
Not applicable for shop repair orders.
OrderFunc - Campaign Priority Code
Not applicable for shop repair orders.
OrderFunc - Allocated Inv Review
If any products are available for reallocation, opens the Backorder Reallocation Availability screen. You can use this screen to determine if you can reallocate stock from one sales order to another.
OrderFunc - Order Line Sort Options
Resorts lines using one of the following methods:
•
Customer product number.
•
Product description.
•
Product number.
OrderFunc - Import RF Batch Order Lines
Allows you to import radio frequency line items for 05 & 06 order types.
Schedule Delivery
Opens the Scheduling Calendar to allow you to view scheduling resources for each day.
JobFunc - Sort Lines By Job Number
Resorts lines so that all products associated with a job appear with that job.
JobFunc - Change Job Number
Moves a product from one job to another.
JobFunc - Move To Another Job
Moves a product to a job on a different order.
JobFunc - Move All Lines to Job
Moves all line items from a job on a shop repair order to another shop repair order.
JobFunc - Import Job From Quote
Imports a previously entered shop repair quote. You can use this if you have a job that requires additional work and you need to provide an estimate to the customer. You can enter the shop repair quote, then once approved, you can import the quote onto the job.
Notes - Header Notes
Allows you to enter notes for the order.
Notes - Line Notes
Allows you to enter notes for the line that your cursor is on.
Notes - Conversion Notes
Not applicable for shop repair orders.
Notes - EDI Line item Notes
Not applicable for shop repair orders.
Notes - EDI Header Notes
Not applicable for shop repair orders.
Notes - Service Problem Notes
Not applicable for shop repair orders.
Notes - Service Solution Notes
Not applicable for shop repair orders.
