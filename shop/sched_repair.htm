Shop/Repair Appointment Entry screen
The Shop Repair Appointment Entry screen allows you to enter or edit an appointment for one of your resources. You can identify the type of appointment, schedule the hours to work, and enter job information. You can access this screen using the detail, weekly, or monthly view from either a shop repair order or Scheduling Calendar Entry
For more information, refer to Scheduling Resources for Shop Repair Jobs.
Field/Function Key Descriptions
Field/Function Key
Description
Resource
The technician’s number and description.
Date
The date on which you are scheduling work.
Hours
The number of hours that you are assigning to this technician for this job.
Start Time
The time the technician should begin work on the job.
End Time
The time the technician should finish the job. This is already filled in if you entered Hours and Start Time.
Schedule Type
The type of work you are scheduling:
•
A - Appointment
•
P - Personal time/sick.
•
S - Schedule work.
•
T - Training.
•
V - Vacation.
Comment
Any comments regarding the technician or the appointment.
Status
The status of the appointment:
•
C - Completed.
•
O - Open.
Order Number
The order number for which you are scheduling a resource.
Required Date
The date the customer requires that the repair be completed.
Last Contact
The date that you last had contact with the customer.
Model
The number of the model being repaired.
Customer
The customer number and name default from the shop repair order you are in.
Job
The job that you are scheduling work for.
Description
The job description.
PType
The code for the pay type that indicates how the job will be billed (e.g., CB, WTY). For more information, refer to Creating Pay Types.
Status
The status of the job. The system updates this field as the order changes. Options are:
•
C - Complete.
•
N - Technician clocked in.
•
O - Open.
•
Q - Quotation.
•
R - Pending warranty response.
•
S - Pending warranty submittal.
•
T - Technician clocked in.
•
W - Waiting on parts.
F2-Save
Saves the schedule resource information.
F4-Del
Deletes the appointment.
F5-Review
Opens the Sales Order Display screen to allow you to view detail information for the order.
