Adding Fees and Taxes
To get a total value of the quote, including freight, handling fees, and taxes:
1
On the Shop Repair Recap Information screen (refer to Saving a Quote), press F2-Save.
2
On the dialog box, choose Complete to open the Quotation Completion screen.
3
Enter any extra charges or discounts (e.g., freight).
4
For Pending Deposit, enter any deposit required for a backorder, if necessary.
5
To change Tax?, press F5-Tax Jur, change the data, and press F2-Save.
6
Press F2-Complete. A prompt appears with the following choices:
•
Print - Flags the quote to print.
•
Fax - Faxes the quote. The Fax Confirmation screen appears. If the customer is not set up to fax, a warning dialog box appears. Select Yes to proceed to the Fax Confirmation screen. The information defaults from the header. Update it as necessary and press F2.
•
E-mail - Emails the quote to the customer. The Email Confirmation screen appears. If the customer is not set up to email, a warning dialog box appears. Select Yes to proceed to the Email Confirmation screen. The information defaults from the header. Update it as necessary and press F2.
•
Suspend - Suspends the quote. Suspended quotes are saved, but are not printed, faxed, or emailed.
•
Return - Returns you to the quote.
