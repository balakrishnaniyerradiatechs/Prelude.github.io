Regular or Assortment Type Kit Parent
If you enter a kit parent that is a regular (Type R in Kit Maintenance) or an assortment (Type A in Kit Maintenance), the list of items is normally assembled at picking time. Usually inventory is kept at the component level, however, you can assemble the kit ahead of time and keep inventory at the parent level. The parent and the components print on the pick ticket and the parent prints on the invoice.
When you enter a regular or assortment type kit parent on an order (refer to Entering a Kit Parent), a dialog box with the following message appears:
Do You Wish To Continue With Kit Entry?
If the kit parent’s S/O flag is set to A or E, this dialog box does not appear. If the flag is set to A, the Line Item Detail screen for the kit parent appears and you cannot edit the components in the Kit Entry screen. If the flag is set to E, the Line Item Detail screen for the kit parent appears, but you can press F2 on the Line Item Detail screen and then press ENTER on the line item to view the Kit Entry screen and edit the components.
1
Choose one of the following:
•
No - To use the kit parent as a regular product.
•
Yes - To change the quantities or components in the Kit Entry screen.
2
For Kit-Qty, enter the quantity to go with each kit.
3
Press F2-Save to complete the kit entry.
