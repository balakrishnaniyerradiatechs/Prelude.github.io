Printing Pickup Tickets
If you are continuously printing pickup tickets, do not use this process. They print immediately after you flag them to print and save the quote.
To batch print pickup tickets:
1
Choose Shop Repair > Shop Quote Subsystem > Quote Repair Pickup Ticket Printing to open the Shop Repair Quote Pickup Printing screen.
2
Enter the following selection criteria as necessary:
•
Shop Quote Number - The number of the quote.
•
Division - The division number for the quote(s).
•
Region - The region number for the quote(s).
•
Selling Warehouse - The warehouse that will receive credit for the repair.
•
Department - The department number for the quote(s).
•
Cost Center - The cost center number for the quote(s).
•
Customer - The customer number(s) for the quotes.
•
Printer Form Queue - The printer where the repair tickets print.
3
Press F2-Save and choose Execute. A dialog box with a message similar to the following appears:
1 Records Selected For Shop Repair Pickup Ticket Printing! Load Shop Ticket Forms In Printer & Align Visually.
4
Choose one of the following:
•
Print Tickets - Continue with printing.
•
Abort Print - Stop printing. No pickup tickets print.
5
After you select Print Tickets, the Print Confirmation dialog box appears. Choose one of the following:
•
Process - All pickup tickets printed successfully.
•
Reprint All - None of the pickup tickets printed successfully. This option is also useful if only a couple of pickup tickets printed successfully from a large print job.
•
Single Reprint - Reprint pickup tickets individually. This option is useful if a couple of pickup tickets jammed in the printer.
•
Abort - Cancel all pickup ticket printing.
6
If you selected Process, the Process Verification dialog box appears. Choose one of the following:
•
Reprompt - Keep this dialog box open.
•
Yes - Complete pickup ticket processing.
•
No - Return to step 5.
