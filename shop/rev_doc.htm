Scheduling Documents screen
The Scheduling Documents screen allows you to review the hours, customer, and job assigned to the technician highlighted in the Daily Resource Schedule screen.
Menu Path: Add. Modules > Shop Repair >> Shop Repair Order Entry > Shop Repair Job Entry > F9-Schedule > F5-Review > F5-Review
Field/Function Key Descriptions
 
Field/Function Key
Description
Start
The time that the technician is to begin work on the job.
End
The estimated time that the technician should finish the job.
Customer
The customer assigned to the job.
Description
The customer description.
Document
The shop repair order number that this job is associated with.
Job
The job number from the shop repair order.
Comment
Any comments entered in the Schedule Resource screen that pertain to this technician or job.
F2-Save
Closes the screen.
F5-Review
Allows you to review the shop repair job.
F6-Edit
Allows you to edit the shop repair job.
