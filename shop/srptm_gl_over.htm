Shop/Repair Payment Type G/L Overrides screen
When you create a pay type, the default company-level general ledger settings apply. You can override those settings in the Shop/Repair Payment Type G/L Overrides screen.
Menu Path: Shop Repair > Maintenance > Pay Type Maintenance > F5-G/L Overrides
Field/Function Key Descriptions
Field/Function Key
Description
G/L Table Headings
The name of the general ledger account.
Div
The highest possible grouping in your company (e.g., Eastern Division, Western Division)
Reg
A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
Loc
The warehouse (e.g., Dallas, Tampa).
Dep
A subset of location/warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
Ccr
A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
Account#
The general ledger account number.
Account Description
A description of the general ledger account.
F2-Save
Save the G/L overrides.
F9-Goto
Allows you to identify a reference number on the screen that you want to jump to. Valid codes are account number, B (bottom), or T (top).
Action Bar
Print
Prints the G/L overrides.
