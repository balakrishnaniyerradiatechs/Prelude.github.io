Shop/Repair Timeclock Inquiry
The Shop/Repair Timeclock Inquiry shows all jobs and codes that a technician clocked into during a day. For more information, refer to Using the Shop TimeClock and Billing Jobs for Technicians.
Viewing a Technicianâ€™s Time
To view jobs clocked into by a technician:
1
Select Shop Repair > Processes > Shop Time Clock> Shop Repair Timeclock Inquiry to open the Shop/Repair Timeclock Inquiry screen.
2
In Technician#, enter the code for the technician for which you want to view timeclock entries.
3
For Date, choose the date for which you want to view timeclock entries.
Timeclock entries for that technician and date appear.
Field/Function Key Descriptions
Field/Function Key
Description
Technician#
The code for the technician clocking into the job or code. For more information, refer to Setting up Technicians.
Date
The current date and day of week.
Job/Code
Either the time clock code for non-job related time (e.g,. SW for start work) or the order number and job being worked on. Order number and jobs are entered using the pattern <order>!<job>. For example, job B for order 98650 is 98650!B.
Description
A description of the time clock code (e.g., start work) or the customer number and name if an order and job number were entered.
Start
The beginning time. The stop time is calculated when the technician clocks into the next job or code.
End
The time that the technician stopped working on the job or the code.
Length
The total time that the technician worked on the job or code.
Total Hours
The total hours the technician worked for the day.
F2-Finish
Closes the screen.
