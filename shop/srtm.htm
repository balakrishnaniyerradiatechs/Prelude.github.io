Shop/Repair Timeclock Maintenance screen
If a technician makes a mistake in Shop Repair Timeclock Entry, you can use Shop/Repair Timeclock Maintenance to correct the error. If the error was made in Technician Job Billing Entry, you should correct the error in Technician Job Billing Entry.
For more information, refer to Changing Time Clock Entries.
Menu Path: Shop Repair > Processes > Shop Time Clock > Shop Repair Timeclock Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Technician#
The code for the technician clocking into the job or code. For more information, refer to Setting up Technicians.
Date
The current date.
Job/Code
Either the time clock code for non-job related time (e.g,. SW for start work) or the order number and job being worked on. Order number and jobs are entered using the pattern <order>!<job>. For example, to clock into job B for order 98650, enter 98650!B.
Description
A description of the time clock code (e.g., start work) or the customer number and name if an order and job number was entered.
Start
The time that the technician started working on the job or the code.
End
The time that the technician stopped working on the job or the code.
Length
The total time that the technician worked on the job or code.
Total Hours
The total hours the technician worked for the day.
F2-Finish
Saves the changes.
