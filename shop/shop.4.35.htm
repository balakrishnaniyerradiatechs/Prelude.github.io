Processing Invoices
 
If you continuously print invoices, you do not need to use this procedure.
To print and process invoices:
1
Select Order Processing > Invoice Preview Printing > Invoice Printing to open the Invoice Print screen.
2
Choose one of the following:
•
Enter an Order Number to print an invoice for only that order.
•
Leave Order Number blank to print invoices for all open invoices.
3
Enter the Warehouse to print invoices from one selling warehouse or leave blank for all, Invoice Date, and Printer Form Queue.
4
For Previewed?, enter Y if you only want to print invoices that were previewed on the Invoice Preview Report (refer to Previewing Invoices).
5
For Print Source, choose which of the following documents to print:
•
B - Invoices and credit memos.
•
C - Credit memos only.
•
I - Invoices only.
•
M - Mail copies of counter invoices to customers whose Mail Invoices flag is set to Y in Customer Maintenance.
6
Press ENTER. The Load Forms dialog box appears.
7
Choose one of the following:
•
Print Invoices - Prints the invoices.
•
Mask Print Test - Tests the alignment of the invoice forms in the printer. Not necessary for laser printers. After the forms are correctly aligned, select Print Invoices.
•
Abort Printing - Cancels the process. No invoices print.
8
After you select Print Invoices, the Invoice Printing Confirmation dialog box appears.
9
Choose one of the following:
•
Process - If the invoices printed successfully.
•
Reprint All - If none of the invoices printed successfully. This option is also useful if only a couple of invoices printed successfully from a large print job. The system reprints the invoices and this dialog box appears again.
•
Single Reprint - If you need to reprint invoices individually. This option is useful if a couple of invoices jammed in the printer. Enter the sales order number for the invoice you want to reprint. This dialog box appears again.
•
Abort - To cancel all invoice updating.
10
If you selected Process, the Process Verification dialog box appears.
11
Choose one of the following:
•
Reprompt - Keeps this dialog box open.
•
Yes - Completes invoice processing, updates the Order Log, and moves the order to Shop Repair Order History (refer to History Shop Repair Inquiry).
•
No - Returns to step 8.
 
If the pay type for an invoiced job is identified as a warranty pay type, a vendor claim is automatically created for the job. Refer to Creating Pay Types.
