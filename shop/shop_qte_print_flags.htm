Shop Document Selection screen (Quotes)
When you save a quote, you can choose to print a pick up ticket and/or delivery ticket. The Shop Document Selection screen allows you to select which of these documents you want to print.
Menu Path: Shop Repair > Shop Quote Subsystem > Shop Repair Quote Entry > F2-Save > F2-Save > Print
Field/Function Key Descriptions
Field/Function Key
Description
Internal Shop Ticket
Indicates if you want an internal repair ticket:
•
Y - Print an internal repair ticket which prints information about each job and lists the products required. Extra space is provided so you can write in any missing data.
•
N - Do not print an internal repair ticket.
External Shop Ticket
The same as an Internal Shop Ticket, but only includes pricing for jobs with a Pay Type for which Charge Customer is set to Y.
Pickup Document
Indicates if you want to print a pickup document:
•
Y - Print a pickup document. Instructions may also be entered.
•
N - Do not print a pickup document.
For more information, refer to Printing Shop Repair Tickets.
Pickup Date
The date on which the model that needs to be repaired should be picked up. Available only if Pickup Document is Y.
Delivery Document
Indicates if you want to print a delivery document:
•
Y - Print a delivery document. Instructions may also be entered.
•
N - Do not print a delivery document.
For more information, refer to Printing Shop Repair Tickets.
Delivery Date
The date on which the repaired model should be delivered. Available only if Delivery Document is Y.
F2- Save
Generates the documents indicated.
