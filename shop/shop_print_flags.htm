Shop Document Selection screen
When you save an order, you can choose to print an internal shop ticket, external shop ticket, pick up ticket, delivery ticket, pick ticket, and/or invoice preview. Use the Shop Document Selection screen to select which of these documents you want to print.
Menu Path: Shop Repair > Shop Repair Order Entry > F2-Save > F2-Save > Print
Field/Function Key Descriptions
Field/Function Key
Description
Internal Shop Ticket
Indicates if you want to print an internal shop ticket:
•
Y - Print an internal shop ticket.
•
N - Do not print an internal shop ticket.
For more information, refer to Printing Shop Repair Tickets.
External Shop Ticket
Indicates if you want to print an external shop ticket:
•
Y - Print an external shop ticket. If the pay type for a job is set to charge the customer, prices appear on the ticket. Refer to Creating Pay Types.
•
N - Do not print an external shop ticket.
For more information, refer to Printing Shop Repair Tickets.
Pickup Document
Indicates if you want to print a pickup document:
•
Y - Print a pickup document. Instructions may also be entered.
•
N - Do not print a pickup document.
For more information, refer to Printing Shop Repair Tickets.
Pickup Date
The date on which the model that needs to be repaired should be picked up. Available only if Pickup Document is Y.
Delivery Document
Indicates if you want to print a delivery document:
•
Y - Print a delivery document. Instructions may also be entered.
•
N - Do not print a delivery document.
For more information, refer to Printing Shop Repair Tickets.
Delivery Date
The date the repaired model should be delivered. Available only if Delivery Document is Y.
Invoice Preview
Indicates if you want to print an invoice preview. Does not invoice the order.
Pick Ticket
Indicates if you want to print a pick ticket:
•
Y - Print a pick ticket.
•
N - Do not print a pick ticket.
For more information, refer to Printing Shop Repair Tickets.
Inspection Ticket
Indicates if you want to print an inspection ticket:
•
Y - Print an inspection ticket.
•
N - Do not print an inspection ticket.
For more information, refer to Printing Shop Repair Tickets.
F2- Save
Generates the documents indicated.
