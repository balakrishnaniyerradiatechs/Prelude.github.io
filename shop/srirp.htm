Printing Inspection Tickets
The inspection ticket lists the equipment required for the repair and the type of inspection that is needed. This information is set up for each piece of equipment in Sub-Serial Number Maintenance (refer to Setting up Sub-Serial Numbers).
If you are continuously printing inspection tickets, do not use this process. They print immediately after you flag them to print. Refer to Saving the Shop Repair Order.
To batch print inspection tickets:
1
Select Shop Repair > Batch Printing > Shop Repair Inspection Report Printing to open the Shop Repair Inspection Report Printing screen.
2
Enter the following selection criteria as necessary:
•
Shop Order Number - The number of the order.
•
Division - The division number for the order(s).
•
Region - The region number for the order(s).
•
Selling Warehouse - The warehouse that will receive credit for the repair.
•
Department - The department number for the order(s).
•
Cost Center - The cost center number for the order(s).
•
Customer - The customer number(s) for the orders.
•
Printer Form Queue - The printer that the repair tickets should print to.
3
Press F2-Save and choose Execute. A dialog box with a message similar to the following appears:
1 Records Selected For Shop Repair Inspection Printing! Load Shop Ticket Forms In Printer & Align Visually.
4
Choose one of the following:
•
Print Tickets - Continue with printing.
•
Abort Print - Stop printing. No inspection tickets print.
5
After you select Print Tickets, the Print Confirmation dialog box appears. Choose one of the following:
•
Process - All inspection tickets printed successfully.
•
Reprint All - None of the inspection tickets printed successfully. This option is also useful if only a couple of inspection tickets printed successfully from a large print job.
•
Single Reprint - Reprints inspection tickets individually. This option is useful if a couple of inspection tickets jammed in the printer.
•
Abort - Cancels all delivery printing.
6
If you selected Process, the Process Verification dialog box appears. Choose one of the following:
•
Reprompt - Keep this dialog box open.
•
Yes - Complete inspection ticket processing.
•
No - Return to step 5.
