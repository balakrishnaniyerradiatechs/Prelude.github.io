Warranty Information screen
If the pay type selected for a job is identified as a warranty pay type you can enter warranty information for the job in the Warranty Information screen.
Menu Path: Shop Repair > Shop Repair Order Entry > Job Entry > F7-Warranty
Field/Function Key Descriptions
Field/Function Key
Description
Submit Electronic
Indicates if the warranty claim should be submitted electronically to the vendor:
•
Y - Submit the transaction electronically to the vendor for warranty approval.
•
N - Do not submit the transaction electronically.
Warranty Vendor
The number for the vendor that should receive the warranty claim.
Ownership Flag
The ownership status type of the equipment being repaired:
•
C - Customer owned.
•
D - Demo unit.
•
P - Parts previously owned.
•
S - Stock unit.
Main SRT Code
The SRT (Standard Rate Table) code published by the vendor, which represents individual labor operations on models. Refer to Entering Standard Repair and Setup Time for Products.
Failed Product #
The product/model number that is defective.
Damage Code
The damage code that describes why the part is broken. Refer to Setting Up Damage Codes.
Warranty Claim #
The warranty claim number from the vendor, if available.
Policy Approval #
The authorization code from the vendor to allow payment of the warranty claim if the item is out of warranty or the repair exceeds the allotted SRT time.
Model Override
The model number of the main part if the part being repaired is an attachment.
Serial Override
The serial number of the main part if the part being repaired is an attachment.
F2-Save
Saves the warranty information.
