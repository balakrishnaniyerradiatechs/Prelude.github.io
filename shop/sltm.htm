Shop Labor Type Maintenance screen
If you use Technician Job Billing Entry to enter time clocked into a job, you must identify shop labor types (e.g., shop labor, field labor). These types are created in the Shop Labor Type Maintenance screen. When a technician enters time against a job, a shop labor type must be specified (each shop labor type must be set up as a product in Product Maintenance). Labor charges are automatically entered on the shop repair order based on the shop labor type entered and the technician’s rate.
Menu Path: Shop Repair > Maintenance > Shop Labor Type Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Default Product
The product number to default into Technician Job Billing Entry. For more information, refer to Billing Jobs for Technicians.
Product
All available product numbers that could be used as labor types in Technician Job Billing Entry.
Description
The description of the type of labor.
Type
The type of pay rate to use with the technician’s rate to calculate the cost for this type of labor:
•
R - Regular.
•
O - Overtime.
•
P - Premium.
Refer to Setting up Technicians.
F2-Save
Saves the labor types.
F4-Del
Deletes the Default Product#.
