Shop Time Clock Code Maintenance screen
If you use Shop Timeclock Entry to record your technicians’ time during the day, you must set up timeclock codes to record time that is not spent working on a job (e.g., cleaning the shop, break time). You can set up these codes in the Shop Time Clock Code Maintenance screen.
Menu Path: Shop Repair > Maintenance > Shop Time Clock Code Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Code
The code the represents the time clock code. Up to 4 alphanumeric characters.
Description
A description of the code.
Paid
Indicates if the time clocked into this code is paid:
•
Y - Time clocked into this code is paid.
•
N - Time clocked into this code is not paid.
PTime
Indicates if the time clocked using this code is considered productive time:
•
Y - The time is considered productive time and will appear under the Actual Hours column on the Productivity By Technician Report.
•
N - The time is not considered as productive time.
F2-Save
Saves the time clock codes.
F4-Del
Deletes all codes.
