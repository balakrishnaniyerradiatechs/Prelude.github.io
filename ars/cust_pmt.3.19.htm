Fast Cash Receipts Work File Report
After you run the Best Match Check (refer to Matching Cash Receipts with Open Invoices), print the Fast Cash Receipts Work File Report to view reference numbers that found a match.
To list direct and potential matches:
1
Select Financial > Accounts Receivable > High Volume Cash Receipts > Work File Report to open the High Volume Cash Match Report screen.
2
For Beginning Batch and Ending Batch, enter the numbers of the first and last batches to include. Leave blank to include all batches that were entered in the Best Match Check.
3
For Display All Possible Matches?, indicate which matches to list:
•
Y - List all direct and potential matches.
•
N - List only direct matches.
4
Press ENTER and choose an output. Refer to the Column Descriptions for an explanation of the report.
After you run and view the report, if you still have unmatched invoices, choose from the following:
•
Re-run the Best Match check and choose different settings for Maximum Matching and Quick Match Only to search on more options. Refer to Matching Cash Receipts with Open Invoices.
•
Manually search for an invoice and edit any discrepancies in High volume Cash Receipts Maintenance. Refer to Manually Matching and Editing Invoices.
Column Descriptions
Column Heading
Description
Reference Number
The reference number entered in the High Volume Cash Receipts Detail screen. Refer to Entering a High Volume Cash Receipts Batch. These numbers appear below the Batch#.
Amount
The amount to be paid toward the invoice. These amounts appear below the Check#.
Inv Number
The matching invoice number, if a direct match was found. If a direct match was not found, asterisks (*) appear.
Discrepancy$
The difference between the invoice amount in High Volume Cash Receipts Detail Entry and the amount on the matched invoice.
Inv List
The list of matching invoices. If left blank, no invoice was found.
Source
Indicates how the Reference Number was matched:
•
CPO - Customer P/O number.
•
ORD - Order number.
•
REL - Release number.
•
INV - Invoice number.
•
GROS - Gross amount.
•
NET - Net amount.
•
Blank - A direct hit was found.
Invoice Bal
The remaining balance due on the invoice.
Batch#
The number of the batch.
Seq#
The transaction number for a customer’s invoices.
Code
The code for the bank.
Check#
The number of the customer’s check.
Check Amount
The amount of the customer’s payment.
Chk Date
The date the payment was issued.
Cust#
The customer’s number.
Description
The customer’s name.
