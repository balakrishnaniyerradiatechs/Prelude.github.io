Warehouse Maintenance
You can set an oversize order limit by warehouse so that any orders over a certain value automatically go on credit hold. This value in Warehouse Maintenance overrides the value set in Order Processing Parameters (refer to Order Processing Parameters), but it can be overridden by customer (refer to Customer Maintenance).
To set an oversize order limit by warehouse:
1
Select Warehouse Maintenance > F5-Add Info to open the Additional Information screen.
2
For Oversized Order Value, enter the value that determines an oversized order. Any amount over this places the order on credit hold.
3
Press F2-Save twice to save the order credit hold information.
