High Volume Cash Receipts Maintenance screen
After you run and view the Fast Cash Receipts Work File Report, you can use High Volume Cash Receipts Maintenance to edit any errors.
Menu Path: Financial > Accounts Receivable > High Volume Cash Receipts > Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Batch Number
The batch number to edit.
Batch Amount
The total amount of checks in the batch.
Deposit Date
The date the cash was deposited.
Entered Amt
The amount of cash applied in the batch.
Bank Code
The code for the bank where the cash is deposited.
Deposit Comments
Any comments regarding the batch.
