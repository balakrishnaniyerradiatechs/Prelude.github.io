Setting up Financial References
Financial references identify who provides financial information about customers. Once you create financial references in Financial Reference Maintenance, you can assign them to customers (Customer Maintenance > F6-A/R > Action Bar > Cred Ref) to track financial requests and responses.
To setup a financial reference:
1
Select Financial > Accounts Receivable > Maintenance > Financial References Maintenance to open the Financial Reference Maintenance screen.
2
For Financial Reference, enter a number that you will use in the Customer Maintenance Financial Reference screen to identify the financial institution. Choose from the following:
•
Enter New to let the system generate the next sequential number.
•
Enter any alphanumeric number, up to 6 digits, to identify the reference.
•
Press F3 to list existing references to select from.
3
Enter a Description of the financial reference.
4
Enter the address, contact, and phone/fax number information for the reference.
5
For Credit Request, determine how you normally transmit requests to the reference. Choose from the following:
•
To be faxed.
•
To be mailed.
•
To be contacted by telephone.
6
Press F2-Save to save the financial reference information.
Refer to “Customer Accounts” in the Order Processing documentation for more information.
