User Id Maintenance
Employees must be set up as a credit manager to be able to release orders from credit hold.
To set up an employee as a credit manager:
1
From the Systems menu, select User Identification Maintenance to open the User Identification Maintenance screen.
2
Enter the User Id Code of the employee to set up as a credit manager.
3
Press F5- Administrative to open the Administrative Settings screen.
4
For Credit Manager, choose from the following:
•
Y - The user is a credit manager and can release orders from credit hold.
•
A - The same as Y but the user also has the option to release an order from credit hold from within Sales Order Entry.
5
In the next field, enter the Credit Manager ID (created in Manager Maintenance) to assign to this user. The user will be able to release orders on credit hold for the warehouses assigned to this credit manager in Warehouse Manager Maintenance.
6
In the next field, choose from the following:
•
Y - The user can enter other manager codes in the Credit Release Inquiry. Refer to Releasing Orders From Credit Hold.
•
N - The user can only view and release orders from his/her own credit release queue.
•
Press F2-Save 2 times to save the credit manager information.
