High Volume Cash Detail Maintenance screen
You can use the High Volume Cash Detail Maintenance screen to identify the correct match for each customer invoice and to edit any errors. Refer to Manually Matching and Editing Invoices for more information.
Menu Path: Financial > Accounts Receivable > High Volume Cash Receipts > Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Transaction Number
The transaction number to edit.
Transaction Count
 
Customer
The customer associated with the batch and transaction number.
Check Number
The check number used by the customer.
Check Amount
The amount of the customer’s check.
Check Date
The date of the customer’s check.
Totals
The totals of the Amount and the Discount.
F2-Save
Saves the cash receipt information.
F5-Get Invoice List
invoices that matched multiple reference numbers. You can identify the correct invoice.
F6-ARI
Opens the Accounts Receivable Inquiry to allow you to view accounts receivable balance and payment information.
F8-Add Comment
Allows you to enter a comment regarding this cash receipts batch.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
