Monthly Cash Receipts History screen
The Monthly Cash Receipts History screen lists the fiscal periods in which checks were received from a customer. You can view detail check information for each check.
Menu Path: Financial > Accounts Receivable > Inquiries > Account Receivable Inquiry > Action Bar > Inquiries > Monthly Cash Receipts History
Field and Function Key Descriptions
Field/Function Key
Description
Period
The fiscal period in which cash was received.
# Cash Rec
The number of checks received in the fiscal period.
First Check
The check number of the first check received in the fiscal period.
F2-Finish
Closes the Monthly Cash Receipts History screen.
F5-Rev Checks
Opens the Monthly Cash Receipts Check Inquiry screen to display the check date, check amount, and associated batch number for each check sent by the customer.
F6-Ent Check#
Allows you to enter a check number to review.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
