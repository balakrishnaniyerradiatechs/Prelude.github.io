Additional Option Parameters
To set additional accounts receivable system options:
1
In Additional Option Parameters, select F5-Addit Info to open the Additional Information screen.
2
For Expiration Days for General Comments, enter a number to override the system default (45) for the number of days before a general comment expires. You can override this value for each note.
When the number of Expiration Days in Credit Collections Schedule Entry or in Credit Collections Notes Entry is reached, the comment is removed from collection history.
3
For Include Disputed Invoices in Collections, indicate if disputed invoices are included in credit collections:
•
Y - Include disputed invoices in the collection queue.
•
N - Do not include disputed invoices in the collection queue.
Marking an Invoice as Disputed
You can mark the status of an invoice as disputed in Manual Invoice Entry (refer to Manually Entering an Invoice) or in Accounts Receivable Maintenance (refer to Editing an A/R Invoice).
4
For Include Credits in Collections, determine if credits are included in the past due balance when Credit Collections Nightly Processing generates credit collection letters:
•
Y - Credit Collections Nightly Processing includes credits to determine if a collection letter should be generated.
•
N - Credit Collections Nightly Processing excludes credits to determine if a collection letter should be generated.
Refer to Generating Scheduled Collection Letters.
5
Press F2-Save twice to save the additional option parameters.
