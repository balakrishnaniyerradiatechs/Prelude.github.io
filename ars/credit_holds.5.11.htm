Setting up Credit Managers
Your user ID must be set up as a credit manager to release orders that are on credit hold. Credit manager IDs are set up in Manager Maintenance and then assigned to a warehouse in Warehouse Manager Maintenance and to customers in Customer Maintenance.
Credit Manager Assignment Hierarchy:
•
Customer Level: Customer Maintenance > F6 > Credit Manager ID. (refer to Customer Maintenance).
•
System Level: Systems Menu > Accounts Receivable Controls > Credit Manager Assignment Matrix > Cred Mgr (defines the credit manager by alphabetical name range).
•
Warehouse Level: Systems Menu > Warehouse Manager Maintenance > Credit Manager (refer to Warehouse Manager Maintenance).
Refer to the following maintenance programs for information on setting up credit managers:
•
Manager Maintenance.
•
Warehouse Manager Maintenance.
•
User Id Maintenance.
