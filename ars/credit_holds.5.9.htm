Credit Authorization Code Maintenance
If Credit Check in Company Maintenance is set to A or O (authorization required), you must designate the user IDs that are allowed to enter sales orders for customers with a Credit Code of A.
To assign Authorization Code user IDs:
1
Select Accounts Receivable Controls > Authorization Codes to open the Credit Authorization Maintenance screen.
2
For Authorization Code, enter the user IDs of the employees that are allowed to enter sales orders for customers with a Credit Code of A (refer to Customer Maintenance).
3
Press F2-Save to save the authorization code information.
