Accounts Receivable Parameters
To set accounts receivable system options:
1
Select Option Parameters > Accounts Receivable Parameters to open the Accounts Receivable Parameters screen.
2
For Min Cred Coll Amt, enter a value so that any customer whose past due A/R balance is below this amount is not included in Credit Collections Nightly Processing. You can override this value for a customer in Customer Maintenance.
3
For Bypass Security Access in Coll, indicate if security access codes are required in Credit Collections Schedule Entry:
•
Y - Bypass security access in the collection subsystem. All customers appear in the activity collection schedule, not just those associated with a security access code.
•
N - Require security codes. Only customers with a matching security code appear in the activity collection schedule.
Refer to Customer Maintenance for information on assigning security access codes to a customer.
4
For Collections Eval Method, determine how customers pass through activities in Credit Collections Schedule Entry.
•
A - Action method. Moves customers through activities only if they were reviewed (for the call queue) or had a letter generated (for a correspondence activity).
•
T - Time method. Moves customers based on the number of days that passed, regardless of whether or not they were reviewed. For example, if the customer is in a queue but no action is taken, they will still appear in the next queue in the number of days assigned to their credit collections type.
5
Press F2-Save to save the A/R parameter information.
