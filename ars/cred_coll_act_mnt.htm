Credit Collections Activity Maintenance screen
You can use the Credit Collections Activity Maintenance screen to create activity codes to identify each type of activity used to collect on past due invoices (e.g., phone call, collection letter). These codes are used in Sequencing Follow-Up Collection Activities to define the sequence of collection activities to follow when invoices are past due.
Menu Path: Financial > Accounts Receivable > Collections Subsystem > Maintenance > Credit Collections Activity Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Activity Code
The alphanumeric code to identify an activity.
Description
The description of the activity.
Correspondence Document?
Indicates if this activity involves a letter:
•
N - This activity does not generate a letter. The activity appears in Credit Collections Schedule Entry (refer to Entering Scheduled Credit Collection Activities).
•
Y - This activity generates a letter (refer to Generating Scheduled Collection Letters).
Number Of Copies
The number of copies of the letter to send.
Salesrep Letter?
Indicates if the letter should go to the customer’s sales representative:
•
Y - Send a copy of the letter to the customer’s sales representative.
•
N - Do not send a copy of the letter to the customer’s sales representative.
F2-Save
Saves the activity code information.
F4-Delete
Deletes the activity code information.
F5-Document Text
Opens the Credit Collections Correspondence Definition screen to allow you to enter or edit a letter regarding the disputed invoices. Refer to Creating Credit Collections Activity Codes for more information.
