Charge Back Reason Code Maintenance screen
The Charge Back Reason Code Maintenance screen allows you to create reason codes to use when you create a charge back to a customer in Cash Receipt Entry. If you want to charge the amount back to the customer, you must enter a charge back reason code. Refer to Creating Charge Back Reason Codes for more information.
Menu Path: Financial > Accounts Receivable > Maintenance > Charge Back Reason Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Charge Back Reason
The 1 or 2 digit alphanumeric code to use when charging back a customer in Cash Receipts Entry.
Description
The explanation for why you are charging the customer back for the unpaid amount (e.g., customer did not pay freight, customer did not pay tax, NSF bank charge).
Comments
Any additional comments to further explain the charge back reason.
F2-Save
Saves the charge back reason code information.
F4-Del
Deletes the charge back reason code.
