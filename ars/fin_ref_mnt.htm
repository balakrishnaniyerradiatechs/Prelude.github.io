Financial Reference Maintenance screen
Financial references identify who provides financial information about customers. Once you create financial references in Financial Reference Maintenance, you can assign them to customers (Customer Maintenance > F6-A/R > Action Bar > Cred Ref) to track financial requests and responses.
Menu Path: Accounts Receivable > Financial > Maintenance > Financial References Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Financial Reference Number
An alphanumeric number, up to 6 digits, used to identify the financial institution. You can assign this number to customers in the Customer Maintenance Financial Reference screen.
Description
A description of the financial institution.
Address One
The first line of address for the financial institution.
Address Two
The second line of address for the financial institution.
Address Three
The third line of address for the financial institution.
City
The city of the financial institution.
State
The state of the financial institution.
Zip
The zip code for the financial institution.
Contact
The name of the contact at the financial institution.
Phone
The contact’s phone number.
Fax
The contact’s fax number.
Credit Request
Determines how you normally transmit requests to the financial reference:
•
F - Fax.
•
M - Mail.
•
T - Telephone.
F2-Save
Saves the financial reference information.
F4-Del
Deletes the financial reference.
