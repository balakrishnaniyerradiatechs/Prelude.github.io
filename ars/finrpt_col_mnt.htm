Financial Report Column Maintenance screen
The Financial Report Column Maintenance screen allows you to define the column descriptions for the financial reports. Refer to the Financial Report Manager documentation in the General Ledger Module for more information.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Column Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Column
The column types for the financial reports.
Abbreviations/Definitions:
•
TM - this month.
•
TQ - this quarter
•
TY or CY - this/current year.
•
LM or PM - last/previous month.
•
LY or PY - last/previous year.
•
YT or YTD - year-to-date.
•
B1-4 - budget 1, 2, 3, or 4 from Chart of Accounts Maintenance.
•
Basis - the G/L account(s) used to compare to each row on the report.
•
Variance - difference in amount (e.g., dollars).
•
Activity - credits and debits to date for the month or fiscal period.
Description
The descriptions of the column.
Column Heading
The column headings:
•
Column Heading 1 - This text is printed at the top of each report as the first line of description.
•
Column Heading 2 - This text is printed at the top of each report as the second line of description.
F2-Save
Saves the column headings.
