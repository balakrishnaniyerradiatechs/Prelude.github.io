Cash Receipts Status Display screen
The Cash Receipts Status Display screen allows you to review the status of each check and to verify that the amounts are correct. If you access a suspended cash receipts batch, you can view check numbers and cash amounts already applied for a customer.
Warning:
Even if you suspend the batch, the general ledger has already been posted to and the invoices no longer show as open on the customerâ€™s A/R aging. Run the Incomplete Cash Receipts Report weekly to view and complete the suspended batches
Menu Path: Financial > Accounts Receivable > Cash Receipts Entry > F6-Status Display
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The numbers for the customers whose invoices are included in the cash batch.
Description
The customer description.
Check Number
The numbers of the checks included in the batch.
Check Amount
The amount of the check.
F2-Finish
Closes the screen and returns the Cash Receipts Entry screen.
F5-Review Detail
Opens the Cash Receipts Detail Display screen to allow you to view detail payment information for each invoice in the batch.
