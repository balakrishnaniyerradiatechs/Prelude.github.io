Best Match screen
After you create a high volume batch and identify invoices (refer to Entering a High Volume Cash Receipts Batch), you can use the Best Match screen to match each reference to an open A/R invoice. Since this process can take a while to run; you may need to run it overnight.
Menu Path: Financial > Accounts Receivable > High Volume Cash Receipts > Best Match Check
Field and Function Key Descriptions
Field/Function Key
Description
Maximum Matching?
Determines the fields to match on:
•
Yes - Find the invoice match using the sales order number, the invoice number, the customer P/O number, the release number, the gross amount, or the net amount.
•
No - Same as the Yes option except that it will not search on the gross amount or the net amount.
Quick Match Only?
Determines if you want to quickly search for a direct match to the invoice number:
•
Yes - Match on invoice numbers and total open balance for a customer. If one invoice fails to match for a customer, all of that customer’s invoices are unmatched. All invoices must be on file for the same customer and they must have the exact open balance. After you match invoices, you can run the Work File Report to determine whether to run a complete Best Match Check or correct any discrepancies manually. Refer to the Fast Cash Receipts Work File Report.
•
No - Match on all reference numbers. May be necessary if the quick match did not find any invoices. You may want to use this option if the customer has multiple invoices with the same amount.
Batch Number
All the batches for which you want to check for a match. Leave blank to select all open batches.
F2-Save
Begins matching invoices and payments. The system checks the open A/R invoices to search by invoice, order, customer P/O, or release number. It also checks for gross or net dollar matches.
