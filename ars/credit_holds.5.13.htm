Warehouse Manager Maintenance
Once you create a credit manager code in Manager Maintenance, you can assign the manager code to a warehouse. This credit manager will be the default credit manager for all customers that have this warehouse as their selling warehouse. You can override a credit manager for a customer in Customer Maintenance.
To assign a credit manager code to a warehouse:
1
From the Systems menu, select Warehouse Manager Maintenance to open the Warehouse Manager Maintenance screen.
2
Enter the Warehouse to which you want to assign a credit manager.
3
Enter the Credit Manager code to assign to this warehouse.
4
Press F2-Save to save the warehouse manager information.
