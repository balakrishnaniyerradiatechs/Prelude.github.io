Credit Collections Schedule Entry screen
The Credit Collections Schedule Entry screen allows you to view A/R information for customers that are scheduled for follow-up credit collection activities. From this screen you can access the Credit Collections Customer Entry screen to view and enter activity information for the customer. Refer to Entering Scheduled Credit Collection Activities for more information.
Menu Path: Financial > Accounts Receivable > Collections Subsystem > Credit Collections Schedule Entry
Field and Function Key Descriptions
Field/Function Key
Description
Credit Collections Type
Determines the customers to include. Credit collections types are created in Credit Collections Type Maintenance (refer to Defining Credit Collections Types) and assigned to customers in Customer Maintenance.
Security Access Code
If you use security codes, the security code (if Bypass Security Access Code is set to N in Accounts Receivable Parameters) so that only customers with the same security code can be viewed.
Activity Code
The activity code (refer to Creating Credit Collections Activity Codes) to list all customers that have this code as their next scheduled activity. You can only enter activity codes that do not generate a document. Activities that generate a letter are scheduled by Nightly Processing (refer to Generating Scheduled Collection Letters).
Customer
The customers that have the defined Activity Code as their next scheduled activity.
Description
The customer description.
State
The customer’s state.
A/R Balance
The customer’s accounts receivable balance. Sales orders invoiced but not yet paid by the customer.
Over 30+
The value of the customer’s invoices that are over 30 days past due.
Reviewed
Indicates if the customer’s collection data was reviewed:
•
* - The customer was reviewed (press F5-Review to open the Credit Collections Customer Entry screen) but the customer was not removed from the queue.
•
N - The customer had not yet been reviewed.
Number Of Customers
The number of customers that have this Activity Code as their next scheduled activity.
F2-Finish
Close the Credit Collections Schedule Entry screen.
F5-Review
Opens the Credit Collections Customer Entry screen to allow you to view and enter credit collection activity information.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
