Credit Collections Activity Report
The Credit Collections Activity Report displays the status of customers who are delinquent on past due invoices. You can sort the report by credit collection type or activity code. You can use this report before beginning collection activities to determine which group of customers you want to contact or which set of activities you want to perform.
Menu Path: Financial > Accounts Receivable > Collections Subsystem > Reports > Credit Collections Activity Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Cred Collections Type. The collections type (refer to Defining Credit Collections Types) and assigned to each customer in Customer Maintenance. Leave blank to include all credit collection types.
•
Cust Security Access. Only accessible if Bypass Security Access in Coll flag is set to N in Accounts Receivable Parameters. If you enter a security code, only customers assigned to that security code are included.
•
Collections Activity Code. The collections activity codes if you want to only include customers with this activity as their next scheduled activity (refer to Creating Credit Collections Activity Codes). Leave blank to include all activity codes.
•
Cuttoff Date. The cutoff date so that any collection activities before this date are included on the report.
•
Days Overdue. The number of days overdue a customer’s invoices must be to appear in the report.
•
Past Due Amount. The dollar amount overdue a customer’s past due invoice must be to appear on the report.
 
Column Descriptions
Column Heading
Description
Customer
Line 1: The customer number.
Line 2: The customer description.
Activity
The credit collection activity code that identifies the type of activity currently assigned to the customer to collect on past due invoices (refer to Creating Credit Collections Activity Codes).
Date
The date the Activity was created for the customer.
Contact
The customer contact name from Customer Maintenance.
Phone
The customer’s phone number from Customer Maintenance.
A/R Bal
The total value of the customer’s open A/R balance, both current and past due.
Current
The amount of the customer’s open A/R balance that is not past due.
1 - 30
The amount of the customer’s open balance that is 1 to 30 days past due.
31 - 60
The amount of the customer’s open balance that is 31-60 days past due.
Over 60
The amount of the customer’s open balance that is over 60 days past due.
