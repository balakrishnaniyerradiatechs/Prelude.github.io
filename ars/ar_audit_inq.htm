A/R Transaction Audit Inquiry
The A/R Transaction Audit Inquiry summarizes all accounts receivable transactions for a customer. A cumulative balance displays for each credit and debit entry.
Viewing Accounts Receivable Transactions
To list accounts receivable transactions for a customer:
1
Select Financial > Accounts Receivable > Inquiries > A/R Transaction Audit Inquiry to open the A/R Transaction Audit Inquiry screen.
2
Enter the Customer whose transactions you want to view.
3
Enter the Sort Sequence to indicate how you want to sort the invoices:
•
Ending date. The newest invoices appear at the top.
•
Beginning date. Displays the oldest invoices first, stating with the beginning date that you enter.
A/R transactions for the customer appear.
Field & Function Key Descriptions
Field/Function Key
Description
Primary Contact
The customer’s contact person that is responsible for paying invoices. Designated in the Customer Maintenance screen.
Phone
The contact’s phone number.
Code
The transaction audit inquiry code that identifies the transaction. Refer to the Transaction Audit Inquiry Codes for a definition of each code.
Date
The date of the transaction.
Source
A code that identifies the source of the sales order or quote. Commonly used order source codes are:
•
C - Customer call in.
•
D - Direct order entry.
•
E - Electronic data interchange.
•
K - Special order-telemarketing.
•
L - Sales representative.
•
M - Mail in.
•
P - Point of sale.
•
R - Remote order entry.
•
S - Sales force automation.
•
T - Telephone sales.
•
W - Will call.
•
X - Special order.
Reference 1
The document number for the associated transaction (e.g., sales order number, invoice number, batch number, service charge number).
Reference 2
A second associated transaction number (e.g., sales order number, invoice number, batch number, service charge number).
Debit Amount
The amount billed to the customer on this transaction.
Credit Amount
The amount credited to the customer’s account (e.g., payment applied in Cash Receipts Entry, Credit memo issued).
Balance
The cumulative balance updated by each debit and credit entry on the screen.
F2-Finish
Closes the inquiry screen.
F5-Review
Allows you to view detail information for the highlighted transaction.
F6-Report
Allows you to print, email, or fax the A/R Transaction Audit Report, which lists the same information as what appears on this inquiry screen.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Notes
Allows you to view:
•
The Credit Collections History Inquiry which displays the credit collection notes entered in Credit Collections Notes Maintenance (refer to Entering Credit Collections Notes).
•
Customer notes entered for the customer in Customer Maintenance > Action Bar > Notes.
Mly Cr Hist
Opens the Monthly Cash Receipts History screen to list fiscal periods in which checks were received from a customer.
