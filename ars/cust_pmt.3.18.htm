Matching Cash Receipts with Open Invoices
After you create a high volume batch and identify invoices (refer to Entering a High Volume Cash Receipts Batch), you must attempt to match each reference to an open A/R invoice. Use Best Match to search for a match to the Reference Number entered in the High Volume Cash Detail Entry screen.
If Best Match does not find enough invoices, you can run it again to search for invoices using different settings. You can also manually search for any missing invoices using High Volume Cash Receipts Maintenance. Refer to Manually Matching and Editing Invoices
To match reference numbers with open invoices:
1
Select Financial > Accounts Receivable > High Volume Cash Receipts > Best Match Check to open the Best Match screen.
2
For Maximum Matching?, determine which fields to match on:
•
Yes - Find the invoice match using the sales order number, the invoice number, the customer P/O number, the release number, the gross amount, or the net amount.
•
No - Same as the Yes option except that it will not search on the gross amount or the net amount. You may want to use this option if the customer has multiple invoices with the same amount.
3
For Quick Match Only?, determine if you want to run a quick search for a direct match to the invoice number:
•
Yes - Match on invoice numbers and total open balance for a customer. If one invoice fails to match, all invoices are unmatched.
•
No - Match on all reference numbers (e.g., invoice number, customer P/O number, order number, or release number).
After you match invoices and run the Fast Cash Receipts Work File Report, you can determine if you need to re-run the Best Match Check and answer N to Quick Match Only?.
4
For Batch Number, enter all the batches for which you want to check for a match. Leave blank to select all open batches.
5
Press F2-Save to begin matching invoices and payments. Since this process can take a while to run, you may need to run it overnight.
