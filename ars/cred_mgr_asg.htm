Customer Credit Manager Assignment screen
The Customer Credit Manager Assignment screen allows you to override the credit manager for a warehouse for a range of customers. You can change the existing credit manager assignment for a range of customers to a new credit manager. You can override this assignment for a customer in Customer Maintenance. Refer to Assigning Credit Managers to a Group of Customers.
If you do not assign a credit manager to a customer, then the credit manager assigned to the warehouse can release the customerâ€™s orders from credit hold.
Menu Path: Financial > Accounts Receivable > Processes > Customer Credit Manager Assignment
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse for the credit manager assignment.
Beginning Customer Name
The beginning customer to include in this credit manager assignment.
Ending Customer Name
The ending customer to include in this credit manager assignment.
Old Credit Manager
The credit manager ID assigned to the customers. When the re-assignment is complete, only customers in the customer name range with this Credit Manger ID currently in Customer Maintenance will have a new credit manager. To re-assign the credit manager for all customers in the name range, leave this field blank.
New Credit Manager
The credit manager ID that you want to assign to customers in the name range, that are assigned to the Old Credit Manager.
Warning: The system does not validate the New Credit Manager that you enter. Check with your systems administrator before using this feature.
F2-Save
Re-assigns the credit manager for the selected customers.
F4-Del
Deletes the credit manager assignment.
