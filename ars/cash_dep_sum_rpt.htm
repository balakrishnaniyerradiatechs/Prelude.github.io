Cash Receipts Deposit Summary Report
The Cash Receipts Deposit Summary Report displays the total value for each batch posted for a date range.
Menu Path: Financial > Accounts Receivable > Reports >Cash Receipts Reports > Cash Receipts Deposit Summary Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Bank Code. The bank code entered when the cash receipts batch was created (refer to Beginning a Cash Receipt Batch).
•
Deposit Date Range. The beginning and ending cash receipt deposit dates to include.
Column Descriptions
Column Heading
Description
Total For Batch Number
The total value of all checks deposited in each batch.
Posted at
The time and date when the batch was posted, the user ID of the person that posted the batch, and the number of invoices to which cash was applied.
Total For Deposit Date
The total value of all batches posted for each date.
