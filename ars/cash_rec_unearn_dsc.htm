Unearned Discount screen
The Unearned Discount screen appears when you enter an unearned discount amount in Cash Receipts Entry (refer to Overriding an Unearned Discount). You must enter the unearned discount code to justify overriding the discount amount.
Related Report
Description
Unearned Cash Discounts Report
lists cash discounts that were allowed and taken even though cash was applied to the invoice after the discount due date.
 
Menu Path: Financial > Accounts Receivable > Cash Receipts Entry > Enter an unearned discount in Disc App
Field and Function Key Descriptions
Field/Function Key
Description
Unearned Disc Code
The reason code to explain why you are overriding the discount (e.g., mail delay, good customer, cash receipts late). Refer to Creating Unearned Discount Reason Codes for information on setting up codes.
F2-Save
Saves the unearned discount code information and returns to the Cash Receipts Entry screen.
