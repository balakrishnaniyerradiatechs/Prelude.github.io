Credit Collections Type Maintenance screen
Credit collection types are used to group similar customers for collections (e.g., slow paying customers, good customers). Once you create the collections types in the Credit Collections Type Maintenance screen, you can assign them to customers in Customer Maintenance.
Menu Path: Financial > Accounts Receivable > Collections Subsystem > Maintenance > Credit Collections Type Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Credit Collections Type
The alphanumeric code to identify a group of customers.
Description
The description of the credit collection type (e.g., slow paying customers, good customers).
F2-Save
Saves the credit collection type information.
F4-Del
Deletes the credit collection type.
