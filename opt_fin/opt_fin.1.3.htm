Company Maintenance
Path: System Parameter > Maintenance > Company Maintenance > F6-Modules

ABC Accounting: determines if Activity Based Costing is used within Prelude. Set this flag to ‘Y’ to use Activity Based Costing. Set this flag to ‘N’ to not use Activity Based Costing.
Activity Unit Costs Maintenance
All features in this section are in the following menu path unless otherwise noted: ABC Accounting > Activity Unit Costs Maintenance.
Company Activity Unit Costs Maintenance
This maintenance routine defines the company activity unit costs.
F5-Add’l Cost Drivers allows additional cost overrides for vendor and receiving costs.
Warehouse Activity Unit Costs Maintenance
This maintenance routine defines the warehouse activity unit costs.
F5-Add’l Cost Drivers allows additional cost overrides for vendor and receiving costs.
Miscellaneous Activity Unit Costs Maintenance
This maintenance routine defines the miscellaneous activity unit costs. These costs are used so that companies who have activities to their business can define these unit costs and allow them to be reflected in the Customer’s final Profit and Loss Inquiry.
Customer Miscellaneous Costs
This maintenance allows for updating the quantities of miscellaneous costs for each customer.
Path: ABC Accounting > Miscellaneous Costs > Customer Miscellaneous Costs
