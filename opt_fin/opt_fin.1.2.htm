ABC Unit Activity Costs and What Drives Them
Cost
Driven By...
Cost of Capital
Two processes on the General Ledger Processes End of Month Closing Menu, Monthend Inventory Balance Save and Monthend Customer A/R Aging Save, capture the monthend inventory balance for each product, product line, vendor and vendor line and the monthend A/R balance for each customer. The Unit Activity Cost defined as Cost of Capital is applied to the average of those balances to derive the actual Capital Costs.
Cost Per Credit Memo Line
Tracks each time a line item is returned to stock.
Cost Per Delivery & Cost Per Delivery Ship-Via
If you track by Ship-Via, they are used to update the Delivery Costs. If you do not track by Ship-Via, then each time an order is invoiced that is for a truck route Ship-Via this cost is updated. Based on the shipping warehouse.
Handling Expense Per Invoice
When an order is invoiced that is either a Warehouse order or an In-direct order, this cost is updated. This is based on the shipping warehouse.
Handling Expense Per Invoice Line (WHSE)
When a line item is invoiced that is a warehouse line item, this cost is updated. This is based on the shipping warehouse.
Handling Expense Per Invoice Line (IND)
When a line item is invoiced that is an indirect line item, this cost is updated. This is based on the shipping warehouse.
Handling Per Volume
When a line item is invoiced this cost is updated based on the quantity shipped and the volume defined on the product at the shipping warehouse. This applies to warehouse and indirect line items only.
Handling Per Weight
When a line item is invoiced this cost is updated based on the quantity shipped and the weight defined on the product at the shipping warehouse. This applies to warehouse and indirect line items only.
Handling Per Selling Pkg
When a line item is invoiced this cost is updated based on the quantity shipped and the selling package quantity defined on the product at the shipping warehouse. This applies to warehouse and indirect items only.
Handling Per Broken Selling Pkg
When a line item is invoiced this cost is updated based on the quantity shipped and the selling package quantity on the product at the shipping warehouse. If the quantity does not equal the selling package quantity, it is a broken package. Applies to warehouse and indirect items only.
Selling Cost Per Invoice
When an order is invoiced this cost is updated.
Selling Cost Per Invoice Line
When an order line is invoiced this cost is updated.
Warehouse Cost Per Invoice
When an order is invoiced this cost is updated based on the selling warehouse.Applies to warehouse and indirect orders.
Warehouse Cost Per Invoice Line
When an order line is invoiced this cost is updated. This is based on the selling warehouse and applies to warehouse and indirect orders.
G&A Per Invoice
When an order is invoiced this cost is updated. This is based on the selling warehouse and applies to warehouse, direct, and indirect orders.
G&A Per Invoice Line
When an order line is invoiced this cost is updated. This is based on the warehouse and applies to warehouse, direct, and indirect orders.
Cost Per A/P Debit Memo
When a receiver is flagged for a Debit Memo this cost is updated. This is based on the warehouse defined when the P/O is created.
Cost Per A/P Cost Discrepancy
When a line item cost on a vendor invoice changes, this cost is updated. This is based on the warehouse defined when the P/O is created.
Cost Per Over Receipt
When a line item is received on a P/O and the quantity received is more than the quantity open, this cost will be updated based on the warehouse when the P/O is created.
Cost Per Partial Receipt
When a line item is received on a P/O and the quantity received is less than the quantity open, this cost is updated based on the warehouse when the P/O is created.
Cost Per Receipt
When a P/O is received this cost will be updated. This is based on the selling warehouse defined when the P/O is created.
Cost Per Receipt Line Item
When a line item is received this cost will be updated. This is based on the selling warehouse defined when the P/O is created.
Cost for Storage Per Unit
When a line item is received this cost will be updated based on the number of items received. This is based on the selling warehouse defined when the P/O is created. This is for warehouse and indirect line items only.
Cost for Storage Per Volume
When a line item is received this cost is updated based on the number of items received and the volume of the product at the warehouse where the P/O is created. This is for warehouse and indirect line items only.
Cost for Storage Per Weight
When a line item is received this cost is updated based on the number of items received and the weight of the product at the warehouse where the P/O is created. This is for warehouse and indirect line items only.
Cost Per Rejection
Each time a line item is received and flagged as rejected this cost will be updated. This is based on the selling warehouse defined when the P/O is created. This is for warehouse and indirect line items only.
Sales Cost Per Invoice
Each time an order is invoiced this cost is updated. This is based on the selling warehouse. This is for warehouse and indirect invoices only.
Sales Cost Per Invoice Line
Each time a line item is invoiced this cost is updated. This is based on the selling warehouse. This is for warehouse and indirect line items only.
Cost Per Sales Visit
Updated from the Contact Diary for contact type MT (Meeting). If the you can use multiple warehouses, the activity unit cost is defined at the company level; otherwise your assigned warehouse is used.
Cost Per Sales Fax
Updated from the Contact Diary for contact type FAX. If you can use multiple warehouses then the activity unit cost is defined by the company level, otherwise your assigned warehouse is used.
Cost Per Sales Call
Updated from the Contact Diary for contact type TH (Telephone) and from the Marketing module when the activity is a phone call. If you can use multiple warehouses then the activity unit cost is defined by the company level, otherwise the warehouse to which that user is assigned defines it.
Cost Per Sales Letter
Updated from the Contact Diary for contact type ML (Mailed Letter). If you can use multiple warehouses then the activity unit cost is defined at the company level, otherwise your assigned warehouse is used.
