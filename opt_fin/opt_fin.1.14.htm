Maintenance
All maintenances are in Financial > Fixed Assets > Maintenance.
Book Maintenance (/BKM)
Use Book Maintenance to add, change or delete depreciation book codes. Book codes define how depreciation is reported to the government. One book is the primary tax book and the others are just ways of viewing it. Normally only one book is used unless depreciation is calculated multiple ways on each asset. The primary book posts depreciation to the G/L during depreciation update. The 2 relevant G/L accounts are for depreciation expense and depreciation allowance.
•
Book Number: Enter the book code to be added or changed.
•
Book Description: Enter the description of the book code.
•
Primary Book: Enter Y if this book updates the G/L for depreciation.
•
F2- Save: Saves the book code input.
Class Maintenance (/CLSM)
Use this program to add, change or delete class codes, which group assets by type (e.g., office or warehouse). Assets are assigned a class and each book is assigned to an asset.
•
Class Number: Enter the class number to add or change. Class numbers are group fixed assets by type.
•
Class Description: If creating a new class number, enter the description.
•
Post Depr to G/L?
-
Y - To automatically post the depreciation to the general ledger.
-
N - To manually post the depreciation to the general ledger.
•
Equipment: Enter Y if this is an equipment class.
•
Asset Type: Enter R to set this asset type to rental. Leave blank for non-rental.
•
F2- Save: Saves the class code input.
•
F4 - Delete: Deletes the class code input.
•
F5 G/L Accounts: Allows the class code to be posted to a different G/L account.
Assets Maintenance (/ASM)
Creates, changes or deletes fixed assets master files.
•
Asset Number: Enter the asset number to be created or maintained.
•
EIN: If the rental or equipment module is being used, this is the equipment I.D.
•
Asset Description: Enter a description of the asset number.
•
Class Number: Enter a class for the asset, which may be used to group fixed assets.
•
Vendor Number: Enter the 6 digit vendor number to identify the source.
•
Serial number: Enter the serial number of the asset.
•
Path: Enter the location number for G/L transactions involving this asset.
•
Warehouse number: Enter the warehouse where the asset is located.
•
New / Used:
-
N - New asset.
-
U - Used asset.
•
Acquire date: Enter the date the asset was acquired by the warehouse. This calculates the life of the assets.
•
Install cost: Enter the original installation cost of the asset.
•
Tax Credit: Designates the allocation of the tax credit. Enter one of the following:
-
A - If the tax credit was assigned to a lessor.
-
N - If the tax credit is not to be taken.
-
T - If the tax credit has already been taken.
-
Y - If the tax credit is to be taken.
•
Post Depr. to G/L?
-
Y - If depreciation should automatically be posted to the General Ledger.
-
N - To not post depreciation to the general ledger.
•
Equipment:
-
Y - If this asset is considered to be equipment.
-
N - If not equipment.
•
Last Post date? This field is updated with end of period depreciation. Not accessible.
•
F5 - G/L Accounts: Assets can be posted to a different G/L number if necessary.
•
F6 - Disposals: Stops updating and depreciating the asset. Enter the disposal date, the disposal code and the disposal proceeds (value of the asset at the time of disposal).
•
F7 - Books: Displays the Assets Book Maintenance screen to depreciate the asset.
•
Book Number: Enter the book number to be used for depreciation of this asset.
•
Depreciation Method: Enter the depreciation method to be used for this asset.
-
M - Manual update, use F6 for Manual entry (see below).
-
U - Units of Usage depreciation, use F8 for units entry (see below).
-
F3 - For a listing of all user defined depreciation methods.
•
Begin Depr. Date: Enter the date the depreciation began on this asset.
•
Install cost: Displays installation cost from the asset record.
•
Salvage Value: Value of the asset after full depreciation.
•
Bonus Depreciation:
-
N - The normal setting in which case the fields under it would not be used.
-
Y - If this asset is eligible for bonus depreciation from the government.
•
Life in YY/MM: Enter the assets life span as a four digit number, where YY is the number of months and MM is the number of years: i.e; 1 year and 4 months would be 0104.
•
Opening Reserve: If this asset is coming from an old system, enter the accumulated depreciation. Enter the value of the asset at the beginning of the current fiscal year.
•
YTD Depreciation: Enter the value of the year-to-date depreciation the asset.
•
LTD Depreciation: Enter the lifetime-to-date depreciation of this asset. Open reserve + YTD = LTD.
•
F5 - Net Value: Displays the book Net Value after depreciation.
•
F6 - Manual Entry: If the Depreciation Method is set to M for manual entry, this can be used to catch up on past depreciation that was not posted. Enter the amount of the current depreciation including the amount for the current fiscal period. Then update depreciation using the Manual Depreciation Update function on the processes menu (/MDU.UPD). This posts the manual depreciation amount to both the YTD and LTD and the amount is cleared from the asset record.
•
F7 - List Books: Lists all available books to select the book you want to view.
•
F8 - Units Entry: If the depreciation method is set to U for units entry, enter the number of units to depreciate. Then update depreciation using Units of Usage Depreciation Update (/UUDU.UPD).
Equipment Maintenance (/EQPM)
Creates and maintains maintenance schedules for equipment assets. Lease information, maintenance information and profitability statistics on equipment repairs is stored.
•
Asset Number: Enter a valid asset number from Asset Maintenance.
•
Next Scheduled Maint: Enter the date of the next scheduled maintenance on this asset. This date and the frequency in days set the dates for future maintenance.
•
Frequency in Days: Specify in days how often this equipment is scheduled for maintenance.
•
Current Repairs
-
YTD Repairs: Total year-to-date expenses for repairs to this equipment.
-
Ltd Repairs: Total lifetime-to-date expenses for repairs to this equipment.
•
Curr Billing Jobs
-
Ytd Bill Jobs: Year-to-date job billings for this equipment.
-
Ltd Billing Jobs: Lifetime to date billings for this equipment.
•
Cur Billing Leases
-
Ytd Billing Leases: Displays the year-to-date billing leases for this equipment.
-
Ltd Billing Leases: Displays the lifetime-to-date billing leases for this equipment.
•
Current Depreciation
-
Ytd Units Used: Displays the number of units used year-to-date.
-
Ltd Units Used: Displays the number of units used lifetime-to-date.
-
Depr. Cost Per Unit: Displays the depreciation cost per unit.
•
F5 - Lease Info: Name and address information for the lessee.
•
F6 - Maint Notes: Any notes concerning maintenance on this equipment.
•
F7 - Profitability: Profitability calculated with installation cost and net value.
Depreciation Maintenance (/DPRM)
Creates and updates depreciation methods and defines the calculation method for each.
•
Depreciation Method: Enter the depreciation method to create or maintain.
•
Method Description: If entering a new method, enter the description.
•
Table/Equation: Enter the type of method:
-
T - Table method, used with F6 -Table Maintenance.
-
E - Equipment method, used with F7 - Equation Maintenance.
•
Eval By Fisc Period?: This is only accessible if the above flag is set to T. It determines if the depreciation for the fixed asset is calculated using the fiscal period table to account for the date the asset was purchased.
-
N - Do not account for the fiscal period the fixed asset was purchased. The depreciation is calculated using the standard 12 months for the percent in the depreciation method table.
-
Y - To account for the fiscal period the fixed asset was purchased. The depreciation is calculated using the number of months remaining in the year from the time the asset was purchased for the percent in the table for the depreciation method.
Note: if you use multiple fiscal periods, the Multi-Fisc Period Table Maintenance to determine the depreciation year and percent from the table in Depreciation Maintenance.
•
F6 - Table Maintenance Brings up the Depreciation Table Percent Schedule screen to enter the depreciation percent for each year.
•
F7 Equation Maintenance Brings up the Depreciation Method Equation Definition screen to allow for the entry of the equation to be used for depreciation.You can create new equations to specify how to calculate annualized depreciation. The following equations are standard:
-
Straight Line.
-
Declining Balance.
-
Sum of Years Digits.
•
Equation: Enter the equation in algebraic notation using the dictionary items of the ASSETS.BOOK file. Use parentheses to group the elements of your equation.
-
(* ) Multiplication
-
(/ ) Division
-
(+) Addition
-
(-) Subtraction
-
(^) Exponent
The following functions are available:
-
(! ) Factorial
-
(INT) Integer
-
(SQR) Square Root
-
(ABS) Absolute value
The following non-dictionary items are available:
-
YEARS REMAINING
-
PERIODS.REMAINING
-
SUM.OF.YEARS.DIGITS
•
HP Equation: Displays the above equation in reverse polish notation.
Disposal Maintenance (/DISM)
Used to add, change or delete disposal code records.
Investment Credit Maintenance (/ITCM)
Allows access to the investment tax credit table to edit the percent for investment credits.
