Create EMail Lists screen
The Create EMail Lists screen allows you to create a list of recipients for a broadcast email or fax based on one of the following files: CUSTOMER, VEND, PROSPECT, or PROS.CONT.
For more information, refer to Selecting a Broadcast List.
Menu Path: Marketing > Broadcast/Export Functions > List Selector
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the list you want to create or edit.
List Desc
A description of the list you want to create or edit.
File Name
The file you want to select from:
•
CUSTOMER.
•
VEND.
•
PROSPECT.
•
PROS.CONT.
Field Name
The field you want to use for selection. For example, enter STATE to select by state in the CUSTOMER file.
Operator
How to select the Field Name. Choose one of the following:
•
= - Equals.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
>= - Less than or equal to.
•
# - Not equal to.
Data Value
The value of the data for the selection of the list.
F2-Save
Saves the list selection and allows you to select the list now.
F4-Del
Deletes the list.
F5-View List Entries
Opens the list of customer, prospects, vendors, or contacts selected in the View Items in List screen to display on the screen.
