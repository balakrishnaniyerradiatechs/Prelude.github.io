Sales Representative Maintenance screen
In the Sales Representative Maintenance screen, you can enter and edit information about sales representatives. Once you create the sales rep’s ID, you can assign it to a prospect/customer in the Prospect Maintenance screen.
Menu: Marketing > Maintenance > Sales Representatives
Field/Function Key Descriptions
Field/Function Key
Description
Salesrep
The alphanumeric code that identifies the sales representative.
Description
The sales representative’s name.
Address One
The first line of the sales representative’s address.
Address Two
The second line of the sales representative’s address.
Address Three
The third line of the sales representative’s address.
Country
The country where the sales representative resides.
City
The sales representative’s city.
State
The sales representative’s state.
Zip
The zip code.
Phone
The sales representative’s phone number.
Ext
The phone extension.
Fax
The fax number.
Pager
The sales representative’s pager number.
Cellular
The sales representative’s cell phone number.
Email Address
The sales representative’s email address.
Sales Manager
The sales representative number for the sales manager assigned to this salesrep, if applicable. For information only.
Engineer
Determines if the sales representative is also set up as an engineer (Shop Repair Module):
•
Y - The sales representative is authorized as an engineer for shop repair orders.
•
N - The sales representative can not work on shop repair orders.
Region
The sales region assigned to this sales representative. Sales regions are optional and are established in Sales Region Maintenance on the Order Processing Menu.
Type
A sales representative type or other data. For information only.
Commission Type
Determines how commission is calculated for this sales representative on sales orders:
•
$ - Commission is based on an amount.
•
* - Commission is based on a percent of sales.
•
G - Commission is based on a percent of gross profit.
•
S - Commission is based on a percent of sale using a Gross Profit Table (established in Gross Profit Range Maintenance). The table allows the commission to vary percent by the sales amount.
•
T - Commission is based on a percent of gross profit using a Gross Profit Table (established in Gross Profit Range Maintenance). The table allows commission to be varied by the gross profit percent of the order.
Refer to the ”Commission Subsystem” in the Order Processing documentation for more information.
Table Number
(Unlabeled field)
If Commission Type is S or T, enter the commission table from Gross Profit Range Maintenance. Refer to the ”Commission Subsystem” in the Order Processing documentation for more information.
Rate
If Commission Type is $, *, or G, enter the default commission rate for the sales representative. For example, if Commission Type is G and you enter 3 for Rate, the sales representative will receive 3% of the gross profit generated on sales orders.
Direct Comm Type
Determines how commission is calculated for this sales representative on direct sales orders:
•
$ - Commission is based on an amount.
•
* - Commission is based on a percent of sales.
•
G - Commission is based on a percent of gross profit.
•
S - Commission is based on a percent of sale using a Gross Profit Table (established in Gross Profit Range Maintenance). The table allows the commission to vary percent by the sales amount.
•
T - Commission is based on a percent of gross profit using a Gross Profit Table (established in Gross Profit Range Maintenance). The table allows commission to be varied by percent by the gross profit percent of the order.
Refer to the ”Commission Subsystem” in the Order Processing documentation for more information.
Table Number
(Unlabeled field)
If Direct Comm Type is S or T, enter the commission table.
Rate
If Commission Type is $, *, or G, enter the default commission rate for the sales representative. For example, if Commission Type is G and you enter 3 for Rate, the sales representative will receive 3% of the gross profit generated on sales orders.
Territory
If you use territories, enter the Territory Number to assign to this salesrep. Territory codes are set up in Territory Maintenance (refer to Assigning Territory Codes).
Royalty Eligibility
Indicates if the sales representative is eligible for royalties:
•
Y - The sales representative is eligible for royalties.
•
N - The sales representative is not eligible for royalties.
A product must also be set up for royalties in order for the sales representative to receive royalty commission.
Prepaid Royalty
The prepaid royalty amount if the sales representative receives royalties and was prepaid.
Teardown Contact
The sales representative responsible for the final price inquiry in the work order module.
F2-Save
Saves the sales representative information.
F4-Del
Deletes the sales representative record.
F5-Perform
Opens the Salesman Performance Planning screen. You can enter project sales budgets for the sales representative.
F7-G/L Override
Opens the Salesman G/L Prefix Override screen to allow you to enter general ledger table headings to apply for this sales representative.
Action Bar
Notes
Opens the Notes Display screen to allow you to view or edit notes regarding the sales representative.
Via
Opens the Salesman Via Options screen to allow you to enter the sales representative’s user ID to access your website, if you use VIA.
