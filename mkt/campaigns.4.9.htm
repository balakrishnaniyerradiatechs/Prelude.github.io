By Other Sales Criteria
You can build a customer and/or prospect list based on sales (e.g., year-to-date, lifetime, date of last and/or first sale). To create a list based on sales criteria:
1
Select Marketing > Campaign System > Campaign Lists > Build Campaign Lists > Other Sales Criteria Selection. The Campaign Lists by Other Criteria screen appears.
2
For List Name, choose one of the following:
•
If you want to create a list, enter the name of the new list.
•
If you want to edit a list, enter the name of the existing list.
3
If you want to select by the customer’s and/or prospect’s sales for the year, identify how you want to select the year-to-date sales in YTD Sales. Enter one of the following:
•
= - Equals.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
>= - Less than or equal to.
•
# - Not equal to.
4
Enter the year-to-date sales you want to select by.
5
If you want to select by the customer’s and/or prospect’s total sales, identify how you want to select the total sales in Lifetime Sales. Enter an option from step 3.
6
Enter the total sales you want to select by.
7
If you want to select by the last date on which the customers and/or prospects placed a sales order, identify how you want to select the date in Date of Last Sale. Enter an option from step 3.
8
Enter the date you want to select by.
9
If you want to select by the first date on which the customers and/or prospects placed a sales order, identify how you want to select the date in Date of First Sale. Enter an option from step 3.
10
Enter the date you want to select by.
11
Press F2-Save. A dialog box with the following message appears:
Do You Wish To Select/Reselect This Save List?
12
Choose Yes. A dialog box appears that tells you how many customers and/or prospects were selected.
13
Choose Continue.
