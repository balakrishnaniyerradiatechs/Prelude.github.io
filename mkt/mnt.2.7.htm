Using SICs
SIC (Standard Industry Classification) codes indicate which industry a contact, prospect, customer or vendor belongs to. Once you create the codes in SIC Number Maintenance, you can assign them to prospects/customers in the Prospect Additional Info Screen.
You can also view a list of existing SICs. To view the list, choose Marketing > Listings > SIC Numbers.
To enter an SIC:
1
Select Marketing > Maintenance > SIC Numbers to open the SIC Number Maintenance screen.
2
For Sic Number, choose from the following:
•
Enter a new code (up to 9 alphanumeric characters).
•
Enter an existing code.
•
Press F3 to search for an existing code (e.g., 3308).
3
For Description, enter or edit a brief description of the code (e.g., Plastic Plumbing Fixtures).
4
Press F2-Save.
