Catalog Literature Maintenance screen
The Catalog Literature Maintenance screen allows you identify what catalogs and literature may be sent to a customer from a marketing activity (refer to Entering a Marketing Activity).
Menu Path: Marketing > Maintenance > Marketing Activity Codes > Catalog Literature
Field/Function Key Descriptions
Field/Function Key
Description
Catalog Number
The code for the catalog (e.g., 01, 02).
Description
A description of the catalog (e.g., Spring Catalog, Christmas Specials).
F2-Save
Saves the code and description.
F4-Del
Deletes the code and description.
Action Bar
User Fields
See your System Administrator.
