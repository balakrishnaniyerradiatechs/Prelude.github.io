Catalog Literature Information screen
The Catalog Literature Information screen allows you to indicate which catalogs and/or literature should be sent to a customer. You can add the catalog literature while entering a marketing activity. Refer to Entering a Marketing Activity for more information.
Menu Path: Marketing > Marketing Activity Entry > F6-Send Lit
Field/Function Key Descriptions
Field/Function Key
Description
Code
The code for the catalog or literature. Refer to Setting Up Catalogs/Literature Codes for more information.
Description
The description of the catalog or literature.
F2-Save
Saves the request for catalogs or literature.
