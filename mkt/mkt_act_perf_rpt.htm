Activity Performance Report by User/Date
The Activity Performance Report By User/Date shows how many marketing activities were closed each day by user ID.
Menu Path: Marketing > Reports > Activity Performance Reports > Activity Performance by User/Date
Selection Criteria
Use the following criteria to limit the information on the report:
•
User Identification. The user ID on the activity.
•
Closed Date range. The date the activity was closed.
Column Descriptions
Column Heading
Description
Usr
The user ID of the sales representative on the activity.
Description
The name on the activity.
Cls Date
The date and day of the week on which the activity was closed.
Total Per Day (not labelled)
The number of activities the user closed that day.
