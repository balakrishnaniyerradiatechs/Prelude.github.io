Salesman G/L Prefix Override screen
In the Salesman G/L Prefix Override screen, you can enter and edit G/L override accounts for the salesman.
Menu: Marketing > Maintenance > Sales Representatives > F7-G/L Override
Field/Function Key Descriptions
 
Field/Function Key
Description
G/L Table Headings
The name of the G/L entry.
Division
The highest possible grouping in your company (e.g., Eastern Division, Western Division)
Region
A subset of Division - if Division is used. (e.g., Southeast Region, Northwestern Region).
Location
The warehouse (e.g., Dallas, Tampa).
Dept
A subset of Location/Warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
Cost Center
A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
Account#
The G/L account number.
Description
The description of the G/L account.
F2-Save
Saves the G/L overrides.
F5-Reset Override
Resets the G/L accounts to what they where when you opened the screen.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are line number, page number, reference number, B (bottom), T (top).
