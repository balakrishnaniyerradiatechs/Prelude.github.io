Building a Campaign List
You can build a list using any of the following selection criteria:
•
By SIC Code.
•
By City, State, or ZIP.
•
By Source Code.
•
By Campaign Code.
•
By Priority Code.
•
By Other Sales Criteria.
•
By Sales Rep.
•
By User-Selected Criteria.
•
By a TCL Select Statement.
