Send Broadcast Email screen
The Send Broadcast Email screen allows you to create an email to send to prospects, customer, vendors, or contacts. The email can be entered manually or based on a form letter.
For more information, refer to Broadcast Emailing.
If the email preferences for the customer, vendor, prospect, or contact indicate that they do not want to receive email, that customer, vendor, prospect, or contact is not emailed even if they are in the email list.
Menu Path: Marketing > Broadcast/Export Functions > Broadcast Email
Field/Function Key Descriptions
Field/Function Key
Description
Msg#
The number of the broadcast email.
Blind Flag
Determines if all email address appear in the To line:
•
Y - Recipients only see their email address.
•
N - Recipients see email address of all other recipients.
Date
The date on which the email was created.
Time
The time at which the email was created.
From
The email address of the sender. Defaults to the email address of the user lD of the person logged in.
To
The email addresses you want to send the email to:
•
Separate email address.
•
A list of customer, vendors, prospects, or contacts.
•
A campaign list name. Refer to Building a Campaign List.
•
A select list name from a saved TCL list.
•
A list created using the list selector. Refer to Selecting a Broadcast List.
File
The file name for the list name entered in the To field and any merged fields:
•
CUSTOMER.
•
PROSPECT.
•
VEND.
•
PROS.CONT.
Subject
The subject of the email. The text entered appears as the subject line of the email.
Attach
The name of the document to attach to the email. The file should be in /tmp on the Prelude server (e.g., \/tmp/attach.pdf). You must have full permissions to that file to attach it.
Body of Message
Enter the text message or choose a form letter to be emailed. This text appears in the body of the email. You can use merged fields to insert data from a file into an email. For example, you may want to use a contact’s name from the PROS.CONT file. To use a merged field, first determine the dictionary name of the field you want to use. The field name of the contact name in the PROS.CONT file is CONT.DESC. In the body of the email, include the field name, surrounded by carets (e.g, ^FIELD.NAME^). To use the contact name, type ^CONT.DESC^.
F2-Save
Saves the email message allows you to send the email now.
F5-Load Form Letter
Allows you to select a form letter to use in Body of Message. Refer to Maintaining Form Letters.
F6-Enter Keys Manually
Opens the Manually enter customers/vendors/etc. screen, which allows you to select customers, vendors, prospects, or contacts to receive the email.
Action Bar
Copy
Opens the Copy Email/Fax screen, which allows you to choose an exisitng email or fax to copy. Refer to Copying a Broadcast Email or Fax.
