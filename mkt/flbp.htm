Form Letter Batch Print screen
The Form Letter Batch Print screen allows you to print multiple form letters at once. These form letters were selected to print for a customer, prospect, or contact from a marketing activity (refer to Entering a Marketing Activity). Refer to Batch Printing Form Letters for more information.
Menu Path: Marketing > Processes > Form Letter Batch Print
Field/Function Key Descriptions
Field/Function Key
Description
Form Letter Number
The form letter number(s) to print (refer to Maintaining Form Letters).
Customer
The customer or prospect number(s) for which you want to print form letters.
Print Type
Indicates which types of form letters to print:
•
P - Only generate form letters that were flagged to print.
•
E - Only generate form letters that were flagged to email.
•
F - Only generate form letters that were flagged to fax.
•
Leave blank to generate all form letters.
Printer Form Queue
The number of the printer you want to print the form letters to.
