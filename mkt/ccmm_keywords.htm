Customer Contact Keywords screen
In the Prospect Keywords screen, you can enter and edit keywords for contacts at a prospect or customer. When a user searches for a contact, these keywords are searched along with other standard keywords (e.g., Description, City, State, ZIP).
Menu Path: Marketing > Maintenance > Customer Contact > F5-Add Info > F5-Keywords
Field/Function Key Descriptions
Field/Function Key
Description
Keywords
Keywords that you want to use for searching.
F2-Save
Saves the keywords.
