Send broadcast fax screen
The Send broadcast fax screen allows you to create a fax to send to prospects, customer, vendors, or contacts. The fax can be entered manually or based on a form letter.
For more information, refer to Broadcast Faxing.
Menu Path: Marketing > Broadcast/Export Functions > Broadcast Fax
Field/Function Key Descriptions
Field/Function Key
Description
Msg#
The number of the broadcast fax.
File
The file name associated with the list name entered in the To field and any merged fields:
•
CUSTOMER.
•
PROSPECT.
•
VEND.
•
PROS.CONT.
Date
The date on which the email was created.
Time
The time at which the email was created.
Cover?
Indicates if you want to include a cover page separate from the text of the fax:
•
Y - Include a cover page.
•
N - Do not include a cover page.
From
The fax number of the sender. Defaults to the fax of the user lD of the person logged in.
Name
The name of the sender. Defaults to the name of the user lD of the person logged in.
Email
The email address of the sender. Defaults to the email of the user lD of the person logged in.
Attach
The name of the document to attach to the email. The file should be in the /tmp on the Prelude server (e.g., \/tmp/attach.pdf). You must have full permissions to that file to attach it.
To
The fax numbers to send the fax to:
•
Separate email address.
•
A list of customer, vendors, prospects, or contacts.
•
A campaign list name. Refer to Building a Campaign List.
•
A select list name from a saved TCL list.
•
A list created using the list selector. Refer to Selecting a Broadcast List.
Fax
The fax number.
Customer Name
The name of the customer to receive the fax.
Company Name
The name of the company to receive the fax.
Subject
The subject of the fax. This text entered appears as the subject line of the fax.
Body of Fax
Enter the text message or choose a form letter to be faxed. This text appears in the body of the fax. You can use merged fields to insert data from a file into an fax. For example, you may want to use a contact’s name from the PROS.CONT file. To use a merged field, first determine the dictionary name of the field you want to use. The field name of the contact name in the PROS.CONT file is CONT.DESC. In the body of the email, include the field name, surrounded by carets (e.g, ^FIELD.NAME^). To use the contact name, type ^CONT.DESC^.
F2-Save
Saves the fax and allows you to send the fax now.
F5-Load Form Ltr
Allows you to select a form letter to use in Body of Fax. Refer to Maintaining Form Letters.
F6-Manual Entry
Opens the Manually enter customers/vendors/etc. screen, which allows you to select customers, vendors, prospects, or contacts to receive the fax.
F7-Coverpage Notes
If Cover? is Y, allows you to enter text for the cover page.
Action Bar
Copy
Opens the Copy Email/Fax screen, which allows you to choose an exisitng email or fax to copy. Refer to Copying a Broadcast Email or Fax.
