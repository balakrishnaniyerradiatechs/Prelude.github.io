By City, State, or ZIP
If you want to create a campaign list based on the location of the customers or prospects, you can select the list by city, state, or ZIP. To build a list based on city, state, or ZIP:
1
Select Marketing > Campaign System > Campaign Lists > Build Campaign Lists > City, State, Zip Selection. The Campaign Lists by City/State/ZIP screen appears.
2
For List Name, choose one of the following:
•
If you want to create a list, enter the name of the new list.
•
If you want to edit a list, enter the name of the existing list.
3
For Customers/Prospects Or Both, enter one of the following:
•
C - Use only customers.
•
P - Use only prospects.
•
B - Use both customers and prospects.
4
Enter the first character of the city you wish to use in City Greater Than Or Equal. For example, if you want to build a campaign list for cities that begin with A through H, enter A. All cities that begin with A are selected.
5
Enter the first character of the city you wish to use in City Less Than Or Equal. For example, if you want to build a campaign list for cities that begin with A through H, enter H. All cities that begin with H are selected.
6
Enter the first character of the state you wish to use in State Greater Than Or Equal. For example, if you want to build a campaign list for states that begin with C through M, enter C. All states that begin with A are selected.
7
Enter the first character of the state you wish to use in State Less Than Or Equal. For example, if you want to build a campaign list for states that begin with C through M, enter M. All states that begin with M are selected.
8
To select by ZIP code:
•
Enter the highest ZIP code in Zip code Greater Than Or Equal.
•
Enter the lowest ZIP code in Zip Code Less Than Or Equal.
9
Press F2-Save. A dialog box with the following message appears:
Do You Wish To Select/Reselect This Save List?
10
Choose Yes. A dialog box appears that tells you how many customers and/or prospects were selected.
11
Choose Continue.
