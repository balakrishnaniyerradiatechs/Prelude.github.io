Change Dates screen
The Change Dates screen allows you to change the date and/or time of a scheduled marketing activity in Marketing Activity Inquiry or Marketing Activity Contact Inquiry.
Menu Path: Marketing > Inquiries > Marketing Activity Inquiry > F6-Time/Date
Field/Function Key Descriptions
Field/Function Key
Description
Enter New Date
The date on which you want to reschedule the activity.
Enter New Time
The time at which you want to reschedule the activity.
F2-Save
Saves the code and description.
