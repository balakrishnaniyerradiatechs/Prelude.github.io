Selecting a Broadcast List
If you want to create a list to use for emailing, faxing, or exporting, you must first select the list. To select the list:
1
Select Marketing > Broadcast/Export Functions > List Selector. The Create EMail Lists screen appears.
2
For List Name, enter the name of the new list or an existing list.
3
Enter a brief description of the list in List Desc.
4
For File Name, enter the name of the file you want to select from:
•
CUSTOMER -Customers only.
•
VEND - Vendors.
•
PROSPECT - Customers and prospects.
•
PROS.CONT - Contacts for customers and prospects.
5
For Field Name, enter the name of the dictionary item you want to use for selection criteria.
6
For Operator, identify how you want to select the Field Name. Choose one of the following:
•
= - Equal to.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
<= - Less than or equal to.
•
# - Not equal to.
7
For Data Value, enter the data you want to select for the Field Name.
8
Repeat steps 5-7 for each selection criteria you want to use.
9
Press F2-Save. A dialog box with the following message appears:
 
Select the list now?
10
Choose Yes to select the list.
