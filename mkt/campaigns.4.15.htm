Exporting a Campaign List
You can export a list of customers and/or products to a DOS file that you can edit as necessary. To export a list:
1
Select Marketing > Campaign System > Campaign Lists > Download to DOS File. The DOS Save of Campaign List screen appears.
2
For List Name, enter the name of the new or existing list.
3
Enter the name of the file you want to export in DOS File Name.
4
For Type of Data, indicate the file format you want to export:
•
F - Fixed.
•
D<x> - Delimited. This option creates a CSV (comma-separated value) file where x is the delimiter. For example, if you enter D, a comma (,) is inserted between each piece of data.
5
Indicate where you want to export the list to in Write To Floppy:
•
Y - Save the list to a floppy.
•
N - Save the list to your UNIX /tmp directory.
6
For Filename, enter the name of the file where the data resides (e.g., CUSTOMER).
7
Enter the name of the dictionary from the file name in which you want (e.g., ADDRESS1) for Field Name.
8
For Format, indicate how to store the data:
•
I - Internal. Stores dates (e.g.,month/day/year) in 4-5 digit number formats and does not include decimals.
•
E - External. Converts dates to MM/DD/YY format and includes decimals in numbers.
9
Enter the number of the data in Attr if you did not enter a Field Name.
10
If you entered a value in Attr, indicate if the data is multivalued in MV.
11
For Beg and End, indicate at what character position or delimiter you want to start and stop pulling data.
