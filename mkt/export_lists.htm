Export Lists screen
The Export Lists screen allows you to export a list of data so that you can edit the list in another program (e.g., Microsoft Excel).
For more information, refer to Exporting a Broadcast List.
Menu Path: Marketing > Broadcast/Export Functions > Export Functions
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the list to create.
File Name
The name of the file to use for selecting the data for the list:
•
CUSTOMER - Customers only.
•
VENDOR - Vendors.
•
PROSPECT - Customers and prospects.
•
PROS.CONT - Contacts for customers and prospects.
List Desc
A description of the list.
Headings
A description of the data to include in the list (e.g., Customer Name, Email).
Field Names
The field name to select (e.g., CUST.DESC, EMAIL).
F2-Save
Saves the parameters for the list name.
F4-Del
Deletes the parameters for the list name.
F7-Export
Processes and exports the list created to a text file. The exported file is called exportlist.txt and is saved to your My Documents folder. Each time you export a list, this file is overwritten. Therefore, if you want to save the exported list, rename the file.
