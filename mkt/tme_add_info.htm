Prospect Additional Info Screen
In the Prospect Additional Info screen, you can enter and edit codes and settings for prospects and customers.
Menu Path: Marketing > Prospect Maintenance > F5-Add Info
Field/Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for the prospect. Used in reports.
Territory
The territory in which the customer is based. For more information on territory codes, refer to Assigning Territory Codes.
Primary SIC Code
The standard industry classification (SIC) code for the account. For more information, refer to Using SICs.
Secondary SIC Code
A secondary standard industry classification (SIC) code for the account. For more information, refer to Using SICs.
Potential
A code that describes the likelihood that the prospect will become a customer. For more information, refer to Using Potential Codes.
Source
A code that indicates how you got the lead for this account. For more information, refer to Identifying Source Codes.
Number Of Employees
The number of employees that work for the customer or prospect.
Sales Volume
An amount that represents the sales volume for the customer or an estimated sales volume for a prospect.
Fax?
Indicates if the customer or prospect will receive broadcast faxes (refer to Broadcast Faxing):
•
N - Customer or prospect will not receive broadcast faxes.
•
Y - Customer or prospect wants to receive broadcast faxes.
E-Mail?
Indicates if the customer or prospect will receive broadcast emails (refer to Broadcast Emailing):
•
N - Customer or prospect does want broadcast emails.
•
Y - Customer or prospect wants to receive broadcast emails.
Call?
Indicates if the customer or prospect wants to receive phone calls:
•
N - Customer or prospect does not want to receive phone calls.
•
Y - Customer or prospect wants to receive phone calls.
The phone call information appears in Marketing Activity and Marketing Contact Activity.
Mail?
Indicates if the customer or prospect wants to receive mail:
•
N - Customer or prospect does not want to receive mail.
•
Y - Customer or prospect wants to receive mail.
The receive mail information appears in Marketing Activity and Marketing Contact Activity.
Custom Fields (unlabeled)
Any customer fields you set up. Refer to Establishing Custom Fields.
F2-Save
Saves the data.
F5-Keywords
Opens the Prospect Keywords screen. Refer to Prospect Keywords screen.
