Using Campaigns
Marketing campaigns allow you to define a set of customers and prospects that will receive promotional material about your products. You can set up special pricing for the campaign. By using a campaign, you can identify which sales are a result of the campaign and measure the effectiveness of the campaign. To create a campaign, you must:
•
Set up lists of customers or prospects who should receive the campaign literature (refer to Working with Campaign Lists).
•
Create a campaign to select customer lists and products (refer to Creating a Campaign).
 
If you want to use campaign pricing, Utilize Campaign Pricing in Marketing Parameters must be set to Y. For more information, refer to Choosing Marketing Options.
