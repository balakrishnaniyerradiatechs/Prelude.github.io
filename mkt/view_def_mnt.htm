View Definition Maintenance screen
The View Definintion Maintenance screen allows you specify product lines and major groups to include in a view (refer to the View Maintenance screen).
Menu Path: Marketing > Maintenance > View Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
View Code
The view to which you are adding product lines and major groups.
Description
The group within the view to which you are adding product lines and major groups.
Major Groups
The code for the major group to add.
Description (unlabeled)
The description of the major group to add.
Include (unlabeled)
