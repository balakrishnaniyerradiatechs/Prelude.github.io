Identifying Source Codes
Source codes indicate how your company got the lead for a customer or prospect. Once you create the codes in Source Code Maintenance, you can assign them to prospects/customers in the Prospect Additional Info Screen. You can then use the source codes to build campaign or broadcast lists (refer to Creating Campaign Lists and Selecting a Broadcast List).
To enter a source code:
1
Select Marketing > Maintenance > Source Codes to open the Source Code Maintenance screen.
2
For Source, choose from the following:
•
Enter a new code (up to 6 alphanumeric characters).
•
Enter an existing code.
•
Press F3 to search for an existing code (e.g., ADV, YP).
3
For Description, enter or edit a brief description of the code (e.g., Advertising, Yellow Pages).
4
Press F2-Save.
