Setting Up Catalog Literature
You can set up codes for catalogs and literature that you want to send to customers. You can then enter the code for the catalog or literature in a marketing activity (refer to Entering a Marketing Activity) and run the Send Literature Report to determine which customers should receive the catalogs and/or literature.
To enter a code for catalog literature:
1
Choose Add. Modules > Marketing > Maintenance > Marketing Activity Codes > Catalog Literature. The Catalog Literature Maintenance screen appears.
2
For Catalog, enter a new code (up to 6 alphanumeric characters) or and existing code or press F3 to search for an existing code (e.g., Spr-04, Xmas).
3
For Description, enter or edit a brief description of the code (e.g., Spring Catalog, Christmas Catalog).
4
Press F2-Save.
