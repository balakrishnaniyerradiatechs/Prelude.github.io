Campaign Lists by Other Criteria screen
The Campaign Lists by Other Criteria screen allows you to create a campaign list based on sales data for customers and/or prospects.
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > Other Sales Criteria Selection.
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Description (unlabeled)
The description of the campaign list.
YTD Sales
Indicates how you want to select the year-to-date sales:
•
= - Equals.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
>= - Less than or equal to.
•
# - Not equal to.
Amount (unlabeled)
The year-to-date sales you want to select.
Lifetime Sales
Indicates how you want to select the lifetime sales:
•
= - Equals.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
>= - Less than or equal to.
•
# - Not equal to.
Amount (unlabeled)
The lifetime sales you want to select.
Date of Last Sale
Indicates how you want to select the date on which the customer last purchased a product:
•
= - Equals.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
>= - Less than or equal to.
•
# - Not equal to.
Date (unlabeled)
The date you want to select.
Date of First Sale
Indicates how you want to select the date on which the customer first purchased a product:
•
= - Equals.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
>= - Less than or equal to.
•
# - Not equal to.
Date (unlabeled)
The date you want to select.
F2-Save
Selects the campaign list.
