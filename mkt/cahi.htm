Contact Activity History Inquiry
The Contact Activity History Inquiry lists all scheduled and closed activities for a customer or prospect and contact
Viewing Closed Contact Activities
To view the Contact Activity History Inquiry:
1
Select Marketing > Inquiries > Contact Activity History Inquiry. The Contact Activity History Inquiry screen appears.
2
For Prospect, enter the number of the customer or prospect for which you want to see scheduled and closed marketing activities.
3
For Contact, enter the number of the contact for which you want to see scheduled and closed marketing activities.
4
If you use mail-to addresses for contacts, indicate which addresses you want to use in Mail-To.
Open and closed marketing activities for the customer or prospect and contact appear.
Field/Function Key Descriptions
Field/Function Key
Description
Tran#
The transaction number for the activity.
Date
The action date for the activity. For closed activities, the action date is the date on which the activity occurred. For open activities, the action date is the date on which the activity is scheduled to occur.
Time
The action time for the activity. For closed activities, the action time is the time at which the activity occurred. For open activities, the action time is the time at which the activity is scheduled to occur.
User
The user ID of the sales representative who completed the activity.
Priority
The priority of the activity (e.g., A, B, C, 1, 2, 3).
Transaction Summary
The action summary for the activity. Refer to Creating Activity Codes for more information.
Status
The status of the activity:
•
O - Open.
•
C - Closed.
F2-Finish
Closes the inquiry.
F5-Review Activity
Opens the Marketing Activity Inquiry.
F9-Goto Line
Identify a reference number on the screen that you want to jump to. Valid codes are transaction number, page number, line number, B (bottom), or T (top).
