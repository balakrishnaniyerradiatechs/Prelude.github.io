Prospect Activity Report
The Prospect Activity Report lists scheduled and/or closed marketing activities. Activities can be entered or scheduled in Marketing Activity Entry or Marketing Activity Contact Entry (refer to Entering a Marketing Activity) or Contact Diary Entry (refer to Creating a Diary Entry).
Menu Path: Marketing > Reports > Activity Report
Selection Criteria
Use the following criteria to limit the information on the report:
•
User Id. The user ID of the person who entered the activity.
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used. (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse for the customer or prospect.
•
Department. A subset of Warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Salesrep. The code for the sales representative assigned to the customer or prospect.
•
Prospect. The customer or prospect number on the activity.
•
Action Date range. The range for which you want to view activities.
•
Activity Status. The status of the activity:
-
O - Open.
-
C - Closed.
-
Null - View all activities.
Column Descriptions
Column Heading
Description
Tran#
The transaction number for the activity.
Act-Date
The action date for the activity. For closed activities, the action date is the date on which the activity occurred. For open activities, the action date is the date on which the activity is scheduled to occur.
Act-Time
The action time for the activity. For closed activities, the action time is the time at which the activity occurred. For open activities, the action time is the time at which the activity is scheduled to occur.
P
The priority of the activity.
Transaction Summary
The action summary for the activity. Refer to Creating Activity Codes for more information.
Notes
Any notes entered for the activity.
Cust #
The number of the customer or prospect on the activity.
Mailto
If you are using mail-to addresses, the mail-to address on the activity.
Description
The description of the customer or prospect on the activity.
Contact
The name of the contact on the activity, if one was entered.
