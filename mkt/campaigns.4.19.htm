Adding a Campaign List to an Existing Campaign
If you want to add a campaign list to an existing campaign and you do not need to edit any other information on the campaign, you can update a campaign with the campaign list. You must edit the campaign to add estimated responses (refer to Adding Campaign Lists to a Campaign).
To add a list to an existing campaign:
1
Select Marketing > Campaign System > Campaign Lists > Update Campaign and Prospects. The Campaign Update Procedure screen appears.
2
Enter the name of the campaign list you want to add to the existing campaign in Campaign List.
3
Enter the name of the campaign to which you want to add the list in Campaign 1.
4
Press F2-Save.
