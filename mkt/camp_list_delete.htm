Campaign List Deletion screen
The Campaign List Deletion screen allows you to delete a campaign list.
Menu Path: Marketing > Campaign System > Campaign Lists > Delete Campaign Lists
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to delete.
Count
The number of customers and/or prospects in the list.
Selection Statement
The selection criteria for the campaign list.
Ok To Delete?
Indicates if you want to delete the list:
•
Y - Delete the list.
•
N - Do not delete the list.
