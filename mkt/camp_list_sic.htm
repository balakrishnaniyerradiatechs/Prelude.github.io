Campaign Lists By SIC Codes screen
The Campaign Lists By SIC Codes screen allows you to create a campaign list based on the SICs (Standard Industry Code) in Customer Maintenance. For more information on SICs, refer to Using SICs.
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > SIC Code Selection
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Description (unlabeled)
The description of the campaign list.
Customer/Prospects Or Both
Indicates what type of data you want to use to create the list:
•
C - Only customers.
•
P - Only prospects.
•
B - Both customers and prospects.
Enter SIC Codes Separated by Spaces
The SICs for which you want to select customers or prospects. For example, if you want to select customers with SICs of 01 and 05, enter 01 05.
F2-Save
Selects the list.
