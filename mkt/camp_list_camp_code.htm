Campaign Lists By Campaign Code screen
You can build a customer and/or prospect list based on a previous campaign. To view a list of available campaigns, refer to Campaign List Inquiry.
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > Campaign Codes Selection
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Customer/Prospects Or Both
Indicates what type of data you want to use to create the list:
•
C - Only customers.
•
P - Only prospects.
•
B - Both customers and prospects.
Description (unlabeled)
The description of the campaign list.
Original Campaign Equal To
The campaign number which was the first campaign purchase for this customer.
Original Campaign Date Equal To
The date of the campaign which was the first campaign purchase for this customer.
Latest Campaign Equal To
Selects customers whose most recent purchases were made on this campaign.
Latest Campaign Date Equal To
Selects customers whose most recent purchases were made on this campaign date.
Successful Campaign Equal To
The campaign number on which customers have purchased from this campaign.
Mailed To Campaigns Equal To
Displays all customers that received mail from this campaign.
F2-Save
Selects the campaign list.
F4-Del
Deletes the campaign list.
