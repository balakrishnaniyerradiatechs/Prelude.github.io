Campaign Lists By User Selected Criteria screen
The Campaign Lists By User Selected Criteria screen allows you to select customers or prospects for a campaign list based on data in the PROSPECT file.
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > User Selected Criteria Selection
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Description (unlabeled)
The description of the campaign list.
Customer/Prospects Or Both
Indicates what type of data you want to use to create the list:
•
C - Only customers.
•
P - Only prospects.
•
B - Both customers and prospects.
Dictionary Id
The field name you want to select by.
Oper
How to select the Dictionary Id:
•
= - Equals.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
>= - Less than or equal to.
•
# - Not equal to.
Value
The data you want to select for the Dictionary Id.
And/Or
Indicates how to select if you are using multiple Dictionary Ids:
•
AND - select customers and/or prospects that meet the all selection criteria.
•
OR - select customers and/or prospects that meet any of the selection criteria.
F2-Save
Selects the list.
