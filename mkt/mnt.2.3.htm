Selecting Files to Update
Flags in File Update Parameters determine the type of marketing data to track.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > File Update Parameters
Field and Function Key Descriptions
Field/Function Key
Description
Contact Maint Audit?
The Contact Maintenance Audit Reports (refer to Contact Maintenance Audit Report by User ID and Contact Maintenance Audit Customer/Contact Report) record each time a contact record was changed, who changed it, and what the data was changed from and to:
•
Y - Track changes to contact records.
•
N - Do not track changes to contact records.
Contact Detail?
If you use Contact Diary Inquiry or Contact Diary Report, choose the type of data to appear in the contact diary:
•
N - Do not track data for the contact diary.
•
Y - Tracks data for the contact diary. You can choose to track quotes, sales orders, shipping, campaign letters, work order completions, marketing actives, and/or sample requests.
