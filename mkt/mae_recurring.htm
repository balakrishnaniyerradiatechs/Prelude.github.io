Recurring Marketing Events screen
The Recurring Marketing Events screen allows you to automatically schedule marketing activities that occur regularly. For example, if you want to call a customer or prospect on the first of every month, you can set a recurring activity to appear on the first of every month for that customer.
For more information, refer to Scheduling a Recurring Activity.
Menu Path: Marketing > Marketing Activity Maintenance > Action Bar > Recurring
Field/Function Key Descriptions
Field/Function Key
Description
Recurring
Indicates if the marketing activity occurs on a regular basis:
•
Y - The activity is recurring.
•
N - The activity is not recurring.
Method
The code for how frequently the marketing activity will occur:
•
NOD - Number of days. A new activity should automatically be scheduled after a set number of days.
•
DOW - Day of week. A new activity should automatically be scheduled once a week on the same day.
•
DOM - Day of month. A new activity should automatically be scheduled once a month.
Number Of Days
If Method is NOD, the number of days between scheduled marketing activities. For example, if the marketing activity should be automatically generated every other day, Number Of Days is 2.
Day Of Month
If Method is DOM, the date of each month that the activity should be scheduled on. For example, if the marketing activity should be automatically generated on the 15th of every month, Day Of Month is 15.
Monday
If Method is DOW, indicates if the marketing activity should occur each Monday:
•
Y - Schedule this activity for each Monday.
•
N - Do not schedule this activity for each Monday.
Tuesday
If Method is DOW, indicates if the marketing activity should occur each Tuesday:
•
Y - Schedule this activity for each Tuesday.
•
N - Do not schedule this activity for each Tuesday.
Wednesday
If Method is DOW, indicates if the marketing activity should occur each Wednesday:
•
Y - Schedule this activity for each Wednesday.
•
N - Do not schedule this activity for each Wednesday.
Thursday
If Method is DOW, indicates if the marketing activity should occur each Thursday:
•
Y - Schedule this activity for each Thursday.
•
N - Do not schedule this activity for each Thursday.
Friday
If Method is DOW, indicates if the marketing activity should each Friday:
•
Y - Schedule this activity for each Friday.
•
N - Do not schedule this activity for each Friday.
Saturday
If Method is DOW, indicates if the marketing activity should occur each Saturday:
•
Y - Schedule this activity for each Saturday.

N - Do not schedule this activity for each Saturday.
Sunday
If Method is DOW, indicates if the marketing activity should occur each Sunday:
•
Y - Schedule this activity for each Sunday.
•
N - Do not schedule this activity for each Sunday.
F2-Save
Saves the recurring marketing activity information.
