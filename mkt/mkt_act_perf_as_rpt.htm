Activity Performance Report by User/Date/Activity
The Activity Performance Report By User/Date/Activity shows how many marketing activities were closed each day by user ID and the activity summary for the transaction.
Menu Path: Marketing > Reports > Activity Performance Reports > Activity Performance by User/Date/Activity
Selection Criteria
Use the following criteria to limit the information on the report:
•
User Identification. The user ID on the activity.
•
Transaction Summary. The activity summary for the activity. For more information, refer to Creating Activity Codes.
•
Closed Date range. The date on which the activity was closed.
Column Descriptions
Column Heading
Description
User
The user ID on the activity.
Description
The name on the activity.
Cls Date
The date and day on which the activity was closed
Transaction Summary
The summary of the activity. For more information, refer to Creating Activity Codes.
Total Per Day (not labelled)
The number of activities the user closed that day.
