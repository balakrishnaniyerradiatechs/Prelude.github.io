Closed Prospect Activity Report
The Closed Prospect Activity Report lists all closed marketing activities.
Menu Path: Marketing > Reports > Closed Activity Report
Selection Criteria
Use the following criteria to limit the information on the report:
•
User Id. The user ID on the activity.
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used. (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse for the customer or prospect.
•
Department. A subset of Warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Salesrep. The user ID of the sales representative for the customer or prospect.
•
Prospect. The customer or prospect number on the activity or transaction.
•
Transaction Date range. The date on which the activity was created.
•
Closed On range. The date on which the activity was closed.
Column Descriptions
Column Heading
Description
Tran#
The transaction number for the activity.
Cls Date
The date on which the activity was closed.
Cls Time
The time at which the activity was closed.
Transaction Summary
The activity summary appears. For more information, refer to Creating Activity Codes.
Customer
The customer or prospect number on the activity or transaction.
Mail-To
The number of the mail-to address on the activity, if you are using mail-to addresses.
Description
The description of the prospect or customer.
Contact
The name of the customer or prospect contact on the activity or transaction.
Notes
Any notes entered for the activity.
