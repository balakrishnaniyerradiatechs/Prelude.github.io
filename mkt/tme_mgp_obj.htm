Major Group Objectives screen
In the Major Group Objectives screen, you can view major group objectives for each of the customer or prospect’s contacts. For more information on contacts, refer to Setting up Customer Contacts.
Menu Path: Marketing > Prospect Maintenance > Action Bar > Add’l Options > Major Group Objectives
Field/Function Key Descriptions
Field/Function Key
Description
Cn#
The contact number.
Description
The contact’s name.
MGroup
The code for the major group for which the contact has an objective.
Description
The description of the major group.
