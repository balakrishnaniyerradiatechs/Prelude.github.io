By Campaign Code
You can build a customer and/or prospect list based on a previous campaign. To view a list of available campaigns, refer to the Campaign List Inquiry.
To create a list based on campaign code:
1
Select Marketing > Campaign System > Campaign Lists > Build Campaign Lists > Campaign Codes Selection to open the Campaign Lists By Campaign Code screen.
2
For List Name, choose one of the following:
•
If you want to create a list, enter the name of the new list.
•
If you want to edit a list, enter the name of the existing list.
3
For Customers/Prospects Or Both, enter one of the following:
•
C - Use only customers.
•
P - Use only prospects.
•
B - Use both customers and prospects.
4
For Original Campaign Equal To, enter the first campaign from which the customer ordered.
5
For Original Campaign Date Equal To, enter the date of the first order the customer placed from a campaign.
6
For Latest Campaign Equal To, enter the most recent campaign from which the customer ordered.
7
For Latest Campaign Date Equal To, enter the most recent date on which the customer ordered from a campaign.
8
For Successful Campaign Equal To, enter the campaign code for which you want to select all customers who ordered from the campaign.
9
For Mailed To Campaign Equal To, enter the campaign code for which you want to select all customers who were mailed campaign literature.
10
Press F2-Save. A dialog box with the following message appears:
Do You Wish To Select/Reselect This Save List?
11
Choose Yes. A dialog box appears that tells you how many customers and/or prospects were selected.
12
Choose Continue.
