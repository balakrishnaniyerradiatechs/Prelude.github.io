Contact Diary Entry screen
The Contact Diary Entry screen allows you to record an activity with a customer or prospect and enter a follow-up activity. For more information, refer to Creating a Diary Entry.
Menu Path: Marketing > Contact Diary System > Contact Diary Entry.
Field/Function Key Descriptions
Field/Function Key
Description
Prospect Number
The number of the customer or prospect for which you want to enter an activity.
Description (unlabeled)
The name of the customer or prospect.
Mail-To
The number of the mail-to address on the activity, if you are using mail-to addresses.
Phone Number (unlabeled)
The phone number of the customer or prospect.
Contact Type
The reason for the contact (e.g., Sales, Credit).
Contact Method
Indicates how the marketing activity was completed:
•
TH - Telephone.
•
MT - Meeting.
•
ML - Mail.
•
FAX - Fax.
•
EM - Email.
Contact
The contact for the activity, if any:
•
P - The primary contact for the customer or prospect, if only one exists.
•
Number - The number of the contact.
•
Name - If the contact record does not exist, the name of the contact.
For more information on contacts, refer to Setting up Customer Contacts.
Diary Code
A code that identifies the activity performed (e.g., 03 Called for service). For more information, refer to Identifying Diary Codes.
Entry Date
The date on which the marketing activity was completed. Defaults to the current date.
Time
The time at which the marketing activity was completed. Defaults to the current date.
User ID
The user ID who completed the marketing activity. Defaults to the user ID of the person logged in.
Follow Up Date
The date on which you want to schedule a follow up activity.
Time
The time at which you want to schedule a follow up activity.
User ID
The user ID that should complete the marketing activity.
Description
A description for the follow-up activity.
Document Type
If applicable, the document type associated with this transaction:
•
S - Sales Order.
•
Q - Quote.
•
W - Work Order.
Document Number
If a Document Type is entered, the corresponding document number.
Product Number
The product that was sold/quoted as a result of this activity, if any.
Serial Number
The serial number of the product(s) sold or quoted, if any.
F2-Save
Saves the marketing activity.
F7-Notes
Allows you to enter notes about the activity.
