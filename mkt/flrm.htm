Form Letter Maintenance screen
The Form Letter Maintenance screen allows you to create and maintain a standard letter that can be mailed, e-mailed, or faxed to to multiple customers or prospects (e.g, for a marketing campaign). You can send form letters to customers or prospects from a:
•
Marketing activity - refer to Sending a Form Letter.
•
Broadcast email or fax - refer to Broadcast Emailing and Broadcast Faxing.
Menu Path: Marketing > Maintenance > Marketing Activity Codes > Form Letter Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Form Letter#
The number of the form letter.
Desqtop Folder
Previously used method. No longer used.
Description
A description of the letter.
Letter Text Content
The text of the letter, including merged fields. Merged fields allow you to insert data from a file into an email. For example, you may want to use a contact’s name from the PROS.CONT file. To use a merged field, first determine the dictionary name of the field you want to use. The field name of the contact name in the PROS.CONT file is CONT.DESC. In the body of the email, include the field name, surrounded by carets (e.g, ^FIELD.NAME^). To use the contact name, type ^CONT.DESC^.
F2-Save
Saves the letter.
F4-Del
Deletes the letter.
