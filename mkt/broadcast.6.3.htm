Exporting a Broadcast List
If you want to create a list and then edit that list in another program (e.g., Microsoft® Excel), you can export the list to a text file.
To export a list to a text file:
1
Select Marketing > Broadcast/Export Functions > Export Functions. The Export Lists screen appears.
2
For List Name, enter the name of the list that you want to export.
3
For File Name, enter the name of the file on which you want to base your selection criteria. Choose one of the following:
•
CUSTOMER - Customers only.
•
VEND - Vendors.
•
PROSPECT - Customers and Prospects.
•
PROS.CONT - Contacts for customers and prospects.
4
Enter a brief description of the list in List Desc.
5
In Headings, enter a description of each piece of data you want to export (e.g., Name, Email, Address).
6
In Field Names, enter the dictionary name of the corresponding heading.
7
Press F7-Export. The list is exported to a file called exportlist.txt in your My Documents folder.
8
Press F2-Save.
Each time you export a list, exportlist.txt is overwritten. Therefore, if you want to keep an exported list, rename the file so that it is not overwritten the next time you export a list.
