Identifying Diary Codes
Diary codes allow you to more specifically identify information about a marketing activity entered through Contact Diary Entry. For more information, refer to Creating a Diary Entry.
To enter a diary code:
1
Choose Add. Modules > Marketing > Maintenance > Contact Diary Codes. The Contact Diary Code Maintenance screen appears.
2
For Diary Code, enter a new code (up to 3 alphanumeric characters) or and existing code or press F3 to search for an existing code (e.g., FAX, TEL.
3
For Description, enter or edit a brief description of the code (e.g., Fax, Telephone).
4
Press F2-Save.
