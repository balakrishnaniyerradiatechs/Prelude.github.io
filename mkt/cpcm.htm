Competitor Code Maintenance screen
The Competitor Code Maintenance screen allows you to identify groups of competitors. You can use these competitor codes to track competitor pricing or in Prospect Maintenance (refer to Setting up Prospect Accounts).
Menu Path: Marketing > Maintenance > Competitor Codes
Field/Function Key Descriptions
Field/Function Key
Description
Competitor Code
The code for the competitors (e.g., HDWR, TOOL).
Description
A description of the group of competitors (e.g., Hardware, Tools).
F2-Save
Saves the code and description.
F4-Del
Deletes the code and description.
