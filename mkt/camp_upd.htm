Campaign Update Procedure screen
The Campaign Update Procedure screen allows you to add a campaign list to an existing campaign when you do not need to edit any other information on the campaign. The update procedure also updates the prospect records in the list with the campaign priority. You must go into the campaign to add estimated responses (refer to Adding Campaign Lists to a Campaign).
Menu Path: Marketing > Campaign System > Campaign Lists > Update Campaign and Prospects
Field/Function Key Descriptions
Field/Function Key
Description
Campaign List
The campaign list that you want to link to a campaign.
Campaign 1
The name of the campaign to be linked to the list.
F2-Save
Updates the campaign with the campaign list.
