Assigning Territory Codes
Territory codes indicate the part of the country where a prospect or customer resides. Once you create a territory code in Territory Maintenance, you can assign it to a prospect/customer in the Prospect Additional Info Screen.
To view a list of existing territory codes, choose Marketing > Listings > Territory.
To enter a territory code:
1
Select Marketing > Maintenance > Territory Code to open the Territory Maintenance screen.
2
For Territory, choose from the following:
•
Enter a new code (up to 6 alphanumeric characters).
•
Enter an existing code.
•
Press F3 to search for an existing code (e.g., 01, 02).
3
For Description, enter or edit a brief description of the code (e.g., Dallas, Houston).
4
Press F2-Save.
