Contact List screen
The Contact List screen lists the contacts that are responsible for purchasing items in a view. You can access this screen from Contact Management Statistics Inquiry. Refer to Establishing Product Views and Setting up Customer Contacts.
Menu Path: Marketing > Inquiries > Contact Management Statistics Inquiry > F9-Who
Field/Function Key Descriptions
Field/Function Key
Description
Contact
The number of the contact.
Description
The name of the contact.
