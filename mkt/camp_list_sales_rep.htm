Campaign Lists by Sales Rep screen
The Campaign Lists by Sales Rep screen allows you to create a campaign list using the sales representative identified in Customer Maintenance or Prospect Maintenance (refer to Setting up Prospect Accounts). For more information on sales representatives, refer to Maintaining Sales Representatives.
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > Sales Rep Selection
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Description (unlabeled)
The description of the campaign list.
Customer/Prospects Or Both
Indicates what type of data you want to use to create the list:
•
C - Only customers.
•
P - Only prospects.
•
B - Both customers and prospects.
Enter Sales Reps Separated by Spaces
The sales representatives for which you want to select customers or prospects. For example, if you want to select customers and prospects with sales representatives MKM and JDH, enter MKM JDH.
F2-Save
Selects the list.
