Customer Contact Management Maintenance screen
In the Customer Contact Management Maintenance screen, you can enter information about the contact and then assign the contact to a customer or prospect.
Menu Path: Marketing > Maintenance > Customer Contacts
or
Menu Path: Marketing > Prospect Maintenance > F6-Contacts
Field/Function Key Descriptions
Field/Function Key
Description
Customer
The customer or prospect number for the contact.
Created
The date on which the contact record was created.
Contact
The code for the contact for the customer or prospect (e.g., 01, 02).
Salutation
The contact’s title (e.g., Mr., Ms., Dr.).
First Name
The contact’s first name.
MI
The contact’s middle initial.
Last Name
The contact’s last name.
Description
The contact’s full name. Defaults to the Salutation, First Name, MI, and Last Name.
Address One
The first line of the contact’s address.
Address Two
The second line of the contact’s address.
Address Three
The third line of the contact’s address.
Country
The country in which the contact is based.
GeoCode
If you are using GeoCode maintenance, the code that represents the city, state, and ZIP code in which the contact is based.
City
The city in which the contact is based.
State
The state in which the contact is based.
Zip
The ZIP code in which the contact is based.
Email Address
The county in which the contact is based.
Ext
The e-mail address of the contact.
Fax
The phone number for the contact.
Cellular
The mobile phone number for the contact.
Pager
The pager number for the contact.
Department
The code for the department in which the contact works.
Title
The code for the title of the contact person. Refer to Assigning Job Titles to Contacts for more information.
Source
The code that specifies how this contact was found. Refer to Identifying Source Codes.
Shipto
The ship-to for the contact if the contact uses a ship-to other than the main ship-to from Customer or Prospect Maintenance.
F2-Save
Saves the contact information.
F4-Delete
Deletes the contact.
F5-Add Info
Opens the Contact Additional Information screen.
F6-Mail Addr
Opens the Mailing Address screen, if you need to add another mailing address.
F7-Default Addr
Brings in address information from Customer or Prospect Maintenance.
F9-Stats
Opens the Contact Statistics screen which shows sales, returns, profit, and freight for the contact.
Action Bar
User Fields
See your System Administrator.
Notes
Allows you to enter internal notes for the contact.
Addl Opts > Activity History
Opens the Contact Activity History Inquiry.
Addl Opts > Form Letter and Literature History
Opens the Form & Literature Activity History screen which lists form letters and marketing literature that was sent to the customer or prospect.
Addl Opts > Product Line Objectives
Opens the Product Line Objectives Information screen, which allows you to enter a potential sale code for product lines.
Addl Opts > Major Group Objectives
Opens the Major Group Objectives Information screen, which allows you to enter a potential code for major groups.
Web User
Opens the Web User Maintenance screen so you can setup a web user ID.
