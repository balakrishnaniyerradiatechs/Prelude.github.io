Form & Literature Activity History screen
In the Form & Literature Activity History screen, you can view which form letters and catalog literature were sent to the customer or prospect from a marketing activity (for more information, refer to Entering a Marketing Activity). For more information on form letters, refer to Maintaining Form Letters. For more information on catalog literature, refer to Setting Up Catalogs/Literature Codes.

Field/Function Key Descriptions
Field/Function Key
Description
Doc#
The transaction number of the marketing activity that requested the literature or form letter.
Description
A description of the catalog or form letter that was sent.
Type
Indicates if the customer received literature or a letter.
User
The user ID of the person who entered on the marketing activity from which the literature was sent.
F2-Finish
Closes the screen.
F5-Review
Displays the text of the letter that was sent.
