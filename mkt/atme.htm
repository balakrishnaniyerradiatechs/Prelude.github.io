Automated Prospect Maintenance screen
The Automated Prospect Maintenance screen allows you to update a group of customers or prospects based on a campaign list (refer to Building a Campaign List).
Menu Path: Marketing > Prospect Maintenance Via Saved List.
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to use to update prospects and/or customers.
Prospect
The number of the prospect or customer you want to start with, if any.
F2-Save
Opens the Prospect Maintenance screen to allow you to edit the prospect or customer.
