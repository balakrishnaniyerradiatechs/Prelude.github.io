Manually enter customers/vendors/etc. screen
The Manually enter customers/vendors/ect. screen allows you to select customers, vendors, prospects, or contacts to receive a broadcast email or fax (refer to Broadcast Emailing and Broadcast Faxing).
Menu Path: Marketing > Broadcast/Export Functions > Broadcast Fax > F6-Manual Entry
Field/Function Key Descriptions
Field/Function Key
Description
File
The name of the file to use to select data:
•
CUSTOMER.
•
VEND.
•
PROSPECT.
•
PROS.CONT.
Number
The number of the customer, vendor, prospect, or contact to include on the list.
Description
The name of the customer, vendor, prospect, or contact you selected.
F2-Save
Saves the selections.
