DOS Save of Campaign List screen
The DOS Save of Campaign List screen allows you to export a list of customers and/or products to a DOS file that you can edit as necessary.
Menu Path: Marketing > Campaign System > Campaign Lists > Download to DOS File
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list to export.
DOS File Name
The name of the file to export.
Type of Data
Indicates how to format the file:
•
F - Fixed.
•
D<x> - Delimited. This option creates a CSV (comma-separated value) file where x is the delimiter. For example, if you enter D, a comma (,) is inserted between each piece of data.
Write To Floppy
Indicates where to save the file:
•
Y - Save the list to a floppy.
•
N - Save the list to your UNIX /tmp directory.
Filename
The name of the file to export data from (e.g., CUSTOMER, PROS.CONT).
Field Name
The name of the dictionary to export (e.g., CUST.DESC, EMAIL).
Format
The file format to use.
•
I - Internal. Do not use decimals.
•
E - External. Use decimals.
Attr
The attribute numbers to use.
MV
The multi value to use.
Beg
Determines where to begin pulling the data. Use a number or delimiter.
End
Determines the ending position when pulling the data. You can use a number or delimiter.
F2-Save
Exports the list.
