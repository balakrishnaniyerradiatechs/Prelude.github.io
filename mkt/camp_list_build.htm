Campaign List Build screen
The Campaign List Build screen allows you to select a list of customers or prospects using a TCL select statement with the PROSPECT file. Customers and/or prospects who match the criteria you enter are added to the campaign list (refer to Creating Campaign Lists).
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > User Entered ACCESS Selection
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Customer/Prospects Or Both
Indicates what type of data you want to use to create the list:
•
C - Only customers.
•
P - Only prospects.
•
B - Both customers and prospects.
SSELECT PROSPECT WITH CO.NUM AND WITH
The TCL select statement you want to use to create the campaign list.
F2-Save
Selects the list.
