Campaign Lists by Priority screen
The Campaign Lists by Priority screen allows you to select a campaign list by priority code The Priority Code is based on the Campaign Number and List Name. For example, if the Campaign is FALL and the List Name is ALLCUST, the Priority Code is FALL-ALLCUST.
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > Priority Codes Selection
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Customer/Prospects Or Both
Indicates what type of data you want to use to create the list:
•
C - Only customers.
•
P - Only prospects.
•
B - Both customers and prospects.
Description (unlabeled)
The description of the campaign list.
Original Priority Equal To
The campaign priority which was the first campaign priority purchase for this customer.
Original Priority Date Equal To
The date of the campaign priority which was the first campaign priority purchase for this customer.
Latest Priority Equal To
Selects customers whose most recent purchases were made on this campaign priority.
Latest Priority Date Equal To
Selects customers whose most recent purchases were made on this campaign priority date.
Successful Priority Equal To
The campaign priority in which customers have purchased from this campaign priority.
Mailed To Priority Equal To
Displays all customers that were mailed this campaign priority.
F2-Save
Selects the campaign list.
F4-Del
Deletes the campaign list.
