Campaign List Inquiry
The campaign list inquiry lists all of the campaign lists in Prelude and the number of customers and/or prospects in the list. For more information, refer to Building a Campaign List.
Viewing a List of Campaigns
To run the Campaign List Inquiry:
1
Select Marketing > Campaign System > Campaign Lists > Campaign List Inquiry. The CAMPAIGN.LISTS Selection Criteria screen appears.
2
Enter the names of the campaign list(s) that you want to view or leave blank to see all campaign lists.
3
Press Accept.
4
At the Output To prompt, select an output method.
Column Descriptions
 
Column Heading
Description
List Name
The name of the campaign list.
Description
The description of the campaign list.
Count
The number of customers and/or prospects in the campaign.
