Using Potential Codes
Potential codes indicate how likely a customer/prospect is to purchase items from a product line or major group (refer to the Product Line Objectives Information screen or the Major Group Objectives screen).
You can set up potential codes in Potential Code Maintenance and then assign them to Prospects/Customers in the Prospect Additional Info Screen. You can then use the potential codes to build campaign or broadcast lists (refer to Creating Campaign Lists and Selecting a Broadcast List).
You can also view a list of existing potential codes. To view the list, choose Marketing > Listings > Potential Codes.
To create a potential code:
1
Select Marketing > Maintenance > Potential Codes to open the Potential Code Maintenance screen.
2
For Potential Code, choose from the following:
•
Enter a new code (up to 2 alphanumeric characters).
•
Enter an existing code.
•
Press F3 to search for an existing code (e.g., A, B).
3
For Description, enter or edit a brief description of the code (e.g., High, Low).
4
Press F2-Save.
