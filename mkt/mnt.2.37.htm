Creating Territory Codes
Territory codes indicate in which part of the country a prospect or customer resides in. For more information, refer to Entering Prospects or Maintaining Contacts.
 
You can also view a list of existing territory codes. To view the list, choose Add. Modules > Marketing > Listings > Territory.
To enter a territory code:
1
Choose Add. Modules > Marketing > Maintenance > Territory Code. The Territory Maintenance screen appears.
2
For Territory, enter a new code (up to 6 alphanumeric characters) or an existing code or press F3 to search for an existing code (e.g., 01, 02).
3
For Description, enter or edit a brief description of the code (e.g., Dallas, Houston).
4
Press F2-Save.
