Source Code Maintenance screen
The Source Code Maintenance screen allows you to set up codes to track how your company got the lead for a customer or prospect
For more information, refer to Setting up Prospect Accounts.
Menu Path: Marketing > Maintenance > Source Codes
Field/Function Key Descriptions
Field/Function Key
Description
Source
The code for how you got the lead for the customer or prostpect (e.g., ADV, YP).
Description
A description of the code (e.g., Advertising, Yellow Pages).
F2-Save
Saves the code and description.
F4-Del
Deletes the code and description.
Action Bar
User Fields
See your System Administrator.
