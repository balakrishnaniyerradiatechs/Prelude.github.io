Product Line Objectives screen
In the Product Line Objectives screen, you can view product line objectives for each of the customer or prospect’s contacts. For more information on contacts, refer to Setting up Customer Contacts.
Menu Path: Marketing > Prospect Maintenance > Action Bar > Add’l Options > Product Line Objectives
Field/Function Key Descriptions
Field/Function Key
Description
Cn#
The contact number.
Description
The contact’s name.
Pline
The code for the product line for which the contact has an objective.
Description
The description of the product line.
Pt
The potential code for this contact and product line. Refer to Using Potential Codes.
