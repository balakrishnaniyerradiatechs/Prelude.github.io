Setting Marketing Option Parameters
You must set Marketing Option Parameters to control how marketing activities are used in Prelude.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Marketing Parameters
Field and Function Key Descriptions
Field/Function Key
Description
Campaign Priority Code
Determines how campaign priority codes are used:
•
Y - You must enter a campaign codes.
•
N - You cannot enter a campaign code.
•
O - You may enter a campaign code, but it is not required.
Refer to Creating Marketing Campaigns.
Utilize Campaign Pricing
Determines if campaign pricing is used for campaigns:
•
Y - Use pricing entered through campaign maintenance for customers participating in the campaign.
•
N - Use regular pricing, not campaign pricing.
Refer to Creating Marketing Campaigns.
Contact Name Override
Determines which contact name appears in the contact field in Sales and Quote Order Entry, if you use contacts:
•
Y - Use the contact name as the contact on the sales order.
•
N - Use the contact based on the Override Contact Name in Order Processing Option Parameters, which determines if the contact should be pulled from the customer Ship To or if it should not load any contact information.
Default Purchasing Contact
Determines if the contact is defaulted in Sales Order and Quote Order Entry:
•
Y - If one purchasing contact exists for the customer, it is the default. If multiple contacts exist, no contact is defaulted.
•
N - Contacts are never defaulted.
Utilize Mail-to
Determines if you want to save Mail-To addresses for prospects:
•
Y - Use Mail-to addresses from the Prospect Maintenance screen.
•
N - Do not use Mail-to addresses in Prospect Maintenance.
The Utilize Mail-to is used in Marketing Activity Entry (refer to Entering a Marketing Activity) and the Contact Diary (refer to Creating a Diary Entry).
Print Form Letters Batch
Determines how form letters print from a marketing activity:
•
A - Allows you to choose to print immediately or by batch.
•
Y - Prints by batch only.
•
N - Prints at the time of entry.
Quote F/U - Activity Batch
Determines if quote follow-ups are included in the marketing activity inquiries:
•
Y - Quote follow-ups are included.
•
N - Quote follow-ups are not included.
About Defaults
If you activate defaults for product, product line, major group, document type, and/or document number, that data defaults into marketing activities. For example, if you enter a marketing activity from a sales order, the document type and document number default. If you enter a marketing activity from Product Maintenance, the product number, product line, and major group (if applicable) default. Refer to Setting Up Activity Defaults.
Activate Product# Default
Indicates if you want to default the product number into a marketing activity:
•
Y - Default the product number.
•
N - Leave the product number blank.
Activate PLine Default
Indicates if you want to default the product line into a marketing activity:
•
Y - Default the product line.
•
N - Leave the product line blank.
Activate MajorGrp Default
Indicates if you want to default the major group into a marketing activity:
•
Y - Default the major group.
•
N - Leave the major group blank.
Activate Document Type
Indicates if you want to default the document type into a marketing activity:
•
Y - Default the document type.
•
N - Leave the document type blank.
Activate Document Number
Indicates if you want to default the document number into a marketing activity:
•
Y - Default the document number.
•
N - Leave the document number blank.
Activate Queue Menu
The name of the custom menu if you use a custom menu for your activities. The menu must be named CUSTOM.<INT>, where INT is your company initials (e.g., CUSTOM.ABC).
