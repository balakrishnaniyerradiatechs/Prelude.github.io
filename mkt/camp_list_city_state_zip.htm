Campaign Lists by City/State/ZIP screen
The Campaign Lists by City/Sate/ZIP screen allows you to create a campaign list based on the location of the customers or prospects by selecting city, state, and/or ZIP.
Menu Path: Marketing > Campaign System > Campaign Lists > Build Campaign Lists > City, State, Zip Selection
Field/Function Key Descriptions
Field/Function Key
Description
List Name
The name of the campaign list that you want to create or update.
Description (unlabeled)
The description of the campaign list.
Customer/Prospects Or Both
Indicates what type of data you want to use to create the list:
•
C - Only customers.
•
P - Only prospects.
•
B - Both customers and prospects.
City Greater Than Or Equal
The value in which the city is selected for the list. For example, if you enter D, all customers and/or prospects in cities that begin with the letter D and greater will display.
City Less Than Or Equal
The value in which the city is selected for the list. For example, if you enter M, all customers and/or prospects in cities that begin with the letter M and lower will display.
State Greater Than Or Equal
The value in which the state is selected for the list. For example, if you enter D, all customers and/or prospects in states that begin with the letter D and greater will display.
State Less Than Or Equal
The value in which the state is selected for the list. For example, if you enter M, all customers and/or prospects in states that begin with the letter M and lower will display.
Zip Code Greater Than Or Equal
The value in which the ZIP code is selected for the list. For example, if you enter 5, all customers and/or prospects in ZIP codes that begin with the number 5 and greater will display
Zip Code Less Than or Equal
The value in which the ZIP code is selected for the list. For example, if you enter 9, all customers and/or prospects in ZIP codes that begin with the number 9 and lower will display
F2-Save
Selects the campaign list.
F4-Del
Deletes the campaign list.
