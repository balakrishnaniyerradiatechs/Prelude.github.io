Maintaining Form Letters
You can use form letters to:
•
Send to a customer or contact from a marketing activity (refer to Entering a Marketing Activity).
•
Broadcast fax or email (refer to Broadcast Emailing and Faxing).
When you create a form letter, you can enter text and merged fields to personalize the letter based on information in one of the following files:
•
CUSTOMER
•
VEND
•
PROSPECT
•
PROS.CONT
Merged fields allow you to insert data from a file into an email. For example, you can use a contact’s name from the PROS.CONT file. To use a merged field, first determine the dictionary name of the field you want to use. The field name of the contact name in the PROS.CONT file is CONT.DESC. In the body of the email, include the field name, surrounded by carets (e.g, ^FIELD.NAME^). To use the contact name, type ^CONT.DESC^.
To create a form letter:
1
Select Marketing > Maintenance > Marketing Activity Codes > Form Letter Maintenance to open the Form Letter Maintenance screen.
2
For Form Letter#, press ENTER over NEW to get the next number, enter an existing number, or press F3 to search for an existing code.
3
For Description, enter or edit a brief description of the letter (e.g., Spring Promotion Letter).
4
In the unlabeled text entry box, enter text and merged fields to create the letter that you want to send.
5
Press F2-Save.
