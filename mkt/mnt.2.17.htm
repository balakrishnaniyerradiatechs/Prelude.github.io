Creating Activity Codes
Activity codes identify what the sales representative does to complete a marketing activity (refer to Entering a Marketing Activity or Creating a Diary Entry).
To create an activity code:
1
Select Marketing > Maintenance > Marketing Activity Codes > Activity Code to open the Activity Code Maintenance screen.
2
For Activity Code, choose from the following:
•
Enter a new code (up to 6 alphanumeric characters).
•
Enter an existing code.
•
Press F3 to search for an existing code (e.g., 01, 02).
3
For Description, enter or edit a brief description of the code (e.g., Phone Call, Send, Delivery, or Drop off Literature).
4
Press F2-Save.
