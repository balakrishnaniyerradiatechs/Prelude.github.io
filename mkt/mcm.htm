Marketing Campaign Maintenance screen
The Marketing Campaign Maintenance screen allows you to create a campaign and associate it with campaign lists (refer to Creating Campaign Lists) and products.
Menu Path: Marketing > Campaign System >Campaign Maintenance
Field/Function Key Descriptions
Field/Function Key
Description
Campaign Number
The number of the campaign.
Description
A description of the campaign.
Pages
The number of pages in the campaign mailer, fax, or flyer.
Mail Date
The date the material was mailed.
Beginning Date
The date that this campaign becomes effective in Sales and Quote Order Entry.
Ending Date
The date that this campaign is no longer valid in Sales and Quote Order Entry.
Campaign Cost
The total cost of the campaign. Used to calculate the gross profit of the campaign and the cost per response.
F2-Save
Saves the campaign.
F4-Del
Deletes the campaign.
F5-Comments
Allows you to enter notes about the campaign.
F6-Lists
Opens the Campaign Lists screen, which allows you to add campaign lists to the campaign. Refer to Adding Campaign Lists to a Campaign.
F7-Product
Opens the Product Detail screen, which allows you to indicate which products are included for the campaign. Refer to Adding Products.
F8-Product Listing
Shows a list of the products associated with the campaign.
