Marketing Activity Inquiry
The Marketing Activity Inquiry allows you to view all scheduled marketing activities for a user ID and cutoff date. Activities can be scheduled in Marketing Activity Entry or Marketing Activity Contact Entry (refer to Entering a Marketing Activity) or Contact Diary Entry (refer to Creating a Diary Entry). You can edit the scheduled activities, change the date and/or time for the scheduled activity, or enter a quote.
Viewing Marketing Activities
To view the Marketing Activity Inquiry:
1
Select Marketing > Inquiries > Marketing Activity Inquiry to open the Marketing Activity Inquiry screen.
2
Enter the User Id for which you want to see scheduled marketing activities. Defaults to your user ID.
3
For Cutoff, enter the date on or before which you want to see scheduled marketing activities or leave blank to see all schedule activities. Defaults to the current date.
4
Choose a Sort option:
•
Action date.
•
Customer number.
•
Contact.
•
Entry date.
•
Action summary.
•
Priority.
5
Choose a Filter option:
•
No.
•
Yes - Use the following selection criteria:
-
Action date range.
-
Customer.
-
Contact.
-
Priority.
-
Major group
-
Product line.
-
Product.
-
Action summary.
6
To enter information about an activity, highlight Doc for the activity you want to edit and press F5-Update. The Marketing Activity Entry screen appears. You can use the arrow keys to move up and down the list. Refer to Entering a Marketing Activity for more information.
7
To reschedule an activity, press F6-Time/Date. The Change Dates screen appears. To change the date, enter the new date in Enter New Date. To change the time, enter the new time in Enter New Time. Press F2-Save.
8
When you finish working with the scheduled activities, press F2-Save.
Field/Function Key Descriptions
Field/Function Key
Description
Doc
The document type associated with each activity:
•
M – Contact diary.
•
P – Phone.
•
Q – Quote follow up.
•
T – To do.
Date
The date for which the activity is scheduled.
Time
The time for which the activity is scheduled.
Prospect
The number of the prospect or customer on the activity.
Description
The description of the prospect or customer.
Contact
The contact for the activity, if any.
Phone
The phone number for the contact, if any.
Ext
The extension for the contact, if any.
Activity Summary
The description of the marketing activity.
F2-Finish
Closes the inquiry.
F5-Update
Opens the Marketing Activity Entry screen for the highlighted activity. Refer to Entering a Marketing Activity for more information.
F6-Time/Date
Allows you to change the date and/or time for the highlighted activity.
F7-Quote
Opens the Quotation Order Entry screen.
F9-Diary
Opens the Contact Diary Inquiry. For more information, refer to Contact Diary Inquiry.
