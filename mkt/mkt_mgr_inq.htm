Marketing Manager Inquiry
The Marketing Manager Inquiry allows you to see how many scheduled activities are coming up for each user (e.g., sales representative, telemarketing personnel). Activities can be scheduled in Marketing Activity Entry or Marketing Activity Contact Entry (refer to Entering a Marketing Activity) or Contact Diary Entry (refer to Creating a Diary Entry).
Viewing All Scheduled Activities
To view the Marketing Manager Inquiry:
1
Select Marketing > Inquiries > Marketing Manager Inquiry to open the Marketing Manager Inquiry screen.
2
Enter the Warehouse for which you want to see scheduled marketing activities or leave blank to view scheduled marketing activities for all warehouses.
3
To view details about the scheduled marketing activities, highlight the user ID and press F5-Review to open the Marketing Activity Inquiry.
Field/Function Key Descriptions
Field/Function Key
Description
User
The user ID of the user with upcoming marketing activities.
Name
The name of the user with upcoming marketing activities.
Current Date
Shows the number of scheduled marketing activities that are due on or before the current date.
Date at the end of the week
The number of scheduled marketing activities that are due on or before the end of the current week.
Date at the end of the next week
The number of scheduled marketing activities that are due on or before the end of the next week.
Date at the end of the month
The number of scheduled marketing activities that are due on or before the end of the month.
F5-Review
Opens the Marketing Activity Inquiry for the highlighted user ID.
