Setting up Competitor Codes
Competitor codes allow you to track information about a competitor and their prices. You can then use these codes in Competitor Price Tracking Maintenance to track a competitor’s price for a part number (refer to ”Maintaining Competitor Prices & Part Numbers” in the Inventory documentation).
Once you set up competitor codes in Competitor Code Maintenance, you can assign them to Prospects/Customers in the Competitor Information screen.
To create a competitor code:
1
Select Marketing > Maintenance > Competitor Codes to open the Competitor Code Maintenance screen.
2
For Competitor Code, choose from the following:
•
Enter a new code (up to 4 alphanumeric characters).
•
Enter an existing code (e.g.,01).
•
Press F3 to search for an existing code.
3
For Description, enter or edit a brief description (e.g., Fasteners, Inc) of the code.
4
Press F2-Save.
