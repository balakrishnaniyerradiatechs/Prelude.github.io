Contact Diary Code Maintenance screen
The Contact Diary Code Maintenance screen allows you create codes to more specifically identify information about a marketing activity entered through Contact Diary Entry. For more information, refer to Creating a Diary Entry.
Menu Path: Marketing > Maintenance > Contact Diary Codes
Field/Function Key Descriptions
Field/Function Key
Description
Diary Code
The contact diary activity (e.g., 01, 02).
Description
A description of the contact diary activity (e.g., Sent catalog, Called about service).
F2-Save
Saves the code and description.
F4-Del
Deletes the code and description.
