Vendor Payable Type Maintenance screen
The Vendor Payable Type Maintenance screen allows you to setup vendor payable types. Payable types in Vendor Maintenance group vendors. For example, you can use Payable Type 02 for expense type vendors and then use Payable Type to select invoices to pay.
Menu Path: Financial > Accounts Payable > Maintenance > Vendor Payable Type Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Payable Type
The vendor payable is used in Vendor Maintenance to group vendors (e.g., Inventory or Expense).
Description
The description of the payable type.
F2-Save
Saves the vendor payable type information.
F4-Del
Deletes the vendor payable type information.
Action Bar
User Fields
See your Systems Administrator.
