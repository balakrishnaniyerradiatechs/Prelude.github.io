Form 1099 Printing screen
The Form 1099 Printing process allows you to print 1099 forms for your company. After the 1099s print, the 1099 dollar amounts from the previous quarter are deleted.
Menu Path: Financial > Accounts Payable >Processes > Form 1099 Printing
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Number
The vendor number.
Use (C)urrent/(P)revious Year Figures?
Determines which year’s information to use:
•
C - Current year’s totals.
•
P - Previous year’s totals.
Enter 1099 Form Type For This Run
Determines the IRS Form Type to print:
•
I - Interest.
•
D - Dividends.
•
M - Miscellaneous. Selecting Miscellaneous will include R-Royalties, O-Other Income, and N-Non Employee Compensation types in the print process.
Printer Form Queue
The form queue where 1099 forms will print.
