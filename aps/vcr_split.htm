Vendor Claims Split screen
The Vendor Claim Split screen allows you to select individual line items to reconcile with a claim. (e.g., vendor ships one large shipment and submits two or more invoices).
Menu Path: Financial > Accounts Payable > AP/PO Subsystem > Vendor Claim Reconciliation > Claim Reconciliation Entry > F5-Split
Field and Function Key Descriptions
Field/Function Key
Description
Claim#
The vendor claim number.
Vendor#
The vendor number associated with the claim.
Ln# (not labeled)
The line count for the product number.
Product Number
The product number on the vendor claim which is to be split.
Description
The description of the product.
Whse
The shipping warehouse number.
U/M
The unit of measure for the product (e.g., EA, CS).
Qty Clm
The quantity of the product on the claim.
Qty Alw
The quantity allowed on the debit memo.
Cost Clm
The product cost on the claim.
Disc% Clm
The discount amount (if any) applied to the claim.
Disc% Alw
The discount amount (if any) allowed for the debit memo.
Cost Alw
The product cost allowed.
Inc?
Determines if this line item is included on the current receiver or if it is to be split and placed on a different receiver:
•
Y - Include the line item on the current receiver.
•
N - Split the line item, and place on another receiver.
If you select N, a message prompts with the new receiver number. This is the receiver number appended with a (e.g., -50, -51, -52).
Qty Inc
The quantity of products from this line item to include on the current receiver. You can split line items by partial line item quantities.
R/S
Indicates the status of the product:
•
R - Warranty return.
•
S - Scrapped.
Vendor P/N
The product number assigned by the vendor.
Linkage
The order number if the item is linked.
Action Bar
Examine
Displays delivery and receipt information for the highlighted item.
Sort
Determines how the line items of the invoice appear:
•
D - By product description.
•
P - By product number.
•
V - By vendor product number.
Misc Entry
Not valid for vendor claims.
F2-Save
Saves the split receiver information, creates a new receiver, and closes the screen.
F5-Notes
Opens the Notes screen to allow you to view, enter, or edit notes.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are line number, page number, reference number, B (bottom), or T (top).
