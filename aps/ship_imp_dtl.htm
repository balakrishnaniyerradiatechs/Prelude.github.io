Invoice Detail screen
The Invoice Detail screen lists vendor invoices for goods purchased and received through the On-The-Water Maintenance. For more information on ships and On-The-Water receiving, refer to the Inventory documentation. For more information on vendor invoices, refer to the Vendor Invoice Reconciliation Entry screen.
Field and Function Key Descriptions
Field/Function Key
Description
Invoice Number
The number of the vendor’s invoice.
Vendor
The vendor’s account code.
Description
The name of the vendor.
Actual Cost
The amount paid toward the invoice.
F2-Save
Closes the screen.
F5-Review Invoice
Opens the Vendor Invoice History Inquiry.
F6-Cost Detail
Opens the Invoice Detail screen to display detailed information for the invoice.
