Manually Entering Expense Invoices
You can create manual invoices for non-inventory or expense (e.g., rent, supplies) items. Manually entered invoices update A/P and G/L only and not inventory. To manually enter an invoice, refer to:
•
Starting a Batch of Manual Invoices.
•
Posting Manual Invoices.
•
Posting Manual Invoices for Assets.
 
Do not use manual invoices for inventory items. Receive the stock (refer to the Inventory documentation) and then reconcile the receiver with the invoice. For more information refer to Reconciling Inventory Invoices.
