Starting a Batch of Manual Invoices
To enter a batch of manual invoices:
1
Select Financial > Accounts Payable > Manual Invoice Entry to access the A/P Invoice Batch Information screen.
A/P Batch Control in Accounts Payable Parameters determines if manual invoices can be entered by batch:
•
Y - You can enter manual invoices by batch. The A/P Batch Information screen appears in Manual Invoice Entry.
•
N - You cannot enter manual invoices by batch.
2
Press ENTER to get the next Batch Number or enter an existing number.
3
Enter the date on which the batch is created for Batch Date.
4
For Batch Amount, enter the total value of the invoices in the batch or enter 0 if you do not know the total.
5
Press F2-Save. The Standard Account Entry screen appears.
If the vendor has default G/L accounts in Vendor Maintenance, you can use those defaults.You can enter the G/L accounts in the G/L Prefix information section of the Accounts Payable Manual Invoice Entry screen.
6
Enter the appropriate number or press F3 to search for:
•
Div (Division), which represents the highest possible grouping in your company, if applicable.
•
Reg (Region), which is a subset of Division if Division is used.
•
Loc (Location), which is the physical location or warehouse.
•
Dep (Department), which is a subset of Warehouse. Typically used to separate expenses within a Warehouse.
•
Ccr (Cost Center), which is a subset of Department if Department is used.
•
Account# (defaults to your Accounts Payable G/L account number) to use for the manual invoice. The description of the account appears.
7
Press F2-Save. The Accounts Payable Manual Invoice Entry screen appears. Enter as many invoices as necessary (refer to Posting Manual Invoices).
