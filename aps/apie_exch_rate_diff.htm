Currency Info screen
The Currency Info screen shows the currency type, source and exchange rate used for a transaction if you use multiple currencies. You can setup currency types in Currency Type Maintenance. For more information, refer to the General Ledger documentation.
Menu Path: Financial > Accounts Payable > Inquiries > Vendor Invoice History Inquiry > Action Bar > Curr Info
Field and Function Key Descriptions
Field/Function Key
Description
Currency Ind.
The currency code assigned to the vendor. Defaults to native currency.
Curr Rate Source
Displays the source of the currency rate:
•
D - Daily currency rate file.
•
S - Currency Code Maintenance.
Exchange Rate
The currency exchange rate, if other than native currency. Currency exchange rates are defined in Daily Currency Exchange Rate Maintenance.
OK
Closes the currency info screen.
