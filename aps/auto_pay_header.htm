A/P Payment Selection Header screen
The A/P Payment Selection Header allows you to select vendor invoices for payment and print checks. You can automatically or manually select invoices to pay. You can also select invoices based on batch types, A/P Floor Plan, Automatic Bank Deduction, Electronic Funds Transfer, or Credit Card.
Refer to Selecting Vendors/Invoices to Pay for more information.
Menu Path: Financial > Accounts Payable > Check Processing > Payment Selection/Print
Field and Function Key Descriptions
Field/Function Key
Description
Check Batch
The batch code for this check run.
Bank Code
The code assigned to the bank. Bank codes are set up in Bank Codes Maintenance (Systems menu > Accounts Receivable Controls).
Check Date
The date when the checks are processed.
Fiscal Period (not labeled)
The fiscal period.
Batch Type
The batch type for this check run:
•
R - Regular.
•
A - A/P floor plan.
•
B - Automatic bank deduction.
•
E - Electronic funds transfer.
•
C - Credit card.
•
F - A/P floor plan - automatic bank deduction.
Due Date
The date the invoices are due. Invoices with a due date on or before this date are selected for payment. Invoices with a Hold (manually placed on hold in Vendor Invoice Reconciliation Entry) status are not selected for payment. If the Batch Type is set to A (floor planned items), this date is used to select floor planned items that have not been sold but have reached the due date.
Discount Date
The discount date of the invoice. This date is used in the selection process for invoices to pay. All invoices with a Discount Date before or on this date are selected. Invoices with a status of Hold are not selected for payment. Typically, this date should be through the next payment date. For example, if you pay on the 10th and 25th, your 10th of the month run must pay all invoices discounted through the 24th or the invoice will not qualify for the discount. If the Batch Type is set to A, this date is used as a ship date rather than a discount date. If a Discount Date exists for an A batch type, all floor planned invoices are selected for merchandise shipped and invoiced on or before this date.
Take Lost Discounts?
Determines if discounts are available regardless of the Discount Date:
•
Yes - Discounts are available regardless of the Discount Date.
•
No - Discounts are not available if the Discount Date has passed.
Additional Selection
Determines if you will include or exclude selections:
•
No - Uses the criteria entered for this check process.
•
Yes - Opens the A/P Check Add Selection screen so you can use additional criteria such as vendor type and G/L prefix information for the A/P account.
•
Exclude - Opens the A/P Check Add Selection screen so you can to exclude criteria such as vendor type and G/L prefix information for the A/P account.
Sort By Option?
The sequence in which invoices should print on the check stub.
•
D - Due date.
•
I - Invoice number.
•
N - Invoice date.
Number of Vendors
The number of vendors included in this check run.
Gross Pay Amount
The total amount to pay to the vendors.
Discount Amount
The discount amount for the invoice (if any).
Net Amount
The net amount of the check (Gross Pay Amount - Discount Amount).
Steps Done
Header Built
Indicates if the header was built for invoices automatically selected for payment. The header is required for paying invoices that are automatically selected. If invoices are manually selected, the header is not built:
•
X - Yes.
•
O - No.
Auto Selection
Indicates if checks were automatically selected based on header criteria:
•
X - Yes.
•
O - No.
Manual Selection
Indicates if checks were manually selected:
•
X - Yes.
•
O - No.
Check Print
Indicates if checks printed:
•
X - Yes.
•
O - No.
Chk Batch Complete
Indicates if the check batch is complete:
•
X - Yes.
•
O - No.
F2-Save
Saves the check batch information and processes the checks.
F4-Del
Deletes the check batch information.
F5-Auto Sel
Selects invoices to pay based on the selection criteria entered in the header. A report prints so you can review it before you print the checks. For more information refer to Automatically Selecting Invoices to Pay.
F6-Manual Sel
Opens the A/P Payment Manual Selection screen to allow you to manually select the invoices to pay. For more information refer to Manually Selecting Invoices to Pay.
F7-Sel Rpt
Prints or displays the Accounts Payable Payment Selection Register which lists the invoices set to pay.
F8-Print Chks
Processes the payment (e.g., prints checks, creates data for the bank). Refer to Printing Checks.
