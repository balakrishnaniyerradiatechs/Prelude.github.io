Monthly Payment History screen
The Monthly Payment History screen displays payment history for a vendor. You can select the period from which payment was made, the number of checks sent to the vendor, and the check number of the first check paid.
Menu Path: Financial > Accounts Payable > Inquiries > Accounts Payable Inquiry > Inquiries > Monthly A/P History
Field and Function Key Descriptions
Field/Function Key
Description
Period
The fiscal period in which you paid the vendor.
# Pays
The number of payments made to the vendor.
First Check
The check number of the first check paid to the vendor.
F2-Finish
Closes the screen.
F5-Review Checks
Opens the Monthly Payment Check Inquiry screen.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are line number, page number, reference number, B (bottom), or T (top).
