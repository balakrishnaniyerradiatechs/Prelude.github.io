Accounts Payable Released Inquiry
The Accounts Payable Released Inquiry contains:
•
Receivers that a buyer released from hold in the Accounts Payable Discrepancy inquiry (in the Buyer Queue). The receivers that you put on hold will appear for your user ID.
•
Invoices and vendor claims on hold because the reconciliation limits were exceeded for the user that tried to reconcile the vendor invoice. You must be set up as the A/P Manager to release these receivers.
Accounts Payable Manager
A/P Manager in Warehouse Manager Maintenance is the user ID of the person responsible for reviewing and releasing invoices/claims on hold in the Accounts Payable Release Inquiry.
Invoices can go on hold in Vendor Invoice Reconciliation Entry if a user exceeds their limits set in A/P Write-Off Limit, Vend Inv Rec Limit, or Vend Inv Recost% set for their user ID.
From this inquiry screen, you can release the receiver and reconcile the vendor invoice to set it up for payment.
Releasing Receivers on Hold
To view or reconcile a receiver in the accounts Payable Released Inquiry:
1
Select Financial > Accounts Payable > Inquiries > Accounts Payable Released Inquiry. The Accounts Payable Released Inquiry screen appears.
2
Enter your User Id to list all receivers that you are authorized to release and reconcile.
3
To reconcile an invoice released from hold, select F5-AP/PO Entry to open the Vendor Invoice Reconciliation Entry screen. You can reconcile or edit the vendor invoice to set it up for payment. Refer to Reconciling a Vendor Invoice to a Receiver
 
Field & Function Key Descriptions
Field/Function Key
Description
User Id
Your user identification number.
Receiver
The receivers on hold.
Vendor
The vendor number on the receiver.
Description
The vendor’s description on the receiver.
Invoice#
The invoice number assigned to the receiver.
Merch Amt
The merchandise amount of the receiver. This amount comes from Merch Amt on the receiver. For more information, see Reconciling a Vendor Invoice to a Receiver.
APPO Amt
The received amount. This amount comes from Received Total on the purchase order. For more information, see Reconciling a Vendor Invoice to a Receiver.
Freight?
Displays the value of the Receiver Freight flag.
F2-Finish
Closes the inquiry.
F5-AP/PO Entry
Opens the Vendor Invoice Reconciliation Entry screen. You can release the receiver from hold after you edit it. For more information, see Reconciling a Vendor Invoice to a Receiver.
F6-Buyer Notes
Allows you to view and enter buyer notes.
F8-Log
Opens the Purchase Order Log Inquiry screen. Allows you to view the purchase orders related to the receiver.
F9-Goto Line
Allows you to enter a reference number. Valid codes are line number, page number, reference number, B (bottom), or T (top).
Action Bar
Refresh
View receivers that were released since you opened this inquiry.
