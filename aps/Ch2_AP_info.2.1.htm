Accounts Payable Maintenance Requirements
Before you can reconcile vendor invoices and claims, create manual invoices, process checks, you must set up payment terms and other Accounts Payable maintenance options. You can also set flags at the system level in Accounts Payable Parameters to control how Prelude operates for A/P functions. For more information on the available options, refer to the following sections:
•
Setting up Accounts Payable Parameters.
•
Using Accounts Payable Terms.
•
Creating Payable Types.
•
Establishing Cost Adjustment Reason Codes.
•
Tracking Floor Plan Inventory.
•
Creating and Assigning Vendor Discount Codes.
You can maintain vendor A/P settings and A/P contact information in Vendor Maintenance. Refer to the Purchasing documentation for more information.
