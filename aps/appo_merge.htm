Vendor Invoice Merge Routine screen
The Vendor Invoice Merge Routine allows you to merge two or more receivers to reconcile with one invoice.
Menu Path: Financial > Accounts Payable > AP/PO Subsystem > Vendor Invoice Reconciliation > Invoice Reconciliation Entry > F6-Merge
Field and Function Key Descriptions
Field/Function Key
Description
Receiver#
The receiver number to merge.
Vendor
Displays the vendor number and name associated with the Receiver Number.
Receive#
The receiver number(s) to merge with the Receiver Number.
Rec Date
The date the stock was received.
Rec Value
The total value of the receiver.
Total
The calculated total for all receivers listed.
F2-Save
Saves the receiver information and closes the screen.
