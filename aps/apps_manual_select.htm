A/P Payment Manual Selection screen
The A/P Payment Manual Selection screen allows you to select which vendor invoices to pay. You can also modify any invoices that were automatically selected.
Refer to Manually Selecting Invoices to Pay for more information.
Menu Path: Financial > Accounts Payable > Check Processing > Payment Selection/Print > F6-Manual Sel
Field and Function Key Descriptions
Field/Function Key
Description
Ln# (not labeled)
The line count for the number of invoices listed.
Vendor
The vendor number.
Description
The vendor description.
#Inv
The number of invoices selected to pay.
Disc Amount
The discount amount deducted from the invoice (if any).
Gross Pay
The gross amount for this invoice.
Total Batch
The total batch amounts for both the Dsc Amount and Gross Pay columns.
F2-Save
Saves the information and closes the screen.
F5-Maintain Invoices
Opens the A/P Payment Manual Selection Audit screen. The audit screen displays detail information such as dates, amounts, discount amounts and balances for the invoice. If an invoice is selected with a payment due to a miscellaneous vendor, the Miscellaneous Vendor Information screen appears.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are line number, page number, reference number, B (bottom), or T (top).
