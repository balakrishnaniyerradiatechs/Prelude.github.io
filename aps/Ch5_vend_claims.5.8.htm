Choosing a G/L Account
When you write-off a discrepancy, you must specify G/L account(s) to post to. You can limit write-off amounts in User ID Maintenance. For example, if the write-off limit for user ID MKM is $100.00 and the discrepancy is $125.00, an error message notifies you that the write-off amount exceeds the limit. You must choose a different way to correct the discrepancy.
To choose G/L account(s):
1
In Vendor Claim Reconciliation Entry, choose G/L Acct. from the Discrepancy Warning dialog box. The G/L Accounts For Discrepancy screen appears.
2
For each G/L account you want to credit or debit, enter or press F3 to search for each of the following:
•
Div #, (Division), which represents the highest possible grouping in your company, if applicable.
•
Reg# (Region), which is a subset of Division if Division is used.
•
Loc# (Location), which is the physical location or warehouse.
•
Dep# (Department), which is a subset of Warehouse. Typically used to separate expenses within a Warehouse.
•
Ccr# (Cost Center), which is a subset of Department if Department is used.
•
Account # to use for the vendor invoice. The description of the account appears.
3
Enter the discount amount applied in Disc. Amount.
4
Press F2-Save.
