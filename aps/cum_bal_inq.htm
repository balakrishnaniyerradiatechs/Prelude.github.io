Cumulative Balance Inquiry screen
The Cumulative Balance Inquiry screen displays the cumulative balance for the vendor.
Menu Path: Financial > Accounts Payable > Inquiries > Detail Payable Aging Inquiry > F6-Cumulative
Field and Function Key Descriptions
Field/Function Key
Description
Invoice#
The invoice number.
Due Date
The date the invoice is due.
Terms
The terms code on the invoice.
Balance
The balance due on the invoice.
Cumulative
The cumulative A/P balance after each invoice.
OK
Closes the Cumulative Balance Inquiry screen.
