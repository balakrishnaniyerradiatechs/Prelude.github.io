Previewing 1099 Forms
The Form 1099 Preview report allows you to preview 1099 information for the current or previous year.
Menu Path: Accounts Payable > Processes > Form 1099 Printing > Form 1099 Preview Report
Selection Criteria
Use the following criteria to limit the information to print:
•
Vendor Number. The vendor number(s) to include.
•
(C)urrent / (P)revious Year?. Determines the year for which you want to view form 1099 information. Choose from the following:
-
C - View current year information.
-
P - View previous year information.
Column Descriptions
Column Heading
Description
Vendor
The vendor number.
Description
An override name appears if one is entered in the F6-1099 Info screen in Vendor Maintenance for this vendor.
SSN
The social security or tax ID number for the vendor.
1099 Category
The 1099 category which is the IRS form type for this vendor:
•
D- Dividends.
•
I - Interest.
•
M - Miscellaneous.
•
N - Non-employee Compensation.
•
O - Other Income.
•
R - Rents.
1099 Amount
The total dollar amount of the 1099 form.
