A/P Check Add Selection screen
The A/P Check Add Selection screen allows you to include or exclude criteria to select invoices for payment in a batch.
Menu Path: Financial > Accounts Payable > Check Processing > Payment Selection Print > Additional Selection = Y (include) or E (exclude)
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Payable Type
The Vendor Payable Type used to select invoices to pay. Only accounts payable items with this payable type in Vendor Maintenance are selected for check processing. If this is blank, all payable types are selected.
Division
The highest possible grouping in your company.
Region
A subset of Division if Division is used.
Location
The warehouse that the code applies to.
Department
Department number. A subset of Location.
Cost Center
A subset of Department if Department is used.
Open Balance >
The selection will include/exclude invoices with a balance more than this amount.
Open Balance <
The selection will include/exclude invoices with a balance less than this amount.
