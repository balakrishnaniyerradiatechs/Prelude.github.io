Adjusting Discrepancies
If a discrepancy exists between the Merch Amt and Received Total, use one of the following tools to resolve the discrepancy:
•
Recost the claim. For more information, see Recosting a Vendor Claim.
•
Enter a credit or debit memo. Use if Merch Amt and Received Total do match. For more information, see Entering a Credit or Debit Memo.
•
Choose a G/L account. For more information, see Choosing a G/L Account.
•
Create an additional invoice. Use if Merch Amt and Received Total don’t match. For more information, see Creating an Additional Debit or Credit Memo.
•
Put the claim on buyer hold. For more information, see Placing the Vendor Claim on Buyer Hold.
