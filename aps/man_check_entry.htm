Manual Check Entry screen
If you write a check that is not processed through Prelude, you must use the Manual Check Entry screen to record information about the check. You can reconcile the check with an existing invoice, create a new invoice, or record the check without an invoice.
Menu Path: Financial > Accounts Payable > Check Processing > Manual Check Entry
