AP/PO Header screen
The AP/PO Header screen displays the header information for an invoice.
Menu Path: Financial > Accounts Payable > Injuries > Vendor Invoice History Inquiry F5-Review Receiver > F5-Hdr Scr
Field and Function Key Descriptions
Field/Function Key
Description
Invoice Number
The invoice number for the receiver.
Invoice Date
The date of the invoice.
Status
The status of the invoice:
•
O - Open.
•
H - On hold.
•
B - Automatic bank deduction.
•
E - Electronic funds transfer.
•
C - Credit cards.
•
F - Floor plan - automatic bank deduction.
•
A - Floor plan - print check
Terms Code
The terms code from the original purchase order.
Discount Date
The discount date from the original purchase order.
Due Date
The date the invoice is due. Due Date is based on the Invoice Date and Terms Code.
Merchandise Amount
The merchandise total for the invoice.
Freight Amount
The freight amount for the vendor invoice (if any).
Include?
Determines if the Freight Amount is included in the payment to the vendor:
•
Y - Include the freight amount.
•
N - Do not include the freight amount.
Duty Amount
The duty amount for the vendor invoice (if any).
Include?
Determines if the Duty Amount is included in the payment to the vendor:
•
Y - Include the duty amount.
•
N - Do not include the duty amount.
Brokerage Amount
The brokerage amount for the vendor invoice (if any).
Include?
Determines if the Brokerage Amt is included in the payment to the vendor:
•
Y - Include the brokerage amount.
•
N - Do not include the brokerage amount.
Tax Amt
The tax amount for vendor invoice (if any).
Include?
Determines if the Tax Amt is included in the payment to the vendor:
•
Y - Include the tax amount.
•
N - Do not include the tax amount.
Discount Amount
The discount amount associated with the terms code.
Misc Amt
Any miscellaneous debits or credits not already included in this entry.
Invoice Amount
The total amount of the invoice. The total includes the merchandise amount and the freight, duty, brokerage, and miscellaneous charges if any. The total amount does not reflect the discount amount.
OK
Closes the AP/PO Header screen.
F5-Hdr Scr
Opens the Purchase Order Header Display screen, which allows you to view the purchase order header information.
F6-Credit Memo
Opens the AP/PO Credit Memo Entry screen, which allows you to view any credit memo information for the invoice.
F7-Remittance Comment
Opens the Remittance Check Comment screen, which allows you to view any check comments for the invoice.
Action Bar
Notes
Opens the Notes Information screen, which allows you to view any notes for the invoice.
Multi Level Discounts
Opens the Multi Discounts screen, which allows you to view discount dates and term percentages for this vendor.
