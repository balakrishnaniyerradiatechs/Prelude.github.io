Consigned Inventory screen
The Consigned Inventory screen displays products on consignment invoices. If you create a new invoice, this screen is blank.
Menu Path: Financial > Accounts Payable > Manual Invoice Entry > Action Bar > Consigned
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Number
The vendor number and description.
##
The line count of the consigned item.
Cons Flag
The type of transaction the consigned product is on (e.g., purchase order, receiver).
Document#
The document number for the consignment item.
Whse
The warehouse number on the transaction.
Product Number
The product number on consignment.
Quantity
The quantity of the consignment item.
Extension
The total amount of the item (Qty X (Cost + Add On Cost).
Total
The total amount for the consignment items.
OK
Saves the consignment product information.
F7-Toggle All
Opens the Consignment Invoice Selection screen to allow you to enter a beginning and ending transaction date to limit the invoices that appear.
F8-Toggle
Allows you to tag the consignment items to pay on this invoice.
F9-Goto
Allows you to enter a reference number for quick access. Valid codes are line number, page number, reference number, B (bottom), or T (top).
Action Bar
Sort
Sorts the consigned inventory records on the screen by product number or by warehouse by product.
