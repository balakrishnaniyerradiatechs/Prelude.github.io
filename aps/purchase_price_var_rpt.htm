Purchase Price Variance Report
The Purchase Price Variance report list purchase orders that were re-costed (refer to Recosting a P/O).
Menu Path: Accounts Payable > AP/PO Subsystem > Vendor Invoice Reconciliation Entry > Purchase Price Variable Report
Selection Criteria
Use the following criteria to limit the information to print:
•
Vendor. The vendor number(s) to include.
•
Process Date range. The date the cost was changed.
•
Period. The fiscal period for the recosted P/Os.
Column Descriptions
Column Heading
Description
Receive#
The receiver number on which cost was changed.
Vend#
The vendor number.
Whse#
The warehouse number where the products were received.
Invoice#
The invoice number.
Product Number
Line 1: The product number.
Line 2: The description of the product.
Line 3: The second line of description of the product.
Qty Rec
The quantity received.
U/M
The unit of measure for each product.
Original Cost/Disc.
Line 1: The original cost of the item.
Line 2: The discount amount for the item (if any).
Extension
The extended amount for the products (Original Cost X Qty-Rec).
Override Cost/Disc.
Line 1: The new cost allocated to this product.
Line 2: The discount amount (if any).
Extension
The extended cost of the item using the override cost for the product (Override Cost X Qty-Rec).
