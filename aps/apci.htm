Accounts Payable Check Inquiry
Accounts Payable Check Inquiry lists vendor invoices paid on a check number. You can view detailed information for each invoice.
Viewing the Accounts Payable Check Inquiry
To view all paid invoices:
1
Select Financial > Accounts Payable> Inquiries > Accounts Payable Check Inquiry to open the Accounts Payable Check Inquiry screen.
2
Enter the Bank Code or press F3 to search.
3
To select the Check Number, choose one of the following:
•
Enter the payment number.
•
Press F3 and enter search criteria. You can search by Vendor Number and/or Check Type. If you select a vendor number and leave the check type blank, all checks for that vendor appear. If you leave the vendor number blank and select a check type, all checks with that check type appear.
The check information appears.
Field & Function Key Descriptions
Field/Function Key
Description
Bank Code
The bank code used when processing the payment.
Check Number
The payment number.
Vendor
The vendor number associated with the payment.
Check Date
The date the payment was generated.
Check Amount
The amount of the payment.
Check Type
The type of payment.
•
A/P payment. Refer to Paying the Vendor.
•
Manual payment. Refer to Recording Manual Checks.
•
Voided payment. Refer to Voiding Payment.
Address (not labeled)
The vendor’s name and address.
Invoice Number
The invoice(s) paid with this payment.
Inv Date
The date the invoice was paid.
Gross Paid
The total amount of the invoice.
Disc Taken
The discount amount deducted from the invoice.
Net Paid
The net amount paid to the vendor (Gross Paid - Disc Taken).
Gross Paid Total (not labeled)
The total gross paid for all invoices listed.
Total Disc Taken (not labeled)
The total discount taken for all invoices listed.
Net Paid Total (not labeled)
The net total paid for all invoices listed.
Total Manual G/L Dist.
The total amount manually distributed to G/L accounts.
F2-Save
Closes the inquiry screen.
F5-Review Invoice
Opens the Vendor Invoice History Inquiry.
F6-Distribute to G/L
Opens the Miscellaneous G/L Display screen, which shows miscellaneous G/L entry information for this invoice.
Action Bar
Paid For Vendor
Opens the Paid For Vendor Information screen, which shows the vendor was paid if using the Pay To Vendor.
