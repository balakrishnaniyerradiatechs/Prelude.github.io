Daily Vendor Invoice Distribution Register
The Daily Vendor Invoice Distribution Register displays all vendor invoices for a date range including the distribution account and distribution amount per vendor invoice.
Menu Path: Accounts Payable > Reports > Daily A/P Reports > Vendor Invoice Distribution Register
Selection Criteria:
Use the following criteria to limit the information to print:
•
Posting Date range. The beginning and ending dates for which you want to view posted invoices.
•
Period. The fiscal period of the vendor invoice.
•
User Id. The user ID code of the person who entered the invoice.
•
Batch Number. The batch number assigned when entering a manual invoice.
•
Vendor. The vendor(s) for which you want to display invoices.
•
Vendor Type. The vendor type(s) for which you want to display invoices.
Column Descriptions
Column Heading
Description
Vendor
The vendor number.
Description
The name of the vendor.
Invoice Number
The vendor invoice number for this transaction.
Inv Date
The invoice date of this transaction.
Due Date
The date the vendor invoice is due.
S
The status of the vendor invoice:
•
O - Open.
•
H - Hold.
•
P - Paid.
Disc. Amt
The discount amount taken from the vendor invoice.
Inv. Amount
The amount of the vendor invoice.
Dist. Prefix
The distribution prefix associated with the vendor invoice.
Account#
The G/L account number posted for this invoice.
Dist. Amount
The dollar amount distributed to the G/L account.
