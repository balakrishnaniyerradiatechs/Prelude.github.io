APPO Import Costs screen
The APPO Import Costs screen displays import costs for an invoice. You can access the APPO Import Costs screen from the Vendor Invoice Reconciliation Entry screen and Vendor Claim Reconciliation Entry screen.
Menu Path: Financial > Accounts Payable > AP/PO Subsystem > Vendor Invoice Reconciliation > Invoice Reconciliation Entry > Action Bar > Import Cost
Field and Function Key Descriptions
Field/Function Key
Description
Import Cost
The Import Cost Code for the vendor invoice. The Import Cost Codes are setup in Import Cost Factor Maintenance.
Description
The description of the Import Cost.
Import Cost
The import cost of the item.
Currency
The currency type from Vendor Maintenance if you use multiple currencies.
Total
The total import cost for this invoice.
OK
Closes the Import Costs screen.
