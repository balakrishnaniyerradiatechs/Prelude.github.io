Daily A/P Distribution Report
The Daily A/P Distribution report shows all vendor invoices by posting date and G/L account.
Menu Path: Accounts Payable > Reports > Daily A/P Reports > A/P Distribution Register
Selection Criteria
Use the following criteria to limit the information to print:
•
Posting Date range. The beginning and ending dates for which you want to view posted invoices.
Column Descriptions
Column Heading
Description
Dist. Prefix
The distribution prefix number associated with vendor invoice.
G/L #
The G/L account number the invoice is posted to.
Vendor
The vendor number.
Description
The name of the vendor.
Invoice Number
The vendor invoice number.
Inv Date
The invoice date of this transaction.
Due Date
The due date of the vendor invoice.
P/O Number
The purchase order number associated with the vendor invoice.
Dist Amt
The distribution amount posted to the G/L account.
