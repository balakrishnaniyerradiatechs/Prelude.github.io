Posting Manual Invoices for Assets
If you choose an asset account for a manual invoice, the Equipment ID Number screen appears.
For each asset invoice, enter the following:
1
For Equip ID#, enter the EIN (equipment ID number) or description or press F3 to search.
If your company tracks serialized equipment and related attachments, you cannot use an attached part as an account for posting (Equipment Status = E).
2
For Update Inv?, choose one of the following:
•
Y - Adjusts cost for the serial number.
•
N - Does not adjust cost for the serial number. Go to step 4.
3
For Reas Code, enter the reason code (e.g., 01 Conversion Error) that represents why you are updating this invoice.
4
For FA Chrg Type, enter the asset tracking charge type for this invoice:
•
A - Add-On Capital.
•
C - Capital.
•
F - Freight.
•
I - Interest.
•
ML - Maintenance Labor.
•
MP - Maintenance Parts.
•
O - Other Maintenance Expense.
•
SL - Sublet Labor.
•
SP - Sublet Parts.
•
T - Taxes.
5
Press F2-Save to update the assets record for the piece of equipment.
Creating Sub Assets
Create Sub Asset From APIE in Accounts Payable Parameters determines if a sub asset is created in Accounts Payable Manual Invoice Entry:
•
Y - Do not update the main asset. Create a sub asset for the add-on capital.
•
N - The main asset for the equipment number is updated with the dollars for add-on capital.
