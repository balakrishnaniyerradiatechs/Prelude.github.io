Void Check Batch Entry screen
The Void Check Batch Entry screen allows you to void a range of payments. Refer to Voiding Multiple Payments for more information.
Menu Path: Financial > Accounts Payable > Check Processing > Void Check Batch Entry
Field and Function Key Descriptions
Field/Function Key
Description
Bank Code
The bank code assigned to the payments to void. You cannot void checks from multiple banks at the same time.
Check Number >=
The beginning check number to void.
Check Number <=
The ending check number to void.
Void Date
The date the checks are voided.
F2-Save
Voids the checks.
