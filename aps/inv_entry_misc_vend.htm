Miscellaneous Vendor Information screen
The Miscellaneous Vendor Information screen displays the address and phone for a miscellaneous vendor.
Menu Path: Financial > Accounts Payable > Inquiries > Vendor Invoice History Inquiry > Action Bar > Misc Vend
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Name
The name of the vendor.
Address One
The first line of the vendor’s address.
Address Two
The second line of the vendor’s address.
Address Three
The third line of the vendor’s address.
City
The city in which the vendor is located.
State
The state in which the vendor is located.
Zip
The zip code in which the vendor is located.
Phone
The vendor’s phone number.
Contact
The contact name for the vendor.
OK
Closes the Address Information screen.
