EDI Freight Invoice Second Level Inquiry
The EDI Freight Invoice work queue contains freight invoices converted through EDI. Once a freight invoice is released from the EDI Freight Invoice work queue in the Buyer Queue Inquiry (refer to the Purchasing documentation), it goes into the EDI Freight Invoice Second Level work queue so that the 2nd Level Frt Manager can set it up for payment.
Freight Manager
2nd Level Frt Manager in Warehouse Manager Maintenance is the code for the manager that can release orders from EDI Freight Invoice Second Level hold for this warehouse. A user ID must have this same code assigned in Freight Manager in the Administrative Settings screen for their user ID to be able to release freight invoices from hold for this warehouse.
Releasing EDI Freight Invoices
To Release EDI freight invoices for payment:
1
Select Financial > Accounts Payable> Inquiries > EDI Freight Invoice Second Level Inquiry. The EDI Freight Invoice Second Level Inquiry screen appears.
2
Enter the Warehouse for which you want to release freight invoices.
3
Enter the Freight Manager ID.
EDI Invoices on Hold appear. You can review the invoices or enter any notes (refer to the Field and Function keys table).
4
Enter a Y under A (approved) for each invoice that is approved for invoicing.
5
Press F2-Finish to update the General Ledger and set the invoice up for payment.

The vendor number associated with the freight invoice.

The vendor description.

The number of pounds the freight invoice is charging for.
Amount Due
The freight amount due.
Ship From/To
The ship-to address for inbound freight or ship-from address for outbound freight.
City/State
The city and state the freight will ship to or was shipped from.
I/O
Indicates if the freight invoice is for outbound or inbound freight:
•
O - Outbound.
•
I - Inbound.
Approved (A)
Indicates that the freight bill is ready for releasing. If you enter Y and press F2-Finish, the invoice will update for payment.
