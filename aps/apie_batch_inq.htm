A/P Vendor Invoice Batch Information screen
The A/P Vendor Invoice Batch Information screen displays the batch information for the vendor invoice.
Refer to Entering a Batch of Recurring Invoices for more information.
Menu Path: Financial > Accounts Payable > Recurring Vendor Invoices > Recurring Vendor Invoice Entry > F9-Batch Info
Field and Function Key Descriptions
Field/Function Key
Description
Batch Number
The batch number.
Batch Date
The date on which the batch was created.
Batch Amount
The dollar amount of the batch.
Vendor
The vendor number.
Description
The vendor description.
Invoice Number
The invoice number for the recurring vendor invoice.
Invoice Amount
The amount for the recurring vendor invoice.
# Of Invoices
The number of invoices in the batch.
Applied Total
Displays the total dollar amount applied to the batch.
OK
Closes the Vendor Invoice Batch Information screen.
