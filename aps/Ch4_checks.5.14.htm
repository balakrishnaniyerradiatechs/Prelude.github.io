Voiding a Payment
To void a single payment:
1
Select Financial > Accounts Payable > Check Processing > Void Check Entry to access the Voided A/P Check Entry screen.
2
Enter the Bank Code for the payment you want to void. Use the same Bank Code as when the payment was generated.
3
In Check Number, enter the number of the payment you want to void. The Vendor, Check Date, and Check Amount appear based in the Check Number you select.
4
Update it as necessary and press F2-Save. The Void Date defaults to today’s date. The following dialog box appears.
Do You Wish To Continue With Voiding Check Number 000027?
5
Choose one of the following:
•
No - Leave the payment in the system and cancel the void.
•
Yes - Void the payment.
