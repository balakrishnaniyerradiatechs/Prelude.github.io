Vendor Invoice Split Routine screen
The Vendor Invoice Split Routine allows you to select which line items are reconciled on an invoice. This is used when vendors ships one large shipment but submits two or more invoices, or if your warehouse receives multiple small shipments at one time (creating one receiver) and invoiced for each shipment separately. You can split receivers to determine which line items are matched with an invoice.
Menu Path: Financial > Accounts Payable > AP/PO Subsystem > Vendor Invoice Reconciliation > Invoice Reconciliation Entry > F5-Split
Field and Function Key Descriptions
Field/Function Key
Description
Receiver#
The receiver number you wish to split.
Vendor
The vendor number and description associated with this receiver.
Product Number
The product number of the item on the vendor invoice.
Description
The product description of the item on the vendor invoice.
Whse
The warehouse number in which the products were received.
Qty Rec
The total product quantity received for each line item on the receiver.
U/M
The unit of measure for each line on the receiver.
Cost
The product cost for the line item.
Disc%
The discount percentage applied to this line item (if any).
Inc?
Determines if this line item is included on the current receiver or if it is to be split and placed on a different receiver:
•
Y - Include the line item on the current receiver.
•
N - Split the line item, and place on another receiver.
If you select N, a message prompts with the new receiver number. This is the receiver number appended with a (e.g., -50, -51, -52).
Qty Inc
The quantity of products from this line item to include on the current receiver. You can split line items by partial line item quantities.
Vendor P/N
The product number assigned by the vendor.
F2-Save
Saves the split receiver information and closes the screen.
F5-Notes
Opens the Notes screen.
F9-GoTo Line
Allows you to enter a reference number for quick access. Valid codes are line number, page number, reference number, B (bottom), or T (top).
Action Bar
Examine
Displays delivery and receipt information for the highlighted item.
Sort
Determines the order in which the lines appear:
•
D - Product description.
•
P - Product number.
•
V - Vendor product number.
