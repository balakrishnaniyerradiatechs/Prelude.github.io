Creating Payable Types
Payable types classify vendors (e.g., Inventory, Expense, Supplies) and are used to select invoices to pay. Refer to Selecting Vendors/Invoices to Pay.
To create payable types:
1
Select Financial > Accounts Payable > Maintenance > Vendor Payable Type Maintenance to access the Vendor Payable Type Maintenance screen.
2
For Payable Type, enter the code for the type or press F3 to search.
3
Enter the Description of the payable type.
4
Press F2-Save.
You can assign Payable Types to vendors in Vendor Maintenance > F6-A/P > Payable Type. For more information, refer to the Purchasing documentation.
