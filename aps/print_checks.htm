Printing Checks
If the Batch Type is R or A (floor plan), checks will print. To print checks to send to the vendor:
1
On the A/P Payment Selection Header screen, press F8-Print Chks. The Print Screen appears.
2
For Printer Name, select a printer setup specifically to print checks (e.g., APCHECK).
3
For Stationery, select the stationery for printing checks. If you receive an error message, contact your Systems Administrator.
4
Press F2-Save. The following message appears.
Now ready to begin check run. Load Check Stock into printer 56 and press Continue.
5
Load your check stock and choose Continue. The following message appears.
Ready to run Alignment Routine
6
Choose one the following:
•
Run Alignment - Tests the alignment of the check forms in the printer. Not necessary if you are using a laser printer. After the forms are correctly aligned or if you do not need to check alignment, select Alignment OK.
•
Alignment OK - Indicates that the check forms are properly aligned in the printer and you are ready to print checks.
•
Quit - Goes back to the A/P Payment Selection Header screen.
7
When you choose Alignment OK, the Get the starting Check Number screen appears.
8
Verify that the number in Enter the Starting Check Number is the correct check number. Change it if necessary.
9
Press ENTER to verify the check number.
10
When you are ready to print checks, press ENTER. After the checks print, the following message appears.
Last Check Number--> 215. Did The Checks Print Correctly?
To stop printing checks, press any key.
11
Verify that each check printed correctly and compare the last check number to the one in the Verify Last Check Number dialog box. Choose one of the following:
•
Yes - If all checks printed correctly and the number on the last check printed matches the number in the Verify Last Check Number dialog box. The Reset Printer dialog box appears. Follow the instructions and choose Continue.
Now ready to reset printer to previous form. After the checks finish printing, remove check stock and press Continue.
•
No - If the checks did not print correctly and/or the number on the last check printed does not match the number in the Verify Last Check Number dialog box. The Get the Last OK Check screen appears.
Last Check Number --> 215. Did The Check Print Correctly?
12
Choose one of the following:
•
If no checks printed correctly, type VOID and press ENTER.
•
If some checks printed correctly, enter the number of the last good check and press ENTER.
13
When you are ready to void checks, press ENTER.
14
Repeat steps 8-14 until the checks print correctly.
MICR Format
Print MICR on A/P Checks in Accounts Payable Parameters determines if the bank account and routing number print on the checks in MICR (magnetic ink character recognition) format:
•
Y - Print the bank information in the MICR format.
•
N - Do not print the bank information in the MICR format.
