Debit Memo Printing screen
The Debit Memo Printing process allows you to print debit memos created from the Additional Invoice option in Vendor Invoice Reconciliation Entry screen.
Refer to Printing Debit Memos.
Menu Path: Financial > Accounts Payable > AP/PO Subsystem > Vendor Invoice Reconciliation > Debit Memo Printing
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Number
The vendor number.
Debit Memo Number
The debit memo to print. If left blank, all debit memos print.
Document Option
Determines how debit memos are sent to the vendor:
•
P - Printed (default).
•
F - Faxed.
•
E - E-mailed.
Printer Form Queue
The form queue where the debit memos print.
