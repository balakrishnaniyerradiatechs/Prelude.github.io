Daily Vendor Invoice Merchandise Register
The Daily Vendor Invoice Merchandise Register displays all vendor invoices for a date range and fiscal period.
Menu Path: Accounts Payable > Reports > Daily A/P Reports > Vendor Invoice Merchandise Register
Selection Criteria
Use the following criteria to limit the information to print:
•
Posting Date range. The beginning and ending dates for which you want to view posted invoices.
•
Period. The fiscal period of the vendor invoice.
•
User Id. The user ID code of the person who entered the invoice.
•
Batch Number. The batch number assigned when entering a manual invoice.
•
Vendor. The vendor(s) for which you want to display invoices.
•
Vendor Type. The vendor type(s) for which you want to display invoices.
Column Descriptions
Column Heading
Description
Vendor
The vendor number.
Description
The name of the vendor.
Invoice Number
The vendor invoice number for this transaction.
Inv Date
The invoice date of this transaction.
Due Date
The due date of the vendor invoice.
S
The status of the vendor invoice:
•
O - Open.
•
H - Hold.
•
P - Paid.
Dsc Date
The date by which the invoice must be paid to deduct the discount.
Inv. Amount
The invoice amount of the vendor invoice.
Disc. Amt
The discount amount taken from the vendor invoice.
Net Amount
The net amount after the discount amount is deducted (Inv. Amt - Disc Amt).
