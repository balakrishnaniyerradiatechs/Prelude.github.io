Automatically Selecting Invoices to Pay
To automatically select invoices to pay:
1
Enter header information and selection criteria (refer to Starting a Batch).
2
Press F5-Auto Sel. Invoices that match your criteria are selected. If you already manually or automatically selected invoices, the following dialog box appears.
A Select has been done already. Running Auto Select will lose any Selections already made.
3
Choose one of the following:
•
Return to Header - does not automatically select any invoices. Any previous manual and automatic selections remain in the batch. The batch is not updated.
•
Continue - automatically selects invoices based on the selection criteria entered. Your previous manual and automatic selections are discarded.
4
If you choose Continue, the Print screen appears. Choose one of the following:
•
Press F2-Save to print the Accounts Payable Payment Selection Register report that lists invoices selected. You can also print the report by selecting F7-Sel Rpt.
•
Press F4-Del or ESC to not generate the report.
You can now manually edit the list of invoices or print checks.
