Establishing Cost Adjustment Reason Codes
Cost adjustment reason codes identify why a cost was adjusted.
Accounts Payable Parameters:
If the S/R Adj. Reason Code in Accounts Payable Parameters is set to Y, you must enter an adjustment reason code when you adjust cost. Refer to Setting up Accounts Payable Parameters.
To create reason codes:
1
Select Financial > Accounts Payable > Maintenance > AP/PO Cost Adjustment Reason Maintenance to access the AP/PO Cost Adjustment Reason Maintenance screen.
2
For Reason, enter the code or press F3 to search.
3
Enter the Description of the reason code.
4
Press F2-Save.
