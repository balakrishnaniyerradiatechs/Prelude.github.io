AP/PO Cost Adjustment Reason Maintenance screen
The AP/PO Cost Adjustment Reason Maintenance screen allows you to create cost adjustment reason codes. Reason codes are used when you adjust costs or discounts on a purchase order in Vendor Invoice Reconciliation Entry (refer to Recosting a P/O).
Cost Adjustment Reason Codes
S/R Adj. Reason Code in Accounts Payable Parameters determines if a cost adjustment reason code is required when a received cost is changed:
•
Y - Require a cost adjustment reason code when cost is adjusted.
•
N - Do not require a cost adjustment reason code when cost is adjusted.
Menu Path: Financial > Accounts Payable > Maintenance > AP/PO Cost Adjustment Reason Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Stk Adj Reason
The one or two-digit reason code that identifies why a cost is adjusted on a purchase order in the Vendor Invoice Reconciliation Entry screen.
Description
The description of the cost adjustment reason code that explains why the cost is adjusted. The description displays on all screens and reports that have the cost adjustment reason code.
Action Bar
User Fields
See your Systems Administrator.
F2-Save
Saves the adjustment reason code information and closes the screen.
F4-Del
Deletes the adjustment reason code information.
