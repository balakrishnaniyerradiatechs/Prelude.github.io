Default Page/Catalog View Maintenance
•
Page Name: The name of the default page.
•
Page Heading: The default page heading to be displayed.
•
Number Of Columns: The number of columns to be displayed across the page.
•
Style Prefix: The style sheet that determines appearance characteristics.
•
URL Text: The URLs that appear when the default page opens.
•
Type: Enter L - URL text is attached to a list, P - URL text is attached to a page, U - URL text is attached to a custom page, X - menu item is attached to an external link.
•
Shopping List Name: The URL name attached to the URL Text. Reserved list names are New Products, Specials, Clearance and the Web User ID.
•
F5-URL Images: If you have an image for the URL, enter the path for the default.
•
F6-Main Image: If you have an image for the main page, enter the path for that image. The image appears to the left or right of the links on the page, based on the Align option in UD.
•
Action Bar
-
Copy: Enter the list name to copy and the name of the new list.
-
Rename: Enter the list name to rename and the new name.
•
Web Image Maintenance: Web Image Maintenance allows you to define the type, path, and description of images associated with a product (e.g., thumbnails, product images). Web images may also be imported from a spreadsheet. Contact VIA support for more information.
•
Products: Select the product associated with the images.
•
Product Lines: Select the product line for the images. Use product lines when all products in a product line have the same image and thumbnail. Images at the product override those at the product line.
•
Product Number: The product number or line for which you have an image.
•
Image Type: The type of image: Schematic, Material Safety Data Sheet, Specifications, Literature, Hazardous Material Sheet, Catalog, Product Image, Thumb Nail Image, Certificates, or Other.
•
New Web User Maintenance
The Web Queue Mgr uses this screen to view newly registered web users. The Web Queue Mgr must be set up in the New Web User Queue in the Company Web Policies Maintenance screen.
•
Web Queue Mgr: The ID of the person who managers new web users.
•
Web User: The ID for the new web user. Not editable.
•
Creation Date: The date on which the web user was created. Not editable.
•
F5-Web User Mnt: Opens the Web User Maintenance screen.
