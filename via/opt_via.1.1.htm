VIA integrates with Prelude to allow your company to enhance your web site. With VIA, you and your customers can browse, purchase, and research products.
Company Web Policies
Company web policies are the global setup for VIA. These flags override those at the user level.
Check any of the following to indicate that…
•
Login Required For Access: Users must login to access the web site.
•
Customer Pricing On Order: Users can see pricing, including discounts or special pricing.
•
Dynamic User Create: Users can register and create a User ID for themselves on the web.
•
Use Cookies For Login: Users can check Remember Me to save their login and password.
•
Allow Line Required: Users can enter required date on sales orders.
•
Allow Line Comments: User can enter comments about products on sales orders.
•
Order/Quote Acknowledgements: Users receive acknowledgement sfor orders and quotes.
•
Enforce Minimum Sell/Pkg Qty: The minimum selling quantity for products is enforced.
•
Enable Budget: You want to use the budgeting module.
•
Auto Purge Cart: Shopping cart contents are purged after the Auto Purge Cart Days.
•
Enable Flat Rate Shipping: Users can determine shipping charges by zip code.
•
Autosplit Transaction: When a quote is converted, non-stock products are deleted.
•
Reprice On Checkout: Product pricing uses current Prelude prices.
•
Use Cc Verification: Credit cards will be verified on-line.
•
Hold All Orders: All sales orders go on admin order hold.
•
Duplicate Cart Items: A product can be entered on order multiple times.
•
Available Policy: The program that calculates product availability.
•
Advanced Search Page Name: URL of the page that users go to for an Advanced Search.
•
Maximum Items Per Page: Maximum lines of products and summaries per page.
•
Webmaster Email Address: The address for support email.
•
Auto Purge Cart Days: The number of inactive days after which shopping carts are purged.
•
New Web User Queue: User ID that reviews web users created on the web site.
•
Credit Card Req. Terms: The A/R terms for web users that pay by credit card.
•
Campaign Number: Marketing campaigning code for discount pricing.
•
Default Page Style: How users view the website. Normal view lists links in a column. Tree view lists links on the left side of the page in a tree-and-branch style.
•
Product Search Sort: The dictionary name used to sort products (ascending or descending).
•
F5-View Options: Refer to View Options.
•
F6-Contact Info: Refer to Contact Info .
•
F7-Def Page: Refer to Default Page.
•
Action Bar Options: Refer to Action Bar.
