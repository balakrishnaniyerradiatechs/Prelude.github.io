Web User Maintenance
Use Web User Maintenance to allow new web users to place orders, view products, or research information. To allow users to register themselves, setup a DEFAULT account with the required information.
•
Web User Id: The ID (3 – 20 characters) for this user to logon to the web site.
•
Creation Date: The date the User ID was created. Not editable.
•
Web Password: The user’s password (1-20 characters).
•
User Type: Determines if user type is default (retain default web user info), no login (not required to login), or normal (required to login).
•
Last Modified: The date the user ID was last changed. Not editable.
•
Salutation: User’s salutation (Mr./Mrs./Ms.)
•
Last Access: The date the user last logged in. Not editable.
•
First Name: The user’s first name.
•
Last Name: The user’s last name.
•
Company: The name of the customer the user works for.
•
Is Internet Interface: Indicates if this user participates in Internet Connectivity Data Transfers. Direct VIA and Consortia Interfaces. This does not apply to EDI transactions.
•
Address 1,2,3: The user’s address.
•
City: The user’s city.
•
State: The user’s state abbreviation.
•
Zip: The user’s zip code.
•
Country: The user’s country.
•
Email: The user’s e-mail address.
•
Phone #, Ext, Fax #: The phone number, extension, and fax where the user can be reached.
•
Company #: The Prelude company for this user (mandatory for the DEFAULT account).
•
SB User: The user’s System Builder ID (mandatory for the DEFAULT account).
•
Customer #: The Prelude customer for this user (mandatory for the DEFAULT account).
•
Ship To: Blank if the ship to address if the same as the address above. The ship-to from CSTM if different.
•
Default Ship Via: The ship via code.
•
Purch. Contact: The user’s contact number. All sales information is tracked by this contact.
•
Sell Warehouse: The user’s selling warehouse (mandatory for the DEFAULT account).
•
Ship Warehouse: The user’s shipping warehouse (mandatory for the DEFAULT account).
•
F5-Access: Refer to Access.
•
F7-Def Page: Select the default page that appears when you log in.
•
F8-Conv Cust: Converts the temporary customer to the customer.
•
Action Bar
-
Copy: Allows information from one web user to be copied to another. Enter the ID of the web user you want to copy and enter the web ID of the new user.
-
Cart: Allows you to review a web users shopping cart. If you are using multi-carts you can choose the cart you want to review.
-
Budgets: Refer to Budgets.
-
Custom: User fields.
-
Info: User fields.
