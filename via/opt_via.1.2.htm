View Options
•
Show Available On Web: Product availability appears on the web.
•
Show Kit Available: Availability is based on the components available.
•
Show Add to List: Users can add a product to a favorites list.
•
Show Featured Product Icons: Icons appear for Clearance, Specials, or New Products lists.
•
Show Suggested Retail: MSRP appears for the products.
•
Show Substitutes: Products with substitutes will have a link to the substitute products.
•
Show Accessories: Products with accessories will have a link to the accessories.
•
Show Priority Code: Users can enter a campaign code at checkout.
•
Show Thumbnail Image: Thumbnail images for products will be displayed.
•
Show Flyover Help: The flyover help appears on the web site.
•
Show Printable Views: Users can view and print product pages without extra information.
•
Null Qualifier: Users can only see products with qualifiers.
•
Show Expired Quotes: Users can view expired quotes in Quote Inquiry.
•
F5-Look/Feel: Refer to Look/Feel.
•
F6-Comments: Refer to Comments.
Look/Feel
The Look/Feel option allows you to determine where images appear on the web page.
•
Search Display: Show the search field in the Advertise, Data, or Menu panel.
•
Menu Display: Show the menus in the Advertise, Data, or Menu panel. To use the Menu area, Menu Display must be set to Menu Area.
•
Logo Display: Show the logo in the Advertise, Data, or Menu panel.
•
Customer Info: Show customer info in the Advertise, Data, or Menu panel.
•
Banner Display: Show the banner in the Advertise, Data or Menu panel.
•
Use Buttons Instead Of Menus: Use buttons, not menus, for options at the top of the page.
•
Where: Choose where you want the data to appear:
-
Advertise Area - Upper frame in the browser window.
-
Data Area - Lower frame or main area where the pages are displayed.
-
Menu Area - Left area in the browser window.
