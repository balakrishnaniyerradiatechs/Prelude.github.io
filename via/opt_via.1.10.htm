List Maintenance
Allows you to view, modify, or delete shopping lists.
•
List Name: The list name to create or edit.
•
List Defined By: If you are editing a list, you see how the list was selected. If you are creating a list, choose how to select the list:
-
Keyword - standard keyword search. Use connective words and/or sort product numbers ascending or descending.
-
Product - sort product numbers (ascending or descending) by product line, by vendor, or by exceptional items. If you use exceptional items, they appear on the web regardless of the Sold on Web Flag in Product Maintenance. For a web user to see these products, the list must be in the Web User or Industry Type Maintenance screen as Search List or on a Default Page/Catalog Page.
-
Select Statement - use the PROD.CAT file to find products in ascending or descending order.
•
Description: A description of the list.
•
F5-Details: The appropriate screen appears depending upon the type of list selected.
•
Action Bar
-
Copy: Enter the list name to copy and the name of the new list.
-
Rename: Enter the list name to rename and the new name.
