Action Bar - Customer List
This screen defines the list of valid customers that a web user can switch to (e.g., sales representatives that need to switch between customers to view and enter orders).
•
Enter List of Valid Customers to Access: Customers the user can access.
•
F5-Customer Selection: The sales rep and state to view customers.
•
F6-Sort by CN: Resort list by customer number.
•
F7-Clear Customers: Delete the list.
Budgets
•
Budget Supervisor: The user can approve orders that exceed their company budget.
•
Auto Generate Cart: The user’s shopping cart automatically becomes an order in nightly processing.
•
View Budget Info: Show budget amount and balance on web site.
•
Portal User: The user is a portal user (contact VIA support for further information).
•
Budget Period: The effective time period for budget: daily, weekly, monthly, quarterly, and yearly.
•
Budget Table Name: The name of the budget table to use if you do not use the defined periods.
•
Budget Action: How the user is notified if they are over budget: R Email & Require Authorization Warn User, E Email. Only Warn User, S Email Only But Don’t Warn User, W Warn Only No E-Mail.
•
Budget Cost Center: The budget cost center associated with this web user.
•
Budget Amount: The total budget for this user.
•
Notify Web User Id: The web user ID to receive email notification if budget is exceeded.
•
Notify Interval: How often the user is notified of budget excess: immediately or nightly.
•
DOE Portal Hook: Subroutine that determines if the product is a portal or normal product.
•
Current Budget Statistics (Display only)
-
Budget Amount: Amount of budget.
-
Budget Used: Amount of budget used to date.
-
Cart Total: Total current cart amount.
-
Budget Balance: Budget amount remaining.
-
Budget Over Highest: The highest dollar amount a web user can exceed their budget by.
-
Budget Reset Date: Date budget will be reset to initial amount.
