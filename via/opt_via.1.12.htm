Industry Type Maintenance
•
Industry Type: The industry type you want to create or edit.
•
Description: The type of industry.
•
Logo Gif: The path of the logo image to be displayed for the industry type.
•
Logo Destination: The destination path of the logo for the industry type.
•
Logo Overtext Resource Id: The Lang Res key to use to display text when the mouse hovers over the logo.
•
Specials List Name: The specials list name created for the industry type.
•
New Product List Name: The new product list name created for the industry type.
•
Clearance List: The clearance list created for the industry type.
•
Advert Banner: The path of the advertising banner to be displayed for the industry type.
•
Base Profile User: The default account with the settings to copy for new web users.
•
No Login User: The no login user for this industry type.
•
Classification Code: The industry type: Normal or Retail.
•
Verbiage Id: Enter guest user verbiage fields.
•
DBA Info Id: See Web Policies DBA Listener.
•
Doc Logo URL: Path to the logo on the printable view of the acknowledgement.
•
Scope List: Scope list for all users in this industry.
•
Checkout Disclaimer: Add information to the checkout screen for the website.
•
F5-Default Page: The web page that appears when a user connects to the site. The default page hierarchy is determined by web user, industry and company.
