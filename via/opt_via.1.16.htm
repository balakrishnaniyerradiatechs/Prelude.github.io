Utilities/Rebuilds
•
Cart Purge Routine: Delete shopping carts based on the date the cart was last updated. Any carts updated before this date are deleted.
•
Fix And Reset UserDocs: Marks all default pages as changed so the images are reloaded to the web site for the user’s next VIA session if default images are not automatically uploaded.
•
Verify/Update Security Options: Verifies menu options are set for each web user and does not change flags for existing menus.
•
Update Shopping List from Order Pad: Place all products a web user ordered in the Favorites List based on a customer range and last sale date. You can append to or override an existing list.
•
Build Product Image References: Updates products and images and/or product lines for the web site by copying the data from the image file to PRODUCT. Rebuild this data when expected images do not appear. Enter a beginning and ending product number(s) for the cross-reference build. Leave blank to rebuild all.
•
Shopping Cart View/Delete: Allows you to view or delete the contents of a user’s chopping cart. You can also convert the cart items into a sales order.
•
Portal Xref Rebuild: Rebuilds the cross reference file for third party sales orders. Please contact a VIA customer support representative for further information about this feature.
•
Budget Fiscal Period Maintenance: Define a fiscal period that overrides the budget period in Web User Maintenance. Works with the Budgeting Module.
•
Rebuild Customer Web User Xref: Rebuilds the web user customer cross-reference file, which tracks associates web users with customers. This rebuild affects pages such as Switch User and Admin Password.
•
Build Web Image Xref From Product File: Updates products and images for the web site by copying data from PRODUCT to the Web Image Xref file. If you update image information in PRODUCT, you must rebuild data to move that information to the Web Image Xref file.
•
Move Portal To Portal History: Moves portal (third party sales order) orders into a portal history file. Contact a VIA customer support representative for more information.
