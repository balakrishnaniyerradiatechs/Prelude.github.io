Maintenance
Product Maintenance
Menu path: Product Maintenance > * Action - Additional Options > Warehousing information (/PM > Action > Add Options > Warehousing)
•
Floored Item: Defines each product for segment classification. A floored item is one that is taken directly to the staging area. Non-floored items are products that are packed prior to truck loading.
•
Box Type: Defines the box type or classification for each product (e.g., Bill of Lading printing.
Ship Via Maintenance
Menu path: Order Processing > Maintenance > Customer Control File > Ship Via Codes (/SVM)
•
Truck Route: Y allows multiple pickers to pick a single truck route stop at the same time. The products for each stop are divided into picking segments with each picker processing a single segment.
Segment Processing Maintenance
Menu path: Warehousing > Segment Processing > Maintenance
•
Box Type Maintenance (/BTM): The box type is used in segment packing and can also be used for classification on the bill of lading. Once box types are setup, they can be added to each product.
•
Segment Ship Via Maintenance (/SSVM): This maintenance program is used to change the ship via of a segment record that has already been generated.
•
Ship Via Matrix Maintenance (/SSMM): This Maintenance program is used to maintain the Ship Via codes for a User Id in priority sequence. This is used in RF picking for Truck route Processing.
•
Ship Via Assignment Maintenance (/SVAM): This is used in radio frequency picking so that each user id can be assigned a ship via code. If the warehousing option, Picker Assignment Required is set to Y, the radio frequency unit automatically assigns the next available picking segment for the assigned ship via. If ship via assignment maintenance is not used, the picker can enter a specific ship via code and route number. The next available segment is assigned based on the criteria.
•
Driver’s Notes Maintenance (/DNM): This allows the truck driver to enter notes for a specified route and stop I.D. Example: check collected.
•
Segment Counter Range Maintenance (/CTR.RANGE.MNT.SEG#): This is used to set the counter for each warehouse when Sales Order Bin Allocation (Segment Generation) is processed. If segment history records exist on file, a warning message is displayed regarding the segment counter starting at the current range.
