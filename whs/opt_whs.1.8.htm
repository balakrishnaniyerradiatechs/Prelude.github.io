Entering, Picking, and Shipping Sales Orders
Radio Frequency - Sales Order Entry
Path: Warehousing > Handheld RF Menu > Processes > Sales Order Entry (/RFSOE)
Radio Frequency scanning devices (RF guns) can be used to enter sales orders in the warehousing module. A regular sales order is created behind the scenes when the RF Sales Order Entry screen is updated, using the Direct Order Entry conversion process. No asterisk items may be entered or scanned using this screen. The sales order is created for the items at a central location.
•
Order#: Press ENTER to get the next available order number. To access an order that was previously entered and suspended, enter the sales order number to be updated.
•
Customer #: Enter the customers number, or press F3 to search by customer name.
•
Ord Type: Enter the order type of the sales order. Press F3 to see order types.
•
Whse#: Defaults to the warehouse number assigned to your user id, or the one entered in the prefix screen.
•
Ship to#: Defaults to D if no ship-to records exist for the customer.
•
WB#: Displays your initials.
•
Terms: Defaults to the terms defined for the customer in Customer Maintenance.
•
Req Date: Enter the requested date of the sales order. The system date is the default.
•
Cust Po#: Enter the customers purchase order number.
•
Route: Enter the truck route code for this sales order. Route codes are in Route Code Maintenance.
Once the header screen information is entered, the detail screen displays:
•
Order#: The order number entered in the header displays for reference.
•
Product#: Displays the scanned product number. The description displays below the product number. If a product is entered on an order twice, the system warns you that the product already exists on the order. No asterisk items may be entered or scanned using the RF Sales Order Entry screen.
•
Ln#: The line number on the order is updated as each item is scanned.
•
Qty: Enter the quantity of the product number to be sold.
•
Ok?:
-
Y - If the product number and quantity entered are correct. The next item appears.
-
N - If changes need to be made to the product number or quantity fields.
•
Price: Displays the price of the product number entered.
•
Ord Tot: Displays the total dollar amount of the order. Each item updates this amount.
•
F5-Header: Changes can be made to the header screen.
•
F2-Save: Completes the order and brings up the following options screen:
•
U: Updates the order and creates a regular sales order for the customer with the next available order number using the DOE conversion process. The RF sales order is deleted at this time and cannot be viewed.
•
S: Suspends the order and allows you to exit without finishing the order. The order can be accessed in RF Sales Order Entry to make changes before updating.
•
D: Deletes the RF sales order. No updating has taken place since a regular sales order has not been created yet. There is no reference to deleted RF sales orders since these orders are deleted once the regular sales orders are created.
