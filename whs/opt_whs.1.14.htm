Pick Confirmation – by Range or by Release
Path: Warehousing >Sales Order processing > Pick Confirmation by Range (/PLRC)
Path: Warehousing > Sales Order processing > Pick Confirmation by Release (/PKCE)
If pick tickets are used instead of a radio frequency terminal and queue, then pick confirmation is used to move the product from the pick bin to a staging bin awaiting shipment. Pick confirmations may be done by a range of label numbers or by a single release number. The Prompt for Dest Bin in PKCE flag in Warehouse Maintenance must be set to Y to allow for the Destination Bin to be changed to STAGE. This new bin will be used for moving bin inventory to the bin commitments and loaded on the release line.
