Batch Pick Ticket Processing
Path: Warehousing > Sales Order Processing > Batch Processing (Wrist Picking) > Batch Pick Ticket Processing (/BPTP)
After sales orders are entered, Batch Pick Ticket Processing selects pick tickets to print.
Enter the selection criteria, then either use the F5- Auto Select key to let the system automatically select pick tickets depending on the criteria entered, or the F6 - Manual Select key to manually enter sales orders for pick ticket release. Press F7 - Report to print the Sales Order Staging report to preview the sales orders before printing. Press F8 - Release to release the selected pick tickets to print.
•
Order Staging Batch: Enter NEW to assign the next available batch number.
•
Warehouse: Enter the shipping warehouse number for the sales orders entered.
•
Customer: Enter the customer number to limit the batch pick ticket processing selection to a customer, or leave blank to include all customers.
•
Required Date: Enter the required date for the sales orders to be released.
•
Maximum # of Orders: Enter the maximum number of orders to be picked.
•
Minimum # Order Lines: Enter the minimum number of sales order lines to include.
•
Maximum # Order Lines: Enter the maximum number of sales order lines to include.
•
Report Form Queue: Enter the printer form queue where the Sales Order Staging Report should be printed when the F7-Report option is selected.
•
Printer Form Queue: Enter the printer form queue where the pick tickets should be printed when the F8-Release option is selected to generate the sales order releases.
•
Number of Orders: The total number of sales orders selected from the auto or manual selection is displayed.
•
Number Lines: The total number of lines selected appears.
•
Header Built:
-
X - Indicates the header was built by pressing F5-Auto select or F6 - Manual Select.
-
O - The header has not been built for the batch pick ticket process.
•
Auto Selection:
-
X - Indicates the F5-Auto Select option was used.
-
O - Indicates the F5-Auto Select option has not been used.
•
Manual Selection:
-
X - Indicates the F6-Manual Select option was used.
-
O - Indicates the F6-Manual Select option has not been used.
•
F2-Save: Saves all information on the Batch Pick Ticket Processing screen for the order staging batch number. This can be used to return to the batch later.
•
F4-Del: Deletes the order staging batch number.
•
F5-Auto Select: Selects the sales orders available for pick ticket release depending on the criteria entered.
•
F6-Manual Select: Manually enter sales order numbers for pick ticket release.
•
F7-Report: Prints, emails, or faxes the Sales Order Staging Report. (Optional).
•
F8-Release: Releases pick tickets for the selected sales orders.
