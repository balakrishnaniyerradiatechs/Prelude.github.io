RF Order Batch History Inquiry
Path: Warehousing > Sales Order Processing > Batch Processing (Wrist Picking) > RF Order Batch History Inquiry (/ROBHI)
Displays closed batches from the Staging Order Batch purge process.
•
Batch #: Enter the batch number to review.
•
Batch Status: Displays the status of the batch:
-
A – Active (active in order picking).
-
S – Suspend (batch is suspended).
-
C – Completed (Wrist RF Order Picking was performed).
•
Picker: Displays the Id of the picker id assigned to the batch number.
•
Emergency: Displays a Y if the batch was an emergency.
•
Release Number / Customer: Displays the sales order release numbers and associated customers included in the batch.
•
F2-Finish: Clears the current batch number displayed on the screen.
•
F5-Review Order: Displays Order History Inquiry for the highlighted sales order.
•
F7-Release Inq. Displays the Release Confirmation Inquiry for the sales order.
•
F8-Log: Displays the Order Log for the highlighted sales order.
