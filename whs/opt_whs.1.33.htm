Maintenance
Warehouse Maintenance
Maintenance > Inventory Controls > Warehouse Maintenance (/WM) > F7 > F5 > F8
•
Allow New Products in RF SRE: Indicates if you can receive products that are not on the P/O:
-
Y - You can receive products that are not on the P/O.
-
N - You cannot receive products that are not on the P/O.
•
Display Count of Open B/Os in RFSRE: Choose whether backorders appear in RF stock receipts entry:
-
Y - Once a product number is entered, any backordered amount displays.
-
N - Backorder does not appear.
•
Action > Additional Options > Bin Locator Information > Choose default bins: Choose a default bin for:
-
Receiving (non-inspection) Dock Bin.
-
Linkage Receipt Bin.
-
Inspection Bin.
-
Return Bin.
-
WO Completion Bin.
•
Bar Code Form#: Enter the barcode form queue where the barcode labels will print.
Bin Location Maintenance
Inventory > Maintenance > Bin Files > Bin Location (/BLM)
•
Bin Type: Indicate the type of staging bin: D - Receiving Dock; I - Inspection.
Product Maintenance
Inventory > Maintenance > Product Maintenance > Product Maintenance > F5 > F7
•
Product label size: Indicates how big the label is: L - Large; S - Small.
•
Qty/product label: How many barcode label should print for this product at stock receipts time.
•
F6 > Alternates: If a vendor barcodes products, that barcode number is an alternate for the product.
•
F8 > Bin Defaults : Choose default bins:
-
Primary Bin - The permanent pick bin for a product.
-
Secondary Bin - For information only when a product has more than one bin.
-
Primary Case Bin - Third bin for this product (e.g., bin allocation, alternative picking).
-
Primary Case Qty - If using Primary Case Bins, the case quantity.
Additional Option Parameters > F6-ICS)
•
Update Primary Bin When Incr Onhand: Determines what is updated when onhand increases for a product without a primary bin:
-
Y - Update the primary bin in the product/warehouse for the received product.
-
N - Do not update the primary bin in the product/warehouse for the received product.
User Id Maintenance (/USER.ID.MNT > Action > User Options > Warehousing)
•
RF Primary Bin Update: Determines what is updated for a product without one in RF Putaway Entry:
-
Y - Update Primary Bin Number in the product warehouse record.
-
N - Do not update the primary bin in the product warehouse record.
