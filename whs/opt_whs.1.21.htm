Sales Order Bin Allocation
Path: Warehousing >Sales Order processing > Sales Order Bin Allocation (/SOBA)
The bin level allocation must be run to determine the correct picking bin for the radio frequency units. The picking segments are created after all the sales orders have been bin level allocated. A segment consists of the products being picked from multiple sales orders for a single customer.
The Sales Order Route Inquiry (/SORI) can be used to view the status of a proposed route before doing the bin level allocation. Displays the total number of orders, line items, and weight.
