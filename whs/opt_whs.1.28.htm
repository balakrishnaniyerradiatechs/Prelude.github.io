Shipping Box Confirmation Entry
Path: Warehousing > Sales Order Processing > Batch Processing (Wrist Picking) > Shipping Box Confirmation Entry (/SBCE)
This process confirms that everything for the specified serial code is in the shipping box. This is the last step in the wrist picking process before invoicing.
•
Serial Ship Code: Enter the box number created at Wrist RF Order Picking.
•
Release #: The sales order release number associated with the serial ship code (box number) is displayed.
•
Continue: Select Y to continue with shipping box confirmation for the sales order.
The Order Log Inquiry is updated with the SBC code for the sales order. The Sales orders can now be invoiced.
