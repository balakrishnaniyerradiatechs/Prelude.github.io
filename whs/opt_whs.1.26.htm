RF Order Batch Queue Inquiry
Path: Warehousing > Sales Order Processing > Batch Processing (Wrist Picking) > RF Order Batch Queue Inquiry (/ROBQI)
Displays the open order staging batches for a warehouse.
•
Warehouse Number: The warehouse number to display the order staging batches.
•
Batch#: Open batch numbers are displayed. R for released.
•
Emg: Enter Y to update the batch number as an emergency batch.
•
Picker: the picker id for the batch, which Prelude updates in Wrist RF Order Picking.
•
Status: Displays the status of the batch:
-
A – Active (currently active in order picking).
-
S – Suspend (batch is suspended).
-
C – Completed (Wrist RF Order Picking was performed).
•
Date: Displays the date the batch was created in Batch Pick Ticket Processing.
•
#Order: the number of open sales orders included in the batch based on the criteria.
•
Batch#Ln: the total number of all lines included in the batch.
•
Pckd#Ln: the total number of all lines picked in the batch.
•
Asg By: the picker id assigned to the batch. If the picker id was manually assigned by a manager, the managers user id is the picker id. If Prelude assigned the picker during Wrist RF Order Picking, an asterisk ‘*’ appears.
•
F2-Save: Saves changes.
•
F5-Review: Individual sales orders included in the batch can be reviewed.
•
F6-Suspend: Displays only the batch numbers that were suspended (Status = S).
•
F7-Emg Only: Displays only the batch numbers that have the Emergency flag set to Y.
•
F8-View All: Displays all batch numbers available for the warehouse number entered.
