Wrist RF Order Picking
Path: Warehousing > Wrist RF Menu > Order Picking Ent (/WROPE)
Once pick tickets were printed, this process performs Sales Order Picking and Serial Shipping Container Code Entry on your wrist device for a batch number.
•
Warehouse: Enter the warehouse number for displaying order batch records.
•
Picker: Enter the picker id to display the available batch for sales order picking. Prelude determines the batch for the picker, who can only work on one batch at a time.
•
Batch: the batch number for the picker id and warehouse number combination.
•
# of Orders: the total number of sales orders being picked for the batch number.
•
Tot/Pk: the total number of sales order lines and the total number of lines picked for the batch.
•
Continue: Enter Y to continue with sales order picking for the batch number. This following screen appears:
•
Mv#: The bin movement number is created for sales order picking.
•
Z: Displays the zone of the bin location.
•
TQ: Displays the total quantity of the product to be picked for this batch.
•
B: Displays the bin location for the product being picked.
•
Product Number: The product number must be scanned for confirmation.
•
Qty: Displays the quantity of the product being picked for a single order.
•
Box: Enter the box number for the product being picked. This requires a scan confirmation to be sure the product is going in the correct box. This number is the serial ship container code for the sales order.
•
F2-Save: Processes the sales order picking entry. If the customer is setup to print serial ship container code labels, the labels are printed.A summary of sales orders picked for the batch appears.
•
F5-PBI: Displays the Product Bin Inquiry screen for the wrist device.
•
F6-BOHI: Displays the Bin Onhand Inquiry screen for the wrist device.
•
F7-Skp: Skips the current line being picked and moves to the next line.
•
F8-Hdr: Displays the Wrist RF Order header screen for the batch.
•
F9 - Suspend: Suspends the batch. Allows the picker to continue later or to have the batch assigned to another picker.
Once the Wrist RF Order Picking process was performed, the picked sales orders are updated with the RFP (Radio Frequency Picking) and & SSC (Serial Shipping Container) codes in the Order Log Inquiry screen.
