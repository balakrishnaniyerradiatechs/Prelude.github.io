User Id Security Setup
A radio frequency menu and a separate user I.D. must be created for radio frequency users. This menu is entered in the System Builder User ID so that RF users only get the programs that are formatted for the radio frequency device screen size when they log on. A second User Id is required for users that also perform terminal transactions. The radio frequency User Id is generally setup for a single warehouse. This User Id must be setup to display error messages as a dialog box and also have the print selection screen disabled.
