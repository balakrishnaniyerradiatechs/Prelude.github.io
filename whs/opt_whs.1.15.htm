Radio Frequency Confirmation Packing
Path: Warehousing > Handheld RF Menu > Doc Picking > Release Cfm Packing (/RRCE)
This confirms a release and assembles all of the products for a release if the products were picked by zone. This process is not necessary if all products are picked by release.
The location of the products appears so that the packer can locate the material in sections of the warehouse. Changes can be made if necessary. After the entire release is confirmed, the products are moved to the INV bin awaiting invoicing.
