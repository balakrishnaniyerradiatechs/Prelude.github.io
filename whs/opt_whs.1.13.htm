Radio Frequency Order Picking
Path: Warehousing > Handheld RF Menu > Doc Picking > Order Picking (/ROPE)
•
Picker Id.: Enter the picker id for the person who will be picking the following orders.
•
Release number: If the warehouse option Require Picker Assignment is set to Y, then a release number must be entered and it must were assigned to this picker. Press F3 to search for releases assigned to this picker. The releases are sorted by the required date. In V18+, multiple releases can be picked in one session. The first five releases are displayed on the screen, but additional releases can be entered.
•
# of lines: If the picker has not entered or selected a release, then the system selects lines to pick based upon the entered criteria. In this situation the picker is gang picking lines regardless of what release they are on, and Radio Frequency Release Confirmation Packing is used to assemble the products by release.
•
Zone: If appropriate, enter a picking zone. Defaults to the RF order picking zone if one exists for the user id.
The lines are selected from the data created during bin allocation, and sorted by bin sort sequence. Each product appears with the description, from bin, and quantity. If Validate Radio Frequency in Warehouse Maintenance is set to Y, then the bin number and the product number must be scanned. If not, then the return key can be used to accept the displayed information. The To bin is prompted for which represents the pickers cart. At that time a bin move is done from the From bin to the cart. After all of the lines selected by the picker were picked, enter a destination bin, which represents where the material is held awaiting shipment. If a partial quantity of a line item is picked, the balance of the pick quantity for that product appears after the first pick. The balance of the product is required for until the entire quantity was picked or a 0 pick quantity is entered.
In V18+, this process was updated to check Auto RCE After RF Picking in Warehouse Maintenance F7 > F5. RCE is performed for complely and partially picked lines.
