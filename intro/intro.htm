Using the Activant Prelude Help System
The frameset view divides the browser window into three areas: an area for navigational controls, an area for toolbar buttons, and an area for Help topics. Activant Prelude help has three areas:
•
Topic Frame - The Topic frame displays the help topic you selected from the Contents, Search, or Index tab. You are reading this help topic in the Topic frame.
•
Navigation Frame - The Navigation frame, which is to the left of the Topic frame, provides three tabs:
-
Contents - show the Table of Contents. Simply click on the item you want to see and the help topic appears in the Topic frame.
-
Index Tab - alphabetical list of keywords associated with topics. To view index entries, select a letter group to display the entries for that group. Then just click an index entry to view the information in the Topic frame.
-
Search - searches all text in the help system (not just keywords). The drop down list below the Search box allows you to narrow your search to a set of documentation or a module within a set of documentation. Simply enter text and click Go to see matching help topics and each topic’s relevancy ranking that indicates how well the item matches your search criteria.
•
Toolbar Frame - The Toolbar is just above the Topic frame and provides buttons to help you navigate the help:
-
Show in Contents (circle arrows): highlight the current topic in the Contents tab.
-
Previous (left arrow): display the topic that precedes the current topic.
-
Next (right arrow): display the topic that follows the current topic.
-
Related Topics (page with down arrow): lists topics related to the current topic.
-
Print (printer): print the currently displayed topic.
-
E-mail (envelope): e-mail feedback on the documentation using your email program.
