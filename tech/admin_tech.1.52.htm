Macros - /UK
A macro is a string of often-used characters that may be associated and activated by a single keystroke. A macro is to each user and may be defined by typing /UK.
Enter a Switch and a Macro String. The switch is the character that, when used with the User Key, will invoke the macro string characters. The Macro String is the string of characters to be invoked when the macro is called.
You must precede process calls with ‘\’. The input may be stacked using \DATA followed by the desired characters to be used as input separated by commas.
In the example above, switch ‘S’ will start a new sales order for location 10 and take you up to the customer number prompt. First we execute /SOE. The ‘10’ after the DATA will be used for the Location prompt. The two nulls after that are for the Department and Cost Center (they must exist even though departments and cost centers are not used). The ‘NEW’ is for a new order number.
To execute, press the User Key followed by the desired switch.
