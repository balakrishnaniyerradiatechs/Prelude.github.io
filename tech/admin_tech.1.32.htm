Taking An Existing Report To Excel
In addition to creating your own downloads by using SBClient’s Smart Query, you can also download existing reports to a PC spreadsheet. Although may seem confusing and require more work, the process gets easie. In our example we will take an Accounts Receivable Aging Report and download it to Microsoft Excel. Once you have filled out the prompts in the A/R Aging Report screen and you begin to run the report you should get a prompt back asking you which printer you want the report to go to.
Note: If you do not get this prompt, a your user security must be changes.
1
Set Assign Options to “HS” to hold the job in the spooler instead of printing. Once this screen is saved you should notice a message on the screen telling you which job number this report will be. (Ex: Hold Entry # 50)
2
Once you know the job number then go to TCL and type: SP-EDIT XX (C where XX is the hold entry job number. In this next step we will take the hold job in the spooler and convert it to a PICK file, you should see:
Entry # 50 Display (y/n/s/d/x)? - Answer N here
String:- - Press Return Here
Spool (y/n=cr/t/tn/f)?- - Answer F, which is for file
File Name? - - Enter a file name for the hold job (e.g., AR) or use “FIX”. Item id
3
Next we will use SBClient Host To PC Transfer to download the FIX AR record to a PC File. The best format is ASCII, though you may experiment with other formats to see if you get better results.
Note: If the ASCII format is used you do not need to specify any display fields.
The hold jobs are kept in a file called _HOLD_ which we can directly access. When you print a report the system should come back and give you the item id of the hold job which looks something like: Hold Entry _HOLD_/ACT50_1234 where ACT is the user id of the person running the report. Then in the Host To PC Data Transfer screen you can put _HOLD_ in the host file and dictionary fields and “ACT50_1234” in the Select Records field using the double quotes before and after the item id.
Now that the file is downloaded to the PC, you can go to your spreadsheet application and open the PC file. When you try to open the file, Excel’s Import Wizard guides you through setting up the appropriate columns.
In the Import Wizard choose Fixed Width for the Data Type and then be sure to set up your columns appropriately. When you are done you should have a spreadsheet that looks something like this:
