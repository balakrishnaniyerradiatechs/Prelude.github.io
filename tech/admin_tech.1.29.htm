Network Setup
Many of our clients use a Windows network with SBClient to access their host computer. To set up SBClient to use your Windows:
1
Specify the network type and IP address under Setup â€“ Communication for a network type SBClient session. The most common network type will be Windows Sockets (SB Telnet).
Note: For SBClient to work properly with the network, the network connection to the host computer must be setup properly and your PC must be able to access the network. To test this, go to MS-DOS and ping the host computer (Ex: PING 192.9.200.7). If the ping is successful, SBClient should work properly with the network.
2
Specify the terminal type of Wyse 60 and make sure the SB+ Mode box is checked.
