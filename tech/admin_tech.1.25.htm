Configuring the Session
If you want to use GUI, you must change one setting for each SB Client session. This setting greatly increases the speed of the system. To configure the speed setting:
1
Double-click on the SB Client icon . The SBClient screen appears.
2
Highlight the session you want to change.
3
Choose Setup > GUI Parameters. The GUI Parameters Setup screen appears.
4
Check Pad TCP Packets and set it to 2048.
 
 
You may increase this number to see if it improves speed.
5
Click OK.
6
On the SBClient screen, choose File > Save Configuration.
