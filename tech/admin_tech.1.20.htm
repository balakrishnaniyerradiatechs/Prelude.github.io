Btree (Indexes)
A ‘B-tree’ (balanced tree) is essentially a system-maintained ordered-list based on any attribute or combination of attributes from items in a file. The unique aspect of a ‘b-tree’ is the database itself maintains this secondary key file based on any update to the file. Whenever the values in a attribute that is used in a ‘b-tree’ is updated - regardless if by a basic program, editor, copy, or file restore from tape - the database will maintain this low level file. The primary use of a ‘b-tree’ is for extremely fast retrieval of records that match the desired secondary key.
