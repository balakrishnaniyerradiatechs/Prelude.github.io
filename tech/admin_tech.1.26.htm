Defining GUI Terminal Types and Preferences
Each user that uses GUI must have terminal types and GUI preferences defined correctly. To define GUI terminal types and preferences:
1
In SBClient, choose Admin > Security > User Security Setup.
2
Enter your password. The User Security Setup screen appears.
3
Enter the User Id Code for the user (e.g., MKM).
4
Press F7-Term/Prnt. The Terminal Ids screen appears.
5
For Terminal ID, choose one of the following:
•
To make GUI optional for this user, enter TU.WYSE60.GUI.
•
To make GUI required for this user, enter TU.WYSE60.GUI and delete TU.WYSE60.
6
Press F2-Accept.
7
Press F8-GUI Prefs. The GUI User Preferences screen appears.
8
Set the preferences to match the following screen.
9
Press F2-Accept and F2-Save.
