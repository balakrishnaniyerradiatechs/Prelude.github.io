Faxing Documents with VSI-Fax
Prelude works in conjunction VSI-FAX, a third party software that enables you to fax documents to vendors and customers. The VSI-FAX software resides on the UNIX box and requires a dedicated modem and port to send faxes. Several maintenance flags must be setup to customize faxes and the faxing process.
If the VSI-FAX server is down when a document (sales order, quote, purchase order, etc.) is faxed, you are notified of the error by email or screen prompt . Jobs that were submitted are held until the server is back up. Also, the system administrator is emailed if VSI-FAX must be restarted, or if one of the modems is down.
User Identificaton Maintenance (/USER.ID.MNT)
Path: Systems >System Parameter > Maintenance > User Identification Maintenance
E-mail notifications are sent to individual users on the results of faxes. to receive fax notifications, the e-mail field in User Identification Maintenance must be filled in. Each buyer in Buyer Maintenance must be established with a user ID.
•
User I.D. code: Enter the User ID of the employee using the VSI-FAX.
•
E-mail Address: Enter the email address where the fax notifications will be sent.
Company Maintenance (COMPANY.MNT > F6)
Path: Systems >System Parameter > Maintenance > Company Maintenance > F6-Modules
•
Fax Integration: Select V to use VSI-FAX for faxing purposes. This brings up the following screen:
•
VSI-FAX Path: Enter the UNIX path to VSI-FAX in this field. The default UNIX path is \/usr/vsifax3/bin/vfx.
•
Coverpage Path: Enter the name of the cover page to use for all faxes for the company. All cover pages for VSI-FAX reside in a directory therefore it is only necessary to specify the name of the cover page, and not the path. This field is not mandatory and if left blank the default cover page is used. to pass the company and warehouse information to VSI-FAX, the cover page tag must be entered in this field.
•
Email Notification Address: Enter the System Administrator's e-mail address. This address is set up for the purpose of receiving any VSI-FAX error or failure messages, as well as any e-mail notifications of faxes sent by a user without an e-mail address setup in User Identification Maintenance. VSI-FAX also includes e-mail confirmations for both successfully and unsuccessfully sent faxes.
•
Document Logo and Document Overlay: These fields are informational and only used for reference.
•
Email Type: Determines when fax notifications should be sent to the user via email:
-
B - Always notify whether the fax was successful or unsuccessful.
-
O - Notify only for successful faxes.
-
F - Notify only for failed faxes.
-
E - Notify on each attempt.
-
N - Never notify and never send an email notification.
•
Whse Info for Coverpage: Determines which records containing information such as Company Name, Address and phone numbers are extracted from.
-
N - The information for the cover page is pulled from the data in Company Maintenance.
-
Y - The data is pulled from the Warehouse Maintenance file. to use this option, the default coverpage tag must be selected in the Coverpage Path field. If you select not to use this feature, the default data in the VSI-FAX setup is used.
Note: VSI-FAX attempts to send out a fax 5 times before it is considered failed or unsuccessful.
Note: In versions 16 and higher, a program change was made to remove the F5-Multi Fax Queues option from VSI-Fax Parameters screen. This feature does not have logic to support standard software.
Fax Control
Path: Systems >System Parameter > Maintenance > Fax Control
To customize your faxing process for each warehouse and fax type, you can set flags for each type of document that will be faxed. The information in each of these programs is very similar to the information setup in company maintenance, but each one is for faxed documents for a warehouse. The information in the Fax Control screen overrides the information in Company Maintenance. Prelude fax types are:
•
Purchase Order
•
Invoice
•
Quote
•
Credit Application
•
Vendor Claim
•
Debit Memo
•
Return Goods Authorization
•
Statement
•
Sales Order Acknowledgement
•
Vendor Request For Quote
•
Credit Collections
•
Fax Vendor Expedite
As each of the fax types listed above is selected, the Fax Control Record Maintenance screen displays. Enter the warehouse number you would like to setup, press ENTER, then select F5-VSI-FAX Options to display the following VSI-FAX Whse Fax Control screen. Enter the information that applies to this warehouse and fax type.
•
Fax Notification Address: All e-mail notifications for a designated fax type for the assigned warehouse can be sent to a user. If this field is left blank, the notification is sent back to the user sending the fax.
•
Document Logo and Document Overlay: These fields are informational only and used for reference.
•
Fax Destination: VSI-FAX allows for the setup and addition of as many classes of fax modems as needed. Each fax modem contains a fax destination (Ex: fax1, fax2). This field allows you to control the destination of a fax (more than 1 modem for VSI-FAX is required to set this flag). For example, if multiple modems are setup, you can send all quotes to be faxed through one modem, allowing another modem available to send all other faxes. The default class is outgoing.
•
Fax Coverpage: This can be used to enter a cover page for a warehouse and fax type. This also follows the same format as Company Maintenance. If this field is left blank the cover page in Company Maintenance is used.
•
Suppress Coverpage: This field allows you to suppress the cover page for a warehouse and fax type.
•
Overnight/Immediate:
-
I - To fax this document type immediately.
-
O - To fax this document overnight. In this case the following field must be filled in.
•
Overnight Time: If the previous flag is set to O, enter the Military time for the fax to be sent. (This can be overridden at the time the fax is entered).
•
F5-Memo Message: Up to 8 lines (48 characters long) of text can be entered to display in the Memo field on the cover page. For example; to request that vendors reply to this P/O in 24 hours to confirm it was received, place the request in the Memo Message screen, and the message automatically shows up on the cover page for all P/Os for this warehouse.
Note: If a VSI-FAX error occurs while sending a fax, you will receive an e-mail notifying you of a failure with the VSI-FAX scheduler. This is different than the e-mail notification of a fax. If you are currently using more than one fax modem, it is possible the fax was completed successfully despite the fact an error message was sent. This error indicates that there is an error on at least one of the fax modems, but not necessarily the one being used to send out the fax. At this point vfxstat must be executed in UNIX to identify the problem.
VSIFax Log (/VP)
Path: Order Processing > Reports > Miscellaneous Reports > VSIFAX Log
This process includes information on VSI-FAX transmissions that were performed or are currently in process.
•
All: Displays all fax jobs being processed and sent in the system.
•
Expired: Displays only fax jobs that did not go through and are expired. For example, if the maximum number of attempts was reached and the fax sent was unsuccessful, it would be included when viewing expired jobs.
•
F2-Exit: Closes the VSI-FAX screen and goes back to the menu.
•
F4-Cancel Job: Cancels the VSI-FAX job. An email will be sent to the user ids email address to notify them that the job was cancelled.
•
F5-Examine Job: Displays the Fax Inquiry screen showing all information for a fax transmission.
•
F6-Retry Job: Prompts for a fax number to be entered. The original fax number is defaulted, but can be changed. This function key is only available for use with expired jobs.
•
F7-Refresh: Updates the VSI-FAX log screen with additional faxes submitted to the queue since accessing the inquiry screen.
•
F8-Modem Status: Displays the status of the fax modems.
