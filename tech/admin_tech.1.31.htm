Smart Query
Allows you to download data from the host computer to your PC similar to the Host to PC download except data can be directed to the screen, printer or sent to a PC file in Word, Excel or Lotus 1-2-3.
•
Dict File, Data File, Select, Sort By: Refer to the Host to PC download
•
Disp Fields: These are the fields in the file that you wish to download to Excel. You cannot download an entire file so you must specify which data elements you want to see.
•
Heading: Creates a header line in the data download.
•
Footing: Creates a footer line in the data download.
•
Options:
-
Id Supp: Suppresses the output of record ids.
-
Detail Supp: Suppresses detail output when used with total or break fields specified.
-
Header Supp: Suppresses the creation of the header line.
-
Col Header Supp: Suppresses the automatic creation of column headings.
Output
•
Terminal: Displays the output on your screen.
•
Printer: Sends the output to the default windows printer.
•
Excel/Word/Lotus: Sends the output directly to these programs. If they are not open, SBClient will open these applications and then download the data into a new spreadsheet. Note: Before SBClient will download correctly into the Excel/Word/Lotus programs a pointer record must be setup on the host computer that contains your PC’s path to these programs. The name of the file is TUSTATE and the records are EXCEL, 123W and WINWORD. These records should only have 1 attribute with your PC’s full path to that program in the attribute. You can setup 1 default path for all users in these files and then setup paths for any individual users that have different paths by editing EXCEL.XX, WINWORD.XX, 123W.XX where XX is the port of that user. A sample Excel path would be: C:\MSOFFICE\EXCEL\EXCEL.EXE
