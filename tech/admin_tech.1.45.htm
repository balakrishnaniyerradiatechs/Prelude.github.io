Optio
Optio is a third party software package that works with Prelude to enable your company to fax and email documents to vendors and customers. Maintenance flags must be set to activate the Optio functions.
Company Maintenance (/COMPANY.MNT > F6)
Path: Systems >System Parameter > Maintenance > Company Maintenance > F6-Modules
•
Fax Integration: Set to O to use Optio for faxing documents.
Warehouse Maintenance (/WM > F6)
Path: Systems >System Parameter > Maintenance > Inventory Controls > Warehouse Maintenance > F6-Cont Setup
If using continuous printing, the form queue printer entered in the Continuous Printing Setup screen must be established as a valid Optio printer in Optio Printer Maintenance.
Form Heading Control Flags (/FM)
Path: M:USER-FORMS > Form Heading Control Flags
•
Heading: Enter O for each document that Optio will be printing using laser output.
•
OptioFax: Enter Y if the document will be faxed using Optiofax.
•
F5-Additional Forms: Additional forms using Optio are in this screen.
Optio Printer Maintenance (/OPTIO.MNT)
Path: M:USER-FORMS > Optio Printer Maintenance
To connect Optio to Prelude, at least one physical printer must be specified as an Optio printer. This is done slightly different depending on whether the system is Pick or Unidata.
•
Pick Systems: The OptioQue needs to be a Pick printer setup to spool to the Optio virtual printer in unix. The FormQue is the Pick Printer that Optio will print to, and the Printer Device Name is the unix printer that this Pick printer spools to.
•
Unidata Systems: For Unidata the setup is more straightforward. The OptioQue needs to be the name of the Optio virtual printer in unix. The FormQue is just any arbitrary number that will print to the unix printer in the Printer Device Name when specified.
In both cases, to print a job through Optio, the printer name in the FormQue needs to be specified as the printer destination. This screen allows Prelude to reroute the job to the Optio virtual printer, then send the final output to the printer specified in the Printer Device Name.
•
F7- Print: Prints a listing of printer queues. (Versions 18 and higher)
