Troubleshooting RF guns with DOS Operating Systems
To configure the RF gun’s network settings to connect to your access point (AP):
1
With the gun off, press the following keys at the same time "A", "B" and "D", and while holding them down, press the PWR/ON button, release the PWR/On button then release "A", "B" and "D”. This is a cold boot.
2
Allow the gun to load files until a screen comes up with the following:
For a PDT6846 RF gun you see:
Using 11 Mbps
Flash Terminal
LWP 11 Hex 5.00-38

For PDT6840, PDT6842 and PDT 3540 series RF guns you see:
Using 802.11
FLASH TERMINAL
LWP Hex V 4.02-07
RF V1.07
3
Once the gun continues to the next screen press control c until you are prompted “Halt Batch Process (Y/N)?” Choose y for yes, which will bring you to D:\> where you would type in cfg11 for PDT6846 units or cfg24 for PDT6840, PDT6842, and PDT3540 units then press Enter.
4
Use the arrow keys to guide you through the menu which looks like the following: 
CONFIGURATOR 2.18 
View config 
ESS Id 
Boot Mode 
Perf Index  
Tx Rate  
Intl Roaming  
Antenna  
MU Sleep Mode 
Buffer Count 
ReXmit Delay 
Encryption 
Exit
View config will display the RF gun’s hardware address, IP, Subnet Mask, Router, ESS ID, whether or not the gun is associated with an access point, and your current encryption status. Use the CLR key to go back to the main menu. If you want to set up how the gun obtains an IP highlight BOOT MODE and press enter. This will bring you to another screen. Highlight Set Mode and press Enter. Use the arrow keys to go through the different options. (EX. DHCP) If you choose DHCP use the CLR key to escape out back to the main menu. If you choose manual press the CLR key once to the previous menu and enter the MU IP, Subnet Mask, and Default Router assigned to the gun. Use the BKSP key to clear any previous entries. Once you have enter the information press the CLR key once to get you back to the main menu.
5
Next you need to make sure the ESS ID is set to your AP’s ESS ID. Highlight ESS ID and press Enter. Use the BKSP key to erase the current ID if incorrect and enter the correct one. Press CLR twice and you save your new setting while exiting the configuration screen. Once you are back at the D:\> power off the gun and perform a cold boot (See Step 1).
To change the host IP in Wavelink on the gun:
1
Power up the gun and wait for all the files to load. You should end up at a screen with the following: 
No host connection 
<Enter> to connect
Note: If you do not see this then you have not connected to your AP and need to follow the steps for configuring your RF gun to connect to your AP.
2
Press FUNC then CTL then r in that order. You will be prompted for a password. The default password is system. Press y when prompted “Warning: Any Active session will be terminated.”, and you will see a menu with different options. Press 2 to configure the host profile. Highlight the host profile you want to configure and press enter. You can change the Host IP Address and Host Profile Name. Then press CLR to exit and save. Press 3 to return to login.

Symbol RF Setup
When you obtain Symbol RF guns to be used with the Prelude Warehousing Module, the following items must be setup to enable each gun to function:
•
An IP address must be established.
•
An ESS identification must be assigned.
•
A host configuration must be setup.
A process called cfg24 is used to set up the IP address and the ESS ID. Then the host configuration screen can be set up. The following steps will guide you through the proper procedures.
1
Cold boot the gun by turning the power off.
2
Hold down the A+B+D keys at the same time while turning the power back on.
3
Release the A+B+D keys. As the screen begins to scroll with information, look for:
-
Symbol Technologies
-
Copyright 1995-1998
-
BOOTP Ver 2.xx
-
Connecting…
4
When the above information appears, press and release the CTL key followed by BKSP.
5
The gun prompts if you want to halt the batch process. Answer Yes and you are taken to the D: prompt.
6
Enter cfg24 to access the configuration menu.
7
Arrow down to the prompt for ESS Id and press ENTER.
8
Set the ESS Id to uppercase PSI.
9
Press ENTER to get back to the main menu.
10
Arrow down to the MU IP Address option and press ENTER.
11
Enter the IP address for the gun then press ENTER to return to the main menu.
12
Arrow down to the Exit option and press ENTER. This brings you to the D: prompt.
13
Cold boot the gun again by turning the gun off.
14
Hold down the A+B+D keys and turn the power back on.
15
Release the A+B+D keys.
16
Once the unit has finished booting, you can set up the host configuration screen by pressing FUNC, CTRL, R.
17
The password is lowercase system.
18
The screen will display a warning, Any Active Session Will Be terminated, Continue?, answer Y to continue.
19
You will then be at the CONFIG HOST screen. Press ENTER to get to the actual host configuration setup.
20
Enter the Host IP address when prompted.
21
Enter the Host Profile Name when prompted. This is the name of the host you are connecting to.
22
When prompted for Port use 23 as the port number.
23
Enter VT220 as the Terminal Type. If you CTRL+N, the terminal types scroll down or you can enter VT220.
24
To save this information, press CTRL+S to save the screen. This brings back the CONFIG HOST screen.
25
Press CTRL+S to save this screen.
The RF gun is now setup to login to an Prelude session at the IP address that you entered in the host configuration screen.
Access Point Setup
Use the following steps to set up the Net ID and IP address for the access point.
1
Power up the AP and plug it into the COMM1 port on your PC with a null modem cable.
2
The Main menu of the Access Point can be accessed using a HyperTerminal session on your PC. The HyperTerminal session must be configured as follows:
-
Phone Number: Direct to COMM1
-
Terminal Keys: Checked
-
Emulation: Auto Detect
-
Bits per Second: 19200
-
Data Bits: 8
-
Parity: None
-
Stop Bits: 1
-
Flow Control: None
3
Save this HyperTerminal session under any name you choose and then double click the HyperTerminal session to start it up. The Main Menu Screen appears.
4
Choose option A for Enter Admin Mode.
5
The password is Symbol (make note of the uppercase S and lower case ymbol).
6
Choose P for AP Installation and change the IP Address, which is the IP address for the Access Point.
7
Change the NET_ID to be set to the same as the ESS Id on the gun, (uppercase PSI). This is a security check so that only RF guns with the ESS Id set to the same name as the Access Points NET _ID can communicate through the AP and into your system.
8
Once you have set these, press F1 to save the settings.
9
At the main menu close the HyperTerminal session and the Access Point is ready.
10
Once the Access Point has an IP address, then you can access the Main Menu of the Access Point by teleneting to the IP address you assigned to the Access Point.
11
Once you have set up the RF gun and AP and you plug the AP into the network, then you can login into the system (if you have the user security setup correctly.)
