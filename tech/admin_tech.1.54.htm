Using the Scheduler
Scheduler Maintenance (/SCHED.MNT)
A user may only access this maintenance routine if the user’s Schedule GRI Process flag in User Id Maintenance is set to ‘A’. This maintenance routine allows you to delete a job, but you cannot change an existing schedule. In the future, the user will be able to change date and time for a schedule in this maintenance routine.
Path: Jobstream > Scheduler > Scheduler Maintenance
