Printer Form Feed Options
If you are upgrading to V17+, you can stop blank pages from ejecting from teh printer by changing settings for the types of printers you use. To continue producing blank pages, follow this process and contact Activant .
NOTE: Once you complete this process, ALL Report Writer reports use the new Form Feed process.
Report Form Feed Maintenance (/RFFM)
Set the fields in Report Form Feed Maintenance to determine if blank pages should eject. Once you run this routine, all processes use the modified Form Feed procedure.
WARNING! Do NOT run this routine for 'single' processes becasue the process could be deleted. Therefore, leave Process Name blank to run this routine for ALL processes.
•
Discontinue Blank Pages: To stop blank pages, set Start to 0 (zero). Note: On V17+, this is the default.
•
Continue Blank Pages: To stop blank pages, set Start to '1' (one). Contact Prelude for programming changes.
Printer Class Definition (/PRINTER.CLASS)
Printer Class Definition allows you to define and maintain printer classes for printers.
Path: Goto Admin > Printers & Terminals > Printer Management > Printer Class Definition
Laser Printer Specifications
For LASER (sheet fed) printers, set IJA to 'Y' for all Stationery Types. Set FF to '0' (zero). The 'N' setting is not the same as '0' (zero); therefore, for this to work correctly for laser printers, FF must be '0' (zero).
•
Discontinue Blank Pages: To stop blank pages, set IJA to 'Y' for all Stationery Types. FF must be '0' (zero).
•
Continue Blank Pages: To continue producing blank pages, set IJA to 'Y' for all Stationery Types. Set FF to '1' (one) and contact Prelude for programming changes.
Dot Matrix Printer Specifications
For Dot Matrix (non-sheet fed) printers, set IJA to 'N' for all Stationery Types. Set FF to '0' (zero). The 'N' setting is not the same as '0' (zero); therefore, for this to work correctly for dot matrix printers, FF must be '0' (zero).
•
Discontinue Blank Pages: To stop blank pages, set IJA to 'N' for all Stationery Types. Set FF to '0' (zero).
•
Continue Blank Pages: To stop producing blank pages, set IJA to 'N' for all Stationery Types. Set FF to '1' (one) and contact Prelude for programming changes.
