AIX and UniData Software Release Policy
When a new release of AIX or UniData software comes out (not point releases like 6.1.5 to 6.1.6, but 6.1.X to 6.2.X as an example) we will bring it in house and upgrade according to the following:
3
Test it for 30 days on our test server.
4
Run it for 30 days on our production system.
5
Install it on new systems for new customers for 30 days.
6
Install it on new systems for existing customers for another 30 days.
7
Upgrade existing customers.
That is a total of 120 days from the software coming in-house to being released to the customer base for upgrades. Progression from one step to the next will only if no problems are found. We will not release the software to our customers if we find problems that will negatively impact their businesses.
