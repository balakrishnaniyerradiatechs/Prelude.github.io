Loading Files to 2 Mbit/s Series RF Guns
This document will list the steps to reprogram PDT6840, PDT6842 and PDT 3540 series RF guns. You can see which model you are working on by looking at the back of the gun. Also these guns will only communicate with access points based on frequency hopping spread spectrum (FHSS) technology. (EX. AP-3021)
Before you start you need a PC with the HEX files - LWP.HEX, UPD802.HEX and VTL46K.HEX - in a directory called C:\symbol and the Symbol utilities loaded (eg. SENDHEX). A cradle connected to PC with serial cable, and a battery.
1
First thing you need to do is format the flash memory on the gun. Turn on the gun and while the files are being loaded press control then c. This will bring you to the D:\> prompt where you will type in flshfmt. Choose yes when you are prompted whether or not you are sure. Once the message format a success appears you can power off the gun.
2
Next on the PC open an MSDOS prompt. To do this click Start/Run, and enter cmd and click OK. Locate the directory containing the HEX files.
3
Then on the RF gun hold down the "F" and "I" keys, press the PWR/On key, release the PWR/On key then release the "F" and "I" keys. This places the gun in a state where you can upload HEX files to it.
4
Use the Up/Down arrow keys on the gun and select “Program Loader” by pressing enter. Use the arrow keys to set the “Baud Rate” to 38400 (IMPORTANT: this figure must match the baud rate setting of the COM port on your PC, so if there are any problems check that setting). The rest of the settings are default so continue to press enter until the message “Start?” appears then stop and move back to the PC.
5
On the MSDOS prompt enter "SENDHEX LWP.HEX 38400 1" but do *not* press Return. This command will take the file LWP.HEX and sends it to the gun at the baud rate 38400 on COM port 1. If the PC you are using differs from this setup you must adjust the numbers accordingly. Do *not* press Enter just yet because you need the gun to be listening first.
6
Press enter on the gun so it can receive the file. If you do not press Enter on the gun quickly enough or forget to do so, the gun will shutdown and you will need to repeat step 3, including setting the baud rate on the gun.
7
Press Enter twice on the PC to execute the command “SENDHEX LWP HEX 384001”.The gun should now start to receive the file. The display on the gun should indicate that it is receiving a file. Allow it to finish - pressing any keys on the gun may cause the upload to fail.
8
When the upload is complete (the screen on the gun will have stopped changing and you should see "0000" on the screen). Press the PWR/On button to switch off the gun. Do not press Enter because this will erase what you have just uploaded.
9
With the gun off, press the following keys at the same time "A", "B" and "D", and while holding them down, press the PWR/ON button, release the PWR/On button then release "A", "B" and "D”. This is a cold boot.
10
When the gun boots back up it will ask you if you want to “REFORMAT FLASH?" It looks like the following and you need to Press 2.
INSTALLING
LWP Hex Ver
Spring: 3.06-07
802.11: 4.02-07
Loading system files
Re-format flash
User flash as-is
Select [1,2]
11
The gun will continue extracting files and return to a message saying “Load UPD802 Strike a key when ready”, do not strike a key. Instead turn off the gun by pressing the PWR/On button.
12
With the RF gun off hold down the "F" and "I" keys, press the PWR/On key, release the PWR/On key then release the "F" and "I" keys. You will select “Program Loader” and will see “<ENTER> to erase NUM”. Press Enter and the gun will beep and erase the NVM. From here you will set “Baud Rate” to 38400, and the rest you will leave the same until you get to the message “Start”. Make sure to stop at the message “Start”.
13
On the PC, enter "SENDHEX UPD802.HEX 38400 1" but do *not* press Enter. This command will take the file UPD802.HEX and sends it to the gun at the baud rate 38400 on COM port 1. If the PC you are using differs from this setup you must adjust the numbers accordingly. Do *not* press Enter just yet because you need the gun to be listening first.
14
Press Enter twice on the PC to execute the command “SENDHEX UPD802.HEX 38400 1”.The gun should now start to receive the file. The display on the gun should indicate that it is receiving a file. Allow it to finish - pressing any keys on the gun may cause the upload to fail.
15
When the upload is complete (the screen on the gun will have stopped changing and you should see "0000" on the screen). Press the PWR/On button to switch off the gun. Do not press Enter because this will erase what you have just uploaded.
16
With the gun off, press the following keys at the same time "A", "B" and "D", and while holding them down, press the PWR/ON button, release the PWR/On button then release the "A", "B" and "D” keys.
17
The gun will extract and inflate files and this can take a few minutes.
18
If the gun beeps and prompts you to remove from cradle go ahead and take the gun off the cradle and allow it to continue. Once the gun has stopped loading files power off the gun.
19
With the RF gun off hold down the "F" and "I" keys, press the PWR/On key, release the PWR/On key then release the "F" and "I" keys. You will select “Program Loader” and will see “<ENTER> to erase NUM”. Press Enter and the gun will beep and erase the NVM. From here you will set “Baud Rate” to 38400, and the rest you will leave the same until you get to the message “Start”. Make sure to stop at the message “Start”.
20
On the PC, enter "SENDHEX VTL46K.HEX 38400 1" but do *not* press Enter. This command will take the file VTL46K.HEX and sends it to the gun at the baud rate 38400 on COM port 1. If the PC you are using differs from this setup you must adjust the numbers accordingly. Do *not* press Enter just yet because you need the gun to be listening first.
21
Press Enter twice on the PC to execute the command “SENDHEX VTL46K.HEX 38400 1”.The gun should now start to receive the file. The display on the gun should indicate that it is receiving a file. Allow it to finish - pressing any keys on the gun may cause the upload to fail.
22
When the upload is complete (the screen on the gun will have stopped changing and you should see "0000" on the screen). Press the PWR/On button to switch off the gun. Do not press Enter because this will erase what you have just uploaded
23
With the gun off, press the following keys at the same time "A", "B" and "D", and while holding them down, press the PWR/ON button, release the PWR/On button then release "A", "B" and "D”. This is a cold boot.
24
The gun will extract and inflate files and this can take a few minutes. If all is well you should see a configuration screen. You should now be able to use the gun to connect to the host server and login to ADS.
