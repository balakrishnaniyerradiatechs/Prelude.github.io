Scheduler Inquiry (/SCHED.INQ)
This inquiry will display all schedules for reports and it will allow the user to drill down into the schedule. The summary screen will display the next run date and time for the report, whether or not the schedule is recurring, and if the schedule is finished.
Path: Jobstream > Scheduler > Scheduler Inquiry
