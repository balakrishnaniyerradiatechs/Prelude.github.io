Setting Up your PC to Program a MC9060 RF Gun
Software to install on PC:
1
Microsoft ActiveSync version 3.7.1. This is available from http://www.microsoft.com/windowsmobile/downloads/default.mspx - select ActiveSync 3.7.1 then select Worldwide English, accept the EULA the "Download now". Install program with default settings.
2
Embedded Visual C++ and Embedded Visual C++ SP1. With this software do a complete install when prompted everything else is default. This is available from Microsoft at: http://www.microsoft.com/downloads/results.aspx?productID=&freetext=visual+c%2B%2B+4. 0&DisplayLang=en. Install program with default settings.
The CD key for Embedded C++ is TRT 7H - KD36T - FRH8D - 6QH8P – VFJHQ.
3
Embedded Visual C++ SP3. This is also available from: http://www.microsoft.com/downloads/results.aspx?productID=&freetext=visual+c%2B%2B+4. 0&DisplayLang=en. IMPORTANT: You will need SP3 because the latest Symbol SDK requires it.
4
Symbol Mobility Developer Kit version 1.3 installed. You can find this on http://devzone.symbol.com, but you will need to create a login name to gain access to the program. Make sure you select the correct one as there are multiple different files – see screenshot below for how to locate this file and download it once you have logged in:
5
A copy of the Wavelink software including the termconfig files that we have created at Prelude will be needed and put in the programs folder on the C drive
6
Create a shortcut on the desktop to the telnetconfig file. The path should be C:\Program Files\Wavelink\TelnetCE\9000_AS.
7
Copies of two Courier font files named “cour.ttf” and “CourierFont.cpy”. These are also available from Prelude.
8
After installing all of the software mentioned above you need to configure Embedded Visual C++ to communicate with Microsoft ActiveSync.
9
Open Embedded Visual C++ and on the menu bar choose tools and go all the way down to the last option which is Configure Platform Manager.
10
A window like the picture below should appear:
11
Double click Default Device under Windows CE .NET Default Platform, and on the window that pops up choose Microsoft ActiveSync for both drop down menus and click OK on both windows to close.
12
You can close Embedded C++ and it is now ready to communicate with the RF gun.
13
An MC9000 programming cradle and a power supply needs to be connected to your PC via a USB cable.
14
You are now setup to program MC9060 RF guns. You can view the document “How to Config MC9060 RF Gun” to learn the process of programming the MC9060.
