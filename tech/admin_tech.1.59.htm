Purging Product & Warehouse Data
Path: System Parameter >Processes >Purge Utility Menu >Inventory >Purge Product & Warehouse Records
The Purge Product & Warehouse Records purge routine will purge products and warehouse records for a product line, number of months with no usage, and date established on or before the date entered.
•
Target File: enter the target file to be used for deletion: P - PRODUCT; C - Catalog (NS.PRODUCT).
•
Deletion Type: enter the deletion type. Enter 'PW' for product and warehouse record deletion. This deletion type will indicate to the system to determine if all warehouse records and only all warehouse records for a product can be deleted. If the conditions are met, the product record and associated warehouse records are deleted. If the conditions are not met, the product record and associated warehouse records are not deleted. If Deletion Type is set to ‘PW’ and a date is entered in the Established On or Before field, the system will use the Est Date field in the product header (Inventory > Maintenance > Product Maintenance) to determine if the product was established on or before that date and if the product is included in the purge process. Enter 'W' for warehouse record deletion only. This deletion type will indicate to the system to determine if a warehouse for a product can be deleted. Only the warehouse record will be deleted.
•
Warehouse: enter the warehouse number to limit the purge routine to a warehouse or leave blank to include all warehouse numbers. This field does not apply if the Deletion Type is set to PW.
•
Product Line: enter the product line to limit the product/warehouse deletion to a product line or leave this field blank to include all product lines.
•
Established Date Start: enter the date the product was established to limit the purge routine to only include products established on or before a date. Est Date in Product Maintenance screen is updated when the product record is created. This date is checked when a date is entered here.
•
Months without usage: enter a number of months the product does not have usage to limit the purge for only those products and product warehouse records.
•
Generate Report: enter 'R' to generate a report for those items to be deleted. No deletion actually takes place when 'R' is selected. Enter 'U' to actually select the items and delete them.
•
Move to Catalog File: this flag determines if the items deleted are moved to the catalog file. Select 'Y' to move the items deleted to the catalog file. Select 'N' to have the items removed from the system completely. This flag does not apply if the Deletion Type is set to 'W' or the Target File is set to ‘C’.
•
Stock Status N: this flag determines if all products with the Status Code flag set to ‘N’ in all warehouse records are deleted and moved to the catalog file. If any of the warehouse records for a product have the Status Code flag set to ‘S’, the product will not be deleted or moved. This flag does not apply if the Move to Catalog File flag is set to 'N’.
Note: Rental products are excluded from the purge process.
