How to Set Defaults for Master Files
Prelude allows you define your own defaults for all master files. The defaults are used when a new customer, product, vendor, etc. is created. To define defaults for the customer file, access customer maintenance. Enter ‘DEFAULTS’ at the customer number prompt. All current defaults will be displayed in their respective places. You may change any information just like you would for a regular customer. There are no required fields. When complete, press F2 to save. To define defaults for all other master files, you will have to use a file layout and the editor. Use /LAYOUTS to determine the attribute (and multi-value if necessary) for the default you want to modify. From TCL, type SE DEFAULTS company:filename (i.e. SE DEFAULTS 001PRODUCT). Move your cursor to attribute using the up/down keys, arrow keys, or F7 to go to a line or attribute.
You may insert, replace, delete, etc. to edit the data. Never press ENTER because this would add a line and push everything down. If you make a mistake, press ESC and don’t save changes. When complete, press F2.
