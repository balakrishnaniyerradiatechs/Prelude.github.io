Point-of-Sale Terminal Set-Up Parameters
Terminal
To enter terminal set-up parameters, press CTRL+ -. Or, if you have a “set up” button, press SHIFT+SETUP. To move from menu to menu, use the function keys.
Important Terminal Parameters
F1 settings - Emulation = Wyse-60 Comm Mode = Full Duplex
Enhanced = On Host/Printer = EIA/Aux
EIA Baud Rate = 9600 Aux Baud Rate = 4800
EIA Data Format = 8/1/N Aux Data Format = 8/1/N
F6 settings - EIA Xmt = Xon-Xoff
(normally) Aux Xmt = Xon-Xoff
To exit set-up, press CTRL+ - or SHIFT+SETUP and ‘Y’ to save changes.
Printer
If printer is not working correctly, verify that in the terminal settings, the Aux Baud Rate is set to ‘4800’.
If this doesn’t correct the problem, open the panel on the bottom of the printer (one screw holds it in place) and set the dip switches as follows: (U = Up, D = Down)
	U D U D U U U U
SW2	1 2 3 4 5 6 7 8
	U U U U U U U U
SW1	1 2 3 4 5 6 7 8
A printer test can be run by holding down the line-feed button and then turning on the printer. When this is done, the printer prints all the fonts it is capable of.
Pole Display
On the bottom of the pole display box, there are two buttons that allow you to set parameters. To enter the set-up, press the two buttons at the same time. Once in the set-up, one button moves through the options, and the other selects the settings. Make sure that the Baud Rate is set to ‘4800’.
Check Scanner
For check scanners to work with Prelude, they must emulate format #65 using the MICR software.
Start-Up Parameter Maintenance
In addition to these settings, it is important that in Start-Up Parameter Maintenance (/SM), the Location Profile be set to the port number being used. The port number must be entered as a three-digit number, so if port 45 is being used, 045 must be entered. Also, in this same screen, Retail Register Type must be set to ULTM.
