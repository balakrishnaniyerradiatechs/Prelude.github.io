Windows Clipboard
You can use the cut, copy and paste capabilities of the windows clipboard to share session data and create graphic screen prints. With an SBClient window open, press the Print Screen key on the keyboard, this will copy the current session window onto the Clipboard. You can then go into a Windows Application such as Word and then use the Edit - Paste function to paste the image from the clipboard onto your document.
