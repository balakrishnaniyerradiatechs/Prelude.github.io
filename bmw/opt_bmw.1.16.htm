Allow Overage in Work Orders
Parameter flags can be set to handle overages received on work orders. You can to allocate the extra quantity to a linked sales order or linked transfer.
BMW/Work Order Parameters (/OPTIONS.BMW.MNT)
Path: Systems > System Parameter > Maintenance > Option Parameters > BMW/Work Order Parameters
•
Allow Overage in W/O: Determines if overages on a work order are passed on to the linked sales order when the work order is completed.
-
Y - To allocate the overage to the line item linked on the sales order. The sales order line item quantity will be increased by the overage quantity.
-
N - To not allocate the overage to the line item linked on the sales order. The sales order line item quantity will not be increased by the overage quantity. The overage quantity is added to stock for the bill of material.
•
Allow WO Over To Linked Xfer: Determines if overages received in Work Order Completion will affect the linked stock transfers. This flag was added for Versions 18 & higher.
-
Y - If the line item is linked, the system automatically allocates the overage to the linked stock transfer. The line item quantity of the stock transfer is increased by the overage quantity.
-
N - If the line item is linked, the system does not allocate the overage to the stock transfer and does not change the quantity of the line item. The overage quantity is then updated into regular stock.
