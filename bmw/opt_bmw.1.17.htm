Allow Completion in W/O Issues
Typically, there are four steps in processing and completing a work order: Work Order Entry, Work Order Printing, Work Order Issue Entry and Work Order Completion. But, it is possible to complete the work order in the third step so that the last step does not have to be performed.
The parameter flag, Allow Comp in W/O Issues, gives you the option to complete and close a work order at the time of Work Order Issues Entry. If this system flag is turned on, you are prompted with the option to complete the order while in Work Order Issues Entry, or to complete the order in Work Order Completions.
BMW/Work Order Parameters (/OPTIONS.BMW.MNT)
Path: Systems > System Parameter > Maintenance > Option Parameters > BMW/Work Order Parameters
•
Allow Comp in W/O Issues: determines if work orders can be completed and closed when the work order is issued in Work Order Issues Entry:
-
Y - For Work Order Issues Entry to prompt for completing the work order. This eliminates the Work Order Completions step.
-
N - For Work Order Issues Entry to not prompt for completing the work order. The Work Order Completions program is required to complete and close the work order.
Work Order Issues Entry (/WOIE)
Path: BMW/Work Order > Work Order Processing > Work Order Issues Entry
When the Allow Comp in W/O Issues flag is set to Y, you are prompted to complete the work order in Work Order Issues Entry.
•
No: To not complete the work order. The Work Order Completion process must be performed to complete and close the work order.
•
Yes: To complete the work order. The work order is updated as complete and closed.
