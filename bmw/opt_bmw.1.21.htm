Repair Orders
Repair orders can be entered using Sales Order Entry (type 10) or Quotation Order Entry (Type 03). These are are designed primarily to interface with the work order module to accumulate material, labor, and other costs associated with the repair. In most cases there would be a product number and associated bill of material (BOM) established for each unit to be repaired. *MISC products can also be established with general BOMs. The existence of a BOM will lead you directly to the Work Order Entry or Work Order Quotation Entry.
The following types of Repair Orders are available:
•
01 - Warranty. Used when a vendor is to be invoiced for part or all of the charges for the repair, or the distributor is to absorb part or all of the cost.
•
02 - Time and Material. Used when the customer is to be charged for material and labor expended on the repair order, based upon standard pricing logic and parameters for the customer.
•
03 - Fixed Fee. Used when a customer is to be charged a fixed amount (usually based on an estimate related to the condition of the product being repaired) regardless of the material and labors expended by the distributor to accomplish the repair.

A Teardown Contact Id is the person responsible for entering a quote and contacting the customer with a price estimate. This Id can be set in two places in the system. The hierarchy to determine which level to use first is:
•
1st level - Customer Maintenance - Teardown Contact Id field.
•
2nd level - Sales Rep Maintenance - Teardown Contact field.
Customer Maintenance (/CM)
Path: Order Processing > Maintenance > Customer Maintenance > F8-Contact
A Teardown Contact Id can be established for each customer.
•
Teardown Contact: Enter the default Teardown Contact for the customer. This contact overrides the contact set at the sales rep level. All repair orders for this Id display in Final Price Inquiry.
Sales Rep Maintenance (/SRM)
Path: Order Processing > Maintenance > Customer Control Files > Sales Representatives
The Teardown Contact Id field must be established for each sales rep.
-
Teardown Contact: Enter the User Id of the person who is the Teardown Contact for this sales representative. This is the Id to use in Final Price Inquiry for repair orders.
Quotation Order Entry (/QOE)
Path: Order Processing > Additional Subsystems > Quotation Subsystems > Quotation Entry
This program can be used when a customer returns a product for an estimate on the repair cost. If the customer does not require a quote before the work is to be performed, you can go directly into Sales Order Entry and enter a type 10 order for Repair.
•
Quote Type: Enter type 03 for Repair Quotation. This prompts for a Repair Type:
•
Repair type: Enter the Repair type:
-
01 - Warranty
-
02 - Time and Repair
-
03 - Fixed Fee
•
F2 - Save: Once all the header information was entered, this brings up the Quotation Order Entry screen:
•
Ln# Product # / Description: Enter the BOM part number and quantity. This brings up the following prompt:
Choose Repair to bring up the following prompt:
•
Enter Repair Warehouse Number: Enter the warehouse where the BOM is to be repaired. This allows for the automatic creation of a non-inventory transfer to move the product to another location if necessary.
The Repair Work Order Quotation is initially presented without components:
•
*Action -> Add Comp: Brings up the Bill of Material Components screen to allow you to add components from the product’s BOM. The Repair Work Order Quotation number may be used as a tag number and attached to the returned product. The quote and associated work order quote may be modified as required.
•
F2 - Accept: If not all items on the BOM components list are to be used, move the cursor to Add for each item to be included, enter the quantity and press F2 - Accept.
•
F5 - Accept All: Selects all items on the BOM for the repair order.
Enter any price changes or additional items and press F2 to finish the quote. Once the customer approves the quote for repair, it can be converted to a sales order upon completion of the quote entry. The work order quote is converted to a live work order.
Work Order Status Entry (/WOSE)
In addition to the normal sales and work order follow-up, the Work Order Status Entry screen can also be used to closely monitor the progress of a customer repair order.
Each activity is entered in this screen as it is completed. The information regarding the customer order, entry, and product arrival are entered when the repair work order quotation is created. The price and cost are automatically updated throughout the quotation process.
Work Order Status Inquiry (/WOSI)
Displays the work order status of a customer repair order.
Final Price Inquiry (/FPI)
Displays the work order status of the repair orders for a Teardown Contact Id.
This inquiry is updated when the final price is updated in Work Order Status Entry.
