One Step Work Order
One Step orders are primarily used for work orders that have been completed but not yet entered in the system for Inventory and G/L purposes.
Work Order Entry (/WOE)
Usually, type ‘07’ orders are used for one step work orders. Once the components are updated on the work order entry screen, F2 will display 'Complete' or 'Suspend' options.
Select ‘Complete’ to complete and close the work order for the quantity entered. The G/L posting will debit and credit inventory given that no work in progress accounts will be used. Direct labor accounts will be included if labor is part of the bill of material. Select ‘Suspend’ to return to the work order at a later time for additional updates or changes prior to completing the work order.
Path: BMW/Work Order > Work Order Entry.
Work Order Log Inquiry (/WLI)
The Reference Number for the ASG code for an ‘07’ work order type is WO TYPE 07 followed by the bill number.
Path: BMW/Work Order > Inquiries > Work Order Log Inquiry.
