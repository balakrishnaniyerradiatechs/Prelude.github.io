Work Order Budget
Work order budgets compare estimated labor and material cost to actual labor and material costs. Only labor (burden items included as part of the bill of material) and material costs (components) entered on the work order in Work Order Entry are considered budget values. Any manual issues performed for the components in Work Order Issues Entry or burden items added from Miscellaneous Entry, Work Order Timesheet Entry, or Work Center Activity Maintenance are updated as actual costs in the Issued column for labor or material. If a partial quantity is released, the system uses the full quantity assembled of the components for budget analysis. This feature does not pertain to work orders using the warehousing module at this time.
Work Order Budget Inquiry (/WOBI)
Path: BMW/Work Order > Inquiries > Work Order Budget Inquiry (V18+ only)
This inquiry displays the budget labor and material costs for all work orders released for a warehouse number. The budget columns for labor and material are updated when the work order is released.
•
Warehouse Number: Enter the warehouse.
•
Sort: Enter the sort criteria for displaying records.
•
Filters: Enter Y to display the filters screen to limit the inquiry to product lines, completion dates, customer numbers, job numbers, etc.
•
W/O#: Displays open work orders for the warehouse number.
•
N: An asterisk (*) displays if header notes are entered for the work order.
•
Labor - Budget: Displays the labor budget costs reference. This column is updated when the work order is released and contains a burden item.
•
Labor - Issued: Displays the labor issued costs for reference. This column is updated when the work order is updated with a Miscellaneous Entry for the burden item.
•
Labor - Var: The labor variance is the difference between Labor Budget and Labor Issued.
•
Materials - Budget: Displays the material costs for reference. This column is updated when the work order is released with the total cost of the components.
•
Materials - Issued: Displays the material issued costs for reference. This column is updated when the work order is issued in Work Order Issues Entry for the components.
•
Materials - Var: The materials variance is the difference between the Materials Budget and Materials Issued.
•
C: For informational purposes only at this time. This Y/N flag can be set in Work Order Entry on the Change Order field on the header screen.
•
Linkage: The sales order number is displayed if the work order is linked.
•
F5-Review: Displays the work order inquiry screen for the work order selected.
•
F6-Rev Budget: Displays the Work Order Budget Detail Display screen for budget information.
•
F7-Rev Link: Displays the sales order if the work order contains a linkage.
•
F8-Log: Displays the Work Order Log Inquiry screen for the selected work order.
Work Order Budget Comparison Report (/WO.BUDGET.COMPARISON.RPT)
Path: BMW/Work Order > Reports > Daily Reports > Work Order Budget Comparison Report
This report compares estimated versus actual costs for completed work orders. Both labor and material costs are included.
•
Warehouse Number =: Enter the warehouse number or leave blank to include all warehouses.
•
Completion Date >=: Enter the beginning completion date or leave blank to include all completion dates. The system defaults to today’s date.
•
Completion Date <=: Enter the ending completion date or leave blank to include all completion dates. The system defaults to today’s date.
•
Bill Number: Enter the bill number or leave blank to include all bill numbers.
