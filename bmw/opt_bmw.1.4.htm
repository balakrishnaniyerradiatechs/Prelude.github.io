Work Order Quote Subsystem
The Work Order Quote Subsystem allows you to create and print a quote on a potential work order for a customer. Once the customer approves the quotation, it can then be converted into a regular work order.
Warehouse Maintenance (/WM > F6-Cont Setup)
Path: System Parameter > Maintenance > Inventory Controls > Warehouse Maintenance > F6-Cont Setup
•
Cont. W/O Qte Flag: Determines if continuous printing is used for work order quotes.
-
Y - To use continuous printing for work order quotes.
-
N - To not use continuous printing for work order quotes. Printing must be performed in the batch printing routine described below.
Work Order Quote Entry (/WOQE)
Path: BMW/Work Order > Work Order Quote Subsystem > Work Order Quote Entry
Work Order quotes can be entered and printed for review before creating the work order in the system. The entry screen works the same as the Work Order Entry screen. Once the quote is ready to be processed, it can be printed if continuous printing is set up. If continuous printing is not set up, use the Work Order Quote Printing process described below. Either way, it must be printed before being converted into a Work Order.
Work Order Quote Printing (/WOQP)
Path: BMW/Work Order > Work Order Quote Subsystem > Work Order Quote Printing
This batch printing process prints the work order quotes selected for printing.
•
Warehouse Number: Enter the warehouse number to print the work order quotes.
•
Work Order Quote Number: Enter the work order quote number to print or leave blank to include all work order quotes available to print.
•
Printer Form Queue: Enter the form queue where these documents should print.
Work Order Entry (/WOE)
Path: BMW/Work Order > Work Order Entry
Once the work order quote was printed it can be converted to a regular work order in Work Order Entry. Before entering a Bill Number, use the F6-Convert W/O Quote key to bring up the following Conversion screen:
•
Work Order Quote Number: Enter the work order quote number to be converted.
•
Product Number: Displays the bill of material number on the work order quote.
•
Correct Work Order Number?:
-
Y - To accept the work order quote number entered for conversion.
-
N - If this is not the correct quote number. Another one can be entered.
Once the correct quote is entered, all the information from the quote will be pulled into the Work Order and it can be processed as usual.
Work Order Quote Inquiry (/WOQI)
Path: BMW/Work Order > Inquiries > Work Order Quote Inquiry
Displays all work order quotes before they are converted to regular work orders.
Work Order Log Inquiry (/WLI)
Path: BMW/Work Order > Inquiries > Work Order Log Inquiry
The QCV code displays if the work order was created using the work order quote conversion method.
