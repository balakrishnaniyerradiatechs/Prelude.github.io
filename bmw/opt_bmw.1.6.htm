Partial Release of Work Orders
A parameter flag can be set to give you the ability to release partial or full quantities in Work Order Entry.
NOTE: This flag does not pertain to 05 Expense work orders.
BMW/Work Order Parameters (/OPTIONS.BMW.MNT)
Path: Systems > System Parameter > Maintenance > Option Parameters BMW/Work Order Parameters
•
Allow W/O Release for Partial: Determines if work orders can be released for a partial quantity.
-
Y - To allow a partial or the full quantity to be released for a work order.
-
N - To not allow a partial quantity to be released for a work order. Only the full quantity of the work order can be released.
Work Order Entry (/WOE)
Path: BMW/Work Order > Work Order Entry
Once the work order is released, select a partial or full release of the work order.
•
Full: To release the full quantity of the work order.
•
Partial: To release a partial quantity of the work order. This brings up the following screen:
•
Work Order Quantity: Displays the quantity of the work order.
•
Previous Release Date & Quantity: Displays any previous release dates and quantities for this work order.
•
Total Partials: Displays the number of partial quantities released for this work order.
•
Release Quantity: Enter the partial quantity to release for this work order. The quantity entered calculates the pick quantities on the pick ticket.
