Miscellaneous Entry in Work Orders
Miscellaneous Entry allows for payroll and accounts payable entries to be processed against a work order. This process must be performed for any burden items that exist on a work order to be added to the cost of the work order. If entering labor for multiple work orders, the Work Order Timesheet Entry can be used. See Mass Entry of Labor for Work Orders document for more information.
Miscellaneous Entry (/ME)
Path: BMW/Work Order > Work Order Processing > Miscellaneous Entries
This process must be performed for burden items on a work order to be added to the total cost of the work order.
•
Burden Number: Enter the burden number to be updated to the work order. The burden code must be established in Burden Maintenance for use in this entry.
•
Counter: ENTER over NEW to bring up the next available counter number.
•
Work Order Number: Enter the work order number that is to be updated with this burden item.
•
Entry Date:Enter the date of the miscellaneous entry. The fiscal period displays next to the date.
•
Amount: Enter the amount of the burden item to be updated to the work order.
•
Transaction Type: Determines the transaction type being entered.
-
A - For accounts payable. Used if the burden type for the burden number entered is set to A.
-
P - For payroll. Used if the burden type for the burden number entered is set to L.
-
B - For billing. Used if the burden type for the burden number entered is set to O.
When Transaction Type is set to P, you can enter employee and payroll information.
•
Employee Number: Enter the employee number associated with this payroll entry.
•
Regular Hours: Enter the regular hours worked on this work order.
•
Regular Rate: Defaults the regular hourly rate established for the burden in Burden Maintenance.
•
Overtime Hours: Enter the overtime hours worked on this work order.
•
Overtime Rate: Enter the overtime-hourly rate for the overtime hours worked.
•
F5-Comments: Comments can be added to the Miscellaneous Entry. These comments update the Line Notes in Work Order Entry when the burden number is added to the work order.
NOTE: In V17+, you can re-open a work order linked to a sales order that was invoiced in Miscellaneous Entry. The G/L posting is updated to the cost of goods sold (COGS). Example:
Work Order Entry (/WOE)
Path: BMW/Work Order > Work Order Entry
The burden number entered in Miscellaneous Entry is added to the work order in Work Order Entry.
Action >Notes > Line Notes displays the comments added in Miscellaneous Entry.
If a work order number and a burden item are entered in A/P Manual Invoice Entry, the Line Notes are updated for that work order number with the vendor number, vendor description, and invoice number. The work order number and burden item are prompted in A/P Manual Invoice Entry if the G/L account entered has the W/O Required Flag, set to Y in General Ledger > Maintenance > Chart of Accounts > Action > Module Interface.
Work Order Log Inquiry (/WLI)
Path: BMW/Work Order > Inquiries > Work Order Log Inquiry
Work Order Log Inquiry displays MIS when a miscellaneous entry is used for the W/O.
Miscellaneous Entries Report (/MLY.MISC.ENTRY.RPT)
Path: BMW/Work Order > Reports > Monthly Reports > Miscellaneous Entries
This report displays all miscellaneous entries entered for a division, selling warehouse, and fiscal period depending on the criteria entered.
•
Division Number: Enter the division number to limit the report to include only divisions or leave blank to include all divisions.
•
Selling Warehouse: Enter the selling warehouse to limit the report to include only a warehouse or leave blank to include all warehouses.
•
Fiscal Period: Enter the fiscal period to include all miscellaneous entries for that fiscal period. The system defaults to the fiscal period the system date is in.
•
Inc. ‘Fixed Type Entries?: Determines if miscellaneous entries for fixed bill of material items or for locations with warehousing are included in the report.
-
Y - To include fixed bill of material items and warehousing location miscellaneous entries.
-
N - To not include fixed bill of material items and warehouse location miscellaneous entries.
