Nonstock Price/Cost Prompt in Work Order Entry
Prelude provides the option to have Cost prompted for when a nonstock item is entered in Work Order Entry.
BMW/Work Order Parameters (/OPTIONS.BMW.MNT)
Path: Systems > System Parameter > Maintenance > Option Parameters > BMW/Work Order Parameters
•
‘* N/S Price/Cost Prompt WOE:
-
N - To not prompt for cost on nonstock items. This allows the system to work as it normally does for ‘* items in Work Order Entry.
-
Y - To prompt for a selling price and cost on nonstock items.
-
C - To prompt for cost first.
Work Order Entry (/WOE)
Path: BMW/Work Order > Work Order Entry
If the ‘* N/S Price/Cost Prompt WO flag is set to Y, you are prompted for both list price and cost when a nonstock (*) item is entered in Work Order Entry. The list price displays as the products selling price on the work order line item screen.
If the ‘* N/S Price/Cost Prompt WO flag is set to C, you are prompted for cost before any other prompts when a nonstock (*) item is entered in Work Order Entry.
If the ‘* N/S Price/Cost Prompt WO flag is set to N, you are not prompted for cost when a nonstock (*) item is entered in Work Order Entry.
