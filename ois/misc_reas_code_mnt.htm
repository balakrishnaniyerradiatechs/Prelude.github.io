Miscellaneous Reason Code Maintenance screen
The Miscellaneous Reason Code Maintenance screen allows you to create reason codes to use in the Miscellaneous Amount Entry screen to identify the miscellaneous charge on a sales order (refer to Entering a Miscellaneous Product). If the miscellaneous charge if for a gift certificate, refer to Creating Reason Codes for Gift Certificates.
Examples of Miscellaneous Charges:
•
Gift Certificates (refer to Gift Certificates).
•
Charge for liability insurance.
•
Credit the customer for a rebate earned.
•
Billing for a trade show (if not set up as a product).
Disadvantages Of a Using a Miscellaneous Charge
•
Billing only, no cost.
•
Does not exist on product lookup inquiries since it is not a product.
•
The wrong G/L account could be used if not set-up correctly.
Menu Path: Order Processing > Maintenance > Misc Code Files Miscellaneous Reason Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Miscellaneous Reason#
The miscellaneous reason code to create.
Description
The description of the miscellaneous reason code (e.g, Gift Certificate, General Purpose).
Comment Line One
The default first line of comment to add to the sales order when the miscellaneous reason code is entered in the Miscellaneous Entry screen on a sales order.
Comment Line Two
If applicable, the default second line of comment to add to the sales order when the miscellaneous reason code is entered in the Miscellaneous Entry screen on a sales order.
Miscellaneous Type
The type of miscellaneous code. For example:
•
GC - Gift certificate. When you enter this Miscellaneous reason code in the Miscellaneous Entry screen on a sales order, the Gift Certificate Maintenance screen appears to allow you to create a new gift certificate. Refer to Creating a Gift Certificate in Sales Order Entry).
•
GP - General purpose. When you enter this reason code on a sales order, the Product Description screen appears to allow you to enter the product description to appear on the sales order. Refer to Entering a Miscellaneous Product.
Miscellaneous Account:
Div
The division to assign to the reason code. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
Reg
The region to assign to the reason code. A region is a subset of Division - if Division is used (e.g., Southeast Region, Northwest Region).
Loc
The warehouse to assign to the reason code.
Dep
The department to assign to the reason code. A department is a subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
Ccr
The cost center assigned to the reason code. A cost center is a subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
Account#
The general ledger account number to post to when this reason code is used on a sales order.
F2-Save
Saves the reason code information.
F4-Del
Deletes the reason code.
