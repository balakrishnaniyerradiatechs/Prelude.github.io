Commission Splits screen
The Commission Splits screen allows you to split the commission on a sales order between 2 sales representatives. Refer to Splitting Commission Between Salesreps for more information.
Menu Path: Order Processing > Sales Order Entry > F5-Hdr Scr > Action Bar > Add Info > Comm Splits
Field and Function Key Descriptions
Field/Function Key
Description
Rep
The code for the sales representative who is to receive part of the commission.
Description
The description of the sales representative.
Rate
The percentage of the sale or gross profit amount or the fixed dollar amount that this sale representative should receive for this order.
F2-Save
Saves the commission split information.
