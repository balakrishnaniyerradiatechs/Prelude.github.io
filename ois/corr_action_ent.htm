Quality Corrective Action Entry screen
Once a complaint is entered (refer to Recording a Product Complaint), you can use the Quality Corrective Action Entry screen to assign the person responsible for the solution and to record the resolution of the problem (refer to Recording Corrective Actions).
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Quality Corrective Action Entry
Field and Function Key Descriptions
Field/Function Key
Description
Action#
The corrective action number.
Entry Date
The entry date of the corrective action.
User Id
The user ID of the person that entered the corrective action.
Status
The status of the corrective action:
•
O - Open.
•
C - Closed.
•
I - In process.
•
U - Unassigned.
•
W - Waiting for response.
Incident#
The incident number that this corrective action is intended to resolve. You can leave this blank if the incident is not yet created (refer to Quality Incident Entry).
Agent
The quality agent responsible for the corrective action (refer to Creating Quality Agent Codes).
Title
A summary title or description of the corrective action.
Description
Any additional information or further explanation of the corrective action taken to solve the problem.
Notes
Displays any notes entered for this corrective action (notes entered using the Notes option on the Action Bar).
F2-Save
Saves the corrective action entry.
F4-Del
Deletes the corrective action entry.
Action Bar
Notes
Open the Notes Information to allow you to enter additional information for this corrective action.
Output
Allow you to print, email, or fax the corrective action information.
