Product Number/State Entry screen
The Product Number/State Entry screen appears when you enter a Type Code of 2 in the Override Shipping Whse Matrix screen, to allow you to override a shipping matrix for an individual product. For example, if a customerâ€™s default shipping warehouse is 30, but they want a product to always ship out of warehouse 10, you can set up a shipping matrix to require that the pick ticket print in warehouse 10. You can also limit shipments to a zip code or zip code range within a state. Refer to Overriding Shipping Locations for Products for more information.
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Override Shipping Whse Matrix > Type Code = 2
Field and Function Key Descriptions
Field/Function Key
Description
Product
The product number that overrides the shipping matrix.
Prod Desc
The description of the product.
State/Province
The state that this matrix applies to, if applicable.
F2-Save
Saves the product number shipping information.
