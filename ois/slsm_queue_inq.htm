Employee Queue Inquiry
The Employee Queue Inquiry holds messages sent to you from other departments when sales orders/quotes are released from hold or vendor quote requests are updated. You can edit the orders if your user ID is the Written By on the sales order/quote.
Retrieving Messages from the Employee Queue
To view/edit orders in your employee queue:
1
Select Order Processing > Inquiries > Employee Queue Inquiry to open the Employee Queue Inquiry screen.
2
The User Id defaults to your user ID. Press ENTER. The number of messages held in each queue displays under Open.
3
Place the cursor in front of the desired Document type and press F5-Review to open the corresponding inquiry screen. Refer to the following Employee Queue Document descriptions for more information on each inquiry.
Employee Queue Document Descriptions
Field/Function Key
Description
User Id
The user ID of the person logged on.
Documents
The order types from which the messages were generated.
Open
The number of open orders/messages in each of the corresponding inquiries.
Order Message Inquiry
Orders that were released from Credit Hold, Administrative Order Hold, or Administrative Invoice Hold. Press F5-Review to open the Employee Order Message Inquiry. You can view any messages entered by the administrative managers and edit or complete the orders as necessary. Refer to the Administrative Hold Inquiries.
Quote Message Inquiry
Quotes that were released from hold. Press F5-Review to open the Employee Quote Message Inquiry. You can view any comments entered by the administrative quote manager and edit or complete the quote as necessary. Refer to the Administrative Quote Inquiry.
Linked Sales Order Inquiry
Sales orders linked to purchase orders, if the purchase order was edited. You can determine if the purchase order change (e.g., cost, delivery date) will affect the sales order and if you should edit the sales order accordingly. Press F5-Review to open the Linked Sales Order Inquiry. You can view the changes and any notes entered by the buyer. If necessary, you can send notes back to the buyer.
This inquiry is only activated if S/O-P/O Linkage Work Queue in Purchasing Parameter is set to Y.
VRFQ Inquiry
Vendor request for quotes for which you generated releases. This queue is updated when the buyer enters the cost received from the vendor. You can generate a purchase order from the quote request. Refer to Creating a Vendor RFQ from a Quote for information on entering quote requests and converting purchase orders.
F2-Finish
Closes the inquiry screen.
F5-Review
Allows you to review/edit any of the selected Documents.
