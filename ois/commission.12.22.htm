Updating and Paying Commission
Depending on how your flags are set in File Update Parameters (refer to Setting System Parameters for Commission) commission due to salesreps can be updated when an order is invoiced during the B Update and/or when Cash is applied in Cash Receipts Entry.
The following reports display commission calculations that you can use to pay your salesreps:
•
Salesman Commission Report
(if Commissions is set to Y in File Update Parameters to record commission when a sales order goes through the B Update).
•
Salesman A/R Commission Report
(if A/R Commissions is set to Y in File Update Parameters to record commission when an invoice is paid in Cash Receipts Entry).
