Pending Cores Due Report
The Pending Cores Due report lists pending core records created when an exchange product was invoiced.
For a pending core to appear on this report, the Auto Core Charge? flag (Order Processing Parameters > F5-Add Info > F5-Add Info) must be set to N. This option allows you to determine whether or not to bill the core charge during invoicing.
A core product is the re-usable part of a product that a customer returns either for credit or for a discount against a new item. If you set up a product as a Core Product (Product Maintenance > F5-Add Info > F5-Add Info), you should also set up a product to exchange for the core product when it is returned.
A pending core is created when you sell an exchange product to a customer but do not charge them for the core (reusable) part of the item. The core charge is in a pending status until it is billed to the customer or a core bank item is applied to it. A pending core is an open receivable.
For more information on cores, refer to “Tracking Pending Cores” in the Inventory documentation.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Core Reports1 > Pending Cores Due Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Customer Range. A beginning and ending set of customer numbers, if you want to limit the report to just these customers.
•
Whse. The warehouse that shipped the core.
•
Due Date <. Include pending core due dates that are before this date.
Column Descriptions
Column Heading
Description
Customer / Description
Line 1: The customer number.
Line 2: The customer description.
Product / Description
Line 1: The product number.
Line 2: The product description.
Order#
The sales order that contains the pending core.
Invoice#
The invoice number created when the exchange product was billed.
Inv Date
The date the order was invoiced.
Due Date
The date the invoice is due for payment.
Core Qty
The quantity of the core product that was billed.
Rec Date
The date the core product was received.
Rec Qty
The quantity of the core product received.
