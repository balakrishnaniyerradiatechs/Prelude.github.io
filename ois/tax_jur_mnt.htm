Tax Jurisdiction Maintenance Screen
The Tax Jurisdiction Maintenance screen allows you to set up tax rates to calculate the amount of tax to add to a sales or rental order. You can also assign the appropriate general ledger accounts to each tax jurisdiction. Once you set up tax jurisdictions, you can assign them to customers in the Customer Sales Tax Entry Screen (Customer Maintenance > F5-Add Info > F6-Sales Tax) and the Customer Job Sales Tax Maintenance Screen (Customer Job Numbers > F5-Add Info > F5-Tax Jurisdiction).
Menu Path: Order Processing > Maintenance > Sales Tax > Tax Jurisdictions
Field and Function Key Descriptions
Field/Function Key
Description
Tax Jurisdiction
The tax jurisdiction code to assign to customers.
Description
The description of the tax jurisdiction.
Tax Rate %
The tax rate percent to use to calculate tax for customers in this tax jurisdiction (e.g., 8.25 %).
Rental Tax Rate %
The tax rate for rentals for customers in this tax jurisdiction. This value overrides the percentage in Tax Rate % for rental orders.
Dollar Limit
The dollar amount after which the Excess Rate percentage is used to calculate the sales tax.
Excess Rate
The tax rate percent to use when a sales order total is over the amount entered in Dollar Limit.
State Abbreviations
The state code set up in the State Abbreviation Maintenance Screen that identifies the tax regulations for this tax jurisdiction.
Additional Tax Code
If using the Canadian GST, this code is the additional Canadian code that is added to the tax jurisdiction code to calculate the total tax.
Sales Tax Account
Div
The division number to use to post to the general ledger for this tax jurisdiction.
Reg
The region number to use to post to the general ledger for this tax jurisdiction.
Loc
The warehouse location to use to post to the general ledger for this tax jurisdiction.
Dept
The department number to use to post to the general ledger for this tax jurisdiction.
Ccr
The cost center number to use to post to the general ledger for this tax jurisdiction.
Account#
The account number to use to post to the general ledger for this tax jurisdiction. The account description displays.
F2-Save
Saves the tax jurisdiction information.
F4-Del
Deletes the tax jurisdiction.
F5-Addit Info
Opens the Additional Tax Jurisdiction Information Screen to allow you to determine the tax status for freight and handling charges.
