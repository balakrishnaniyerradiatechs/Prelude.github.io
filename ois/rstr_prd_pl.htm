Restricted Product Matrix Input screen
The Restricted Product Matrix Input screen allows you to enter zip code ranges to include in a product/product line restriction. Refer to Restricting Products for a State or Zip Code.
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Restricted Product Matrix > enter a Matrix Type and State Abbr
Field and Function Key Descriptions
Field/Function Key
Description
Product Line/
Product Number
Depending on the Matrix Type you chose, the Product Number or Product Line to restrict customers in this state from purchasing.
Req Permit Flg
Indicates if the sale of this product/product line is allowed if the customer has a permit on file:
•
Y - Sales are allowed to customers with a permit.
•
N - Sales are not allowed to any customers in this state.
A customer’s permit information is located in the Customer Permit Information Screen (Customer Maintenance > F7-Order > F5-Add Info > Action Bar > Permits). A customer’s ship-to address permit information is located in the Permit Information Screen (Customer Ship-to Maintenance > F5-Add Info > F5-Permit Info).
Zip Code Ranges - You can enter multiple zip code ranges to include in the restriction.
From Zip
The beginning zip code to restrict.
To Zip
The ending zip code to restrict.
F2-Save
Saves the zip code information.
F4-Del
Deletes the zip code information.
F5-Zip Exclusion
Opens the Zip Exclusions screen to allow you to enter any zip codes excluded from the restriction matrix. Customer’s whose shipping address is within these zip codes can purchase the products listed on the Restricted Product Matrix Input screen.
F6-Whse Exclusion
Opens the Warehouse Exclusions screen to enter one or more warehouses that can purchase the products, even if their zip code is within the restriction.
