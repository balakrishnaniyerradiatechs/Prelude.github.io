Open Quotation Report
The Open Quotation Report lists open quotes for customers that are not yet purged. The information on this report is the same as the information on the Quote Order Inquiry.
Menu Path: Order Processing > Additional Subsystems > Quotation Subsystem > Open Quotation Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Selling Warehouse. The warehouse that receives credit for the sale, if the quote is converted to a sales order.
•
Customer. The customer number if you want to limit the report to one customer.
•
Salesrep. The sales representative assigned to the quotes, if you want to limit the report to one sales representative.
•
Written By. The user ID of the salesperson who entered the order, if you want to limit the report to one salesperson.
•
Quote Type. The Quote Type if you want to limit the report to one of the following quote types:
-
01 - Regular quote.
-
02 - Planned quote.
-
03 - Repair type quote.
-
04 - Standing quote.
•
Quote Date >. The oldest quote date to include.
•
Quote Date<. The most recent quote date to include.
•
By Cust, Slsrep, W/B. Indicates how the report is sorted. Choose from the following:
-
C - Sort by customer.
-
S - Sort by sales representative.
-
W - Sort by written by.
Column Descriptions
Column Heading
Description
Cust#/Name
Line 1: The customer number.
Line 2: The customer name.
W/B
The written by user ID. The person that entered the sales order.
Slsm
The sales representative assigned to the quote.
Qte Date
The date the quote was entered.
Quote#
The quotation number.
P
Indicates the print/fax/email status of the quote:
•
P - Printed.
•
E - Emailed.
•
M - Printed/emailed.
•
B - Printed/faxed.
•
F - Faxed.
Ty
The quote type:
•
01 - Regular quote.
•
02 - Planned quote.
•
03 - Repair type quote.
•
04 - Standing quote.
Da
The disposition analysis code that indicates if the quote is converted or deleted. If the quote was converted to a sales order, 00 displays. If it was deleted, the disposition codes displays (refer to Creating Quote Disposition Codes).
Product Number
The product quoted.
Description
Line 1: The product description.
Line 2: The second line of product description, if applicable.
C
Indicates if the line item is part of a kit:
•
K - Kit parent.
•
C - Kit component.
•
Blank - Not a kit.
Opn-Qty
The quantity quoted.
U/M
The unit of measure of the product, as quoted.
Price/Ext.
Line 1: The unit price of the product, as quoted.
Line 2: The extended price of the product (Opn-Qty x Price).
