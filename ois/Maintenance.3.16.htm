Overriding Shipping Locations for Products
If you want a product or group of products to always ship out of a designated warehouse, you can use a matrix to override a shipping warehouse. This matrix allows you to assign a default shipping warehouse either for a product, all products within a product line, or all products purchased from a vendor. For example, if a customer uses warehouse 30 as their shipping warehouse but you have a product that always has to ship out of warehouse 10, you can set up a shipping matrix to require that the product be picked and shipped in warehouse 10. You can also limit shipments to a zip code or zip code range within a state.
When a product is entered on a sales order or quote, the shipping warehouse is determined according to the following hierarchy:
•
Product.
•
Product Line.
•
Vendor.
Ship Complete Orders
An 02 (Ship Complete) Sales Order overrides the shipping warehouse matrix since all products must ship from the same warehouse.
Auto Split Orders
If a shipping matrix is in effect for a product, the Auto Order Split screen will not appear if quantities are available in other warehouses. Refer to Shipping Split Quantities out of Other Warehouses for more information.
File Update Parameters
Ovr Ship Whse Matrix in File Update Parameters must be set to Y to activate override shipping warehouses.
To set up an override shipping warehouse matrix:
1
Select Order Processing > Maintenance > Order Entry Control Files > Override Shipping Whse Matrix to open the Override Shipping Whse Matrix screen.
2
For Type Code, choose an override type:
•
1 - Product line. Opens the Product/State Entry screen. You can enter the Product Line and if applicable, the State/Province information.
•
2 - Product. Opens the Product Number/State Entry screen. You can enter the Product and if applicable, State/Province information.
•
3 - Vendor/selling warehouse. Opens the Vendor/Selling Warehouse Entry screen. You can enter the Vendor Number and Selling Warehouse information.
3
After you enter the appropriate parameters for the Type Code, the Shipping Warehouse Entry screen appears.
4
For Whse, enter the warehouse number to use as the default shipping warehouse for the selected product, product line, or vendor/warehouse.
5
You can limit the shipping information to a zip code or enter a From Zip and To Zip to include a zip code range.
6
Enter as many warehouses and zip code ranges as necessary.
7
Press F2-Save to save the warehouse matrix information.
