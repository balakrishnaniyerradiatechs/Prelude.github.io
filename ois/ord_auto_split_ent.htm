Auto Order Split screen
Depending on system settings, the Auto Order Split screen may appear if a product that is backordered in the shipping warehouse, is available in other warehouses. Refer to Shipping Split Quantities out of Other Warehouses.
System Options for Auto Split Quantities
Order Processing Parameters > F5 > F5 > Shipping Whse Split /Reassignment
•
Y - If the quantity is available in another warehouse, automatically prompt with the Auto Order Split screen in Sales Order Entry to let the user decide if they want to ship part of the quantity from other warehouses.
•
N - Do not allocate any inventory from another warehouse. The product will be backordered.
•
A - Automatically ship part of the line item from a warehouse where it is available.
Menu Path: Order Processing > Sales Order Entry >
Field and Function Key Descriptions
Field/Function Key
Description
ReqQty
The quantity that will ship out of each warehouse.
Whse
The warehouse where the product is stocked.
Description
The warehouse description.
Usage
The calculated Forecast quantity of the product from the Product Purchasing Parameters Maintenance Screen.
Ldt
Lead time. The average number of days required to replenish the product from the day it is ordered to the day it is received.
Onordr
The quantity on-order (transfers and purchase orders) for the warehouse.
Bkordr
The quantity backordered at the warehouse.
Surplus
The surplus quantity of the product. If using the EOQ purchase method, this is any quantity available over the EOQ plus the line point. If using the min/max purchase method, this is the quantity over the maximum recommended quantity to keep in stock.
Available
The quantity of the item available to sell in the warehouse.
F2-Save
Closes the screen and applies the ReqQty for each warehouse to the sales order.
F5-Warehouse Select
Allows you to ship an available quantity out of the highlighted warehouse. The quantity is added to the Sales Order Entry screen as a separate line item.
F6-Auto Allocation
Lets the system automatically select the available quantities from warehouses where it is available. The available quantity in each warehouse that is needed to fill the order appears under ReqQty. You can change the quantities as necessary. Each quantity is added to the Sales Order Entry screen as a separate line item.
F9-Invty Inq
Opens the Inventory Inquiry screen to allow you to view available and on-order information for the product.
