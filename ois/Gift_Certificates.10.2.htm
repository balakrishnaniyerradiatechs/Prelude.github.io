Setting System Options for Gift Certificates
The following system options must be set up to use gift certificates.
Gift Certificate Option Maintenance
You can let the system generate gift certificate numbers or you can use pre-printed forms with the numbers already on them. To set options to create gift certificate numbers:
1
On the Miscellaneous Controls menu, select Gift Certificate Option Maintenance to open the Gift Certificate Option Maintenance screen.
2
For Number of Characters In Cert#, enter the number of characters to use to create a gift certificate number. This number can include a check digit field (see step 5).
3
For Gift Certificate Print Routine, enter the print program used to print gift certificates from Sales Order Entry or Gift Certificate Maintenance.
4
For Default Expiration Days, enter the number of days to add to today’s date to calculate the default gift certificate expiration date. You can override this date when you create the gift certificate. When a gift certificate expires, a warning message displays but the certificate can still be used.
5
For Utilize Check digit, choose from the following:
•
Y - Insert a check digit at the end of the generated gift certificate number. This is used for security purposes so that fake gift certificates cannot be created.
•
N - Do not use a check digit.
Since the check digit is added to the end of the number, the gift certificate number may not be a sequential number. For example, if a gift certificate is generated with number 00000014, the next gift certificate number may be 00000028. The first seven digits are sequential and the eighth digit is the calculated field for the check digit. The check digit is only used when the system generates the next available gift certificate number.
Payment Methods
To set up a gift certificates as a type of payment:
1
In Order Processing Controls, select Payment Methods to open the Payment Methods screen.
2
For Valid Payment Methods, if it does not already exist, enter 11 for Gift Certificate.
3
Press F2-Save to save the payment method information.
When you use a gift certificate as a form of payment on a sales order, enter a Pay Type of 11 in the Cash Application screen when completing the sales order. Refer to Applying a Gift Certificate to a Sales Order.
