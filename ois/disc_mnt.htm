Discount Break Table Maintenance screen
You can use the Discount Break Table Maintenance screen to define the quantity discount levels to determine product prices and costs. Each price or cost is expressed as a set value or as a discount off of a column price. Once a table is set up, you can reference it in Product Maintenance, Product Warehouse Maintenance, Product Vendor Maintenance, Vendor Line Maintenance, or Price Matrix Maintenance to calculate prices and costs.
Menu Path: Order Processing > Maintenance > Discount Break Table Maintenance > Pricing Structures > Discount Break Table Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Table Number
Identifies this set of rules to determine discounts. This number is referenced in Price Matrix Maintenance, Product Maintenance, Product Warehouse Maintenance, or Vendor Maintenance to generate prices and costs.
Warehouse
The warehouse number if this table only applies to one warehouse. Leave this field blank if the discount break table applies to all warehouses.
Description
The description of the discount break table so that you can easily identify it when choosing a table.
Sum And Recost?
Determines if you will use the sum & recost method to cost line items:
•
Yes - Accumulate line items and recost the line items if a quantity break level is met.
•
No - Recost line items based on their individual quantity or dollar value.
Table Method
Determines how to calculate price and cost. You can express break-level pricing as a set price or as a discount off of a column price. Choose from the following:
•
$ - Each break has a set price. Refer to the Discount Table - Dollar screen.
•
* - Price is a discount percent off of a column price. Refer to the Discount Table - Percent screen.
•
C - Rebate base cost. Use current cost to calculate a rebate.
•
D - Chain discount.
•
M - Net margin. Calculate the price as a percent of a gross profit.
•
P - Rebate percent - price basis. Calculate the rebate as the difference between actual and rebated cost (e.g., the price is higher than the cost).
•
R - Replacement cost. Use rebate base cost (replacement) to calculate the rebate.
Break Level Type
Defines how break levels are applied:
•
U - Units.
•
$ - Dollars.
•
M - Unit of measure.
