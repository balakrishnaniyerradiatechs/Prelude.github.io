Release History Inquiry
The Release History Inquiry displays picking and shipping information for invoiced sales order releases. Refer to Invoicing.
Viewing Sales Order Releases
To list sales order release information:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Release History Inquiry to open the Release History Inquiry screen.
2
Enter the Order Number.
Sales order release information appears.
Field & Function Key Descriptions
Field/Function Key
Description
Release#
The sales order release number.
Whse
The shipping warehouse assigned to the order.
Prt-Date
The date the release printed.
Prt-Time
The time the release printed.
Proc-Date
The date the pick ticket was processed in Release Confirmation Entry.
Ship-Dte
The date the pick ticket release shipped from the warehouse.
Picker
The initials of order filler. Entered in Release Confirmation Entry.
Packer
The initials of the person who packed the order. Entered in Release Confirmation Entry.
F2-Finish
Closes the inquiry screen.
F5-Review Release
Opens the Release History Confirmation Inquiry screen to display shipping information.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
