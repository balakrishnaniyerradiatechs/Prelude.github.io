Linked Sales Order Inquiry
The Linked Sales Order Inquiry is updated for your user ID when a purchase order that you linked to a sales order is edited. You can review the changes and any notes sent by the buyer and edit the sales order as necessary. You cannot change or cancel the purchase order but you can send notes back to the buyer.
This inquiry is only updated if S/O-P/O Linkage Work Queue in Option Parameters > Purchasing Parameters is set to Y.
Reviewing and Releasing Linked Sales Orders
To review linked sales orders and to remove them from your queue:
1
Select Order Processing > Inquiries > Linked Sales Order Inquiry to open the Linked Sales Order Inquiry screen (this screen is also accessible from the Employee Queue Inquiry).
2
Enter your User Id. Linked sales orders appear.
3
To review or release the order:
•
Select F5-Review to open the Sales Order Display screen to review the sales order.
-
Press F7-Rev Link to view the linked purchase order.
-
Press F2-Save to return to the Linked Sales Order Inquiry.
•
Select F8-Release to release the order from the queue.
The following message appears:
Do You Wish To Flag This Line As Reviewed?
4
Choose from the following:
•
No - Retains the sales order in your queue. The Status on the Linked Sales Order Inquiry screen remains as N.
•
Yes - Changes the Status on the Linked Sales Order Inquiry screen to R and opens the Notes Entry screen so you can enter notes regarding the linked order.
The line item is removed from the queue when you exit the Linked Sales Order Inquiry screen.
Field & Function Key Descriptions
Field/Function Key
Description
User Id
Defaults to the user ID of the person logged in.
Order#
The sales order linked to a purchase order that has been edited.
Counter
The line on the sales order.
Customer
The customer who ordered the material.
Description
The customer description.
User
The user ID of the person that entered the order.
Link-Date
The date the sales order was linked to a purchase order.
N
An asterisk (*) indicates that notes exist in the order header.
Link Notes
Notes sent from the buyer when the purchase order was edited.
Status
Indicates if the order was reviewed in this work queue:
•
R - Reviewed.
•
N - Not reviewed.
Reason
The reason code that represents the type of change on the purchase order. You can define your own reason codes in Linkage Queue Reason Code Maintenance (Systems Menu > Misc Controls). Some examples are:
•
P - Change in price.
•
D - Change in delivery date.
F2-Save
Closes the inquiry screen.
F5-Review
Opens the Sales Order Display screen to allow you to view the sales order. Press F7-Rev Link to view the linked purchase order. When you exit the Sales Order Display screen, you can choose to mark the item as reviewed.
F7-Notes
Opens the Notes Entry screen to allow you view any notes entered by the buyer when the purchase order was edited.
F8-Release
Marks the order as reviewed (Status = R) and allows you to enter notes to the buyer regarding the order. When you exit the screen, the released orders are removed from the queue.
