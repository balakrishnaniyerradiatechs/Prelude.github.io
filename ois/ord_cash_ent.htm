Cash Customer Information screen
The Cash Customer Information screen allows you to enter name, address, phone number, and payment terms for a cash sale order. Existing cash customers are stored in the system based on their phone number.
Menu Path: Order Processing > Sales Order Entry > Customer = CASH
Field and Function Key Descriptions
Field/Function Key
Description
Phone Number
The customer’s phone number. You can use the phone number to search for existing cash customers.
Cash Customer Name
The person or company name.
Price Group
The price group classification for the cash customer, if applicable. The price groups are established in the Customer Price Group Maintenance Screen to group customers using price matrixes for discounts or special pricing.
Address Line One
The cash customer’s address.
Address Line Two
The second line of address.
Address Line Three
The third line of address.
Country
The country of residence.
Geographic Code
The geographical code assigned to this customer location, if you use geo codes.
City
The customer’s city.
State
The customer’s state.
Zip
The customer’s zip code.
Payment Terms
The accounts receivable payment terms code for orders that determines when payment is due, payment conditions, and any applicable discounts. These codes are set up in A/R Terms Maintenance.
Language Indicator
The code that indicates which language should appear on pick tickets, packing lists, quotes, and invoices for this customer.
Email Address
The customer’s email address.
F2-Save
Save the cash customer information and opens the Sales Order Header screen.
