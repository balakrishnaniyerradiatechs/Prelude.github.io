Quality Inspection Default Maintenance screen
The Quality Inspection Default Maintenance screen allows you to assign a default inspection procedure for each inspection type. Inspection types are used in Inspection Results Entry to record when the inspection is performed.
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Quality Inspection Procedure Default Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Inspection Type
The inspection type to which you want to assigned a default procedure:
•
C - Customer return.
•
P - PO receipts.
•
V - Vendor performed.
•
W - Work order manufacturing.
Proc#
The default procedure number set up in Quality Inspection Procedure Maintenance (refer to Setting Acceptable Quality Levels for Inspection Procedures). This is the procedure number that will appear in Inspection Results Entry.
Description
The description of the default procedure.
F2-Save
Saves the quality inspection procedure information.
F4-Del
Deletes the quality inspection procedure default information.
