Geographical County/City Lines Screen
A geographic code may contain a zip code that covers more than one city and/or county. Once you set up the geographic codes in the Geographic Code Maintenance Screen, you can use the Geographical County/City Lines screen to set up all city/county geographic codes for a zip code.
When you enter a multi-city/county geographic code, all available county/city geographic codes display to allow you to choose the correct code. Once you select the correct code, the system updates the address, sales representative, warehouse, and tax information according to the values assigned for the geocode in Geocode Maintenance.
Menu Path: Order Processing > Maintenance > Sales Tax > Geographic Codes > F5-County/City Lines
Field and Function Key Descriptions
Field/Function Key
Description
Geo Code
The geographic codes that share a zip code with the original geographic code.
City
The city for this geographic code.
State
The state abbreviation for this geographic code.
Zip Code
The zip code for this geographic code.
County
The county for this geographic code.
F2-Save
Saves the county/city lines information.
