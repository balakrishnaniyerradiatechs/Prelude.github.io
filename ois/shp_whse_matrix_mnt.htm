Override Shipping Whse Matrix screen
If you want a product or group of products to always ship out of a designated warehouse, you can set up an override shipping warehouse matrix in the Override Shipping Whse Matrix screen. This matrix allows you to assign a default shipping warehouse for a product, all products within a product line, or all products purchased from a vendor. For example, if a customer’s shipping warehouse is 30 but you want a product to always ship out of warehouse 10, you can set up a shipping matrix to require that the product is picked and shipped in warehouse 10. You can also limit shipments to a zip code or zip code range within a state. Refer to Overriding Shipping Locations for Products for more information.
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Override Shipping Whse Matrix
Field and Function Key Descriptions
Field/Function Key
Description
Type Code
Indicates the criteria used to override the shipping warehouse:
•
1 - Product line. Opens the Product/State Entry screen. You can enter the Product Line and if applicable, the State/Province information.
•
2 - Product. Opens the Product Number/State Entry screen. You can enter the Product and if applicable, State/Province information.
•
3 - Vendor/selling warehouse. Opens the Vendor/Selling Warehouse Entry screen. You can enter the Vendor Number and Selling Warehouse information.
Refer to Overriding Shipping Locations for Products for more information on shipping warehouses.
F2-Save
Saves the warehouse matrix information.
F4-Del
Deletes the warehouse matrix.
