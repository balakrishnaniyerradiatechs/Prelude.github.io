Promotion Matrix Maintenance Header screen
The Promotion Matrix Maintenance Header screen allows you to define a vendor and buy quantities for a customer promotion. The use of the screen varies depending on whether you are Creating a Promotion Price Matrix by Product or Creating a Promotion Price Matrix by Order Total.
Menu Path: Order Processing > Maintenance > Pricing Structures > Promotional Pricing > Promotion Type Matrix Maintenance > F5-Hdr
Field and Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor associated with the promotion.
Start Date
The day the product promotion begins.
Expire Date
The day the promotion ends.
Campaign Number
If the promotion is associated with a marketing campaign, you can enter the campaign number. This number must be entered on a sales order to activate the promotion. Leave blank if you do not use campaigns or if all marketing campaigns use the promotion.
Audit Sales
Determines if sales using this promotion are tracked:
•
Y - Track promotion sales. Informational only for custom reporting.
•
N - Do not track promotion sales.
Blanket Contract
If applicable, you can enter a blanket P/O contract if one was set up for this product in Blanket Contract Maintenance (refer to Blanket Contracts). If you enter a value, only sales orders with this blanket contract number in Customer P/O# will receive the promotional pricing.
One Time Promo
Indicates if this promotion is a one-time offer. Choose from the following:
•
Y - This is a one-time offer for each customer.
•
N - The promotion is available for all orders up until the Expire Date.
Buy Qty
The quantity that the customer must buy to qualify for the promotion. For example, enter 1 if the customer must buy 1 product in order to receive the promotion. This field only applies if you are Creating a Promotion Price Matrix by Product.
Get Qty
The quantity of the promotional product to receive if the Buy Qty is met.
Buy Amt
The value a customer’s order must be to get a discount off of their purchase. This field only applies if you are Creating a Promotion Price Matrix by Order Total.
Disc%
The discount percent the customer will receive if they purchase the Buy Amt.
Description
The description of the promotion. You can use this description to search for existing promotional matrices.
F2-Save
Saves the promotional information and returns to the Promotion Type Matrix Screen.
F7-Cust Exc
Opens the Promotion Exception Maintenance screen to allow you to include or exclude a customer from a promotion matrix. Refer to Excluding/Including Customers in a Promotion Matrix.
F8-Prd Exc
Opens the Promotion Product Exception Maintenance screen to allow you to exclude or include products from a promotion matrix. Refer to Excluding or Including Product Lines on a Matrix.
