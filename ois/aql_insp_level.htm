Sample Size Code Letter Maintenance screen
Samples refer to products selected at random to test for quality. A sample size is the quantity of a product in an entire lot that should be inspected. Inspection levels for lot sizes are already set up in your system based on the American National Standard (ANSI) recommendations. When you enter inspection results for products (refer to Inspection Results Entry), the sample size recommended for testing is based on the value that you enter for Lot Size.
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Sample Size Code Letter Maintenance

Represents the relationship between the lot size and the sample size. The inspection level to use (e.g., Level 1, Level 2, Level 3) is determined by your company.
The code letters listed below each level (e.g., A, B, C) are established by American National Standards and indicate the sample size for each lot size. Refer to Using Single Sampling Tables to view the acceptable quality level for each Code.
