Rate Shop Results screen
If you have rate shopping set up in your system, the Rate Shop Results screen allows you to compare 3rd party shipping costs for the ship via code used on this sales order. The weights established in Product Maintenance for each product are used to calculate the delivery rate. You can change the number of cartons, weight, and ship via method to compare difference scenarios.
Menu Path: Order Processing > Sales Order Entry > Action Bar > Order Func > Rate Shop
Field and Function Key Descriptions
Field/Function Key
Description
Lbs
The number of pounds calculated so far for the products on the Sales Order Entry screen.
Ctns
The estimated number of boxes it will take to fill this order. The total weight is divided by the number of boxes to get an average weight per box. The rate comparison for each shipping method is calculated based on both the average weight per box and the extended weight.
Ship Via
The ship via method to use to shop for rates. You can use one the codes set up in the Ship Via Maintenance Screen or compare the best rate for just one delivery method. For example:
•
BA - Use the best rate from all ship via types.
•
B1 - Use the best rate for next day delivery.
•
B2 - Use the best rate for 2nd day delivery.
•
B3 - Use the best rate for 3 day (ground) delivery.
•
B1A - Use the best rate for next day A.M. delivery.
•
B2A - Use the best rate for 2nd day A.M delivery.
Shipper
The company name and type of delivery service (e.g., UPS Blue, UPS Red, Fedex 2nd day, Fedex Express Saver).
Service Type
The type of service this shipper description represents (e.g., 1 for next day delivery, 2 for 2nd day delivery, G for ground delivery).
Rate
The rate charged by this carrier for one carton, based on average weight per carton.
Ext Rate
The extended rate charged by the carrier to ship the entire order.
F2-Exit
Closes the Rate Shop Results screen.
F5-Commit
Allows you to commit the selected shipper to the sales order.
F6-Re-Rate
Re-calculates the rate if changes are made.
