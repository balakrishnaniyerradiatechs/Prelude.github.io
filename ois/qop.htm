Printing Quotes by Batch
If quotes are not set up for continuous transmission, you must print, fax, or email them by batch.
To transmit quotes by batch:
1
Select Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Printing to open the Quotation Printing screen.
2
Enter the Quotation Number or leave blank to print all quotes for a warehouse.
3
Enter the selling Warehouse, which is the warehouse that receives credit if the quote is converted to a sales order.
4
Enter the Printer Form Queue where the quotes will print.
5
For Quote Message, you can enter a message to print at the bottom of the quotes. The following message appears:
Load Quote Forms In Printer & Align Visually
6
Choose from the following:
•
Print Quotes - Print the quotes.
•
Mask Print Test - Allows you to test the alignment of the quote forms in the printer. Not necessary if you are using a laser printer. After the forms are correctly aligned, select Print Quotes.
•
Abort Printing - Cancel the quote printing.
7
After you select Print Quotes, the following message appears:
Quote Printing Complete...
8
Choose from the following:
•
Process - If all quotes printed successfully.
•
Reprint All - If none of the quotes printed successfully. This option is also useful if only a couple of quotes printed successfully from a large print job.
•
Single Reprint - To reprint quotes individually. This option is useful if a couple of quotes jammed in the printer.
•
Abort - Cancel the quote printing.
