Gift Certificate Maintenance screen
The Gift Certificate Maintenance screen allows you to create a gift certificate to sell to a customer. You can access this screen either when you enter the gift certificate reason code in the Miscellaneous Amount Entry screen (refer to Creating a Gift Certificate in Sales Order Entry) or in Gift Certificate Maintenance (refer to Creating a Gift Certificate in Gift Certificate Maintenance).
Menu Path: Order Processing > Sales Order Entry > Enter M or MN for Product.
or
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Gift Certificate Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Certificate Number
The gift certificate number. It can either be your own gift certificate number or a system-generated gift certificate number (refer to Setting System Options for Gift Certificates).
Recipient Name
The name of the person who will receive the gift certificate. You can use this name when searching for a gift certificate.
Address One
The first line of address for the recipient.
Address Two
The second line of address for the recipient.
Address Three
The third line of address for the recipient.
City
The recipient’s city of residence.
State
The recipient’s state of residence.
Zip
The zip code.
Message
A message to appear on the certificate (e.g., Happy Birthday!).
Expiration Date
The date the gift certificate will expire (optional). When the gift certificate is used, the expiration date is validated if one exists. If the gift certificate is expired, a warning displays but the gift certificate can still be used.
Issuing Customer
The account number of the customer that is purchasing the gift certificate.
Certificate Amount
The value of the gift certificate.
Remaining Balance
Defaults to the Certificate Amount. This balance is reduced each time the gift certificate number is entered in the Cash Application screen on the sales order.
F2-Update
Creates or updates the gift certificate.
F4-Delete Certificate
Deletes the gift certificate.
F8-Print Certificate
Prints a copy of the gift certificate.
