Order Log Inquiry
The Order Log Inquiry lists all transactions that affected a sales order. Each transaction is preceded by a Code that defines the change or process that affected the quote. The date, time, and user who entered the transaction display also.
Order Log in File Update Parameters must be set to Y to track order lot detail information.
Viewing Sales Order Transactions
To list all transactions that occurred for a sales order:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Order Log Inquiry to open the Order Log Inquiry screen.
2
Enter the Sales Order Number to view.
All transactions for the order appear.
Field & Function Key Descriptions
Field/Function Key
Description
Code
The order log inquiry code that identifies the transaction. Press F1 to list the codes, or refer to the Order Log Inquiry Codes for code definitions.
Act Date
The date the corresponding action took place.
Time
The time the corresponding action took place.
Customer
The customer number.
User
The user ID of the person that entered the transaction.
Reference Number
The associated order, product, ID, or other reference number associated with the transaction. Refer to the Reference column in the Order Log Inquiry Codes for full descriptions.
F2-Finish
Closes the inquiry screen.
F5-Review
Depending on the Code, it may open either:
•
The Sales Order Display screen to display detail information for the highlighted sales order.
•
The Order Log Inquiry notes screen to reference changes made.
F6-Consolidation
Opens the Consolidation Order View screen to display the shipping conditions of all products on both open and invoiced orders.
F9-Goto line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Output
Allows you to print, fax, email, or save (to an Excel spreadsheet) the order information.
