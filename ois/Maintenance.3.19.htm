Creating the Package Insert Codes
Once you create the package insert products (refer to Creating the Package Insert Product), you can assign one or more of the products to a code in Package Insert Code Maintenance. The package insert code determines when the free products are sent to a customer.
To create a package insert code:
1
Select Order Processing > Maintenance > Order Entry Control Files > Package Insert Code Maintenance to open the Package Insert Code Maintenance screen.
2
For Insert Code, enter an alphanumeric code that identifies the package insert.
3
Enter a Description of the package insert (e.g., tool insert, automotive insert).
4
To determine when a customer should receive a free product, choose one or more of the following methods:
•
To give the customer a free product for their:
-
First Order - Enter the package insert product that they should receive the first time they place an order with your company.
-
Second Order - Enter the package insert product that they should receive the second time they place an order with your company.
-
Third Order - Enter the package insert product that they should receive the third time they place an order with your company.
•
To give a customer a free product based on the number of days since their last order:
-
For Last Order Days, enter the package insert product that they should receive.
-
For # Days, enter the number of days since their last order that determines when they should receive the package insert product entered in Last Order Days.
•
To give a customer a free product based on the value of the order:
-
For Order Value, enter the package insert product that they should receive.
-
For Dollars, enter the amount that determines when they receive the package insert product entered in Order Value.
•
To give a customer a free product based on the product that they buy, press F5-Products to open the Insert Code Product Numbers screen.
-
Enter the Product# for which the customer will receive the following Insert Product.
-
Enter the Insert Product the customer receives when they buy the Product#.
-
Press F2-Save to save the product information and to return to the Package Insert Code Maintenance screen.
•
To give a customer a free product based on products purchased within a product line, press F6-Product Lines to open the Insert Code Product Lines screen.
-
Enter the Product Lines that determine when the customer receives the free product or use Plines Omitted to enter the product lines for which you want to exclude the customer from receiving free products.
-
For Product Number, enter the package insert product number to ship when the customer buys products within the designated product lines.
-
Press F2-Save to save the product line information and to return to the Package Insert Code Maintenance screen.
5
If a customer is receiving too many inserts, you can limit the number to prevent hundreds of free products from being given away. To limit the number of inserts:
•
Press F7-Addtl Options to open the Additional Options screen.
•
For Maximum Inserts, enter the maximum number of insert products that should be sent out with any order. This number is used with the Ranking order to determine which products to add to an order.
•
For Ranking, enter the codes in order of importance to determine which inserts are sent if there is a value in Maximum Inserts. The first, second, and third order insert products are automatic and do not need to be included in the ranking. Choose one or more of the following codes:
-
D - Days since last order.
-
L - Product lines.
-
P - Product numbers.
-
V - Total order value.
•
If you want to notify the sales person any time a package insert is added to an order, for Notification, enter the codes that cause a message to appear on the sales order when a package insert is sent:
-
D - Days since last order.
-
L - Product lines.
-
P - Product numbers.
-
F - First order.
-
S - Second order.
-
T - Third order.
-
V - Total order value.
6
After you assign all the package insert information to the insert code, press F2-Save.
