Skeleton Product Maintenance screen
The Skeleton Product Maintenance screen allows you to create a product number while entering a sales order. You can enter basic information about the product and then complete the detail information in Product Maintenance at a more convenient time. Refer to Creating Products in Sales Order Entry for more information.
Order Processing Parameters
Allow New Prod in S/O determines if products can be created within Sales Order Entry or Quote Order Entry:
•
N - You cannot create a new product on a sales order.
•
Y - If you enter a non-existing part number, a dialog box appears to ask if you want to create a new product. If you select Yes, the Skeleton Product Maintenance appears and you can create a new product to add to the product file.
•
C - If you enter a non-existing part number, a dialog box appears to ask if you want to create a comment line to reference the product. You can enter a description of the product that appears as a comment. No inventory is affected.
•
B - If you enter a non-existing part number, a dialog box appears to ask if you want to create a new product or add a comment line. If you choose to create a product, the Skeleton Product Maintenance appears and you can create a new product to add to the product file. If you choose to add a comment, You can enter a description of the product that appears as a comment.
Menu Path: Order Processing > Sales Order Entry
Field and Function Key Descriptions
Field/Function Key
Description
Product
The product number to create.
Description
The description of the new product number.
Description Two
The second line of description for the product number.
Product Line
The product line that the product belongs to. Product lines group similar products for selling and are required for every part number.
Product Type
Product types are used to group products for reports. Choose from the following:
•
F - Finished good.
•
R - Raw material.
•
B - Both a finished good and a raw material.
•
E - Expense item.
•
N - Rental.
•
U - Used item.
•
P - Part. Same as raw material but is used for customer reporting purposes.
U/M Table
Indicates if a unit of measure table is required for this product. If the product is purchased, stocked, and sold in different units of measure, you may require a unit of measure table:
•
* (Asterisk) - A unit of measure table already exists for the product.
•
Y - A unit of measure table is required for the product. The U/M Table Maintenance screen appears to allow you to enter a unit of measure table. The unit of measure table uses Buy, Stock, and Price to affect how the system buys, stocks, and sells the item. It can also track inventory using multiple units of measure and inventory factors.
Refer to ”Establishing a Unit of Measure” in the Inventory documentation.
•
N - A unit of measure table is not required.
You cannot use Stock/Buy Ratio with a unit of measure table.
Weight
The weight of one unit of the product.
Buy
The unit of measure to use when you purchase the product from a vendor.
Stock
The unit of measure to use for stock in your warehouse. Enter all costs for the product and product-warehouse level in the terms of the Stock unit of measure.
Price
The unit of measure to use to sell to your customers. Enter all prices at the product and the product-warehouse level in terms of the Price unit of measure.
Replacement Cost
The cost to replace the product.
Product Prc Group
The product price group established in Product Price Group Maintenance to group products for pricing discounts. Refer to ”Establishing Product Price Groups” in the Inventory documentation for more information.
Primary Vendor
The primary vendor used by Nightly Processing to recommend purchase orders.
Vendor LIne
The vendor line for the product. Vendor lines group products for purchasing. Refer to “Vendor Line Maintenance” in the Purchasing module.
Vendor Product
The vendor’s product number, if applicable.
F2-Save
Completes the skeleton maintenance information and opens the Line Item Detail screen opens to allow you to enter a quantity and to verify or change other information for the line item (refer to Entering Sales Order Line Items).
F5-Add Info
Opens the Additional Information screen to allow you to enter detail information for the product.
F6-Xref
Opens the Product Xref/Notes screen to allow you to enter cross reference information for the product.
F7-U/M Table
Opens the U/M Table Maintenance screen to allow you to enter or edit a unit of measure table.
Action Bar
Price Qty Break
Opens the Quantity Break Table screen to allow you to enter quantity break pricing for the product.
Cost Qty Break
Opens the Cost Quantity Break Table to allow you to enter quantity break cost for the product.
Scan Image
Allows you to scan images to attach to the product information, if you have the imaging software.
Language Code
Opens the Product Language Code Mnt screen to allow you to define the language to use for the product description.
