Customer Prescription Maintenance screen
Before you can enter a prescription product on a sales order, you must use Customer Prescription Maintenance to set up a prescription number that identifies the customer, physician, products, quantities, refills, and expiration date information. Refer to Creating a Customer Prescription.
Menu Path: Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > Customer Prescription Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer to receive the prescription.
Prescription#
The number of the prescription:
•
Enter NEW for a system generated number (refer to Setting Prescription and Facility Counter Numbers).
•
Enter any prescription number as determined by your company.
•
Enter an existing prescription number to view or track refill information.
•
Press F3 to search for existing prescriptions for this customer.
Ship To
The Ship To account number if the prescription is limited to just one ship-to account set up for the customer.
Issued
The date the physician issued the prescription.
Expiration
The date the prescription expires. The default is the date set up for each state in Maximum RX Month Maintenance (refer to Identifying Maximum RX Months by State). If no default was set up for the state, the system default is 6 months.
Status
The status depends on if today’s date is before or after the expiration date:
•
Open - The prescription is not yet expired.
•
Closed - The prescription is closed.
Comment
Comments to include with the prescription, if applicable.
Contact
The name of the person the prescription is for.
Renewal
The Renewal date of the prescription.
Phoned
The date the physician was contacted for renewal.
Phone Ok’d
Y indicates that the prescription was phoned in and confirmed.
Species
The species that the prescription is for. Only valid species may be entered (refer to Creating Valid Species Responses).
Physician
The Physician that issued the prescription. The address and other information set up for the physician will default into the remaining fields (refer to Setting up Physician Information).
Address
The address of the physician.
City/State
The city of state of the physician
Zip
The physician’s zip code.
Phone
The physician’s phone number.
Fax
The physician’s fax number.
E-mail
The physician’s email address.
License
The physician’s license number.
Facility
The facility that the physician is associated with (refer to Setting up Facility Information).
F2-Save
Save the prescription information.
F4-Del
Deletes the prescription.
F5-Products
Opens the Customer Prescription Product Maintenance screen to allow you to enter the product and refill information for the prescription.
