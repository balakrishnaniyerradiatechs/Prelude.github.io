Creating Administered-By Codes
Administered-by codes may be required when you confirm a DD250 sales order (refer to Confirming a Release that Requires a DD250 Invoice). The administered-by code prints in block 10 on the DD250 invoice (refer to Printing Invoices Using the DD250 Form).
To create an administered-by code:
1
Select Order Processing > Maintenance > Misc Code Files > Administered By Maintenance to open the Administered By Maintenance screen.
2
For Admin By Number, enter the 1 to 6 alphanumeric code to use for sales orders set up to print DD250 invoices.
3
For Description, enter the name of the administered-by company or person.
4
Enter the address information for the administered-by company or person.
5
Press F2-Save to save the code.
