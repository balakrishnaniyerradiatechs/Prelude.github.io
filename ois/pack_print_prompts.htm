Printing Packing Lists by Batch
A packing list is similar to a pick ticket except that it shows confirmed pick quantities, substitute items, and freight charges. You can print the packing list after it is confirmed (refer to Confirming Releases) for just one release or for all shipments by customer and/or warehouse. If you do not continuously print packing lists (refer to Setting Packing List Maintenance Options), you must print the packing lists by batch.
To print packing lists by batch:
1
Select Order Processing > Batch Ticket Printing > Packing List Printing to open the Packing List Print screen.
2
Enter the Warehouse Number for which you want to print packing lists.
3
Enter the Release Number or leave blank to print all packing lists for the warehouse.
4
For Printer Form Queue, enter the printer number where the packing lists should print.
5
Choose from the following:
•
Print Packing Lists - Print the packing lists.
•
Mask Print Test - To test the alignment of the packing lists in the printer. Not necessary if you are using a laser printer. After the forms are correctly aligned, select Print Packing Lists.
•
Abort Printing - Cancel the packing list printing.
6
After you select Print Packing Lists, the following message appears:
Packing List Printing Complete...
7
Choose from the following:
•
Process - If all packing lists printed successfully.
•
Reprint All - If none of the packing lists printed successfully. This option is also useful if only a couple of packing lists printed successfully from a large print job.
•
Single Reprint - To reprint packing lists individually. This option is useful if a couple of packing lists jammed in the printer.
•
Abort - Cancels the packing list update.
8
If you select Process, the Process? screen appears.
9
Choose from the following:
•
Y - The packing lists printed successfully. The Order Log Inquiry is updated.
•
N - The packing lists did not print successfully. Return to step 6 to reprint the packing list.
To reprint a packing list that already printed, select Order Processing > Batch Ticket Printing > Packing List Reprinting.
