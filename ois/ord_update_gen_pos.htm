Generate P/Os for Backorders screen
If you select to generate a purchase order for a backorder on a sales order, the Generate P/Os for Backorders screen appears after you complete the sales order to list the purchase orders for each vendor you chose. You can generate purchase orders for one or more of the vendors listed. Refer to Linking a Purchase Order to a Backorder for more information.
When you enter a sales order product that is backordered, you can create a purchase order for the product if:
•
The Sc (supply code) is B, E, or D.
•
Generate P/O from S/O is set to Y for your user ID.
•
S/O-P/O, N/S Items Only is set to Y in Purchasing Option Parameters. You can only generate purchase orders for non-stock items.
Menu Path: Order Processing > Sales Order Entry > Generate a purchase order for a backorder
Field and Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor you selected to purchase the backorder from.
Description
The vendor description.
Whse
The warehouse to receive the material.
Type
The type of backorder:
•
B - Buyout.
•
D - Direct ship.
•
E - Emergency buyout
Line
The number of lines to create for the purchase order.
P/O#
The purchase order number generated which appears after you select one of the options to generate a purchase order.
Printed
Indicates if the purchase order printed which appears after you select one of the options to generate a purchase order.
F2-Generate All
Allows you to generate purchase orders for all vendors listed.
F5-Edit
Allows you to edit the highlighted purchase order.
F6-Generate
Allows you to generate the purchase order for the highlighted vendor.
F8-Review
Allows you to review the purchase order for the highlighted vendor.
F9-Continue
Allows you to exit the screen after the purchase orders are generated.
