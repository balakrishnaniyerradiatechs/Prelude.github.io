Point of Sale Report
If you use the Point of Sale Subsystem, the Point of Sale report displays cash sales by vendor and zip code. Before you run the report, you must run the Point of Sale Report Build to create the records.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Point of Sale > Point Of Sale Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Enter vendor number. The vendors to include on the report.
•
Enter product line. The product lines to include on the report.
•
Do you wish to report Price/Cost/Neither Both. Choose from the following:
-
P - Price.
-
C - Cost.
-
N - Neither price or cost.
-
B - Both price and cost.
•
Do you wish to report quantity?. Choose from the following:
-
Y - Display quantities.
-
N - Do not display quantities.
•
Do you wish to report extensions?. Choose from the following:
-
Y - If you choose to display prices or costs, the associated extensions display (e.g., Price x Quantity).
-
N - No price/cost extensions display.
•
Do you wish to report customers?. Choose from the following:
-
Y - List customers.
-
N - Do not list customers, only show product information.
•
Enter customer number. If you chose to list customers, enter the customer numbers.
•
Do you wish to report customer address?. If you chose to list customers, indicate if you want to include their ship-to addresses:
-
Y - Include addresses.
-
N - Exclude addresses.
•
Do you wish to report customer types?. If you chose to list customers, indicate if you want to display the customer type:
-
Y - Display customer type.
-
N - Do not display customer type.
Refer to the Customer Type Maintenance Screen.
•
Do you wish to sort by Price Group/Zip Code. Indicate how you want to sort the report:
-
P - Sort by price group.
-
Z - Sort by zip code.
-
N - Neither (sorts by customer number).
Column Descriptions
Column Heading
Description
Customer / Address / Type
The customer’s name and address. The customer type also displays, if you set up types to classify customers in the Customer Maintenance Additional Customer Information Screen. Type codes are established in the Customer Type Maintenance Screen to categorize customers.
Product Number
The product purchased by the customer.
Inv-Date
The date the order was invoiced.
Inv#
The invoice number.
Price
The price billed to the customer.
Cost
The product’s replacement cost.
Quantity
The quantity purchased by the customer.
Price Extension
The total price paid for the entire quantity (Price x Quantity).
Cost Extension
The total replacement cost for the quantity sold (Cost x Quantity).
Price Group
Price group assigned to the customer.
