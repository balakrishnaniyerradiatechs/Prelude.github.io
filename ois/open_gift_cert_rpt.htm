Open Gift Certificates Report
The Open Gift Certificates report lists open gift certificates with an open balance. For more information, refer to Gift Certificates.
Menu Path: Order Processing > Reports > Daily Order Detail Reports > Open Gift Certificates Report
Selection Criteria

Column Descriptions
Column Heading
Description
Certificate#
The gift certificate number.
Recipient Name
The name of the person that received the gift certificate.
Exp-Date
The date the gift certificate expires.
Orig Amt
The original value of the gift certificate.
Balance
The value remaining on the gift certificate.
