Point of Sale Report Build
The Point of Sale Report Build creates the records for the Point of Sale Report.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Point of Sale > Build Point Of Sale
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Fiscal Period Range Equal To. The beginning and ending fiscal periods to rebuild.
•
And Include Direct Ships?. Indicate if you want to include direct ship sales orders in the rebuild:
-
Y - Include direct ships.
-
N - Exclude direct ships.
•
And Include BOM Components?. Indicate if you want to include bill of material components in the rebuild:
-
Y - Include BOM components.
-
N - Exclude BOM components.
•
And Include Kit Components. Indicate if you want to include kit components in the rebuild:
-
Y - Include kit components and kit parents.
-
N - Only include kit parents. Exclude kit components.
-
O - Only include kit components. Exclude kit parents.
