Discount Table - Dollar screen
The Discount Table - Dollar screen appears when you enter a Table Method of $ in Discount Break Table Maintenance. You can enter a price for each break level.
Menu Path: Order Processing > Maintenance > Pricing Structures > Discount Break Table Maintenance > Table Method = $
You can also access this screen from the Inventory and Purchasing menus.
Field and Function Key Descriptions
Field/Function Key
Description
Break Level
Defines the minimum break value that qualifies for the corresponding price:
•
If the Break Level Type is M, the units of measure to determine the discounts.
•
If the Break Level Type is U, the minimum quantity values to determine the discounts.
•
If the Break Level Type is $, the minimum dollar amount to determine the discounts.
Price
The price for the corresponding Break Level. This value appears under Gross and Net-Cost on the P/O Detail Information screen when you re-cost an order.
F2-Save
Saves the break table information.
F4-Del
Deletes the break table.
