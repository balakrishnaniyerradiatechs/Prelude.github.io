Auto Consignment Billing Update screen
If the customer sells or decides to keep the material on an 08 consignment order, you can use the Auto Consignment Billing Update to set the existing quantity for invoicing. If you need to change any quantities before billing the order, refer to Manually Billing Consignment Orders.
Menu Path: Order Processing > Processes > Auto Consignment Billing Update
Field and Function Key Descriptions
Field/Function Key
Description
Consign Order #
The consignment order number to bill.
F2-Update
Marks all product lines on the consignment order as shipped and sets the order up for invoicing.
