Suspended Order Inquiry
The Suspended Order Inquiry displays suspended sales orders for a user ID. Suspended orders do not allocate inventory or print pick tickets.
Viewing Suspended Orders
To list the suspended orders:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Incomplete (Suspended) Order Inquiry to open the Suspended Orders Inquiry screen.
2
Enter a User Id, (defaults to your user ID).
Suspended orders appear.
Field & Function Key Descriptions
Field/Function Key
Description
Order#
The suspended order number.
N
An asterisk (*) indicates that notes exist for the sales order header.
Whse
The selling warehouse.
Customer
The customer number.
Description
The customer description.
AOI
AO indicates that the order is on Administrative Order Hold.
All
AI indicates that the order is on Administrative Invoice Hold.
C
H indicates that the order is on Credit Hold.
Ord-Date
The date the order was entered.
Req-Date
The date the customer requires the order.
S/O Dollars
The total value (selling price) of the order.
F2-Finish
Closes the inquiry screen.
F5-Review Order
Opens the Sales Order Display screen to display detailed information for the sales order.
F6-SOE
Opens the Sales Order Entry screen for the highlighted sales order, so that you can edit and/or finish the order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Output
Allows you to print, fax, email, or save (to an Excel spreadsheet) the order information.
