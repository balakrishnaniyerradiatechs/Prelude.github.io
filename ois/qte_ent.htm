Quote Order Entry screen
The Quote Order Entry screen allows you to enter products for which you want to quote price and availability information to a customer. Refer to Entering a Quote.
Menu Path: Order Processing > Additional Subsystems Quotation Subsystem > Quotation Entry
Field and Function Key Descriptions
Field/Function Key
Description
Quote Number
The number assigned to the quote.
Customer
The customer number and description.
Product Number
The product for which the customer would like a quote.
Description
The product description.
N
An asterisk (*) indicates that notes exist for the product in the Line Item Detail screen.
R
If the product is from Vendor Request for Quote Entry, a Q indicates that a RFQ (request for quote) was sent to the vendor. Refer to Creating a Vendor RFQ from a Quote.
AQI
If the quote is on Administrative Quote Hold, the reason code to indicate why the order went on Hold (refer to Defining Options for Administrative Quote Hold):
•
D - Override freight dollars.
•
F - Override freight terms.
•
I - Incompatible ship via.
•
J - Exceeded the quote limit for the Written By user ID that entered the quote.
•
L - All quotes for the customer go on Administrative Quote Hold (If Admin Hold Override is set to Q for the customer in the A/R Information Screen).
•
M - Manual hold.
•
I - Incompatible ship via.
•
N - Price overridden below minimum price.
•
V - Manager low level/high level limit GP%.
C
C indicates that line item on the quote was converted to a sales order.
Linkage
The document number if the product is linked to another order. The prefix indicates the document type:
•
P - Purchase order.
•
T - Transfer.
•
W - Work order.
Quantity
The quantity requested by the customer.
U/M
The unit of measure for the product.
Whse
The warehouse that will ship the material.
Price
The sell price quoted to the customer.
Bottom of the screen
Quantity
The total quantity of products on the quote.
Weight
The total weight of the products on the quote.
Profit
The total profit earned for the products quoted.
Gp%
The total gross profit percent earned for the products quoted.
Dollars
The total sales dollars quoted to the customer for the accumulative order.
F2-Save
Saves the information on the quote entry screen and allows you to choose one of the following:
•
Complete - Opens the Quotation Completion screen to allow you to add freight and extra charges (refer to Using the Sales Order Completion Screen for detail information on each field). You can print, email, or fax the quote to the customer (refer to Printing Quotes) but no sales order is generated. You can return to Quote Order Entry later to edit the quote or convert it into a sales order. If the quote is not eventually converted, it may be purged.
•
Order - Opens the Quote Conversion screen to allow you to convert the quote to a sales order. Refer to Converting a Quote to a Sales Order.
•
Suspend - Suspends the quote. You can use this option if the customer does not need a copy or confirmation. Or, if you are waiting for a response from a vendor on a vendor RFQ (refer to Creating a Vendor RFQ from a Quote).
•
Return - Allows you to return to the quote to edit or verify the products and prices.
F4-Del
Deletes the quote.
F5-Hdr
Opens the Quotation Header screen to allow you to view or edit header information for the quote.
F6-Total
Opens the Quote Totals screen to allow you to view the total quantity, weight, gross profit, and sales amounts on the quote.
F7-Log
Opens the Quote Log Inquiry to allow you to view every transaction that affected the quote.
F8-Order Pad
Opens the Order Pad Inquiry to allow you to use a keyword to search for products purchased by a customer. You can also add the selected products to the quote.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Examine
Allows you to view pricing and shipping information for the quote.
Review
Opens the Quotation Review screen to display detail availability and pricing information for all the products on the quote.
Prod Func
Provides access to the following product functions:
•
Add Comments Lines. Opens the Comment Description Entry screen to allow you to enter multiple comments for a line item.
•
Product Image. If you use the imaging software, you can scan images of documents related to the product.
•
Line Item Split. Opens the Order Line Split screen to allow you to split the quantity of an item for multiple delivery dates (refer to Splitting Line Items for Delivery Dates).
Quote Func
Provides access to the following sales order functions:
•
Resort Quote. Allows you to resort the line items on the quote by product number, description, or customer product number.
•
Reprice Quote Options. Refer to Re-Pricing an Entire Sales Order.
•
Manual Administrative Order Hold. Refer to Figure 15.4.2.
•
Campaign Priority Code. Allows you to view or change the campaign priority code assigned to the sales order. Refer to the Marketing documentation for more information.
Notes
Allows you to enter:
•
Header Notes. Notes regarding the entire sales order.
•
Line Notes. Notes regarding the highlighted line item.
•
Service Problem Notes. Problems notes on a service quote.
•
Service Solution Notes. Solution notes on a service quote.
•
Header Attachments. Allows you to attach a document or picture to the quote.
•
Line Attachments. Allows you to attach a document or picture to the line item.
