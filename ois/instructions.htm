Instructions Entry Screen
The Instructions Entry screen allows you to enter instructions to appear on sales orders, quotes, and pick tickets for the customer.
Automatically Displaying Instructions on Sales Orders and Quotes
Disp. Shp.Inst. In SOE/QOE in Order Processing Parameters determines if the instructions screen automatically displays when the header screen comes up in Sales Order Entry and Quotation Entry.
•
A - Always display the Instructions screen.
•
N - Never display the Instructions screen.
•
Y - Only display the Instructions screen if instructions exist in the Customer’s Instructions Entry screen.
Menu Path: Order Processing > Maintenance > Customer Maintenance > Action Bar > Add’l Opts > Instructions
Field and Function Key Descriptions
Field/Function Key
Description
Instructions
Up to 3 lines of default instructions to display on the Sales Order Header > F8 Instructions screen and appear on the pick ticket when you enter an order for this customer. You can override the instructions on the sales order. Press F3 to view standard instructions messages already set up to use on sales orders. Refer to Creating Standard Instructions
F2-Save
Saves the instruction information.
