Kit Entry screen
The Kit Entry screen displays the components and quantities that make up the kit parent. This screen appears when you enter a Type R (regular) or A (assorted) kit parent on a sales order and you choose to continue with kit entry (refer to Entering Kit Parent Products). You can change the components and quantities as necessary.
Menu Path: Order Processing > Sales Order Entry > enter a kit parent product
Field and Function Key Descriptions
Field/Function Key
Description
Kit Parent
The kit parent entered on the sales order.
Available
The quantity of the kit parent available to sell.
Product
The product components that make up the kit.
Description
The description of each product component.
Kit Qty
The quantity of each component product required to make the kit.
Quantity
The quantity of the component that is available.
U/M
The selling unit of measure for the kit component.
Price
The price of the kit component.
Linkage
The document number if the product is linked to another order. The prefix indicates the document type:
•
S - Sales order.
•
T - Transfer.
•
W - Work order.
Adm
Administrative Order Inquiry
F2-Save
Saves kit information.
F5-Recalc Kit Available
Re-calculates the number of kit parents available to sell.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Kit Entry
Examine
Displays price, gross profit, and date information for the highlighted product.
Review
Opens the Kit Review screen to allow you to view the products in the kit.
Prod Func
Opens the Comments Description Entry screen to allow you to enter a comment line after the highlighted product.
SumReprc
Sum & Reprice allows you to combine products in a defined group and give a discount if the total number of items purchased in the group meets a quantity break level.
Notes
Allows you to enter notes regarding the kit products.
