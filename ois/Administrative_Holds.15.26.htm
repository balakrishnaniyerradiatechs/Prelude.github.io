User Level Administrative Quote Hold Options
You can set restrictions by user ID to place an order on Administrative quote Hold for price overrides.
To define Administrative Quote Hold options for a user:
1
Select User ID Maintenance > F5-Administrative to open the Administrative settings screen.
2
For Quote Limit, enter the largest amount for which this user can enter a quote without it going on Administrative Quote Hold. If this field is left blank, the user can enter a quote for any value.
3
Press F2-Save to return to the User Identification Maintenance screen.
4
Press F6-Add Info to open the Additional Information screen.
5
For Price Override Code, determine if this user can override a price on a quote:
•
A - The user can change the price, even below cost.
•
C - The user can change the price, but not below cost.
•
G - The user can change the price, but if the gross profit falls below the minimum for the product line, the order or quote goes on hold.
•
N - The user cannot access any price fields.
•
R - The user can raise the price above the customer’s default price, but if he/she lowers it below the default price, the order or quote goes on hold.
6
Press F2-Save 2 times to save the price override information.
The Price Override Audit Report displays all overridden prices.
