Serial Shipping Container Inquiry
The Serial Shipping Container Inquiry displays serial shipping container information for a release number. Serial Shipping Container Codes (SSCC) are a unique identification system used to identify a carton or pallet. SSCCs provide quality assurance by assigning codes and printing labels to verify shipped products and quantities. Refer to Using Serial Shipping Container Codes for more information.
Viewing Serial Shipping Container Information
To display serial shipping container information for a release:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Serial Shipping Container Inquiry to open the Serial Shipping Container Inquiry screen.
2
For Document Type, choose from the following:
•
R - Release confirmation.
•
T - Transfer shipping confirmation.
•
V - Vendor claim confirmation.
3
For Status, indicate whether you want to view invoiced or open orders:
•
H - Historical (invoiced) orders.
•
O - Open orders.
4
For Document Number, enter the release number.
Serial shipping package information appears.
Field & Function Key Descriptions
Field/Function Key
Description
Package Number
The serial ship code assigned in Serial Shipping Container Entry (refer to Assigning Serial Shipping Container Codes).
Date
The date the release shipped.
User
The user ID of the person that packed the order.
C/P
Package type:
•
C - Carton.
•
P - Pallet.
F2-Continue
Returns to the Document Type field so you can view another document.
F5-Review
Allows you to view detail product information for the release number.
