Permit Information Screen
If you require permits from your customers for restricted products, you can enter and update the customer’s permit and restricted product information for this ship-to address. Refer to the Restricted Usage Code Maintenance Screen.
Menu Path: Order Processing > Maintenance > Customer Ship-to Maintenance > F5-Add Info > F5-Permit Info
Field and Function Key Descriptions
Field/Function Key
Description
State Permit Number
The state permit number for the ship-to address, if applicable. You can use this as a free-form field for custom reports.
Expire Date
The expiration date of the state permit.
Restricted Use Permit
Indicates if this ship-to address can purchase restricted products and which type of restricted products they can buy (defined in Product Maintenance>F5-Add Info > F6-MSDS/Permits > Restricted Product):
•
Y - The customer can purchase normal restricted products only (Restricted Product set to Y in Product Maintenance). The Usg Codes must match the usage codes for the product in the Restricted Usage Code Maintenance Screen.
•
M - The customer can purchase matrix and normal restricted products (Restricted Product set to M or Y in Product Maintenance). The customer must have a state permit for matrix-restricted products and the restriction level must be defined for each state usage code. If you enter a restricted product on an order, this flag must match the code for the product.
•
N - The customer may not purchase restricted products.
If the customer does not have a restricted product permit, you can add the permit information in Sales Order Entry which updates the customer’s record. Refer to “Assigning MSDS Numbers to Products” in the Inventory documentation for more information.
The MSDS/Ship-To Specific flag in Order Processing Parameters determines if this information is stored for a customer at the ship-to level or the bill to level.
Usg Codes
The usage code established in the Restricted Usage Code Maintenance Screen that defines the type of restricted product the customer is certified to buy:
•
If Restrict Use Permit =Y, enter the restricted usage codes. A restricted product must have the same usage code for the customer to be able to buy it.
•
If Restrict Use Permit = M, enter each state where this customer has a license to purchase matrix-restricted products. A sales order’s ship-to state must be defined here before the customer can buy a matrix-restricted product.
Permit Holder
The name of the person who is authorized to buy this type of restricted product for this customer ship-to address. If this name does not exist in this customer’s employee list, then you have the option to add it.
If Restrict Use Permit is set to M or Y, or if the Req Permit Holder flag in Restricted Usage Code Mnt is set to Y, then you must enter a permit holder before continuing.
Permit Number
The permit number that allows the customer to buy this product.
Exp Date
The date this restricted use product permit expires.
Level
If Restricted Use Permit is set to M, the level of restriction:
•
0 - No license.
•
1 - General.
•
2 - Restricted.
Address One
The first line of address for the permit holder.
Address Two
The second line of address for the permit holder.
City
The city of the permit holder.
State
The state of the permit holder.
Zip
The zip code.
F2-Save
Saves the permit information.
