Order Dates screen
The Order Dates screen allows you to change the order date for invoicing and aging purposes.
Menu Path: Order Processing > Sales Order Entry > F2-Save > Complete > F6-Dates
Field and Function Key Descriptions
Field/Function Key
Description
Invoice Number
The invoice number if the order is already invoiced.
Invoice Date
The date the order was invoiced.
Fiscal Period (unlabeled field)
The fiscal period in which the order was invoiced.
Ship Date
The date the order shipped.
Future Aging Date
 
# Of Cartons
The number of cartons included in the shipment.
Picked Up By
The name of the person picking up the order. The default is the Purchasing Agent assigned in the sales order header Order Additional Information Screen, but can be changed.
Additional COD Amount
The additional amount to collect on a COD order. For information only (unless interfaced with a third party manifest system).
F2-Save
Saves the order date information.
