Universal Customer Replacement /Merge Preview Report
After you enter the customers whose information will be merged (refer to Merging/Replacing Customer Accounts) you can run the Universal Customer Replacement /Merge Preview Report to see the To and From customer information that will be merged when you run the Universal Customer Replacement /Merge Update.
Menu Path: Order Processing > Maintenance > Universal Customer Replacement/Merge > Universal Customer Replacement/Merge Preview Report

Column Descriptions
Column Heading
Description
Old Customer
The number and description of the customer whose information is being copied/merged.
New Customer
The customer who will receive the information from the Old Customer.
Type
The type of replacement:
•
C - Copy. If you entered a new customer number for To Customer.
•
M - Merge. If you entered an existing customer number for To Customer.
