Manufacturer Rep Inquiry
This inquiry lists open manufacturer rep orders for a vendor, that are not updated in Manufacturer Rep Invoice Processing. You can view detail information for each order.
Menu Path: Order Processing > Additional Subsystems > Manufacturer Rep Subsystem > Manufacturer Rep Inquiries > Manufacturer Rep Inquiry
Field & Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor who has open manufacturer rep orders.
Order#
The manufacturer rep order number.
Ent-Date
The date the order was entered.
Customer
The customer that purchased the material from the vendor.
Description
The customer description.
Customer P/O
The customer’s purchase order number.
Comm Dollars
The amount of commission you can expect to receive from the vendor for this order.
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the M/R Display screen to allow you to view detail shipping and commission information for each item on each order.
F6-Enter Order
Allows you to enter a manufacturer rep order number for direct access to the order.
F9-Goto Line
Allows you to enter a reference number, page number or line number to go directly to.
