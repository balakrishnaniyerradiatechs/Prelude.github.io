Shipping Label Printing screen
If you need to attach shipping address labels to send shipments to your customer, you can use Shipping Label Printing to print labels for confirmed releases.
Menu Path: Order Processing > Batch Ticket Printing > Shipping Label Printing
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse Number
The warehouse where the labels print.
Sales Order Number
The sales order for which you want to print labels.
Number of Labels
The number of labels to print for the shipment.
Printer Form Queue
The printer number where the labels print.
