Identifying Maximum RX Months by State
If a state regulates the number of months a prescription is valid, you can set up the default number of valid months for each state in Maximum RX Month Maintenance.
To identify the number of months a prescription is valid for a state:
1
Select Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > Maximum RX Month Maintenance to open the Maximum RX Month Maintenance screen.
2
Enter the State abbreviation.
3
For #Months, enter the number of months a prescription is valid in the state.
4
Press F2-Save to save the information.
