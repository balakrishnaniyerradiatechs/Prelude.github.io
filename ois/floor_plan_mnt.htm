Floor Planning Maintenance Screen
If you use floor plan financing, you can use the Floor Planning Maintenance screen to enter the floor plan customer who is financially responsible for orders for this customer. Refer to Using Floor Plan Financing for more information.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F6-A/R > F8-Floor Plan
Field and Function Key Descriptions
Field/Function Key
Description
Financial Institution Floor Plan
Indicates if this customer uses floor plan financing:
•
Y - This customer uses financial floor planning. Any time you enter a sales order for this customer, you are asked if this is a floor plan order.
•
N - This customer does not use floor planning.
Floor Plan Customer
The customer responsible for payment, whose credit limit is checked, and whose A/R balance is updated when an order is entered for this customer. If you do not enter a floor plan customer number here, any customer can be used as the floor plan customer on the sales order header (refer to Entering a Floor Plan Sales Order).
Description
The description of the customer responsible for payment.
Plan Number/Approval Req
Determines if an approval and a floor plan number are required by the floor plan customer when this customer places an order:
•
Y - The Floor Plan Approval/Plan Number screen appears when you enter a sales order for this customer. You must enter the Floor Plan number (refer to Creating Floor Plan Numberst) to be able to exit the Sales Order Header Order Additional Information Screen.
•
N - A plan number and approval are not required.
F2-Save
Saves the floor plan information.
