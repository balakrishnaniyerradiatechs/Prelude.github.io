Commission Matrix Types & Sequence
To determine the commission matrix types to use and the sequence the system should follow to calculate commission:
1
In Order Processing Controls, select Commission Matrix Types & Sequences to open the Commission Matrix Types & Sequence Maintenance screen.
2
For Evaluate WHS types after DIR types, determine if the system should check for a warehouse type matrix for direct orders:
•
Y - Use a WHS type matrix if a DIR type matrix is not found.
•
N - Do not use a WHS matrix if a DIR matrix is not found.
3
For Comm Matrix Types, enter any of the following applicable matrix types in the sequence it should be used to calculate commission:
•
1 - Customer/Product.
•
2 - Customer/Vendor.
•
3 - Customer/Product Price Group.
•
4 - Customer Price Group/Product.
•
5 - Customer Price Group/Vendor.
•
6 - Customer Price Group/Product Price Group.
•
7 - Warehouse/Vendor.
•
8 - Warehouse/Product Price Group.
•
9 - Customer/Product Line.
•
10 - Customer/Major Group.
•
11 - Territory/Major Group.
•
12 - Customer Ship-To/Product Group.
•
13 - Territory/Product Line.
•
14 - Warehouse/Product Line.
•
15 - Warehouse/Route/Product Line.
•
16 - Product Line/Sales Rep.
•
17 - Product.
•
18 - Customer.
Refer to Creating a Commission Matrix for information on using matrix types in commission matrices. Any matrices not used should be removed from this table to help improve processing time.
4
Press F2-Save to save the matrix sequence information.
