Warehousing Information Screen
If you use the Warehousing Module, the Warehousing Information screen allows you to establish staging and picking information for a customer.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F7-Order > F5-Add Info > F7-Whsing
Field and Function Key Descriptions
Field/Function Key
Description
Sales Order Staging
Determines if sales orders for this customer are staged for picking:
•
Y - The customer’s orders are staged.
•
N - The customer’s orders are not staged.
Primary Case Picking
If you use Advanced Warehousing Bin Allocation, indicate if the customer requires primary full case picking in the bin allocation hierarchy:
•
Y - The customer requires full case picking.
•
N - The customer does not require full case picking.
Ord Batch Pick Method
If you use Advanced Warehousing and Batch Pick Ticket printing, indicate the method to use to pick orders for the customer:
•
P - Paper pick tickets.
•
R - Radio frequency guns.
F2-Save
Saves the warehousing information for the customer.
