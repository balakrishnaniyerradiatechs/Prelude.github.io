System Level Administrative Invoice Hold Options
You can set option parameters for your company to define the restrictions that place an order on Administrative Invoice Hold. Administrative hold options set at the customer level override the options set at the system level.
Order Processing Parameters
To define Administrative Invoice Hold order entry parameters:
1
Select Option Parameters > Order Processing Parameters > F5-Add Info to open the Additional Information screen.
2
For Administrative Invoice Hold, select Y to open the Administrative Invoice Hold Parameters screen.
3
Under AIH Types, enter one or more of the following codes to determine what criteria places an order on Administrative Invoice Hold:
•
A - Credit Card Order – No Authorization: If a credit card order is entered without an authorization code. Refer to Credit Cards.
•
P - Price Violations: If a price is changed to a value that is lower than the default price or lower than the cost. This only applies if Price Override Code for the User ID is set to C or R (refer to User Level Administrative Invoice Hold Options).
•
S - Salesrep Changed On Order: If the Salesrep One or Salesrep Two is changed on the Sales Order Header screen.
•
T - Order With No Tax Jur Codes: If an order does not contain Tax Jurisdiction Codes (set in the sales order header Tax Jurisdiction Information Screen)
•
U - User Defined.
4
For Exceed Preauth % Inv Hold, if you pre-authorize credit card orders, enter the percent a credit card invoice can be over the pre-authorized amount before it goes on hold.
5
Press F2-Save 3 times to save the Order Processing Parameter Information.
Purchasing Parameters
To define Administrative Invoice Hold purchasing parameters:
1
Select Option Parameters > Purchasing Parameters to open the Purchasing Parameters screen.
2
For Allow P/O Link For Inv Hold, determine if a sales order on Administrative Invoice hold can be linked to a purchase order.
•
Y - Allow sales orders on Administrative Invoice Hold to be linked to purchase orders.
•
N - Do not allow sales orders on Administrative Invoice Hold to be linked to purchase orders.
•
D - Allow only direct ship orders that are on Administrative Invoice Hold to be linked to purchase orders.
3
Press F2-Save to save the Purchasing Parameter information.
