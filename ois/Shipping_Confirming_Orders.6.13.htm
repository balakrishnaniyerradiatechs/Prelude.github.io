Maintenance Requirements for SSCCs
Set the following maintenance options to use serial shipping container codes:
Startup Parameter Maintenance
To set up printers for SSCCs:
1
In Startup Parameter Maintenance, select F5-Printers.
2
For SSCC Label Queue, enter the printer queue for the SSCC label printing.
3
For SSCC PC Label Queue, enter the printer queue for the SSCC PC label printing.
4
Press F2-Save twice to save the printer information.
Company Maintenance
To assign a UCC number to your company:
1
In Company Maintenance select F5-Add-Info to open the Additional Information screen.
2
For Ucc Mfg Number, enter the Uniform Code Council Manufacturer Number assigned to your registered organization by the Uniform Code Council. This code is part of the Serial Shipping Container Code identification number. This number is not used if a UCC Mfg Number is defined for the warehouse.
3
Press F2-Save 2 times to save the UCC information.
Customer Maintenance
To set up SSCC label information for a customer, refer to the Customer Labels Information Screen.
