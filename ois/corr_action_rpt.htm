Quality Corrective Action Report
The Quality Corrective Action report list collective action entries (refer to Quality Corrective Action Entry).
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Reports > Quality Corrective Action Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Corrective Action Number. The corrective action to view. Leave blank to view all action numbers that meet the remaining criteria.
•
Description. The corrective action description to limit the report to a description. Leave blank to include all descriptions.
•
Title. The corrective action entry title to limit the report to a specific title. Leave blank to include all titles.
•
Agent. The quality action agent assigned to the corrective action. Leave blank to include all agents.
•
Entry Date range. The beginning and ending corrective action entry dates to include.
•
User Id. The user ID number for which you want to view corrective action entries. Leave blank to include entries for all user IDs.
•
Status. The status of the feedback entry. Leave blank to include all feedback entries. Options include:
-
O - Open.
-
C - Closed.
-
I - In process.
-
U - Unassigned.
-
W - Waiting for response.
Column Descriptions
Column Heading
Description
CAction#
The corrective action number (refer to Quality Corrective Action Entry).
Incident
The incident number associated with the corrective action (refer to Quality Incident Entry).
Title
The title of the corrective action, if one exists.
Ent-Date
The date the corrective action was entered.
Agnt
The quality agent assigned to the problem.
Description
The agent description
Status:
The status of the case.
•
O - Open.
•
C - Closed.
•
I - In process.
•
U - Unassigned.
•
W - Waiting for response.
Notes
Any notes entered in Quality Corrective Action Entry.
