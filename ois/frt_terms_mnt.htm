Freight Terms Maintenance Screen
The Freight Terms Maintenance screen allows you to create and define freight terms codes that determine if and when a customer is to pay freight. You can assign a default freight terms code to each customer in the Customer Maintenance Freight Information Screen and in the Customer Ship-To Maintenance Freight Information Screen.
You can override a customer’s default freight terms on a sales order, but all orders require a freight terms code. If a customer’s freight terms require that they pay inbound and/or outbound freight and it was not included on the order, you can enter it in Release Confirmation Entry in the Sales Order Completion screen.
System Options for Freight Charges
Freight Handling (Order Processing Parameters > F5 > F5) determines how freight terms calculate freight in Sales Order Entry and Release Confirmation Entry.
•
Y - Skips the Inbound Freight and Outbound Freight fields in the Sales Order Completion screen and in Release Confirmation Entry if the freight terms code assigned to the sales order has Inbound Freight and Outbound Freight set to N. The freight charge is automatically calculated according to the freight terms.
•
N - Allows you to enter the Inbound Freight and Outbound Freight in both the Sales Order Completion screen and in Release Confirmation Entry, to override the system calculated freight no matter how the freight terms are set.
•
A - Works the same as N except that the freight amount entered in the release header is only carried to the completion screen if the freight terms codes has Inbound Freight and Outbound Freight set to Y. This option is used to allow you to enter freight without having to determine if the freight should be billed to the customer. The system calculates freight charges according to the freight terms.
Menu Path: Order Processing > Maintenance > Customer Control Files > Freight Terms
Field and Function Key Descriptions
Field/Function Key
Description
Freight Terms#
The 1 or 2 digit alpha-numeric code that identifies freight terms for a customer.
Description
The description of the freight terms.
Inbound Freight
Determines if you can enter inbound freight when confirming or invoicing an order, if freight was not already entered:
•
Y - Prompt to add inbound freight in Release Confirmation Entry.
•
N - Do not prompt to add inbound freight in Release Confirmation Entry.
Inbound Freight Cost
If Inbound Freight is set to N, the following options apply:
•
Y - Prompt for a freight amount during invoicing to add to commission cost.
•
C - Track freight for custom reports.
If Inbound Freight is set to Y, the following options apply:
•
N - Do not prompt for freight during invoicing.
•
C - Track freight for custom reports.
Outbound Freight
Determines if you are prompted to enter outbound freight when confirming or invoicing an order, if freight was not already entered:
•
Y - Prompt to add outbound freight in Release Confirmation Entry.
•
N - Do not prompt to add outbound freight in Release Confirmation Entry.
Outbound Freight Cost
If Outbound Freight is set to N, the following options apply:
•
Y - Prompt for a freight amount during invoicing, to add to commission cost.
•
C - Track freight for custom reports.
If Outbound Freight is set to Y, the following options apply:
•
N - Do not prompt for freight during invoicing.
•
C - Track freight for custom reports.
Post Freight With PN
Determines if freight is added to a sales order as a service product.
•
Y - Freight is added a service product, using the Freight Prod Number designated in Freight Prod Number. Outbound freight is updated.
•
N - Freight is not added as a service product.
Freight Prod Number
If Post Freight With PN is set to Y, you can enter the product number (from Product Maintenance). The price and cost of this service product applies to the order, depending on how Outbound Freight and Outbound Freight Cost are set.
PSS Collect Type
If you use PSS (Parcel Shipping System) to ship orders, determine if collection information from the customer’s account is passed to PSS:
•
Y - Pass collection information to PSS.
•
N - Do not pass collection information to PSS.
F2-Save
Saves the freight terms information.
F4-Del
Deletes the freight terms code.
