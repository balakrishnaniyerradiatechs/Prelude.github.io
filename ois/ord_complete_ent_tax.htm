Sales Tax Jurisdictions screen
The Sales Tax Jurisdictions screen allows you to view or edit tax exempt information for a sales order.
Menu Path: Order Processing > Sales Order Entry > F2-Save > Sales Order Completion Screen > F5-Tax Jur
Field and Function Key Descriptions
Field/Function Key
Description
Tax Juris
The tax jurisdiction code set up in the Tax Jurisdiction Maintenance Screen to calculate tax rates. The default comes from the Customer’s Tax Jurisdiction Maintenance Screen.
Description
The description of the tax jurisdiction (e.g., state, city, county).
Tax
Determines if this transaction in this jurisdiction is taxable:
•
Y - The transaction is taxable.
•
N - The transaction is not taxable.
Resale Type
If Tax is set to Y, this field indicates if only resale products sold to this customer in this tax jurisdiction are tax exempt:
•
R - Only products flagged as resale are tax exempt.
•
Blank - All products are tax exempt.
Exempt Number
The tax exemption number from the taxing authority, if the customer is tax exempt.
Taxable
The taxable amount of merchandise on this sales order.
Non Taxable
The non-taxable amount of merchandise on this sales order.
Tax Amount
The amount of tax charged for each corresponding jurisdiction.
F2-Save
Saves the tax jurisdiction information.
F5-Tax Flags
Opens the Sales Tax Jurisdictions screen to allow you to view or edit default tax flag settings for freight and handling charges.
