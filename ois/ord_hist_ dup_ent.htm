Order Duplication History Entry Screen
The Order Duplication History Entry screen allows you to duplicate an invoiced sales order.
Menu Path: Order Processing > Sales Order Entry > Action Bar > Dup Order Hist
Field and Function Key Descriptions
Field/Function Key
Description
Duplicate Order History#
The sales order number to duplicate.
Customer
The customer assigned to the order that you are duplicating.
Invoice#
The invoice number assigned to the order that you are duplicating.
Invoice Date
The date the order was invoiced.
Reprice Sales Order?
Indicates if you want to use current pricing for the customer or use the pricing from the original sales order. Choose from the following:
â€¢
Y - Use the current pricing set up for the customer for the products on the order.
â€¢
N - Use the same pricing that was entered on the original sales order.
New Customer?
The customer number if you are duplicating the order for a different customer. Leave blank if the duplication is for the same customer.
New Warehouse?
The warehouse number if you are duplicating the order for a different customer. Leave blank if the duplication is for the same warehouse.
F2-Save
Duplicates the order into a new order. You can verify and edit the header and line items as needed.
