Sales Tax Flags Entry screen
The Sales Tax Flags Entry screen identifies how the freight, handling, restocking, and other flags are set for a sales order.
Menu Path: Order Processing > Sales Order Entry > F2-Save > Sales Order Completion Screen > F5-Tax Jur > F5-Tax Flags
Field and Function Key Descriptions
Field/Function Key
Description
Tax Jur
The tax jurisdiction code set up in the Tax Jurisdiction Maintenance Screen to calculate tax rates. The default comes from the Customer’s Tax Jurisdiction Maintenance Screen.
Description
The description of the tax jurisdiction (e.g., state, city, county).
Inbound
Determines if inbound freight is taxable for this tax jurisdiction:
•
Y - Inbound freight is taxable.
•
N - Inbound freight is not taxable.
Outbound
Determines if outbound freight is taxable for this tax jurisdiction:
•
Y - Outbound freight is taxable.
•
N - Outbound freight is not taxable.
Handling
Determines if handling charges are taxable for this tax jurisdiction:
•
Y - Handling charges are taxable.
•
N - Handling charges are not taxable.
Min Charge
Determines if a minimum charge is taxable for this tax jurisdiction:
•
Y - A minimum charge is taxable.
•
N - A minimum charge is not taxable.
Restock
Determines if restocking charges are taxable for this tax jurisdiction:
•
Y - Restocking charges are taxable.
•
N - Restocking charges are not taxable.
Delivery
Determines if delivery charges are taxable for this tax jurisdiction:
•
Y - Delivery charges are taxable.
•
N - Delivery charges are not taxable.
Pick Up
Determines if pick up charges are taxable for this tax jurisdiction:
•
Y - Pick up charges are taxable.
•
N - Pick up charges are not taxable.
Labor
Determines if labor is taxable for this tax jurisdiction:
•
Y - Labor is taxable.
•
N - Labor is not taxable.
F2-Save
Saves the tax flag information.
