Customer Tax Certificates Report
The Customer Tax Certificates report lists tax exempt numbers, and expiration dates for a customer. This information is set for a customer in the Sales Tax Certificate Information Screen.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Tax Exemption Certificates Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Customer. The customer whose tax exempt information you want to view.
Column Descriptions
Column Heading
Description
Customer
The customer number.
St
The number or reference that points to the state.
Exempt Number
The sales tax exempt number for issued to the customer.
Type
The customer tax exempt type set up in Valid Response Maintenance.
IssDte
The date the tax exempt certificate was issued to the customer.
ExpDte
The date the tax exempt certificate expires.
AddDte
The date the tax exempt information was added for this customer.
