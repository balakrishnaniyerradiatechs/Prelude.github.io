Restocking Fee Ranges screen
The Restocking Fee Ranges screen allows you to set multiple restocking charges for a credit reason code depending on the number of days since the invoice date. For example, you can have a 5 percent restock charge for a product returned less than 30 days since the invoice date, a 10 percent restock charge for a product returned 30 to 60 days since the invoice date, etc.
Menu Path: Order Processing > Maintenance > Misc Code Files > Credit Reason Codes > F7-Restock
Field and Function Key Descriptions
Field/Function Key
Description
Number of Days Since Invoice Date For Restocking Percent
Beginning
The beginning number of days since the invoice date that incurs the corresponding Percent restock charge.
Ending
The ending number of days since the invoice date that incurs the corresponding Percent restock charge.
Percent
The percent restock charge for the corresponding range of days.
F2-Save
Saves the restock charge information.
