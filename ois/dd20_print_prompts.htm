DD250 Invoice Printing screen
You must use the DD250 Invoice Printing screen to print invoices that require a government DD250 form. Refer to Printing Invoices Using the DD250 Form for information on setting flags and entering information required for DD250 forms.
Menu Path: Order Processing > Invoice Preview & Printing > DD250 Invoice Printing
Field and Function Key Descriptions
Field/Function Key
Description
Sales Order Number
The order number for which you want to print a DD250 invoice. Leave blank to print invoices for all confirmed sales orders that are set up to print DD250 invoices.
Warehouse
The shipping warehouse numbers to include.
Invoice Date
The date to print on invoices and to calculate due dates.
Printer Form Queue
The printer where the invoices will print.
Invoice Cycle
The invoice print cycle for which you want to include invoices. Each customer is assigned an Invoice Print Cycle in the Customer Maintenance Additional Order Information Screen. Only orders for customers whose Invoice Print Cycle matches this selection are included with these invoices. Choose from the following:
•
D - Daily.
•
W - Weekly.
•
M - Monthly.
•
S - Semi-monthly.
Print Source
Indicates which types of invoices to include:
•
I - Invoices only.
•
C - Credit memos only.
•
B - Both invoices and credit memos.
•
M - Mail copies of counter invoices to customers whose Mail Invoices flag is set to Y in the Customer Maintenance Additional Order Information Screen.
F2-Save
Allows you to load the forms and print the invoices.
F5-Invoice Reprint
Allows you to re-print an invoice using either the invoice number or the original sales order number.
