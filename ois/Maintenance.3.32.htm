Creating Prescription Groups
Before you can create a prescription product (refer to Identifying a Product as a Prescription), you must first set up prescription groups to assign products to. You can use prescription groups to determine if a product requires a prescription if it is entered on a sales order.
You can also use prescription groups to assign a standard set of default instructions that applies to multiple prescriptions. For example, if a vet has a standard set of instructions for a cat medication, you can set up the instructions in RX Group Maintenance, then assign the RX group to a product (refer to Identifying a Product as a Prescription). When you create a prescription for a customer in Customer Prescription Maintenance (refer to Creating a Customer Prescription), these instructions default for the prescription.
To set up prescription groups:
1
Select Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > RX Group Maintenance to open the RX Group Maintenance screen.
2
For RX Group, enter a name or number for the group.
3
For RX Required, choose from the following:
•
A - All. Any product assigned to this group code (refer to Identifying a Product as a Prescription) will require a prescription number when it is entered on a sales order. If a product in this group is entered on a sales order that does not have a prescription number, it will go on Administrative Order Hold.
You must also set an Administrative Order Hold type of X (Refer to System Level Administrative Order Hold Options).
•
E - Excluded. A prescription is required for all products except for orders entered for customers assigned to the Customer Type indicated.
4
For Scientific Name, enter the full scientific name that pertains to the RX group. This information goes on the label. For example:
•
Group name: ECP.
•
Scientific name: Estradiol Cypionate 2mg/ml
5
If you entered E for RX Required, enter the customer types that do not require a prescription number when a sales order is entered for products in this group. Refer to Customer Type Maintenance Screen for information on setting up customer types.
6
Under Standard Instructions by Species, enter the Species for which you want to create instructions. You must enter a valid species as set up in Species Maintenance (refer to Creating Valid Species Responses).
The Notes Entry screen opens.
7
Enter any instructions that apply to this prescription group. This would be legal instructions that the vet has prescribed for the product.
Press F2-Save to return to the RX Group Maintenance screen.
An asterisk (*) appears under Instructions.
8
If you want to enter instructions from just one physician, enter the Physician number under Physician Specific Instructions.
9
Enter the Species for which you want to create instructions for this physician. You must enter a valid species as set up in Species Maintenance (refer to Creating Valid Species Responses).
The Notes Entry screen opens.
10
Enter any instructions that apply to this prescription group.
11
Press F2-Save to return to the RX Group Maintenance screen.
An asterisk (*) appears under Instructions.
12
Press F2-Save again to save the prescription group information.
