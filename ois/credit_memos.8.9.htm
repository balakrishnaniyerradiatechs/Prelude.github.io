Creating Warranty Tags for Returned Products
When a customer returns an item for which you will receive a credit or a replacement, you can enter a credit memo to issue credit to the customer and to create a warranty tag to designate the item as warranty inventory. A warranty tag tracks the inventory as it goes through Vendor Claim Entry, Vendor Claim Update Processing, and Warranty Replacement Entry. Refer to “Vendor Claims” in the Inventory documentation.
To create a warranty tag while issuing a credit memo or RGA:
1
In the Line Item Detail screen, for QtyWty, enter a negative number for the quantity to put into warranty inventory. The Warranty Tag Information screen appears.
2
For Returned/Scrapped, indicate where the material will go:
•
R - The material will be returned to the vendor.
•
S - The material will be scrapped.
In either case you will receive a credit or a replacement from the vendor.
3
For Vendor RMA #, enter the vendor’s return material authorization (RMA) number, if applicable.
4
For Tag Number, enter N for a new tag number or enter your own tag number.
5
For Tag Quantity, enter the defective quantity. You can enter more than one tag number per line item, but the total in the Tag Quantity column must equal the Quantity on the Line Item Credit Detail screen.
Vendor Return Rules
To view notes regarding return policies for this vendor, press F5-Vendor Return Policy.
6
Press F2-Save to save the tag information and return to the Line Item Credit Detail screen.
7
If necessary, change the pricing information from the original invoice, if you entered one. If you did not enter an original invoice, the net price comes from the pricing set up for the customer.
8
Press F2-Save to save the credit detail information.
9
After you enter all the line items and comments, press F2-Save to complete the credit memo or RGA.
You can now return the items to the vendor on a vendor claim. Refer to ”Entering a Vendor Claim For Warranty Inventory” in the Inventory documentation.
Comments
To enter comments to print on the credit memo, use the CV warranty tag comment code. When the credit memo is invoiced, the CV comment lines become the warranty tag notes. You can view these notes in Warranty Tag Inquires > F5-Review Tag > F7-Wty Tag Notes.
Warranty Tag Bins
If you use bins to track inventory, you can manually enter a warranty tag product in the warranty tag bin using the Warranty Tag Bin Maintenance screen (Inventory > Vendor Claims > Warranty Tag Bin Maintenance). The inventory in this bin does not affect available inventory but is only for reference in the Warranty Tag Inquiries in the Inventory module.
