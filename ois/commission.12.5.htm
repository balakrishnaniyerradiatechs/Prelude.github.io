Order Processing Parameters
To determine commission cost calculations and Written By commission information:
1
In Option Parameters, select Order Processing Parameters to open the Order Processing Parameters screen.
2
For Curr/Orig Cost for Comm, determine if current or original cost is used to calculate commission:
•
C - Use the cost of the product at the time it was invoiced.
•
O - Use the cost of the product at the time the order was entered.
3
For Commission Cost, define the cost to use to calculate gross profit for salesrep commissions:
•
A - Average cost.
•
S - Standard cost.
•
C - Cost For Sales. Opens the Commission W/Linkage Cost Options screen so you can determine if commission is calculated using the cost from a linked purchase order, regardless of the cost of sales.
•
R - Replacement cost.
•
U - Original accounting cost.
This cost can be overridden in Price Matrix Maintenance.
4
Press F5-Add Info to open the Additional Information screen.
5
For Written By Commission, determine if commission is also calculated for the sales order’s Written By (order taker):
•
Y - Calculate commission for the Written By in addition to Salesrep One and/or Salesrep Two.
•
N - Do not calculate commission for the Written By.
6
Press F2-Save 2 times to save the commission information.
