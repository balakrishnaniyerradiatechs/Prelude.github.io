Quote Log Inquiry
The Quote Log Inquiry displays every transaction that affected a quote. Each transaction is preceded by a Code that defines the change.
Viewing Quote Transactions
To list all transactions for a quote:
1
Select Order Processing > Additional Subsystems > Quotation Subsystem > Inquiries > Quotation Log Inquiry to open the Quote Log Inquiry screen.
2
Enter the Quote Number.
Transaction codes for the quote display.
Field & Function Key Descriptions
Field/Function Key
Description
Code
The quote log inquiry code that defines the transaction. Refer to the Quote Log Inquiry Codes for a definition of each code.
Customer#
The customer who requested the quote.
Act Date
The date that the transaction occurred.
Time
The time that the transaction occurred.
User
The user ID of the person that entered the transaction.
Reference Number
The associated order, product, ID, or other reference number associated with the transaction. Refer to the Reference column in the Quote Log Inquiry Codes for full descriptions.
F2-Finish
Closes the Quote Log Inquiry screen.
F5-Review
Opens the Quotation Display screen to display detail information.
F9-Goto Ln
Allows you to enter a line number, page number, or reference number.
