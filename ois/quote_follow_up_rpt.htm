Quote Follow Up Report
The Quote Follow-Up Report lists open quotations that will expire between the date you enter and the number of quote expiration days in Order Processing Parameters.
Menu Path: Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Follow-Up Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwest Region).
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Selling Warehouse. The warehouse that receives credit for the sale, regardless of which warehouse ships the merchandise.
•
Follow Up Date <. Include quotes that have a follow up date before this date. The Follow Up Date is entered in the Quotation Header screen to determine when to follow up on a quote if it is not yet converted or deleted.
•
Written By. The user ID of the person who entered the quote, if you want to limit the report to one or more users.
•
Salesrep. The number for the sales representative assigned to the quotes if you want to limit the report to one or more sales representatives.
•
Sort By W/B or Slsrep. Indicates how the report is sorted. Choose from the following:
-
S - Sort by sales representative.
-
W - Sort by written by.
Column Descriptions
Column Heading
Description
Cust#/Name
The customer number.
Description
The customer description.
Quote#
The quotation number.
Qte Date
The date the quote was entered.
Fu-Date
The Follow Up Date entered in the Quotation Header screen to determine when to follow up on a quote if it is not yet converted or deleted.
W/B
The written by user ID. The person that entered the quote.
Sell
The selling warehouse.
Contact
The customer’s purchasing contact name.
Phone
The customer’s phone number.
Dollars
The total value of the quote.
