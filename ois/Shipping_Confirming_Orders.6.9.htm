Setting Packing List Maintenance Options
Maintenance options determine when and where packing lists print for sales order releases.
Customer Maintenance
To set packing list information for a customer:
1
From Customer Maintenance, select F7 Order -> F5 Add Info to open the Additional Order Information screen.
2
For Packing List, determine if packing lists print for this customer. Choose from the following:
•
Y - Packing lists print for confirmed releases for this customer.
•
N - Packing lists do not print for releases.
3
Press F2-Save 3 times to save the packing list information.
Warehouse Maintenance
To set up packing list printing information for a warehouse:
1
From the Warehouse Maintenance screen, press F6-Printing to open the Continuous Printing Setup screen.
2
For S/O Pack List, determine if packing lists print, continuously or by batch. Choose from the following:
•
Y - Packing lists print when the release is confirmed in Release Confirmation Entry or Warehouse Release Confirmation Entry.
•
N - Packing lists print by batch (refer to Printing Packing Lists by Batch).
•
D - If you use multiple printers, enter the printer queue in Release Confirmation Entry or Warehouse Release Confirmation Entry.
3
For Prt, enter the port number (created in Continuous Printer Port Maintenance) used to control continuous printing.
4
For Form Queue, enter the printer number where packing lists print.
5
Press F2-Save twice to save the printer information.
