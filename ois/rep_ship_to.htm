M/R Ship-To Address screen
If you need to change a customer’s ship-to address on a Manufacturer’s Rep order, you can use the M/R Ship-To Address screen to enter a different shipping address.
Menu Path: Order Processing > Additional Subsystems > Manufacturers Rep Subsystem > Manufacturer Rep Entry > F5-Hdr Scr > F5-Ship-To
Field and Function Key Descriptions
Field/Function Key
Description
Ship-To Name
The name of the ship-to address.
Ship-To Address One
The first line of the ship-to address.
Ship-To Address Two
The second line of the ship-to address.
Ship-To Address Three
The third line of the ship-to address.
City
The city of the ship-to address.
State
The state of the ship-to address.
Zip
The zip code of the ship-to address.
F2-Save
The state of the ship-to address.
