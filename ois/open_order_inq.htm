Sales Order Inquiry
The Sales Order Inquiry lists all open (not invoiced) sales orders for a customer. You can access detail product and shipping information for each order.
Selecting Open Sales Orders
To list open sales orders for a customer:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Sales Order Inquiry to open the Sales Order Inquiry screen.
2
Enter the Customer.
3
Select the Sort sequence for the customer’s orders. Choose from the following to sort the list by:
•
None.
•
Order number.
•
Required date.
•
Customer purchase order number.
•
Order type.
•
Customer release number.
•
Sales order dollars.
•
Descending order number.
•
Descending order date.
•
Descending required date.
•
Descending customer purchase order number.
•
Descending order type.
•
Descending customer release number.
•
Descending sales order dollars.
4
If you want to use any Filters to limit the list of orders, choose from the following:
•
No. Lists all open sales orders for the customer.
•
Yes. Opens the Sales Order Inquiry Filter Options screen to allow you to use any combination of the following selection criteria to limit the list:
-
Required Date Range.
-
Quote Number.
-
Order Number.
-
Cust Release Number.
-
Purchasing Contact.
-
Customer P/O Number.
-
Job Number.
•
Customer P/O Number.
•
Customer Job Number.
•
Customer Release Number.
•
Order X/Ref.
•
Quote.
Open sales orders for the customer appear.
Field & Function Key Descriptions
Field/Function Key
Description
Order#
The customer’s open order number.
N
An asterisk (*) indicates that notes exists in the order header.
Type
The order type:
•
01 - Regular.
•
02 - Ship complete.
•
03 - Tag & hold.
•
04 - Direct ship.
•
05 - Counter invoice.
•
06 - Confirmation.
•
07 - Regular consignment.
•
08 - Confirmation consignment.
•
09 - Return goods.
•
10 - Repair order.
•
11 - Lease orders.
•
12 - Invoice when complete.
•
13 - Ship backorder complete.
•
14 - Ship line item complete.
•
15 - Show order.
•
16 - Order projection.
•
17 - Manufacturer’s representative.
•
18 - Crib replenishment.
Refer to Sales Order Types for a detailed explanation of each order type.
Excl
An asterisk (*) indicates if the order is excluded from the customer’s credit limit test. A Credit Manager can exclude a sales order from a customer’s credit limit check so that a pre-approved large order does not cause subsequent orders to go on credit hold. You must set Exclude From Credit Limit in Order Processing Parameters to Y to allow the credit manager from excluding an order (Refer to “Credit holds” in the Accounts Receivable documentation for more information).
Adm
Indicates if the order is on administrative hold:
•
AO - The order is on Administrative Order Hold. No pick ticket printed.
•
AI - The order is on Administrative Invoice Hold. No invoice printed.
•
AQ - The order is on Administrative Quote Hold. No quote printed.
•
Blank - The order is not on hold.
Refer to Administrative Hold Inquiries for information on holds.
Inv
Indicates the order’s invoicing status:
•
Y - The order is flagged for invoicing.
•
H - The order is on hold and cannot be invoiced until released.
•
Blank - The order is not yet flagged for invoicing.
Cred
Indicates if the order is on credit hold:
•
H - The order is on credit hold.
•
R - Released from hold.
•
Blank - The order is not on credit hold.
If the order is on hold, no pick ticket prints.
Ord-Date
The date the order was entered.
Late
An asterisk (*) indicates that the order was not shipped by the Required Date on the Sales Order Header screen.
Req-Date
The date the customer requires the order.
Customer P/O#
The customer’s purchase order number.
Release#
The customer’s purchase order release number, if applicable.
S/O Amount
The total value (selling price) of the sales order.
F2-Finish
Closes the Sales Order Inquiry screen.
F5-Review
Opens the Sales Order Display screen to allow you to review detail information for the highlighted order.
F6-Order#
Allows you to enter a sales order number to go directly to.
F7-Release
Opens the Open Release Inquiry. You can review printing, picking, and shipping information for the highlighted order if it has a release.
F8-Log
Opens the Order Log Inquiry. You can view each transaction that affected the order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Returns to the previous page if more than one page of sales orders are included in the inquiry.
sF6-Next Group
Proceeds to the next page if more than one page of sales orders are included in the inquiry.
Action Bar
Output
Allows you to print, fax, email, or save (to an Excel spreadsheet), the order information.
