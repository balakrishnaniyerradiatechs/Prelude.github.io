Insert Code Product Lines screen
The Insert Code Product Lines screen in Package Insert Code Maintenance allows you to give a customer a free product based on products purchased within a product line. Refer to Creating Package Inserts.
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Package Insert Code Maintenance > F6-Product Lines
Field and Function Key Descriptions
Field/Function Key
Description
Product Lines
The product lines that determine when the customer receives the free product. If you enter a value here, leave PLines Omitted blank.
PLines Omitted
If Product Lines is blank, you can identify the product lines for which you want to exclude the customer from receiving the free products. If the customer buys products in any product lines other than the ones listed here, they will still receive the free products.
Product Number
The package insert product to ship when the customer buys products within the designated product lines.
F2-Save
Saves the product line information and to returns to the Package Insert Code Maintenance screen.
