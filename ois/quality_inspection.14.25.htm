Quality Control Inspection Inquiry
The Quality Control Inspection Inquiry Displays Quality Control Products received into inventory waiting to be inspected. You can use Product Maintenance to designate the products that require inspection upon receipt (refer to Assigning Inspection Procedures for Products). You can view the products and/or enter inspection results from this inquiry (refer to Inspection Results Entry).
Releasing Products From the Quality Inspection Queue
To review and or record inspection results for products:
1
Select Inventory > Inquiries > Stock Receipt Inquiries > Quality Control Inspection to open the Quality Inspection Management Queue screen.
If you use the Quality Inspection Subsystem on the Order Processing module (Quality Inspection AQL flag set to Y in Inventory Parameters > F5-Add Info), the Quality Inspection Management Queue displays but the functionality is different.
2
Enter the Quality Inspection Manager ID.
Your user ID must be set up as an Inspection Manager to release the items into available inventory. Refer to Authorizing Users to Release Products for information on activating quality control inspections.
3
For Sort, choose the order in which you want to sort the products:
•
P - Priority. Sort products by priority so that with orders with linked backorders are listed first.
•
R - Receipt date. Sort products in order by receipt date.
Products awaiting inspection display.
4
For each Product listed, highlight the product and choose from the following:
•
F5-Review - Allows you to view the document associated with the product (e.g., the Purchase Order Display screen if the product was received on a purchase order).
•
F6-Perform Inspection - Opens the Inspection Results Entry screen to allow you to enter inspection results. Refer to Inspection Results Entry for information on entering inspections.
•
F7-Assign To Agent - Allows you to assign the product to a Quality Inspection Agent. Before you can assign any products to an agent, you must set Asg for the product to Y. The agent must be a valid agent set up in Quality Agent Maintenance (refer to Creating Quality Agent Codes). Once you assign the product to an agent, it no longer appears in the Quality Inspection Management Queue.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product awaiting inspection.
Whse
The warehouse where the product was received.
Description
The product description.
Doc
The type of document on which the product was received:
•
P - Purchase order.
•
W - Work Order.
Ln#
The line number from the associated document on which the products exists.
Document#
The document number for the product that requires inspection.
Quantity
The quantity of the product awaiting inspection.
Bin
The bin where the product is being held, if you use bins.
Req
The linkage requirements for the product that should be reviewed:
•
S - Linked to a sales order.
•
T - Linked to a transfer.
•
W - Linked to a work order.
•
V - Linked to a vendor claim.
Asg
Indicates if you want to assign the product to a quality inspection agent:
•
No - The product remains in the Quality Inspection Management queue.
•
Yes - The product will be assigned to a quality inspection agent once you press F7-Assign to Agent and enter the agent number.
F2-Finish
Closes the Quality Inspection Management Queue.
F5-Review
Opens the document number for the highlighted product, to allow you to view product information.
F6-Perform Insp
Opens the Inspection Results Entry screen to allow you to enter inspection result information. Refer to Inspection Results Entry
F7-Assign To Agent
Allows you to assign the product to a quality inspection agent if Asg is set to Y.
F8-Inv Inq
Opens the Inventory Inquiry screen to allow you to view the inventory status of the product.
