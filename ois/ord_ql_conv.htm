Quote Line Item Conversion screen
The Quote Line Item Conversion screen appears if you are in Sales Order Entry and you enter a product for which a quote already exists for the customer. Depending on system flags, you can convert the quote directly onto the sales order (refer to Converting Line Items from Quotes).
Quote Conversion Message Flags
Qte Line Conversion (Order Processing Parameters > F5 > F5) determines if a message appears in Sales Order Entry to ask if you want to convert existing quote line items for the customer onto the sales order.
•
Y - You are prompted to convert existing quote line items to the sales order. Display Open Quote/RGA Message (Order Processing Parameters) must be set to Y and Open Quote Message (Order Processing Parameters > F5) must be set to L or B for this flag to work.
•
N - A message appears to let you know that quotations exist but does not allow you to convert the quote onto the sales order.
Menu Path: Order Processing > Sales Order Entry > enter a product on a sales order for which a quote exists
Field and Function Key Descriptions
Field/Function Key
Description

The product number for which a quote already exists.

The package quantity of the product.
Quote#

The date the quote was entered.

Cust Po#

C
•
C - Converted.
•
Blank - Not converted.
Qty
The quantity quoted to the customer.
U/M
The unit of measure as quoted to the customer
Whse
The warehouse that will ship the material.
Net Price
The net price for the product as quoted to the customer.
Conv
Indicates if you want to convert the quote line item onto the sales order:
•
Y - Convert the quote line item to the sales order. The price and quantity are pulled from the quote. If the quote is on Administrative Quote Hold, it must be released before it can be converted.
•
N - Do not convert this quote.
Contact
The purchasing contact for the order.
F2-Save
Closes the screen:
•
If you chose Y for Conv, the line item appears on the Sales Order Entry screen with the quantity and price from the quote.
•
If you chose N for Conv, the Line Item Detail screen appears to allow you to enter quantity and price information.
F5-Review
Allows you to review the quote.
F6-Line Notes
Opens the Notes screen to allow you to view any line item notes entered for the line item on the quote.
F7-Header Notes
Opens the Notes screen to allow you to view any header notes entered for the quote.
Action Bar
Template
Opens the Product Characteristics screen. Product characteristics define special ordering requirements for a product. When you sell an item containing product characteristics, you must enter the characteristic information necessary to fill the order. These characteristics are then stored for the next time this customer buys this product.
Add’l Desc
Displays any additional description set up for the product, if applicable.
