Quotation Additional Information screen
The Quotation Additional Information screen displays default information for the customer from Customer Maintenance. You can change the information as necessary. This screen may automatically appear when you save the main header screen if any required information is missing for this customer.
Menu Path: Order Processing > Additional Subsystems Quotation Subsystem > Quotation Entry > F5-Hdr > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Fob Description
The freight on board description.
Frt Shipping Type
The freight shipping type of arrangement you have with this customer for orders shipped by UPS, FedEx, and RPS, if you use the Parcel Shipping System (PSS):
•
CBS - Consignee billing.
•
COL - Freight collect.
•
TPB - Third party billing.
The default is set in the customer’s Freight Information Screen.
Account# (unlabeled field)
The customer’s account number with the shipper, if applicable.
Floor Plan
Indicates if this customer uses Floor Plan Financing (i.e., another customer is responsible for payment):
•
Y - This customer uses another customer as a financial institution (refer to Creating Floor Plan Numbers).
•
N - This customer does not use floor plan financing.
Informational only and does not affect Sales Order Entry. The default for the customer is set up in the Floor Planning Maintenance Screen (Customer Maintenance > F6-A/R > F8-Floor Plan).
Floor Plan Customer Number (unlabeled field)
If you entered Y for Floor Plan, enter a floor plan customer who is responsible for payment, whose credit limit is checked, and whose A/R balance is updated for this sales order. The default for the customer is set up in the Floor Planning Maintenance Screen (Customer Maintenance > F6-A/R > F8-Floor Plan).
Specifier#
The customer number associated with the book for this sales order. This field is only accessible if you use the Book/Sample subsystem.
Add. Discount%
An additional discount percent that applies to each product sold to this customer. The default comes from the customer’s Additional Order Information Screen.
Chain Discount
Indicates if chain discounts exist for the customer:
•
Y - Opens the Chain Discounts screen to allow you to enter the chain discount percent and description. Chain discounts allow you to add up multiple discounts for a final price. For example, if you have chain discounts of 10% and 20%, the system multiplies the sales price by 9 and then takes that value and multiplies it by .8.
•
N - Chain discounts do not apply for this customer.
The default comes from the customer’s Additional Order Information Screen.
Price Pick Tick
Determines if prices should appear on picking tickets and packing lists for this sales order:
•
Y - Print prices.
•
N - Do not print prices.
The default is from the Additional Order Information Screen (Customer Maintenance > F7-Order > F5-Add Info).
Blind Shipment
Determines if the sales order is sent as a blind shipment:
•
Y - The sales order is sent as a blind shipment. Blind shipment packing lists print the customer information in the logo area. The bill-to and ship-to addresses are both updated to print the sales order’s ship-to address.
•
A - The same as Y except that when you enter a sales order for this customer, after you enter the customer number, a message appears to ask whether to send the order as a blind shipment.
•
N - Sales orders for this customer are not blind shipments. The company logo is included on the pick ticket and packing list.
A blind shipment means that the sales order will ship anonymously to this customer, without your company logo.
The default is from the Additional Order Information Screen (Customer Maintenance > F7-Order > F5-Add Info).
Deliver/Pick Up
Indicates if delivery and/or pick-up charges are billed to the customer:
•
D - Delivery charges.
•
P - Pick-up charges.
•
B - Both delivery and pick-up charges.
•
N - Neither delivery or pick-up charges.
This field is used primarily with the rental module when equipment is delivered or picked up. The default comes from the customer’s Additional Order Information Screen (Customer Maintenance > F7 > F5).
Language Indicator
The code that indicates which language appears on pick tickets, packing lists, quotes, and invoices for this customer. The default comes from the customer’s Order Information Screen or the Ship-To Additional Information Screen.
Charge Frt/Hnd Surchg
Indicates if additional freight and handling surcharges (calculated in Order Processing Parameters > F7-Add Frt/Handling) are added to this order:
•
Y - Freight surcharges apply.
•
N - Freight surcharges do not apply.
Two flags in Customer Maintenance (F7-Order > F7-Frt Info) determine if and when surcharges are added to a customer’s order:
•
Charge Frt/Hndlg Surcharge determines if surcharges are added.
•
Addl Frt/Handling Chg determines whether surcharges are added to Outbound Freight or Handling on the Sales Order Completion screen.
Refer to the customer’s Freight Information Screen for more information on calculating surcharges.
Pro Forma Quote
Indicates if you will use a pro-forma quote to print the quote:
•
Y - Use a pro-forma quote. A forms modification is required.
•
N - Use a regular quote.
F2-Save
Closes the screen.
