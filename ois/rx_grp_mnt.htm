RX Group Maintenance screen
If you have a standard set of instructions that applies to multiple prescriptions, you can set up prescription groups to define default instructions (refer to Creating Prescription Groups. For example, if a vet has a standard set of instructions for a cat medication, you can set up the instructions in RX Group Maintenance, then assign the RX group to a product (refer to Identifying a Product as a Prescription).
Menu Path: Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > RX Group Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
RX Group
The name or number for the group.
RX Required
Determines if a prescription is required:
•
A - All. Any product assigned to this group code (refer to Identifying a Product as a Prescription) will require a prescription number when it is entered on a sales order. If a product in this group is entered on a sales order that does not have a prescription number, it will go on Administrative Order Hold.
You must also set an Administrative Order Hold type of X (Refer to System Level Administrative Order Hold Options).
•
E - Excluded. A prescription is required for all products except for orders entered for customers assigned to the Customer Type indicated.
Scientific Name
The full scientific name that pertains to the RX group. This information goes on the label. For example:
•
Group name: ECP.
•
Scientific name: Estradiol Cypionate 2mg/ml.
Cust Type
If RX Required is set to E, enter the customer types that do not require a prescription number when a sales order is entered for products in this group. Refer to the Customer Type Maintenance Screen for information on setting up customer types.
Description
The description of the customer type.
Standard Instructions by Species
Species
The Species for which you want to create instructions. You must enter a valid species as set up in Species Maintenance (refer to Creating Valid Species Responses).
Instructions
An asterisk (*) indicates that instructions exist for the this RX Group/Species. To view or edit instructions, enter Y in this field.
Physician Specific Instructions
Physician
The physician number if this group number refers to standard instructions for a physician for a species.
Species
The species that this physician has instructions for.
Instructions
An asterisk (*) indicates that instructions exist for the this Physician/ RX Group/Species. To view or edit instructions, enter Y in this field.
F2-Save
Saves the prescription group information.
F4-Del
Deletes the prescription group.
F5-Copy
