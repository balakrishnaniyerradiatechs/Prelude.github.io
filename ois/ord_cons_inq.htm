Consignment Order History Inquiry
The Consignment Order History Inquiry screen displays invoiced, 08 consigned orders for a customer. Consignment orders contain products that are on loan to the customer. If the customer sells the product, the customer is invoiced for the products sold. If the customer cannot sell the products, they are returned back into inventory. Refer to Using Consignment Sales Orders for more information.
Viewing Invoiced Consignment Orders
To list the consigned orders:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Consignment Order History Inquiry to open the Consignment Order History Inquiry screen.
2
Enter the Customer.
Invoiced, consignment orders appear.
Field & Function Key Descriptions
Field/Function Key
Description
Order #
The consignment order number.
N
An asterisk (*) indicates that notes exist in the order header.
Invoice#
The invoice number associated with the sales order.
Ord-Date
The date the order was entered.
Inv-Date
The date the order was invoiced.
Customer P/O
The customer’s purchase order number.
Release Number
The customer’s purchase order release number, if applicable.
Order Amount
The total value of the sales order.
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the Consignment Order History Inquiry screen screen to display detail information for the consigned order.
F6-Enter
Allows you to enter a consignment order number for direct access.
F7-Invoice#
Allows you to enter a consignment invoice number for direct access.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Lists the previous group of orders, if applicable.
sF6-Next Group
Lists the next group of orders, if applicable.
