Setting Up Customer Product Numbers
If a customer wants to use their own part numbers when they place an order, you can use Customer Product Number Maintenance to reference customer part numbers to your own product numbers. If you enter the customer part number on a sales order, your own part number automatically appears. If several customers use the same part numbers to reference products (e.g., divisions of a large company), you can create the part numbers for the parent customer and then share the information with other customers.
You can also use Customer Product Number Maintenance to set up customer part numbers for a customer price group so that the products may be used for all customers assigned to the same Price Group in the Customer Maintenance Order Information Screen.
To set up customer part numbers for a customer or customer price group:
1
Select Order Processing > Maintenance > Order Entry Control Files > Customer Product Numbers to open the Customer Product Number Maintenance screen.
2
For Customer/Cust Prc Grp, choose from the following:
•
Enter a customer number if you want to enter a product number based on a customer. To share this customer part number with other customers, enter this parent customer number in the CPN Customer Number field in the Order Information Screen (Customer Maintenance > F7-Order).
•
Enter a customer price group if you want to set up a product number by price group. Price groups allow you to assign the same pricing, prices matrices, or discounts to a group of customers. You can create price groups in the Customer Price Group Maintenance Screen and then assign one or more price group to each customer in the Customer Maintenance Order Information Screen.
3
For Cust Product Number, enter the customer’s part number that corresponds to your system part number. When you enter this number on a sales order, your corresponding Product number automatically appears.
4
For Product, enter your system product number that corresponds to the customer’s part number.
5
For Description One and Description Two, enter the first and second line of description for the product if they are different than what is in Product Maintenance
6
For Code Number One, enter any special code that may apply to this customer’s product number. This information can be used on reports, pick tickets, or invoices.
7
For Code Number Two, enter any additional codes, if applicable.
8
If the customer requires an accounting code (Customer Maintenance > F7-Order > Require Acct Code = Y), enter the Accounting Code for this customer part number. This code is the default for Customer Acct Code on the Line Item Detail Additional Information screen when the product is entered on a sales order. Refer to the Acct Code Pattern Match Screen for more information on accounting codes.
9
For Resale Exemption, indicate if the customer product number is considered tax exempt:
•
Y - The customer part number is tax exempt.
•
N - The customer part number is taxable.
10
You can enter the Package Qty of the item if it is different than what is in Product Maintenance and if you want it to appear on reports.
11
Press F2-Save to save the customer product number information.
