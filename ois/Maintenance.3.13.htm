Creating Discount Reason Codes
If you enter a discount amount or percentage in Disc/Allowance in the Sales Order Completion screen or the Quotation Completion screen, a Discount Reason Code may be required. You can set up the discount reasons in Discount Reason Code Maintenance.
Order Processing Parameters
Prompt for Disc.Allow Reas (Order Processing Parameters > F5-Add Info > F5-Add Info) must be set to Y to require a discount reason code when a discount is entered in the Sales Order Completion screen.
To create a discount reason code:
1
Select Order Processing > Maintenance > Misc Code Files > Discount Reason Code Maintenance to open the Discount Reason Code Maintenance screen.
2
For Discount Reason#, enter a code to represent the reason for the discount.
3
Enter a Description of the discount reason (e.g., promotion, manager approved).
4
Press F2-Save to save the reason code information.
