Duplicating a Sales Order for Multiple Addresses
You can use Sales Order Duplication to duplicate an order for multiple ship-to addresses for a customer, if all orders are shipped from the same warehouse. The ship-to addresses must already be set up for the customer before you can select one for this process. Refer to Customer Ship-To Maintenance.
To duplicate an order for multiple ship-to locations:
1
Select Order Processing > Optional Entry and Printing > Sales Order Duplication to open the Sales Order Duplication screen.
2
For (Ord/Hist) File, indicate if you are duplicating an open or an invoiced order. Choose from the following:
•
O - Open order.
•
H - Invoiced (historical) order.
3
For Source Order#, enter the order number to duplicate. The Customer associated with the order displays.
4
For Ship To (M)(A)(S), choose from the following to determine how to select ship-to locations:
•
M - Manually enter each valid ship-to number for the customer.
•
A - Use all ship-to addresses available for the customer.
•
S - Display all ship-to addresses for the customer and select the one(s) you want to include.
5
Press F2-Save to duplicate the sales order for the new ship-to addresses. If not all orders can be filled with available inventory, the following message appears:
Warning! Insufficient inventory for all orders to be created. Do you wish to?
6
Choose from the following:
•
Process w/Backorders. Creates the sales orders for all the ship-to locations and backorders the un-available products.
•
Return to Screen. Allows you to return to the Sales Order Duplication screen to edit the list of ship-to locations.
•
Abort. No new orders are created.
