Customer Level Administrative Order Hold Options
You can set up restrictions for a customer to determine when their orders should go on Administrative Order Hold.
To set Administrative Order Hold options for a customer:
1
Select Order Processing > Maintenance > Customer Maintenance to open the Customer Maintenance Screen.
2
Enter the Customer Number.
3
Select F6-A/R to open the A/R Information Screen.
4
For Admin Hold Override, choose from the following:
•
B - Always place sales orders for this customer on both Administrative Invoice and Administrative Order Hold.
•
I - Always place sales orders for this customer on Administrative Invoice Hold.
•
N - Never place an order for this customer on hold. You can override this option using Manual Administrative Order Hold while in Sales Order Entry. Refer to Manually Placing Orders on Administrative Order Hold.
•
O - Always place sales orders and quotes for this customer on hold.
•
Q - Always place quotes for this customer on hold.
5
For Admin Order Mgr I.D., enter a manager ID if you want to override the manager ID set up at the system and warehouse level. All orders for this customer go to this manager’s work queue.
6
Press F2-Save to save any changes and to return to the Customer Maintenance screen.
7
Select F7-Order to open the Order Information screen.
8
For Require P/O Number, choose from the following:
•
Y - The customer requires a purchase order. Opens the Customer Blanket P/O Numbers Screen. You can enter blanket purchase order information for this customer. You can also establish purchase order patterns that must match the pattern on sales orders for the customer.
•
N - Purchase orders are not required for this customer.
•
O - Places an order with no purchase order on Administrative Order Hold.
•
I - Opens the Customer Blanket P/O Numbers Screen. You can set an Administrative Invoice Hold limit for sales orders.
9
Select F8-Inc Ship to open the Incompatible Ship Via Maintenance screen.
10
For Via, enter the ship via codes that represent carriers that cannot be used to ship to this customer. If you enter an order for this customer using one of these ship-via codes, the order will go on Administrative Order hold. This applies whether the incompatible ship via is entered at the header level or at the line item level.
11
Press F2-Save three times to save any changes to the customer’s record.
