Customer Level Administrative Quote Hold Options
You can set up restrictions for a customer to determine if/when their quotes go on Administrative Quote Hold.
To set Administrative Quote Hold options for a customer:
1
Select Order Processing > Maintenance > Customer Maintenance.
2
Enter the Customer Number.
3
Select F6-A/R to open the A/R Information screen.
4
For Admin Hold Override, choose from the following:
•
B - Always place sales orders for this customer on both Administrative Invoice and Administrative Order Hold.
•
I - Always place sales orders for this customer on Administrative Invoice Hold.
•
N - Never place an order for this customer on hold. You can override this option using Manual Admin Order Hold while in Sales Order Entry. Refer to Manually Placing Orders on Administrative Order Hold.
•
O - Always place sales orders and quotes for this customer on hold.
•
Q - Always place quotes for this customer on hold.
5
For Admin Qte Mgr Id., enter a manager ID if you want to override the manager ID set up at the system level. All quotes for this customer will go to this manager’s work queue.
6
Press F2-Save to save any changes and to return to the Customer Maintenance screen.
7
Select F7-Order to open the Order Information screen.
8
To set up incompatible ship via codes, select F7-Order > F8-Inc Ship to open the Incompatible Ship Via Maintenance Screen.
9
For Via, enter the ship via codes that represent carriers that cannot be used to ship to this customer. If you enter an order for this customer using one of these ship-via codes, the order will go on Administrative Quote hold. This applies whether the incompatible ship via is entered at the header level or at the line item level.
10
Press F2-Save three times to save any changes to the Customer’s record.
