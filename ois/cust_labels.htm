Customer Labels Information Screen
If you print retail price labels for this customer, you can use the Customer Labels Information screen to print the labels and then charge them either per label or by a percentage of sales. You can also set up SSCCs (serial shipping container codes) to print labels that list the detailed content information for each order shipped to the customer (refer to Assigning Serial Shipping Container Codes).
Only products that have Print Label (Product Maintenance > F5-Add Info > F7-Labels) set to Y will print labels. Refer to “Product Labels Information” in the Inventory documentation.
Menu Path: Order Processing > Maintenance > Customer Maintenance > Action Bar > Add’l Opts > Label information
Field and Function Key Descriptions
Field/Function Key
Description
Print Price Label
Determines if price labels print for a customer and how the customer is charged:
•
Y - Print price labels for the customer. Opens the Price Label Information screen to allow you to determine how to charge the customer for the labels and how to format the labels. Refer to Price Labels for more information.
•
N - Do not print price labels for the customer.
Contents Tracking
Determines if you use Serial Shipping Container Tracking for all orders shipped to this customer:
•
Y - Every pick ticket release includes a detail breakdown of the contents of the carton or pallet. The release is not available for confirmation until all products are tracked under an SSCC. Refer to Using Serial Shipping Container Codes.
•
N - The customer does not require Serial Shipping Container Tracking.
Ucc Mfg Number
The customer’s UCC (Uniform Council Code) used in the SSCC code. This number overrides the UCC Manufacturer Number at the company level.
SSCC Label Program
The name of your custom program that prints SSCC labels. The labels include a detail listing of the contents of the shipment. Refer to Using Serial Shipping Container Codes.
SSCC Label Execution
Determines when the labels print:
•
N - Labels print when the SSCC is assigned in Serial Container Entry.
•
E - Labels print after each SSCC entry.
•
F - Labels print after all the SSCCs for a release number are entered.
PC Label Program
The name of your custom program that prints SSCC labels. The labels include a detail listing of the contents of the shipment.
PC Label Execution
Determines when the labels print:
•
E - Labels print after each SSCC entry.
•
F - Labels print after all the SSCCs for a release number are entered.
F2-Save
Saves the label information.
