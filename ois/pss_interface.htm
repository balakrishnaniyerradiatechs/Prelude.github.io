PSS Manifest Interface Screen
If you use the Parcel Shipping System (PSS), you can enter shipping cross reference information in the PSS Manifest Interface screen.
Prelude no longer supports the PSS Manifest.
Menu Path: Order Processing > Maintenance > Customer Control Files > Ship Via Codes > Action Bar > Addt Optns > PSS Manifest Interface
Field and Function Key Descriptions
Field/Function Key
Description
PSS Ship Via Code
The Parcel Shipping System ship via code that corresponds with this ship via code. This information is transmitted to PSS when the order is entered on the Shipment Rating Station Screen.
LTL Carrier
Determines if this carrier is an LTL (less than truckload) carrier:
•
Y - The carrier will ship an order even if it is not a full truckload.
•
N - The carrier will not ship an order that less than a full truckload.
Saturday Delivery
Indicates if the ship via type includes Saturday delivery:
•
Y - The ship via code includes Saturday delivery. An additional charge may apply.
•
N - The ship via code does not include Saturday delivery.
Early AM Delivery
Indicates if the ship via type represents an early morning delivery:
•
Y - The ship via code includes early morning deliveries. An additional charge may apply.
•
N - The ship via code does not include early morning delivery.
Hazardous Material
Indicates if the ship via is authorized to handle hazardous material:
•
Y - The ship via code is compatible for hazardous material. An additional charge may apply.
•
N - The ship via code is not for an hazardous material.
Non-Rated Courier#
The non-rated courier number assigned in PSS that refers to this ship via (i.e., the LTL Carrier number assigned to his ship via within the PSS configuration setup.).
Residence/
Commercial
Indicates the type of address:
•
C - Commercial.
•
R - Residential.
Charge Customer For
The rate that applies for this ship via code. For example, if you ship an order UPS Red, but you want to only charger for UPS Ground.
List Rate Adjustment
Allows you to add a percent to apply a markup or discount to freight for sales orders and shop repair orders:
•
Markup - Add a positive percent.
•
Discount - Add a negative percent.
