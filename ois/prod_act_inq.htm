Accessory Inquiry
The Accessory Inquiry lists accessory products that you can sell to go with another product. When you sell an item that has accessory products, you can view the list of accessories and select the ones you want to add to the order (refer to Adding Accessory Products to an Order).
Once accessory products are assigned to a product (in Product Maintenance > F6-Xref/notes), any time you enter the product on a transaction or in an inquiry, ACC displays to let you know that accessories are available. If accessory groups are set up (refer to Accessory Groups in the Inventory documentation), you can assign an accessory group so that an entire list of products display any time you enter the original product on a sales order.
Viewing Accessories for a Product
To list the accessory products assigned to a product:
1
Select Order Processing > Inquiries > Accessory Inquiry to open the Accessory Inquiry screen.
2
Enter the Product.
Any accessories assigned to the product appear.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number. The product description appears below.
Accessory Product
The products that can be sold to go with the original product.
Description
The description of the accessory product.
OK
Closes the inquiry.
