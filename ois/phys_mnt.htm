RX Physician Maintenance screen
Each physician that you use when setting up customer prescriptions, must be set up with a physician number in RX Physician Maintenance (refer to Setting up Physician Information).
Menu Path: Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > RX Physician Maintenance
Field and Function Key Descriptions

Physician
The number to assign to the physician. Enter NEW to assign the next system generated number.
Valid numbers for physicians for each warehouse must be set up in Physician Number Range Setup (Order Processing > Maintenance > Customer Control Files, Physician/RX Maintenance).

The number for the facility associated with this physician. Facilities group physicians that work at the same hospital or location (refer to Setting up Facility Information. If you enter a facility number, you can let the default address and phone number information from the facility update the address and phone number for this physician.
