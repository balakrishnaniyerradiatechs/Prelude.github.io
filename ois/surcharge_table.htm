Freight Surcharge Table Screen
If you want to make a profit on freight charges, you can use the Freight Surcharge Table to assign a freight surcharge to orders using this ship via code. This table uses the information from the Addl Frt/Handling Chg and Charge Frt/Hndlg Surcharge fields in the customer’s Freight Information Screen.
Menu Path: Order Processing > Maintenance > Customer Control Files > Ship Via Codes > F7-Surch Tbl
Field and Function Key Descriptions
Field/Function Key
Description
Freight $
The maximum freight amount that incurs the corresponding Surcharge %.
Surcharge %
This percent is multiplied by the freight amount on the sales order to calculate the additional freight/handling surcharge. Two flags in Customer Maintenance (F7-Order > F7-Frt Info) determine if and when surcharges are added to a customer’s order:
•
Charge Frt/Hndlg Surcharge determines if surcharges are added.
•
Addl Frt/Handling Chg determines whether surcharges are added to Outbound Freight or Handling on the Sales Order Completion screen.
Refer to the customer Freight Information Screen for more information on calculating surcharges.
F2-Save
Saves the freight surcharge information.
