Sourcing a Backorder to an Existing Purchase Order
If you enter an unavailable stock product on a sales order, you can use the Allocate Backorders to Existing P/Os screen to see if the product is on a purchase order you can source from. If you choose to link the backorder to the purchase order (and the product is not already linked to another sales order, transfer, or work order) when the backorder arrives in your warehouse, it automatically allocates to your customer’s order. You must source the whole backorder quantity.
Sourcing Options
Purchasing Parameters
Multiple P/O Sourcing in Option Parameters > Purchasing Parameters determines if a multiple sales orders can be linked to a purchase order line or if one-to-one linkage is required:
•
Y - A single purchase order line can be linked to multiple sales orders. This method is referred to as the Multiple Document Sources Method. The Purchase Order Display screen does not list the linked sales orders but displays the word Multiple for the linked product.
•
N - A purchase order line item can only be linked to one sales order. This method is referred to as the Split Sourced Quantity Method, meaning the quantity allocated to the sales order must be split out on a separate purchase order line. The Purchase Order Display screen lists the linked sales order number for each line with a sourced quantity.
User ID Maintenance
Source P/O from S/O in User ID Maintenance > F6-Add Info determines if your user ID allows you to link a sales order line item to an existing purchase order:
•
Y - You can link a sales order line item to an open purchase order.
•
N - You cannot link a sales order line item to an open purchase order.
To link a backorder on a sales order to an existing purchase order:
1
From the Line Item Detail screen (refer to Entering Stock Products), press F8-Source P/O to open the Allocate Backorders To Existing P/O’s screen. Open purchase orders that contain the backordered product appear.
2
Select the P/O Number that you want to source the backorder from and enter the quantity to source under Linked.
3
For New P/O#, choose one of the following to indicate if you want to split the linked quantity off the original purchase order onto a new or existing purchase order:
•
To split a quantity onto an existing purchase order, enter the purchase order number.
•
To split a quantity onto a new purchase order, enter NEW for the next available purchase order number. Refer to Linking a Purchase Order to a Backorder.
•
Leave blank to keep the product assigned to the original purchase order.
4
Press F2-Save to return to the Line Item Detail screen and to continue entering line items. Refer to Entering Sales Order Line Items.
When the purchase order is received, the backorders are allocated to the linked sale orders until all backorders are filled. The remaining quantity goes into stock.
