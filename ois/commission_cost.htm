Commission Cost Entry screen
The Commission Cost Entry screen allows you to override a commission cost for a product on a sales order. This commission cost overrides any commission cost defined in the Purchase Cost screen in Price Matrix Maintenance (refer to Defining Purchasing Cost).
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Maintenance > F6-Commission Cost
Field and Function Key Descriptions
Field/Function Key
Description
Cost Method
Determines how purchase cost is calculated for line items that meet this matrix criteria:
•
* - Use a percentage of a price/cost field.
•
$ - Price/Cost. Use only if the matrix is for a specific product.
•
T - Use a discount break table (refer to Discount Break Tables). The system determines a cost based upon the discount level.
Cost
If you chose $ for Cost Method, enter the cost.
Basis
If you chose * for Cost Method, enter the price or cost column (from Product Maintenance > F7-Pricing) to use as the base amount to calculate the discount or markup.
Percent
If you choose * for Cost Method and entered a price column in Basis, enter the percentage to take off of the price. If you entered cost column in Basis, enter a percentage over the cost to calculate the purchase order cost.
Table
If you chose T for Cost Method, enter the discount break table number.
F2-Save
Saves the commission cost information.
