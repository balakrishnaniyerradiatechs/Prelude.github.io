Blanket Contract Detail screen
The Blanket Contract Detail screen allows you to enter detail price and/or rebate information for a product.
Menu Path: Order Processing > Maintenance > Pricing Structures > Blanket Contract Maintenance > Blanket Contract Maintenance > enter a Product
Field and Function Key Descriptions
Field/Function Key
Description
Ln#
The line number of the blanket contract.
Customer Product
If applicable, the customer’s part number for your part number (refer to Setting Up Customer Product Numbers for information on setting up customer part numbers). If the customer part number that you enter is new, cross reference information but not the price matrix is updated.
Whse
The Supplying Warehouse entered in the Header Information screen.
U/M
The unit of measure of the product as entered on the contract.
Cln#
The cross reference line number. Only used when you first go live on Prelude and cross-reference old contract information.
Ent-Date
The date the blanket contract is entered.
Price
Mtd
The method to assign a price to the product:
•
$ - Allows you to enter a set price for the product.
•
* - Calculates the price as a percent of a price or a multiplier over a cost.
•
A - Allows you to add a discount to a product in addition to the discount set up for the customer in Add. Discount% in the Customer Maintenance Additional Order Information Screen.
Price
The price for the product, if Mtd = $.
Basis
The price or cost column to use to calculate the price, if Mtd = *
Pct
The percentage to use to calculate price:
•
If Mtd = *, this percent is applied to the value in Basis to calculate the price for each line item. The value entered is a percentage off a price column if you entered a price in the Basis field or a percentage over a cost if you entered a cost in the Basis field.
•
If Mtd = A, the percentage to add to the customer’s discount. The system calculates the discount as a chain discount.
Rebate
Mtd
The method to assign a rebate amount to the product:
•
$ - Allows you to enter a set rebate amount for the product.
•
* - Calculates the rebate amount as a percent of a price or a multiplier over a cost.
•
M - Allows you to enter a rebate amount based on a net margin.
Rebate
The rebate amount for the product, if Mtd = $.
Basis
The price or cost column to use to calculate the price, if Mtd = * or M.
Pct
The percentage to use to calculate price:
•
If Mtd = *, this percent is applied to the value in Basis to calculate the price for each line item. The value entered is a percentage off a price column if you entered a price in the Basis field or a percentage over a cost if you entered a cost in the Basis field.
•
If Mtd = M, the percentage to add to the net margin.
BOH Min
The beginning on-hand minimum quantity required by the customer.
Boh Beg
The on-hand balance of the customer when the contact was created.
NOQ
The customer’s normal order quantity.
TargetPrice
 
UseCode
 
Type
 
Lst-Date
The last time the contract was manually updated.
F2-Save
Saves the blanket contract information.
F5-Line Notes
Opens the Notes Display screen to allow you to enter notes for the line item.
