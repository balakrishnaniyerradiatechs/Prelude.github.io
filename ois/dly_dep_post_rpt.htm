Daily Deposit Received Report
The Deposit Balance report lists sales orders for one day, for which a deposit was applied in the Sales Order Completion screen (refer to Using the Sales Order Completion Screen). The report is sorted by pay type (e.g., cash, check, bank card).
Menu Path: Order Processing > Reports > Daily Accounting Reports > Daily Deposit Received Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Post Date. The day the deposit posted to the general ledger.
•
Selling Warehouse. The warehouse that receives credit for the sale.
Column Descriptions
Column Heading
Description
Order#
The sales order number.
Cust#
The customer number.
Description
The customer description.
Deposit Detail
The deposit information.
Deposit Amount
The total amount of the customer deposit for this posting date.
