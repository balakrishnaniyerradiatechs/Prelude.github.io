Release Header screen
The Release Header screen allow you to enter picker, packer, and shipping information for a sales order release in Release Confirmation Entry.
Menu Path: Order Processing > Release Confirmation Entry > F5-Hdr Scr
Field and Function Key Descriptions
Field/Function Key
Description
Picker
The initials of the person who picked the merchandise on this pick ticket. Pickers are set up in Picker Abbreviation Maintenance (refer to Assigning Order Picker Abbreviations).
Packer
The initials for the person who packed the merchandise. This field defaults to your user ID.
Ship Via
The Ship Via code that indicates how the order was sent to the customer. The Ship Via from the Sales Order Header screen appears but can be changed if necessary.
Freight Terms
The freight terms from the Sales Order Header Order Additional Information Screen.
Inbound Freight
Any freight charges incurred to bring the material into your warehouse. For example, if the vendor shipped this material UPS red just for this customer, you may want to pass on the freight charge (typically handled through linked freight).
Outbound Freight
Any freight charges incurred to bring the material into your warehouse. For example, if the vendor shipped this material UPS red just for this customer, you may want to pass on the freight charge.
Handling
Any applicable handling charges. These charges appear when the order is invoiced.
Weight
The total weight of the shipment calculated from the weights set up for each product in Product Maintenance.
Number Of Cartons
The number of cartons included in the shipment.
Number Of Pallets
The number of pallets included in the shipment.
Waybill
The waybill tracking number, if applicable.
Ship Date
The date the order shipped. Defaults to today’s date.
F2-Save
Saves the header information and opens the Release Confirmation Entry screen.
F5-Freight Detail
Opens the Release Shipment Data screen. You can enter freight tracking numbers for the shipping company.
Action Bar
Freight Cost
Opens the Freight Cost screen to allow you to add any known inbound or outbound freight costs to a customer’s shipment.
Currency
Allows you to change the currency if you use multiple currencies.
