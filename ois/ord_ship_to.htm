Ship-To Address Screen
The Ship-To Maintenance screen allows you to verify the shipping address for a sales order. If you are shipping to an address that is not set up for the customer in Customer Maintenance or Customer Ship-To Maintenance, you can enter a one-time shipping address for the order.
Menu Path: Order Processing > Sales Order Entry > F5-Hdr Scr > F6-Ship-To
Field and Function Key Descriptions
Field/Function Key
Description
Ship-To Name
The name that represents this ship-to location.
Address One
The first line of address for this location.
Address Two
The second line of address for this location.
Ship To Attention
The name of the person or department that should receive orders for this ship-to address.
Country
The country of this ship-to address.
Geographic Code
The geographical code assigned to this customer location if you are using geo codes set up in the Geographic Code Maintenance Screen. If you use Vertex to calculate taxes, this information must be correct or you cannot complete the sales order. If more then one code exists for a zip code, a selection screen appears to allow you to select the correct code.
City
The city of this ship-to location.
State
The state for this ship-to location.
Zip
The zip code for this ship-to location.
Bill To Attention
The name of the person or department that should receive invoices for this ship-to address, if applicable.
Phone
The phone number for this ship-to address.
Ext
The phone extension.
Route Number
If you use delivery trucks and follow delivery routes, you can enter a route number set up in the Route Maintenance Screen that contains valid delivery day and time information for each route.
The default comes from the customer’s Customer Route Maintenance Screen or the Ship To Route Maintenance Screen, depending on how the order is entered.
DelSeq
If you entered a Route Number, this number indicates where in the delivery sequence the customer will receive the order.
F2-Save
Closes the Ship-To Information screen.
