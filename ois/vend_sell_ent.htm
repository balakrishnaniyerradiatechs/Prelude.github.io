Vendor/Selling Warehouse Entry screen
The Vendor/Selling Warehouse Entry screen allows you to designate the vendor and selling warehouse for an override shipping warehouse Matrix (refer to Overriding Shipping Locations for Products).
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Override Shipping Whse Matrix > Type Code = 3
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Number
The vendor number for the override matrix.
Selling Warehouse
The selling warehouse that applies for this override. The selling warehouse comes from the sales order header screen.
F2-Save
Saves the vendor/warehouse information and opens the Shipping Warehouse Entry screen
