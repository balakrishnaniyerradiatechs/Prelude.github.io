Vendor Information screen
If you enter vendor contract number or quote number information in the Vendor Information screen in Price Matrix Maintenance, the vendor information carries over to the following:
•
The Line Item Detail Additional Information screen on the sales order.
•
EDI transmissions for linked purchase orders.
•
The purchase order line item for sales orders linked to purchase orders.
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Maintenance < Action Bar > Vend Info
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Contract#
The vendor contract number to assign to the line item.
Vendor Quote#
The vendor’s quote number, if applicable.
Vendor Salesman#
The vendor’s assigned salesman number, if applicable.
F2-Save
Saves the vendor information.
F4-Del
Deletes the vendor information.
