Credit Information screen
The Credit Information screen allows you to enter a credit reason code and original invoice number for returned material.
Require an Invoice Number
Require Original Invoice# in User Identification Maintenance > F6-Add Info determines if a user must enter the original invoice number when entering an RGA or credit memo for returned material.
Menu Path: Order Processing > Optional Entry and Printing > RGA Entry > Line Item Credit Detail screen > enter a Quantity
Field and Function Key Descriptions
Field/Function Key
Description
Credit Reason
The credit reason code to define why a customer is returning material, to update inventory, to update usage a statistics, and to post to the appropriate general ledger accounts (refer to Creating Credit Reason Codes).
Original Inv#
The original invoice number on which the customer purchased this item.
