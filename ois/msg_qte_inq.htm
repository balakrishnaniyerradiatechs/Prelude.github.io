Employee Quote Message Inquiry
The Employee Quote Message Inquiry contains orders released from Administrative Quote Hold. You can view messages sent from the Administrative Quote Manager, then edit or complete the quotes as necessary.
Menu Path: Order Processing > Inquiries > Employee Queue Inquiry > Quote Message Inquiry > F5-Review
Field & Function Key Descriptions
Field/Function Key
Description
User Id
The user ID of the employee assigned to this queue.
Quote#
The quote released from hold.
Notes
An asterisk (*) indicates that notes or messages exist for the quote.
Customer
The customer number and description.
AQ
Indicates if the order was reviewed/released by the manager:
•
AQ - The order was reviewed but not released from quote hold. Notes were entered by the manager to explain why it was not released.
•
Blank - The order was released from quote hold.
Refer to the Administrative Quote Inquiry for more information on Administrative Quote Holds.
Slsm
The sales representative assigned to the order.
Quote Amount
The total value of the quote.
F2-Save
Closes the inquiry screen.
F4-Del
Deletes all messages from the Quote Message Inquiry.
F5-Review
Opens the Quote Display screen to allow you to view detail information on the quote.
F6-QOE
Opens the Quote Order Entry screen to allow you to edit and print the quote.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Refresh
Clears and re-sets the screen if changes were made.
Alternate User
Allows you to send the message to another user ID.
Notes
Allows you to view any header notes, administrative invoice hold notes, or administrative order hold notes for the order.
