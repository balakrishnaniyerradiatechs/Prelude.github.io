Customer Ship-To Maintenance Screen
If your customer receives material at more than one location, Customer Ship-To Maintenance allows you to set up as many additional ship-to addresses as necessary. You can enter unique contact, phone number, salesrep, and other information for each ship-to location. When you enter an order for this customer, you can choose which Ship-To Number to use. The information in the ship-to address takes precedence over the customer address when you select it for a sales order.
Menu Path: Order Processing > Maintenance > Customer Ship-to Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The number assigned to this customer in the Customer Maintenance Screen.
Ship to Number
The number or name that represents this ship-to location. This number appears when you list ship-to addresses.
Once you enter the ship-to number, you can select a warehouse associated with this address. The warehouse for the ship-to address determines the range of document numbers to assign to orders entered for this customer. The warehouse number also sets a default selling location for customers that have no assigned location.
Description
A description of the ship-to location (e.g., the customer name or location).
Address One
The first line of address for this location.
Address Two
The second line of address for this location.
Address Three
The third line of address for this location.
Ship To Attention
The name of the person or department that should receive orders for this ship-to address, if applicable.
Country
The country of this ship-to address.
GeoCode
The geographical code assigned to this customer location, if you are using geo codes set up in the Geographic Code Maintenance Screen.
City
The city of this ship-to location.
State
The state for this ship-to location.
Zip
The zip code for this ship-to location.
County
The county of this ship-to location.
Territory
The territory number used to group customers for sales reports. These codes are set up in the Territory Maintenance Screen.
Status
Indicates if you can ship orders to this address:
•
A - Active. You can enter sales orders using this customer ship-to address number.
•
I - Inactive. You can not enter a sales order using this customer ship-to number.
Contact
The contact person’s name for this ship-to address.
E-Mail Address
The contact’s email address.
Phone
The contact’s phone number.
Ext
The phone extension.
Fax
The contact’s fax number.
Salesrep One
The primary sales representative that appears as the default Salesrep One on sales orders for this ship-to address. You can override this ID on sales orders, but if sales representatives are assigned in the Authorized List of SalesRep #1 Screen, then you can only use one of those names.
Salesrep Two
The second sales representative assigned to the customer (e.g., inside sales rep), if applicable. This sales representative number appears as the default Salesrep Two on sales orders with this ship-to address. You can override this ID on sales orders, but if sales representatives are assigned in the Authorized List of SalesRep #2 Screen, then you can only override Salesrep Two with one of those names.
Utilize SalesRep#2 in Company Maintenances > F5-Add Info determines if Salesrep Two is accessible in Customer Maintenance:
•
Y - You can enter a Salesrep Two in Customer Maintenance.
•
N - You cannot enter a Salesrep Two in Customer Maintenance.
Customer Job Number
If this ship-to account is for a customer job, you can enter a default Job Number to use for all orders going to this ship-to address.
F2-Save
Saves the customer ship-to information.
F4-Del
Deletes the customer ship-to record.
F5-Add Info
Opens the Ship-To Additional Information Screen. You can enter additional data unique to this ship-to address.
F6-Sales Tax
Opens the Ship-To Tax Information Screen. You must enter tax information unique to this ship-to address. If you do not use Vertex to calculate taxes, the sales tax jurisdiction and flag will not default back to the value in Customer Maintenance.
F7-Cust P/O’s
Opens the Customer Purchase Orders Screen. You can enter purchase order numbers associated with this ship-to address.
Action Bar
Pros Mail-To Cnv
Opens the Mail-To to Ship-To Conversion screen to allow you to convert a prospect mail-to into a ship-to customer address.
Add’l Opts
Allows access to the following:
•
The Customer Service Information Screen. You can enter service contact and technician information.
•
The Via Overrides screen. You can set up the customer ship-to address as a valid address on your website.
