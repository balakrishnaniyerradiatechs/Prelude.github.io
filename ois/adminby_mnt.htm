Administered By Maintenance screen
Administered-by codes may be required when you confirm a DD250 sales order (Refer to Confirming a Release that Requires a DD250 Invoice). The administered-by code prints in block 10 on the DD250 invoice (refer to Printing Invoices Using the DD250 Form).
Menu Path: Order Processing > Maintenance > Misc Code Files > Administered By Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Admin By Number
The 1 to 6 digit alphanumeric code to use for sales orders set up to print DD250 invoices.
Description
The name of the administered-by company or person.
Address One
The first line of address for the company.
Address Two
The second line of address.
Address Three
The third line of address.
Address Four
The fourth line of address.
City
The city where the company or person is located.
State
The state where the company or person is located.
Zip
The zip code.
F2-Save
Saves the administered-by information.
F4-Del
Deletes the administered-by number.
