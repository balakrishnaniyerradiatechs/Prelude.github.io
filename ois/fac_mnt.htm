Facility Maintenance screen
If more than one physician works at a hospital or doctor’s office, you can use the Facility Maintenance screen to set up a facility number to use to group the physicians. When you create a physician number (refer to Setting up Physician Information) you can enter the facility number which automatically fills in the physician’s address and phone number information.
Menu Path: Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > Facility Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Facility#
The number to identify the facility. Up to 8 alphanumeric characters. Used in the RX Physician Maintenance screen to group physicians.
Facility Name
The name of the facility.
Address One
The first line of address for the facility.
Address Two
The second line of address for the facility.
Address Three
The third line of address for the facility.
City
The city where the facility is located.
State
The state where the facility is located.
Zip
The zip code for the facility.
Phone
The phone number for the facility.
Fax
The fax number for the facility.
Email Address
The email address for the facility.
Manager
The manager at the facility, if applicable.
F2-Save
Saves the facility information.
F4-Del
Deletes the facility information.
