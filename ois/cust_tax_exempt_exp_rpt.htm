Customer Tax Exemptions Expiration Report
The Customer Tax Exemption report lists, by sales territory, customers whose tax exempt numbers will expire within the number of days that you indicate.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Customer Tax Exempt Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Territory. The code for the sales territory assigned to customers or customer ship-to accounts.
•
Customer Type. The type code used to group similar types of customers for reporting and sales analysis.
•
Expiration Days. Include all customer whose tax exempt code will expire within this number of days.
Column Descriptions
Column Heading
Description
Customer
The customer number.
Description
The description of the customer.
Phone
The customer’s phone number
Exempt Number
The customer’s tax exempt number.
Exp Date
The date the tax exempt number expires.
Tax Jur
The tax jurisdiction assigned to the customer.
Description
The description of the tax jurisdiction code.
