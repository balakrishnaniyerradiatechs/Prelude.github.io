Order Source Matrix Screen
The Order Source Matrix screen allows you to give a customer an additional discount on products depending on how they submit their order.
Menu Path: Order Processing > Maintenance >Customer Maintenance > F7-Order > Action Bar > Add’l Options > Order Source Matrix
Field and Function Key Descriptions
Field/Function Key
Description
Order Source
The order sources that will earn the customer additional discounts on sales orders and quotes. The following are commonly used order sources:
•
C - Customer call in.
•
D - Direct order entry.
•
E - Electronic data interchange.
•
K - Special order-telemarketing.
•
L - Sales representative.
•
M - Mail in.
•
R - Remote order entry.
•
S - Sales force automation.
•
T - Telephone sales.
•
W - Will call.
•
X - Special order.
You can add or edit order source codes in Order Source Code Maintenance (refer to Defining Order Source Codes).
Description
The description of the order source.
Add.%
The additional discount percent to apply to each product that the customer orders using the corresponding method.
F2-Save
Saves the order source discount information.
