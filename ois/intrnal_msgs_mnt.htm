Instruction Messages Maintenance screen
If you have standard sets of instructions that you want to appear on sales documents, quotes, or purchase orders, you can use the Instruction Messages Maintenance screen to set up codes to reference the standard message. You can then enter the code in the Instructions field in the maintenance and order entry programs.
Menu Path: Order Processing > Maintenance > Misc Code Files > Instruction Messages
Field and Function Key Descriptions
Field/Function Key
Description
Message Code
The 1 or 2 digit alpha-numeric code to reference a set of instructions.
Instruction Line#1
The first line of instruction to appear on documents.
Instruction Line#2
The second line of instruction to appear on documents, if applicable.
Instruction Line#3
The third line of instruction to appear on documents, if applicable.
F2-Save
Saves the instruction message.
F4-Del
Deletes the instruction message.
