Release Shipment Data screen
The release Shipment Date screen allows you to enter freight tracking numbers for the shipping company.
Menu Path: Order Processing > Release Confirmation Entry > F5-Hdr Scr > F5-Freight Detail
Field and Function Key Descriptions
Field/Function Key
Description
Freight Amount
The total amount of outbound freight.
Number Of Cartons
The number of cartons included in the shipment.
Box#
The number for each box.
Weight
The weight of each box.
Tracking#
The tracking number assigned by the shipping company.
Amount
The freight amount charged for each box.
F2-Save
Saves the shipment release freight information.
