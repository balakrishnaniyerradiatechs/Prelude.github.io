Sales Tax Exempt Code Maintenance Screen
The Sales Tax Exempt Code Maintenance screen allows you to set up sales tax exemption codes to use in Customer Maintenance (F5-Add Info > F6-Sales Tax) and Customer Ship-To Maintenance (F5-Add Info > F6-Sales Tax).
Menu Path: Order Processing > Maintenance > Sales Tax > Sales Tax Exempt Codes
Field and Function Key Descriptions
Field/Function Key
Description
Sales Tax Exempt Code
The 1 or 2 digit alphanumeric code to represent a tax exempt code to assign to customer records.
Description
A description of the tax exempt code.
F2-Save
Saves the tax exempt code information.
F4-Del
Deletes the tax exempt code.
