Customer Prescription Product Maintenance screen
The Customer Prescription Product Maintenance screen allows you to enter the products included in a customerâ€™s prescription. The products that you enter must be set up as a prescription product (refer to Identifying a Product as a Prescription). You can also enter the quantity, refill, and label information.
Menu Path: Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > Customer Prescription Maintenance > F5-Products
Field and Function Key Descriptions
Field/Function Key
Description
Product
The product included in the prescription. The product must be a valid prescription product (refer to Identifying a Product as a Prescription).
Description
The description of the product.
Qty
The quantity of the product to fill the initial prescription with.
Refills
The number of refills allowed for this prescription.
Labels
The number of labels to print on the prescription.
Available
The quantity needed to fill the prescription (e.g., the original quantity plus the number of refill quantities).
Qty + (Qty x Refills)
Inst
An asterisk (*) displays if instructions exist for the product.
F2-Save
Saves the prescription information and closes the screen.
F5-Instructions
Opens the Notes Entry screen to allow you to view or edit any instructions for the prescription.
F6-Quantities
Opens the Prescription Quantities screen to allow you to view the quantities and dates for the initial prescription and refills.
F7-History
Opens the Prescription History screen to allow you to view the status of open and invoiced sales orders for a customer prescription.
F8-Select Inst
If no instructions are set up for the product, you can add default instructions already set up for the RX Group.
