Territory Maintenance Screen
The Territory Maintenance screen allows you to create territory codes to represent a region where customers are located. Once you create a territory, you can assign it to a customer in the Customer Maintenance Additional Customer Information Screen, or the Customer Ship-To Maintenance Screen. You can use territories to group customers in sales reports.
Menu Path: Order Processing > Maintenance > Customer Control Files > Territory Codes
Field and Function Key Descriptions
Field/Function Key
Description
Territory
The 1 to 4 digit alpha-numeric code to represent a sales territory.
Description
A description of the territory (e.g., East, West, Texas, Coastal).
F2-Save
Saves the territory information.
F4-Del
Deletes the territory code.
