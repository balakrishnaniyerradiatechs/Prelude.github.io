Confirming Releases
When a customer’s order is filled by the order pickers, the material that is available may not necessarily be the quantity that printed on the pick ticket. Release Confirmation Entry allows you to confirm the products, quantities, substitute information, freight amounts, and other shipping information before the order is released for invoicing.
To confirm a sales order release:
1
Select Order Processing > Release Confirmation Entry. The Release Confirmation Entry screen appears.
2
Enter the Release Number, which is the sales order number with a dash and the release (e.g., 10000178-1). The Release Header screen appears.
3
For Picker, enter the initials of the person who picked the merchandise on this pick ticket. Pickers are set up in Picker Abbreviation Maintenance (refer to Assigning Order Picker Abbreviations).
Picker ID Required
The Pick Error flag in File Update Parameters determines if a picker ID is required in Release Confirmation Entry:
•
Y - A picker ID is required in Release Confirmation Entry.
•
N - A picker ID is not required in Release Confirmation Entry,
4
For Packer, enter the initials for the person who packed the merchandise. This field defaults to your user ID.
5
Verify or change the Ship Via code that indicates how the order was delivered to the customer.
6
Verify or change the Freight Terms. These terms come from the Sales Order Header Order Additional Information Screen.
Changing the Ship Via and Freight Terms
The Suppress Order Ship Via/Frt Upd In RCE flag in Order Processing Parameters > F5 > F5 indicates if you can update the ship via code and the freight terms when you confirm a shipment in Release Confirmation Entry:
•
Y - If you change the ship via and freight terms on a sales order in Release Confirmation Entry, the changes will not update the sales order.
•
N - If you change the ship via and the freight terms when confirming an order in Release Confirmation Entry, the changes will update the sales order.
•
S - If you change the ship via and the freight terms when confirming an order in Release Confirmation Entry, only the freight terms are updated.
7
For Inbound Freight, add any charges incurred to bring the material into your warehouse. For example, if the vendor shipped this material UPS red just for this customer, you may want to pass on the freight charge.
8
For Outbound Freight, add any charges incurred to ship the material to the customer. If the customer is set up to always pay an outbound freight amount and you do not enter an amount here, a message appears to remind you to enter the freight before saving the screen.
Posting Outbound Freight
The Post O/B Frt to Shp Whse? flag in Order Processing Parameters > F5 > F5 determines if Outbound Freight, entered in Release Confirmation Entry to bill to the customer, posts to the general ledger of the shipping warehouse:
•
Y - Post Outbound Freight to the sales order’s shipping warehouse.
•
N - The entire amount is income for the selling warehouse. No general ledger entries are created.
9
For Handling, add any applicable handling charges. These charges appear when the order is invoiced.
10
Accept the Weight calculated from the weights set up for each product in Product Maintenance, or override it if necessary.
11
Enter the Number of Cartons and the Number of Pallets included in the shipment, if applicable.
12
Enter the Waybill tracking number, if applicable.
13
Accept the Ship Date which defaults to today’s date or change if necessary.
14
To further define freight tracking information, press F5-Freight Detail to open the Release Shipment Data screen. You can enter freight tracking numbers for the shipping company.
15
Press F2 - Save to save the header information and to open the Release Confirmation Entry screen.
If the sales order is set up to print a DD Form 250 for an invoice, the DD250 Information screen appears to allow you to enter information required for government customers. Refer to Confirming a Release that Requires a DD250 Invoice.
For each Product:
•
Qty-Opn displays the quantity entered on the sales order.
•
Qty-Pck displays the quantity that the system calculated as available and ready to pick on this pick ticket release.
16
For Qty-Shp, verify or enter the quantity that actually shipped out of the warehouse.
Substitute Products
If a substitute product was shipped rather than the product on the picking ticket, enter the substitute product when you confirm the release. If the product you enter is not a substitute product for the original product, (depending on company settings) you can choose to create the product as a valid substitute or to return to the line item and enter another product. The original product’s pricing is used. Refer to “Substitute Products in the Inventory Documentation for more information.
Substitutes are not available for the following when you confirm a release:
•
Comment lines.
•
Kit parent items.
•
Serial numbered products.
•
Lot products.
•
Sales order lines items linked to a purchase order, transfer or work order.
•
Rental order.
•
Job accounting orders.
•
Core product linked to core exchange.
•
Core exchange product linked to core.
•
Surcharge item.
Components
To confirm kit components for a kit parent, select Component Entry from the Action Bar. The Component Entry screen appears to allow you to enter shipped component quantities.
17
After all shipped quantities are confirmed or changed, press F2-Save. The following message appears:
Do You Wish To....
18
Choose from the following:
•
Update/Invoice - Update the release as confirmed so it can be invoiced.
•
Update/Hold - Save the confirmed information but do not release it for invoicing until you come back into the order and release it. Do not use this option unless instructed by your manager.
•
No Update - Do not update the release as confirmed (if you entered the wrong release number or do not want to update the order).
•
Return - Allows you to return to the Release Confirmation Entry screen to verify or change quantities.
19
Once you choose Update/Invoice, the Sales Order Completion screen appears. You can verify or change and freight or handling charges.
20
Press F2-Complete to complete the release confirmation.
Shipment Processing Entry
If you need to enter, confirm, and invoice an order all in one step, you can use Shipment Processing Entry (Order Processing > Optional Entry & Printing > Shipment Processing Entry). The order is set up for invoicing and no pick ticket is generated. You may want to use this if the customer already has the material.
