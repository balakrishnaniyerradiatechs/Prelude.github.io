Restricting Products for a State or Zip Code
If you want to restrict customers in a state from buying a product or product line (e.g, for competition laws), you can create a restricted product matrix to identify the products or product lines that are not allowed. Within each state, you can limit the restrictions by zip code or warehouse. The system checks the state where the product ships from to determine if the product is allowed. If you change the ship-to on a sales order or quote, the system will check to see if the new ship-to address has a permit for the restricted product.
Refer to the Customer Permit Information Screen for information on restricting products for a customer.
Order Processing Parameters
Geographic Prod Restrict (Order Processing Parameters > F5 > F5) must be set to Y so that Sales Order Entry will check for a restricted product matrix.
The MSDS/Ship-To Specific flag determines if restricted product information is stored for a customer at the ship-to level or the bill to level.
To create a restricted product matrix:
1
Select Order Processing > Maintenance > Order Entry Control Files > Restricted Product Maintenance to open the Restricted Product Matrix Maintenance screen.
2
For Matrix Type, choose one of the following types of restrictions:
•
1 - Product line.
•
2 - Product.
3
For State Abbr., enter the abbreviation for the state where the product or product line cannot be sold. The Restricted Product Matrix Input screen appears.
4
Depending on the Matrix Type you chose, enter either the Product Number or Product Line to restrict customers in this state from purchasing.
5
For Req Permit Flag, indicate if the sale of this product/product line is allowed as long as the customer has a permit on file:
•
Y - Sales are allowed to customers with a permit.
•
N - Sales are not allowed to any customers in this state.
A customer’s permit information is in the Customer Permit Information Screen. A customer’s ship-to address permit information is located in the Permit Information Screen.
6
If you want to limit the restriction to just one or more zip codes; under Zip Code Ranges, enter the From Zip and the To Zip codes where customers are not allowed to buy the products.
7
If you want to exclude zip codes from the zip code range that you entered, press F5-Zip Exclusion to open the Zip Exclusions screen.
8
For Zip Exclusions, enter zip codes not included in the restricted product matrix. Customers whose shipping address is in these zip codes, can purchase the products listed on the Restricted Product Matrix Input.
9
Press F2-Save to save the zip exclusion information.
10
To exclude one or more warehouses from the zip code range, press F6-Whse Exclusion to open the Warehouse Exclusions screen.
11
Enter any warehouses that you want to exclude from the restricted product matrix in Whse Exclusions. These Warehouses can to receive the products listed on the Restricted Product Matrix Input screen.
12
Press F2-Save twice to save the warehouse exclusion information and the restricted product information.
