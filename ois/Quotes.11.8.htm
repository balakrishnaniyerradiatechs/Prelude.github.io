Quote Conversion Flags
•
User Identification Maintenance > F6-Add Info > Disallow Quote Conversion. Determines if a user can convert quotes.
•
Order Processing Parameters > Display Open Quote/RGA Message: Determines if a message displays when you enter a sales order or quote if the customer has existing open quotations and/or RGAs.
•
Order Processing Parameters > F5 > Open Quote Message: Determines if the message displays at the header level, line item level, or both. Display Open Qte/RGA Message in Warehouse Maintenance overrides this flag
•
Order Processing Parameters > F5 > Include Converted Quotes: Determines if converted quotes are included in the warning message
•
Order Processing Parameters > F5 > F5 > Line By Line Conversion In QOE > Determines if upon completion of a quote, you can convert the quote to a sales order by line item or if you have to convert the entire quote.
•
Order Processing Parameters > F5 > F5 > Qte Line Conversion in SOE : Determines if you are prompted in Sales Order Entry by line item, if a quote exists for a product.
•
Order Processing Parameters > F5 > F5> F5 > Quote Message in All Warehouses: Determines if a message displays when you enter a sales order or quote if the customer has existing open quotations in the selling warehouse only or if the message displays if open quotations exist on all warehouses.
