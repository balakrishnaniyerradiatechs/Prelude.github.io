Modifying Multiple Matrices for a Customer
Quick Price Matrix Entry allows you to modify multiple price matrices for a customer/customer price group at the same time. You can modify the beginning and ending dates and the pricing method information. Only price matrix types 1, 3, 6, and 8 can be updated with this matrix option.
If Price Matrix Maint Audit is set to Y in File Update Parameters, the Price Maintenance Audit Report by User Id and the Price Maintenance Audit Report by Type are updated with prices modified by Quick Price Matrix Entry.
To modify multiple price matrices at one time:
1
Select Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Quick Price Matrix Entry to open the Quick Matrix Entry screen.
2
For Type, enter the price matrix type (refer to Creating a Price Matrix) to update. You can only update the following matrix types:
•
1 - Customer/Product.
•
3 - Customer/Product Line.
•
6 - Customer Price Group/Product.
•
8 - Customer Price Group/Product Line.
3
Enter the Customer whose price records you want to update.
4
For Customer Prc Group, if you are modifying matrix types 6 or 8, enter the customer price group for which to want to modify matrices.
5
Enter the Effective Date for which you want to search for the price matrix records. For example, enter 8/1/06 to include all price matrices for the customer or customer price group that begin on 8/1/06 or include 8/1/06 in the valid date range.
6
Enter the Contract Number if you only want to update matrices for an existing pricing contract. Depending on the matrix type entered, one of the following screens displays:
•
For matrix types 1 or 6, the Quick Matrix Entry Product/Number screen appears.
•
For matrix types 3 or 8, the Quick Matrix Entry/Product Line screen appears.
7
Add or delete products/product lines as necessary. When you delete a product, you can also delete the price matrix record. If the price matrix record contains multiple date ranges, only the date range displayed is removed.
8
Press F2-Save to save any changes.
