Price Matrix Mass Update screen
You can use the Price Matrix Mass Update screen to copy price matrices for a product to another product. Only matrix types 1 (customer / product) and 6 (customer price group / product) are copied when you use this procedure. Refer to Copying all Price Matrices for a Product.
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Mass Update
Field and Function Key Descriptions
Field/Function Key
Description
Copy From:
Product
The product with matrices to copy.
Warehouse
The warehouse number if you just want to copy matrices set up for the product in a single warehouse.
Job/Customer P/O
The job or customer purchase order number if you just want to copy the matrices that apply to one job or purchase order.
Contract
The contract number if you just want to copy the matrices that apply to one contract. Otherwise, all type 1 and 6 matrices are copied.
Drop Ship
Indicates if this matrix is limited to direct ship orders only:
•
Y - Copy only matrix types 1 and 6 that have Drop Ship set to Y.
•
N - Copy all type 1 and 6 matrices.
Currency
The currency type if you only want to copy product matrices for a currency type.
Copy Ctr#
Indicates if you want to copy the contract number from the old price matrix to the new one:
•
Y - Copy the contract number to the new matrix.
•
N - Do not copy the contract number from the old matrix to the new one.
Copy Exp Date
Indicates if the expiration date from the old matrix should be copied to the new matrix:
•
Y - Copy the expiration date from the old contract.
•
N - Do not copy the expiration date. The new matrices are created with no expiration date.
Override Price
Indicates if the price for the product/matrix can be overridden on a sales order:
•
Y - The price can be overridden on a sales order.
•
N - The price cannot be overridden on a sales order.
•
H - The price can be raised but not lowered below gross price.
Copy To:
Product
The product that is to receive the price matrix.
Whse
The warehouse number if the price matrix only applies to one warehouse. Leave blank to copy the matrix to all warehouses.
D/S
Indicates if the new product matrix should be for drop ship orders only:
•
Y - The new matrix only applies to drop ship orders.
•
N - The new matrix applies to all orders.
Job/Cust P/O
The job or customer purchase order number if the new matrix is only for a specific job or purchase order.
Contract
The contract number if the new matrix is only for a specific contract.
F2-Save
Copies all price matrices for the product to the new product number.
