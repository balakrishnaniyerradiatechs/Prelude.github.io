Incompatible Ship Via Maintenance Screen
The Incompatible Ship Via Maintenance screen allows you to identify ship via codes that should not be used to ship material to this address. Ship via codes are established in the Ship Via Maintenance Screen.
Menu Path: Order Processing > Maintenance > Customer Ship-to Maintenance > F5-Add Info > F6-Incomp Ship Via
Field and Function Key Descriptions
Field/Function Key
Description
Ship Via
The ship via codes that cannot be used to ship to this customer ship-to address.
Description
The description of the ship via code.
F2-Save
Saves the incompatible ship via information.
