Pending Lien/Bond Report
The Pending Lien/Bond report lists lien and bond expiration dates for customer jobs.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Lien / Bond Reports > Pending Lien/Bond Expiration Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Expiration Date <. The latest expiration date to include. All pending bonds and liens that expire before this date are included.
Column Descriptions
Column Heading
Description
Cust#
The customer number.
Description
The customer description.
Job Number
The customer’s job number.
Description
The job description.
Lien Exp Date
The expiration date of the lien, if applicable.
Bond Exp Date
The expiration date of the bond, if applicable.
Last Shp Date
The last time a product shipped for this job.
