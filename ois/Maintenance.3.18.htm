Creating the Package Insert Product
Before you can create the package insert code, you must first create a product number for each free product (package insert) that you will give to customers.
To create the package insert product:
1
Select Inventory > Maintenance > Product Maintenance > Product Maintenance to open the Product Maintenance screen.
For complete detailed information on creating a product, refer to “Product Maintenance” in the Inventory documentation.
2
Enter the Product and Description that refers to the package insert.
3
Enter the Product Line set up for package inserts.
4
Set Service Item Flag to Yes and the next field to Material so that inventory is not tracked for the insert product.
5
Press F5-Add Info > F5-Add Info to open the Additional Product Maintenance Information screen.
6
Set S/O PTP Dependent to Y so that the package insert only ships if the sales order has products available to ship.
7
Press F2-Save 3 times to save the package insert product information.
