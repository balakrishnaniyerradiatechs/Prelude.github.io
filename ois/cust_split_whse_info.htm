Alternate Shipping Warehouse Screen
If you use EDI or DOE transmissions, you can use the Alternate Shipping Warehouse screen to designate additional warehouses that can ship products if inventory is not available at the customer’s designated shipping warehouse.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F7-Order > Action Bar > Add’l Options > Alternate Shipping Warehouse
Field and Function Key Descriptions
Field/Function Key
Description
Warehouses
The warehouses that EDI/DOE transmissions can check to ship available inventory if stock is not available at the customer’s regular shipping warehouse.
F2-Save
Saves the warehouse information.
