Adding a Markup to the Pre-Authorization Amount
If you want to pre-authorize sales orders for more than the merchandise amount (e.g., to account for freight charges), you can set up a credit card table to determine the markup percent to add to the merchandise amount pre-authorized on a credit card. Then if the freight charge ends up being more that what was pre-authorized, it won’t be rejected when you post-authorize. You can enter multiple levels of markups.
Order Processing Parameters
To access the CC Table Maintenance screen, Add Markup To Pre-Auth Amt in Order Processing Parameters > F5-Add Info must be set to Y. Refer to Integrating Protobase with Prelude.
1
Select Credit Card Table Maintenance in Order Processing Controls to open the CC Table Maintenance screen.
2
Enter the Table Number to create.
3
For Markup Limit, indicate how large an order must be to be excluded from adding a markup.
4
Enter the cutoff Amount that determines when to add a markup percent. For example, if you enter 10, all orders that are $10 or less will have the corresponding Markup % added to their order. You can enter more than one cutoff amount.
5
Enter the Markup % the customer will receive when they purchase up to the corresponding Amount.
6
Press F2-Save to save the markup information.
Once you set up credit card tables, you can assign the table number to:
•
A ship via code in the Credit Card Options screen (refer to the Ship Via Maintenance Screen).
•
All credit card sales orders: Set Add Markup To Pre-Auth Amt to Y in the Add CC Software Info screen (refer to Integrating Protobase with Prelude).
