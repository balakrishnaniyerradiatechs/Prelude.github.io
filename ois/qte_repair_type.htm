Repair Type Screen
The Repair Type screen appears for a type 03 repair quote. This type of quote interfaces with the Work Order module to accumulate material, labor, and other costs associated with the repair.
Menu Path: Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Order Entry > F5-Hdr Scr > Order Type = 03
Field and Function Key Descriptions
Field/Function Key
Description
Repair Type
The type of repair that determines payment:
•
01 - Warranty: The vendor, distributor, and/or customer are charged all, part, or none of the charges depending on the product’s warranty policy.
•
02 - Time and Material: The customer is charged for material and labor expended on the repair order based upon standard pricing logic and parameters for the customer.
•
03 - Fixed Fee: The customer is charged a fixed amount, usually based on an estimate related to the condition of the repaired product, regardless of the materials and labor.
F2-Save
Saves the repair type and returns to the Quotation Header screen.
