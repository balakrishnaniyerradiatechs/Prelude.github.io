Assigning Credit Bureau Codes
Prelude has standard credit card bureau codes (e.g., AMEX, MC, VISA) already set up to post to general ledger accounts. For each credit card type you accept, you can accept or change the general ledger chart of account, bank, and tax information.
Since the credit card functionality was originally written to work with MAPP, some fields do not apply to Protobase. Prelude no longer supports MAPP.
To set up the credit bureau code information:
1
Select Order Processing > Maintenance > Misc Code Files > Credit Bureaus to open the Credit Bureau Maintenance screen.
2
Enter the Bureau Code or press F3 to see the codes set up in Prelude. The Description of the card displays.
3
If you use the MAPP verification system, enter the Online Phone#, Baud Rate, Voice Authorization, and Station Id. This information is no longer supported by Prelude.
4
Enter the credit card Bank Code to determine which checking account to use for cash receipts and payables. This code is also used as a prefix for check numbers to make sure they are unique.
5
For Post To Account, enter G/L accounts if you want override the existing accounts.
6
Press F5-Alt State Code to enter alternate state tax jurisdiction codes for this credit card type.
7
Press F6-Alt TIDs to enter alternate terminal IDs for each warehouse. This information is mandatory if you use multiple merchant numbers with Protobase.
8
Press F7-G/L Prefix Override to post to a different general ledger prefix for this credit card.
9
Press F2-Save to save the credit bureau information.
