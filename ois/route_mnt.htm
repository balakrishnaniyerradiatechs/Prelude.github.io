Route Maintenance Screen
You can use Route Maintenance to create route codes that identify truck delivery routes and picking days. A sales order must have a Ship Via code that has Truck Route set to Y in Ship Via Maintenance, to use truck routes.
Once you set up the route codes, you can assign a default route to customers in the Customer Route Maintenance Screen (Customer Maintenance > F7-Order > Action Bar > Route).
Menu Path: Order Processing > Maintenance > Customer Control Files > Route Codes
Field and Function Key Descriptions
Field/Function Key
Description
Route Code
The 1 to 7 digit alpha-numeric code that identifies a truck route.
Description
A description of the route.
Route Group
The route group code that groups the routes. These codes are established in the Route Code Group Maintenance Screen and are applicable if you use Route Processing to assign a freight surcharge to the first sales order release.
Pick Ticket Def Days
The number of days to subtract from the required date to calculate the pick ticket date for this route code. The hierarchy for calculating pick ticket days is as follows:
•
1st - Pick Ticket Def Days in the Route Maintenance screen.
•
2n - Pick Days in Customer Maintenance.
•
3rd - Pick Days in Customer Maintenance.
•
4th - Pick Ticket Default Days Warehouse Maintenance.
•
5th - Pick Ticket Default Days in Order Processing Parameters.
Refer to Setting Pick Ticket Rules.
Monday - Sunday (Y/N)
Determines which days the delivery truck drives this route:
•
Y - This route is included in this day’s deliveries.
•
N - This route is not included in this day’s deliveries.
Cutoff Time
If the delivery day is set to Y, orders must be entered on the day before the delivery day. For example, if the truck delivers on Wednesday and the cutoff time is 17:00, then any orders entered before 5:00 P.M. on Tuesday will be included in this route.
F2-Save
Saves the route information.
F4-Del
Deletes the route code.
F5-Picking Days
Opens the Picking Rules screen to allow you to define the days that this warehouse picks and fills sales orders. The hierarchy to determine picking days is as follows:
•
1st - Order Processing > Maintenance > Customer Controls Files > Route Codes > F5-Picking Days
•
2nd - Order Processing > Maintenance > Customer Control Files > Ship Via Codes > Action Bar > Addtl Optns > Picking Days
•
3rd - Inventory Controls > Warehouse Maintenance > F5-Add Info > F6-Picking Days
Refer to “Printing Pick Tickets” in the Order Processing documentation for detailed information on pick ticket rules and the hierarchy used to determine when to print pick tickets.
