Additional Information screen
The Additional Information screen allows you to enter additional information for a line item on a quote.
Menu Path: Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Entry > Quote Entry Detail screen > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Expected Date
The date by which you expect to hear back from the customer regarding the quote. Defaults from the Follow Up Date in the Quotation Header screen. The date defaults from the header screen.
Customer P/O Line#
The line number for the product on the customer’s purchase order.
Customer Product#
The customer’s part number for the product, if applicable. Refer to Setting Up Customer Product Numbers.
Customer Acct Code
The customer’s accounting code for the product, if required by the customer. Require Acct Code in the customer’s Order Information Screen (Customer Maintenance > F7-Order) determines if the customer requires an accounting code for each line item on a sales order or quote. If the customer’s Acct Code Pattern Match Screen has an account code pattern, you must also match the pattern.
Ship Via
The ship via method for this line item, if it is to be shipped by a different method than the other items on the order. If the quote is converted to a sales order, a separate release generates for each ship via method.
Campaign Priority Code
The campaign priority code that the customer is responding from, if you use the Marketing Module and campaigns.
Vendor Contract#
If you use purchase order EDI transmissions and this line is linked to a purchase order, this field represents the vendor’s contract number. The price defaults from the price matrix but can be overridden until the line is linked to a purchase order.
Vendor Quote#
if you use purchase order EDI transmissions and this line is linked to a new purchase order, this field represents the vendor’s quote number. The price defaults from the price matrix but can be overridden up until the line is linked to a purchase order.
Vendor Salesman#
The vendor’s sales person number if you use EDI transmissions and this line is linked to a purchase order.
F2-Save
Saves the line item information.
