Assigning Order Picker Abbreviations
When you confirm a shipped/packed order in Release Confirmation Entry, you can enter the picker’s initials to identify who picked the order. You can create the picker abbreviations in Picker Maintenance.
Picker ID Required
The Pick Error flag in File Update Parameters determines if a picker ID is required in Release Confirmation Entry:
•
Y - A picker ID is required in Release Confirmation Entry.
•
N - A picker ID is not required in Release Confirmation Entry,
To assign picker abbreviations:
1
Select Order Processing > Maintenance > Misc Code Files > Picker Abbreviations to open the Picker Maintenance screen.
2
Enter the 1 to 3 digit alpha-numeric Picker Number that identifies the order picker.
3
For Description, enter the name of the order picker.
4
Press F2-Save to save the picker information.
