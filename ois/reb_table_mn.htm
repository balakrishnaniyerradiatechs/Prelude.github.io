Rebate Table Maintenance Screen
The Rebate Table Maintenance screen allows you to define rebate percentages based on the total amount of products purchased. You can set up multiple rebate percentages that the customer will earn based on the quantities purchased. If you choose to use rebate tables, you can view the results in the Customer Rebate Detail Report (Table) and the Customer Rebate Summary Report (Table).
Menu Path: Order Processing > Maintenance > Customer Control Files > Customer Rebates > F5-Rebate Table
Field and Function Key Descriptions
Field/Function Key
Description
Total Purchases
The amount the customer must purchase to qualify for the corresponding Rebate %.
Rebate %
The percent rebate of total goods purchased the customer will receive if they purchase the amount defined in Total Purchases.
F2-Save
Saves the rebate table information.
