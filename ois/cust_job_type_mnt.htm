Customer Job Type Maintenance Screen
The Customer Job Type Maintenance screen allows you to establish customer job types that group jobs for custom reporting. You can assign these to jobs in the Customer Job Maintenance Screen.
Menu Path: Order Processing > Maintenance > Customer Control Files > Customer Job Types
Field and Function Key Descriptions
Field/Function Key
Description
Job Type
The job type code to identify a group of jobs.
Description
A description of the job type.
F2-Save
Saves the job type information.
F4-Del
Deletes the job type.
