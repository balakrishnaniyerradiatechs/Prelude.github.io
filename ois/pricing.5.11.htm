Defining Purchasing Cost
You can use Price Matrix Maintenance to define the purchase cost for a purchase order. If a line item on a sales order has a B supply code and a purchase cost matrix exists, then this cost is used when the line item is linked to a purchase order.
To set up a purchase cost matrix:
1
Follow the procedures for Creating a Price Matrix and fill in the appropriate pricing information.
2
In the Price Matrix Input screen, enter Y for Ct to designate this matrix as a contract.
3
Press F5-Purchase Cost to open the Purchase Cost screen.
4
For Cost Method, choose from the following to determine how the purchase cost is calculated for line items that meet this matrix criteria:
•
* - Use a percentage of a price/cost field. Enter the following:
-
For Basis, enter the price or cost column (from Product Maintenance > F7-Pricing) to use as the base amount to calculate the discount or markup.
-
For Percent, if you entered a price column in Basis, enter the percent to take off of the price. If you entered a cost column in Basis, enter a percent over the cost to calculate the purchase order cost.
•
$ - Price/Cost. Use this option if the matrix is for a specific product. The cursor moves to the Cost field to allow you to enter the cost.
•
T - Use a discount break table (refer to Discount Break Tables). The system determines a cost based upon the discount level. Enter the discount break Table number to use. The system determines a purchase cost based upon the discount level.
5
For Commission Cost Override, determine if the purchase cost calculated in this screen overrides the commission cost:
•
N - Purchase cost does not override commission cost.
•
Y - Commission cost for the line item is calculated using the parameters defined in this screen. This calculated commission cost may be overridden if a cost is defined in the Commission Cost Entry screen (refer to Overriding Commission Cost).
6
Enter the Vendor Number to assign to the purchasing cost. When a line item with a B supply code is sold with this primary vendor, this purchase cost is used on the linked purchase order.
7
Press F2-Save to save the purchase cost information.
