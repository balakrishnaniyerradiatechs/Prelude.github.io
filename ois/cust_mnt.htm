Customer Maintenance Screen
The Customer Maintenance screen allows you to set up name, address, and contact information for a customer. From this screen you can access accounts receivable information, sales order requirements, sales statistics, and purchasing contact information.
Menu Path: Order Processing > Maintenance > Customer Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Notes Indicator
(unlabeled field)
Indicates if notes or attachments exist in the customer record (Action Bar > Notes):
•
A - Attachments.
•
* - Notes
•
+ - Both notes and attachments.
Customer
The customer number.
Description
The description of the customer.
Shipto Ct
The number of ship-to addresses set up for this customer. For information on setting up ship-to accounts, refer to Customer Ship-To Maintenance.
Desc. (To Sort By)
A different description of the customer name to use for sorting if you want the customer to appear under a different description. For example, if the company name is The Golf Shop, you would not want it to appear under The. Therefore, enter Golf Shop as the sort key.
Address One
The first line of the customer’s address.
Address Two
The second line of the customer’s address.
Address Three
The third line of the customer’s address.
Bill To Attention
The name of the person that receives the invoices.
Country
The country where the customer is located.
GeoCode
The geographical code if you are using geo codes from the Geographic Code Maintenance Screen.
City
The customer’s city.
State
The state abbreviation.
Zip
The customer’s zip code.
County
The customer’s county.
Salesrep One
The primary sales representative that defaults as Salesrep One on sales orders for the customer. You can override this ID on orders, but if sales representatives are assigned in the Authorized List of SalesRep #1 Screen, then you can only override Salesrep One with one of those codes.
Salesrep Two
The secondary sales representative assigned to the customer (e.g., inside sales rep), if applicable. This sales representative number defaults as Salesrep Two on orders assigned to the customer. You can override this ID on orders, but if sales representatives are assigned in the Authorized List of SalesRep #2 Screen, then you can only override Salesrep Two with one of those codes.
Utilize SalesRep#2 in Company Maintenance > F5-Add Info determines if Salesrep Two is accessible in Customer Maintenance.
Payment Terms
The default accounts receivable payment terms code for orders and quotes that indicates when payment is due, payment conditions, and any applicable discounts. These codes are set up in A/R Terms Payment Terms Maintenance. Refer to the Accounts Receivable documentation for more information.
Contact
The person to contact for accounts receivable information. This name prints on accounts receivable reports and inquiries.
Email Address
The email address for invoices, quotes, and acknowledgements.
Website address
The customer’s website, if applicable. For reference only.
Phone
The phone number for accounts receivable. This number prints on A/R reports and inquiries.
Ext
The phone extension for the contact person.
Fax
The customer’s fax number for accounts receivable.
Cellular
The cell phone number for the contact person.
Pager
The pager number for the contact person.
Other Phone
Any other phone number for the contact person.
F2-Save
Saves the information on the Customer Maintenance screen.
F4-Del
Deletes the customer record. If the customer has a sales or A/R history, a warning message appears to allow you to confirm that you want to delete the record. If open invoices or sales orders exists, you cannot delete the record.
F5-Add Info
Opens the Additional Customer Information Screen. You can access additional customer/owner information.
F6-A/R
Opens the A/R Information Screen to access accounts receivable and payment information for the customer.
F7-Order
Opens the Order Information Screen to access information regarding shipping, job, and freight information for the customer’s sales orders.
F8-Contact
Opens the Customer Contacts Screen to access purchasing contact information for the customer.
F9-Statistics
Opens the Customer Statistics Screen to access sales, returns, profit, receipts, discount, and freight information for the customer.
Action Bar
Add’l Opts
Provides access to the following:
•
Customer Labels Information Screen.
•
Customer Service Information Screen.
•
Instructions Entry Screen.
•
Authorized List of SalesRep #1 Screen.
•
Authorized List of SalesRep #2 Screen.
•
Marketing Notification List Screen.
•
Via Overrides screen (if you use VIA).
Notes
Allows access to the following:
•
Customer Notes: Opens the Notes Display screen to allow you to enter, edit, or view notes regarding this customer.
•
Attachments: Opens the Prelude Attach Viewer to allow you to attach a document to the customer record. You can attach any document/picture that you can access on your computer. If a document is attached to the customer record, an A or + displays to the left of Customer on the Customer Maintenance screen.
•
Imaging: If your company uses CoStar ® imaging, you can access images associated with the customer.
