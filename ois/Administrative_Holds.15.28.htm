Product Level Administrative Quote Hold Options
You can set up an individual product to always go on Administrative Quote Hold or to go on hold if the ship via is not compatible for shipping the product.
To set Administrative Quote Hold options for a product:
1
Select Inventory > Maintenance > Product Maintenance > Product Maintenance.
2
Enter the Product Number.
3
Press F5-Add Info to open the Additional Information screen.
4
If you want this product to always go on hold, set Always Hold Order to Y. Any time a quote is entered with this product on it, it will go on hold.
5
If you want to set up incompatible ship via codes for the product; from the Action Bar, select Incomp Ship Via to open the Incompatible Ship Vias screen.
6
For Via, enter any ship via codes not acceptable for shipping this product. Any time a quote is entered for this product using this ship via code, it will go on hold.
7
For Class, enter the class of the associated incompatible ship via (e.g., for UPS, the class might be next day or second day delivery). Not all ship vias have associated classes.
8
Press F2-Save 3 times to save the information.
