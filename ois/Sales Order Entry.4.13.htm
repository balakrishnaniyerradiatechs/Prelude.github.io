Entering a Miscellaneous Product
If you need to enter a charge on a sales order that is not a stock product, non-stock product, service charge, or freight charge, you can enter a miscellaneous product (taxable or non-taxable) in the Sales Order Entry screen.
Examples of Miscellaneous Charges:
•
Gift Certificates (refer to Gift Certificates).
•
Charge for liability insurance.
•
Credit the customer for a rebate earned.
•
Billing for a trade show (if not set up as a product).
 
Disadvantages Of a Using a Miscellaneous Charge
•
Billing only, no cost.
•
Does not exist on product lookup inquiries since it is not a product.
•
The wrong G/L account could be used if not set-up correctly.
To enter a miscellaneous product on a sales order:
1
At the Product prompt on the Sales Order Entry screen (refer to Entering Stock Products), enter one of the following miscellaneous entry codes:
•
M - Miscellaneous product (taxable if the order is taxable).
•
MN - Miscellaneous product (non-taxable, even if the order is taxable).
•
MNP - Miscellaneous product (non-taxable, Canada).
The Miscellaneous Amount Entry screen appears.
2
For Misc. Amount, enter the amount to charge the customer for the miscellaneous product.
3
For Misc. Reason Code, enter the reason code that identifies what the charge is for. These are created in Miscellaneous Reason Code Maintenance (Order Processing > Maintenance > Misc Code Files > Miscellaneous Reason Code Maintenance).
4
The general ledger account information under Miscellaneous Account defaults from the account information set up for the code in the Miscellaneous Reason Code Maintenance screen.
5
Press F2-Save. The Product Description screen appears.
6
Enter as much of a description as necessary in Description 1 and Description 2.
7
Press ENTER. The miscellaneous product and description appear on the Sales Order Entry screen.
