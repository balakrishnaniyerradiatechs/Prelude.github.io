Copying Multiple Price Matrices
To expedite the entry of multiple price matrices, Price Matrix Mass Copy allows you to copy all matrices for a customer or customer price group to another customer or customer price group. You can also copy to multiple customers or price groups. All price matrix records for the Copy From customer/customer price group are selected and duplicated to the Copy To customer/customer price group in the same format.
The Price Matrix Mass Copy excludes matrices set up for customer ship-to accounts.
To copy multiple price matrices:
1
Select Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Mass Copy to open the Price Matrix Mass Copy screen.
2
Under Copy From, for Type, choose from the following:
•
C - Copy a price matrix from a customer.
•
G - Copy a price matrix from a customer price group.
3
For Cust/Grp, depending on the Type, select one of the following:
•
If Type is set to C, enter the customer number to copy from.
•
If Type is set to G, enter the customer price group to copy from.
4
For Whse, if the price matrix is only for one warehouse, enter the number. Only matrices with the same warehouse number are copied. If a warehouse number is not entered, only matrices not assigned to a warehouse are copied.
5
For D/S, determine if drop ship (direct) matrices are copied:
•
Y - Only matrices with the Drop Ship flag set to Y are copied.
•
N - Only matrices with the Drop Ship flag set to N are copied.
6
Enter a Currency type to limit the matrices that are copied to only one currency. This field is only applicable if you use multiple currency.
7
For Job/Cust P/O, determine if matrices that specify a job number or customer P/O are copied. If you enter a job or customer P/O, only matrices matching that job or customer P/O are copied. If you do not enter a job or cust P/O, then only matrices not assigned a job or cust P/O are copied.
8
Enter a Contract Number to only copy line items with the same contract number. If you do not enter a contract number, only matrix line items without a contract number are copied. For example, if the Copy From matrix has one line referencing a contract number and one without a contract, then when the matrix is copied, only the line item with the same contract number is copied to the new matrix.
9
If you are creating a new contract, Ov allows you to determine if prices can be overridden on a sales order:
•
Y - Prices can be overridden on sales orders.
•
N - The prices cannot be overridden on a sales order.
•
H - Prices can be raised but not lowered below gross price.
10
Under Copy To, for Cust/Grp, choose from the following:
•
If Type is set to C, enter the customer numbers to copy to.
•
If Type is set to G, enter the customer price groups to copy to.
11
For Whse, enter a warehouse number if the new matrices are for just one warehouse. Leave blank if the new price matrices are for all warehouses.
12
For D/S, choose from the following:
•
Y - The new matrices should be set up as drop ship only.
•
N - The new matrices should not be set up as drop ship.
13
For Job/Cust P/O, enter a job number or customer P/O number if you only want to copy prices matrices to a specific job or customer P/O.
14
Enter a Contract number to copy matrices to a specific customer contract. Leave blank if the new price matrices do not have a contract number. If you enter a contract number, it will be assigned to each new price matrix created and on every line of the new matrices. Both the CT and OV flags will be set to Y.
15
Repeat steps 10 - 14 for any additional matrices to copy.
16
Press F2-Save to copy the price matrix information.
