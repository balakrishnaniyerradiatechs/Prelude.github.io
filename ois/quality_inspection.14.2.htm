•

•

•
Creating Quality Agent Codes.
•
Setting Acceptable Quality Levels for Inspection Procedures.
•
Assigning Default Procedures to Inspection Types.
•
Setting up Inspection Levels for Sample Sizes.
•
Using Single Sampling Tables.
