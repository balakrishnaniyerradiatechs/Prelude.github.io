Price Matrix Mass Copy screen
To expedite the entry of multiple price matrices, the Price Matrix Mass Copy screen allows you to copy all matrices for a customer or customer price group to another customer or customer price group. You can also copy to multiple customers or price groups. All price matrix records for the Copy From customer/customer price group are selected and duplicated to the Copy To customer/customer price group in the same format.
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Mass Copy
Field and Function Key Descriptions
Field/Function Key
Description
Copy From:
Type
The type of matrix to copy:
•
C - Copy a price matrix from a customer.
•
G - Copy a price matrix from a customer price group.
Cust/Grp
Customer or customer price group, depending on the Type:
•
If Type is set to C, the customer number to copy from.
•
If Type is set to G, the customer price group to copy from.
Description
The description of the customer or customer price group.
Whse
If the price matrix is only for one warehouse, enter the number. Only matrices with the same warehouse number are copied. If a warehouse number is not entered, only matrices not assigned to a warehouse are copied.
D/S
Determines if drop ship (direct) matrices are copied:
•
Y - Only matrices with the Drop Ship flag set to Y are copied.
•
N - Only matrices with the Drop Ship flag set to N are copied.
Currency
The Currency type to limit the matrices that are copied to only one currency. Only applicable if you use multiple currency.
Job/Customer P/O
The job or customer purchase order number if you just want to copy the matrices that apply to one job or purchase order.
Contract Number
The contract number if you only want to copy line items with the same contract number. If you do not enter a contract number, only matrix line items without a contract number are copied. For example, if the Copy From matrix has one line referencing a contract number and one without a contract, then when the matrix is copied, only the line item with the same contract number is copied to the new matrix.
Ov
If you are creating a new contract, determines if prices can be overridden on a sales order:
•
Y - Prices can be overridden on sales orders.
•
N - The prices cannot be overridden on a sales order.
•
H - Prices can be raised but not lowered below gross price.
Copy To:
Cust/Grp
The customer or price group to copy the contract to:
•
If Type is set to C, the customer numbers to copy to.
•
If Type is set to G, the customer price groups to copy to.
Description
The description of the customer or price group.
Whse
The warehouse number if the price matrix only applies to one warehouse. Leave blank to copy the matrix to all warehouses.
D/S
Indicates if the new product matrix should be for drop ship orders only:
•
Y - The new matrix only applies to drop ship orders.
•
N - The new matrix applies to all orders.
Job/Cust P/O
The job or customer purchase order number if the new matrix is only for a specific job or purchase order.
Contract
The contract number to copy matrices to a specific customer contract. Leave blank if the new price matrices do not have a contract number. If you enter a contract number, it will be assigned to each new price matrix created and on every line of the new matrices. Both the CT and OV flags will be set to Y.
F2-Save
Copies all price matrices for the product to the new product number.
