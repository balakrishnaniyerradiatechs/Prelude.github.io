Promotion Exception Maintenance screen
The Promotion Exception Maintenances screen allows you to include or exclude a customer from a promotion matrix. When you create a promotion matrix, all customers are automatically included. But, you can exclude customers from a promotion matrix or create a list of customers that are the only ones to receive the promotion.
Menu Path: Order Processing > Maintenance > Pricing Structures > Promotional Pricing > Promotion Type Matrix Maintenance > F7-Cust Exc
Field and Function Key Descriptions
Field/Function Key
Description
Exception Type
C for customer number is the only option available at this time.
Exception Method
Determines whether you will include or exclude customers from the promotional pricing:
•
I - Include the customers listed under Exception List in this promotion. All other customers will not receive the promotional pricing.
•
E - Exclude the customers listed under Exception List in this promotion. All other customers will receive the promotional pricing.
Exception List
 
The customer numbers to include or exclude from the promotion depending on the Exception Method setting.
F2-Save
Closes the Promotion Exception Maintenance screen.
F5-Load From Saved List
Opens the Promotion Exception List Name screen to allow you to enter an existing Exception List Name from TCL to use to include or exclude customers.
F6-Clear Exception List
Allows you to clear the exception list.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
