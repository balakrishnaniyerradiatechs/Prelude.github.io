Setting Prescription and Facility Counter Numbers
If you want to use system generated numbers to identify facilities and prescriptions, you must use RX Counter Maintenance to identify the next available facility number and the next available prescription number.
To establish counter numbers for prescriptions and facilities:
1
Select Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > RX Counter Range Setup to open the RX Counter Range Setup screen.
2
For Current Rx#, enter the next prescription number to use in Customer Prescription Maintenance (refer to Creating a Customer Prescription).
3
For Current Facility#, enter the next facility number to use in Facility Maintenance (refer to Setting up Facility Information).
4
Press F2-Save to save the counter information.
