Duplicating a Sales Order for One Ship-To
An existing open sales order can be duplicated for the same customer or for another customer.
To duplicate an existing open sales order:
1
Select Order Processing > Sales Order Entry.
2
Press ENTER over Order Number.
3
From the Action Bar, select Dup Order to open the Order Duplication Entry screen.
Duplicate an Invoiced Order
To duplicate an order that is already invoiced, select Dup Order Hist.
4
For Duplicate Order#, enter the sales order number that you want to duplicate. The Customer assigned to the order appears below.
5
For New Customer?, if you are duplicating the order for a different customer, enter the customer number. Leave blank if the new order is for the same customer.
6
For New Warehouse?, if you are duplicating the order for a different customer, enter the warehouse number. Leave blank if the new order is for the same warehouse.
7
For Reprice Sales Order?, indicate if you want to use current pricing for the customer or use the pricing from the original sales order. Choose from the following:
•
Y - Use the current pricing for the customer and products on the order.
•
N - Use the same pricing that was entered on the original sales order.
8
Press F2-Save. The line items from the original order number are copied to the new order. The Sales Order Header screen appears. You can verify and edit the header and line items as needed.
