Price Maintenance Audit Report by User Id
The Price Maintenance Audit Report by User Id displays, for a user ID, all changes a user made in Price Matrix Maintenance, Quick Matrix Entry, or in Quote/Order Created Price Matrix. Refer to Product Pricing for information on setting up price matrices.
After you run the report, you can choose to purge the product information so that it is not included the next time you select the report.
Track Changes
Price Matrix Maint Audit in File Update Parameters must be set to Y to record changes made in Price Matrix Maintenance.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Pricing Reports > Pricing Audit Reports > Price Maintenance Audit Report by User
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
User Identification. The user ID number of the person who changed price matrix.
•
Date Range. The beginning and ending dates for which you want to view changes.
Column Descriptions
Column Heading
Description
Matrix Type
The matrix type. Refer to Price Matrices.
Cust/CPG
The customer number or customer price group, depending on the matrix type.
PN/PPG/PLINE/VN/VLINE/MLINE/
PRC RNG
The part number, product line, vendor number, vendor line, or product price range, depending on the matrix type chosen.
Whse
The selling location for the price matrix, if it is warehouse specific.
Type
Indicates the type (level) of price change:
•
P - Product level.
•
V - Product warehouse/vendor level.
•
W - Product warehouse level.
Ent-Date
The date the matrix was modified.
Ent-Time
The time of day that the matrix was modified.
User
The user ID of the person that modified the matrix.
Description
The description of the user ID.
Description
The description of the field that was changed.
Before Value
The data in the field before the change.
After Value
The data in the field after the change.
