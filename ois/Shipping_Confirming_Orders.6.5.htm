Setting Shipping Confirmation Maintenance Options
The Email Ship Notice flag in the sales order header Order Additional Information Screen indicates if and how a shipping confirmation is transmitted for the sales order. You can set a default for each customer or customer ship-to account and also determine the person, fax number, or email address where the confirmation should be sent.
The following hierarchy is used to determine which email address to use:
•
1st - E-Mail Address in Customer Ship-To Maintenance.
•
2nd - Purchasing Email in Customer Maintenance > F8-Contact.
•
3rd - Email Address in Customer Contact Management Maintenance.
•
4th - Email Address in Customer Maintenance.
Email Document Text Maintenance
Before you can use text emailing for shipping confirmations, you must first set up a document code.
To set up a document code for shipping confirmations:
1
From the Systems menu choose Miscellaneous Controls > Email Document Text Maintenance to open the Email Document Text Maintenance screen.
2
For Document Code, enter ORDSHP for shipping acknowledgements.
3
If the customer has a language indicator (set in Customer Maintenance > F7-Order), enter the language code to use for this type of text email.
4
Enter the description of the text email code (e.g., Shipping Confirmation).
5
Press F2-Save to save the document code.
Customer Maintenance
To set shipping confirmation information for a customer:
1
From Customer Maintenance press F7-Order > F5 Add-Info to open the Additional Order Information screen.
2
For Email Ship Notice, determine how shipping confirmations are transmitted to the customer. Choose from the following:
•
E - Email
•
F - Fax.
•
P - Print.
•
Y - Text email (refer to Text Emailing an Individual Shipping Confirmation).
•
N - No shipping confirmation is sent.
3
Press F2-Save 3 times to save the shipping confirmation information.
Warehouse Maintenance
To set up shipping confirmation information for a warehouse:
1
From the Warehouse Maintenance screen, press F6-Printing to open the Continuous Printing Setup screen.
2
For Shipping Conf, determine if shipping confirmations print, fax, or email continuously or by batch. Choose from the following:
•
Y - Shipping confirmations print, fax, or email when the order is released for invoicing in Release Confirmation Entry or Warehouse Release Confirmation Entry. Text email can only be set up for continuous printing.
•
N - Shipping confirmations print, fax, or email by batch (Refer to Transmitting Shipping Confirmations by Batch).
•
D - If using multiple printers, prompts for the printer queue in Release Confirmation Entry or Warehouse Release Confirmation Entry.
3
For Prt, enter the port number (created in Continuous Printer Port Maintenance) used to control continuous printing.
4
For Form Queue, enter the printer number where shipping confirmations print.
5
Press F2-Save twice to save the printer information.
