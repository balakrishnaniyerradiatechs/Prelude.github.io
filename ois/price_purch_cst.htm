Purchase Cost screen
The Purchase Cost screen in Price Matrix Maintenance allows you to define the cost to use on a purchase order. If a line item on a sales order has a B supply code and a purchase cost matrix exists, then this cost is used when the line item is linked to a purchase order.
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Maintenance > Price Matrix Input screen > F5-Purchase Cost
Field and Function Key Descriptions
Field/Function Key
Description
Cost Method
Determines how the purchase cost is calculated for line items that meet this matrix criteria:
•
* - Use a percentage of a price/cost field. Enter the Basis and the Percent.
•
$ - Price/Cost. Use this option if the matrix is for a specific product. The cursor moves to the Cost field to allow you to enter the cost.
•
T - Use a discount break table (refer to Discount Break Tables). The system determines a cost based upon the discount level. Enter the discount break Table number to use. The system determines a purchase cost based upon the discount level.
Cost
The cost of the product, if you chose $ for Cost Method.
Basis
The price or cost column (from Product Maintenance > F7-Pricing) to use as the base amount to calculate the discount or markup.
Percent
If you entered a price column in Basis, enter the percentage to take off of the price. If you entered cost column in Basis, enter a percentage over the cost to calculate the purchase order cost.
Table
The discount break table number to use if you chose T for Cost Method.
Commission Cost Override
Determines if the purchase cost calculated in this screen overrides the commission cost in Sales Order Entry:
•
N - Purchase cost does not override commission cost.
•
Y - Commission cost for the line item is calculated using the parameters defined in this screen. This calculated commission cost may be overridden if a cost is defined in the Commission Cost Entry screen (refer to Overriding Commission Cost).
Vendor Number
The Vendor Number to assign to the purchasing cost. When a line item with a B supply code is entered on a sales order that has this vendor as its primary vendor, this purchase cost is used on the linked purchase order.
F2-Save
Saves the purchase cost information.
