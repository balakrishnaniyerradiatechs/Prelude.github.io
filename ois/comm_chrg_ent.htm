Commission Charges Entry screen
You can use the Commission Charges Entry screen to add freight or any other charges to a sales order to adjust commission either before or after it is invoiced. Refer to Adding Charges to an Invoice for more information.
Update Commissions in Accounts Payable Parameters (refer to Setting System Parameters for Commission) determines how to update commission when cost is adjusted in Commission Charges Entry.
•
C - Create a commission record for the additional charge. The charge appears on the commission report as C followed by a 6-digit invoice number.
•
A - Delete the original commission record and re-create it to include the additional charge. The invoice date is updated to the date of the new invoice.
Menu Path: Order Processing > Additional Subsystems > Commission Subsystem > Commission Charges Entry
Field and Function Key Descriptions
Field/Function Key
Description
Order Number
The order number to change, if the sales order is not yet invoiced.
Order History Number
The order number to change, if sales order is already invoiced.
Customer
The customer assigned to the order.
Invoice Number
The invoice number, if the order is already invoiced.
Charge Type
The type of charge to add to the order:
•
F - Freight.
•
L - Labor.
•
M - Miscellaneous.
•
R - Remnant.
Charge Amount
The value of the charge to add to the order.
F2-Update
Saves and updates the order information.
