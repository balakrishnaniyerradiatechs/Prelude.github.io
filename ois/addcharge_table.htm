Ship Via Additional Charge Screen
The Ship Via Additional Charge screen allows you to set merchandise amounts that incur additional freight charges. These charges appear on the Sales Order Completion screen for sales orders that use this ship via type.
Menu Path: Order Processing > Maintenance > Customer Control Files > Ship Via Codes > F8-Add Chrg
Field and Function Key Descriptions
Field/Function Key
Description
Additional Charge Type
The type of charge to display on the Sales Order Completion screen for orders that use this ship via code:
•
Delivery.
•
Handling.
•
Inbound freight.
•
Minimum charge.
•
Outbound freight.
Refer to Using the Sales Order Completion Screen for more information.
Merchandise $
The merchandise amount that incurs an additional freight charge.
Charge $
The amount to charge for the corresponding merchandise amount.
F2-Save
Saves the additional charge information.
