Quotation Header screen
The Quotation Header screen allows you to enter header information for a quote. Refer to Entering a Quote.
Menu Path: Order Processing >Additional Subsystems Quotation Subsystem > Quotation Entry > F5-Hdr
Field and Function Key Descriptions
Field/Function Key
Description
Quote Type
Indicates the type of quote:
•
01 - Regular Quote. This type is purged based on the expiration date when you run the Disposition Analysis (refer to the Quotation Disposition Detail Report).
•
02 - Planned Quote. When you convert the quote into a sales order, you can use a multiplier to change all the quantities. This type of quote is not purged.
•
03 - Repair Type Quote. Allows you to quote repair costs to the customer.
•
04 - Standing Quote. Opens the Standing Order Maintenance screen to allow you to enter information to convert the quote to a sales order on a regular basis for using Standing Quote Conversion. Refer to Converting Standing Quotes.
Purch Contract
The person at the customer’s location who is purchasing the material. If you use Contact Management, you can track sales information by purchasing contact.
Ship-To Number
The customer’s ship-to address. If this is not correct, choose from the following:
•
Enter an existing ship-to address for this customer.
•
D - Duplicate the customer’s main address for the ship-to address. This is the default if there are no ship-to addresses set up for the customer.
•
M - The Ship-To Address Screen appears to allow you to manually enter a new, one-time address for this order.
•
P - Customer pick-up.
•
C - Use the purchasing contact address for the ship-to address.
•
CP - End user pick-up.
Contact
The name of the person placing the order. The default comes from the Customer Contacts Screen (Customer Maintenance > F8-Contact) if one is available. Otherwise it defaults from the Contact name on the Customer Maintenance screen.
Customer P/O#
The customer purchase order number.
Cust Release#
The customer’s release number. This number may or may not be mandatory depending on how Require Release Number is set in the customer’s Order Information Screen.
Job Number
The customer’s job number. A job number may or may not be mandatory depending on if you use Job Accounting and how Require Job Number is set in the customer’s Order Information Screen.
Ship Via
Indicates how the customer will receive the material. Ship via codes are set up in the Ship Via Maintenance Screen. Incompatible ship vias for a customer are set up in the Customer Maintenance Incompatible Ship Via Maintenance Screen or the Customer Ship-to Maintenance Incompatible Ship Via Maintenance Screen
Freight Terms
The freight terms code that determines if inbound or outbound charges apply to the sales order when it is confirmed in Release Confirmation Entry. The default comes from the customer’s Freight Information Screen (Customer Maintenance > F7 > F7). Freight codes are established in the Freight Terms Maintenance Screen.
Payment Terms
The accounts receivable payment terms code for orders that indicate when payment is due, payment conditions, and any applicable discounts. These codes are set up in A/R Terms Maintenance and assigned to customers as a default in the Customer Maintenance Screen.
Email Address
The email address for the associated Contact.
 
Fax
The fax number for the customer. The default comes from the purchasing contact for the customer if one exists in the Customer Contacts Screen. Otherwise it uses the fax number on the Customer Maintenance Screen.
Phone
The phone number for the customer. The default comes from the purchasing contact for the customer if one exists in the Customer Contacts Screen. Otherwise it uses the phone number on the Customer Maintenance Screen.
Ext
The phone extension number.
Sell Warehouse
The default warehouse for this customer as assigned in the Customer Maintenance Order Information Screen. You can change the selling warehouse if you want a different warehouse to receive credit for the sale.
Ship Warehouse
The default shipping warehouse for this customer as assigned in the Customer Maintenance Order Information Screen. You can change the shipping warehouse if you want the material to ship out of a different warehouse.
Written By
Defaults to the person logged on.
Salesrep One
The primary sales representative assigned to this customer. The default comes from Salesrep One in the Customer Maintenance Screen. Sales representative are defined in the Sales Representative Maintenance Screen.
Salesrep Two
The secondary sales representative assigned to this customer (e.g., an inside salesrep). The default comes from Salesrep Two in the Customer Maintenance Screen. Sales representative are defined in the Sales Representative Maintenance Screen.
Entry Date
The date the quote was entered. Assigned by the system.
Quote Date
Defaults to today’s date. Change as necessary.
Required
The date the customer is requesting to have the material.
Expiration
The date the quote will expire if not accepted by the customer.
Follow Up
The date that you want to be reminded to follow up on the quote if it is not yet converted or deleted (the default is set at the company level). The quotes appear in the Quote Follow Up Inquiry and the Quote Follow Up Report. Leave blank if you do not want to be notified to follow up on the quote.
Route Number
If you use delivery trucks and follow delivery routes, you can enter a route number set up in the Route Maintenance Screen that contains valid delivery date and time information for each route.
The default comes from the customer’s Customer Route Maintenance Screen or the Ship To Route Maintenance Screen, depending on how the order is entered.
Delivery Seq
If you entered a Route Number, this number indicates where in the delivery sequence the customer will receive the order.
Currency
The country code that indicates the type of currency that this customer uses, if you use multiple currency. The default is set in the customer’s A/R Information Screen.
Curr Rate
The exchange rate for the currency type.
F2-Save
Saves the quotation header information.
F5-Add Info
Opens the Quotation Additional Information screen to allow you to verify salesrep and delivery information.
F6-Ship-To
Opens the Ship-To Address Screen to allow you to view or edit the ship-to address.
F7-Tax Jur
Opens the Tax Jurisdiction Information Screen to allow you to verify or edit tax exempt information for the quote.
F8-Instructions
Opens the Instructions screen to allow you to enter any applicable instructions for this quote (refer to Creating Standard Instructions). You can set up standard instructions for each customer to default for all of their quotes (refer to the Customer Maintenance Instructions Entry Screen).
F9-Credit
Opens the Credit Information Screen to allow you to view the credit limit and remaining credit balance information for the customer.
Action Bar > Add Info
Repairs
Opens the Repair Type Screen to allow you to designate the type of repair, for 03 repair type quotes.
Future Commitments
Opens the Future Commitment Override screen to allow you to commit inventory for future orders.
