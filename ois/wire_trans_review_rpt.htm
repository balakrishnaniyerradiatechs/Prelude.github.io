Wire Transfer Review Report
The wire transfer review report displays information on type 05 (wire transfer) sales orders. Type 05 sales orders require a payment by wire transfer before you can ship the material. If the transfer is not received by the cancellation date, the order will be cancelled.
Menu Path: Order Processing > Reports > Daily Order Detail Reports > Wire Transfer Review Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
S/O Cancel Date <. The date by which payment in advance should be received, or the order will be cancelled.
•
Sort By Customer Or Salesrep. Determines the sort order:
-
S - Sort by sales representative.
-
C - Sort by customer.
•
Customer. The customers to include.
•
Salesrep. The sales representative assigned to the sales order.
Column Descriptions
Column Heading
Description
Cust#
The customer number.
Description
The customer description.
Contact
The customer’s purchasing contact (Customer Maintenance > F8-Contacts).
Phone
The customer’s phone number.
Order#
The sales order number.
Ord Date
The date the sales order was entered.
Dollars
The total dollar amount of the sales order.
Canc Dte
The date the order will be cancelled if payment is not received.
