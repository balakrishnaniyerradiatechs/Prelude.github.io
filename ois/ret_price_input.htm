Retail Price Matrix Input screen
The Retail Price Matrix Input screen allows you to create a retail price matrix to give discounts and markups to customer for a period of time by using one of the following:
•
Net price.
•
Percentage markup on a gross price.
•
Gross profit percent.
•
Markup above the net price.
Refer to Creating a Retail Price Matrix.
Menu Path: Order Processing > Maintenance > Pricing Structures > Retail Price Matrix Maintenance > Retail Price Matrix Maintenance > select the matrix type
Field and Function Key Descriptions
Field/Function Key
Description
Beg-Date
The date that the special pricing is to begin. The start date for the first line defaults to the system date. All remaining lines default to the day after the previous expiration date. For example, if the previous line has an expiration date of 06/30/06 then the next line’s beginning date defaults to 07/01/06. If the previous line does not have an expiration date then the next line is not available.
Exp Date
The date that the special pricing is to end. Leave blank if there is no expiration date.
Mtd
Determines the method to calculate the retail prices:
•
$ - Allows you to enter a set Price for the product. Not valid for product range matrices.
•
* - Allows you to enter a markup or a discount to calculate the price.
•
G - Allows you to enter the % Markup for the gross profit percentage that you want to earn.
•
N - Allows you to enter a % Markup above net price.
Price
If you chose $ for Mtd, enter the price.
%Markup
If you chose *, G, or N for Mtd, you can enter the markup percent.
Basis
If you chose * for Mtd, you can enter the column price or cost to use from Product Maintenance.
Percent
The percent to multiply against the basis to calculate the price. The discount entered will be a percentage over a price column if a price column was entered in Basis or a percentage over a cost if a cost was entered in Basis.
Key Code
The key code to print retail price stickers.
PPFlg
Indicates if this price is for a promotion:
•
Y - Flag the price as a promotion (information only).
•
N - Do not flag the price as a promotion.
F2-Save
Saves the retail price information.
F4-Del
Deletes the retail price matrix.
Action Bar
Copy
Opens the Customer Matrix Copy screen to allow you to copy a customer or customer price group matrix to a single customer, multiple customers, or to a list of customers. Refer to Copying a Retail Price Matrix.
