Assigning Default Procedures to Inspection Types
You can assign a default inspection procedure for each inspection type. Inspection types are used Inspection Results Entry to indicate when the inspection is performed.
To set up a default procedures for inspection types:
1
Select Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Quality Inspection Procedure Default Maintenance to open the Quality Inspection Default Maintenance screen.
2
For each Inspection Type, enter a default procedure number set up in Quality Inspection Procedure Maintenance (refer to Setting Acceptable Quality Levels for Inspection Procedures). This is the procedure number that will appear in Inspection Results Entry.
3
Press F2-Save to save the default procedure information.
