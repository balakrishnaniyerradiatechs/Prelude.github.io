Entering a Direct Ship Sales Order
A direct ship sales order is shipped directly from the vendor to the customer. Each direct ship line item, regardless of its stock status, is backordered since it is shipped directly from the vendor and not from stock.
To create a direct ship sales order:
1
Follow the steps for Selecting a Customer.
2
In the Sales Order Header screen (refer to Entering Sales Order Header Information), enter an Order Type of 04 (direct).
If a line item is to ship direct in an order type that is not 04 (e.g., 01 regular order), enter the direct line item using a supply code of D and then you can generate the purchase order for the line item (refer to Linking a Purchase Order to a Backorder).
3
In the Sales Order Entry screen, enter each Product to ship to the customer. The Line Item Detail screen shows the quantity for each product as backordered since it is coming from the vendor and inventory is not affected. The Sc (supply code) is D (direct) for each line item.
Since the order is going direct to the customer from the vendor, all line items should be from the same vendor.
4
Press F2-Save for the line item. The Cost Prompt screen appears.
5
Enter a cost and press ENTER. A dialog box with the following message appears:
Do You Wish To Automatically Generate A
Purchase Order For This Product?
6
Choose one of the following:
•
Yes - Generate the P/O at the end of the order.
•
No - Allow the P/O to generate as part of Nightly Processing or enter it manually.
7
If you select Yes, the Automatic Purchase Order Vendor screen appears.
8
The primary vendor for the product appears. Enter a different vendor number if necessary. Press F2-Save.
9
Repeat steps 3-8 for each item to add to the order.
10
After you enter all of the products on the order, press F2-Save to complete the order. The following message appears:
Do You Wish To ......
11
Choose from the following:
•
Suspend - Holds the order in suspension. You can generate the purchase order independently (refer to Entering a Direct Ship Purchase Order).
•
Return - Allows you to return to the order to view or edit if necessary.
•
Complete - Opens the Sales Order Completion screen to allow you to enter freight, delivery, or handling charges before you bill the customer. Refer to Using the Sales Order Completion Screen. Choose this option if you want to generate the purchase order or add freight and handling charges.
12
After you press F2-Complete from the sales order completion screen, the following message appears:
Are You Ready To Release This Order For Invoicing?
13
Choose from the following:
•
Yes - The order is held to print an invoice when the vendor invoice is reconciled. Check with your systems administrator to use this option.
•
No - Suspends the order so you can wait to invoice the customer when the vendor sends their invoice to you. You would normally want to choose this option to continue with the purchase order generation.
In either case, a dialog box with the following message appears:
Do You Wish To Proceed WIth Purchase Order Update?
14
Choose one of the following:
•
Yes - Immediately generate the purchase order.
•
No - Allow the purchase order to generate as part of Nightly Processing.
15
If you select Yes, the Generate P/Os for Backorders screen displays the purchase orders for the vendor you chose.
16
Choose from the following:
•
F2-Generate All: Generates purchase orders for the vendors listed.
•
F5-Edit: Allows you to edit the selected purchase order.
•
F6-Generate: Generates one purchase order at a time for the highlighted vendor.
17
When the orders are generated, press F9-Continue. The output dialog box appears.
18
Select the appropriate method for transmitting the purchase order to the vendor.
If you chose not to generate a purchase order during Sales Order Entry, refer to Entering a Direct Ship Purchase Order.
