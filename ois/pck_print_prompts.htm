Pick Ticket Printing screen
The Pick Ticket Printing screen allows you to print tickets by batch. Refer to Printing Pick Tickets.
Menu Path: Order Processing > Batch Ticket Printing > Pick Ticket Printing
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse Number
The warehouse for which you want to print pick tickets.
Sales Order Number
The sales order number to print, if you want to print tickets for just one sales order. Leave blank to print pick tickets for all orders ready to pick.
Printer Form Queue
The printer where the pick tickets are to print.
F5-Pick Ticket Reprint
Allows you to re-print a single pick ticket.
F6--Price Label Printer
Opens the Price Label Printer Prompt screen to allow you to enter the printer number where the price labels are to print. Refer to (refer to Price Labels for information on setting up customers and products to print price labels).
