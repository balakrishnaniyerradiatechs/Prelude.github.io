Setting up Physician Information
Each physician that you use for a customer prescription must have a physical number.
To create a physician number:
1
Select Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > RX Physician Maintenance to open the RX Physician Maintenance screen.
2
Enter the Physician number or enter NEW to assign the next system generated number
Valid numbers for physicians for each warehouse must be set up in Physician Number Range Setup (Order Processing > Maintenance > Customer Control Files, Physician/RX Maintenance).
3
If applicable, enter the number for the Facility associated with this physician. Facilities group physicians that work at the same hospital or location (refer to Setting up Facility Information). If you enter a facility, the following message appears:
Do you wish to update the address line from this facility record?
4
Choose from the following:
•
Yes - The address information from the facility updates the address information for this physician.
•
No - You must manually fill in the address information for the physician.
5
Enter the Physician Name.
6
If you did not use the default facility information, enter the address and phone number information for the physician.
7
Enter the physician’s Email Address.
8
For Rx Renewal Method, enter the method for contacting the physician to renew prescriptions. Choose from the following:
•
P - Print.
•
E - Email.
•
F - Fax.
9
For Rx Copies, if you entered P for Rx Renewal Method, enter the number of copies of the renewal request to print.
10
For Report Method, enter the method for transmitting the prescription report to the physician. Choose from the following:
•
P - Print.
•
E - Email.
•
F - Fax.
Refer to the Monthly Prescription Report and the Expiring Prescription Report.
11
Enter the physician’s License number.
12
Press F2-Save to save the physician information.
