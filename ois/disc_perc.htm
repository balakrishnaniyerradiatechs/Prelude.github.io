Discount Table - Percent screen
The Discount Table - Percent Method screen appears when you enter an asterisk (*) for the Table Method in the Discount Break Table Maintenance screen. You can specify the column and type of break level to calculate discounts.
Menu Path: Order Processing > Maintenance > Pricing Structures > Discount Break Table Maintenance > Table Method = *
You can also access this screen from the Inventory and Purchasing menus.
Field and Function Key Descriptions
Field/Function Key
Description
Basis Field
The price or cost column to use as the base amount to calculate discounts:
•
0 - Suggested retail.
•
1 - Price1.
•
2 - Price2.
•
3 - Price3.
•
4 - Price4.
•
5 - Price5.
•
A - Average cost.
•
B - Product quantity break. Uses the product’s quantity break table to apply the discount defined in this table. Only used for percentage discounts.
•
C - Customer default column. Uses the customer’s default price level before applying the discount defined in this table.
•
O - Cost for sales.
•
R - Replacement cost.
•
S - Standard costs.
Break Level
Defines the minimum break value that qualifies for the corresponding discount percent or price:
•
If the Break Level Type is M, enter the unit of measure that sets this discount.
•
If the Break Level Type is U, enter the minimum quantities to determine the discount.
•
If the Break Level Type is $, enter the minimum value to determine the discount.
%
The discount percentage received when you reach the corresponding Break Level. This percent is applied to the value in Basis to calculate the price for each product. This value displays under Disc% on the Line Item Detail screen of the purchase order or sales order. The discount is a percent off of a price if a price is specified in Basis or a percent over the cost if a cost is specified in Basis. A negative percent gives the opposite result (a percent over a price or a percent off of a cost).
F2-Save
Saves the break table information.
F4-Del
Deletes the break table.
