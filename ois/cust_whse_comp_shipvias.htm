Customer Compatible Ship Vias Inquiry
The Customer Compatible Ship Vias Inquiry lists ship via codes that are valid for a customer/customer ship-to account and warehouse. Incompatible ship vias codes are assigned in Customer Maintenance > F7-Order > F8-Inc Ship Via (refer to the Incompatible Ship Via Maintenance Screen).
If an order is entered with an incompatible ship via code for the product/customer, the order will go on Administrative Order Hold and must be released before the pick ticket will print (refer to the Administrative Order Inquiry).
Viewing Compatible Ship Vias
To list the ship via codes that are available for a customer:
1
Select Order Processing > Inquiries > Customer Info Inquiries > Customer Compatible Ship Vias to open the Customer Compatible Ship Vias screen.
2
Enter the Customer.
3
Enter the Customer Ship-to number, if applicable.
4
Enter the Warehouse.
Ship via codes compatible with this customer/ship-to/warehouse appear.
Field & Function Key Descriptions
Field/Function Key
Description
Code
The ship via code set up for the customer. Ship via codes are set in the Ship Via Maintenance Screen.
Description
The description of the ship via code.
F2-Exit
Closes the inquiry screen.
