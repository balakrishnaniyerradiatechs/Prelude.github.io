Sales Orders With No Credit Card Authorization Report
The Sales Order With No Authorization Code Report lists all open orders with a credit card Pay Type (03) that are not yet pre-authorized.
Menu Path: Order Processing > Processes > Credit Card Processes > Additional Batch Functions > Sales Order With No Authorization Code Report
Column Descriptions

Description

The sales order number.

The total amount of the sales order that is awaiting approval.

The warehouse where the order was entered.
