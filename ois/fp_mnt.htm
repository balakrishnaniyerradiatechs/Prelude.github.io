Floor Plan Number Maintenance screen
If you use floor plan financing and the customer requires a floor plan number in Sales Order Entry, you can use Floor Plan Number Maintenance to create a floor plan number.
Requiring Floor Plan Numbers
A floor plan number is required if both Financial Institution Floor Plan and Plan Number/Approval Req are set to Y in Customer Maintenance > F6-A/R > F8-Floor Plan. Refer to Setting up a Customer to use Floor Plan Financing.
Menu Path: Order Processing > Order Processing > Maintenance > Misc Code Files > Floor Plan
Field and Function Key Descriptions
Field/Function Key
Description
Floor Plan Number
Enter the Floor Plan Number that references the floor plan agreement. You are required to enter this number when you enter a sales order for the customer that uses floor plan financing. Refer to Entering a Floor Plan Sales Order.
Customer Number
The floor plan customer who is set up as a financial institution for other customers and is responsible for payment, whose credit limit is checked, and whose A/R balance is updated when orders are entered for customers assigned to them. Refer to Setting up a Customer to use Floor Plan Financing.
Description
The description of the floor plan.
Expiration Date
The date the floor plan financing expires.
F2-Save
Saves the floor plan information.
F2-Del
Deletes the floor plan information.
