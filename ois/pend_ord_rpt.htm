Pending Orders Report
The Pending Orders report is for warehouses that use the standard pick ticket process, rather than the Warehousing method. The report displays, by ship via type, picking information for sales orders that are available to pick at the time the report is run. Once a sales order release generates from Pick Ticket Printing, the order is no longer included in the report. The report does not include any suspended orders or orders on hold.
Menu Path: Order Processing > Reports > Daily Order Summary Reports > Pending Orders Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The shipping warehouse.
•
Ship Via Code. The ship via codes to include.
•
Show Order Detail. Determines if the sale order numbers for each ship via type display on the report. Choose from the following:
-
Y - Include sales order numbers for each ship via type.
-
N - Do not include sales order numbers for each ship via type.
Column Descriptions
Column Heading
Description
Ship Via
The Ship Via Code as entered in the Sales Order Header screen. The ship via description displays.
#Orders
The number of sales order releases generated in Pick Ticket Printing that are ready to pick, for this ship via type.
#Lines
The number of line items included on the orders that are ready to pick.
#Units
The number of units included on the orders that are ready to pick.
Single Line
Orders
The number of sales order releases generated in Pick Ticket Printing that have only one line item to pick.
Orders
The sales order numbers that are ready to pick. These only display if you chose Y for Show Order Detail in the Selection Criteria.
