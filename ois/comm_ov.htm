Commission Header Override screen
The Commission Header Override screen within the sales order header allows you to override commission information for a sales representative. You can calculate commission as a percentage of the total sale or of the gross profit amount. Refer to the Overriding Commission For a Salesrep for more information.
Menu Path: Order Processing > Sales Order Entry > F5-Hdr Scr > Action Bar > Add Info > Comm Override
Field and Function Key Descriptions
Field/Function Key
Description
Rep
The sales representative to receive commission for this sales order.
Description
The description of the sales representative.
Meth
The method to use to calculate commission:
•
* - Calculate commission as a percentage of the total sale.
•
G - Calculate commission as a percentage of the gross profit.
Rate
The percentage of the sale or gross profit to use to calculate commission.
F2-Save
Saves the commission override information.
