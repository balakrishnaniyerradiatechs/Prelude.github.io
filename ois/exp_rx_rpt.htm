Expiring Prescription Report
The Expiring Prescription Report lists expiration dates for customer prescriptions.
Menu Path: Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance >Expiring Prescription Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Customer. The customers to include.
•
Warehouse. The warehouse that shipped the products
•
Expiration Date Range. The beginning and ending expiration dates to include.
•
Output Options. Determines how to transmit the report
-
P - Print all prescriptions included in the report.
-
E - Email the report if the Report Method for a physician is set to email. Print the report for all other physicians.
-
F - Fax the report if the Report Method for a physician is set to fax. Print the report for all other physicians.
-
D - Use the default Report Method set up for each physician to transmit the reports.
•
Printer. The printer to send the report to.
Column Descriptions
Column Heading
Description
Customer
The customer number that the prescription is issued to.
Description
The customer description.
Physician
The number of the physician that prescribed the medicine.
Phone
The physician’s phone number.
RX#
The prescription number.
Product
The product on the prescription.
Description
The product description.
Qty
The quantity included in the prescription.
Refills
The number of refills available for the prescription.
Exp Date
The date the prescription expires.
Instructions
Instructions set up for the prescription.
