Automatically Billing Consignment Orders
If the customer decides to keep the material on an 08 consignment order, you can use the Auto Consignment Billing Update to set the ship quantity for each line item and to release the order for invoicing. If you need to change any quantities before billing the order, refer to Manually Billing Consignment Orders.
To automatically invoice an 08 consignment order:
1
Select Order Processing > Processes > Auto Consignment Billing Update to open the Auto Consignment Billing Update screen.
2
For Consign Order #, enter the consignment order number to bill.
3
Press F2-Update. All products on the consignment order are marked as shipped and the order is set up for invoicing (refer to Invoicing) to bill the customer.
