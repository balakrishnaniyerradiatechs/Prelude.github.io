Using the Sales Order Completion Screen
When you finish a sales order and choose Complete (refer to Finishing the Sales Order), the Sales Order Completion screen appears.
1
Verify or change the Terms Code. This code represents the accounts receivable payment terms that indicate when payment is due, payment conditions, and any applicable discounts.
2
The total value of the Merchandise displays. You cannot change the amount in this screen. If you need to change any prices, you must save the order, suspend it, then go back into Sales Order Entry.
3
Enter the Inbound Freight charge if you want to bill the customer for any inbound freight charges that you paid.
4
Enter the Outbound Freight if you charge to ship material to the customer (if known, otherwise this information can be entered when the order is confirmed).
5
For Handling, you can manually add any applicable handling charges or let the system calculate the handling charges according to settings in the customer’s Freight Information Screen. Refer to the following explanation for defining handling charges at the system level.
Freight/Handling Charges Maintenance Options
You can automatically add freight and handling charges to customer’s orders using the merchandise amount or the outbound freight amount.
Addl Frt/Handling Chg setting (Customer Maintenance > F7-Order > F7-Frt Info) determines how the charges apply to each customer. The Freight Class setting in the Freight Information screen for a product (refer to the Product Maintenance in the Inventory documentation) also determines the amount to charge for freight. Refer to the Freight Information Screen for more information.
The following flags in Order Processing Parameters > F7-Add/Frt Handling determine how freight and handling charges are calculated on sales orders:
•
Add Frt/Handling Min: Defines the minimum amount to add to each invoice. If the result of the base amount (defined in Add Frt/Handling Bas) multiplied by the percentage (defined in Add Frt/Handling Prc) is less than this amount, then this value is used.
•
Add Frt/Handling Max: Defines the maximum amount to add to each invoice. If the result of the base amount (defined in Add Frt/Handling Bas) multiplied by the percentage (defined in Add Frt/Handling Prc) is more than this amount, then this value is used.
•
Add Frt/Handling Prc: The percentage to multiply by the base (defined in Add Frt/Handling Bas) to determine the amount of the freight or handling charge. If left blank, no additional freight or handling charge is calculated.
•
Add Frt/Handling Bas: Defines the basis to use to multiply by the percent defined in Add Frt/Handling Prc to calculate freight. Set to M to use the Merchandise amount from the Sales Order Completion screen. Set to O to use the Outbound Freight amount from the Sales Order Completion screen.
Freight Handling (Order Processing Parameters > F5-Add Info > F5- Add Info) determines how freight terms are used to calculate freight in Sales Order Entry and Release Confirmation Entry.
•
Y - Skips the Inbound Freight and Outbound Freight fields in the Sales Order Completion screen and in Release Confirmation Entry if the freight terms code assigned to the sales order has Inbound Freight and Outbound Freight set to N. The freight charge is automatically calculated according to the freight terms.
•
N - Allows you to enter the Inbound Freight and Outbound Freight in both the Sales Order Completion screen and in Release Confirmation Entry, to override the system calculated freight no matter how the freight terms are set.
•
A - Works the same as N except that the freight amount entered in the release header is only carried to the completion screen if the freight terms codes has Inbound Freight and Outbound Freight set to Y. This option is used to allow you to enter freight without having to determine if the freight should be billed to the customer. The system calculates freight charges according to the freight terms.
6
Enter the Delivery charge, if known. You can also enter this information when the order is confirmed).
7
Enter the Pickup charge (primarily used with the rental module).
8
If this is a credit memo, enter a Restocking Charge, if applicable.
9
A calculated charge may appear in Minimum Charge to increase the order amount to the minimum amount defined by one of the following:
•
The Min Invoice Amount in the customer’s Order Information Screen.
•
The Min Order Amount for the ship via code (refer to the Ship Via Freight Info Screen).
•
The minimum order amount defined at the company level.
You can manually enter a Minimum Charge if necessary.
10
For Disc/Allowance, if applicable, enter a discount amount to apply to the entire order by using one of the following methods:
•
By dollar amount (e.g., 10.75).
•
By percentage of the Merchandise amount (e.g., P15 for a 15 percent discount).
In either case, you may be required to enter a Discount Reason Code. Refer to Creating Discount Reason Codes.
Discounts Allowed
The following flags in Option Parameters > Order Processing Parameters > F5-Add Info > F5-Add Info affect discounts:
•
Maximum Order Discount defines the maximum amount that can be entered in Disc/Allowance on the Sales Order Completion and the Quotation Completion screens. If this field is blank, then any Disc/Allowance can be entered. You can override this value for a user ID In User Identification Maintenance > F5-Administrative > Max Order Disc.
•
Prompt for Disc.Allow Reas must be set to Y to require a discount reason code when a discount is entered in the Sales Order Completion screen.
11
The Sales Tax and Total Sale amounts are automatically calculated. If necessary, you can change or verify the tax calculations by pressing F5-Tax Jur to open the Sales Tax Jurisdictions screen.
12
Depending on how the A/R Terms are set up for the customer, you may be required to enter a Deposit Required amount for backordered items.
13
Press F2-Complete. Depending on the Order Type (refer to Sales Order Types), choose one of the following:
•
For an 05 counter order with cash terms, refer to Taking Payment for an Order.
•
For an 04 (direct), 06 (confirmation), or 08 (confirmation consignment) order, choose Suspend. You cannot invoice the customer until the vendor invoice is reconciled (refer to Invoicing).
•
For all other order types, you can print the pick ticket (refer to Printing Pick Tickets).
