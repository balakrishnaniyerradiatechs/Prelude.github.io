Quote Order Inquiry
The Quotation Order Inquiry lists all open quotes for a customer that are not purged.
Selecting Open Quotes
To list open quotes for a customer:
1
Select Order Processing > Additional Subsystems > Quotation Subsystem > Inquiries > Quotation By Customer to open the Quote Order Inquiry screen.
2
Enter the Customer.
3
Select the Sort sequence for the customer’s quotes. Choose from the following to sort the list by:
•
None.
•
Quote number.
•
Quote type.
•
Quote date.
•
Expiration date.
•
Customer P/O number.
•
Customer release number.
•
Quote total.
•
Descending quote number.
•
Descending quote type.
•
Descending quote date.
•
Descending expiration date.
•
Descending customer purchase order number.
•
Descending customer release number.
•
Descending quote total.
4
Determine if you want to use any Filters to limit the list of quotes. Choose from the following:
•
No. Lists all open quotes for the customer.
•
Yes. Allows you to use the following selection criteria to limit the list:
-
Quote Number.
-
Cust Release Number.
-
Purchasing Contact.
-
Customer P/O Number.
-
Expiration Date Range.
Open quotes for the customer appear.
Field & Function Key Descriptions
Field/Function Key
Description
Quote#
The customer’s quote number.
Notes
An asterisk (*) indicates that notes exists in the quote header.
Type
The quotation type:
•
01 - Regular quote.
•
02 - Planned quote.
•
03 - Repair type quote.
•
04 - Standing quote.
Disp#
The disposition analysis code that indicates if the quote is converted or deleted. If the quote was converted to a sales order, 00 displays. If it was deleted, the disposition code displays (refer to Creating Quote Disposition Codes). Converted or deleted quotes appear in this inquiry until they are purged in the Disposition Analysis reports (refer to the Quotation Disposition Detail Report).
Hold
Indicates if the quote is on hold:
•
Q - The quote is on Administrative Quote Hold. Refer to the Administrative Quote Inquiry.
•
Blank - The quote is not on Administrative Quote Hold.
Qte Date
The date the quote was entered.
Exp Date
The date the quote expires. If the customer does not accept the quote by this date, the quoted price is no longer valid.
Customer P/O#
The customer’s purchase order number.
Release#
The customer’s release number for the purchase order, if applicable.
Merchandise
The total value of merchandise quoted to the customer.
Conv
Indicates if the quote was converted to a sales order:
•
Y - The quote was converted to a sales order.
•
N - The quote was not converted to a sales order.
F2-Finish
Closes the Quote Order Inquiry screen.
F5-Review
Opens the Quotation Display screen to allow you to view detail information for the quote.
F6-Quote#
Allows you to enter a quote number to go directly to.
F7-QOE
Opens the Quote Order Entry screen to allow you to review, edit, or convert the quote.
F8-Log
Opens the Quote Log Inquiry to allow you to view every transaction that occurred for this quote.
F9-Goto Ln
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Returns to the previous page if more than one page of quotes are included in the inquiry.
sF6-Next Group
Proceeds to the next page if more than one page of quotes are included in the inquiry.
