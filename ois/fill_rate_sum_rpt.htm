Fill Rate Summary Report
The Fill Rate Summary Report displays the percentage of line items that were filled when sales orders shipped to customers. The information is updated during invoice processing.
Kit Components
Fill rate statistics for kits are based on the components filled, not the kit parent itself. For example, if one kit parent has 2 components, the number completed is 2.
Menu Path: Order Processing > Reports > Daily Order Summary Reports > Fill Rate Summary
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Shipping Warehouse. The warehouse that shipped the products.
•
Customer. The customers for whom you wish to view the sales order fill rate.
Column Descriptions
Column Heading
Description
Order #
The sales order number.
Invoice#
The invoice number.
Cust#
The customer number.
Description
The customer description.
# Lines
The number of lines on the sales order.
# Compl
The number of lines that shipped complete.
Perc
The percentage of sales order lines that shipped complete (fill rate).
