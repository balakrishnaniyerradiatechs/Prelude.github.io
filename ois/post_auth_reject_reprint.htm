Credit Card Post Authorization Rejection Report
The Credit Card Post Authorization Rejection Report lists transactions that were rejected during Credit Card Post Authorization (refer to Post-Authorizing Credit Card Transactions). You can also access this report using Post Authorization Rejection Report Reprint if you need to re-print the report.
Menu Path: Order Processing > Processes > Credit Card Processes > Additional Batch Functions > Post Authorization Rejection Report Reprint
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Whse. The selling warehouse where credit card transactions took place.
•
Cc Type. The credit card type to include.
Column Descriptions
Column Heading
Description
Type
The type of credit card. Refer to Assigning Credit Bureau Codes.
Account Number/Error
Line 1: The credit card number.
Line 2: The error message that explains why it was rejected (e.g., past the authorization date, an additional amount added since post-authorization).
Exp
The expiration date of the credit card.
Appr
The approval code assigned in Authorization Code on the Additional Credit Card Information screen when the sales order was pre-authorized. Refer to Pre-Authorizing a Sales Order.
AVS
The Address Verification Service approval number if you use AVS.
Order
The sales order number.
Ref Key
The credit card batch used in Rejection Maintenance.
Pay Amt
The amount approved and paid on the invoice.
Tran Dte
The date the order was invoiced and post-authorized.
