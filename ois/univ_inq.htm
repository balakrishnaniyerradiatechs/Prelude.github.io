Universal Inquiry
The Universal Inquiry allows you to look up multiple types of sales orders, purchase orders, or work orders. You can enter selection criteria to limit the orders to view.
Universal Inquiry in File Update Parameters must be set to Y to track information for the Universal Inquiry.
Using the Universal Inquiry
To look up sales orders, purchase orders, or work orders:
1
Select Order Processing > Inquiries > Universal Inquiry to open the Universal Inquiry screen.
2
For Document Type, indicate the types of orders that you want to look up. Choose from the following:
•
S - Sales orders.
•
P - Purchase orders.
•
W - Work orders.
3
For Source, select the types of documents that you want to view. Depending on which Document Type you chose, the following options are available to limit the types of orders to view:
•
Sales orders:
-
A - All sales orders and quotes.
-
S - Open sales orders.
-
H - Historical (invoiced) sales orders.
-
D - Deleted sales orders.
-
Q - Quote orders.
-
T - Converted quote.
-
B - Invoiced crib orders.
-
SH - Open and invoiced sales orders.
-
SQ - Open sales orders and quotes.
-
QH - Quotes and invoiced sales orders.
-
QT - Quotes and converted quotes.
-
SB - Open and invoiced crib orders.
•
Purchase orders:
-
A - All documents.
-
P - Open purchase orders.
-
H - Historical (reconciled) purchase orders.
•
Work orders.
-
A - All documents.
-
W - Open work orders.
-
H - Historical (completed) work orders.
-
Q - Work order quotes.
-
SH - Open and invoiced work orders.
-
SQ - Open work orders and quotes.
-
QH - Quote and completed work orders.
Depending on which Document Type you chose, one of the following screens appear:
 
Sales Orders/Quotes - Field & Function Key Descriptions
Field/Function Key
Description
Start Date
The beginning order entry date for which you want to view orders.
End Date
The ending order entry date for which you want to view orders.
Customer
The customer number, if you want to limit the inquiry to one customer.
Ship-To Number
The customer’s ship-to number, if you want to limit the inquiry to one ship-to address.
Header Keyword
A header keyword, if you want to search by a keyword in the order header.
Product
A product number, if you want to lookup orders by product.
Line Keyword
A line item keyword, if you want to lookup orders by a product number keyword.
F2-Execute
Opens the Sales Order/Quote Review screen to list all sales orders that meet the criteria defined in the Sales Order/Quotes Lookup screen.
F5-Enter Order#
Allows you to enter a sales order number for direct access to the Sales Order Display screen.
F6-Enter Order History#
Allows you to enter an invoiced sales order number for direct access to the Order History Inquiry Display screen to view detail information on the invoiced order.
F7-Enter Quote#
Allows you to enter a quote number for direct access the Quotation Display screen.
 
Purchase Orders - Field & Function Key Descriptions
Field/Function Key
Description
Start Date
The beginning purchase order entry date for which you want to view orders.
Vendor
The vendor number, if you want to limit the inquiry to one vendor.
Ship-Frm Number
The vendor’s ship-to number, if you want to limit the inquiry to one ship-from address.
Header Keyword
A header keyword, if you want to lookup up orders by a keyword in the purchase order header.
Product
A product number, if you want to lookup orders by product.
Line Keyword
A line item keyword, if you want to lookup orders by a product number keyword.
F2-Execute
Opens the Purchase Order/Purchase Order History Review screen to list all open and reconciled purchase orders that meet the criteria defined in the Purchase Order/History Lookup screen.
F5-Enter P/O#
Allows you to enter the purchase order number for direct access to the Purchase Order Display screen.
F6-Enter P/O History#
Allows you to enter the received purchase order number for direct access to the Purchase Order History Display screen.
Work Orders - Field & Function Key Descriptions
Field/Function Key
Description
Start Date
The beginning work order entry date for which you want to view orders.
Bill Of Material
The work order BOM, to limit the inquiry to one BOM.
Header Keyword
A header keyword, if you want to lookup up orders by a keyword in the work order header.
Product
A product number, if you want to lookup orders by product.
Line Keyword
A line item keyword, if you want to lookup orders by a product number keyword.
F2-Execute
Opens the Work Order Review screen to list all work orders that meet the criteria defined in the Work Order/Quote/History Lookup screen.
F5-W/O#
Allows you to enter the work order number for direct access to the Work Order Display screen.
F6-Enter W/O History#
Allows you to enter the invoiced work order number for direct access to the Workorder History Display screen.
F7-Enter W/O Quote#
Allows you to enter the work order quote number for direct access to the Work Order Quotation Display screen.
