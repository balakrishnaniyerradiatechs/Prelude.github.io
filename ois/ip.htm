Printing Invoices
Once the releases are confirmed in Release Confirmation Entry (refer to Confirming Releases) and set up for invoicing, you can print, email, or fax the invoices to the customers. The method of transmission for each invoice depends on how each customer is set up to receive invoices (refer to Setting Maintenance Options for Invoices). To preview the invoices before sending them out, run the Invoice Preview Report (refer to Previewing Invoices).
To print and update invoices:
1
Select Order Processing > Invoice Preview & Printing > Invoice Printing to open the Invoice Print Prompts screen.
2
Enter the Sales Order Number for which you want to print an invoice or leave blank to generate all invoices that are ready to print.
3
Enter the selling Warehouse for which you want to print invoices or leave blank to generate invoices for all warehouses.
4
Enter the Customer Number for whom you want to print invoices or leave blank to generate invoices for all customers.
5
Enter the Invoice Date for the invoices. This date and the Payment Terms calculate the discount and due date for each invoice.
6
Enter the Printer Form Queue where the invoices should print.
7
For Previewed?, choose from the following:
•
Y - Only print invoices that were previewed on the Invoice Preview Report (refer to Previewing Invoices).
•
N - Print all invoices that are ready to print.
8
Select the Invoice Cycle to print (Only invoices for customers with this invoice cycle are generated):
•
M - Monthly.
•
S - Semi-monthly.
•
W - Weekly.
•
D - Daily.
9
For Print Source, indicate which types of invoices to print. Choose from the following:
•
I - Print invoices only.
•
C - Print credit memos only.
•
B - Print both invoices and credit memos.
•
M - Mail duplicates of counter invoices to customers whose Mail Invoices flag is set to Y in the Customer Maintenance Additional Order Information Screen.
10
Enter an Invoice Message if you want a message to print at the bottom of all invoices in this print run (e.g., Thank you for your Business!). Once you enter through all the fields. The following message appears:
Load Invoice Forms In Printer & Align Visually
11
Choose one of the following:
•
Print Invoices - Print the invoices.
•
Mask Print Test - To test the alignment of the invoice forms in the printer. Not necessary for laser printers. After the forms are correctly aligned, select Print Invoices.
•
Abort Printing - Cancel the invoice print process.
12
After you select Print Invoices, the following message appears:
Invoice Printing Complete....
13
Choose one of the following:
•
Process - The invoices printed successfully.
•
Reprint All - None of the invoices printed successfully. This option is also useful if only a couple of invoices printed successfully from a large print job. The system re-prints the invoices and this message appears again.
•
Single Reprint - To re-print invoices individually. This option is useful if a couple of invoices jammed in the printer. Enter the sales order number for the invoice you want to re-print.
•
Abort - Cancel all invoice updating.
14
Once you select Process, the following message appears:Choose one of the following:
Are You Sure You Want to Process?
•
Reprompt - Keep this dialog box open.
•
Yes - Complete invoice processing, update the Order Log Inquiry and move the order to the Sales Order History Inquiry.
•
No - Return to step 14.
