Printing Pick Tickets by Batch
If you don’t continuously print pick tickets (refer to Setting Pick Ticket Rules), the tickets are held in a queue until you print them in Pick Ticket Printing.
To batch print pick tickets:
1
Select Order Processing > Batch Ticket Printing > Pick Ticket Printing to open the Pick Ticket Printing screen.
2
Enter the Warehouse Number for which you want to print pick tickets.
3
Enter the Sales Order Number to print a single pick ticket or leave blank to print all pick tickets in the queue.
4
Enter the Printer Form Queue where you want the tickets to print.
5
If you use price labels, press F6-Price Label Printer and enter the Price Label Printer where labels should print (refer to Price Labels for information on setting up customers and products to print price labels).
6
Press ENTER.
7
Once you choose Execute the following message appears:
Load Pick Ticket Forms In Printer & Align Visually
8
Choose from the following:
•
Print Pick Tickets - Print the pick tickets and create releases.
•
Mask Print Test - To test the alignment of the pick ticket forms in the printer. Not necessary if you are using a laser printer. After the forms are correctly aligned, select Print Pick Tickets.
•
Abort Printing - Cancel the pick ticket printing.
9
After you select Print Pick Tickets, the following message appears:
Pick Ticket Printing Complete...
Last Release Number ---> ###
10
Choose from the following:
•
Continue - If all pick tickets printed successfully.
•
Reprint All - If none of the pick tickets printed successfully. This option is also useful if only a couple of pick tickets printed successfully from a large print job.
•
Single Reprint - To reprint pick tickets individually. This option is useful if a couple of pick tickets jammed in the printer.
To reprint a pick ticket release, choose one of the following:
•
Order Processing > Batch Ticket Printing > Pick Ticket Printing > F5-Pick Ticket Reprint.
•
Order Processing > Batch Ticket Printing > Pick Ticket Reprinting.
If a pick ticket does not print, it may be because the order or product is:
•
On hold.
•
Suspended.
•
Not available.
•
Not at or past the pick date.
