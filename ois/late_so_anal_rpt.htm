Late Sales Order Report
The Late Sales Order report displays, by warehouse, invoiced sales orders that were shipped after the Required Date. Non-stock products and direct ship orders are not included on the report.
Late Orders in File Update Parameters must be set to Y to track late sales order information.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Late Sales Order Report
Column Descriptions
Column Heading
Description
CMTD
The number of orders in each column that shipped in the current month-to-date.
Last MO
The number of orders in each column that shipped last month.
NLAST
The number of orders in each column that shipped 2 months ago.
TODAY
The number of orders in each column that shipped today.
Total Orders
The total number of orders that shipped for each category.
Orders on Time
The number of orders that shipped on-time for each category and the percent of the total shipped that this number represents.
Percent on time = The number of orders shipped on-time / the total number of orders shipped for the corresponding category.
1-5 Days Late
The number of orders that shipped 1 - 5 days late for each category and the percent of the total shipped that this number represents.
Percent 1- 5 days late = The number of orders shipped 1- 5 days late / the total number of orders shipped for the corresponding category.
6-10 Days Late
The number of orders that shipped 6-10 days late for each category and the percent of the total shipped that this number represents.
Percent 6-10 days late = The number of orders shipped 6-10 days late / the total number of orders shipped for the corresponding category.
11-20 Days Late
The number of orders that shipped 1 - 5 days late for each category and the percent of the total shipped that this number represents.
Percent 11-20 days = The number of orders shipped 11-20 days late / the total number of orders shipped for the corresponding category.
21+Days Late
The number of orders that shipped 21 or more days late for each category and the percent of the total shipped that this number represents.
Percent 21 or more days late = The number of orders shipped 21 or more days late / the total number of orders shipped for the corresponding category.
