Creating Quote Disposition Codes
When you delete a quote, you must enter a Disposition Analysis Code that explains why the quote is being deleted.
To create disposition analysis codes:
1
Select Order Processing > Maintenance > Misc Code Files > Quote Disposition Reason Codes to open the Quote Disposition Maintenance screen.
2
For Qte Disp Number, enter a code to identify the quote disposition reason.
3
Enter a Description of the quote disposition reason (e.g., out of stock, price too high, delivery too long).
4
Press F2-Save to save the reason code.
