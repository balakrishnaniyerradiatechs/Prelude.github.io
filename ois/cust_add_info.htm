Additional Customer Information Screen
The Additional Customer Information screen allows you to set up additional data for the customer including the owner’s address and contact information.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Start Date
The date the customer account was created.
Customer Type
If you use customer type codes from the Customer Type Maintenance Screen, you can enter the type code to categorize customers for reports. You can also use type codes for Price Matrix Maintenance.
Territory
The territory number used to group customers for sales and marketing reports. These codes are set up in the Territory Maintenance Screen.
Owner Name
The owner of the business.
Address One
The first line of the owner’s address.
Address Two
The second line of the owner’s address.
Address Three
The third line of the owner’s address.
City
The city of the owner’s address.
State
The state of the owner’s address.
Zip
The zip code of the owner’s address.
Owner’s Phone
The owner’s phone number.
Soc. Sec. Num
The owner’s social security number. This field is masked if viewed from Customer Inquiry.
Date Founded
The date the company was founded.
Vendor Id Number
The vendor number assigned to your company by the customer.
Our Vendor Number
The vendor number from Vendor Maintenance, if this customer is also one of your vendors. Not validated in the system.
Potential Code
The Potential Code that defines the sales potential that this customer may have. Refer to the Marketing documentation for more information.
Broadcast Fax
Determines if promotional broadcast faxes are sent to this customer:
•
Y - The customer receives broadcast faxes.
•
N - The customer does not receive broadcast faxes.
Refer to the Marketing documentation for more information.
Mail?
Determines if the customer receives promotional mailings:
•
Y - Include the customer in promotional mailings.
•
N - Do not include the customer in promotional mailings.
Refer to the Marketing documentation for more information.
Email?
Determines if the customer is included in mass emails to customers:
•
Y - The customer receives broadcast emails.
•
N - The customer will not receive broadcast emails.
Refer to the Marketing documentation for more information.
F2-Save
Saves the customer information.
F5-Keywords
Opens the Additional Keywords Screen. You can create keywords to help when searching for a customer.
F6-Sales Tax
Opens the Customer Sales Tax Entry Screen. You can enter tax jurisdiction codes and tax exempt numbers for the customer.
