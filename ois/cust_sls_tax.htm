Customer Sales Tax Entry Screen
The Customer Sales Tax Entry screen allows you to enter tax exempt or tax information for a customer. Once this information is added for a customer, you can look up exempt information in Customer Maintenance, Customer Ship-To Maintenance, Sales Order Entry, and Quotation Entry.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F5-Add Info > F6-Sales Tax
Field and Function Key Descriptions
Field/Function Key
Description
Tax Jur
The tax jurisdiction code from the Tax Jurisdiction Maintenance Screen.
Description
The description of the tax jurisdiction code.
Taxable
Indicates if this tax jurisdiction applies to this customer’s purchases:
•
Y - Tax is calculated.
•
N - Tax is not calculated.
Sls Tax Type
If Taxable is set to Y, this field indicates if only resale products sold to this customer are tax exempt:
•
R - Only products flagged as resale are tax exempt.
•
Blank - All products are tax exempt.
Exempt Code
Exempt code for the state of Illinois. You can only access this field if the customer and tax jurisdiction are in Illinois.
Exempt Number
The tax exemption number from the taxing authority, if the customer is tax exempt.
Exempt Description
The description of the exempt code that explains why the customer is exempt.
Exp Date
The date the tax exempt number expires. Not required.
F2-Save
Saves the tax information.
F5-Certificate Info
Opens the Sales Tax Certificate Information Screen to allow you to enter tax certificate information on file for this customer.
