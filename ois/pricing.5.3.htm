Price Source Codes
As you enter each product on a sales order for a customer, the Src (source) field in the Line Item Detail screen indicates where the price for this product for this customer is derived from.
The Prelude source codes include the following:
•
C# - Contract price from matrix (e.g., C1=contract from Price Matrix type 1).
•
CB - Contract price from Blanket Contract Maintenance.
•
LT - Lot pricing.
•
M1 - Price Matrix Type 1 - Customer/Product.
•
M2 - Price Matrix Type 2 - Customer/Customer Product Number.
•
M3 - Price Matrix Type 3 - Customer/Product Line.
•
M4 - Price Matrix Type 4 - Customer/Vendor Number.
•
M5 - Price Matrix Type 5 - Customer/Product Price Group.
•
M6 - Price Matrix Type 6 - Customer Price Group/Product.
•
M7 - Price Matrix Type 7 - Customer Price Group/Customer Product Number.
•
M8 - Price Matrix Type 8 - Customer Price Group/Product Line.
•
M9 - Price Matrix Type 9 - Customer Price Group/Vendor Number.
•
M10 - Price Matrix Type 10 - Customer Price Group/Product Price Group.
•
M11 - Price Matrix Type 11 - Customer/Major Group.
•
M12 - Price Matrix Type 12 - Customer Price Group/Major Group.
•
M13 - Price Matrix Type 13 - Customer Number.
•
M14 - Price Matrix Type 14 - Product Number.
•
M15 - Price Matrix Type 15 - Vendor Number.
•
M16 - Price Matrix Type 16 - Product Price Group.
•
M18 - Price Matrix Type 18 - Order Source Product.
•
M19 - Price Matrix Type 19 - Customer Ship-to/Product Number.
•
M20 - Price Matrix Type 20 - Customer Ship-to/Product Price Group.
•
M21 - Price Matrix Type 21 - Customer Number/Product Range.
•
M26 - Price Matrix Type 26 - Customer Price Group/Product Range.
•
M28 - Price Matrix type 28 - Customer/Vendor/Product Line.
•
M29 - Price Matrix Type 29 - Customer Price Group/Vendor Number/Product Line.
•
M30 - Price Matrix Type 30 - Customer Type/Product Price Group.
•
M34 - Price Matrix Type 34 - Product Range.
•
M35 - Price Matrix Type 35 - Product Line.
•
M36 - Price Matrix type 36 - Major Group.
•
M40 - Price Matrix Type 40 - Product/Zip Code.
•
M50 - Price Matrix Type 50 - Customer Number/Rental Product Numbers.
•
M51 - Price Matrix Type 51 - Customer Number – Rental.
•
M52 - Price Matrix Type 52 - Customer Number/Rental Product Line.
•
M53 - Price Matrix Type 53 - Customer Number/Product Prefix.
•
M54 - Price Matrix Type 54 - Customer Price Group/Product Prefix.
•
M55 - Price Matrix Type 55 - Customer Price Group/Rental Product.
•
OV - Override in Order Entry.
The following are from Product Maintenance or Product Warehouse Maintenance:
•
P0 - Retail Price.
•
P1 - Selling Price 1.
•
P2 - Selling Price 2.
•
P3 - Selling Price 3.
•
P4 - Selling Price 4.
•
P5 - Selling Price 5.
•
SP - Sum and Reprice.
•
VDS - Volume Discount.
