Commission Matrix Maintenance screen
The Commission Matrix Maintenance screen is the first screen that appears when you create a commission matrix. Once you enter the matrix type and the matrix criteria, you can use a commission matrix to vary the commission rate and/or the salesrep by customer, customer price group, warehouse, product number, vendor number, product price group, product line, major group, or a combination of these criteria. Refer to Creating a Commission Matrix for more information.
Menu Path: Order Processing > Additional Subsystems > Commission Subsystem > Commission Matrix Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Matrix Type
The type of commission matrix to create. Press F3 to view the types set up for your company in Commission Matrix Types & Sequence Maintenance (refer to Setting System Parameters for Commission).
