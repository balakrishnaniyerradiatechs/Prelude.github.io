Package Insert Code Maintenance screen
You can use the Package Insert Code Maintenance screen to assign package insert code products to a package insert code. The package insert code determines when the free products or literature are sent to a customer (refer to Creating Package Inserts).
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Package Insert Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Insert Code
An alphanumeric code that identifies the package insert.
Description
A Description of the package insert (e.g., tool insert, automotive insert).
Products.
First Order
The package insert product a customer should receive the first time they place an order with your company.
Second Order
The package insert product a customer should receive the second time they place an order with your company.
Third Order
The package insert product a customer should receive the third time they place an order with your company.
Last Order Days
The package insert product the customer should receive if you want to give a free product based on the number of days since their last order. This field works with # Days to give a customer a free product based on the number of days since their last order.
# Days
The number of days since their last order that determines when they should receive the package insert product entered in Last Order Days.
Order Value
The package insert product the customer should receive if you want to give them a free product based on the value of the order. Works with Dollars to determine when to give the customer a free product.
Dollars
The value that determines when the customer receives the package insert product in Order Value.
F2-Save
Saves the package insert information.
F4-Del
Deletes the package insert information.
F5-Products
Opens the Insert Code Product Numbers screen to allow you to give a customer a free product based on a specific product that they buy.
Refer to Creating Package Inserts.
F6-Product Lines
Opens the Insert Code Product Lines screen to allow you to give a customer a free product based on products purchased within a product line. Refer to Creating Package Inserts.
F7-Addtl Options
Opens the Additional Options screen to allow you to limit the number of inserts if a customer is receiving too many inserts. Refer to Creating Package Inserts.
