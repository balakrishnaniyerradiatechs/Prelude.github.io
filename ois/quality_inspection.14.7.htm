Setting Acceptable Quality Levels for Inspection Procedures
For each type of inspection procedure (e.g., by vendor, customer return, work order or P.O. receipt), you can set up acceptable quality levels (AQL) for testing.
To set up acceptable quality levels for inspection procedures:
1
Select Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Quality Inspection Procedure Maintenance to open the Quality Inspection Procedure Maintenance screen.
2
For Procedure#, choose from the following:
•
To display default procedures already set up for each inspection type, press F3.
•
To assign the next available procedure number, press ENTER. Additional procedure numbers may be set up for each inspection type.
3
Enter a Description of the inspection procedure.
4
For Inspection Type, indicate which of the following types of inspections this procedure number represents. Choose from the following:
•
C - Customer return.
•
P - PO receipts.
•
V - Vendor performed.
•
W - Work order manufacturing.
5
For Version, enter the inspection procedure version that is currently active.
6
In the next (unlabeled) field, enter the date this version went into production.
7
Previous Version is updated when you enter a new version number in Version.
8
Under Acceptable Quality Levels (AQL), indicate the type of sample Table to use for accept and reject levels for this procedure number. Choose from the following:
•
S1 – Single Sampling (normal). Refer to the Normal Inspection Table.
•
S2 – Single Sampling (tightened). Refer to the Tightened Inspection Table.
•
S3 – Single Sampling (reduced). Refer to the Reduced Inspection Table.
9
For Standard Level, enter the ANSI General Inspection Level to use from Sample Code Letter Maintenance (refer to Setting up Inspection Levels for Sample Sizes).
10
Enter the default Critical percent for this procedure.
11
Enter the default Major percentage for this procedure.
12
Enter the default Minor percentage for this procedure.
