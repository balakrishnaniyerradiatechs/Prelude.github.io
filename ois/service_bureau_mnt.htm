Credit Bureau Maintenance screen
Prelude has standard credit card bureau codes (e.g., AMEX, MC, VISA) already set up to post to general ledger accounts. For each credit card type you accept, you can accept or change the general ledger chart of account, bank, and tax information.
Menu Path: Order Processing > Maintenance > Misc Code Files > Credit Bureaus
Field and Function Key Descriptions
Field/Function Key
Description
Bureau Code
The credit bureau code to assign to a credit card company (e.g., AMEX, MC, VISA).
Description
The name of the credit card company (e.g. American Express, Master Card, Visa).
Online Phone
The phone number to call if you use the MAPP verification system.
Baud Rate
The baud rate for the phone line if you use the MAPP verification system.
Voice Authorization
The voice authorization code if you use the MAPP verification system.
Station Id
The station ID number if you use the MAPP verification system.
Bank Code
Determines which checking account to use for cash receipts and payables. This code is also used as a prefix for check numbers to make sure they are unique.
Post To Account
Div
The division where the bureau code is valid. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
Reg
The region where the bureau code is valid. A region is a subset of Division - if Division is used (e.g., Southeast Region, Northwest Region).
Loc
The warehouse or location where the bureau code is valid.
Dep
The department where the bureau code is valid. A department is a subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
Ccr
The cost center where the bureau code is valid. A cost center is a subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
Account
The general ledger account to post to for this credit bureau code.
F2-Save
Saves the bureau code information.
F4-Del
Deletes the bureau code information.
F5-Alt State Code
Opens the Alternate State Code screen to allow you to enter alternate state tax jurisdiction codes for this credit card type.
F6-Alt TIDs
Opens the Additional Terminal IDs Per Location screen to allow you to enter alternate terminal IDs for each warehouse. This information is mandatory if you use multiple merchant numbers with Protobase.
F7-G/L Prefix Override
Opens the Credit Bureau G/L Prefix Override screen to allow you to post to a different general ledger account for this credit card.
