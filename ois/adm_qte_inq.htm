Releasing Orders from Administrative Quote Hold
The Administrative Quote Inquiry holds quotes that were put on hold for pricing or other ordering restrictions defined by your company (refer to Defining Options for Administrative Quote Hold for information on how to place quotes on hold). The quote must be released from hold before it can be printed, faxed, or emailed. Your user ID must be set up as an Administrative Quote Manager to allow you to release quotes.
To review or release orders on Administrative Quote Hold:
1
Select Order Processing > Inquiries > Administrative Quote Inquiry to open the Administrative Quote Inquiry screen. The Administrative Quote Manager ID defaults if you are set up as an Administrative Quote Manager.
2
Press ENTER. Quotes on hold appear.
3
Move the cursor to the quote that you wish to review or release and choose from the following:
•
F5-Review - Opens the Quotation Display screen to allow you to review the quote.
•
F6-QOE - Opens the Quote Order Entry screen to allow you to modify the quote. You can also release the quote and send a message to the queue of the employee who entered it.
•
F7-Toggle - Allows you to move back and forth between the original list of quotes on hold and the orders in the hold queue. To send a quote to the hold queue, enter one of the following under Reviewed.
-
C - Customer review.
-
I - Internal review.
-
S - Salesrep review.
•
F8-Release - Allows you to release the quote from hold and send a message to the queue of the employee that entered the quote.
4
Repeat step 3 for each quote that you want to review, edit, or release.
5
Press F2-Save to close the inquiry screen.
 
Refer to the following field and function key descriptions for information on each field in the inquiry screen.
Field & Function Key Descriptions
Field/Function Key
Description
Administrative Quote Manager
Defaults to the Admin Quote Manager ID if you are set up as an Admin Quote Manager.
Quote#
The quotes on Administrative Quote Hold.
N
An asterisk (*) indicates that either admin quote hold notes or quote notes exist for this order.
Customer
The customer number and description.
Slsm
The code for the sales representative assigned to this quote.
SV
The ship via code on this quote. Refer to the Ship Via Maintenance Screen.
Quote Amount
The customer’s total quoted price.
User
The user ID of the person that entered the quote.
Hdr
The reason code to indicate why the quote went on Administrative Hold (refer to Defining Options for Administrative Quote Hold). Multiple codes may be displayed for each quote. Codes in this column refer to the holds affecting the order header:
•
D - The freight dollars were overridden on the Quotation Completion screen.
•
F - The Freight Terms were overridden on the Quotation Additional Information screen in the quotation header.
•
I - The Ship Via Code is incompatible for the customer (refer to Customer Level Administrative Quote Hold Options) or the product (refer to Product Level Administrative Quote Hold Options).
•
J - The Quote exceeded the Quote Limit for the Written By user ID that entered the quote. Refer to User Level Administrative Quote Hold Options.
•
L - All quotes for the customer go on Administrative Quote Hold (If Admin Hold Override is set to Q for the customer (refer to Customer Level Administrative Quote Hold Options).
•
M - The quote was place on Manual Quote hold (refer to Manually Placing Quotes on Hold).
•
O - All quotes and orders for the customer go on Administrative Quote Hold (If Admin Hold Override is set to O for the customer (refer to Customer Level Administrative Quote Hold Options).
•
P - The customer requires a P/O number. Refer to Customer Level Administrative Order Hold Options.
•
Q - The invoice amount is more than the Admin Invoice Amt in the Customer Blanket P/O Numbers Screen, if Require P/O Number (Customer Maintenance > F7-Order) is set to I.
Line
The reason code to indicate why the quote went on Administrative Quote Hold (refer to Defining Options for Administrative Quote Hold). Multiple codes may display for each order. Codes displayed in this column refer to the holds affecting line items within the quote:
•
A - The product is always on hold. Set in Product Maintenance > F5-Add Info > Always Hold Order = Y.
•
C - The price was overridden below cost.
•
G - The price was overridden below GP%.
•
I - The Ship Via Code is incompatible for the customer (refer to Customer Level Administrative Quote Hold Options) or the product (refer to Product Level Administrative Quote Hold Options).
•
N - The price was overridden below the product’s Minimum Price (Product Maintenance > F7-Pricing or Product Warehouse Maintenance >F7-Price/Cost).
•
R - The price was overridden below the customer’s default price.
•
V - A line item’s gross profit percent is below the Mgr Low Level Minimum GP% or more than the Mgr High Level GP% (designated in Order Processing parameters and can be overridden in Product Line Maintenance).
Message
Any notes sent to the administrative quote manager by the Written By sales person when the quote went on hold.
Reviewed
Indicates if the quote was reviewed within this inquiry:
•
Y - This quote was reviewed.
•
N - This quote was not reviewed.
If you want to move the quote to a hold queue for review, enter one of the following:
•
C - Customer review.
•
I - Internal review.
•
S - Salesrep review.
Once moved, use F7-Toggle to see quotes in the hold queue. You can then release the quote from the hold queue if necessary.
F2-Save
Closes the inquiry screen.
F5-Review
Opens the Quotation Display screen to allow you to view detail information for the quote.
F6-QOE
Opens the Quotation Order Entry screen to allow you to modify and/or release the quote from hold.
F7-Toggle
Goes back and forth between the original list of quotes on hold and the hold queue that quotes were sent to (refer to Reviewed). Quotes can be released from either queue.
F8-Release
Allows you to release the quote from hold and send a message to the employee’s queue.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Refresh
Clears and re-enters the data on the screen if changes were made.
Alternate Mgr
Allows you to send the highlighted quote to another manager’s work queue.
Notes
Opens the Notes Display screen to allow you to enter or view notes regarding this quote.
Sort
Allows you to re-sort the order by:
•
Customer number.
•
Quote date.
•
Salesman number.
•
Quote amount.
•
User ID.
•
Quote header reason code.
•
Quote line reason code.
•
Reviewed quotes.
