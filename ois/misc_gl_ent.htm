Miscellaneous Amount Entry screen
The Miscellaneous Amount Entry screen allows you to enter the amount to charge the customer for a miscellaneous product and a reason code that identifies the purpose of the charge.
Menu Path: Order Processing > Sales Order Entry > enter M, MN, or MNP for a product.
Field and Function Key Descriptions
Field/Function Key
Description
Misc. Amount
The amount to charge the customer for the miscellaneous product.
Misc. Reason Code
The reason code that identifies what the charge if for (refer to Creating Reason Codes for Gift Certificates).
Miscellaneous Account: This information defaults from the account information set up for the code in the Miscellaneous Reason Code Maintenance screen.
Div
The division assigned to the account. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
Reg
The region assigned to the account. A region is a subset of Division - if Division is used (e.g., Southeast Region, Northwest Region).
Loc
The warehouse assigned the account.
Dep
The department assigned to the account. A department is a subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
Ccr
The cost center assigned to the account. A cost center is a subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
Account#
The general ledger account assigned to the reason code. The reason code displays.
F2-Save
Saves the miscellaneous amount information.
