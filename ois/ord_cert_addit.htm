Additional Gift Certificate Information screen
The Additional Gift Certificate Information screen appears when you enter a Pay Type of 11 in the Cash Application screen when completing a sales order. You can enter the gift certificate number and view the balance information.
Menu Path: Order Processing > Sales Order Entry > Complete > Cash Application screen > Pay Type = 11-Gift Certificate
Field and Function Key Descriptions
Field/Function Key
Description
Gift Certificate#
The gift certificate number to use for payment (refer to Creating Gift Certificates).
Certificate Balance
The remaining balance on the gift certificate. If the balance is zero, the certificate is void.
Expiration Date
The date the gift certificate expires. If the gift certificate is expired, a warning appears but you can still use the certificate.
F2-Save
Saves the gift certificate information.
