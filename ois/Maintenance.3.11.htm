Defining Order Source Codes
An order source code defines where a sales order or quote came from. Once you create an order source code in Order Source Code Maintenance. you can assign it to a sales order in the Order Additional Information Screen, to identify how you obtained the order. You can also set up a default order source code for a customer in Customer Maintenance (Order Source Matrix Screen) and then give the customer an additional discount depending on how they submit their order.
Do not use V as an order source code. V is reserved for void sales and will automatically flag the order as an auto-reverse.
To create an order source code:
8
Select Order Processing > Maintenance > Misc Code Files > Order Source Codes to open the Order Source Code Maintenance screen.
9
For Order Source, enter a one-digit code that identifies the source of the sales order or quote. Commonly used order source codes are as follows:
•
C - Customer call in.
•
D - Direct order entry.
•
E - Electronic data interchange.
•
K - Special order-telemarketing.
•
L - Sales representative.
•
M - Mail in.
•
P - Point of sale.
•
R - Remote order entry.
•
S - Sales force automation.
•
T - Telephone sales.
•
W - Will call.
•
X - Special order.
10
Enter a Description of the order source (e.g., customer call in, will call, sales rep).
11
Enter the code to display in the Order Log Inquiry. The default is ASG (assignment of order).
12
Press F2-Save to save the order source code.
