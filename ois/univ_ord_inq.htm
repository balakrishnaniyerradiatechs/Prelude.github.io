Universal Order Inquiry
You can use the Universal Order Inquiry to access the following information for a sales order:
•
The Order Log Inquiry.
•
The Sales Order Display screen.
•
The Sales Order History Inquiry.
•
The Open Release Inquiry.
Maintaining Historical Cross Reference Information
Universal Inquiry in File Update Parameters determines if historical cross reference information is maintained for sales orders, purchase orders, work orders, and quotes for the Universal Inquiry (Order Processing module).
•
Y - Maintain cross reference information for sales orders, purchase orders, and work orders for the Universal Inquiry.
•
N - Do not maintain cross reference information for the Universal Inquiry.
Viewing Universal Sales Order Information
To view inquiry screens for a sales order:
1
Select Order Processing > Inquiries > Universal Order Inquiry to open the Universal Order Inquiry screen.
2
Enter the sales Order Number to view.
3
Place the cursor in front of any of the Sources and press F5-Review to open the corresponding inquiry screen.
Field & Function Key Descriptions
Field/Function Key
Description
Sources
The inquiry screens that contain information for the sales order. Move the cursor a source and press F5-Review to open the corresponding inquiry screen.
Order Log Inquiry
Opens the Order Log Inquiry to list every transaction that occurred for a sales order.
Sales Order Inquiry
If the sales order is still open, you can view the Sales Order Display screen which shows detail information for the open sales order.
Sales Order History Inquiry
Opens the Sales Order History Inquiry to list all invoiced sales order for the customer. You can review detail information on each order.
Release Confirmation Inquiry
Opens the Open Release Inquiry to show picking and packing information for sales order releases not yet invoiced.
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the corresponding inquiry screen depending on which Source is selected.
