Zip Exclusions screen
The Zip Exclusions screen allows you to exclude one or more zip codes from the zip code range indicated in the Restricted Product Matrix Input screen.
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Restricted Product Matrix > enter a Matrix Type and State Abbr > F5-Zip Exclusion
Field and Function Key Descriptions
Field/Function Key
Description
Zip Exclusions
The zip codes within the zip code range indicated in the Restricted Product Matrix Input screen that you want to exclude from the restricted product matrix.
F2-Save
Saves the zip exclusion information.
