Freight Information Screen
The Freight Information screen allows you to enter freight terms and other freight information for a customer ship-to address.
Menu Path: Order Processing > Maintenance > Customer Ship-to Maintenance > F5-Add Info > F7-Frt Info
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer associated with this ship-to address.
Ship to Number
The ship-to number assigned to this address.
Freight Terms
The default freight terms code that indicates whether inbound and/or outbound freight is normally charged to this customer ship-to address. You can override freight terms in Sales Order Entry.
Freight codes are established in Freight Terms Maintenance and ensure that applicable freight charges are passed to the customer when the order is confirmed in Release Confirmation Entry. Refer to the Freight Terms Maintenance Screen.
Exc From Small Ord Chk
Determines if the customer is charged a fee when an order does not meet the Small Order Limit established for a ship via code in Ship Via Maintenance (refer to the Ship Via Maintenance Screen):
•
Y - The Small Order Fee is not added to the customer’s orders if the Small Order Limit is not met.
•
N - If an order total is less that the Small Order Limit established for the Ship Via, then the Small Order Fee is added to Inbound Freight in the Sales Order Completion screen.
Freight Shipping Type
The freight shipping type of arrangement you have with this customer for orders shipped by UPS, FedEx, and RPS, if you use the Parcel Shipping System (PSS):
•
CBS - Consignee billing.
•
COL - Freight collect.
•
TPB - Third party billing.
Account#
The consignee’s UPS account number, if applicable.
Freight Pd Limit
The minimum dollar amount the customer’s order must be to receive pre-paid freight. Any sales order over this value does not incur a freight charge. This value overrides the Prepaid Frt Minimum in Ship Via Maintenance, but can be overridden in Customer Ship-To Maintenance.
ShipVia
The ship via code associated with the Freight Paid Limit for prepaid freight. If this field is left blank, the system checks the Prepaid Frt Minimum for all ship via codes with Prepaid Freight set to Y.
Review Date
A review date to check the Freight Paid Limit and Ship Via. This field is optional and is only used for custom reporting.
F2-Save
Saves the freight information for this ship-to address.
