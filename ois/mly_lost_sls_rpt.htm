Monthly Lost Sales Report
The Monthly Lost Sales report lists lost sales information for a warehouse, sales representative, and/or product. Refer to Tracking Lost Sales for more information on lost sales.
Track Lost Sales
Lost Sales? in File Update Parameters must be set to Y to require a lost sales reason code.
Menu Path: Order Processing > Reports > Monthly Detail Reports > Monthly Lost Sales
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Selling Warehouse. The warehouse that receives credit for the sale.
•
Salesrep #1. The Salesrep One assigned to the sales order.
•
Product. The product numbers for which you want to view lost sales.
•
Reason Code. The lost sale reason codes to include (e.g., out of stock, price too high). These codes are set in the Lost Sales Reason Maintenance screen.
•
Document Type. The document types to include:
-
S - Lost sales orders.
-
M - Lost sales created manually in Lost Sales Entry.
-
Q - Lost quotes.
•
Order Date Range. The order entry date range for which you want to view lost sales.
Column Descriptions
Column Heading
Description
Rep1
The Salesrep One assigned to the sales order.
Customer Number
Line 1: The customer number.
Line 2: The customer description.
Doc Num
The order number.
Type
The lost order document type:
•
S - Lost sales orders.
•
M - Lost sales created manually in Lost Sales Entry.
•
Q - Lost quotes.
Upd Date
The date the order was cancelled.
Product Number
Line 1: The product number.
Line 2: The product description.
Line 3: The second line of description for the product, if applicable.
Reason
Line 1: The lost sales reason code from Lost Sales Reason Maintenance that indicates why the order was cancelled.
Line 2: The description of the reason code if one exists.
Refer to Creating Lost Sales Reason Codes for more information.
Units
The number of units ordered.
U/M
The unit of measure for the product as entered on the sales order.
Ext. Price
The extended price of the total quantity of the product.
Gross Prof
The gross profit amount lost due to the lost order.
