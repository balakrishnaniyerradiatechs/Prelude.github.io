Geographic Code Maintenance Screen
The Geographic Code Maintenance screen allows you to set up geographic codes and zip codes for tax jurisdictions to calculate taxes on sales orders. This information is required on sales orders if you use Vertex to calculate taxes.
Cash Sale
The Geographic code tax jurisdiction does not apply for a type 05 (Counter) cash sale. The tax jurisdiction from startup parameters is used first. If one does not exist in startup parameters, then the tax jurisdiction from the selling warehouse is used.
Menu Path: Order Processing > Maintenance > Sales Tax > Geographic Codes
Field and Function Key Descriptions
Field/Function Key
Description
Geographic Code
The geographical code that identifies a region for taxing purposes.
City
The city for this geographic code.
State
The state abbreviation for this geographic code.
Zip
The zip code for this geographic code.
County
The county for this geographic code.
Selling Warehouse
The warehouse that becomes the selling warehouse when this code is entered on an order.
Shipping Warehouse
The warehouse that becomes the shipping warehouse when this code is entered on an order.
Salesrep #1
The primary sales representative that appears as the default Salesrep One on sales orders when you use this geographic code. Leave this blank to use the Salesrep One from Customer Maintenance or Customer Ship-To Maintenance.
Salesrep #2
The secondary sales representative that appears as the default Salesrep Two on sales orders when you use this geographic code. Leave this blank to use the Salesrep Two from Customer Maintenance or Customer Ship-To Maintenance.
Territory
The territory associated with this geographic code. This only applies if you are not using Vertex. Territories are established in the Territory Maintenance Screen.
Tax Jurisdictions
The tax jurisdictions included in this geographical area that determine the tax rate. Tax jurisdictions are established in the Tax Jurisdiction Maintenance Screen.
Description
The description of each tax jurisdiction code.
F2-Save
Saves the geographical information.
F4-Del
Deletes the geographic code.
F5-County/City Lines
Opens the Geographical County/City Lines Screen. You can establish multiple geographic codes that contain zip codes for multiple cities and counties.
