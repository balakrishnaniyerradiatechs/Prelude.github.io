Invoice Print Prompts screen
Once the releases are confirmed in Release Confirmation Entry and set up for invoicing, you can use the Invoice Print Prompts screen to print, email, or fax the invoices to the customers. The method of transmission for each invoice depends on how each customer is set up to receive invoices (refer to Setting Maintenance Options for Invoices). If you want to preview the invoices before sending them, you can run the Invoice Preview Report (refer to Previewing Invoices), but it is not required.
Menu Path: Order Processing > Invoice Preview & Printing > Invoice Printing
Field and Function Key Descriptions
Field/Function Key
Description
Sales Order Number
The sales order number to invoice if you only want to print one invoice. Leave blank to print all invoices ready for printing.
Warehouse
The selling Warehouse for which you want to print invoices or leave blank to print invoices for all warehouses.
Customer Number
The customer number for whom you want to print invoices or leave blank to print invoices for all customers.
Invoice Date
The date to print on the invoices as the date of invoicing. This date and the Payment Terms are used to calculate the discount and due date for each invoice.
Printer Form Queue
The printer where the invoices will print.
Previewed?
Indicates whether you want to print all invoices or only invoices that were previously previewed:
•
Y - Only print invoices that were previewed on the Invoice Preview Report (refer to Previewing Invoices).
•
N - Print all invoices that are ready to print.
Invoice Cycle
Indicates how often invoices print. Each customer is assigned to one of the following:
•
M - Monthly.
•
S - Semi-monthly.
•
W - Weekly.
•
D - Daily.
Print Source
Indicates which types of invoices to print:
•
I - Invoices only.
•
C - Credit memos only.
•
B - Both invoices and credit memos.
•
M - Mail copies of counter invoices to customers whose Mail Invoices flag is set to Y in the Customer Maintenance Additional Order Information Screen.
Invoice Message
A message to print at the bottom of all invoices in this print run (e.g., Thank you for your Business!).
F5-Invoice Reprint
Opens the Invoice Reprint Entry screen to allow you to re-print, re-email, or re-fax the invoice.
F6-Batch Reprint
Allows you to re-print a batch of invoices.
