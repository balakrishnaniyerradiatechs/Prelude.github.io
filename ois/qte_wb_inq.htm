Quote Written By Inquiry
The Quote Written By Inquiry lists all open quotes for a Written By user ID. The Written By ID is assigned in the Quotation Header screen. It defaults to the person who entered the quote.
Viewing Open Quotes For a Written by ID
To list open quotes for an order taker:
1
Select Order Processing > Additional Subsystems > Quotation Subsystem > Inquiries > Quotation By Written By to open the Quote Written By Inquiry screen.
2
Enter the Written By user ID number. This is the number assigned in the Quotation Header screen to the person who is entering the quote.
3
Select the Sort sequence for the quotes. Choose from the following to sort the list by:
•
None.
•
Quote number.
•
Quote date.
•
Expiration date.
•
Customer P/O number.
•
Customer release number.
•
Quote total.
•
Descending quote number.
•
Descending quote date.
•
Descending expiration date.
•
Descending customer purchase order number.
•
Descending customer release number.
•
Descending quote total.
4
Determine if you want to use any Filters to limit the list of quotes. Choose from the following:
•
No. Lists all open quotes for the date.
•
Yes. Allows you to use the following selection criteria to limit the list:
-
Customer.
-
Quote Number.
-
Customer Release Number.
-
Purchasing Contact.
-
Customer P/O Number.
-
Expiration Date Range.
Quotes for the Written By ID display.
Field & Function Key Descriptions
Field/Function Key
Description
Quote#
The quote number.
Customer#
The customer who requested the quote.
Description
The description of the customer.
Exp Date
The date the quote expires. If the customer does not accept the quote by this date, the quoted price is no longer valid.
DA
The disposition analysis code that indicates if the quote is converted or deleted. If the quote was converted to a sales order, 00 displays. If it was deleted, the disposition code displays (refer to Creating Quote Disposition Codes). Converted or deleted quotes appear in this inquiry until they are purged in the Disposition Analysis reports (refer to the Quotation Disposition Detail Report).
Merchandise
The total price for all the merchandise quoted.
Notes
An asterisk indicates that header notes exist for the quote.
F2-Finish
Closes the Quote Written By Inquiry screen.
F5-Review
Opens the Quotation Display screen to allow you to view detail information for the quote.
F6-Enter Quote
Allows you to enter a quote number to go directly to.
F7-QOE
Allows you to convert the quote into a sales order (refer to Converting Quotes into Sales Orders).
F8-Log
Opens the Quote Log Inquiry to allow you to view every transaction that occurred for this quote.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
