Creating Quality Agent Codes
Each person that enters quality inspection complaints or corrective actions must have an agent code set up. An agent could be a customer, employee, or vendor. If the agent is an employee that already has a user ID, the existing information defaults into the agent code.
To create an agent code:
1
Select Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Quality Agent Maintenance to open the Quality Agent Maintenance screen.
2
For Agent Number, enter the name, code, or number to represent the agent.
3
Enter the User Id if the agent is an employee and one already exists. If so, information from User Identification Maintenance defaults into the following fields. If the agent does not have a user ID, leave this field blank.
4
For Description, enter the name of the agent.
5
Enter the agent’s Company Name.
6
Enter the agent’s Email Address, Phone, and Fax information.
7
Enter a valid Job Type to describe the agent’s position. Use F3 to search for existing codes from Valid Response Maintenance.
8
For Manager?, indicate if the agent is a manager:
•
Y - The agent is a manager and has the authority to close incidents.
•
N - The agent is not a manager.
9
If you use bins for inspected material, press F5-Add Info to open the Additional Information screen.
10
For Default Pass Bin, enter the default bin for this agent where items are placed that pass inspection in Inspection Results Entry (refer to Inspection Results Entry).
11
For Default Fail Bin, enter the default bin for this agent where items are placed that fail inspection in Inspection Results Entry (refer to Inspection Results Entry).
The default pass/fail bins set up for an agent override the default pass bins set up in Warehouse Maintenance > Action Bar > Addl Opts > Bin Locator Information > F6-Other Bins
12
Press F2-Save to close the Additional Information screen.
13
Press F2-Save to close the Quality Agent Maintenance screen and to save the agent information.
