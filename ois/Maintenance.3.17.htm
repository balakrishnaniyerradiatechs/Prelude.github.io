Creating Package Inserts
If you want to send out free gifts, flyers, advertisements, or other material with customer orders, you can create package insert codes to designate when a product or sales order should receive the package insert. Once you create the package insert codes, you can assign the code to customers so that only designated customers receive the free material.

Creating the Package Insert Product.
•
Creating the Package Insert Codes.
•
Assigning the Package Insert Code to a Customer.
