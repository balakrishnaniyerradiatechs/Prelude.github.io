Quotation Disposition Detail Report
The Quotation Disposition Detail Report lists quotes that are converted to sales orders, expired, or deleted. If deleted, the disposition code displays also to explain why it was deleted.
Purging Quotes
You should purge quotes regularly so that the quotation inquiries do not become cluttered with irrelevant quotes. When you purge quotes (Purge Utility Menu > Order Processing > Quotations), you can choose to purge only quotes with specific disposition codes.
The quotes are deleted from the open quote file and updated to the quote history file. The purged quotes appear in the Quote Order History Inquiry and the Quote History Product Inquiry. Once a quote is purged, you can no longer convert it. However, you can still duplicate a purged quote.
Menu Path: Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Disposition Analysis > Quotation Disposition Detail Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwest Region).
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Selling Warehouse. The selling warehouse that receives credit for the sale, if the quote is converted into a sales order.
•
Quotation Type. The Quote Type if you want to limit the report to one of the following quote types:
-
01 - Regular quote.
-
02 - Planned quote.
-
03 - Repair type quote.
-
04 - Standing quote.
•
Expiration Date <. Include all quotes with an expiration date before this date. You may want to use a date from a previous week or month to allow a grace period for recent quotes.
•
Disposition Analysis. The disposition codes to include, if you want to limit the report to one or more disposition codes. The disposition codes identify why a quote was deleted (refer to Creating Quote Disposition Codes).
•
Quote Number. The quote number if you want to limit the quotes to print and/or purge.
Column Descriptions
Column Heading
Description
Slm
The number for the sales representative on the quote.
Quote#
The quote number.
Cust #
The customer number. The customer description displays at the bottom of each section.
Dispos
Line 1: The disposition code (refer to Creating Quote Disposition Codes).
Line 2: The description of the disposition code, that explains why it was deleted. If the quote was converted rather than deleted, then Converted displays.
Exp Dt
The date the quote expires.
Product Nbr / Description
Line 1: The product number.
Line 2: The product description.
Quant
The quantity quoted to the customer.
Unit Price
The price per unit, as quoted to the customer.
Ext Price
The extended price of the total quantity of the product (Quant x Unit Price).
Unit Cost
The unit cost of the product.
Ext Cost
The extended cost of the total quantity of the product (Quant x Unit Cost).
Gr Profit
The total gross profit amount potentially earned.
G.P. %
The gross profit percentage potentially earned.
