Assigning Serial Shipping Container Codes
Once you set up your system to use serial shipping container codes, you can use Serial Shipping Container Entry to assign the codes to the cartons or pallets in a shipment while you are confirming the release. If you use the regular Release Confirmation Entry (refer to Confirming Releases) to confirm a release that requires detailed content labels (refer to the Customer Labels Information Screen), a warning message appears to allow you to choose to go directly into the Serial Shipping Container Entry screen.
To assign an SSCC to a release:
1
Select Order Processing > Optional Entry & Printing > Serial Shipping Container Entry to open the Serial Shipping Container Entry screen.
2
Enter the pick ticket Release Number to register the product detail on this SSCC.
3
For Serial Ship Code, press ENTER to generate a new SSCC.
4
Enter the Package Type to define the type of container. Choose from the following:
•
C - Carton.
•
P - Pallet.
5
Enter the Product to register. Press F5-Review Release to view items on the release. If the item is serialized, you must enter the serial number information.
6
Enter the Quantity packed in this container.
From the Action Bar, choose NonScan Inquiry to display all items on the release that are not yet assigned an SSCC.
 
7
After all products are registered, choose from the following to complete the transaction:
•
If the release has more line items, enter ENDSCAN on a blank line or scan a label with the ENDSCAN text. A new SSCC appears to assign to the next item on the release.
•
If this is the last item on the release, type FINALSCAN on a blank line or scan a label with the FINALSCAN text. The system checks to make sure all items on the release have an SSCC. The SSCC Label Program defined for the warehouse executes to print labels. You can attach the labels to all the registered cartons/pallets.
•
If you want to re-check the product information, press F2-Save. The following prompt appears:
Are you Finished With Registering All
Contents for This Release?
•
Choose from the following:
-
Yes: Allows you to check the product information for each code to make sure it matches the expected merchandise. Once everything matches, the transaction is complete. The SSCC Label Program defined for the warehouse executes to print labels. You can attach the labels to all the registered cartons/pallets.
-
No: Allows you to continue registering the next SSCC.
-
Suspend: Saves the information so you can assign more SSCCs later.
-
Return: Returns to the screen to allow you to enter additional information.
