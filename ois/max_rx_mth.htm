Maximum RX Month Maintenance screen
If a state regulates the number of months a prescription is valid, you can set up the number of valid months for each state in the Maximum RX Month Maintenance screen.
Menu Path: Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > Maximum RX Month Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Sate
The state abbreviation code for the state with the limited number of months for prescriptions.
#Months
The number of months a prescription is valid in the state.
F2-Save
Saves the information.
