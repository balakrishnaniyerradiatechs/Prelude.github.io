Written By Order Inquiry
The Sales Order Inquiry by Written By displays all open orders entered by a user ID.
Selecting Open Sales Orders
To list all open orders for a user:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Written by Order Inquiry to open the Written By Order Inquiry screen.
2
Enter the Written By user ID.
3
Enter the Sort sequence for the report. Choose from the following:
•
None.
•
Order number.
•
Order date.
•
Required date.
•
Customer P/O number.
•
Order type.
•
Customer release number.
•
Sales order dollars.
•
Descending order number.
•
Descending order date.
•
Descending required date.
•
Descending customer P/O number.
•
Descending order type.
•
Descending customer release number.
•
Descending sales order dollars.
4
If you want to use any Filters to limit the list of orders, choose from the following:
•
No. Lists all invoiced orders for the customer.
•
Yes. Opens the Sales Order Inquiry Filter Options for Written By screen to allow you to limit the list by the following options:
-
Customer number.
-
Required date range.
-
Order number.
-
Customer release number.
-
Customer purchase order number.
-
Job number.
Open sales orders for the Written-By user ID appear.
Field & Function Key Descriptions
Field/Function Key
Description
Order#
The sales order number.
Customer
The customer assigned to the order.
Description
The customer description.
N
An asterisk (*) indicates that notes exist in the sales order.
Type
The order type:
•
01 - Regular.
•
02 - Ship complete.
•
03 - Tag & hold.
•
04 - Direct ship.
•
05 - Counter invoice.
•
06 - Confirmation.
•
07 - Regular consignment.
•
08 - Confirmation consignment.
•
09 - Return goods.
•
10 - Repair order.
•
11 - Lease orders.
•
12 - Invoice when complete.
•
13 - Ship backorder complete.
•
14 - Ship line item complete.
•
15 - Show order.
•
16 - Order projection.
•
17 - Manufacturer’s representative.
•
18 - Crib replenishment.
Refer to Sales Order Types for a detailed explanation of each order type.
E
Indicates if the order is excluded from the customer’s credit limit. (e.g., order with a long term future ship date).
Inv
Invoice flag. Indicates the order’s invoice status:
•
Y - The order is flagged for invoicing.
•
H - The order is on hold and cannot be invoiced until released.
•
Blank - The order is not yet flagged for invoicing.
Cred
Indicates if the order is on credit hold:
•
H - The order is on credit hold.
•
R - Released from credit hold.
•
Blank - The order is not on credit hold.
Ord-Date
The date the order was entered.
L
Late flag. An asterisk (*) indicates that the order was not shipped by the Required Date on the sales order header.
Req-Date
The date the customer requires the order.
Customer P/O#
The customer’s purchase order number.
Release #
The customer’s purchase order release number, if applicable.
S/O Dollars
The total value (selling price) of the sales order.
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the Sales Order Display screen to display detail information for the highlighted order.
F6-Order#
Allows you to enter a sales order number to go directly to.
F7-Releases
Opens the Open Release Inquiry to display printing, picking, and shipping information for the highlighted order number.
F8-Log
Opens the Order Log Inquiry to list each transaction that affected the order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Lists the previous group of orders, if applicable.
sF6-Next Group
Lists the next group of orders, if applicable.
Action Bar
BO Only
Lists only sales orders that contain backorders.
