Order Log Report
The Order Log report lists all transactions that affected a sales order or a group of sales orders. Refer to the Order Log Inquiry Codes for a definition of each transaction code that appears on the report.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Order Log
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Customer. The customers to include.
•
Order. The sales order numbers to include.
•
Action Code. The transaction code to limit the list to all transactions of one type.
•
Action Date Range. The beginning and ending transaction dates to include.
Column Descriptions
Column Heading
Description
Order#
The sales order number.
Customer
The customer number.
Description
The customer description.
Code
The code that identifies the transaction. Refer to the Order Log Inquiry Codes for transaction code definitions.
Act Date
The date the transaction took place.
Time
The time the transaction took place.
User
The user ID of the person that entered the transaction.
Reference number
The order number, warehouse, code, or other values associated with the transaction. Refer to the reference column in the Order Log Inquiry Codes for the references descriptions.
