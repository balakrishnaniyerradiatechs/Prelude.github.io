Linked Sales Order Manager Inquiry
The Linked Sales Order Manager Inquiry is updated when someone edits a sales order that is linked to a purchase order. You can review the sales order and determine whether or not to update the purchase order.
This inquiry is only updated if S/P-P/O Linkage Work Queue in Option Parameters > Purchasing Parameters is set to Y.
Reviewing Linked Sales Orders
To list edited, linked sales orders:
1
Select Order Processing > Inquiries > Linked Sales Order Manager Inquiry to open the Linked Sales Order Manager Inquiry screen.
2
Press F5-Review to open the Sales Order Display screen to review the highlighted sales order.
3
If necessary, press F7-Rev Link to view the linked purchase order.
4
Press F2-Save to return to the Linked Sales Order Manager Inquiry.
The following message appears:
Do You Wish To Flag This Line As Reviewed?
5
Choose from the following:
•
No - Retains the sales order in your queue. The Status on the Linked Sales Order Manager Inquiry screen remains as N.
•
Yes - Changes the Status on the Linked Sales Order Manager Inquiry screen to R and opens the Notes Entry screen so you can enter notes regarding the linked order.
The line item is removed from the queue when you press F2-Save on the Linked Sales Order Manager Inquiry screen.
Field & Function Key Descriptions
Field/Function Key
Description
Order#
The sales order that has been edited.
Cntr
The line on the sales order that is linked to the purchase order.
Customer
The customer who ordered the material.
Description
The customer description.
User
The user ID of the person that entered the order (Written By).
Link-Date
The date the sales order was linked to a purchase order.
N
An asterisk (*) indicates that notes exist in the order header.
Status
Indicates if the order was reviewed in this work queue:
•
R - Reviewed.
•
N - Not reviewed.
The orders remain in the queue until you press F2-Save.
Link Notes
Notes sent from the sales person when the order was edited.
Reason
The reason code (set by user) that represents the type of change made to the sales order.
Line-Buy
The vendor line buy number if the purchase order was a line buy.
F2-Save
Closes the inquiry screen.
F5-Review
Opens the Sales Order Display screen to allow you to view the sales order. Press F7-Rev Link to view the linked purchase order. When you exit the Sales Order Display screen, you can mark the item as reviewed.
F7-Notes
Opens the Notes Entry screen to allow you view any notes entered by the salesperson when the sales order was edited.
