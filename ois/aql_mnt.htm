Acceptable Quality Levels screen
The American National Standards (ANSI) publishes AQL (acceptable quality level) tables that determine acceptable levels of products that must pass inspection. The number of products to inspect for each lot of products is determined by the Sample Size Code Letter (refer to Setting up Inspection Levels for Sample Sizes).

Field and Function Key Descriptions
Field/Function Key
Description
Code
The code levels established by American National Standards that indicates the sample size for each lot size.
Size
The sample size to inspect, based on the lot size.
Percent Acceptance Levels
The acceptable quantity that must pass inspection for this code and lot size.
F2-Save
Saves the acceptable quality levels.
F4-Del
Deletes the acceptable quality levels.
