State Abbreviation Maintenance Screen
Since each state may have different tax regulations, you can set up an abbreviation code for each state that contains tax information for the state. You can then assign these codes to the appropriate tax jurisdiction in the Tax Jurisdiction Maintenance Screen.
Menu Path: Order Processing > Maintenance > Sales Tax > State Abbreviations
Field and Function Key Descriptions
Field/Function Key
Description
State Abbreviation
The state abbreviation code to use in the Tax Jurisdiction Maintenance Screen.
Description
The state name.
Tax Authority Rules
Determines if state tax laws are based on source or destination rules:
•
D - Destination. The ship-to location is responsible for the tax.
•
S - Source. The shipping location is responsible for the tax.
•
M - Mixed. A combination of both destination and source.
Incl Tax/Cash Disc
Determines how cash discounts are calculated:
•
Y - Cash discounts are based on the merchandise total plus the sales tax.
•
N - Cash discounts are based on the merchandise total only.
Refer to “Cash Receipts” in the Accounts Receivable documentation for more information on cash discounts.
Private Liens
Lien Expire Days
The number of days after the last shipment that private liens will expire (1-365).
Bond Expire Days
The number of days after the last shipment that private bonds will expire (1-365).
Public Liens
Lien Expire Days
The number of days after the last shipment that public liens will expire (1-365).
Bond Expire Days
The number of days after the last shipment that public
liens will expire (1-365).
Certigy Code
The code assigned by Certigy for this state, if you use Certigy for authorization through Protobase.
Certigy Abbreviation
The state abbreviation used by Certigy.
Telecheck Code
The code assigned by Telecheck for this state.
F2-Save
Saves the state abbreviation information.
F4-Del
Deletes the state abbreviation code.
Action Bar
VIA Retail Info
Opens the Via Retail Info Maintenance screen to allow you to enter VIA tax code information, if you use VIA.
