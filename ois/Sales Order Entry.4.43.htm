Setting Maintenance Options for Acknowledgements
You can set maintenance options to determine when, where, and how acknowledgements are transmitted to your customer.
Email Document Text Maintenance
Before you can use text emailing for order acknowledgements, you must first set up a document code.
To set up a document code for shipping confirmations:
1
From the Systems menu choose Miscellaneous Controls > Email Document Text Maintenance to open the Email Document Text Maintenance screen.
2
For Document Code, enter ORDACK for shipping acknowledgements.
3
If the customer has a language indicator (set in Customer Maintenance > F7-Order), enter the language code to use for this type of text email.
4
Enter the description of the text email code (e.g., Order Acknowledgment).
5
Press F2-Save to save the document code.
Warehouse Maintenance
To set printing options for acknowledgments:
1
From Warehouse Maintenance, select F6-Continuous to open the Continuous Printing Setup screen.
2
For Acknowledgement, determine if acknowledgements print continuously or by batch. Choose from the following:
•
Y - Acknowledgements print, fax, or email upon completion of each sales order.
•
N - Acknowledgements print by batch (refer to Sending Acknowledgements by Batch).
3
For Prt, enter the port number (created in Continuous Printer Port Maintenance) to control continuous printing.
4
For Form Queue, enter the printer number where acknowledgements print.
5
Press F2-Save twice to save the printer information.
Customer Maintenance
To set up a customer to receive acknowledgements:
1
Select Order Processing > Maintenance > Customer Maintenance to open the Customer Maintenance screen.
2
Enter the Customer.
3
Select F7-Order > F5-Add Info to open the Additional Order Information screen.
4
For Ack Document Option, determine how acknowledgments transmit to the customer. Choose from the following:
•
B - Print each time a backorder is received.
•
C - Print and fax.
•
E - Email only.
•
F - Fax only.
•
M - Email and print.
•
N - No acknowledgment.
•
T - Text email (ORDACK must be set up in Email Document Text Maintenance on the Systems menu).
•
Y - Print only.
5
Press F2-Save three times to save the customer information.
Sales Order Entry
To transmit an acknowledgement from a sales order:
1
From the Sales Order Header screen, select F5-Hdr > F5-Add Info to open the Order Additional Information Screen.
2
For Acknowledge Flag, determine how acknowledgments are transmitted to the customer:
•
C - Print and fax.
•
E - Email only.
•
F - Fax only.
•
M - Email and print.
•
N - No acknowledgment.
•
T - Text email (ORDACK must be set up in Email Document Text Maintenance on the Systems menu).
•
Y - Print only.
The system searches for the correct email address in the following order:
•
Customer Ship-To Maintenance > E-Mail Address.
•
Customer Maintenance > F8-Contact > Purchasing Email.
•
Customer Maintenance > Email Address.
