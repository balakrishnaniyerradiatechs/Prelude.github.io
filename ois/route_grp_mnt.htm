Route Code Group Maintenance Screen
Route Code Group Maintenance allows you to group the route codes set up in the Route Maintenance Screen. The surcharge amount for the route group is added to outbound freight for the first sales order release for the customer/ship via /route.
Menu Path: Order Processing > Maintenance > Misc Code Files > Route Code Group Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Route Group
The code that represents the route group used in the Route Maintenance Screen to group multiple routes.
Description
The description of the route group.
Surcharge Amount
The surcharge amount to add to the outbound freight for the first sales order release when the route is processed.
F2-Save
Saves the route group information.
F4-Del
Deletes the route group.
