Masking Credit Cards for a User
To determine if credit cards are masked for a user ID:
1
In User Identification Maintenance, enter the User Id Code and select F5-Administrative to open the Administrative settings screen.
2
For View Credit Card#, choose from the following:
•
Y - The user can view the entire credit card number.
•
N - The first 12 digits of the credit card number are masked (e.g., xxxx-xxxx-xxxx-1234).
3
Press F2-Save 2 times to save the credit card information.
