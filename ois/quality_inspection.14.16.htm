Quality Feedback Entry
When you receive a product complaint from a customer, vendor, employee, or other source, you can enter the information in Quality Feedback Entry. This information is later tied into an incident (refer to Quality Incident Entry) so that all accumulative information on the product is recorded in one place.
To record a product complaint:
1
Select Order Processing > Additional Subsystems > Quality Inspection Subsystem > Quality Feedback Entry to open the Quality Feedback Entry screen.
2
For Feedback#, choose from the following:
•
Press ENTER over NEW to create a new feedback number.
•
Enter an existing feedback number.
•
Press F3 to search for previously created feedback entries.
3
Enter a short Description of the problem.
4
For Agent, enter the agent code of the person assigned to the complaint. The agent’s name displays to the right of the number. Refer to Creating Quality Agent Codes for information on agents.
5
Enter the Product that has a complaint.
6
For Source Type, enter the code for the source of the feedback (i.e., who made the complaint).
•
E – Employee.
•
C – Customer.
•
V – Vendor.
•
M – Miscellaneous.
7
For Source Number, choose from the following:
•
If Source Type is an employee, enter the employee number.
•
If Source Type is a customer, enter the customer number.
•
If Source Type is a vendor, enter the vendor number.
•
If Source Type is miscellaneous, this field is not accessible, but you can enter a description of the source for Source Description.
8
For Type, enter the type of feedback entry that tells what the complaint is related to (e.g., product or employee).
9
If applicable, enter the Location of the defective product in the warehouse.
10
When you are first creating a feedback entry, the Notes Information screen appears to allow you to enter any notes regarding the problem. The more descriptive the information, the better the ability to fix the problem. Any subsequent time that you access this same feedback number, you can access the Notes Information screen by selecting the Notes option on the Action Bar.
11
Enter the current Status feedback entry. Choose from the following:
•
O - Open (default).
•
C - Closed.
•
I - In process.
•
U - Unassigned.
•
W - Waiting for response.
12
If you want to create a new incident record at this time (refer to Quality Incident Entry) you can press F5-New Incident to access the Quality Incident Entry screen. This screen is a collection of all feedback entries and correction entries regarding a complaint. You can also wait to create the incident after more information is gathered.
13
Press F2 - Save to save the feedback entry.
