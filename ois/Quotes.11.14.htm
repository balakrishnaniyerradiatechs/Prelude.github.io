Deleting a Quote
To delete a quote:
1
From the Quote Order Entry screen, select F4-Del. The following message appears:
Are You Sure You Wish To Delete?
2
If you choose Yes, the following message appears:
Update Lost Quotations?
3
Choose from the following:
•
Yes - Opens the Lost Quote Reasons screen to allow you to enter a Disposition Analysis Code that explains why the quote was cancelled (refer to Creating Quote Disposition Codes).
•
No - Lost quotations are not updated.
4
In either case, the following message displays:
Do You Want To Update Lost Sales?
5
Choose from the following:
•
No - Exits the sales order.
•
Yes - Opens the Lost Sale Reason Code screen to allow you to enter a Lost Sale Reason Code. Refer to Creating Lost Sales Reason Codes.
Track Lost Sales
To track lost product sales, set Lost Sales in File Update Parameters to Y to require the lost sales reason code.
