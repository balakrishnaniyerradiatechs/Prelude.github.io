Duplicating Quotes
An existing quote for a customer can be duplicated for the same customer or for another customer.
To duplicate an existing quote:
1
Select Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Entry to open the Quotation Order Entry screen.
2
Press ENTER over Quote Number.
3
From the Action Bar, select Dup Quote to open the Quote Duplication Entry screen.
Duplicate a Converted Quote
To duplicate a quote that is already converted, select Dup History Quote.
4
For Duplicate Quote#, enter the quote number that you want to duplicate. The Customer assigned to the quote appears below.
5
For New Customer, if you are duplicating the quote for a different customer, enter the customer number. Leave blank if the new quote is for the same customer.
6
For New Warehouse?, if the new quote is for a different warehouse, enter the warehouse number. Leave blank if the new quote is for the same warehouse.
7
For Reprice Quote?, indicate if you want to use current pricing assigned to the customer or use the pricing from the original quote. Choose from the following:
•
Y - Use the current pricing for the customer for the products on the quote.
•
N - Use the same pricing that was entered on the original quote.
8
Press F2-Save. The line items from the original quote are copied to the new quote. The Quotation Header screen appears. You can verify and edit the header and line items as needed.
