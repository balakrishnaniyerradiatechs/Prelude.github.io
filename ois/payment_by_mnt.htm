Payment By Maintenance screen
A Payment By code indicates who will make payment for an invoice and is required when you confirm a DD250 sales order (Refer to Confirming a Release that Requires a DD250 Invoice). The Payment By code prints in block 12 on the DD250 invoice (refer to Printing Invoices Using the DD250 Form).
Menu Path: Order Processing > Maintenance > Misc Code Files
Field and Function Key Descriptions
Field/Function Key
Description
Payment By Number
The 1 to 6 alphanumeric code to use for sales orders set up to print DD250 invoices.
Description
The name of the payment company or person.
Address One
The first line of address for the payment company or person.
Address Two
The second line of address for the payment company or person.
Address Three
The third line of address for the payment company or person.
Address Four
The fourth line of address for the payment company or person.
City
The city of the payment company or person.
State
The state of the payment company or person.
Zip
The zip code.
F2-Save
Saves the payment by information.
F4-Del
Deletes the payment by code.
