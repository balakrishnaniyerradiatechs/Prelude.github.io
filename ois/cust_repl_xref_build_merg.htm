Universal Customer Merge Entry screen
The Universal Customer Merge Entry screen allows you to copy or merge information from an existing customer to a new customer. You must run the Universal Customer Replacement Merge Update to complete the merge/change. Refer to the Universal Customer Replacement Merge for more information.
Field and Function Key Descriptions
Field/Function Key
Description
From Customer
The existing customer that you want to change.
To Customer
The new customer that will be updated with the information from the From Customer in the Universal Customer Replacement /Merge Update. You can enter either a new customer number that does not already exist or you can enter an existing customer. If you enter an existing customer, a warning message appears to let you know that the data from the From Customer will merge to the To Customer.
Merge Type
The type of merge.
•
C - Copy. Appears if you enter a new customer number for To Customer.
•
M - Merge. Appears if you enter an existing customer number for To Customer.
This field is not accessible.
F2-Save
Saves the merge information.
F4-Del
Deletes the merge information.
