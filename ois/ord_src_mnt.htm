Order Source Code Maintenance screen
An order source code identifies where a sales order or quote came from. Once you create an order source code in Order Source Code Maintenance, you can assign it to a sales order in the Order Additional Information Screen (Sales Order Header > F5-Add Info), to identify how you obtained the order. You can also set up a default order source code for a customer in the Customer Maintenance Order Source Matrix Screen, and then give the customer an additional discount depending on how they submit their order.
Menu Path: Order Processing > Maintenance > Misc Code Files > Order Source Codes
Field and Function Key Descriptions
Field/Function Key
Description
Order Source
The one-digit code that identifies the source of the sales order or quote. Commonly used order source codes are:
•
C - Customer call in.
•
D - Direct order entry.
•
E - Electronic data interchange.
•
K - Special order-telemarketing.
•
L - Sales representative.
•
M - Mail in.
•
P - Point of sale.
•
R - Remote order entry.
•
S - Sales force automation.
•
T - Telephone sales.
•
V - VIA
•
W - Will call.
•
X - Special order.
Description
A description of the order source (e.g., customer call in, will call, sales rep).
Order Log Code
The code to display in the Order Log Inquiry. The default is ASG (assignment of order).
F2-Save
Saves the order source code information.
F4-Del
Deletes the order source code.
