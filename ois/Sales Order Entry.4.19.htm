Entering Comments
Always insert comments after the line item they are related to. Comments inserted after a line item are always linked to that line item and print on documents according to their comment code.
To add comments to a line item:
1
Move the cursor to the blank line following the related line item.
If the item is not the last line on the order, move the cursor to the line below where you want the comment to appear and press Cntrl and Insert at the same time to create a blank line.
2
Enter the appropriate Comment Code (refer to the Comment Code Definitions).
3
Press ENTER. The Comment Description Entry screen appears.
4
Enter a Description.
5
Press F2-Save. Continue entering line items as necessary.
