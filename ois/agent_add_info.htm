Additional Information screen
If you use bins for inspected material, you can enter the default bins where items are placed that pass or fail inspections for a quality agent.
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Quality Agent Maintenance > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Default Pass Bin
The default bin for this agent where items are placed that pass inspection in Inspection Results Entry (refer to Inspection Results Entry).
Default Fail Bin
the default bin for this agent where items are placed that fail inspection in Inspection Results Entry (refer to Inspection Results Entry).
F2-Save
Saves the bin information and returns to the Quality Agent Maintenance screen.
