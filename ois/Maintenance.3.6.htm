Creating Standard Instructions
If you have standard sets of instructions that you want to appear on sales orders, quotes, or purchase orders, you can set up codes to reference the standard message. You can then use the code in the Instructions field in Customer Maintenance, Customer Ship-To Maintenance, Vendor Maintenance, Product Maintenance, or any of the order entry programs.
Automatically Displaying Instructions on Sales Orders and Quotes
Disp. Shp.Inst. In SOE/QOE in Order Processing Parameters determines if the instructions screen automatically displays when the header screen comes up in Sales Order Entry and Quotation Entry.
•
A - Always display the Instructions screen.
•
N - Never display the Instructions screen.
•
Y - Only display the Instructions screen if instructions exist.
 
To set up standard instruction messages:
1
Select Order Processing > Maintenance > Misc Code Files > Instruction Messages to open the Instruction Messages Maintenance screen.
2
For Message Code, enter the 1 or 2 digit alpha-numeric code to reference a set of instructions.
3
Enter up to 3 lines of instructions to appear on transactions when this message code is entered.
4
Press F2-Save to save the instructions.
