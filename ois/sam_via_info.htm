Via Retail Info Maintenance screen
If you use VIA for tax reporting, the Via Retail Info Maintenance screen allows you to enter VIA tax code information.
Menu Path: Order Processing > Maintenance > Sales Tax > State Abbreviations > Action Bar > Via Retail Info
Field and Function Key Descriptions
Field/Function Key
Description
VIA Retail Tax Codes
The tax jurisdiction code used by VIA.
Description
The description of the tax code.
Retail Shipments Allowed?
Determines if VIA retail shipments are allowed for this state:
•
Y - VIA retail shipments are allowed for this state.
•
N - VIA retail shipments are not allowed.
Freight Table Number
The freight table number used by VIA to calculate freight for this state. If left blank, table 00 is used.
Wholesale Frt Table Id
The wholesale freight table used by VIA to calculate freight. If left blank, table 00000W is used.
F2-Save
Saves the VIA retail tax information.
