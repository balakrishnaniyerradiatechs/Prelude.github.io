Finishing the Sales Order
Once you finish entering the line items on the sales order (refer to Entering Sales Order Line Items) and you are ready to complete the order:
1
Press F2-Save. The following message appears:
Do You Wish To...
2
Choose from the following:
•
Pick Ticket - The pick ticket is set up to print if inventory is available, the pick ticket date is reached, and if the order is not on hold. If you go back into the sales order and choose this option again, it will only print if additional items have become available since the first time you printed it.
If your company continuously prints pick tickets, the pick ticket automatically prints. If your company does not continuously print pick tickets, you must use Pick Ticket Printing to print the pick tickets. Refer to Printing Pick Tickets by Batch.
•
Complete - Opens the Sales Order Completion screen to display the order total and to allow you to add freight, delivery, or handling charges or to take payment for the order. Refer to Using the Sales Order Completion Screen.
•
Suspend - Suspends the sales order so you can access it later. No pick ticket prints until you go back into Sales Order Entry to edit the order and choose the Pick Ticket option, even if inventory becomes available. Depending on system flags, inventory may be allocated while the order is suspended.
•
Return - Allows you to return to the order to verify or edit as necessary.
Suspend Orders From Activities in Order Processing Parameters determines if an order that is suspended can go on Credit Hold or Administrative Hold.
3
If you choose to generate a purchase order, transfer, work order, or transfer for any line items on backorder, you are prompted to proceed with the order generation. Refer to the following:
•
Linking a Purchase Order to a Backorder.
•
Linking a Transfer to a Backorder.
Shipment Processing Entry
If you need to enter, confirm, and invoice an order all in one step, you can use Shipment Processing Entry (Order Processing > Optional Entry & Printing > Shipment Processing Entry). The order is set up for invoicing and no pick ticket is generated. You may want to use this if the customer already has the material.
