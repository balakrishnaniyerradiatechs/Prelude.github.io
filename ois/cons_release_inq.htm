Consignment Release Inquiry
The Consignment Release Inquiry displays release information for type 07 consignment sales orders. Consignment orders contain products on loan to a customer. If the customer sells the product, they are invoiced for the products. If the customer does not sell the products, they are returned back into inventory. Refer to Using Consignment Sales Orders.
Viewing Consigned Releases
To list releases for a consignment order:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Consignment Release Inquiry to open the Consignment Release Inquiry screen.
2
Enter the Order Number.
The customer number and consigned order release information appears.
Field & Function Key Descriptions
Field/Function Key
Description
Release#
The consignment order release number.
Whse
The shipping warehouse assigned to the order.
Prt-Date
The date the consignment release printed.
Prt-Time
The time the consignment release printed.
Proc-Date
The date the release was confirmed in Release Confirmation Entry.
F2-Finish
Closes the inquiry screen.
F5-Review Release
Displays detail order information for the highlighted consignment release number.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
