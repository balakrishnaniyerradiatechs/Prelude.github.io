Setting up a Crib Warehouse
A crib warehouse is managed as a separate warehouse. Products are maintained, tracked, purchased, and received into the crib warehouse.
To set up a warehouse as a crib:
1
Select Inventory Controls > Warehouse Maintenance > Action Bar > Addl Opts > Crib Information to open the Crib Information Maintenance screen.
2
For Crib Flag, enter Y to identify this warehouse as a crib.
3
For Crib Type, indicate if the crib is managed by warehouse or bin location:
•
W - Warehouse.
•
B - Bin. You must set up the customer as a crib bin in Bin Location Maintenance (Bin Type set to C for Crib Warehouse).
4
If Crib Type is set to W, enter the Crib Customer Number where the crib warehouse is located.
5
Enter the Crib Shipto Number associated with the Crib Customer Number.
6
Press F2-Save to update the crib information for the customer and customer ship-to.
