Reduced Inspection Table
The reduced inspection table shows the acceptance and rejection levels for each sample size of a product considered a reduced inspection. Reduced inspection is used when a pre-determined number of preceding batches have passed normal inspections without any defects.
These values are already set up by Prelude but can be changed if necessary.
To view or edit the reduced acceptable quality levels:
1
Select Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Single Sampling - Reduced Inspection Table to open the Acceptable Quality Levels screen.
2
Press F2-Save to save any changes and to close the screen.
