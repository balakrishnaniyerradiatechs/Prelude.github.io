Order Processing Overview
The Prelude Order Processing Module allows you to set up customer accounts to accommodate shipping, tax, and pricing information for sales orders. You can determine ordering restrictions for each customer and branch so that unauthorized orders do not ship. The Order Processing features also allow you to create sales orders, print pick tickets and acknowledgements, confirm and invoice sales orders, and issue RGAs and credit memos.
