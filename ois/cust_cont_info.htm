Customer Contacts Screen
The Customer Contacts screen allows you to enter phone and address information for both the purchasing contact and hazardous material coordinator for a customer.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F8-Contact
Field and Function Key Descriptions
Field/Function Key
Description
Purchasing Contact
The name of the customer’s purchasing contact.
Address One
The first line of address for the purchasing contact.
Address Two
The second line of address for the purchasing contact.
Address Three
The third line of address for the purchasing contact.
City
The purchasing contact’s city.
State
The purchasing contact’s state.
Zip
The zip code for the purchasing contact.
Purchasing Phone
The purchasing contact’s phone number.
Purchasing Fax
The purchasing contact’s fax number.
Purchasing Email
The purchasing contact’s email number.
Primary Sic Code
The primary Standard Industrial Classification code assigned to this customer.
Secondary
The secondary Standard Industrial Classification code assigned to this customer.
Haz. Matl. Contact
The name of the customer’s hazardous material’s coordinator.
Address One
The first line of address for the hazardous material contact.
Address Two
The second line of address for the hazardous material contact.
Address Three
The third line of address for the hazardous material contact.
City
The city of the hazardous material contact.
State
The state of the hazardous material contact.
Zip
The state of the hazardous material contact.
Default View
If you are using the Marketing Module, you can enter a default view for the Contact Management Statistics Inquiry.
Teardown Contact Id
The person responsible for communicating with the customer. Also the user ID for the Final Price Inquiry screen.
F2-Save
Saves the contact information.
F5-Employees
Opens the Customer Employees Screen. You can enter the names of the employees for this customer.
F6-Purch Agents
Opens the Customer Authorized Purchasing Agents Screen. You can enter the employees (that are already set up in the Customer Employees Screen) that are authorized purchasing agents for this customer.
