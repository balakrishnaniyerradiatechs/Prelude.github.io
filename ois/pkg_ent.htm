Serial Shipping Container Entry screen
You can use Serial Shipping Container Entry to assign serial shipping container codes to the cartons or pallets in a shipment while you are confirming the release. If you use the regular Release Confirmation Entry (refer to Confirming Releases) to confirm a release that requires detailed content labels, a warning message appears to allow you to go directly into the Serial Shipping Container Entry screen. Refer to Using Serial Shipping Container Codes for instructions on setting up SSCCs.
Menu Path: Order Processing > Optional Entry & Printing > Serial Shipping Container Entry
Field and Function Key Descriptions
Field/Function Key
Description
Release Number
The pick ticket release number to register for the SSCC.
Customer
The customer assigned to the sales order.
Serial Ship Code
The serial shipping container code to assign to the shipment.
Package Type
Defines the type of container:
•
C - Carton.
•
P - Pallet.
Product
The product to register. Press F5-Review Release to view items on the release. If the item is serialized, you must enter the serial number information.
Description
The product description.
U/M
The unit of measure for the product as entered on the sales order.
Quantity
The quantity packed in this container.
F2-Save
Completes the transaction and gives you the following choices:
•
Yes. Allows you to check the product information for each code to make sure it matches the expected merchandise. Once everything matches, the transaction is complete. The SSCC Label Program defined for the warehouse executes to print labels. You can attach the labels to all the registered cartons/pallets.
•
No. Allows you to continue registering the next SSCC.
•
Suspend. Saves the information so you can assign more SSCCs later.
•
Return. Returns to the screen to allow you to enter additional information.
F4-Del
Deletes the serial ship container code.
F5-Rev Release
Opens the Release Confirmation Inquiry screen to allow you to view the products on the release.
F6-Del Release
Deletes the release from within the SSCC screen.
F9-Goto Ln
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
NonScan Inquiry
Displays all items that are not completely packed or scanned.
