Thaw Date Formula
Product Warehouse Maintenance: Pick Date - (Lead Time + Safety Days) = Thaw Date (A).
Vendor Line Maintenance: Next Review Cycle + Review Cycle Days = B.
•
If Next Review Cycle is empty, the system uses today’s date.
•
If Review Cycle Days is empty, the system uses 28.
 
If B > A, the line is not a future.
If B </= A, the line is a future.
Menu Path: Order Processing > Sales Order Entry > F5-Hdr Scr > Action Bar > Add Info > Future Commit
Field and Function Key Descriptions
Field/Function Key
Description
Commitment Override
Indicates if you want to commit inventory now for future requirements:
•
Y - Commit inventory for a future shipment.
•
N - Do not commit inventory for a future shipment.
User Id
The user ID of the person entering the sales order.
F2-Save
Returns to the Sales Order Header screen.
