Monthly Lost Sales by Vendor Report
The Monthly Lost Sales By Vendor report lists lost sales information by primary vendor. Refer to Creating Lost Sales Reason Codes for more information on lost sales.
Track Lost Sales
Keep Lost Sales? in File Update Parameters must be set to Y to require a lost sales reason code.
Menu Path: Order Processing > Reports > Monthly Detail Reports > Monthly Lost Sales By Vendor
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Selling Warehouse. The warehouse that receives credit for the sale.
•
Primary Vendor. The primary vendor assigned to a product in Product Vendor Maintenance.
•
Reason Code. The lost sale reason code to include (e.g., out of stock, price too high). These codes are created in the Lost Sales Reason Maintenance screen.
•
Document Type. The document types to include:
-
S - Lost sales orders.
-
M - Lost sales created manually in Lost Sales Entry.
-
Q - Lost quotes.
•
Lost Sale Date Range. The order date range for which you want to view lost sales.
Column Descriptions
Column Heading
Description
Cust#/Vend#/ Description
Line 1: The customer number and description.
Line 2: The vendor number and description.
Doc Num
The cancelled order number.
Type
The lost order document type:
•
S - Lost sales orders.
•
M - Lost sales created manually in Lost Sales Entry.
•
Q - Lost quotes.
Upd Date
The date the order was cancelled.
Product Number
Line 1: The product number.
Line 2: The product description.
Line 3: The second line of description for the product, if applicable.
Reason
Line 1: The lost sales reason code, created in Lost Sales Reason Maintenance that indicates why the order was cancelled.
Line 2: The description of the reason code.
Refer to Creating Lost Sales Reason Codes for more information.
Units
The number of units ordered.
U/M
The unit of measure for the product as entered on the sales order.
Ext. Price
The extended price of the total quantity of the product.
Gross Prof
The gross profit amount lost when the order was cancelled.
