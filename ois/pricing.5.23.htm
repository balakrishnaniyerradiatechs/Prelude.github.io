Copying a Retail Price Matrix
If you want to copy a customer matrix or customer price group retail matrix, use the Copy feature in Retail Price Matrix Maintenance to copy a matrix to a single customer, multiple customers, or to a list of customers.
To copy a retail price matrix:
1
Follow steps 1 - 5 in Creating a Retail Price Matrix to open the Retail Price Matrix Input screen.
2
Select Copy from the Action Bar to open the Customer Matrix Copy screen.
If you entered a price matrix for a customer price group, the Customer Price Group Copy screen opens. The procedures are the same except that you enter a customer price group to copy the information to, rather than a customer number.
3
For Copy Option, choose from the following:
•
O - Overwrites the target matrix with the source matrix. The original target matrix is deleted. You are warned if the target matrix exists and allowed to exit without overwriting.
•
I - Inserts the source matrix at the beginning of the target matrix and retains the existing target price matrix data. You can either insert the entire source matrix or a single line of the source matrix. You may want to use this option to update promotional or temporary pricing.
•
A - Appends the source matrix to the end of the target matrix and retains the existing target price matrix data. You can either append the entire source matrix or a single line of the source matrix into the target matrix. You may want to use this option to update future prices that are effective after the existing matrix expires.
•
D - Deletes all the price matrices for the customers defined only if the same combination of matrices exists for the customer or customer price groups. For example, to delete a price matrix for a product, the customers must have the same product on their matrices as the source matrix. Only records with the same product are deleted. The price matrices that have a range of numbers as part of the matrix must have the same range of numbers for the customers or customer price groups to allow you to delete them.
4
If you choose A (append) or I (insert) for the Copy Option, enter the Line Number from the existing matrix to insert or append. Leave blank to insert or append all line items for the current matrix.
5
If applicable, enter a previously created Listname of customers or customer price groups to copy the current matrix to.
6
For Customer Number, enter a single customer or multiple customers to copy the current matrix to.
7
Press F2-Save to copy the price matrix information.
