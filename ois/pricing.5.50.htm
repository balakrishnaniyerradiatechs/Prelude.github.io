Duplicating a Catalog ID Code
After you create a price catalog for your products (refer to Creating a Catalog ID Code) you can duplicate the catalog if the products in both the old and the new catalogs are the same. The following occurs when you duplicate a catalog:
•
Products that are identified as alternate codes for the catalog being duplicated are copied.
•
New alternate codes are created by appending the default prefix character of the new catalog code to the product.
•
The new alternate code receives the default product price group and catalog ID from the old catalog.
To duplicate an existing catalog:
1
Select Order Processing > Maintenance > Pricing Structures > Catalog Price Maintenance > Catalog Duplications to open the Copy Existing Catalogs screen.
2
For From Catalog Number, enter the catalog that you want to copy the information from.
3
For To Catalog Number, enter the new catalog ID number. The new catalog number must be already created (refer to Creating a Catalog ID Code). New alternate part numbers are created using the prefix of the new catalog ID code.
