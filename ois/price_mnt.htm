Price Matrix Maintenance screen
You can use Price Matrix Maintenance to give discounts and rebates for customers for a group of products for a period of time. Refer to Creating a Price Matrix.
Price Matrix Inquiry allows you to view existing price matrices. The field and function key descriptions are the same as in Price Matrix Maintenance.
The Price Matrix Listings list the existing price matrices for the matrix type that you choose.
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Matrix type
A valid matrix type set up in Price Matrix Types & Sequence Maintenance. Refer to Setting System Options for Price Matrices.
Warehouse
The selling location if you want this price matrix to apply to only one warehouse. Leave blank if the price matrix is not warehouse specific.
Drop Ship
Indicates if the matrix applies to drop ship orders:
•
Y - The matrix applies to drop ship orders only. You must create a separate matrix for orders shipped from stock.
•
N - The matrix applies to both stock and direct orders.
This field is only accessible if the Drop Ship Price Matrix Option Parameter is set to Y.
Job#/Customer P/O#
The job or a customer P/O if this pricing is limited to just one job or purchase order. Leave blank if you want the price matrix to apply to all orders for the customer. The system looks for a customer P/O number first then for a job number.
F5-Existing Matrix
Opens the Price Matrix Selection screen to allow you to view or modify an existing matrix. You can enter any of the selection criteria to find a matrix or leave any of the fields blank to find multiple matrices. Once you select a matrix, the data automatically appears in the Price Matrix Input screen.
