EDI Transmission Info Screen
If you use EDI to transmit orders and a customer is not set up to automatically convert DOE orders to open orders, you can use the EDI Transmission screen to manually convert orders for the customer.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F7-Order > Action Bar > Add’l Options > EDI Information
Field and Function Key Descriptions
Field/Function Key
Description
Direct Order User Id
The user ID assigned to enter direct EDI orders for this customer.
This Cust Sends An 855
Determines if the customer receives detailed functional acknowledgments:
•
Y - The customer receives acknowledgements.
•
N - The customer does not receive acknowledgements.
F2-Save
Saves the EDI transmission information.
