Ship Via Freight Info Screen
The Ship Via Freight Info screen allows you to enter freight minimums and terms information for a ship via code.
Menu Path: Order Processing > Maintenance > Customer Control Files > Ship Via Codes > F6-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Prepaid Freight
Works with Prepaid Frt Minimum to determine if prepaid freight is allowed for this ship via type (if the customer is set to auto pay freight):
•
Y - Freight is not charged with this ship-via.
•
N - Freight is charged with this ship-via.
•
C - Freight is not charged if the Prepaid Frt Minimum is met.
Prepaid Frt Minimum
If Prepaid Freight is set to C, this is the minimum amount an order can be to earn prepaid freight. Any order below this amount is charged freight. You can override the minimum in Customer Maintenance or Customer Ship-to Maintenance.
Calc Frt On Credits
Determines if the freight table in the Ship Via Maintenance Screen is used to calculate freight for customer credits:
•
Y - Use the ship via table.
•
N - Do not use the ship via table.
Min Order Amount
The minimum order amount for this ship via type.
Small Order Limit
Orders below this amount are charged a small order fee in Outbound Freight on the Sales Order Completion screen. This flag does not apply if the customer’s Exclude Small Ord Amt Chk flag is set to Y.
Small Order Fee
The fee to add to the order if it does not meet the Small Order Limit.
Freight Terms
The freight terms code (established in Freight Terms Maintenance Screen) for this ship via type. A customer’s freight terms override the ship via’s freight terms. The ship via freight terms only apply if the ship via is manually changed on the order header.
Delivery Days
The number of days to allow for delivery for orders with this ship via code. This information appears in #Days on the Shipping Consolidation screen when completing a sales order (if Shipping Whse Consolidation in Order Processing Parameters is set to S).
Dynamic Truck Routing
Determines if this ship via is available to interface with the 3rd party dynamic routing software (custom modification):
•
Y - Yes.
•
N - No.
F2-Save
Saves the ship via freight information.
