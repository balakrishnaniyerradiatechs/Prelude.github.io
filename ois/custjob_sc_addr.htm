Subcontractor Information Screen
If a subcontractor is associated with the job, you can enter the subcontractor’s address in the Subcontractor Information screen.
Menu Path: Order Processing > Maintenance > Customer Job Numbers > Action Bar > Add’l Opts > Subcontractor Information
Field and Function Key Descriptions
Field/Function Key
Description
SubContractor
The name of the subcontractor.
Address Line
The subcontractor’s address.
City
The subcontractor’s city.
State
The subcontractor’s state.
Zip
The zip code.
Phone
The subcontractor’s phone number.
F2-Save
Saves the subcontractor information
