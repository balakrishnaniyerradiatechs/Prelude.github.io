Customer Job Additional Information Screen
If you use the Job Accounting Module, you can enter retainage, invoice, tax, and salesrep information for a customer job that requires change order control. A change order is used to add new items or delete existing items but does not allow you to change quantities and prices.
Menu Path: Order Processing > Maintenance > Customer Job Numbers > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Change Order Control
Determines if this job requires change order control:
•
Y - The job requires change order control. Sales Order Entry prompts for a Change Order Number which allows you to add new items or delete existing items but not change quantities and prices.
The customer’s Require Job Number flag in the Order Information Screen must be set to Y, V, or S to enable change order control.
•
N - Change order control is not required for this job. The rest of the fields in this screen are not applicable.
Retainage Percent
The percent of each invoice that the customer will retain until the job is complete. For example, enter 5 for 5%.
Retainage Date
The estimated date when the customer will pay the retainage amount.
AIA Invoice
Indicates if an AIA (American Institute of Architects) invoice report is required:
•
Y - An AIA invoice report is required.
•
N - An AIA report is not required.
Job Line Item Tax Flag Default
The default tax flag for line items entered for this customer job:
•
Y - Taxable.
•
N - Non-taxable.
•
U - Use tax.
Salesrep One
The sales representative to assign to orders for this customer job.
Salesrep Two
The second sales representative to assign to orders (e.g., inside sales).
F2-Save
Saves the job information.
F5-Tax Jurisdictions
Opens the Customer Job Sales Tax Maintenance Screen. You can enter tax jurisdiction and tax exempt information.
