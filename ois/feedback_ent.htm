Quality Feedback Entry screen
When you receive a product complaint from a customer, vendor, employee, or other source, you can enter the information in the Quality Feedback Entry screen (refer to Recording a Product Complaint). This information is later tied into an incident (refer to Quality Incident Entry) so that all accumulative information on the product is recorded in one place.
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Quality Feedback Entry
Field and Function Key Descriptions
Field/Function Key
Description
Feedback#
The feedback entry number.
Entry Date
The qualify feedback date of entry.
User Id
The user ID of the person that entered the quality feedback.
Status
The status of the feedback entry:
•
O - Open.
•
C - Closed.
•
I - In process.
•
U - Unassigned.
•
W - Waiting for response.
Description
A description of the problem.
Agent
The agent code of the person assigned to the complaint. The agent’s name displays to the right of the number. Refer to Creating Quality Agent Codes for information on agents.
Product
The number for the product for which you are recording a complaint.
Source Type
The code for the source of the feedback (i.e., who made the complaint).
•
E – Employee.
•
C – Customer.
•
V – Vendor.
•
M – Miscellaneous.
Source Number
Depending on the Source Type, one of the following:
•
If Source Type is an employee, the employee number.
•
If Source Type is a customer, the customer number.
•
If Source Type is a vendor, the vendor number.
•
If Source Type is miscellaneous, this field is not accessible, but you can enter a description of the source for Source Description.
Source Description
The description of the source.
Type
The type of feedback entry that tells what the complaint is related to (e.g., product or employee).
Location
The warehouse location of the defective product.
Notes
Displays any notes entered for this feedback entry (notes entered using the Notes option on the Action Bar).
F2-Save
Saves the feedback entry.
F4-Del
Deletes the feedback entry.
Action Bar
Notes
Open the Notes Information to allow you to enter additional information for this feedback entry.
Output
Allows you to print, email, or fax the feedback information.
