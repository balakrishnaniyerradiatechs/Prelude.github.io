Setting Maintenance Flags for DD250 Form Printing
The following maintenance flags determine what prints on a DD250 invoice.
Warehouse Maintenance > F5-Add Info
Prime Contractor Code. The code to print in the Prime Contractor Code block on the DD250 Form (block 9).
Vendor Maintenance > F5-Add Info
Cage Code. The code to print on the DD250 Form (block 16) immediately following the product number, if the vendor is the product’s primary vendor.
Customer Maintenance
The Government Information Screen allows you set up information to print a customer’s invoices print on the DD250 form.
Product Maintenance > F6-Xref/Notes > Alternates
If a product has an alternate product number and the alternate product is entered on a sales order, it prints above the product description (block 16) on the DD250 form.
