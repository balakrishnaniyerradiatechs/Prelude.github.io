Customer Ship-To Verification screen
The Customer Ship-To Verification screen allows you to verify if the correct ship-to account is selected for the customer’s sales order.
Verify Customer in SOE in Order Processing Parameters determines if the Customer Verification or the Customer Ship-To Verification screen appears. Choose from the following:
•
N - By-pass the customer verification screens.
•
Y - Display the customer’s name and address from the Customer Maintenance Screen and require verification.
•
S - Display the customer ship-to name and address from the Customer Ship-To Maintenance Screen and require verification.
Menu Path: Order Processing > Sales Order Entry > Enter a Customer number
Field and Function Key Descriptions
Field/Function Key
Description
Ship-To
A ship-to account address set up for the customer (refer to Customer Ship-To Maintenance). Enter one of the following:
•
The ship-to address account number.
•
D - Use the customer’s main address from Customer Maintenance.
•
P - The customer is picking up the order,
•
M - Opens the Customer Ship-To Maintenance Screen to allow you to create a new ship-to account.
•
C - Use a purchasing contract account.
Sales Rep1
The primary sales representative assigned to the customer in the Customer Maintenance Screen.
Sales Rep2
The secondary sales representative assigned to the customer in the Customer Maintenance Screen.
Customer Type
The customer type created in the Customer Type Maintenance Screen which is used to create a code to group similar types of customers for reporting and sales analysis. Once you create the code, you can assign it to customers in the Customer Maintenance Additional Customer Information Screen.
Is This The Correct Ship-To (Y/N)
Indicates if this is the correct ship-to address:
•
Y - The correct ship-to address. The Sales Order Header screen appears.
•
N - Not the correct ship-to address. The cursor returns to Ship-To to allow you to enter another ship-to account.
