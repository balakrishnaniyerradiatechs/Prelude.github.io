Component Entry screen
The Component Entry screen allows you to confirm kit component quantities for a shipped kit parent product on a sales order release.
Menu Path: Order Processing > Release Confirmation Entry > Action Bar > Component Entry
Field and Function Key Descriptions
Field/Function Key
Description
Parent
The parent product that shipped.
Product
The component product.
Description
The description of the component product.
U/M
The unit of measure of the component product as entered on the sales order.
Qty-Pck
The quantity shown on the sales order release as picked.
Qty-Shp
The quantity of the kit component that actually shipped.
F2-Save
Confirms the shipped quantities of the component products and returns to the Release Confirmation Entry screen.
