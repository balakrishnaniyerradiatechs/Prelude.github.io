Creating a Catalog ID Code
To create a catalog ID code:
1
Select Order Processing > Maintenance > Pricing Structures > Catalog Price Maintenance > Catalog ID Codes to open the Catalog ID Code Maintenance screen.
2
Enter a Catalog Code to create or modify. This code is used in Product Maintenance (F6-Xref/Notes > Alternates) to indicate which catalog the products belong to.
3
Enter the Catalog Description.
4
For Default Price Group, enter the product price group for the catalog code. This allows you to offer different pricing for each catalog. Product price groups are set up in Product Price Group Maintenance (refer to the Inventory documentation) and are used to group products for pricing. Any special pricing for the price group is maintained by a price matrix. When prices go up and you create a new catalog (refer to Duplicating a Catalog ID Code) the price group information is copied to the new catalog.
5
For Default Prefix, enter the 1 character prefix to use for alternate codes that belong to this catalog. The prefix allows the system to create a series of unique alternate codes for multiple catalogs (refer to Duplicating a Catalog ID Code). You can enter the catalog prefix and the part number as an alternate in Product Maintenance (F6-Xref/Notes > Alternates) so that when the customer orders the part number, they will get the pricing associated with the catalog designated by the prefix.
6
Press F2-Save to save the catalog ID information.
Delete a Catalog
To delete an expired catalog, select Order Processing > Maintenance > Pricing Structures > Catalog Price Maintenance > Delete Expired Catalog. Enter the catalog ID code to delete.
