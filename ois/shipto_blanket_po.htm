Customer Purchase Orders Screen
If the customer has specific purchase orders for a ship-to address, you can set up the purchase order, valid dates, and total value in the Customer Purchase Orders screen.
Menu Path: Order Processing > Maintenance > Customer Ship-to Maintenance > F7-Cust P/Os
Field and Function Key Descriptions
Field/Function Key
Description
Customer P/O#
A default customer purchase order number that can be used with this ship-to address, if applicable.
Start Date
The beginning date that this purchase order is valid.
Exp Date
The ending date that this purchase order is valid.
P/O Amount
The amount allowed on this purchase order. A warning message appears in sales order entry if a sales order for this customer exceeds this amount.
P/O Balance
The balance remaining on the purchase order, which is the P/O Amount less the value of any orders generated using this purchase order number.
P/O Desc
A description to reference this purchase order number.
F2-Save
Saves the purchase order information.
