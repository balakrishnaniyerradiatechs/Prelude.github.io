Open Release Inquiry
The Open Release Inquiry displays sales orders that are released for picking and are confirmed in Release Confirmation Entry but not yet invoiced.
Viewing Sales Order Releases
To list releases for a sales order:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Open Release Inquiry to open the Open Release Inquiry screen.
2
Enter the sales Order Number.
Open sales order releases appear.
Field & Function Key Descriptions
Field/Function Key
Description
Release#
The sales order release number.
Whse
The shipping warehouse assigned to the order.
Prt-Date
The date the pick ticket printed.
Prt-Time
The time the pick ticket printed.
Proc Dte
The date the pick ticket was confirmed in Release Confirmation Entry. Refer to Confirming Releases.
Ship Dte
The date the product was shipped from the warehouse.
Picker
The picker abbreviation of the person who picked the order (refer to Assigning Order Picker Abbreviations). Entered in Release Confirmation Entry.
Packer
The user ID of the person who packed the order. Entered in Release Confirmation Entry.
F2-Finish
Closes the inquiry screen.
F5-Review Release
Opens the Release History Confirmation Inquiry screen to display shipping information.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
