Quote Order History Inquiry
The Quotation Order History Inquiry lists purged quotes for a customer. Converted, expired, and cancelled quotes are purged when you run the Quotation Disposition Detail Report.
Viewing Purged Quotes
To list purged quotes for a customer:
1
Select Order Processing > Additional Subsystems > Quotation Subsystem > Inquiries > Quotation Order History Inquiry to open the Quote Order History Inquiry screen.
2
Enter the Customer for whom you want to view purged quotes.
3
Select the Sort sequence for the customer’s quotes. Choose from the following to sort the list by:
•
None.
•
Quote number.
•
Quote type.
•
Quote date.
•
Expiration date.
•
Customer P/O number.
•
Customer release number.
•
Quote total.
•
Descending quote type.
•
Descending quote number.
•
Descending quote date.
•
Descending expiration date.
•
Descending customer purchase order number.
•
Descending customer release number.
•
Descending quote total.
4
Determine if you want to use any Filters to limit the list of quotes. Choose from the following:
•
No. Lists all open quotes for the date.
•
Yes. Allows you to use the following selection criteria to limit the list:
-
Quote Number.
-
Customer Release Number.
-
Purchasing Contact.
-
Customer P/O Number.
-
Job Number.
Purged quotes for the customer display.
Field & Function Key Descriptions
Field/Function Key
Description
Quote#
The quote number.
Notes
An asterisk indicates that header notes exist for the quote.
Type
The quotation type:
•
01 - Regular quote.
•
02 - Planned quote.
•
03 - Repair type quote.
•
04 - Standing quote.
Disp#
The disposition analysis code that indicates if the quote was converted or deleted. If the quote was converted to a sales order, 00 displays. If it was deleted, the disposition code displays (refer to Creating Quote Disposition Codes). Converted or deleted quotes appear after they are purged in the Disposition Analysis reports (refer to the Quotation Disposition Detail Report).
Qte Date
The date the quote was entered.
Exp Date
The date the quote expires. If the customer does not accept the quote by this date, the quoted price is no longer valid.
Customer P/O#
The customer’s purchase order number.
Release#
The customer’s release number, if applicable.
Merchandise
The total price for all the merchandise quoted.
Conv
Indicates if the quote was converted to a sales order:
•
Y - The quote was converted to a sales order.
•
N - The quote was not converted to a sales order.
F2-Finish
Closes the Quote Order History Inquiry screen.
F5-Review
Opens the Quotation Display screen to allow you to view detail information for the quote.
F6-Enter Quote#
Allows you to enter a quote number to go directly to.
F8-Log
Opens the Quote Log Inquiry to allow you to view every transaction that occurred for this quote.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Returns to the previous page if more than one page of quotes are included in the inquiry.
sF6-Next Group
Proceeds to the next page if more than one page of quotes are included in the inquiry.
