Order Cancellation Reason Maintenance screen
The Order Cancellation Reason Code Maintenance screen allows you to create reason codes to use when an order is cancelled or deleted in Sales Order Entry. The reason codes explain why an order is cancelled. You can also update lost sales statistics when you cancel an order (refer to Creating Order Cancellation Reason Codes).
Menu Path: Order Processing > Maintenance > Misc Code Files > Order Cancellation Reason Codes
Field and Function Key Descriptions
Field/Function Key
Description
Ord Del Reason#
The 1 or 2 digit alphanumeric code that represents an order cancellation reason code.
Description
The description of the code that explains why an order is cancelled. This description displays on the Deleted Orders Report.
F2-Save
Saves the reason code.
F4-Del
Deletes the reason code.
