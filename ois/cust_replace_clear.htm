Clearing the Cross Reference File
After you update the customer records with the new information in the Universal Customer Replacement /Merge Update, you must clear the cross reference file before any new entries can be made in the Universal Customer Change/Merge Entry.
To clear the cross reference file:
1
Select Order Processing > Maintenance > Universal Customer Replacement/Merge > Clear Cross Reference File.
2
The following message appears.
This Process will clear out all the work files for Customer Replacement/merge and will clear out the audit record. This process must be run before any new customer replacement/merges may be entered. Do you wish to continue?
Choose from the following:
•
No - Cancels the process.
•
Yes - Clears the cross reference file and allows new entries in the Universal Customer Change/Merge Entry.
