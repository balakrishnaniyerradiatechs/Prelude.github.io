Sales Order Entry screen
The Sales Order Entry screen allows you to enter the products on a sales order. Refer to Entering Sales Order Line Items for detailed explanations of the features available in this screen.
Menu Path: Order Processing > Sales Order Entry
Field and Function Key Descriptions
Field/Function Key
Description
Order Number
The assigned sales order number.
Customer
The customer number and description.
Product
The product to sell to the customer. If you do not know the exact part number you can enter a partial number, product description, customer part number, or other keyword to search for products and press ENTER to open the Product Tagging Entry screen.
Description
The product description.
Notes
An asterisk (*) indicates that line item notes exist for the product on this order (Action Bar > Notes > Line Notes).
RFQ
If the product is from Vendor Request for Quote Entry, a Q indicates that a RFQ (request for quote) was sent to the vendor. Refer to Creating a Vendor RFQ from a Quote.
AOI
If the line item is placed on Administrative Order hold, the code for the violation appears:
•
A - Product always on hold.
•
C - Price overridden below cost.
•
G - Price overridden below GP%.
•
I - Incompatible ship via.
•
N - Price overridden below minimum price.
•
R - Price overridden below customer default price.
•
S - Quantity over month’s supply test.
•
V - A line item’s gross profit percent is below the Mgr Low Level Minimum GP% or more than the Mgr High Level GP% (designated in Order Processing parameters).
Refer to the
AII
If the line item is placed on Administrative Invoice hold, the code for the violation appears.
•
A - Credit Card Order – No Authorization: A credit card order is entered without an authorization code.
•
P - Price Violations: A price is changed to a value that is lower than the default price or lower than the cost. This only applies if Price Override Code for the User ID is set to C or R (refer to User Level Administrative Invoice Hold Options).
•
S - Salesrep Changed On Order: Salesrep One or Salesrep Two was changed on the Sales Order Header screen.
•
T - Order With No Tax Jur Codes: The order does not contain Tax Jurisdiction Codes.
•
U - User Defined.
Refer to the Administrative Invoice Inquiry.
Linkage
The document number if the product is linked to another order. The prefix indicates the document type:
•
P - Purchase order.
•
T - Transfer.
•
W - Work order.
Quantity
The quantity ordered by the customer.
U/M
The unit of measure for the product.
Whse
The warehouse that will ship the material.
Price
The customer’s price for the order.
Totals
Quantity
The total quantity of products on the order.
Weight
The total weight of the products on the order.
Profit
The total profit earned for the sales order.
Gp%
The total gross profit percent earned for the products sold.
Amount
The total sales dollars sold to the customer for the accumulative order.
F2-Save
Saves the information on the Sales Order Entry screen and allows you to do one of the following:
•
Print the pick ticket - The pick ticket is set up to print if inventory is available and if the order is not on hold. If your company uses continuous printing, the pick ticket automatically prints. If your company does not continuously print pick tickets, you must use Pick Ticket Printing to print the pick tickets. Refer to Printing Pick Tickets.
•
Complete - Opens Sales Order Completion screen to display the order total and to allow you to add freight, delivery, or handling charges or to take payment for the order. Refer to Using the Sales Order Completion Screen.
•
Suspend - Suspends the sales order so you can access it at a later time. No pick ticket prints until you go back into Sales Order Entry and choose to print the order.
•
Return - Allows you to return to the order to verify or edit any information.
F4-Del
Deletes the sales order. Refer to Manually Deleting a Sales Order.
F5-Hdr Scr
Opens the Sales Order Header screen to allow you to view or edit header information.
F6-Totals
Opens the Order Totals screen to allow you to view the total quantity, weight and profit amount of the order.
F8-Order Pad
Opens the Order Pad Inquiry to allow you to view products purchased by this customer.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Examine
Allows you to view pricing and shipping information for the order.
Review
Opens the Sales Order Review screen to display detail availability and pricing information for all the products on the order.
Prod Func
Provides access to the following product functions:
•
Add Comments Lines. Opens the Comment Description Entry screen to allow you to enter multiple comments for a line item.
•
Line Item Split. Opens the Order Line Split screen to allow you to split the quantity of an item for multiple delivery dates (refer to Splitting Line Items for Delivery Dates).
•
Rental Unit Sold. Applies to rental orders.
•
Buyer Message. Allows you to enter a message to send to the buyer, if the line item is linked to a purchase order.
Order Func
Provides access to the following sales order functions:
•
Reprice Order Options. Refer to Re-Pricing an Entire Sales Order.
•
Manual Administrative Order Hold (refer to Manually Placing Orders on Administrative Order Hold).
•
Manual Administrative Invoice Hold (refer to Manually Placing Orders on Administrative Invoice Hold).
•
Rate Shop. Opens the Rate Shop Results screen to allow you to compare 3rd party shipping rates for the ship via code used on this sales order.
•
Campaign Priority Code. Allows you to view or change the campaign priority code assigned to the sales order. Refer to the Marketing documentation for more information.
•
Allocated Inv. Review. Opens the Backorder Reallocation Availability screen to allow you to re-allocate backordered products from future orders. Only available if enough inventory of the backordered item will be received before the required date of the product on the future sales order. You can view the committed and available quantities and reallocate the products if necessary. refer to Re-Allocating Available Backorders.
•
Order Line Sort Options. Allows you to re-sort the lines on the sales order by product number, product description, or customer product number. Any comments associated with a part number will remain attached to the line item.
•
Import RF Batch Order Lines. Allows you to import RF batch sales orders into the sales order. All available sales orders for your user ID display to allow you to select the orders to include (05 & 06 confirmation orders only).
Notes
Allows you to enter:
•
Header Notes. Notes regarding the sales order header information.
•
Line Notes. Notes regarding the line item that your cursor is on.
•
Conversion notes for DOE conversion.
•
EDI Header notes for EDI transmissions.
•
Service problem notes for service orders.
•
Service solution notes for service orders.
