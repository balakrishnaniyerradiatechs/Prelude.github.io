Assigning the Package Insert Code to a Customer
Once you create a package insert code, you can assign it to a customer to determine when they should receive the free products.
To assign a package insert code to a customer:
1
Select Order Processing > Maintenance > Customer Maintenance to open the Customer Maintenance Screen.
2
Press F7-Order to open the Order Information Screen.
3
For Package Insert Code, enter the insert code created in Package Insert Code Maintenance.
4
Press F2-Save twice to save the insert information for the customer.
