Restricted Usage Code Maintenance Screen
The Restricted Usage Code Maintenance screen allows you to create usage codes that work with the usage codes in Customer Maintenance. The usage codes define the type of restricted products the customer can buy. These codes also work with the MSDS/Restricted Permits Information screen in Product Maintenance. The customer’s restricted usage codes must match the product’s restricted usage code for the customer to purchase the product. For more information, refer to the Customer Permit Information Screen.
You can also set up a restricted product matrix to restrict products for a state or zip code, refer to Restricting Products for a State or Zip Code.
Menu Path: Order Processing > Maintenance > Customer Control Files > Restricted Usage Codes
Field and Function Key Descriptions
Field/Function Key
Description
Usage Code
 
The restricted usage codes to assign to customers and products that determine if a customer can purchase a product that has sales restrictions. The customer’s usage codes must match the product’s usage codes. These codes can be either:
•
Normal restrictions. A code that is used in Product Maintenance to define products with restrictions and in Customer Maintenance to define which restricted product a customer can purchase.
•
Matrix restrictions. A state abbreviation for each state you sell or ship to. These codes are used in Company Maintenance (Action Bar > Restricted Lic Info > Usg Cde) to define which states you can sell to and in the Customer Maintenance Customer Permit Information Screen, to define which states the customer can purchase restricted products from.
Description
The description of the usage code or the state.
Req Permit Holder
Determines if this usage code requires you to enter a Permit Holder in the Customer Permit Information Screen in Customer Maintenance:
•
Y - A permit holder name is required. The permit holder is the name of the person who is authorized to buy a restricted usage product.
•
N - A permit holder name is not required.
F2-Save
Saves the restricted usage code information.
F4-Del
Deletes the restricted usage code.
