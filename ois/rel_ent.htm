Release Confirmation Entry screen
The Release Confirmation Entry screen allows you to confirm products, quantities, substitute information, freight amounts, and other shipping information before a sales order is released for invoicing.
If you access this screen through Warehouse Release Confirmation Entry, (refer to Printing Packing Lists for Non-Confirmed Releases), you can print the packing list but the order is not updated.
Menu Path: Order Processing > Release Confirmation Entry
Field and Function Key Descriptions
Field/Function Key
Description
Release Number
The sales order number with a dash and the release (e.g., 10000178-1).
Customer
The customer number and description.
Product
The product on the sales order release.
Description
The description of the product.
Qty-Opn
The quantity entered on the sales order.
U/M
The unit of measure of the product as entered on the sales order.
Qty-Pck
The quantity that the system calculates as available and ready to pick on this pick ticket release.
Qty-Shp
The quantity that actually shipped to the customer. You can change the quantity as necessary.
F2-Save
Saves the information and allows you to choose one of the following:
•
Update/Invoice - Update the order as confirmed so it can be invoiced.
•
Update/Hold - Save the confirmed information but do not release it for invoicing until you re-enter the order and release it. Do not use this option unless instructed by your manager.
•
No Update - Do not update the order as confirmed. You may want to use this if you entered the wrong release number or if for some reason this order is not to be updated.
•
Return - Allows you to return to the Release Confirmation Entry screen to verify or change quantities.
F4-Del
Deletes the release.
F5-Hdr Scr
Opens the Release Header screen to allow you to view or edit the shipping information.
F7-Consol Shipment (Only accessible before you enter a release number)
If you use PSS (parcel shipping system) Manifest Interface, the Consolidation Release Confirmation Entry screen appears to allow the available sales order releases for a customer ship-to /warehouse to update as one shipment through the PSS. The freight amount for the entire shipment is included on the first sales order release. The remaining releases will have a comment line referencing the freight information on the first release.
F9-Goto-Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Component Entry
Opens the Component Entry screen to allow you to confirm kit component quantities for a shipped kit parent product on a sales order release.
Notes
Allows you to enter or edit notes regarding the sales order release.
Image
Allows you to copy a proof of delivery image if you use the imaging software.
