Check For Existing Orders
Check for Existing Orders (Option Parameters > Order Processing Parameters > F5 > F5) allows you to determine if you want to add to open orders for a customer before you create a new order for the same customer:
•
Y - When you save the Sales Order Header screen, a message displays to allow you to add to non-released orders to the customer ship-to. If you select Yes, the first type 01 non-released sales order opens to allow you to add products.
•
P - The same as Y except that the system only searches for orders that have a pickable quantity.
•
N - No message displays if open orders exist.
This feature only applies to type 01 sales orders that are going to the same ship-to address and for which the customer does not require a purchase order.
