Quote Follow Up Inquiry
The Quote Follow Up Inquiry displays open quotes for a Written By user ID that have a Follow Up Date (entered in the Quotation Header screen) on or before the date that you specify. You can use this information to trigger a follow-up call to the customer to determine if:
•
The quote was approved. You can convert the quote to a sales order (refer to Converting Quotes into Sales Orders).
•
The quote was not accepted. You can delete the quote and update lost quotations (refer to Deleting Quotes).
•
The outcome of the quote is not yet determined. You can re-set the follow up date on the Quotation Header screen so that the quote will appear in the Quote Follow Up Inquiry again.
Follow Up Quotes
QFUI Dollar Limit (Order Processing Parameters > F5 > F5) determines the quotes that appear in this inquiry. You can enter a quote follow up limit so that only quotes that are above the limit are included in the inquiry. For example, enter 15 so that only quotes above $15.00 appear in the Quote Follow Up inquiry.
Viewing Follow Up Quotes For a Written By ID
To list open quotes for a salesperson:
1
Select Order Processing > Additional Subsystems > Quotation Subsystem > Inquiries > Quote Follow Up to open the Quote Follow Up Inquiry screen.
2
Enter the Written By user ID number from the Quotation Header screen.
Follow up quotes for the Written By ID display.
Field & Function Key Descriptions
Field/Function Key
Description
Quote#
The quote number.
Notes
An asterisk indicates that header notes exist for the quote.
Sell
The selling warehouse for the quote.
Customer#
The customer who requested the quote.
Description
The description of the customer.
Followup Date
The Follow Up Date entered in the Quotation Header screen that determines when to follow up on a quote if it is not yet converted or deleted.
Exp Date
The date the quote expires. If the customer does not accept the quote by this date, the quoted price is no longer valid.
Merchandise
The total price for all the merchandise quoted.
F2-Finish
Closes the Quote Follow Up Inquiry screen.
F5-Review Quote
Opens the Quotation Display screen to allow you to view detail information for the quote.
F7-QOE
Allows you to convert the quote into a sales order (refer to Converting Quotes into Sales Orders).
F8-Log
Opens the Quote Log Inquiry to allow you to view every transaction that occurred for this quote.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
