Shipping Consolidation screen
The Shipping Consolidation screen appears upon completion of a sales order (if Shipping Whse Consolidation in Order Processing Parameters is set to S) to allow you to consolidate as many lines as possible from a single warehouse.
Menu Path: Order Processing > Sales Order Entry > F2-Save > Complete
Field and Function Key Descriptions
Field/Function Key
Description
Order Number
The sales order number.
Customer
The customer number and name.
Ship Warehouse
The shipping warehouse from the sales order.
Order Total
The total value of the sales order.
Outbound Freight
The cost of outbound freight added to the sales order.
Whse
The warehouses where the line items are originally set to ship from.
B
An asterisk indicates that the product is backordered in the original warehouse.
Ship Via
The method of shipping the material to the customer.
Description
The description of the ship via code.
#Days
The number of days to allow for delivery.
Weight
The total weight of the products shipping from each warehouse.
# Items
The number of line items that will ship out of each warehouse.
TotQty
The total quantity of products that will ship from each warehouse.
F2-Save
Saves the shipping consolidation information.
F5-Detail
Opens the Shipping Consolidation Detail screen to display the detail product information for all items shipping on the order.
F6-Summary
Opens the Shipping Consolidation Summary screen.
F7-Consolidate
Consolidates as many lines as possible to a single warehouse.
