Open Order Release Inquiry
The Open Order Release Inquiry displays the status of open sales orders in a selling warehouse, for which pick tickets are printed but not yet invoiced.
Viewing Open Releases
To select open releases for a warehouse:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Open Order Release Inquiry to open the Open Order Release Inquiry screen.
2
Enter the Warehouse.
3
Select a Sort sequence to display the orders by:
•
None.
•
Customer number.
•
Print date.
•
Route/delivery sequence.
•
Ship via.
•
Descending customer number.
•
Descending print date.
•
Descending route/delivery sequence.
•
Descending ship via.
4
If you want to use any Filters to limit the list of orders, choose from the following:
•
No. Lists all open orders for the warehouse.
•
Yes. Opens the Open Order Release Inquiry Filter Options screen to allow you to limit the list by the following options:
-
Warehouse.
-
Customer.
-
Order number.
-
Print date range.
-
Customer P/O number.
-
Ship via.
-
Route.
-
Status.
Open releases for the warehouse appear.
Field & Function Key Descriptions
Field/Function Key
Description
Release#
The sales order release number.
Customer
The customer assigned to the sales order release.
Print Date
The date the release printed.
Route
The delivery route for fleet deliveries if shipping via company truck.
DelSeq
The delivery sequence within the route.
User
The default is the ship via code but you can use this column for custom information (e.g., zone, picker, packer).
Status
The status of the shipment if using RF picking or SSCE:
•
Waiting. The release is waiting for a picker.
•
M-Pick. The release has a picker.
•
Picked. The release has a picker if RF picking is complete.
•
PT-Pick. The release has a picker, RF picking is not done if lines still need to be picked.
•
Boxing. The release has a packer but no shipping date.
•
Boxed. The release has a packer and a process date.
•
Shipped. The release has a waybill number.
•
Loaded. The boxes are loaded (for RF truck loading).
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the Release Confirmation Inquiry screen to display picking, shipping, and box number information for the order.
F7-Refresh
Refreshes the data on the screen if changes were made.
F8-Log
Opens the Order Log Inquiry which lists each transaction that affected the order.
F9-Goto Ln
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Lists the previous group of orders, if applicable.
sF6-Next Group
Lists the next group of orders, if applicable.
Action Bar
Output
Prints, e-mails, or faxes the information on the screen.
