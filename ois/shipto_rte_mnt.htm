Ship To Route Maintenance Screen
If you use delivery trucks and follow delivery routes, you can use the Ship-To Customer Route Maintenance screen to establish the route and time information for a customer ship-to address, including any special delivery requirements they may have.
Menu Path: Order Processing > Maintenance > Customer Ship-to Maintenance > F5-Add Info > Action Bar > Routes
Field and Function Key Descriptions
Field/Function Key
Description
Route#
The default route number, that indicates the route normally used to deliver merchandise to this customer. The route numbers are set up in the Route Maintenance Screen and contain valid delivery day and time information for each route.
Description
The description of the route.
Del Seq
If you have delivery trucks and standard delivery routes, you can assign a number that indicates where in the delivery sequence the customer will receive their orders.
Beginning Delivery Time
The earliest time that the customer accepts deliveries.
 
Restricted Start Time
The beginning of the time frame during which the customer does not accept deliveries, if applicable.
Restricted End Time
The ending of the time frame during which the customer does not accept deliveries, if applicable.
Ending Delivery Time
The latest time in the day that the customer accepts deliveries.
Delivery Requirements
The delivery requirement code if the customer has special delivery requirements.
Dynamic Routing Longitude
The longitude to use with interfacing dynamic routing software (custom modification used with 3rd party software).
Dynamic Routing Latitude
The latitude to use with interfacing dynamic routing software (custom modification used with 3rd party software).
Dynamic Routing Zone
The zone to use with interfacing dynamic routing software (custom modification used with 3rd party software).
F2-Save
Saves the customer route information.
