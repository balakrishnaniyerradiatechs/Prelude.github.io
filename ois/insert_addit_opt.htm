Additional Options screen
To prevent excessive product inserts from being sent with customer orders, you can use the Additional Options screen in Package Insert Code Maintenance to limit the number of inserts given away.
Menu Path: Order Processing > Maintenance > Order Entry Control Files > Package Insert Code Maintenance > F7-Addtl Options
Field and Function Key Descriptions
Field/Function Key
Description
Maximum Inserts
The maximum number of insert products to send out with any order. This number is used with the Ranking order to determine which products to add to an order.
Ranking
The codes, in order of importance, to determine which inserts are sent if value exists in Maximum Inserts. The first, second, and third order insert products are automatic and do not need to be included in the ranking. You can add one or more of the following:
•
D - Days since last order.
•
L - Product lines.
•
P - Product numbers.
•
V - Total order value.
Notification
The codes that will add a message to the sales order to let the salesperson know that an insert is being sent:
•
D - Days since last order.
•
L - Product lines.
•
P - Product numbers.
•
F - First order.
•
S - Second order.
•
T - Third order.
•
V - Total order value.
F2-Save
Saves the insert information and returns to the Package Insert Code Maintenance screen.
