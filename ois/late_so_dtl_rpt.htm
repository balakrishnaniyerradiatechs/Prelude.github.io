Late Sales Order Detail Report
The Late Sales Order Detail report lists open sales orders for which the pick ticket printed after the Required Date on the sales order. You can limit the report to list all orders that are less than a number of days late. The report sorts by descending number of days late.
Late Orders in File Update Parameters must be set to Y to track late sales order information.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Late Sales Order Detail Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The shipping warehouse.
Column Descriptions
Column Heading
Description
Customer
The customer number.
Description
The customer description.
Order#
The sales order number that printed past the required date.
Cust PO
The customer purchase order number, if applicable.
Days Late
The number of days the pick ticket printed after the required date.
Ord Date
The date the order was entered.
Req Date
The date entered in Required Date on the Sales Order Header screen that indicates when the customer required the order.
Order-Total
The total selling price of the sales order.
Order Cost
The total cost for sales of the order.
Gross Prof
The total gross profit amount earned on this order.
GP%
The gross profit percent earned on this order (Gross Prof / Order Total).
