Creating Order Cancellation Reason Codes
When you delete an order in Sales Order Entry (refer to Manually Deleting a Sales Order), if your company tracks deleted sales orders, you must enter a reason code that explains why you are deleting the order. You can also update lost product sales usage when you delete an order (refer to Creating Lost Sales Reason Codes).
File Update Parameters
To track deleted sales orders, set Deleted Orders in File Update Parameters to Y so that the salesperson is required to enter an order cancellation reason code.
To create order cancellation reason codes:
1
Select Order Processing > Maintenance > Misc Code Files > Order Cancellation Reason Codes to open the Order Cancellation Reason Maintenance screen.
2
For Ord Del Reason#, enter the 1 or 2 digit alphanumeric code that represents an order cancellation reason code.
3
Enter the Description of the code that explains why an order is cancelled. This description displays on deleted order reports (e.g, price too high, out of stock, late delivery).
4
Press F2-Save to save the reason code.
The DEL code displays in the Order Log Inquiry when a sales order is deleted.
