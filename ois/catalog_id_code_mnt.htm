Catalog ID Code Maintenance screen
If you use catalogs to sell your products, you can use the catalog ID Code Maintenance screen to set up catalog identification codes to assign to products.
Menu Path: Order Processing > Maintenance > Pricing Structures > Catalog Price Maintenance > Catalog ID Codes
Field and Function Key Descriptions
Field/Function Key
Description
Catalog Code
The code that identifies a catalog. You can assign this code to products in Product Maintenance (F6-Xref/Notes > Alternates) to indicate which catalog a product belongs to.
Catalog Description
A description of the catalog.
Default Price Group
The product price group for the catalog code. You can have different pricing for each catalog. Product price groups are set up in Product Price Group Maintenance (refer to the Inventory documentation) and are used to group products for pricing. Any special pricing for the price group is maintained by a price matrix. When prices go up and you create a new catalog (refer to Duplicating a Catalog ID Code) the price group information is copied to the new catalog.
Default Prefix
The character prefix to use for alternate codes that belong to this catalog, if applicable. This creates a series of unique alternate codes for multiple catalogs (refer to Duplicating a Catalog ID Code). You can enter the catalog prefix and the part number as an alternate in Product Maintenance, so that when the customer orders the product, they get the pricing associated with the catalog designated by the prefix.
F2-Save
Saves the catalog ID information.
