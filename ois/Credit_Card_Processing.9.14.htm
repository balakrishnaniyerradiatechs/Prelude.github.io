Manually Adding an Authorization Code
If your credit card authorization system is down and you must pre-authorize a credit card sale, you can manually enter the authorization number that you receive over the phone.
To manually enter a pre-authorization code for a credit card sale:
1
Select Order Processing > Processes > Credit Card Processes > Additional Batch Functions > Add Manual Auth Code to Order to open the Sales Order Credit Card Authorization Entry screen.
You cannot add credit card information in this screen, it must already exist in the sales order.
2
Enter the Order Number for which you need to add an authorization code. The customerâ€™s name and address information displays.
3
For Auth Num, enter the authorization number that you received over the phone.
4
Press F2-Save to complete the authorization.
