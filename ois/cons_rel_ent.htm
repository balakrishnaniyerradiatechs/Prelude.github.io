Consolidation Release Confirmation Entry screen
If you use the PSS (Parcel Shipping System) Manifest Interface, the Consolidation Release Confirmation Entry screen allows you to combine the available sales order releases for a customer ship-to/warehouse to update as one shipment through PSS. The freight amount for the entire shipment is included on the first sales order release. The remaining releases will have a comment line referencing the freight information on the first release.
Menu Path: Order Processing > Release Confirmation Entry > F7-Consol Shipment (before you enter a release number)
