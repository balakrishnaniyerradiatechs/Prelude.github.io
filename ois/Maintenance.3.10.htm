Entering a Floor Plan Sales Order
Once a customer is set up to use floor plan financing (refer to Setting up a Customer to use Floor Plan Financing), the following message appears when you enter the customer’s number on a sales order:
This Order will be a Floor Plan Order?
1
Choose from the following:
•
No - The order is entered and billed as normal. The customer pays their own invoice.
•
Yes - Floor Plan is automatically set to Y on the Sales Order Header Order Additional Information Screen.
2
If only one Floor Plan Customer is set up for this customer in the Floor Planning Maintenance screen (refer to Setting up a Customer to use Floor Plan Financing), the customer name and description appear in the next (unlabeled) field. If the customer has more than one floor plan customer, you must enter a valid floor plan customer number.
3
If Plan Number/Approval Req is set to Y for the customer (refer to Setting up a Customer to use Floor Plan Financing), the Floor Plan Approval/Plan Number screen appears.
4
Enter Approval Number issued from the Floor Plan customer.
5
Enter the Floor Plan Number (refer to Creating Floor Plan Numbers) that references the agreement set up with the Floor Plan customer.
6
If applicable, enter the Suffix for the floor plan number.
7
Press F2-Save to save the header information and to continue the order as usual. You cannot exit the header screen until all of the floor plan information complete.
