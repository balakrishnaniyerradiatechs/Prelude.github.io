Quick Matrix Entry Product/Number screen
The Quick Matrix Entry Product/Number screen allows you to modify matrix types 1 and 6. When you delete a product, you can also delete the price matrix record. If the price matrix record contains multiple date ranges, only the date range displayed is removed.
If Keep Price Matrix Maint Audit is set to Y (File Update Parameters -> F6-Add Info) the Pricing Audit Reports (Order Processing Reports -> Miscellaneous Reports menu) are updated with the modified price matrix.
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Quick Price Matrix Entry > Matrix types 1 or 6
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer whose prices are being updated.
Product
The products included in the matrix.
Beg-Date
The starting date for the prices to take effect.
Exp Date
The ending date for the special pricing.
Mt
Determines the method to calculate the price:
•
$ - The product has a set price. The cursor moves to Price to allow you to enter the price.
•
* - The price is a percentage of a column price or cost. Enter the Basis and Pct.
•
T - A Quantity Discount Table calculates the price. The cursor moves to Table to allow you to enter a discount break table. Refer to Discount Break Tables for information on setting up a discount break table.
Table
The discount break table number to use.
Price
If you entered $ for Mt, enter the price.
Basis
If you entered * for Mt, indicate which price or cost column (from Product Maintenance > F7-Pricing) to use for the calculation. The discount calculated is a percentage over a price column if a price is referenced or a percentage over a cost if a cost is referenced.
Pct
If you entered a price column in Basis, enter the discount to give off the price. This value appears in Disc% on the Line Item Detail screen on the sales order. If you entered cost column in Basis, enter a percentage over the cost to calculate the sales price. For example if you enter 20 for Pct, the cost is divided by .80 to determine the sales price. This value appears in Net Price on the Line Item Detail screen on the sales order.
F2-Save
Saves the price matrix information.
