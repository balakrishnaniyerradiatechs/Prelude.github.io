Releasing Orders from Branch Hold
The Branch Release Inquiry lists sales orders and transfers on Branch Hold. The branch manager can review each order and determine whether to release or cancel the order or transfer.
•
A sales order is placed on hold when a line item’s entire quantity cannot be filled from surplus in the shipping location (and the shipping location is different from the selling location). Interbranch/Surplus Only in Order Processing Parameters must be set to Y.
•
A transfer is placed on hold when a line item’s quantity lowers stock below the safety level in the shipping location. Transfer Branch Hold in Inventory Parameters must be set to Y or M.
To review and/or release an order on Branch Hold:
1
Select Order Processing > Inquiries > Branch Release Inquiry to open the Branch Release Inquiry screen.
2
Enter a Branch Manager Id or accept the default.
3
Sales orders (S prefix) and transfers (T prefix) on Branch Hold appear.
4
Choose one of the following:
•
To view an order, press F5-Review to open the Branch Release Inquiry screen. You can view availability information for each product and determine if you want to release each item on the order:
-
To release an item, enter Y under Accept.
-
To cancel an item, enter N under Accept.
•
To release or cancel the entire order without reviewing it, press F8-Release. The following message appears:
Do You Wish To Release Items For This Document From Branch Hold?
Choose from the following:
-
Return - Goes back to the inquiry screen.
-
Release All - Releases all the products on the sales order or transfer from Branch Hold.
-
Reject All - No products are released.
5
Once you release or cancel the products on each order, the Branch Release Message screen appears so you can enter a message to send to the salesperson assigned to the order.
6
Repeat steps 4-5 for each order in the Branch Release Inquiry.
7
Press F2-Finish to close the screen.
Field & Function Key Descriptions
Field/Function Key
Description
Document#
The order number that is on Branch Hold. The prefix indicates the document type:
•
S - Sales order.
•
T - Transfer.
Whse
The selling warehouse.
Customer
The customer or warehouse assigned to the order.
Description
The customer description.
Rep
The sales representative assigned to the order.
User
The user ID of the person that entered the order.
Reason
 
The reason code that identifies why the order is on branch hold:
•
R - Reserved inventory at insufficient level. The sales order or transfer line item quantity places the product below its reserved quantity and the shipping location is different from the selling location.
•
S - For sales orders: If a line item’s entire quantity cannot be filled from surplus in the shipping location (and the shipping location is different from the selling location).
For transfers: If a line item’s quantity lowers stock below the safety level in the shipping location.
Message
A message sent by the person that entered the order when it went on Branch Hold.
F2-Finish
Closes the inquiry screen.
F5-Review
Sales orders: Opens the Branch Release Display screen so you can review the order and available quantities. You can also release or cancel line items from this screen and send messages to the employee queue of the sales representative assigned to the order (refer to the Employee Queue Inquiry).
Transfers: Opens the Transfer Branch Release Inquiry screen so you can review available quantities and send a message to the employee queue of the sales representative assigned to the order (refer to the Employee Queue Inquiry).
F7-Refresh Scr
Refreshes the screen if changes are made.
F8-Release
Allows you to release or reject the entire order without reviewing it. You can also send a message to the sales representative assigned to the order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
