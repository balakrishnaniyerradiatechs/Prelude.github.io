Customer Price Group Copy screen
The Customer Price Group Copy screen allows you to copy a customer price group matrix to another customer.
Menu Path: Order Processing > Maintenance > Pricing Structures > Price Matrix Maintenance > Price Matrix Maintenance (select a matrix type for a customer price group) > Action Bar > Copy
Field and Function Key Descriptions
Field/Function Key
Description
Copy Option
 
Determines the copy method:
•
O - Overwrites the target matrix with the source matrix. The original target matrix is deleted. You are warned if the target matrix exists and allowed to exit without overwriting.
•
I - Inserts the source matrix at the beginning of the target matrix and retains the existing target price matrix data. You can either insert the entire source matrix or a single line of the source matrix. You may want to use this option to update promotional or temporary pricing.
•
A - Appends the source matrix to the end of the target matrix and retains the existing target price matrix data. You can either append the entire source matrix or a single line of the source matrix into the target matrix. You may want to use this option to update future prices that are effective after the existing matrix expires.
•
D - Deletes all the price matrices for the customers defined. To delete the matrix, the same combination of matrices must be setup for the customer or customer price groups. For example, if deleting Price Matrix Type 1, customers 981234, 001000, and 000100 must have the same product on their matrices as the source matrix. Only records with the same product are deleted. The price matrices that have range numbers as part of the matrix, must have the same range number for the customers or customer price groups to be deleted.
Line Number
The line number from the existing matrix to insert or append. Only applicable if you chose A (append) or I (insert) for Copy Option. Leave blank to insert or append all line items for the current matrix.
Listname
The select list of customers or customer price groups to copy the current matrix to (if applicable).
Customer Price Grp
The customer price group to copy the current matrix to.
F2-Save
Copies the price matrix information.
