Historical Percentage Credit Reason Report
The Historical Percentage Credit Reasons report lists the total amount of credit issued to customers for each month. The report lists each credit reason code (refer to Creating Credit Reason Codes).
You can also view the percentage of total credit that each credit reason code was used for each month.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Historical % Credit Reasons
Column Descriptions
Column Heading
Description
Code
The credit reason code which identifies why the credit was issued.
Description
The description of the credit reason code.
Months
(horizontal)
The percentage of the total credit that each credit reason code was used for each month.
Months
(vertical)
The total amount of credit issued for all credit memos for each month. Refer to Entering a Credit Memo for more information on credit memos.
