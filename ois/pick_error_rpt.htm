Pick Error Report
The Pick Error report lists picking errors by warehouse. Picking errors are recorded when the shipped quantity is changed from the pick ticket quantity in Release Confirmation Entry.
After you run the report, you can choose to purge the product information so that it is not included the next time you select the report.
Pick Error in File Update Parameters must be set to Y to capture pick error information.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Pick Error
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse that shipped the material.
•
Picker Number. The number assigned to the person that picked the order (refer to Assigning Order Picker Abbreviations).
Column Descriptions
Column Heading
Description
Pkr
The picker’s initials.
Description
The description (name) of the picker.
Product Number
The product that had a picking/inventory error.
Description
The product description.
Release#
The sales order release number that contains the products with the incorrect quantity.
Cust#
The customer who purchased the product.
Qty-Pck
The quantity that printed on the pick ticket.
Qty-Shp
The quantity that actually shipped.
