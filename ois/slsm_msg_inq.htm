Employee Order Message Inquiry
The Employee Order Message Inquiry contains orders released from Credit Hold, Administrative Order Hold, or Administrative Invoice Hold. You can view messages sent from the administrative managers then edit or complete the sales orders as necessary. You can only edit orders for which your user ID is the Written By.
Menu Path: Order Processing > Inquiries > Employee Queue Inquiry > Order Message Inquiry > F5-Review
Field & Function Key Descriptions
Field/Function Key
Description
User Id
The user ID of the employee for this queue.
Order#
The sales order number released from hold.
N
An asterisk (*) indicates that notes/messages exist for the order. Select the Notes option on the Action Bar to view:
•
Header notes.
•
Administrative Order Hold notes.
•
Administrative Invoice Hold notes.
Customer
The customer number and description.
CH
Indicates the order’s credit hold status:
•
R - Released from credit hold.
•
H - Still on credit hold.
•
Blank - Was not on credit hold.
AO
Indicates the order’s administrative order hold status:
•
AO - The order was reviewed but not released from order hold. Notes were entered by the manager to explain why it was not released.
•
Blank - Not on order hold.
Refer to the Administrative Order Inquiry for information on order on administrative hold.
AI
Indicates the order’s administrative invoice hold status:
•
AI - The order was reviewed but not released from invoice hold. Notes were entered by the manager to explain why it was not released.
•
Blank - Not on invoice hold.
Refer to the Administrative Invoice Inquiry.
Message
Indicates which type of hold the order was released from. Could also be a message sent by the manager.
Sources
Indicates if the sales order was linked to a purchase order or transfer:
•
Y - The sales order was linked to a purchase order or transfer that could not be processed because it was on hold.
•
N - The sales order was not linked to a purchase order or transfer.
F2-Save
Closes the inquiry screen.
F4-Del
Deletes all lines from the Order Message Inquiry.
F5-Review
Opens the Sales Order Display screen to access detail information on the sales order.
F6-SOE
Opens the Sales Order Entry screen to allow you to edit the sales order and print the pick ticket, unless the hold manager has released it.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Refresh
Clears and re-sets the screen if changes were made.
Alternate User
Allows you to send the message to another user ID.
Notes
Allows you to view any header notes, administrative invoice hold notes, or administrative order hold notes for the order.
