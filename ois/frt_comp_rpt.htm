Freight Comparison Report
The Freight Comparison report allows you to compare inbound, outbound, and manifest freight amounts for orders shipped to customers within a time period. You must use the Parcel Shipping System (PSS) to record manifest freight.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Freight Comparison Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Invoice Date Range. The beginning and ending invoice dates to include in the freight comparison.
•
Warehouse. The warehouse that shipped the orders.
•
Grand Totals Only?. Determines if you want to only display total freight amounts for the time period or if you want to show freight information for each order that shipped during the time period.
-
Y - Only show totals for each type of freight.
-
N - Show freight for each order shipped.
Column Descriptions
Column Heading
Description
Customer
The customer that received the shipments.
Description
The customer description.
Inbound Frt
The total inbound freight amount for orders shipped to this customer.
Outbound Frt
The total outbound freight amount for orders shipped to this customer.
Manifested
The total manifested freight amount for orders shipped to this customer. This value is updated from the Outbound Freight in Release Confirmation Entry.
Difference
The difference between the freight amounts entered on the sales order and the freight amount recorded in the manifest (Inbound Frt + Outbound Frt) - Manifested.
Merchandise
The total value of merchandise sold to this customer in the selected time frame.
