M/R Order Header screen
The M/R Order Header screen allows you to enter the customer information for a manufacturer’s rep order. Refer to Entering a Manufacturer’s Rep Order.
Menu Path: Order Processing > Additional Subsystems > Manufacturers Rep Subsystem > Manufacturer Rep Entry > F5-Hdr Scr
Field and Function Key Descriptions
Field/Function Key
Description
Customer Number
The number for the customer that placed the order with the vendor.
Ship-To Number
The customer’s ship-to number. Enter D to use the customer’s default address from Customer Maintenance.
Sell Warehouse
The warehouse that will receive credit for the sale.
Customer P/O#
The customer’s purchase order number.
Salesrep #1
The sales representative assigned to the customer (from Customer Maintenance). Can be changed for this order.
Written By
The person entering the order.
Entry Date
The first date the vendor notified you of the order.
F2-Save
Saves the header information and returns to the Manufacturer Rep Order Entry screen.
F5-Ship-To
Opens the M/R Ship-To Address screen to allow you to enter or change shipping information for the customer.
