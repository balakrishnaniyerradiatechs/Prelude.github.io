Linking a Purchase Order to a Backorder
When you enter a sales order for a stock or non-stock product that is backordered, company settings determine if and how you can create a purchase order for the product.
Requirements to Generate a Purchase Order From a Sales Order
Line Item Detail screen
The Sc (supply code) for the line item must be set to B, E, or D.
User Identification Maintenance > F6- Add Info
Enter P/O from S/O must set to Y for your user ID.
Option Parameters > Purchasing Parameters
Set the following Purchasing Parameter options
S/O-P/O, N/S Items Only.:
•
Y - You can only generate purchase orders for non-stock items.
•
N - You can generate purchase orders for stock and non-stock items.
Automate P/O Generation from SOE::
•
Y - The purchase order is automatically generated without the option to edit it.
•
N - You have the option to edit the purchase order before generating a P/O.
Cancel Sls Order Backorder?:
•
Y - Cancel the remaining backorder on a linked sales order if the backordered quantity on the purchase order is cancelled.
•
N - Do not cancel the backorder on the sales order.
To generate purchase orders for line items:
1
If the above requirements are met, the Cost Prompt screen appears.
2
Enter a cost and press ENTER. A dialog box with the following message appears:
Do You Wish To Automatically Generate A Purchase Order For This Product?
3
Choose one of the following:
•
Yes - Generate the P/O at the end of the order.
•
No - Allow the P/O to generate as part of Nightly Processing.
4
If you select Yes, the Automatic Purchase Order Vendor screen appears.
5
The primary vendor for the product appears; enter a different vendor number if necessary.
6
Press F2-Save.
7
After you complete the order (refer to Finishing the Sales Order), if the above requirements are met, a dialog box with the following message appears:
Do You Wish To Proceed WIth Purchase Order Update?
8
Choose one of the following:
•
Yes - Immediately generate the purchase orders.
•
No - Generate the purchase orders in Nightly Processing.
9
If you select Yes, the Generate P/Os for Backorders screen lists the purchase orders for each vendor you chose.
10
Choose from the following:
•
F2-Generate All: Generates purchase orders for the vendors listed.
•
F5-Edit: Allows you to edit the selected purchase order.
•
F6-Generate: Generates one purchase order at a time for the highlighted vendor.
11
After the orders generate, press F9-Continue. The output dialog box appears for each vendor.
12
Select the appropriate method to deliver the purchase order to the vendor.
