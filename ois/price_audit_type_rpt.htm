Price Maintenance Audit Report by Type
The Price Maintenance Audit Report by Type displays, by matrix type, all changes to a price matrix either in Price Matrix Maintenance or in Quick Matrix Entry.
After you run the report, you can choose to purge the product information so that it is not included the next time you run the report.
Track Changes
Price Matrix Maint Audit in File Update Parameters must be set to Y to record price matrix changes.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Pricing Reports > Pricing Audit Reports > Price Maintenance Audit Report By Type
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Matrix Type. The price matrix type for which you want to view changes.
•
Date Range. The beginning and ending dates for which you want to view changes.
Column Descriptions
Column Heading
Description
Matrix Type
The matrix type set up in Price Matrix Maintenance (Refer to Creating a Price Matrix).
Cust/CPG
The customer number or customer price group, depending on the matrix type.
PN/PPG/PLINE/VN/VLINE/MLINE/
PRC RNG
The part number, product line, vendor number, vendor line, or product price range, depending on the matrix type chosen.
Whse
The selling location for the price matrix, if it is warehouse specific.
Type
 
Ent-Date
The date the matrix was modified.
Ent-Time
The time of day that the matrix was modified.
User
The user ID of the person that modified the matrix.
Description
The description of the user ID.
Description
(2nd line)
The description of the field that changed.
Before Value
The value of the field before the change.
After Value
The value of the field after the change.
