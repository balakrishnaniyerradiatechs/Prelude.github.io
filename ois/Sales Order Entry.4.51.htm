Creating a Warehouse Hold Area
A warehouse hold area allows you to stage material for ship-complete and tag & hold order types.
Standard Hold Area Listing
To view a list of warehouse hold areas set up in Warehouse Hold Area Maintenance, select Inventory > Listings > Miscellaneous Listings > Warehouse Hold Areas.
To create a warehouse hold area:
1
Select Inventory > Maintenance > Misc Code Files > Warehouse Hold Areas to open the Warehouse Hold Area Maintenance screen.
2
Enter the Warehouse where you want to set up the hold area.
3
For Hold Area Number, enter a code to indicate where hold orders are assigned.
4
Enter the Description of the hold area.
5
If necessary, enter a second line of description in Description Two.
6
Press F2-Save to save the warehouse hold area information.
