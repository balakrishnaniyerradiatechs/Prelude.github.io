Stand-Alone Email Shipping Notice screen
You can use the Stand-Alone Email Shipping Notice screen to text email a shipping confirmation to a customer for a sales order after it ships by just entering a release number and an email address. The customer does not have to be set up to automatically receive text email shipping confirmations.
Text Email Documents
Email Document Text Maintenance (Miscellaneous Controls) must be set up with the correct document codes before you can text email a shipping confirmation.
Menu Path: Order Processing > Optional Entry & Printing > Stand Alone Email Shipping Notice
Field and Function Key Descriptions
Field/Function Key
Description
Release Number
The release number for the shipped material.
Customer
The customer receiving the material.
Email Address
The email address from the Sales Order Header screen. Change as necessary.
F2-Send Email
Transmits the email.
