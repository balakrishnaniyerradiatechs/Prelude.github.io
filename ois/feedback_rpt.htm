Quality Feedback Report
The Quality Feedback report lists detail information for feedback entries.
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Reports > Quality Feedback Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Feedback Number. The feedback number to view. Leave blank to view all feedback numbers that meet the remaining criteria.
•
Feedback Description. The description entered for the feedback. Leave blank to include all descriptions.
•
Entry Date range. The beginning and ending entry dates to include.
•
User Id. The user ID number for which you want to view feedback entries. Leave blank to include feedback entries for all user IDs.
•
Source Type. The feedback sources to include on the report. Leave blank to include all source types. Source types include:
-
C - Customer.
-
E - Employee.
-
V - Vendor.
-
M - Miscellaneous.
•
Product Number. The part number of the product being reported. Leave blank to include feedback entries for all products.
•
Status. The status of the feedback entry. Leave blank to include all feedback entries. Options include:
-
O - Open.
-
C - Closed.
-
I - In process.
-
U - Unassigned.
-
W - Waiting for response.
Column Descriptions
Column Heading
Description
Feedback
The feedback number from Quality Feedback Entry.
Description.
A description of the complaint.
Ent-Date
The date the feedback/complaint was entered.
Agnt
The quality agent assigned to the incident (refer to Creating Quality Agent Codes).
Description
The description of the quality agent.
Status
The status of the feedback entry. Options include:
•
O - Open.
•
C - Closed.
•
I - In process.
•
U - Unassigned.
•
W - Waiting for response.
Src
The source of the complaint:
•
C - Customer.
•
E - Employee.
•
V - Vendor.
•
M - Miscellaneous.
Source Description
The description of the source.
•
If Src is an employee, the employee number.
•
If Src is a customer, the customer number.
•
If Src is a vendor, enter the vendor number.
•
If Src is miscellaneous, this description of the source.
Notes
Extended notes entered in Quality Incident Entry regarding the complaint or problem.
