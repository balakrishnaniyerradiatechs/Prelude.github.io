Credit Authorization Report
The Credit Authorization report lists sales orders released from credit hold.
Capture Credit Authorization Information
Credit Authorization in File Update Parameters must be set to Y to capture credit authorization history information to appear on the Credit Authorization Report Refer “Credit Holds” in the Accounts Receivable documentation for more information.
After you run the report, you can choose to purge the product information so that it is not included the next time you run the report.
Menu Path: Order Processing > Reports > Miscellaneous Reports > Credit Authorization
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Customer. The customers to include in the report.
•
Release Date. The date the orders were released from credit hold.
Column Descriptions
Column Heading
Description
Customer
The number for the customer whose order was released.
Description
The customer description.
Date
The date the order was released from credit hold.
Time
The time of the day that the order was released from credit hold.
Order#
The order number that was released from credit hold.
Credit Bal
The amount of credit the customer has available after the orders are released.
