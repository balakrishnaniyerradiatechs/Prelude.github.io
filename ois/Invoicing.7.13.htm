Creating Payment-By Codes
A payment-by code indicates who will pay for an invoice and is required when you confirm a DD250 sales order (refer to Confirming a Release that Requires a DD250 Invoice). The payment-by code prints in block 12 on the DD250 invoice (refer to Printing Invoices Using the DD250 Form).
To create a Payment By code:
1
Select Order Processing > Maintenance > Misc Code Files > Payment By Maintenance to open the Payment By Maintenance screen.
2
For Payment By Number, enter the 1 to 6 alphanumeric code to use for sales orders set up to print DD250 invoices.
3
For Description, enter the name of the payment company or person.
4
Enter the address information for the payment company or person.
5
Press F2-Save to save the code.
