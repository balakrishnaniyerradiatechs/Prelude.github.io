Creating Valid Species Responses
If you set up standard instructions for prescriptions (refer to Creating Prescription Groups) you must first identify the species that the instructions refer to. For example, if you set up a species as CAT, you can set up standard instructions for all prescriptions for cats within one prescription group. You can also set up instructions by physician by species.
To set up valid responses for a species:
1
Select Order Processing > Maintenance > Customer Control Files > Physician/RX Maintenance > Species Maintenance to open the Valid Responses Maintenance screen.
2
The Validation Code already displays SPECIES, to allow you to enter all valid responses for species.
3
Under Response, enter valid responses that can be entered when a species is required for the physician maintenance programs (e.g., Cat, Dog, Bird).
4
Enter the Description of the response (e.g., Cat, Dog, Bird).
5
For Abbr, if applicable, you can enter an abbreviation of the response, so that whenever the abbreviation is entered, the full response appears on the screen.
6
Press F2-Save to save the valid response codes.
