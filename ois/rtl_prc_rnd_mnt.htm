Retail Price Rounding Maintenance screen
If you want to round the prices calculated by a retail price matrix, you can use Retail Price Rounding Maintenance to determine how the last cent is rounded.
Menu Path: Order Processing > Maintenance > Pricing Structures > Retail Price Matrix Maintenance > Retail Price Rounding Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Customer
Indicates the customer that the rounding information applies to:
•
The customer number if you want the rounding information to apply to one customer only.
•
DEF sets the rounding information for all customers.
From
Thru
The beginning price and ending values to define rounding. For example, for 1.00 - 10.00, define a set of rounding rules. For 10.01 - 20.00, define a new set of rounding rules.
N0
Determines how to round a retail price if the calculated value ends in 0:
•
Ux - To round the price up to x. For example, enter U1 to round the decimal up to the nearest 1.
•
Dx - To round the price down to x. For example, enter D0 to round the decimal down to the nearest 1.
•
X - To leave the price as calculated.
N1-N9
Same as above except for values ending in 1 - 9.
F2-Save
Saves the rounding information.
F4-Del
Deletes the rounding information.
