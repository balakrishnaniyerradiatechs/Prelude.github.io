Automatically Cancelling a Backorder
You can enter an auto cancellation date either in the sales order header or by line item so that a product is automatically canceled if it is not received by this date. Refer to the following for setting an auto cancellation date:
•
Setting an Auto Cancellation Date at the Header Level.
•
Setting an Auto Cancellation Date by Line Item.
You must create a lost sales reason code of CD (cancellation date) to use on sales orders when backorders are not received by a certain date (refer to Creating Lost Sales Reason Codes). The CAN code displays in the Order Log Inquiry when a sales order is automatically cancelled.
