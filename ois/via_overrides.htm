Via Overrides Screen
If you use VIA, you can use the Via Overrides screen to enter information necessary to track orders placed on your website. For each ship via code, you can fill in the tracking Url prefix and suffix from the 3rd-party carrier website (e.g., UPS, FedEx, DHL). For each carrier, you must access their website and follow their directions for tracking a package and finding a package number.
Menu Path: Order Processing > Maintenance > Customer Control Files > Ship Via Codes > Action Bar > Addt Optns > Via Overrides
Field and Function Key Descriptions
Field/Function Key
Description
Available On Web
Indicates if this ship via type is available on your website:
•
Y - The ship via type is available on the website.
•
N - The ship via type is not available on the website.
Tracking Url Prefix
The prefix to the website for the 3rd-party carrier to allow the customer to track VIA orders placed on your website (e.g., everything everything from the first character up to but NOT including the Package#).
Tracking Url Suffix
The suffix to the website for the 3rd-party carrier to allow the customer to track VIA orders placed on your website (e.g., everything after the Package# to the end of the url).
F2-Save
Saves the VIA information.
F5-Zip Code Maintenance
Opens Zip Code Range Maintenance screen. You can enter a shipping charge for a zip code range.
F10- Action
Not applicable.
