Freight Information Screen
The Freight Information screen allows you to enter freight terms, calculations, and pre-paid freight information for a customer.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F7-Order > F7-Frt Info
Field and Function Key Descriptions
Field/Function Key
Description
Freight Terms
The default freight terms code that indicates whether inbound and/or outbound freight is normally charged to this customer. You can override the freight terms in Sales Order Entry.
Freight codes are set in the Freight Terms Maintenance Screen and ensure that freight charges are passed to the customer when the order is confirmed in Release Confirmation Entry.
Target Ship By Days
The number of days to add to the order date on a sales order for this customer to calculate the shipping date.
Charge Freight
Indicates if freight is:
•
Y - Automatically calculated and charged to this customer based on the freight table in the Ship Via Maintenance Screen for the ship-via code assigned to the sales order. You may want to use this when freight is calculated based on the size of the order rather than the actual freight charged. Refer to Ship Via Family Code Maintenance Screen.
•
N - Not automatically charged to this customer. You can enter actual freight charges when the order is released.
Frt Calculation Method
The freight calculation method or additional freight requirements:
•
T - User-defined flat rate table.
•
S - Ship via rate table (refer to the freight table in the Ship Via Maintenance Screen).
•
M - Manifest system freight.
Frt Rate Multiplier
The multiplier to use with the Frt Calculation Method to calculate freight.
Addl Frt/Handling Chg
Indicates if and where additional freight and handling charges (calculated in Order Processing Parameters > F7-Add Frt/Handling) are added to the customer’s orders:
•
F - Add any additional charges to Outbound Freight on the Sales Order Completion screen.
•
H - Add any additional charges to Handling on the Sales Order Completion screen. Refer to Using the Sales Order Completion Screen for information on this flag.
•
N - Do not add additional charges to the sales order.
Additional charges can also be calculated by the Ship Via on the sales order. Refer to the Freight Surcharge Table Screen .
Charge/
backorders?
Determines if the additional freight/handling charge is added to backorder items on invoices following the first invoice cycle:
•
Y - Charge the additional freight/handling charge to the backordered items after invoicing.
•
N - Do not charge additional freight/handling charges to backordered items after invoicing.
Exclude Small Ord Amt Chk
Determines if a fee is added to the customer’s order if it does not meet the Small Order Limit for the ship via code (assigned in Ship Via Maintenance):
•
Y - The Small Order Fee is not added to the customer’s orders if the Small Order Limit is not met.
•
N - If an order total is less that the Small Order Limit for the Ship Via in the Ship Via Freight Info Screen, then the Small Order Fee is added to Inbound Freight on the Sales Order Completion screen.
Charge Frt/Hndlg Surcharge
Determines if additional freight or handling surcharges from the ship via Freight Surcharge Table Screen are automatically calculated and added to orders for this customer:
•
Y - Additional surcharges are added.
•
N - Additional surcharges are not added.
Freight Shipping Type
The freight shipping type of arrangement you have with this customer for orders shipped by UPS, FedEx, and RPS, if you use (PSS):
•
CBS - Consignee billing.
•
COL - Freight collect.
•
TPB - Third party billing.
Account#
The customer’s UPS account number, if applicable.
PSS Consolidation
Determines if releases are consolidated into one shipment for PSS:
•
Y - Releases are automatically set to Update/Hold in Release Confirmation Entry and accumulated for the customer, warehouse, and ship-to. You can send a batch of releases through PSS to determine the freight amount for the entire shipment.
•
N - Releases are not consolidated for PSS.
Freight Paid Limit
The minimum dollar amount the customer’s order must be to receive pre-paid freight. Any sales order over this value does not incur a freight charge. This value overrides the Prepaid Frt Minimum in Ship Via Maintenance in the Ship Via Freight Info Screen, but can be overridden in Customer Ship-To Maintenance.
Ship Via
The ship via code associated with the Freight Paid Limit for prepaid freight. If this field is left blank, the system checks the Prepaid Frt Minimum for all ship via codes with Prepaid Freight set to Y.
Review Date
A review date to check the Freight Paid Limit and Ship Via. This field is optional and is only used for custom reporting.
