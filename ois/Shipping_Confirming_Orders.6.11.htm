Printing Packing Lists for Non-Confirmed Releases
Warehouse Release Confirmation Entry allows you to print a packing list without updating or invoicing the release. It works the same as Release Confirmation Entry except that the release is not updated and you cannot enter substitutes. You may want to use this option if the customer requires a packing list but you are not ready to confirm the shipped material.
To print packing lists without confirming an order:
1
Select Order Processing > Optional Entry & Printing > Warehouse Release Confirmation Entry to open the Warehouse Release Confirmation Entry screen.
2
Follow the steps for Release Confirmation Entry (refer to Confirming Releases), except that when you press F2-Save to generate the packing list, you cannot update the release or set it up for invoicing.
3
If you continuously print packing lists (refer to Setting Packing List Maintenance Options), you can choose to print the packing list. Or, you can manually print the packing list (refer to Printing Packing Lists by Batch).
You must still use Release Confirmation Entry to confirm the release and set it up for invoicing.
Shipment Processing Entry
If you need to enter, confirm, and invoice a release all in one step, you can use Shipment Processing Entry (Order Processing > Option Entry & Printing > Shipment Processing Entry). The order is set up for invoicing and no pick ticket is generated. You may want to use this if the customer already has the material.
