Sales Order History Inquiry
The Sales Order History Inquiry lists invoiced sales orders for a customer.
Selecting Invoiced Sales Orders
To list invoiced sales orders:
1
Select Order Processing > Inquiries > Sales Order Inquiries > Sales Order History Inquiry to open the Sales Order History Inquiry screen.
2
Enter the Customer.
3
Indicate the Sort sequence to list the customer’s orders. Choose from the following to sort the list by:
•
None.
•
Order number.
•
Order date.
•
Invoice date.
•
Customer purchase order number.
•
Customer release number.
•
Invoice number.
•
Invoice amount.
•
Descending order number.
•
Descending order date.
•
Descending invoice date.
•
Descending customer purchase order number.
•
Descending customer release number.
•
Descending invoice number.
•
Descending invoice amount.
4
Indicate if you want to use any Filters to limit the list of orders. Choose from the following:
•
No. Lists all open sales orders for the customer.
•
Yes. Opens the Sales Order History Inquiry Filter Options screen to allow you to use the following selection criteria to limit the list:
-
Invoice date range.
-
Quote number.
-
Order number.
-
Cust Release number.
-
Purchasing contact.
-
Customer P/O number.
-
Job number.
•
Customer P/O number.
•
Customer job number.
•
Customer release number.
•
Quote.
•
Consolidated invoice.
Invoiced sales orders for the customer appear.
Field & Function Key Descriptions
Field/Function Key
Description
Order#
The original order number for this invoice. A dash (-) followed by a number indicates that the invoice was generated from a backorder.
N
An asterisk (*) indicates that notes exist in the order header.
Invoice#
The invoice number for the sales order.
Ord-Date
The date the sales order was entered.
Inv-Date
The date the sales order was invoiced.
Customer P/O
The customer’s purchase order number.
Release Number
The customer’s purchase order release number, if applicable.
Invoice Amount
The total amount of this invoice.
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the Order History Inquiry Display screen to show detail information for the highlighted order.
F6-Enter Ord#
Allows you to enter a sales order number to go directly to.
F7-Enter Inv#
Allows you to enter an invoice number to go directly to.
F8-Log
Opens the Order Log Inquiry which lists each transaction that affected the order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Returns to the previous page if more than one page of sales orders are included in the inquiry.
sF6-Next Group
Proceeds to the next page if more than one page of sales orders are included in the inquiry.
Action Bar
Output
Allows you to print, fax, email, or save (to an Excel spreadsheet), the order information.
Route Info
Displays shipping route history information, if applicable.
Releases
Opens the Release History Inquiry to display printing, picking, and shipping information for the highlighted order if releases exists.
