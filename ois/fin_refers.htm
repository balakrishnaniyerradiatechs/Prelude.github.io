Financial References Screen
The Financial Reference screen allows you to enter detail information for a company that is used as a financial reference for a customer. Financial References are created in Financial Reference Maintenance on the Accounts Receivable menu.
Menu Path: Order Processing > Maintenance > Customer Maintenance > F6-A/R > Action Bar > Cred Ref
Field and Function Key Descriptions
Field/Function Key
Description
Credit Review Complete Date
The date the credit review was completed.
Ref#
The financial reference number created in Financial Reference Maintenance that identifies the institution where financial references are sent. Enter M to create a new one-time financial reference.
Description
The description of the financial reference.
Requested
Determines how the request will be sent:
•
F - Fax.
•
M - Mail.
•
T - Telephone.
Req-Date
The date the financial reference was requested.
Sent
Indicates if the request was sent:
•
Y - Yes.
•
N - No.
Resp
Indicates if a response was received:
•
Y - Yes.
•
N - No.
Resp-Date
The date the response was received.
F2-Save
Saves the financial reference information.
F5-Fin Ref Detail
Opens the Financial Reference Detail screen to allow you to view or enter address and contact information for the reference.
F6-Fax Credit Application
Allows you to fax the credit application to the customer. Defaults to the fax number from the Customer Maintenance screen.
