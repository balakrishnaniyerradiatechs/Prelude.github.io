Quality Agent Maintenance screen
Each person that enters quality inspection complaints or corrective actions must have an agent code set up in Quality Agent Maintenance. An agent could be a customer, employee, or vendor. If the agent is an employee that already has a user ID, the existing information defaults into the agent code.
Menu Path: Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Quality Agent Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Agent Number
The name, code, or number to represent the agent.
User Id
The user ID if the agent is also an employee. The information from User Identification Maintenance defaults into the following fields. If the agent does not have a user ID, leave this field blank.
Description
The agent’s name.
Company Name
The agent’s company name.
Email Address
The agent’s email address.
Phone
The agent’s phone number.
Ext
The agent’s phone extension.
Fax
The agent’s fax number.
Job Type
The job type code (from Valid Responses Maintenance) that describes the agent’s position.
Manager?
Indicates if the agent is a manager:
Y - The agent is a manager and has the authority to close incidents.
N - The agent is not a manager.
F2-Save
Saves the quality agent information.
F4-Del
Deletes the quality agent information.
F5-Add Info
Opens the Additional Information screen to allow you to enter default pass and fail bins for this agent.
