Inspection Procedures screen
The Inspections Procedures screen allows you to enter the type of inspection procedures required for the product. Refer to the Quality Inspection Subsystem for more information.
This screen only appears if the Quality Inspection - AQL flag in Inventory Parameters is set to Y to activate the Quality Inspection Subsystem in the Order Processing Module.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > Qual Insp Required = Y
Field and Function Key Descriptions
Field/Function Key
Description
Proc#
The inspection procedure to use as the default for this product. Inspection procedures are set up in Quality Inspection Procedure Maintenance (Order Processing > Additional Subsystems > Quality Inspection Subsystem > Maintenance > Quality Inspection Procedure Maintenance).
Description
The description of the inspection procedure.
Version
The inspection version that is currently active.
Ver-date
The date this version went into effect.
Inspection Type
Indicates which type of inspections this procedure number represents:
•
C - Customer return.
•
P - PO receipts.
•
V - Vendor performed.
•
W - Work order manufacturing.
Instructions
Any instructions regarding the inspection that you want to appear for this product in Inspection Results Entry > F6-Instructions. Refer to Recording Inspection Results for more information.
F2-Save
Saves the inspection procedure information.
