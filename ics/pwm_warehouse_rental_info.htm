Warehouse Rental Information Screen
The Warehouse Rental Information screen allows you to enter rental rates for each type of time period (e.g., hours, days, weeks). This information overrides the rental information in Product Maintenance.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Warehouse Maintenance > Action Bar > Addtl Optns > Rental Information
Field & Function Key Descriptions
Field/Function Key
Description
Hours Shifted To Day
The number of hours before the hourly rental rate shifts to a daily rate. Only applies if the Rental Rate Method in Rental Option Parameters is A or B.
Days Shifted to Week
The number of days before the daily rental rate shifts to a weekly rate. Only applies if the Rental Rate Method in Rental Option Parameters is A or B.
Weeks Shifted To Month
The number of weeks before the weekly rate shift to the monthly rate. Only applies if the Rental Rate Method in Rental Option Parameters is A or B.
F2-Save
Saves the rental information.
