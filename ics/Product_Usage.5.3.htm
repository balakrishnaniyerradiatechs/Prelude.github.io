Maintaining Product Usage
The following screens within Product Usage Maintenance allow you to view and/or edit monthly usage information for a product for each warehouse.
•
Product Usage Maintenance Screen.
•
Usage Summary Screen.
•
Warehouse Hits Inquiry Screen.
•
Current Year Weekly Usage Screen.
Statistics Sequence
Reverse Statistics Sequence (Inventory Parameters > F5-Add Info) determines the order to display statistics screens:
•
Y - Usage and statistics screens appear in reverse order (December to January) for each calendar year.
•
N - Usage and statistics screens appear in regular order (January - December).
Usage History Listing
To obtain a listing product usage history information, select Inventory > Listings > Product Listings > Usage History.
