Warehouse Quantity Breaks Screen
The Warehouse Quantity Breaks screen allows you to enter quantity break prices for a product.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Warehouse Maintenance > F7-Price/Cost > F8-Prc Qty Brks
Field & Function Key Descriptions
Field/Function Key
Description
Qty Break Type
Indicates how quantity break prices are calculated:
•
$ - Price. Opens the Price Column Quantity Break Screen to allow you to calculate prices based on quantities or discounts off of a column.
•
* - Discount percent. Allows you to enter a percent discount for each quantity break.
•
T - Discount table. Allows you to enter a discount break table number to calculate column quantity breaks. Refer to Setting Up Discount Break Tables.
Quantity Discount Break Table Number (unlabeled field)
If T is entered for Qty Break Type, enter the discount table number set up in Discount Break Table Maintenance.
F2-Save
Saves the warehouse quantity break pricing information and closes the screen.
