Discount Break Table Maintenance screen
The Discount Break Table Maintenance screen allows you to establish break levels to determine discounts to generate prices and costs. Each price or cost is expressed as a set price or as a discount off of a column price. Once a table is set up, you can reference it in Product Maintenance, Product Warehouse Maintenance, Product Vendor Maintenance, Vendor Line Maintenance, or Price Matrix Maintenance to calculate prices and costs.
Menu Path: Inventory > Maintenance > Discount Break Table Maintenance
