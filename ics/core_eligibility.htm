Core Eligibility Inquiry
The Core Eligibility Inquiry lists pending cores for a customer so you can determine if the customer is eligible to return the core product.
Viewing Eligible Cores
To inquire on a customerâ€™s eligibility to return a core product:
1
Select Inventory > Inquiries > Core Inquiries > Core Eligibility Inquiry to open the Core Eligibility Inquiry screen.
2
Enter the Customer for which you want to view returned core products.
3
Enter the Product number.
4
Enter the Warehouse where the customer bought the product.
Core information for this customer/product number appear.
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer for which you want to inquire on eligibility for core products.
Product
Line 1: The exchange product number.
Line 2: The exchange product description.
Warehouse
The warehouse number. The warehouse description displays.
Core Product
The core product number associated with the exchange product number.
Month (unlabeled field)
The month for the corresponding transactions.
Qty Shp
The quantity of the core product shipped in that month.
Qty Credit
The quantity credited for the core product in that month.
Qty Avail
The quantity of the core product available to return in that month.
OK
Closes the inquiry screen.
