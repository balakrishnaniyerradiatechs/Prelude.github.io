Bin Products Onhand Report
The Bin Products Onhand report displays on-hand quantities for each product in a bin. You can use this report to identify items that should not be in a bin. For example, if you schedule the report to print at the end of the day, you can review all items left on the receiving dock, in an inspection bin, or in any other bin that should be empty by the end of the day.
Menu Path: Inventory > Reports > Miscellaneous Reports > Bin Products Onhand Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse containing the bin products.
•
Bin Number Range. The beginning and ending bins to view. Leave blank to view all bins for the warehouse.
•
Bin Type. The bin location type to include. Leave blank to include all bin types or choose from the following:
-
B - Bulk.
-
C - Crib.
-
D - Receiving dock.
-
I - Inspection.
-
K - Bulk un-packaged.
-
L - Work-in-process line stock.
-
O - On-the-water.
-
P - Pick.
-
R - Replenishment repackaging.
-
S - Staging.
-
T - Retail.
-
U - Unallocated.
-
W - Work-in-process.
•
Storage Type. Indicates how the products in this bin are stored. Leave blank to include all storage types or choose from the following:
-
P - Pallet racks.
-
R - Flow racks.
-
C - Carousel.
-
F - Floor stock.
-
S - Shelf.
-
M - Mobile.
-
W - Work center.
-
T - Truck.
-
O - Outplant.
-
G - Pre-stage.
•
Product List Sort (P/Null). Indicates how the products are sorted on the report:
-
P - Sorts the products in each bin in alphanumeric order.
-
Blank - Sorts the products in the order by the date they were added to the bin.
Column Descriptions
Column Heading
Description
Bin Location
The bin number.
Product Number
The products in the corresponding bin.
Description
The product description.
VPN
The vendor part number, if applicable.
Bin OH
The quantity of each of the products in the bin.
Bin Cmt
The quantity of the product in the bin that is committed to sales orders, work orders, transfers, and vendor claims.
Bin Avl
The quantity of the product in the bin that is available to sell.
U/M
The stocking unit of measure for the product.
Lots
The lot number assigned to the product in the bin, if applicable.
Cls
The inventory class for the product. Refer to Classifying Products.
