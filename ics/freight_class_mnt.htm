Freight Class Maintenance screen
The Freight Class Maintenance screen allows you to enter freight classes used to calculate weight when a sales order release is confirmed as shipped. Once you set up freight classes, you can assign them to product lines in Product Line Maintenance and to products in Product Maintenance. The freight class in Product Maintenance overrides the freight class in Product Line Maintenance.
Menu Path: Inventory > Maintenance > Product Control Files > Freight > Freight Class
Field and Function Key Descriptions
Field/Function Key
Description
Freight Class
The freight class to assign to products to calculate weight in Release Confirmation Entry.
Description
The description of the freight class.
F2-Save
Saves the freight class information.
F4-Del
Deletes the freight class.
