Work Order Commitments screen
The Work Order Commitments screen displays the work orders to which this product is committed.
Menu Path: Inventory > Inquiry > Inventory Inquiries > Work Orders (under Commitments) > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
W/O#
The work order number.
N
An asterisk (*) indicates that notes exist for the product.
Req-Date
The date the work order is required.
Status
The status of the work order:
•
Entered.
•
Released.
•
Picked in RF picking, but not complete. Only applicable with the warehousing module.
Bill Number
The bill of material number.
F
Indicates if the work order is a future:
•
F - A future work order.
•
Blank - Not a future work order.
Qty-Opn
The quantity of the product committed to the work order.
U/M
The unit of measure of the product as entered on the work order.
Qty-Pck
The quantity of the product that is picked but not yet issued.
Qty-B/O
The quantity of the product backordered for this work order.
Qty-Iss
The quantity of the product that is issued on a work order.
S
The supply code that appears if the product is on backorder. The code varies depending on the product’s replenishment path and product settings. Possible supply codes include:
•
B - Purchase order.
•
E - Emergency purchase order.
•
T - Transfer.
•
R - Emergency transfer
•
W - Work order.
•
D - Direct ship purchase order.
Linkage S
The document number of the source of the product, if it is linked to another transaction. The prefix indicates the document type:
•
P - The product was purchased on this purchase order number.
•
T - The product was ordered on this transfer number.
Linkage D
The document number for the destination of the product, if it is linked to another transaction. The prefix indicates the document type:
•
S - When the product arrives it will be assigned to this sales order number.
•
T - When the product arrives it will be shipped out on this transfer number.
F2-Finish
Closes the screen and returns to the Inventory Inquiry screen.
F5-Review W/O
Opens the Work Order Display screen to display detail information on the highlighted work order.
F7-Log
Opens the Work Order Log Inquiry to display every transaction that has occurred for the highlighted work order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
