Copy Warehouse Utility screen
The Copy Warehouse Utility feature copies product/warehouse information from one warehouse to another. If you access this screen from Product Warehouse Maintenance, just the current product is copied to the new warehouses. If you access this screen within Copy Warehouse Utility, all product are created in the new warehouse. Existing product records are not overlaid. Refer to Copying Products for more information.
Field & Function Key Descriptions
Field/Function Key
Description
Copy From Whse
The warehouse number of the product warehouse record(s) being copied.
Copy To Whse
The new warehouse numbers to copy the product warehouse information to.
F2-Save
Adds the new product warehouse record(s) to the selected warehouses.
