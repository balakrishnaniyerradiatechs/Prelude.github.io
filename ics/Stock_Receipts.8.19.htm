Entering UPCs When Receiving Stock
If UPCs are required in Stock Receipts Entry and a UPC is not set up as an alternate part number, you must create the UPC before you can complete the receiver.
Requiring UPCs in Stock Receipts Entry
UPC Required (Option Parameters > Inventory Parameters > F5-Add Info) must be set to Y to indicate that a UPC code is required in Stock Receipts Entry. If you receive a product that does not have a UPC, you must enter the UPC code before you can complete the receiver.
To create UPCs in Stock Receipts Entry:
1
Refer to Receiving Stock for detailed information on receiving stock. When you press F2-Save to complete the receiver, the Stock Receipt UPC Code Entry screen appears. All products on the receiver, without UPCs set up as an alternate part number, display under Product Number.
2
Enter the UPC Code for each product. You can either enter the UPC or enter N to let the system assign the next available UPC number. All products must have a UPC before you can complete the receiver.
3
Press F2-Save to complete the receiver.
The UPC is now stored as an alternate part number. The next time you receive the item, you are not required to enter a UPC.
To view the UPC code for a product, select Inventory > Inquiries > Product Inquiries > Product Alternates Inquiry.
