Warranty Replacement Entry screen
The Warranty Replacement Entry screen allows you to reconcile the items returned on a vendor claim with the replacement items sent from the vendor.
Refer to Entering Warranty Replacements for more information.
Menu Path: Inventory > Warranty Replacements > Warranty Replacement Entry
Field and Function Key Descriptions
Field/Function Key
Description
Entry Number
The warranty replacement number.
Vendor
The vendor the items were returned to on the vendor claim.
Vendor Invoice#
The vendor’s invoice/credit memo number for the replaced items, if applicable.
Received By
The user ID of the person entering the replacement.
Replacement Date
The date the warranty replacements were received.
Vendor Claim Number
The number for the vendor claim on which the products were returned to the vendor.
Value - Replaced
The value of the products sent to the vendor for replacement.
Value - Replacing
The value of the products the vendor sent to replace the returned products.
Discrepancy
The difference in value between what was sent to the vendor and what the vendor replaced.
F2-Save
Saves the warranty replacement information and gives you the following choices:
•
Process - Adds the replacement products to available inventory and updates the vendor claim as replaced.
•
No update - Suspends the transaction. Inventory is not updated. You can view suspended warranty entries in the Warranty Replacement Inquiry.
•
Return - Returns to the entry screen.
F4-Del
Deletes the warranty replacement entry.
F5-Replaced Products
Opens the Warranty Replaced Products Entry screen to allow you to enter the products returned to the vendor on a vendor claim.
F6-Replacing Products
Opens the Warranty Replacing Products Entry screen to allow you to enter the products replaced by the vendor for the vendor claim.
