Vendor Verification screen
The Vendor Verification screen appears when you enter a Vendor Number in Purchase Order Entry or Vendor Claim Entry. The vendor’s name and address display. You can choose to select the vendor or to go back to the Vendor Number field to pick another vendor.
Field and Function Key Descriptions
Field/Function Key
Description
Vend Cust#
Displays the customer number if this vendor is also a customer.
Is This the Correct Vendor (Y/N)
Indicates if this is the correct vendor.
•
Y - If the name and address displayed are correct. Opens the Header screen.
•
N - If the name and address displayed are not for the correct vendor. Returns to the Vendor Number field to allow you to try another number.
