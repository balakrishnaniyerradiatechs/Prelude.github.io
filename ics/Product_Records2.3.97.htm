Setting Up Customer Ration Types
Customer ration types determine the quantity a customer is allowed to buy in a designated time period. You can set up customer ration types by product or by product line.
To set up customer ration types:
1
Select Inventory > Maintenance > Product Control Files > Rationing > Rationing Maintenance to open the Ration Maintenance screen.
2
For Ration Type, indicate how you want to assign rationing:
•
P - By product.
•
L - By product line.
3
For Vendor, enter the primary vendor number that the ration quantity applies to. This vendor number must be established in the Product Vendor Maintenance Screen as the primary vendor.
4
For Product/Pline:
•
If Ration Type is set to P, enter the product number.
•
If Ration Type is set to L, enter the product line.
5
Enter the Start Date, which is the date to begin rationing for this set of criteria.
6
Enter the End Date, which is the date to end rationing for this set of criteria.
7
For Ration Days, enter the number of days the rationing cycle is in effect. The customer can only purchase the ration quantity during this cycle. Once the number of days ends, the next ration cycle starts and the ration quantity is available again. The number of ration days keeps cycling until the End Date is reached.
8
For Cust Ration Type, enter the type codes (from Ration Type Code Maintenance) to correspond to each Ration Quantity. These types are used in the customer’s Ration Type field in Customer Maintenance.
9
For Ration Qty, enter the ration quantity associated with each Customer Ration Type. For example, type A could be setup with a ration quantity of 5000, type B with a quantity of 7500, etc.
10
For Comments, enter a comment to print on the sales order after the rationed product.
Customer Maintenance
Ration Type in Customer Maintenance > F7-Order activates rationing for a customer. Enter the ration type or enter O for override to exclude this customer from sales order rationing (normally a temporary setting to allow you to enter a sales order).
Customer Rationing Inquiry
To display sales order rationing information for a customer/product number, select Order Processing > Inquiries > Customer Rationing Inquiry.
