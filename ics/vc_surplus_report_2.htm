Vendor Claim Surplus Worksheet screen
The Vendor Claims Printing screen allows you to generate a vendor claim report for all surplus items for a vendor. You can run the report by warehouse, vendor, and inventory class to limit the items to review. The report also recommends other warehouses that may be candidates for the surplus inventory so that you can determine if you want to transfer the material rather than return it to the vendor.
Menu Path: Inventory > Vendor Claims > Vendor Claim Surplus Report and Update
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse that you want to check for surplus products.
Vendor
The vendor whose surplus products you want to review.
Class Greater Than
The class that is 1 number lower than the classes you want to include in the report. The report will only print products with a class greater than this number so that you only have to review less active products. For example: Inventory classes range from 1 to 13, with class 1 being the most active and class 13 being dead stock. To only show items in classes 7-13, enter a 6. Refer to Classifying Products for information on inventory class.
Split Claims By Line
Indicate if you want to create a separate claim for each vendor line, if the vendor is set up to create a new purchase order for each vendor line:
•
Y - A separate claim generates for each vendor line.
•
N - One claim generates for all vendor lines.
