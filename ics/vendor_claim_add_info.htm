Vendor Claim Additional Information screen
The Vendor Claim Additional Information screen allows you to enter the vendor’s order number and type of currency.
Menu Path: Inventory > Vendor Claims > Vendor Claim Entry > F5-Hdr Scr > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Language Indicator
The default language for documents sent to the vendor (e.g., purchase orders) if using Optio:
•
1 - English.
•
2 - French.
•
3 - Spanish.
•
4 - Portuguese.
Currency Ind.
The type of currency used by this vendor on this claim.
Rate
The currency exchange rate, if applicable. Currency exchange rates are set up in Currency Maintenance on the Systems menu. Refer to “Using Multiple Currencies in the General Ledger Documentation for more information.
F2-Save
Saves the vendor claim information.
