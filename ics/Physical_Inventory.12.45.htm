Taking Physical Inventory Using RF Devices
Radio Frequency guns allow you to enter physical inventory counts directly into the system without printing count sheets or tags.
Use the following steps to make sure that all products are counted accurately:
1
Merge products (refer to Merging Catalog Products into Inventory Count Lists).
2
Build lists (refer to Building the Physical Inventory Lists).
3
Display lists (refer to Displaying Physical Inventory Lists).
4
Save on-hand quantities (refer to Saving Current Onhand Quantities).
5
Enter counts (refer to Entering Counts on RF Devices).
6
Run the Physical Count Report (refer to the Physical Inventory Radio Frequency Count Report).
7
Consolidate bins (refer to Consolidating Bin Counts).
8
Identify products with a large deviation in value (refer to the Physical Inventory Deviation Exception Report).
9
Verify count and value changes (refer to the Physical Inventory Shrinkage Report).
10
Identify shrinkage values for counted products (refer to the Physical Inventory Shrinkage Valuation Report).
11
Verify uncounted products (refer to Verifying Products With no Count).
12
Update Inventory (refer to Updating Physical Inventory Counts).
13
Print a report for the total inventory value (refer to the Physical Inventory Valuation Report).
14
Print count sheets for serialized products (refer to the Physical Inventory Serial Number Report).
15
Clear count files (refer to Clearing the RF Count File).
