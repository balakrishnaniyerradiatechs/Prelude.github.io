Transfer Fill Rate Detail Report
The Transfer Fill Rate Detail report displays the percentage of transfer line items that are 100% filled before shipping from a warehouse. The report includes the detail shipping information for each line item on the transfer.
Fill rate for a transfer is tracked from the time the first pick ticket prints to the time the first release ships. If a transfer number has more than one release in the same day, the fill rate is recalculated for that transfer. Lines added to the transfer do not update the fill rate. Lines deleted from the transfer in Transfer Shipment Entry are considered zero percent shipped.
Tracking Transfer Fill Rate
Transfer Fill Rate? in File Update Parameters must be set to Y to track fill rate information.
After the report runs, a dialog box appears to allow you to purge the fill rate information so that it does not appear the next time you run the report.
Menu Path: Inventory > Reports > Stock Movements Reports > Transfer Fill Rate Detail Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
From Warehouse. The warehouse that shipped the material.
•
Beginning and Ending Ship Date. The ship date range to include.
Column Descriptions
Column Heading
Description
Transfer
The transfer number that shipped.
To
The receiving warehouse.
Shp Date
The date the transfer shipped.
Product
Line 1: The product number that shipped.
Line 2: The serial or lot number of the product, if applicable.
Description
Line 1: The product description.
Line 2: The second line of description for the product, if applicable.
Ordered
The quantity of the item ordered for the receiving warehouse.
Shipped
The quantity of the product shipped to the receiving warehouse.
#Lines
The number of lines on the transfer that contain the product number. Only the total amount for the transfer calculates fill rate.
#Compl
The number of lines shipped complete. Only the total amount for the transfer is the valid number used to calculate fill rate.
Perc
The fill rate per line. If a line is 100% shipped, then the fill rate for that line is 100%. If the line is less than 100% shipped then the fill rate for that line is zero.
