Displaying Physical Inventory Lists
After you select the products for the inventory lists, you can view the lists created for each warehouse. You can also view the selection criteria used to create each list (refer to Building the Physical Inventory Lists).
To display the inventory lists:
1
Select Inventory > Physical Inventory > Inventory Sheet Procedure > Display Physical Lists to open the Physical Inventory List Display screen.
2
Enter the Warehouse.
3
Press ENTER to display the physical inventory lists created for the warehouse.
Deleting Inventory Lists
To delete a list, enter a back-slash (\) over the Physical Listname and press ENTER. To delete all of the lists displayed for the warehouse, press F4-Delete All Lists.
Field & Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for the created lists.
Physical Listname
The physical inventory lists created for the warehouse.
Item Ct
The number of items included on the corresponding list.
Upd Cnt
Indicates the number of items on the list with a count already entered (refer to Entering Physical Inventory Counts). Since you can use a list multiple times, this field changes each time the list is updated (refer to Updating Physical Inventory Counts).
Upd?
Indicates if the count was updated into on-hand inventory by the Physical Update Procedure (refer to Updating Physical Inventory Counts):
•
U - On-hand was updated by the Physical Update Procedure.
•
N - On-hand is not yet updated by the Physical Update Procedure.
Criteria
The selection criteria used to create the list.
List Created
The date and time the list was created.
On-hands Saved
The date and time the current on-hand quantities were saved. Refer to Saving Current On-Hand Quantities.
F2-Save
Saves changes if any lists were deleted.
F4-Delete All Lists
Deletes all of the physical inventory lists for the warehouse.
F5-Review Criteria
Displays the selection criteria and sort method used to create the list.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
