Transfer Fill Rate Summary Report
The Transfer Fill Rate Summary report displays the percentage of line items on a transfer that are 100% filled before shipping from a warehouse. Fill rate for a transfer is tracked from the time the first pick ticket prints to the time the first release ships. If a transfer has more than one release in the same day, the fill rate is recalculated for that transfer. Lines added to the transfer do not update the fill rate. Lines deleted from the transfer in Transfer Shipment Entry are considered zero percent shipped.
Tracking Transfer Fill Rate
Keep Transfer Fill Rate? in File Update Parameters must be set to Y to track fill rate information.
After the report runs, a dialog box appears to allow you to purge the fill rate information so that it does not appear the next time you run the report.
Menu Path: Inventory > Reports > Stock Movements Reports > Transfer Fill Rate Summary Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
From Warehouse. The warehouse that shipped the transfers.
•
Beginning and Ending Ship Date. The ship date range to include.
Column Descriptions
Column Heading
Description
Transfer
The transfer number that shipped.
Shp Date
The date the transfer shipped.
To
The receiving warehouse.
# of Lines
The number of lines on the order.
# Complete
The number of lines shipped complete.
Percent
The percentage of lines items that shipped 100% complete (#Complete / #Lines).
