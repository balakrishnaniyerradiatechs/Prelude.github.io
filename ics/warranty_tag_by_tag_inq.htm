Warranty Tag by Tag Inquiry
The Warranty Tag by Tag Inquiry allows you to view detail product, vendor, and claim information for a tag number.
Viewing Warranty Tags by Tag Number
To view warranty tag information:
1
Select Inventory > Inquiries > Warranty Tag Inquiries > Warranty Tag By Tag to open the Warranty Tag By Tag Inquiry.
2
Enter the Tag Number.
Detail information for the tag number displays.
Field & Function Key Descriptions
Field/Function Key
Description
Tag Number
The warranty tag number. Refer to Moving Products into Warranty Inventory for information on assigning a tag number.
Product
The product number on the warranty tag.
Description
The product description.
Whse
The warehouse associated with the tag.
Vendor
The vendor number for the tag.
Customer
The customer number for the tag if the warranty tag was entered on a credit memo. Refer to Creating Warranty Tags in Credit Memo Entry.
Document#
The document number where the tag number was created. The prefix indicates how the tag number was assigned:
•
R - The tag number was assigned in Stock Receipts Entry. Refer to Creating Warranty Tags in Stock Receipts Entry.
•
S - The tag number was assigned on a credit memo to a customer. Refer to Creating Warranty Tags in Credit Memo Entry.
•
A - The tag number was assigned in Inventory Adjustment Entry. Refer to Creating Warranty Tags in Inventory Adjustment Entry.
Doc-Date
The date the warranty tag was assigned.
Claim#
The vendor claim number if the tag is on a vendor claim.
Clm-Date
The date the vendor claim was issued, if applicable.
App
Indicates if the claim was reconciled:
•
Y - The vendor claim was reconciled.
•
Null - The vendor claim is not reconciled.
Vendor RMA#
The vendor’s return material authorization number.
F2-Finish
Closes the inquiry screen.
F5-Review Tag
Opens the Warranty Tag Inquiry Screen to allow you to view detail warranty information for a product on the warranty tag.
F6-Enter Prd#
Allows you to enter a product number to go directly to.
F9-Goto Line
Allows you to enter a line number, reference number, or page number to go directly to.
