Cycle Count Entry 1st Level screen
After the first round of counts, you can use the Cycle Count Entry 1st Level screen to enter the counts from the count sheets. If you enter any quantities that differ from the previous on-hand quantity by more than the tolerance range defined in Cost Tolerance Maintenance, the item goes into the second level queue for a recount.
Refer to Entering First Level Cycle Counts for more information.
Menu Path: Inventory > Physical Inventory > Cycle Counting Procedure > Paper Site Cycle Count Entry (1st Level)
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number.
Counter Id
The user ID of the person who counted the products. The user ID must be set up for cycle counting in User Identification Maintenance > F5-Administrative.
# Of Bins
The number of bins assigned to the counter.
Zone
The zone assigned to the counter.
Current Bin
The bin where the products exist.
Product Number
The product number counted.
Lot Number
The lot number assigned to the product, if applicable.
Quantity
The quantity counted. If the count differs from the current on-hand quantity by more than the defined cost tolerance percent, you can re-enter the count or accept the quantity.
F2-Update Bin
Updates the count for each product in the bin for which the count quantity is within the defined tolerance levels.
F6-Exit
Closes the screen.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
