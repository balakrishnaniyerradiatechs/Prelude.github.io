Royalty Commission Information Screen
The Royalty Commission Information screen allows you to calculate royalty commissions to pay to a salesrep when they sell a product.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > F5-Add Info > F6-Royalty
Field & Function Key Descriptions
Field/Function Key
Description
Slsm
The number for the sales representative who will receive the royalty commission for this product. This sales representative must be set up with their Royalty Eligibility flag set to Y.
Description
The description of the sales representative.
Mtd
The method used to calculate royalty:
•
$ - By dollar amount.
•
Q - By quantity.
Perc
The percent of the price to use to calculate the royalty fee.
Guarantee
The annual guaranteed amount for the sales representative (e.g., a set rate is guaranteed no matter how much is sold).
Renewal
The renewal date for the royalty commission.
QLimit
The quantity limit for the royalty commission.
Comm Limit
The maximum amount of commission that can be earned for this product.
F2-Save
Saves the royalty commission information.
