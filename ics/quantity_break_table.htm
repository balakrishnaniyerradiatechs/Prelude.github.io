Quantity Break Table Screen
The Quantity Break Table Screen allows you to enter prices for a product based on quantity breaks.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F7- Pricing > F8-Prd QtyBrk
Field & Function Key Descriptions
Field/Function Key
Description
Qty Break Type
Determines how prices are calculated:
•
$ - Opens the Quantity Break By Dollar Amount Screen to allow you to enter prices for each quantity level. You can enter a set price or calculate the price as a percentage off of a basis field.
•
* - Calculates prices as a percentage off of a default price.
•
T - Uses a quantity discount table established in Discount Break Table Maintenance. Refer to Setting Up Discount Break Tables.
Quantity Discount Table (unlabeled)
If you chose T for the Column Qty Break Type, enter the table number established in Quantity Discount Break Table Maintenance.
Break Level Type
Indicates how break levels are determined.
If you entered $ for Column Qty Break Type, the Price Column Quantity Break Screen opens to allow you to enter break levels:
•
$ - By dollar amount.
•
M - By unit of measure.
•
U - By quantity.
If you entered * for Column Qty Break Type, you can enter the quantity and the percent to determine the break levels:
•
$ - By dollar amount.
•
M - By unit of measure.
•
U - By quantity.
F2-Save
Saves the price column quantity break information.
