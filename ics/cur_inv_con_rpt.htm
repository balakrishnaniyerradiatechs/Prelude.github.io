Current Consigned (Prod Type) Value Report
The Consigned Inventory Valuation Report by Product displays the value of all vendor consigned products (Stock Status Code = C) in inventory for each warehouse. Consigned inventory refers to products in stock that you do not pay for until they are sold.
Menu Path: Inventory > Reports > Inventory Valuation Reports > Current Consigned (Prod Type) Value Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse containing the consigned inventory.
Column Descriptions
Column Heading
Description
Product Number/ Serial Number
Line 1: The consigned product number.
Line 2: The serial number, if applicable.
Description
The product description.
Whse
The warehouse containing the consigned inventory.
On-Hand
The quantity of the consigned product on hand.
U/M
The product’s unit of measure.
Ave-Cost
The average cost of the product.
Val@Cost
The extended value of the consigned inventory (Ave-Cost x On-Hand).
