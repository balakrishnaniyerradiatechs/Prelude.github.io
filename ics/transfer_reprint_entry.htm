Transfer Reprint Entry screen
The Transfer Reprint Entry screen allows you to reprint a transfer picking ticket.
Menu Path: Inventory > Stock Movements > Transfer Pick Ticket Reprinting
Field and Function Key Descriptions
Field/Function Key
Description
From Warehouse
The shipping warehouse for the transfer pick ticket.
Transfer Release #
The transfer release number to reprint.
Printer Form Queue
The printer to print the transfer pick ticket.
