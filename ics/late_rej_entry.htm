Late Rejection Entry screen
The Late Rejection Entry screen allows you to mark items as rejected if you track vendor shipping performance. The Vendor On-Time Receipts/Quality Report shows the on-time shipping performance for a vendor, including late and rejected shipments.
If the stock receipt is late, the on-time percentage is affected. If any stock is rejected, the quality percentage is affected.
Menu Path: Inventory > Processes > Late Rejection Entry
Field and Function Key Descriptions
Field/Function Key
Description
Rejection#
The system generated number that references the rejection.
Product
The product number marked for rejection.
Vendor
The number for the vendor that shipped the material.
P/O Number
The purchase order receiver number for the original receipt of this product.
Receive Location
The warehouse that received the product.
Date Received
The date the product was received in the warehouse.
Received By
The user ID of the person that received the material.
Reject Date
The date the product was rejected.
Rejection Reason
The reason code to indicate why the item is rejected (refer to Setting Up Rejection Claim Reason Codes).
Rejection Qty
The quantity rejected.
Reject Notes
Any additional information explaining why the product was rejected. These notes appear on the Vendor On-Time Receipts/Quality Report.
F2-Save
Saves the rejection information.
F4-Del
Deletes the rejection number.
