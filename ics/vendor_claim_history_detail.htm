Vendor Claim History Detail screen
The Vendor Claim History Detail screen displays detailed information for the products on an updated vendor claim.
Menu Path: Inventory > Inquiries > Vendor Claim Inquiries > Vendor Claim History Inquiry > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Clm
The vendor claim number. An asterisk (*) in front of the claim number indicates that notes exist in the header of this vendor claim.
Vendor RMA#
The vendor’s return material authorization number.
Vendor
The number and description of the vendor receiving the returned merchandise.
Product
The product returned to the vendor.
Description
The product description.
N
An asterisk (*) indicates if notes were entered for the highlighted item on the vendor claim.
Whse
The warehouse that shipped the products on the vendor claim.
Del-Date
The delivery date from the Vendor Claim Detail Information screen.
Qty-Ord
The quantity requested to return to the vendor.
Qty-Opn
The quantity returned to the vendor.
U/M
The unit of measure of the product as entered on the vendor claim.
Cost
The vendor cost of the product as entered on the claim.
Disc%
The vendor’s discount, as entered on the claim.
Linkage
Not applicable for vendor claims.
RMA Number
The vendor’s return material authorization number if the product was returned on a warranty tag. Refer to Moving Products into Warranty Inventory.
Vendor Part#
The vendor’s part number, if applicable.
F2-Finish
Closes the screen.
F5-Hdr Scr
Opens the Vendor Claim Header Display screen to allow you to view shipping information for the vendor claim.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Examine
Displays detail shipping and receiving information on the vendor claim.
Notes
Displays header or line item notes for the vendor claim.
Doc Func
Displays General Ledger postings.
SSCI
Displays the serial shipping container information to identify the contents of the vendor claim, if applicable.
