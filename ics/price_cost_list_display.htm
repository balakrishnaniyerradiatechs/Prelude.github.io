Displaying Price/Cost Lists
After you create the price/cost lists to update (refer to Building Price/Cost Lists), you can display the available lists for each warehouse. This step is not required but can help to find list names when you are ready to run the Price/Cost Pre-Save report (refer to Price/Cost Pre-Save Report) or to enter the price/cost updates (refer to Updating Prices/Costs).
To display the price/cost lists:
1
Select Inventory > Processes > Price/Cost Update Procedure > Price Cost Changes by Lists > Display Lists to open the Price Cost List Display screen.
2
Enter the Warehouse for which you want to display lists.
The price/cost update lists created for the selected warehouse display.
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for which you want to display lists. The warehouse description displays.
Physical Listname
The name of the list created in Building Price/Cost Lists.
Item Ct
The number of items on the list.
Upd?
Indicates if the list is already updated with new prices/cost values. Refer to Updating Prices/Costs:
•
Y - The price/cost list is updated.
•
N - The price/cost list is not yet updated.
Criteria
The selection criteria used to build the list.
Type
Indicates if the list was created from inventory items or catalog items:
•
I - Created using inventory items.
•
C - Created using catalog items.
F2-Save
Clears the screen.
F4-Delete All Lists
Deletes all the lists displayed for the warehouse.
F9-Goto Line
Allows you to enter a list name for direct access.
