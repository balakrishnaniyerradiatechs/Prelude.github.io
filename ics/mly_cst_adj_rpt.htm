Monthly Cost Adjustment Journal
The Monthly Cost Adjustment Journal displays all cost adjustments entered for a warehouse for a month or fiscal period.
Menu Path: Inventory > Reports > Inventory Adjustment Reports > Monthly Cost Adjustment Journal
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where inventory cost was adjusted.
•
Period: The month or fiscal period when the cost was adjusted.
Column Descriptions
Column Heading
Description
Cst Adj#
The adjustment number assigned to the transaction. Refer to Adjusting Inventory Cost.
Ent-Date
The date of the cost adjustment.
User
The user identification code of the person that made the adjustment.
Product
Line 1: The product number.
Line 2: The product description.
Line 3: The second line of description for the product, or the lot number information, if applicable.
Old Cost
The cost of the product before the cost adjustment.
New Cost
The adjusted cost of the product.
Onhand
The quantity on-hand in the warehouse.
Ext. Amount
The extended amount of the cost adjustment (New Cost - Old Cost ) x Onhand.
A positive number indicates an increase in cost. A negative number indicates a decrease in cost.
Cost Adjustment Reason
The cost adjustment reason code and description. Refer to Establishing Cost Adjustment Reason Codes.
