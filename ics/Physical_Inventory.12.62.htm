Physical Inventory Shrinkage Report
The Physical Inventory Shrinkage Report displays all items on a list. The report includes the saved on-hand quantity, the new physical count, and the difference between the two. This report is similar to the Physical Inventory Deviation report except that it displays all items, not just the ones with a count deviation. It also includes lot numbers.
You must run this report before you update inventory.
Menu Path: Inventory > Physical Inventory > Radio Frequency Procedure > Shrinkage Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse(s) where the inventory was counted.
•
Physical List Name. The list name to include on the report. You can leave this blank to include all the lists for a warehouse.
Column Descriptions
Column Heading
Description
Product Number
The product numbers from the count sheets.
Description / Lot Number
The product description and the lot number, if applicable.
Cost
The unit cost of the item.
BookOH
The saved on-hand quantity of the item before the update.
Book$Value
The extended dollar amount of the total saved on-hand quantity.
Count
The quantity entered from the physical inventory count.
Phys$Value
The extended dollar value of the total quantity counted.
Dev
The difference between the saved on-hand quantity and the counted quantity.
Dev$Value
The extended dollar amount of the total deviation quantity (Dev x Cost).
