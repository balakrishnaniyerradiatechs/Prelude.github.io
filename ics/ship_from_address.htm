Ship-From Address screen
The Ship-From Address screen in the Vendor Claim header allows you to change the ship-from address if the vendor claim is shipping from someplace other than your warehouse address.
Menu Path: Inventory > Vendor Claims > Vendor Claim Entry > F5-Hdr Scr > F7-Ship-From
Field and Function Key Descriptions
Field/Function Key
Description
Ship-From Name
The description of where the vendor claim is shipping from.
Address Line One
The first line of the ship-from address.
Address Line Two
The second line of the ship-from address.
Address Line Three
The third line of the ship-from address.
City
The city of the ship-from address.
State
The state of the ship-from address.
Zip
The zip code of the ship-from address.
F2-Save
Saves the ship-from address information on the vendor claim.
