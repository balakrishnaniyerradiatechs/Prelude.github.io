Setting Up Configuration Codes
Before you can configure child products from a base product, you must set up configuration codes to identify all of the possible options available for the base product. You can use these same configuration codes for other base products with the same Options.
To set up a configuration code:
1
Select Inventory > Maintenance > Misc Code Files > Configuration Code Maintenance to open the Configuration Code Maintenance screen.
2
Enter the Configuration Code to create or edit (e.g., color, size, length).
3
Enter a Description of the code.
4
For Options, enter all of the possible variations available for this code. For example: if the Configuration Code is color, enter all of the product colors that are available for sale.
5
For Abbr, enter an abbreviation for the option to use to create a child product. For example, if the Configuration Code is for color and the Option is red, enter R to designate the color red. If the Configuration Code is for size and the Option is Extra Large, enter XL to designate an extra large size.
6
For Desc Abbr, enter the abbreviation to use to create the child product description. For example:
-
Base product: TS2309
-
Base product description: T-Shirt
-
Child product: TS2309-XL-R
-
Child product description: T-Shirt X-Large Red
If the description is longer than 30 characters, the extra characters go into Description Two for the child product.
7
Press F2-Save to save the configuration code.
You can assign these to a base product in Product Maintenance. Refer to Creating the Base and Child Products for information on setting up base products.
