Product Whse Transfer Information Screen
The Product Whse Transfer Information screen allows you to enter package information that Nightly Processing checks to recommend stock transfers. This field overrides the transfer package quantity in Product Maintenance.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Warehouse Maintenance > Action Bar > Addtl Optns > Transfer Information
Field & Function Key Descriptions
Field/Function Key
Description
Transfer Package Qty
The package quantity to use when transferring this product from this warehouse. Nightly Processing checks this quantity and the Round Transfer to Pkg Qty field in Inventory Parameters to calculate the package quantity when recommending stock transfers.
F2-Save
Saves the package information
