Transfer History Inquiry
The Transfer History Inquiry allows you to view received transfers for a warehouse. To view open transfers, refer to the Transfer Inquiry.
Viewing Received Transfers
To view transfers already received in a warehouse:
1
Select Inventory > Inquiries > Transfer Inquiries > Transfer History Inquiry to open the Transfer History Inquiry screen.
2
Enter the Warehouse.
Received transfers for the warehouse display.
Field & Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number. The warehouse description displays to the right.
Transfer#
The transfer release number.
N
An asterisk (*) indicates that notes exist in either the header or the line item detail for the transfer.
From
The shipping warehouse.
To
The receiving warehouse.
W/B
The user ID of the person who entered the transfer.
Type
The transfer type:
•
01 - Regular order.
•
02 - Non-inventory.
•
03 - Warranty inventory.
•
04 - Crib replenishment.
•
05 - Ship line item complete.
Refer to Defining Transfer Types for extended descriptions of transfer types.
Via
The code that indicates how the transfer shipped.
Ent-Date
The date the stock transfer was entered.
Shp-Date
The date the transfer shipped out of the From warehouse.
Rec-Date
The date the material was received at the To warehouse.
F2-Finish
Closes the inquiry screen.
F5-Review
Allows you to view detail product and quantity information for the transfer.
F6-Enter T/R
Allows you to enter a transfer number to go directly to.
F8-Log
Opens the Transfer Log Inquiry to allow you to view all transactions for the highlighted transfer number.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Allows you to view the previous page if multiple pages of transfers are listed.
sF6-Next Group
Allows you to view the next page if multiple pages of transfers are listed.
Action Bar
Output
Allows you to view, print, email, or fax the usage information.
