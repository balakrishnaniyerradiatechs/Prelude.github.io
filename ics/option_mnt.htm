Equipment Option Maintenance screen
The Equipment Option Code Maintenance screen allows you to create and maintain equipment codes to assign to a piece of equipment. You can assign each code to a group of equipment codes (Equipment Option Group Maintenance screen) and then assign the Option group to a product in Product Maintenance. Refer to Using Equipment Option Codes for more information.
Menu Path: Inventory > Maintenance > Misc Code Files > Equipment Option Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Option Code
The equipment option code to assign to a piece of equipment when purchased.
Description
The description of the option code.
F2-Save
Saves the option code.
F2-Del
Deletes the option code.
