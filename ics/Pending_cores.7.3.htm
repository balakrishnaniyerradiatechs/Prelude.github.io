Creating Core Type Codes
Core type codes group core products for core banking. You can create a code in Core Type Code File Maintenance and then assign it to a product in the Additional Product Maintenance Information Screen.
To create a core type code:
1
Select Inventory > Maintenance > Misc Code Files > Core Type Code File Maintenance to open the Core Type Code Maintenance screen.
2
Enter the Core Type Code to create or edit.
3
Enter the Description of the core type code.
4
Press F2-Save to save the core type information.
