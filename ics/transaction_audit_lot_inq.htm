Transaction Audit By Lot Inquiry
The Transaction Audit by Lot Inquiry displays the transactions that affected the on-hand quantity or average cost of a product/lot number for a warehouse. You can also view the detail information on each transaction.
Viewing Lot Number Transactions
To view the transactions for a lot number:
1
Select Inventory > Inquiries > Transaction Audit Inquiries > Transaction Audit by Lot to open the Transaction Audit By Lot Inquiry screen.
2
Enter the Product.
3
Enter the Warehouse number.
4
Enter the Lot number. Press F3 to list the lot numbers for this product.
5
For Sort Sequence, determine how to display the transaction information:
•
B - Display entries in ascending order, from the oldest to the newest. If you enter a beginning date, only transactions on or after the date display.
•
E - Display entries in descending order, from the newest to the oldest.
•
F - Filter the transactions to display. You can enter beginning and ending dates and the transaction codes to display (Refer to the list of Transaction Audit Inquiry Codes). In this case, the true on-hand, average cost, and linked cost may not represent a sequential audit of the product’s activities.
The transaction codes display along with reference numbers, values, and on-hand quantities.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number. The product description displays.
Warehouse
The warehouse number. The warehouse description displays.
Lot
The lot number for this product and warehouse.
Sort Sequence
Determines the order for the transaction information:
•
B - Displays entries in ascending order, from the oldest to the newest. If you enter a beginning date, only transactions on or after the date display.
•
E - Displays entries in descending order, from the newest to the oldest.
•
F - Filters the transactions to include. You can enter beginning and ending dates and the transaction codes to display (Refer to the list of Transaction Audit Inquiry Codes). In this case, the true on-hand, average cost, and linked cost may not represent a sequential audit of the product’s activities.
U/M
The stocking unit of measure of the product from Product Maintenance.
Linked Value
The value of all lot numbers of this product in on-hand inventory linked to sales orders, work orders, or transfers.
Stock Value
The value of all lot numbers of this product in on-hand (unlinked) inventory.
Code
The code representing the transaction that affected the product. Press F3 to see a list of codes. Refer to Transaction Audit Inquiry Codes for more information.
Date
The date the transaction took place.
Reference 1
A reference number for the associated transaction (e.g., sales order number, purchase order number, invoice number, receipt number). Refer to Reference 1 on the list of Transaction Audit Inquiry Codes.
Reference 2
A reference number for the associated transaction (e.g., sales order number, purchase order number, invoice number, receipt number). Refer to Reference 2 on the list of Transaction Audit Inquiry Codes for an explanation of each reference number.
CN/VN/Whse
The customer number, vendor number, or warehouse number associated with the transaction.
Units
The number of units going into or out of inventory. A negative value means the amount was deducted from inventory.
Value/Unit
The cost per unit for the transaction.
Cost
The average cost of the item.
Onhand
The total value of on-hand linked and unlinked inventory.
F2-Finish
Closes the inquiry screen.
F5-Review
Allows you to view the detail and historical information of the highlighted transaction. The type of transaction code selected determines the type of display screen that appears.
sF5-Prev Group
Returns to the previous page if more than one page of transactions are listed. The number of records that display depends on Records to Block in Company Maintenance > Action Bar > GUI Options.
sF6-Next Group
Forwards to the next page if more than one page of transactions are listed. The number of records that display depends on Records to Block in Company Maintenance > Action Bar > GUI Options.
