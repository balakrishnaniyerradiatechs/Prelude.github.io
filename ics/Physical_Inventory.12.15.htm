Printing Physical Inventory Count Sheets
After you create the inventory lists for a warehouse (refer to Building the Physical Inventory Lists) and save the on-hand quantities (refer to Saving Current On-Hand Quantities), you can print the count sheets and begin counting inventory.
To print inventory count sheets:
1
Select Inventory > Physical Inventory > Inventory Sheet Procedure > Physical Count Sheet Printing to open the Physical Count Sheet Printing screen.
2
Enter the Warehouse for the count sheets.
3
Enter the Physical ListName for the count sheets. Leave blank to print count sheets for all lists created for the warehouse.
4
For #/Spaces Between Lines, enter the number of blank lines to print between each product. Enter 0 for single spacing.
5
For Show Onhand Quantity, indicate if you want the saved on-hand quantity to display on the count sheets. Choose from the following:
•
Y - Display the saved on-hand quantities of each product.
•
N - Suppress the saved on-hand quantities of each product.
6
For Count Sheet Size, indicate the paper size for the count sheets. Choose from the following.
•
1 - Print the count sheets on 8 ½” X 11” (letter) paper.
•
2 - Print the count sheets on 11” x 14” (legal) paper.
7
Press ENTER to print the count sheets.
After the count sheets print, you can begin counting inventory. Be sure to count inventory in staging or tag-and-hold areas also. You can begin entering counts into the system (refer to Entering Physical Inventory Counts) as each sheet is completed.
Count Sheet Column Descriptions
Column Heading
Description
Product Number
The product numbers included in the physical list name.
Description
The product description.
Bin
The bin number for the product, if applicable.
On-Hand
The saved on-hand quantity. Refer to Saving Current On-Hand Quantities.
Count
A blank space to enter the count.
