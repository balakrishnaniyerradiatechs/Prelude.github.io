Loading a Price Tape
Once you setup the price tape, (refer to Setting Up a Price Tape), use the following procedures to load the price tape:
1
Select Inventory > Processes > Price/Cost Update Procedure >Price Tape Update > Price Tape Load to open the Price Tape Load Routine screen.
2
For the Name on Diskette or Data File, enter the full name of the price tape. Press F3 to select a setup.
3
Enter the Vendor for the price tape. The information from the Price Tape displays.
4
For File Type, indicate where the file is coming from:
•
C - CD. You first must go to UNIX as root and type
mount /cdrom. When finished with the load, type unmount /cdrom.
•
D - DOS diskette or the /tmp UNIX directory.
•
P - Data already stored in the UniData file.
5
Press F2-Load to read the price tape. If you chose D for File Type, the following prompt displays:
Are You Loading From a Diskette?
6
Choose from the following:
•
Yes - Loading from a diskette or floppy.
•
No - Loading from a UNIX /tmp directory.
The product numbers are built and stored in the proper format in the PRICE.WORK file.
The Pre-Update report (refer to Running the Price Tape Pre-Update Report) allows you to verify the prices before you update the products.
