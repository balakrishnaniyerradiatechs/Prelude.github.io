Reserved Inventory - Corporate View screen
The Reserved Inventory - Corporate View screen displays the quantity reserved for each customer for each warehouse (refer to Reserving Inventory for a Customer).
Menu Path: Inventory > Inquiries > Availability Inquiries > Product Warehouse Availability Inquiry > Action Bar > Rsrv Inv
Inventory > Inquiries > Inventory Inquiry > Inquiries > Reserved Inventory-Corp view
Field and Function Key Descriptions
Field/Function Key
Description
Whse
The warehouse where the product is reserved.
Customer
The customer for whom the product is reserved.
Description
The customer description.
Qty
The quantity reserved for the customer.
OK
Closes the screen.
