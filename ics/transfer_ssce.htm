Transfer Serial Shipping Container Entry screen
The Transfer Serial Shipping Container Entry screen allows you to assign a serial shipping container code (SSCC) to items packed in a carton or pallet. The products are tracked under the SSCC for barcode printing and EDI transmissions.
Refer to Using Serial Shipping Container Codes for Transfers for more information.
Menu Path: Inventory > Stock Movements > Transfer Serial Shipping Container Entry
Field and Function Key Descriptions
Field/Function Key
Description
Transfer Release#
The release number for the transfer.
Serial Ship Code
The transfer serial ship container code.
Package Type
Defines the type of container:
•
C - Carton.
•
P - Pallet.
Product
The product numbers included in the container.
Description
The product description.
U/M
The unit of measure for the product.
Quantity
The quantity of the product packed in the container.
F2-Save
Completes the serial shipping container entry and allows you to check the accumulated product information to make sure it matches the products on the release. If everything matches, the transaction is complete. If discrepancies exist, you can print a report of the discrepancies and update the release with the quantities on the SSCC records.
F4-Del
Deletes the transfer serial shipping container entry.
F5-Review Release
Opens the Stock Transfer Display screen to allow you to review the items and quantities on the release.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
