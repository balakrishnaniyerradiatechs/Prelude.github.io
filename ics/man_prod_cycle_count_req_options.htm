Manual Product Cycle Count Request Options screen
The Manual Product Cycle Count Request Options screen allows you to select products to send to the second level count queue. You can select products by a product number range, bin number range, status, or sales ranking.
Menu Path: Inventory > Physical Inventory > Cycle Counting Procedure > Product Cycle Count Request Entry > Request Type - M
Field and Function Key Descriptions
Field/Function Key
Description
Product >
The beginning product number to include in the second level count.
Product <
The ending product number to include in the second level count.
Stocking Status
The stock status of products to include:
•
S - Stock.
•
N - Non-stock.
•
C - Consigned.
Onhand not equal to 0
Indicate if you want to only include products with an on-hand balance not equal to zero:
•
Y - Do not include products with a zero on-hand balance.
•
N - Include products with a zero on-hand balance.
Primary Bin >
The beginning bin number to include in the second level count.
Primary Bin <
The ending bin number to include in the second level count.
Rank By Sales >
The beginning product by sales ranking to include in the second level count.
Rank By Sales <
The ending product by sales ranking to include in the second level count.
F2-Save
Saves the product selection information.
