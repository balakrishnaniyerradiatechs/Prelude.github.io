Resetting Physical Inventory Attributes
You must clear and reset the physical inventory attributes and statistics for each warehouse before you start the physical inventory tag procedure to make sure that the old data is cleared.
To reset the physical inventory attributes:
1
Select Inventory > Physical Inventory > Inventory Tag Procedure > Reset Physical Inventory Attributes.
2
Enter the Warehouse to clear.
3
Press ENTER.
