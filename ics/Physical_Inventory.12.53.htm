Creating a User-Defined Inventory List
To create an inventory count list from your own set of selection criteria:
1
Select Inventory > Physical Inventory > Radio Frequency Procedure > Build Physical Inventory Lists > By Range > User Defined Criteria to open the General Report Interface for user defined physical lists.
2
Enter the Warehouse for the inventory list.
3
For Physical ListName, enter a name to identify the count list. You will use this name to enter counts, update inventory, and run inventory reports.
4
For Only Onhand not equal to 0, choose from the following:
•
Y - Only include products with an on-hand quantity above or below zero. Do not include products with zero quantity.
•
N - Include all products regardless of the quantity.
5
For Sort By? (P, B, V, L, M), indicate how you want to sort the count list. Choose from the following:
•
P - By product number.
•
B - By primary bin location.
•
V - By vendor number.
•
L - By product line.
•
M - By major group.
6
For Product Type, indicate the type of products to include. Choose from the following or leave blank to include all products:
•
N - Rental.
•
F - Finished.
•
R - Raw.
•
B - Both finished and raw.
•
E - Expense.
•
U - Used.
7
Enter the Stocking Status Code to include on the list. Choose from the following or leave blank to include products of all stock status codes:
•
S - Stock.
•
N - Nonstock.
•
C - Consigned.
8
Press F2-Create List. The following prompt appears:
## Items Selected. Press Return.
9
Press ENTER to create the lists for the selected bins.
To view the list, refer to Displaying Physical Inventory Lists.
