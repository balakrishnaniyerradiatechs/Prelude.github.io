Setting Quality Control Inspection Flags
The following flags determine which products require quality control inspection upon receipt and who is authorized to release the products into inventory. Once the quality inspection flags are established for products, the hierarchy to determine which level to use first is:
-
1st level - Product Warehouse Maintenance.
-
2nd level - Product Maintenance.
-
3rd level - Vendor Maintenance.
When a product that requires inspection is received, it must be released by an inspection manager in the Quality Control Inspection Inquiry before it is available for sale. Refer to Releasing Products from Quality Control Inspection.
Inventory Parameters
> F5-Add Info: Quality Inspection - AQL must be set to N to use inspection levels. Setting the flag to Y activates the Quality Inspection Subsystem in the Order Processing module.
Warehouse Maintenance
Action Bar > Additional Options > Bin Locator Information: Set Multiple Bin Locations to N to activate multiple bins. Enter the Default Inspection Bin where inspection products should be received into in Stock Receipts Entry.
Warehouse Manager Maintenance
Enter the Inspection Manager for each warehouse. This ID is overridden if a Manager ID exists in the Qual Insp Required field in Product Warehouse Maintenance.
User ID Maintenance
> F5-Administrative: Enter Y for Inspection Mgr to allow the user to view and release inspection products in the Quality Control Inspection Inquiry. Enter the Inspection Manager ID established in Warehouse Manager Maintenance.
> F6-Add Info: Enter Y for Inspection Bin Movement to allow the user to approve the quality of received material. This flag controls bin movements from inspection bins.
Product Warehouse Maintenance
Enter Y for Inspection? to require inspection of the product by a quality control inspector. Enter the Inspection Manager Id established in Warehouse Manager Maintenance.
Product Maintenance
> F5-Add Info: Enter Y for Qual Insp Required to require inspection of the product by a quality control inspector. Indicate how many levels of inspectors must release this item in Quality Control Inspection Inquiry before it is released into available inventory.
Vendor Maintenance
> F5-Add Info: The ISO Qualified flag determines if the vendor is included in the ISO Quality Preference and if inspection is required when products are received. Choose from the following:
•
Blank - The default setting if ISO Qualified vendors are not applicable for your company. No inspection is required.
•
Y - Marks the vendor as Qualified according to the ISO Qualified Preference. The Inspection Required flag for purchase orders to this vendor is set to N. You must enter the date the vendor was ISO Qualified and the user ID of the person who qualified the vendor for ISO certification.
•
P - Marks the vendor as Non-Preferred according to the ISO Qualified Preference. The Inspection Required flag for the purchase orders to this vendor is set to N.
•
N - Marks the vendor as Disqualified according to the ISO Qualified Preference. The Inspection Required flag for the purchase orders to this vendor is set to Y.
