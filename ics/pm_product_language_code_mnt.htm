Product Language Code Mnt Screen
The Product Language Code Maintenance screen allows you to enter the description of a product in foreign languages.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Language Information
Field & Function Key Descriptions
Field/Function Key
Description
Language Code
The code to define the language of the following product description.
Description One
The first line of the product’s description in the language defined by the Language Code.
Description Two
The second line of the product’s description in the language defined by the Language Code.
F2-Save
Saves the product language code information.
