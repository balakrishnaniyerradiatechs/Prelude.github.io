Create an Update Template
Once you create a list of products to update (refer to Create a Product List) you can create a template of the product changes (e.g., product line number, description, primary vendor).
To create a template:
1
Select Inventory > Maintenance > Product Maintenance > Global Product Maintenance Menu > Template Product Maint Entry to access the Template Product Maintenance Entry screen.
2
Enter a Product to use for the template. The product you select must be in the list you created. Refer to Create a Product List.
3
Change the fields as necessary. The field descriptions are the same as the fields in the Product Maintenance Screen. These changes become part of the template.
4
Press F2-Save to create the template product and open the Template Name Entry screen.
5
Enter the Template Name. If the name already exists, a message notifies you to either override the existing name or to enter a new one.
6
Press F2-Save to save the template.
Refer to Update Products for information on the final step in globally updating products.
