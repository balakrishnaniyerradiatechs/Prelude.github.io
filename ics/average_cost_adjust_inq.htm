Average Cost Adjustment Inquiry
The Average Cost Adjustment Inquiry displays the detail information for a cost adjustment.
Viewing Average Cost Adjustments
To view cost adjustments for a warehouse:
1
Select Inventory > Inquiries > Average Cost Adjustment Inquiry to display the Average Cost Adjustment Inquiry screen.
2
Enter the Adjustment Number.
Cost adjustment information displays.
Field & Function Key Descriptions
Field/Function Key
Description
Adjustment Number
The cost adjustment number.
Entry Date
The date the cost was adjusted. The fiscal period displays to the right.
Product
Line 1: The product number.
Line 2: The product description.
Line 3: The second line of description for the product.
Warehouse
The warehouse number and description.
Current Cost
The current average or standard cost for the product.
Revised Cost
The new cost entered for the product.
Onhand
The quantity currently on-hand in the warehouse.
Extended Amount
The extended dollar amount of the difference between the old cost and the new cost. (Onhand) x (Current Cost - Revised Cost).
Cost Adj Reason#
The cost adjustment reason code to explain why the cost was adjusted. The description of the reason code displays to the right. Refer to Establishing Cost Adjustment Reason Codes.
Lot
If the cost was adjusted for a lot item, the lot number displays.
Serial Number
If the cost was adjusted for a serialized item, the serial number displays.
