Serial Number History Inquiry
The Serial Number History Inquiry displays the history of all transactions for a product/serial number combination.
Viewing Serial Number Transactions
To view the Serial Number History Inquiry:
1
Select Inventory > Inquiries > Product/Serial Number Inquiries > Serial Number History Inquiry to open the Serial Number History Inquiry screen.
2
Enter the Product.
3
Enter the Serial Number.
All transactions that occurred for the product/serial number display.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number. The description of the product displays.
Serial Number
The serial number.
Warehouse
The number for the warehouse where the serialized item was received.
Last Activity
The date of the last transaction containing this serialized product.
Bin
The bin location of the serialized item.
Type
Inventory status for the serialized product:
•
S - Scrapped.
•
D - Damaged.
•
W - Warranty.
•
Blank - Regular inventory.
FP
Indicates if this product/serial number has a floor plan established:
•
Y - A floor plan exists for the product. A floor plan means that you do not have to pay for until a specified date, or when it is sold.
•
N - No floor plan exists for the product.
Onhand
The total quantity in inventory for this product/serial number.
Doc#
The current document that the serialized product exists on.
WtyExp/Mat
The expiration date for the warranty material if this is a warranty service item (Service Item Flag set to Y, type set to W in the Product Maintenance Screen). The expiration date is calculated using the Warranty Period information in the Additional Product Maintenance Information Screen).
WtyExp/Lab
The expiration date for the warranty labor if this is a warranty service item (Service Item Flag set to Y, type set to W in the Product Maintenance Screen). The expiration date is calculated using the Warranty Labor Period information in the Additional Product Maintenance Information Screen).
Equip#
The equipment identification number.
Trans#
The transaction number containing this product/serial number.
Code
The serial number transaction code regarding the product serial number. For an explanation of codes, refer to Serial Number Transaction Codes.
TransDate
The date the transaction occurred.
Reference
The transaction number (e.g., order number, purchase order number, transfer number).
Description
Varies according to the Reference number. For an explanation of descriptions, refer to Serial Number Transaction Codes.
Cost
The cost of the product.
Price
The selling price of the product.
Rental Ext.
The extended value of the product if it is on a rental order (code RTL).
From
The warehouse that shipped the item if the transaction is a transfer.
To
The warehouse receiving the item.
User
The user ID of the employee who entered the transaction.
F2-Finish
Closes the screen.
F5-Review
Opens the corresponding inquiry screen, depending on the type of document that the product is on.
F6-Floor Plan
Opens the Serial Number Floor Plan Inquiry to allow you to view A/P floor plan information.
F7-Shop
Opens the Shop Repair Information screen to allow you to view shop repair information, if applicable.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Book Value
Opens the Book Value screen to display the transaction date, cost and book value for the selected transaction. This information may be updated by stock receipts, depreciation, cost adjustments, equipment attachments, or capital repairs.
Opts
Opens the Equipment Options screen to list equipment options that exist for the product/serial number. Refer to Using Equipment Option Codes.
