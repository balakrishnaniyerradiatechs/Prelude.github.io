Manufacturer Maintenance screen
The Manufacturer Maintenance screen allows you to enter manufacturer abbreviations for lot controlled items. Once you create the abbreviation codes in Manufacturer Maintenance, you can assign the codes to:
•
A lot product in Product Warehouse Maintenance > F9-Lot > Manufacturer Code.
•
A vendor in Vendor Maintenance > F5-Add Info > Manufacturer Code.
•
A product when it is received in Stock Receipt Entry > Detail Information screen > Lot Information Screen > Mfg#.
Menu Path: Inventory > Maintenance > Misc Code Files > Manufacturer Abbreviations
Field and Function Key Descriptions
Field/Function Key
Description
Manufacturer Number
The manufacturer code that identifies the manufacturer of the product. You can assign this code to lot products in the Lot Information Screen in Product Warehouse Maintenance for lot controlled items.
Description
The description of the manufacturer.
F2-Save
Saves the manufacturer code.
F4-Del
Deletes the manufacturer code.
