Physical Inventory List By Bin screen
The Physical Inventory List By Bin screen allows you to create an inventory count list by bin location.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Build Physical Inventory Lists > By Bin Location(s)
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse for the inventory list.
Physical ListName
The name that identifies the count sheet. You will use this name to print count sheets, enter counts, update inventory, and run inventory reports.
Bin
The bin location(s) to include on the count sheet. Only products with this primary bin assigned in the Product Warehouse Maintenance Screen will print on the count sheets.
Description
The bin description.
F2-Create List
Generates the inventory lists.
