Lot Information Screen
The Lot Information screen appears when you set Lot Number Flag in the Product Maintenance Additional Information Screen to Y for lot items. You can enter lot, license plate, and reel information.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > Lot Number Flag = Y
Field and Function Key Descriptions
Field/Function Key
Description
Lot Exp Date Flag
Indicates if an expiration date is required to receive lot or license plate items:
•
Y - An expiration date is required.
•
M - The date of manufacture is required. The date of manufacture plus the Shelf Life Days in the Product Maintenance Additional Information Screen determine the expiration date.
•
N - No expiration date is required.
For more information, refer to Receiving Lot Number Products and Receiving License Plate Products.
Minimum Remnant Quantity
The minimum quantity considered practical to sell. Lots with less than this quantity are not considered as available by Nightly Processing. Example: Pipes are usually cut to meet a customer requirement. Once the pipe is cut down to about 2 feet it may not be a commonly used size. The pipe is still in stock and affects the purchasing of new pipes. This flag allows the 2-foot pipe to be ignored (as if it didn’t exist) when determining a recommended purchase quantity.
Lot Priority Code
Determines which lots are picked first:
•
N - No sequence.
•
S - Smallest lot that fills the requirement.
•
O - Oldest single lot first. Searches for a single lot (starting with the oldest) that will fill the order.
•
F - Oldest lot first. All lots are combined, starting with the oldest, to fill the order.
Lot Scrapped Qty
Indicates the minimum on-hand quantity the product can have in stock before it is automatically removed from available inventory.
Lot Pack Flag
Determines if the product is sold in a pack:
•
Y - The product is sold in a pack. The Lot Additional Info screen screen opens to allow you to enter package information.
•
N - The product is not sold in a pack.
Price/Cost Required?
Indicates if you are prompted to enter commission cost and selling price when you enter this lot product on a purchase order:
•
Y - When you enter this product on a purchase order, a dialog box displays to allow you to enter commission cost and selling price.
•
N - Commission cost and selling price are not required.
Reel Optimization
Indicates if the product looks for the optimal remaining reel quantity when allocating lots to sales orders:
•
Y - Displays the Lot Reel Prompting screen to show all available lot quantities of the product. Refer to Allocating Optimal Reel Quantities for more information.
•
N - Does not display the Lot Reel Prompting screen.
Reel Optimization Minimum
The minimum quantity that can be left on the reel for each lot before the system allocates another lot. Refer to Allocating Optimal Reel Quantities for more information.
Lic Plate-SOE Lot Prompt
Determines if the Lot Number Information screen appears when you enter through the Qty-Pck field in the Line Item Detail screen in Sales Order Entry. This only applies if you have the License Plate option set to Y.
F2-Save
Saves the lot information.
