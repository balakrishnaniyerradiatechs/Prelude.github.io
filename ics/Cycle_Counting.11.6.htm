Bin Location Maintenance
Once you set up zones for cycle counting, you must enter the Cycle Count Zone and the Bin Type for each bin. Bin locations are assigned to products in the Product Warehouse Maintenance Screen.
To set up bins for cycle counting:
1
Select Inventory > Maintenance > Bin Files > Bin Locations to open the Bin Location Maintenance screen. For a full explanation of all the fields, refer to Defining Bins.
2
For Cycle Count Zone, enter the zone to assign to this bin for cycle counting so you can print cycle count sheets by zone. If this field is left blank, the system uses the Zone already established for the bin.
3
For Bin Type, indicate the type of bin this location represents. Only the following are used for cycle counting:
•
B - Bulk.
•
P - Pick.
•
T - Retail.
4
Press F2-Save to save the bin location information.
