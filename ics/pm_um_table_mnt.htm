U/M Table Maintenance screen
The U/M Table Maintenance screen allows you to define how a product’s unit of measure impacts pricing, costing, and inventory. If you buy, stock, and sell in different units of measure, you can enter a factor to divide or multiply by to convert the unit of measure. Before you set up the unit of measure table, you must set up the default units of measure for Buy, Stock, and Price (refer to Establishing a Unit of Measure).
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > U/M Table
Field and Function Key Descriptions
Field/Function Key
Description
Allow U/M Override
Indicates if you can change the unit of measure on all transactions:
•
Y - Allow overrides to the unit of measure table.
•
N - Disallow overrides to the unit of measure table.
Default S/O U/M
The default unit of measure for sales orders. If you leave this blank, the Price unit of measure from the Product Maintenance screen is used for sales orders.
Default Web U/M
The default unit of measure for web orders. If you leave this blank, the Price unit of measure from the Product Maintenance screen is used for web orders.
U/M Code
The unit of measure code for the product.
Factors: Allow you to define how to convert one price, cost, or inventory unit to another. The Price/Cost factor and the Inventory factor are independent of each other.
Price/Cost
The factor to divide or multiply by the prices/costs to convert from one unit of measure to another. Leave this field blank if a factor already exists in the company unit of measure table and the Price/Cost factor is the same for this product.
Inventory
The factor to divide or multiply by the inventory quantity to convert from one unit of measure to another. Leave this field blank if a factor already exists in the company unit of measure table and the Inventory factor is the same for this product.
Valid in SOE
Determines if the unit of measure can be used in Sales Order Entry:
•
Y - The unit of measure is allowed in Sales Order Entry.
•
N - The unit of measure not allowed in Sales Order Entry.
Qty Per U/M
The quantity for each unit of measure. For information only and does not affect the calculations.
F2-Save
Saves the unit of measure information.
