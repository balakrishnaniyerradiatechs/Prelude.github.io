Stock Transfer Header screen
The Stock Transfer Header screen allows you to enter or view shipping information for a stock transfer.
Menu Path: Inventory > Stock Movements > Transfer Entry (or Transfer Shipments Entry) > F5-Header
Field and Function Key Descriptions
Field/Function Key
Description
Transfer Type
Indicates the type of transfer:
•
Regular Order-Type 01: Moves inventory items from one warehouse to another.
•
Non-Inventory - Type 02: Moves non-inventory items such as office supplies or packing materials. The products still appear in the system but are not tracked as inventory.
•
Warranty Inventory - Type 03: Moves warranty tag inventory from one location to another. The items must exist in the system as warranty inventory. Refer to Transferring Warranty Inventory.
•
Crib Replenishment - Type 04: If the receiving warehouse is a crib warehouse, the transfer type defaults to 04. Replenishment is handled as a standard transfer for product/bin locations. Nightly processing generates a recommended transfer to re-stock the crib.
•
Ship Line Item Complete - Type 05: The transfer will not ship a line item if the line is not completely available to ship.
Written By
The user ID of the person entering the order.
Ship Via
The ship via code to indicate how to transfer the material. Defaults from Warehouse Maintenance for the receiving warehouse.
Entry Date
The date the transfer was entered.
Pick Ticket Date
The date to print the pick ticket, if a quantity is available.
Unusual Usage Flag
Indicates if the transfer should update usage and lead-time:
•
N - The transfer of the material is not unusual; therefore, usage and lead-time are updated.
•
Y - The transfer of the material is unusual so usage and lead-time are not updated for the products included on the transfer. Refer to Recording Product Usage for information on transfer usage.
Picker
The user ID of the person that filled the order. This ID must be a valid ID set up in Picker Maintenance.
Packer
The user ID of the person that packed the order.
Loader
The user ID of the person that loaded the truck. Defaults to the person logged on to the terminal.
Freight Amount
Opens the Transfer Freight Detail Info screen to allow you to enter freight information. This information is normally entered when you ship the transfer. Refer to Shipping Stock Transfers.
Weight
The weight of the transfer.
Number of Cartons
The number of cartons in the transfer.
Waybill Number
The waybill number if known.
Instructions
Shipping instructions regarding the transfer.
F2-Save
Saves the header information.
F5-Freight Detail
Opens the Transfer Freight Detail Info screen to allow you to enter freight information.
F7-Ship To
Opens the Manual Ship-To Entry screen to allow you to change the shipping address for the transfer.
Action Bar
Fut Commit Ov
Opens the Future Commitment Override screen to allow you to commit the inventory for future transactions.
