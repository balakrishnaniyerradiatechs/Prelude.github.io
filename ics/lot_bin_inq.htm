Lot Bin Inquiry screen
The Lot Bin Inquiry screen displays bin information for a lot/license plate number in a warehouse.
Menu Path: Inventory > Inquiries > Miscellaneous Inquiries > Lot Inquiry > F6-Bin Inq
Field and Function Key Descriptions
Field/Function Key
Description
Product Number
The lot product number.
Warehouse
The warehouse number and name that contains the lot product.
Lot Number
The lot number assigned by the system or the vendor.
Expiration Dte
The expiration date of the lot product in this bin, if the lot product was set up to require an expiration date in the Lot Information Screen in Product Maintenance.
Vendor
The vendor that shipped the lot product.
Bins
All bins that contain the product.
Bin OH
The amount in each bin that is in on-hand inventory.
Bin Cmt
The amount in each bin that is committed to sales orders, work orders, or transfers.
Bin Avl
The amount in each bin that is available to sell.
F2-Save
Closes the Lot Bin Inquiry screen.
