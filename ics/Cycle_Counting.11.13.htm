Adding Bins to the First Level Cycle Count
In addition to the cycle counts generated by Nightly Processing, you can manually generate cycle count requests to include in the first level cycle count queue. The Bin 1st Level Cycle Count Request Entry allows you to request cycle counts for a range of bins outside of the normal counting cycle.
To manually request first level cycle counts by bin location range:
1
Select Inventory > Physical Inventory > Cycle Counting Procedure > Bin First Level Cycle Count Request Entry to open the Bin 1st Level Cycle Count Request Entry screen.
2
Enter the Warehouse for the cycle count.
3
Enter the Beginning Bin and the Ending Bin for the products to include in the cycle count request.
4
Press F2-Continue to place the products into the first level cycle count queue.
