Universal Product Replacement Entry screen
The Universal Product Replacement Entry screen allows you to modify a part number so that all occurrences of the old part number are replaced with the new number. (e.g., a vendor changes their part number or if a part number was incorrectly created).
When the old part number is entered on any transaction, the new part number automatically appears. The inquiry screens display the old part number as an alternate for the new part number.
Refer to Replacing and Merging Products for more information.
Menu Path: Inventory > Maintenance > Universal Product Replacement & Merge > Universal Product Replacement Entry
Field and Function Key Descriptions
Field/Function Key
Description
Old Product
The old or incorrect number to replace.
New Product
The new number to replace the old product number for all occurrences in the system.
F2-Save
Saves the replacement information. The replacement is not updated until you run the Universal Product Replacement/Merge Update (refer to Update Replaced/Merged Products).
