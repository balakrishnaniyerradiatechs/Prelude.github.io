Daily FIFO Cost Adjustment Journal
The Daily FIFO Cost Adjustment Journal displays FIFO cost adjustments for a day. If your company uses FIFO cost (refer to Using FIFO Cost), a cost adjustment is created when you change FIFO layers in FIFO Maintenance.
Menu Path: Inventory > Reports > Inventory Adjustment Reports > Daily FIFO Cost Adjustment Journal
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where FIFO costs were adjusted.
•
Entry Date. The date the FIFO costs were adjusted.
Column Descriptions
Column Heading
Description
Cst Adj#
The cost adjustment number assigned to the transaction.
Ent-Date
The date the FIFO cost was adjusted.
Id
The user ID of the person who adjusted the cost.
Product Number
The product number adjusted.
Old Cost
The cost of the product FIFO layer before the adjustment.
New Cost
The adjusted cost of the FIFO layer.
Onhand
The quantity of the product on-hand.
Ext. Amount
The extended dollar amount of the adjusted FIFO inventory layer
(New Cost - Old Cost ) x Onhand.
Cost Adjustment Reason Code
The code and description of the cost adjustment entered when the cost was adjusted.
