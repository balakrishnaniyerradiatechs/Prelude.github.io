Kit Where Used Inquiry
The Kit Where Used Inquiry identifies the parent product numbers associated with a component product number.
Kit Where Used Listing
To obtain a list of kit components and the parent kits they are a part of, select Inventory > Listings > Product Listings > Kit Whereused Listing.
Viewing Kits
To find the parent product for a kit component:
1
Choose Inventory > Inquiries > Miscellaneous Inquiries > Kit Where Used Inquiry to open the Kit Where Used Inquiry screen.
2
Enter the Component Product for which you want to find the kit parent product.
The kit parent product number displays.
Field & Function Key Descriptions
Field/Function Key
Description
Component Product
The kit component part number associated with a kit parent product.
Parent Product
The kit parent part number that contains the Component Product Number.
Description
Description of the Parent Product Number.
OK
Closes the screen.
