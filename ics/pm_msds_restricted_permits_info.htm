MSDS/Restricted Permits Information Screen
The MSDS/Restricted Permits Information screen allows you to assign MSDS (Material Safety Data Sheet) numbers and customer usage codes to a product. Refer to Using Material Safety Data Sheets for information on setting up MSDS codes.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > F6-MSDS Permits
Field & Function Key Descriptions
Field/Function Key
Description
MSDS Flag
Determines if a material safety data sheet must accompany this product when sold:
•
Y - The item requires an MSDS.
•
N - The item does not require an MSDS.
MSDS Number
The MSDS number in Material Safety Data Maintenance for this product.
Restricted Product
Indicates the type of restrictions for the product:
•
Y - Normal restrictions. The product requires that the customer have a restricted use permit to purchase this product. You must enter Usage Codes to define the restrictions. The customer must have the same usage codes in Customer Maintenance (Permits Information) to buy this product.
•
M - Matrix restrictions. Each state where this product is sold must be defined in Usage Codes along with the level of restriction.
•
N - No restrictions exist for this product.
Usage Codes
The code from Restricted Usage Code Maintenance that describes the type of restriction for the product (e.g. hazardous, prescription drug, freon). The customer must have the same usage code assigned in Customer Maintenance to buy the product (Customer Maintenance > F7-Order > F5-Add Info > Action Bar > Permits).
Level
The level of restriction. Only applies if Restricted Product is set to Y.
F2-Save
Saves the MSDS restricted permit information.
