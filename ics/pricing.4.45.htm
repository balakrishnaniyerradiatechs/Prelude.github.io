Correcting Price Tape Updates
If you find any errors on the Price Tape Pre-Update Report (refer to Running the Price Tape Pre-Update Report), use Price Tape Work Maintenance to correct the data.
If you need to correct an extensive number of items, you can enter the file in a spreadsheet, correct the data, run the Price Tape Load, and then run the Price Tape Pre-Update Report again. Refer to Running the Price Tape Pre-Update Report.
To correct data in the temporary work file:
1
Select Inventory > Processes > Price/Cost Update Procedure > Price Tape Update > Price Tape Work Maintenance to open the Price Tape Work Maintenance screen.
2
For Diskette/File Name, enter the diskette or name from the report. The Vendor displays.
3
For Record Id, enter the record ID number from the Pre-Update report, which is the sequence number at the end of the file name. For example: if the file name is STAN2001Price.csv-1006, enter 1006 for the record ID. Press F3 to list file names and record IDs for the vendor.
The Product and the Data display.
4
Correct the Data as necessary.
5
Press F2-Save to save the price tape changes.
After you correct the data, you can update the new prices and costs. Refer to Updating the Price Tape Data.
