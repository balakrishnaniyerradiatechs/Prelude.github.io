Parker Report
The Parker Report displays sales information for products set up with Parker product codes. The Parker Hannifin manufacturer requires its distributors to track and report all sales of Parker products. You can set up Parker division codes and Parker distributor codes, then assign the codes to product lines or products. Refer to Parker Reporting Codes.
Menu Path: Inventory > Reports > Miscellaneous Reports > Parker Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Period. The fiscal period for the parker information.
•
Product Codes. The Parker product codes to include on the report (refer to Setting Up Parker Marketing Codes).
•
Report, Diskette, ASCII File?. Determines the report destination. Choose from the following:
-
R - Printer.
-
D - Diskette.
-
A - ASCII file.
•
Kit Reporting Option. The method for reporting kits, if kit items are included:
-
P - Compile the report using the parent product for kits.
-
C - Compile the report using the kit component products. The calculation for the components takes a percentage of the parent price.
-
B - Use both the parent and component products if they are included in the selected Product Codes.
Column Descriptions
Column Heading
Description
Cust Num
The number for the customer who purchased the Parker product.
Cde
The Parker reporting codes set up for the product. Refer to Setting Up Parker Marketing Codes.
Name
The customer name.
City
The customer’s bill-to city.
St
The customer’s bill-to state.
Zip
The customer’s bill-to zip code.
Sales
The total sales of Parker products for this customer.
Prod Desc
The Parker division code description. Refer to Setting Up Parker Divisions.
