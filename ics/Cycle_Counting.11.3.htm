Inventory Parameters
To activate cycle counting:
1
In Inventory Parameters, set Multiple Bin Locations to Y to indicate that a product may exist in multiple bins and that the system tracks the quantity on-hand in each bin.
2
Enter the default reason code for inventory adjustments in cycle counting (refer to Establishing Cost Adjustment Reason Codes).
3
Press F2-Save to save the bin information.
You can override this flag at the warehouse level in Warehouse Maintenance.
