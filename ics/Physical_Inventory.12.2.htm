Taking Physical Inventory Using Count Sheets
The Inventory Sheet Procedure provides a method to take physical inventory using count sheets to effectively record the count for each item in your warehouse. After you enter the new counts, you can run the Physical Inventory Deviation Exception Report to compare the system quantity to the count quantity. Once you verify the counts, the Physical Update Procedure adjusts the on-hand quantity of each item by the difference between the saved on-hand amount and the counted amount.
If Multiple Bins for the Warehousing module is set to Y, you cannot use count sheets for physical inventory. You must either use cycle counting (refer to Cycle Counting) or tags (refer to Taking Physical Inventory Using Tags) to take inventory.

Enter counts (refer to Entering Physical Inventory Counts).

12

13
