Previous Year Weekly Usage screen
The Previous Year Weekly Usage screen displays the weekly usage totals for sales orders, work orders, and transfers for a product for the previous year.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Usage Maintenance > F9-Weekly Usage > F5-Previous Year
Field & Function Key Descriptions
Field/Function Key
Description
Grand Totals
The total amount of each column.
Begin
The beginning date (Monday) of the week.
End
The ending date of the week.
S/O Usage
Product usage recorded from sales orders for the week.
Override
A space to enter an override to S/O Usage for the corresponding week.
W/O Usage
The usage quantity calculated from work orders for the week.
Override
A space to enter an override to W/O Usage for the corresponding week.
T/R Usage
The usage quantity calculated from transfers for the week.
Override
A space to enter an override to T/R Usage for the corresponding week.
F2-Save
Closes the screen.
