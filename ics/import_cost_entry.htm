Import Cost Entry screen
The Import Cost Entry screen in On The Water Maintenance allows you to view or override import costs calculated for the OTW purchase order.
Menu Path: Inventory > Stock Receipts > On the Water Maintenance > F5-Import Costs
Field and Function Key Descriptions
Field/Function Key
Description
Import Code
The import code to represent importing fees such as freight, duty, or brokerage fees. These costs are set up in Import Cost Factor Maintenance in the Purchasing module.
Description
The description of the code.
Currency
The currency used by the vendor.
Estimate
The estimated amount of the import cost.
Actual
The actual amount of the import cost, if known. This value overrides the Estimate amount.
F2-Save
Saves the import cost information.
