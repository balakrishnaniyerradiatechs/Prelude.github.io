Product Classification Rebuild Procedure screen
The Product Classification Rebuild Procedure screen allows you to rebuild product classifications independently of the End of Month Calendar Close Process.
Menu Path: Inventory > Processes > Rebuild Product Classifications
Field and Function Key Descriptions
Field/Function Key
Description
Enter Calendar Month End Closing Date
The last month-end date. For example, if today is November 10, enter October 31.
Is This the First Execution For This Month
Determines if the products were already classified this month:
•
Y - If this is the first time you are rebuilding classifications this month.
•
N - If you already rebuilt classifications this month. The existing calculations are overridden.
