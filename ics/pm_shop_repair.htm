Shop Repair Information Screen
The Shop Repair Information screen allows you to enter SRT (standard repair time) and SST (standard setup time) information published by the manufacturer of the product. The Shop Repair module uses the SRT and SST information to determine default labor charges for setup and repair operations.
Shop Repair Module
Shop Repair is an additional module that you must purchase from Activant.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Shop Repair Information
Field & Function Key Descriptions
Field/Function Key
Description
SRT Product
Table #
The Standard Repair Time table number that applies to this product. This number is published by the manufacturer and refers to a model number or a group of similar models.
SRT Repair Group
The repair group code for this type of labor (e.g., maintenance, diagnostics, attachments). The codes are set up in the Shop Repair module under Repair Group Maintenance.
SRT Time
The standard repair time (in hours) for this product as published by the manufacturer. You can use this information to estimate repair time when you enter a shop repair order.
SST Time
The standard time needed to set up this product. You can use this information to estimate setup time when you enter a shop repair order.
Setup Required
Indicates if the product requires a shop repair set-up order when sold:
•
Y - The product requires setup. A shop repair order generates when this item sold.
•
N - No set-up is required for this product.
F2-Save
Saves the shop repair SRT and SST information.
