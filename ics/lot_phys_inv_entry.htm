Lot Physical Inventory Entry screen
The Lot Physical Inventory Entry screen displays when you enter a count for a lot product during your physical inventory. You can enter the count for each lot number for a product. The accumulated count of all the lots becomes the total count for the product.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Physical Count Sheet Entry > lot product number
Field and Function Key Descriptions
Field/Function Key
Description
Lot Number
The lot numbers assigned to the product.
Ln#
The line number for each lot quantity.
Book-Oh
The on-hand quantity before the physical inventory.
Count
The count for each corresponding lot number of the product.
F2-Finish
Saves the count entry. If the accumulated count for all the lot numbers does not equal the Count entered for the product on the Physical Inventory Entry screen, you can either adjust the individual counts or let the system adjust the total Count.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
