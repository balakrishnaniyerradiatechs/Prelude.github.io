Releasing Products from Quality Control Inspection
The Quality Control Inspection Inquiry displays received products that require inspection. If the received product requires multiple levels of inspection, it must be released from all inspection levels before it updates inventory and prints labels.
Once the products are released, you can print the Receiving Inspection Completion Report to show you which products to move from the inspection bin to the picking bins.
Your user ID must be set up as an Inspection Manager to release the items into available inventory. Refer to Setting Quality Control Inspection Flags for information on activating quality control inspections.
If you use the Quality Inspection Subsystem on the Order Processing module (Quality Inspection AQL flag set to Y in Inventory Parameters > F5-Add Info), the Quality Inspection Management Queue displays but the functionality is different.
Quality Control Inspection Inquiry
To release products from the Quality Control Inspection queue:
1
Select Inventory > Inquiries > Stock Receipt Inquiries > Quality Control Inspection to open the Quality Control Inspection Inquiry screen.
2
Enter the Quality Control Inspector ID number to list Quality control products awaiting inspection.
3
Press F5-Review to view the detail information for each product.
4
Enter a Y under Release? for each item to release from the inspection queue. Depending on inspection level codes, one of the following occurs:
•
If the product does not have an inspection level code in Product Maintenance, the item is released from inspection hold and added to available inventory.
•
If the product has an inspection level code in Product Maintenance, the following message displays:
Press Continue to move the product to the hold queue for another level of inspection. When the product is released from the second level queue, follow the instructions under Second Level Inspection.
This Product Will Be Sent to the IN1.HOLD Queue While Inspection Tests Are Being Performed!
5
Press F2-Finish. You can print receiving inspection labels and the Receiving Inspection Completion Report for the released products.
Second Level Inspection
If any products went to the hold queue for a second level of inspection, use the following steps to release them into available inventory after they pass inspection:
1
From the Quality Control Inspection Inquiry screen, press F7-Toggle to view the products sent to the hold queue.
2
Enter a Y under Release? for each item to release. The Quality Inspection Tests Results screen appears.
3
For Test Results, enter one of the following test result codes for the product number:
•
A - Accepted.
•
W - Waived.
•
R - Rejected.
4
Enter any Inspection Comments that pertain to the test results. These comments print on the inspection reports.
5
Press F2-Save to release the product into available inventory. You can print receiving inspection labels and the Receiving Inspection Completion Report for the released products.
Field & Function Key Descriptions
Field/Function Key
Description
Quality Control Inspector
The inspection manager ID authorized to release products from Quality Inspection Hold. If your user ID has an inspection manager ID, it defaults into this field.
Product
The product numbers in the quality inspection queue.
Description
The product description.
Type
The document type for the received material:
•
P - Purchase order.
•
W - Work order.
Document#
The document number.
Quantity
The quantity on the order.
Bin
The bin where the product is held to await inspection.
Release?
Enter Y to release the product.
F2-Finish
Completes the release.
F5-Review
Opens the Purchase Order History Display screen to allow you to view the purchase order containing the product.
F7-Toggle
Displays the hold queue for products requiring inspection tests, if you use inspection level codes for released items.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
