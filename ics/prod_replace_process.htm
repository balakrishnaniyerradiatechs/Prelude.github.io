Update Replaced/Merged Products
Once you verify the product information using the Universal Product Replacement Preview Report, run the Universal Product Replacement/Merge Update to replace and merge the products. Open transactions as well as history records are updated for the products.
To update replaced and merged products:
1
Select Inventory > Maintenance > Universal Product Replacement & Merge > Universal Product Replacement/Merge Update.
2
Choose Execute. The following message displays:
The Following Procedure Will Update All Occurrences Of An Old Product Number With Its Corresponding New Product Number. This Process Will Only Update Standard Fields And Standard Fields. This Process Will Require Modifications To Handle Them Correctly. The Update Procedure Should Only Be Executed When There Are No Operators On The System. Do You Wish to Continue?
3
Choose from the following:
•
No: Returns to the menu and does not update the products.
•
Yes: Updates the products. The update takes a few minutes as updated files flash across the bottom of the screen. The following message displays when complete.
The Product Number Replacement Confirmation Report Has Been Printed. If ‘Y’ is Entered, The Cross-Reference File And The Confirmation Message File Will Be Cleared.
If ‘N’ Is Entered, The Process Of Clearing The Files Should Be Performed From The Universal Product Replacement Menu.
4
Choose from the following:
•
Yes: Clears the cross-reference and confirmation message files created during the update. The cross reference information is the data displayed on the The Universal Product Replacement Preview Report. Clear this file before the next time you update products so that the information does not show up again.
•
No: Does not clear the cross-reference confirmation message files created during the update. You must clear the files manually (refer to Clear the Cross-Reference File) before you update products again.
The order history descriptions for replaced products remain available to view after you update them.
