Transfer Ship-To Maintenance screen
The Transfer Ship-To Maintenance screen allows you to set up codes to reference a ship-to address to use for stock transfers. Once you set up a ship-to address, you can use it to transfer stock from any warehouse.
Menu Path: Inventory > Maintenance > Transfer Ship-To Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Transfer Ship-To
The code to represent this transfer ship-to address. You can use this code in Transfer Entry > F5-Header > F7-Ship-To, to enter a transfer to any warehouse.
Description
The description of the ship-to address.
Address One
The first line of address for the ship-to code.
Address Two
The second line of address for the ship-to code.
Address Three
The third line of address for the ship-to code.
City
The city for the ship-to code.
State
The state for the ship-to code.
Zip
The zip code for the ship-to code.
F2-Save
Saves the ship-to address information.
F4-Del
Deletes the ship-to code and address information.
