Running the FIFO Revaluation
The FIFO Revaluation Process verifies the replacement cost of all items for a division or warehouse. This process is only available if the FIFO Method flag is set to W in Company Maintenance. Refer to Setting System Flags for FIFO Costing.
For accurate revaluation, do NOT re-evaluate FIFO replacement cost while any users are logged on to the system or while Nightly Processing is running.
To run the FIFO Revaluation Process:
1
Select Inventory > Processes > FIFO Revaluation Process > FIFO Revaluation Process. The following message displays:
Warning - This process should only be run
When there is no activity on the system!
2
Choose from the following:
•
Return - Cancels the process.
•
Continue - Allows you to continue with the process.
3
Enter a Division or leave blank to include all divisions. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
4
Enter the Warehouse or leave blank to include all warehouses.
5
For ‘A’udit or ‘U’pdate, choose from the following:
•
A - Generates information for the FIFO Revaluation Audit Report to allow you to view the changes that will be made to replacement cost before you actually update the products.
•
U - Runs the process and automatically updates the products that meet the criteria for FIFO revaluation. After the process is complete, the FIFO Revaluation Audit Report prints to allow you to view the updated products. Refer to the FIFO Revaluation Audit Report.
The Transaction Audit Inquiry displays FRP for products adjusted in the FIFO Revaluation Process.
