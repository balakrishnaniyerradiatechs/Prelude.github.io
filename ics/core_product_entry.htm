Core Product Entry screen
The Core Product Entry screen allows you to enter a product number to exchange for a core product when it is returned. The product set up as an exchange product then displays for the core product the next time you press ENTER in the Core Product field in Product Maintenance.
Refer to Pending Cores for more information on core products.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F6-Xref/Notes > Core Product
Field and Function Key Descriptions
Field/Function Key
Description
Core Product
The product to exchange when the product set up as a Core Product is returned. Core products are set up in Product Maintenance in the Additional Product Maintenance Information Screen.
Description
The description of the core product.
F2-Save
Saves the core product information.
