License Plate Receiving Report
The License Plate Receiving Report displays the purchase orders, part numbers, and license plate numbers included in an advance shipping notice transmitted from the vendor.
Menu Path: Inventory > Stock Receipts > Batch Stock Receipts > License Plate Receiving Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Bill of Lading Number. The bill of lading number from the advance shipping notice sent from the vendor.
Column Descriptions
Column Heading
Description
Bill of Lading#
The bill of lading number from the vendor’s shipping notification.
Seal#
The seal number.
Order#
The order number from the vendor.
Shipment Date
The date the vendor shipped the material.
Total Weight
The total weight of the material shipped.
Scac Code
The standard carrier code that identifies the shipping company.
P/O#
The purchase orders included in each seal, by license plate number.
Product#
The products included in the shipment.
Serial#
The serial number for each product.
Shp Qty
The quantity shipped, according to the vendor.
U/M
The unit of measure for the item.
