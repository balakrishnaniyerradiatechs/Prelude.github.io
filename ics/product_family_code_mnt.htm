Product Family Code Maintenance screen
The Product Family Code Maintenance screen allows you to create family codes that group similar products. When one product in the group needs replenishment, you can review all the products in the group to determine if anything else can be ordered to meet any minimum ordering requirements. Refer to Product Family Codes for more information.
Menu Path: Inventory > Maintenance > Misc Code Files > Product Family Codes
Field and Function Key Descriptions
Field/Function Key
Description
Family Code
The code to identify a product family. Up to 20 alphanumeric characters.
Description
The description of the family code.
F2-Save
Saves the product family code information.
F4-Del
Deletes the product family code information.
