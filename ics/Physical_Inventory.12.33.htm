Entering Inventory Counts on an Expanded Screen
The expanded count screen allows you to enter multiple quantities or a series of package quantities for a product. The counts are rolled up and a single count is recorded for the product.
Inventory Parameters
To activate the Expanded Physical Entry screen, set the Phys. Inv Expanded Count flag in Inventory Parameters > F5-Add Info to Y.
Once you set the Phys Inv Expanded Count flag in Inventory Parameters to Y, the Expanded Physical Entry screen displays after you enter the bin number in Physical Count Entry. The expanded screen does not apply for lot products (refer to Entering Counts From Tags for Lot Products).
To enter package quantities on the Expanded Physical Entry screen:
1
For Shelf Pkg Qty, enter the quantity contained in each package of the item. For example, if the item comes in a single unit quantity, enter 1. If the item comes in packages of 10, enter 10.
2
For Units, enter the number of packages counted.
For example, if the item comes as a single unit, enter the total counted. If the item comes in packages of 10, enter the number of packages counted. The Shelf Pkg Qty is multiplied by the number of Units to calculate the Total Count for that package.
3
Continue to enter as many packages and units as necessary to complete the count for the item. The quantities in Total Count are added together to calculate the Grand Total for the item.
4
Press F2-Save. The amount of the Grand Total becomes the Count for the product on the Physical Count Sheet Entry screen.
The Expanded Physical Entry screen does not retain the count information. If you need to change a count, you must re-enter the count information.
