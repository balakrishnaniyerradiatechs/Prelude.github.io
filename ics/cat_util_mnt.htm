Catalog Page Utility Maintenance screen
The Catalog Page Utility Maintenance screen allows you to enter multiple products and their catalog page numbers. The information that you enter in this screen updates each product record individually (Product Maintenance > Action Bar > Addtl Opts > Catalog Information).
Menu Path: Inventory > Maintenance > Product Maintenance > Catalog Page Utility Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Catalog#
The number for the catalog that contains the products.
Catalog Page
The page number that the product is on in the corresponding catalog.
Product
The product number that is in the catalog.
Description
The product description.
F2-Save
Save the catalog page number information.
