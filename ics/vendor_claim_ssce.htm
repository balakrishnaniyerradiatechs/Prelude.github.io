Vendor Claim Shipping Serial Container Entry screen
The Vendor Claim Shipping Serial Container Entry screen allows you to assign a serial shipping container code and print labels to identify the contents of a vendor claim shipment.
Menu Path: Inventory > Vendor Claim > Vendor Claim Serial Shipping Container Entry
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Claim#
The vendor claim number to register on this serial shipping container code.
Serial Ship Code
The code to assign to serial shipping container.
Package Type
Defines the type of container:
•
C - Carton.
•
P - Pallet.
Product
The product number to register. If the item is serialized, you must enter the serial number information.
Description
The product description.
U/M
The unit of measure of the product as entered on the vendor claim.
Quantity
The quantity of the product packed in this container.
F2-Save
Completes the registration of the product and allows you to verify the product information.
F4-Del
Deletes the SSCC entry.
F5-Review Vendor Claim
Allows you to view the items on the vendor claim.
F9-Goto line
Allows you to enter a page number, reference number, or line number to go directly to.
