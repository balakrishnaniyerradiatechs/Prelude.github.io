Material Safety Data Sheet Maintenance screen
The Material Safety Data Sheet Information screen allows you to enter the MSDS numbers issued by manufacturers that require material data safety sheets to be sent with their products. Once you set up the MSDS numbers, you can assign the numbers to products in Product Maintenance.
When a product with an MSDS number is sold to a customer, a comment appears on the pick ticket instructing the picker to include the indicated MSDS. The customer record is flagged as having received the MSDS and the customer will not receive another one unless the MSDS is revised.
Refer to Using Material Safety Data Sheets for more information.
If you use Co-Star imaging, the MSDS will print.
Menu Path: Inventory > Maintenance > Product Control Files > MSDS Numbers
Field and Function Key Descriptions
Field/Function Key
Description
MSDS Number
The material safety data sheet number assigned by the vendor. This number identifies the safety information that you are required to send when you ship products with this MSDS number assigned in Product Maintenance.
Description One
An explanation of the material safety data sheet.
Description Two
The second line of description for the data sheet, if applicable.
Last Revision Date
The last time the MSDS number was changed. Customers who purchased the product with the MSDS before this date will receive a new MSDS with orders after this date. The revision date prints on the pick ticket.
Revision Number
The MSDS revision number, if applicable. The revision number prints on the back of the pick ticket.
Bin
The Bin Number where the MSDS is stored. The bin number prints on the pick ticket for reference.
Hazardous Matl
Indicates if the MSDS is for hazardous material:
•
Y - The MSDS refers to hazardous material.
•
N - The MSDS does not refer to hazardous material.
Flammable Matl
Indicates if the MSDS is for flammable material:
•
Y - The MSDS refers to flammable material.
•
N - The MSDS does not refer to flammable material.
Combustible Matl
Indicates if the MSDS is for combustible material:
•
Y - The MSDS refers to combustible material.
•
N - The MSDS does not refer to combustible material.
Vendor Name
The vendor/manufacturer who originally produced the material.
MSDS Sheet Images
The path to the MSDS image scanned into the system (only applicable if your company is using imaging). The system only accepts single page images so you must enter an image for each MSDS number. For example, if the MSDS sheet is 3 pages, you must scan each page of the MSDS separately so each page is stored as its own image. You can enter an unlimited number of image paths. The MSDS images print with the pick ticket.
F2-Save
Saves the MSDS information.
F4-Del
Deletes the MSDS information.
