Discount Table - Percent screen
The Discount Table - Percent Method screen appears when you enter (*) for the Table Method in Discount Break Table Maintenance. This screen allows you to specify the column and type of break level to calculate discounts.
Refer to Setting Up Discount Break Tables for more information.
Menu Path: Inventory > Maintenance > Discount Break Table Maintenance > Table Method = *
Field and Function Key Descriptions
Field/Function Key
Description
Basis Field
The price or cost column to use as the base amount to calculate discounts:
•
0 - Suggested Retail.
•
1 - Price1.
•
2 - Price2.
•
3 - Price3.
•
4 - Price4.
•
5 - Price5.
•
A - Average cost.
•
B - Product quantity break. Uses the product’s quantity break table to apply the discount defined in this table. Only used for percentage discounts.
•
C - Customer default column. Uses the customer’s default price level before applying the discount defined in this table.
•
O - Cost for sales.
•
R - Replacement cost.
•
S - Standard costs.
Break Level
Defines the minimum break value that qualifies for the corresponding discount percent or price:
•
If the Break Level Type is M, enter the unit of measure that sets this discount.
•
If the Break Level Type is U, enter the minimum quantities to determine the discount.
•
If the Break Level Type is $, enter the minimum value to determine the discount.
%
The discount percentage received when you reach the corresponding Break Level. This percentage is applied to the value in Basis to calculate the price for each product. This value displays under Disc% on the line item detail screen of the purchase order or sales order. The discount is a percentage off of a price if a price is specified in Basis or a percentage over the cost if a cost is specified in Basis. A negative percentage gives the opposite result (a percentage over a price or a percentage off of a cost).
F2-Save
Saves the break table information.
F4-Del
Deletes the break table.
