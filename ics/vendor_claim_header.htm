Vendor Claim Header screen
The Vendor Claim Header Screen allows you to designate the claim type and to enter shipping information for a vendor claim.
Menu Path: Inventory > Vendor Claims > Vendor Claim Entry > F5-Hdr Scr
Field and Function Key Descriptions
Field/Function Key
Description
Claim Type
Indicates the type of vendor claim:
•
01 - Regular vendor claim. Refers to regular or warranty inventory.
•
02 - Vendor show claim. Refer to Entering a Vendor Show Claim.
•
03 - Vendor consigned inventory. Refers to products in stock that you do not pay for until they are gone.
•
04 - Non-inventory expense. For non-inventory items (requires a G/L account number).
Ship-To
The ship-to address.
Claim Date
The date the claim was entered.
Buyer
The buyer responsible for this vendor claim. The default comes from Vendor Maintenance for this vendor.
Contact
The contact name from Vendor Maintenance.
Email Address
The email address of the contact name in Vendor Maintenance.
Fax
The number to fax the vendor claim to.
Phone
The contact’s phone number.
Ship Via
The ship via code that determines how the claim will ship to the vendor. The default is from Vendor Maintenance.
Payment Terms
The terms code for the vendor from Vendor Maintenance.
Freight Terms
The freight terms assigned to this vendor in Vendor Maintenance
Fob Description
The freight on board description.
Freight Amount
The freight cost to ship this claim to the vendor.
Weight
The total weight of the products that shipped on this vendor claim.
#/Cartons
The number of cartons that shipped.
Waybill
The waybill number assigned to the shipment.
Vendor Order
The vendor’s order number, if applicable.
Instructions
Any shipping instructions regarding this vendor claim.
F2-Save
Saves the vendor claim header information.
F5-Add Info
Opens the Vendor Claim Additional Information screen to allow you to enter the vendor’s order number and currency.
F6-Ship-To
Opens the Ship-To Address screen to allow you to verify or change the address where you will ship the material.
F7-Ship-From
Opens the Ship-From Address screen to allow you to verify or change the address of the location that is shipping the material.
F8-Freight Detail
Opens the Freight Detail Info screen to allow you to enter detail tracking information for reach carton shipped.
