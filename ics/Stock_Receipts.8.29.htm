Receiving Lot Products Not Sold in Packs
The Lot Number Information screen appears in Stock Receipts Entry when the Lot Pack flag is set to N in Product Maintenance. Refer to Setting Up a Lot Product.
1
For Lot Number, enter the lot number from the package or enter N to assign a new lot number.
2
Enter the Quantity received of this lot number.
3
Enter the Bin, if applicable.
4
For Exp-Date, enter the date the lot expires. The default depends on how the Lot Date Flag is set in the Lot Information Screen. If a Manufacture Code is established in Vendor Maintenance for this vendor, it displays in Mfg#. A Description of the manufacturer code displays.
5
Repeat steps 1-4 for each lot number and quantity received for this product. The total quantities of all the lot numbers must equal the quantity received in the Detail Information screen. If not, you must re-enter the lot number quantities or default the quantity received to the total of the lot quantities.
6
Press F2-Save to save the lot information and to receive the next item on the purchase order.
