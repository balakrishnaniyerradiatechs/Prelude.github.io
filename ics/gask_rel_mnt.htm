Gasket Related Product Maintenance Screen
The Gasket Related Product Maintenance screen allows you to enter bulk products associated with a remnant product. This screen is only applicable if you are using the Gasket/Hose Subsystem
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Special Operation = G
Field and Function Key Descriptions
Field/Function Key
Description
Product
The remnant product number.
Stock U/M
The stocking unit of measure set up for the remnant product.
Sl Per Pound
The square inches per pound, if the unit of measure is in SI (square inches).
Gasket Bulk Related Products
The gasket bulk products associated with the remnant product. Products entered here must be defined with a gasket (G) special operation code.
Description
The description of the bulk products.
Qty Per
The quantity of the bulk product per stocking unit of measure.
F2-Save
Saves the gasket-related product information,
