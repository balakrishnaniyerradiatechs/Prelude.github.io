Production Control Parameters screen
The Production Control Parameter screen allows you to enter product manufacturing information for work orders.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Purchasing Parameters Maintenance > Replenishment Mtd = M
Field and Function Key Descriptions
Field/Function Key
Description
Planner
The buyer number of the planner responsible for producing this item.
Production Multiple
The quantity of the product to manufacture.
Production Minimum Months.
The minimum number of months usage to manufacture at one time.
Production Group
The production group that determines the instructions to print on the work order. Refer to Production Control Groups.
F2-Save
Saves the production control parameters.
