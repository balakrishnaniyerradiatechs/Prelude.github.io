Printing Core Receipt Tickets
You can print a core receipt ticket if a customer wants a receipt for a returned core item. If core receipt tickets are not set up to print automatically when you adjust inventory (refer to Receiving Cores in Inventory Adjustment Entry), you can print the receipt manually in Core Receipt Ticket Printing.
Automatic Core Receipt Ticket Printing
Set Print Core Receiving Tkt in Inventory Parameters to Y if you want to print a core receipt ticket for the customer automatically when inventory is adjusted.
To print a core receipt ticket:
1
Select Inventory > Processes > Print Core Receipt Ticket to open the Core Receipt Ticket Printing screen.
2
Enter the Adjustment Number that contains the core item.
3
Enter the Warehouse where the inventory was adjusted.
4
Enter the Printer Form Queue to designate where the ticket prints.
Only core products print on the core receiving ticket.
