Building the Physical Inventory Lists
Depending on your counting procedures and which products you want to include in your physical inventory count, you can create count sheets in any sequence or by any grouping to accommodate your warehouse layout.
Choose one of the following options to create inventory lists:
•
Creating an Inventory List by Bin Location.
•
Creating an Inventory List by Product Line
•
Creating an Inventory List by Vendor.
•
Creating an Inventory List by Product Number.
•
Creating an Inventory List by Range.
•
Creating a User-Defined Inventory List.
•
Creating an Inventory List Using a TCL Command.
•
Creating an Inventory List for All Products.
These lists exclude asterisk (*) products, serial products, rental products, belting products, service products, and products containing a Fill With Product.
You can re-use lists for future counts; but if you add any new products after a list is generated, you must create the list again.
