Printing Count Sheets
The first step in cycle counting is to print a list of products to count. You can print count sheets by defining the zone or bin location for the desired products.
To print count sheets:
1
Select Inventory > Physical Inventory > Cycle Counting Procedure > Paper Site Count Sheet Printing. The Cycle Count Sheet Printing screen appears.
2
Enter the Division. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
3
Enter the Region. A region is a subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
4
Enter the Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
5
Enter the Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
6
Enter the Warehouse where the products are being counted.
7
For Counter Id, enter the user ID of the person who will count the products in this area. This user must be set up as a cycle counter in User Identification Maintenance.
8
Enter the Zone for the products to count.
9
For # of Bins, enter the maximum number of bins to count for this session.
10
For Bin Page Break, indicate if a separate count sheet should print for each bin. Choose from the following:
â€¢
Y - A new page prints for each bin.
â€¢
N - Bins print in order with no page breaks.
11
For Print Bin Onhand, indicate if you want the bin on-hand quantity to print on the count sheets:
â€¢
Y - Bin on-hand quantities print on the count sheets.
â€¢
N - Bin on-hand quantities do not print on the count sheets.
12
For Print â€˜Invâ€™ Bin, indicate if you want the on-hand quantity for the Inv bin to print on the count sheets. The Inv bin is the bin a product moves to when it is confirmed as shipped but not yet invoiced. This option is only available if you have the Warehousing Module.
â€¢
Y - Print the quantity in the Inv bin for each product on the count sheets.
â€¢
N - Do not print the Inv bin quantity.
13
Press F2-Save to print the count sheets.
The count sheets print the product numbers in alphanumeric order. Depending on the selection criteria, they may include lot numbers, bins, on-hand quantities, and quantities in the Inv bin quantities. A space is provided to allow you to enter the counts.
