Parker Division/Marketing Maintenance screen
The Parker Division/Marketing Maintenance screen allows you to assign Parker marketing codes for each division on the Parker report. Refer to Parker Reporting Codes for more information.
Menu Path: Inventory > Maintenance > Product Control Files > Parker Reporting Codes > Parker Division/Marketing Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Parker Division Number
The Parker Division number established in the Parker Division Maintenance screen. Refer to Setting Up Parker Divisions.
Cde
The 2-digit Parker Marketing code for this division.
Description
A description that identifies the products for the Marketing code.
F2-Save
Saves the marketing code.
F4-Del
Deletes the marketing code.
