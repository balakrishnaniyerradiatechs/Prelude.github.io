Lot Additional Info screen
The Lot Additional Info screen allows you to fill in default package information to display when you enter a lot product on a purchase order.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > Lot Number Flag, set to Y > Lot Pack Flag set to Y
Field and Function Key Descriptions
Field/Function Key
Description
Description
The description of the package (e.g., pack, box, case). For information purposes only.
Std Lot Pack
The quantity in each lot pack. This number is used for the default on purchase orders.
Lot Pack Per Pallet
The number of packs usually bought at a time for this product. For example, one box of screws contains 100 screws. Screws are normally bought by the case, which contain 10 boxes. Therefore, the number of packs is 10.
Tariff Commodity Code
The Tariff Commodity Codes used to incorporate harmonized tariffs into duty calculations for material received from overseas vendors.
Competitor Price
The competitor’s price if you are Tracking Competitor Prices.
Split Lot Pack
Indicates if you can split a lot pack in Sales Order Entry. Choose from the following:
•
Y - Allow lot packs to be split on a sales order. When the Lot Pack Availability screen prompts for the lot pack(s) to commit to the sales order, press F5-Split Lot Pack to sell the product in quantities less than the lot pack quantity set in Product Maintenance. The remaining quantity is assigned to a new lot number.
•
N - Do not allow lot packs to be split on a sales order. When the Lot Pack Availability screen prompts for the lot pack(s) to commit to the sales order, you can only sell the product in multiples of the lot pack quantities.
For example, if all the available lots have a lot pack quantity of 10, you can only sell multiples of 10. If multiple lots of 5 and 10 exist, you can enter a sales order for 10, 15, 20, or 25, etc., but not for a quantity of 12.
F2-Save
Saves the lot pack information and to returns to the Lot Information screen.
