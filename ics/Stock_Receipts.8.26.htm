Setting Up a Lot Product
If you want to make sure that lot numbers are assigned to products before they are part of inventory, you can set up products to requires a lot number when received.
Option Parameters
The following flags in Additional Option Parameters apply to lot numbers:
Use Whse Over Lot Cost determines which cost to use for lot numbers in the general ledger. Choose from the following:
•
Y - Use warehouse cost for general ledger entries
•
N - Use average lot cost for general ledger entries.
Disallow Same Lot# For Diff P/N’s determines if the same lot number is allowed for 2 part numbers. Choose from the following:
•
Y - You cannot use the same lot number for different product numbers.
•
N - You can use the same lot number for different product numbers.
You can override this setting for sales orders in the Disallow Same Lot# For Diff P/N’s field in Order Processing Parameters > F5-Add Info.
To set up a product to require a lot number in Stock Receipts Entry:
1
Select Inventory > Maintenance > Product Maintenance > Product Maintenance.
2
Enter the Product Number and press F5-Add Info.
3
For Lot Number Flag, enter Y. The Lot Information Screen appears.
4
For Lot Exp Date Flag, choose from the following:
•
Y - Expiration dates are required for each lot in stock receipts. You can review product expiration dates in the Lot Shelf Life Expiration Report.
•
M - Date of manufacture is required for each lot in stock receipts. In this case, the date of manufacture plus the Shelf Life Days in the Product Maintenance Additional Information Screen determines the expiration date. For example, if the date of manufacture is May 1, and the shelf life days is 90; the expiration date is August 1.
•
N - Expiration dates are not required.
5
For Minimum Remnant Quantity, you can enter the minimum quantity that is considered suitable to sell if you want to exclude small remnants from availability calculations.
Example: Pipes are usually cut to meet a customer’s request. Once the pipe is cut down to about 2 feet, it may not be a commonly used size. The pipe is still in stock and affects the purchasing of new pipes. This flag allows Nightly Processing to ignore the 2-foot pipe (as if it didn’t exist) to recommend a purchase quantity.
6
For Lot Priority Code, choose from the following to determine which lots are picked first:
•
N - No order.
•
S - Smallest lot that fills the requirement.
•
O - Oldest lot first.
7
For Lot Pack Flag, indicate if you want to prompt for Lot Pack Quantity and Number of Lot Packs in Purchase Order Entry. Choose from the following:
•
N - The product is not sold in a pack. Go to step 15.
•
Y - The product is sold in a pack. The Lot Additional Info screen opens to allow you to fill in default package information to display when you enter the lot product on a purchase order.
8
Enter the Description of the package (e.g., pack, box, case). For information purposes only.
9
For Std Lot Pack, enter the quantity in each lot pack. This number is the default on purchase orders.
10
For Lot Pack Per Pallet, enter the number of packs usually bought at a time for this product. This number is used for the default on purchase orders. For example, one box of screws contains 100 screws. Screws are normally bought by the case, which contain 10 boxes. Therefore, the number of packs is 10.
11
Enter the Tariff Commodity Code if this is an overseas vendor. Refer to Tariff Commodity Codes for more information.
12
Enter the Competitor Price for the lot item if you track competitor prices. (Refer to Tracking Competitor Prices).
13
For Split Lot Pack, indicate if you can split a lot pack in Sales Order Entry. Choose from the following:
•
Y - Allow lot packs to be split on a sales order. When the Lot Pack Availability screen prompts for the lot pack(s) to commit to the sales order, press F5-Split Lot Pack to sell the product in quantities less than the lot pack quantity set in Product Maintenance. The remaining quantity is assigned a new lot number.
•
N - Do not allow lot packs to be split on a sales order. When the Lot Pack Availability screen prompts for the lot pack(s) to commit to the sales order, you can only sell the product in multiples of the lot pack quantities.
For example, if all the available lots have a lot pack quantity of 10, you can only sell multiples of 10. If multiple lots of 5 and 10 exist, you can enter a sales order for 10, 15, 20, or 25, etc, but not for a quantity of 12.
14
Press F2-Save to save the lot pack information and to return to the Lot Information screen.
15
For Price/Cost Required, choose from the following:
•
Y - Prompt for commission cost and selling price when receiving lot products.
•
N - Do not prompt for commission cost and selling price when receiving lot products.
16
Press F2-Save to save the lot information.
