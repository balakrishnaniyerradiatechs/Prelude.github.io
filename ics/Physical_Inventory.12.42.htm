Physical Inventory Valuation Report
The Physical Inventory Valuation Report displays the total value of inventory for a warehouse after you enter and update the physical counts (refer to Updating Physical Inventory Counts). The count for each product is multiplied by the cost of the product at the time of the update.
The new count for each product is saved and is available until the next time you update inventory. If no count was entered for a part number, the on-hand amount becomes the count quantity.
Menu Path: Inventory > Physical Inventory > Inventory Tag Procedure Physical Inventory Valuation Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse for which you want to display the physical inventory value.
•
Exclude Class 13?. Determines if class 13 products (dead inventory) are excluded from the report. Choose from the following:
-
Y - Exclude class 13 products.
-
N - Include class 13 products.
•
Rental Products?. Determines if rental products are included on the report. Choose from the following:
-
B - Include rental products with all other products.
-
Y - Only include rental products. Exclude all other products.
-
N - Exclude rental products.
•
Sort ‘V’endor/’P’roduct Line. Determines the sort sequence for the report. Choose from the following:
-
V - Sort by vendor.
-
P - Sort by product line.
Column Descriptions
Column Heading
Description
Product Number
The product numbers included in the total inventory valuation.
Description
Line 1: The product description.
Line 2: The second line of description for the product.
U/M
The unit of measure for the product.
Phys-dte
The date the Physical Update Procedure updated the counts into inventory.
Count
The count entered from the physical inventory count sheets.
LinkOH
The quantity linked to sales orders, work orders, or transfers.
LinkValue
The dollar amount of the LinkOH quantity.
StockOh
The updated on-hand quantity.
Cost
The cost of the item at the time of the Physical Update Procedure.
Ext-Value
The extended value of the total quantity of the product (StockOh x Cost).
