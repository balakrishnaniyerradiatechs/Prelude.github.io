Requesting Additional Cycle Counts
In addition to the recounts generated from the first level count entry (refer to Entering 1st Level RF Cycle Counts) you can also manually add a product, product line, vendor, or cycle count zone to the second level count queue. Refer to Cycle Count Second Level Report.
To manually request recounts:
1
Select Inventory > Physical Inventory > Cycle Counting Procedure > Product Cycle Count Request Entry to open the Cycle Count Request Entry screen.
2
For Request Type, choose from the following methods to select products to add to the second level count queue. :
•
B - Bin location. Opens the Cycle Count Request Entry By Bin screen to allow you to select a bin number to include in the second level count. If a product has no primary bin assigned, it is not included.
•
C - Cycle count zone. Opens the Cycle Count Request Entry by Cycle Count Zone screen to allow you to designate a cycle count zone to include in the second level count queue.
•
P - Product. Opens the Cycle Count Request Entry by Product screen to allow you to enter an individual product number to send to the second level queue.
•
L - Product line. Opens the Cycle Count Request Entry by Product Line screen to allow you to designate a product line to include.
•
M - Manual selection. Opens the Manual Product Cycle Count Request Options screen to allow you to select products by product number range, bin number range, status, or sales rank.
•
V - Primary vendor. Opens the Cycle Count Request by Primary Vendor screen to allow you to select a primary vendor whose products you want to include in the second level count.
3
Enter the Warehouse for the request.
4
Press F2-Continue. Depending on the Request Type entered in step 2, enter the required information to add the products to the second level count queue.
The selected products appear on the Cycle Count Second Level Report.
