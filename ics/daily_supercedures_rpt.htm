Daily Supercedures Report
The Daily Supercedures report displays all transactions containing superceded products. A superceded product is a product that is replaced with another product, usually by the vendor (e.g., ABC is replaced by ABC-1). You can set up a new (superceding) part number using the usage from the old (superceded) part number. Refer to Supercedure Products for information on setting up and using supercedures.
After the report runs, a dialog box appears to allow you to purge the product information so that it does not appear the next time you run the report.
Menu Path: Inventory > Reports > Miscellaneous Reports > Daily Supercedures Report

Column Descriptions
Column Heading
Description
Superceded Part#
The part number that was replaced by a new part number.
Superceding Part#
The new part number that replaces the old (superceded) part number.
Port
The port number for the terminal where the supercedure was entered.
Related Sales Orders
Ord#
The sales order number containing the superceded product.
Tran#
The linked transfer number, if applicable.
PO#
The linked purchase order number, if applicable.
Qty
The quantity of the superceded product entered on the sales order.
Cust#
The number for the customer that purchased the superceded product.
Name
The customer description.
Related Transfers
Trans#
The transfer number containing the superceded product.
Qty
The quantity of the superceded product transferred.
To-Whse
The warehouse that shipped the superceded product.
From-Whse
The warehouse that received the superceded product.
Related Purchase Orders
PO#
The purchase order number containing the superceded product.
Qty
The quantity of the superceded product purchased.
Vend#
The vendor number for the purchase order.
Vendor
The vendor description.
