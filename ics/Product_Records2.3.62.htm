Using Equipment Option Codes
If you stock serialized equipment that comes with optional features (e.g., alarm, heater, clock), you can set up option codes to identify each feature. You can use option groups to identify the type of equipment (e.g., tractor, backhoe) that each option applies to. When you purchase the piece of equipment, you are prompted to add the appropriate option codes that are available for this type of equipment. The codes are added as comments on the purchase order. When you receive the purchase order, you can confirm the equipment options and can change them as necessary. The option information is stored in the product serial number record.
To use equipment options, you must:
1
Set up option codes in Equipment Option Code Maintenance. Refer to Setting up Equipment Option Codes.
2
Assign the option codes to groups in Equipment Option Group Maintenance. Refer to Setting up Equipment Option Groups.
3
Assign the option groups to products in the Option Group field in the Product Maintenance Additional Information Screen.
