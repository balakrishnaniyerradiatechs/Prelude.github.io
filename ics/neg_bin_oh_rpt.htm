Negative Bin Onhand Report
The Negative Bin Onhand report displays all bin locations with an on-hand quantity of less than zero. You can run this report regularly (e.g., once a week) to detect and correct any inventory discrepancies. All on-hand balances should have a quantity of zero or above so that they will not appear on this report.
Menu Path: Inventory > Reports > Stock Status Reports > Negative Bin Onhand Bin
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse(s) containing negative bin on-hand balances.
•
Print All Product Bins: Determines which bins to print. Choose from the following:
-
Y - If a product has a negative quantity, print all bins that contain the product.
-
N - If a product has a negative quantity, only print the bins with a negative on-hand quantity.
Column Descriptions
Column Heading
Description
Product Number
The product number.
Description
The product description.
Bin
The product’s bin location.
Onhand
The on-hand quantity of the product in this bin.
