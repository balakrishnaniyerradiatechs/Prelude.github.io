Physical Inventory Serial Number Report
If serialized products are not included in your regular inventory count sheets, you can print separate count sheets for all products that contain serial numbers. If the count quantity does not equal the book on-hand quantity, you must manually adjust inventory to correct the on-hand balance. You can run this report at any time during the physical inventory process.
Serialized Products
You can only print count sheets for serialized products if the Phys. Inv. Include Serial# flag in Inventory Parameters > F5-Add Info is set to N, in which case serialized products are not included in your physical inventory count process.
Menu Path: Inventory > Physical Inventory > Inventory Tag Procedure > Physical Inventory Serial Number Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse containing the serial numbers.
•
Show Onhand Qty?. Determines if the on-hand quantity is displayed. Choose from the following:
-
Y - Show on-hand quantities.
-
N - Suppress on-hand quantities.
Column Descriptions
Column Heading
Description
Product Number
The product number that is serial number controlled.
Description
Line 1: The product description.
Line 2: The second line of description for the product.
U/M
The unit of measure for the product.
On-Hand
The saved on-hand quantity for the product (including all serial numbers) at the time the report is run.
Phys-Count
A blank space to enter the count.
Serial Number
The serial number of the product. Write in any additional serial numbers found or mark off any listed serial numbers not found. Adjust inventory to correct the serial number information.
Bin Location
The bin location for the product, if applicable.
