Transfer Putaway Document Report
You can print the Transfer Putaway Document report after receiving a transfer to list the products received. The report shows bin locations and/or linkages to help determine where to put the material.
After you print the report, a dialog box appears to allow you to purge the received product information so that it does not print the next time you run the report.
Menu Path: Inventory > Reports > Stock Movements Reports > Transfer Putaway Document Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse that received the transfer.
•
Received By: The user ID of the person that received the transfer.
•
Sort By: The sort order of the report. Choose from the following:
-
P - Sort by product number.
-
B - Sort by primary bin.
Column Descriptions
Column Heading
Description
Product Number
The product number as it is entered on the transfer.
Description
Line 1: The product description.
Line 2: The second line of description for the product, if applicable.
Document#
The transfer receiver number.
Linkage
The document number linked to the product. The prefix indicates the order type:
•
S - Sales order.
•
W - Work order.
•
T - Transfer.
Qty-Rec
The quantity received for the corresponding receiver number.
U/M
The unit of measure for the product as entered on the transfer order.
Stk-Qty
The quantity to put into stock.
Prim/Sec Bin
Line 1: The primary bin location for the product.
Line 2: The secondary bin location for the product.
