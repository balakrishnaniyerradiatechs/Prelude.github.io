Purchasing Controls Screen
The Purchasing Controls screen allows you to establish a default purchasing formula for a product for all warehouses where it is stocked.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Purchasing Controls
Field & Function Key Descriptions
Field/Function Key
Description
Default Formula Method
The default formula to calculate the product’s forecast usage for all warehouses:
•
AWU - Average Weekly Usage. (Refer to Tracking Average Weekly Usage). You can either update all warehouses or only warehouses with the same formula.
•
CLS - Classification.
•
DRP - Distribution Requirements Planning.
•
DS - Days Supply.
•
EOQ - Economic Order Quantity.
If you change this formula, you must also choose from the following:
•
All Whses. Updates all warehouses.
•
Whses With Same Meth. Updates only warehouses currently using this Default Formula Method.
•
No Changes.
You can override this setting in the Product Purchasing Parameters Maintenance Screen.
Vendor
The vendor automatically assigned to this product. You can add vendor/vendor part numbers to use for product searches.
Vendor Product
The vendor part number as established in the Product Vendor Maintenance Screen, if applicable. If you override this number, it will not affect the Product Vendor Maintenance record.
F2-Save
Saves the purchasing control information.
F5-Recalc Pur-chase Parameters
Recalculates the purchasing parameters according to the new Default Formula Method.
