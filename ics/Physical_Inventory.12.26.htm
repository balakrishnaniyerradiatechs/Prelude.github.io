Taking Physical Inventory Using Tags
The tag inventory procedure creates a tag for each product for its primary bin. If a product exists in more than one bin, a tag also prints for any additional bins that contain a quantity of the product. As you count each product, you can pull off the tag and write in the count. As you count each area, you can accumulate the tags, put them in numerical sequence, and enter the counts into the system. You can also create manual tags for any product/bin combination without a tag.
