Requirements for OTW Receiving
You must establish the following system options to create and receive OTW purchase orders.
Additional Option Parameters
To activate OTW Receipts Maintenance, set On The Water Conversion to P.
Purchasing Controls > Purchase Order Entry Types
Purchase order entry types 12 OTW-Stock and 13 OTW-Direct are required to create OTW purchase orders.
Import Cost Factor Maintenance
To estimate import costs on shipments from overseas vendors (e.g., duty, brokerage fees, freight), you can set up import cost factors to apply to each vendor. Import Cost Factor Maintenance allows you to maintain your own cost factors to satisfy unique shipping costs incurred when you import merchandise. You can calculate cost factors based on dollars, pounds, or units. Refer to “Maintaining Import Cost Factors” in the Purchasing documentation for information on creating import cost factors.
You can assign these import factors to each overseas vendor in Vendor Maintenance. If necessary, the buyer can override the import factors for a vendor in Purchase Order Entry > F5-Hdr Scr > F5-Add Info > F5-Import Factors.
Vendor Maintenance
To set up a vendor with default import codes to apply when you enter a purchase order:
1
Select Purchasing > Maintenance > Vendor Maintenance.
2
Enter the Vendor.
3
Press F5-Add Info to open the Additional Information screen.
4
Press F7-Importing to open the Import Information screen.
5
Enter P in Overseas Vendor so that OTW purchase orders (types 12 and 13) are allowed for this vendor.
6
For Country, enter the country code that represents where the product is manufactured.
7
For Import Code, enter the code representing the charges incurred when purchasing from this vendor. Refer to the Purchasing documentation for information on maintaining import cost factors. You can override this code in the Purchase Order Header screen. The following are examples of commonly used codes:
•
BRK - Overseas brokerage (dollars).
•
CONS - Overseas consolidation (units).
•
DUTY - Overseas duty (dollars).
•
IF - Inland freight (pounds).
•
LF - Overseas loading fee (pounds).
Duty
If a product is set up with a Commodity Code to calculate duty in the Additional Product Maintenance Information Screen, it will override the duty calculated in Vendor Maintenance.
8
Enter the Currency code to indicate the type of currency this vendor uses.
9
If applicable, enter a Factor to override the factor for the import code as established in Import Cost Factor Maintenance. The factor is multiplied by the cost of the product to calculate the corresponding charge.
10
Press F2-Save 3 times to save the import information.
