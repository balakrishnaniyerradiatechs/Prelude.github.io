Balancing Inventory
To make sure inventory is balanced before you start the physical inventory, you must run the Physical Data Integrity Process to clear out old data to put inventory back in balance.
To balance inventory:
1
Select Inventory > Physical Inventory > Inventory Tag Procedure > Physical Data Integrity Process.
2
Enter the Warehouse to balance.
3
Press ENTER.
