Warranty Replacements History Inquiry
The Warranty Replacement History Inquiry displays completed warranty replacement entries for a vendor.
Viewing Historical Warranty Replacements
To view completed warranty replacement entries:
1
Select Inventory > Warranty Replacements > Warranty Replacement History Inquiry to open the Warranty Replacement History Inquiry screen.
2
Enter the Vendor for which you want to view updated warranty replacements.
Updated warranty replacement entries for the vendor display.

F2-Finish
Closes the inquiry screen.
F5-Review W/R
Opens the Warranty Replacement History screen to allow you to view the original entry information. You can view the replaced products on the Warranty Replaced History Review screen and the replacing products on the Warranty Replacing History Review screen.
F6-Enter W/R
Allows you to enter a warranty entry number if several pages of warranty replacements are listed.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
