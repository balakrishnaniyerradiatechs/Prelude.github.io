Putaway Documents Report
The Putaway Documents Report displays bin location and/or linked sales order information for received products. You can run this report after you receive one or more purchase orders to help your warehouse personnel determine where to put away stock.
After you print the report, a dialog box appears to allow you to purge the received product information so that it does not print the next time you run the report.
Putaway Document in File Update Parameters Maintenance determines if putaway documents print automatically after each receiver is saved and if the information is tracked.
Choose from the following:
•
Y - Prints the Putaway Documents Report automatically after receiving stock.
•
A - Prints the Putaway Documents Report after receiving stock and keeps an audit trail of the allocated material.
•
N - Does not print the Putaway Documents Report after receiving stock.
Menu Path: Inventory > Reports > Stock Receipts Reports > Putaway Documents Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
•
Selling Warehouse. The warehouse that received the material.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Document Receiver Number. The receiver number.
•
Received By: The user ID of the person that received the order.
•
Sort By: The sort order of the report. Choose from the following:
-
P - Sort by product number.
-
B - Sort by primary bin.
Column Descriptions
Column Heading
Description
Product
The product number as it is entered on the purchase order.
Description
Line 1: The product description.
Line 2: The second line of description for the product, if applicable.
Document
The receiver number.
Linkage
The document number linked to the product. The prefix indicates the order type:
•
S - Sales order. The customer description displays on the next line.
•
W - Work order.
•
T - Transfer. The receiving warehouse description displays on the next line.
Qty-Rec
The quantity received for the corresponding receiver number.
U/M
The unit of measure for the product as entered on the purchase order.
Stk-Qty
The quantity to go into stock based on the stocking unit of measure. For example, if the quantity received was 10 and the stocking unit of measure is DZ (dozen); the quantity received is 10 and the quantity in stock is 120. Refer to Establishing a Unit of Measure for more information on unit of measure.
Primary Bin
The primary bin location for the product. If a bin other than the primary bin was entered when the item was received, it displays above the primary bin number.
