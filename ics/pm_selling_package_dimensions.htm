Selling Package Dimensions Screen
The Selling Package Dimensions screen allows you to enter the measurements of a productâ€™s selling package. When you enter the product on a sales order, the selling package dimensions determine if the product is compatible with the ship via method on the sales order.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > Action Bar > Sell Pk Dim
Field & Function Key Descriptions
Field/Function Key
Description
Selling Package Length
The length of the product when packaged for sale.
Selling Package Width
The width of the product when packaged for sale.
Selling Package Depth
The depth of the product when packaged for sale.
Linear Measure Description
The linear measurement description of the product when packaged for sale (e.g., feet, inches, yards, meters).
F2-Save
Saves the package dimension information.
