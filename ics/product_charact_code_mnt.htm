Product Characteristics Code Maintenance screen
The Product Characteristics Code Maintenance screen allows you to create codes to identify the characteristic information required when a product is sold. After you set up the code, you can assign it to a product or multiple products in Product Maintenance.
Refer to Product Characteristics for more information.
Menu Path: Inventory > Maintenance > Product Control Files > Product Characteristics Codes
Field and Function Key Descriptions
Field/Function Key
Description
Characteristics Code
The code to assign for the characteristic information. For example, if you create a code called WTY and assign it to a product, you must enter warranty information each time you sell the product.
Description
The description of the product characteristics code (e.g., warranty information).
Style Template
Determines how product characteristics appear on a sales order when you sell a product with this code:
•
1 - The Product Characteristic Information screen appears and requires you to enter an additional description of the product.
•
2 - The product characteristic information this customer/contact/product combination used last.
•
3 - A list of previous sales orders for the customer/product combination displays to allow you to select product characteristics from a previous order.
Service Product
The service product number to add to the sales order when a new product characteristic is created. This field is only accessible if you choose 3 for Style Template.
Question Template
Description
The types of information to request when an item with this characteristic code is sold. For example, for warranty information you may request a serial number, model number, and/or expiration date.
Mandatory
Indicates if the type of information requested for each Description is mandatory when an item with this product characteristic is sold:
•
Y - Requires the corresponding Description information to sell the product. For example, if you enter Serial # for Description and Y for Mandatory, then you must enter a serial number any time you sell a product with this characteristics code.
•
N - Does not require the Description information to sell the product.
History
Indicates if the product characteristic information is available on future sales orders:
•
Y - The product characteristic information is available for future orders.
•
N - The product characteristic information is not available for future orders. You must re-enter the product characteristic information each time the customer buys the product.
Add Info
Indicates if validation information is entered in the Product Characteristics Additional Information screen.
F2-Save
Saves the product characteristics code information.
F4-Del
Deletes the product characteristic code.
F5-Add Info
Opens the Product Characteristics Additional Information screen to allow you to enter validation information for each of the characteristic Descriptions.
Action Bar
Copy
Opens the Product Characteristics Maintenance Copy screen to allow you to copy information from an existing product code to create a new product characteristics code. Refer to Copying a Product Characteristic Code.
