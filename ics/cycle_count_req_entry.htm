Cycle Count Request Entry screen
The Cycle Count Request Entry screen allows you to enter a bin location, zone, product, product line, or vendor to send to the second level count queue for cycle counting.
Menu Path: Inventory > Physical Inventory > Cycle Counting Procedure > Product Cycle Count Request Entry.
Field and Function Key Descriptions
Field/Function Key
Description
Request Type
Indicates how to select products to add to the second level count queue:
•
B - Bin location.
•
C - Cycle count zone.
•
P - Product.
•
L - Product line.
•
M - Manual selection.
•
V - Primary vendor.
Warehouse
The warehouse for the cycle count.
F2-Continue
Depending on the Request Type, you can enter the Bin Location, Zone, Product, Product Line, or Vendor to add to the second level count queue. If you entered M for manual selection, you can enter a range of products or bin locations. The selected products appear on the Cycle Count Second Level Report.
