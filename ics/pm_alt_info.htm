Alternates Information screen
The Alternates Information screen allows you to view and/or set up alternate part numbers (e.g., catalog numbers, UPCs, UCCs) which can be used as an additional method to enter a product.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F6-Xref/Notes > Alternates
Field and Function Key Descriptions
Field/Function Key
Description
Alternates
The alternate part number, catalog number, UPC, or UCC code associated with the Product Number. If the alternate is a UPC, enter N for the next system assigned UPC.
UPC
Indicates if the alternate part number is a UPC (Universal Product Code):
•
Y - The alternate part number is a UPC.
•
N - The alternate part number is not a UPC.
UCC
Indicates if the part number is a UCC (Uniform Code Council) code:
•
N - The alternate part number is not a UCC code.
•
Y - The alternate part number is a UCC code.
•
M - The alternate part number is for a master pack UCC code.
•
P - The alternate part number is for a pallet pack UCC code.
•
S - The alternate part number is for a UCC code shelf pack.
UCC Qty
The quantity to use as the default to print labels in Serial Ship Container Entry.
UCC U/M
The UCC unit of measure, if applicable.
Catalog Id
The code for the catalog that contains this product, if applicable. Catalog IDs are set up in Catalog Identification Code Maintenance.
Atl Ppg
The product price group for the alternate part number. Product price groups are used to group products for pricing matrices. This field defaults to the product price group for the catalog identification code entered in Catalog Id.
F2-Save
Saves the alternate product information.
