Stock Transfer Log Report
The Stock Transfer Log report lists all transactions that affected a transfer. This report contains the same information as the Transfer Log Inquiry.
Menu Path: Inventory > Reports > Stock Movement Reports > Stock Transfer Log Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
â€¢
Transfer number. The transfer for which you want to view activity.
Column Descriptions
Column Heading
Description
Transfer
The transfer number.
From
The shipping warehouse.
To
The receiving warehouse.
Code
The transfer log inquiry code. Refer to the Transfer Log Inquiry for an explanation of each code.
Act Date
The date the action occurred.
Time
The time the action occurred.
User
The user ID of the person who performed the transaction.
Prt
The port ID for the SB client session number that the user was logged into when the transfer was entered.
Reference Number
The document number or product number for the corresponding action. Refer to the Transfer Log Inquiry for an explanation of the reference numbers.
