Consignment Sources screen
The Consignment Sources screen allows you to view open consignment (type 07 and 08) sales orders. The products are on loan to the customer and will go back into on-hand inventory when the sales order is invoiced.
Menu Path: Inventory > Inquiries > Inventory Inquiry > Consignments (under Sources) > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
Order#
The consignment sales order number.
N
An asterisk (*) indicates that notes exist for the product on this order.
Customer
The customer number assigned to the order.
Description
The customer description.
Req-Date
The date the customer requires the order.
Qty-Opn
The quantity not yet shipped.
U/M
The unit of measure as entered on the consignment order.
Qty-Shp
The quantity updated as shipped.
Qty-Stk
The quantity to be updated into on-hand inventory for each line item.
S
The supply code.
C
The kit component supply code.
Linkage
The purchase order number if the item is linked to a purchase order.
Order Type
Consignment order type:
•
Regular consignment (sales order type 07).
•
Confirm consignment (sales order type 08).
F2-Finish
Closes the screen and returns to the Inventory Inquiry screen.
F5-Review Order
Opens the Consignment Order Inquiry screen to display detail information on the order.
F7-Log
Opens the Consignment Log Inquiry to display the transactions for the consignment order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
