Parker Division Maintenance screen
The Parker Division Maintenance screen allows you to assign division codes to identify the distributor divisions defined by Parker Hannifin. Refer to Parker Reporting Codes for more information.
Menu Path: Inventory > Maintenance > Product Control Files > Parker Reporting Codes > Parker Division Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Parker Division Code
The code that represents a division or location defined by Parker Hannifin.
Description
The description of the Parker division (e.g., southwest, northeast).
F2-Save
Saves the parker division code information.
F4-Del
Deletes the parker division code.
