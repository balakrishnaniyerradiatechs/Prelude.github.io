Multiple Bin Inquiry
The Multiple Bin Inquiry allows you to view all the bins where a product number exists and includes the on-hand balance and min/max settings for each bin.
Inventory Parameters
Multiple Bin Locations in Inventory Parameters must be set to Y to allow a product to exist in multiple bins.
Finding Products in Multiple Bins
To view all bins that contain a product:
1
Select Inventory > Inquiries > Miscellaneous Inquiries > Multiple Bin Inquiry to open the Multiple Bin Inquiry screen.
2
Enter the Product. The product description displays.
3
Enter the Warehouse number.
The bin location and availability information for the product displays.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product for which you want to see multiple bin locations. The product description displays.
Warehouse
The warehouse number. The warehouse description displays.
Bin List
The bins where the product is kept in the warehouse.
Onhand
The quantity on-hand in each bin. Including quantities committed to sales orders, work orders, and transfers.
Reference#
Indicates the type of transaction that moved the on-hand quantity into the bin:
•
ADJ - Inventory adjustment.
•
BMV - Bin move.
•
CONV - Inventory conversion.
•
PWM - Product Warehouse Maintenance.
•
REC - Stock receipt
•
Receiver number, if the product is still on the receiving dock.
Date
The date the quantity was received into the bin.
Min Qty
The minimum quantity to stock in this bin (for bin replenishment in the Warehousing module).
Max Qty
The maximum quantity to stock in this bin (for bin replenishment in the Warehousing module).
OK
Closes the inquiry screen.
