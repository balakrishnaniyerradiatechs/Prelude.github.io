Import Cost Inquiry
The Import Cost Inquiry displays estimated and actual import costs for a ship or an on-the-water purchase order.
Viewing Import Costs
To view import costs for an overseas shipment:
1
Select Inventory > Inquiries > Miscellaneous Inquiries > Import Cost Inquiry to open the Import Cost Inquiry screen.
2
For Ship/Ref# enter ship number (if using the shipping bins) or the reference number (if using OTW purchase orders). Refer to Receiving On-The-Water Merchandise.
The import costs associated with the shipment display.
Field & Function Key Descriptions
Field/Function Key
Description
Ship/Ref #
The ship number (if using the shipping bins) or the reference number (if using OTW purchase orders). Refer to Receiving On-The-Water Merchandise.
Description
The description of the ship.
Merchandise
The total dollar amount of the merchandise that shipped from the vendor.
Import Codes
The import code that references the type of import charge from the overseas vendor. These codes are set up in Import Cost Factor Maintenance and then set up as a default import codes in Vendor Maintenance.
Description
The description of the import code.
Expected
The estimated value of the corresponding import code, calculated from the product weight and the vendor’s import codes.
Actual
The actual import cost for each import code (updated in A/P Manual Invoice Entry).
Variance
The difference between the Expected and the Actual import cost.
Totals
The total amount of each of the cost columns.
OK
Closes the screen.
F5-Review Receivers
Opens the Receiver Detail screen to display the import cost for each receiver.
F6-Review Invoices
Displays the invoice information, if the receiver was reconciled with the vendor invoice.
