Setting Up Discount Break Tables
Discount break tables define the quantity discount levels to determine product prices and costs. You can use discount break tables for products in one warehouse or for the whole company. Each price or cost is expressed as a set value or as a discount off of a column price. Once a table is set up, you can reference it in Product Maintenance, Product Warehouse Maintenance, Product Vendor Maintenance, Vendor Line Maintenance, or Price Matrix Maintenance to calculate prices and costs.
Discount Break Table Listings
To print a listing of discount break tables, select Inventory > Listings > Miscellaneous Listings > Discount Break Table Listing then choose one of the following:
•
Discount Percent Method.
•
Discount Price Method.
To define discount break tables:
1
Select Inventory > Maintenance > Discount Break Table Maintenance to access the Discount Break Table Maintenance screen.
2
Enter a Table Number to identify this set of rules to determine discounts. This is the number you will reference in Product Maintenance, Product Warehouse Maintenance, or Vendor Maintenance to generate prices and costs.
3
Enter a Warehouse if this table only applies to one warehouse. Leave this field blank if the discount break table applies to all warehouses.
4
Enter a Description of the discount break table so that you can easily identify it when applying it to products.
5
For Sum And Recost?, choose from the following:
•
Yes - Accumulate purchase order line items by vendor line and recost the line items if a quantity break level is met.
•
No - Recost purchase order line items based on their individual quantity or dollar value.
6
Enter a Table Method to determine how to calculate price and cost. You can express break-level pricing as a set price or as a discount off of a column price. Choose from the following:
•
$ - Each break has a set price.
•
* - Price is a discount percent off of a column price.
Vendor Line Maintenance
When referencing a table in Vendor Line Maintenance, if products within a vendor line have different costs, use the * (percent) method. The $ method sets one cost for all products using the table.
7
Enter the Break Level Type to define break levels. Choose from the following:
•
U - Define break levels based on units.
•
$ - Define break levels based on dollars.
•
M - Define break levels based on unit of measure.
8
Press ENTER. If you chose the * (percent) Table Method, go to step 9. If you chose the $ (price) method, go to step 14.
9
If you chose to enter values using a percent, the Discount Table - Percent screen displays.
10
For Basis Field, enter the price or cost column from Product Maintenance to use as the base amount to calculate discounts. If you select a price column, enter a discount percent off of the column in step 12. If you select a cost column, enter a percentage over the cost in step 12. Options are:
•
0 - Suggested retail.
•
1 - Price1.
•
2 - Price2.
•
3 - Price3.
•
4 - Price4.
•
5 - Price5.
•
A - Average cost.
•
B - Product quantity break. Uses the product’s quantity break table to apply the discount defined in this table. Only used for percentage discounts.
•
C - Customer default column. Uses the default Price Code set up for each customer in Customer Maintenance to determine which column to apply discounts to.
•
O - Cost for sales.
•
R - Replacement cost.
•
S - Standard cost.
11
Enter the Break Level to define the break value that qualifies for the corresponding discount percent or price:
•
If the Break Level Type is M, enter the units of measure that determine the discounts.
•
If the Break Level Type is U, enter the minimum quantities to determine the discounts.
•
If the Break Level Type is $, enter the minimum value to determine the discounts.
12
Enter the % to define the discount percentage for the corresponding break level. This percentage is applied to the value in Basis to calculate the price or cost for each product. This value displays under Disc% on the line item detail screen of the purchase order or sales order. The discount entered is a percentage off of a price if a price is specified in Basis or a percentage over the cost if a cost is specified in Basis. A negative percentage gives the opposite result (a percentage over a price or a percentage off of a cost).
13
Press F2-Save to save the discount break table information. The remaining steps do not apply for the percent method.
14
If you choose to enter values using the $ (price) Table Method, the Discount Table - Dollar screen displays.
15
Enter the Break Level to determine the break quantity that qualifies for the corresponding discount percent or price:
•
If the Break Level Type is M, enter the units of measure to determine the discounts.
•
If the Break Level Type is U, enter the minimum quantity values to determine the discounts.
•
If the Break Level Type is $, enter the minimum value to determine the discounts.
16
Enter the Price for the corresponding break level. This value appears for Gross and Net-Cost in the P/O Detail Information screen when you use the Recost option on a purchase order.
17
Press F2-Save to save the discount break table information.
