Consolidating Bin Counts
After you enter, verify, and adjust inventory counts, you must consolidate bin counts before you update physical inventory. The Physical RF Consolidation Detail Audit Report shows the total count and deviation amount. If a product exists in multiple bins, the counts consolidate into a single count. This step is necessary for the shrinkage and deviation reports to calculate correctly.
To consolidate bin counts:
1
Select Inventory > Physical Inventory > Radio Frequency Procedure > Consolidation Procedure to open the Consolidation Procedure screen.
2
Enter the Warehouse for which you want to consolidate bins.
3
Indicate if you want to include Rental Products in the consolidation procedure. Choose from the following:
•
B - Consolidate all products, including rental.
•
Y - Consolidate only rental products. Exclude all other products.
•
N - Consolidate all products, except rental.
4
Press ENTER. The following message appears:
Consolidation will print the Detail Audit
Report, and needs to be run again if there
are any additional bins entered before running the Physical Update Procedure. Do you wish to continue?
5
Choose from the following:
•
No - Cancels the procedure.
•
Yes - Consolidates the items in multiple bins and prints the Physical RF Consolidation Detail Audit Report.
If you change any counts, you must consolidate the bin counts again.
Column Descriptions
Column Heading
Description
Product Number/ Description
Line 1: The product number.
Line 2: The product description.
Bin Location
The bin location for the product. If the product exists in multiple bins, all bins display.
U/M
The unit of measure for the product.
Count
The count entered for each bin.
Booked
LinkOH
The quantity of the product linked to a sales order, work order, or transfer.
Link $Val
The value of the linked quantity.
StkOH
The saved on-hand quantity.
Dev
The deviation quantity between the count and the saved on-hand quantity.
Fifo-Avg
The cost of the item in this warehouse. The cost used depends on whether your company uses FIFO or average cost.
Dev $ Value
The total dollar amount of the deviation quantity.
