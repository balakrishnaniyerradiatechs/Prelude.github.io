Warranty Tag Bin Maintenance screen
The Warranty Tag Bin Maintenance screen allows you to assign a bin location for a product that has a warranty tag. This warranty tag bin must be updated manually, not by sales orders or inventory adjustments. Any quantity added here does not affect on-hand inventory, but is only for reference in the Warranty Tag Inquiries.
Menu Path: Inventory > Vendor Claims > Warranty Tag Bin Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse where the warranty tag product exists.
Warranty Tag
The warranty tag number assigned to the product.
Product
The product that is in warranty inventory.
Vendor
The vendor that should be sending credit or a replacement for the warranty item.
Bin
The bin where the warranty tag product exists. This does not affect on hand inventory but displays for reference in the Warranty Tag Inquiries.
F2-Save
Saves the warranty tag bin information.
F5-TWTI
Opens the Warranty Tag by Tag Inquiry to display product and vendor claim information for a tag number.
