Setting System Flags for FIFO Costing
To set up your system to use FIFO cost, you must set the following flags.
Company Maintenance
Set G/L Cost to F to use FIFO costing for accounting cost. Then you can choose how to track FIFO layers:
•
C - At the company level.
•
W - At the warehouse level (default).
For Maintain FIFO Layers, determine if you want to keep FIFO information for reporting purposes:
•
Y - Maintain FIFO layers for all incoming material. Enough layers are stored to support the current onhand quantity of a product. The date, document number, reference number, quantity, cost, freight, duty, and brokerage are all saved for reports and inquiries.
•
N - Do not store FIFO layers.
Chart of Account Maintenance
If the G/L Cost flag in Company Maintenance is set to W, you must set up a FIFO Revaluation Clearing account in Chart of Accounts Maintenance to use the FIFO Revaluation Process.
Additional Option Parameters
Use Serialized Cost with FIFO determines if serialized items use the serial cost of the product for FIFO costing on sales orders, transfers, work orders, vendor claims, inventory adjustments, and physical tag entries. FIFO layers are relieved based on the first in first out method and the cost is the layer being relieved. If you have serialized items, FIFO normally ignores the serial cost and updates by layer. Choose from the following:
•
Y - Use the serial cost of the item instead of the FIFO layers cost. The layer’s on-hand cost is relieved accordingly.
•
N - Use the FIFO layer cost of the item.
