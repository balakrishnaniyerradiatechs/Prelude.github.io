Current Inventory Value Report - Reserved
This report displays the value of on-hand inventory that is reserved for customers. If a quantity of a product is reserved for a customer, it cannot be allocated to any other customers’ orders. Refer to Reserving Inventory for a Customer for information on how to put a product into reserved inventory.
Menu Path: Inventory > Reports > Inventory Valuation Reports > Current Inventory Value Report - Reserved
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse(s) with the reserved inventory.
•
Reserved Customer. The customer number assigned to the reserved inventory.
•
Primary Vendor. The primary vendor(s) to include on the report. Leave blank to include all vendors.
•
Product. The product number(s) to include on the report. Leave blank to include all products.
•
Sort by (P)roduct (C)ustomer. Determines how you want to sort the report. Choose from the following:
-
P - Sort by product.
-
C - Sort by customer.
Column Descriptions
Column Heading
Description
Customer
The customer number and description.
Product Number
The product number in reserved inventory.
Description
The product description.
Qty
The quantity of the product reserved for the customer.
Avg Cost
The average cost of the product.
Extended
The total extended value of the product reserved for the customer (Qty x Avg Cost).
