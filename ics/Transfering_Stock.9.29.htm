Assigning SSCCs to Transfers
You can assign Serial Shipping Container Codes in Transfer Serial Shipping Container Entry. Depending on how you set up the transfer printing information in Warehouse Maintenance (refer to Printing SSCC Labels for Transfers), the SSCC labels can print with or without a detail listing of the contents of the carton or pallet.
If you use the regular Transfer Shipment Entry program to ship a transfer from a warehouse that requires detailed content labels (refer to Printing SSCC Labels for Transfers), the Transfer Serial Shipping Container Entry screen appears to allow you to enter the contents.
To assign SSCCs to identify the contents of a carton or pallet on a transfer:
1
Select Inventory > Stock Movements > Transfer Serial Ship Container Entry to open the Transfer Serial Shipping Container Entry screen.
2
For Transfer Release#, enter the transfer release number. If you accessed this screen from Transfer Shipment Entry, the release number already appears.
3
For Serial Ship Code, press ENTER to generate a new SSCC.
4
Enter the Package Type to define the type of container. Choose from the following:
•
C - Carton.
•
P - Pallet.
5
Enter the Product and Quantity for each item in the container.
F5-Review Release opens the Stock Transfer Display screen to allow you to view the items and quantities on the original release.
Action Bar > Nonscan Inquiry displays all items not completely packed or scanned. Once each item is completely packed or scanned, it no longer displays in this inquiry.
6
After all products are registered, choose from the following to complete the transaction:
•
If the transfer has more line items, enter ENDSCAN on a blank line or scan a label with the ENDSCAN text. A new SSCC appears to assign to the next item on the release.
•
If this is the last item on the transfer release, type FINALSCAN on a blank line or scan a label with the FINALSCAN text. The system checks to make sure all items on the release have an SSCC. The SSCC Label Program defined for the warehouse executes to print labels. You can attach the labels to all the registered cartons/pallets.
•
If you want to recheck the product information, press F2-Save. The following prompt appears:
Choose from the following:
Are you Finished With Registering All
Contents for This Release?
-
Yes: Allows you to check the product information for each code to make sure it matches the expected transfer merchandise. Once everything matches, the transaction is complete. The SSCC Label Program defined for the warehouse executes to print labels. You can attach the labels to all the registered cartons/pallets.
-
No: Allows you to continue registering the next SSCC.
-
Suspend: Saves the information so you can assign more SSCCs later.
-
Return: Returns to the screen to allow you to enter additional information.
If you accessed Transfer Serial Shipping Container Entry from Transfer Shipments Entry, you cannot return to Transfer Shipment Entry until the transfer is complete. A warning message appears to let you know it is not complete.
Printing labels
Depending on how you set up label printing in Warehouse Maintenance (refer to Printing SSCC Labels for Transfers), the SSCC labels will print either after you register each item or after you’ve entered all the items on the transfer release.
