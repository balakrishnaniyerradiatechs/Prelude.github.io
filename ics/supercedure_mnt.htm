Supercedure Maintenance screen
The Supercedure Maintenance screen allows you to create a new part number to supercede an existing part number. A superceded product is a product that is replaced with another product, usually by the vendor (e.g., ABC is replaced by ABC-1). You can set up a new (superceding) part number to replace the superceded product. The usage from the old (superceded) product number is moved to the new product number. Refer to Supercedure Products for more information.
Menu Path: Inventory > Maintenance > Supercedures
Field and Function Key Descriptions
Field/Function Key
Description
Superceded Product
The product number that is obsolete.
New Product
The superceding product number that replaces the Superceded Product Number. You can use either an existing part number or create a new part number.
Description
The first line of description of the new product number.
Description Two
The second line of description of the new product number.
Vendor Product
The vendor’s corresponding part number, if applicable.
Date Superceded
The date the New Product Number supercedes the Superceded Product Number.
F2-Save
Updates the product supercedure information.
