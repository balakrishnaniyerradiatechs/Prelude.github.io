Bin Location Sort Sequence Codes Update screen
The Bin Location Sort Sequence Codes Update screen allows you to logically sort bins in your warehouse to help streamline order picking (used with the Warehousing module).
Menu Path: Inventory > Processes > Bin Location Sort Sequence Codes Update Procedure
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse for the sort sequence.
Beginning Bin
The first bin to pick.
Ending Bin
The last bin to pick.
Sequence Order
The sequence order:
•
A - Ascending.
•
B - Descending.
Beginning Sequence Number
The Beginning Sequence Number to assign to bins. When a pick ticket prints, the products are listed in the sequence you define. For example, if you enter 1, the Beginning Bin Location that you define becomes the first bin to pick in this bin location range. This number displays under Sort Sequence Code in Bin Location Maintenance (refer to Defining Bins).
F2-Begin Update
Sorts bins in the defined picking order and assigns a sort sequence code to each bin.
