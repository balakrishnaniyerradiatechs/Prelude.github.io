Commodity Codes
If you receive merchandise from overseas vendors, you must pay import duty fees to the U.S. government. Duty is calculated either as a rate per quantity of the product or as a percent of the value of the product. Prelude provides commodity codes to allow you to group products that have the same duty rate. Once you set up the commodity codes in Commodity Code Maintenance, you can assign the codes to each product.
Commodity Code Listing
To view a list of commodity codes, select Inventory > Listings > Miscellaneous Listings > Commodity Code Listing.
To set up a commodity code:
1
Select Inventory > Maintenance > Product Control Files > Commodity Codes to open the Commodity Code Maintenance screen.
2
Enter a Commodity Code to assign to products with the same duty rate.
3
For Duty Rate, enter either a rate per quantity or a percentage of cost to use to calculate duty for these products.
4
Enter a Description of the commodity code.
5
Press F2-Save to save the commodity code information.
To assign a commodity code to a product:
1
Select Inventory > Maintenance > Product Maintenance > Product Maintenance to open the Product Maintenance screen.
2
Enter the Product.
3
Press F5-Add Info > F5-Add Info to open the Additional Product Maintenance Information Screen.
4
Enter the Commodity Code set up in Commodity Code Maintenance.
5
Press F2-Save to save the commodity code information.
