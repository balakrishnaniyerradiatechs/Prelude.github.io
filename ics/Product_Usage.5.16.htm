Product Parameters Calc Skipped Audit Report
The Product Parameters Calc Skipped Audit Report displays products whose purchasing parameters were not recalculated during End of Month Calendar Close. You can either run this report with the End of Month Calendar Close or you can run it independently any time during the month.
The products on this report appear in the Buyer Queue Inquiry. The buyer can review and/or manually recalculate the purchase parameters if necessary. For information on recalculating skipped purchase parameters refer to the “Buyer Queue Inquiry” (Purchase Parameters Skipped) in the Purchasing documentation.
Skipped Purchase Parameters
Purchasing parameters are not recalculated if:
•
Monthly usage was greater than the total of the last 5 months of combined usage (U code).
•
A product is out of stock for 2 weeks or more in one month (O code).
•
Usage overrides were not entered for the months with usage exceptions. Usage overrides are entered either in the Buyer Queue Inquiry (Purchase Parameters Skipped) or in Usage Override Maintenance (refer to Adjusting Usage for Skipped Product Parameters).
Out of Stock Exception
The Out of Stock Exception flag in Purchasing Controls > Purchasing Parameters determines if purchasing parameters are reviewed when a product is out of stock:
Unusual Usage Exception
The Unusual Usage Exception flag in Purchasing Controls > Purchasing Parameters determines if purchasing parameters are reviewed when a product has unusual usage.
Menu Path: Inventory > Processes > Usage Exception Menu > Product Parameters Calc Skipped Audit Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse containing products with skipped purchase parameter.
•
Buyer. The buyer assigned to the vendor for the products with the skipped purchase parameters. Leave blank to view products for all buyers.
•
Auto Override Usage Only?. Determines if you want to include products whose usage was automatically overridden:
-
Y - Include products with overridden usage.
-
N - Do not include products with overridden usage.
Column Descriptions
Column
Description
Product
The product skipped during End of Month Calendar Close. Purchasing parameters are not recalculated for the product.
Description
The product description.
Months 1-6
The usage for the last 6 months.
Byr
The number for the buyer responsible for the product.
Auto
Indicates if usage was automatically recalculated:
•
Y - Usage was automatically recalculated.
•
N - Usage was not automatically recalculated.
