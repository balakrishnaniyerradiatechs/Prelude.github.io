Lot Description Maintenance screen
The Lot Description Maintenance screen allows you to enter vendor lot number information when you receive a lot product.
Menu Path: Inventory > Stock Receipts > Stock Receipts Entry > Lot Number Information screen > F5-Description
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Lot Number
The lot number assigned by the vendor.
Lot Description
The description of the lot product.
Country of Origin
The country code for the country of origin of the lot product.
Comments
Comments regarding the lot product.
F2-Save
Saves the lot description.
