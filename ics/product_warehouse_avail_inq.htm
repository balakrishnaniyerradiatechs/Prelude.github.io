Product Warehouse Availability Inquiry
The Product Warehouse Availability Inquiry displays available quantities of a product in other warehouses.
Warehouse Sequence
The warehouses and the sequences in which they are displayed are defined in Inventory Controls > Warehouse Sequence Maintenance.
Checking Product Warehouse Availability
To view the available quantities of a product in all warehouses:
1
Select Inventory > Inquiries > Availability Inquiries > Product Warehouse Availability Inquiry to open the Product Warehouse Availability Inquiry screen.
2
Enter the Product.
3
Enter the Warehouse.
Warehouse availability for a product displays along with forecast, on-order, and backorder information.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number. The product description displays to the right.
Warehouse
The warehouse for which you want to check availability for the product. The warehouse description displays.
Whse
The warehouses where the product is normally stocked.
Description
The warehouse description.
Forecast
The forecast for the upcoming month.
Op/Min
The order point or minimum of the product. If available stock falls below this quantity, the system recommends replenishment.
Ldt
Leadtime. The average time required to replenish a stock item. The number of days from the date the purchase order was entered until the date the item is received.
OnOrder
The quantity on order from purchase orders, transfers, and work orders.
Backorder
The quantity backordered on sales orders, transfers, and work orders.
Surplus
The surplus quantity of the product. If using the EOQ purchase method, this is any quantity available over the EOQ plus the line point. If using the min/max purchase method, this is the quantity over the maximum recommended quantity to keep in stock.
Available
The quantity available to sell.
An R displays if the product has reserved inventory in the warehouse (refer to Reserving Inventory for a Customer)
Rsvd Inv
Opens the Reserved Inventory - Corporate View screen (only accessible if an R displays next to an Available quantity) to display the quantity reserved for each customer for each warehouse (refer to Reserving Inventory for a Customer).
OK
Closes the inquiry.
F5-Kit Available
If the product is a kit parent, you can view the kits available in each warehouse. Press F5-Component Availability to view the Product Kit Availability Inquiry, which shows the availability of each component of the kit. Refer to Kit Parent Products for information on setting up kits.
F6-Bom Available
If the item is a bill of material, you can view availability of this BOM in each warehouse. Press F5-Components to view the availability of each component on the BOM.
