Physical Inventory List By Other screen
The Physical Inventory List By Other screen allows you to enter a select statement to define the selection criteria for an inventory list.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Build Physical Inventory Lists > Manually Defined Criteria
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for the inventory list.
List Name
The name that identifies the count sheet. You will use this name to print count sheets, enter counts, update inventory, and run inventory reports.
Enter the English Statement That Will Be Used To Generate The List: SSelect Product With Whse = “<Whse>” AND WITH
The TCL command to create the list of products.
