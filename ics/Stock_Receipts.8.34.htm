Splitting a License Plate
If you need to break a box quantity to fill an order, you can split a license plate number. A new license plate number is assigned and a new label prints for the split quantity. The original license plate is reduced to the remaining quantity.
To split a license plate quantity:
1
Select Inventory > Inventory Adjustments > License Plate > License Plate Split Entry to open the License Plate Split Entry screen.
If you have access to more than one warehouse, the Location Retrieval screen appears first. Enter the appropriate Selling Warehouse, Department, and/or Cost Center.
2
For License Plate #, enter the license plate number or press F3 to search for license plate numbers by product. The Product, Lot (license plate number), and Quantity display.
3
For New Package Quantity, enter the total quantity to split off into a into new license plate.
For example: Enter 25 to split off a quantity of 25 from the original license plate.
The License Plate Information screen displays to allow you to enter the new number of labels and quantity per label.
4
For #/Lbls, enter the number of new license plates to create.
5
For Qty/Label enter the number of units to include in each license plate.
The number of labels multiplied by the quantity per label must not exceed the New Package Quantity on the License Plate Split Entry screen.
6
Press F2-Save.
A new license plate number is created and printed with the new quantity. The original license plate number still contains the remaining quantity.
Lot Inquiry
Lot Inquiry displays information on lot/license plate numbers and quantities.
Transaction Audit Inquiry
The Transaction Audit Inquiry displays the LSP code for the product to indicate that the license plate was split.
