Establishing Inventory Adjustment Reason Codes
Inventory adjustment reason codes identify the reason for adjusting inventory. Each code references a general ledger account to use as the offset account for the inventory adjustment. Once the codes are set up, you can use them in Inventory Adjustment Entry (refer to Adjusting Inventory Quantities).
Inventory Adjustment Reason Code Listing
To view a list of inventory adjustment reason codes select Inventory > Listings > Miscellaneous Listings > Inventory Adjustment Reason Codes.
To create or maintain inventory adjustment reason codes:
1
Select Inventory > Maintenance > Misc Code Files > Inventory Adjustment Reason Codes to open the Inventory Adjustment Reason Maintenance screen.
2
For Inv Adj Reason, enter a 1 or 2 character, alphanumeric code to use in Inventory Adjustment Entry.
3
Enter the Description that identifies why inventory is being adjusted. This description displays when the inventory adjustment code is entered.
4
For Update Product Usage, indicate if product usage is updated when this code is used in Inventory Adjustment Entry (refer to Recording Product Usage). Choose from the following:
•
Y - Product usage is updated for items with this adjustment code. The usage is adjusted for sales orders and the change displays in the sales order usage column in Product Usage Inquiry. Refer to the Product Usage Maintenance Screen.
•
N - Product usage is not updated for items with this adjustment code.
5
Under Adjustment Account, enter the general ledger account number to use as the offset account for the inventory adjustment general ledger posting:
•
Div - The division number for this account. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Reg - The region number for this account, if applicable. A region is a subset of division - if division is used. (e.g., Southeast Region, Northwest Region).
•
Loc - The location or warehouse for this account.
•
Dep - The department number for this account. A department is a subset of warehouse, typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Ccr - The cost center number for this account. A cost center is a subset of department - if department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Account# - The general ledger account number to use as the offset when this inventory adjustment code is used in Inventory Adjustment Entry. If left blank, then the Physical Inventory Adjustment account from General Ledger Table Maintenance is used.
6
Press F2-Save to save the inventory adjustment reason code.
