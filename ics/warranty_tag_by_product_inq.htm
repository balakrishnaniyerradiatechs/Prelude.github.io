Warranty Tag by Product Inquiry
The Warranty Tag by Product Inquiry displays detail tag, vendor, and claim information for a product number. This information is helpful if your warehouse has a designated area for warranty returns and you are not sure whether or not a claim was generated.
Viewing Warranty Tags by Product Number
To view warranty tag information:
1
Select Inventory > Inquiries > Warranty Tag Inquiries > Warranty Tag By Product to open the Warranty Tag by Product Inquiry screen.
2
Enter the Product.
Detailed tag information for the product displays.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number for which you want to view warranty tag information. The product description displays.
Tag Number
The tag number assigned to the product. Refer to Moving Products into Warranty Inventory for information on assigning a tag number to a product.
Whse
The warehouse where the tagged product exists.
Vendor
The primary vendor for the product.
Customer
The customer number for the tagged product if the warranty tag was entered on a credit memo for a customer. Refer to Creating Warranty Tags in Credit Memo Entry.
Document#
The document number where the tag number was created. The prefix indicates how the tag number was assigned:
•
R - The tag number was assigned in Stock Receipts Entry. Refer to Creating Warranty Tags in Stock Receipts Entry.
•
S - The tag number was assigned on a credit memo to a customer. Refer to Creating Warranty Tags in Credit Memo Entry.
•
A - The tag number was assigned in Inventory Adjustment Entry. Refer to Creating Warranty Tags in Inventory Adjustment Entry.
Doc-Date
The date the warranty tag was assigned.
Claim#
The vendor claim number if the tag is on a vendor claim.
Clm-Date
The date the vendor claim was issued, if applicable.
App
Indicates if the claim was reconciled:
•
Y - The vendor claim was reconciled.
•
Null - The vendor claim is not reconciled.
Vendor RMA#
The vendor’s return material authorization number.
F2-Finish
Closes the inquiry screen.
F5-Review Tag
Opens the Warranty Tag Inquiry Screen to allow you to view detail warranty information for a product on the warranty tag.
F6-Enter Tag#
Allows you to enter a tag number to go directly to.
F9-Goto Line
Allows you to enter a line number, reference number, or page number to go directly to.
