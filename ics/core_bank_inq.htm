Core Bank Inquiry
The Core Bank Inquiry allows you to view the status of a customer’s core bank. You can view the number of cores received, applied, and remaining in the core bank.
Viewing Core Banks
To view the status of a customer’s core bank:
1
Select Inventory > Inquiries > Core Inquiries > Core Bank Inquiry to open the Core Bank Inquiry screen.
2
Enter the Customer whose core bank records you want to review.
3
For Filters, indicate if you want to limit the inquiry to a return type or exchange type. Choose from the following:
•
N - Does not use filters to limit the inquiry.
•
Y - Opens the Core Bank Inquiry Filter Options screen to allow you to enter a return type and/or an exchange type to limit the inquiry.
The core bank status appears.
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer number. The customer description displays.
Filters
Determines if you want to limit the inquiry to a return type or exchange type:
•
N - Do not use filters to limit the inquiry.
•
Y - Use filters to limit the inquiry. The Core Bank Inquiry Filter Options screen opens to allow you to enter a return type and/or an exchange type to limit the inquiry.
Inv Adj#
 
The Inventory Adjustment number for the core bank credit and the core bank record ID (refer to Receiving Cores in Inventory Adjustment Entry).
Product
The core product number.
Core Cd
The core type code from the core’s product record. Refer to Creating Core Type Codes.
T
The exchange type for the corresponding record:
•
R - Repair and return.
•
E - Exchange.
•
W - Warranty.
•
U - Unused.
•
O - Other.
Date Ret
The date the core was placed in the core bank.
Rec’d
The number of cores placed into the core bank on this adjustment.
Applied
The number of cores from the core bank that are applied against a pending core.
Remain
The number of cores still available to apply on the core bank.
F2-Save
Clears the screen to allow you to enter another customer number.
F5-Review Received
Opens the Inventory Adjustment Display screen to allow you to view the adjustment that placed the core items into the core bank.
F6-Review Applied
Opens the Applied Info screen to allow you to view the order numbers/pending core records that the banked cores were applied against. Multiple orders are listed if the core bank record was applied against multiple sales orders/pending cores.
F9-Goto Line
Allows you to enter a line number, page number, or order number to go directly to.
