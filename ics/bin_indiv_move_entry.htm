Bin Individual Move Entry screen
The Bin Individual Move Entry screen allows you to move a product from one bin to another. You must have the Multiple Bin Locations Flag in Inventory Option Parameters set to Y to change bin locations.
Refer to Entering Individual Bin Moves for more information.
Menu Path: Inventory > Inventory Adjustments > Bin Individual Move Entry.
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The number for the warehouse where the bin move is taking place.
Move Number
The system-assigned number to track the bin move information.
Product Number
The product number to move.
Description
The product description.
Description 2
The second line of product description, if applicable.
Class
The inventory class for the item. Inventory class is based on how many dollars of the item move through inventory in a year. Refer to Classifying Products.
Lot Number
The lot number for the product, if using lot numbers.
From Bin
The bin number where the product is moving from.
Description
The description of the From Bin.
Quantity
The quantity of the product to move.
U/M
The unit of measure of the product as entered in Product Warehouse Maintenance.
To Bin
The bin location to receive the product quantity.
Description
The description of the To Bin.
F2-Save
Saves the bin move information.
F5-Load Bin
Opens the Bin Move Default Bin Entry screen to allow you to load default bin and quantity information for a bin move.
F8-PBI
Opens the Product Bin Inquiry to allow you to view bin information for a product.
F9-Goto Line
Allows you to enter a page number, reference number, or line number to go directly to.
