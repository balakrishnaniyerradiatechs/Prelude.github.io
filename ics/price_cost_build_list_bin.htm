Price/Cost Build List By Bin screen
The Price/Cost Build List by Bin screen allows you to enter one or more bin numbers for which you want to create a list of prices and costs to update.
Menu Path: Inventory > Processes > Price/Cost Update Procedure > Price Cost Changes by Lists > Build List > By Bin Location(s)
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for which you want to create the list. Enter ALL to update prices for all warehouses at the product level.
List Name
The descriptive name to assign to the list. Do not use spaces in the name.
Inventory/Catalog
Indicates if the list is created from the inventory file or from the catalog file:
•
I - Creates the list from the active inventory file (stock and non-stock).
•
C - Creates the list from the catalog file which includes products sold by the vendor but are not active in your system.
Bin
The bin(s) to include in the list.
Description
The bin description.
F2-Create List
Creates the list. You can view the list in Displaying Price/Cost Lists.
