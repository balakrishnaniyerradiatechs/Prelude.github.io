Product Characteristics screen
The Product Characteristics screen allows you to assign a product characteristic code to a product number. Product characteristics define the information required to sell a product. When you sell a product that has characteristic codes assigned, you must enter the information necessary to fill the order. Refer to Product Characteristics for more information.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F6-Xref/Notes > Characteristics
Field and Function Key Descriptions
Field/Function Key
Description
Code
The product characteristics code to assign to this product number (refer to Setting Up Product Characteristic Codes). Any time you sell this product, the Product Characteristics Information screen opens so that you can enter the information necessary to fill the order.
Description
The description of the product characteristic code.
F2-Save
Saves the product characteristic code information.
