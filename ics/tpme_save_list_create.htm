Create a Product List
To update a group of products, you must first create the list of products. You can make a list either for a range of product numbers or for a product line.
To create a product list:
1
Select Inventory > Maintenance > Product Maintenance > Global Product Maintenance Menu > Save-List Creation Process to access the Global Product Maint Save List Creation Screen.
2
For Save-List Name, enter a unique name that represents the range of products or the product line included in this update. If the name you choose already exists, a warning message appears to inform you that the name already exists but still allows you to override it.
3
Choose one of the following to determine how the list is created:
•
Enter the Product range to include in the list.
•
Enter a Product Line to limit the list to one product line.
4
If you want to use any other field to select products, press F5-Modify Criteria and enter the Dictionary Name.
5
Choose Execute to create the list. The number of products included on the list displays.
To change all products on the list, refer to Create an Update Template.
