E-Commerce Information Screen
The E-Commerce Information screen allows you to set up product, description, URL, and packaging information so that you can sell the product on your VIA website.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > E-Commerce Information
Field & Function Key Descriptions
Field/Function Key
Description
Sold On Web
Indicates if the product is available for sale on the VIA website for all customers:
•
Y - The product is available for sale (default).
•
V - The product can be viewed but not purchased.
•
P - The product is available for sale but does not show a price. The message “Call For Price” displays underneath the product.
•
N - The product is not available on the VIA website.
Sold On Web Retail
Indicates what displays for retail customers:
•
Y - The product is available for sale (default for initial setup of VIA products).
•
V - The product can be viewed but not purchased.
•
P - The product is available for sale but does not show a price. The message “Call For Price” displays underneath the product.
•
N - The product is not available on the VIA website.
Shopping List
The shopping list name that this product belongs to. You can also tag all the lists applicable for this product.
Packaging Description
Describes how the product is packaged. This information displays on the product page on the website.
Urls
The URL of a web page with more information on the product.
Url Descriptions
The description of the web page.
Auto Generated Urls
The URL links that automatically update the product.
Thumbnail Image
Product Page Images
The URLs to the product description pages.
F2-Save
Saves the E-commerce information.
F5-Web Desc
Allows you to enter a wholesale web description or a retail web description.
F6-Qualifiers
Allows you to enter the E-commerce qualifiers to filter products by site, vendor, or customer.
