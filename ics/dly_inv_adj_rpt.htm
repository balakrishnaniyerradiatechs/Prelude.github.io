Daily Inventory Adjustments Report
The Daily Inventory Adjustment Report displays all inventory adjusted for a warehouse for a day (refer to Adjusting Inventory). You can choose to view either the detail line item information or only the total value for each adjustment.
Menu Path: Inventory > Reports > Inventory Adjustment Reports > Daily Inventory Adjustments
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse where inventory was adjusted.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Transaction Type: The inventory adjustment types to include in the report. Leave blank to list all types of inventory adjustments or choose from the following:
-
INV - Regular inventory.
-
WTY - Warranty inventory.
-
DMG - Damaged inventory.
-
SCR - Scrapped/lost inventory.
•
Entry Date: The date when inventory was adjusted.
Column Descriptions
Column Heading
Description
Adj Number
The adjustment number assigned to the transaction.
Whse
The number for the warehouse where inventory was adjusted.
Type
The type of inventory adjustment:
•
INV - Regular inventory.
•
WTY - Warranty inventory.
•
DMG - Damaged inventory.
•
SCR - Scrapped/lost inventory.
User
The user ID of the person who adjusted the inventory.
Product
Line 1: The product number.
Line 2: The product description.
Line 3: The second line of description, lot number, or serial number, if applicable.
Quantity
The quantity added to or deducted from inventory. A positive quantity indicates that inventory was added. A negative quantity indicates that inventory was reduced.
U/M
The unit of measure for the adjusted item.
Cost
The unit cost for the adjusted item as entered in the Inventory Adjustment Line Detail screen.
Ext Cost
The total extended cost of the adjusted inventory cost (Quantity x Cost). A positive value indicates that inventory cost was increased. A negative value indicates that inventory cost was reduced.
Adjustment Reason
The adjustment reason code and description. These codes are set up in Inventory Adjustment Reason Code Maintenance (refer to Establishing Inventory Adjustment Reason Codes).
