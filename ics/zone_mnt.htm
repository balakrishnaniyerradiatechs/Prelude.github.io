Zone Maintenance screen
The Zone Maintenance screen allows you to set up zones to group similar bins within a warehouse and to expedite picking and cycle counting.
Menu Path: Inventory > Maintenance > Bin Files > Zone Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse for the zones.
Zone
The zone number (up to 4 alphanumeric characters).
Description
The description of the zone to help identify the location in the warehouse.
Warehouse Area
The warehouse area that includes this zone, if you use warehouse areas. Refer to Defining Warehouse Areas.
Pick Quota Lines/Hr
The number of lines to pick per hour when picking in this zone (informational purposes only - not required).
Bin Count Frequency
The number of times per year to cycle count this zone. This field is required if you are using cycle counting.
Sort Sequence Code
Optional user defined field.
Advance Warehousing Options
Auto Rce After Rf Picking
Determines if release confirmation is automatic after radio frequency order picking.
Floored Cubed Segment
Defines how many cubes a segment should contain for a floored segment. For example, a if pallet is ‘x’ cubes, the segment is generated with enough products to meet this cube value. Then a new segment is started.
NonFloored Cube/Segment
Defines how many cubes a segment should contain for a non-floored segment. For example, a if pallet is ‘x’ cubes, the segment is generated with enough products to meet this cube value. Then a new segment begins.
Bin Fill Level Percent
The fill level percent for bins in this zone for the cube logic used (volume method) in stock receipts putaway.
Bin Putaway Qty Minimum Percent
The minimum percent used to determine which bin to recommend for a putaway quantity (if you are using radio frequency stock receipts putaway - V Method). For example, if you enter 50 for the minimum percent, stock receipts putaway will only recommend a bin that can hold at least 50% of the putaway quantity.
SSCE Required
Determines if Serial Shipping Container Entry (SSCE) is required for this zone to identify the entire contents of any container.
Rf Pck Ship Label
Determines if a radio frequency pick label prints:
•
L - Print labels for each line.
•
N - Do not print labels.
•
Y - Print labels after the whole order is picked.
F2-Save
Saves the zone information.
F4-Del
Deletes the zone information.
