Price / Cost Build List By Other Criteria screen
The Price / Cost Build List by Other Criteria screen allows you to manually enter a TCL statement to define the selection criteria to use to create a list of prices and costs to update.
Menu Path: Inventory > Processes > Price/Cost Update Procedure > Price Cost Changes by Lists > Build List > Other Criteria
Field and Function Key Descriptions
Field/Function Key
Description
Product or Warehouse
Determines whether you want to update prices by product or warehouse:
•
P- Update by product.
•
W - Update by warehouse.
Warehouse
The warehouse number for which you want to create the list. Enter ALL if updating price and cost at the product level.
List Name
The descriptive name to assign to the list. Do not use spaces in the name.
Inventory/Catalog
Indicates if the list is created from the inventory file or from the catalog file:
•
I - Creates the list from the active inventory file (stock and non-stock).
•
C - Creates the list from the catalog file which includes products sold by the vendor but are not active in your system.
Enter The English Statement That Will Be Used To Generate The List:
The TCL command to create the list of products.
