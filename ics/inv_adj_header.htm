Inventory Adjustments Header screen
The Inventory Adjustments Header screen allows you to enter a ship via and a reference number for an item that is shipping out of the warehouse.
Menu Path: Inventory > Inventory Adjustments > Inventory Adjustments Entry > F5-Header
Field and Function Key Descriptions
Field/Function Key
Description
User Id
The user ID number of the person entering the adjustment.
Ship Via
The ship via code to indicate how the item is shipping out of the warehouse.
Reference Number
A packing list number or other reference number, if applicable.
F2-Save
Saves the header information and closes the screen.
