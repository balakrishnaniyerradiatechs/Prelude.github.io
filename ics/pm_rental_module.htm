Rental Module Screen
The Rental module screen appears when you set the Product Type to N (for Rental) on the Product Maintenance screen. This screen is also accessible from the Action Bar.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Rental Information
Field & Function Key Descriptions
Field/Function Key
Description
Training Required
Indicates if a customer requires training to rent this product:
•
Y - Training is required. This information prints on the comment line on the pick ticket.
•
N - Training is not required.
Rental Return Status
The default status code used in Rental Order Entry that will be used when the rental product is returned to indicate what status the product is in:
•
A - Available to rent.
•
C - Check in.
•
H - Hold.
•
O - Off rent.
•
R - Shop/repair order. Creates a shop repair order.
•
W - Work orders. Generates an expense work order.
This information prints on the invoice and rental acknowledgement
Liability Insurance Required
Indicates if the customer must purchase insurance when renting the product:
•
Y - Insurance is required. A warning message appears if the customer does not have an insurance limit or if the insurance is expired.
•
N - Insurance is not required.
The Liability Insurance Limit for a customer is set up in Customer Maintenance > F5-Add Info > F6-Rental.
Measurements Required
Indicates if measurements are required:
•
H - Meter readings are required (e.g., you must record the hours in and the hours out).
•
O - Odometer readings are required.
•
W - Wear measurements are required (e.g., diamond blades).
Leave blank if measurements are not required.
Rental Retail Value
The retail value for this type of rental equipment, in case it is damaged. This value is compared to the customer’s insurance coverage (Customer Maintenance > F5-Add Info > F6-Rental > Liability Insurance Limit) to determine if a damage waiver should be charged on a rental order.
Damage Waiver Product Number
The service product number set up to identify the damage waiver charge and the percentage to calculate the damage waiver charge. This charge is billed to a customer for every periodic billing cycle (if they don’t have liability insurance). The Liability Insurance Limit for a customer is set up in Customer Maintenance > F5-Add Info > F6-Rental.
Environmental Charge P/N
The service product number for environmental charges on a rental order. Added at the time of rental return.
You can determine if an Environmental Charge should be billed to a customer in Customer Maintenance > F5-Add Info > F6-Rental.
Liability Ins Charge P/N
The service product number to use for liability insurance charges on a rental order. If the customer has no Physical Damage Coverage set in
Customer Maintenance > F5-Add Info > F6-Rental., you would want to add a liability insurance charge for the product.
Fuel Charge Applies
Indicates if the customer will be charged for fuel:
•
Y - Fuel charges apply and print on the invoice and rental acknowledgement. You must enter a Fuel Charge Product Number and a Fuel Type. Following fields:
•
N - Fuel charges do not apply.
•
T - Fuel charges apply and are calculated from the number of gallons in the tank for this equipment ID from Equipment Maintenance.
When the rental product is returned, you must enter the amount of fuel consumed.
Fuel Charge Product Number
The product number that designates fuel charges for rental orders. the product is set up as a service item.
Fuel Type
The fuel type used for this equipment. Fuel types are set up in Fuel Type Maintenance.
Minimum Rental Period
The minimum number of days the customer is allowed to rent the piece of equipment. For example, if the minimum is one day and they only rent if for 2 hours, they are still charged for a day.
U/M
The unit of measure for the Minimum Rental Period (e.g., days, months, hours).
Rent Rate Method
The default rental rate assignment and billing method for rental orders with this product (Rental Order Entry > F5-Hdr Scr > Action Bar > Rental > Rental Rate Method):
•
A - Best rental rate always. The rate is automatically evaluated (and pro-rated if necessary) and moved to the best rate based on the actual time material is rented. For example: 5 days moves to one week, 17 days moves to 2 weeks and 3 days.
•
B - Best rental rate - first billing. The rate is automatically evaluated and moved to the best rate based on the actual time material is rented up to the first billing. This billing is either the return of the material or the first periodic billing. Once first billing occurs, when material is returned, the billing is pro-rated by day.
•
C - Best rental rate - use single. Use the rate and rental period as a single rate based on the billing days (F5-Add Info). For example, if billing days are Monday through Friday and the rental item is billed for 7 days, the billing rate is calculated as 1.4 (1 week + 2/5). The total number of billing days is the denominator.
•
D - Daily rental - no exceptions. Always use the rate and rental period as a daily rate. The item is billed at the daily rate for the number of days between the start date and return date (Monday through Sunday plus holidays). For periodic billing, the daily rate is calculated according to the number of days set for the Monthly Billing Cycle in Rental Parameters.
•
N - No best rental rate. Use the rate and rental period as defined on the rental document and round to the nearest whole rental (e.g., 10 days on a daily weekly rental bills as 2 weeks).
You can override the rental rate for a customer or for a rental order. Refer to the Rental documentation for more information.
Hours Shifted To Day
The number of hours before the hourly rental rate shifts to a daily rate. Only applies if Rental Rate Method in Rental Option Parameters is A or B.
Days Shifted to Week
The number of days before the daily rental rate shifts to a weekly rate. Only applies if Rental Rate Method in Rental Option Parameters is A or B.
Weeks Shifted To Month
The number of weeks before the weekly rate shift to the monthly rate. Only applies if Rental Rate Method in Rental Option Parameters is A or B.
Square Footage Factor
The numeric square footage amount to calculate in order entry if you bill by square foot (e.g., concrete forms). This value calculates the rental rate based on price per square foot, percent off list, or a total dollar amount.
F2-Save
Saves the rental information.
F5-Safety Instr
Allows you to enter notes regarding safety instructions for the rental equipment. This information prints on the rental agreement.
F6-Check-in Notes
Allows you to enter notes regarding check-in (return) for the rental equipment. Prints when the rental product is returned.
F7-Asset Info
Opens the Rental Asset Information Screen to allow you to assign rental asset number information, if you use the Fixed Assets Module for rental products.
