Consolidating Product Counts
After you enter the counts from the tags, you must consolidate counts (for products with multiple tags) before you update physical inventory. The Physical Tag Consolidation Detail Audit Report prints to show the total count and deviation amount. This step is necessary in order for the shrinkage and deviation reports to calculate correctly.
To consolidate product counts:
1
Select Inventory > Physical Inventory > Inventory Tag Procedure > Physical Tag Consolidation Procedure to open the Physical Tag Consolidation Procedure screen.
2
Enter the Warehouse.
3
Indicate if you want to include Rental Products in the consolidation procedure. Choose from the following:
•
B - Consolidate all products, including rental.
•
Y - Consolidate only rental products. Exclude all other products.
•
N - Consolidate all products, except rental.
4
Press ENTER, the following message appears:
Consolidation will print the Detail Audit
Report, and needs to be run again if there
are any additional tags entered before running the Physical Update Procedure. Do you wish to continue?
5
Choose from the following:
•
No - Cancels the procedure.
•
Yes - Consolidates the items with multiple tags and prints the Physical Tag Consolidation Detail Audit Report.
If you change any counts, you must consolidate the products again.
Column Descriptions
Column Heading
Description
Product Number/ Description
Line 1: The product number.
Line 2: The product description.
Tag #
The tag numbers assigned to the product. If the product has multiple tags, all tag numbers display.
Lot #
If lot numbers exist for the product, the number and count for each lot display.
Bin Location
The bin location for the product.
U/M
The unit of measure for the product.
Count
The count entered for the tag number.
Booked
LinkOH
The quantity of the product linked to a sales order, work order, or transfer.
Link $Val
The value of the linked quantity.
StkOH
The saved on-hand quantity.
Dev
The deviation amount between the count and the saved on-hand quantity.
Avg-Cost
The cost of the item in this warehouse.
Dev $ Value
The total dollar amount of the deviation quantity (Dev x Avg-Cost).
