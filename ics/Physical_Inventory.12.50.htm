Creating an Inventory List by Vendor
To build a physical inventory list by vendor:
1
Select Inventory > Physical Inventory > Radio Frequency Procedure > Build Physical Inventory Lists > By Vendor(s) to open the Physical Inventory List By Vendor screen.
2
Enter the Warehouse for the inventory list.
3
For Physical ListName, enter a name to identify the count list. You will use this name to enter counts, update inventory, and run inventory reports.
4
For Vendor, enter one or more vendors to include in the count lists. Only products with these primary vendors assigned in the Product Warehouse Maintenance Screen will be included. Press F3 to search for vendor numbers or leave blank to include all vendors.
5
Press F2-Create List. The following prompt appears:
## Items Selected. Press Return.
6
Press ENTER to create the lists for the selected bins.
To view the list, refer to Displaying Physical Inventory Lists.
