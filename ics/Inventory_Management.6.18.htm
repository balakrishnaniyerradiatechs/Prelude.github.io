Defining Warehouse Areas
Before setting up bins and zones, you can divide your warehouse so that each area contains a group of zones and each zone contains a set of similar bins.
Warehouse Areas Listing
To view a list of warehouse areas for a warehouse, select Inventory > Listings > Miscellaneous Listings > Warehouse Areas.
To set up warehouse areas:
1
Select Inventory > Maintenance > Bin Files > Warehouse Areas to open the Warehouse Area Maintenance screen.
2
Enter the Warehouse that contains the warehouse areas.
3
For Warehouse Area, enter up to 4 alphanumeric characters to identify this area.
4
Enter a Description of the warehouse area.
5
You can enter a Group Code for information purposes only.
6
Press F2-Save to save the warehouse area information.
