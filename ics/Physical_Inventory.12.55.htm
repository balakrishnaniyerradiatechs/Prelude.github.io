Creating an Inventory List for All Products
You can create an inventory list for all products, sorted by vendor, product line, or bin for each warehouse. Depending on which sort option you choose, a separate list generates for each primary vendor, product line, or bin. If any of these lists contain more than 1000 products, a new list begins after 1000 products.
Choose one of the following methods to create an inventory list for all products:
•
To sort the inventory list by vendor, select Inventory > Physical Inventory > Radio Frequency Procedure > Build Physical Inventory Lists > Entire Product File Create > By Vendor(s). A Physical Listname is created for each primary vendor number in each warehouse.
•
To sort the inventory list by product line, select Inventory > Physical Inventory > Radio Frequency Procedure > Build Physical Inventory Lists > Entire Product File Create > By Product Line(s). A Physical Listname is created for each product line in each warehouse.
•
To sort the inventory list by bin location, select Inventory > Physical Inventory > Radio Frequency Procedure > Build Physical Inventory Lists > Entire Product File Create > By Bin Location(s). A Physical Listname is created for each bin location in each warehouse.
To view the lists, refer to Displaying Physical Inventory Lists.
