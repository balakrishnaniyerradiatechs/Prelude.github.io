Product Book Sample Maintenance Screen
The Product Book Sample Information screen allows you to enter patterns and book codes to track book samples sent to customers. This screen is only accessible if you are using the Book/Sample Subsystem in the Order Processing module.
Book Sample Subsystem
The Book Sample Subsystem in the Order Processing module is an additional module that you must purchase from Activant.
The Product Book Sample Maintenance screen is only accessible if Books/Samples is set to Y in Company Maintenance > F6-modules.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Book /Sample Information
Field & Function Key Descriptions
Field/Function Key
Description
Type
The book type to setup for the product:
•
B - Book.
•
O - Other.
Book Code
The code created in Book Maintenance that represents the products that make up a book.
Pattern
If Type is O, you can enter the pattern of the book code to use to track book samples. The pattern becomes a product keyword.
F2-Save
Saves the book sample information.
