Purchase Parameters Additional Information Screen
The Purchase Parameters Additional Information screen allows you to enter additional factors that may affect recommended purchase quantities.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Purchasing Parameters Maintenance > F7-Add Info
Field & Function Key Descriptions
Field/Function Key
Description
T/R Reserve Formula
The formula to calculate the sales forecast. You can use this to reserve inventory for sales for this warehouse so that it is not available for transfers.
Percentage
The percentage of the calculated forecast to reserve for customer orders that are not available for warehouse transfers.
Quantity
If T/R Reserve Formula is blank, you can enter a set Quantity that is not available for transfers.
Ann. Usage Override
The override to annual usage. The system divides this value by 12 to get an average monthly usage override.
W/O Mfg. Production
Indicates if work order production calculates usage:
•
Y - Only sales orders and transfers calculate usage.
•
N - Sales orders, transfers, and work orders calculate usage.
This setting overrides company & product level settings.
Single Customer
Indicates if the product’s usage is controlled primarily by one customer:
•
Y - This product’s usage is controlled primarily by one customer. You must set purchasing parameters manually.
•
N - This product’s usage is not controlled by one customer.
Single Customer Number
If you set Single Customer to Y, enter the customer that primarily controls the product’s usage.
Distribution Ctr
Defines the warehouse that tracks product usage for replenishment purposes. If this product is shipped from any warehouse other than the selling warehouse, usage is updated in the distribution center’s product warehouse record so that a warehouse only maintains enough stock to meet their demands. Overrides the warehouse-level distribution center. Refer to Recording Product Usage.
F2-Save
Saves the additional purchase parameter information and returns to the previous screen.
