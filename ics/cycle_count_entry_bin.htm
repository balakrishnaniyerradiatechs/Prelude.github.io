Cycle Count Entry screen
The Count Entry screen allows you to enter the second level count for a product.
Menu Path: Inventory > Physical Inventory > Cycle Counting Procedure > Paper Site Cycle Count Entry (2nd Level) > ENTER on Bin Number
Field and Function Key Descriptions
Field/Function Key
Description
Bin
The bin counted.
Product Number
The product counted.
System Onhand
The quantity that the system currently shows as on-hand.
1st Level Disc
The discrepancy amount between the System Onhand and the first level count.
Actual Count
The second level count.
