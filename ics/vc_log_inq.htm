Vendor Claim Log Inquiry
The Vendor Claim Log Inquiry lists all transactions that affected a vendor claim, in order from the first to the most recent. You can view the detail information for each transaction.
Viewing Vendor Claim Transactions
To display transactions for a vendor claim:
1
Select Inventory > Inquiries > Vendor Claim Inquiries > Vendor Claim Log Inquiry to open the Vendor Claim Log Inquiry screen.
2
Enter the Vendor Claim Number for which you want to display transactions. Press F3 to search for claims by vendor.
Transaction codes for the vendor claim display.
Field & Function Key Descriptions
Field/Function Key
Description
Vendor Claim Number
The vendor claim number for which you want to view transactions.
Code
The vendor claim log inquiry code that references the type of transaction. Refer to the Vendor Claim Log Inquiry Codes for code definitions.
Vendor
The vendor for whom the claim was issued.
Act Date
The date the corresponding action occurred.
Time
The time the corresponding action occurred.
User
The user ID of the person who entered the transaction.
Reference Number
The document number or vendor number for the transaction, depending on the transaction code. Refer to Vendor Claim Log Inquiry Codes for an explanation of the reference numbers.
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the Vendor Claim History Detail screen to allow you to view detail information for each vendor claim.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Output
Allows you to view, print, email, or fax the usage information.
