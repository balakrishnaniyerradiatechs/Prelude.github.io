Receiving Cores in Inventory Adjustment Entry
When the customer returns a core, you can adjust inventory to receive the core by applying it to a pending core record or by placing it in the core bank. You can also adjust inventory to correct a core bank error.
You can adjust core inventory and print a core receipt for the customer by:
•
Applying a Returned Core to a Pending Core.
•
Returning a Core Item to a Core Bank.
•
Adjusting a Core Item Out of a Core Bank.
