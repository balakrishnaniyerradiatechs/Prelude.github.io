Merging Catalog Products into Inventory Count Lists
The inventory count file contains stock items and non-stock items that your warehouse stocks and tracks. The catalog file contains all products that your vendors carry that you store for reference or products that you no longer sell. If any products in the catalog file show an on-hand quantity in your warehouse, you can temporarily move these products into your inventory count file so that they will appear on your count sheets.
Although these products are automatically moved to the inventory file when sold, you should merge the files before you build the physical inventory count lists to catch any inconsistencies that may occur. You can merge products more than once.
To merge catalog products to the inventory count list, select Inventory > Physical Inventory > Inventory Sheet Procedure > Merge Non-Stock Products W/Onhand.
After your physical inventory is complete, you can use the N/S Product Merged Report & Process to view the products and move the ones with a zero on-hand quantity back to the catalog file.
