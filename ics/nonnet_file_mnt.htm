Nonnet File Maintenance screen
The Nonnet File Maintenance screen allows you to move items from non-nettable inventory into nettable (available) inventory. This procedure is necessary if you use the carousel interface and receive material into non-nettable inventory.
Refer to Receiving Carousel Non-Nettable Products for more information.
Menu Path: Inventory > Processes > Nonnettable File Update Entry
Field and Function Key Descriptions
Field/Function Key
Description
Product
The product number to move into nettable inventory.
Warehouse
The warehouse number for the non-nettable product.
Type
The transaction that received the product into non-nettable inventory:
•
PO - The product was received on a purchase order.
•
TR - The product was received on a transfer.
Document#
The document number that the non-nettable product was received on. The first three digits identify the receiving warehouse.
Qty
The quantity received on this document number.
Rec Date
The date the product was received.
Rec Time
The time the product was received.
F4-Del
Deletes the non-nettable information.
