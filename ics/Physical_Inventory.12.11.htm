Creating an Inventory List Using a TCL Command
To build a physical inventory list using a TCL command (see your systems administrator for information on TCL commands):
1
Select Inventory > Physical Inventory > Inventory Sheet Procedure > Build Physical Inventory Lists > By Manual Defined Criteria to open the Physical Inventory List By Other screen.
2
Enter the Warehouse for the inventory list.
3
For List Name, enter a name to identify the count sheet. You will use this name to print count sheets, enter counts, update inventory, and run inventory reports.
4
Enter the TCL command to create the list of products.
5
Press F2-Create List. The following prompt appears:
## Items Selected. Press Return.
6
Press ENTER to create the lists for the selected product lines.
To view the list, refer to Displaying Physical Inventory Lists.
