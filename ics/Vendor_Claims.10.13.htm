Assigning SSCCs to Vendor Claims
A Serial Shipping Container Code (SSCC) is a unique identification system used to identify a carton or pallet. SSCCs provide quality assurance by assigning codes and printing labels to verify shipped products and quantities.
An SSCC is a 20 character number where each number represents container information. For example, the code 00006123450000000517 breaks down as follows:

Vendor Claim Serial Shipping Container Entry allows you to assign SSCCs to identify the contents of a vendor claim.
You must print a vendor claim before you can assign an SSCC.
To assign an SSCC to a vendor claim:
1
Select Inventory > Vendor Claim > Vendor Claim Serial Shipping Container Entry to open the Vendor Claim Shipping Serial Container Entry screen.
2
Enter the Vendor Claim# to register the product detail on this SSCC. Press F3 to search by vendor number.
3
For Serial Ship Code, press ENTER to generate a new SSCC.
4
Enter the Package Type to define the type of container. Choose from the following:
•
C - Carton.
•
P - Pallet.
Warehouse Maintenance
If SSCC Label Execution in Warehouse Maintenance is set to N, the vendor’s SSCC Label Program prints the correct label. You can then put the label on the carton/pallet to register.
5
Enter the Product to register. Press F5-Review Vendor Claim to view items on the vendor claim. If the item is serialized, you must enter the serial number information.
6
Enter the Quantity packed in this container.
From the Action Bar, choose NonScan Inquiry to display all items that are not completely packed or scanned.
7
After all products are registered, choose from the following to complete the transaction:
•
If the claim has more line items, enter ENDSCAN on a blank line or scan a label with the ENDSCAN text. A new SSCC appears to assign to the next item on the claim.
•
If this is the last item on the claim, type FINALSCAN on a blank line or scan a label with the FINALSCAN text. The system checks to make sure all items on the claim have an SSCC. The SSCC Label Program defined for the warehouse executes to print labels. You can attach the labels to all the registered cartons/pallets.
•
If you want to re-check the product information, press F2-Save. The following prompt appears:
Choose from the following:
Are you Finished With Registering All
Contents for This Release?
-
Yes: Allows you to check the product information for each code to make sure it matches the expected vendor claim merchandise. Once everything matches, the transaction is complete. The SSCC Label Program defined for the warehouse executes to print labels. You can attach the labels to all the registered cartons/pallets.
-
No: Allows you to continue registering the next SSCC.
-
Suspend: Saves the information so you can assign more SSCCs later.
-
Return: Returns to the screen to allow you to enter additional information.
