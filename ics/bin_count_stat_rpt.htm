Bin Count Statistics Report
The Bin Count Statistics Report displays the number of times bins were counted for each month and for the year, the total number of actual product counts, and the total number of product count rejects. The calculated percentage of accuracy shows you how accurate the system on-hand inventory was compared to the counts.
Menu Path: Inventory > Physical Inventory > Cycle Counting Procedure > Bin Count Statistics Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Warehouse. The warehouse to display bin count information for.
•
Cycle Count Zone. The cycle count zone to view. Leave blank to include all zones.
•
Last Cycle Count Date Range. The beginning and ending Last Cycle Count dates to include in the report. Any bins that were last counted on or between these dates are included in the report.
•
Bin Number. The bin location to view. Leave blank to include all bins.
•
Count Month Range. The beginning and ending months to include in the report. Any bins with a last cycle count date within this date range are included in the report.
Column Descriptions
Column Heading
Description
Bin
The bin numbers included in a zone.
Description
The description of the bin.
Freq
The frequency to count the bins per year. Determined in Zone Maintenance.
Monthly
Freq Bin Ct
The number of times the bin was counted this month.
Prd Ct
The number of products counted in the bin.
Prd Rej
The number of times a product count was rejected, meaning it was sent to the second level count queue, or had inventory adjusted. Refer to the Update 1st level Adj as Rejects flag in Warehouse Maintenance.
Accur%
The percentage of accuracy:
(Prd Ct - Prd Rej) / Prd Ct.
Year To Date
Bin Ct
The number of times the bin was counted this year.
Prd Ct
The number of products counted in the bin.
Prd Rej
The number of times a product count was rejected, meaning it was sent to the second level count queue, or had inventory adjusted. Refer to the Update 1st level Adj as Rejects flag in Warehouse Maintenance.
Accur%
The percentage of accuracy:
(Prd Ct - Prd Rej) / Prd Ct.
