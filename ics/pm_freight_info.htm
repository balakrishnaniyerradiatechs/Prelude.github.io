Freight Information Screen
The Freight Information screen allows you to enter freight class and code information for a product.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > F8-Freight
Field & Function Key Descriptions
Field/Function Key
Description
Freight Class
The freight class for the item. Refer to Freight Class for information on setting up freight classes. The freight class uses the weight of the product to determine the amount to charge a customer for freight. This freight class overrides the Freight Class in Product Line Maintenance.
Haz Material Class
The Hazardous Material Classes set up by the Office of Hazardous Materials Safety within the U.S. Department of Transportation. These classes identify products that require hazardous material information to print on a bill of lading. Refer to Hazardous Material Class for information on setting up the classes.
Haz Material Code
The hazardous material codes assigned in the Hazardous Material Class Maintenance screen that may apply to this hazardous material class. These codes also determine the packing requirements for the product.
NMFC Code
The National Motor Freight Classification that describes what the material is and how easy it is to transport. This information is used with the Freight Class to determine shipping rates. Refer to NMFC Codes for information on setting up NMFCs.
Slap Shipper
Indicates if a special label is required for products shipped in one box and not repacked:
•
Y - This product requires a slap ship label.
•
N - This product does require a slap ship label.
Freight Rate
The freight to charge for this product, if you estimate freight. If you enter a value here, it is excluded from the evaluation for freight charges in the ship via record.
Exclude From Freight
Determines whether to include/exclude this product to estimate freight using a ship via freight table:
•
Y - The product is not used to estimate freight. You can enter the Start Date, End Date, and the Campaign Codes that apply when excluding freight (e.g., promotional items).
•
N - The product is used to estimate freight. The remaining fields are not applicable.
Start Date
The start date to exclude from freight estimate calculations if you entered Y in Exclude From Freight.
End Date
The end date to exclude from freight estimate calculations if you entered Y in Exclude From Freight.
Campaign Codes
The campaign codes to exclude from freight calculations if Exclude From Freight is set to Y. If campaign codes exists for the product, it is excluded from freight calculations between the Start Date and End Date.
F2-Save
Saves the freight information.
