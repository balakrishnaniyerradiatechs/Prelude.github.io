Batch Stock Receipts by Seal screen
The Batch Stock Receipts by Seal screen allows you to receive stock for all purchase orders listed on a license plate seal. Refer to Receiving Stock For a Seal Number for more information.
Menu Path: Inventory > Stock Receipts > Batch Stock Receipts > Batch Stock Receipts by Seal
Field and Function Key Descriptions
Field/Function Key
Description
Seal #
The seal number from the shipping notice that includes all of the purchase orders for a license plate.
