Establishing the Weekly Usage Calculation Hierarchy
Before you calculate weekly usage, you must indicate the number of weeks to use in the calculation and determine if usage is captured at the company level, the product level, or the product-warehouse level.
Capturing Weekly Usage at the Company Level
To calculate weekly usage for all products and warehouses:
1
Select Inventory Parameters from the Systems menu.
2
Set Capture Weekly Usage to Y to open the Weekly Usage Options screen.
3
For Number of Weeks - Usage, indicate the number of weeks to use to calculate average weekly usage. You can override this setting at the product level.
4
For Classify by Week, enter Y to classify products using weekly usage (refer to Classifying Products). If you enter N the rest of the fields do not apply.
5
For Number of Weeks - Class, indicate the number of weeks to use to classify products.
6
For Classification Cutoff, enter the classification level for products that should use the Average Weekly Usage (AWU) formula. Each product in a warehouse that is this classification or higher will have its purchasing formula automatically set to AWU. Each product in a warehouse with this classification or lower, automatically has its purchasing formula set to EOQ.
 
Capturing Weekly Usage at the Product Level
To calculate weekly usage for a product in all warehouses:
1
Select Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Purchasing Controls to open the Purchasing Controls Screen.
2
Set Default Formula Method to AWU (average weekly usage). The following message appears:
Change methods on
3
Choose from the following:
•
All Whses - Updates this product with the AWU purchasing formula for all warehouses.
•
Whses With Same Meth - Updates this product with the AWU formula only in warehouses that contain the old purchase method.
•
No Changes - Does not update the purchasing formula.
4
Press F2-Save twice to save the weekly usage information.
You can override this information for the product in each warehouse in Product Purchasing Parameters Maintenance.
 
Calculating Weekly Usage at the Product/Warehouse Level
To calculate weekly usage for a product for a warehouse (and override the weekly usage at the product level):
1
Select Inventory > Maintenance > Product Purchasing Parameters Maintenance to open the Product Purchasing Parameters Maintenance Screen.
2
Set Purchase Method to AWU to record average weekly usage.
3
In the next (unlabeled) field, enter the number of weeks to use to calculate usage. If you leave this field blank, the system uses the number of weeks established in Inventory Parameters. The recommended quantity is calculated just like the EOQ, except it uses 6 weeks instead of 6 months and 7 days instead of 30.
4
Press F2-Save to save the weekly usage information.
