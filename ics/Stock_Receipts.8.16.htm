Entering Label Information When Receiving Stock
If your products are set up to print labels, the Labels Information screen appears after you receive the quantity for each line item. This screen is only accessible if you chose Line Item to receive the stock.
If you chose Complete or Exception to receive stock, the Labels Information screen is not accessible and the quantity of each product received determines the number of labels that print.
To enter label information:
1
For Num Lbls, indicate the number of labels to print for the product. This number controls how many labels print instead of just using the quantity received. For example, you may receive a quantity of 50, but you only want to print one label because all 50 items are stored in the same box.
2
For Qty Lbl, enter the quantity to print on each label.
3
For Size, indicate the label size to print. If you use both of the following label sizes, additional programming is required:
•
S - Small.
•
L - Large.
4
Press F2-Save to save the label information and to continue receiving stock.
The Purchase Order Receiving Report prints barcode labels for each product and purchase order number according to the maintenance flag settings.
To print barcodes on the report, you must set the Stock Receipts Printer Type to the appropriate printer type in Warehouse Maintenance > F7-RTS Maint.
