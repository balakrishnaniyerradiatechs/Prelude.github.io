Bin Move Default Bin Entry screen
The Bin Move Default Bin Entry screen allows you to move all the products from one bin to another and to change quantities or primary bins for each product.To move products from one bin to another without changing quantities or primary bins, refer to Entering Bulk Bin Moves.
Menu Path: Inventory > Inventory Adjustments > Bin Individual Move Entry > F5-Load Bin
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for the bin move.
Bin
The bin that contains the products.
Load Quantities
Determines if you want to move existing on-hand quantities for each product:
•
Y - Defaults the current on-hand quantity for each product to the designated To Bin.
•
N - Allows you to manually enter each quantity to move to the To Bin when you return to the Bin Individual Move Entry screen.
Default To Bin Type
Determines if you want to default the quantities to the product’s primary bin or if you want to manually enter each To Bin:
•
P - Automatically defaults the product’s primary bin for the To Bin.
•
M - Allows you to manually enter a To Bin for each product. You can change products within the bin when you return to the Bin Individual Move Entry screen.
Default To Bin
If you entered M for Default To Bin Types, enter a default bin to move all the products in a bin to. You can change these bins when you are in the Bin Individual Move Entry screen.
F2-Finish
Saves the default bin information and brings you back to the Bin Individual Move Entry screen.
