Printing a Transfer Packing List
If your warehouse does not continuously print transfer packing lists after transfers are shipped, you can batch print the packing lists.
Transfer Shipments Maintenance
If you use the Transfer Shipments Maintenance method, (refer to Transfer Shipments Maintenance), you must continuously print packing lists.
To batch print transfer packing lists:
1
Select Inventory > Stock Movements > Transfer Packing List Printing .
2
Enter the From Warehouse for the shipping warehouse.
3
For Transfer Number, enter the transfer release number.
4
For Printer Form Queue, enter the printer number.
5
Once you choose EXECUTE, the following prompt appears:
Load Packing List Forms in Printer & Align Visually
6
Choose from the following:
•
Print Packing Lists: Prints the packing lists.
•
Mask Print Test: Tests the alignment of the packing lists in the printer (not necessary if using a laser printer). After the forms are correctly aligned, select Print Packing Lists.
•
Abort Printing: Cancels the print job.
7
After you select Print Packing Lists, the following prompt appears:
Packing List Printing Complete
Choose one of the following:
•
Process: Completes the print process if all packing lists printed successfully.
•
Reprint All: Reprints all packing lists if none printed successfully.
•
Single Reprint: Allows you to reprint one or more packing lists individually.
•
Abort: Cancels the print process.
Reprinting a Packing list
If you want to reprint a packing list, select Inventory > Stock Movements > Transfer Packing List Reprinting. Enter the warehouse and the transfer release number.
