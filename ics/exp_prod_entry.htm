Expanded Physical Entry screen
The Expanded Physical Entry screen allows you to enter multiple counts for an item in the Physical Count Sheet Entry screen. This screen only displays if the Phys Inv Expanded Count flag is set to Y in Inventory Parameters.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Physical Count Sheet Entry
Field and Function Key Descriptions
Field/Function Key
Description
Product Number
The product number on the count sheet.
U/M
The unit of measure of the product.
Shelf Pkg Qty
The quantity contained in each package of the item.
Units
The number of packages counted. For example, if the item comes in a single unit, enter the total counted. If the item comes in packages of 10, enter the number of packages counted.
Total Count
The total quantity of the product in each package (Shelf Pkg Qty x Units).
Grand Total
The total count of the product (the sum of all the quantities under Total Count).
F2-Save
Saves the count information.
