Batch Transfer Receipts Entry screen
The Batch Transfer Receipts Entry screen allows you to receive multiple transfers under one batch number. To help minimize possible errors, you can run a report to verify products and quantities before you update inventory. Refer to Batch Transfer Receipts Entry for more information.
Menu Path: Inventory > Stock Movements > Batch Transfer Receipts > Batch Transfer Receipts Entry
Field and Function Key Descriptions
Field/Function Key
Description
Transfer Release#
The release number from the transfer packing list (e.g., 10000079-1).
Receipt Date
The date the material arrived in the warehouse.
From
The branch number and description for the shipping warehouse.
To
The branch number and description for the receiving warehouse.
Product
The product on the transfer.
N
An asterisk (*) indicates that notes exist for the transfer receipt.
Description
The product description.
Emg
Indicates if the transfer is an emergency. Used with the warehousing module.
Linkage
Indicates if the product is linked to a sales order or work order.
Qty-Opn
The quantity not yet received on the transfer.
U/M
The unit of measure of the product.
Qty-Shp
The quantity shipped according to the shipping warehouse.
Qty-Rec
The quantity received according to the receiving warehouse.
F2-Save
Saves the transfer receipts information.
F4-Del
Deletes the transfer receipt information.
F5-Header
Opens the Stock Transfer Header screen to allow you to change any shipping information.
F7-Wty Tag
Not accessible when receiving a transfer.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Examine
Displays the cost, supply code, and the original transfer number.
Review
Opens the Transfer Review screen to allow you to view the shipping information for a transfer.
Notes
Allows you to enter or view header or line item notes for the transfer.
