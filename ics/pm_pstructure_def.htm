PStructure Definition screen
The PStructure Definition screen allows you to define the combination of kit parent items that create a product structure. A product structure or pstructure number is a string of characters that represent multiple kits using an infinite combination of kit component items.
Refer to PStructures for more information.
Although product structures are still a valid option, you may find it easier to use one of the following newer Prelude features to accomplish the same results:
•
Product characteristic codes, which define special ordering requirements for each product you sell (refer to Product Characteristics).
•
Configured products, which allow you to create child products for all the possible combinations of a product (refer to Configuring Base and Child Products).
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F6-Xref/Notes > PStructure
Field and Function Key Descriptions
Field/Function Key
Description
Length
The length of the product structure (i.e., the number of digits a pstructure definition must be when this product is entered on a sales order).
Product Line
The product line of the kit parent.
Segment Definition
The kit parent item that joins other kit parent items to comprise the pstructure. When you enter this product and pstructure on a sales order, the system builds a component list of all the combined kit items.
F2-Save
Saves the pstructure information.
