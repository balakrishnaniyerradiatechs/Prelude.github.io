Configuration Code Maintenance screen
The Configuration Code Maintenance screen allows you to set up configuration codes to identify the types of choices you have when you create a child product from a base product. Each base product can have multiple child products. For example, if you sell a t-shirt that comes in many sizes and colors, the configuration codes used to create child products might be Size and Color. Refer to Configuring Base and Child Products for more information.
Menu Path: Inventory > Maintenance > Product Maintenance > Misc Codes Files > Configuration Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Configuration Code
The code to identify the types of choices available (e.g., color, size, length). You can use these codes in the Product Configuration Maintenance screen to create all the possible combinations of a product that you stock (refer to Creating the Base and Child Products).
Description
The description of the code.
Options
All possible configurations available for this code. For example, if the Configuration Code is COLOR, enter all colors available to sell.
Abbr
The abbreviation for this option with a child product. For example, if the code is for color and the Option is Red, enter R to designate the color red. If the code is for size and the Option is Extra Large, enter XL to designate an extra large size.
Desc Abbr
The abbreviation to use to create the child product description. For example:
•
Base product: TS2309.
•
Base product description: T-Shirt.
•
Child product: TS2309-XL-R.
•
Child product description: T-Shirt X-Large Red.
If the description is longer than 30 characters, the extra characters go into Description Two for the child product.
F2-Save
Saves the configuration code information.
F4-Del
Deletes the configuration code.
