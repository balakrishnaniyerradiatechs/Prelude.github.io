Updating Multiple Products
The Global Product Update Process allows you to update a range of products or product lines at the same time to simplify the process of changing product descriptions, prices, or other routine information.
File Update Parameters
Set Product Maint Audit in File Update parameters to Y to track the product updates. The changes to the products appear on the Product Maintenance Audit Report by User ID and the Product Maintenance Audit Report by Product.
To globally update products, you must:
•
Create a Product List.
•
Create an Update Template.
•
Update Products.
