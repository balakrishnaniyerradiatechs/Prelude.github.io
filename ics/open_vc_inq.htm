Open Vendor Claim Inquiry
The Open Vendor Claim Inquiry lists all claims entered for a vendor that are not yet updated in Vendor Claim Update Processing (refer to Updating Vendor Claims). The quantity on the claim still exists in on-hand inventory. You can view the detail information for each claim.
Viewing Open Vendor Claims
To view detail information on open vendor claims:
1
Select Inventory > Inquiries > Vendor Claim Inquiries > Open Vendor Claim Inquiry to open the Open Vendor Claim Inquiry screen.
2
Enter the Vendor.
Open vendor claims for the vendor display.
Field & Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor with open inquiries.
V/C#
The vendor claim number.
N
An asterisk (*) indicates that notes exist on the vendor claim.
Type
The vendor claim type:
•
Regular. A regular vendor claim for on-hand or warranty inventory.
•
Vendor Show Claim. Refer to Entering a Vendor Show Claim.
•
Vendor Consigned Inventory. Consigned inventory refers to products in stock that you do not pay for until sold.
•
Non-Inventory Expense.
V/C Date
The date the vendor claim was entered.
S
Status:
•
Y - Printed.
•
P - Reprinted.
•
F - Faxed.
•
B - Printed and faxed or emailed.
Emg
Indicates if the vendor claim is an emergency. Used with the warehousing module.
Vendor Order#
The vendor’s order number for the claim, if applicable.
Units
The number of units on the vendor claim.
V/C Dollars
The value of the vendor claim.
F2-Finish
Closes the vendor claim inquiry screen.
F5-Review V/C
Opens the Vendor Claim Display screen to allow you to view detail information for each vendor claim.
F6-Enter V/C
Allows you enter a vendor claim number for direct access.
F8-Log
Opens the Vendor Claim Log Inquiry to display all transactions for a vendor claim.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Previous Group
Displays the previous page of claims if more than one page of claims display.
sF6-Next Group
Displays the next page of claims if more than one page of claims display.
Action Bar
Output
Allows you to view, print, email, or fax the usage information.
