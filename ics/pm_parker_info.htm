Parker Information Screen
The Parker Information screen allows you to enter Parker division and product code information required by the manufacturer, Parker Hannifin, to track and report all sales of Parker products. Refer to Parker Reporting Codes for more information.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Parker Information
Field & Function Key Descriptions
Field/Function Key
Description
Parker Division Code
The Parker division code assigned to the product. Refer to Setting Up Parker Divisions.
Parker Marketing Code
The Parker marketing code that represents the type of product.
Refer to Setting Up Parker Marketing Codes.
Parker Product Code
The Parker product code for the type of product. Refer to Assigning Product Codes to Parker Divisions.
F2-Save
Saves the Parker information.
