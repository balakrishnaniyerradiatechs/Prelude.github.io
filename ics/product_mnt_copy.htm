Product Maintenance Copy screen
The Product Maintenance Copy screen allows you to copy product information from an existing product to a new one. If you have products that are very similar except for color, size, or other repetitive characteristics, you can use this feature to create a new product without having to re-enter most of the detail information.
Refer to Copying a Product to Create a New Product for more information.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Copy
Field and Function Key Descriptions
Field/Function Key
Description
From Product Number
The original product number from which you want to copy information.
To Product Number
The new product number to create.
Description
The description for the new product. Change as necessary.
Description Two
The second line of description for the new product number, if applicable. Change as necessary.
F2-Save
Adds the new product number to your master product file.
