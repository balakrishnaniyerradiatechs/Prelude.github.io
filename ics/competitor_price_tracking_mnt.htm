Competitor Price Tracking Maintenance screen
The Competitor Price Tracking Maintenance screen allows you to set up competitors’ part numbers and prices to cross-reference any part number. This product number becomes an alternate part number in Product Maintenance.
Refer to Maintaining Competitor Prices & Part Numbers for more information.
Menu Path: Inventory > Maintenance > Product Maintenance > Competitor Price Tracking Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Product
The existing product number for which you want to set up a competitor’s part number.
Code Number
The code from Competitor Code Maintenance (refer to Setting Up Competitor Codes) for the competitor whose prices you want to track.
Competitor Product
The competitor’s product number. The number is automatically converted to uppercase. Dashes and special characters are deleted.
Description
The description of the competitor’s product number.
Product Match
The product match code:
•
E - Exact match. The products are exactly the same.
•
P - Private label match. The products are the same but have different manufacturer labels.
•
S - Similar match. The products are similar but not exactly the same.
Product Match codes are established in Valid Response Maintenance using the validation code PROD.MATCH.
Retail Price
The competitor’s retail price.
Good Price
The competitor’s best price.
Unit of Measure
The product’s unit of measure.
F2-Save
Saves the competitor price information.
F4-Del
Deletes the competitor price information.
