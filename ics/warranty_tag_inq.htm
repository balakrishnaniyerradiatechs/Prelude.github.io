Warranty Tag Inquiry Screen
The Warranty Tag Inquiry screen displays detail status information for a product. You can view the date each transaction occurred, along with the current status of the warranty tag product.
To access this information, press F5-Review Tag from any of the warranty tag inquiries.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number assigned to the warranty tag.
Customer
The customer number if the warranty tag was assigned in Credit Memo Entry on a return. Refer to Creating Warranty Tags in Credit Memo Entry for information on assigning warranty tags.
Vendor
The primary vendor for the product.
Warehouse
The warehouse where the tagged product exists.
Tag Number
The tag number assigned to the product. Refer to Moving Products into Warranty Inventory for information on assigning a tag number to a product.
Status
The warranty tag status code that represents the last transaction that occurred for the item:
•
A/P - Vendor claim reconciled with the vendor’s credit memo.
•
AUPD - Credit memo processed in Invoice Processing.
•
CLM - Vendor claim entered for the tagged item.
•
IAE - Inventory adjusted to put the item into warranty inventory.
•
R - Tagged item rejected in Stock Receipts Entry.
•
SOE - Credit memo entered to the customer for the tagged item.
•
TE - Stock transfer entered for the tagged item.
•
TR - Stock transfer received for the tagged item.
•
TRR - Inventory adjusted to take the warranty item out of warranty inventory and put it into regular inventory.
•
TS - Stock transfer shipped for the tagged item.
•
VCP - Vendor claim updated.
•
WTR -Warranty replacement entered for the returned item.
Price
The selling price of the tagged item if it was returned from a customer.
Cost
The cost of the tagged item.
Orig. Qty
The quantity entered on the tag on the original transaction.
Order#
The credit memo number if the warranty tag was created in Credit Memo Entry.
Ord-Date
The date of the credit memo.
Original Ord#
The original sales order number when the customer bought the product.
Invoice#
The invoice number for the credit memo.
Inv-Date
The date the credit memo was processed.
Original Inv#
The customer’s invoice number for the original purchase of the item.
Inv Adj#
The inventory adjustment number if the product was put into warranty inventory using Inventory Adjustment Entry. Refer to Creating Warranty Tags in Inventory Adjustment Entry.
Adj-Date
The date the inventory was adjusted.
Debit#
The vendor credit memo number from Vendor Claim Reconciliation Entry.
Return#
The vendor order number from F5-Add info on the Vendor Claim Header screen.
Rtn-Date
The date the vendor claim was entered.
Return/Scrap
Indicates if the product was returned to the vendor or scrapped:
•
R - Returned.
•
S - Scrapped.
Transfer#
The transfer number if the returned item was entered on a stock transfer.
Trn-Date
The date of the stock transfer.
Wty Claim#
The vendor claim number.
Clm-Date
The date the vendor claim was entered. Refer to Creating Vendor Claims.
Process Date
The date the vendor claim was updated. Refer to Updating Vendor Claims.
Approved
Indicates if the warranty claim was approved by the vendor:
•
Y - The warranty claim was approved.
•
Null - The warranty claim was not approved.
Wty Replace#
The warranty replacement entry number if the returned item was replaced by the vendor. Refer to Entering Warranty Replacements.
Rpl-Date
The date the warranty replacement was entered for the tagged item.
Qty Replaced
The quantity replaced by the vendor.
Receiver#
The stock receiver number if the item was rejected in Stock Receipts Entry. Refer to Creating Warranty Tags in Stock Receipts Entry.
Rec-Date
The date of the stock receiver which is the date the item was originally received into the warehouse.
Bin
The bin location updated from the Warranty Tag Bin Maintenance screen (Inventory > Vendor Claims > Warranty Tag Bin Maintenance). The inventory in this bin does not affect on-hand inventory, but is only for reference in the warranty tag inquiry screens
Serial#
The serial number(s) if the tagged product was a serialized item.
Remaining
Quantity
The quantity remaining on the tag. If this value is zero, the inventory was all returned or adjusted out.
Cost Ext.
The extended cost of the quantity remaining.
OK
Closes the inquiry screen.
F5-Test Reports
Opens the Test Results Inquiry screen to allow you to view any test results entered for the tagged item.
F6-Rejection Notes
Displays any rejection claim notes if the item was rejected in Stock Receipts Entry (refer to Creating Warranty Tags in Stock Receipts Entry).
F7-Wty Tag Notes
Displays any warranty tag notes for the item if it was tagged for return on a credit memo to a customer. Refer to Creating Warranty Tags in Credit Memo Entry.
