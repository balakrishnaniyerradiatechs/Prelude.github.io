Alternate Products
Alternate products are additional numbers you can use to access a product. When you enter a part number for which alternate products exist, ALT displays at the bottom of the Line Item Detail screen on sales transactions and on the Inventory Inquiry screen.
Examples of alternate part numbers are:
•
A competitor’s part number. Refer to Maintaining Competitor Prices & Part Numbers.
•
The UPC (universal product code). Refer to Receiving Products with UPCs.
•
The UCC (Uniform Code Council) codes that reference the product. This code is helpful when different UCCs exist for different case quantities of the same product.
•
Catalog numbers. You may use a different number for a product when it is used in the catalog. The Order History Report by Catalog Id (Order Processing module) lists sales history information by catalog number.
•
Part numbers set up with a prefix in Scan Prefix Maintenance (System Parameters). Each product in the system has alternate part number added that includes the prefix.
Product Alternates Inquiry
To view alternates set up for a product, select Inventory > Inquiries > Product Inquiries > Product Alternates Inquiry.
To set up alternate part numbers for a product:
1
Select Inventory > Maintenance > Product Maintenance > Alternate Product Maintenance to access the Alternate Product Maintenance screen.
You can also access this screen in Product Maintenance > F6-Xref/Notes > Alternates. Refer to the Product Xref/Notes Screen.
2
Enter the Product for which you want to add an alternate part number.
3
For Alternates, choose from the following:
•
Enter an alternate part number (e.g., a catalog number, competitor’s part number, or any other number to use as a cross-reference on sales orders). The alternate number cannot be a product that already exists in the system.
•
Enter N to assign the next sequential UPC number.
4
For UPC, choose from the following:
•
Y - The alternate part number is a UPC. Refer to Receiving Products with UPCs for information on UPC codes.
•
N - The alternate part number is not a UPC.
5
For UCC (Uniform Code Council), indicate if the alternate product number is for one of the following:
•
M - A master pack UCC code.
•
P - A pallet pack UCC code.
•
S - A UCC code shelf pack.
•
N - Not a UCC code.
•
Y - A UCC code.
6
If the product is for a UCC code, enter the UCC Qty. This quantity defaults in Serial Ship Container Entry when the labels print.
7
For UCC U/M, enter the unit of measure for the UCC code, if applicable.
8
For Catalog ID, enter the identification code for the catalog that contains this product, if applicable. Catalog codes are set up in Catalog Identification Code Maintenance in the Order Processing module.
9
For Alt Ppg, enter the product price group for the alternate part number. Product price groups are used to group products for pricing matrices. This field defaults to the product price group established for the catalog identification code entered in Catalog ID.
10
Press F2-Save to save the alternate product number information.
