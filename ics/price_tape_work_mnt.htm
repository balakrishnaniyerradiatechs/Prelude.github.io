Price Tape Work Maintenance screen
If you find any errors on the Price Tape Pre-Update Report (refer to Running the Price Tape Pre-Update Report), you can use the Price Tape Work Maintenance screen to correct any errors.
Menu Path: Inventory > Processes > Price/Cost Update Procedure > Price Tape Update > Price Tape Work Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Diskette/File Name
The diskette or PICK name from the report.
Vendor
The vendor number associated with the update.
Record Id
The record ID number from the Pre-Update report, which is the sequence number at the end of the file name.
For example: If the file name is STAN2001Price.csv-1006, enter 1006 for the record ID. Press F3 to list file names and record IDs for the vendor.
Product
The product number to correct.
Vendor Product
The vendor’s part number, if applicable.
Ln
The counter number that corresponds to the Fld# on the Price Tape Setup Maintenance screen.
Data
The data for the part number.
File Name
The file to update.
Dict Name
The name of the field where the tape information is stored.
Format
The number of decimal places stored on the field. For example, enter 3 to use 3 places after the decimal (12000 becomes 12.000). Enter D to specify that this field is a date.
F2-Save
Saves the price tape changes.
F4-Del
Deletes the price tape record.
