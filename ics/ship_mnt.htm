Ship Maintenance screen
The Ship Maintenance screen allows you to change the description of a ship or view the status of an on-the-water shipment.
Menu Path: Inventory > Maintenance > Bin Files > Ship Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Ship/Ref #
The same name or number as the ship bin in Bin Location Maintenance.
Description
The ship description.
F2-Save
Saves the ship information.
F4-Del
Deletes the ship information.
F5-Ship Inq
Opens the Shop/Boat Inquiry screen to display the estimated value and the actual value (after it arrives) of each import factor for the ship.
