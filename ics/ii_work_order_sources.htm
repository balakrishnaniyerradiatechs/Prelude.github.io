Work Order Sources screen
The Work Order Sources screen lists work orders in process.
Menu Path: Inventory > Inquiries > Inventory Inquiry > Work Orders (under Sources) > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
W/O#
The work order number that contains the item.
Cre-Date
The date the work order was created.
N
An asterisk (*) indicates that notes exist for the work order.
Req-Date
The date the finished good is required.

Qty-Ord
The quantity of this bill of material that exists on open work orders.
Linkage
The sales orders linked to this bill of material.
F2-Finish
Closes the screen and returns to the Inventory Inquiry screen.
F5-Review W/O
Opens the Work Order Inquiry screen to allow you to view detail information on the work order.
F7-Log
Opens the Work Order Log Inquiry to allow you to view every transaction that has occurred for the highlighted work order.
F9-Goto Line
Allows you to enter a line number, page number or reference number to go directly to.
