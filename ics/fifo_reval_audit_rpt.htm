FIFO Revaluation Audit Report
This FIFO Revaluation Audit report displays projects audited or updated during the FIFO Revaluation Process. You can view the original and new replacement costs and the variances between the two.
Menu Path: Inventory > Processes > FIFO Revaluation Process > FIFO Revaluation Audit Report
Column Descriptions
Column Heading
Description
Product Number
The product number updated.
Description
The product description.
U/M
The unit of measure of the product.
Whse
The warehouse where the product is stocked.
Tier
The FIFO layer created.
Qty
The quantity in the layer.
Cost
The total landed cost of the FIFO layer.
Ext Cost
The extended cost of the FIFO layer (Qty x Cost).
