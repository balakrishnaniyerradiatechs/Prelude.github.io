Product Group Codes Maintenance screen
The Product Group Codes Maintenance screen allows you to set up product group codes used to combine major groups in Commission Matrix Maintenance.
Menu Path: Inventory > Maintenance > Misc Code Files > Product Group Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Product Group Code
The code to reference a product group.
Description
A description of the product group.
F2-Save
Saves the product group code.
F4-Del
Deletes the product group code.
