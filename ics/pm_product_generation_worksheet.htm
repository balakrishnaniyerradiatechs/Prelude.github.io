Product Generation Worksheet screen
The Product Generation Worksheet displays all product combinations available based on the configuration codes and responses you entered in the Product Configuration Maintenance screen. You can edit or delete any of the proposed child products before you link them to the base product.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > Special Operation = CB > F6-Configuration Worksheet.
Field and Function Key Descriptions
Field/Function Key
Description
Proposed Child Product
The products that can be created based on the Configuration Codes and Configuration Responses entered in the Product Configuration Maintenance screen.
Description One
The description of the child product.
Description Two
Additional description of the child product.
F2-Save
Saves the new child products created.
F5-Create Children
Creates the child products. These products are linked to the base product so that when you sell the base product, you can choose which of the child products to sell.
The new child products numbers have a Special Operation Code of CC (configured child) in the Product Maintenance > F5-Add Info screen. Any changes to the base product, such as price or package information, are automatically applied to the child products (unless the child product already has something different entered).
