Standard Description Codes
You can set up standard comments to use for sales orders, quotes, work orders, and purchase orders. First, you must create a code for each comment in Standard Description Code Maintenance; then when you enter an order, you just have to enter the code to pull in the standard comment.
Standard Descriptions Listing
To view a list of standard description codes set up in Standard Description Maintenance, select Inventory > Listings > Miscellaneous Listings > Standard Descriptions.
To set up codes to reference standard comments:
1
Select Inventory > Maintenance > Misc Code Files > Standard Descriptions to open the Standard Description Maintenance screen.
2
For Std Desc Number, enter an alphanumeric code (up to 10 digits) to identify the comment.
3
For Description, enter the standard comment that you want to display whenever the code is entered on a transaction.
4
Press F2-Save to save the standard description code.
Using Standard Description Codes
To enter a standard description code on a transaction; enter a dollar sign, followed by the comment code, followed by a period, followed by the standard description comment code. For example; $C.02
