Physical Tag Deviation Report
The Physical Tag Deviation Report lists items that show a difference between the saved on-hand quantity and the physical count quantity. The report only includes products with a count entered.
This report is only valid if you run it before you update inventory (refer to Updating Physical Inventory From Tag Counts) and after you consolidate counts (refer to Consolidating Product Counts).
Menu Path: Inventory > Physical Inventory > Inventory Tag Procedure > Physical Tag Deviation Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where the inventory was counted.
•
Rental Product. Determines if rental products are included. Choose from the following:
-
Y - Include rental products.
-
N - Exclude rental products.
Column Descriptions
Column Heading
Description
Product Number
The product number.
Description
The product description.
Avg-Cost
The cost of the product (FIFO or average cost).
Booked
LinkOH
The quantity of the product linked to a sales order, work order, or transfer.
StockOH
The saved on-hand quantity.
PhysCnt
The new physical count entered for the product.
Dev-Qty
The deviation in quantity between the Count and the StockOH.
Dev $ Value
The total dollar amount of the Dev-Qty (Dev-Qty x Fifo Avg).
