Cost Quantity Break Table Screen
The Cost Quantity Break Table screen allows you to enter costs based on quantity breaks. You can base the costs on a set dollar amount, a discount percent off of a cost, or a quantity discount table.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F7- Pricing > F9-Cst Qtybrk
Field & Function Key Descriptions
Field/Function Key
Description
Cost Qty Break Type
Indicates how you want to set up the cost quantity breaks:
•
$ - Enter a cost for each quantity break.
•
* - Use a percentage of a default cost.
•
T - Use a discount break table. Refer to Setting Up Discount Break Tables.
Quantity Discount Break Table Number (unlabeled)
If you entered T for Cost Qty Break Type, enter the discount table number set up in Discount Break Table Maintenance. Refer to Setting Up Discount Break Tables.
F2-Save
Saves the cost quantity break information.
