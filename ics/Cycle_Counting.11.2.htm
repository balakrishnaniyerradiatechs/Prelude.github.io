Setting up Your System for Cycle Counting
To use cycle counting, you must set the appropriate flags in the following maintenance programs:
•
Inventory Parameters.
•
Warehouse Maintenance.
•
Zone Maintenance.
•
Bin Location Maintenance.
•
Cost Tolerance Maintenance.
•
Cycle Count Control Maintenance.
