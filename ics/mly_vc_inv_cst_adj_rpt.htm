Monthly Cost Adjustment Journal - Vendor Consigned
The Monthly Cost Adjustment Journal - Vendor Consigned lists vendor consigned products whose cost was adjusted in Vendor Consigned Cost Adjustment Entry. You can run the report monthly.
Menu Path: Inventory > Reports > Inventory Adjustment Reports > Monthly Cost Adjustment Journal - Vendor Consigned
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Vendor. The vendor that owns the vendor consigned inventory.
•
Period. The fiscal period in which inventory was adjusted.
Column Descriptions
Column Heading
Description
Cst Adj#
The adjustment number assigned to the transaction. Refer to Adjusting Vendor Consigned Inventory Cost.
Ent-Date
The date the cost was adjusted.
Product Number
Line 1: The vendor consigned product whose cost was adjusted.
Line 2: The product description.
Line 3: The serial number or lot information, if applicable.
Old Cost
The cost of the product before the adjustment.
New Cost
The revised cost of the product.
Onhand
The quantity of the product in on-hand inventory.
Ext. Amount
The total extended value of the adjusted inventory cost.
(New Cost - Old Cost) x Quantity.
A positive value indicates that inventory cost was increased. A negative value indicates that inventory cost was reduced.
Cost Adjustment Reason
The adjustment reason code and description. These codes are set up in Inventory Adjustment Reason Code Maintenance (refer to Establishing Inventory Adjustment Reason Codes).
The default reason code from Inventory Parameters is used if inventory is adjusted during cycle counting. Refer to Cycle Counting.
