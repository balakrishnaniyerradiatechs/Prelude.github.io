Via Overrides Screen
The Via Overrides screen allows you to determine if inventory from all warehouses or just this warehouse is used to calculate availability for this product/warehouse on the VIA website.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Warehouse Maintenance > Action Bar > Addtl Optns > Via Overrides
Field & Function Key Descriptions
Field/Function Key
Description
Include in Available Calculation
Determines if inventory from all warehouses or just this warehouse calculates availability for this product/warehouse on the VIA website:
•
Y - Include all warehouses in the available quantity.
•
N - Only include this warehouse to calculate availability (standard VIA option).
