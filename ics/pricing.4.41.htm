Updating Prices/Costs Using a Price Tape
You can use a tape, file, diskette, or CD to load and update prices and costs automatically. You can load the information using either the vendor’s part number or your own part number. To update prices and costs using a price tape, refer to the following sections:
•
Setting Up a Price Tape.
•
Loading a Price Tape.
•

•
Correcting Price Tape Updates.
•
Updating the Price Tape Data.
•
Price Tape Update Log.
If you use Trade Services price tapes, the Trade Services tape update procedures on the Price Tape Update menu work the same as the regular price tape update procedures.
Inventory Parameters
Vendor Part# Xref (Tape) indicates if a cross reference file for the vendor part number is maintained when using the price tape updates:
•
Y - Maintain a cross reference file.
•
N - Do not maintain a cross reference file.
 
Disp Save Prev Pricing determines if the “Save Previous Price?” prompt displays when you change a price. Choose from the following:
•
Y - Displays the message to allow you to save the previous price when you change a price.
•
N - Does not display the message or update previous prices.
•
U - Does not display the message, but automatically saves previous prices.
Additional Option Parameters
Mult PN’s Per VPN For Price Tapes determines if all part numbers that are assigned to one vendor product number are updated when you run the vendor price tape update:
•
Y - The vendor price tape will update all of the part numbers that are assigned to one vendor product number.
•
N - If more than one part number is assigned to a vendor product number, the vendor price tape will not update any of the part numbers.
