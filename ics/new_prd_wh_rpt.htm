New Product Warehouse Audit Report
The New Product Warehouse Audit report displays products automatically created for a warehouse when:
•
A stock product in one warehouse is transferred to another warehouse that does not stock the product.
•
A non-stock product that is a stock product at any other warehouse is entered on a sales order, work order, or transfer.
You can use this report to complete the maintenance of the product for the new warehouse.
After the report runs, a dialog box appears to allow you to purge the product information so that it does not appear the next time you run the report.
Menu Path: Inventory > Reports > Miscellaneous Reports > New Product Warehouse Audit Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse with the new automatically created product/warehouse records.
Column Descriptions
Column Heading
Description
Product Number
The product number created for a warehouse.
Description
The product description.
Description Two
The second line of description for the product, if applicable.
Whse
The warehouse containing the new product warehouse record.
