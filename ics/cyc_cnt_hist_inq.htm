Cycle Count History screen
The Cycle Count History screen displays cycle count and product information for a bin.
Menu Path: Inventory > Inquiries > Miscellaneous Inquiries > Bin Location Inquiry > F5-Cycle Count History
Field and Function Key Descriptions
Field/Function Key
Description
Date
The date the products in the bin were counted.
Time
The time the bin was counted.
Level
Indicates the cycle count level:
•
1 - The count is from the 1st level cycle count. Refer to Entering First Level Cycle Counts.
•
2 - The count is from the 2nd level cycle count. Refer to Entering Second Level Cycle Counts.
User
The user ID of the employee who counted the products.
Prd Ct
The number of products counted.
Product (2nd Level)
The product number counted if it was a second level cycle count.
Description
The description of the product.
Description Two
The second line of description for the product, if applicable.
F2-Save
Closes the screen.
