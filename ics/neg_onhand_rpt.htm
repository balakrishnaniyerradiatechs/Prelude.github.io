Negative Onhand Report
The Negative Onhand report displays all products with a negative on-hand balance. You can run this report regularly (e.g., once a week) to detect and correct any inventory discrepancies. All on-hand balances should have a quantity of zero or above so that they will not appear on this report.
Menu Path: Inventory > Reports > Stock Status Reports > Negative Onhand
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse(s) containing negative on-hand balances.
Column Descriptions
Column Heading
Description
Product
The product number.
Description
The product description.
Onhand
The on-hand quantity of the product in this warehouse.
Bin
The product’s bin location.
