Inventory Adjustment Line Detail screen
The Inventory Adjustment Line Detail screen allows you to enter the quantity, cost, and reason code for an inventory adjustment.
Menu Path: Inventory > Inventory Adjustments > Inventory Adjustments Entry > Product Number
Field and Function Key Descriptions
Field/Function Key
Description
Ln#
References the line number on the Inventory Adjustment Entry screen.
Quantity
The quantity to add or subtract from inventory. Depending on the adjustment type entered in the Inventory Adjustments Entry screen:
•
If the adjustment type was INV (inventory adjustment), enter a positive or a negative quantity to adjust inventory.
•
If the adjustment type was DMG (damaged inventory), enter a positive quantity to decrease the on-hand inventory of the product and increase the damaged inventory amount for the product.
•
If the adjustment type was SCR (scrapped/lost inventory), enter a positive quantity to decrease the on-hand inventory of the product and increase the scrapped inventory for the product.
•
If the adjustment type was WTY (Warranty Inventory), enter a positive quantity to decrease the on-hand inventory of the product and increase the warranty inventory for the product. Refer to Creating Warranty Tags in Inventory Adjustment Entry.
U/M
The unit of measure for the product.
Accept
Determines if everything in the screen is correct:
•
Y - Returns to the Inventory Adjustment Entry screen if everything is correct.
•
N - Allows you to change any of the information on the screen.
Cost
The cost of the item being adjusted.
Reason
The reason code that identifies why the item’s on-hand quantity is being adjusted. Refer to Establishing Inventory Adjustment Reason Codes.
Available
The quantity available for sale.
Onorder
The quantity on order.
Stock Status
The stock status of the product:
•
S - Stock.
•
N - Non-stock.
•
C - Consignment.
F2-Save
Saves the detail information and returns to the Inventory Adjustment Entry screen.
F9-Inv Inq
Opens the Inventory Inquiry to display detail on-hand, on-order, and committed information for the product.
