Product Warehouse Maintenance Screen
Once a product is set up in Product Maintenance, you can use Product Warehouse Maintenance to enter product information that is unique to each warehouse (e.g., purchasing parameters, bin locations, pricing). Any data entered in Product Warehouse Maintenance overrides the data in Product Maintenance.
The Product Warehouse Information Listing (Inventory > Listings > Product Listings > Warehouse Information) lists detailed information for each product in a warehouse.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Warehouse Maintenance
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number.
Warehouse
The number for the warehouse that stocks the product.
Primary Bin
The primary bin location within the warehouse where the product is stocked. If Validate Bin Locations in Option parameters is set to Y, then the bin number must be a valid bin established in Bin Location Maintenance. If Validate Bin Locations is set to N, you can enter any free form bin number. Refer to Setting Up Bins.
Second Bin
The secondary bin location within the warehouse where the product is stocked. If Validate Bin Locations in Option parameters is set to Y, then the bin number must be a valid bin in Bin Location Maintenance. If Validate Bin Locations is set to N, you can enter any bin number. Refer to Setting Up Bins.
Default WIP Bin
The default work-in-process bin if you use Work Order Entry in the Warehousing module. Refer to Setting Up Bins.
Primary Case Bin
The primary case bin location if you use the Advanced Warehousing Allocation Rules. Refer to Setting Up Bins.
Primary Case Qty
The full case primary bin quantity if you use the Advanced Warehousing Allocation Rules in the Warehousing module.
Status Code
The stocking status code:
•
S - Stock. The product is a regular stock item for this warehouse and is replenished according to purchase parameters.
•
N - Nonstock. The product is not a stock item for this warehouse. You can only purchase this product if it is linked to a sales order, work order or transfer.
•
C - Consignment. The product is on consignment from the vendor.
Price Group
The price group code for this product used to group products together for pricing. Refer to Establishing Product Price Groups. This price group overrides the price group in Product Maintenance.
Tax Group
The tax group for sales tax overrides.
Freight Rate
The freight rate for the product. Freight rate is the amount per pound, dollar, unit, or volume depending on the freight allocation method for the product’s product line. Refer to Freight Allocation in the Product Line Maintenance Screen.
Taxable
Indicates if the product is taxable when sold from this warehouse:
•
Y - The product is taxable. Does not apply if the customer is tax exempt.
•
N - The product is not taxable.
Print Barcode Lbl
Determines if barcode labels print when the product is received:
•
N - Barcode labels do not print for the product regardless of the vendor flag.
•
Y - Barcode labels print for the product regardless of the vendor barcode setting.
•
V - Barcode labels print based on the vendor setting.
•
R - Barcode labels only print on reports and documents (requires customized programing).
You can override this flag in Product Vendor Maintenance, but it overrides the barcode options in Vendor Maintenance and in Product Maintenance.
Refer toPrinting Barcode Labels when Receiving Stock.
Inspection?
Determines if the product requires inspection by a quality control inspector:
•
Y - The product must be inspected by a quality control inspector. You can enter the inspection manager ID in the following field. Refer to Requiring Quality Control Inspections in Receiving.
•
N - The product does not need to be inspected.
Inspection Manager ID
(unlabeled field)
If Inspection? is set to Y, the inspection manager ID assigned to the product at this warehouse.
Date Established
The date the product was set up in this warehouse.
Last Sale Date
The date the last time the product was sold from this warehouse.
Last Pur. Date
The date the last time the product was purchased for this warehouse.
F2-Save
Saves the product warehouse information.
F4-Del
Deletes the product warehouse information.
F5-Usage
Opens the Product Usage Maintenance Screen. You can view or override usage for the product.
F6-Purch Param
Opens the Product Purchasing Parameters Maintenance Screen. You can view or override purchasing parameters for this product/warehouse combination.
F7-Price/Cost
Opens the Warehouse Pricing Screen. You can view or edit pricing for this product/warehouse combination.
F8-Prd Vendor
Opens the Product Vendor Maintenance Screen. You can enter a vendor number to view or edit information for this product/warehouse/vendor combination.
F9-Lot
Opens the Lot Maintenance Screen to allow you to enter lot information for the product. This function key only displays if the Lot Number Flag in Product Maintenance > F5-Add Info is set to Y.
Action Bar
Notes
Opens the Notes Display screen to allow you to view or edit notes regarding the product. These notes are the same as the notes in the Purchasing Parameters screen and the Distribution Requirements Evaluation Inquiry screen in the Forecasting module.
Addtl Optns
Provides access to the following screens:
•
Warehouse Rental Information Screen.
•
Product Whse Crib Information Screen.
•
Via Overrides Screen.
•
Reserved Inventory Screen. Refer to Reserving Inventory for a Customer.
•
Warehouse Product Notes Screen.
•
Product Whse Transfer Information Screen.
Copy
Opens the Copy Warehouse Utility screen. You can copy the product warehouse information to another warehouse. Refer to Copying a Product to a New Warehouse.
