Ship-To Address screen
The Ship-To Address screen on a vendor claim allows you to change the ship-to address if the material is shipping to someplace other than the vendor’s default shipping address or warranty address.
Menu Path: Inventory > Vendor Claims > Vendor Claim Entry > F5-Hdr Scr > F6-Ship-To
Field and Function Key Descriptions
Field/Function Key
Description
Ship-To Name
The description of the vendor’s ship-to location.
Address Line One
The first line of the vendor’s receiving address.
Address Line Two
The second line of the vendor’s receiving address.
Address Line Three
The third line of the vendor’s receiving address.
City
The city of the vendor’s receiving location.
State
The state of the vendor’s receiving location.
Zip
The zip code of the vendor’s receiving location.
F2-Save
Saves the ship-to address information for the vendor claim.
