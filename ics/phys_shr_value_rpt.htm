Physical Inventory Shrinkage Valuation Report
The Physical Inventory Shrinkage Valuation Report displays the dollar amount of inventory shrinkage for each product, sorted by vendor. The difference between the saved on-hand and the physical count is multiplied by the cost of the product. This report also provides a complete valuation report of your inventory once you update physical counts to on-hand inventory.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Shrinkage Valuation Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse for which you want to display shrinkage values.
•
Rental Product. Determines if rental products are included on the report. Choose from the following:
-
Y - Include rental products.
-
N - Do not include rental products.
Column Descriptions
Column Heading
Description
Product Number
The product number counted.
Description / Lot Number
The product description and the lot number, if applicable.
Fifo-Avg
The cost of the item (FIFO or average).
Book-Oh
The saved on-hand quantity of the item, before the count entry and update.
Book$Value
The extended dollar amount of the total saved on-hand quantity.
P-Count
The quantity entered from the physical inventory count.
Phys$Value
The extended dollar amount of the total quantity counted.
Dev
The difference between the saved on-hand quantity and the counted quantity.
Dev$Value
The extended dollar amount of the total deviation quantity (Dev x Fifo-Avg).
