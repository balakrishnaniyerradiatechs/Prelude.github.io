Product Group Codes
You can use product group codes to combine major groups in Commission Matrix Maintenance. Refer to Major Group Codes.
To set up a product group code:
1
Select Inventory > Maintenance > Misc Code Files >Product Group Code Maintenance to open the Product Group Codes Maintenance screen.
2
For Product Group Code, enter a code to reference a product group.
3
Enter a Description that identifies the product group.
4
Press F2-Save to save the product group code information.
You can use these codes to set up major group codes in Major Group Codes Maintenance. Refer to Major Group Codes.
