Physical Inventory Radio Frequency Count Report
The Physical Inventory Radio Frequency Count Report verifies the counts for the scanned products.
Menu Path: Inventory > Physical Inventory > Radio Frequency Procedure > Physical Count Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where the inventory was counted.
•
Product Number Range. The beginning and ending part numbers to include in the report. Leave blank to print all part numbers.
•
Bin Location Range. The beginning and ending bin locations to include in the report. Leave blank to include all bin locations.
•
Sort ‘P’roduct/ ‘B’in. Indicate the sort sequence for the list:
-
P - Sort by product number.
-
B - Sort by bin location.
Column Descriptions
Column Heading
Description
Bin Location
The bin where the products are located.
Product Number
The product numbers in each bin.
Description
Line 1: The product description.
Line 2: The second level of description for the product, if applicable.
Phys RF Qty
The count entered in Physical RF Entry.
