Transfer Freight Detail Info screen
The Transfer Freight Detail Info screen allows you to enter freight information for a stock transfer.
Menu Path: Inventory > Stock Movements > Transfer Entry (or Transfer Shipments Entry > F5-Header Information > F5-Freight Detail
Field and Function Key Descriptions
Field/Function Key
Description
Freight Amount
The dollar amount of the freight as entered in the Stock Transfer Header screen.
Number of Cartons
The number of cartons shipped.
Box#
The box number for each of the cartons shipped.
Weight
The weight of each box.
Tracking Number
The tracking number.
F2-Save
Saves the freight information.
