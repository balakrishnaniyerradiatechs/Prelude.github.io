Adjusting a Core Item Out of a Core Bank
To adjust a core out of the core bank to correct an error:
1
Select Inventory > Inventory Adjustments > Inventory Adjustment Entry. Refer to Adjusting Inventory Quantities for information on adjusting inventory.
2
For Adj Type, enter INV.
3
For Quantity, enter a negative quantity. The Return Type Info screen displays.
4
For Core Return Type, enter CB. The Core Bank Credit Info screen displays.
5
Enter the Customer Number that the core was applied to.
6
For Adjustment #, enter the original adjustment number.
7
Press F2-Save to save the credit information.
8
Press F2-Save again to complete the inventory adjustment. The general ledger postings are reversed, the received quantity is decreased, and the remaining quantity is increased.
If the Print Core Receiving Tkt in Inventory Parameters is set to Y, you can print a core receiving ticket for the customer. Otherwise you can print the ticket manually (refer to Printing Core Receipt Tickets).
