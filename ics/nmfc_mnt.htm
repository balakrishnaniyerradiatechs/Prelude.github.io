NMFC Maintenance screen
The NMFC Maintenance screen allows you to set up National Motor Freight Classifications to compare transportation cost for commodities. Commodities are grouped into 18 classes (50-500) based on 4 transportation characteristics: Density, stowability, handling, and liability. Once you set up an NMFC, you can assign it to a product in Product Maintenance (refer to NMFC Codes).
This information is used with the freight class of a product for billing purposes (refer to Freight Class for information on setting up the freight class).
Menu Path: Inventory > Maintenance > Product Control Files > Freight > NMFC Code
Field and Function Key Descriptions
Field/Function Key
Description
NMFC Code
