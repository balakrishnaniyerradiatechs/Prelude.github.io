Stock Transfer Pick Error Report
The Stock Transfer Pick Error report displays stock transfer picking errors in which the receiving branch received more or less than the quantity shown as shipped by the shipping branch.
Menu Path: Inventory > Reports > Stock Movements Reports > Stock Transfer Pick Error Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Period. The fiscal period when the transfers were picked.
•
Receiving Branch. The branch that received the material.
•
Shipping Branch. The branch that shipped the material.
Column Descriptions
Column Heading
Description
Product
The product number that shipped.
Description
Line 1: The product description.
Line 2: The second line of description for the product, if applicable.
Qty Shp
The quantity shipped according to the shipping warehouse.
Date Shp
The date the transfer shipped.
Qty Rec
The quantity received according to the receiving warehouse.
Rec Date
The date the material was received at the to warehouse.
Qty Os
The quantity over-shipped according to the receiving warehouse.
Qty Us
The quantity under-shipped according to the receiving warehouse.
Picker User
The ID of the person that picked the order.
