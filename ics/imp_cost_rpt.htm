Import Cost Report
The Import Cost Report lists the expected and actual import costs associated with merchandise received from overseas vendors.
Menu Path: Inventory > Reports > Miscellaneous Reports > Import Cost Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Stock Receipt Date Range. The beginning and ending receipt dates to include in the report.
•
Vendor. The vendor numbers to include on the report.
•
(D)etail or (S)ummary. Determines if you want to display the detail individual import cost factors or just the summary of all import costs for each receiver.
Column Descriptions
Column Heading
Description
Ship Number
The ship number updated when the merchandise was received. Refer to Receiving Stock for OTW Shipping Bins.
Vendor#
The vendor that shipped the merchandise.
Vendor Description
The vendor description.
Merch Amount
The value of the received merchandise.
Receiver Number
The stock receiver number for the imported merchandise.
Stock Rec Date
The date the merchandise was received.
Import Code Number
The import cost factors entered for the import codes associated with the shipment. Import cost factors are created in Import Cost Factor Maintenance and assigned to vendors in Vendor Maintenance.
Description
The description of the import code.
Expected Amount
The estimated value of the import cost, calculated according to the factor set up in Import Cost Factor Maintenance, and the weight of the product.
Actual Amount
The actual amount of the import cost update when the vendor invoice was reconciled for the overseas vendor.
Variance
The difference between the expected and the actual import cost.
