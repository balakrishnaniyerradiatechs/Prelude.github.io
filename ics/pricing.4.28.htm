Establishing Cost Adjustment Reason Codes
Cost adjustment reason codes identify the reasons for adjusting inventory cost. Each code references a general ledger account to use as the offset account for the cost adjustment. Once these codes are set up, you can use them in Cost Adjustment Entry (refer to Adjusting Inventory Cost).
Cost Adjustment Reason Code Listing
To view a list of inventory adjustment reason codes select Inventory > Listings > Miscellaneous Listings > Cost Adjustment Reason Codes.
To create or maintain cost adjustment reason codes:
1
Select Inventory > Maintenance > Misc Code Files > Cost Adjustment Reason Codes to open the Cost Adjustment Reason Maintenance screen.
2
For Cost Adj Reason, enter a 1 or 2 character, alphanumeric code to identify the cost adjustment reason code.
3
Enter the Description to display when the cost adjustment code is entered.
4
Under Adjustment Account, enter the general ledger account number to use as the offset account for the cost adjustment general ledger posting.
•
Div - The division number for this account. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Reg - The region number for this account, if applicable. A region is a subset of division - if division is used. (e.g., Southeast Region, Northwest Region).
•
Loc - The location or warehouse for this account.
•
Dep - The department number for this account. A department is a subset of warehouse, typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Ccr - The cost center number for this account. A cost center is a subset of department - if department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Account# - The general ledger account number to use as the offset when this inventory adjustment code is used in Inventory Adjustment Entry. If left blank, then the Cost Adjustment account from General Ledger Table Maintenance is used.
5
Press F2-Save to save the cost adjustment reason code.
