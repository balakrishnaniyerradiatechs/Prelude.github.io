Daily Stock Transfer Receiving Report
The Daily Stock Transfer Receiving report displays all products received from transfers in a warehouse for a day.
Menu Path: Inventory > Reports > Stock Movements Reports > Daily Stock Transfer Reports > Daily Stock Transfer Receiving Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Receiving Warehouse. The warehouse that received the transfers.
•
Receipt Date. The date the transfers were received.
Column Descriptions
Column Heading
Description
Transfer#
The received transfer release number.
Rec Date
The date the transfer was received.
Product Number
Line 1: The product number that was received.
Line 2: The serial or lot number of the product, if applicable.
Description
Line 1: The product description.
Line 2: The second line of description for the product, if applicable.
FWhse
Line 1: The shipping warehouse number.
Line 2: The shipping warehouse description.
Qty-Rec
The quantity received.
U/M
The unit of measure as entered on the transfer.
Cost
The cost of the product.
Ext-Cost
Line 1: The extended cost of the total quantity received (Qty Rec x Cost).
Line 2: The linked sales order or work order number.
Freight
The freight incurred on the transfer.
