Vendor Claim History Inquiry
The Vendor Claim History Inquiry displays updated claims for a vendor (refer to Updating Vendor Claims). When the claim is updated (shipped out of the warehouse), the quantities are removed from inventory. You can view the detail information for each claim.
Viewing Historical Vendor Claims
To view updated vendor claims:
1
Select Inventory > Inquiries > Vendor Claim Inquiries > Vendor Claim History Inquiry to open the Vendor Claim History Inquiry screen.
2
Enter the Vendor.
Updated claims for the vendor display.
Field & Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor number for the updated claims.
V/C#
The vendor claim number.
N
An asterisk (*) indicates that notes exist in the vendor claim
Type
The vendor claim type:
•
Regular. A regular vendor claim for on-hand or warranty inventory.
•
Vendor Show Claim. Refer to Entering a Vendor Show Claim.
•
Vendor Consigned Inventory. Consigned inventory refers to products in stock that you do not pay for until sold.
•
Non-Inventory Expense.
Ent Date
The date the vendor claim was entered.
Vendor Order#
The vendor’s order number for the claim, if applicable.
Units
The number of units on the vendor claim.
V/C Dollars
The value of the vendor claim.
F2-Finish
Closes the Vendor Claim Inquiry screen.
F5-Review
Opens the Vendor Claim History Detail screen to allow you to view detail information for each vendor claim.
F6-Enter V/C
Allows you enter a vendor claim number for direct access.
F8-Log
Opens the Vendor Claim Log Inquiry to display all transactions for a vendor claim.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Previous Group
Displays the previous page of claims if more than one page of claims display.
sF6-Next Group
Displays the next page of claims if more than one page of claims display.
Action Bar
Output
Allows you to view, print, email, or fax the usage information.
