Cycle Count Entry 2nd Level screen
The Count Entry 2nd Level screen displays the bin locations and quantities for items in the 2nd level count queue. When you press ENTER on each bin location, you can enter the 2nd level count.
Refer to Entering Second Level Cycle Counts for more information.
Menu Path: Inventory > Physical Inventory > Cycle Counting Procedure > Paper Site Cycle Count Entry (2nd Level)
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number.
Counter Id
The user ID of the person who counted the products. This person must be set up as a cycle counter in User Identification Maintenance.
Product
The product number from the count sheet.
Bin
The bin containing the product.
Line
The line number from the count sheet.
Problem
An asterisk (*) displays if the discrepancy exceeded the tolerance limits.
Counted
The second level count.
Qty
The quantity entered in the first level count.
F2-Update
Displays the adjusted quantity and value and gives you the option to change the quantities or to update the system with the new count.
F6-Exit
Exits the screen after all the entries are complete.
