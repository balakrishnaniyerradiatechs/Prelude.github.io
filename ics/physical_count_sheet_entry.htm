Physical Count Sheet Entry screen
The Physical Inventory Entry screen provides a space to enter the counts from the inventory count sheets.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Physical Count Sheet Entry
Field and Function Key Descriptions
Field/Function Key
Description
Whse
The warehouse where the material was counted.
List Name
The list name created for the products on the count sheets.
Product
The product counted.
Description
The description of the product.
Last Upd
The date the last time the on-hand balance for the product was updated.
U/M
The unit of measure for the product from Product Warehouse Maintenance.
Book-Oh
The saved on-hand quantity.
Count
The count from the count sheet. This value becomes the new on-hand quantity when you update inventory counts.
If a part number was skipped on the count sheet, leave the field blank so that the part number is not updated. If you enter a zero, the new on-hand will be zero. If a count already exists in the field, possibly entered from a manual list, you must determine whether the count should replace the previous quantity.
F2-Finish
Closes the entry screen and saves the count information.
F5-Switch Entry Mode
Allows you to change the method of entering counts: either straight down the Count column or back and forth between the Count and the Product Number.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
