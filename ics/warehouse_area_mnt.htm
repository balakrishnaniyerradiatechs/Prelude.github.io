Warehouse Area Maintenance screen
The Warehouse Area Maintenance screen allows you to define areas in your warehouse for cycle counting or picking purposes. Each area contains a group of zones and each zone contains a set of similar bins.
Menu Path: Inventory > Maintenance > Bin Files > Warehouse Areas
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The number for the warehouse that is being divided into areas.
Warehouse Area
The code to identify the warehouse area (up to 4 alphanumeric characters).
Description
The description of the warehouse area.
Group Code
An optional code you can use to group areas. For information only.
F2-Save
Saves the warehouse area information.
F4-Del
Deletes the warehouse area.
