Lot Pack Confirmation screen
The Lot Pack Confirmation screen appears when you receive a lot product with the Lot Pack Flag set to Y. The lot pack quantity, number of lot packs, and description default from the Lot Additional Info screen in Product Maintenance > F5-Add Info > Lot Number = Y.
Menu Path: Inventory > Stock Receipts > Stock Receipts Entry > Detail Information screen
Field and Function Key Descriptions
Field/Function Key
Description
Lot #
The lot number from the package. Press N to generate a system assigned lot number.
Bin Number
The number for the bin where the product is located.
LP Qty
The lot package quantity:
•
Defaults from the purchase order if you are in Stock Receipts Entry.
•
Defaults from Product Maintenance if you are in Inventory Adjustment Entry.
Desc
The type of package (e.g., pallet, carton, case). You can accept or change the default.
CO
The default country of origin.
#LP
The number lot packs per pallet.
Price
The price of the lot product.
Comm Cost
The commission cost for the product.
Quantity
The quantity to receive or adjust for this lot number.
F2-Save
Saves the lot pack information.
