Bin Location Inquiry
The Bin Location Inquiry allows you to view the bin information entered in Bin Location Maintenance. Refer to Defining Bins for more information on setting up bins.
Viewing Bins
To view information for a bin:
1
Select Inventory > Inquiries > Miscellaneous Inquiries > Bin Location Inquiry to open the Bin Location Inquiry screen.
2
Enter the Warehouse.
3
Enter the Bin for which you want to view information.
Detailed information on the bin location and specifications display.
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The number for the warehouse.
Bin
The bin number for which you want to display information.
Description
The description of the bin.
Zone
The zone where the bin is located.
Cycle Count Zone
The cycle count zone for this bin. If this field is blank, then this bin is not cycle counted. Only Pick, Bulk, and Retail Bin Types are used for cycle counting.
Permanent Code
Indicates if this is a permanent bin:
•
P - Permanent bin (default).
•
T - Temporary Bin. Deletes a product from the list of products in this bin when the quantity reaches zero. You can determine whether or not to delete the bin when the last product is removed.
•
R - Retained Temporary Bin. Automatically deletes the products from the list of products in this bin when the quantity reaches zero (unless the bin is the primary bin for the product). Does not delete the bin when the last product is removed. You must delete the product record manually in Bin Location Maintenance.
Bin Type
The type of bin:
•
B - Bulk.
•
C - Crib.
•
D - Receiving dock.
•
I - Inspection.
•
K - Bulk un-packaged.
•
L - Work in progress line stock.
•
O - On-the-water (refer to Receiving Stock for OTW Shipping Bins).
•
P - Pick.
•
R - Replenishment repackaging.
•
S - Staging.
•
T - Retail.
•
U - Unallocated.
•
W - Work in process.
Pick Reserve
The pick status of this bin (used with the Warehousing module):
•
N - Not available for picking.
•
R - Pickable.
•
R - Reserved for a customer.
Environment Code
The type of controlled environment for this bin (used for the Warehousing module):
•
E - EPA controlled.
•
F - Fire rated.
•
N - Normal.
•
R - Refrigerated.
•
S - Secured.
Storage Type
Indicates how the products in this bin are stored:
•
P - Pallet racks
•
R - Flow racks.
•
C - Carousel.
•
F - Floor stock.
•
S - Shelf.
•
M - Mobile.
•
W - Work center.
•
T - Truck.
•
O - Outplant.
•
G - Pre-stage.
Vendor Number
The vendor number (if you are using the outplant processing bin method).
Sort Sequence
Indicates how the bins are sorted for picking. Refer to Sorting Bins by Warehouse Location for information on the Bin Location Sort Sequence Procedure (for the Warehousing module).
Putaway Sort Seq
The sort sequence code for sorting putaway bins. If this field is blank, the Sort Sequence Code applies.
Nettable Flag
Indicates if this bin is considered nettable:
•
Y - The bin is nettable (the material is available to sell).
•
N - The bin is non-nettable (the material is not available to sell).
Auto Replenish Flag
Indicates if this bin in included in the auto replenish procedure (for the Warehousing module).
•
Y - The bin is included in the auto replenish procedure.
•
N - The bin is not included in the auto replenish procedure.
Putaway Status
Indicates if this bin is active:
•
A - The bin is active:
•
I - The bin is inactive.
Date Established
The date the bin was created.
Aisle
The aisle where the bin is located.
Bin Width
The width of the bin.
Bin Height
The height of the bin.
Bin Depth
The depth of the bin.
Bin Volume
The cubic volume of the bin.
Bin Weight
The total weight of the products that you can store in this bin.
OK
Closes the inquiry screen.
F2-Del
Deletes the bin information.
F5-Cycle Count History
Opens the Cycle Count History screen to display cycle count date a product information for the bin.
