Transfer Duplication screen
The Transfer Duplication screen allows you to duplicate an open transfer.
Menu Path: Inventory > Stock Movements > Transfer Entry > Action Bar > Dup Transfer
Field and Function Key Descriptions
Field/Function Key
Description
Duplicate Transfer#
The transfer to duplicate:
•
The transfer number if the transfer has not printed.
•
Transfer release number (-1) if the transfer already printed.
From Warehouse
The warehouse shipping the material. The default is the From warehouse from the original transfer but you can change it as necessary.
To Whse
The receiving warehouse. The default is the To warehouse from the original transfer but you can change it as necessary.
F2-Save
Copies the original transfer information to create a new transfer.
