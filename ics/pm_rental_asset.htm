Rental Asset Information Screen
The Rental Asset Information screen allows you to enter default asset and depreciation information to assign to a new asset record for an item converted to a rental product (if you use the Fixed Assets module). Asset records are only created for rental products that have the Equipment Flag set to Y in the Additional Information Screen.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Rental Information > F7-Asset Info
Field & Function Key Descriptions
Field/Function Key
Description
Asset Number Assignment
Indicates how to assign asset numbers:
•
R - Use the rental unit number.
•
E - Use the equipment number.
Asset Class
The default asset class to assign to new asset records. Asset classes group assets and separate rental assets from other fixed assets.
Depr On Rent Revenue?
Determines if the Depr Percent calculates the depreciation on the rental product. Set the depreciation method for the fixed asset to M (manual) so that the system calculates depreciation when the rental order is invoiced. After invoicing, run the Manual Depreciation Update to post to the G/L and to update the fixed asset record YTD and LTD values.
Depr Percent
The depreciation percentage if Depr On Rent Revenue is set to Y.
Book#
The default book number code to create for the general ledger.
Life
The default lifespan (years) of the asset.
Salv%
The percent of the installation cost that determines the salvage value of the asset. For example, if the installation cost is $1000 and the Salv% is 10, then the salvage value is $100.
Method
The default depreciation method.
F2-Save
Saves the rental asset information.
