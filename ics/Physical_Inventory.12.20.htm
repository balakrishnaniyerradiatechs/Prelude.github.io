Printing Physical Inventory Post-Entry Count Sheets
The Physical Inventory Post-Entry Count Sheets allow you to compare the counts entered in the system against the original count sheets to identify entry errors and to audit the physical counts. You are not required to print post-entry count sheets; but if you do, you must print them before you update inventory.
To print post-entry inventory count sheets:
1
Select Inventory > Physical Inventory > Inventory Sheet Procedure > Physical Post-Entry Count Sheet Printing to open the Physical Post-Entry Count Sheet Printing screen.
2
Enter the Warehouse for the count sheets.
3
Enter the Physical ListName for the count sheets. Leave blank to print count sheets for all lists created for the warehouse.
4
For #/Spaces Between Lines, enter the number of blank lines to print between each product. Enter zero for single spacing.
5
For Show Onhand Quantity?, indicate if you want the saved on-hand quantity to display on the count sheets. Choose from the following:
•
Y - Displays the saved on-hand quantities of each product.
•
N - Suppresses the saved on-hand quantities of each product.
6
For Count Sheet Size, indicate the paper size for the count sheet. Choose from the following:
•
1 - Prints the count sheets on 8 ½” X 11” paper.
•
2 - Prints the count sheets on 11” x 14” paper.
7
Press ENTER to print the post-entry count sheets.
Count Sheet Column Descriptions
Column Heading
Description
Product Number
The product numbers included in the physical list name.
Description
The product description.
Bin
The bin number for the product, if applicable.
On-Hand
The saved on-hand quantity. Refer to Saving Current On-Hand Quantities.
Count
The count entered in the Physical Inventory Entry screen. Refer to Entering Physical Inventory Counts.
