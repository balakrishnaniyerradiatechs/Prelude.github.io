Core Exchange Type Maintenance screen
The Core Exchange Type Maintenance Screen allows you to change the exchange type for a core bank record or a pending core record.
Refer to Maintaining Core Exchange Types for more information.
Menu Path: Inventory > Maintenance > Misc Code Files > Core Exchange Type Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer associated with the core.
Core Product
The core product number.
Select By
Determines which cores to display:
•
P - Only display pending cores.
•
C - Only display core bank records.
•
A - Display all pending core and bank cores.
Document
The document containing the core.
Doc Date
The document date.
Product
The core product number.
Description
The core product description.
Source
Indicates where the received core was applied:
•
Core Bank - The customer returned the core and it was applied to the customer’s core bank.
•
Pending - The customer returned the core and it was applied to a pending core invoice for the customer.
Current
The current core exchange type assigned to the product:
•
E - Exchange. The customer will exchange the core item for another product.
•
R - Repair and return. The customer wants you to repair and return the core product.
•
U - Unused. The core product was not used.
•
W - Warranty. The core exchange product is under warranty.
•
O - Other.
New Type
The new core exchange type to assign to the product:
•
E - Exchange. The customer will exchange the core item for another product.
•
R - Repair and return. The customer wants you to repair and return the core product.
•
U - Unused. The core product was not used.
•
W - Warranty. The core exchange product is under warranty.
•
O - Other.
F2-Save
Saves the core exchange type information.
