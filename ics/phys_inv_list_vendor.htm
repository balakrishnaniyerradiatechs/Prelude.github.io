Physical Inventory List By Vendor screen
The Physical Inventory List By Vendor screen allows you to create an inventory count list by vendor.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Build Physical Inventory Lists > By Vendor(s)
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for the inventory list.
Physical ListName
The name that identifies the count sheet. You will use this name to print count sheets, enter counts, update inventory, and run inventory reports.
Vendor
The vendor(s) whose products you want to include on the count sheet. Only products with this primary vendor will be included on the count sheet.
Description
The vendor description.
F2-Create List
Creates the inventory list.
