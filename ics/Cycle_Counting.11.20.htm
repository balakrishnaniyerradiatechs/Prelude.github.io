Cycle Counting using Radio Frequency Devices
If you use RF guns to cycle count, the procedures are similar to the manual count sheet procedures, except that you use a different method to enter counts.
Use the following steps to cycle count using RF devices:
1
Print the Cycle Count First Level Report (refer to Cycle Count First Level Report).
2
Enter counts (refer to Entering 1st Level RF Cycle Counts).
3
Add products to the second level count (refer to Requesting Additional Cycle Counts).
4
Print the Cycle Count Second Level Report (refer to Cycle Count Second Level Report).
5
Enter recounts (refer to Entering 2nd Level RF Cycle Counts).
