Physical Inventory List By Range screen
The Physical Inventory List By Range screen allows you to build a physical inventory list using a combination of bin location range, product line range, product number range, and/or vendor number range.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Build Physical Inventory Lists > By Range
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number for the inventory list.
Physical ListName
The name that identifies the count sheet. You will use this name to print count sheets, enter counts, update inventory, and run inventory reports.
Beginning Bin
The beginning bin number if you want to create a list for a bin number range. Leave blank to include all bins.
Ending Bin
The ending bin number if you want to create a list for a bin number range. Leave blank to include all bins.
Beginning Pline
The beginning product line number if you want to create a list for a range of product lines. Leave blank to include all product lines.
Ending Pline
The ending product line number if you want to create a list for a range of product lines. Leave blank to include all product lines.
Beginning Vendor
The beginning vendor number if you want to create a list for a range of vendors. Leave blank to include all vendors.
Ending Vendor
The ending vendor number if you want to create a list for a range of vendors. Leave blank to include all vendors.
Beginning Product
The beginning product number if you want to create a list for a range of products. Leave blank to include all products.
Ending Product
The beginning product number if you want to create a list for a range of products. Leave blank to include all products.
F2-Create List
Create the physical inventory list based on the selection criteria. To view the list, refer to Displaying Physical Inventory Lists.
