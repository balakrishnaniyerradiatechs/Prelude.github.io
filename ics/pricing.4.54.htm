Setting Up Competitor Codes
Before you can enter competitor prices and part numbers, you must first set up a code for each competitor. You can then use these codes in Competitor Price Tracking Maintenance (refer to Maintaining Competitor Prices & Part Numbers) to track a competitor’s price for a part number.
To set up competitor codes:
1
Select Inventory > Maintenance > Misc Code Files > Competitor Code Maintenance to open the Competitor Code Maintenance screen.
2
For Competitor Code, enter a code (up to 4 alphanumeric characters) to represent the competitor.
3
For Description, enter a description for the competitor code (e.g., the competitor’s name).
4
Press F2-Save to save the competitor code information.
Refer to Maintaining Competitor Prices & Part Numbers for information on using the competitor code to track prices.
