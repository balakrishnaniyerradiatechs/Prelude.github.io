Kit Component Availability Inquiry screen
The Kit Component Availability screen in the Inventory Inquiry displays the on-order, backorder, availability, and delivery dates for the components on a kit product.
Menu Path: Inventory > Inventory Inquiries > Inventory Inquiry > Action Bar > Inquiries > Kit Component Availability
Field and Function Key Descriptions
Field/Function Key
Description
Product
The kit product.
Warehouse
The warehouse for which you want to view availably and delivery information.
Available
The quantity of the kit available in the selected warehouse.
Component
The components that make up the kit.
Description
The description of each component.
Onorder
The total quantity of the component that is on order (e.g., on transfers, purchase orders).
Backorder
The total quantity of the product that is on backorder.
Available
The total quantity of the product that is available to sell in this warehouse.
Delivery
The closest expected delivery date of the product if it is Onorder.
F2-Save
Closes the inquiry screen.
F5-Inv Inq
Opens the Inventory Inquiry screen for the highlighted component.
