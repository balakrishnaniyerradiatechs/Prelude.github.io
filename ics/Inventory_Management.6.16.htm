Inventory Option Parameters
Multiple Bin Locations must be set to Y for an item to exist in multiple bins. The system then tracks the quantity on-hand in each bin. Validate Bin Locations is automatically set to Y to require that a bin be created before it can be used.
Related Reports
Description
Bin Products Onhand Report
The products and on-hand quantities included in each bin.
Negative Bin Onhand Report
All bins containing products with a negative on-hand balance.
Bin Quantity Discrepancy Report
All products in which the total on-hand does not equal the total of on-hand quantities of each of the product’s bins.
Primary Bin Stockout Report
Products whose primary bin contains zero quantity.
Bin Product Onhand On The Water Report
Products moved to on-the-water bins when you use the shipping bin method to receive on-the-water merchandise.
Bin Product Valuation Report
The extended average cost of each product in a bin, and the total cost for the combined quantities in a bin.
Product Bin Onhand Report
Bin and on-hand quantity information for a product or list of products, to see all the bins where a product exists.
Bin Replenishment Report
Bins that require replenishment from other bins.
List Products With No Primary Bins
Products that have no primary bin number assigned in Product Warehouse Maintenance.
