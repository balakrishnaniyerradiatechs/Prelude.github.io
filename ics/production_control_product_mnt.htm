Production Control Product Maintenance screen
The Production Control Product Maintenance screen allows you to create a code to represent standard manufacturing instructions for work order production groups. Once the codes are established, you can assign them to the bill of material products in the Production Control Parameters screen (Inventory > Maintenance > Product Maintenance > Product Purchasing Parameters Maintenance > Replenishment Mtd = M > Production Group). Refer to Production Control Groups for more information.
Menu Path: Inventory > Maintenance > Product Control Files > Production Control Product Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Product Group
The code to represent a group of products.
Description
The description of the product group.
Instruction
The instructions for the production group. These instructions appear on the work order pick tickets and the Open Work Order report.
F2-Save
Saves the production group code.
F4-Del
Deletes the production group code.
