Current Inventory Value Report - Replacement
This report displays the value of on-hand inventory using replacement cost. You can select to print the extended value for each product for each vendor or only the total value of inventory for each vendor.
Menu Path: Inventory > Reports > Inventory Valuation Reports > Current Inventory Value Report - Replacement
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse(s) to include on the report.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Product. The product(s) to include on the report. Leave blank to include all products.
•
Product Type. The products type(s) to include on the report. Leave blank to include all product types or choose from the following:
-
N - Rental item.
-
F - Finished good.
-
R - Raw material.
-
B - Both finished and raw material.
-
E - Expense item.
-
U - Used item.
•
Stocking Status: The stock status codes to include on the report. Leave blank to include all status codes or choose from the following:
-
S - Stock.
-
N - Nonstock.
-
C - Consigned.
•
Only Onhand not equal to 0. Determines if you want to see only products with an on-hand quantity. Choose from the following:
-
Y - Display only products with an on-hand quantity or negative on-hand quantity.
-
N - Display all products, including products with an on-hand balance of zero.
•
Sort by (P)rodLine (V)endor. Determines how to sort the report. Choose from the following:
-
P - Sort by product line.
-
V - Sort by vendor.
•
Primary Vendor. The primary vendor(s) to include on the report. Leave blank to include all vendors.
•
Product Line. The product line(s) to include on the report. Leave blank to include all product lines.
•
Period. The fiscal period to include on the report (e.g., 05-01 to see the value of the inventory for January 2005).
Column Descriptions
Column Heading
Description
Product
The product number.
Description
The product description.
Onhand
The quantity of the product in stock.
U/M
The product’s unit of measure.
Rep Cost
The replacement cost of the product.
Rep Value
The extended replacement value of the product in on-hand inventory
(Onhand x Rep Cost).
