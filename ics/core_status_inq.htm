Core Status Inquiry
The Core Status Inquiry displays the number of pending cores sold, returned, and remaining for a customer.
Viewing Core Status
To view the cores status for a customer:
1
Select Inventory > Inquiries > Core Inquiries > Core Status Inquiry to open the Core Status Inquiry screen.
2
Enter the Customer number.
3
For Core Product, enter the core product number for the customer or leave blank to view all core products for a customer.
4
For Ret. Type, enter the return type to limit the report or leave blank to list all return types. Choose from the following:
-
A - All core bank records for a customer.
-
E - Only core bank records with a remaining quantity eligible to use.
The core status information for the customer appears.
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer number.
Core Product
The core product number. You can leave this field blank to view all pending cores for the customer.
Ret.Type
Determines the return record types to display:
•
A - All core bank records for a customer.
•
E - Only core bank records with a quantity eligible to use.
Order#
The original order number that the pending core created.
Inv Date
The date the pending core was created.
Due Date
The date from Sales Order Entry that the pending core should be returned.
Product
The core product number for the pending core.
Shipped
The number of cores shipped on this order.
Applied
The number of cores applied against the pending core.
Remain
The number of cores to return or bill on the pending core.
Totals
The totals of the cores in the Shipped, Applied and Remain columns.
F2-Save
Clears the inquiry screen.
F5-Review Shipped
Opens the Order History Inquiry Display for the sales order that the core was shipped on. Use the Examine option on the Action Bar to view the credit memo if one was applied against the original invoice.
F6-Review Applied
Opens the Receipt Info screen to allow you to view the core bank items that were applied against the current pending core. Write-off displays if a quantity of the core was written off. Credit Memo displays when a credit memo is applied against the original invoice number.
F9-Goto Line
Allows you to enter a line number, reference number, or page number to go directly to.
