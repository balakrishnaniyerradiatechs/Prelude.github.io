Primary Bin Stockout Report
The Primary Bin Stockout Report displays products with zero quantity in their primary bin. You can use this report to monitor bin stockouts.
Menu Path: Inventory > Reports > Stock Status Reports > Primary Bin Stockout
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse(s) containing the primary bins with stock-outs.
Column Descriptions
Column Heading
Description
Area
The warehouse area assigned to the product. Refer to Defining Warehouse Areas.
Description
The description of the warehouse area.
Primary Bin
The primary bin assigned to the product.
Product
The product number.
Description
The product description.
Case Qty
The case quantity of the product.
