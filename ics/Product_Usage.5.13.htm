Evaluating and Adjusting Unusual Usage
Unusual usage is defined as:
•
A product with a month’s usage that is greater than the previous five months combined.
•
A product that is out of stock for more than 2 weeks.
•
Products with a spike in usage above or below the minimum or maximum levels set for the product by the percentage defined in Spike Exception - Min/Max set in Purchasing Parameters Maintenance (only applicable if you have the Forecasting Module).
The End of Month Calendar Close determines which products have unusual usage. The Usage Exception menu provides a means to detect unusual monthly usage for each product in each warehouse. Refer to the following:
•
Evaluating Unusual Usage.
•
Usage Exception Report.
•
Product Parameters Calc Skipped Audit Report.
•
Adjusting Usage for Skipped Product Parameters.
•
Spike Exceptions Inquiry.
