Cycle Counting Using Count Sheets
Once you set up the cycle count requirements (refer to Setting up Your System for Cycle Counting), you can begin counting. If any product has a new count that differs from the on-hand balance by more than the defined tolerance limit, the item goes into a second level count queue for a recount. You can also manually add products to the second level count queue.
Use the following steps to cycle count using count sheets:
1
Print count sheets (refer to Printing Count Sheets ).
2
Enter counts (refer to Entering First Level Cycle Counts).
3
Add products to the second level count (refer to Requesting Additional Cycle Counts).
4
Print a report for recount items (refer to Cycle Count Second Level Report).
5
Enter recounts (refer to Entering Second Level Cycle Counts).
