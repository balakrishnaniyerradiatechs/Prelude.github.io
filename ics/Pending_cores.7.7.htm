Returning a Core Item to a Core Bank
To apply a returned core to the core bank:
1
Select Inventory > Inventory Adjustments > Inventory Adjustment Entry. Refer to Adjusting Inventory Quantities for detail instructions on adjusting inventory.
2
For Adj Type, enter INV.
3
Enter the Product Number, which must be set up as a core product for the customer. The Inventory Adjustment Line Detail screen appears.
4
After you enter the Quantity, the Return Type Info screen appears.
5
For Core Return Type, enter CB to return the core to the core bank. The Exchange Type Info screen appears.
6
For Exchange Type, enter the exchange type for this return. Choose from the following:
•
E - Exchange. The customer will exchange the core item for another product.
•
R - Repair and return. The customer wants you to repair and return the core product.
•
U - Unused. The core product is not used.
•
W - Warranty. The core exchange product is under warranty.
•
O - Other.
7
Enter the Customer Number to use for banking the core.
8
Press F2-Save to save the exchange type info.
9
Press F2-Save again to create a core bank record for the items returned to the bank. The cost defaults to the core product’s replacement cost. The core bank cost is used in the adjustment.
If the Print Core Receiving Tkt in Inventory Parameters is set to Y, you can print a core receiving ticket for the customer. (refer to Printing Core Receipt Tickets).
