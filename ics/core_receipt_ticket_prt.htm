Core Receipt Ticket Printing screen
The Core Receipt Ticket Printing screen allows you to print a core receiving ticket when you receive a core item by adjusting inventory. You can give the receipt ticket to the customer when they return the item.
Refer to Pending Cores for more information on core products.
Menu Path: Inventory > Processes > Print Core Receipt Ticket
Field and Function Key Descriptions
Field/Function Key
Description
Adjustment Number
The adjustment number from Inventory Adjustment Entry (refer to Receiving Cores in Inventory Adjustment Entry) for which you want to print a receiving ticket.
Warehouse
The number for the warehouse that contains the core item.
Printer Form Queue
The printer location where you want to print the receiving ticket.
