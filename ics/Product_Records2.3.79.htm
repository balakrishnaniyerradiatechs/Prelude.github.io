Manufacturer Abbreviations
Manufacturer abbreviations identify the manufacturer for lot-controlled items. Once you create the abbreviation codes in Manufacturer Maintenance, you can assign the codes to:
•
A lot product in Product Warehouse Maintenance > F9-Lot > Manufacturer Code.
•
A vendor in Vendor Maintenance > F5-Add Info > Manufacturer Code.
•
A product when it is received in Stock Receipt Entry > Detail Information screen > Screen > Mfg#.
Manufacturer Abbreviations Listing
To view a list of manufacturer abbreviations set up in Manufacturer Maintenance, select Inventory > Listings > Miscellaneous Listings >Manufacturer Abbreviations.
To create manufacturer abbreviations:
1
Select Inventory > Maintenance > Misc Code Files > Manufacturer Abbreviations to open the Manufacturer Maintenance screen.
2
For Manufacturer Number, enter a code to represent a manufacturer of lot-controlled products.
3
Enter a Description of the manufacturer.
4
Press F2-Save to save the manufacturer abbreviation code.
