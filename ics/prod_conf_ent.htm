Product Configuration Lookup screen
The Product Configuration Lookup screen appears when you enter a base product on a sales order or when you enter a count for a configured product on a count sheet during physical inventory. A base product is a product that can be configured into child products based on the options available. For example, if the base product is a shirt, a child product could be a Red XL Menâ€™s shirt.
Refer to Configuring Base and Child Products for more information.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Physical Count Sheet Entry
Field and Function Key Descriptions
Field/Function Key
Description
Product
The base product entered on the sales order or count sheet.
Questions
The types of options (configuration codes) set up for the product. You must enter a Response for each Question to select a child product.
Responses
The options available for each Question.
F2-Update
Selects the correct child product based on the Responses entered. If the selected product is correct, press ENTER.
