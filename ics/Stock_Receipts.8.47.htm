Requirements to Receive into Ship Bins
To use shipping bins for on-the-water receiving, you must be using bins and/or warehousing and set the following maintenance flags:
Additional Option Parameters
Set On The Water Conversion to B to activate shipping bins. When you receive from an overseas vendor, you are required to enter a Ship Bin.
Chart of Accounts Maintenance
To properly post the value of ship bin merchandise, expense accounts must be set up in Chart of Accounts Maintenance for each import cost factor that applies to on-the-water receiving.
Set Overseas Information Required to Y (Module Interface option on the Action Bar) for each account to require a ship number and cost code when Accounts Payable enters a manual invoice for overseas material.
Import Cost Factor Maintenance
To estimate import costs on shipments from overseas vendors (e.g., duty, brokerage fees, freight), you can set up import cost factors to apply to each vendor. Import Cost Factor Maintenance allows you to maintain your own cost factors to satisfy unique shipping costs incurred when you import merchandise. You can calculate cost factors based on dollars, pounds, or units. Refer to “Maintaining Import Cost Factors” in the Purchasing documentation for information on creating import cost factors.
You can assign these import factors to each overseas vendor in Vendor Maintenance. If necessary, the buyer can override the import factors for a vendor in Purchase Order Entry > F5-Hdr Scr > F5-Add Info > F5-Import Factors.
Vendor Maintenance
To set up a vendor with default import codes to apply when you enter a purchase order:
1
Select Purchasing > Maintenance > Vendor Maintenance.
2
Enter the Vendor.
3
Press F5-Add Info to open the Additional Information screen.
4
Press F7-Importing to open the Import Information screen.
5
Enter Y in Overseas Vendor to prompt for a ship number when products from this vendor are received.
6
For Country, enter the country code that represents where the product is manufactured.
7
For Import Code, enter the code representing the charges incurred when purchasing from this vendor. Refer to the Purchasing documentation for information on maintaining import cost factors. You can override this code in the Purchase Order Header screen. The following are examples of commonly used codes:
•
BRK - Overseas brokerage (dollars).
•
CONS - Overseas consolidation (units).
•
DUTY - Overseas duty (dollars).
•
IF - Inland freight (pounds).
•
LF - Overseas loading fee (pounds).
8
Enter the Currency code to indicate the type of currency this vendor uses.
9
If applicable, you can enter a Factor to override the factor for the import code established in Import Cost Factor Maintenance. The factor is multiplied by the cost of the product to calculate the corresponding charge.
10
Press F2-Save 3 times to save the import information.
Bin Location Maintenance
You must create a ship bin in Bin Location Maintenance to receive the on-the-water merchandise for each on-the-water shipment. Refer to Defining Bins for complete information on creating a bin.
To set up a ship bin:
1
Select Inventory > Maintenance > Bin Files > Bin Locations to open the Bin Location Maintenance screen.
2
Enter the Warehouse to receive the on-the-water shipments.
3
For Bin, enter a reference number for the ship.
4
For Bin Type, enter O for to establish the ship as an on-the-water bin. The Ship Information screen appears.
5
For Description, enter the name of the ship containing the merchandise.
6
For Ship Eta, enter the date the ship is expected to arrive.
7
Press ENTER to return to the Bin Location Maintenance screen.
8
For Nettable, enter N so the on-the-water inventory is on-hand but not available for sale. Always select this option for on-the-water bins.
9
Press F2-Save to save the bin information.
Ship Maintenance
You can change the description of a ship or view the status of an on-the-water shipment in Ship Maintenance.
To view the status of a ship bin:
1
Select Inventory > Maintenance > Bin Files > Ship Maintenance to open the Ship Maintenance screen.
2
For Ship/Ref#, enter the same name or number as the ship bin in Bin Location Maintenance.
3
Press F5-Ship Inq to open the Ship/Boat Inquiry screen. You can view the estimated value and the actual value (after it arrives) of each import factor for the ship.
