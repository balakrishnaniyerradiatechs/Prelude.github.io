Setting Up Parker Divisions
You can set up the Parker divisions, defined by Parker Hannifin, in Parker Division Maintenance.
Parker Product Code Listing
To view a list of Parker Product codes, select Inventory > Listings > Miscellaneous Listings > Parker Product Code Listing.
To set up Parker division codes:
1
Select Inventory > Maintenance > Product Control Files > Parker Reporting Codes > Parker Division Maintenance to open the Parker Division Maintenance screen.
2
Enter the Parker Division Code to define a division or location.
3
Enter a Description of the Parker division (e.g., Southwest, West, Northeast).
4
Press F2-Save to save the Parker division information.
Refer to Setting Up Parker Marketing Codes for information on assigning Parker product codes to a division code.
