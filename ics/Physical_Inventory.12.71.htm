Physical Linked Asterisk Report
The Physical Linked Asterisk report displays the customer and order information for linked (stock and nonstock) items counted during physical inventory.
Menu Path: Inventory > Physical Inventory > Inventory Linked Procedure > Physical Linked Asterisk Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where the inventory was counted.
Column Descriptions
Column Heading
Description
Product Number
The product number linked to a customer order.
Customer
The customer associated with the linked product.
Doc Id
The order number for the linked item. The prefix indicates the order type:
•
S - Sales order.
•
W - Work order.
•
T - Transfer.
Description
The description of the product number
Description Two
The second line of description for the product number.
Qty Opn
The quantity still open (not yet shipped) of the item for this order number.
Qty Pck
The quantity of the item picked for this order number.
Link Qty
The quantity designated for the customer order.
