Batch Transfer Receipts Report
The Batch Transfer Receipts report allows you to view and verify received stock information for a batch. If the report shows any discrepancies from what was actually received, you can adjust the quantities as necessary in Batch Transfer Receipts Entry. If you do change quantities, be sure to run the report again. Once everything is correct, run the Batch Transfer Receipts Update.
Menu Path: Inventory > Stock Movements > Batch Transfer Receipts > Batch Transfer Receipts Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Batch Number. The batch number(s) to review.
•
Receiving Warehouse. Warehouse(s) to include on the report.
•
Only Lines With Qty. Rec’d?. Indicate which line items to include on the report:
-
Y - Only include lines with a received quantity.
-
V - Only include line items with an over- or under-received quantity.
-
Leave blank to include all line items on the report.
Column Descriptions
Column Heading
Description
Transfer#
The transfer receiver numbers included in the batch.
FWhs
The shipping warehouse.
TWhs
The receiving warehouse.
Product/Description/Serial#
Line 1: The product number.
Line 2: The product description.
Line 3: The second line of product description, if applicable.
Line 4: The serial number, if applicable.
Ord Qty
The original quantity ordered on the transfer.
Opn Qty
The quantity shipped (in Transfer Shipments Entry) on this transfer.
Rec Qty
The quantity received in Batch Transfer Receipts Entry.
B/O Qty
The quantity backordered. The quantity still remaining open after this receiver.
Cost
The cost of the item.
U/M
The unit of measure of the item as entered on the transfer.
Extension
The total dollar amount of the quantity received of the line item.
