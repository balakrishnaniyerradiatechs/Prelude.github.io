Bin Product Onhand On The Water Report
The Bin Product Onhand On The Water report displays products moved to the on-the-water bin. This report is only applicable if you use the shipping bin method to receive on-the-water merchandise (refer to Receiving Stock for OTW Shipping Bins).
Menu Path: Inventory > Reports > Miscellaneous Reports > Bin Products OTW Onhand Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse containing the on-the-water bin products.
•
Bin Number Range. The beginning and ending bin number to view. Leave blank to view all bins for the warehouse.
•
Bin Type. The bin location type defaults to O for on-the-water bin locations.
•
Storage Type. Indicates how the products in this bin are stored. Leave blank to include all storage types or choose from the following:
-
P - Pallet racks.
-
R - Flow racks.
-
C - Carousel.
-
F - Floor stock.
-
S - Shelf.
-
M - Mobile.
-
W - Work center.
-
T - Truck.
-
O - Outplant.
-
G - Pre-stage.
Column Descriptions
Column Heading
Description
Bin Location
The bin number.
Product Number
The products in the corresponding bin location.
Description
The product description.
Bin OH
The quantity of the product in the bin.
Bin Cmt
The quantity of the product in the bin that is committed to sales orders, work orders, transfers, and vendor claims.
Bin Avl
The quantity of the product in the bin that is available.
U/M
The unit of measure for the product.
Lots
The lot number assigned to the product, if applicable.
Cls
The inventory class for the product. Refer to Classifying Products.
Cost
The replacement cost for the product.
Ext Cost
The cost for the total onhand quantity of the product (Bin OH x Cost).
