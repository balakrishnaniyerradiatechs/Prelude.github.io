Tariff Commodity Codes
The U.S. Customs Service publishes a Harmonized Tariff Rate Schedule to regulate import duties and taxes on imports and to track import and export statistics. Tariff commodity codes define favored nation rates, non-favored nation rates, and agreement code rates to work with harmonized tariff calculations. Rates can either be calculated as a percentage of the value of a product or an amount per unit or weight.
Prelude uses tariff codes to incorporate harmonized tariffs into duty calculations for material received from import vendors. When you purchase a product with a harmonizing code, a tariff charge is calculated for the item. The Harmonizing Unit in the Additional Product Maintenance Information Screen, checks the tariff commodity code to determine if the product is subject to a tariff charge.

Country Codes
Country Codes Maintenance (Order Processing Controls > Country Codes) allows you to maintain tariff rates, agreement codes, and incompatible ship-vias for a country.
Most Favored Nation determines if tariffs for a country are calculated at the most-favored- nation rate or at the non-favored-nation rate.
•
Y - Tariffs are calculated at the Favored Nation Rate set in Tariff Commodity Code Maintenance.
•
N - Tariffs are calculated at the Non-Favored Nation Rate set in Tariff Commodity Code Maintenance.
Agreement Codes are used to calculate tariff charges for this country as defined in the Harmonized Tariff Schedule of the United States.
To establish tariff commodity codes:
1
Select Inventory > Maintenance > Product Control Files > Tariff Commodity Codes to access the Tariff Commodity Code Maintenance screen.
2
Enter the Tariff Commodity Code from the U.S Harmonized Tariff rate schedule.
3
Enter a Description that identifies the products associated with the code.
4
For Fav Nation Perc Rate, enter the percentage rate to calculate the harmonized tariff for a country with the Most Favored Nation flag set to Y (Order Processing Controls > Country Codes).
5
For Fav Nation Per Unit Rate, indicate the rate per unit to calculate the harmonized tariff for a country with the Most Favored Nation flag set to Y.
6
For Nonfav Nation Perc Rate, indicate the percentage rate to calculate the harmonized tariff for a country with the Most Favored Nation flag set to N (Order Processing Controls > Country Codes). Agreement Codes do not apply.
7
For Nonfav Nation Per Unit Rate, indicate the rate per unit to calculate the harmonized tariff for a country with the Most Favored Nation flag set to N. Agreement Codes do not apply.
8
For Agreement Codes, indicate the programs under which special tariff treatments are provided as defined in the Harmonized Tariff Schedule of the United States. You can assign multiple agreement codes to a harmonized tariff code.
For example, if the country is a favored nation and contains an agreement code that matches one of these agreement codes, then the Agree% and the Agree Rate are used to calculate the tariff.
9
For Agree%, define the percentage rate for the agreement code. This percentage is used to calculate the tariff if the country is a most favored nation and has the same agreement code.
10
For Agree Rate, enter the unit rate for the agreement code. This rate is used to calculate the tariff if the country is a most favored nation and has the same agreement code.
11
Press F2-Save to save the tariff commodity code information.
