Unusual Activity Report
The Unusual Activity report displays all items with unusual usage. An item is considered to have unusual usage when:
•
A product with a month’s usage that is greater than the previous five months combined.
•
A product that is out of stock for more than 2 weeks.
•
Products with a spike in usage above or below the minimum or maximum levels set for the product by the percentage defined in Spike Exception - Min/Max set in Purchasing Parameters Maintenance (only applicable if you have the Forecasting Module).
Refer to Evaluating and Adjusting Unusual Usage for more information on unusual usage.
Menu Path: Inventory > Reports > Miscellaneous Reports > Unusual Activity
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse with unusual usage.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Month End Date. The last month-end date to include on the report.
•
Executed End of Year? (Y/N). Indicate if the End-Of-Year Procedure already ran to determine if current or previous months of the year are used to show unusual usage:
-
Y - The End of Year Procedure already ran. The system uses the months from the previous year to review usage.
-
N - The End of Year Procedure has not yet run. The system uses the months from the current year to check usage.
Column Descriptions
Column Heading
Description
Product#
The product number with unusual usage.
Description
The description for the product.
Pline#
The product line number.
Sea
Indicates if the product is seasonal:
•
Y - The product is seasonal (the Seasonal Flag is set to Y in the Product Purchasing Parameters screen).
•
N - The product is not seasonal.
Months: (Calc/Override
/Exceptions)
Line 1: The last 6 months of usage.
Line 2: The 6 months before the last 6 months of usage.
Line 3: The usage code:
•
O - Usage was overridden.
•
U - Unusual usage.
Stat
The stocking status of the product:
•
S - Stock item.
•
N - Nonstock item.
•
C - Consigned item.
Frz
Indicates if the product has frozen controls. Refer to Freezing Purchasing Controls.
