Hazardous Material Class Maintenance screen
If you sell products classified as Hazardous Materials, you can use the Hazardous Material Class Maintenance screen to enter codes to assign to products that require hazardous material information to print on a bill of lading. Once you set up hazardous material codes, you can assign them to products in Product Maintenance.
Refer to Hazardous Material Class for more information.
Menu Path: Inventory > Maintenance > Product Control Files > Freight > Hazardous Materials Class
Field and Function Key Descriptions
Field/Function Key
Description
Haz Material Class
The hazardous material class defined by the Office of Hazardous Materials Safety within the U.S. Department of Transportation. These classes identify products that require hazardous material information to print on a bill of lading when hazardous material is shipped.
Description
The description of the hazardous material class.
Hazmat Codes
The hazardous material codes to determine the packing requirements for the product.
Manifest Charge
Indicates if hazardous material charges should apply on the manifest system:
•
Y - Hazardous charges apply for the manifest.
•
N - Hazardous charges do not apply on the manifest.
This flag is only applicable if using the PSS interface.
F2-Save
Saves the hazardous material information.
F4-Del
Deletes the hazardous material class.
