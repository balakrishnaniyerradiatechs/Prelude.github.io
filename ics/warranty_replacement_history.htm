Warranty Replacement History screen
The Warranty Replacement History screen displays the information entered on a warranty replacement.
Menu Path: Inventory > Warranty Replacements > Warranty Replacement History Inquiry > F5-Review W/R
Field and Function Key Descriptions
Field/Function Key
Description
Entry Number
The warranty replacement number.
Vendor
The vendor the items were returned to on the vendor claim.
Vendor Invoice#
The vendorâ€™s invoice/credit memo number for the replaced items, if applicable.
Received By
The user ID of the person who entered the replacement.
Replacement Date
The date the warranty replacements were received.
Vendor Claim Number
The number from the vendor claim on which the products were returned to the vendor.
Value - Replaced
The value of the products sent to the vendor to be replaced.
Value - Replacing
The value of the products the vendor sent to replace the products.
Discrepancy
The difference in value between what was sent to the vendor and what the vendor replaced.
OK
Closes the inquiry screen.
F5-Replaced Products
Opens the Warranty Replaced History Review screen to display the products returned to the vendor on the vendor claim.
F6-Replacing Products
Opens the Warranty Replacing History Review screen to display the products replaced by the vendor.
