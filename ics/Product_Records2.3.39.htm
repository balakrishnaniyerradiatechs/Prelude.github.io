Configuring Base and Child Products
If you sell a product in multiple sizes, colors, lengths, or other options, you can set up one product as the base product and then create child products from all of the available configurations of the product. For example, if you sell a t-shirt that comes in many sizes and colors, the base product is the t-shirt and the child products are all of the size and color combinations available for sale.
You must first set up configuration codes to identify all options available for a product. Then when you enter a base product on a sales order or quote, you can choose the size and color options. The price for the base product is the same as for the child products. If you change a base product, the change affects all of the linked child products.
