Advanced Warehousing Product Information Screen
The Advanced Warehousing Product Information screen only applies if you use the Warehousing module. You can enter rules for segment picking and work order allocation.
Warehousing
Warehousing is an additional module that you must purchase from Activant.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Action Bar > Addtl Opts > Warehousing Information
Field & Function Key Descriptions
Field/Function Key
Description
Floored Item
Indicates if the item is floored, which means it is part of a logical order for segment picking:
•
Y - The item is a floored and goes directly to the staging area.
•
N - The item is a non-floored item.
Floored Quantity (unlabeled field)
The quantity of the item to use to build a segment.
Box Type
Defines the box type or classification for each product. You can use this type to classify products when you print a bill of lading.
W/O Over Allocation %
The percent to over-allocate to each component in Work Order Bin Allocation. This percent applies if Warehousing Allocation in Bill of Material Maintenance is set to O for the bill of material and the components of the BOM have a W/O Over Allocation %.
Work Order Bin Allocation increases the required quantity by the W/O Over Allocation % and rounds up to the nearest selling package quantity for each component. The overall percentage is not used if the bin available is less than the amount calculated.
F2-Save
Saves the advanced warehousing product information.
