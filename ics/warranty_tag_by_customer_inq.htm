Warranty Tag by Customer Inquiry
The Warranty Tag by Customer Inquiry displays detail tag, vendor, and claim information for a customer number.
Viewing Warranty Tags by Customer Number
To view warranty tag information:
1
Select Inventory > Inquiries > Warranty Tag Inquiries > Warranty Tag By Customer to open the Warranty Tag by Customer Inquiry.
2
Enter the Customer.
Detailed tag information for the customer displays.
Field & Function Key Descriptions
Field/Function Key
Description
Customer
The customer number.
Product
The product number assigned to a warranty tag.
Description
The product description.
Tag Num
The tag number assigned to the product. Refer to Moving Products into Warranty Inventory for information on assigning a tag number to a product.
Whse
The warehouse where the tagged product exists.
Customer
The customer number for the tagged product. Refer to Creating Warranty Tags in Credit Memo Entry.
Document
The document number on which the tag number was created. The prefix indicates how the tag number was assigned:
•
R - The tag number was assigned in receiving. Refer to Creating Warranty Tags in Stock Receipts Entry.
•
S - The tag number was assigned on a credit memo to a customer. Refer to Creating Warranty Tags in Credit Memo Entry.
•
A - The tag number was assigned when inventory was adjusted. Refer to Creating Warranty Tags in Inventory Adjustment Entry.
Doc-Date
The date the warranty tag was assigned.
Claim#
The vendor claim number if the tag is on a vendor claim.
Clm-Date
The date the vendor claim was issued, if applicable.
App
Indicates if the claim was reconciled:
•
Y - The vendor claim was reconciled.
•
Null - The vendor claim is not reconciled.
Vendor RMA
The vendor’s return material authorization number.
F2-Finish
Closes the inquiry screen.
F5-Review Tag
Opens the Warranty Tag Inquiry Screen to allow you to view detail warranty information for a product on the warranty tag.
F6-Enter Tag#
Allows you to enter a tag number to go directly to.
F9-Goto Line
Allows you to enter a line number, reference number, or page number to go directly to.
