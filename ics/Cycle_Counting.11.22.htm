Entering 1st Level RF Cycle Counts
RF devices allow you to enter counts as you count. The list of items to count is generated from the Cycle Count First Level Report.
To enter counts using RF guns:
1
Select Inventory > Physical Inventory > Cycle Counting Procedure > Radio Frequency Cycle Count Entry (1st Level) to display the Cycle Count Entry screen.
2
Enter the number for the Warehouse you are counting.
3
Enter your Counter ID.
4
Enter the Zone to count.
5
Enter the # Of Bins to count for this session.
6
Press ENTER to display the next Cycle Count Entry screen.
The Zone and the first Bin to count display.
7
Scan the barcoded product. The Product Number displays. Always scan the product as you see it; however, you can press F3 to list the products in the bin. The products are sorted in alphanumeric order for each bin. If the product is lot controlled, the Lot number displays.
Scan First Level Counts in Warehouse Maintenance determines if the product number displays or if you have to enter it.
8
For Qty, enter the quantity of the scanned product. If the count differs from the current on-hand quantity, depending on how close the count is, one of the following occurs:
•
If the quantity is within the defined tolerance range, the count is accepted and the next product in the bin displays. The inventory is adjusted for the item when you press F2-Update to update the bin count.
•
If the quantity differs from the current on-hand quantity by more than the allowable cost tolerance percent (refer to Cost Tolerance Maintenance), the following message displays:
Choose from the following:
This Count is Out of Tolerance. Do You
Wish to Recount?
-
Yes - Allows you to re-enter a new count.
-
No - Sends the item to the second level count queue for a recount. Continue with the next item.
•
If the quantity is within the defined tolerance range, the count is accepted and the next product in the bin displays. The inventory is adjusted for the item when you press F2-Update to update the bin count.
9
Repeat steps 7-9 for each product.
10
When you finish entering counts for the bin, press F2-Upd Bin. Depending on whether all the counts for the bin are entered, one of the following displays:
•
If you did not enter counts for all of the items in the bin, the following message displays:
Choose from the following:
## Product/Lot Combinations Have Not Been Counted! Do You Wish To Continue?
-
No - Allows you to go back and enter counts for the rest of the items.
-
Yes - Returns uncounted bins to the count queue.
-
Skp - Skips the current bin and goes to the next bin.
•
If you did enter counts for all the items in the bin, inventory is adjusted for any changes in quantity that are within the acceptable tolerance limits. The product numbers are updated with a transaction audit code of ADJ in the Transaction Audit Inquiry.
11
If another bin location exists in the defined zone, repeat steps 7-10 for each bin.
If you select F6-Exit any time while entering counts, any quantities entered in that bin are not updated.
