Warranty Tag Information screen
The Warranty Tag Information screen allows you to enter warranty tag information when moving a defective item into warranty inventory.
Field and Function Key Descriptions
Field/Function Key
Description
Return/Scrapped
Indicates where the material will go:
•
R - Returned to the vendor.
•
S - Scrapped.
In either case, you will receive a credit or replacement from the vendor.
Vendor RMA#
The vendor’s return material authorization number, if available.
Tag Number
The number to identify and track the warranty item. You can enter your own number or enter N for a system assigned number.
Ln#
The warranty tag line number.
Tag Quantity
The defective quantity. You can enter more than one tag number per line item but the total in the Tag Quantity column must equal the Quantity entered on the Line Item Credit Detail screen.
F2-Save
Saves the warranty tag information.
F5-Vendor Return Policy
Opens the Vendor Return Policy Inquiry to allow you to view the return policies for the vendor.
