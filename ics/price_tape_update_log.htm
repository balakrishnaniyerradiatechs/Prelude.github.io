Price Tape Update Log
The Price Tape Update Log provides an audit trail of all the price tapes that updated your product prices (refer to Updating the Price Tape Data). For each price tape, you can view which files were updated.
Menu Path: Inventory > Processes > Price/Cost Update Procedure > Price Tape Update > Price Tape Update Log
Field & Function Key Descriptions
Field/Function Key
Description
Price Tape Name
The price tape name (refer to Setting Up a Price Tape).
Vendor
The number for the vendor whose prices were updated.
Description
The vendor description.
User
The user ID of the person that ran the update.
BegDate
The date the price tape started running.
BegTime
The time the price tape started running.
EndDate
The date the price tape completed.
EndTime
The time the price tape completed.
F2-Finish
Closes the inquiry.
F5-Files
Displays the files that were updated for each price tape.
