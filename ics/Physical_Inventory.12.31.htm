Printing Physical Inventory Tags
You can print physical inventory tags for all items in your inventory after you save the current on-hand quantities (refer to Saving Current Onhand Quantities for Tags). If you want to print tags before your physical inventory, you do not have to save the on-hand quantities before printing the tags, but the current on-hand quantities do not print on the tags.
To print physical inventory tags for counting:
1
Select Inventory > Physical Inventory > Inventory Tag Procedure > Physical Tags Printing to open the Physical Inventory Tag Printing screen.
2
Enter the Warehouse. Leave blank to include all warehouses.
3
For Physical Tag Type, determine the type of tags to print. Choose from the following:
•
T - Print tags for each product in each bin in the warehouse.
•
R - Reprint a tag or group of tags. You may need to use this if tags are missing (refer to the Missing Tag Report).
•
M - Print manual tags for any product/bin without a tag.
•
N - Print tags for rental products only.
4
For Tag Form Method, indicate the type of page breaks. Choose from the following:
•
B - Print a new page for each bin.
•
T - Print a new page for each product.
5
For Beginning Tag Number, enter the beginning tag number to print. If you are printing manual tags, the beginning number must be over 800,000.
6
For Ending Tag Number:
•
If you chose T for Physical Tag Type, the ending tag number is generated by the system.
•
If you chose any of the other options for Physical Tag Type, enter the ending tag number to print.
7
Press ENTER. The following message displays:
Load Physical Tags in Printer & Align Visually
8
Choose from the following:
•
Print Physical Tags: Starts the print process.
•
Mask Print Test: Runs a test print to see if the printer is correctly aligned.
•
Abort: Cancels the process.
Summary Page/Tag Number Listing
At the end of the tag count sheets, a list of tag numbers and corresponding page numbers prints. You can use this summary listing to track count sheets to make sure that all pages are turned back in.
9
Once you print the tags you can distribute them to the proper bins. Re-print tags using the M (manual) Physical Tag Type for any bins without tags.
Default Printer
The tags automatically print to the default printer set up for your user ID. Contact your system administrator for information on changing your default printer.
