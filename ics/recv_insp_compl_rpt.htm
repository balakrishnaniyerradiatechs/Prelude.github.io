Receiving Inspection Completion Report
The Receiving Inspection Completion report displays products released from the Quality Control Inspection Inquiry. You can use this information to determine which products to move from inspection bins to picking bins.
After the report runs, a dialog box appears to allow you to purge the product information so that it does not appear the next time you run the report.
Refer to Requiring Quality Control Inspections in Receiving for information on releasing products from Quality Control Inspection
Menu Path: Inventory > Reports > Stock Receipts Reports > Receiving Inspection Completion Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse that received the material.
•
Beginning and Ending Completion Date Range. The inspection completion (release) date range to include in the report. Leave blank to include all released items not yet purged.
Column Descriptions
Column Heading
Description
Product#
The product released from the inspection queue.
Description
Line 1: The product description.
Line 2: The second line of description for the product, if applicable.
Qty Recv
The quantity received, inspected, and released.
Vendor#
The vendor part number, if applicable.
P/O#
The receiver number for the product.
Release Dte
The date the product was released from the inspection queue.
Results
The test result code for the product number:
•
A - Accepted.
•
W - Waived.
•
R - Rejected. The product does not go to the picking bin and is assigned a warranty tag for return to the vendor.
Comments
Inspection comments entered when the product was released from the Quality Control Inspection Inquiry.
