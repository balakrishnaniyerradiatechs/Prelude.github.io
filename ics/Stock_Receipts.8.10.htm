Setting Up Rejection Claim Reason Codes
Rejection codes identify why an item is being returned to the vendor.
For a listing of Rejection Claim Reason codes, select Inventory > Listings > Miscellaneous Listings > Rejection Claim Reason Codes.
To set up rejection claim reason codes:
1
Select Inventory > Maintenance > Misc Code Files > Rejection Claim Reason Codes to open the Rejection Claim Reason Code Maintenance screen.
2
For Reason Code, enter a 1 or 2-digit alphanumeric code to use in Stock Receipts Entry or Late Rejections Entry when an item is rejected.
3
Enter a Description of the reason code to identify why the material is being returned (e.g., damaged, over-shipped, or an incorrect item).
4
For Exclude from Quality Rpt?, determine if you want items with this rejection code to show up on the Vendor On-Time Receipts/Quality Report:
•
Y - Include the item on the report.
•
N - Do not include the item on the report.
5
Press F2-Save to save the rejection code information.
