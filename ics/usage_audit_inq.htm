Usage Audit Inquiry
The Usage Audit Inquiry screen displays inventory transactions for a product by warehouse and by month. The usage includes sales orders, work orders, and transfers.
The Usage Audit Xref program on the Rebuild menu rebuilds the usage data. You can run it periodically if data is missing.
Viewing Usage Transactions
To view inventory transactions for a product:
1
Select Inventory > Inquiries > Transaction Audit Inquiries > Usage Audit Inquiry to open the Usage Audit Inquiry screen. You can also access this screen by selecting Inventory > Maintenance > Product Maintenance > Product Usage Maintenance > (select a month) > F7-Audit.
2
Enter the Product.
3
Enter the Warehouse that contains the product.
4
Enter the Usage Month month to view.
5
Enter the Year to view.
6
For Type, select the type of transactions usage to view:
•
A - All usage.
•
S - Sales order usage.
•
T - Transfer usage.
•
W - Work order usage.
The transactions for the selected criteria display.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number.
Warehouse
The warehouse that contains the product.
Usage Month
The month to view.
Year
The year to view.
Type
The types of usage:
•
A - All usage.
•
S - Sales order usage. Includes inventory adjustments.
•
T - Transfer usage.
•
W - Work order usage.
Code
The audit inquiry transaction code representing the type of transaction that affected the product usage:
•
ADJ - Inventory adjusted.
•
C/M - Credit memo issued.
•
CMV - Move Customer Usage Procedure performed (Forecasting module).
•
INV - Sales order invoiced.
•
L/S - Lost sales recorded.
•
MOV - Product usage moved.
•
PUA -Usage moved to another part number when the other part number was set up as the Usage Product in the Related Product Information screen (refer to Updating Usage for Another Product).
•
SOE - Sales order entered.
•
SUP - Product superseded.
•
T/R - Transfer shipped.
•
UPR - Universal product replaced/merged.
•
WOI - Work order completed.
Trn-Date
The date of the transaction.
Document#
The document number on which the product was sold or adjusted.
CN/Whse
The customer number if the document is a sales order. The warehouse number if the document is a transfer.
Description
The name of the customer or warehouse.
Type
The type of order:
•
S - Sales order.
•
W - Work order.
•
T - Transfer.
Quantity
The quantity of the product sold on this order.
F2-Finish
Closes the Usage Audit Inquiry screen.
F5-Review
Opens the Order History Inquiry Display screen to display detail information for the highlighted sales order.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
