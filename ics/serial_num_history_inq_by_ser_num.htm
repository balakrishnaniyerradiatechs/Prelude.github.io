Serial Number History Inquiry By Serial Number
The Serial Number History Inquiry by Serial Number displays product history for a serial number for a transaction code.
Viewing Serial Number History
To view the history of a serial number:
1
Select Inventory > Inquiries > Product/Serial Number Inquiries > Serial Number History Inquiry By Serial Number to open the Serial Number History Inquiry By Serial Number screen.
2
Enter the Serial Number. The associated product displays.
3
For Activity Code, enter the transaction code for which you want to view historical transactions (Refer to theSerial Number Transaction Codes).
The history information for this serial number/transaction code displays.
Field & Function Key Descriptions
Field/Function Key
Description
Serial Number
The serial number for which you want to display transactions.
Product
The product number associated with the serial number. The description of the product displays.
Activity Code
The transaction code for the serial number. Refer to Serial Number Transaction Codes.
Last Activity
The date of the last transaction involving this serialized product.
Bin
The bin of the serialized item.
Type
Inventory status for the serialized product:
•
S - Scrapped.
•
D - Damaged.
•
W - Warranty.
•
Blank - Regular inventory.
FP
Indicates if this product/serial number has a floor plan set up (a product that you do not pay for until sold):
•
Y - A floor plan exists for the product.
•
N - No floor plan exists for the product.
Onhand
The on-hand quantity of this product/serial number.
Warehouse
The warehouse number where the transaction took place. The warehouse description displays.
Trans#
The transaction number containing this product/serial number.
Code
The serial number transaction code regarding the product serial number. For an explanation of codes, refer to Serial Number Transaction Codes.
Trn-Date
The date the transaction occurred.
Reference
References the transaction number (e.g., customer number, vendor number, warehouse number, serial number). For an explanation of references, refer to Serial Number Transaction Codes.
Description
The customer, vendor, warehouse for the corresponding to the Reference number.
Price
The gross selling price of the product.
Extension
The extended price of the total quantity invoiced.
Cost
The cost of the product.
From
The warehouse that shipped the item if the transaction is a transfer.
To
The warehouse receiving the item.
User
The user ID of the employee who entered the transaction.
F2-Finish
Closes the screen.
F5-Review
Opens the corresponding inquiry screen, depending on the type of document that contains the product.
F6-Floor Plan
Opens the Serial Number Floor Plan Inquiry to allow you to view A/P floor plan information.
F7-Shop
Opens the Shop Repair Information screen to allow you to view shop repair information, if applicable.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Book Value
Opens the Book Value screen to display the transaction date, cost and book value for the selected transaction. This information may be updated by stock receipts, depreciation, cost adjustments, equipment attachments, or capital repairs.
Opts
Opens the Equipment Options screen to list equipment options assigned to the product/serial number. Refer to Using Equipment Option Codes.
