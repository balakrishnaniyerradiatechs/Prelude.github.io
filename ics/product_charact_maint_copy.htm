Product Characteristics Maintenance Copy screen
The Product Characteristics Maintenance Copy screen allows you to copy information from an existing product characteristic code to create a new code.
Menu Path: Inventory > Maintenance > Product Control Files > Product Characteristics Codes > Action Bar > Copy
Field and Function Key Descriptions
Field/Function Key
Description
Characteristics Code
The characteristics code to copy (refer to Setting Up Product Characteristic Codes).
To Characteristics Code
The new characteristic code to create.
Description
The description of the new product characteristic code.
F2-Save
Saves the new product characteristic code.
