Equipment Options screen
If you purchase or receive a product with an Option Group assigned in the Product Maintenance Additional Information Screen, the Equipment Options screen appears to allow you to enter or verify any options available for this product. The option information is stored in the product serial number record.
Refer to Using Equipment Option Codes for more information.
Menu Path: Inventory > Stock Receipts Entry > Receive a product with equipment options.
Field and Function Key Descriptions
Field/Function Key
Description
Option
The option code assigned to the product that identifies the optional features available with this piece of equipment. Refer to Using Equipment Option Codes.
Description
The description of the feature.
F2-Save
Saves the equipment option information.
