Entering Physical Inventory Counts from Tags
As each area of the warehouse is counted, you can collect the tags and enter the counts.
Serial Number Products
To include serialized products in the physical inventory tag procedure, set Phys. Inv. Include Serial# to Y in Inventory Parameters > F5-Add Info.
To enter physical inventory counts from tags:
1
Select Inventory > Physical Inventory > Inventory Tag Procedure > Physical Tag Entry to open the Physical Tag Header screen.
2
Enter the Warehouse number for the tags.
3
Enter the Picker Initials from the tags.
4
Enter the Starting Tag Number. The Physical Inventory Tag Entry screen appears.
The Warehouse, Tag Number, and Product display for the first item on the list.
5
Enter the Physical Count for each product for each bin. This count is consolidated later for a total on-hand count for this product. If the count was zero, enter a zero. If a count already exists, verify and change the count as necessary.
Lot Products
Refer to Entering Counts From Tags for Lot Products for information on entering lot product counts.
Expanded Entry Screen
Refer to Entering Inventory Counts on an Expanded Screen for information on entering multiple counts for the same item.
6
Press ENTER after you enter each count. The next tag number appears and you can continue to enter counts.
7
After you enter all the counts, press ESC. The following prompt appears:
Do You Wish To Exit From Physical Tag Entry?
8
Choose from the following:
•
No - Allows you to continue entering counts.
•
Yes - Exits the count entry screen and saves all counts already entered.
•
Return to Header - Returns to the Physical Tag Header screen to allow you to enter a different starting tag number.
The following reports are available to check counts before you update inventory. If you adjust the count entries, you can run the reports as many times as necessary before you update inventory.
Related Reports
Description
Physical Tag Deviation Report
Compares the saved on-hand quantities to the new count entries and prints the products that fall within the defined deviation range.
Physical Inventory Shrinkage Valuation Report
Displays the saved on-hand quantity and the counts for all inventory items, and displays the difference in value.
 
After you verify counts, run the Physical Update Procedure (refer to Updating Physical Inventory From Tag Counts) to update the counts into the on-hand balance for each item.
