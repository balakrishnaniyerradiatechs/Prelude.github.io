Product Classification Report
Product classifications are based on the total value of a product that moves through inventory each month. The last 12 months of usage and the cost of each stock item are used to calculate a usage value. Class 1 is assigned to items with the greatest turns and value and class 13 is assigned to items with no sales. You can set purchase parameters to determine which products are assigned to each class. Refer to Classifying Products for more information on product classifications.
Menu Path: Inventory > Reports > Miscellaneous Reports > Product Classification
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse(s) to include.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Inventory Class. The inventory class to include (1-13) or leave blank to display all classes.
•
By Cls, Usg, Val?. Determines the sort sequence for the report. Choose from the following:
-
C - Sorts the report by inventory class, starting with class 1. All products in the same class are then sorted by the total usage value of the product for the month.
-
U - Sorts the report by usage (the total number of units sold for the month).
-
V - Sorts the report by usage value (the extended value of the total units sold).
Column Descriptions
Column Heading
Description
Product Number
The product numbers sorted by classification, usage, or usage value, depending on the sort option chosen.
Description
The description for the product.
Usage
The total quantity of the product allocated to sales orders, work orders, and transfers in the last 12 months.
Unit-Cost
The replacement cost of the product.
Usage Value
The extended cost of the usage quantity for the month (Usage x Unit-Cost).
Class
The product classification based on the value of the item that moves through inventory each month. Class 1 is assigned to items with the highest turns and value; class 13 represents items with low turns. Refer to Classifying Products for more information.
