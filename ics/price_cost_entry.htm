Price / Cost Entry screen
You can use the Price / Cost Entry Screen to specify the prices and costs to update for a list name.
Refer to Updating Prices and Costs for more information.
Menu Path: Inventory > Processes > Price/Cost Update Procedure > Price Cost Changes by Lists > Price/Cost Entry & Update
Field and Function Key Descriptions
Field/Function Key
Description
Product or Warehouse
Determines whether you are updating prices and costs at the product level or the warehouse level:
•
P - Update prices/costs at the product level.
•
W - Update prices/costs at the warehouse level (for products with unique prices for each warehouse).
Warehouse
The warehouse number for which you want to create the list.
Save List Name
The name of the list for the products you want to update. Refer to Building Price/Cost Lists.
Starting Product
The beginning part number if you want to update only select products on the list. For example, you may want to use a starting part number if you had to stop when previously updating. Leave blank to update all products.
Future or Current
Indicate which prices and costs to update:
•
F - Future prices and costs.
•
C- Current prices and costs.
Future Effective Date
The date the prices take effect. Only accessible if Future or Current is set to F.
Update Retail Price
Indicates if retail price should be updated:
•
Y - Update the retail price.
•
N - Do not update the retail price.
Update Price 1
Indicates if price 1 should be updated:
•
Y - Update price 1.
•
N - Do not update price 1.
Update Price 2
Indicates if price 2 should be updated:
•
Y - Update price 2.
•
N - Do not update price 2.
Update Price 3
Indicates if price 3 should be updated:
•
Y - Update price 3.
•
N - Do not update price 3.
Update Price 4
Indicates if price 4 should be updated:
•
Y - Update retail price 4.
•
N - To not update the price 4.
Update Price 5
Indicates if price 5 should be updated:
•
Y - Update price 5.
•
N - Do not update price 5.
Update Replace Cost
Indicates if replacement cost should be updated:
•
Y - Update replacement cost.
•
N - Do not update replacement cost.
Update Standard Cost
Indicates if standard cost should be updated:
•
Y - Update standard cost.
•
N - Do not update standard cost.
Update Cost For Sls
Indicates if cost for sales should be updated:
•
Y -Update the cost for sales.
•
N - Do not update the cost for sales.
Update Dir Shp Cost
Indicates if direct ship cost should be updated:
•
Y - Update the direct ship cost.
•
N - Do not update the direct ship cost.
F2-Save
Opens the Price / Cost Update screen.
F4-Delete
Deletes the price/cost update information.
