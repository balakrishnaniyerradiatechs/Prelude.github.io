Core Inquiry
The Core Inquiry displays the combined status of core bank records and pending core records for a core item. You can view the inventory adjustments to see which cores are applied.
Viewing Core Records
To display the status of core bank records and pending core records:
1
Select Inventory > Inquiries > Core Inquiries > Core Product Inquiry to view the Core Inquiry screen.
2
Enter the Product.
3
For Ret. Type, enter the return type to limit the report or leave blank to list all return types. Choose from the following:
-
A - All core bank records for a customer.
-
E - Only core bank records with a remaining quantity.
The bank core and pending core status information appears.
Field and Function Key Descriptions
Field/Function Key
Description
Product
The product number.
Ret.Type
Determines the return types to display:
•
A - All core bank records for a customer.
•
E - Only core bank records with a remaining quantity.
Document#
The pending core record ID or the core bank ID depending on which type you are viewing.
Doc Date
The date the record was created.
Due Date
The date by which the pending core should be returned (pending core records only).
Type
The type of record:
•
P - Pending core.
•
B - Core bank.
Customer
The customer number associated with the transaction.
Ship/Rec
The number of cores shipped if the corresponding record is a pending core. The number of cores received into the core bank if the record is a core bank.
Applied
The number of cores that are applied. If the corresponding record is a pending core, then this is the number of cores applied against it. If this is a core bank record, this is the number of cores that are applied against a pending core.
Remain
The number of cores that still need to be returned or billed if the corresponding record is a pending core. The number of cores still available to apply if the corresponding record is a core bank record.
F2-Save
Clears the inquiry screen.
F5-Review Ship/Rec
Opens the Sales Order History Inquiry for pending core items, or the Inventory Adjustment Display screen for core bank items.
F6-Review Applied
Displays the applied core bank record for pending core items, or the inventory adjustment record for core bank items.
F9-Goto Line
Allows you to enter a line number, reference number, or page number to go directly to.
