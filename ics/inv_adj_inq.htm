Inventory Adjustment Inquiry
The Inventory Adjustment Inquiry displays all the inventory adjustments for a warehouse. You can view detail information for each adjustment.
Viewing Adjusted Inventory
To view inventory adjustments for a warehouse:
1
Select Inventory > Inquiries > Inventory Adjustment Inquiry to display the Inventory Adjustment Inquiry screen.
2
Enter the Warehouse number.
Inventory adjustments for the warehouse display.
Field & Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number. The warehouse description displays.
Adjustment#
The adjustment number.
N
An asterisk (*) indicates that notes were entered in the header screen when inventory was adjusted.
Ent-Date
The date for the adjustment.
Period
The fiscal period of the adjustment.
Type Of Transaction
The type of inventory adjustment:
•
INV - Regular Inventory.
•
DMG - Damaged inventory.
•
SCR - Scrapped/lost inventory.
•
WTY - Warranty Inventory. Refer to Creating Warranty Tags in Inventory Adjustment Entry.
User
The user ID of the person who entered the transaction.
Cost
The total amount adjusted.
F2-Finish
Closes the inquiry screen.
F5-Review
Opens the Inventory Adjustment Display screen to allow you to view detail information for the adjustment.
F6-Enter Adjustment
Allows you to enter an adjustment number for direct access.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Returns to the previous page if more than one page of transactions are listed. The number of records that display depend on Records to Block in Company Maintenance > Action Bar > GUI Options.
sF6-Next Group
Forwards to the next page if more than one page of transactions are listed. The number of records that display depend on Records to Block in Company Maintenance > Action Bar > GUI Options.
