FIFO Inquiry
The FIFO Inquiry displays the FIFO cost of each layer of inventory for a product.
Viewing FIFO Cost
To view FIFO cost layers for a product:
1
Select Inventory > Inquiries > Miscellaneous Inquiries > FIFO Inquiry to open the FIFO Inquiry screen.
2
Enter the Product for which you want to display FIFO costs.
3
Enter the Warehouse if the FIFO Method flag is set to W in Company Maintenance. Refer to Setting System Flags for FIFO Costing.
The FIFO costs and related document numbers for the product appear.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number.
Warehouse
The warehouse where the product is stocked if the FIFO Method flag is set to W in Company Maintenance. Refer to Setting System Flags for FIFO Costing.
Onhand
The quantity on-hand for this warehouse.
Stock Value
The dollar amount of the quantity on-hand.
Fifo-Dte
The date the product entered the warehouse on the corresponding Document#.
Document#
The receiving document number for the product.
C/V/W
The number of the customer, vendor, or warehouse that shipped the product.
Qty
The quantity received on the corresponding Document#.
Cost
The unit cost for each of the quantities received on the corresponding Document#.
Freight
The freight charge amount added to the cost of the product.
Duty
The duty charge amount added to the cost of the product.
Remain
The quantity remaining for the corresponding layer.
OK
Closes the inquiry screen.
