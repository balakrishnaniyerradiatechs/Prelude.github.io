Product Usage Maintenance Screen
The Product Usage Maintenance screen allows you to view and adjust sales order, work order, and transfer usage for a product. You can also access summary, audit, product hit, and weekly usage information from this screen.
Menu Path: Inventory > Maintenance > Product Maintenance > Product & Usage Maintenance
You can also access the Product Usage Maintenance screen from within Product Maintenance or Product Warehouse Maintenance > F5-Usage.
Product Usage Inquiry
The Product Usage Inquiry (Inventory > Inquiries > Maintenance Inquiries > Product Inquiries > Product Usage Inquiry) displays the same information as Product Usage Maintenance.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number and description.
Warehouse
The warehouse number.
Month
The months of the year.
S/O Usage
The usage quantity calculated from sales orders for the month. This number also reflects changes made to S/O Usage in Product Usage Maintenance.
Override
A space to enter an override to S/O Usage for the corresponding month, if applicable.
W/O Usage
The usage quantity calculated from work orders for the month.
Override
A space to enter an override to W/O Usage for the corresponding month, if applicable.
T/R Usage
The usage quantity calculated from transfers for the month.
Override
A space to enter an override to T/R Usage for the corresponding month, if applicable.
Forecast
The calculated forecast quantity.
Formula
The forecast formula for this product, if applicable.
Exc
Indicates if exceptions exists:
•
U - Unusual usage. Monthly usage is greater than the total of the last 5 months of combined usage.
•
O - Out of stock. A product is out of stock for 2 weeks or more in one month.
•
S - Spike. Monthly usage is above or below the minimum or maximum levels set for the product by the percentage defined in Spike Exception - Min/Max set in Purchasing Parameters Maintenance (Systems Menu). Refer to “Entering Formulas/Parameters by Company” in the Forecasting documentation for more information on spikes.
F2-Save
Saves the override information and closes the screen.
F6-Summary
Opens the Usage Summary Screen. You can view the total usage for all sales orders, work orders, and transfers.
F7-Audit
Opens the Usage Audit Inquiry. You can view the inventory transactions that updated usage.
F8-Prd Hits
Opens the Warehouse Hits Inquiry Screen. You can view the number of times the item was entered on a sales order, work order, or transfer.
F9 - Weekly Usage
Opens the Current Year Weekly Usage Screen to allow you to view the week by week usage of the item on sales orders, work orders, and transfers. From here you can also access the Previous Year Weekly Usage screen.
Action Bar
Graphs
Allows you to view graphs which display usage per month for the following:
•
Current total usage.
•
Current year sales order usage.
•
Current year work order usage.
•
Current year transfer usage.
•
Previous total usage.
•
Previous year sales order usage.
•
Previous year work order usage.
•
Previous year transfer usage.
Parameters
Opens the Production Control Parameters screen to display manufacturing information for work orders.
Output
Allows you to view, print, email, or fax the usage information.
Company Usg
Opens the Company Wide Usage screen to display product usage and override information for the entire company.
