Accessory Group Maintenance screen
The Accessory Group Maintenance screen allows you to identify a group of products that are accessories for a product or multiple products. You can assign the accessory group to a product in Product Maintenance (refer to Accessory Products), so that when you enter the product on a sales order, all the items included in the group display. You can then decide which products, if any, to add to the order. Refer to Accessory Groups for more information on setting up accessory groups.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance F6-Xref/Notes > Accessories > F5-Group
Field and Function Key Descriptions
Field/Function Key
Description
Accessory Group
The name assigned to the group of accessory products.
Description
A description of the group (e.g., Golf Accessories, Tool Kits).
Description 2
The second line of description for the product, if necessary.
Product
The stock product numbers to include in the accessory group.
Description
The product description.
Script
Determines if a script message displays when you enter the product on a sales order or quote:
•
Y - Opens the General Notes Entry screen to allow you to enter a message that could help the sales person sell the accessory items.
•
N - No script message is included for this product group.
•
* - A script message already exists for this group.
F2-Save
Saves the accessory group information.
F4-Del
Deletes the accessory group.
