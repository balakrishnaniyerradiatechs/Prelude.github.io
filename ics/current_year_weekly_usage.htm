Current Year Weekly Usage Screen
The Current Year Weekly Usage screen allows you to view and adjust sales order, work order, and transfer weekly usage values for a product, for the current year.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Usage Maintenance > F9-Weekly Usage
Statistics Sequence
Reverse Statistics Sequence (Inventory Parameters > F5-Add Info) determines the order to display statistics screens:
•
Y - Usage and statistics screens appear in reverse order (December to January) for each calendar year.
•
N - Usage and statistics screens appear in regular order (January - December).
Field & Function Key Descriptions
Field/Function Key
Description
Grand Totals
The total amount of each column.
Begin
The beginning date (Monday) of the week.
End
The ending date of the week.
S/O Usage
Product usage recorded from sales orders for the week.
Override
A space to enter an override to S/O Usage for the corresponding month.
W/O Usage
The usage quantity calculated from work orders for the month.
Override
A space to enter an override to W/O Usage for the corresponding month.
T/R Usage
The usage quantity calculated from transfers for the month.
Override
A space to enter an override to T/R Usage for the corresponding month.
F2-Save
Closes the screen.
F5-Previous Year
Displays the Previous Year Weekly Usage screen.
