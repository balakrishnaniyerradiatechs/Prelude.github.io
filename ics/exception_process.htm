Evaluating Unusual Usage
To detect unusual usage for each product in a warehouse, you can run the Usage Exception Evaluation Procedure. You can also run this as part of the calendar month end.
To detect unusual usage:
1
Select Inventory > Processes > Usage Exception Menu > Usage Exception Evaluation Procedure to open the Usage Exception Evaluation Procedure screen.
2
For Enter Month End Closing Date, enter the last month-end to include. For example, if today is September 15, 2005 enter 8/31/05.
3
Enter the Warehouse numbers for which you want to evaluate unusual usage.
4
Select Execute to calculate unusual usage. To view the items with unusual usage, run the Usage Exception Report.
