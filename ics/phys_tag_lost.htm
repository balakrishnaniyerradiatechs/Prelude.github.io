Missing Tag Report
After you enter counts for all tags, run the Missing Tag report to find any tags skipped during the count entry. To reprint the missing tags, refer to Printing Physical Inventory Tags and choose R for the Physical Tag Type.
If Full Physical in Save Current Onhand Quantities is set to Y, you do not need to run the Missing Tag report. Refer to Saving Current Onhand Quantities for Tags.
Menu Path: Inventory > Physical Inventory > Inventory Tag Procedure > Missing Tag Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where you want to check for missing tags.
•
Print Manual Tags on Report. Determines if you want to print the missing manual tags. Choose from the following:
-
Y - Print manual tags. You may not want to use this option because several hundred un-used manual tags may show up as blank on the report
-
N - Do not print manual tags. You can reprint the missing tags later in Physical Tags Printing (R option). Refer to Printing Physical Inventory Tags.
Column Descriptions
Column Heading
Description
Tag #
The tag numbers with no counts entered.
Bin Location
The bin number for the missing tags.
Product Number
The product number with no count entered.
Description
The product description.
Vendor Product Number
The vendor’s product number, if applicable.
