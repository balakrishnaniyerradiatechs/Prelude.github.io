Defining Zones
Zones group similar bins within a warehouse to help streamline picking and cycle counting. If warehouse areas are established (refer to Defining Warehouse Areas). You can also group zones by warehouse area.
Zone Listing
To view a listing of zones set up for each warehouse area, select Inventory > Listings > Miscellaneous Listings > Zone Listing.
To set up warehouse zones:
1
Select Inventory > Maintenance > Bin Files > Zone Maintenance to open the Zone Maintenance screen.
2
Enter the Warehouse.
3
For Zone, enter up to 4 alphanumeric characters to define the zone location.
4
Enter a Description of the zone to help identify the location in the warehouse.
5
Enter a Warehouse Area (refer to Defining Warehouse Areas) to identify the area for this zone (optional).
6
For Zone Type, indicate if this zone is selected for pick ticket printing. Choose from the following:
•
P - Pick. Include the zone in the picking process.
•
C - Cycle count. Do not include the zone in the picking process. This setting does not affect any processes associated with cycle counting but notifies pick ticket printing to check for picking.
In order for the P option in Zone Type to function correctly, the Generate Release By flag in Warehouse Maintenance > F5-Add Info must be set to Z (zone)
7
For Pick Quota Lines/Hr, enter the number of lines to pick per hour when picking in this zone (informational purposes only - not required).
8
For Bin Count Frequency, enter the number of times per year to cycle count this zone. This field is required if you are using cycle counting. Refer to Setting up Your System for Cycle Counting.
9
Press F2-Save to save the zone information.
