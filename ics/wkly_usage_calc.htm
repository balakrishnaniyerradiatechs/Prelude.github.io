Calculating Weekly Usage
After you set up your products with the AWU formula (refer to Establishing the Weekly Usage Calculation Hierarchy), you can calculate new recommended purchase quantities. You can then review and update product classifications as necessary.
Anytime you create a new product with the AWU formula, calculate weekly usage again.
To calculate weekly usage:
1
Select Inventory > Processes > Weekly Usage Evaluation Process to open the Weekly Usage Evaluation Process screen.
2
For Calculation Date, enter the Sunday that begins the week to use for calculations.
3
Enter the Number of Weeks for which you want to calculate average weekly usage. If you leave this field blank, the system uses the setting in Inventory Parameters. Refer to Establishing the Weekly Usage Calculation Hierarchy.
4
For Product Classification, indicate if you want to re-classify products before calculating average weekly usage. Refer to Classifying Products:
•
Y - Reclassify before calculating weekly usage.
•
N - Do not reclassify before calculating weekly usage.
5
Select Execute to calculate weekly usage and reclassify products.
You can review the new weekly usage in the Product Usage Inquiry. Refer to the Product Usage Maintenance Screen.
After you set up and calculate weekly usage, Nightly Processing continues to calculate and update usage (refer to Nightly Processing in the Purchasing documentation). The inventory End of Month Calendar Close reviews and updates the product classifications.
