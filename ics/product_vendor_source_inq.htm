Product Vendor Source Inquiry
The Produce Vendor Source Inquiry displays all the vendors established for a product for each warehouse. The inquiry also identifies which vendor is the primary vendor and the vendor part number, if applicable.
Running the Product Vendor Source Inquiry
To view the vendors assigned to a product:
1
Select Inventory > Inquiries > Maintenance Inquiries > Product Inquiries > Product Vendor Source Inquiry to open the Product Vendor Source Inquiry screen.
2
Enter the Product.
3
Enter the Warehouse number.
The vendors established for the product/warehouse display.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The product number. The product description displays to the right.
Warehouse
The warehouse where the product is stocked. The warehouse description displays to the right.
Vendor
The vendors that supply the product.
Description
The vendor description.
Vendor Part Number
The vendor part number, if applicable.
P
Indicates if the vendor is the primary vendor for this product:
•
Y - The primary vendor for this product. Nightly Processing uses this vendor to recommend purchase orders.
•
N or Null - Not the primary vendor.
Unit Cost
The cost per unit of the product when bought from this vendor.
ALt
The average number of days of lead time required to purchase this product from this vendor.
LLt
The number of days lead time it took the last time this product was purchased from this vendor.
Curr
The currency used to purchase from this vendor.
F2-Finish
Closes the inquiry.
F5-Vendor Inquiry
Opens the Vendor Inquiry screen for the highlighted vendor.
F6-Product Vendor Inquiry
Opens the Product Vendor Inquiry screen. Refer to the Product Vendor Maintenance Screen for information on each field.
