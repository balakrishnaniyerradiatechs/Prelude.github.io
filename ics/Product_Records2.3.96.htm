Customer Ration Types
Sales order rationing allows you to limit the quantity of a product or product line that a customer can buy within a designated number of days. For example, if the customer’s ration quantity of an item is set at 100 units and the ration days for that item is 30, the customer can purchase up to 100 units of that item within the 30 day time frame. Ration comments appear on the sales order when the ration quantity is exceeded. Refer to “Entering Rationed Products” in the Order Processing documentation for more information.
For information on entering ration types and rationing comments, refer to the following sections:

•

Option Parameters
Sales Order Rationing in Order Processing Parameters > F5-Add Info must be set to Y to use Rationing Maintenance.
Kit Components
Kit components are not included as part of this feature. Only kit parent items are checked for rationing.
