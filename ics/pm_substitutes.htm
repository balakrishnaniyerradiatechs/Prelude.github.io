Substitutes screen
The Substitutes screen allows you to enter substitute product numbers for a product. You can set flags to determine when and how the substitute products fill backorders. Refer to Substitute Products for more information.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F6-Xref/Notes > Substitutes
Field and Function Key Descriptions
Field/Function Key
Description
Product Substitution Type
Determines how the substitute product is added to the order. The first three options only apply to customers that allow substitutes (Accept Substitutes set to Y in Customer Maintenance):
•
Substitute Backordered Product. If you enter a backordered quantity of the product, the first substitute product with an available quantity is selected for the order. Pricing is based on the substitute product.
•
Price on Substituted Product. Works the same as the previous but the pricing is based on the original (substituted) product.
•
Supersedes All Other Types. Automatically substitutes regardless of the availability of the original product. For example, the vendor provides a bonus pack of four instead of the usual pack of three. Once the bonus product runs out, the system defaults back to the original product. Pricing is based on the substituted product.
•
Substitute Screen Notification - or none: When you enter the product on a sales order, work order, or transfer, the substitution is not automatic. You can press F6 to view and/or select a substitute product from the available list. Pricing is based on the substitute product.
Auto Sub Prompt in SOE/WOE/TE?
Determines how substitute products are added to a sales order, work order, or transfer:
•
Y - When an item is backordered, the first available substitute item is automatically placed on the sales order or work order. No message displays regarding the substitution.
•
A - The same as Y but also includes transfers.
•
N - When an item is backordered, the Substitute Product? message displays so you can use the substitute product.
Product Substitution Comment 1
The first line of the comment to add to the order when a product is substituted.
Product Substitution Comment 2
The second line of comment to add to the order when a product is substituted.
Substitute Product #
Valid product numbers to use as substitutes for this product. If you enter multiple products, they are checked for availability in the order in which they are entered.
Description
The description of the substitute product.
F2-Save
Saves the product substitute information.
F5-Equivalents
Opens the Equivalents screen to allow you to enter possible future substitute part numbers. An equivalent product cannot be a valid part number in your system. If the equivalent product later becomes a stock product, it will then be changed to a substitute product for this item.
