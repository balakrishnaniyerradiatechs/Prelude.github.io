Product Prefix Description Maintenance screen
The Product Prefix Description Maintenance screen allows you to set up prefixes to identify the manufacturer for a line of products. When you use a prefix to set up part numbers in Product Maintenance, standard information (e.g., price columns, price tape information, product line, vendor number) defaults from the prefix to help expedite the entry of new products. Refer to Using Product Prefixes for more information.
Inventory Parameter Flag
Set Product# Prefix in Inventory Parameters to Y to require product number prefixes for all part numbers. You must enter the Length, which is the number of digits for the product prefix.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Prefix Maintenance.
Field and Function Key Descriptions
Field/Function Key
Description
Product Prefix
The prefix to identify the manufacturer. Typically, an abbreviation of the vendor’s name. The length of the prefix depends on the value of Length in Inventory Parameters.
Description
The description of the product prefix (e.g., the vendor’s name).
Maximum Price Column
The maximum number of price columns to default for items with this product prefix.
Sugg. Retail Price
The suggested retail price description to default in the pricing screen in Product Maintenance when a new product is created with this prefix.
Price 1 Desc. - Price 5 Desc.
The price column descriptions to default in the pricing screen in Product Maintenance when a new product is created with this prefix.
Product Line
The default product line in Product Maintenance for new products created with this prefix.
Vendor Number
The vendor number to default for a product in the Vendor Number field when a new product warehouse record is created with this prefix.
Vendor Line
The vendor line number to default in the Vendor Line field when a new product vendor record is created with this product prefix.
Price Tape Flag
The default price type to use in Product Maintenance > F5 Add Info when a new product is created with this prefix:
•
Y - The product pricing is updated by a price tape.
•
N - The product pricing is not updated by a price tape.
F2-Save
Saves the product prefix information.
F4-Del
Deletes the product prefix record.
F5-Add Price Columns
Allows you to add extra price column descriptions if the Additional Price Columns feature is activated.
