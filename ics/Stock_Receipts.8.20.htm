Generating UPCs for New Products
If UPCs are required for all products, you can set maintenance flags to automatically generate UPCs when a new product is created.
To automatically create UPCs for all products in inventory that do not already have a UPC, select Inventory Utilities > Create UPC Codes For Products.
To automatically generate UPCs for new products:
1
Select Option Parameters > Inventory Parameters.
2
For Generate UPC Code, choose from the following:
•
Y - Generate a UPC each time a new product is created.
•
C - Generate a UPC using the 12-digit checksum logic. The first 11 digits identify the manufacturer and item; the 12th digit is for validation.
•
R - Generate a UPC for products received with a radio frequency device.
•
D - Generate a UPC using the 12-digit checksum logic for products received with a radio frequency device. The first 11 digits identify the manufacturer and item: the 12th digit is for validation.
•
N - Do not generate a UPC code.
3
In the next (unlabeled) field, enter a numeric prefix for the UPC code. For example, if the prefix is 333 and the next system generated number is 22338, the UPC is 33322338.
4
In the next (unlabeled) field, determine how to find UPC numbers using product keywords. Choose from the following:
•
Y - Check keywords before alternate part numbers.
•
N - Check alternate part numbers before keywords.
5
Press F2-Save to save the UPC information.
Every product created will now generate and store a UPC as an alternate part number. When you receive the item, you will not be required to enter a UPC.
