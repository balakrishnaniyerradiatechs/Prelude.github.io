Accessory Information screen
The Accessory Information screen allows you to enter items that you can sell to go with a product. When you enter a sales order or quote for a product that has accessory products, you are notified that accessories exist. You can choose to display the accessories and then add them to the order. Refer to Accessory Products for more information on setting up accessory products.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance F6-Xref/Notes > Accessories
Field and Function Key Descriptions
Field/Function Key
Description
Accessory Product#
The accessory product number that may be sold with the original product. Any time you sell the original product, you can choose to add the Accessory Product Number to the sales order.
Description
The description of the Accessory Product Number.
Script
Determines if a script message displays each time the product is entered on a sales order or quote:
•
Y - Opens the General Notes Entry screen to allow you to enter a message that could help the sales person to sell the accessory items.
•
N - No script message is included for this product group.
•
* - A script message already exists for this group.
Accessory Group
The accessory group code if you want to assign an accessory group to the product. All products in the accessory group will display when you enter the original product on a sales order. Refer to Accessory Groups for information on setting up an accessory group.
Description
The description of the accessory group.
F2-Save
Saves the accessory information.
F5-Group
Opens the Accessory Group Maintenance screen to allow you to view and maintain products included in the group if you entered a valid Accessory Group.
