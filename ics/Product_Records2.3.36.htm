Copying a Product Characteristic Code
You can copy a product characteristic code to create a new code containing similar information.
To copy an existing product characteristic code:
1
Select Inventory > Maintenance > Product Control Files > Product Characteristic Codes to open the Product Characteristics Code Maintenance screen.
2
From the Action Bar, select Copy to open the Product Characteristics Maintenance Copy screen.
3
Enter the Characteristic Code that you want to copy.
4
For To Characteristic Code, enter the new code that you want to create.
5
For Description, enter the product characteristic code description to assign to the new code.
6
Press F2-Save to save the new code. You can then go back into Product Characteristics Code Maintenance to modify the new code as necessary.
