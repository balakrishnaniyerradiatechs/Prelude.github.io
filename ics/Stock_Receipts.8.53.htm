Allocating Inventory
You can set your system to automatically allocate backordered merchandise to sales orders, transfers, and work orders as material is received into your warehouse (refer to Receiving Stock). Even if you allocate inventory automatically, you should still manually allocate periodically so that increases in inventory due to credit memos will also fill backorders.
If you do not use automatic inventory allocation, you can manually fill backorders for all products with an increase in on-hand inventory. An exception report generates to allow you to review backorders.

Allocation Priority
To set a priority sequence for each customer to determine whose backorders to fill first:
•
Select Order Processing > Maintenance > Customer Maintenance > F7-Order > F5-Add Info.
•
Enter a code in Allocation Priority for each customer using letters from A to Z; A is the highest priority and Z is the lowest priority. Leave blank if not using allocation by priority.
Received items are allocated first to customers with the highest priority (A), then (B) then down the list to customers with the lowest priority (Z). Customers with no priority code are selected last.
•
Press F2-Save 3 times to save the allocation information.
You can override the allocation sequence in Sales Order Entry to set an individual order to a higher/lower priority than normal (F5-Hdr Scr > F5-Add Info > F5 Add Info).
