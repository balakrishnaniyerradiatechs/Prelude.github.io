Carousel NONNET Status Report
The Carousel NONNET Status report displays all products in the carousel bin. If you use the carousel interface for non-nettable products, you can receive products into the carousel bin to remain non-nettable until you move them to the correct bin. A non-nettable product is part of on-hand inventory but is not available to sell until it is moved into a nettable bin. Refer to Receiving Carousel Non-Nettable Products for information on the carousel interface.
Menu Path: Inventory > Reports > Miscellaneous Reports > Carousel NONNET Status Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse containing the non-nettable products.
Column Descriptions
Column Heading
Description
Prod Num
The product in non-nettable status.
Description
Line 1: The product description.
Line 2: The second line of description for the product, if applicable.
Type
Indicates how the product was received into the warehouse:
•
PO - Received on a purchase order.
•
TR - Received on a transfer.
Doc Num
The document number on which the item was received.
Qty
The quantity of the product received into non-nettable status on this receiver.
Rec Date
The date the product was received into non-nettable status.
Rec Time
The time the product was received into non-nettable status.
