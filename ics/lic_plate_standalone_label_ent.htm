License Plate Standalone Label Entry screen
If continuous printing is not set up for license plate labels, you can print labels one at a time in the License Plate Standalone Label Entry screen.
Menu Path: Inventory > Inventory Adjustments > License Plate > License Plate Standalone Label Entry
Field and Function Key Descriptions
Field/Function Key
Description
License Plate #
The license plate number for which you want to print labels.
Product
The product to which the license plate is assigned.
Description
The product description.
Warehouse
The warehouse containing the license plate product.
Lot
The license plate number.
Quantity
The quantity of labels to print.
Continue (Y/N)
Determines if you want to print the label:
•
Y - Print the label.
•
N - Do not print the label.
