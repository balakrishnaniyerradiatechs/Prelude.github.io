Lot Number Information screen
The Lot Number Information screen appears when you receive a lot product with the Lot Pack Flag set to N in the Lot Information Screen (Product Maintenance > F5-Add Info > Lot Number Flag =Y). You can assign lot number information to the product.
Menu Path: Inventory > Stock Receipts > Stock Receipts Entry > Detail Information screen
Field and Function Key Descriptions
Field/Function Key
Description
Lot Number
The lot number from the package.
Quantity
The quantity of the lot product received in this shipment.
Bin
The bin for the lot product, if applicable.
Exp-Date
The expiration date of the lot product, if the lot product was set up to require an expiration date in the Lot Information Screen in Product Maintenance.
Mfg#
The manufacture code is set up for the vendor in Vendor Maintenance.
Description
The description of the manufacturer code.
F2-Save
Saves the lot number information and returns to the Detail Information screen.
F5-Description
Opens the Lot Description Maintenance screen to allow you to enter vendor lot number information.
