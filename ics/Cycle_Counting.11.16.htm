Entering First Level Cycle Counts
After the count sheets print (refer to Printing Count Sheets ) and the products are counted, you can enter the counts directly from the count sheets.
To enter counts:
1
Select Inventory > Physical Inventory > Cycle Counting Procedure > Paper Site Cycle Count Entry (1st Level) to open the Cycle Count Entry 1st Level screen.
If you have access to more than one warehouse, the Location Retrieval screen appears first. Enter the appropriate Selling Warehouse, Department, and/or Cost Center.
2
Enter the Warehouse number.
3
For Counter Id, enter the user ID of the person who counted the products. This user ID must have access to this warehouse (refer to User Identification Maintenance). The counter’s ID The following fields automatically display for any count sheets printed for this counter. If more than one bin is assigned to the counter, then the bins display in alphanumeric order:
•
# Of Bins - The number of bins assigned to the counter.
•
Zone - The zone assigned to the counter.
•
Current Bin - The bin assigned to the products displayed.
•
Product Number - The products in the Current Bin.
•
Lot Number - The lot number assigned to the products, if applicable.
Lot Products
If a product is lot controlled, you can enter the quantity for each product/lot, but you cannot enter a product/lot combination more than once.
4
Enter the Quantity counted for each product. If the count differs from the current on-hand quantity, depending on how close the count is, one of the following occurs:
•
If the quantity is within the defined tolerance range, the count is accepted and inventory automatically adjusts for the item in this bin when you press F2-Update Bin. The product shows a transaction audit code of ADJ in the Transaction Audit Inquiry.
•
If the quantity entered differs from the system on-hand quantity by more than the tolerance range defined in Cost Tolerance Maintenance, the following prompt displays:
Choose from the following:
This Count Is Out Of Tolerance. Do You Wish To Recount?
-
Yes - The cursor returns to Quantity. Re-enter the count.
-
No - The item goes into the second level queue for a re-count.
5
Repeat step 4 for each product.
6
After you enter all the counts for the bin, press F2-Update Bin. Inventory is adjusted for all the items whose count quantities were within the defined tolerance levels. Products whose count equals the current on-hand quantity are not adjusted.
7
If another count sheet exists for this counter, the next bin location appears and you can repeat steps 4-6 for each bin.
8
After updating all the bins assigned to the counter, press F6-Exit. If you exit before entering all counts, the following message displays:
All Product/Lot Combinations Have Not Been Counted! Do You Still Wish To Exit The Process
9
Choose from the following:
•
No - Allows you to continue entering counts.
•
Yes - Displays the following message:
The session is not complete and the remaining items are returned to the cycle count queue to be counted the next day.
All Assigned Bins Were Not Counted! Uncounted Bins Will Be Returned To Queue!
Date Stamp for Zero Count Products
If a bin has a zero count, press F2 to continue to the next product. A date stamp shows on the report the next time you cycle count this product to let you know it was counted.
