Stock Receipt UPC Code Entry screen
The Stock Receipt UPC Code Entry screen appears if any of the products on a receiver require a UPC but do not have one. You must create the UPC before you can save the receiver.
Menu Path: Inventory > Stock Receipts > Stock Receipts Entry > F2-Save
Field and Function Key Descriptions
Field/Function Key
Description
Product Number
The product numbers from the receiver that do not have a UPC code set up as an alternate part number.
UPC Code
The UPC code to assign to the product. Enter N to let the system assign a number.
