Major Group Maintenance screen
The Major Group Maintenance screen allows you to combine a large number of products or product lines for sales analysis reports. You can use major groups to calculate product classifications within each warehouse.
Menu Path: Inventory > Maintenance > Misc Code Files > Major Group Codes
Field and Function Key Descriptions
Field/Function Key
Description
Major Group
The code that identifies a group of products.
Description
The description that represents the group of products.
Product Group
The code for the product group that the major group belongs to. Refer to Establishing Product Price Groups. You can use this to group products in Commission Matrix Maintenance.
F2-Save
Saves the major group maintenance information.
F4-Del
Deletes the major group code.
F5-G/L Accounts
Opens the Account Number Entry screen to allow you to enter overrides to general ledger accounts for products in this major group. These accounts are used if no G/L account is set up at the product line level.
