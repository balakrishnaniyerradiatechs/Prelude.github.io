HConfigurator Maintenance Screen
If the product is a gasket remnant product, the HConfigurator Maintenance screen allows you to set up components for material and labor to use on sales orders and quotes. When you enter this product on a sales order, the Gasket Entry screen appears so you can assign components and options.
If the product is a hose kickoff code product, you can use the HConfigurator Maintenance screen to set up questions to ask when the customer purchases a hose product. The product must also be set up as a kit parent.
This screen is only applicable if you are using the Gasket/Hose Subsystem.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Special Operation = GC or HK
Field and Function Key Descriptions
Field/Function Key
Description
Kickoff Code
The kickoff code that triggers the set of questions to ask when a customer purchases a hose. This is normally the same as the product.
Config Components
The configuration components (e.g., material, labor) that are options for the kickoff code product when the product is entered on sales orders and quotes.
Description
The description of the component.

F2-Save

Opens the Hconfigurator Decision Filter Maintenance screen to allow you to define the business rules for each hose class.
