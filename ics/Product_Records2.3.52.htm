Clear the Cross-Reference File
The cross-reference information that appears on the Universal Product Replacement Preview Report must be cleared before the next time you update products, so that the information does not show up again. If you did not choose to clear the cross-reference files during the update (refer to Update Replaced/Merged Products), you can manually clear the cross reference file.
To clear the cross-reference file created during the Universal Product Replacement/Merge Update:
1
Select Inventory > Maintenance > Universal Product Replacement & Merge > Clear Cross-Reference File. The following message displays:
The Following Procedure Will Clear Out The Universal Product Replacement Cross Reference Files. Do You Want To Continue?
2
Choose from the following:
•
No: Does not clear the cross-reference files.
•
Yes: Clears the cross-reference files created during the Universal Product Replacement/Merge Update.
Transaction Audit Inquiry
The Transaction Audit Inquiry displays a transaction audit by product number, including all transactions for a product in a warehouse. The code MRG indicates a merged product number. Inventory is updated for the original products added to the existing merged product number.
Usage Audit Inquiry
The Usage Audit Inquiry displays all usage information for a product number. The code UPR indicates a replaced product number. Usage is updated for the original products added to the existing replacing product number. If the product number is replaced, the usage is copied to the new product number.
