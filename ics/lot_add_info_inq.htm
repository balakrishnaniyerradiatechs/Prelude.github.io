Lot Additional Info Inquiry screen
The Lot Additional Info screen displays detailed lot information for a lot/license plate number.
Menu Path: Inventory > Inquiries > Miscellaneous Inquiries > Lot Inquiry > F7-Addit Info
Field and Function Key Descriptions
Field/Function Key
Description
Lot Number
The lot number.
Primary Bin #
The first bin location for the products in this lot.
Secondary Bin #
The secondary bin location for the products in this lot.
Lotpack
The package quantity of the lot if it is sold in package quantities.
Lot Pack Desc
The description of the package (e.g., case, carton, box).
Onhand
The quantity of the lot product still in on-hand inventory.
Average Cost
The average (received) cost of the products in this lot.
Comm Cost
The cost used for sales commission for the products in this lot.
Price
The selling price for the products in the lot.
Cert Of Comp
Indicates if this lot product has a certificate of compliance:
•
Y - A certificate of compliance was received with this lot product (if the Mfg Cert Of Compliance flag in Purchasing Parameters is set to Y to require a certificate of compliance when lot products are received).
•
N - The lot product does not have a certificate of compliance.
Chem & Physical
Indicates if this lot product has a chemical and physical test report:
•
Y - A chemical and physical test report was received with this lot product (if the Chem/PHys Test Report) flag in Purchasing Parameters is set to Y to require a chemical and physical test report when lot products are received).
•
N - The lot product does not have a chemical and physical test report.
Expire Date
The expiration date of the lot product, if the lot product was set up to require an expiration date in Product Maintenance in the Lot Information Screen.
Manufacturer Code
The manufacture code set up for the vendor in Vendor Maintenance. This code defaults into the Lot Number Information screen when the lot product is received.
Primary Vendor
The primary vendor for this lot product assigned in Product Warehouse Maintenance.
Vendor Lot Number
The lot number assigned by the vendor, if applicable.
Country Orig
The country where the product was manufactured.
OK
Closes the screen.
