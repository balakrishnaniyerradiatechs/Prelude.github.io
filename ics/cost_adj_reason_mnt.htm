Cost Adjustment Reason Maintenance screen
The Cost Adjustment Reason Maintenance screen allows you to set up codes to explain why a cost is adjusted. You can use these codes in Cost Adjustment Entry.
Menu Path: Inventory > Maintenance > Misc Code Files > Cost Adjustment Reason Codes.
Field and Function Key Descriptions
Field/Function Key
Description
Cost Adj Reason
The 1 or 2-character alphanumeric code to use when adjusting cost in Cost Adjustment Entry.
Description
The description of the reason code to display when you enter the cost adjustment reason code.
Adjustment Account
The general ledger account information to use to offset inventory.
Div
The division number for this account. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division).
Reg
The region number for this account, if applicable. A region is a subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
Loc
The location or warehouse for this account.
Dep
The department number for this account. A department is a subset of warehouse, typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
Ccr
The cost center number for this account. A cost center is a subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
Account#
The G/L account number to use as the offset when this cost adjustment code is used to adjust inventory.
Description
The description of the general ledger account.
F2-Save
Saves the cost adjustment reason code.
F4-Del
Deletes the cost adjustment reason code.
