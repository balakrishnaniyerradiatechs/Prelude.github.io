Product Kit Availability Inquiry
The Product Kit Availability Inquiry displays the quantity available of each of the components in a kit.
Kit Component Listing
To obtain a list of all kits and their components, select Inventory > Listings > Product Listings > Kit /Component Listing.
Kit Component Availability Inquiry
To view on-order, backorder, available, and delivery date information for kit components, select Inventory > Inventory Inquiries > Inventory Inquiry > Action Bar > Inquiries > Kit Component Availability to open the Kit Component Availability Inquiry screen
Viewing Kit Availability
To view availability for each of the components in a kit:
1
Choose Inventory > Inquiries > Availability Inquiries > Product Kit Availability Inquiry to open the Product Kit Availability Inquiry screen.
2
For Product, enter the kit parent part number.
3
Enter the Warehouse number.
Components available for the kit display.
Field & Function Key Descriptions
Field/Function Key
Description
Product
The kit parent part number and description
Warehouse
The warehouse containing the kits.
Available
The quantity of whole kits that can be assembled.
Component
The product numbers that make up the components of the kit.
Description
The description of the component product numbers.
Quantity
The quantity of each of the components required for the kit.
Available
The quantity available of each of the components included in the kit.
Extended Cost
The extended cost of the Quantity of each component required to make the kit.
OK
Closes the screen.
