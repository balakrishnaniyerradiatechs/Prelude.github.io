RX Group Maintenance screen
The RX Group Maintenance screen allows you set up a product as part of a prescription group. Refer to “Tracking Physician Prescriptions” in the Order Processing documentation.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > Special Operation =RX
Field and Function Key Descriptions
Field/Function Key
Description
RX Group
The RX Group to assign to the product for setting up prescriptions. Refer to ”RX Group Maintenance” in the Order Processing documentation.
F2-Save
Saves the RX information and returns to the Product Maintenance screen.
