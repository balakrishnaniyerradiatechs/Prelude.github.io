Physical Post-Entry Count Sheet Printing screen
The Physical Post-Entry Count Sheet Printing screen allows you to re-print the count sheets after the inventory counts are entered. You can compare the counts entered against the original count sheets to identify entry errors and to audit the physical counts. You are not required to print post-entry count sheets; but if you do, you must print them before you update inventory.
Menu Path: Inventory > Physical Inventory > Inventory Sheet Procedure > Physical Post-Entry Count Sheet Printing
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse where the inventory was counted.
Physical ListName
The list name from the count sheet.
#/Spaces Between Lines
The number of blank lines to print between each product. Enter zero for single spacing.
Show Onhand Quantity
Indicates if you want the saved on-hand quantity to display on the count sheets:
•
Y - Displays the saved on-hand quantities of each product.
•
N - Suppresses the saved on-hand quantities of each product.
Count Sheet Size
Indicates the paper size for the count sheet:
•
1 - Print the count sheets on 8 ½” X 11” (letter) paper.
•
2 - Print the count sheets on 11” x 14” (legal) paper.
