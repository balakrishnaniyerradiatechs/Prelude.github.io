Equipment Option Group Maintenance screen
The Equipment Option Group Maintenance screen allows you to group options that are available for a piece of equipment. Different types of equipment may have unique groups of options. For example, a lawn mower may have different optional features than a snowblower. You would want to group the options so that when you buy a lawn mower, you only have to choose from the features available for a lawn mower.
First you must identify the available options (refer to Setting up Equipment Option Codes), then you can assign the options to a group in Equipment Option Group Maintenance. You can then assign the option group to a product in the Option Group field in the Product Maintenance Additional Information Screen.
Menu Path: Inventory > Maintenance > Misc Code Files > Equipment Option Group Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Option Group
The option group which identifies the group of options available for a product.
Description
A description of the type of equipment that the option group applies to (tractor, lawn mower).
Option
The equipment option codes set up in the Equipment Option Maintenance screen, that are available for this type of equipment.
Description
The description of the equipment option code.
F2-Save
Saves the equipment option group.
F4-Del
Deletes the equipment option group.
