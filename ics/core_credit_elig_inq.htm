Core Credit Eligibility Inquiry
The Core Credit Eligibility Inquiry lists pending cores invoiced to a customer. You can determine if the customer is eligible to return the cores.
Viewing Eligible Core Credits
To display eligible core credits:
1
Select Inventory > Inquiries > Core Inquiries > Core Credit Eligibility Inquiry to view the Core Credit Eligibility Inquiry screen.
2
Enter the Customer.
3
Enter the Product.
Core credits for the customer/product appear.
Field and Function Key Descriptions
Field/Function Key
Description
Customer
The customer number.
Product
The product number.
Order#
The sales order number for the core.
Invoice#
The invoice number generated for the core.
Inv-Date
The date the sales order was invoiced.
Product
The exchange product.
Core Product
The core product number.
Description
The description of the core product.
Qty-Shp
The quantity of the core that shipped.
Qty-Credit
The quantity of the core that was returned for credit.
Qty-Avail
The quantity of the core that is available.
Net Price
The net price of the core on the sales order.
F2-Finish
Closes the screen.
F5-Review
Opens the Sales Order History Inquiry for the selected order.
F9-Goto Line
Allows you to enter a page number, part number or line number to go directly to.
