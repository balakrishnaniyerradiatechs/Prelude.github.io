Maintaining Core Exchange Types
Core exchange types define why a customer returned a core product. You can change the exchange type for a core bank record or a pending core record in Core Exchange Type Maintenance.
To change the exchange type on a core bank record:
1
Select Inventory > Maintenance > Misc Code Files > Core Exchange Type Maintenance to open the Core Exchange Type Maintenance screen.
2
Enter the Customer.
3
For Core Product, enter a core product number or leave blank to display all core records for the customer.
4
For Select By, choose from the following:
•
P - Only display pending core records.
•
C - Only display core bank records.
•
A - Display all pending core and bank core records.
The current core exchange types display for pending core and bank core records.
5
Press ENTER on the core document whose exchange type you want to to change.
6
For New Type, enter the new core exchange type. Choose from the following:
•
E - Exchange. The customer will exchange the core item for another product.
•
R - Repair and return. The customer wants you to repair and return the core product.
•
U - Unused. The core product was not used.
•
W - Warranty. The core exchange product is under warranty.
•
O - Other.
7
Press F2-Save to update the exchange codes.
