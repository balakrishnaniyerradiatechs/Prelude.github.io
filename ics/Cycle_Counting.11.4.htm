Warehouse Maintenance
To enable cycle counting for a warehouse:
1
In the Warehouse Maintenance screen, enter the Warehouse Number for which you want to set up cycle counting.
2
Select the Action Bar > Addl Opts > Bin Location Information to open the Bin Locator Options screen.
3
Set Multiple Bin Locations to Y to indicate that a product may exist in multiple bin locations and that the system tracks the quantity on-hand in each bin. This flag overrides the Multiple Bin Locations flag at the system level.
4
Under Maintenance Cycle Counting, for Cycle Count Days, enter the estimated number of days this warehouse will count bins. The default is 200 days.
5
For Update Bin Count 2nd Levl Queue, determine if the bin count updates all of the YTD Bin Counts:
•
Y - The bin count updates YTD Bin Counts for every bin on the list in the 2nd level cycle count entry.
•
N - The bin count does not update YTD Bin Counts for every bin on the list in the 2nd level cycle count entry. Only the bins with counts are updated.
6
For Update 1st Level Adj As Rejects?, determine if the product is marked as a reject in the first level of cycle counting. Choose from the following:
•
Y or Null - A product is marked as a reject in the first level queue if the counted quantity differs from the on-hand, but it is within the tolerances allowed in Cost Tolerance Maintenance. It does not go to the second level queue but inventory is adjusted to correct the on-hand inventory. The Bin Count Statistics Report displays these products in the Prd Rej column.
•
N - The product is not marked as a reject if the counted quantity is within the cost tolerance limits. Inventory is still adjusted but it does not show up in the Prd Rej column on the Bin Count Statistics Report.
7
For Default P/N in 2nd Level Entry?, determine if the product number is defaulted in the Product Number field when 2nd level counts are entered. Refer to Entering Second Level Cycle Counts. Choose from the following:
•
Y - A product number defaults in the Part Number field in the second level count entry.
•
N - A product number does not default in the second level of count entry. You must manually enter the part number.
8
Leave Update /ECCI Queue? blank unless you use emergency cycle counting with the Warehousing module.
9
For Exclude Non-Net Bins 2nd Level, determine if the non-nettable bins are included in the list of bins for the product for the second level count. A non-nettable product is part of on-hand inventory but is not available to sell until it is moved into a nettable bin. Choose from the following:
•
Y - The non-nettable bins are excluded from the list of bins for the product for second level counts.
•
N - The non-nettable bins are included in list of bins for the product for second level counts.
10
If you use RF devices, Scan First Level Counts indicates if products are scanned into the system for first level counts or if the total count is entered after all products are counted. Refer to Entering 1st Level RF Cycle Counts. Choose from the following:
•
Y - Products are scanned in the first count. If you find additional quantities of the product in other locations, you can keep scanning the additional quantities to add to the count.
•
N - The total count of the product is entered at one time. The count you enter overrides any count already entered.
11
For Cycle Count Auto Update, indicate if you want to automatically update products from a bin during the first level cycle count if the items were not initially physically counted. Choose from the following:
•
Y - Products not counted that have an on-hand balance will go to the second level count queue. If the product has a zero on-hand, it is updated as zero and not sent to the second level count queue.
•
N - If items in the bin are skipped during counting, you are warned and prompted to fill in counts for the remaining items.
12
For 2nd Level Bin Sorting, choose the sort order for the bins in the 2nd level cycle count:
•
S - Sorts the problem bin first in the 2nd level cycle queue bin list.
•
B - Sorts the 2nd level cycle queue with by ascending bin number.
13
Press F2-Save to save the bin information.
