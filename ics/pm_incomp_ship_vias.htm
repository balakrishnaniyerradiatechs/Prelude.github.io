Incompatible Ship Vias Screen
The Incompatible Ship Vias screen allows you to enter shipping methods that are not compatible for a product (e.g., weight or size restrictions on overnight shipments).
Order Processing Parameters > F5-Add Info > Administrative Order Hold = Y
If the Administrative Order Hold Parameters screen (refer to ”Defining Options for Administrative Order Hold” in the Order Processing documentation) includes the I option, incompatible ship vias can be set up for a product. If you enter a sales order for this product using one of these ship vias, the order will go on hold. This applies whether the incompatible ship via is entered at the header or line item level.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > Action Bar > Incomp Ship Via
Field & Function Key Descriptions
Field/Function Key
Description
Via
The ship via codes not acceptable for shipping this product.
Description
The description of the ship via code.
Class
The class of the associated incompatible ship via (e.g., for UPS, the class might be next day or second day delivery).
Not all ship vias have associated classes.
F2-Save
Saves the incompatible ship via information.
