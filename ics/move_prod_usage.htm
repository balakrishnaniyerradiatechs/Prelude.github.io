Move Product Usage screen
The Move Product Usage screen allows you to move usage from one product to another. You can determine whether you want to keep or delete the usage of the original product number. You may want to use this process if you start buying a regular stock item from a new vendor (but different part number) or if usage is incorrectly recorded for a product.
Menu Path: Inventory > Processes > Move Product Usage
Field and Function Key Descriptions
Field/Function Key
Description
Product
The product to move usage from (the old part number).
To Product
The product to receive the usage information.
Description
The description of the To Product.
Description Two
The second line of description of the To Product.
(A) Add Usage or (C) Add Usage and Clear
Indicates if usage remains for the original product:
•
A - Adds usage to the To Product. The usage also remains in the original product number.
•
C - Adds usage to the To Product and clears usage from the original product number.
F2-Save
Completes the product usage move.
