Commodity Code Maintenance screen
The Commodity Code Maintenance screen allows you to enter commodity codes to group products to calculate duty. You can assign these codes to products in Product Maintenance.
Menu Path: Inventory > Maintenance > Product Control Files > Commodity Codes
Field and Function Key Descriptions
Field/Function Key
Description
Commodity Code
The code to identify a group of products with the same duty rate.
Duty Rate
The percentage used to calculate duty when purchasing or selling an item.
Description
The description of the commodity code.
F2-Save
Saves the commodity code information.
F4-Del
Deletes the commodity code information.
