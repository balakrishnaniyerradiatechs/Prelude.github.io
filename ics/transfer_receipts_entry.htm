Transfer Receipts Entry screen
The Transfer Receipts Entry screen allows you to receive a stock transfer shipped from another warehouse to update inventory in the receiving warehouse.
Menu Path: Inventory > Stock Movements > Transfer Receipts Entry
Field and Function Key Descriptions
Field/Function Key
Description
Transfer Release#
The release number from the transfer packing list.
Receipt Date
The date the material arrived in the warehouse.
From
The branch number and description for the shipping warehouse.
To
The branch number and description for the receiving warehouse.
Product
The product number shipped.
N
An asterisk (*) indicates that notes exist in the header.
Description
The product description.
Emg
Indicates if the transfer is an emergency. Used with the Warehousing module.
Linkage
Indicates it the product is linked to another order. The prefix indicates the document type:
•
S - Sales order.
•
W - Work order.
•
T - Transfer.
Qty-Opn
The quantity open (not yet received) on the transfer.
U/M
The unit of measure of the product as entered on the transfer.
Qty-Shp
The quantity shipped according to the shipping warehouse.
Qty-Rec
The quantity received according to the receiving warehouse.
F2-Save
Updates inventory with the received quantities.
F4-Del
Deletes the transfer receiver.
F5-Header
Opens the Stock Transfer Header screen to allow you to view or edit the shipping information.
F7-Wty Tag
Not applicable when receiving a transfer.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
Action Bar
Examine
Displays cost and supply information for the highlighted item.
Review
Opens the Transfer Review screen to allow you to view picked, backordered, and shipped information for the transfer.
Notes
Allows you to enter or view header or line item notes for the transfer.
