Product Labels Information Screen
The Product Labels Information screen allows you to enter label printing information for a product. This information works with the Price Label Format Maintenance screen in the Order Processing module to determine the number of labels to print.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Maintenance > F5-Add Info > F7-Labels
Field & Function Key Descriptions
Field/Function Key
Description
Print Price Label
Determines if customer price labels print for this item when sold:
•
Y - Price labels print.
•
N - Price labels do not print.
If a product is set up to print price labels when you sell it, labels only print for customers with Print Price Label set to Y (Customer Maintenance > Action Bar > Add’l Opt > Label Information).
Price Sticker Mult
The multiplier to calculate the number of price labels to print for the product. When the product is sold, this number is multiplied by the quantity to determine the number of labels. If the customer orders 10 and the Price Sticker Mult is 10, 100 labels print.
Print Cust/Prod Lbl
Determines if product labels print when the product is sold.
•
Print product labels when sold.
•
Do not print product labels when sold.
Case Pack
The quantity of the product packed in a shipping case.
Pallet Pack
The quantity of the product contained in a pallet.
Qty/20’ Container
The quantity of the product included in a 20-foot container.
Qty/40’Container
The quantity of the product included in a 40-foot container.
Product Label Size
The default label size if you print labels when the product is received:
•
L - Large.
•
S - Small.
Qty/Product Label
The default number of labels to print when the product is received.
Print Barcode Label
Determines if the price labels print barcodes when this item is received:
•
Y - Barcode the product label regardless of the vendor barcode setting.
•
N - Do not barcode the product label regardless of the vendor barcode setting.
•
V - Barcodes print based on the vendor barcode setting.
•
R - Barcodes print only on reports or documents (requires a custom forms modification).
Refer to Printing Barcode Labels when Receiving Stock.
This flag setting overrides the flag setting in Vendor Maintenance, but is overridden in Product Warehouse Maintenance and Product Vendor Maintenance.
F2-Save
Saves the product labels information.
