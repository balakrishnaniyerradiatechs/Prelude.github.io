Crib Bin Customer/Ship-To Maintenance screen
If you manage crib bins by bin location, the Crib Bin Customer/Ship-To Maintenance screen allows you to establish a crib bin as a crib warehouse within a customer facility. Inventory is managed at the bin location level. This bin number must be entered in the Bin Location field in the Crib to Warehouse Cross Reference Maintenance screen (Inventory > Maintenance > Customer Control Files > Customer Crib Maintenance > Crib to Warehouse Maintenance). Refer to “Using Crib Warehouses” for more information on Cribs.
Crib Type in Warehouse Maintenance > Action Bar > Addl Opts > Crib Information must be set to B to manage cribs by bin.
Menu Path: Inventory > Maintenance > Bin Files > Bin Locations > Bin Type = C
Field and Function Key Descriptions
Field/Function Key
Description
Crib Customer #
The customer that the represents the location of this bin.
Crib Ship-To #
The customer’s ship-to account number. If the Crib Customer # is a master account, a ship-to number is not required.
Crib Replenish Whse
The crib bin replenishment warehouse to use with this bin location. This warehouse is used in Nightly Processing for crib bin replenishments.
F2-Save
Saves the crib customer information.
F4-Del
Deletes the crib customer.
