License Plate Combine Entry screen
The License Plate Combine Entry screen allows you to combine multiple license plates into one license plate. A new license plate number is created with the combined quantities and the original license plate numbers are changed to zero.
Menu Path: Inventory > Inventory Adjustments > License Plate > License Plate Combine Entry
Field and Function Key Descriptions
Field/Function Key
Description
License Plate
The license plate numbers to combine.
Product
The product number associated with the license plate.
Lot
The lot (license plate) number.
Quantity
The quantity on the license plate.
F2-Save
Combines the listed license plates into a new license plate and assigns a new number.
