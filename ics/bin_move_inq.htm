Bin Move Inquiry
The Bin Move Inquiry allows displays information on a bin move. Refer to Moving Products to New Bins for information on how to change bins for a product.
Viewing Bin Moves
To view information on a bin move:
1
Select Inventory > Inquiries > Miscellaneous Inquiries > Bin Movement Inquiry to open the Bin Move Inquiry screen.
2
Enter the Move Number.
The information for the moved products displays.
Field & Function Key Descriptions
Field/Function Key
Description
Move Number
The move number assigned by the system in Bin Individual Move Entry or in Bin Bulk Move Entry. Refer to Moving Products to New Bins.
Warehouse
The warehouse number where the product was moved.
User Id
The user ID of the person who entered the bin move.
Date
The date the product moved.
Beg Time
The beginning time of the bin move (if using RF devices).
End Time
The ending time of the bin move entry (if using RF devices).
Status
The status of the bin move:
•
In process.
•
Completed.
•
Suspended.
Product Number
The product that moved.
Description
The product description.
From Bin
The bin the product was originally in.
Quantity
The quantity of the product that moved.
To Bin
The new bin for the product.
UOM Stk
The stocking unit of measure for the product.
Lot Number
The lot number of the product, if applicable.
Cls
The inventory class for the item. The inventory class is based on how many dollars of the item move through inventory in a year. Refer to Classifying Products.
End Time
The ending time of the bin move (if using RF devices).
Misc
The warranty tag number if the bin move is associated with a vendor claim.
F2-Finish
Closes the inquiry screen.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
