Bin Onhand Inquiry
The Bin Onhand Inquiry allows you to view the products and on-hand quantities contained in a bin.
Viewing Bin Onhand Quantities
To view the contents of a bin:
1
Select Inventory > Inquiries > Miscellaneous Inquiries > Bin Onhand Inquiry to open the Bin Onhand Inquiry screen.
2
Enter the Warehouse number.
3
Enter the Bin for which you want to view the contents.
The on-hand and committed quantities for the products in the bin display.
Field & Function Key Descriptions
Field/Function Key
Description
Warehouse
The number for the warehouse where the bin exists.
Bin
The bin number.
Product
The products that exist in the bin.
Description
The description for each product in the bin.
Bin OH
The quantity on-hand in the bin. Includes committed quantities.
Bin Cmt
The quantities committed to sales orders, work orders, and transfers. Used with the Warehousing module.
Bin Avl
The quantity of the product available to sell. Used with the Warehousing module.
Cls
The inventory class for the item. The inventory class is based on how many dollars of the item move through inventory in a year.
Refer to Classifying Products for information on inventory classes.
Lot Numbers
The lot numbers for products in the bin, if applicable.
F5-Orders
Opens the Sales Order Commitments screen for the highlighted bin to allow you to view detail information on sales orders containing the product. Used with the Warehousing module.
F6-Transfers
Opens the Stock Transfer Commitments screen for the highlighted bin to allow you to view detail information on transfers containing the product. Used with the Warehousing module.
F7-Work Orders
Opens the Work Order Commitments screen for the highlighted bin to allow you to view detail information on work orders containing the product. Used with the Warehousing module.
F8-Repl
Displays bin replenishment information for the highlighted bin. Used with the Warehousing module.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
