Transfer Receipts Label Reprint screen
The Transfer Receipts Label Reprint screen allows you to reprint a receipt label for a received transfer.
Menu Path: Inventory > Processes > Transfer Receipts Label Reprint
Field and Function Key Descriptions
Field/Function Key
Description
Transfer History Number
The received transfer number for which you want to reprint a label.
Transfer History Line#
The line number for the product on the receiver for which you want to reprint labels.
Number Of Labels
The number of labels you want to print for the product.
F2-Save
Opens the Print Manager screen to allow you to print the labels.
