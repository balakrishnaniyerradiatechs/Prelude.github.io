User Identification Maintenance
Each employee that cycle counts must have their user ID set up to enable cycle counting. Employees who use RF devices to cycle count must have their user ID set to access only one Location so that the count information is posted to the correct general ledger account.
To set up a user for cycle counting:
1
Select Systems > Maintenance > User ID Maintenance to open the User Identification Maintenance screen.
2
Enter the User Id Code of the employee that is a cycle counter.
3
For Location, determine if the counter has access all warehouse locations or just one or more designated warehouses. Choose from the following:
•
A - The counter has access to all locations.
•
S - Opens the Warehouse Selection screen to allow you to enter the warehouses that this employee has access to. This field is validated in the cycle count entry programs so that this Counter ID can only be entered for one of these warehouses.
4
Select F5-Administrative to open the Administrative Settings screen.
5
Set Cycle Counter to Y to allow the employee to enter cycle counts.
6
Press F2-Save to return to the User Identification Maintenance screen.
7
Press F2-Save to save the cycle count information.
