Tariff Commodity Code Maintenance screen
The Tariff Commodity Code Maintenance screen allows you to set up Tariff Commodity Codes for harmonized tariffs. The U.S. Customs Service publishes a Harmonized Tariff Rate Schedule to regulate duties and taxes on imports and to track import and export statistics. Tariff commodity codes define favored nation rates, non-favored nation rates, and agreement code rates to work with harmonized tariff calculations. You can calculate a rate either as a percentage of the value of a product or as an amount per unit or weight.
Prelude uses tariff codes to incorporate harmonized tariffs into duty calculations for material received from import vendors. When you purchase a product with a harmonizing code, a tariff charge is calculated for the item. The Harmonizing Unit in the Additional Product Maintenance Information Screen, checks the tariff commodity code field to determine if the product is subject to a tariff charge.
Refer to Tariff Commodity Codes for more information.
Menu Path: Inventory > Maintenance > Product Control Files > Tariff Commodity Codes
Field and Function Key Descriptions
Field/Function Key
Description
Tariff Commodity Code
The code published by the U.S. Harmonized Tariff rate schedule.
Description
The description that identifies the products associated with the code.
Fav Nation Perc Rate
The percentage rate to calculate the harmonized tariff for a country with the Most Favored Nation flag set to Y (Country Codes Maintenance in Order Processing Controls).
Fav Nation Per Unit Rate
The rate per unit to calculate the harmonized tariff for a country with the Most Favored Nation flag set to Y.
Nonfave Nation Perc Rate
The percentage rate to calculate the harmonized tariff for a country with the Most Favored Nation flag set to N (Country Codes Maintenance in Order Processing Controls). Agreement Codes are not checked.
Nonfav Nation Per Unit Rate
The rate per unit to calculate the harmonized tariff for a country with the Most Favored Nation flag set to N. Agreement Codes are not checked.
Agreement Codes
The programs under which special tariff treatments are provided as defined in the Harmonized Tariff Schedule of the United States. You can assign multiple agreement codes to a harmonized tariff code.
For example, if the country is a favored nation and contains an agreement code that matches one of these agreement codes, then the Agree% and the Agree Rate are used to calculate the tariff.
Agree%
The percentage rate for the agreement code. This percentage is used to calculate the tariff if the country is a most favored nation and has the same agreement code.
Agree Rate
The unit rate for the agreement code. This rate is used to calculate the tariff if the country is a most favored nation and has the same agreement code.
F2-Save
Saves the tariff commodity code.
F4-Del
Deletes the commodity code.
F5-Add’l Description
Opens the General Notes Entry screen to allow you to enter any additional notes regarding the tariff code.
