Lot Maintenance Screen
The Lot Maintenance screen allows you to enter and edit lot information for a product. This screen is only accessible if Lot Number Flag in the Product Maintenance Additional Information Screen is set to Y or S.
Menu Path: Inventory > Maintenance > Product Maintenance > Product Warehouse Maintenance > F9-Lot
Field & Function Key Descriptions
Field/Function Key
Description
Lot Number
The lot number assigned to the product.
Description
The description of the lot number.
Lot Description 2
The second line of description for the lot number.
Primary Bin Number
The bin where the lot item is stored.
Second Bin Number
The secondary bin for the lot.
On Hand Balance
The quantity of the item in the warehouse.
Average Cost
The average cost of the item.
Cert. of Comp
Indicates if a manufacturer certificate of compliance is required when you purchase the product (informational only):
•
Y - A certificate of compliance is required.
•
N - A certificate of compliance is not required.
Mfg Cert Of Compliance in Purchase Parameters (Option Parameters) must be set to Y to require a certificate of compliance when the product is entered on a purchase order.
Chem & Physical
Indicates if a chemical and physical test is required when you purchase the product (informational only):
•
Y - Chemical and physical tests are required.
•
N - Chemical and physical tests are not required.
Chem/Phys Test Reports in Purchase Parameters (Option Parameters) must be set to Y to require chemical and physical tests when the product is entered on a purchase order.
Expire Date
The date the shelf life for the lot expires.
Manufacturer Code
The manufacturer code for the product, if applicable. These codes are established in the Manufacturer Maintenance screen. Refer to Manufacturer Abbreviations.
Primary Vendor
The primary vendor assigned to this lot item.
Lot Vendor Number
The lot number assigned to the product from the vendor if it is different than the lot number assigned to the product when it was received.
County Of Origin
The country where the product is manufactured.
F2-Save
Saves the lot number information.
F5-Comments
Allows you to enter comments regarding the lot.
