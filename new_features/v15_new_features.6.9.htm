Bin Replenishment Min/Max Update
The Bin Replenishment Min/Max Update is a new process that was added to calculate bin replenishment minimums and maximums. Select all product whse. stat records for that warehouse. If a product/bin is set with. If a product/bin is set with frozen controls, check for a thaw date. If a thaw date is less than or equal to the system date, clear the thaw date, set the frozen control flag to N, and continue with the Min/Max recalculation. If the product/bin is set with frozen controls and no thaw date that exists or the thaw date has not yet been reached, skip this product/bin and continue to the next product/bin evaluation. The monthly usage from the product purchasing parameters screen for that warehouse is multiplied by the minimum and maximum percentages established in Warehouse Maintenance. If an average monthly usage override exists for the product, use the override value in the calculation.
Maintenance
Product Warehouse Maintenance
Generate/Create New Warehouse now has the capability to copy all existing warehouse records from one warehouse to all warehouses established in Warehouse Maintenance.
Segment Counter Range Maintenance
A new maintenance option was added to the Segment Processing Maintenance menu. This new routine allows the range counter for segment generations that are processed during Sales Order Bin Allocations to be maintained. This Segment Counter Range Maintenance process has the ability to set up the counter for the segment counter range to be used for each warehouse.
Radio Frequency Menu
Sales Order Entry
The Radio Frequency menu was enhanced with the ability to enter Sales Orders utilizing the Radio Frequency devices. This option allows Sales Orders to be generated from one location by scanning the items at another. A standard sales order will be created behind the scenes at the time the RF Sales Order Entry screen is being updated utilizing the Direct Order Entry conversion process. 040100I
