General Ledger
Rebuild CA Description Prefixes
This is a new process that was added to rebuild the Chart of Account prefixes. It is recommended that this process is completed once chart of accounts are created using the Chart of Account Duplication process or manually created or deleted in Chart of Account Maintenance. Once this process was completed, the Chart of Accounts Balance Inquiry will display the G/L prefixes of a account number in Division/ Region/ Location/ Department/ Cost Center order.
Chart of Accounts Balance Inquiry
This is a new inquiry that was added to ADS to display the available balances of the prefixes of a G/L account. You can enter a G/L account number as well as the period ending date. This inquiry displays the account balance information as well as the Division, Location and Department numbers. Note* For this inquiry to display accurate balance information, you must run Rebuild CA Description Prefixes (/REBUILD.CADF.PREFIX)
