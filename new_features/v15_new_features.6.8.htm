New Marketing Features
Contact Diary Codes
Contact Diary Codes Maintenance was added to the Marketing Maintenance menu in the Contact Diary System. This maintenance option is to provide the setup of Diary codes to display the Contact type on the Contact Diary Report / Inquiry. This new option allows you to enter the contact code and description to be used in Contact Diary Entry. Contact Diary Entry has also added a new Diary Code flag to enter the desired contact diary code to display. The Marketing Activity flag that was added to the File Update Parameters as well. This flag determines if marketing activities entered are updated to the contact detail file. The Contact Diary Report's Sort Options flag has also added 3 new sort options to be selected. The user now can enter E for entry date, U for user id, or S for serial number. The F-6 key in the Contact Diary Inquiry now displays the contact inquiry for the contact entered on the Marketing Activity screen.
