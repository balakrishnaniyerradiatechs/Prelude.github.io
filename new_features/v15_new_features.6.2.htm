New Purchasing Features
Purchase Order Entry - Header Level
For a type 02 Direct purchase order, the Drop Ship Order# field was enhanced to accept a non-direct type sales order that contains a line item with a D supply code. Previously this field would only accept a direct sales order. Now the system will search the sales order for any line items with a D supply code with the same primary vendor as the vendor on the purchase order and add those items to the purchase order. A warning will be displayed if no line items with a D supply code were found and you will be returned to the Drop Ship Order# field.
Daily Processing
A new Stock Transfers flag was added to the Daily Processing Routine to determine if stock transfers are to be reviewed during the daily processing routine.
Vendor Maintenance
The Vendor Maintenance screen was updated with a new F-9 Selection option. This new feature enables the user to review the last list in memory of a vendor search. Once a user completes the search for a vendor using the F3 key, the F-9 option will allow you to retrieve the same list without having to perform the entire search
Inquiries
Purchase Order Inquiry/ Purchase Order History Inquiry
These inquiries now include Sort and Filter options. These functions allow the user to limit the inquiries displayed on the screen to sales orders, based on the criteria selected in the sort and filter fields. You also have the ability to set a Default sort for the inquiry screens in Default Parameters Maintenance (/DSS).
•
The Sort field determines how the purchase order records are to be sorted. You can sort by P/O Number, P/O Date, Vendor Order Number, or by Order Type.
•
The Filter field determines if filters are to be used to limit the sales orders displayed on the screen. If this field is set at N, no filters will be used, and all sales orders for that customer number will be displayed.
Purchase Order Log Inquiry
Two new inquiry codes were added to the Purchase Order Log Inquiry screen.
•
ADD - Line Item added Reference- Product#
•
CHG- Header/Line Item changed
The Purchase Order Log Inquiry has added a new drill down feature utilizing the F-5 Review option in the inquiry screen. This allows the user to view each transaction that has taken place during the purchasing process. In addition, a Purchase Order Log Notes screen can also be displayed to show additional information regarding any transactions displaying the CHG code. For the detailed information on the CHG transaction updates to be viewed, the fields that are to be tracked, must be established in the Purchase Order Change Log Setup and the Purchase Order Line Change Log Setup in Log Inquiry Setup.
