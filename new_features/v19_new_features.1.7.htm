Miscellaneous
Marketing
Creating Universal Link Numbers
Use Starting Universal Link Assignment in Customer Contacts to set up a starting number for creating universal link numbers.
Using Updated Fields
The following fields have changed in the Marketing module:
•
Prospect Number to Prospect.
•
Phone Number to Phone.
•
Fax Number to Fax.
Finding Addresses for Broadcast Emails
The F3 search for To in Marketing > Broadcast/Export Functions > Broadcast Email uses the GRIM interface.
Job Accounting Reports
The following headings have changed on the AIA Invoice Report:
•
The product description 1 was moved to print below the product number.
•
The product description 2 was added to the report to print next to product description 1.
EDI
Updating Customers/Prospects
You can now convert customer, prospect contacts, and prospects into Prelude through the following menu: E-Commerce > Electronic Entry > Entry Programs.
Updating Transfer Receipts
Convert Transfer Receipts (E-Commerce > Electronic Entry > Entry Programs) allows you to convert transfer receipts updated by DOE convert files.
Service
Service Order Parts Report (new)
Displays information for all parts invoiced from service orders.
Service Order History Summary Report (new)
The Service Order History Summary report displays information on invoiced service orders for an invoiced date range, customer, and engineer.
Systems Listing
Lists all information for each systems number in Systems Maintenance.
Sales Analysis
Sales Rep Break Added
The Forecast & GP By Sales Rep/Customer and Forecast By Sales Rep/Customer reports page break by sales rep.
Forecast & GP By Customer/Rep/Major Group
List sales forecast and gross profit information by customer, representative, or major group.
