Transfer Rental Return
to correctly update the G/L posting for master asset items transferred to another location using the Transfer Entry or Transfer Rental Return process, the logic for the G/L posting was modified. This was changed to incorporate the accumulated depreciation (LTD depreciation) of the sub asset. Previously, if a master asset were moved to another location, the depreciation value would remain in the prior location and not post the correct G/L information. This feature works with the new Master Asset Entry and Sub Asset Disp options.
Rollup Billing
Rental Order Entry has a new Rollup Billing flag in the Rental Information screen. If this flag is set to Y, the pricing is rolled up at invoicing. If this flag is N, the pricing is not rolled up. Use this feature only with non-serialized items. Note: to use this new feature, the Rental Rate Method flag in the Rental Parameters screen must be set to D (Daily).
Reports
Utilization Summary Report
The Utilization Summary Report is a new report that was added to the Rental module. This report is identical to the Utilization Report with the exception of the format. The Utilization Summary Report will display a summarized view of the same information as is displayed in detail on the Utilization Report. However, both reports display rental information that is calculated in the exact same way. This displays utilization for Dollars or Time pertaining to Rental Equipment. When this report is run by dollars, the percent is calculated by the revenue dollars divided by the potential revenue in Rental Equipment Maintenance. Revenue comes from the system calculated Rental Incom in Rental Equipment Maintenance. When the utilization report is generated by time, the utilization percent is calculated by the number of days rented divided by the number of days available. Days available are the number of days the item has available for rent in the month, year, or lifetime of the product. This is the number of total days available and includes days you are not open for business and days the item is rented out. Days Rented comes from the system calculated Rental Days fields in Rental Equipment Maintenance.
Rental Billed Thru Date
to control how far out the pre-billed rental order will be scheduled for, a new Rental Billed Thru Date was created in Rental Start Info in the Release Confirmation and Invoice Printing screens. You can enter a date to determine the period. However, if this field is blank, the standard pre-bill date is calculated per the unit of measure (hourly, daily, weekly, monthly) for the rental product.
Asset Maintenance
Master Asset Entry
The Master Asset Entry process allows you to create an asset and link it to a master asset. Therefore, the cost of the linked asset is rolled up to the master asset. Once a master asset is transferred through Transfer Entry or Transfer Rental Return, the accumulated depreciation (LTD depreciation) of the sub asset is included in the depreciation amount. Once Inventory Type Conversion Entry is run and the asset is created, the Master Asset is not automatically created. You must manually link the sub assets by using the Master Asset Entry for each EIN.
Sub Asset Disp
This new Sub Asset Disp feature was created with the new Master Asset Entry option. This process will display the asset records that were processed in the Master Asset Entry process and are now sub assets linked to the master asset.
