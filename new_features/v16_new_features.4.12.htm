Rental
Rental Order Entry
•
Due to the new Pct Off flag in Damage Waiver Code Maintenance screen, Rental Order Entry was updated to incorporate this new feature. If a damage waiver charge is added to the order along with the liability insurance charge, the system will use the percentage entered in the Pct Off field to reduce the percentage amount. This will calculate the damage waiver and liability insurance charges added to the rental order. If a damage waiver is in fact added to the order, a dialog box will be displayed notifying the user the amount of the charge.
•
The Rental Parameters was updated to include a new Liability Ins Charge flag. In addition, a new Pct Off flag was added to the Damage Waiver Code Maintenance screen as well as a new Liability Ins Charge P/N flag in Product Maintenance. To incorporate these new features, Rental Order Entry was updated. Once the rental return is completed and the rental order’s Liability Ins Charge flag is set to Y, the system will total up the rental retail value of all rental products and compare that total to the physical damage coverage amount established for the customer. If the total exceeds the customer’s coverage or if the physical damage coverage has expired, the system will insert a service charge line item for the liability insurance charge. The liability insurance charge amount is calculated as a percentage of the rental dollars. The percentage is determined from the percentage established in Damage Waiver Code Maintenance. If a damage waiver charge is added to the order along with the liability insurance charge, the system will use the percentage entered in the Pct Off field to reduce the percentage amount to calculate the damage waiver and liability insurance charges added to the rental order. A dialog box will be displayed, notifying the user of the amount of the charge.
•
The Rental Order Entry screen was updated for Version 16. Two new Warning messages were added for rental items entered on a rental order that is subject to rental damage waiver charges and/or rental liability charges. If a product is entered on a Rental Order that is required to have Damage Waiver charges applied, the user will be prompted with a Damage and Theft Required. Do You Wish To Continue? message. At that point, the user has the option of selecting Yes with the awareness that a damage waiver charge will be applied to this order. If the user selects No, the user will then be brought back to the order line to remove the rental item from the order. A new Warning message has also been added for rental orders requiring rental liability insurance. If a Rental Order contains a product that requires a Liability Charge, the message Liability Additional Insured Required. Do You Wish To Continue? will be displayed. This Y/N prompt works identical to the Damage Waiver prompt. If a product entered on the order requires both damage waiver and liability charges, both warning messages will be displayed.
Maintenance
Damage Waiver Code Maintenance> Pct Off
A new Pct Off flag was added to the Rental Maintenance screen. This new field allows you to enter the percentage amount that is to be deducted when damage waiver and liability insurance charges are added to a rental order. For example, if a damage waiver product was set at 10% and the Pct Off field was established at 2%, the system would calculate the damage waiver charge on the rental order as 8%, if damage waiver and liability insurance charges were both added to the order. If only damage waiver charges are added to the rental order, the system would use the Percent field (10%) to calculate the damage waiver charges.
Rental Number Change Maintenance
A new maintenance routine was added to the Rental module. The Rental Number Change Maintenance feature allows users to change rental id numbers on products. Changes may be required due to receiving a rental item under the incorrect rental id number. Therefore this option allows the modifications to be made.
Processes
Periodic Billing
The Periodic Billing process was modified to evaluate each rental order to determine if Liability Insurance charges should be added to orders. This billing process was updated to use the new Liability Ins Charge feature in Rental Parameters.
F5-Remove From Billing
A new F5-Remove From Billing function key was added to the Periodic Billing Entry screen in the Rental module. This new key enables the user to remove the entire order from the billing process. Previously the user would be required to manually select N for each item to ally not include these line items in the billing process. Utilizing this new feature will automatically update each line item on the order with a N as the Bill flag setting as well as delete any additional charges that may were applied, such as damage waivers, environmental charges and liability charges. Once this was performed, the Order Log Inquiry will then be updated with a LDE code for each additional charge that was deleted when using the F5-Remove From Billing process. In addition, a PBR code will be displayed indicating that the F5-Remove From Billing function was performed for this rental order.
Equipment Subsystem
The Equipment Maintenance screen was updated to reflect the new rental id number when entering the original equipment id and asset number in Equipment Maintenance. This maintenance routine was updated due to the new Rental Number Change Maintenance option, which allows rental id numbers to be changed for existing rental id products.
Miscellaneous Features
The Book# field in Product Maintenance was updated with a new functionality for Version 16. Previously, the Book Number field would only allow one book number to be entered for this rental item. In version 16, this was updated to allocate multiple book numbers to a product. The Book # is established in Product Maintenance> F10-Action> Addtl Opts> Rental Information> F7-Asset Info.
