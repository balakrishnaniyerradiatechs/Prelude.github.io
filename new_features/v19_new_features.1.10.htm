Rental
Rental Order Entry
Maintaining Rental Equipment with Shop/Repair Orders
When you enter Y in Maintenance Needed for a returned Rental item, you can automatically create a Shop/Repair order. A message appears to create a Shop/Repair order for each rental item to be maintained.
Choosing Rental Type
Enter the one of the following rental types for Rental Type:
•
C - consignment.
•
D - demo.
•
L - long term.
•
O - loaner.
•
P - rental purchase.
•
R - reservation.
•
S - short term.
•
W - warranty loaner.
Prelude automatically checks the Rental Type set up in Rental Parameters. If you do not choose a type, Prelude defaults to O (On-rent) rental type.
Rental Parameters
Selling Rental Items
Specify if you want to include rental items on sales orders and quotes with the Allow Rental Items in Soe field in Rental Parameters. Choose one of the following:
•
Y - allow rental items on sales orders or quotes.
•
N - do not allow rental items on sales orders and quotes.
Calculating Credit Lines
Set the Auto Credit Calc Method in Rental Parameters to specify how Prelude should automatically calculate the credit lines for early return rental orders. Choose one of the following to allow credit for early returns:
•
D - use the daily rate.
•
B - use the best rate.
Entering Rental Exchange Repair Information
When you remove an item in Rental Exchange Entry and enter Y in the M field, you can enter rental exchange repair information to create a shop/repair order.
Attaching Equipment
When you attach equipment to rental items, Prelude changes the equipment status to E for Equipment Attachment. This new feature affects the following programs:
•
Rental > Equipment Subsystem > Equipment Attachment Entry.
•
Accounts Payable > Manual Invoice Entry.
Maintaining Vendor Claims
When equipment is on a vendor claim, Prelude changes the equipment status to V for Vendor Claim. After you update the vendor claim, Prelude changes the status to R for Vendor Return.
Calculating Damage Charges
In Damage Waiver Code Maintenance, enter a value in the Minimum Amount field to update the damage waiver or liability insurance code for the line. Use this field only when the total amount for the line item multiplied by the percent established for the damage waiver or liability insurance code does not exceed the minimum amount entered.
Reports
Equipment Sales History Report
This report includes information on equipment sold for the date range entered.
ROI Equipment Inventory Report
This report includes return on investment information for rental equipment.
Equipment Sold Inquiry
View return on investment information for rental equipment.
Equipment Model Inquiry
This inquiry includes equipment information for a product. The Stat column shows In Transit if an item is in transit from one location to another. In Default Parameters Maintenance, you can define the default sort for the Equipment Model Inquiry. Choose one of the following:
•
H - Hours
•
O for Options.
•
P for Product.
•
S for Price.
•
Y for Year.
Choose Inventory > Inquiries > Miscellaneous > Equipment Model Inquiry to view the data.
