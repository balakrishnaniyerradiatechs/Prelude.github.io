Reports
Transfer Picking Report
The Transfer Picking Report has recently been modified to now include the Transfer Number on the report. The transfer numbers will now be displayed for each item under the HM & UM columns of the report.
Work Order Picking Report
The Work Order Picking Report was updated to display assembly instructions in the heading section of the report. In addition, page breaks have also been included to print each work order on a separate page. Assembly Instructions will be printed as well as using Page Breaks for the Work Order Picking Report regardless if a Work Order Number was selected for the report.
Radio Frequency
Bin Onhand Inquiry
The Warehouse # field has recently been modified to automatically default to the Warehouse location that is assigned to this User Id. If this user is not assigned to a warehouse, the Warehouse # field will remain blank and allow you to enter the warehouse number. However, if a warehouse is established, and the default warehouse is displayed, you can override the warehouse number. Previously, you must enter a warehouse number.
Recv & Putaway> Putaway Ent
A new N (Manual Primary) Putaway Method was added to the list of available Putaway Methods. If the N option is selected, the Primary Bin allocated for this product will be defaulted into the TO Bin (TBn) field for this putaway process. You can change the default bin location if needed. Although, if the default bin is overridden with a pickable bin, the message The Bin You Are Moving The Product To Is A Pick Bin But Is Not The Primary Bin For This Product. Do You Wish To Continue? will be displayed. You can still change the bin.
