New Sales Analysis Features
G/P Based On Cost (A,C,R)
This field was changed for Version 16. Previously this field was labeled G/P Based on Acct/Comm Cost. This field has changed to the G/P Based On Cost (A,C,R) field, as well as added a new R for Replacement Cost selection. This allows the user to select if the report will be generated to display gross profit by Accounting Cost, Commission Cost or Replacement Cost.
By Sales Rep1> By Sales Rep / Major Group
The Sales Analysis menu was updated with a new report. The By Sales Rep / Major Group report was created to provide a report that would display both Gross Profit and Sales Analysis information for the Sales Rep as well as Major Group.
By Ship To
new reports were added to the Monthly Sales & GP Variance reports menu. These reports will be displayed only if the new Ship To field in the F5-Sales Analysis screen in the File Update Parameters option is set to A. This A setting determines if customer ship to information will be kept. Only if this flag is set to A, will these reports be available. Please note that if this flag is set to A and the reports to be activated, it is required that the Sales Analysis rebuild is processed. These reports were only added to retrieve Current Salesreps from the Ship-to file if defined, they are not created to display ship-to information. The new reports added are ‘By Sales Rep, ‘By Sales Rep/Customer, ‘By Sales Rep/Customer/Product Number, ‘By Sales Rep/Customer/Product Line, ‘By Sales Rep/Customer Ship-To, ‘By Sales Rep/Product Line, ‘By Sales Rep/Major Group, ‘By Sales Rep/Vendor.
