Work Center Activity Maintenance
The Work Center Activity Maintenance routine was created for V17 to enable a work center (which is created as a Burden Code in Burden Code Maintenance) to be allocated to a work order. This routine is ideally used for updating the system with information pertaining to the length of time a technician has spent in a work center for a work order. Once a work order number is entered in the work center, the start date and time will begin for the work order. When the work order number is re-entered, the date and time for this work order will be stopped. However, when both entering and re-entering the work order number, you must verify the time. Select No to choose a different work order. Choose Yes to either start or stop the time. This maintenance screen also displays your open work orders. You cannot modify either the start or stop time. The system will then calculate the estimated time (hours and minutes) the technician spent in the work center and updates the work order with a Miscellaneous Entry. Note: Multiple work centers are not allowed to be in process for a work order. The work center in process must be stopped prior to accessing the work center for the next work order being started. In addition to this new maintenance screen, a new Work In Process Inquiry has also been created.
Release Pick Tickets
Release Pick Tickets uses GRIM. You can change the selection criteria or schedule the process.
Bill of Material Maintenance
Bill of Material Maintenance screen now accommodates kit parents without exploding the kit components. Previously you had to manually add the kit parent item to the work order since BOMM only exploded the kit components. Now you can simply select Continue to enter just the kit parent item.
Work Order Type Filter
Work Order Type in the Filters screen now recognizes 07 type orders in Work Order Inquiry, Work Order Quote Inquiry, Work Order History Inquiry, Work In Process Inquiry.
Work In Process Inquiry
with the new Work Center Activity Maintenance option that was created in the Work Order Processing menu, a new Work In Process Inquiry was added as well. This inquiry will display any work orders by a warehouse that are currently in process for a work center. This inquiry will also allow you to input Sort and Filter options to determine how the information is to be displayed. Due to a modification in logic in several of the current Work Order Inquiry screens, the Work Order Type field in the Filters screen will also recognize type 07 work orders as a valid order type to search by.
Work Order Completions Report
Previously, this report did not display totals for the completion quantity column at the warehouse or grand total level. This has since been modified and now has the capability of displaying both the completion quantity for the each warehouse as well as grand total of the completion quantity for all warehouses.
