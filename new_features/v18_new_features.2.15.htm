Bill of Material/Work Orders
Work Order Entry
Product Number Previously Entered - New Message
If you see the message Warning!! This Product Number was Previously Entered in WOE, you can choose Return To Product in addition to Return to Order to go back to the order and enter a different part andContinue to return to the Line Item Detail screen for the duplicate product. Return To Product takes you to the first instance of the product and removes the duplicate.
Total Issue Cost – New Field
Total Issue Cost in Work Order Issues Entry has a running total of the components issued for the work order. This total includes components only, not labor from Miscellaneous Entry or Work Order Timesheet Entry.Total Issue Cost relects the totals for the previous release, (e.g., totals on the second release (-2) are for the first release; the third release (-3) for releases 1 and 2).
Teardown Contact ID – New Field
ForTeardown Contact Id in Customer Maintenance > F8-Contact, enter a contact id for the customer that overrides the salesrep for the Final Price Inquiry in Work Order Status Entry.
Budget Labor and Material Costs
•
The Work Order Budget Inquiry displays the budget labor and material costs for all work orders released for a warehouse. Only labor and material costs are considered budget values and are updated when the work order is released. Any manually issued components in Work Order Issues Entry or burden items from Miscellaneous Entry, Order Timesheet Entry, or Work Center Activity Maintenance are updated. If a partial quantity is released, the full assembled quantity is used.
•
Change Order in the Work Order Entry header can be set to display in the Work Order Budget Inquiry under the C column.
•
The Work Order Budget Comparison Report includes the estimated vs. actual labor and material costs for each work order completed.
Linked Sales Orders On Credit Hold
When using F6-Release in Work Order Release Inquiry, the system now checks the linked sales order to see if it is on credit hold. You must release the linked sales order from credit hold in Credit Release Inquiry before the work order can be released.
Work Order Completions
Work Order Completions Report – New Fields
On the Work Order Completions Report:
•
Completion Date was added to the selection criteria.
•
Cpl-Qty (completed quantity) includes a total for each warehouse and a grand total.
•
Lnk-Qty (linked quantity) was added to display the linked quantity of each work order completion. A total displays for each warehouse and a grand total.
Fixed Asset Record For Type 05 Expense Work Orders
You can create a fixed asset record for Type 05 expense work orders when the work order is complted. The Fixed Asset record is recognized as a sub asset of the original fixed asset. Add on Capital in the Work Order Expense Reason Code Maintenance screen must be set to Y.
Work Order Log Inquiry
Vendor Claim Number
•
Header Notes in Work Order Log Inquiry shows the Vendor Claim Number when a work order is completed and the vendor claim is created for the expense items on the work order.
•
The Examine screen (from the Action Bar) shows the Expense Reason Code when you complete the work order and create the vendor claim for the expense item on the work order.
BOM Component Replacement Entry – ADD & DEL Codes Modified
When a bill of material is modified in BOM Component Replacement, the Bill of Material Log Inquiry is updated with ADD for added components and DEL for deleted components. Previously, the inquiry was only updated if you changed Bill Of Material Maintenance.
BMW/Work Order Parameters
•
Auto W/O Deletion From SOE - determines if a work order is deleted when its linked sales order line is deleted.
•
Allow WO Over To Linked Xfer - Determines if overages are allocated to a transfer or to stock.
