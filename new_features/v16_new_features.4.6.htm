General Ledger
Sort & Break By Location
The Trial Balance report was updated with a new Sort & Break Location field. This is a Y/N flag that will determine if this report will sort and break by location number as well as display the totals for each location. Note: If the Multiple Currency flag is set to Y, the Trial Balance Report contains the standard options plus a few additional options that pertain to currency only. When the Multiple Currency flag is set to N, the Trial Balance report contains the standard options only.
Chart of Accounts Balance Report
The Chart of Accounts Balance Report was added to the reports option in the General Ledger module. This new report enables the user to enter a range of account numbers and a period to display account number and balance information. This report is displays the same information as the Chart of Accounts Balance Inquiry with the exception, that this report can be printed as the inquiry cannot.
