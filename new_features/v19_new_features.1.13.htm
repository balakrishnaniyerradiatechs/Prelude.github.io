Warehousing
RF Sales Order Entry
Entering RF Sales Orders
You can enter sales orders by batch on RF devices if you use multiple bins to enter RF Sales Orders by batch.
Using Customer Keywords
You can enter customer keywords in the Customer # field to search for customers. When you enter a keyword in Customer #, Prelude displays the available customer keyword list.
Selecting Product Numbers
The Handheld RF Order and Transfer Picking display the product lookup when you enter an alternate number or keyword in a Product Number field that is set up for multiple products.
Picking and Packing
Suspending RF Order Picking
You can now suspend RF Order Picking while picking line items. When the picker’s user ID is entered in RF Order picking, the remaining line items from the suspended session appear. You must complete suspended orders before beginning new RF Order Picking sessions.
Setting Packing Ship Via Codes
Enter the packing ship via priority codes for the user ID entered in Picking Ship Via Maintenance.
Allocating Sales Order Bins
If you are using Reel Optimization, Prelude assigns a lot number and bin location in Sales Order Bin Allocation when you enter a sales order. If you delete a release, Prelude will keep the original lot number and bin location to use when the pick ticket is released.
Choosing Staging Bins
When you choose NEXT to default for the release number, the next available release for packing appears on the Release Confirmation Entry screen. After the release is updated, you can choose a staging bin, which is the final destination bin for packing.
Releasing Confirmation Automatically
Enter Y in Auto RCE After RF Picking to automatically enter confirm shipment after RF Order Picking. Set Generate Release By to Z in Warehouse Maintenance to confirm shipment by zone.
Entering Default Release Numbers
Enter X (Next) in the Default Release# field to choose the next available release number in Release Confirmation Entry. Prelude updates the user’s packer ID to determine shipping priority.
Viewing Quantity Discrepancies
A message appears in Handheld RF Stock Receipts and Multi P/O SRE if the full quantity received does not match the quantity of serial or lot numbers entered. Enter one of the following:
•
R - return to the RF Serial Number screen and re-enter the remaining serial numbers.
•
D - default to the quantity of serial numbers entered
Updating Freight Class
Use Order Picking to update the freight class for the sales order release when the order is picked. To use this new feature, enter Y in Support Weight By Freight Class, Z in Generate Release By, and Y in Auto RCE After RF Picking. This feature affects the following programs:
•
Systems > System Parameter > Maintenance > Option Parameters > Additional Option Parameters.
•
Warehousing > Sales Order Processing > Release Shipments Entry.
Entering Package Numbers
You can enter package number in RF Hold Area Assign in the Package # field. Enter the complete package number or the last ten digits or a release number.
Putaway
Selecting Putaway Methods
You can select the Drop putaway method for bin replenishment for the following programs:
•
Warehousing > Bin Replenishments Processing > Replenishment Picking Entry.
•
Warehousing > Handheld RF Menu > Doc Picking > Bin Repl Picking.
•
Warehousing > Truck Mount RF Menu > Doc Picking > Bin Repl Picking.
Updating Bin Information
When you enter putaways for a dock or inspection bin, Prelude now updates the linked transfer or work order with the new To bin
Using N and K Putaway Methods
You can use putaway method N (manual primary) or K (manual bulk) in Validate Radio Frequency Entry.
Using Lead Identifiers
The following screens in the Recv & Putaway menu were updated to use the lead identifier P for the product number field in:
•
Stock Receipts (Handheld and Truck Mount RF Menu).
•
Multi P/O SRE.
For example, if you enter the lead identifier P before the product number RFPROD as
PRFPROD, Prelude updates the product number in the Product Number field for these screens.
Bins
Printing Labels
Using Product Label By Bin Location, you can select bins and product labels for warehouses.
Creating Quick Bins
Create multiple bin locations based on a warehouse and bin type.
Listing Warehouses and Destination Bins
You can enter a list of receiving warehouses and destination bins in Warehouse Maintenance.
Selecting Destination Bins
If you define ship via codes for a release in Ship Via Maintenance, Prelude only recognizes the defined bins for sales order picking and picking confirmation entry.
Maintaining Warranty Tag Bins
Use Warranty Tag Bin Maintenance to update the bin for a warranty tag. This new feature affects the following programs in Prelude:
•
Warehousing > Handheld RF Menu > Maintenance > Wty Tag Bin Mnt.
•
Inventory > Vendor Claims > Warranty Tag Bin Maintenance.
Moving Work Order Bins
Moves bins for warehousing work orders that do not use work order picking. You cannot change the To Bin.
Entering Maximum Bin Quantities
Prelude checks the maximum quantity for the bin when you select P (primary) or N (manual primary) as the putaway method.
Allocating Work Order Bins
When you set the Allow W/O Release Partial to Y, Prelude allocates bin commitments based on the partial quantity entered for the work order in Work Order Entry.

Route Processing
Entering Route Processing Dates
You now enter a date in Route Processing. The Route Process Date defaults to the current date.
Starting Route Processing
Use the Whse Start Time field to set default start time for Route Processing, picker statistics, and packer statistics as the current day's date. For example, if you enter 09:00 in Whse Start Time, Prelude considers all processing after 9:00 a.m. as the current date and all processing before 9:00 a.m. as the previous day's date. This new feature affects the following programs in Prelude:
•
Systems > System Parameter > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint > F5-Add Info.
•
Systems > System Parameter > Maintenance > User Identification Maintenance > F10-Action > User Options > Warehousing.
Reports
Bin Location Product Availability Report
The report was updated to include items with negative quantities.
Transfer Picking Report
Pickable weight was added to each detail line of the Transfer Picking Report. The total pickable weight displays at the bottom of the last report page.
Empty Bins Report
You can enter bin type when running an Empty Bins Report. Default bin types are:
•
Pick.
•
Bulk.
•
Bulk unpackaged.
•
Retail.
•
Linestock.
Replenishment Requirements Detail Report
The Src column was added to the Replenishment Requirements Detail Report.
Zone Recap Report
This report was updated to check the RELEASE file for the zone.
Inquiries
Open Order Release Inquiry
The following status options appear on the Open Order Release Inquiry:
•
WAITING - order released but not assigned to a picker.
•
IN-PICK - order assigned to a picker but not picked.
•
PT.PICK - order partially picked in RF Picking.
•
BOXING - order assigned to a packer but not packed.
•
BOXED - order updated in Release Confirmation Entry.
•
SHIPPED - order updated in Release Shipment Entry or updated with a shipping or waybill number.
•
LOADED - order updated in Truck Loading Entry.
Transfer Warehouse Picking Inquiry
Enter the packing ship via priority codes for the user ID entered in Picking Ship Via Maintenance
Inventory Inquiry
You can view the Inventory Inquiry screen on RF devices. Prelude defaults to the warehouse set up for the user's ID in Inventory Inquiry on handheld and truck mount RF menus.
Delivery Preview Inquiry
You can add a sales order to a route in F5- Review. This inquiry now excludes sales orders on credit hold, admin order hold, and future orders. Only orders available for picking will be included. The Delivery Preview Inquiry includes new columns and buttons:
•
Weight.
•
Volume.
•
F5- Review- Volume.
•
F5- Review > F8- Change Route: Change the route for an order.
•
F5- Review > F9- Change Req Date: Change the required date for an order.
User Forms Changes
These new features require billable user forms changes.
Printing RF Picking Shipping Labels
Enter Y in RF Picking Shipping Label in Warehouse Maintenance to print shipping labels for the release number during RF Order Picking:
•
Systems > System Parameter > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint > F5-Add Info.
•
Inventory > Maintenance > Bin Files > Zone Maintenance.
Printing Vendor Claim Packing Lists
After you finish picking, you can choose a printer for the vendor claim packing lists. This new feature affects:
•
Warehousing > Handheld RF Menu > Doc Picking > Vendor Claim Picking.
•
Warehousing > Truck Mount RF Menu > Doc Picking > Vendor Claim Picking.
Reprinting Route Documents
Use Route Document Reprint to reprint documents from Route Processing. To use this option, Enter Y in the following fields in Warehouse Maintenance > F7 > F5 > F10:
•
Bill of lading.
•
Packing list.
•
RGA.
Entering a Batch by License or Seal Number
You can update batches by license plate or seal number. Prelude updates the onhand for the bin for the license plate or seal number. This new feature affects:
•
Warehousing > Handheld RF Menu > Processes > Recv & Putaway > LicPlate SRE.
•
Inventory > Stock Receipts > Batch Stock Receipts > Batch Stock Receipts by Seal.
