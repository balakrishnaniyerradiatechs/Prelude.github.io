Shop Repair
Shop Repair Option Parameters
The following flags are new in Shop Repair Option Parameters:
•
Update Equip Expense With List’ determines if the price on the order updates the equipment expense.
•
Auto Shop Charges indicates if each pay type on the order creates a separate shop charge:
-
Y - Automatically creates a shop charge using the pay type for each job.
-
N - Functions as before so that the Shop Repair Job Entry screen appears to allow you to manually add each shop charge.
•
Update Asset For Capital Repair determines if a sub-asset or a main asset is created for a shop repair order:
-
Y - Updates the main asset for the shop/repair order with the dollars for add-on capital updated in the Asset Additions in Asset Maintenance > F7-Books.
-
N - Does not update the main asset. The system creates a sub-asset for the add-on capital.
•
Continuous Shop Invoice Print determines if shop invoices should print continuously.
•
Repair Group Optional indicates if Repair Group is optional or required in the Shop Repair Job Entry screen. The Technician’s skill set in Technician Maintenance determines if a repair group is required.
•
Prorate Labor Cost determines if the actual labor hours for each job are used to calculate the labor cost. Labor cost is only pro-rated for the first service labor item for each shop/repair job.
•
Use Pline for Capital Postings determines whether to use the G/L account from the product line or the A/R offset account for capital postings:
-
Y - Use the product line.
-
N - Use the A/R offset account.
This flag only applies to shop repair orders with a pay type that has Capitalize set to Y. If the unit is still on hand, the posting goes to the inventory account. If the inventory is sold, the posting goes to the cost of goods account.
•
For Process Inventory as WIP, indicate if inventory is processed using the work-in-process inventory method:
-
Y - Process inventory as work-in-process (WIP). Inventory is relieved from the on-hand balance during Release Confirmation Entry and put into the WIP. When the order in invoiced, the quantities move out of WIP.
-
N - Do not process inventory as work-in-process.
•
Convert Estimated Labor determines if estimated labor on a shop repair quote will be converted to the order.
Jobs
New Repair Amount Type for Job Entry
The new Repair Amount Type in Job Code Maintenance indicates if the costs included in the Repair Amount for a job comes from labor, parts, or both.
Import Jobs From Quotes
While in Shop Repair Order Entry, you can import a job from a shop repair quote (e.g., a repair requires additional work and you need to provide an estimate). You can enter the shop repair quote and then import the job quote onto the existing order by choosing Action Bar > Job Func > Import Job from Quote from Shop Repair Order Entry.
Schedule Jobs
When you enter a job, you can use the new scheduling calendar to assign technicians and time periods to the job. From the Shop Repair - Job screen, select F9-Schedule to display the total amount of all available technician time for each day of the month. You can then select F5-Review > F6-Schedule to schedule a technician and a length of time. Once selected, the information displays on the Job Entry screen.
Equipment/Serial Numbers
Customer Equipment Maintenance/Inquiry
When a customer buys a serialized product, the serial number is assigned to that customer to help track return and repair information. If the product is sold or for any reason becomes the property of another customer, you can now use Customer Equipment Maintenance to assign the product and serial number to the new customer. The EIN also appears on in Customer Equipment Maintenance and Customer Equipment Inquiry.
Only Change Status of Rental Serialized Products
Only a rental serialized item can be set to D (down for repair) in Shop Repair Order Entry (Action Bar > Rent Info). If you try to change the status of a non-rental serialized item, the message Access Denied! Repair Must be for A Rental Product displays. The status of the item changes to A (available) after the order is invoiced.
Generating Shop Repair Orders for Rental Returns
When a rental return is completed for a product that is scheduled for routine maintenance, you can generate a shop repair order with a shop repair job with a default description generates for each part returned. If the rental return is for another warehouse, the serial number is updated for the correct warehouse.
Basing the Warranty Period on Hours
The new Wty Hours field in Product Maintenance > F5 > F5 allows you to set a warranty period based on the number of hours on the equipment. For example, the warranty period could be one year or 1500 hours, whichever comes first. In Shop Repair Order Entry, a warning message alerts you that the unit is under warranty.
Technician Shop Rate Maintenance Renamed
The Technician Shop Rate Maintenance program was renamed to Technician Maintenance. The new F5-Repair Skill Set key allows you to identify authorized repair groups. address, and phone information for the technician. The repair skill set information was removed from Sales Representative Maintenance.
Accessing Tracking Information
Freight Detail on the Shop Repair inquiries allows you to access the freight tracking and PSS manifest data. To view, choose F5-Review > Action Bar > Order Func in Open Shop Repair Inquiry or History Shop Repair Inquiry.
Reports
Open Repair Quote Report (new)
Lists quotes that were not converted to orders. Displays detail product, model, and serial number information.
Shop Inventory Work in Process Report (new)
The new Shop Inventory Work in Process Report lists inventory allocated to Work in Process (WIP) during Release Confirmation Entry. When the order is invoiced, WIP inventory is relieved rather than on-hand.
Printing
Print Inspection Tickets
The Shop Document Selection screen that appears when you complete a shop repair order has a new flag that allows you to print an inspection ticket. Inspection tickets list the equipment required for the repair and the type of inspection that is needed. The Shop Inspection flag (Warehouse Maintenance > F6-Continuous) determines if the inspection tickets print continuously. If printing inspection tickets by batch, you can use the new Shop Repair Inspection Report Printing option on the batch printing menu.
Override Continuous Printing by Location
New options in Startup Parameters > F5-Printers allow you to change continuous printing from WM for:
•
Shop Repair Quote Queue
•
Shop Repair Queue
•
Shop Repair Pickup Que
•
Shop Repair Delivery Que
•
Shop Repair Inspection
