Systems
Financial
Editing Credit Comments
Use the Edit Credit Comments flag in A/R Parameters to allow users to edit credit comments. Enter Y or Null to allow users to edit credit comments. If you enter N, Prelude does not display the Review Date field because the standard Notes Entry screen appears. Enter the date and time for credit comments in Notes Entry in:
•
Systems > System Parameter > Maintenance > Option Parameters > A/R Parameters.
•
Financial > Accounts Receivable > Inquiries > Customer Inquiry.
•
Financial > Accounts Receivable > Inquiries > Customer Invoice Inquiry.
•
Financial > Accounts Receivable > Inquiries > A/R & Customer Credit Inquiry.
•
Financial > Accounts Receivable > Maintenance > Customer Maintenance.
Entering G/L Account Numbers
You can enter up to 12 digits for the G/L Acct# in Company Maintenance.
Saving A/P Balances
Use Month-End Vendor A/P Aging (General Ledger > Processes > End of Month Closing > Month-End Vendor A/P Aging to save vendor A/P balances and run A/P aging reports based on fiscal data. You can purge this data.
Purging Vendor Fiscal Period Data
Use the Vendor Fiscal Period Data Purge to delete the fiscal period data from the associated vendor stat files that are updated using Month End Vendor A/P Aging.
Purging Customer/Warehouse Fiscal Period Data
Purge the fiscal period data from the associated customer stat and warehouse stat files using Customer/Warehouse Fiscal Period Data Purge.
Setting Default Sort Options
You can choose the default sort for invoices in Cash Receipts Entry in Default Parameters Maintenance:
•
I - Invoice Date.
•
D - Due Date (default).
Tracking Administrative Releases
Enter Y the Admin Mgr. Release field in File Update Parameters to track admin hold releases.
User Identification Maintenance
Viewing the DID Fax Number
DID (direct inward dialing) Fax Number was added to the Personal screen in User Identification Maintenance.
Restricting Transfer Entry
Prelude now checks location number in User Identification Maintenance to stop users from entering transfers.
Warehousing
Generating Emergency Bin Replenishment
Enter Y in the Generate Emergency Bin Repl in Warehouse Maintenance to generate emergency bin replenishment during Transfer Bin Allocation or Transfer Pick Ticket Printing if the entire pick quantity cannot be allocated. This information also appears in the Replenishment Requirements Detail Report.
Renamed Transfers Key
The F5-Transfers/Work Orders key in Warehouse Maintenance was renamed to F5-Transfers. Only the transfer fields appear on the screen.
Updating Freight Class
To use Order Picking to update the freight class for each release when it is picked enter:
•
Y for Support Weight By Freight Class in Additional Option Parameters.
•
Z for Generate Release By in Warehouse Maintenance.
•
Y for Auto RCE After RF Picking in Warehousing Maintenance.
Adjusting Inventory
Use RF Inv Adj Reason Code in Warehousing Parameters to adjust inventory on RF devices.
System Parameters
Exploding Kit Components on Work Orders
Use Auto Kit Explosion in WOE in BMW/Work Order Parameters to create kit components when you create work orders.
Limiting Email Size
Enter a value in megabytes or kilobytes in the Size Limit field in General Parameters to limit broadcast and standard report email size. For example, enter 20M to limit the size to 20 megabytes or 1024K to limit the size to 1024 kilobytes.
Verifying Printer Queues
Prelude checks for a valid printer queue when you use a default printer to print continuously.
Viewing a Logoff Message
In ADS Utilities, you can choose to display the message, Do You Wish To Logoff? when you press ESC or close the main menu. When you choose to use the message, Prelude will always display the logoff question. Contact Prelude if you want to turn the message off.
Assigning Starting Product Numbers
You can set up a starting product number for new products in Starting Product Number Assignment.
Entering Inventory Notes
Enter the code PWM in Notes Edit Authorizations to allow users to edit notes in Product Warehouse Maintenance, Product Purchasing Parameters Maintenance, and Distribution Requirements Evaluation Inquiry.
Setting Default AVS for Credit Card Payment
You can set AVS information to default from the customer record when you enter credit card payment on a sales order. Enter Y in Default AVS For Credit Card to enter information from the customer record by default.
Calculating Credit Lines for Rental Customers
Set the Auto Credit Calc Method in Rental Parameters to specify how Prelude should automatically calculate the credit lines added for early return rental orders. Enter D in Auto Credit Calc Method to calculate the credit line using the daily rate. Enter B to calculate the credit line using the best rate. You must set the Allow Auto Credit field to Y or A to allow credits for early returns on pre-billed or periodically billed orders. This new feature affects:
Rebuild Audit Inquiry
Use the Rebuild Audit Inquiry to view the date, time, and user ID for the last time data was rebuilt.
Printing Customer Logos
Use Logo Name in Warehouse Maintenance to enter the name of the logo to use on forms. Prelude prints the Activant logo by default if you do not enter a logo name. Requires a billable user forms change.
GRIM Interface
Scheduling Reports by Fiscal Date
You can now schedule reports by the first or last day of the fiscal period. Enter F in the Recurring Pattern field to schedule the report for the first day of the fiscal period. Enter L to schedule the report for the period’s last day.
Entering Dates
When entering dates in a GRIM screen, press F1 to enter the date of the first day of the previous month.
Choosing Date Range
When you enter a date range in a GRIM report or process that is scheduled, you must choose Yes to continue.
