Marketing
Marketing Notification List
The Marketing Notification List is a new option in Customer Maintenance (Action Bar> Addl Options > Marketing Notification List). This feature allows you to establish user ids and email addresses for a customer or employee to notify them of marketing activities. A valid email address must be in User Identification Maintenance if you are using a user id. Any marketing activities performed for the customer automatically sends an email notification to the user ids, and email addresses entered here.
The Marketing Activity Entry and Marketing Activity Contact Entry processes were updated to check the customer’s Marketing Notification List.
Activity Function Changed in Prospect Maintenance
The F9-Activity function key in Prospect Maintenance, now opens the Marketing Activity Entry screen, rather than the Prospect Activity Entry screen .
Broadcasting
Additional Email Addresses Available in Broadcast Email
The F3 lookup feature in the To field in Broadcast Email was updated to display email addresses for user IDs, customers, contacts, prospects, or vendors.
Broadcast Email Form Letters – Program Fix
A program fix was made to correct the problem of form letter emails losing characters when sent to customers.
Contact Management
Web User – New Customer Contact Option
The new Web User feature (Maintenance > Customer Contacts > Action Bar > Web User) allows you to create a Web User Id for the customer contact number. You can enter a Web User Default to set the appropriate system flags for the new Web User Id. Once you press F2-Save, the web user id is created using the customer number, contact number and personal information such as address, phone, fax etc. The rest of the settings are pulled from the web user id entered in the Web User Default field.
Change to Address Field Descriptions – Program Fix
A program fix was made to the Contact Maintenance Audit Reports to update the field descriptions for Mailing Address 1, Mailing Address 2, and Mailing Address 3, for each change made in Customer Contacts. When you make an address change for a customer contact, the changes made to the customer address now only show up once on the report.
Contact Statistics – Display Updated
The Contact Statistics screen, accessible from the Customer Contact Maintenance Action Bar, was updated to reflect the same screen format as the rest of the statistics screens in the system (customer, vendor, vendor line, etc).
