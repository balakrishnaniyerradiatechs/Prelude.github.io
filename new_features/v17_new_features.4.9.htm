Shop/Repair
The Shop/Repair module is a new module created for V17. This new feature was added to better accommodate your company's needs of expediting billing by processing completed invoices immediately, therefore, generating a more consistent cash flow. This process will provide users the ability to create orders that will be automatically maintained for multiple repair jobs while keeping track of additional parts, technician repair time, extra repair costs and service requirements. You can access to warranty information, repair cost/ time estimates and repair history on model and serial numbers while entering the repair order. In addition, the capability of viewing the status of a job and the productivity of the technicians is available at any time.
Note: If you would like to purchase this module, please contact the Prelude's Sales Department. Considering that this is a new module for this version, the new features are not currently listed throughout the New Feature documentation. However, the Shop/Repair document will review all the steps in setting up and using this new module.
