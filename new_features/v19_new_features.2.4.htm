Forecasting
Excluding a Warehouse from a Formula
When you set up or modify a formula in Forecast Formula Maintenance, you can exclude one or more warehouses from the formula (Action Bar > Exclusion) on the Comparison Forecast Report.
Set Maximum Allowable Trend Calculation by Warehouse
The new Trend % Maximum field in Warehouse Maintenance > F5 allows you to set a maximum percentage that the system can calculate for each warehouse. This percent overrides the Trend % Maximum in Product Purchasing Parameters Maintenance. The Trend % Maximum is used with End of Month Calendar Close.
Customer Usage
Adjust Customer Usage by Warehouse/Product
The selection criteria in the Remove/Restore Customer Usage Procedure now allows you to adjust usage for a customer by warehouse and product number.
Move Customer Usage
If an existing customer is assigned to a different warehouse in your company, you can use the new Move Customer Usage Procedure to move the usage recorded for a product, vendor, or product line to the new warehouse. You can move the customer’s usage to a new warehouse to correctly forecast usage for both warehouses.
DRP
View Corporate DRP Requirements
The DRP Inquiry has a new Consolidate option on the Action Bar to allow you to choose two or more warehouses to include in the distribution requirement calculations.
Automatically Generate a DRP Purchase Order
After you run and review the Distribution Requirements Planning Report, you can use the same selection criteria in the DRP Purchase Order Generation Process to automatically create a purchase order for the recommended early buy quantities. The generated purchase order appears in the Buyer Queue Inquiry under Recommended Purchase Orders.
