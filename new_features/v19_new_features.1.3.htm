Bills of Material/Work Orders
Bills of Material
BOM Quantity Display Includes Multi-Level Components
The available BOM quantity at the bottom of Inventory Inquiry includes multi-level bill of material components. Previously, the quantity only included components for the first level BOM.
Exploding Bills of Material
If you enter Y in Auto BOM Explosion in WOE, Prelude automatically explodes the components set up as a bill of material onto the work order.
BOM Exploded Availability Inquiry
This inquiry displays components for the BOM entered depending on the number of levels.
New Batch Work Order Entry
The New Batch Work Order Entry provides a quick way to create multiple work orders for bill of material items, without having to re-enter all of the products. You can choose whether you want to create one or multiple work orders for one bill of material.
Access to Cost Field on Out-Plant Work Order
You can now access the cost field for the out-plant burden code on a type 03 out-plant work order. This option does not apply to Miscellaneous Entry.
Pick Number Field Added
The Picker Number field was added to the Work Order Issues Entry screen.
Changing Totals in Work Order Quotes
If you press F6-Totals to change the GP% or Total Dollars in Work Order Quote Entry the lines are recalculated according to the new amount. GP% and W/O Cost were added to the screen.
Technician Job Billing Entry
You can enter work orders in Technician Job Billing Entry.
