Rental
Updated Rental Status Code
The functionality of the status code S was updated in Rental Status Maintenance. Previously, the S status code was only for rental-serialized items sold. With this modification, the status of the serial number is set to S for sold non-rental serialized items. In addition, you can enter non-rental serialized items.
Updated Entry Functionality
With the new H Rental Return Status option, the Rental Order and Rental Confirmation Entry processes were updated. The system no longer allows serial numbers with andH (Hold) status to be entered for rental orders. The message Invalid Entry. This unit is on hold â€“ status must be changed manually is displayed.
Asset Maintenance
Updated Manual Entry Functionality
The functionality of the F6-Manual Entry in the F7-Books screen was updated with a new Depr Percent flag (referenced in the Inventory New Features list). Now when a rental item is invoiced that has the Depr on Rent Revenue flag set to Y and a Depr Percent value in Product Maintenance, the system updates the rental asset record with a manual depreciation entry. This entry is posted to the G/L and the fixed asset record (YTD & LTD Depreciation) the next time the Manual Depreciation Update process is performed. Note: The depreciation method for the rental asset record MUST be set to M (manual) for this feature to work.
Updated Master Asset Entry Functionality
with the recently added ability to create a fixed asset record at the time of Work Order Completions (referenced in the BMW/WO New Features list), the functionality of the Master Asset Entry has also been updated. The Master Asset Entry screen will now be automatically updated for the sub asset record created for Type 05 expense work orders at the time of the Work Order Completion process. Therefore, the master asset record would be displayed for reference when entering the sub asset.
Updated Sub Asset Display Functionality
with the recently added ability to create a fixed asset record at the time of Work Order Completions (referenced in the BMW/WO New Features list), the functionality of the Sub Asset Disp has also been updated. The Sub Assets Display screen will now be automatically updated for the sub asset record created for Type 05 expense work orders at the time of the Work Order Completion process. Therefore, the sub asset record would be displayed for reference when referring to the master asset.
Equipment Maintenance
Updated Financial Functionality
The functionality of the F8-Financial option was modified. The equipment statistics are now updated when the shop/repair order is invoiced for the rental-serialized item. The Maintenance Expense-parts column is updated with the total cost of parts when the shop/repair order is invoiced. The Maintenance Expense-Labor column is updated with the total cost of labor (including shop charges) when the shop/repair order is invoiced.
Updated Review Maintenance Functionality
The F5-Review Maint Exp screen was updated to now display shop/repair orders that are invoiced for the renal serialized item. Previously, the screen only displayed expense work orders.
Updated F3 Lookup Functionality - New Sold? Field
The F3- Lookup option for the Equipment ID Number in Equipment Maintenance and Equipment Inquiry now use GRIM selection screen interface. In addition, a new Sold? flag was added to the GRIM screen. Select Y to include equipment id numbers that were sold (Status S). Select N to include only equipment id numbers that has not been sold (all other status codes). Leave blank to see all equipment id number results.
New Comments Option
The Equipment Maintenance screen was updated with a new Comments option in the Action Bar. The comments can also now be viewed in the Equipment Inquiry.
