Bin Individual Move Entry
This process was modified to update the linked sales order line item with the new To Bin if a bin move is done after the stock receipt was processed and an order line(s) exists with the dock type From Bin loaded on the order line. After the To Bin is selected and if the From Bin is a dock type bin, the system will check for open sales orders for the product that have linkage history and currently have a line item bin that matches the From Bin. If there is only one sales order then the bin for that order line item will be updated behind the scenes. If there is more than one order, the system will display the order numbers and the user must choose one to be updated. Only one order per bin move line entry can be updated at a time. A new bin move entry line must be entered to take care of multiple sales orders. This allows the bin allocation rules to continue to work as they do now by allocating to the bin on the order line record. This works with both standard bin moves and RF bin moves and with RF Putaway Entry.

Putaway Entry
This process was modified to update the linked sales order line item with the new To Bin if a bin move is done after the stock receipt was processed and an order line(s) exists with the dock type From Bin loaded on the order line. See Bin Individual Move Entry for more information.
Putaway Method has a new option of V for Volume, which is used to determine bin putaways according to the volume of the bin. Previously, the putaway methods were B for Bulk, M for Manual, and P for Primary. These methods do not check for available room like the new V method does. The hierarchy rules for directing putaway using this method are as follows:
•
For special order products, there will be no directed putaway. The operator will indicate a final destination bin location. This bin location will be updated to the sales order for bin allocation process. The Primary bin is considered first.
•
If the putaway quantity is less than or equal to the quantity required to fill the bin using bin maximum minus bin onhand, then the primary bin established in Product Maintenance is recommended.
•
If the quantity in the primary bin is less than or equal to the bin minimum, then the primary bin is filled to maximum and the rest is put into a reserve location (following the logic below).
•
If the putaway quantity is greater then the quantity required to reach the bin maximum and the quantity in the primary bin is greater than the bin minimum, then the system does not direct the putaway to the primary bin but goes to the next putaway rule.
•
If there is a single bulk reserve bin location where the product already exists that can hold the entire quantity to be putaway up to the fill level percent for that zone (defined in Zone Maintenance) and it resides in the same aisle as the primary bin, then that bulk bin is recommended.
•
If there is a bulk reserve location where the product already exists on the same aisle as the primary bin location that can take part of the putaway using the partial putaway quantity minimum percent (defined in Zone Maintenance), then that bulk bin is recommended. The remaining quantity is placed in another reserve location where the product already exists. The first preference is a bin on the same aisle, then the system looks for the closest aisle.
Note: if the putaway quantity cannot be completely placed in existing reserve locations, the system will recommend an empty bin instead of the existing reserve locations. We do not want to receive 10 boxes of a product, place 5 in the only reserve location and place the remaining 5 in an empty bin. This example should place all 10 boxes in one empty bin.
•
If there is an empty bin in the same aisle as the primary bin location, then that empty bin is recommended.
•
If there is a bulk reserve bin location in any aisle where the product already exists that can hold the entire quantity to be putaway up to the fill level percent for that zone (defined in Zone Maintenance), then that bulk bin is recommended. The system will attempt to select a reserve location in the aisle closest to the primary bin location.
•
If there is a bulk reserve bin location in any aisle where the product already exists that can take part of the putaway using the partial putaway quantity minimum percent (defined in Zone Maintenance), then that bulk bin is recommended. The system will attempt to select a reserve location in the aisle closest to the primary bin location. The remaining quantity is placed in another reserve location where the product already exists.
Note: if the putaway quantity cannot be completely placed in existing reserve locations, the system will recommend an empty bin instead of the existing reserve locations.
•
The system recommends an empty bin as close to the primary bin as possible.
•
If none of the criteria could facilitate the recommended putaway, no bin will be recommended and the system will prompt the user to manually enter the bin.
Note: with the V putaway method, the system will not recommend a bin that does not have a cube value.
