Warehousing
New Menus
Radio Frequency Menu Replaced by New RF Menus
The Radio Frequency Menu previously on the Warehousing menu, was replace by three new menus; the Handheld RF Menu, the Wrist RF Menu, and the Truck Mount RF Menu. The addition of these new menus allows the functionality of the RF processes to be more to handheld, wrist mount, and truck mount radio frequency devices.
F5 Key on the Handheld and Truck Mount RF Menus
All of the F5 function on these screens were changed to another function key (usually F6 or F7). The Wrist RF menu has not changed.
Batch Processing (Wrist Picking) – New Menu
The procedures in this menu were added to accommodate the new Wrist RF procedures:
•
Automatic Batch Generation Maintenance
•
Sales Order Entry
•
Batch Pick Ticket Processing
•
Automatic Batch Generation
•
RF Order Batch Queue Inquiry
•
Shipping Box Confirmation Entry
•
RF Order Batch History Inquiry
Route Delivery Menu – New Menu
The procedures on this menu were added to accommodate the warehousing functions related to routes:
•
Zone Recap Report
•
Delivery Preview Inquiry
•
Daily Manifest Review Report
•
Driver Manifest Report
Sales Orders
Hold Area Assign – New Menu Option
The Hold Area Assign option allows you to record the hold area for a sales order release.
Credit Code Checked in RF Order Entry
The Customer’s Credit Code Field is now checked when an order is entered in the RF Sales Order Entry program (handheld). If the Credit Code is set to N, orders are not allowed for the customer.
Average Pick Times per Line/Unit added to Picker Statistics Report
The Picker Statistics Report was updated to include average pick times per line/unit for the RF picking processes. The Bin Movement Inquiry displays the beginning and ending times for each bin movement.
Auto RCE After RF Picking – New Flag
The new Auto RCE After RF Picking flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing) determines if the release confirmation process, to set up an order for invoicing, is automatic after RF Order Picking.
Release Field in Order Picking – Functionality Changed
The Rel# field now allows you to enter multiple releases in one RF session. Only the first 5 releases are displayed on the screen but additional releases can be entered. The lines are sorted properly for picking.
Partial Quantities in Document Picking - Functionality Changed
These processes were modified so that if a partial quantity of a line item is picked, the balance of the pick quantity for the product comes up after the first pick.
•
Order Picking (Handheld, Truck Mount, and Wrist RF Menu)
•
Transfer Picking (Handheld and Truck Mount RF Menu)
•
Work Order Picking (Handheld RF Menu)
Prompt for Route/Ship Via - New Flag
The new Prompt for Route/Ship Via flag in Warehouse Maintenance ( F7-RTS Maint > F5-Warehousing) determines if the system should prompt for route and ship via information in RF Order Picking. If specified, only lines that match the route and ship via are assigned to the user.
Release Confirmation Entry – New Menu Option
The Release Cfm option on the Handheld RF Menu, under Processes, now allows you to perform release confirmations using radio frequency devices. You can use this to perform a quick release when no changes are necessary. If changes are required, you must use the terminal version of Release Confirmation Entry.
Prompt For Destination Bin In RF Picking – New Flag
The new Prompt for Dest Bin in RF Pick flag (Warehouse Maintenance > F7-RTS Maint > F5-Warehousing) allows you to store the destination bin for the 1st picking session of the release. This default is then used for the remaining items picked in a later picking session.
Validate Radio Frequency Entry Flag– Functionality Modified
•
The Validate Radio Frequency Entry flag in Warehouse Maintenance ( F7-RTS Maint > F5-Warehousing) was modified so the product number is not validated for asterisk items.
•
If you choose the S (selective) option for the Validate Radio Frequency Entry flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing), a new L option is available to allow you to require scanning for Lot items.
Work Orders
W/O Prompt for RF Picker – New Flag
The W/O Prompt for RF Picker flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing > F5-Transfers/Work Orders) determines if the picker Id is required in Work Order Picking after you enter the W/O number.
W/O Deletion Bin – New Flag
The new W/O Deletion Bin field in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing > F5-Transfers/Work Orders) allows you to enter a bin location to be updated when a line is deleted off of a work order. If this field is left blank, a bin move is not performed. Note: the work in process bin entered at the time of RF Work Order Picking must a have a Storage Type set to W in Bin Location Maintenance, for the system to allow the work order deletion to take place in Work Order Entry.
When a work order or line is deleted and the work order was picked using one of the RF picking screens, the system automatically moves each line item deleted to the deletion bin location in Warehouse maintenance.
W/O Allocation Priority Type - New Types
The W/O Allocation Priority Type flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing > F5-Transfers/Work Orders), has new bin types available to use in Work Order Bin Allocation. These options are:
•
D – Dock Bin
•
I – Inspection Bin
•
K - Bulk Unpackaged
•
T – Retail Bin
Work Order Completions - Warning Message
These processes now prompt with a warning message if all lines have not been picked.
Parts Issued Date – Updated in Work Orders
The Work Order Picking process updates Parts Issued Date on the Work Order Status Inquiry when the last line item is picked for the work order. If Parts Issued Date is blank, the date is updated with RF Work Order Picking.
Zone – New Field in Work Order Picking
The new Zone field in Work Order Picking allows multiple zones for picking.
W/O Radio Frequency Picking – New Flag
The new W/O Radio Frequency Picking flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing > F5-Transfers/Work Orders), determines if work order picking records are created for the work order when the release is generated and the bin allocation is performed. If you set the flag to N, the Work Order Printing process is used to generate the release and perform the bin allocation. If you set this flag to Y or null, you must use the Work Order Bin Allocation Process to generate the release and allocate the bins.
Transfers
Transfer Picking – Program Fix
A program fix was made to calculate the correct totals for the transfer created/updated at the time of Transfer Picking Entry or RF Transfer Picking.
S/T Bin Priority Type Flag – New I (Inspection) Type Available
The S/T Bin Priority Type flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing > F5-Transfers Work Orders) has a new I (inspection type) option available. This allows you to include inspection bin types in Transfer Bin Allocation.
Stock Receipts
Multi P/O SRE – Functionality Changed
When a purchase order is received in RF Multi Stock Receipts Entry, the P/O type is checked to see if it is set to 04 (multi branch). If the product on the purchase order is a valid product in the RF receiving warehouse, the product can be received. If the product is not a valid product in the receiving warehouse, it can only be added to the purchase order if the Allow New Products in RF SRE (Warehouse Maintenance) is set to Y.
Stock Receipts Putaway - Functionality Changes
Several changes were made to the SR Putaway Ent processes (handheld & truck):
•
You can choose a Putaway method and the bins print according to the Putaway Sort Seq field in Bin Location Maintenance.
•
If the quantity you enter as put away is less than the full quantity received, you can choose to keep the partial quantity or change the quantity.
•
If the Validate Radio Frequency Entry flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing) is Y, the numbers you scan are validated.
RF Receiving Information Available
When a purchase order is received in one of the RF stock Receipt screens, the RF receiving information (user Id, received quantity, bin location, date and time) is updated for each purchase order line item. Once the Batch Update Procedure updates the purchase order, the RF receiving information can be viewed in Purchase Order History Inquiry (F5-Review > Action > Examine). Applies to handheld and truck RF.
Putaway Sort Sequence – New Field
The new Putaway Sort Seq flag in Bin Locations allows you to enter the sort sequence code for putaway (handheld and truck). If blank, the Sort Sequence Code is used.
Display Count of Open B/Os in RF SRE – New Flag
The new Display Count of Open B/Os in RF SRE flag in Warehouse Maintenance (F7-RTS Maint > F5-Warehousing > F8-Receiving) determines if the prompt displays for a count of open backorder quantities in RF Stock Receipts (handheld & truck).
Negative Quantities Allowed in RF Stock Receipts
You can now enter a negative receiver in RF stock receipts with the following restrictions:
•
The stock receipt must not be processed through the Batch Update Procedure.
•
The negative quantity cannot be greater than the original quantity received.
•
If the item is lot controlled, you must enter the entire quantity but not the lot number.
•
If the item is serialized, you do not need to enter the serial number.
Default Receiving Bin Range – New Function Key
The new Default Receiving Bin Range function key (Warehouse Maintenance > Action Bar > Addl Opts > Bin Locator Information > F5-Default Receiving Bin Range) allows you to enter a beginning and ending bin location range for a default receiving bin.
Document Option – New Field for Printing Labels
The USER FORMS programs must be modified. The new Document Option field allows you to choose between open or received documents when printing labels.
