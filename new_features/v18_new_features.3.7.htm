Job Accounting
Updated Unit Cost and Actual Cost Functionality
An update was made to the Unit Cst and Act Cost columns. Previously the full dollar amount including cents were displayed for these columns. The Unit Cst and Act Cost columns now round up or down to the nearest dollar. If the cost has a decimal value of .50 or greater (9.50, for example), the system rounds up to the nearest dollar value of (10). If the cost has a decimal value of .49 or lower (9.25, for example), the system rounds down to the nearest dollar value of (9).
Updated Change Order Control Functionality
The Change Order Control flag was modified for versions 18 and higher. This flag now allows a customer with the Require Job Number flag (Customer Maintenance > F7 Order screen) set to S the ability of setting up the change order control for the job number. Previously, the system only allowed the Change Order Control flag to be set to Y if the customerâ€™s Require Job Number flag was set to Y or V.
New Job Sales G/L Account
A new Job Sales general ledger override account is now available. This new account was created for use with the job accounting sales at the product line level. This account is posted to during Job Invoice Printing and Processing. Previously, the Job Sales account in the General Ledger Table was used for job accounting sales.
