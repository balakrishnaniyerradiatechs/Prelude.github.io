Accounts Receivable
Credit Release Inquiry
Credit Release Inquiry was updated with a new suspended order feature. You can require the credit manager to enter a password for any orders on credit hold that are to be suspended. To activate this feature, the ORD.SUSPEND process must be established in the Program & Menu PassWords maintenance screen.
Invoice Detail Inquiry
A new Other column was added to the Invoice Detail Inquiry screen. This column was added to reflect additional charges that were added on the sales order completion screen. Any additional charges entered in the Handling, Delivery, Pick Up, Restocking Charge, Minimum Charge, or Disc/Allowance fields in the sales order completion screen will be displayed in the new Other column in this inquiry.
Transaction Audit Inquiry
This is a new inquiry that was added to the Accounts Receivable module in ADS. The Transaction Audit Inquiry displays a summary of all the A/R transactions that involve a customer number entered. In addition, and accumulative balance is also displayed with each debit and credit entry updated to the inquiry. in this inquiry, you also have to option to view transaction by Beginning date or Ending date entries. This can be selected in the Sort Sequence field, or you have the option of establishing a default sort in Default Parameters Maintenance. A new FWD was added to the Transaction Audit Inquiry screen. This FWD code is displayed indicating that transaction audit records were purged.
Listings
In Version 16, the Customer Labels option was updated to be processed using the GRIM (General Report Interface Maintenance) screen. This allows the user the options of modifying the criteria of the report, as the users did not have this option in previous versions.
