General Ledger
Saving A/P Balances
Use Month-End Vendor A/P Aging to save all vendor A/P balances as of the specified date and allow A/P aging reports to be run based on fiscal date. The following programs are affected.
•
General Ledger > Processes > End of Month Closing > Month-End Vendor A/P Aging.
•
Systems > System Parameter > Processes > Purge Utility Menu > Miscellaneous Purges > Month-End Vendor A/P Aging.
•
Accounts Payable > Inquiries > Accounts Payable Inquiry.
Entering G/L Account Numbers
You can enter up to 12 digits for the G/L Acct# in Company Maintenance.
Journal Entry
User ID
The User ID field displays in Journal Entry. You cannot edit this field.
Net Balance
The Net Balance displays in Journal Entry and Journal Entry Inquiry. You cannot edit this field. Prelude updates the balance after you enter a Debit Amount and Credit Amount. Sorting by Deposit Date
Prelude sorts deposit dates in Bank Reconciliation Entry starting with the earliest deposit date.
Financial Reports
Financial Report Group Listing
This listing includes financial report groups in Financial Report Group Maintenance.
Financial Report Parameters Listing
This listing includes financial report parameters in Financial Report Master Maintenance.
Printing Financial Reports
You can now override the title for financial reports when you print.
