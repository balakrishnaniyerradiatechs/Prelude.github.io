Bill of Material/Work Order
Work Order Entry/Work Order Quote Subsystem - Line Item Level
•
A new function key was added to the Product Tagging screen in Work Order Entry. F7- Resort enables the user to modify the sort option that was established as the default. The sort options available are P for product number or D for product description. For example, if the Product Tagging Default Sort flag in Default Parameters Maintenance is set to P in which the products will be displayed in ascending order by product number, and the user prefers the products to be displayed in ascending order by product description, this function key can be used to change the product sort in the tagging screen.
•
A new F8- Switch Entry Mode function key was added to the Product Tagging screen in Work Order Entry. This option allows the user to modify the position of the cursor from product number to quantity or vice versa when tagging products. A default of P for product number or Q for quantity is established in the Product Tagging Entry Mode field in Default Parameters Maintenance. For example, if the default is set to P, the cursor will be positioned at the product number, and the user prefers the cursor to be positioned at the quantity field, this function key can be used to change the cursor positioning in the tagging screen.
Work Order Issues Entry
A modification was made to Work Order Issues Entry if Vendor Consigned Inventory is being used. The components entered on the work order in Work Order Issues Entry are now updated to the CONSIGNED (Attribute 6) and CONSIGNED.SOLD (Attribute 10) files. Previously, these files were not updated when work order issues were processed for Vendor Consigned Inventory.
