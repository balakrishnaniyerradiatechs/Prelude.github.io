Sporadic Usage
The Purchasing Parameters Maintenance function on the Systems menu has a new F5-Forecasting key to allow you to enter information on spikes, sporadic usage, and target information for your company. Refer to the new Help feature on the Action Bar for a full explanation of each of the new fields on the screen.
Purchasing Forecast Parameters Maintenance
This maintenance option is on the Forecasting menu in addition to the Inventory menu.
Customer Usage Adjustment Procedure - New Feature
The Customer Usage Adjustment Procedure allows you to remove or restore a customer’s product usage. You can use this procedure when you have lost a customer and want to remove the usage recorded for a vendor or product line, so that forecasting numbers no longer reflect the customer’s usage. Once the customer’s usage is removed, you can restore it if you regain the customer’s business.

Forecast Formula Copy - The Copy feature on the Action Bar in Forecast Formula Maintenance now lets you copy an existing formula to create a new one.

Reports
•
Distribution Requirements Planning Report – Primary Vendor Number was added to the selection criteria.
•
Comparison Forecast Report - Changed the Detail or Summary selection criteria to Output Presentation. This includes a U option to provide a summary report displaying the current and suggested formula as well as the usage for the past 12 months. Added Display Formula Change Only to the selection criteria to only display products on the report if the system would recommend a different formula than the one defined.
•
Forecast Formula Utilization Report - This new report list formulas available for your company, and illustrates the percentage breakdown of the formulas used versus the formulas recommended for each warehouse. Only products with a Forecast Formula override from the Comparison Forecast report appear on this report.
•
Minimum Maximum Report - This new report includes Min/Max products and displays the Min/Max settings with the last 12 months of usage. It also shows the forecast quantity and indicates if the usage is sporadic.
