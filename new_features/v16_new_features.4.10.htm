Web User Maintenance
The F8-Conv Cust is a new flag that was added to the Web User Maintenance screen. This flag allows web users that have created themselves as customers manually over the web, as well as new users that were created through the system to be converted to ADS Customers. For the web user to be converted to an ADS customer, the Valid ADS Customer field in this maintenance screen is required to be set to N. Once a web user was converted, all open documents for this user will be updated with the correct ADS customer number, as well as automatically updating the Valid ADS Customer field to Y.
