Report Title Maintenance screen
The Report Title Maintenance screen allows you to create a title for the financial report.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Master Maintenance > F6-Titles
Field and Function Key Descriptions
Field/Function Key
Description
Description
The report description that displays on the report.
F2-Save
Saves the report description.
