Financial Report Group Maintenance screen
The Financial Report Group Maintenance allows you to establish report groups such as assets, liabilities, income, and expenses. You can also assign general ledger account numbers to the group.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Group Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Group Name
The name of the financial report group.
Description
The description of the report group.
Reverse Sign?
Determines how debits and credits print:
•
Y - Reverses credits to print as positive and debits to print as negative.
•
N - Prints credits as negatives and debits as positive.
Account#
The accounts that are included in the financial report group.
Description
The account description.
F2-Save
Saves the financial report group information added or modified.
F4-Del
Deletes the financial report group.
