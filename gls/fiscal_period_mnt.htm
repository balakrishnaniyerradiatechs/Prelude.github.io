Fiscal Period Maintenance screen
The Fiscal Period Table contains all valid periods necessary to post accounting data and allows you to maintain accounting records by date without requiring a month-end cutoff. You must construct the fiscal period table before you can post any transactions to the general ledger. You can continue to enter transactions in a new month, even if the old month is still open.
For example, at end-of-month, you can open a new period but still post cash receipts and pay vendor invoices against previous periods. Then, on the 5th of the month, you can change to allow journal entries only, then on the 10th of the month, you can close it completely.
Menu Path: Financial > General Ledger > Processes > End of Month Closing > Fiscal Period Table
Field and Function Key Descriptions
Field/Function Key
Description
Period
The fiscal period.
BeginDte
The beginning date of the fiscal period.
End Date
The ending date of the fiscal period. If you change the date, the next period’s beginning date is automatically adjusted. The change date and the user ID of the person that changed the date display.
For leap year, you would have to change the month end date for February to 02/29.
Open?
The status of the fiscal period for transaction postings:
•
Y - Open for all module transactions to post within the date range.
•
N - Closed. If a transaction attempts to post to a closed period, the next available open period is assigned. If you use multiple currency, refer to Assigning Exchange Rates for Monthly G/L Reports for information on how to update multiple currency.
•
P - Open only for accounts payable transactions and journal entries. If a transaction attempts to post within the period range for any other module, the next available open period is assigned.
•
A - Open only for journal entries. If a transaction attempts to post within the period range for any other module, the next available open period is assigned.
ChngDate
The date the status of the fiscal period changed.
ChangeUser
The user ID of the person that changed the fiscal period.
F2-Save
Saves the changes to the fiscal period and closes the screen.
F4-Del Year
Opens the Delete a Fiscal Year screen to allow you to delete the entire fiscal year from the table.
F5-Create New Year
Opens the Add a Fiscal Period screen to allow you to add the next year to the fiscal period table. The number of fiscal periods per year is determined in Company Maintenance > F5-Add Info. Fiscal Period Table Maintenance appends a YE (year end) period to use for Year-End Closing (refer to End-of-Year Closing Procedures).
F7-Maintain Open?
Opens the Maintain Open? screen to allow you to change the status of all the fiscal periods.
