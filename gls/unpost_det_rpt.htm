Unposted G/L Detail Report
The Unposted G/L Detail report lists debits and credits that did not post to the general ledger. The report sorts by period, by module, with the total debits and credits for each account number. You can use the G/L Balances Posting Routine to post these transactions to the general ledger. Refer to Posting to the General Ledger for more information.
Menu Path: Financial > General Ledger > Reports > Unposted G/L Detail Report
Column Descriptions
Column Heading
Description
Period
The fiscal period of the unposted general ledger transaction. If a T appears (e.g., 07T04), then this transaction will not post to the G/L until the issue is resolved. Contact Activant Prelude Support.
Mod
The general ledger module. Refer to the General Ledger Module Code Definitions for an explanation of each module code.
Document Id
The document number of the associated transaction.
Trn-Date
The date of the transaction.
Account
The general ledger account prefix and account number.
Description
The description of the general ledger account. Refer to Maintaining the General Ledger Chart of Accounts for a full explanation of each account.
Debit
The unposted debit amount for the general ledger account.
Credit
The unposted credit amount for the general ledger account.
