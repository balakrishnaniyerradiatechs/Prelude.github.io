A/P Reconciliation Outstanding Checks Report
Run the A/P Reconciliation Outstanding Checks report after reconciling and updating the bank statement with your records (refer to Reconciling Checks and Deposits) to list checks still outstanding. You can use this report to balance the bank statement to the general ledger.
Menu Path: Financial > General Ledger Bank Reconciliation > Outstanding Checks Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Check Date. The cutoff date for payments to include. All payments generated before or on this date are included.
•
Bank Code. The bank code assigned to the bank. You can include more than one for the report.
Column Descriptions
Column Heading
Description
Check#
The number of the check paid by your company.
Chk-Date
The date the check was generated.
Vendor
The number of the vendor that the check was sent to.
Description
The vendor description.
Check Amount
The amount of the check paid to the vendor.
