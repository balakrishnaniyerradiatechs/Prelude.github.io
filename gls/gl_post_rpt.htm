G/L Summary Posting Report
You can use the G/L Summary Posting Report to re-print any of the general ledger update reports produced in the G/L Balance Posting Routine (refer to Posting to the General Ledger).
Menu Path: Financial > General Ledger > Reports > G/L Summary Posting Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Counter. The counter number assigned to the posting.
•
Posted Date Range. The beginning and ending transaction post dates to include.
Column Descriptions
Column Heading
Description
Counter
The sequential counter assigned to the posting.
Posted Date
The date the transactions posted to the general ledger.
Time
The time of the day the G/L Balance Posting Routine ran.
Account
The general ledger prefix account.
Description
The description of the general ledger account.
Debit Amt
The amount debited to the general ledger account.
Credit Amt
The amount credited to the general ledger account.
