Financial Report Printing screen
The Financial Report Printing screen allows you to define the report parameters and to generate the financial reports created in the Financial Report Master Maintenance screen. After you set up report parameters, you can save them to re-use the information again in the future. Refer to Printing Financial Reports.
Menu Path: Financial > General Ledger > Financial Report Manager >Financial Report Printing
Field and Function Key Descriptions
Field/Function Key
Description
Period
The fiscal period.
Report Name
The name of the financial report.
Ending Date
The ending date of the fiscal period.
Report Parameters
Entity
Determines the information to include on the report. If the Divisions, Regions, Locations, Departments, or Cost Centers flags are set to Y in Hierarchy Maintenance, (refer to Setting up the General Ledger Hierarchy) they are displayed as available entities for the report:
•
COM - Company.
•
DIV - Division.
•
LOC- Location.
•
REG - Region.
•
DEP - Department.
•
CCR - Cost Center.
Options
Determines the type of entities included on the report. Works with Selection. For example, Entity is set to LOC, Option is B, Selection is set to I with locations 10, 20, and 30 listed:
•
C - Consolidated - Generate only a consolidated report that groups all entities. For example, if you have locations 10, 20, and 30, then one report is produced with all 3 locations consolidated.
•
I - Individual - Generate an individual report for each entity. For example, if you have locations 10, 20, 30, three separate reports are generated.
•
B - Both - Generate a report for each entity and a consolidated report. For example, if you have locations 10, 20 , 30, 4 reports are produced; 3 individual reports and one consolidated report.
Selection
Determines the entities to include in or exclude from the report:
•
All - Include all entities. For example, if Entity is location, all locations are included).
•
Include - Opens the Entity Include/Exclude Entry screen to allow you to enter the entities to include. For example if the Entity is location, enter 10, 20, and 30 to only include locations 10, 20, and 30.
•
Exclude - Opens the Entity Include/Exclude Entry screen to allow you to enter the entities to exclude. For example if the Entity is location, enter 10, 20, and 30 to exclude only locations 10, 20, and 30.
If Entity is set to COM (company), this field is not accessible.
Explode Group?
Determines how the report is generated:
•
S - Creates a summary report in which a single line prints displaying the total for the group. Summary is the default if the field is left blank.
•
D - Lists detailed information on the report including each assigned account as a line on the report.
Suppress Comma?
Determines if commas appear when printing amounts:
•
Y - Suppress commas when displaying amounts (e.g., 52865.73).
•
N - Show commas when displaying the amounts (e.g., 52,865.73).
Suppress Zero?
Determines if zeros are suppressed:
•
Y - Does not include zero amounts. The account with zero amounts are left blank.
•
N - Includes zero dollar amounts. The accounts with zero amounts display the zeros (e.g., 0.00).
Report Columns
The columns used on the report. Columns are the financial column codes used to define which accounts are used for reporting (e.g., TMTY - Activity This Mnth This Yr).
Decimal
Determines if decimals appear on the report:
•
Y - Show dollars and cents on the report (e.g., 2356.73).
•
N - Round dollar values for the report (e.g., 2357).
Width
The width of each column on the report. The default width is 13.
F2-Output
Choose how to generate the report.
F6-Inquiry
Displays the Financial Report Inquiry. Press F5-Review on an account detail line with a total to display the Expanded G/L Summary Inquiry.
F7-Load Config
Allows you to use saved report configuration.
F8-Save Config
Allows you to save the report configuration.
F9-Delete Config
Allows you to delete the report configuration.
