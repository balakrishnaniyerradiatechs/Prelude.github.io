Using Multiple Fiscal Periods
If you have multiple locations that share inventory but post to different fiscal periods (e.g., if you acquire a new company that uses different fiscal periods), you can set up a parent account for the corporation and then fiscal accounts for each fiscal year-end month or location.
WARNING Carefully consider using multiple fiscal periods. It is not recommended for most companies.
