Equipment (EQ)
The Equipment Subsystem maintains equipment ID numbers (EIN) along with any associated products, parts lists, and scheduled maintenance. The EINs can be created internally when the Inventory Type Conversion creates an asset record for an item converted to a rental product. When converted, the following postings occur:
•
Inventory is credited.
•
Fixed Assets is debited. The Fixed Assets account is set in Class Maintenance for the asset class from the Rental Product Number.
