Maintaining Report Column Descriptions
Prelude provides a list of report columns to use for financial reports. Each column definition represents a calculation used to show activity, account variance, or balance information. Although you cannot change the formulas, you can use Financial Report Column Maintenance to change the column heading descriptions. When you run the reports (refer to Printing Financial Reports), use these column definitions under Report Columns. The associated descriptions appear as the column headings on your report.
To edit or view report column descriptions:
1
Select General Ledger > Financial Report Manager > Financial Report Column Maintenance to access the Financial Rpt Column Maintenance screen.
A list of Column headings appear.
Abbreviations/Definitions:
•
TM - this month.
•
TQ - this quarter.
•
TY or CY - this/current year.
•
LM or PM - last/previous month.
•
LY or PY - last/previous year.
•
YT or YTD - year-to-date.
•
B1-4 - budget 1, 2, 3, or 4 from Chart of Accounts Maintenance.
•
Basis - the G/L account(s) used to compare to each row on the report.
•
Variance - difference in amount (e.g., dollars).
•
Activity - credits and debits to date for the month or fiscal period.
•
Ending balance - the total of all credits and debits for the YTD ending with that month.
2
Change the Description, Column Heading 1, and/or Column Heading 2 as necessary. Description is for internal reference. Column Headings 1 and 2 appear on the financial reports.
3
Press F2-Save to save any changes.
