Saving Inventory Balances for Month-End
Before you run the Inventory Valuation reports in the Inventory module, you should run the Month-End Inventory Balance Save to save all inventory balances and cost information for a fiscal period. You should only run this process once a month, overnight, on the month-end cutoff date.
To run the Month-End Inventory Balance Save:
1
Select Financial > General Ledger > Processes > End of Month End Closing > Month-End Inventory Balance Save. The following message appears:
This procedure will capture all inventory balance figures as of the point in time this process is run. We save the average cost, replacement cost, standard cost, onhand balance, linked onhand balance and value. This process should only be run at the month end cutoff and should be executed overnight. This process should only be run once a fiscal period. This will allow you to run the inventory valuation reports for a fiscal period end. Continue With Procedure?
2
Choose from the following:
•
No - Cancels the procedure. Returns to the menu.
•
Yes - Opens the Month End Inventory Balance Save GRIM screen.
3
Enter the fiscal Period period that you are closing.
4
Enter the Warehouse to close, or leave blank to close all warehouses.
5
Press ENTER. The following message appears:
Do You Wish To..............
6
Choose from the following:
•
Execute: Begins the month-end save.
•
Schedule: Allows you to schedule the process to run later when nobody is on the system.
•
Save Params: Allows you to save the input parameters to use again later.
•
Return: Returns to the selection criteria.
•
Exit: Cancels the process.
