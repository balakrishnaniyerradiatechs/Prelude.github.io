Chart of Accounts Maintenance
Use Chart of Accounts Maintenance to create a new general ledger account. You can also edit or add locations to an account that exists in General Ledger Table Maintenance.
To create a general ledger account:
1
Select Financial > General Ledger > Maintenance > Chart of Accounts Master to open the Chart of Accounts Maintenance screen.
2
For Account, enter the general ledger account number to create or edit.
General Ledger Account Number Length
The following flags in Company Maintenance > F5-Add Info determine account number length:
G/L Acct # Length. The length of your general ledger account numbers (must be the same for all companies). Twelve is the maximum amount.
G/L# Suppress Zero Fill. Determines if zeros are used to fill in general ledger account numbers to the G/L Acct# Length.
•
Y - Do not fill in zeros for general ledger account numbers.
•
N - Fill in zeros for general ledger account numbers to make it the G/L Acct# Length. For example, if the G/L Acct# Length is 6 and you enter an account number of 2222, it automatically changes to 222200.
3
Enter the Description of the account.
4
For Acct Type, enter the type of account. Choose from the following:
•
Asset.
•
Liability.
•
Capital/Equity.
•
Income/Revenue.
•
Expense.
The income and expense account types are set to zero during the End-of-Year Closing Procedures.
5
Enter the Currency type used for this account type. Defaults to the currency type in Location Maintenance. If this field is left blank, your native currency is used. You cannot change the currency for an account once transactions are posted to it. Refer to Using Multiple Currency for more information on multiple currency requirements.
6
For Fin Rpt Group, if you use the Financial Report Manager, enter the code used to group accounts. These groups are created in Financial Report Group Maintenance (refer to Creating Financial Report Groups).
7
For Interim/Memo A/C, indicate if this account is included in Financial Report Printing if you use the Financial Report Generator: Financial Report Manager is the recommended method:
•
Y - If Financial Report Printing is executed in the interim mode, this account number is included. If Financial Report Printing is executed in the final mode, this account number is excluded.
•
N - This account is included in Financial Report Printing regardless of whether it is executed in the interim or the final mode.
8
Depending on how your general ledger hierarchy is set up, you can add each Division, Region, Location, Department and Cost Center that uses this account. Enter X for a wildcard if the division, region, department or cost center should be taken from the transaction being posted.
9
Edit the Description of the account if necessary.
10
Verify the Status of the account:
•
A - Active. You can post transactions to this account with no restrictions.
•
I - Inactive. You cannot post to this account in Journal Entry, Accounts Payable Invoice Entry, or Vendor Invoice Reconciliation Entry. You may want to mark the account as inactive before deleting it, to make sure that it is no longer necessary.
•
D - Marked for deletion. You cannot post to this account. After an account is inactive for the amount of time determined by your company, you can mark it for deletion.
11
You can also access the following from Chart of Accounts Maintenance:
•
F5-Summary Posting - Opens the Summary of Chart Posting screen to display the debits, credits, net actual amount, and cumulative amounts for each fiscal period.
•
F6-Detail - Opens the Chart Detail Display to display the posting date and debit or credit amount for each transaction posted during a date or fiscal period range.
•
F7-Budgets - Opens the Chart Budget Maintenance screen to allow you to enter up to four budget amounts for each fiscal period for this account. If you set up budgets, you can run reports to compare what is posted to what is budgeted for each account.
•
Action Bar > Module Interface - Opens the Module Interface Options screen. Refer to Interfacing a G/L Account With Other Modules.
12
Press F2-Finish to complete the account information.
