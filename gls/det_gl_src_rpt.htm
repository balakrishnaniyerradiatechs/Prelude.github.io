Detail General Ledger Source Report
The Detail General Ledger Source Report lists general ledger information for a fiscal period, general ledger range, and posting date for all module postings.
Menu Path: Financial > General Ledger > Reports > Detail General Ledger Source Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Period range. The beginning and ending fiscal periods to include.
•
G/L Account Number range. The beginning and ending general ledger account numbers to include. Leave blank to include all periods.
•
Posting Date Range. The beginning and ending posting dates to include:
Column Descriptions
Column Heading
Description
G/L Prefix
The general ledger account number posted to. Refer to Setting up the General Ledger Hierarchy for an explanation of how the prefix is determined for your company.
Mod
The general ledger module posted to. Refer to General Ledger Module Code Definitions for information on general ledger modules.
Inv# Chk
The invoice or check number depending on the module posted to.
Date
The date the transaction posted.
Ref 1
Depending on the module posted to, the customer number, bank code, or part number.
Description
The description for the type of transaction.
Ref 2
Depending on the module posted to, the customer number, release number, or user ID.
Debit Amount
The amount debited to the general ledger account for the module on the post date.
Credit Amount
The amount credited to the general ledger account for the module on the post date.
