Posting to the General Ledger
If you do not automatically post transactions to the general ledger, the G/L Balances Postings Routine allows you to manually post transactions from each module to the general ledger. You can post one module at a time or all at the same time. All transactions that occurred since the last posting are posted. You can use the scheduler to post at night and on a recurring basis so that balances are updated when no one is entering any transactions.
Automatically Posting to the General Ledger
Automatic General Ledger Posting in General Ledger Option Parameters determines when transactions post to the general ledger:
•
Y - Opens the Automatic G/L Posting Module Maintenance screen to allow you to enter G/L modules that should automatically post to the general ledger as each transaction is processed. You cannot use this option if Multiple Fiscal Periods (Company Maintenance > F5-Add Info) is set to Y.
•
A - Audit mode only. Checks for any potential out of balance postings, marks the general ledger posting, and sends a notification to the person indicated in the G/L Email Address. If necessary, notify Activant of out-of-balance conditions before posting to the general ledger. A report generates but nothing posts.
•
N - Transactions are not automatically posted to the General Ledger. You must use the G/L Balances Posting Routine to post transactions from each module to the general ledger.
To post balances to the general ledger:
1
Select Financial > General Ledger > G/L Balances Posting to open the G/L Balance Posting Routine screen.
2
For Requested Module To Post, enter the G/L module codes (refer to General Ledger Module Code Definitions) to post or leave blank to post all modules.
3
Enter the fiscal Period to post.
4
For Print G/L Summary Report, choose from the following:
•
Y - Print the Summary Balance report after all balances are posted (not applicable if using the scheduler).
•
N - Do not print the Summary Balance report.
5
If you are using the scheduler to run this program and you want to notify someone if an error occurs, enter their email address in Email Address Notification.
6
Press ENTER to begin posting.
