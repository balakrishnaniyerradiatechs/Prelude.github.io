Printing Financial Reports
Once you define the financial report information (refer to Designing Financial Reports), use the Financial Report Printing screen to define the report parameters and to generate the reports. After you set up report parameters, you can save them to re-use the information again in the future.
You can also define the print parameters by pressing F8-Print Params in the Financial Report Master Maintenance screen. See Designing Financial Reports.
To print financial reports:
1
Select Financial > General Ledger > Financial Report Manager > Financial Report Printing to open the Financial Report Printing screen.
2
Enter the fiscal Period to use for this report. Ending Date displays the last day of the fiscal period entered.
3
For Report Name, choose from the following:
•
Enter the report name or press F3 to list reports created in Financial Report Master Maintenance. The report description appears.
•
To use a set of report parameters that are already created, press F7-Load Config. Enter the configuration you want to use and press F2-Save to load the configuration. Skip to step 15.
4
For Entity, determine how to group the data on the report:
•
COM - Company - skip to step 7.
•
DIV - Division.
•
LOC - Location.
•
REG - Region.
•
DEP - Department.
•
CCR - Cost Center.
5
In Options, indicate how you want to break up the reports:
•
C - Consolidated - Generate only a consolidated report that groups all entities. For example, if you have divisions 001, 002, 003, then one report will generate with all 3 divisions consolidated.
•
I - Individual - Generate an individual report for each entity. For example, if you have divisions 001, 002, 003, three separate reports will generate.
•
B - Both - Generate a report for each entity and a consolidated report. For example, if you have divisions 001, 002, 003, 4 reports will generate: 3 individual reports and one consolidated report.
6
For Selection, indicate which entities to use for the report:
•
All - Include all entities. For example, if Entity is division, all locations are included).
•
Include - Opens the Entity Include/Exclude Entry screen to allow you to enter the entities to include. For example if the Entity is division, enter 001 and 002 to only include divisions 001 and 002.
•
Exclude - Opens the Entity Include/Exclude Entry screen to allow you to enter the entities to exclude. For example if the Entity is division, enter 001 and 002 to exclude only division 001 and 002.
7
For Explode Group?, choose from the following:
•
Summary - Generate a line with a total for the group (default).
•
Detail - Generate each assigned accounts as a line on the report.
8
For Suppress Comma?, choose one of the following:
•
Yes - Do not use commas for amounts (e.g., 52865.73).
•
No - Use commas for amounts (e.g., 52,865.73).
9
For Suppress Zero?, choose from the following:
•
Yes - Use blank spaces in place of zero values.
•
No - Use the zero values (e.g., 0.00).
10
Enter Report Columns to print on the financial reports. For information on the description of each column and the formula used for each of these column definitions, refer to Maintaining Report Column Descriptions.
11
For Decimal, choose one of the following:
•
Y - Yes - Use decimals (e.g., 2356.73).
•
N - No - Round to a whole number (e.g., 2357).
12
Enter the Width of each column for the report. The width can be different for each column heading. The default is 13.
13
Enter the number of Spaces between each column. The default is 2.
14
To save these printing parameters, press F8-Save Config. If you want to use this configuration the next time you print this report, you can use F7-Load Config.
15
To view the lines and totals before printing the report, press F6-Inquiry to open the Financial Report Inquiry. For more detail, highlight any of the columns and press F5-Review. The Expanded G/L Summary Inquiry appears.
16
Press F2-Output and choose how you want to generate the report.
To view a list of existing financial report parameters, select, Financial > General Ledger > Financial Report Manager > Financial Report Parameter Listing.
