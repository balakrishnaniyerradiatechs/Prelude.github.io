Create New G/L Accounts screen
If you want to create a new general ledger account using the same prefixes for division, region, location, cost center, and department as an existing account, you can use New G/L Account Maint to create the new account. The new account will have a new account number and a new description.
Menu Path: Financial > General Ledger > Maintenance > Create New G/L Accounts
Field and Function Key Descriptions
Field/Function Key
Description
Existing Account Number
The existing general ledge account number to use to create the new general ledger account number.
New Account Number
The new general ledger account number to create.
New Account Description
The description of the new account number.
F2-Save
Creates the new general ledger account number.
