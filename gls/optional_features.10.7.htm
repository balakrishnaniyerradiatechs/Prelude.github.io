Activating Multiple Currency
To set your system to use multiple currency:
1
Select Option Parameters > General Ledger Parameters to open the General Ledger Parameters screen.
2
For Multiple Currency, enter Y to activate multiple currency.
3
For Currency Method, determine where the exchange rate comes from:
•
S - Use the exchange rate from Currency Type Maintenance (refer to Creating Currency Types).
•
D - Use the exchange rate from Daily Currency Exchange Maintenance (refer to Maintaining Currency Exchange Rates). The most recent exchange rate is used. However, if no data exists for the past 9 days, the exchange rate from Currency Type Maintenance applies.
If an exchange rate exists in Vendor Maintenance (refer to Assigning Currency Information to Vendors), it overrides the exchange rate in either Currency Type Maintenance or Daily Currency Exchange Rate Maintenance.
4
For Auto Calc Currency Clrng Amt?, determine how to calculate the amount to post to the currency clearing account in Accounts Payable Manual Invoice Entry:
•
Y - Use the currency clearing account from Currency Type Maintenance to calculate the currency clearing amount (refer to Creating Currency Types).
•
N - Manually calculate the currency clearing amount.
5
For Repl Cost in Vend Currency?, determine how replacement cost displays on a purchase order:
•
Y - Show replacement cost in the vendor’s currency.
•
N - Show replacement cost in your native currency.
6
For Display Currency Rate Screen?, determine if the currency rate screen appears (e.g., Vendor Invoice Reconciliation Entry, Vendor Claim Entry, Stock Receipts Entry, Manual Check Entry) so you can change the currency rate.
•
Y - The currency rate screen appears so you can change the currency rate.
•
N - The currency rate screen does not appear.
7
For Re-eval Invoice Currency?, determine if the currency rate is re-evaluated when an order is invoiced:
•
Y - Re-price sales orders to the currency rate that is in effect at the time of invoicing.
•
N - Do not re-price a sales order when it is invoiced.
8
Press F2-Save to save the parameter information.
