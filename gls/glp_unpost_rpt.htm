Unposted General Ledger Summary Report
The Unposted General Ledger Summary report lists debits and credits that did not post to the general ledger. The report sorts by period, by module, with the total debits and credits for each module. You can use the G/L Balances Posting Routine to post these periods to the general ledger. Refer to Posting to the General Ledger for more information.
Menu Path: Financial > General Ledger > Reports > Unposted G/L Summary Report
Column Descriptions
Column Heading
Description
Period
The fiscal period of the unposted general ledger transaction. If a T appears (e.g., 07T04), then this transaction will not post to the G/L until the issue is resolved. Contact Activant Prelude Support for assistance.
Module
The general ledger module. Refer to the General Ledger Module Code Definitions for an explanation of each module code.
Debit
The unposted debit total for the module.
Credit
The unposted credit total for the module
