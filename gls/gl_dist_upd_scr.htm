G/L Redistribution Update Process screen
The G/L Redistribution Update Process updates the accounts designated in G/L Redistribution Maintenance with the redistributed account balances. You can choose to view the audit report to view the proposed redistribution amounts before redistributing the balances. Refer to the G/L Redistribution Update Process for more information.
Menu Path: Financial > General Ledger > Processes > Redistribution Processes > Redistribution Update
Field and Function Key Descriptions
Field/Function Key
Description
Enter Fiscal Period
The fiscal period for which you want to post the redistribution entries made in G/L Redistribution Maintenance.
Distribution Type
The distribution type for which you want to post all redistribution entries:
•
% - Percentage.
•
P - Proportion.
•
Blank - Post entries for all distribution types.
Perform Update Only?
Indicates if the audit report should print before posting:
•
Y - Post the balance redistribution to the general ledger without previewing the audit report.
•
N - Run the audit report for review and then decide if you want to post the redistribution amounts.
