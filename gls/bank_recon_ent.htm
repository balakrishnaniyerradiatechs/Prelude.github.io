Bank Reconciliation Entry screen
The Bank Reconciliation Entry screen lists outstanding checks and deposits for a bank so that you can balance your checks and deposits with a bank statement (refer to Reconciling Checks and Deposits).
Menu Path: Financial > General Ledger > Bank Reconciliation > Reconciliation Entry
Field and Function Key Descriptions
Field/Function Key
Description
Bank Code
The Bank Code for the bank whose statement you are reconciling. The bank code becomes the prefix for check numbers to make sure they are unique. Bank codes are set up in Bank Code Maintenance on the systems menu.
Cutoff Date
The statement Cutoff Date for which you want to include un-reconciled checks and deposits to reconcile. All checks and deposits entered on or before this date appear.
‘Clr’ Column Date
The default for the Cleared column:
•
Y - Sets the default for Cleared to Y for all checks and deposits for all transactions before the cutoff date.
•
N - Sets the default for Cleared to N for all checks and deposits for all transactions before the cutoff date. You must manually mark the checks and deposits as cleared.
Beg. Balance
The beginning balance from the bank statement.
Other Charges
Any additional charges that the bank issued since the last statement. For example:
•
Service charges - Negative.
•
Interest charges - Positive.
Document
The check or transaction number if the transaction was a payment to a vendor. The batch number if the transaction is a deposit (e.g., B plus the batch number).
Cleared
Determines if the deposit or payment cleared the bank:
•
N - The deposit did not clear the bank.
•
Y - The deposit cleared the bank.
Defaults according to the value in ‘Clr” Column Date.
Description
The vendor number and description if the Document is a payment to a vendor. The description of the batch if the Document is a deposit.
Date
The date of the deposit if the Document is a cash receipts deposit. The date posted to the G/L if the Document is paid to a vendor.
Amount
The amount of the deposit or check.
Deposits
The total deposits that cleared the bank (Cleared set to Y).
Checks
The total checks that cleared the bank (Cleared set to Y).
End. Balance
The running total of the Beg. Balance plus Deposits minus Checks.
F2-Save
Updates all transactions that have Cleared set to Y and reconciles the bank statement with your general ledger.
F5-Edit Report
Allows you to print or view the Bank Reconciliation Edit Report before updating the reconciliation. You can access the report at any time.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
