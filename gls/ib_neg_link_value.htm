Out Of Balance - Negative Linked Value Report
This report lists products that have a negative linked on-hand value.
Menu Path: Financial > General Ledger > Reports > Inventory Balancing > Inventory Balancing Reports - Misc > Out of Balance - Negative Linked Value
Column Descriptions
Column Heading
Description
Whse
The warehouse where the product has a negative value.
Product
The product number with zero on-hand value.
Description
The description of the product.
Onhand
The on-hand balance of the product.
N/S-oh
The linked quantity.
U/M
The unit of measure of the product.
N/S Value
The value of the linked product.
