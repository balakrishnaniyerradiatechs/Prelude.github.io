Chart of Accounts Maintenance screen
You can use Chart of Accounts Maintenance to create a new general ledger account. You can also edit or add locations to an existing account.
Menu Path: Financial > General Ledger > Maintenance > Chart of Accounts Master
Field and Function Key Descriptions
Field/Function Key
Description
Account
The general ledger account number.
Description
The description of the general ledger account.
Acct Type
The type of account to group general ledger accounts for financial reports. Also to let the system know what accounts are zeroed out during year-end and to allow flexibility in numbering chart items:
•
Asset.
•
Liability.
•
Capital/Equity.
•
Income/Revenue.
•
Expense.
Currency
The type of currency used for this account type. Defaults to the currency type in Location Maintenance. If left blank, your native currency is used. You cannot change the currency for an account once transactions are posted to it. Refer to Using Multiple Currency for more information on multiple currency requirements.
Fin Rpt Group
The group codes used for financial reports. The groups are created in Financial Report Group Maintenance (refer to Creating Financial Report Groups).
Interim/Memo A/C
Indicates if this account is included in Financial Report Printing (only if you use the Financial Report Generator. Financial Report Manager is the recommended method):
•
Y - If Financial Report Printing is executed in the interim mode, this account number is included. If Financial Report Printing is executed in the final mode, this account number is excluded.
•
N - This account is included in Financial Report Printing regardless of whether it is executed in the interim or the final mode.
Ln#
The line number to reference the general ledger prefix information.
Div
The division number if your company uses divisions. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division). Enter X for a wildcard if the division should be taken from the transaction being posted
Reg
The region number, if your company uses divisions. Regions are a subset of Division - if Divisions are used (e.g., Southeast Region, Northwest Region). Enter X for a wildcard if the region should be taken from the transaction being posted.
Loc
The location number which could be a warehouse location (e.g., Denver, Dallas) where inventory is kept, or it could be a corporate location where transactions may post to. Enter X for a wildcard if the location should be taken from the transaction being posted.
Dep
The department number if your company uses departments. Departments are a subset of Locations, typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse). Enter X for a wildcard if the department should be taken from the transaction being posted.
Ccr
The cost center number if your company uses departments. Cost centers are a subset of Departments - if Departments are used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales). Enter X for a wildcard if the cost center should be taken from the transaction being posted.
Description
 
Status
The status of the account:
•
A - Active. You can post transactions to this account with no restrictions.
•
I - Inactive. You cannot post transactions to this account in Journal Entry, Vendor Invoice Reconciliation Entry, and Accounts Payable Invoice Entry. You may want to leave it as inactive for a while before deleting it to make sure that it is no longer necessary.
•
D - Marked to be deleted. You cannot post to this account. After an account is inactive for the amount of time determined by your company, you can mark it to be deleted.
Balance
The balance of the account, which is updated as transactions are posted.
F2-Finish
Closes the inquiry screen.
F4-Del
Allows you to delete the account.
F5-Summary Posting
Opens the Summary of Chart Posting screen to display the debits, credits, and net actual amount for each fiscal period.
F6-Details
Opens the Chart Detail Display to display the posting date and debit or credit amount for each transaction posted during a date or fiscal period range.
F7-Budgets
Opens the Chart Budget Maintenance screen to allow you to enter up to four budget amounts for each fiscal period for this account. If you set up budgets, you can run reports to compare what is posted to what you have budgeted for each account.
Action Bar
Module Interface
Opens the Module Interface Options screen. Refer to Interfacing a G/L Account With Other Modules.
