Project G/L Inquiry
The Project G/L Inquiry lists debits and credits posted to general ledger accounts that have Project ID Required set to Y (refer to Setting a G/L Account to Require a Project ID).
General Ledger Project Selection Criteria
To list debits and credits for a general ledger project:
1
Select Financial > General Ledger > Inquiries > Project General Ledger Inquiry to open the Project General Ledger Inquiry GRIM screen.
2
Enter the Beginning and Ending fiscal Period to include in the inquiry or leave blank to list all fiscal periods.
3
Enter the Beginning and Ending Project numbers to include in the listing or leave blank to list all projects (refer to Creating Valid Responses for Project ID Codes.),
4
Enter the Division, Region, Location, Department, and Cost Center as necessary, or leave blank to list all projects.
5
Enter the general ledger Account number to view or leave blank to list all accounts.
Once you select Execute, the Project G/L Inquiry displays the debits and credits to the general ledger accounts for the selected projects.
Field and Function Key Descriptions
Field/Function Key
Description
Account
The general ledger account number if you entered an Account in the selection criteria.
Description
The description of the general ledger account.
Beg Project
The beginning project number if you entered a beginning number in the selection criteria.
End Project
The ending project number if you entered an ending number in the selection criteria.
Period
The fiscal period in which the account was posted to.
Document Reference
The document number associated with the general ledger entry. The document depends on the type of entry:
•
The batch number if the entry was made in Cash Receipts Entry.
•
The journal entry number if the entry was made in Journal Entry.
•
The vendor number and invoice number if the entry is from A/P Manual Invoice Entry
•
The receiver number if the entry is from Vendor Invoice Reconciliation Entry.
Refer to Entry Programs that use Project IDs.
Module
The Prelude module in which the general ledger was posted to:
•
CR (Cash Receipts Entry).
•
JE (Journal Entry).
•
AP (Accounts Payable: Vendor Invoice Reconciliation Entry or A/P Manual Invoice Entry).
Refer to Entry Programs that use Project IDs.
Account
The general ledger account number posted to.
Amount
The debit, credit, or distribution amount posted to the general ledger account.
Project
The ID of the project being tracked. Refer to Creating Valid Responses for Project ID Codes..
F2-Save
Closes the inquiry screen.
F5-Review
Displays detail information for the general ledger entry for the associated project.
