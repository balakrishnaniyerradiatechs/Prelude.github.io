Wty Tags With Open GL - Type R Report
This report lists warranty tag inventory (Type R - return) that is not yet reconciled in Vendor Claim Reconciliation Entry, Warranty Replacement Entry, or Inventory Adjustment Entry. Refer to Vendor Claims and Warranty Replacements in the Inventory documentation for more information. You can view detail open tag information for each product or only the total value of open warranty tag products for each warehouse.
Menu Path: Financial > General Ledger > Reports > Inventory Balancing > Vendor Claim/Warranty Inventory > Wty-Tags With Open GL - Type R
Column Descriptions
Column Heading
Description
Whse
The warehouse with the open warranty inventory tags.
Type
The type of warranty inventory:
•
R - Return to the vendor for credit or replacement.
Product
The product returned to the vendor.
Doc Date
The date the warranty tag was created.
Tag Number
The tag number assigned to the product. Refer to ”Moving Products into Warranty Inventory” in the Inventory documentation for information on assigning a tag number to a product.
Status
The warranty tag status code that represents the last transaction that occurred for the item:
•
A/P - Vendor claim reconciled with the vendor’s credit memo.
•
AUPD - Credit memo processed in Invoice Processing.
•
CLM - Vendor claim entered for the tagged item.
•
IAE - Inventory adjusted to put the item into warranty inventory.
•
R - Tagged item rejected in Stock Receipts Entry.
•
SOE - Credit memo entered to the customer for the tagged item.
•
TE - Stock transfer entered for the tagged item.
•
TR - Stock transfer received for the tagged item.
•
TRR - Inventory adjusted to take the warranty item out of warranty inventory and put it into regular inventory.
•
TS - Stock transfer shipped for the tagged item.
•
VCP - Vendor claim updated.
•
WTR -Warranty replacement entered for the returned item.
Inv Number
The invoice number for the credit memo if the tag was created in a credit memo.
Claim Number
The vendor claim number if the tag was created in a vendor claim.
Wty Qty-Rem
The quantity remaining on the tag. If this value is zero, the inventory was all returned or adjusted out.
Acct-Cost
The accounting cost of the product.
Ext Opn Gl
The total extended value of the open warranty tags on the general ledger (Wty Qty-Rem x Acct-Cost).
