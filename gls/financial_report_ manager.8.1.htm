Financial Report Manager
The Prelude Financial Report Manager allows you to design financial reports (e.g., balance sheets, income statements) using data from the general ledger accounts. You can customize the reports by assigning financial groups to G/L accounts and defining columns for each report. Refer to the following sections for information on setting up and using the Financial Report Manager:
•
Creating Financial Report Groups.
•
Maintaining Report Column Descriptions.
•
Designing Financial Reports.
•
Printing Financial Reports.
You can use wild cards for any G/L hierarchy in Prelude. Enter X in Division, Region, Location, Department or Cost Center to select all available entities related to these fields. For example, enter XX in Region to use all regions.
Financial Report Generator
The Financial Report Manager programs replace the Financial Report Generator programs as the recommended method to create financial reports. You cannot convert reports from the Financial Report Generator to the Financial Report Manager. You must re-create reports in the Financial Report Manager.
