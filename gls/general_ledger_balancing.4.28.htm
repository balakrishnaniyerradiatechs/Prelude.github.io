Work Order (WO)
A work order created in Work Order Entry allows you to build a finished good that consists of multiple components. A bill of material (BOM) is a list of component products used to build the completed item on the work order. A variable work order is entered, issued, and then completed.
When a work order is issued, the material is pulled from the shelf and put in a work area to be worked on or built:
•
Inventory is credited.
•
Work In Process is debited.
Fixed work orders do not require an issues step. The work order is entered and then completed so there are no postings to Work In Process.
When the job is completed and Work Order Completions is run:
•
Work In Process is credited for the components.
•
Inventory is debited for the complete BOM. If your product lines have different general ledger accounts, those are used. Average cost is updated when components are returned to stock (negative issues) and the BOM is completed.
Miscellaneous entries allow burden items to be made against a work order.
•
Absorption from the burden is credited.
•
Work in Process Inventory is debited.
