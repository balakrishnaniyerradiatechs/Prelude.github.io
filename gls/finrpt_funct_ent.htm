Function Operation Entry screen
The Function Operation Entry screen allows you to choose which calculations to use. You can define a calculation for each reference line or a range of reference lines. You can select add, subtract, multiply, or divide.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Master Maintenance > Type = F
Field and Function Key Descriptions
Field/Function Key
Description
Add/Subtract
•
To add a sequential range of lines, enter the beginning and ending line numbers in Range From Ref# and To Ref#. Leave the remaining fields blank.
•
To add or subtract individual lines, enter the first line in Reference # and + for Operator. Enter the second line of the calculation on the next line under Reference # and either (+) to add it to the previous line or (- ) to subtract if from the previous line. Leave the remaining fields blank.
Range From Ref#
The first Ref# line to use for the range of lines to add.
To Ref#
The last Ref# line to use for the range of lines to add.
Reference #
The Ref# line used to add or subtract lines. For example, to subtract line 50 from line 60, enter 60 on the first line under Reference# and set Operator set to (+ ). Enter 50 on the second line and set Operator to (- ).
Operator
Determines the type of operation used for the Ref# lines:
•
(+) - Adds the reference lines.
•
(-) - Subtracts the reference line .
Multiply/Divide
•
To multiply lines, enter Multiplication for Operator and enter the 2 lines to multiply in Reference #1 and Reference #2. Leave the remaining fields blank.
•
To divide lines, enter Division for Operator and enter the line to divide into in Reference #1 and the line to divide by in Reference #2. Leave the remaining fields blank.
Operator
Select the type of operation to use when calculating reference lines:
•
Multiplication - To multiply reference lines.
•
Division - To divide reference lines.
Reference #1
The first reference line to use for the calculation.
Reference #2
The second reference line to use for the calculation.
F2-Save
Saves the calculation method.
