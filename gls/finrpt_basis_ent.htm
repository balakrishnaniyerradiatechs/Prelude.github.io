Financial Report Basis Maintenance screen
The Financial Report Basis Maintenance screen allows you to create basis information for financial reports. For example, you can enter all the sales accounts in Basis. Under the column description %MTH, every detail line and subsequent accumulated fields reports the total as a percentage of sales activity for the current month. Using the %YTH column displays all detail lines reported as a percentage of the sales ending balance.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Master Maintenance > F7-Basis
Field and Function Key Descriptions
Field/Function Key
Description
Ln#
The line count for accounting distribution. For reference only.
Div
Division number. If your company uses divisions, this represents the highest possible grouping in your company (e.g., Eastern Division, Wester Division).
Reg
Region number. A Subset of Division if Division is used (e.g. Southwestern Region, Northeastern Region).
Loc
Location number. The warehouse location that represents where the general ledger posting should post.
Dep
Department number. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
Ccr
Cost Center number. A subset of Department if Department is used. Typically used to separate expenses within a warehouse or department. (e.g., shipping, receiving, outside sales, inside sales).
Account
The general ledger account number to use for the basis for this report line.
Description
The description of the account number.
(+/-)
Determines if addition or subtraction is used for the basis calculation.
F2-Save
Saves the basis account information.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
