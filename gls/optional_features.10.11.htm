Maintaining Currency Exchange Rates
To maintain currency rates daily, use Currency Exchange Rate Maintenance to change the exchange rate for each currency type.
To change the exchange rate for a currency type:
1
Select General Ledger Controls > Currency Maintenance > Daily Currency Exchange Rate Maintenance to open the Currency Exchange Rate Maint screen.
2
Enter the Currency Type to update.
3
For Effective Date, enter the date the currency rate change becomes effective.
4
Enter the Exchange Rate to use for this Currency Type starting on the Effective Date.
5
Press F2-Save to save the exchange rate information.
Currency Exchange Rate Listing
To view or print a list of exchange rates for a date range, select General Ledger Controls > Currency Maintenance > Currency Exchange Rate Listing.
