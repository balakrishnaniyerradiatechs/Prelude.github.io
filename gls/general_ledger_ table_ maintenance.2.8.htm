Interfacing a G/L Account With Other Modules
You can use the Module Interface Options screen to set up a general ledger account to require additional information when it is used with the following:
•
Work orders.
•
Overseas purchase orders.
•
Asset tracking.
•
Project ID Tracking.
To interface a general ledger account:
1
Select Financial > General Ledger > Maintenance > Chart of Accounts Master to open the Chart of Accounts Maintenance screen.
2
Enter the Account number.
3
Select Action Bar > Module Interface to open the Module Interface Options screen.
4
For W/O Required Flag, indicate if a work order number is required when a manual invoice is entered for this general ledger account number:
•
Y - A work order number is required for manual invoices with this account number.
•
N - A work order number is not required for manual invoices.
5
For Oversea Information Required, indicate if a ship number and cost code are required for each line when a manual vendor invoice is entered for this general ledger account:
•
Y - A ship number and cost code are required for manual invoices. You must enter the default Cost Code in the next field.
•
N - A ship number and cost code are not required for manual invoices.
6
Enter the default import Cost Code if Oversea Information Required is set to Y (e.g., freight, duty, brokerage, loading fees). Import codes are created in Import Cost Factor Maintenance to calculate cost factors based on dollars, pounds, or units. Refer to “Maintaining Import Cost Factors” in the Purchasing documentation for information on creating import cost factors.
You can assign these import factors to each overseas vendor in Vendor Maintenance. If necessary, the buyer can override the import factors for a vendor in Purchase Order Entry > F5-Hdr Scr > F5-Add Info > F5-Import Factors. Refer to “Receiving On-The-Water Merchandise” in the Inventory documentation for more information.
7
For Asset Tracking, indicate if the account is used to track assets (e.g., equipment ID numbers, work order repairs or maintenance, rentals, freight):
•
Y - The account tracks assets for repairs or maintenance. A work order uses the asset number instead of a part number so that inventory is not affected. You must enter the Asset Tracking Charge Type in the following field.
•
N - The account does not track assets.
8
If Asset Tracking is set to Y, enter the Asset Tracking Charge Type to expense for the equipment. Choose from the following:
•
C - Capital.
•
A - Add-on capital.
•
F - Freight.
•
MP - Maintenance parts.
•
ML - Maintenance labor.
•
SP - Sublet parts.
•
SL - Sublet labor.
•
O - Other maintenance expense.
•
I - Interest.
•
T - Taxes.
9
For Project Id Required, indicate if a project identification number is required when this general ledger account is entered (refer to Project ID Tracking). Choose from the following:
•
Y - A project ID is required when this account is used.
•
N - A project ID is not required for this account number.
10
Press F2-Save twice to save the interface options.
