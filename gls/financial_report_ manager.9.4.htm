Designing Financial Reports
Financial Report Master Maintenance allows you to design the layout for your financial reports. You can choose which groups to use for the report, how you want to calculate totals, and which account(s) to use for comparison.
To design financial reports:
1
Select General Ledger > Financial Report Manager > Financial Report Master Maintenance to access the Financial Report Master Maintenance screen.
2
Enter the Report Name to modify or add.
3
Enter or change the Description of this report.
Ref# defaults sequentially by increments of 10, which allows you to insert lines between existing lines. For example, to add an account between reference lines 30 and 40, press CTRL+ INSERT to add a reference line of 35.
4
For each line, enter a Type, to define what appears on each line (row) of the report (e.g, a space, a line, a description). Choose from the following:
•
T - Text/space. Enter a description of the line (e.g., Assets, Total Current Assets). Leave blank if you want a blank space.
•
C - Column footer. Enter separators to separate lines. For example, enter - to produce -----------.
•
F - Function. Add, subtract, multiply, or divide lines. For more information see Calculating Lines.
•
D - Detail line. Allows you to enter a description of the line (appears on the left side of the report) and G/L prefix information. After you enter the Description, press ENTER to open the Financial Report G/L Acct Entry screen.
-
If you want to use a financial group, enter the Group Name to include. Refer to Creating Financial Report Groups.
-
For Rev Sign?, determine how to print credits and debits (i.e., Y - credits are positive/debits are negative; N - credits are negative/debits are positive).
-
If you want to add accounts to the financial group indicated, enter the account prefix information. Or, if you did not enter a Group Name, enter the accounts to use for the report.
-
Press F2-Save to save the account information and to return to the Financial Report Master Maintenance screen.
F5-Preview
To view what your report will look like as you create it, press F5-Preview to see how your descriptions, spaces, and lines will appear.
5
To calculate a line based on another line or group of lines, press F7-Basis. For more information see Choosing Basis Accounts.
6
To enter a title for the report, press F6-Titles and enter the title.
7
If you want to set up printing parameters for this report, press F8-Print Params. When you print the report (refer to Printing Financial Reports), after you enter the report name, go directly to F2-Output to generate the report using these parameters.
8
Press F2-Save to save the report information.
