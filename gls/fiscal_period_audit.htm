Viewing Fiscal Period Changes
The Fiscal Period Maintenance Audit Record lists the date, time, and user ID for each change made to the status of a fiscal period in Constructing the Fiscal Period Table.
 
Menu Path: Financial > General Ledger > Inquiries > Fiscal Period Maintenance Audit Inquiry
Field and Function Key Descriptions
.
Field/Function Key
Description
Audit Date
The date the fiscal period was changed.
Audit Time
The time the fiscal period was changed.
Period Audit Message
The explanation of the change to the fiscal period.
User
The user ID of the person that changed the fiscal period.
F2-Save
Closes the screen.
