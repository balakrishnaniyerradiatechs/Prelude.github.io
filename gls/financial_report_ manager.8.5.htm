Calculating Lines
When you enter a function line (type F) in Financial Report Master Maintenance (refer to Designing Financial Reports), the Function Operation Entry screen appears.
For each line, you can either add and subtract or multiply and divide to calculate the value to appear on the report. Refer to the following:
Adding and Subtracting Lines
To add a sequential range of lines:
1
Enter the first line in the range in Range From Ref# and enter the last line in the range in To Ref#.
For example, to add lines 30 through 60, enter 30 in Range From Ref# and 60 in To Ref#.
2
Press F2-Save.
To add or subtract non-sequential lines or to subtract lines:
1
In Reference#, enter the number of the line to begin calculations with.
2
For Operator, indicate whether you want to add (+) or subtract (-).
3
Repeat steps 1-2 for each line in the calculation. For example, to subtract line 50 from 60, enter 60 in Reference# with Operator set to +, then enter 50 in Reference# with Operator set to -.
4
Press F2-Save to save the calculation method.
Multiplying Lines
To multiply lines:
1
Set Operator to Multiplication.
2
Enter the fist line number to multiply in Reference #1 and the second line number in Reference #2.
For example, to multiply line 20 by line 40, enter 20 in Reference #1 and 40 in Reference #2.
3
Repeat steps 2 for each calculation.
4
Press F2-Save to save the calculation method.
Dividing Lines
To divide lines:
1
Set Operator to Division.
2
In Reference #1, enter the line number you want to divide into. For example, to divide line 50 by line 30, enter 50 in Reference #1.
3
In Reference #2, enter the line number you want to divide by. For example, to divide line 50 by line 30, enter 30 in Reference #2.
4
Repeat steps 2-3 for each calculation.
5
Press F2-Save to save the calculation method.
