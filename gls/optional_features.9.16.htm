Assigning Currency Types to Bank Codes
You must set a currency type for each bank code if you use multiple currency.
To set the currency type for a bank code:
1
Select Accounts Receivable Controls > Bank Code Maintenance to open the Bank Account Code Maintenance screen.
2
Enter the Bank Code.
3
For Currency Ind, enter the currency type to use for the bank account.
4
Press F2-Save to save the currency information.
