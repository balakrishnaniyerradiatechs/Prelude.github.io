Creating Currency Types
You must create a currency type for each country in Currency Type Maintenance. You can then assign general ledger accounts and exchange rates for each type of currency other than native currency.
To create a currency type:
1
Select General Ledger Controls > Currency Maintenance > Currency Type Maintenance to open the Currency Type Maintenance screen.
2
Enter the Currency code to add or edit (e.g., CDN - Canadian dollars, EU - Euros).
3
For Description, enter the name of the country whose currency is used by your company.
4
For Receiving Exch Rate, enter the exchange rate to use when stock is received in Stock Receipts Entry. This rate is used to convert costs on purchase orders to native currency.
5
For Invoicing Exch Rate, enter the exchange rate to use to convert prices on sales orders to native currency at invoicing.
6
For Customs Exch Rate, enter the exchange rate to use to convert duty charges on purchase orders to native currency. If this is blank, the Receiving Exch Rate is used.
7
For each of the G/L Table Headings in the following table, enter the appropriate G/L prefix and currency account.
