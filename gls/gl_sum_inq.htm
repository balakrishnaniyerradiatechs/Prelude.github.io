G/L Summary Transaction Inquiry screen
The G/L Summary Transaction Inquiry lists the document numbers, dates, and amounts related to the posted transactions on the Chart Detail Display.
Menu Path: Financial > General Ledger > Inquiries > Detail General Ledger > F5-Posting Review
Field and Function Key Descriptions
Field/Function Key
Description
Module
The general ledger module the transaction posted to. Refer to General Ledger Module Code Definitions.
Document Reference
The document associated with the transaction. For example, if Stock Receipts (SR) was the module posted, the Document Reference is the receiver number. Refer to General Ledger Module Code Definitions.
TranDate
The date the transaction occurred.
Debit Amount
The amount debited to the general ledger if the transaction posted a debit.
Credit Amount
The amount credited to the general ledger if the transaction posted a credit.
F2-Exit
Closes the inquiry screen.
F5-Review
Displays a review screen for the associated document. For example, if the document is a purchase order or receiver number, the Purchase Order History Display screen appears.
F6-Summary By Tran Date
Allows you to view, print, email, or fax a summary list of the transactions for the selected document.
F7-Dtl Rpt
Allows you to view, print, email, or fax a detailed list of the transactions for the selected document.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
sF5-Prev Group
Returns to the previous page if more than one page of transactions appear. The number of records that display depends on Records to Block in Company Maintenance > Action Bar > GUI Options.
sF6-Next Group
Forwards to the next page if more than one page of transactions appear. The number of records that display depends on Records to Block in Company Maintenance > Action Bar > GUI Options.
