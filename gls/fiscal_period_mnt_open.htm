Maintain Open? screen
The Maintain Open? screen allows you to change the status of a fiscal period. Refer to Constructing the Fiscal Period Table.
Menu Path: Financial > General Ledger > Processes > End of Month Closing > Fiscal Period Table > F7-Maintain Open?
Field and Function Key Descriptions
Field/Function Key
Description
Period
The fiscal period for which you want to change the status.
Open?
The status of the fiscal period:
•
Y - Open for all module transactions to post within the date range.
•
N - Closed. If a transaction attempts to post within the period range, the next available open period is assigned. If you use multiple currency, refer to Assigning Exchange Rates for Monthly G/L Reports for information how to update multiple currency.
•
P - Open only for accounts payable transactions and journal entries. If a transaction attempts to post within the period range for any other module, the next available open period is assigned.
•
A - Open only for journal entries. If a transaction attempts to post within the period range for any other module, the next available open period is assigned.
F2-Save
Saves the fiscal period information and returns to the Fiscal Period Maintenance screen.
