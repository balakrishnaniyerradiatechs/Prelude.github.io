Setting a G/L Account to Require a Project ID
To require a project ID for a general ledger account:
1
In Chart of Accounts Maintenance (refer to Chart of Accounts Maintenance), enter the Account that requires a project ID.
2
From the Action Bar, select Module Interface to open the Module Interface Options screen.
3
For Project ID Required, enter Y so that a project ID is required when this account number is entered (refer to Entry Programs that use Project IDs).
4
Press F2-Save twice to save the project ID information.
