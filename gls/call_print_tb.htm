Trial Balance Report
The Trial Balance report lists credit or debit information for each general ledger account for a fiscal period. If the account is a Balance Sheet account, the balance sheet debit or credit balance is listed. If the account is a Profit and Loss account, the P&L debit or credit balance is listed. This information is the same as the information on the Expanded Trial Balance Inquiry.
Menu Path: Financial > General Ledger > Reports > Trial Balance
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Period to Report. The fiscal period to include.
•
Print Only This Period?. Choose from the following:
-
Y - Run the trial balance for only the fiscal period in Period to Report.
-
N - Run the year-to-date trial balance, up to the fiscal period in Period to Report.
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwest Region).
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Location. A warehouse location (e.g., Denver, Dallas) where inventory is kept or a corporate location where transactions may post to.
•
Sort and Break By Location. Choose from the following:
-
Y - Sort the balances by Location.
-
N - Only sort the balances by account.
•
Suppress Zero Balances?. Choose from the following:
-
Y - Exclude accounts with a zero balance.
-
N - Include accounts with a zero balance.
•
Include Memo Accounts?. Choose from the following:
-
Y - Include interim/memo accounts in the listing if Financial Report Printing is executed in the interim mode (only applies if you use the Financial Report Generator. Financial Report Manager is the recommended method)
-
N - Exclude interim/memo accounts in the listing.
•
(N)ative, (F)non-Native, All. Choose from the following:
-
N - Print listings with native currency only.
-
F - Print listings with non-native currency only.
-
Blank - Print all listings, regardless of the currency type.
•
Currency. If you selected F (non-native currency), enter the currency types.
Column Descriptions
Column Heading
Description
Div
The highest possible grouping in your company (e.g., Eastern Division, Western Division). Refer to Setting up the General Ledger Hierarchy.
Reg
A subset of Division - if Division is used (e.g., Southeast Region, Northwest Region). Refer to Setting up the General Ledger Hierarchy.
Loc
A warehouse location (e.g., Denver, Dallas) where inventory is kept or a corporate location where transactions may post to.
Dep
A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse). Refer to Setting up the General Ledger Hierarchy.
Ccr
A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales). Refer to Setting up the General Ledger Hierarchy.
Account
The general ledger account number.
Description
The description of the general ledger account.
BalSht Debits
The total of balance sheet debits, if the account is a balance sheet account.
Balsht Credits
The total of balance sheet credits, if the account is a balance sheet account.
P&L Debits
The total of profit and loss debits, if the account is a P&L account.
P&L Credits
The total of profit and loss credits, if the account is a P&L account.
