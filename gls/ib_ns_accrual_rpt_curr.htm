Linked Inventory Accrual - Currency Report
This report lists linked inventory that is received but not invoiced. The report also includes the currency rate and the extended value in native currency. Refer to Using Multiple Currency for more information on currency.
Menu Path: Financial > General Ledger > Reports > Inventory Balancing > Currency Reports > Linked Inventory Accrual - Currency
Column Descriptions
Column Heading
Description
Whse
The warehouse where the inventory was received.
P/O #
The purchase order receiver number on which the inventory was received.
Product Number
The product number
Rec-Date
The date the product was received.
Qty-Rec
The quantity received.
Net Cost
The cost of the received product.
U/M
The unit of measure of the product.
Tot Frt
The total amount of freight for the received product.
Tot Rec Ext
The extended value of the received quantity (Qty-Rec x Net Cost).
CRate
The currency exchange rate.
US-Rec Ext
The extended value of the received merchandise in native currency.
