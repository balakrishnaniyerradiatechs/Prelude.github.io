Module Interface Options screen
The Module Interface Options screen allows you to set up a general ledger account to require specific information when it is used with:
•
Work orders.
•
Overseas purchase orders.
•
Asset tracking.
•
Project I.D. tracking.
Menu Path: Financial > General Ledger > Maintenance > Chart Of Accounts Master > Action Bar > Module Interface
Field and Function Key Descriptions
Field/Function Key
Description
W/O Required Flag
Indicates if a work order number is required when a manual invoice is entered for this general ledger account number:
•
Y - A work order number is required for manual invoices.
•
N - A work order number is not required for manual invoices.
Oversea Information Required
Indicates if a ship number and cost code are required for each distribution line when a manual vendor invoice is entered for this general ledger account:
•
Y - A ship number and cost code are required for manual invoices. You must enter the default Cost Code in the next field.
•
N - A ship number and cost code are not required for manual invoices.
Cost Code
The default import Cost Code if Oversea Information Required is set to Y (e.g., freight, duty, brokerage, loading fees). Import codes are created in Import Cost Factor Maintenance to calculate cost factors based on dollars, pounds, or units. Refer to “Maintaining Import Cost Factors” in the Purchasing documentation for information on creating import cost factors.
You can assign these import factors to each overseas vendor in Vendor Maintenance. If necessary, the buyer can override the import factors for a vendor in Purchase Order Entry > F5-Hdr Scr > F5-Add Info > F5-Import Factors. Refer to “Receiving On-The-Water Merchandise” in the Inventory documentation for more information.
Asset Tracking
Indicates if the account is used to track assets (e.g., equipment ID numbers, work order repairs or maintenance, rentals, freight):
•
Y - The account is used to track assets for repairs or maintenance. A work order uses the asset number instead of a part number so that there is no impact on inventory. You must enter the Asset Tracking Charge Type in the following field.
•
N - The account is not used to track assets.
Asset Tracking Charge Type
The Asset Tracking Charge Type to expense for the equipment:
•
C - Capital.
•
A - Add-on capital.
•
F - Freight.
•
MP - Maintenance parts.
•
ML - Maintenance labor.
•
SP - Sublet parts.
•
SL - Sublet labor.
•
O - Other maintenance expense.
•
I - Interest.
•
T - Taxes.
Project Id Required
Indicates if a project identification number is required when this general ledger account is entered (refer to Project ID Tracking):
•
Y - A project ID is required when this account is used.
•
N - A project ID is not required for this account number.
F2-Save
Saves the module interface options and returns to the Chart of Accounts Maintenance screen.
