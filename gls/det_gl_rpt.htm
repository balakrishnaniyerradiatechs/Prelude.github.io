Detail General Ledger Report
The Detail General Ledger Report lists a summary of the debits and credits posted to each general ledger module for a range of fiscal periods. You can view the beginning balance, ending balance, and summary of postings for each period. The report is sorted by period, module, posting date.
Menu Path: Financial > General Ledger > Reports > Detail General Ledger Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Period range. The beginning and ending fiscal periods to include.
•
G/L Account Number range. The beginning and ending general ledger account numbers to include. Leave blank to include all periods.
•
Sort By Prefix Or G./L. The sort order for the report:
-
P - Sort by general ledger prefix.
-
G - Sort by general ledger account number.
Column Descriptions
Column Heading
Description
G/L Acct#
The general ledger account number posted to.
Prefix
The general ledger prefix number. Refer to Setting up the General Ledger Hierarchy for an explanation of how the prefix is determined for your company.
Description
The description of the general ledger account. Refer to the Default Chart of Accounts for a complete description of the account.
Period
The fiscal period in which the module posted.
PostDate
The date the module posted to the general ledger.
Document#
The document number if the transaction is a journal entry.
Src
The general ledger module posted to. Refer to the General Ledger Module Code Definitions for a complete list and explanation of modules.
Debits
The amount debited to the general ledger account for the module on the post date.
Credits
The amount credited to the general ledger account for the module on the post date.
Balance
The account balance at the end of the selected fiscal period.
