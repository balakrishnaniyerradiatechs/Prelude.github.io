Creating Valid Responses for Project ID Codes.
When you enter a general ledger account number that requires a project ID (refer to Entry Programs that use Project IDs), you must enter a valid project ID.
To create valid responses for project tracking:
1
In System Parameters select Maintenance > Miscellaneous Controls > Valid Response Maintenance to open the Valid Responses Maintenance screen.
2
For Validation Code, enter GL.PROJECT.ID.
3
Under Response, enter the project code to use whenever a G/L account number is entered that requires a project ID.
4
Enter the Description of the ID to display any time the valid response code is entered.
5
For Abbr., enter an abbreviation of the response to use instead of the response code (optional).
6
Repeat steps 3-5 for each code to add.
7
Press F2-Save to save the response code information.
