Duplicating a Single G/L Account
To create a new general ledger account using the same prefixes for division, region, location, cost center, and department as an existing account, you can use New G/L Account Maint to create the new account. The new account will have a new account number and a new description.
To create a new G/L account from an existing account:
1
Select Financial > General Ledger > Maintenance > Create New G/L Accounts to open the Create New G/L Accounts screen.
2
Enter the Existing Account Number that you want to duplicate.
3
Enter the New Account Number that you want to create.
4
Enter the New Account Description.
5
Press F2-Save to create the new account number that will use the same prefixes as the existing account.
