Financial Report Master Maintenance screen
The Financial Report Master Maintenance screen allows you to design the layout for your financial reports. You can choose which groups to use for the report, how you want to calculate totals, and which account(s) to use for comparison. Refer to Designing Financial Reports.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Master Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Report Name
The report name of the financial report.
Description
The description of the financial report.
Type
The type of line to use for this report:
•
T - Text/space. A description of the line (e.g., Assets, Total Current Assets). Leave blank if you want a blank space.
•
C - Column footer. Separators (e.g., -, = *). For example, enter - to produce --------------.
•
F - Function. Add, subtract, multiply, or divide lines. For more information see Calculating Lines.
•
D - Detail line. A description of the line (appears on the left side of the report) and G/L prefix information. After you enter the Description, press ENTER to open the Financial Report G/L Acct Entry screen so that you can enter the financial group or account numbers.
Ref#
Sequential increments of 10. You can insert lines.
Description
The description of the reference line such as text, general ledger account groups, dash, or calculation.
F2-Save
Saves the financial report information.
F4-Delete
Deletes the financial report.
F5-Preview
Allows you to preview the financial report.
F6-Titles
Displays the Report Title Maintenance screen, which allows you to enter text that prints as the title of the financial report. See Printing Financial Reports.
F7-Basis
Allows you to select the account that is the basis for the line of the report. For more information, see Choosing Basis Accounts.
F8-Print Params
Displays the Report Printing Parameters screen, which allows you to set up the report parameters for the financial report.
