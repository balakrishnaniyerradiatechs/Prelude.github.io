Expanded G/L Detail Inquiry screen
The Expanded G/L Detail Inquiry screen displays each document number, general ledger module, post date, transaction date and period, and debit and credit amount for the highlighted account.
Menu Path: Financial > General Ledger > Inquiries > Expanded Trial Balance Inquiry > F5-Review
Menu Path: Financial > General Ledger > Inquiries > Expanded Detail General Ledger Inquiry > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
Div
Division number. If your company uses divisions, this represents the highest possible grouping in your company (e.g., Eastern Division, Wester Division).
Reg
Region number. A Subset of Division if Division is used (e.g. Southwestern Region, Northeastern Region).
Loc
Location number. The warehouse location that represents where the general ledger posting should be posted.
Dep
Department number. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
CCr
Cost Center number. A subset of Department if Department is used. Typically used to separate expenses within a warehouse or department. (e.g., shipping, receiving, outside sales, inside sales).
Account
The general ledger account number.
Description
The general ledger account description.
Document
The transaction document number.
Module
The general ledger module the document posted to. Refer to the General Ledger Module Code Definitions for a complete description of each code.
Post Date
The date the general ledger account was posted to.
TransDate
The transaction date of the document.
Period
The fiscal period for which the report was run.
Debit Amt
The dollar amount debited to the general ledger account.
Credit Amt
The dollar amount credited to the general ledger account.
Totals
The totals of all the debits and credits.
F2-Finish
Closes the Expanded G/L Detail Inquiry screen.
F5-Review
Displays the inquiry screen based on the type of module the account posted to. For example, if ST Stock Transfers displays as the module, F5-Review displays the Transfer History Inquiry.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are order number, page number, line number, B (bottom), and T (top).
