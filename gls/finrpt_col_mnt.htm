Financial Rpt Column Maintenance screen
Prelude provides a list of report columns to use for financial reports. Each column definition represents a calculation that shows activity, account variance, or balance information. Although you cannot change the formulas, you can use Financial Report Column Maintenance to maintain the column heading descriptions. When you run the reports (refer to Printing Financial Reports), you enter these column definitions under Report Columns. The associated descriptions appear as the column headings on your report. Refer to Maintaining Report Column Descriptions for more information.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Column Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Column
The column types for the financial reports. Refer to Column Descriptions/Calculations for an explanation of each code.
Description
The descriptions of the column to appear on the financial reports.
Column Heading
The column headings:
•
Column Heading 1 - The text that appears at the top of each report as the first line of description.
•
Column Heading 2 - The text that appears at the top of each report as the second line of description.
F2-Save
Saves the column headings.
