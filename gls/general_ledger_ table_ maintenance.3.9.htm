Changing a G/L Account Description
If you want to change the description of an account without using Chart of Accounts Maintenance, you can use Chart of Account Description Maintenance to edit the description only. You may want to use this for employees that you do not want to have access to Chart of Accounts Maintenance.
To change the description of a general ledger account:
1
Select Financial > General Ledger > Maintenance > Chart of Account Description to open the Chart Of Account Description Maintenance screen.
2
Enter the Account for which you want to change the description.
3
Edit the Description as necessary.
4
Press F2-Save to save the account description information.
