Assigning Default G/L Currency Types
Default currency types for new accounts (created in Chart of Accounts Maintenance) can be set up in Location Maintenance.
To set a default currency type:
1
Select General Ledger Controls > Location Maintenance to open the Location Address Maintenance screen.
2
Enter the Division, Region, and Location information as required by your company (refer to Setting up the General Ledger Hierarchy).
3
For Account Type, choose from the following:
•
A - Asset.
•
C - Capital/Equity.
•
E - Expense.
•
I - Income/Revenue.
•
L - Liability.
4
Enter the default Currency Type to use for each account type in Chart of Accounts Maintenance. For example, if Location 20 has an Account Type of A and a Currency Type of EUR, then any asset account created in Chart of Accounts Maintenance will have a default Currency Type of EUR.
5
Press F2-Save to save the currency information.
