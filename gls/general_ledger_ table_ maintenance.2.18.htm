G/L Redistribution Update Process
After you enter the general ledger redistribution information (refer to G/L Redistribution Maintenance), you can run the G/L Redistribution Update Process to update the designated general ledger accounts with the new balances.
If you want to view the distribution information before you complete the update, select Financial > General Ledger > Processes > Redistribution Process > Redistribution Listing to list the entries made in G/L Redistribution Maintenance.
To complete the general ledger redistribution:
1
Select Financial > General Ledger > Processes > Redistribution Process > Redistribution Update to open the G/L Redistribution Update Process screen.
2
For Enter Fiscal Period, enter the fiscal period for which you want to post the redistribution entries made in G/L Redistribution Maintenance.
3
For Distribution Type, enter the distribution type for which you want to post all redistribution entries. Choose from the following:
•
% - Percentage.
•
P - Proportion.
•
Blank - Post entries for all distribution types.
4
For Perform Update Only?, indicate if the audit report should print before posting. Choose from the following:
•
Y - Post the balance redistribution to the general ledger and bypass the audit report.
•
N - Generate the audit report . The following message appears:
Do You Wish To Continue With Update to G/L With The Redistributions Calculated On This Report?
5
Choose from the following:
•
N - Do not redistribute amounts to the general ledger.
•
Y - Redistribute amounts to the general ledger.
G/L Redistribution Update Proof Report
Column Heading
Description
G/L Prefix
The general ledger prefix number.
Account
The general ledger account number that will be redistributed to other accounts.
Description
The description of the general ledger account.
Type
The type of re-distribution:
•
% - Percent. Distributed based on a percent to each account.
•
P - Proportion. Distributed based on a proportion of another account.
Description
The explanation for the redistribution.
Balance
The balance of the account that will be redistributed.
Distribution Acct.
The accounts that received the redistribution.
Description
The description of the accounts that received the redistribution.
Percent
If the distribution Type is %, the percent of the redistribution received by each account.
Proportion Acct.
If the distribution Type is P, the account proportioned from.
Description
The description of the Proportion Acct.
Allocation
The amount of the general ledger balance distributed to the new accounts.
