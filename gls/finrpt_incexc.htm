Entity Include/Exclude Entry screen
The Entity Include/Exclude Entry screen allows you to include or exclude entities (e.g., company, division, location, region, department, cost center) from the report that you are printing in Financial Report Printing. Refer to Printing Financial Reports.
Menu Path: Financial > General Ledger > Financial Report Manager > Financial Report Printing > enter I or E for Selection
Field and Function Key Descriptions
Field/Function Key
Description
Entity
The general ledger entity selected for the report (e.g., company, division, location, region, department, cost center).
Selection
The entities to include or exclude. For example if the Entity is location, enter 10 and 20 to exclude/include only locations 10 and 20.
F2-Save
Saves the include/exclude selection and returns to the Financial Report Printing screen.
