Creating Journal Entries
A journal entry is a manual adjustment to a general ledger entry (e.g., year-end adjustments, corrections to general ledger accounts). If the same account must be adjusted each month, you can set up a recurring journal entry to adjust the same amount for a range of fiscal periods.
