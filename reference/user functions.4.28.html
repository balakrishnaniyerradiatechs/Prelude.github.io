Using Work Queues and Alerts
You can set your user ID to determine how you receive alerts when an issue requires your attention (e.g., credit holds, invoice holds, emergency ordering requirements). For example, you can set the Credit Manager’s user ID to have access to the credit hold queue, the buyer to have access to a buyer queue, and a Sales or Branch Manager to have access to administrative order queues. Each salesperson or clerk can have access to individual work queues according to their user ID and warehouse settings.
Prelude uses the following three methods to send out alerts and work queue information:
•
Toolbar Alerts and Inquiries.
•
Prelude Notify.
•
E-Mail Alerts.
