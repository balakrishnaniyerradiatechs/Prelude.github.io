Warehousing
The Warehousing module provides an efficient method to process inventory transactions. A real time environment allows you to view the on-hand, committed, and available status of each bin. You can use Warehousing for:
•
Bin replenishment: Occurs when Nightly processing detects a bin level below minimum. Warehousing Maintenance flags determine the type of bin to replenish, the type of bin to replenish from, and in what sequence the bins are used
•
Bin level allocation: Analyzes committed information from sales orders, transfers, work orders, and bin replenishment to determine the available bin quantity. Once inventory in a bin is completely committed, the next available bin is committed for subsequent transactions.
•
The bin queue: Maintains the information in a logical sequence for picking. Once you request transactions to process, a report prints with the product information.
The Warehousing module also gives you alternative options to use radio frequency (RF) units for the following:
•
Sales Order Picking / Shipping: Sales orders can be picked according to warehouse layout with a greater level of accuracy. Radio Frequency Release Confirmation Packing allows you to select a release for confirmation.
•
Stock Receipts & Putaway: An RF unit can be used for stock receipts, depending on whether or not the products are bar coded. Putaway by purchase order indicates where to put the orders.
•
Cycle Counts: The RF cycle count entry shows the counter what to count and records the counts entered.
•
RF- Alternate Part Number Maintenance: Products can be set up with alternate product numbers and corresponding UCC codes which can be maintained using RF devices.
•
Transfer Picking / Shipping: RF transfer picking is used to pick products from pick and/or bulk bins. The products are accumulated on a single transfer until the shipment is processed.
•
Work Order Picking / Completion: RF work order picking is used to pick products from bins in a logical picking sequence.
•
Wrist Picking - Batch Processing: Wrist Picking RF devices combine multiple steps to simplify the process of sales order picking and shipping confirmation. Batch processing prepares orders for picking. The wrist picking RF devices combine the picking process and serial ship container entry steps into one process.
