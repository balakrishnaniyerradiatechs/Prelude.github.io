Using Slash Commands
In addition to the mouse and hot keys, you can use slash commands for direct access to a program. Slash commands (also called processes) are shortcuts that allow you to jump to another screen or process from anywhere in the system. To use a slash command type a / (forward slash) followed by the process name. Most slash commands are acronyms or abbreviations of the menu item. For example, to get to the Sales Order Entry screen, type /SOE; to get to Purchase Order Inquiry, type /POI.
If you do not know the slash command for a menu item, highlight the item on the menu, type /MS, and press ENTER. The Process Name Display screen appears.
Process Name is the slash command that you can use to access that menu item.
Combining Slash Commands and Input Criteria for Quick Access
A slash command may be followed by an input string. For example, to do an Inventory Inquiry for product 4CP16 in warehouse 10, type /II 4CP16 10 and press ENTER. The Inventory Inquiry screen appears with information for product 4CP16 in warehouse 10.
This does not work if you have spaces within your product numbers. It takes the first word of the product and does a keyword search, ignoring the rest of the input.
