E-Mailing Documents
You can set up vendors and customers to receive documents (e.g., reports, invoices, and purchase orders, confirmations) by e-mail. If you choose the email option, the Email Information screen appears.
For emailing to work properly, you may need to update your user forms. For more information, see your System Administrator.
 
 
If you e-mail an order confirmation, the E-mail Confirmation screen appears. You can set a default email address for vendors and customers that receive e-mail confirmations.
To email a document:
1
Change Email From, if necessary. Defaults to the e-mail in your user ID.
2
Enter as many addresses as necessary in Email To.
3
Enter a Subject.
4
Enter Coverpage Notes which will appear as the body of the email.
5
For File Type, enter the file extension for the attachment:
•
TXT - Opens in Microsoft® Notepad.
•
HTM - Opens as a webpage.
•
RTF - Opens in Microsoft Word or Microsoft WordPad.
•
PDF - See your System Administrator before using this option.
6
For Format, choose the stationery type for the attachment:
•
P - Portrait.
•
L - Landscape.
•
U - User-defined. Typically used to send data to another computer.
7
For File Name, enter the name of the attachment. If you do not enter a name, Prelude generates a name.
8
Press F2.
