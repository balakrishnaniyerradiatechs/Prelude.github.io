Viewing Report Output
The first 3 lines of each report contain the same type of information:
•
Line 1 - Company name, user ID, page number.
•
Line 2 - Report name, date, time.
•
Line 3 - Selection criteria.
The body of the report contains columns of data. Refer to the Prelude documentation manuals or the Help keys for an explanation of each column in each report.
