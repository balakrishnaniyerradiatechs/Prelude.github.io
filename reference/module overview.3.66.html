Using Multiple Currency
If your company uses more than one type of currency to post transactions, you can set your system to handle multiple foreign currency rates. You can maintain the currency rates and assign currency types and values to customers, vendors, and bank codes. The Monthly General Ledger Reports also reflect currency information.
If you use multiple currencies, the currency type, currency rate as of the transaction date, and the currency source are saved with each transaction. Entry programs that use currency information include:
•
Purchase Order Entry.
•
Stock Receipts Entry.
•
Vendor Invoice Reconciliation Entry.
•
Manual Invoice Entry.
•
Accounts Payable Payment Selection/Print.
•
Manual Check Entry.
•
Sales Order Entry.
•
Cash Receipts Entry.
•
Service Charge Processing.
