Work Orders (Bill of Material)
A bill of material (BOM) is the combination of products used to build a finished good using a work order. Once you create a BOM, you can modify it or replace the components if necessary. The following types of BOMs are available:
•
Variable Bill of Material: In a variable BOM, you can add, change backorder, or delete products. Work Order releases print for available components and show what is actually used to create the finished good. The components are marked as work-in-process until the work order is complete. Actual components and quantities are issued and the completion process updates what was issued. You can determine if the cost of the finished goods updates actual or standard cost.
•
Fixed Bill of Material: In a fixed BOM, the components and quantities do not change. Once you enter and print the work order, inventory is adjusted and standard cost is updated. Fixed BOMs do not include a work-in-process stage or an issue process, which makes editing a BOM more difficult.
