Running End-of-Month, End-of-Year Procedures
The following procedures are necessary to record and track sales, product, and inventory level information:
•
End-of-Month Procedures: The End-of-Month Calendar Close updates forecasts, line points, order points, EOQ, and FIFO information. It also determines unusual usage and notifies buyers of any product whose purchase parameters were not recalculated.
You must run the calendar close every month to keep history and performance records up-to-date. You can run month-end more than once if necessary, but be sure to use the same month-end closing date each time you run it for the same month. If at all possible, run the process when very few or no users are on the system so that the product information is accurate.
•
End-of-Year Procedures. You must run the End-of-Year Procedure at the end of the calendar year to roll your current year’s data into the previous year. The generated information is not related to your general ledger or sales analysis data.
