Physical Inventory
Whether you use count sheets, tags, or radio frequency devices to take your physical inventory, Prelude provides a set of procedures to accurately count, record, verify, and update the new on-hand balances into inventory.
•
Count Sheets: The Inventory Sheet Procedure provides a method to perform a physical inventory using count sheets to effectively record the count for each item in your warehouse. After you enter the new counts, you can run the Physical Inventory Deviation Exception Report to compare the system quantity to the count quantity. Once you verify the counts, the Physical Update Procedure adjusts the on-hand quantity of each item by the difference between the saved on-hand amount and the counted amount.
•
Tags: The tag inventory procedure creates a tag for each product for its primary bin. If a product exists in more than one bin, a tag also prints for any additional bins that contain a quantity of the product. As you count each product, you can pull off the tag and write in the count. As you count each area, you can accumulate the tags, put them in numerical sequence, and enter the counts into the system. You can also create manual tags for any product/bin combination without a tag.
•
Radio Frequency Guns: If you use RF guns you can enter physical inventory counts without printing count sheets or tags.
