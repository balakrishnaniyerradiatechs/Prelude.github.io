Modifying Selection Criteria
 
You may or may not be able to modify selection criteria based on your user ID. If you cannot modify selection criteria, but would like to, see your System Administrator.
You can use GRIM to modify selection criteria and then save that criteria. To modify selection criteria:
1
On the main report screen, press F5. The Modify General Report Interface Parameters screen appears.
2
Enter the Dictionary Name that you want to add or press F3 to search.
3
Change the Description if necessary.
4
For Oper, choose an operator:
•
= - Equal (default).
•
# - Not equal.
•
> - Greater than.
•
< - Less than.
•
>= - Greater than or equal to.
•
<= - Less than or equal to.
5
For Mult, choose one of the following:
•
Y - Allow multiple values to be entered (default).
•
N - Allow only one value to be entered.
6
For Man, choose one of the following:
•
Y - Make this field mandatory.
•
N - Make this field optional (default).
7
For Cnv, accept the default conversion code.
8
For Sel, choose one of the following:
•
Y - Use this criteria to select data (default).
•
N - Do not use this criteria to select data.
9
For W, choose one of the following:
•
B - Allow brackets at the beginning and end of a string as wildcards (e.g., [text]).
•
L - Allow brackets at the left (beginning) of the string (e.g., [text).
•
R - Allow brackets at the right (end) of the string (e.g., text]).
•
N - Do not allow wildcards (default).
10
Repeat steps 2-9 for each Dictionary Name you want to add.
11
Press F2. Your changes appear in the GRIM screen.
