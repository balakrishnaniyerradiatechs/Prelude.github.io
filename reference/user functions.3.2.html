Using Prelude On-Line Documentation
Prelude provides user documentation in the following forms:
•
User Manuals for each module which provide detailed step-by-step, field-by-field instructions of each procedure. The manuals include screen examples, report examples, inquiry definitions, a table of contents, and an index.
•
Activant Customer Web Site (CWS):
-
Click Prelude > Documentation.
-
Select the module and section to view or click Search or Index to search by topic or procedure.
•
On-line Help For Each Screen. From most Prelude screens, click Help from the action bar to open a web page with information about the screen you are in. From this page you can access topics related to the screen and all other information in the module. Use Contents to see all the available information included in the module. Use Search to look up topics, words, procedures, reports, inquiries or any other required information.
