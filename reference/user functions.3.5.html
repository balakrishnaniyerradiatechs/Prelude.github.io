Understanding the Security Hierarchy
Prelude uses the following four levels of system security:
1
Unix Security:
•
Establishes login information.
•
Establishes groups.
2
System Builder Group Security:
•
Sets shell access.
•
Sets login times.
•
Determines login times.
•
Sets menu and process restrictions.
3
System Builder User Security:
•
Assigns a group.
•
Resets passwords.
•
Sets keyboard time-outs.
•
Assigns terminal and printers to users.
4
Prelude User Security:
•
Determines functions each user can perform.
•
Maintains personal information.
•
Sets system defaults.
•
Determines access to manager work queues.
