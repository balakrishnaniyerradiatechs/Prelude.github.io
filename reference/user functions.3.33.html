Entering Selection Criteria
When you select a report to run, the standard GRIM screen appears.
 
Only hierarchy levels that your company uses appear in GRIM screens. For example, if your company does not use division number but does use cost center number, Cost Center Number appears instead of Division Number in the screen above.
To run a report:
1
Enter selection criteria in any or all fields.
•
If you wish to use all available criteria for a field (e.g., all divisions or locations), leave that field blank. If data is mandatory, a warning dialog box appears.
•
To select multiple criteria, separate the data with commas. For example to view results for warehouse 10 and 20, enter 10,20 for Warehouse Number.
•
Put your cursor in the field you want to search and press F3 to search for criteria, if available.
•
Use brackets ([ ]) as wild cards, if allowed. For example, to lookup P/Os that begin with 123, enter 123].
2
After entering the selection criteria, press F2 to run the report. The following dialog box appears.
3
Choose one of the following:
•
Schedule to schedule a report. Refer to Scheduling Reports.
•
Save Params to save report criteria. Refer to Saving Report Parameters.
•
Return to return to the GRIM screen and change criteria.
•
Exit to close the GRIM screen and return to the menu without running the report.
•
Execute to run the report. The following prompt appears.
4
Choose an output option:
•
Scrn - Displays the report on your desktop.
•
Print - Sends the report to a printer.
•
Email - Emails the report to the email address(es) you specify. For more information on e-mailing, refer to E-Mailing Documents.
•
Fax - Faxes the report to the fax number you specify. For more information on faxing, refer to Faxing Documents.
•
Exit - Closes GRIM without generating the report.
To use Aux, Dif, File, or Bkgrd, see your System Administrator. You may also have the option to export to Microsoft® Excel. See your System Administrator for more information.
