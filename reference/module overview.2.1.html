Prelude Module Overview
The Prelude distribution software package created by Activant can help you fully integrate all inventory, purchasing, sales, and financial requirements in your company. The immediate transfer of information between departments and flexible on-line inquiries into daily transactions allow you to access up-to-the-minute information on your operations.
This chapter gives a brief description of each module in Prelude. Refer to the Activant Prelude documentation manuals and on-line help for detailed, step-by-step, information and procedures on each module.
