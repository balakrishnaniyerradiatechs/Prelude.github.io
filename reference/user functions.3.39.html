Printing Documents
Prelude provides two ways to print documents:
•
Continuous - documents that are set to print continuously are printed as soon as they are ready. For example, if your pick tickets are set to print continuously and the criteria are met, pick tickets print when you select Pick Ticket.
•
Batch - documents that are set to batch print are held until a process is run to print them. For example, if your pick tickets are set to batch print, they go into a queue when you select Pick Ticket. The pick tickets are held in the queue until you run Pick Ticket Printing.
Although Prelude uses the term “print,” most documents can be faxed (refer to Faxing Documents) and e-mailed (refer to E-Mailing Documents) both continuously and by batch. For example, if a customer is set to have invoices faxed, when you print invoices, the customer receives a fax.
Other documents, such as purchase orders, allow you to print, fax, and/or email when you save the order.
The first option provided is the customer or vendor’s preferred method of receiving the document; however, you may choose one of the other options.
