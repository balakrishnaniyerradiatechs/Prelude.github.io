Daily Processing
Daily Processing performs the same calculations and recommendations as Nightly Processing, except that you must specify each vendor and warehouse. All recommended transactions for the vendors listed are re-evaluated and regenerated to update the buyer queues.
You may want to use Daily Processing to generate recommended purchase orders, transfers, and work orders during the day, after you correct any errors generated from Nightly Processing (e.g., assigning primary vendors). You can run this program manually using the Daily Processing option, or you can schedule it to run every day using the Daily Processing - Scheduled option.
