Using Saved Report Parameters
To use saved report parameters:
1
On the GRIM screen, press F6. The Report Parameter Load screen appears.
x
2
In Parameter Name, enter the name of the saved report parameters you want to use or press F3 to search. Continue with the report as usual.
 
You can also run a report with saved parameters by typing /<report.name>,<parameter.name>. For example, if you saved parameters named DIV1 for the Monthly Invoice Register Report, type /MLY.INV.REG.RPT,DIV1 to open the GRIM screen and load the parameters.
