Marketing
The Marketing module provides a wide range of features and options to help track interactions with current and prospective customers. System options allow you set up data and codes specific to your business. Marketing features allow you to:
•
Record Marketing Activities: A marketing activity is a process that records events for a customer or prospect that you want to track (e.g., site visits, phone calls, product demonstrations, form letters, and other literature).
•
Create Promotion Campaigns: Marketing campaigns allow you to define a set of customers and prospects that will receive promotional material and special pricing. You can measure the effectiveness of the campaign by identifying which sales are a result of the promotions.
•
Email/fax to Multiple Customers/Prospects. You can email or fax letters (including form letters) to a list of customers and/or prospects based on a list you create. You can also export a previously created list.
The Marketing module works with CRM to manage customer/prospect relationships.
