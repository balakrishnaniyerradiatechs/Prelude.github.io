Vendor Currency Screen
The Vendor Currency screen allows you to enter a currency code and an override exchange rate for the vendor.
Menu Path: Purchasing > Maintenance > Vendor Maintenance > F5-Add Info > F8-Currency
Field and Function Key Descriptions
Field/Function Key
Description
Currency
The currency code for the type of currency used by this vendor. If you leave this field blank, native currency is used.
Override Exch Rate
The currency override exchange rate negotiated for this vendor if the Currency for the vendor is other than native currency. This rate overrides the exchange rate set for the currency type in Currency Type Maintenance (General Ledger Controls) or in Currency Exchange Rate Maintenance (General Ledger Controls) for purchase orders, stock receipts, and accounts payable transactions.
F2-Save
Saves the vendor currency information.
