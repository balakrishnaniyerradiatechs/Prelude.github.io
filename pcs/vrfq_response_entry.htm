Vendor RFQ Response Entry screen
The Vendor RFQ Response Entry screen allows you to enter response information received from a vendor RFQ. Refer to Entering Responses from Vendor RFQs.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Response Entry
Field and Function Key Descriptions
Field/Function Key
Description
Release#
The release number which consists of the RFQ number and a dash and the release number (910051-1). Each vendor on the RFQ has a separate release number.
Vendor Number
The vendor number and description.
Product Number
The product number.
To enter cost and delivery information, highlight each line item and press ENTER to open the VRFQ Detail Information screen.
Description
The product description.
N
An asterisk (*) indicates that notes exist for the vendor RFQ release.
Qty-Opn
The quantity from the RFQ displays but you can change it in the VRFQ Detail Information screen.
U/M
The unit of measure from the RFQ displays but you can change it in the VRFQ Detail Information screen.
Cost
The cost from the RFQ displays but you can change it in the VRFQ Detail Information screen.
Delivery Date
The delivery date from the RFQ displays but you can change it the VRFQ Response Detail Information screen.
F2-Save
Saves the response entry information for this vendor.
F4-Del
Deletes the response entry.
F5-Hdr Scr
Opens the VRFQ Release Header screen to view or edit.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are order number, page number, line number, B (bottom), T (top).
