Vendor Line Turn By Buyer Report
The Vendor Line Turn By Buyer report displays the average annual inventory turnover calculated for products within a vendor line, for a buyer. This report can help you to analyze how well a buyer is maintaining inventory levels. Lower than normal turnover for a inventory item may indicate that the buyer is stocking too much of the product. The turnover is calculated by dividing the cost of stock sold in the last 12 months by the inventory value of the current on-hand quantity. The higher the turns the faster the products are selling, which usually indicates which products are more profitable to sell.
Menu Path: Purchasing > Reports > Vendor Turn Reports > Vendor Line Turn by Buyer
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division - if Division is used (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse responsible for selling the material.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Buyer. Typically the buyer assigned to the vendor. The buyer number can be overridden in Purchase Order Entry.
•
Stocking Status. The stocking status of the products to include on the report:
-
S - Stock.
-
N - Non-stock.
-
B - Both stock and non-stock.
-
C - Consignment.
-
A - All (stock, non-stock, consignment).
•
Include Transfer Usage?. Determines if product usage from transfers is included to calculate usage value (cost of goods sold):
-
Y - Include transfer usage to calculate vendor turns.
-
N - Do not include transfer usage. Only include sales order and work order usage.
Column Descriptions
Column Heading
Description
V-LINE
The vendor line number set up for in Vendor Line Maintenance.
Description
The description of the vendor line.
Vend#
The vendor number.
Description
The description of the vendor.
Product Number
The product number.
Description
Description of the product.
Whse
The warehouse where the products were sold.
Turns
The number of turns calculated for the product. The turnover is calculated by dividing the cost of stock sold in the last 12 months by the inventory value of the current on-hand quantity.
