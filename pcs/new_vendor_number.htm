New Vendor Number screen
The New Vendor Number screen allows you to change the vendor on a purchase order.
Menu Path: Purchasing > Inquiries > Buyer Queue Inquiry > Recommended Purchase Orders > F5-Edit > Action Bar > P/O Func > Change Vendor Number
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Number
The number for the new vendor that you want to purchase the items from.
F2-Save
Opens the Vendor Verification screen to allow you to view the name and address information for the selected vendor. Once you accept the vendor number, the Purchase Order Header screen appears so that you can verify the header information.
