Contacts Maintenance screen
The Contacts Maintenance screen allows you to enter and edit address and phone number information for a vendor contact. Vendor contacts are used in Purchase Order Entry.
Menu Path: Purchasing > Vendor Maintenance > Action Bar > Contacts > F5 > Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Contact #
Displays the contact number from the Contact Vendor Entry screen.
Name
The vendor contact’s name.
Address One
The first line of the vendor contact’s address.
Address Two
The second line of the vendor contact’s address.
Address Three
The third line of the vendor contact’s address.
City
The city for the vendor contact.
State
The state for the vendor contact.
Zip
The zip code for the vendor contact.
Email Address
The e-mail address for the vendor contact.
Phone
The phone number for the vendor contact.
Ext
The phone extension for the vendor contact.
Fax
The fax number for the vendor contact.
Pager
The pager number for the vendor contact.
Cell Phone
The cell phone number for the vendor contact.
F2-Save
Saves the vendor contact information.
F4-Del
Deletes the vendor contact information.
