Vendor Rebate Report
The Vendor Rebate report prints the detail or summary listing of the vendor invoices that are used to calculate rebates for a period of time. Only reconciled vendor invoices are included on the report. You can use the Rebate Maintenance Screen to set up rebate percentages and purchasing target information for each vendor to calculate rebates.
Menu Path: Purchasing > Reports > Vendor Rebate Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Vendor. The vendor(s) to select for the report.
•
Rebate Schedule. The schedule that determines the frequency for calculating rebates:
-
M - Monthly.
-
Q - Quarterly.
-
Y - Yearly.
•
Invoice Date Range. The beginning and ending vendor invoice dates to include on the report.
Column Descriptions
Column Heading
Description
Vend#
The vendor number.
Description
The vendor description.
Invoice Number
The vendor’s invoice number that is reconciled with the stock receiver.
Inv Dte
The date of the vendor invoice.
Program
The description of the rebate program.
Purch Target
The minimum purchase order value needed to qualify for the rebate percent shown in the Rebate Maintenance Screen.
Merch Amount
The total cost of the merchandise on the purchase order.
Reb%
The rebate percent received if the purchasing target is met.
RebateAmt
The value of the rebate that you will receive (Merch Amount x Reb%).
Comments
Comments pertaining to the rebate program from Rebate Maintenance Screen.
