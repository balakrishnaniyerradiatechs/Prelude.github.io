Delivery Log Entry screen
The Delivery Log Entry screen allows you to schedule a delivery and enter information regarding a delivery. Refer to “Scheduling Deliveries in the Inventory documentation for more information.
Menu Path: Inventory > Stock Receipts > Delivery Log Entry
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse where the delivery appointments are scheduled.
Date
The date of the delivery appointment.
Sort
The sort sequence of the delivery schedule:
•
Received date/time.
•
Vendor Name.
•
Appointment Number.
•
Carrier
Filters
Determines if you want to use additional selection criteria to filter delivery appointments:
•
N - Do not use additional filters.
•
Y - Opens the Delivery Log Inquiry Filter Options screen to allow you to filter the appointments by date, vendor, carrier, contact, phone number, document type, document number, appointment number, or status.
Date
The estimated date of the delivery.
Time
The estimated time of the delivery.
Vendor
The vendor number associated with the purchase order or the warehouse description.
Carrier
The ship via carrier for the delivery.
Pallets
The number of pallets in the delivery.
Cartons
The number of cartons associated with the delivery.
T
The document type:
•
P - Purchase order.
•
T - Transfer.
•
M - Miscellaneous.
Document
The document number. If multiple documents are associated with the delivery, only the first document displays and a plus sign (+) appears at the end of the document number to indicate that multiple documents exist.
S
The status of the delivery appointment:
•
A - Arrived.
•
C - Cancelled.
•
L - Late.
•
R - Received.
P
The priority code assigned to the delivery. Informational only.
Appt
The appointment number assigned to the delivery
F2-Finish
Closes the screen
F5-Review
Allows you to review information for a delivery appointment.
F6-New Entry
Opens the Delivery Log Entry screen (F6-New Entry) to allow you to enter a new delivery appointment.
