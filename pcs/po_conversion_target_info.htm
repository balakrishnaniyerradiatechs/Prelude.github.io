Target Information screen
The Target Information screen in Purchase Order Conversion Entry compares the total of the purchase order to the targets set up in Vendor Line Maintenance > F5-Targets. You can increase the purchase order amount to meet the target value to receive better pricing.
Menu Path: Purchasing > Inquiries > Buyer Queue Inquiry > Recommended Purchase Orders > F5-Edit > F7-Target
Field and Function Key Descriptions
Field/Function Key
Description
Vline#
The vendor line number.
Description
The description of the vendor line.
Type
The type of target set up in the Vendor Line Target Parameters Screen (e.g., dollars, units pounds, volume). Vendor Line Maintenance > F5-Targets > Target Type - Ov.
Gross Amt
The total amount of the purchase order.
Target
The target amount for this vendor line.
Difference
The difference between the target amount and the purchase order amount.
Flag
Indicates if the target amount is met:
•
Y - The total meets the target amount.
•
N - The total does not meet the target amount.
Notes
Displays notes regarding the vendor line.
F2-Save
Saves the target information.
F5-Evaluate Line
Opens the P/O Vendor Line Evaluation screen to allow you to view items in the vendor line not included on the recommended purchase order because they have not met target requirements for the products. You can review the products and add quantities to the recommended purchase order.
F6-Proportion Increase
Allows you to proportionally increase all items on the purchase order to try to meet the target amount. Stock items are only increased up to the number of months usage maximum. Nonstock and back ordered items are not increased.
F6- Comments
Opens the Vendor Line Comments Inquiry screen to view comments regarding this vendor line.
