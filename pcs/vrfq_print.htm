Vendor RFQ Printing screen
You can use the Vendor RFQ Printing screen to batch print, email, or fax vendor RFQ releases, if your warehouse is not set up to transmit continuously upon completion of the order. All vendor RFQ releases are held in a queue until you print them.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Printing
Field and Function Key Descriptions
Field/Function Key
Description
Vendor RFQ Release Number
The vendor RFQ release number to print if you want to print just one release. Leave blank to print all releases.
Warehouse
The warehouses for which you want to include releases.
Printer Form Queue
The printer where the releases should print.
