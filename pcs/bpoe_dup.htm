Duplicate Blanket P/O screen
The Duplicate Blanket P/O screen allows you to duplicate an existing blanket P/O so that you do not have to re-enter a list of part numbers and prices. You can duplicate the order for the same vendor or a different vendor. Refer to Duplicating a Blanket Purchase Order.
Menu Path: Purchasing > Blanket Purchase Order Subsystem > Blanket Purchase Order Entry > Action Bar > Dup Blanket
Field and Function Key Descriptions
Field/Function Key
Description
Duplicate P/O#
The existing blanket P/O number to duplicate.
Vendor Number
The vendor number and description from the blanket P/O you are duplicating.
New Vendor Number?
The vendor number for the new blanket P/O. Leave blank if the new purchase order is for the same vendor.
New Warehouse Number?
The ship-to warehouse number.
Recost P/O?
Determines if the new blanket P/O uses cost from the original purchase order, or from the current system cost:
•
Y - To recost the line items with the current system cost for the designated vendor.
•
N - To use the cost from the original blanket P/O.
F2-Save
Completes the blanket P/O duplication and opens the Blanket P/O Header screen to edit and complete the order.
