VRFQ Detail Information screen
The VRFQ Detail Information screen allows you to enter quantity and date information for each item in a vendor RFQ.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Entry > Product Number
Field and Function Key Descriptions
Field/Function Key
Description
Ln#
The line number from the Vendor Request for Quote screen.
Qty-Opn
The quantity to quote.
U/M
The unit of measure from the product file.
A
Accept. Indicates if the information is correct:
•
Y - Everything in the detail information screen is correct.
•
N - Allows you to change the Req-Date or the Qty-Open.
Req-Date
The date the material needs to be delivered to your warehouse. Defaults from the Delivery Date in the header screen.
F2-Save
Saves the detail information.
F5-Add’l Qty
Opens the Add’l Qty screen to allow you to obtain prices for an additional quantity of the product.
F6-Addit Info
Opens the VRFQ Additional Information screen to allow you to add or delete vendors on the RFQ.
