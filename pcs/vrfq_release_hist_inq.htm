VRFQ Release History Inquiry screen
The VRFQ Release History Inquiry screen allows you to view historical information for Vendor RFQ releases.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Inquiries > RFQ Release History Inquiry > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
Release Num
The release number assigned to this vendor.
Vendor Number
The vendor number and description.
Product#
The product number.
Description
The product description.
Qty-Opn
The quantity of the quoted item.
U/M
The unit of measure of the quoted item.
FOB Cost
The freight on board cost.
Tariff Code
The tariff code assigned to the product (from Tariff Commodity Code Maintenance) used to calculate Duty.
Tariff Rate
The rate from the Tariff Code used to calculate Duty.
Duty
The duty dollar amount for the item calculated from the Tariff Code.
Freight Cost
The cost of freight quoted by the vendor.
Net Landed Cost
The cost of the item including duty and freight.
Cost Ty
Cost type. For information only.
Select
Indicates if the vendor was select for the quote.
