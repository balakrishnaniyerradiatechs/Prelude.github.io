Vendor Request For Quote screen
The Vendor Request For Quote screen allows you to enter product and quantity information on a quote request to a vendor.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Entry
Field and Function Key Descriptions
Field/Function Key
Description
RFQ Number
The RFQ number.
Product Number
The product numbers to quote.
Description
The product description.
Whse
The warehouse number.
Quantity
The quantity for the vendor to quote.
U/M
The unit of measure from the product file.
F2-Save
Saves the information on the vendor RFQ and prompts you to generate vendor releases.
F4-Del
Deletes the vendor RFQ.
F5-Hdr Scr
Opens the VRFQ Header Information screen.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are order number, page number, line number, B (bottom), T (top).
