Purchase Parameters Skipped
The Purchase Parameters Skipped work queue holds the products with usage exceptions and for which purchasing parameters were not recalculated during the End of Month Calendar Close. These items are added to the Product Parameters Calc Skipped Audit Report in the Inventory Module. If a product is skipped, purchasing parameters are not calculated until the buyer approves or overrides them. Purchasing Parameters are not calculated if:
•
Monthly usage was greater than the total of the last 5 months combined usage.
•
A product is out of stock for 2 weeks or more in one month.
Out of Stock Exception
The Out of Stock Exception flag in Purchasing Controls > Purchasing Parameters determines if purchasing parameters are reviewed when a product is out of stock:
Unusual Usage Exception
The Unusual Usage Exception flag in Purchasing Controls > Purchasing Parameters determines if purchasing parameters are reviewed when a product has unusual usage.
Recalculating Skipped Purchase Parameters.
To view and maintain items that were skipped during Inventory Control Month End Close:
1
Select one of the following to access the Purchase Parameters Skipped Maintenance screen:
•
Purchasing > Inquiries > Buyer Queue Inquiry. Place the cursor in front of Purchase Parameters Skipped Items and press F5-Review.
•
Purchasing > Inquiries > Buyer Queue Inquiries > Purchase Parameters Skipped. Enter the Buyer.
A list of warehouses containing items whose Purchase Parameters were skipped appear.
2
Highlight the warehouse number to view and press F5-Perform Maintenance. The Product Usage Maintenance screen for the first item appears. Refer to the “Product Usage Maintenance Screen” in the Inventory documentation for more information on maintaining usage.
3
You can view and override product usage for this item for each month. The override applies to any subsequent purchasing calculation. If you change product usage, the following question appears:
Changes Were Detected! Would You Like To Recalculate Purchase Parameters With New Usage Overrides?
4
Choose from the following:
•
Y - Recalculates purchase parameters. The product is removed from the work queue.
•
N - Purchase parameters are not recalculated but the product is removed from the work queue.
5
Press F2- Save after reviewing each item and the next item with skipped Purchase Parameters appears.
6
After reviewing each warehouse, press F2-Finish to close the screen.
Field & Function Key Descriptions
Field / Function
Description
Whse
The number for the warehouse that contains the skipped items.
Description
The description/name of the warehouse location.
#Items
The number of items skipped during month end close that did not have their purchasing parameters recalculated.
F2-Finish
Closes the screen.
F5-Perform Maintenance
Displays the Product Usage Maintenance and allows you to view and override product usage values. Refer to the “Product Usage Maintenance Screen” in the Inventory documentation for more information on maintaining usage.
