Request for Quote Inquiry screen
The Request for Quote Inquiry screen allows you to review information for an RFQ release to a vendor.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Inquiries > Open RFQ Release Inquiry > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
Release Num
The RFQ number and release.
Vendor Number
The vendor number and description for the release.
Product Number
The product number.
Description
The product description.
Qty-Opn
The quantity on the RFQ release for the item.
U/M
The unit of measure from the product file.
FOB Cost
Freight on board cost.
Tariff Code
The tariff code assigned to the product (from Tariff Commodity Code Maintenance).
Tariff %
The tariff rate used to calculate Duty.
Duty
The duty amount calculated from the Tariff Code.
Freight Cost
The cost of freight quoted by the vendor.
Net Landed
The cost of the item including duty and freight.
F2-Finish
Closes the screen.
F5-Hdr
Opens the RFQ Header screen for review.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are order number, page number, line number, B (bottom), T (top).
