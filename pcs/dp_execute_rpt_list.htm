Daily Processing Execute Report List screen
The Daily Processing Execute Report List screen allows you to determine which reports to include in Daily Processing.
Menu Path: Purchasing > Processes > Daily Processing > Warehouse Number
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse Number
The warehouse number for running assigned reports.
Report Heading
The name of the report to run.
Print
Determines if the report is to be printed in this Daily Processing run:
•
Y - Print this report in this Daily Processing run.
•
N - Do not print this report in this Daily Processing run.
F2-Save
Saves the execute report list information.
