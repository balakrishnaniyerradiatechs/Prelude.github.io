Import Codes screen
The Import Codes screen in the Purchase Order Header displays the import cost codes established in Vendor Maintenance > F5-Add Info > F7-Importing. You can delete, change, or enter new currency or import factors. Refer to Maintaining Import Cost Factors for more information.
Menu Path: Purchasing > Purchase Order Entry > F5-Hdr Scr > F9-Import
Field and Function Key Descriptions
Field/Function Key
Description
Import Code
The import cost factors from Vendor Maintenance that determine the import costs on this purchase order. The following are examples of commonly used codes:
•
BRK - $ Dollars - Overseas brokerage.
•
CONS - U Units - Overseas consolidation.
•
DUTY - $ Dollars - Overseas duty.
•
IF - V Volume - Inland freight.
•
LW - V Volume - Overseas loading and wharfage.
Description
Description of the import code.
Currency
The currency code to identify how the charge is expressed. The following examples are commonly used currencies:
•
CAN - Canadian dollar.
•
ENG - English pound.
•
ELI - Euro.
•
US - United states dollars.
Factors
The factor or percentage to use with the allocation method to calculate the import cost. Refer to Maintaining Import Cost Factors.
F2-Save
Saves the import code information.
