Delivery Log Entry screen (F6-New Entry)
The Delivery Log Entry screen allows you to schedule a delivery and enter information regarding a delivery. Refer to “Scheduling Deliveries in the Inventory documentation for more information.
Menu Path: Inventory > Stock Receipts > Delivery Log Entry > F6-New Entry
Field and Function Key Descriptions
Field/Function Key
Description
Document Type
The type of delivery:
•
P - Purchase Order.
•
T - Transfer.
•
M - Miscellaneous delivery.
Document#
The list of documents to include in the delivery (e.g., if Document Type is P, enter the purchase order numbers that are being delivered).
Vendor
The vendor delivering the material.
Estimated Date
The date the delivery is expected to arrive.
Estimated Time
The time the delivery is expected to return.
Number/Cartons
The number of cartons expected on the delivery.
Number/Pallets
The number of pallets expected on the delivery.
Carrier
The carrier that will be making the delivery
Contact
The name of the person that is the contact for the delivery.
Phone
The contact’s phone number.
Priority
The code to prioritize the shipment for receipts. Information only.
Status
The status of the delivery:
•
A - Arrived.
•
C - Cancelled.
•
L - Late.
•
R - Received.
Call Date
The call date of the delivery appointment.
Appointment
The next available appointment number assigned for delivery.
Comment
Any comment associated with the delivery.
F2-Save
Saves the delivery information.
F4-Del
Deletes the delivery appointment.
