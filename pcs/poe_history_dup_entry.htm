P/O History Duplication Entry screen
The P/O History Duplication Entry screen allows you to duplicate a received purchase order from the same vendor or for a different vendor. This prevents you from having to re-enter a list of part numbers and prices.
Menu Path: Purchasing > Purchase Order Entry > Action Bar > P/O Func > Duplicate Purchase Order History
Field and Function Key Descriptions
Field/Function Key
Description
Duplicate P/O History#
The purchase order number to duplicate. Only received purchase orders are duplicated in this screen. To duplicate an open purchase order, refer to the P/O History Duplication Entry screen.
Vendor Number
The vendor number and description from the purchase order you are duplicating.
New Vendor Number?
The vendor number for the new purchase order. Leave blank if the new purchase order is for the same vendor.
New Warehouse Number?
The ship-to warehouse number.
Recost P/O?
Determines if the new purchase order uses cost from the original purchase order or from the current system cost:
•
Y - Recost the purchase order line items with the current system cost for the designated vendor.
•
N - Use the cost from the original purchase order.
F2-Save
Duplicates the purchase order and opens the Purchase Order Header screen to edit and complete the order.
