Ship-To Address screen
The Ship-To Address screen allows you to enter a ship-to address on a purchase order if the receiving location is different than your normal ship-to address.
Menu Path: Purchasing > Purchase Order Entry > F5-Hdr Scr > F7-Ship To
Field and Function Key Descriptions
Field/Function Key
Description
Ship-To
The code (up to 8 alphanumeric characters) assigned to a frequently used override ship-to address for your company. Leave blank if no code is available for the desired ship-to address. Ship-to addresses are set up in Vendor Ship-To Maintenance.
Description
The description of the ship-to location.
Address One
The first line of the receiving location address.
Address Two
The second line of the receiving location address.
Address Three
The third line of the receiving location address.
Ship To Attention
The person or department to receive the material.
City
The city of the receiving location.
State
The state of the receiving location.
Zip
The zip code of the receiving location.
F2-Save
Saves the ship-to address information on the purchase order.
