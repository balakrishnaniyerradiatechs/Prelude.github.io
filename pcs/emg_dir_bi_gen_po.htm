Generate Purchase Orders screen
The Generate Purchase Orders screen allows you to edit and/or generate purchase orders for all items in the Emergency/Direct Buyer Inquiry screen with the Buy? flag set to Y. Refer to Emergency/Direct Line Items for information on generating a purchase order for emergency items.
Menu Path: Purchasing > Inquiries > Buyer Queue Inquiry > Emergency Direct Line Items > F5-Review > F6-Auto Convert
Field and Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor number.
Description
The vendor description.
Whse
The warehouse number where the order was placed.
Type
The status of the items:
•
E - Emergency.
•
D - Direct.
•
B - Both emergency and direct.
•
Blank - Neither emergency or direct.
Lines
The number of lines to go on the purchase order.
P/O#
The purchase order number once the order is generated.
Printed
Indicates if the order was printed, faxed, or emailed.
F2-Generate All
Generates purchase orders for all items with the Buy flag set to Y.
F5-Edit
Opens the Purchase Order Entry screen to allow you to change the highlighted purchase order before generating it.
F6-Generate
Generates a purchase order of all items with the Buy flag set to Y, for just the highlighted order.
F8-Review
Opens the Purchase Order Display screen for review.
F9-Continue
Exits the Generate Purchase Orders screen and opens the Emergency/Direct Buyer Inquiry screen.
