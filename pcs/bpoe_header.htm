Blanket P/O Header screen
The Blanket P/O Header screen allows you to enter vendor information and beginning and ending dates for the blanket P/O. Refer to Entering a Blanket Purchase Order for more information.
Menu Path: Purchasing > Blanket Purchase Order Subsystem > Blanket Purchase Order Entry > F5-Hdr Scr
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Quote#
The vendor’s reference number for the quote or contract pricing.
Ship-From Number
You can choose from multiple ship-from addresses for each vendor:
•
If the system does not find a ship-from address for the vendor, then this field is set to D to use the default address in Vendor Maintenance.
•
If 1 ship-from address is established for the vendor, the system uses the ship-from address from Vendor Ship-From Maintenance.
•
If multiple ship-from addresses are available for the vendor, you must choose a ship-from address.
See Vendor Ship-From Maintenance.
Fax Number
The fax number for the purchasing contact.
E-Mail Address
The email address for the purchasing contact.
Written By
The user ID of the person entering the blanket P/O.
Buyer Number
The buyer number of the person assigned to this vendor in Vendor Maintenance.
Contact
The name of the vendor’s purchasing contact.
Type
Free-form reference number.
Beginning Date
The default beginning date for all items on the blanket P/O. If you change this date in the header when open line items exist on the order, the system prompts you to update all lines with the new date.
Expiration Date
The default ending date for all of the items on the blanket P/O. If you change this date in the header when open line items exist, the system prompts you to update all lines with the new date.
Comment
The default comment for items on the Blanket P/O. If you change the comment when open items exist on the P/O, you have the option to update all the items on the order with the new comment.
F2-Save
Saves the header information and returns to the Blanket Purchase Order Entry screen.
F5-New Vendor
Allows you to enter a different vendor number for the blanket P/O.
F6-Ship-From
Opens the Ship-From Address screen to allow you to change the ship-from address.
