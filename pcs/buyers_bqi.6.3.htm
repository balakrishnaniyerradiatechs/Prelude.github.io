Creating Buyers
You can create and maintain buyer codes in Buyer Maintenance. To set up a buyer:
1
Select Purchasing > Maintenance > Buyer Maintenance to access the Buyer Maintenance screen.
2
Enter the code for the Buyer to create or edit. This code is assigned to vendors to designate purchasing responsibility. You may want to use a number (rather than initials) to accommodate employee turnover easily without having to change each vendor in Vendor Maintenance or Vendor Line Maintenance.
3
Enter a Description or name of the buyer.
4
Enter the User Id of this buyer. This must be a valid user ID as assigned in User Identification Maintenance.
5
Press F2-Save.
