Warranty Information Screen
If warranty address information for a vendor is different than the main address for a vendor, you can use the Warranty Information screen to enter warranty information for merchandise returned to the vendor. You can also enter return rules and information.
Menu Path: Purchasing > Maintenance > Vendor Maintenance > F5-Add Info > F6-Warranty
Field and Function Key Descriptions
Field/Function Key
Description
Attention
The name of the person who receives the merchandise at the vendor location.
Address One
The first line of the vendor’s address. This address is used when returning merchandise to the vendor in Vendor Claim Entry. If left blank, the address from Purchase Order Entry is used.
Address Two
The second line of the vendor’s address.
Address Three
The third line of the vendor’s address.
City
The vendor’s city.
State
The vendor’s state.
Zip
The vendor’s zip code.
Phone
The vendor’s phone number.
Fax
The vendor’s fax number.
Contact
The contact name for warranty returns. This information is used in the Rules Vendor Claim Creation Process.
Email Address
The warranty return contact’s email address used to email vendor claim documents.
Buyer
The buyer code assigned to the person responsible for warranty returns to this vendor. Refer to Setting Up a Buyer.
Months
The number of months this vendor warranties the majority of their products. Null or zero indicates that this vendor does not have a warranty period.
Prior Auth. Req
Determines if the vendor requires authorization to return merchandise:
•
Y - The vendor requires a return authorization number before you can return the merchandise.
•
N - The vendor does not require a return authorization number before you can return the merchandise.
Warranty Po
Determines if the vendor requires a copy of the purchase order for returned merchandise:
•
Y - The vendor requires a copy of the purchase order when you return merchandise.
•
N - The vendor does not require a copy of the purchase order when you return merchandise.
Warranty Direct
Indicates if the vendor accepts returns directly from the customer:
•
Y - The vendor accepts returns directly from the customer.
•
N - The vendor does not accept returns directly from the customer.
A warning message appears in RGA Entry if the primary vendor for the item has the Warranty Direct flag set to Y. You can continue or enter another item.
Return Indicator
Indicates if the vendor normally replaces or credits merchandise:
•
R - The vendor normally replaces defective merchandise.
•
C - The vendor normally credits defective merchandise.
F2-Save
Saves the warranty information for this vendor.
F5-Rules
Opens the Vendor Return Rules Screen to allow you to list any requirements the vendor has for returning defective merchandise.
