P/O Vendor Line Evaluation screen
The P/O Vendor Line Evaluation screen displays products in the vendor line that are not included on the recommended purchase order because they do not meet target requirements for the products that are in a buy condition. You can review the line items and enter quantities to add to the recommended purchase order. The products are sorted by ascending days to line point/minimum order for the product.
Menu Path: Purchasing > Inquiries > Buyer Queue Inquiry > Recommended Purchase Orders > F5-Edit > F7-Target > F5-Evaluate Line
Field and Function Key Descriptions
Field/Function Key
Description
Target Type
The purchasing target override type set up in the Vendor Line Target Parameters Screen to calculate the target amount:
•
$ - Dollars.
•
# - Pounds.
•
U - Units.
•
O - Other. A user defined method to reach a minimum for purchasing (e.g., volume or cube).
Amount
The purchasing target amount entered in the Vendor Line Target Parameters Screen that corresponds with the Target Type.
Actual Amount
The actual amount from the recommended purchase order.
Diff
The difference between the purchasing target amount and the recommended purchase order amount.
Product number
The product number.
Description
The product description.
PrimVend
Indicates if the vendor on the purchase order is the primary vendor for the corresponding product.
Cost
The product cost.
Net Avail
The purchasing quantity available calculated from Nightly Processing.
LP/Min
The line point minimum. When the available quantity falls below the order point, the entire line is reviewed for a possible line buy. The re-order point depends on the type of product:
•
Min/Max products. When the available product falls to the minimum quantity or below, order back up to the max. In the case of replenishment consideration, the minimum is the re-order point.
•
Non min/max products. When the available product falls to the line point or below, order the replenishment quantity. In the case of replenishment consideration, the line point is the re-order point.
Forecast
The forecasted quantity for the current month.
Reccmd
The recommended order quantity calculated from Nightly Processing.
Days
The number of days to the line point/minimum.
(Net Avail - LP/Min) / Forecast
Quantity
The quantity of the product that you want to order.
F2-Save
Saves the quantity ordered information.
F5-Invty Inq
Opens the Inventory Inquiry screen to allow you to view the inventory status of the item.
