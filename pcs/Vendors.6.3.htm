Vendor Maintenance Screen
The main screen in Vendor Maintenance holds the basic name and address information for a vendor. You can access the additional vendor information screens from here.
Menu Path: Purchasing > Maintenance > Vendor Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Vendor
An 8-digit alphanumeric identification.
Description
The vendor’s company name, which appears on screens, reports, and purchase orders for the vendor.
Desc. (To Sort By)
The description to sort by if the name of the vendor’s company is not useful for sorting. For example, if the Description is The Supply Company, the system sorts on the first word “The”. If you want to search using a different word (e.g., Supply), you can enter it here.
Address One
The first line of the vendor’s address. This address is for accounts payable. To enter a different address for purchasing functions, use F7-Contacts (refer to the Vendor Contacts Screen). If accounts payable and purchasing address information is the same, you can use this address for both.
Address Two
The second line of the vendor’s address.
Address Three
The third line of the vendor’s address.
Country
The vendor’s country.
GeoCode
The vendor’s geographic code if you use Vertex. You can assign the city, state, zip code and country of the vendor’s address based on the address cross-reference setup in Geo Codes/Zip Codes Maintenance.
City
The vendor’s city.
State
The vendor’s state.
Zip
The vendor’s zip code.
County
The vendor’s county.
Contact
The name of the vendor’s accounts payable contact person.
Email Address
The accounts payable email address for emailing accounts payable documents such as debit memos.
Website Address
The vendor’s website. This is for information only and does not access the web page.
Phone
The accounts payable contact’s phone number. This phone number is also a keyword to help look up a vendor, both with or without the area code and special characters.
Ext
The extension for the accounts payable contact’s phone number.
Fax
The accounts payable contact’s fax number used to fax accounts payable documents such as debit memos.
Other Phone
The accounts payable contact’s additional phone number.
Payment Terms
The accounts payable terms code as set up in A/P Terms Maintenance. Used as the default terms codes on purchase orders and manual invoices to this vendor.
Buyer
The buyer who is responsible for inventory for this vendor. Recommended purchase orders generated by Nightly Processing are sent to this buyer’s work queue. You can override this number at the vendor line level (refer to Vendor Line Maintenance).
Transfer Buyer
The buyer responsible for transfers for this vendor’s products. Recommended stock transfers generated by Nightly Processing are sent to this buyer’s work queue. You can override this buyer at the vendor line level.
Inventory Parameters > F5-Add Info > Rec. Transfers By Buyer must be set to Y.
Ship Via
The method the vendor uses to ship material to your warehouse. Used for the default on purchase orders to this vendor.
Start Date
The date the vendor record was created.
F2-Save
Saves changes to the vendor record.
F4-Delete
Deletes the vendor record. You cannot delete a vendor record if open documents exist (e.g., purchase orders, vendor claims, receivers) or if open accounts payable balances exist for this vendor.
F5-Add Info
Opens the Additional Information Screen, which includes settings for purchasing and inventory controls for this vendor.
F6-A/P
Opens the A/P Information Screen, which includes payment and discount information.
F7-Contacts
Opens the Vendor Contacts Screen to allow you to enter purchasing contacts.
F8-Statistics
Opens the Vendor Statistics Screen to display statistical information captured by month over the last 3 years.
Action Bar
Notes
You can enter or edit notes to appear in the following areas in the system:
•
Vendor Notes: Reference notes pertaining to the vendor.
•
Purchasing Comments: Notes to appear when this vendor is used on purchase orders.
•
A/P Comments: Notes to appear when this vendor is entered in Vendor Invoice Reconciliation Entry or Manual Invoice Entry.
•
Warranty Comments: Notes to appear when this vendor is entered on vendor claims.
•
Attachments: Opens the Prelude Attach Viewer to allow you to attach a document to the vendor record. You can attach any document/picture that you can access on your computer.
•
Imaging: If your company is using imaging, you can access invoices, letters of credit, shipping documents, P/O related documents, Vendor RFQs, checks, contracts, and other correspondence.
Contacts
Opens the Contact Vendor Entry screen to add or edit contacts for the vendor. These contacts can be accessed in Purchase Order Entry in the Purchase Order Header screen by pressing F3 in Contact.
Add’l Options
Opens the following screens:
•
Label Information Screen.
•
Vendor Product Types Screen.
•
Bank Information Screen.
•
Instructions Screen.
•
Vendor VIA Properties Screen.
