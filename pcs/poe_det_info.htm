P/O Detail Information screen
The P/O Detail Information screen allows you to enter quantity, cost, and delivery information for a product on a purchase order.
Refer to Entering Products for more information.
Menu Path: Purchasing > Purchase Order Entry > Product Number
Field and Function Key Descriptions
Field/Function Key
Description
Ln#
The line number on the purchase order.
Qty-Opn
The quantity to order. If a is value entered in Package Quantity in Product Vendor Maintenance for this vendor/product combination, you are prompted to round to the nearest package quantity.
U/M
The purchasing unit of measure of this product from Product Maintenance. If U/M Table in Product Maintenance is set to Y, you can change this value.
Whse
The shipping warehouse.
Accept
Verifies if everything else on the screen is correct:
•
Y - If everything else in this screen is correct. The P/O Detail Information screen closes and you can enter the next line item on the P/O.
•
N - If you need to change any of the remaining fields.
Qty-B/O
The quantity open on this purchase order.
Del-Date
The expected delivery date of the product (product lead time plus today’s date). The default is from the purchase order header but you can override it.
If you change the Delivery Date after the P/O is already entered, you must choose one of the following status codes:
•
C - Date originally confirmed by vendor.
•
E - Date confirmed by vendor after expediting.
•
O - Date overridden by operator.
If the Suppress Freight Days Prompt in Purchasing Parameters is set to N, you must enter the estimated Number Of Freight Days it will take for the merchandise to ship to your warehouse.
Promised
The date the merchandise is scheduled to arrive as promised by the vendor. This information is used to calculate vendor performance statistics.
Gross
The gross cost of the product before any vendor or product discounts.
Disc%
The discount percent if applicable. Refer to Costing Purchase Orders for more information.
Net-Cost
The net cost of the item after any vendor or product discounts.
Src
The source code to identify where the price came from. Refer to Identifying Purchase Order Source Codes.
Vendor Product#
If applicable, the vendor product number to cross reference your product number and to print on the purchase order.
F2-Save
Saves the detail information for the line item and brings you back to the Purchase Order Entry screen to continue adding items.
F5-Add-Info
Opens the P/O Additional Info screen to allow you to make changes to information regarding Inspection, Thaw Dates, Vendor Contracts, Vendor Quotes etc.
F7-Notes
Opens the Notes Display screen to review or enter any notes regarding this product.
