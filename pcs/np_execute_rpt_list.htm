Nightly Processing Execute Report List screen
The Nightly Processing Execute Report List screen appears as you press ENTER through each Whse number in the Nightly Processing screen. You can determine which reports to execute as each warehouse is evaluated.
Menu Path: Purchasing > Processes > Daily Processing > Warehouse Number
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse Number
The warehouse number to assign reports for.
Report Heading
The name of the report to run. Refer to Nightly Processing Reports.
Print
Determines if the report prints in this Nightly Processing run:
•
Y - Print this report in this Nightly Processing run. Refer to Defining Nightly Purchasing Reports to Print.
•
N - Do not print this report in this Nightly Processing run.
F2-Save
Saves the execute report list information.
