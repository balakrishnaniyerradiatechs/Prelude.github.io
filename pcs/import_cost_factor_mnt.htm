Import Cost Factor Maintenance screen
The Import Cost Factor Maintenance screen allows you to create import cost factor codes to account for additional cost for overseas vendors. The codes are assigned to each vendor in Vendor Maintenance > F5 Add Info > F7 Importing, but are overridden in Purchase Order Entry in the Purchase Order Header > F5-Add Info > F5-Import Factors screen.
Refer to Maintaining Import Cost Factors for more information.
Menu Path: Purchasing > Maintenance > Import Cost Factor Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Import Code
The import code to use when placing a purchase order to a vendor who has extra costs associated with importing.
Description
The description of the import cost.
Alloc Method
The allocation method to determine how the cost factor is calculated and allocated to products on purchase orders:
•
# - Pounds.
•
$ - Dollars.
•
U - Units.
•
V - Volume.
Factor
The factor (percentage) used to calculate the charge. Used with the freight allocation method in Vendor Maintenance.
Div
Division number. The highest possible grouping in your company.
Reg
Region number. A subset of Division if Division is used.
Loc
Location number. The warehouse that the code applies to.
Dep
Department number. A subset of Location.
Ccr
Cost Center Number. A subset of Department if Department is used.
Account#
The general ledger account number. When the vendor invoice is reconciled, the charge is posted to a clearing account.
F2-Save
Saves the import code information.
F4-Del
Deletes the import code.
