Inventory Analysis Report
The Inventory Analysis report lists products ranked by cost of goods sold. You can use this information to analyze the most important products based on cost of sales. You can also determine the percent of inventory that accounts for the most sales and the percent of inventory that does not generate any sales.
When you run this report, a dialog box appears to warn you to run the Current Product Classifications Rebuild Execution before you execute the report.
Menu Path: Purchasing > Reports > Inventory Analysis
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse containing products to be ranked.
Column Descriptions
Column Heading
Description
Product Number
The product numbers ranked by cost of goods sold.
Description
The description of the products.
Usage Per Year
The quantity of the product sold per year.
Weighted Usage Cost
The average cost per unit.
Usage Extension
The extended dollar amount of the product shipped per year.
Cum %
The percent of sales for the products for each line item and all lines previous. For example, the percent of sales for each line item is added to the next line item for a running cumulative total.
Current On-Hand
The quantity of the product currently on-hand.
Current Cost Ext
The extended dollar amount of the product on hand,
Cum %
The percent of cost for each line item and all lines previous. For example, the percent of cost for each line item is added to the next line item for a running cumulative total.
IC
The inventory class. Refer to “Classifying Inventory” in the Inventory Module.
