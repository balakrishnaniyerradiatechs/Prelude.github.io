Vendor RFQ Employee Inquiry
The Vendor RFQ Employee Inquiry holds the vendor request for quote releases for which a vendor cost was entered by the buyer. You can view the release and cost and turn the request into a purchase order if necessary.
Menu Path: Order Processing > Inquiries > Employee Queue Inquiry > VRFQ Inquiry > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
User Id
The user ID of the employee who entered the response.
Release Number
The VRFQ release number that was sent to the vendor for a price quote.
Vendor Number
The number for the vendor that responded to the quote request.
Req Date
The date the products on the request are required.
Exp Date
The date the quote request expires.
Quote#
The quote number issued by the vendor.
Customer
The customer for whom the quote request was generated.
F2-Finish
Closes the Vendor RFQ Inquiry.
F5-Edit
Opens the Vendor RFQ Conversion screen to allow you to view the vendor information and convert the request into a purchase order if necessary.
F6-Enter
Allows you to enter a VRFQ number, for direct access.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
