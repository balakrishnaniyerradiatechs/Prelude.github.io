Merge With Suspended P/O screen
The Merge With Suspended P/O screen allows you to combine the order you are entering with a suspended purchase order for the same vendor. Refer to Merging With a Suspended Purchase Order.
Menu Path: Purchasing > Purchase Order Entry > Action Bar > Prod Func > Merge With Suspended P/O
Field and Function Key Descriptions
Field/Function Key
Description
P/O#
The suspended purchase order number available for merging.
P/o Date
The date the suspended purchase order was entered.
Vend#
The vendor number for the suspended purchase order.
Description
Description of the vendor.
Buy#
The buyer number from the suspended purchase order.
SV
Ship via code.
Lns
The number of lines on the suspended purchase order.
P/o Dollars
The total dollar amount of the suspended purchase order.
F2-Save
Saves the suspended purchase order information and brings you back to the Purchase Order Entry screen. No items are merged.
F5-Review
Opens the Purchase Order Display screen to allow you to view cost and delivery information for each item on the suspended order.
F6-Merge
Prompts you with the following options:
•
Individual: Merges the product that was highlighted when Merge w/Suspended P/O was selected.
•
All: Merges all products on the current purchase order with the suspended purchase order.
•
Return: Goes back to the Purchase Order Entry screen without merging any items.
