Frozen Control Thaw Date Report
The Frozen Control Thaw Date report displays products with frozen controls, sorted by thaw date. You can use this information to determine if the products should be left frozen or if they should be released to let the system calculate the purchasing requirements.
Menu Path: Purchasing > Reports > Frozen Control Date Thaw
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division if Division is used (e.g., Southeast Region, Northwestern Region).
•
Warehouse. The warehouse containing the products.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Primary Vendor. The primary vendor(s) for the products to include on the report.
•
Frozen Control Thaw Date. The date to include all products with a frozen control cancellation date up to or on this date.
Column Descriptions
Column Heading
Description
Product
Line 1: The product number.
Line 2: The product description.
Vendor
The vendor number.
Description
The vendor description.
Type
The purchase method set up for the product in Purchasing Parameters:
•
EOQ - Economic Order Quantity.
•
CLS - Class (always used for Replenishment Method T).
•
AWU - Average Weekly Usage. Refer to Tracking Average Weekly Usage” in the Inventory documentation.
•
DRP - Distribution Requirements Planning. Used with the Forecasting module primarily for products with long lead times.
•
DS - Days Supply. A number of days supply determines the quantity to keep on hand.
Minimum
Minimum quantity to keep in stock as set in Product Warehouse Maintenance.
Maximum
Maximum quantity to keep in stock as set in Product Warehouse Maintenance.
Forecast
The forecasted quantity calculated by the system.
Order Pt
The order point calculated by the system.
Line Pt
The line point calculated by the system.
Rec Qty
The recommended buy quantity recommended by the system.
Thaw Dte
The frozen control thaw date (cancellation date) for this product.
