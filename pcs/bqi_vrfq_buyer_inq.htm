Vendor Request for Quotes
The Vendor Request for Quotes queue is updated when a Vendor RFQ release is generated (refer to Generating a Vendor Release) to request costs from a vendor. When the vendor replies with a quote, you can use this work queue to enter the response (refer to Entering Responses from Vendor RFQs). You also have the option to send the response to the Employee Queue Inquiry to the person that requested the quote.
Enter a Response to a Vendor Request for Quote
To review and respond to a Vendor RFQ:
1
Select one of the following to access the Buyer VRFQ Inquiry screen:
•
Purchasing > Inquiries > Buyer Queue Inquiry. Place the cursor in front of Vendor Request for Quote and press F5-Review.
•
Purchasing > Inquiries > Buyer Queue Inquiries > Vendor Request For Quotes. Enter the Buyer Number.
The open Vendor RFQ releases appear.
2
Highlight the VRFQ Number for which you want to enter a response and press F6-Response. The VRFQ Release Header screen appears.
3
Refer to Entering Responses from Vendor RFQs for step by step instructions on entering the vendor response.
4
Press Esc to save the response. After each vendor response is entered, the following message appears:
Do You Wish To Send Notification to the Employee Queue?
5
Choose from the following:
•
Yes - The response is sent to the Employee Queue Inquiry of the person that entered the vendor RFQ.
•
No - The Employee Queue Inquiry is not updated.
6
Press F2-Finish to close the screen.
Field & Function Key Descriptions
Field / Function
Description
VRFQ Number
The vendor request for quote release number.
Vendor Number
The vendor that the release was sent to.
Description
The vendor name.
VRFQ Date
The date the vendor request for quote was entered.
Req Date
The date a response from the vendor is requested.
F2-Finish
Closes the screen to return to the Buyer Queue Inquiry screen.
F5-Review
Opens the Request for Quote Inquiry screen which lists detail information for products on the request.
F6-Response
Opens the VRFQ Release Header screen to allow you to enter the response information for the vendor RFQ. Refer to Entering Responses from Vendor RFQs.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
