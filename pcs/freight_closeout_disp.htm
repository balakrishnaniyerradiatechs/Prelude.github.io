Freight Closeout Display screen
The Freight Closeout Display screen displays information from the Freight Closeout screen in Purchase Order Entry.
Menu Path: Inventory > Stock Receipts > Stock Receipts Entry > F2-Save
Field and Function Key Descriptions
Field/Function Key
Description
Freight Amount
The calculated freight amount on the purchase order.
Duty Amount
The duty amount if applicable.
Brokerage Amount
The brokerage amount if applicable.
Currency Rate
The currency rate from either Currency Code Maintenance or the daily rate based on company options.
Currency Ind.
The currency indicator for this vendor.
OK
Closes the Freight Closeout Display screen.
