Including Suspended, Linked Sales Orders
You can determine if you want Nightly/Daily Processing to create a temporary purchase order for nonstock products on a work order linked to a suspended sales order.
To include suspended, linked sales orders:
1
Select Option Parameters > Purchasing Parameters.
2
For Incl. Susp. S/O for Linkage?, choose one of the following:
•
Y - Nightly Processing generates a temporary purchase order for the components of a work order linked to a suspended sales order.
•
N - Nightly Processing does not generate a temporary purchase order for components of a work order linked to a suspended sales order.
3
Press F2-Save to save any changes.
