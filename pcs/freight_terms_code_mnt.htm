Freight Terms Code Maintenance screen
The Freight Terms Code Maintenance screen allows you to create codes to identify freight consolidators who pay your freight bills.
Menu Path: Purchasing > Maintenance > Freight Terms Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Freight Terms
The code to identify the freight consolidator.
Description
The description of the code.
Carrier Billed Frt?
Indicates if freight is billed by the carrier:
•
Y - Freight is billed by the carrier.
•
N - Freight is not billed by the carrier.
Vendor Billed Frt?
Indicates if freight is billed by the vendor:
•
Y - Freight is billed by the vendor.
•
N - Freight is not billed by the vendor.
F2-Save
Saves the freight terms code information.
F4-Del
Deletes the freight terms code.
