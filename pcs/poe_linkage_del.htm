Purchase Order Linkage Deletion screen
The Purchase Order Linkage Deletion screen allows you to delete a link to a backorder on a sales order, work order, or transfer. Refer to Manually Linking an Item.
Menu Path: Purchasing > Inquiries > Purchase Order Inquiry > Action Bar > Prod Func > Linkage Deletion
Field and Function Key Descriptions
Field/Function Key
Description
Dest Doc Id
The document and the line item linked to the product. The prefix indicates the document type:
•
S - Sales order.
•
T - Transfer.
•
W - Work order.
Destination Description
The customer or branch that was to receive the linked backorder.
F2-Save
Saves the purchase order linkage deletion information.
