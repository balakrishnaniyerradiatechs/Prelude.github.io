Deleted Reason Code screen
If you delete a purchase order, the Deletion Reason Code screen appears. You can enter a code which tracks the reason for deletion.
Menu Path: Purchasing > Purchase Order Entry > F4-Del
Field and Function Key Descriptions
Field/Function Key
Description
Deleted P/O Reason Code
The reason code from P/O Deletion Code Maintenance that explains why the purchase order is being deleted.
