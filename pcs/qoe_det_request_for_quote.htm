Request for Quotation Entry screen
The Request for Quotation Entry screen allows you to enter product and quantity information on a quote request to a vendor.
Menu Path: Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Entry > Quote Entry Detail Screen > Action Bar > Entries > Request for Quote Entry
Field and Function Key Descriptions
Field/Function Key
Description
RFQ Number
The assigned request for quote number.
Product Number
The product number and description for the item on the quote request. Press ENTER on each line item to open the RFQ Detail Information screen to enter quantity and cost information.
Ln#
The line number on the request for quote.
Vendor #
The vendor number.
Description
The vendor description.
Quantity
The quantity that you are requesting for the vendor to quote a price. You can change the quantity in the RFQ Detail Information screen.
U/M
The unit of measure from the product file.
F2-Save
Saves the request for quote information and prompts you to print the RFQ. Refer to Creating a Vendor RFQ From a Sales Quote for more information.
F4-Del
Deletes the request for quote entry.
F5-RFQ Header
Opens the RFQ Header screen to allow you to enter response date information.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are order number, page number, line number, B (bottom), T (top).
