Freight Closeout screen
The Freight Closeout screen allows you to override the calculated Freight Amount.
Menu Path: Purchasing > Purchase Order Entry > F2-Save
Field and Function Key Descriptions
Field/Function Key
Description
Freight Amount
The estimated freight amount for the merchandise on the purchase order.
Duty Amount
The duty amount if applicable.
Brokerage Amount
The brokerage amount if applicable.
Currency Rate
The currency rate from either Currency Code Maintenance or the daily rate based on company options.
Currency Ind.
The currency indicator for this vendor.
F2-Save
Saves the freight information.
