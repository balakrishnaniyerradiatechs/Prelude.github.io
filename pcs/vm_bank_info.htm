Bank Information Screen
The Vendor Bank Information screen allows you to enter the vendor’s bank account information for EFT (electronic funds transfer) payments.
Menu Path: Purchasing > Maintenance > Vendor Maintenance > Action Bar > Add’l Options > Bank Information
Field and Function Key Descriptions
Field/Function Key
Description
ABA (Routing) Number
The ABA or routing number for the vendor’s bank. Used for EFT payments.
Account Number
The vendor’s bank account number for EFT payments.
Account Type
Bank account type:
•
C - Checking account.
•
S - Savings account.
F2-Save
Saves the vendor bank information.
