Daily Processing screen
The Daily Processing screen allows you to determine which replenishment processes to run for selected warehouses and vendors.
Menu Path: Purchasing > Processes > Daily Processing
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse number to include in this Daily Processing run. Opens the Daily Processing Execute Report List screen to determine which reports to run.
Regular Purchase Orders
Determines if you want to generate regular recommended purchase orders:
•
Y - Generate regular recommended purchase orders.
•
N - Do not generate regular recommended purchase orders.
Emergency Purchase Orders
Determines if you want to generate emergency (E supply code on the sales order, work order, or transfer) recommended purchase orders:
•
Y - Generate emergency recommended purchase orders.
•
N - Do not generate emergency recommended purchase orders.
Drop Ship Purchase Orders
Determines if you want to generate direct/drop ship (D supply code on the sales order or work order) recommended purchase orders:
•
Y - Generate drop ship recommended purchase orders.
•
N - Do not generate drop ship recommended purchase orders.
Work Orders
Determines if you want to generate recommended work orders:
•
Y - Generate recommended work orders.
•
N - Do not generate recommended work orders.
Stock Transfers
Determines if you want to generate recommended stock transfers:
•
Y - Generate recommended stock transfers.
•
N - Do not generate recommended stock transfers.
Vendor
The vendor numbers to evaluate in the Daily Processing run. Vendor numbers are not necessary if you are generating transfers only.
Description
The vendor description.
F2-Start
Starts Daily Processing. Messages appear at the bottom of the screen until Daily Processing is complete. The screen automatically closes.
