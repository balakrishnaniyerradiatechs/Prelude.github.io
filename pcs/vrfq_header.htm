VRFQ Header Information screen
The Header Information screen in the Vendor RFQ Entry program allows you to enter the vendor and date information for the quote request. Refer toStarting or Editing a Vendor RFQ.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Entry > F5-Hdr Scr

Vendor
The vendor names or numbers to include in the quote request.

The vendor description.
Frt Rate
The freight rate if freight is to be considered in the quote.
F2-Save
Saves the header information and opens the Vendor Request For Quote screen.

Opens the VRFQ Duplicate screen to allow you to duplicate an RFQ.
