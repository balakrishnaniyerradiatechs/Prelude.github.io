Label Information Screen
If your company uses the Serial Shipping Container Code (SSCC) identification system, you can use the Label Information screen to set up label printing and contents tracking information.
Menu Path: Purchasing > Maintenance > Vendor Maintenance > Action Bar > Add’l Options > Label Information
Field and Function Key Descriptions
Field/Function Key
Description
Cage Code
The code assigned to the vendor that prints on the DD250 form (block 16) immediately after each product number.
Contents Tracking
Determines if vendor claim serial shipping is required for all material shipped to this vendor:
•
Y - Serial shipping container tracking is required for all material shipped to this vendor. In this case, every vendor claim for the vendor must be itemized and tracked by SSCC.
•
N - Serial Shipping Container tracking is not required for this vendor.
Ucc Mfg Number
The UCC manufacturer number for this vendor for assigning UCC labels. If this field is blank, the system uses your company’s UCC number.
SSCC Label Program
The custom Serial shipping Container Code (SSCC) program name that defines the label required by the vendor. This field works with SSCC Label Execution to print a label based on the vendor’s label layout requirements. Refer to “Assigning Serial Shipping Container Code Labels to Vendor Claims” in the Inventory Module.
SSCC Label Execution
Determines when SSCC labels print:
•
N - Print when a new SSCC number is assigned. Only used if the SSCC label only contains shipping address information and does not print product information.
•
E - Print at the end of each SSCC entry.
•
F - Print at the end of all SSCC entries.
PC Label Program
The name of the custom program that prints SSCC labels. The labels include a detail listing of the contents of the shipment.
PC Label Execution
Determines when SSCC labels print:
•
E - At the end of each SSCC entry.
•
F - At the end of all entries for a shipment.
F2-Save
Saves the vendor label information.
