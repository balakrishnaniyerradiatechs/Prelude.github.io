Discount Break Table Maintenance screen
The Discount Break Table Maintenance screen allows you to establish the break levels for determining discounts. For more information, refer to Setting up Discount Break Tables.
Menu Path: Purchasing > Maintenance > Discount Break Table Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Table Number
The table number to define a set of rules for determining discounts. You can reference this table in Vendor Line Maintenance, Product Maintenance, Product Warehouse Maintenance, or Product Vendor Maintenance.
Warehouse
The warehouse number if the break table only applies to one warehouse. Leave blank if the table applies to all warehouses.
Description
The description of the break table so that you can easily identify it from other maintenance programs.
Sum And Recost?
Determines how purchase order line items are accumulated for recosting:
•
Y - Purchase order line items are accumulated by vendor line and recosted if a break level is met.
•
N - Purchase order line items are recosted based on individual quantity or dollar value.
Table Method
Determines how the discount is calculated:
•
$ - Each break has a set price.
•
* - The price is a discount off of a column price.
•
C - Rebate base cost - current.
•
R - Rebate base cost - replacement.
•
D - A chain discount.
•
M - Net margin.
•
T - Quantity discount table.
Break Level Type
Defines the type of break level:
•
U - The break level is based on units.
•
$ - The break level is based on dollars.
•
M - The break level is based on unit of measure.
