Vendor RFQ Conversion screen
The vendor RFQ Conversion screen displays the items on a vendor RFQ along with the number of vendor responses received for each item. You can select a vendor for each item and convert the RFQ into one or more purchase orders. Refer to Converting a Vendor RFQ to a Purchase Order.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Conversion
Field and Function Key Descriptions
Field/Function Key
Description
RFQ Number
The RFQ number to convert.
Product Number
The product number.
Description
The product description.
Whse
The warehouse number.
Recv
The number of response received from vendors on this RFQ.
Opn
The number of responses not yet received from vendors.
Quantity
The quantity of the line item quoted.
U/M
The unit of measure as entered on the RFQ response.
Selected Vendor
The number of the vendor selected for the corresponding product number.
Description
The description of the vendor selected for the corresponding product number.
F2-Save
Prompts you to generate a purchase order. If you answer yes, the Generate P/O’s For Vendor RFQ screen appears.
F4-Del
Allows you to delete the RFQ.
F5-Hdr Scr
Displays the VRFQ Header Information screen for the RFQ.
F6-Vendor Detail
Opens the Vendor Detail Information screen for the highlighted item. Displays the vendors that you have received responses from along with the quoted costs and freight charges. Allows you to choose a vendor for each item.
F9-Goto Line
Allows you to enter a reference number for quick access. Valid codes are order number, page number, line number, B (bottom), T (top).
Action Bar
Upd Hist
Prompts you to move the vendor RFQ to history:
•
Y - Moves the RFQ to the history file.
•
N - Does not move the RFQ to the history file.
