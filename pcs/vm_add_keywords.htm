Additional Keywords Screen
The Additional Keywords screen allows you to enter keywords to use as cross references to help in vendor searches. These keywords are added to those already created using the vendor description and all other fields in Vendor Keyword Maintenance. Refer to Maintaining Vendor Keywords.
Menu Path: Purchasing > Maintenance > Vendor Maintenance > F5-Add Info > F5-Keywords
Field and Function Key Descriptions
Field/Function Key
Description
Keywords
Additional keywords to assist in searches for this vendor. For example, if the vendor changed their name, you can enter their old name as a keyword. Or, you can enter the products sold by the vendor as keywords.
F2-Save
Saves the additional keywords for the vendor.
