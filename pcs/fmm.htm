Freight Matrix Maintenance screen
The Freight Matrix Maintenance screen allows you to assign a freight matrix to define the percentage to calculate estimated freight.
Menu Path: Purchasing > Maintenance > Freight Matrix Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
From Warehouse
Depending on whether the matrix is for a transfer or stock receipt, one of the following applies:
•
For stock transfers, the shipping warehouse where estimated freight is calculated.
•
For stock receipts, REC calculates estimated freight on purchase orders.
To Warehouse
The warehouse where the merchandise is received.
Product Line
The product line to assign the matrix to:
•
ALL indicates that the matrix is for all product lines.
•
A single product line indicates that the matrix is for just one line. A matrix for a single product line is used before a matrix for all product lines.
Freight Rate
The rate to calculate the estimated freight amount. Used with the Freight Allocation method from Product Line Maintenance.
F2-Save
Saves the freight matrix.
