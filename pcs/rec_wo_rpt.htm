Recommended Work Order Report
The Recommended Work Order report generated from Nightly Processing lists the temporary work orders for BOMs (bill of material) that need to be built.
Menu Path: Purchasing > Reports > Nightly Processing Reports > Recommended Work Order Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
