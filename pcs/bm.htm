Buyer Maintenance screen
The Buyer Maintenance screen allows you to create and maintain buyer codes.
Menu Path: Purchasing > Maintenance > Buyer Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Buyer
The alphanumeric code that represents a buyer.
Description
The name of the buyer.
User Id
The user ID for this buyer. This must be a valid user ID number assigned in User Identification Maintenance.
F2-Save
Saves the buyer information.
F4-Del
Deletes the buyer record.
