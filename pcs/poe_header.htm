Purchase Order Header screen
The Purchase Order Header screen allows you to review or edit information regarding the vendor, P/O type, delivery date, and payment terms. Refer to Entering Purchase Order Header Information.
Menu Path: Purchasing > Purchase Order Entry > F5-Hdr Scr
Field and Function Key Descriptions
Field/Function Key
Description
P/O Type
The type of purchase order, (e.g. Regular, Direct, Will Call). Refer to Purchase Order Types for a listing and explanation of the 13 P/O types available.
Ship-From
You can choose from multiple ship-from addresses for each Vendor:
•
If the system does not find a ship-from address for the vendor, then this field is set to D to use the default address established in Vendor Maintenance.
•
If 1 ship-from address is established for the vendor, the system uses the ship-from address from Vendor Ship-From Maintenance.
•
If multiple ship-from addresses are available for the vendor, you are prompted to choose a ship-from address.
See Vendor Ship-From Maintenance.
Buyer
The buyer code to represent the person responsible for this purchase order. The default is from the buyer assigned to this vendor in Vendor Maintenance or Vendor Line Maintenance, but you can override it. Refer to Setting Up a Buyer.
Contact
The purchasing contact name from Vendor Maintenance > F7 Contacts.
Ship Via
The ship-via code from the vendor record to indicate the method for shipping the merchandise. The code prints on the purchase order.
Freight Terms
The freight code used to identify freight consolidators. Refer to Entering Freight Terms Codes. The default for the vendor is established in Freight Terms in the Additional Information Screen.
Payment Terms
The accounts payable terms code from Vendor Maintenance to determine the payment terms for this purchase order. The codes are set up in A/P Terms Maintenance.
FOB Desc
Freight on board description.
Vendor Order
The vendor’s order number, if applicable.
Email Address
Defaults to the email address for the Contact.
Fax
The vendor’s fax number. The system looks for the fax number in the following order:
•
The fax number from the contacts established in Vendor Maintenance > Action bar > Contacts.
•
The fax number from Vendor Maintenance > F7-Contacts.
•
The fax number from vendor’s main address in Vendor Maintenance.
Phone
The Contact’s phone number.
Std. Disc%
The standard discount deducted from the vendor’s cost on this purchase order.
Freight Flag
Indicates to account payable if you are responsible for freight:
•
Y - You are responsible for freight if freight charges are billed on this purchase order. If the Frt Paid Amount in the Vendor Line Target Parameters Screen is met, this setting automatically changes to N.
•
N - You are not responsible for any freight charges billed on this purchase order. This setting is the default on recommended purchase orders created by Nightly Processing that meet the Frt Paid Amount in the Vendor Line Target Parameters Screen.
For more information on freight charges, refer to Applying Estimated Freight to P/Os and Receivers.
Confirmed P/O
Determines if the purchase order is confirmed with the vendor:
•
Y - The purchase order is confirmed to the vendor. In this case the message “Confirming P/O” prints on the purchase order.
•
N - The purchase order is not confirmed with the vendor.
Unusual Del
Determines if the delivery time for this purchase order is out of the ordinary:
•
Y - The delivery time on this purchase order is not the normal length of time for this vendor and is not used to calculate lead time.
•
N - The delivery time for this order is normal and is used to calculate lead time.
Emergency P/O
A Y in this field indicates that the purchase order was created in the Emergency/Direct buyer work queue and action should be taken immediately. In this case the delivery date is not used to calculate lead time.
Instructions
Any special instructions regarding the purchase order. These instructions print on the purchase order.
If standard instructions are set up in the Vendor Maintenance Instructions Screen, these instructions will default for each purchase order for this vendor.
D/S
The direct ship sales order number if this purchase order is linked to a sales order.
P/O Date
The date the purchase order is created. The system uses this date and the received date to calculate average lead time for each product.
Delivery
The date the purchase order is scheduled for delivery and used as the delivery date for each line item. If left blank, the date on each line item is calculated from actual lead time. This date is also used if no Promised Date exists when ranking the vendor’s performance. Refer to Tracking Vendor Performance.
If you change the Delivery Date after the P/O is already entered, you must choose one of the following status codes:
•
C - Date originally confirmed by vendor.
•
E - Date confirmed by vendor after expediting.
•
O - Date overridden by operator.
If the Suppress Freight Days Prompt in Purchasing Parameters is set to N, you must enter the estimated Number Of Freight Days it will take for the merchandise to ship to your warehouse.
Def Delivery Date Days in Purchasing Parameters sets number of default days to add to the current date to display as the Delivery Date in the Purchase Order Header screen.
Promised
The date the merchandise is scheduled for delivery at the warehouse. This date is used with the receipt date to calculate vendor performance and ranking. Refer to Tracking Vendor Performance.
Currency
Displays the currency used by this vendor. The default is from Currency Ind. in Vendor Maintenance > F5-Add Info > F8- Currency.
Curr Rate
Displays the currency exchange rate for this vendor. The default is from Currency Exch Rate in Vendor Maintenance > F5-Add Info > F8-Currency. This rate is the exchange rate negotiated for this vendor and overrides the Daily Exchange Rate or the Currency Rate Table on purchase orders, stock receipts, and accounts payable transactions.
F2-Save
Saves the purchase order header information and opens the Purchase Order Entry screen.
F5-Add Info
Opens the P/O Additional Information screen to enter additional information regarding this purchase order.
F6-Ship-From
Opens the Ship-From Address screen to view or edit the vendor’s ship-from address for this purchase order.
F7-Ship-To
Opens the Ship-To Address screen to view or edit the address where the vendor will ship the order.
F9-Import
Opens the Import Codes screen to allow you to enter or verify the import cost factors from Vendor Maintenance, if this is an overseas vendor.
