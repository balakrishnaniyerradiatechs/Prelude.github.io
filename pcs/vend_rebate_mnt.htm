Rebate Maintenance Screen
The Rebate Maintenance screen allows you to enter the information necessary to calculate the rebate amount due from a vendor for purchases during a scheduled time frame.
Menu Path: Purchasing > Maintenance > Vendor Maintenance > F5-Add Info > Action Bar > Rebates
Field and Function Key Descriptions
Field/Function Key
Description
Rebate%
The rebate percentage to apply to the purchases from the vendor during the time frame indicated by the Schedule to calculate the total rebate from the vendor.
Purch Target
The purchase order value that should be met to obtain this rebate percent. If the purchasing target is met for the criteria entered on the Vendor Rebate Report, the system calculates the rebate amount according to the Rebate%. If it is left blank, the rebate amount still is calculated according to the Rebate%. This field is not required.
Schedule
The frequency by which the purchasing target should be met to obtain the rebate:
•
M - Monthly.
•
Q - Quarterly.
•
Y - Year End.
Program
The description of the rebate program.
Comments
Any comments pertaining to the rebate program. These print on the Vendor Rebate Report.
F2-Save
Saves the vendor rebate information.
