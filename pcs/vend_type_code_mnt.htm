Vendor Type Code Maintenance screen
The Vendor Type Code Maintenance screen allows you to create a code to assign to a vendor that restricts purchase order types for the vendor.
Menu Path: Purchasing > Maintenance > Vendor Type Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Vendor Type Code
The 2-digit alphanumeric code to identify the type code. You can enter this code in the Vendor Types field in Vendor Maintenance to restrict the vendor to the Valid P/O Types listed OK.
Description
A description of the type code.
Valid P/O Type
The purchase order types that can be entered for vendors with this Vendor Type Code. Refer to Purchase Order Types for a list of purchase order types.
F2-Save
Saves the vendor type code information.
F4-Del
Deletes the vendor type code.
