Additional Information Screen
The Vendor Maintenance Additional Information screen includes document options, rounding options, vendor line settings, P/O information, keywords, warranty information, importing information, and currency.
Menu Path: Purchasing > Maintenance > Vendor Maintenance > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Fax Expedite Flag
Determines if the vendor is faxed an expedite report for past due items. Only applies if the Fax Vendor Expedite Reports flag for the selected warehouse in Nightly Processing is set to Y:
•
Always - A report is always faxed to the vendor every single night that purchase order line items are past due.
•
Yes - Fax a report to the vendor at the time a purchase order line item becomes past due.
•
No - Do not fax a report to the vendor. The information goes into the buyer’s queue and on the Nightly Processing reports for expediting.
Accept Backorders
Indicates if the vendor ships backordered quantities:
•
Y - Backordered quantities remain on the purchase order when stock receipts are updated.
•
N - Backorders are deleted when stock receipts are updated. The Purchase Order Log Inquiry displays the LBC code when a line item is deleted due to a backordered quantity.
Charge Duty
Indicates if duty is charged on products purchased from this vendor:
•
Y - Duty is charged. Typically used for imported merchandise.
•
N - Duty is not charged.
Direct Shipments Allowed
Determines if the vendor allows direct shipments to your customer:
•
Y - The vendor direct ships to customers.
•
N - The vendor does not direct ship to customers. You cannot enter a direct order in Purchase Order Entry, Recommended Purchase Order Conversion, or Sales Order Entry.
V/C RMA# Required?
Determines if you must enter a Return Materials Authorization (RMA) number when you enter a vendor claim for this vendor:
•
Y - The vendor requires an RMA number on vendor claims.
•
N - The vendor does not require an RMA number on vendor claims.
Price Tape Obsolete
Determines if a product is flagged as obsolete if it is not on a price tape for this vendor:
•
Y - The product is flagged as obsolete if it is not on the current price tape.
•
N - The product is not flagged as obsolete even if it is not on the current price tape.
Purchase Hold?
Determines if all purchase orders for this vendor should go on purchase hold for manager approval:
•
Y - All purchase orders to this vendor go on purchase hold.
•
N - Purchase orders do not automatically go on purchase hold unless another setting forces the hold.
Refer to Purchase Hold Release.
Add.P/O Per Dir Whse
Indicates if purchase orders to this vendor are split into separate orders when products are flagged as direct warehouse replenishment (e.g., D replenishment methods):
•
Y - Split direct warehouse replenishment purchase orders into separate orders. Only applies to purchase orders created by Daily/Nightly Processing.
•
N - Do not split direct warehouse replenishment purchase orders into separate orders.
Add.P/O Per Vline
Indicates if purchase orders created by Daily/Nightly Processing allow different vendor lines on one purchase order or if they must be split onto separate purchase orders:
•
N - Neither. Create one purchase order for all vendor lines.
•
R - Regular. Create one purchase order for each vendor line for normal stock orders.
•
D - Direct. Create a purchase order for each vendor line on direct-ship purchase orders.
•
B - Both. Create a purchase order for each vendor line for normal stock and direct ship orders.
Std. Discount%
The standard discount percent for all products purchased from this vendor. Expressed as a discount percentage from the gross price to determine the net price of the merchandise. You can override this value at the vendor line level. Refer to Std.Discount% in Vendor Line Maintenance.
Vend Security Access
Allows you to control which products users can view. You must set Vend Security Access to Y in Order Processing Parameters > F5-Add Info to use this feature.
You can create security codes in Security Access Code Maintenance and then assign the codes to a user in User Identification Maintenance > F6-Add Info > F5-Cust Sec. Each user only has access to vendors with a matching security code.
If this field is left blank, all users have access to this vendor.
Imprint Mgr
Override manager ID for imprinting requirements for products requested on sales orders purchased from this vendor. This ID overrides the Imprinting Manager set at the warehouse level (Systems menu > Maintenance > Warehouse Manager Maintenance).
Purchasing Manager
Override manager ID for the manager who reviews this vendor’s purchase orders and releases them from purchase hold. This ID overrides the ID at the warehouse level.
Cage Code
A code to print on the DD250 form (block 16) immediately following the product number, if the vendor is the product’s primary vendor.
Freight Terms Code
Provides a default freight consolidator for purchase orders entered for this vendor (Purchase Order Entry > F5-Hdr Scr > F5-Add Info > Freight Terms). Refer to Entering Freight Terms Codes.
P/O Document Option
The default to determine how purchase orders, blanket purchase orders, debit memos, and vendor requests for quotes are sent to the vendor:
•
B - Print and fax.
•
E - Email only.
•
M - Print and email.
•
N - Print only.
•
Y - Fax only.
You can override the default for each of these when you enter each transaction type, except for debit memos.
Vendor Claims Doc Option
The default to determine how vendor claims are sent to the vendor:
•
B - Print and fax.
•
E - Email only.
•
M - Print and email.
•
N - Print only.
•
Y - Fax only.
You can override the default in Vendor Claim Entry.
Warranty Tag Type Default
The default to use in Inventory Adjustments Entry and Sales Order Entry for warranty items returned to this vendor. You can override this value at the product level:
•
R - Return.
•
S - Scrap.
Does Vendor Barcode
Defines when barcode print:
•
Y - The vendor barcodes so barcodes do not print when stock is received.
•
N - The vendor does not barcode so barcodes print when stock is received.
•
L - The vendor does not barcode and barcodes print when stock is received, but only on receiving documents.
The setting can be overridden in Product Maintenance, Product Warehouse Maintenance, and Product Vendor Maintenance.
Language Indicator
The default language for documents sent to the vendor (e.g., purchase orders) if using Optio:
•
1 - English.
•
2 - French.
•
3 - Spanish.
•
4 - Portuguese.
Def Qty Type For Credit
Default inventory for credit memos when items from this vendor are returned from customers:
•
D - Damaged.
•
R - Return to stock.
•
S - Scrapped.
•
W - Warranty.
Round Pack Qty
Indicates how to round recommended package quantities in Nightly Processing:
•
N - Rounds to the nearest standard package quantity unless the quantity is less than one whole package quantity or it exceeds annual usage.
•
Y - Round up to the nearest package quantity. Recommends at least one package quantity if the required quantity exceeds annual usage.
•
U - Round up to the nearest package quantity even if it exceeds annual usage. Stock products only.
•
R - Round up to the nearest package quantity even if it exceeds annual usage. Stock and nonstock products.
•
B - Do not round to the nearest package quantity.
Only applies if the vendor is the primary vendor in Product Vendor Maintenance for a product, and a package quantity is established. Refer to Assigning a Primary Vendor.
ISO Qualified
If your company requires ISO Qualified vendors, you can indicate if this vendor is included in the ISO Qualified preference, and if inspection is required when products are received:
•
Blank - The default setting if ISO Qualified vendors are not applicable for your company. No inspection is required.
•
Y - To include this vendor in the ISO qualified preference as qualified. (Inspection Required in the purchase order P/O Detail Information screen > F5-Add Info is set to N).
The Vendor ISO Information screen appears to allow you to enter the Date Qualified and the Qualified By user ID.
•
P - To include the vendor in the ISO qualified preference as Non-Preferred. (Inspection Required in the purchase order P/O Detail Information screen > F5-Add Info is set to N). Enter the Date Qualified and the Qualified By user ID.
•
N - To include the vendor in the ISO Qualified preference as Disqualified. (Inspection Required in the purchase order P/O Detail Information screen > F5-Add Info is set to Y). Products received from this vendor must be inspected and will not automatically be allocated to backorders until released.
Floor Plan Vendor
Determines if the vendor uses A/P floor planning:
•
Y - The vendor is an A/P floor plan vendor. With A/P floor planning, your company acts as a dealer and receives material financed through the vendor for an agreed amount of time and conditions.
•
N - The vendor is not an A/P floor plan vendor.
Vendor Type
The vendor type code that defines valid purchase order types for this vendor. Leave blank to accept all P/O types. Nightly Processing checks this field to determine the P/O type to recommend for the Recommended Purchase Order Action Report. Refer to Creating and Assigning Vendor Type Codes.
P/O Minimum
The minimum dollar amount required for purchase orders to this vendor. If you enter a purchase order that does not meet this minimum, a warning displays and the purchase order goes on hold.
Direct P/O Minimum
The minimum dollar amount required for direct purchase orders to this vendor. If you enter a direct purchase order that does not meet this minimum, a warning displays and the purchase order goes on hold.
Manufacturer Code
The Manufacturer Code to default in the MFG# field on the Lot Number Information screen in Stock Receipts Entry for lot-controlled items. These codes are established in Manufacturer Maintenance in the Inventory Module.
Vend Consign Flag
 
Determines the default setting for the Vend Consign Flag in the P/O Additional Information screen and the Purchase Order Requisition Header screen:
•
N - The Vend Consign Flag defaults to N when purchasing from this vendor.
•
Y - The Vend Consign Flag defaults to Y when purchasing from this vendor. When the sales order linked to the purchase order is invoiced, the vendor consigned inventory is updated as sold and available for payment.
F2-Save
Saves the additional information for this vendor.
F5-Keywords
Opens the Additional Keywords Screen to allow you to assign keywords to use as cross references to help in vendor searches.
F6-Warranty
Opens the Warranty Information Screen. You can enter warranty information for defective merchandise returns to the vendor.
F7-Importing
Opens the Import Information Screen to allow you to enter the import factors established in Import Cost Factors Maintenance. Refer to Maintaining Import Cost Factors.
F8-Currency
Opens the Vendor Currency Screen to enter currency information for the vendor.
Action Bar
Rebates
Opens the Rebate Maintenance Screen to allow you to enter the rebate percentages applicable if purchasing targets are met for this vendor.
