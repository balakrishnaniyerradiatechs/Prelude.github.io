Customer Fill Rate Report
The Customer Fill Rate report displays the percentage of stock products that were available when sales orders were first entered for a customer, not when invoiced. This report can help to analyze how well you are servicing a customer.
Products That Will Not Appear on the Fill Rate Reports
If the following conditions exist for a product/warehouse, it will not update the fill rate:
•
The stocking status is other than S (stock).
•
Both the min and max quantities are set to zero.
•
It has an order point override of zero.
•
The calculated order point is zero and it does not have an override (other than zero).
Fill Rate in File Update Parameters must be set to Y to capture fill rate information.
 
Menu Path: Purchasing > Reports > Fill Rates at Order Entry Reports > Fill Rate By Customer
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Customer. The customers to include on the report
•
Warehouse. The warehouses to include on the report.
•
Order Date Range. The beginning and ending order entry dates to include.
•
Product Line. The product lines to include on the report.
Column Descriptions
Column Heading
Description
Whse
The selling warehouse.
Def Whse
The default shipping warehouse number, if one is established for the customer in Customer Maintenance.
Cust#
The customer number on the sales order.
Product
The products entered on the sales order.
Vendor
The primary vendor established for this product/vendor record.
OrdQty
The quantity of each product ordered within the selected date range.
Available
The quantity available to pick the first time the pick ticket printed.
Reorder Point
The quantity set up as the re-order point for this product/warehouse record. When the available quantity of the product falls below this number, Nightly Processing recommends a buy quantity.
Thaw Date
If the product/warehouse has frozen purchasing parameters, then this is the date the product goes off frozen controls.
Fill Rate%
The fill rate compared to the number of lines on the sales orders (Filled/#Lines).
If a line is 100% shipped then the fill rate for that line is 100%. But if the line is less than 100% shipped, then the fill rate for that line is zero.
