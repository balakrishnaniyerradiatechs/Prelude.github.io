Setting Up a Buyer
Each employee responsible for purchasing material from vendors is set up with a buyer identification. The buyer must first have a user ID from User Identification Maintenance. Once the Buyer is set up in Buyer Maintenance you can go back into User Identification Maintenance, enter the Buyer, then indicate if they can enter other buyers’ numbers in the Buyer Queue Inquiry.
You can then assign buyers to each vendor in Vendor Maintenance or Vendor Line Maintenance. All open issues for each vendor are updated in the buyer’s work queue. Refer to the Buyer Queue Inquiry.
To set up a buyer number:
1
Select Purchasing > Maintenance > Buyer Maintenance to access the Buyer Maintenance screen.
2
Enter the code for the Buyer to create or edit. This code is assigned to vendors to designate purchasing responsibility. You may want to use a number (rather than initials) to accommodate employee turnover easily without having to change the buyer for each vendor in Vendor Maintenance or Vendor Line Maintenance.
3
Enter a Description or name of the buyer.
4
Enter the User Id of the buyer. This must be a valid user ID as assigned in User Identification Maintenance.
5
Press F2-Save.
Buyer Inquiry
To view the buyer assigned to a buyer number, select Purchasing > Maintenance > Maintenance Inquiries > Buyer Inquires.
