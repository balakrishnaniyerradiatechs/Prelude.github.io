Vendor Line Maintenance Screen
The Vendor Line Maintenance screen allows you to enter discount and buying information for a vendor line. You can also set buying targets and view vendor statistics.
Menu Path: Purchasing > Maintenance > Vendor Line Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor associated with the vendor line.
Warehouse
The warehouse where this vendor line is used. Leave blank to use this vendor line for all warehouses.
Vendor Line
The vendor line name or number to identify the group of products the vendor sells. Vendor lines group products for discounting or forecasting. The vendor line number can have up to 6 alphanumeric characters.
Description
The extended description of the vendor line.
Buyer
The buyer code for the person responsible for purchasing products from this vendor line. Recommended purchase orders generated by Nightly Processing are sent to this buyer’s work queue. If this field is blank, the buyer assigned in Vendor Maintenance applies.
Transfer Buyer
The buyer number for the person responsible for transfers generated within this vendor line. Recommended stock transfers generated by Nightly Processing are sent to this buyer’s work queue. This overrides the Transfer Buyer set in Vendor Maintenance.
Inventory Parameters > F5-Add Info > Rec. Transfers By Buyer must be set to Y.
DS/Emg Buyer
The buyer code for the person responsible for direct ship and emergency purchase orders for the vendor line.
Std. Discount %
The standard discount percent that applies to all products purchased in this vendor line. Expressed as a discount percentage from the gross price to determine the net price of the merchandise. If left blank the discount percentage from Vendor Maintenance is used. Refer to Std.Discount in Vendor Maintenance in the Additional Information Screen. The percentage is added to any additional discount from the target at the vendor line level.
Seasonal Flag
Determines if the vendor line is seasonal, meaning a peak in sales during certain months of the year (e.g., Christmas, summer):
•
Y - The vendor line is seasonal.
•
N - The vendor line is not seasonal.
Rev Cycle Target Type
The review cycle target type that identifies how the Rev Cycle Target Amt (following field) is displayed:
•
$ - Dollars.
•
# - Pounds.
•
U - Units.
•
O - Other. You can enter any type of purchase target to calculate the review cycle days. It can define the volume of products such as a truckload or an overseas shipping container. The Volume Disc Mult in Product Vendor Maintenance determines the multiplier to use to provide volume discounts.
Rev Cycle Target Amt
The amount used to calculate the Review Cycle Days (In the Vendor Line Target Parameters Screen) for the vendor line. For example, if you enter $ in the Rev Cycle Target Type field, enter a dollar amount in this field.
Pur Ov. Lead Time
The purchasing override lead-time number of days to calculate order controls. This field overrides the average lead-time calculated for each product within each warehouse for this vendor. Provides the easiest way to establish a new lead-time for all products from this vendor within this vendor line, regardless of past lead-time calculations.
F2-Save
Saves the vendor line information.
F4-Del
Deletes the vendor line record.
F5-Targets
Opens the Vendor Line Target Parameters Screen to allow you to establish purchasing parameters for a vendor line.
F6-Statistics
Opens the Vendor Line Statistics Screen to display purchasing information in dollars, units, pounds, and other amounts for this vendor line. You can enter override values.
