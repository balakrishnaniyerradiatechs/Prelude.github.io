P/O Additional Info screen
The P/O Additional Info screen allows you to enter additional information for each line item on the purchase order.
Menu Path: Purchasing > Purchase Order Entry > Detail Information screen > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Inspection Req
Determines if the item is to be inspected upon receipt if quality inspection procedures are used by your company. Refer to the ISO Qualified flag in Vendor Maintenance > F5-Add Info.
Thaw Date
The date when the product is considered on order for purchasing. Until this date, the quantity of this item on this purchase order is excluded from the Nightly Processing recommended purchasing calculations. Refer to Entering Line Item Thaw Dates.
Original Line#
You can enter a different number here than the system assigned number for situations where the vendor refers to a line number.
If you enter a decimal before the line number, the system recognizes this line in Stock Receipts Entry.
Vendor Contact#
The vendor contract number associated with this line item, if applicable.
Vendor Quote#
The vendor quote number associated with the line item, if applicable.
Vendor Salesman#
The vendor salesman number associated with this line item, if applicable.
Unusual Delivery
Determines if this purchase order line item has an unusual delivery time and if the delivery date should be used to calculate lead time:
•
Y - The lead time on this item is unusual and should not be considered to calculate lead time.
•
N - The lead time on this item is normal and should be used to calculate lead time.
This setting overrides the setting at the header level.
F2-Save
Saves the data in the additional information screen.
