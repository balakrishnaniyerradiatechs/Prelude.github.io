Vendor Line Target Parameters Screen
The Vendor Line Target Parameters screen allows you to set up target requirements and discount opportunities provided by the vendor for products within this vendor line. Nightly Processing uses this information to generate recommended purchase orders. Refer to Using Discount Tables in Vendor Line Maintenance and Re-costing Purchase Orders. You can generate the Vendor Line Missed Target Report during the End Of Month procedures to view and analyze vendor targets.
Menu Path: Purchasing > Maintenance > Vendor Line Maintenance > F5-Targets

Frt Paid Amount
The freight paid amount of the Frt Paid Type that you must order for the vendor to pay the freight. For example, if $ is entered in Frt Paid Type, enter a dollar amount.
Review Cycle Days
The calculated review cycle for this vendor line. Represents the frequency with which you typically place a purchase order with the vendor.
The field is calculated using the Rev Cycle Target Type and Rev Cycle Target Amount compared to the statistics for this type over the last 12 months.
For example, if the Rev Cycle Target Type is defined as $, Rev Cycle Target Amount is $1000, you purchased $24,000 in the last year, and you placed a purchase order every 2 weeks of the year; your Review Cycle Days is 14.
Refer to Including Future Price Increases for information on how review cycle days affects the recommended order quantity on future price increases.
Rev Cyc Override
An override to the Review Cycle Days for Nightly Processing. Press F6-Recalc Product Parameters to recalculate the purchasing parameters for all products in this vendor line.
Next Review Cycle
The date when this vendor line should be reviewed again. This date is updated when you convert a recommended purchase order.

Target Amount - Ov
The purchasing target override for Target Type – Ov value. For example, if you enter $ in Target Type – Ov, enter a dollar amount.
Turn Target
The number of turns you would like to have for the vendor line for the fiscal year. Turns is calculated by dividing annual usage by the inventory dollars invested in this vendor line. The system uses this value to compare against actual turns on the Vendor Line Turns Report.
Discount Type
The discount type that defines how the system determines break levels. You can enter a discount table to calculate discounts in the (unlabeled) Discount Break Table Number field or you can manually enter the discount amount and percentage in Disc Amt/Disc%. Refer to Using Discount Tables in Vendor Line Maintenance for more information. Choose from:
•
$ - Dollars.
•
# - Pounds.
•
U - Units.
•
O - Other. A predetermined method you define to reach a minimum for purchasing (e.g., volume or cube). Enter a description in the next field.
Description (unlabeled)
Only accessible if O is entered in Discount Type. Enter a description of the method defined for the discount table amount. This description prints as the column heading on the Recommended Purchase Order Action Report.
Discount Break Table Number
(unlabeled)
The discount break table number to use for vendor discounts (refer to Setting up Discount Break Tables). If left blank, you can define the discount break tables below in Disc Amt/Disc%.
Disc Amt/Disc%
Target Amount
If no discount break table number is entered, this is the quantity break level you must reach to receive a discount. Nightly Processing attempts to reach this amount when recommending a purchase order.
Discount%
The discount percent of cost that you will receive when the Target Amount is met on a purchase order.
F2-Save
Saves the vendor line target parameter information.
F5-Add Info
Opens the Vendor Line Targets - Additional Information Screen to allow you to enter lead-time defaults, new product defaults, and forecasting information.
F6-Recalc Product Parameters
Recalculates the product parameters using the new information and any overrides entered.
F8-Comments
Allows you to enter comments regarding the vendor line. The comments print on the Recommended Purchase Order Action Report. To view vendor line comments, refer to Vendor Line Comments Inquiry.
