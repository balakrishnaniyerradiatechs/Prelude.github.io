Vendor Discount Code Maintenance screen
The Vendor Discount Code Maintenance screen allows you to enter on and off discount codes. An on-discount code reflects a negotiated discount amount from a vendor that appears on the invoice. An off-discount code reflects a negotiated amount from a vendor that is not shown in the invoice amount. Refer to Creating and Assigning Vendor Discount Codes.
Menu Path: Purchasing > Maintenance > Vendor Discount Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Discount Code
The code that identifies the on or off discount amount.
Description
The description of the discount code.
Discount %
The percentage discount for this code.
Ln#
The line count.
Div
The division number. Represents the highest possible grouping in your company.
Reg
The region number. A subset of division if division is used.
Loc
The location number. The physical location or warehouse.
Dep
The department number. A subset of warehouse. Typically used to separate expenses within a warehouse.
Ccr
The cost center number. A subset of department if department is used.
G/L Account#
The general ledger account number to use for the discount.
Description
The description of the general ledger account number.
Percent
The percentage of the discount to apply to the G/L Account #.
F2-Save
Saves the vendor discount code information.
F4-Del
Deletes the vendor discount code.
F5-Notes
Displays any notes regarding the vendor discount code.
