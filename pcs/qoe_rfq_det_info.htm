RFQ Detail Information screen
The RFQ Detail Information screen allows you to enter quantity and cost information when entering an RFQ to a vendor from within Quote Order Entry.
Menu Path: Order Processing > Additional Subsystems > Quotation Subsystem > Quotation Entry > Quote Entry Detail Screen > Action Bar > Entries > Request for Quote Entry > ENTER on the Quote number.
Field and Function Key Descriptions
Field/Function Key
Description
Ln#
The line number from the Request for Quotation Entry screen.
Quantity
The quantity that you are requesting for the vendor to quote a price.
U/M
The unit of measure from the product file.
A
Accept. Indicates if everything on the screen is correct:
•
Y - If everything on the screen is correct.
•
N - To make changes to any of the fields.
Rfq Cost
Cost quoted from the vendor.
Ldt
The lead time number of days.
Vendor Product Number
The vendor’s product number for cross reference, if applicable.
F2-Save
Saves the detail information for the product.
