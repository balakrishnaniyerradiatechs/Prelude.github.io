Vendor Verification screen
The Vendor Verification screen appears when you enter a Vendor Number in Purchase Order Entry. The vendor’s name and address appear giving you the option to choose the vendor shown, or to go back to the Vendor Number field to pick another vendor.
The Verify Vendor in POE in Option Parameters > Purchasing Parameters determines if the Vendor Verification screen appears.
Menu Path: Purchasing > Purchase Order Entry
Field and Function Key Descriptions
Field/Function Key
Description
Vend Cust#
If the vendor is also a customer, the customer number displays.
Is This the Correct Vendor (Y/N)
Indicates if the correct vendor was selected:
•
Y - The name and address displayed is correct. Opens the Purchase Order Header screen.
•
N - The name and address displayed are not for the correct vendor.
