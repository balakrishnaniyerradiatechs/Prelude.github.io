Vendor Line Turns Report
The Vendor Line Turns report displays the average annual inventory turnover calculated for a vendor line. This report compares the actual inventory turnover for each vendor line to the target turnover in the Vendor Line Target Parameters Screen. The turnover is calculated by dividing the cost of stock sold in the last 12 months by the inventory value of the current on-hand quantity. The higher the turns the faster the products are selling, which usually indicates which products are more profitable to sell.
Menu Path: Purchasing > Reports > Vendor Turn Reports > Vendor Line Turn
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Division. The highest possible grouping in your company (e.g., Eastern Division, Western Division).
•
Region. A subset of Division if Division is used (e.g., Southeast Region, Northwestern Region).
•
Selling Warehouse. The warehouse responsible for selling the material for the selected vendor lines.
•
Department. A subset of warehouse. Typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse).
•
Cost Center. A subset of Department - if Department is used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales).
•
Vendor Line. Groups products for purchasing related functions (e.g., minimum order, free freight, quantity discounts).
•
Stocking Status. The stocking status of the products to include on the report:
-
S - Stock.
-
N - Non-stock.
-
B - Both stock and non-stock.
-
C - Consignment.
-
A - All (stock, non-stock, consignment).
•
Include Transfer Usage?. Determines if product usage from transfers is included to calculate usage value (cost of goods sold):
-
Y - Include transfer usage to calculate vendor turns.
-
N - Do not include transfer usage. Only include sales order and work order usage.
Column Descriptions
Column Heading
Description
Vline
The vendor line number set up for in Vendor Line Maintenance.
Description
The description of the vendor line.
Vendor
The vendor number
Description
The vendor description.
Ext. Usage
Total cost of the products sold (sales orders, work orders, transfers) in the past 12 months not including the current month, extended by the average cost.
Total Inv
The total cost of the vendor line products currently in on-hand inventory (average cost x on-hand quantity).
Target
The desired number of turns for the vendor line. Established in Vendor Line Maintenance > F5-Targets > Turn Target.
Actual
The actual number of turns for the vendor line for the last 12 months. Calculated by dividing the Annual Cost Of Sales by Inventory Dollars.
Less than Target?
(unlabeled column)
Indicates if the actual turns meets target turns:
•
(*) asterisk - Actual turns are less than target turns.
•
Blank - Actual turns meet or exceed target turns.
