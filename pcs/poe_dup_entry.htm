P/O Duplication Entry screen
The P/O Duplication Entry screen allows you to duplicate an open purchase order from the same vendor or for a different vendor. This option prevents you from having to re-enter a list of part numbers and prices.
Refer to Duplicating a Purchase Order for more information.
Menu Path: Purchasing > Purchase Order Entry > Action Bar > P/O Func > Duplicate Purchase Order
Field and Function Key Descriptions
Field/Function Key
Description
Duplicate P/O #
The open purchase order number to duplicate. To duplicate a purchase order that is received, refer to Duplicating Received Purchase Orders.
Vendor Number
The vendor number and description from the purchase order you are duplicating.
New Vendor Number?
The vendor number for the new purchase order. Leave blank if the new purchase order is for the same vendor.
New Warehouse Number?
The ship-to warehouse number.
Recost P/O?
Determines if the new purchase order uses cost from the original purchase order or from the current system cost:
•
Y - To recost the purchase order line items with the current system cost for the designated vendor.
•
N - To use the cost from the original purchase order.
F2-Save
Duplicates the purchase order and opens the Purchase Order Header screen to edit and complete the order.
