Field Definitions by screen
Additional Information screen
The Additional Information screen allows you to enter vendor payment information for a purchase requisition.
Menu Path: Purchasing > Processes > Purchase Requisition Entry F5> Hdr Scr > F5-Add Info
Field and Function Key Descriptions
Field/Function Key
Description
Std. Discount%
The standard discount deducted from the vendor’s cost on this purchase order.
Vendor Order #
The vendor’s order number, if applicable.
Fax Number
The vendor’s fax number. The system looks for the fax number in the following order:
•
The fax number from the contacts established in Vendor Maintenance > Action Bar > Contacts.
•
The fax number from Vendor Maintenance > F7-Contacts.
•
The fax number from vendor’s main address in Vendor Maintenance.
Freight Terms
The freight code used to identify freight consolidators for a vendor. Refer to Entering Freight Terms Codes.
Fob Description
Freight on board description.
Replacement Equip
 
Determines if the purchase order is for replacement equipment:
•
Y - For replacement equipment.
•
N - Not for replacement equipment.
Language Indicator
Indicates the language to print on the purchase order. The default is from Language Indicator in Vendor Maintenance > F5 Add Info:
•
1 - English.
•
2 - French.
•
3 - Spanish.
•
4 - Portugese.
Currency Ind.
Displays the currency used by this vendor. The default is from Currency Ind. in Vendor Maintenance > F5-Add Info > F9 Currency.
Currency Factor
Displays the currency exchange rate for this vendor. The default is from Currency Exch Rate in Vendor Maintenance > F5-Add Info > F9 Currency. This rate is the exchange rate negotiated for this vendor and overrides the Daily Exchange Rate or the Currency Rate Table on purchase orders, stock receipts, and accounts payable transactions.
Ovr Pay To Vendor
The vendor that you pay for the merchandise on this purchase order. You can override the default set up for Pay to Vendor in Vendor Maintenance > F6 A/P.
Vend Consign Flag
Determines if the purchase requisition is for vendor consigned inventory:
•
Y - The purchase requisition is for vendor consigned inventory. When the sales order linked to the purchase order is invoiced, the vendor consigned inventory is updated as sold and available for payment.
•
N - The purchase requisition is not for vendor consigned inventory.
The Vend Consign Flag in the Vendor Maintenance Additional Information Screen determines the default setting for this flag.
On Discount Code
The on-discount code for the purchase order. You can override the default On Discount Codes set up in Vendor Maintenance > F6-A/P. Refer to Creating On-Discount Codes.
F2-Save
Saves the P/O Additional Information data.
F5-Import Factors
Opens the Import Codes screen to allow you to enter or change import codes.
