Vendor Ship-From Maintenance
Vendor Ship-From Maintenance allows you to set up additional ship-from addresses for each vendor. You can access these addresses from Purchase Order Entry to identify where the material is shipping from. You can also identify which warehouses are authorized to use each ship-from address. These addresses are in addition to the addresses used for purchasing and accounts payable.
When you enter a purchase order:
•
If the system does not find a ship-from address for the vendor, then the Ship-From Number in the Purchase Order Header is set to D to use the default address in Vendor Maintenance.
•
If the vendor has one ship-from addresses, all purchase orders default to this address.
•
If the vendor has multiple ship-from addresses, you must choose a ship-from address in Purchase Order Entry.
Menu Path: Purchasing > Maintenance > Vendor Ship-From Maintenance
The information in the Vendor Ship-From Inquiry screen (Purchasing > Inquiries > Maintenance Inquiries >Vendor Ship-From Inquiry) is the same as the following information in Vendor Ship-From Maintenance.
To print a list of Vendor Ship-From addresses, select Purchasing > Listings > Vendor Ship-From Listing.
Field and Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor number. Refer to Vendor Maintenance.
Ship From
The ship-from number to identify this address for the vendor. Up to 6 alphanumeric characters.
Description
The vendor ship-from location name to display on purchase orders.
Address One
The first line of the ship-from address.
Address Two
The second line of the ship-from address.
Address Three
The third line of the ship-from address.
County
The country of the ship-from address.
City
The city of the ship-from address.
State
The state of the ship-from address.
Zip
The zip code of the ship-from address.
Phone
The phone number of the ship-from address.
Fax
The fax number of the ship-from address.
E-Mail Address
The e-mail address for this ship-from address.
Ship Via
The default recommended ship via code for this ship-from address. If this is blank, the ship via from Vendor Maintenance is used.
F2-Save
Saves the vendor ship-from information.
F4-Del
Deletes this vendor ship-from record.
Action Bar
Contacts
Opens the Contact Vendor Entry screen to allow you to enter additional vendor contact names for this ship-from address, including address information.
Auth Whses
Opens the Authorized Warehouses screen to allow you to specify which warehouses are allowed to use this ship-from address. If this screen is left blank, all warehouses can use this ship-from address.
