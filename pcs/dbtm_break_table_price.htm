Discount Table - Dollar screen
The Discount Table - Dollar screen allows you to enter a price for each break level.
Menu Path: Purchasing > Maintenance > Discount Break Table Maintenance > Table Method = $
Field and Function Key Descriptions
Field/Function Key
Description
Break Level
Defines the minimum break value that qualifies for the corresponding price:
•
If the Break Level Type is M, enter the unit of measure that activates this discount.
•
If the Break Level Type is U, enter the quantity values to determine the discount.
•
If the Break Level Type is $, enter the dollar amount to determine the discount.
Price
The price for the corresponding Break Level. This value appears in Gross and Net-Cost on the re-costed purchase orders.
F2-Save
Saves the break table information.
F4-Del
Deletes the break table.
