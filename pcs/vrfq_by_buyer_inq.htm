Vendor RFQ By Buyer Inquiry
The Vendor RFQ By Buyer Inquiry displays open vendor RFQs for a buyer, that are not yet converted to a purchase order. The inquiry includes open RFQs for all vendors assigned to the Buyer in the Vendor Maintenance Screen.
Open Vendor Requests for Quotes for which no response has been entered also appear in the assigned buyerâ€™s work queue in the Buyer Queue Inquiry. Refer to Vendor Request for Quotes.
Viewing Open Vendor RFQ By Buyer
To view open Vendor RFQs for a buyer:
1
Select Purchasing > Vendor RFQ Subsystem > Vendor RFQ Inquiries >Vendor RFQ By Buyer Inquiry.
2
Enter the Buyer Number or press F3 to search.
The open vendor RFQs assigned to this buyer appear.
Field & Function Key Descriptions
Fields/Function Keys
Description
VRFQ#

VRFQ Date
The date the RFQ was entered.
Response Date
The date a response from this vendor was entered for this RFQ.
F5-Review
Accesses the Request for Quote Inquiry screen to show product and cost information for this vendor RFQ release.
F6-Enter
Allows you to enter a vendor RFQ number for direct access.
