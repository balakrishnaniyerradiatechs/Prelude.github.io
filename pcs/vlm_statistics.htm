Vendor Line Statistics Screen
This screen displays monthly statistical information products from the last 3 years. The statistics are defined in dollars, units, weight, or other established methods and are used to calculate Review Cycle Days if Review Cycle Targets are defined. You can override the values, then press F6-Recalc (in the Vendor Line Target Parameters Screen) to recalculate the Review Cycle Days.
Menu Path: Purchasing > Maintenance > Vendor Line Maintenance > F6-Statistics
Field and Function Key Descriptions
Field/Function Key
Description
Month
The months of the year.
Purchase Amounts
The dollar amount received from the vendor line in the corresponding month.
Override
An override dollar amount to calculate the Review Cycle Days.
Units
The number of units received in this vendor line for the corresponding month.
Override
An override number of units received to calculate the Review Cycle Days.
Pounds
The weight received in this vendor line for the corresponding month.
Override
An override weight to calculate the Review Cycle Days.
Other
The quantity if any other type of purchase order type was defined.
Override
An override quantity if any other type of purchase target was defined. Used to calculate the Review Cycle Days.
F2-Save
Saves the vendor lines statistics information.
