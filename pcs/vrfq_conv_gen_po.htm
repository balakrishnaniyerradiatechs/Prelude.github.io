Generate P/O’s For Vendor RFQ screen
The Generate P/O’s For Vendor RFQ screen allows you to generate purchase orders for the vendors you selected from the vendor RFQ responses. Refer to Converting a Vendor RFQ to a Purchase Order.
Menu Path: Purchasing > Vendor RFQ Subsystem > Vendor RFQ Conversion > F2-Save
Field and Function Key Descriptions
Field/Function Key
Description
Vendor
The vendor number you selected for the purchase order conversion.
Description
The vendor description.
Whse
The warehouse where the material is to ship.
Type
Cost type description. For information only.
Line
The number of line items included on the purchase order.
P/O#
The purchase order number once the purchase order is generated.
Printed
Indicates if the purchase order printed (or faxed or emailed).
F2-Generate All
Generates purchase orders for all the vendors listed.
F5-Edit
Opens the Purchase Order Entry screen to allow you to edit or add items on the purchase order. You can generate the purchase order from this screen.
F6-Generate
Generates a purchase order for only the vendor that is highlighted.
F8-Review
Opens the Purchase Order Display screen once the purchase order is generated.
F9-Continue
Closes the screen.
