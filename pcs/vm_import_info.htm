Import Information Screen
If you import merchandise from overseas vendors, you can use the Import Information screen to enter the import codes from Import Cost Factor Maintenance (refer to Maintaining Import Cost Factors).
Menu Path: Purchasing > Maintenance > Vendor Maintenance > F5-Add Info > F7-Importing
Field and Function Key Descriptions
Field/Function Key
Description
Overseas Vendor
Determines if the vendor is an overseas vendor:
•
C - The vendor is an overseas vendor. Used for Container tracking. Refer to “Receiving On The Water Merchandise” in the Inventory Module.
•
N - The vendor is not an overseas vendor.
•
P - The vendor is an overseas vendor. You can use On The Water Maintenance to create a new purchase order when receiving from this vendor. Refer to “On The Water Receiving” in the Inventory Module.
•
Y - The vendor is an overseas vendor. Import cost factors are required in Purchase Order Entry. This option applies if you use shipping bins for On-The-Water receiving. Refer to “On The Water Receiving” in the Inventory Module.
Country
The country code that represents where the product is manufactured.
Import Code
The import cost code that represents the charges incurred when purchasing from this vendor. You can override this code in the Purchase Order Header Import Codes screen. To define your own import codes, refer to Maintaining Import Cost Factors.
The following are examples of commonly used codes:
•
BRK - $ Dollars - Overseas brokerage.
•
CONS - U Units - Overseas consolidation.
•
DUTY - $ Dollars - Overseas duty.
•
IF - V Volume - Inland freight.
•
LW - V Volume - Overseas loading and wharfage.
Description
The description of the import code.
Currency
The currency code for expressing import charges. For example, the currency for expressing brokerage fee charges when importing from this vendor.
Factor
A factor (percentage) to multiply by the cost to calculate the import fee. This overrides the factor in the Import Cost Factor Maintenance screen.
F2-Save
Saves the import code information.
