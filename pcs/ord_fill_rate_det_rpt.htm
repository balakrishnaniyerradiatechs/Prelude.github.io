Fill Rate Detail Report
The Fill Rate Detail report displays the number and percent of stock items that were available when sales orders were first entered, not when invoiced. This report can help determine if stock outages are a common problem. You can run the report by product line and/or by warehouse.
Products That Will Not Appear on the Fill Rate Reports
If the following conditions exist for a product/warehouse, it will not update the fill rate:
•
The stocking status is other than S (stock).
•
Both the min and max quantities are set to zero.
•
It has an order point override of zero.
•
The calculated order point is zero and it does not have an override (other than zero).
Fill Rate in File Update Parameters must be set to Y to capture fill rate information.
 
Menu Path: Purchasing > Reports > Fill Rates at Order Entry Reports > Fill Rate Detail Report
Selection Criteria
Use the following criteria to limit the information to print on the report:
•
Warehouse. The warehouse where the orders were filled.
•
Order Date Range. The beginning and ending order entry dates to include.
•
Product Line. The product lines to include in the report.
Column Descriptions
Column Heading
Description
Whse
The warehouse that shipped the items.
Ord Date
The date the sales order was entered.
Order #
The sales order number that contains the product(s).
Pline
The product line number.
Product
The product number.
Description
The description of the product.
OrdQty
The quantity originally ordered on the sales order.
PckQty
The quantity of the item that was available for picking the first time the pick ticket printed.
#Lines
The number of lines on the sales order that contain the corresponding product number.
Filled
The number of lines on the sales order that were shipped complete.
Percent
The fill rate per line. If a line is 100% shipped then the fill rate for that line is 100%. But if the line is less than 100% shipped then the fill rate for that line is zero. The total fill rate for each warehouse or product line is the percentage of lines items that shipped 100% complete (#Compl divided by #Lines).
