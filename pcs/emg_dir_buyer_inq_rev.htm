Emergency/Direct Buyer Inquiry Review screen
The Emergency/Direct Buyer Inquiry Review screen displays emergency and direct line items and allows you to generate purchase orders. Refer to Emergency/Direct Line Items in the Buyer Queue Inquiry for more information.
Menu Path: Purchasing > Inquiries > Buyer Queue Inquiry > Emergency Direct Line Items > F5-Review > F5-Review
Field and Function Key Descriptions
Field/Function Key
Description
Buy?
Set to Y for each item to be purchased. If the item displays one of the following hold codes in Buy?, it cannot be placed on a purchase order:
•
H - Credit hold.
•
O - Administrative order hold.
•
I - Administrative invoice hold.
•
M - Imprint hold.
•
S - Suspended.
•
L - Locked by another process.
Cost
The cost of the item from the sales order.
Via
The ship via method from the sales order.
Description
The description of the ship via method.
Quantity
The quantity of the item as entered on the sales order.
U/M
The unit of measure of the item as entered on the sales order.
Req-Date
The date the item is required by the customer.
S
The status of the items:
•
E - Emergency.
•
D - Direct.
•
B - Both emergency and direct.
Product Number
The product number.
Description
Description of the product.
Linkage
Document that the product is linked to. The prefix indicates the document type:
•
S - Sales order.
•
T - Transfer.
•
W - Work order
Customer
The customer on the linked order.
F2-Save
If you entered a Y in Buy? for any of the line items, the Generate Purchase Orders screen opens to allow you to edit and generate the purchase orders. Otherwise, you are brought back to the Emergency/Direct Buyer Inquiry screen.
F5-Review
Opens the Sales Order Display screen to allow you to review the linked order.
F6-Resort P/O Disp
Resorts the purchase order display.
Action Bar
Change Buyer
Allows you to change the buyer number to send the item to another buyer’s work queue, if you have access to their work queue.
Change Vendor
Allows you to assign a new vendor to the backordered item.
