Rental Parameters
Path: System Parameter > Maintenance > Option Parameters > Rental Parameters
•
Periodic Billing: determines if rental orders are considered for periodic billing. This option is usually used for long term contracts. This is the default for Periodic Billing in Rental Order Entry and may be changed for an order.
•
Hourly Billing Cycle: the number of hours in an hourly rental to determine the next periodic billing cycle. This is usually the number of rent hours in a day (e.g., 09).
•
Daily Billing Cycle: the number of days in a daily rental to determine the next periodic billing cycle. This is usually set to 01 for one day.
•
Weekly Billing Cycle:the number of days in a weekly rental to determine the next periodic billing cycle. This is usually set to 07 (1 week = 7 days).
•
Monthly Billing Cycle: the number of days in a monthly rental to determine the next periodic billing cycle. This is usually set to 28 (1 month = 28 days).
•
Cred Chk in Periodic Billing?: determines if periodic billing is included in the credit check process. Select Y to include periodic billing in credit checks. Select N to not include periodic billing in credit checks.
