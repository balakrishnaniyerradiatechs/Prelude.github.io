Periodic Billing
This process is used to bill customers on a monthly basis for items rented over an extended period of time. This procedure is run to select open rental orders to see if they can be billed. The process tries to bill for any line item on the rental order that has a unit of measure within the maximum days in cycle. A periodic billing report may be run to check orders to be billed. These orders are picked up on the next batch invoice run and billed.
