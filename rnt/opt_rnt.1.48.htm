Invoice Processing (/IP)
Invoice processing will pick up all orders set to invoice through periodic billing and print an invoice for them. It updates the rental order with the bill-through date, which can be seen in Sales Order Inquiry in Billed Thru on the F10 > Examine screen for the rental line.
