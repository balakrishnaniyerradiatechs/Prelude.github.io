Fixed Assets Menu
This menu provides Fixed Asset processes to use with the rental equipment. See documentation on Fixed Assets for information on these procedures.
Manual Invoice Entry (/APIE)
Path: Financial > Accounts Payable > Manual Invoice Entry
This program is used to pay non-inventory invoices. If an invoice is entered using the General Ledger account number for Equipment Id Numbers, the following screen displays to enter equipment Id information.
•
Equipment ID#: Enter the EIN to be maintained.
•
Update Inv? Determines whether or not to update this invoice with a new cost.
•
Reas Code: Enter the reason code for the cost adjustment.
•
FA Charge Type: Enter a fixed asset tracking charge type to be expensed for the amount of this transaction (defaults from Asset Tracking Charge Type in Chart of Accounts Maintenance).
•
Serial #: Displays the serial number for this equipment Id.
•
Product #: Displays the product number associated with this equipment Id.
•
Description: Displays the description for the product number.
