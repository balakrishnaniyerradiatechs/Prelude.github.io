Fuel Type Maintenance (/FUTM)
To use fuel charges in the rental module, set up fuel types and prices in Fuel Type Maintenance.
Path: Rental > Maintenance > Fuel Type Maintenance
•
Type: the type of fuel charge.
•
Description: the description of the fuel type that appears when this fuel type is used.
•
Price: the selling price of the fuel type.
