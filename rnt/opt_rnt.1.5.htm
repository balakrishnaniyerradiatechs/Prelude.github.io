Rental Lost/Damaged Quantity
Rental Order Entry recognizes and automatically bills accordingly for any rental items lost or damaged. If two pieces of a rental item are on rent and one piece is damaged, you must return both and enter a quantity in Lst/Dmg. The rental fee for the item that was lost or damaged applies because the item was rented. The rental income of the lost/damaged piece should be accounted for in the course of the rental return.
Default Parameters Maintenance (/DSS)
Rental lost/damage comments in Default Parameters Maintenance default when updating the quantity in LST/DMG in Rental Order Entry.
Path: System > Maintenance > Option Parameters > Default Parameters Maintenance > F5-Add Info
•
Rental Lost/Damage Comment: the first line of the default comment for lost or damaged items.
•
Rental Lost/Damage Comment: the second line of the default comment for lost or damaged items.
Rental Order Entry (/ROE)
Once you enter a quantity in Lst/Dmg, the Lost/Damage Information screen is displayed.
Path: Rental > Rental Order Entry
•
Selling Price: the selling price of the item (default is the product’s Rental Retail Value).
•
Comments: the comments about the lost or damaged material (defaults to comments from Default Parameters Maintenance). You can enter multiple comment lines, which are added as C type comment lines.
Once you save the Lost/Damaged Information screen, a new line item is created for the quantity lost or damaged. The rental order includes the comments from /DSS.
