Rental Equipment Subsystem
The Equipment Subsystem, in the Rental module, maintains equipment ID numbers (EIN) along with any associated products, parts lists and scheduled maintenance. The EINs are created internally when the Inventory Type Conversion process is run to create an asset record for an item converted to a rental product.
