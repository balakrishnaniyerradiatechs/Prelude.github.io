Scheduled Maintenance Code Maintenance (/SMCM)
This maintenance program is used to define scheduled maintenance routines performed on equipment, including the days and hours between each scheduled maintenance. You can use these codes in the Parts List in Equipment Maintenance.
•
Maintenance Code: Enter the scheduled maintenance code to be added or changed. This code defines a scheduled maintenance routine performed on equipment.
•
Description: Enter the description of the maintenance code.
•
Days: Enter the number of days between each maintenance period.
•
Hours: Enter the amount of use between each maintenance period. This could be hours of use, mileage, or wear measurements for items such as drill bits.
•
F2- Save: Saves the scheduled maintenance code information.
