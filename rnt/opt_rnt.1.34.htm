Asset Maintenance (/ASM)
Path: Rental > Equipment Subsystem > Fixed Assets > Maintenance > Asset Maintenance
Each rental asset record can be maintained in Asset Maintenance.
•
Action > Master Asset Entry: Allows an asset record to be established and linked to a master asset record. The cost of the linked asset is rolled up to the master asset.
•
Action > Sub Asset Disp: Displays the asset records that were processed in Master Asset Entry. These sub assets are linked to the master asset.
NOTE: When the Inventory Type Conversion Entry process creates the asset record, the Master Asset Entry screen is not automatically updated with the sub assets. A manual step is required to link each master asset to a sub asset for each equipment id.
You can create a fixed asset record for 05 type expense work orders from in Work Order Completions. These asset records are recognized as sub asset records. The Master Asset Entry screen references the sub asset record and the Sub Asset Disp screen now references the Master Asset record.
•
Depreciation Method: This must be set to M to access the F6-Manual Entry screen.
•
F6-Manual Entry: When a rental item is invoiced that has the Depr on Rent Revenue flag set to Y and a Depr Percent value established in Product maintenance, the system updates the rental asset record with a manual depreciation entry to post to the G/L and fixed asset record the next time the Manual Deprecation Update process is performed.
