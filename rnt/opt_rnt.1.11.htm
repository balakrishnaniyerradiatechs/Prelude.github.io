Partially Checking In a Rental Item
Path: Rental > Processes > Rental Checkin Processing (/RPUE)
You can check in rental items partially or completely using Rental Checkin Processing. If an order is partially checked in, use the rental checkin history number to return the remaining items. The following message appears:
This checkin document has been processed. Do you wish to reopen?
Select No to choose another checkin number or choose Yes to reopen the checkin document and return the remaining items on the rental order. The original rental checkin history record is erased and updated with the new rental checkin information.
Reports & Injuries
Path: Rental > Inquiries
Rental Checkin Inquiry (/RCKI)
This inquiry displays rental orders with a rental check-in document no actual checkin.
Rental Checkin Document History Inquiry (/RPHI)
This inquiry displays rental orders that were processed through Rental Check-In Entry.
Order Log Inquiry (/OLI)
Path: Order Processing > Inquiries > Sales Order Inquiries > Sales Order Log Inquiry
The following codes were added to Order Log Inquiry for rental check-in documents.:
•
SPL - the rental order is split due to a partial return of the rental items. Each line item that is partially returned creates a SPL record. For example, if two rental items are partially returned, two split entries are displayed in Order Log Inquiry. Press F5 to view the split line items on partially returned orders.
•
CKO - the checkin document is reopened in Rental Checkin Processing to receive the remaining rental items or allow the checkin document to be deleted.
•
CHD - the checkin history document in deleted.
Open Rental Checkin Report (/OPEN.CHECKIN.RPT)
This report lists open rental checkin documents based on the criteria entered.
Path: Rental > Reports Open Rental Checkin Report
Rental Checkin History Xref Rebuild
A rebuild is available for rental checkin history transactions.
Path: System Parameter > Processes ReBuild Menu Ordering Control Order Processing Inquiry Xref Rental Checkin History Xref
Rental Checkin History Reprint
Path: Rental > Document Printing > Rental Checkin History Reprint (/RCHR)
This option allows you to reprint rental checkin history documents based on the Checkin History Number.
For example, If a checkin was reopened and multiple checkin history records exist (-1,-2), the last checkin history record is the only checkin history number available to be reprinted. For example, if checkin #000011 was processed as a partial return (not all items were returned) and reopened to return the remaining items, rental checkin #000011-1 would be created. Checkin #000011-1 is now the checkin document that is available for reprint. If rental checkin #000011 was entered to be reprinted, a message would display informing the user that reprints are not available for reopened history records.
