Rental Rate Method
The rental rate method determines the rate the rental item is charged. You can define the rental rate method at the customer or system level.
Rental Parameters
Path: System Parameter > Maintenance > Option Parameters > Rental Parameters
In Rental Parameters, you can set the default Rental Rate Method that is used if a customer does not have a rental rate method: Options are:
•
A - automatically review the rates and use the best rate based on the time material is rented. This method would auto prorate and calculate the best rate for the rental item.
•
B - automatically review the rates and use the best rate based on the time material is rented up to the first billing (either return of the material or the first periodic billing). Once the first bill in generated and material is returned, the billing is prorated by day.
•
N - use the rate and rental period from the rental order and round to the nearest whole rental. For example, 10 days on a daily weekly rental would charge 2 weeks.
•
C - use a single rate based on the Billing Days. If Billing Days are M-F and rental item is billed for 7 days, the billing rate is 1.4 (1 week + 2/5). The total number of billing days is the denominator (M-F = 5 billing days). The Billing Days are set in F5-Add Info.
Warehouse Maintenance (/WM)
The Billing Days in Warehouse Maintenance overrides the system parameter flags.
Path: Warehouse Maintenance > Action > Addl Opts > Rental Information
Customer Maintenance (/CM)
You can override the default Rental Rate Method in Customer Maintenance.
Path: Order Processing > Maintenance > Customer Maintenance > F7-Order > F5-Add Info > F6-Rental
Rental Order Entry (/ROE)
The rental rate method defaults in Rental Header screen in Rental Order Entry from the Customer Maintenance, if applicable, or Rental Parameters.
Path: Rental > Rental Order Entry
You can change the method for each rental item in Line Item Rental Detail > F5-Add Info.
