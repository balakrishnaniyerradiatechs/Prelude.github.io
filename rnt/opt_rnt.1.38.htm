Rental Parameters
Path: System Parameter > Maintenance > Option Parameters > Rental Parameters
•
Rental Pre-Billing: determines if rental orders should be pre-billed for the expected rental period. This field is the default for Rental Pre Billing in Rental Order Entry and may be changed by order.
•
Switch to Post Bill: determines if the rental order changes from pre-billed to post-billed after the first billing. Set it to Y - the order is immediately billed for the first rental period (e.g., month, quarter) but there after the order is billed through the Periodic Billing after each rental period.
•
Allow Auto Credit: determines if a credit memo line is created for an early return on a pre-billed rental order:
-
Y - a prorated price is calculated for the early return and a credit memo line is added to the rental order. You must enter a default credit reason code.
-
A - all rental lines are given a credit for early return even if the rental order was not pre-billed. For example, if the rental order was processed thru periodic billing at the end of the month and the customer returns the items early, a credit is automatically created.
-
N -no automatic credit will be issued for a rental order that is returned early.
