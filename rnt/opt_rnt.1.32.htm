Product Maintenance (/PM)
Path: Inventory > Maintenance > Product Maintenance > Product Maintenance
to automatically create the asset record in Inventory Type Conversion Entry, the rental asset information must be established for the rental item in Product Maintenance.
•
Product Type: Set this flag to N to identify this product as a rental item.
•
F5-Add Info:
•
Serial# Flag: Enter Y to require a serial number in Release Confirmation Entry.
•
Detail: Enter Y to require detail serial number tracking for this product.
•
Equipment Flag: This must be set to Y to create asset records for rental products.
•
*Action -> Addtl Optns -> Rental Information -> F7- Asset Info: Opens the Rental Asset Information screen to enter the financial and depreciation data for the product.
•
Asset Number Assignment: Determines how the asset number is assigned when the asset record is created in Inventory Type Conversion Entry. This must be set to R or E to create an asset record. If left blank, no record is created.
-
R - Creates the asset record using the rental unit number.
-
E - Creates the asset record using the equipment ID number.
•
Asset Class: Defines the default class assigned to new asset records created through this procedure. The Asset Class groups assets to separate rental assets from other fixed assets.
•
Depr. On Rent Revenue?
-
Y - calculate depreciation for the rental item when it is invoiced.
-
N - do not calculate depreciation for the rental item when it is invoiced.
•
Depr. Percent: If the previous flag is set to Y, enter the depreciation percent to use when calculating depreciation for the rental item during Invoice Printing.
•
Book#: Defines the default book codes to be assigned to new asset records created through this procedure. When a fixed asset is created from Inventory Type Conversion Entry (Rental > Processes menu), the fixed asset is updated in each book established for the rental item. F3 for options.
•
Life: Enter the default life span of the asset when the new asset record is created through ITCE. Enter the assets effective lifesaving as a four-digit number in a YYMM format. For example, 0060 would indicate six months, 0200 would indicate two years, 0106 would indicate one year, six months.
•
Salv%: Enter the percent used to determine the salvage value of the asset. The salvage value is a percent of the installation cost. For example, if the installation cost is $1,000 and the salvage percent is 10%, the salvage value will be $100.
•
Method: Enter the default depreciation method to assign to new asset records.
