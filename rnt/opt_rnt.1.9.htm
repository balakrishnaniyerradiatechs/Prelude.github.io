Returning Rentals to a Different Warehouse
When a rental item is returned at a different location, a rental return warehouse creates an auto transfer to move the rental item to the receiving location for inventory purposes.
Rental Order Entry (/ROE)
Use Rental Warehouse in the Rental Return Defaults to specify the return location.
Path: Rental > Rental Order Entry
•
Return Warehouse: the return warehouse where the rental item is being returned (defaults to the warehouse on the rental order). Leave this field at the default if the rental item is returned at the same location the rental order was placed or change the return warehouse to the current location of the return.
When the master asset is transferred using standard Transfer Entry or Transfer Rental Return from Rental Order Entry, the accumulated depreciation (LTD depreciation) of the sub asset is included in the depreciation posted to the G/L. Refer to the ITCE Asset Creation documentation for a full explanation of rental asset creation, master assets, sub assets, and G/L postings for rental asset transfers.
Transfer Rental Return Inquiry (/TRTI)
This inquiry will display the auto transfer transactions generated at invoicing when the rental item is returned to a different location. The inventory is decreased for the originating warehouse location and increased at the return warehouse location.
Path: Rental > Inquiries > Transfer Rental Return Inquiry
•
Product#: enter the rental product number to limit the report to or leave this field blank to include all rental products with rental transfers.
•
Customer#: enter the customer to limit the lookup to only the customer number entered or leave this field blank to include all customers with rental transfers.
When you choose a transaction, the Transfer Rental Return Inquiry screen appears.
Several auto transfer transactions may exist for the same rental order. An auto transfer is created for each rental item to record the transfer of the rental item from between warehouses. For example, if a rental order had two rental items returned at a different location, two auto transfer transactions are generated at invoicing to show each rental item transferred. The ext cost on the inquiry is calculated by cost amount of the rental item multiplied by the quantity of the rental item. The cost and ext cost will be the same if the quantity returned is 1.
Sales Order Log Inquiry (/OLI)
The code TRT is added to the Order Log Inquiry for the auto transfer.
Path: Order Processing > Inquiries > Sales Order Inquiries > Sales Order Log Inquiry
Reports
Path: Rental > Reports
Daily Transfer Rental Return Report (/TRN.RENT.RTN.RPT)
This report will display all rental orders returned to a different warehouse that have been invoiced for a transaction date, G/L levels, and To and From warehouse range.
Monthly Transfer Rental Return Report (/MLY.TRN.RENT.RTN.RPT)
This report will display all rental items returned to a different warehouse that have been invoiced for a fiscal period, G/L Levels, and To and From warehouse range.
