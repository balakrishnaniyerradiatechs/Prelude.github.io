Billing the Charges
Path: Rental > Processes > Periodic Billing
Periodic Billing bills customers each month for items rented over an extended period. It will also evaluate each rental order to determine which charges to add. If a damage waiver charge is required, the system will automatically add the damage waiver to the rental order. Periodic Billing Entry allows you to modify an entry. If a line item is set to not bill through this process (Bill? changed to N), the charge is removed from the rental order. If a line item is to bill through this process (Bill? changed to Y), then the charge is added.
