Equipment Maintenance (/EM)
This program is used to maintain rental EINs and provide access to the following:
•
Acquisition and disposal information.
•
Asset maintenance.
•
Attachments associated with the unit.
•
Financial information on the unit.
•
Rental information.
•
Parts listings for scheduled maintenance.
•
Additional serial numbers.
•
Equipment ID#: Enter the Equipment I.D. number (EIN). This is the number the system assigned during the inventory conversion program. V17+: When a serialized item is entered as a trade-in (negative quantity on sales order) with a new serial number, the system creates an equipment record with the new serial number in Equipment Maintenance when the sales order is invoiced.
•
Original Cost: Displays the original cost for this piece of equipment.
•
Serial Number: Displays the manufacturers serial number for this piece of equipment.
•
Product Number: The EIN’s product number appears with its description and manufacturers part number.
•
Make: If applicable, enter the make of the equipment. This field is informational only.
•
Model: Enter the model of the equipment. This field is informational only.
•
Asset Number: the asset number if this equipment is a fixed asset or enter the asset number and create the fixed asset record.
•
Rental Unit #: The rental unit number if this equipment is part of the rental fleet.
•
Product Line: the product line established in product maintenance.
•
Warehouse Number: the warehouse where the equipment is currently in stock.
•
Equipment Status: the stages are:
-
A – Available/Ready to Rent.
-
H – Hold for Customer Will Call.
-
R – On Rent.
-
C – Check in/ Make Ready.
-
W – Work Order.
-
X – Scrapped.
-
S – Sold.
•
Customer Equip Code: Enter the customer equipment code (information only).
•
License/Registered: Y indicates that the unit requires licensing or registration (e.g., state registration).
•
License Expiration: the date licensing expires if Licensed/Registered is set to Y.
•
Warranty Expiration: Enter the date the manufacturers warranty on this unit expires.
•
Operator Required: Enter Y if the unit requires an operator (for example, fork lift).
•
Customer Owned: Enter Y if the unit was not purchased from your company, but service is provided.
•
Hours Metered Flag: Set to Y if a meter is installed on this unit.
•
Current Meter Hours: based on usage of the unit. Calculated by Prelude.
•
Attachment: Displays Y if there were attachments entered for this unit.
•
Consignment Flag: Displays Y if this unit is on a consignment order.
•
Unit Weight: This field is maintained by the system.
•
Total Weight: This field is maintained by the system.
•
Tank Size: Enter the tank size in gallons. This number is used when calculating the amount of fuel when a rental item is returned and Fuel Charge Applies is set to T.
•
F5 – Acq/Dis: View acquisition and disposal information:
V18+ - Disposal Information: When a sales order for a non-rental serialized item is invoiced, the system updates the equipment record and fixed asset record with the disposal information for the serial number in Equipment Maintenance. When a vendor claim for a non-rental serialized item is updated in Vendor Claim Update Processing, the system updates the equipment and fixed asset records with the disposal information.
•
F6 – Asset: If an asset number is assigned to this unit, the Asset Maintenance screen appears.
•
F7 – Attachments: Displays any attachments associated with this piece of equipment.
•
F8 – Financial: Displays financial information for the unit. If this product is a rental unit, the same information appears in the financial section of Rental Equipment Maintenance. The equipment statistics and parts and labor maintenance expenses are updated when a Shop Repair order is invoiced for the item. Also the F5 - Review Maint key shows the invoiced Shop Repair orders.
•
F9 – Rental: Brings up the Rental Equipment Maintenance screen for rental items.
•
* F10 Action > Parts List: Sets up the parts needed for each maintenance routine for a piece of equipment.
•
Product#/Description: Enter the valid product numbers needed for each scheduled maintenance routine performed on the piece of equipment.
•
Maint Code: Defines which scheduled maintenance routine the corresponding product is used for. For example: in the above screen, all parts used in an oil change are identified with the code of OC1. These codes are set up in Scheduled Maintenance Code Maintenance.
•
Quantity: The quantity of each product to include in this scheduled maintenance.
•
F5 – Maint Required: This option in the Parts List screen is used to view or change the dates used to determine when a piece of equipment is next scheduled for maintenance.
•
Maint Code: Identifies each maintenance routine scheduled for the piece of equipment. These codes represent the products already entered and cannot be added, changed, or deleted in this screen.
•
Last Date: Defines the last date the corresponding maintenance routine was performed on the piece of equipment. This field is updated when a work order for the scheduled maintenance routine is completed in Work Order Completions.
•
Next Date: Defines the next date the corresponding maintenance routine should be performed on the piece of equipment. This field is updated when a work order for the scheduled maintenance routine is completed in Work Order Completions. The system calculates the next scheduled maintenance date based on the Days defined in Scheduled Maintenance Code Maintenance for the corresponding maintenance code. For example: if the last maintenance date is 11/05/02 and the number of days between maintenance periods is 30, the next scheduled maintenance date is 12/05/02.
If setting up a piece of equipment for the first time, use this field to enter the date each scheduled maintenance entry should first be performed. If both fields are left blank the system schedules maintenance the next time Nightly Processing is run.
•
Last Hours: Defines the measurement (hours, mileage, wear) of the piece of equipment the last time the corresponding maintenance routine was performed. This field is updated when a work order for the scheduled maintenance routine is completed in Work Order Completions and the current measurement is entered.
•
F6 – Parts Copy: This option in the Parts List screen allows a parts list to copy from an equipment ID.
•
From Equipment Number: Enter the EIN to copy the parts list from.
•
Product Number: Displays the product number associated with the EIN.
•
Serial Number: Displays the serial number associated with the EIN.
•
Include Maintenance Schedule: Determines if the maintenance information in the F5-Maint screen is included when the parts list is copied.
•
Continue With Copy?: Determines if the copy process is to be completed. Select Y to update the Parts List Entry screen, which remains blank until the copy process is performed again or you manually enter the part numbers.
•
Action > Serial Numbers: enter the additional serial numbers for the EIN.
•
Action > Comments: In V18+ you can enter comments .
