Periodic Billing Entry (/PBE)
This option allows you to modify a periodic billing entry created by the Periodic Billing process. You can set certain items to not bill in the next periodic billing. Only the items set to bill will appear on the invoice. Enter the rental order number to view. The following screen will appear displaying all rental items that will be billed.
Path: Rental > Processes Periodic Billing Entry
You can set certain line items up to not bill the customer by setting Bill to N.
•
Bill?: Defines if the line item is included in the billing: Y - bill the customer; N - do not bill the customer.
•
Per: the number of periods the customer will be billed for. Defaults from the rental order but may be changed.
•
U/M: the rental period for the item. This field defaults from the rental order but may be overridden if necessary.
•
From and Time: the beginning date and time for the current billing period.
•
To and Time: the ending date and time for the current billing period.
•
Rate and Extension: the amount per unit and the total amount being billed for the current line item.
•
F5-Remove From Billing (UnBill): remove the rental order from billing. Previously, you had to remove each line. The rental items all have Bill = N and charges such as damage waivers, environmental charges, and liability charges are deleted. The rental order should be setup to the condition the rental order was before Periodic Billing.
