Release Confirmation (/RCE) & Invoice Printing (/IP)
When the rental order is released for invoicing in Release Confirmation Entry, you are prompted for the on-rent information as usual. The order will be invoiced for the full amount of the items rented plus any incidentals.
Path: Rental > Release Confirmation and Order Processing > Invoice Preview & Printing > Invoice Printing
Rental Billed Thru Date allows for a billed thru date on the pre-billed rental order. This date controls how far out to pre-bill the rental order. If this field is blank, the standard pre-bill date is calculated according to the unit of measure (hourly, daily, weekly, monthly) for the rental product. Invoice processing will bill the units on-rent but does not close the rental order. If the item is returned before the rental period expires, the final invoice is for zero unless an early return credit is issued. If the rental order is set up for periodic billing and the rental item is not returned by the start of the next rental period, then periodic billing picks it up and pre-bills for the next rental period unless Switch To Post Bill is Y; then periodic billing will not bill the order till the second rental period expires.
