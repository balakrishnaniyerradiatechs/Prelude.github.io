Inventory Type Conversion Entry (/ITCE)
Path: Rental > Processes > Inventory Type Conversion
This procedure converts an inventory product to a rental item. The ASSETS and ASSETS.BOOK records are created if the all of the flags are set correctly for the product as explained in Product Maintenance. If the Asset Number Assignment flag for the To Product Number is set to R, the asset record is created using the rental unit number. If the Asset Number Assignment flag for the To Product Number is set to E, the asset record is created using the equipment ID number.
For example:
•
Rental unit number : 9801-008
•
Equipment ID number - 0000145
•
Book number - BOOK1
•
If the Asset Number Assignment flag is set to R: ASSETS = 001!9801-008; ASSETS.BOOK = 001!9801-008!BOOK1
•
If the Asset Number Assignment flag is set to E:ASSETS = 001!0000145; ASSETS.BOOK = 001!0000145!BOOK1.
When exiting the Inventory Type Conversion Entry screen, the rental unit number and equipment ID number are displayed. The asset records are created behind the scenes using the defaults set up in Product Maintenance.
