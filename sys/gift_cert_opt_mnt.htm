Gift Certificate Option Maintenance screen
If you issue gift certificates to customers and you want to use system-generated numbers, you can use the Gift Certificate Option Maintenance screen to generate the gift certificate numbers. Refer to “Gift Certificates” in the Order Processing documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Gift Certificate Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Number Of Characters In Cert#
The number of characters to use to create a gift certificate number. This number can include a check digit field if Utilize Check Digit is set to Y.
Gift Certificate Print Routine
The print program to use to print gift certificates from Sales Order Entry or Gift Certificate Maintenance. Enter a standard gift certificate print routine, GIFTCERT.DOC, to print the gift certificate in Sales Order Entry or Gift Certificate Maintenance.
Default Expiration Days
The number of days to add to today’s date to calculate the default gift certificate expiration date. You can override this date when you create the gift certificate. When a gift certificate expires, a warning message displays but the certificate can still be used.
Utilize Check Digit
Determines if a check digit is used for security so that fake gift certificates cannot be created:
•
Y - Insert a check digit at the end of the generated gift certificate number.
•
N - Do not use a check digit.
Since the check digit is added to the end of the number, the gift certificate number may not be a sequential number. For example, if a gift certificate is generated with number 00000014, the next gift certificate number may be 00000028. The first seven digits are sequential and the eighth digit is the calculated field for the check digit. The check digit is only used when the system generates the next available gift certificate number.
F2-Save
Saves the gift certificate number information.
F4-Del
Deletes the gift certificate information.
