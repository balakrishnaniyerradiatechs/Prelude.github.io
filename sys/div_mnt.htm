Division Address Maintenance screen
If your general ledger hierarchy includes divisions, (General Ledger Controls > Hierarchy Maintenance > F5-Div List), you can use the Division Address Maintenance screen to add or edit address and manager information for each division.
You can only access this screen if Divisions is set to Y in the Hierarchy Level Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Division Maintenance
Field and Function Key Descriptions
