Test Results Maintenance screen
The Test Results Maintenance screen allows you to set up and maintain test result codes to use when receiving product that require inspection.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Test Results Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Test Results Code
The code to identify the test results for a product.
Description
The description of the test results code.
F2-Save
Saves the test results information.
F4-Del
Deletes the test results information.
