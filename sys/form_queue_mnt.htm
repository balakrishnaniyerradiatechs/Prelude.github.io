Form Queue Maintenance screen
The Form Queue Maintenance screen allows you to set up form queues and to identify the warehouses that may print to the form queues when you release pick tickets. If you do not set up a form queue, the warehouse is not validated when you release pick tickets (refer to “Releasing Pick Tickets” in the Stock Receipts chapter of the Inventory documentation).
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Printers > Form Queue Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Form Queue
The form queue number.
Printer Desc
The name, description, or location for the printer.
Printer Type
The printer driver to use to print bar codes on this document.
Warehouse# List
The warehouses that are allowed to use this form queue. If no warehouse is entered, then any warehouse can use this form queue.
