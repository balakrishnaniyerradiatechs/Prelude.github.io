Price Description Defaults screen
The Price Description Defaults screen allows you to edit the headings of price fields in the pricing screens if your company defines price levels differently than the way Prelude defines them (e.g., Retail could be changed to List).
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Price Descriptions
Field and Function Key Descriptions
Field/Function Key
Description
Sugg. Retail Desc.
The heading to use for suggested retail price.
Price 1 Desc.
The heading to use for Price 1.
Price 2 Desc.
The heading to use for Price 2.
Price 3 Desc.
The heading to use for Price 3.
Price 4 Desc.
The heading to use for Price 4.
Price 5 Desc.
The heading to use for Price 5.
F2-Save
Saves the price description information.
F4-Del
Deletes the price description information.
F5-Add. Price Columns
Opens the Add. Prc Columns screen to allow you to enter price descriptions for additional columns. This screen is only accessible if Maximum Price Columns in the Inventory Parameters screen has a value greater than 5.
