Country Codes Maintenance screen
The Country Codes Maintenance screen allows you to maintain tariff rates, agreement codes, and incompatible ship via codes for each country.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Country Codes
Field and Function Key Descriptions
Field/Function Key
Description
Country Code
The code for each country.
Description
The description of the country.
Most Favored Nation
Determines if tariffs for a country are calculated at the most-favored- nation rate or at the non-favored-nation rate.
•
Y - Tariffs are calculated at the Favored Nation Rate set in Tariff Commodity Code Maintenance.
•
N - Tariffs are calculated at the Non-Favored Nation Rate set in Tariff Commodity Code Maintenance (e.g., Afghanistan, Cuba, Kampuchea, Laos, North Korea and Vietnam).
Tariff commodity codes define favored nation rates, non-favored nation rates, and agreement code rates to work with harmonized tariff calculations. Refer to “Tariff Commodity Codes” in the Inventory documentation for more information.
Agreement Codes
The agreement codes used to calculate tariff charges for this country as defined in the Harmonized Tariff Schedule of the United States.
If the country is a favored nation and contains an agreement code that matches one of these agreement codes, then the Agree% and the Agree Rate in Tariff Commodity Code Maintenance are used to calculate the tariff.
Refer to “Tariff Commodity Codes” in the Inventory documentation for more information.
Incompatible Ship Via
Ship Via codes that can not be used to ship orders to this country.
Description
The description of the ship via code.
F2-Save
Saves the country code information.
F4-Del
Deletes the country code.
