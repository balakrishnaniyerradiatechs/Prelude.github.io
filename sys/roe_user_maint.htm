Remote User Id Maintenance screen
The Remote User Id Maintenance screen allows you to set up IDs for your customers that use Remote Order Entry to enter or review their sales orders in your system. You can identify the warehouses and customers for which the user can enter orders. The flags in this screen override the flags set up for the user in the User Identification Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Remote Order Entry User ID
Field and Function Key Descriptions
Field/Function Key
Description
User Id
The user ID of the person that enters remote orders.
Company Number
The company that this user is confined to for order entry.
Customer
The customer number that this user is confined to for order entry. If you want to add more customers that this user can access, use F5-Add Cust.
Ship-To
The ship-to address number for the customer that this user is confined to entering orders for, if applicable. Leave blank to allow access to all ship-to addresses for this customer.
Warehouse
The warehouse location that this user is confined to for order entry.
Allow Price Change
Determines if this user can change prices on sales orders:
•
Y - The user can change prices.
•
N - The user cannot change prices.
Allow Whse Change
Determines if this user can change the shipping warehouse number on a sales order:
•
Y - The user can change the shipping warehouse.
•
N - The user cannot change the shipping warehouse.
Allow Non-Stock PN
Determines if this user can enter non-stock part numbers on a sales order:
•
Y - The user can enter non-stock part numbers.
•
N - The user cannot enter non-stock part numbers.
Allow “*” Type PN
Determines if this user can enter “*” (asterisk) part numbers on a sales order:
•
Y - The user can enter “*” part numbers.
•
N - The user cannot enter “*” part numbers.
Allow Change Tax Flag
Determines if this user can change the tax flag on a sales order:
•
Y - The user can change the tax flag.
•
N - The user cannot change the tax flag.
Allow Order# Prompt
Determines if this user can enter an existing sales order to edit:
•
Y - The user can enter an existing sales order number.
•
N - Force a new sales order number each time for this user.
Allow F10 Access
Determines if this user can access the Action Bar in Sales Order Entry and Sales Order History Inquiry:
•
Y - The user can access the Action Bar.
•
N - The user cannot access the Action Bar.
F2-Save
Saves the remote user information.
F4-Del
Deletes the remote user.
F5-Add Cust
Allows you to add any additional customers that this user can access when logged onto your system.
