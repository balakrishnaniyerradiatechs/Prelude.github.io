Location Address Maintenance screen
You can use the Location Address Maintenance screen to add or edit address, manager, currency, and fiscal year information for each location.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Location Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Division
The division number that the location is assigned to, if divisions are used by your company.
Region
The region assigned to the location, if locations are used by your company.
Location
The warehouse or corporate location where transactions can post to.
Description
The description of the location.
Address One
The first line of address for the location.
Address Two
The second line of address for the location.
Address Three
The third line of address for the location.
Address Four
The fourth line of address for the location.
Manager
The manager assigned to the location.
Phone
The manager’s phone number.
GST/HST Tax Flg
Determines the Canadian tax amount to calculate in Manual Invoice Entry and in Vendor Invoice Reconciliation Entry:
•
G - GST (goods and services tax).
•
H - HST (harmonized sales tax).
Separate Fiscal
If you use multiple fiscal periods and this location posts to a different fiscal period than the other locations, you can define the fiscal account this location belongs to and point to the fiscal period to post to. Refer to “Using Multiple Fiscal Periods” in the General Ledger documentation.
Currency Defaults: If you use multiple currency (refer to “Using Multiple Currency” in the General Ledger documentation) you can set up default currency types to use for new accounts.
Account Type
The account types to assign a default currency type to in Chart of Accounts Maintenance:
•
A - Asset.
•
C - Capital/Equity.
•
E - Expense.
•
I - Income/Revenue.
•
L - Liability.
Currency Type
The default Currency Type to use for each account type in Chart of Accounts Maintenance. For example, if Location 20 has an Account Type of A and a Currency Type of EUR, then any asset account created in Chart of Accounts Maintenance will have a default Currency Type of EUR.
F2-Save
Saves the location information.
F4-Del
Deletes the location information.
