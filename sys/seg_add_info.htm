Warehousing Information screen
If you use the Warehousing module, you can set defaults and warehousing information for a user in the Warehousing Information screen.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance > Action Bar > User Options > Warehousing
Field and Function Key Descriptions
Field/Function Key
Description
Pick Only Bin Available
Determines if the user can pick only the available quantity for a bin or if they can over pick and allow the available quantity to go negative:
•
Y - If the user enters a quantity larger than the available quantity in an RF picking screen, a warning message displays and the entry is not allowed. The user must zero pick the line item, choose another bin, or correct the entered quantity.
•
N - If the user enters a quantity larger than the available quantity in an RF picking screen, a warning message displays but the entry is allowed.
Rf Primary Bin Update
Determines if the user can update the primary bin in RF Putaway Entry:
•
Y - The user can update the primary bin.
•
N - The user cannot update the primary bin.
Default To Bin
The default TO bin to appear for this user.
Whse Start Time
The time this user begins work (MM:HH military time). This information is for picker/packer statistics.
Rf Order Picking Zones
The radio frequency picking zones assigned to this user.
Receiving
Default Rec Dock Bin
The default receiving dock bin to use for this user when receiving products in stock receipts.
Default Linkage Bin
The default linkage bin to use when this user receives linked items.
Default Inspection Bin
The default inspection bin to use when this user receives items.
 
Segment Information:
Pack Hazardous Materials?
Determines if this user can select segments containing hazardous materials:
•
Y - The user may only pack hazardous materials.
•
N - The user cannot pack hazardous materials.
•
B - The user can pack both hazardous and non-hazardous materials.
Picking Ship Via
The ship via codes to assign to this user for warehouse segment picking.
Packing Ship Via
The ship via codes to assign to this user for warehouse segment packing.
F2-Save
Saves the warehousing information.
