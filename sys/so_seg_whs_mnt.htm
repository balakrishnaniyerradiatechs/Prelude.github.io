Warehousing Segment Maintenance screen
If you use truck route processing with the Warehousing module, you can use the Warehousing Segment Maintenance screen to set up picking and packing options for truck route segments.
Radio Frequency Truck Route Processing allows you to organize sales orders by truck route. Multiple sales orders are combined to allow picking for the same customer/ship-to, known as a stop. Each truck route consists of multiple stops. A stop consists of all the products for a customer and is divided into picking segments. A single segment may contain all of the products to be picked for that stop, or you may generate multiple segments to allow multiple pickers to gang pick the products for that stop.
For each ship via, you can create gang-picking segments.
Example: 100 products must be picked for customer ABC Supply.
Gang Picking Method: The 100 products are assigned to 4 pickers in segments of 25 products each, allowing this stop to be completed sooner than if just a single picker was assigned the entire order.
Non Gang Picking Method: A single segment is created with all 100 products. A single picker picks all of the products.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint > F5-Warehousing > Action Bar > Segments
