Product Security Access Codes screen
If you want to set up security so that only certain employees can access specific products, the Product Security Access Codes screen allows you enter security access codes that a user can enter to access products that have the same security access codes set in Product Maintenance > F5-Add Info > Prod Security Access.
Security codes are created in the Security Access Code Maintenance screen.
You can only access this screen if Prod Security Access in the Order Processing Parameters Additional Information screen is set to Y.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance > F6-Add Info > F6-Prod Sec.
Field and Function Key Descriptions
Field/Function Key
Description
Code
The security access code assigned to products that this user has access to. Security codes are created in the Security Access Code Maintenance screen.
Description
The description of the security access code.
F2-Save
Saves the security access code information.
