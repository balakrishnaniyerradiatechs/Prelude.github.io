Continuous Forms Procedure screen
The Continuous Forms Procedure screen is used to start and stop the continuous printer ports (refer to the Continuous Port Maintenance screen). For example, you must stop and start continuous printers if you take down the spooler.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Printers > Continuous Port Start/Stop
Field and Function Key Descriptions
Field/Function Key
Description
Continuous Port
The continuous port number to start or stop.
