Warehousing Receiving Maintenance screen
If you use the Warehousing module, you can set options for RF stock receipts and put-away in the Warehousing Receiving Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint > F5-Warehousing > F8-Recv
Field and Function Key Descriptions
Field/Function Key
Description
Stock Receipts Options
Allow New Products In RF SRE
Determines if a product that does not exists on the original purchase order can be received in RF Stock Receipts Entry:
•
Y - You can receive products that were not on the original purchase order, even if the product does not exist in the receiving warehouse.
•
N - You cannot add a product in RF Stock Receipts Entry if the product does not exist in the receiving warehouse.
Your user ID must have Add New Products in SRE set to Y in the User Identification Maintenance Additional Information screen to allow you to enter new products in Stock Receipts Entry.
Display Count Of Open B/Os In RFSRE
Determines if the open backorder quantity displays in RF Stock Receipts Entry:
•
Y - Once you enter the product number, the backordered amount appears. The amount does not include backordered quantities from linked lines or supply codes.
•
N - The backordered quantity does not display.
Prompt For Batch Number
Determines if you must enter a batch number in RF Stock Receipts Entry:
•
Y - A batch number is required.
•
N - The batch number screen does not appear.
Stock Receipts Putaway Options
SR Putaway Ent-2 Method
Determines if a batch number is requested in the RF and Truck Mount Stock Receipts Putaway Entry screens:
•
B - Request a batch number.
•
N - List all items in the dock bin that are available for putaway.
Vendor Claims Options
RF Vendor Claim Picking
Determines if RF picking is used for vendor claims:
•
Y - When you complete a vendor claim, you can choose to release the claim for Warehousing Processing. Bins are allocated and picking records are created. For serialized items, the serial/bin information is cleared during allocations. Warranty items appear in RF Vendor Claim Picking.
•
N - RF picking is not used for vendor claims.
F2-Save
Saves the Warehousing receiving options.
