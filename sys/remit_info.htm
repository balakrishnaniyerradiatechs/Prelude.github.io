Remit Information screen
The Remit Information screen allows you to enter the remit address for a warehouse to print on the accounts receivable statements. This overrides the remit address set for a company in the Company Maintenance screen.
A custom modification is required to print the remit address.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > Action Bar > Remit Info
Field and Function Key Descriptions
Field/Function Key
Description
Remit Company Name
The remit company name for this warehouse.
Remit Address1
The first line of the remit address.
Remit Address2
The second line of the remit address.
Remit Address3
The third line of the remit address.
Remit City
The city for the remit address.
Remit State
The state for the remit address.
Remit Zip
The zip code for the remit address.
Remit Contact Name
The contact name for the remit address for this warehouse.
Remit Phone
The phone number for the remit contact name.
F2-Save
Saves the remit address information for this warehouse.
