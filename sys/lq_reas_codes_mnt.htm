Linkage Queue Reason Code Maintenance screen
The Linkage Queue Reason Code Maintenance screen allows you to set up reason codes to use in the Linkage Queue Parameter Maintenance screen. These reason codes apply when someone edits a sales order that is linked to a purchase order or edits a purchase order that is linked to a sales order. These changes appear in the Linked Purchase Order Inquiry (refer to “Linked P/O Changes” in the Buyer Queue Inquiry section of the purchasing documentation) and the Linked Sales Order Inquiry (refer to the Inquiries section of the Order Processing documentation).
Purchasing Parameters
S/O-P/O Linkage Work Queue in the Purchasing Parameters screen activates the Linked Purchase Order Inquiry and the Linked Sales Order Inquiry.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Linkage Queue Reason Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Linkage Queue Reason Code
The 1-digit code that defines the reason for a change to a linked sales order or a linked purchase order.
Description
The description of the reason code (e.g., change in delivery date, change in quantity, canceled item).
F2-Save
Saves the reason code.
F4-Del
Deletes the reason code.
