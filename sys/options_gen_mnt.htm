Option Parameters
General Parameters screen
The General Parameters screen allows you to set flag and option parameters regarding notes, mailroom address verification, Optio, and Ingenico.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > General Parameters
Field and Function Key Descriptions
Field/Function Key
Description
Update Notes In History
Determines how notes are entered and edited in the entry programs that accept notes (e.g., sales orders, purchase orders, quotes, customers, vendors):
•
Y - New notes are entered with a user, time, and date stamp. Existing notes cannot be edited.
•
N - New notes do not have a user, time, and date stamp, but can be edited.
You can also set up a user ID to allow notes to be edited in the User Identification Maintenance Notes Edit Authorization screen.
Insert New Notes At Top
Determines if new notes are added at the top of the notes screen so that the most recent notes appear at the top:
•
Y - Update notes at the top of the notes inquiry screen.
•
N - Update notes at the bottom of the notes inquiry screen.
•
R - Re-sequences notes that were previously at the bottom of the screen, to appear with the new notes at the top.
If you select the Y or R option, Update Notes History should be set to Y so that the system updates the last notes entry with the user id, date, and time stamp.
Address Verify Installed?
Determines if the mailroom address verification system is used:
•
Y - Use the mailroom address verification system which is an optional module (e.g., to send out catalogs). Opens the Address Verification Server screen to allow you to enter the Ip Address and mailroom information.
•
N - Do not use the mailroom address verification system.
Optio Image Path
The path where Optio logos and graphics are stored for use on forms.
Signature Scale
The size of the signature capture depending on the DPI of your signature capture unit. Can be from 1 to 100 percent (default is 10 percent).
Email Size Limit
The email size limit for report/broadcast type emails (e.g., 1024K for 1024 kilobytes or 20M for 20 megabytes). Attachments larger than this cannot be emailed.
Email Page Limit
The maximum number of pages that can be transmitted in an email (report/broadcast type emails only).
Ingenico Settings
3rd party software interface for credit cards signatures.
Unix Id
The Ingenico Integration Unix User ID to connect as.
Unix Pwd
The Ingenico Integration Unix password to connect as.
Ftp Passive
Indicates if FTP access requires the passive mode:
•
Y - Require passive mode.
•
N - Do not require passive mode.
Signature Wait
Determines if the signature capture waits until after the manual close:
•
Y - Capture signature after manual close.
•
N - Capture signature before manual close.
Filemode
Determines if the file mode is used:
•
Y - Use the file mode.
•
N - Do not use the file mode.
F2-Save
Saves the general parameter settings.
