More Additional Options screen
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Order Processing Parameters > F5-Add. Info > F5-Add Info> F5-Add More Info
Field and Function Key Descriptions
Field/Function Key
Description
000JOB Quote Expanded Screen
Determines if you can print, email, or fax customer quotes to contacts for the 000JOB customer:
•
Y - Default the 000JOB Quote screen that allows you to print, email, or fax a quote.
•
N - You can only print the quote (not email or fax).
Encrypt CC Data
Determines if credit card data is encrypted:
•
Y - Encrypt credit card data using the DES3 (3rd party software) encryption techniques. A warning message appears to let you know that you must run a conversion to encrypt all credit card data (System Parameters > Processes > ADS Utilities > Miscellaneous Utilities > Encrypt all credit card numbers). Contact Activant Prelude technical support for assistance).
•
N - Do not encrypt credit card data.
Ship-To Keyword Search
Determines which screen appears when you press F3 in the Ship-To field on the Sales Order Header screen:
•
Y - Display the Ship-To Keyword Selection screen. You can enter a keyword to lookup ship-to numbers.
•
N or blank - Display the GRIM Screen to lookup up customer ship-to addresses using a description, address, or zip code.
Utilize Best Contract Price
Determines if the best contract price is used to price customer sales orders:
•
Y - Use the best contract price.
•
N - Use the first contract price found.
This flag is in addition to the Utilize Best Price flag in the main Order Processing Parameters screen that searches for the best of all prices except for contracts
Quote Message All Warehouse
Determines if a message displays in Quotation Order Entry if a product has existing quotes in any warehouse or only the selling warehouse:
•
Y - Display message if quotes exist in any warehouse.
•
N - Display message if quotes exist in the warehouse only.
Display Open Quote/RGA Message (Order Processing Parameters) must be set to Y and Open Quote Message (Order Processing Parameters > F5) must be set to L or B for this flag to work.
Route Evaluation on Release
Determines if a backorder release uses the route from the original sales order, or the earliest route set up for the customer:
•
E - Use existing (original) route.
•
R - Re-evaluate the route to find the earliest route established for the customer.
Convert Kit on Consignment
Determines if kit parent items are confirmed/billed as one item for consignment orders:
•
Y - Bill the kit parent as one item for type 08 consignment orders. The kit parent is billed as a regular item instead of prompting for kit components in Release Confirmation Entry. The components are removed when the original release is invoiced.
•
N - Do not bill kit parents as one item for type 08 consignment orders. You must confirm each component quantity in Release Confirmation Entry.
Ship Complete By Warehouse
Determines if you can enter a different warehouse for each line item on a type 02 (ship complete) sales order:
•
Y - Allow entry of multiple warehouses for line items on type 02 sales orders.
•
N - Do not allow entry of multiple warehouses for type 02 sales orders. A message appears to let you know that you cannot change the warehouse.
F2-Save
Saves the additional option information.
