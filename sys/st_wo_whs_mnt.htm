Warehousing Stock Transfer Maintenance screen
If you use the Warehousing module, you can use the Warehousing Stock Transfer Maintenance screen to determine sorting and allocation methods for transferring material from one warehouse to another, using either pick tickets or RF (radio frequency) guns.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint > F5-Warehousing > F5-Transfers
Field and Function Key Descriptions
Field/Function Key
Description
Transfer Picking Options
S/T FIFO Lot Allocation Flag
Determines how transfers use FIFO lot allocation:
•
Y - Use FIFO lot allocation and always allocate the oldest lot first even if another single lot can completely fill a requirement.
•
N - If possible, use the oldest lot that can completely fill an order. If no single lot can fill an order, allocate partial lots using the oldest quantity first.
S/T Radio Frequency Picking
Determines if stock transfers are picked using RF guns:
•
Y - Transfers are picked with RF units.
•
N - Transfers are picked with pick tickets.
Prioritize By Date
Indicates if the date determines transfer picking priority:
•
Y - Assign pick tickets according to the oldest date, then by S/T Bin Sorting Method.
•
N - Only use the S/T Bin Sorting Method to determine transfer pick ticket priority.
Generate Emergency Bin Repl
Determines if an emergency bin replenishment is created if transfer bin allocation or transfer pick ticket printing cannot allocate an entire quantity:
•
Y - If an entire quantity cannot be allocated, an emergency bin replenishment is created with a T replenishment source. The line item is held for picking until the entire quantity can be allocated.
•
N - No emergency bin replenishment is created. Use standard bin allocation.
S/T Bin Priority
Determines the priority sequence for the corresponding Type of replenishment method.
Type
The stock transfer replenishment method for the corresponding S/T Bin Priority sequence:
•
B - Bulk bin.
•
D - Dock bin. The system only allocates from the dock bin if the Pick Reserve flag is set to P in Bin Location Maintenance.
•
I - Inspection bin.
•
K - Bulk un-packaged.
•
L - Line stock bin.
•
P - Pick bin.
•
T - Retail bin.
This sequence is overridden if you enter a value in the following S/T Bin Sorting Method.
S/T Bin Sorting Method
The bin sort sequence for non-lot transfer bin allocation:
•
AA - Ascending bin available quantity.
•
AO - Ascending bin on-hand quantity.
•
AD - Ascending bin transaction date.
•
DA - Descending bin available quantity.
•
DO - Descending bin on-hand quantity.
•
DD - Descending bin transaction date.
This sort sequence overrides any values entered in S/T Bin Priority and Type.
Valid Destination Bins:
Receiving Whse
The warehouse numbers that will receive transfers from this warehouse.
Bin Location
The bin number in the corresponding Receiving Whse that will be used to receive transfers.
F2-Save
Saves the warehousing stock transfer information.
