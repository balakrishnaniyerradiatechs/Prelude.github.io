Nightly Purchasing Setup Maintenance screen
The Nightly Purchasing Setup Maintenance screen allows you to identify which Nightly Processing reports to run for each warehouse. You can change any of these settings when you run Nightly Processing. Refer to Nightly Processing in the Purchasing documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Purchasing Controls > Nightly Purchasing Setup
Field and Function Key Descriptions
Field/Function Key
Description
Whse
The warehouse numbers to include in the Nightly Processing routine. As you enter each warehouse, the Nightly Purchase Report Execution Maintenance screen displays to allow you to choose which reports to print. Refer to “Nightly Processing” in the Purchasing documentation for more information.
Description
The description or location of the warehouse.
F2-Save
Saves the Nightly Processing warehouse setup information.
F4-Del
Deletes the Nightly Processing warehouse setup information.
 
Purchasing Controls:Nightly Purchasing Setup Maintenance screen
