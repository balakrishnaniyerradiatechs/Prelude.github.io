Keyword Exclusion Maintenance screen
The Keyword Exclusions screen allows you to identify which words should not be used as keywords (i.e., words that are too common to use in searches).
You must run the related rebuild to activate keyword changes (Systems > System Parameters > Processes . ReBuild Menu).
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Keyword Exclusions
Field and Function Key Descriptions
Field/Function Key
Description
Keywords
Words too common to your industry that should not be used as keywords (e.g., pipe, supplier, the).
F2-Save
Saves the keyword exclusion information.
F4-Del
Deletes the keyword exclusion information.
