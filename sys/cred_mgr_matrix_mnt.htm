Credit Manager Assignment Matrix screen
The Credit Manager Assignment Matrix screen allows you to assign a credit manager to a group of customers by alphabetical range.
The hierarchy to determine which credit manager to use is as follows:
•
Customer Level: Customer Maintenance > F6 > Credit Manager ID.
•
System Level: Systems Menu > Accounts Receivable Controls > Credit Manager Assignment Matrix.
•
Warehouse Level: Systems Menu > Warehouse Manager Maintenance > Credit Manager.
Menu Path: Systems > System Parameters > Maintenance > Accounts Receivable Controls > Credit Mgr Assignment Matrix
Field and Function Key Descriptions
Field/Function Key
Description
Ln#
The line number to reference the range of customers.
From
The beginning letter or name in the range of customers to assign a credit manager to.
To
The ending letter or name in the range of customers to assign a credit manager to.
Cred Mgr
The credit manager ID to assign this range of customers to. Credit Managers are set up in Manager Maintenance.
F2-Save
Saves the credit manager information.
F4-Del
Deletes the credit manager assignment.
