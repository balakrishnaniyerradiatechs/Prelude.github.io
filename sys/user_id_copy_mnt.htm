User Id Copy screen
The User Id Copy screen allows you to copy all information for an established user ID to create a new user ID. You can only access this screen before you enter a User Id Code in the User Identification Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance > Action Bar > Copy
Field and Function Key Descriptions
Field/Function Key
Description
From User Id
The user ID of the user whose information you want to copy.
New User Id
The new user ID that you are creating.
New User Id Name
The name of the new user that you are setting up.
New Desktop User Id
The desktop ID for the new user.
F2-Save
Saves the new user information.
