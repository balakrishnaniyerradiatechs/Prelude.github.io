Department Address Maintenance screen
If your general ledger hierarchy includes Departments, (General Ledger Controls > Hierarchy Maintenance > F8-Dep List), you can use the Department Address Maintenance screen to add or edit address and manager information for each department.
You can only access this screen if Departments is set to Y in the Hierarchy Level Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Department Maintenance
