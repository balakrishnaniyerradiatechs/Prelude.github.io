Slash Process Security screen
The Slash Process Security screen allows you to allow or disallow one or more slash commands for a user. These restrictions do not prevent the user from accessing the programs from a menu.
Leave this screen blank to allow the user to use all slash commands.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance > Action Bar > User Options > Process Security
Field and Function Key Descriptions
Field/Function Key
Description
Allow Or Disallow Slash Commands
Determines if you want to allow or disallow a list of slash commands for this user:
•
A - Allow the user to only use the slash commands for the programs listed under the Process List.
•
D - Disallow only the slash commands for the programs listed under the Process List for this user.
Process List
The process names allowed or disallowed for this user.
F2-Save
Saves the slash security information.
