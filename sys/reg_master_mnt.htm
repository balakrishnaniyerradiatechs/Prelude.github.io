Region Master Maintenance screen
The Region Master Maintenance screen defines the regions for your company. Regions are a subset of Division - if Divisions are used. Use the Region Maintenance screen to add or edit address, division, and manager information for the region.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Hierarchy Maintenance > F6-Reg List
Field and Function Key Descriptions
Field/Function Key
Description
Reg
The region number for each valid region.
Description
The description of each region (e.g., Southeast region, Northwest region).
F2-Save
Saves the region information.
F4-Del
Deletes all the regions listed on the screen.
