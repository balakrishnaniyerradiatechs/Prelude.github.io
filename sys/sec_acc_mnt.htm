Security Access Code Maintenance screen
If you use security codes to determine which users can access specific customers, products, and/or vendors, use the Security Access Code Maintenance screen to set up the security codes.
For information on how to use security access codes, refer to the following:
•
The Customer Security Access Codes screen.
•
The Product Security Access Codes screen.
•
The Vendor Security Access Codes screen.
Activating Security Codes
Cust Security Access, Prod Security Access, and Vend. Security Access in the Order Processing Parameters Additional Information screen determine if security access codes are activated.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Security Access Codes Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Security Access Code
The security access code to identify a group of customers, vendors, or products.
Description
The description of the security code.

Saves the security access code.
F4-Del
Deletes the security access code.
