Radio Terminal System Maint screen
If you use radio frequency devices in your warehouse, you can enter the radio terminal information in the Radio Terminal System Maint screen.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint
Field and Function Key Descriptions
Field/Function Key
Description
FCC Number
The federal communications commission license number issued to this warehouse for the radio frequency equipment.
Multiplexor Port Number
The pick port number the radio terminal network controller and base radio are connected to.
Continuous Port Number
The pick port number dedicated to the background job responsible for processing requests submitted by radio terminals.
Small Prod. Label Print Queue
The print queue for the small product labels.
Large Prod. Label Print Queue
The print queue for the large product labels.
Bin Label Print Queue
The print queue for bin location bar code labels.
Stock Receipts Printer Queue
The print queue for the radio frequency receiving and stock putaway reports.
Physical Tag Printer Type
The printer driver for physical inventory tag bar codes.
Pick Ticket Printer Type
The printer driver for pick ticket bar codes.
Stock Transfer Printer Type
The printer driver for transfer ticket bar codes.
 
Stock Receipts Printer Type
The printer driver for bar codes on the radio frequency receiving and stock putaway reports.
F2-Save
Saves the radio terminal information.
F5-Warehousing
Opens the Warehousing Maintenance screen to allow you to set options for the warehousing module.
