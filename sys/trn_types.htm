Stock Transfer Types screen
The Stock Transfer Types are set up by Prelude and used in Stock Transfer Entry to determine how the general ledger is affected when inventory is transferred. You can use the Stock Transfer Types screen to set the default transfer type that appears in Transfer Entry and to determine which types are valid for your company. Refer to “Stock Transfers” in the Inventory documentation for more information on transfer types.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Stock Transfer Entry Types
Field and Function Key Descriptions
Field/Function Key
Description
Transfer Type Default
The transfer type to default in the Stock Transfer Header screen.
Valid Transfer Types
The transfer types that are valid in Transfer Entry for your company. The available types in Prelude are:
•
Regular Order - Type 01: Moves inventory items from one warehouse to another.
•
Non-Inventory - Type 02: Moves non-inventory items (e.g., office supplies, packing materials) from one warehouse to another. The products do not exist and are not tracked as inventory.
•
Warranty Inventory - Type 03: Moves warranty tag inventory from one warehouse to another. The items must exist in the system as warranty inventory. Refer to ”Transferring Warranty Inventory” in the Inventory documentation.
•
Crib Replenishment - Type 04: If the receiving warehouse is a crib warehouse, the transfer type defaults to 04. Replenishment is handled as a standard transfer for product/bin locations. Nightly processing generates a recommended transfer to restock the crib. Transfer Entry requires you to enter a Destination Bin. Refer to the ”Product Whse Crib Information Screen” in the Inventory documentation for information on setting up a crib warehouse.
•
Ship Line Item Complete - Type 05: The transfer will not ship a line item if it is not completely available to ship.
•
One Step Transfer - Type 09: Uses the one-step transfer process for inventory.
•
One Step Transfer -Type 10: Uses the one-step transfer process for consignment items.
F2-Save
Saves the transfer type information.
F4-Del
Deletes the transfer type information.
