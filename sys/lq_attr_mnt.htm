Linkage Queue Parameter Maintenance screen
The Linkage Queue Parameter Maintenance screen allows you to determine which fields are checked when a linked sales order or a linked purchase order is modified. You must first set up reason codes in the Linkage Queue Reason Code Maintenance screen to identify the reason for a change.
Changes to a linked sales order appear in the Linked Purchase Order Inquiry (refer to “Linked P/O Changes” in the Buyer Queue Inquiry section of the purchasing documentation). Changes to a linked purchase order appear in the Linked Sales Order Inquiry (refer to the Order Processing documentation).
Purchasing Parameters
S/O-P/O Linkage Work Queue in the Purchasing Parameters screen activates the Linked Purchase Order Inquiry and the Linked Sales Order Inquiry.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Linkage Queue Parameter Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Attribute Reference
The attribute number for the corresponding screen that will be checked for a change. See your systems administrator for help.
Reason Code
The reason code that represents the change for the attribute. These codes are set up in the Linkage Queue Reason Code Maintenance screen.
Sales Order Header
The attribute number in ORDER that is checked for changes. For example, if the required date is changed on a sales order that is linked to a purchase order, the change appears in the Linked Purchase Order Inquiry.
Sales Order Line
The attribute number in ORDER.LINE that is checked for changes. For example, if a quantity is changed on a sales order that is linked to a purchase order, the change appears in the Linked Purchase Order Inquiry.
Purchase Order Header
The attribute number in PO that is checked for changes. For example, if a date is changed on a purchase order that is linked to a sales order, the change appears in the Linked Sales Order Inquiry.
Purchase Order Line
The attribute number in PO.LINE that is checked for changes. For example, if a price is changed on a purchase order that is linked to a sales order, the change appears in the Linked Sales Order Inquiry.
F2-Save
Saves the linked parameter information.
