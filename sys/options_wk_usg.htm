Weekly Usage Options screen
The Weekly Usage Options screen allows you to set up weekly usage calculations rules at the system level. You can override this information at the product level or the product/warehouse level.
If you stock products that have an extremely high sales volume for just a few months of the year, you can use the AWU (average weekly usage) formula to calculate recommended purchase quantities. Refer to “Tracking Average Weekly Usage” in the Inventory documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Inventory Parameters > Capture Weekly Usage = Y
Field and Function Key Descriptions
Field/Function Key
Description
Number Of Weeks - Usage
The number of weeks to use to calculate average weekly usage.
Classify By Weeks
Determines if usage is classified by week:
•
Y - Classify products according to weekly usage.
•
N - Do not classify products according to weekly usage. The rest of the fields do not apply.
Number of Weeks - Class
The number of weeks to include to calculate weekly usage.
Classification Cutoff
The classification level for products that should use the Average Weekly Usage (AWU) formula. Each product in a warehouse that is this classification or higher will have its purchasing formula automatically set to AWU. Each product in a warehouse with this classification or lower, automatically has its purchasing formula set to EOQ.
F2-Save
Saves the weekly usage information.
F4-Del
Deletes the weekly usage information.
