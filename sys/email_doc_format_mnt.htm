Email Document Format Maintenance screen
The Email Document Format Maintenance screen allows you to set up email formats for the text email documents created in the Email Document Text Maintenance screen, for an entire company or for each warehouse.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Email Document Format Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Company/
Warehouse
Determines if this email format record contains defaults at the company level or warehouse level:
•
C - Company defaults.
•
W - Warehouse defaults.
Warehouse Number
The warehouse number if Company/Warehouse is set to W.
Default From User
The user ID to use as the default sender of all emails for all document types. This ID must be an operating system level user. You can override this for each document.
Document Type
The email document type to use with the text and detail format.
Lang
The language code if one was set up for this document type in the Email Document Text Maintenance screen.
Code
The email document code set up in the Email Document Text Maintenance screen that represents the text in the text emails.
Detail
The format for the product detail section:
•
STD - Standard. Prints detail values for email text document setup.
•
USER - User detail. Custom modification required.
OV From User
The override email sender to appear on emails generated for this type of transaction type. This field overrides Default From User.
F2-Save
Saves the email format information.
F4-Del
Deletes the email format information.
