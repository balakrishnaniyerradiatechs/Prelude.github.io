Administrative Quote Hold Types screen
The Administrative Quote Hold Types screen allows you to set option parameters for your company to define the restrictions that place an order on Administrative Quote Hold. You must use Administrative Quote Inquiry to release a quote from hold before the quote can print or be converted into a sales order.
Administrative hold options set at the customer level override the options set in this screen. Refer to the “Administrative Quote Inquiry” in the Order Processing documentation for more information on how to hold and release quotes.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Order Processing Parameters > F5-Add. Info > Administrative Quote Hold = Y
Field and Function Key Descriptions
Field/Function Key
Description
AQH Types
Defines criteria that places a quote on hold:
•
F - Override Freight Terms: The Freight Terms code is changed in the quote header.
•
I - Incompatible Ship Via: The Ship Via Code is incompatible for the customer (Customer Maintenance > F7-Order > F8-Incomp Ship Via) or the product (Product Maintenance > F5-Add Info > Action Bar > Incomp Ship Via).
•
P - Price Violations: A price is changed to a value that is lower than the default price, lower than the cost, or below the acceptable gross profit percent. This option must be selected as quote hold type to activate the N price violation type.
•
N - Minimum Price Violations: A line item’s selling price is below the product’s Minimum Price (Product Maintenance > F7-Pricing).
•
V - Manager High/Low Level Limit GP%: A line item’s gross profit percent is below the Mgr Low Level Minimum GP% or more than the Mgr High Level GP%. Contract pricing is excluded.
•
W - Web quotes. If an quote is entered through VIA, the quote will go on hold.
Mgr Low Level Minimum Gp%
The minimum gross profit percent allowed for a line item before it is placed on hold. This value can be overridden if a Low Level Minimum GP% exists for the product line. AQH Type is V which is only activated if the P (price violation) option is also in effect.
Mgr High Level GP%
The maximum gross profit percentage that a line item can be before it is placed on hold. This value can be overridden if a High Level GP% exists for the product line. AQH Type is V which is only activated if the P (price violation) option is also in effect.
F2-Save
Saves the quote hold information.
