Nightly Processing Printer Assignments screen
The Nightly Processing Printer Assignments screen allows you to set up printer options for the purchasing, transfer, and work order reports generated by Nightly Processing.
 
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F6-Printers > F6-NP Printers
Field and Function Key Descriptions
Field/Function Key
Description
Physical Printer
The name of the physical printer where the corresponding Nightly Processing report should print.
Stationary
The stationery type for the corresponding report.
Options
The printer assignment options for the corresponding Nightly Processing report:
•
H - Send to the HOLD file and send output to the corresponding printer.
•
HS - Send to the HOLD file and suppress printing.
•
P - Send output to the corresponding printer.
Copies
The number of copies that nightly processing should print of this report.
Purchasing
Indicates the printer information for purchasing reports generated by Nightly Processing.
Transfer
Indicates the printer information for transfer reports generated by Nightly Processing.
Work Order
Indicates the printer information for work order reports generated by Nightly Processing.
F2-Save
Saves the Nightly Processing printing information.
