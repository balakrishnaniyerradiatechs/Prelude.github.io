System Options for Gasket screen
If you use the Related Item Transfer Entry program in the Gasket/Hose Subsystem to transfer product quantities from bulk to remnant, you must set up a default inventory adjustment reason code.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Gasket Parameters
Field and Function Key Descriptions
Field/Function Key
Description
Gasket Adjustment Reason Code
The inventory adjustment reason code to default in Related Item Transfer Entry. Inventory adjustment reason codes are set up in Inventory > Maintenance > Misc Code Files > Inventory Adjustment Reason Code.
F2-Save
Saves the gasket parameter information.
