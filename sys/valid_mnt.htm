Valid Responses Maintenance screen
The Valid Responses Maintenance screen allows you to set up response codes that are acceptable choices to use for many of the standard and custom programs in Prelude.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Valid Response Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Validation Code
The code that is validated in a Prelude program and that must have a valid Response entered.
Description
The description of the Validation Code.
Response
The choices that can be used any time a response is required in the field indicated by the Validation Code.
Description
The description of the Response (e.g., if the response for a color is code is YL, the description is Yellow.
Abbr.
If applicable, an abbreviation of the response, so that whenever the abbreviation is entered, the full response appears on the screen.
F2-Save
Saves the valid response information.
F4-Del
Deletes the valid response code.
F9-Goto Line
Allows you to enter a line number, page number, or reference number to go directly to.
