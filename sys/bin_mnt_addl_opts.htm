Receiving Default Bins screen
If no default bins are assigned in the Bin Locator Options screen, you can set up a default bins for items received in Stock Receipts Entry and Transfer Receipts Entry.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > Action Bar > Addl Opts > Bin Locator Information > F5-Default Receiving Bin Range
Field and Function Key Descriptions
Field/Function Key
Description
Receiving Default Bins:
Beginning Range
The beginning bin for which you want to send received items into the Default Bin Number.
Ending Range
The ending bin for which you want to send received items into the Default Bin Number.
Default Bin Number
If the primary bin assigned to a product falls within the Beginning Range and Ending Range, it is received into this Default Bin Number.
This bin is not used if default bins are already assigned in the Bin Locator Options screen.
F2-Save
Saves the receiving default bin information.
