Startup Parameter Maintenance screen
The Startup Parameter Maintenance screen allows you to set startup parameters by port or location profile.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Startup Parameter Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Location Profile
A location profile, up to 20 alphanumeric characters (e.g., a port number or a string of characters that define a set of users). A location profile allows you to set up default information for a group of users (e.g, printers, terms codes, tax jurisdictions).
Selling Warehouse
The selling warehouse for this port.
Entry Warehouse
The default entry location for orders. Only users with access to multiple locations can change the entry warehouse.
Printer Port
The counter or continuous invoice printer override for this terminal. The default is assigned in Warehouse Address Maintenance.
Invoice Date
The invoice date to use for counter or continuous invoices.
Payment Terms
The default accounts receivable payment terms code to use for cash customers.
Tax Jurisdictions
Tax jurisdiction codes allowed for counter invoices and pick-up sales orders (four maximum allowed).
Description
The description of the tax jurisdiction code.
Order Source
The default order source for sales order for this port. You can override this information for each session.
Shared Terminal
Indicates if this terminal is shared by multiple users:
•
Y - Require a Written-By ID for every new order.
•
A - Require a Written-By ID any time an existing order is edited.
•
N - The Written-By ID will default to the person assigned to the terminal.
Prt Acknowledge
Indicates if orders entered from this terminal print acknowledgements:
•
Y - Print acknowledgments.
•
N - Do not print acknowledgements.
Promo Order
Determines if orders entered from this terminal are promotional orders:
•
Y - Orders entered from this terminal are promotional orders.
•
N - Orders entered from this terminal are not promotional orders.
Counter Terminal
Determines if this terminal is a counter terminal:
•
Y - This is a counter terminal. Cost and gross profit do not appear on sales orders.
•
N - This is not a counter terminal.
Data Entry Mode
Determines if this terminal is used for data entry only:
•
Y - This terminal is used for straight data entry only, detail information boxes do not display.
•
N - This terminal is used for more than just data entry.
Replacement Equipment
Determines if the Replacement Equipment default should be set to Y for this terminal:
•
Y - Sales Order Entry is set to default to replacement equipment entry.
•
N - This terminal should not default to replacement equipment entry.
Rental Terminal
Determines if this terminal only allows rental orders:
•
Y - Only allow entry of rental orders on this terminal. The rental order numbers will default to RENT instead of NEW.
•
N - Allow entry of both rental and sales orders on this terminal. Sales order numbers default to NEW, but you can enter R to enter a rental order in Sales Order Entry (Rental Order Entry numbers default to RENT).
Show Order
Determines if this terminal defaults to Show orders:
•
Y - Special show order priorities are set for orders entered from this terminal.
•
N - This terminal does not default to Show sales orders.
Scan Mode
Determines if the user for this terminal will scan products for return goods:
•
Y - The user will RF scan return goods.
•
N - The user will not RF scan return goods.
F2-Save
Saves the startup parameter information.
F4-Del
Deletes the startup parameter information.
F5-Printers
Opens the Startup Port Queue Assignment screen to allow you to enter the default printer number for all Prelude process queues that print documents.
F6-Retail Setups
Opens the Retail Register Setups screen to allow you to enter retail register information for the Point of Sale module.
