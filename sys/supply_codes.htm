Supply Code Maintenance screen
Supply codes are set up by Prelude and appear on a sales order when a product is on backorder and varies depending on the product’s replenishment path and product setting. The Supply Code Maintenance screen allows you to define valid supply codes for your company.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Supply Codes
Field and Function Key Descriptions
Field/Function Key
Description
Valid Supply Codes
The supply codes that are valid in for your company. The available types in Prelude are:
•
B - Purchase order.
•
E - Emergency purchase order.
•
D - Direct ship purchase order.
•
R - Emergency stock transfer.
•
T - Stock transfer.
•
W - Work order.
F2-Save
Saves the supply code information.
F4-Del
Deletes the supply codes.
F5-Goto Line
Allows you to enter a line number to go directly to.
F6-Alpha Sort
Sorts the codes in alphabetical order.
