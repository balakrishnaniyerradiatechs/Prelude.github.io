Currency Type Maintenance screen
If your company uses more than one type of currency to post transactions, you can set your system to handle multiple foreign currency rates (refer to “Using Multiple Currency” in the General Ledger documentation).
General ledger accounts and exchange rates are assigned in Currency Type Maintenance for each type of currency other than native currency.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Currency Maintenance > Currency Type Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Currency
The currency code that identifies the country whose currency is used by your company (e.g., CDN - Canadian, EU - European Union).
Description
The name of the country for the corresponding currency code.
Receiving Exch Rate
The exchange rate to use when stock is received in Stock Receipts Entry. This rate is used to convert costs on purchase orders to native currency.
Invoicing Exch Rate
The exchange rate to use to convert prices on sales orders to native currency at invoicing.
Customs Exch Rate
The exchange rate to use to convert duty charges on purchase orders to native currency. If this is blank, the Receiving Exch Rate is used.
G/L Table Headings
The general ledger account number to use for the corresponding currency account. Refer to the Currency Accounts for G/L Table Headings for a listing of currency accounts.
Div
The division number if your company uses divisions. A division is the highest possible grouping in your company (e.g., Eastern Division, Western Division). Enter X for a wildcard if the division should be taken from the transaction being posted.
Reg
The region number, if your company uses divisions. Regions are a subset of Division - if Divisions are used (e.g., Southeast Region, Northwest Region). Enter X for a wildcard if the region should be taken from the transaction being posted.
Loc
The location number which could be a warehouse location (e.g., Denver, Dallas) where inventory is kept, or it could be a corporate location where transactions may post to. Enter X for a wildcard if the location should be taken from the transaction being posted.
Dep
The department number if your company uses departments. Departments are a subset of Locations, typically used to separate expenses within a warehouse (e.g., accounting, sales, warehouse). Enter X for a wildcard if the department should be taken from the transaction being posted.
Ccr
The cost center number if your company uses departments. Cost centers are a subset of Departments - if Departments are used. Typically used to separate expenses within a warehouse or department (e.g., shipping, receiving, outside sales, inside sales). Enter X for a wildcard if the cost center should be taken from the transaction being posted.
Account
The general ledger currency account number to use. Refer to the Currency Accounts for G/L Table Headings for a listing of currency accounts.
F2-Save
Saves the general ledger currency information.
F4-Del
Deletes the general ledger currency type.
F5-History
Opens the Exchange Rate History screen which lists the exchange rates for past fiscal periods.
