CC Markup Maintenance screen
If you want to pre-authorize credit card sales orders for more than the merchandise amount (e.g., to account for freight charges), you can set up a credit card table to determine the markup percent to add to the merchandise amount pre-authorized on a credit card. If the total order amount ends up being more that what was pre-authorized, it wonâ€™t be rejected when you post-authorize. You can enter multiple levels of markups.
Order Processing Parameters
To access the CC Table Maintenance screen, Add Markup To Pre-Auth Amt in the Add CC Software screen (Order Processing Parameters > F5-Add Info > Credit Bureau Auth Code = P) must be set to Y.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Direct Order Entry Maintenance > Credit Card Markup Table Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Table Number
The table number that references a markup.
Markup Limit
The amount that excludes an order from having a markup added. No markup will be added to any orders greater than this amount.
Amount
The cutoff amount that determines when to add a markup percent. For example, if you enter 10, all orders that are $10 or less will have the corresponding Markup % added to their order. You can enter more than one cutoff amount.
Markup %
The markup percent the customer will receive when they purchase up to the corresponding Amount.
F2-Save
Saves the markup table information.
F4-Del
Deletes the markup table.
