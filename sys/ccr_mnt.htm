Cost Center Address Maintenance screen
If your general ledger hierarchy includes Cost Centers, (General Ledger Controls > Hierarchy Maintenance > F9-Ccr List), you can use the Cost Center Address Maintenance screen to add or edit address and manager information for each cost center.
You can only access this screen if Cost Centers is set to Y in the Hierarchy Level Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Cost Center Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Division Number
The division number that the location is assigned to, if divisions are used by your company.
Region Number
The region assigned to the location, if locations are used by your company.
Location Number
The warehouse or corporate location that transactions can post to.
Department Number
The department number for this location.
Cost Center Number
The cost center to assign as a subset of this department.
Description
The description of the cost center (e.g., shipping, receiving, outside sales, inside sales).
