Scan Prefix Maintenance screen
You can use the Scan Prefix Maintenance screen to set up prefix codes to use with alternate part numbers.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Scan Prefix Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Type
The type of code for which you are setting up a prefix.
Code
The code for the corresponding types of prefixes.
Product Code
The product code to use as a prefix for the alternate product number. The alternate product number uses this code as a prefix to the product number. For example, if Product Code is P and a part number is set up as AS100, the alternate part number for the product is PAS100. New part numbers automatically include the prefix code.
Qty Code
The quantity code to use as a prefix for the quantity field in the Lot Information screen for a lot-controlled item. For example if Qty Code is Q then you can enter Q5 in the Lot Information screen for a quantity of 5.
Lot# Code
The lot code to use as a prefix to the lot number field in the lot information screen for a lot controlled item. For example, if Lot# Code is L, then you can enter L101 in the Lot Number field in the Lot Information screen as an alternate number for lot 101.
Serial# Code
The serial number code to use as a prefix. For display only.
F2-Save
Saves the prefix code information.
F4-Del
Deletes the prefix codes.
