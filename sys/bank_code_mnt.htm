Accounts Receivable Controls
Bank Account Code Maintenance screen
The Bank Code Maintenance screen allows you to create a code to identify each bank account that you use for Accounts Payable and Accounts Receivable. You can also set up general ledger, routing, and currency information for each account.
Menu Path: Systems > System Parameters > Maintenance > Accounts Receivable Controls > Bank Code Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Bank Code
The bank code to identify the bank where accounts receivable checks are deposited and where accounts payable vendors are paid from. The bank code is the prefix for the check numbers to make sure they are unique.
Bank Account Desc
The name of the bank.
Address One
The first line of address for the bank.
Address Two
The second line of address for the bank.
City
The city where the bank is located.
State
The state where the bank is located.
Zip
The zip code.
Bank Account Number
Your account number at this bank.
Div
The division to use to post to this account, if you use divisions.
Reg
The region to use to post to this account, if regions are used.
Loc
The warehouse location to use to post to this account.
Dep
The department to use to post to this account.
Ccr
The cost center to use to post to this account.
Account
The general ledger account number to use when posting to this account.
Currency Ind.
The type of currency used by this bank. This information is required if Multiple Currency is set to Y in the General Ledger Parameters screen. Refer to “Using Multiple Currency” in the General Ledger documentation for more information.
Bank Routing Number
The bank transfer routing number for this bank code. This information may appear on forms and reports.
Fractional Routing#
The fractional routing number for this bank code.
EFT File Build Pgm
The electronic file transfer build program executed in A/P check printing for EFT batches.
F2-Save
Saves the bank code information.
F4-Del
Deletes the bank code.
F5-Cash Post Acct
Opens the Cash Sales Posting Account Maintenance screen to allow you to enter the account for posting cash sales.
