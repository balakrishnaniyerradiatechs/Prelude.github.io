Warehouse Sequence Maintenance screen
The Warehouse Sequence Maintenance screen allows you to set the order to display warehouses when checking inventory in other warehouses.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Sequences Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The warehouse for which you want to set up a sequence of warehouses to view.
Whse
The warehouse numbers in the sequence to display in inventory inquiries. The first warehouse on the list must be the warehouse for which you are setting up the list (i.e., the warehouse entered in the previous field). The second warehouse on the list might be the warehouse that is the closest, or the warehouse that you are most likely to transfer inventory from.
Description
The warehouse description.
F2-Save
Saves the warehouse viewing sequence information.
F4-Del
Deletes the warehouse sequence information.
F5-Purchasing Seq
Opens the Purchasing Warehouse Sequence Maintenance screen to allow you to set up a sequence of warehouses to display for purchasing inquiries.
F6-AutoSplit Seq
Opens the Auto Split Warehouse Sequence Maintenance screen to allow you to enter an override list of warehouses to use for the Auto Order Split screen appears which appears in Sales Order Entry if a backordered quantity exists in another warehouse. The customer’s alternate shipping warehouse (Customer Maintenance > F7-Order > Action Bar > Alternate Shipping Warehouse) overrides the warehouse sequence.
This works with the Shipping Whse Split/Reassignment field in the Order Processing Parameters Additional Options screen.
Refer to “Shipping Split Quantities out of Other Warehouses” in the Order Processing documentation for more information.
