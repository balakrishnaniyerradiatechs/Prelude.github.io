Inventory Adjustment Types screen
Inventory Adjustment Types are set up by Prelude and used in Inventory Adjustment Entry to determine how the general ledger is affected when inventory is adjusted. You can use the Inventory Adjustment Types screen to set the default adjustment type that appears in Inventory Adjustment Entry and to determine which types are valid for your company. Refer to “Adjusting Inventory Quantities” in the Inventory documentation for more information on adjusting inventory.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Inventory Adjustment Entry Types
Field and Function Key Descriptions
Field/Function Key
Description
Type Default
The inventory adjustment type to default in Inventory Adjustment Entry (Inventory > Maintenance > Inventory Adjustments > Inventory Adjustment Entry).
Valid Types
The adjustment types that are valid in Inventory Adjustment Entry for your company. The available types in Prelude are:
•
INV - Regular on-hand inventory.
•
DMG - Damaged inventory.
•
SCR - Scrapped/lost inventory.
•
WTY - Warranty inventory.
F2-Save
Saves the adjustment type information.
F4-Del
Deletes the adjustment type information.
