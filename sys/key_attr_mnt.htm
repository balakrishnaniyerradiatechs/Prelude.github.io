Miscellaneous Controls
Keyword Maintenance screen
The Keyword Controls Menu allows you to set up keywords to identify which fields to use to lookup information (e.g., customers, products, vendors, orders). For example, you can add a sales representative as a search criteria for a customer, or a buyer as a keyword for a vendor.
You must run the related rebuild to activate keyword changes (Systems > System Parameters > Processes . ReBuild Menu).
