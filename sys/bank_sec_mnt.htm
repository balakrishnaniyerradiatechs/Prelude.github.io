Bank Code Authorizations screen
The Bank Code Authorizations screen allows you to limit the bank codes that a user can enter in accounts receivable and accounts payable programs.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance > F6-Add Info > F8-Bank Sec.
Field and Function Key Descriptions
Field/Function Key
Description
Approved Bank Codes
The bank codes that this user can enter in financial programs. Bank codes are set up in the Bank Account Code Maintenance screen.
F2-Save
Saves the bank code information.
