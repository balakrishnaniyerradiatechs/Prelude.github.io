Transfer Information screen
The Transfer Information screen allows you to set up serial shipping container code information if you use SSCCs to ship transfers. A Serial Shipping Container Code is a unique identification system used to identify a carton or pallet. SSCCs provide quality assurance by assigning codes and printing labels to verify shipped products and quantities.
Transfer Serial Shipping Container Entry allows you to assign SSCC codes to print on labels for transfer shipments that contains pallets or cartons. You can determine whether you want to print just the shipment label or to print a listing of the contents of the shipment. Refer to “Using Serial Shipping Container Coders for Transfers” in the Inventory documentation.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > Action Bar > Addl Opts > Transfers
Field and Function Key Descriptions
Field/Function Key
Description
Customer Number
The customer number, if this warehouse represents a customer site.
Contents Tracking
Determines if detail tracking labels print in Transfer Serial Ship Container Entry:
•
Y - Require SSCC detail contents’ tracking labels to print when shipping transfers in Transfer Serial Ship Container Entry. If you ship a transfer in Regular Transfer Shipment Entry, you are automatically brought into the Serial Shipping Container Inquiry screen so you can enter the detail contents of the shipment.
•
N - Detail tracking labels do not print in Transfer Serial Ship Container Entry.
Ucc Number
The UCC manufacturer number for the warehouse to use in the SSCC identification number, if it is different than the UCC Mfg Number defined at the company level in the Company Maintenance Additional Information screen.
The UCC Mfg Number is the Uniform Code Council manufacturer number assigned to your registered organization by the Uniform Code Council. This code is part of the SSCC identification number.
Sscc Label Pgm
The name of your custom program that prints SSCC labels in Transfer Serial Container Entry. The labels will not include a detail listing of the contents of the shipment. Refer to “Printing SSCC Labels Without Listing Contents” in the Inventory documentation.
Sscc Label Execution
Determines when the SSCC labels print:
•
N - Labels print when the SSCC is assigned in Transfer Serial Container Entry.
•
E - Labels print after entering each transfer SSCC entry.
•
F - Labels print after entering all the SSCCs for a transfer release number.
Pc Label Pgm
The name of your custom program that prints SSCC labels in Transfer Serial Container Entry. The labels will include a detail listing of the contents of the shipment. Refer to “Printing Shipment Contents on SSCC Labels” in the Inventory documentation.
Pc Label Execution
Determines when the Pc labels print:
•
E - Labels print after each transfer SSCC entry.
•
F - Labels print after all SSCCs for a transfer release number are entered.
Bin Loc Sort -TRN
Determines the sort sequence for transfer pick tickets:
•
Y - Sort by bin.
•
N - Follow company settings.
•
P - Sort by product.
•
L - Sort by product line.
This flag overrides the Document Sort T/R company level setting in the Company Maintenance Additional Information screen.
F2-Save
Saves the transfer information.
