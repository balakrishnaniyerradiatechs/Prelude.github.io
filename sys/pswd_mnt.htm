Password Maintenance screen
The Password Maintenance screen allows you to restrict access to fields, programs, and menus. For each Process Name in Prelude, you can enter a Password. The Password Process table below lists the processes that you can password protect.
Menu Path: Systems > System Parameters > Maintenance > Program and Menu Passwords
Yes/No Dictionary Items
You can password protect items with a validation code of YES.NO.VAL using the Process Name "filename.fieldname". For example, to protect Prohibit Sales in Customer Maintenance, use CUSTOMER.PROHIBIT.SLS.
Menus
You can password protect menus by using the menu name as the Process Name and assigning a Password. For example, to restrict the Accounts Payable menu, enter APS as the Process Name.
WARNING!!! When you password protect a menu, only that menu restricted; you can slash directly to a submenu and bypass the password protection. For example, if the Accounts Payable menu is restricted, you can use /APS.APPO to go directly to the AP/PO Subsystem menu without a password.
