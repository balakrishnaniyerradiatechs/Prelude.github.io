Canada Info Entry screen
The Canada Info Entry screen allows you to enter Canadian tax identification information for the company.
Menu Path: Systems > System Parameters > Maintenance > Company Maintenance > Action Bar > Canada Info
Field and Function Key Descriptions
Field/Function Key
Description
Tax Id 1
The first Canadian tax identification number.
Tax Id 2
The second Canadian tax identification number.
Tax Id 3
The third Canadian tax identification number.
F2-Save
Saves the tax ID information.
