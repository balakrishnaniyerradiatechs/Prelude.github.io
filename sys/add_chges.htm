Additional Frt/Handling screen
The Additional Frt/Handling screen defines how freight terms are used to calculate freight on sales orders.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Order Processing Parameters > F7-Add. Frt/Handling
Field and Function Key Descriptions
Field/Function Key
Description
Add Frt/Handling Min
Defines the minimum amount to add to each invoice. If the result of the base amount (defined in Add Frt/Handling Bas) multiplied by the percentage (defined in Add Frt/Handling Prc) is less than this amount, then this value is used.
Add Frt/Handling Max
Defines the maximum amount to add to each invoice. If the result of the base amount (defined in Add Frt/Handling Bas) multiplied by the percentage (defined in Add Frt/Handling Prc) is more than this amount, then this value is used.
Add Frt/Handling Prc
The percentage to multiply by the base (defined in Add Frt/Handling Bas) to determine the amount of the freight or handling charge. If left blank, no additional freight or handling charge is calculated.
Add Frt/Handling Bas
Defines the basis to use to multiply by the percent defined in Add Frt/Handling Prc to calculate freight. Set to M to use the Merchandise amount from the Sales Order Completion screen. Set to O to use the Outbound Freight amount from the Sales Order Completion screen.
F2-Save
Saves the freight and handling charge information.
