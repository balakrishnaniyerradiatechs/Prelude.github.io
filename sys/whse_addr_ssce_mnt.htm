Warehouse Address SSCE Information screen
The Warehouse Address SSCE Information screen allows you to enter default bin information for Serial Shipping Container Entry. Refer to “Assigning Serial Shipping Container Codes” in the Release Confirmation and Shipping section of the Order Processing documentation.
Requirements for Bin Moves in SSCE
•
Multiple Bin Locations must be set to Y in the Bin Locator Options screen
•
Warehousing Flag must be set to N in the Warehousing Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > Action Bar > Addl Opts > SSCE Information
Field and Function Key Descriptions
Field/Function Key
Description
SSCE Bin Move
Determines if you can perform a bin move when you confirm a release in SSCE (Serial Shipping Container Entry):
•
Y - You can perform a bin move.
•
N - A bin move is not allowed.
SSCE Default Shipping Bin
The default bin to use as the destination bin for the SSCE bin move. You can also enter the bin number when you confirm the shipment.
SSCE Picking Bin Prompt
Determines if the bin box displays for each line in SSCE:
•
Y - Display the bin box for each line.
•
N - Only display the bin box if a line references more than one bin.
F2-Save
Saves the SSCE information.
