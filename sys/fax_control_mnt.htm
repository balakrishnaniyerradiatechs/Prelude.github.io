Fax Control Record Maintenance
The Fax Controls menu allows you to set up header information for each type of document that can be sent by fax. You can also use this information to track when each type of document is faxed.
These options may not apply due to custom modifications related to faxing and/or user programs created before the fax controls were developed, or if your USER FORMS do not include logic to honor the hierarchy.

Field and Function Key Descriptions
Field/Function Key
Description
Fax Control Type
Depending on the option you chose on the Fax Controls menu, the type of document to fax.
Warehouse
The warehouse where you want to add or change the fax information.
Contact Name
The contact name to appear on all documents for this fax control type. Leave blank to always use the name of the person sending the fax.
Main Phone
The phone number to appear on the cover sheet, if applicable.
Main Fax
The fax number to appear on the cover sheet, if applicable.
Email Address
The email address to use to confirm the fax or notify of errors, if the user does not have an email address defined in User Identification Maintenance (not used by VSI-Fax).
Desqtop User ID
The Desqtop user ID of the person who should received fax error and confirmation messages, if using Desqtop (no longer supported by Prelude).
Immediate/
Overnight
Determines when the documents are faxed:
•
I - Fax the document immediately.
•
O - Fax the document overnight.
F2-Save
Saves the fax control information.
F4-Del
Deletes the fax control information.
F5-Vsifax Options
Opens the VSI-FAX Whse Fax Control screen to allow you to enter faxing information.
