VSI-FAX Whse Fax Control screen
The VSI-FAX Whse Fax Control screen allows you to enter information for faxes or emails that you send by VSI-fax.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Fax Controls > fax control menu option > F5-Vsifax Options
Field and Function Key Descriptions
Field/Function Key
Description
E-Mail Notify Address
The email address to receive VSI-fax notifications.
Document Logo Path
The path to the location of the logo, if you use a logo on this type of document. If your operating system is Unix, use a forward slash (\) before the path-name (e.g., \/usr/vsifax3/lib/images/demo.tif).
Must be uploaded through vsifax\/upload.
Document Overlay Path
The path to the location of the overlay, if you use an overlay image on this document. If your operating system is Unix, use a forward slash (\) before the pathname (e.g., \/Usr/Vsifax3/lib/images/demo.tif).
Must be uploaded through vsifax\/upload.
Fax Destination
Override for default VSI-Fax device or class.
Fax Coverpage
Override for Company default coverpage template. VSI-Fax only.
Suppress Coverpage
Determines if you want to include or suppress the cover page:
•
Y - Suppress (do not fax) a cover page.
•
N - Fax a cover page.
Overnight/
Immediate
Determines when the documents are faxed:
•
I - Fax the document immediately.
•
O - Fax the document overnight at the time entered in Overnight Time.
Overnight Time
If you entered O for Overnight/Immediate, enter the time the faxes are to be sent.
F2-Save
Saves the VSI-Fax information.
F4-Del
Deletes the VSI-Fax information.
F5-Memo Message
Opens the VSIFAX MEMO screen to allow you to enter a message to appear on the coverpage.
