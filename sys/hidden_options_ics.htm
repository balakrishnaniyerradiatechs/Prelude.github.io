ICS Additional Option Parameters screen
The ICS Additional Option Parameters screen allows you to set option parameters for the inventory module.
Menu Path: Additional Options Menu > Additional Option Parameters > F6-ICS
Field and Function Key Descriptions
Field/Function Key
Description
Inventory Control Options
Allocation Audit Report Flag
No longer applicable.
Display Receiver# In Stock Receipts
Determines if the assigned receiver number appears at the bottom of the screen when you save the Stock Receipts Entry screen:
•
Y - Display the receiver number.
•
N - Do not display the receiver number.
Generate Sequential Serial Numbers
Indicates if you want to generate sequential serial numbers when you receive products that require inspection:
•
Y - Generate sequential serial numbers.
•
N - Do not generate sequential serial numbers. You must manually enter the serial numbers.
Refer to “Entering and Tracking Serial Numbers” in the Inventory documentation for more information.
Master Product Number Flag
Determines if the product line description is a keyword for a product:
•
Y - Use the product line description as a keyword to search for a product.
•
N - Do not use the product line description as a keyword.
Product lines are assigned to products in Product Maintenance.
Product Keyword Lookup Menu Options
Determines if you can search for a product by product keyword and/or a lot keyword:
•
L - Allows you to search for products by product keyword or lot keyword.
•
Blank - You can only search by product keyword.
Warehouse Specific Defaults
Determines if the system checks for warehouse defaults before checking for a non-warehouse default record when creating new product, product/warehouse, product/vendor, or customer records:
•
Y - Check for warehouse defaults when creating a new record.
•
N - Use non-warehouse defaults.
This only applies if the user ID has a defined warehouse. The defaults file is maintained in TCL. See your system administrator for more information.
Disallow Same Lot# For Diff P/N’S
Determines if the same lot number is allowed for 2 part numbers:
•
Y - You cannot use the same lot number for different product numbers.
•
N - You can use the same lot number for different product numbers.
You can override this flag for sales orders in Disallow Same Lot# For Diff P/N’s in the Order Processing Parameters Additional Information screen.
Refer to “Setting Up a Lot Product” in the Stock Receipts section of the Inventory documentation.
Proposed Product Numbers
Determines if you can use an abbreviated skeleton entry program for new products in Quotation Entry:
•
Y - When a new product is entered on a quote, an abbreviated version of Skeleton Product Maintenance is used to build the product. If the quote is converted to a sales order, you must enter the remaining information as required in Skeleton Product Maintenance.
•
N - When a new product is entered on a quote, Skeleton Product Maintenance is used to build a new product.
Refer to “Creating a Product in Sales Order Entry” in the Order Processing documentation for more information.
Use Whse Over Lot Cost
Determines which cost to use for lot numbers in the general ledger:
•
Y - Use warehouse cost for general ledger entries.
•
N - Use average lot cost for general ledger entries.
Use Whse Over Serial Cost
Indicates whether you will use average cost or the serial number cost to update the general ledger:
•
Y - Use average cost to update the general ledger.
•
N - Use serial number cost to update the general ledger.
Refer to “Entering and Tracking Serial Numbers” in the Inventory documentation for more information.
Use Serialized Cost With FIFO
Determines if transactions for serialized items use the serial cost of the item for FIFO costing:
•
Y - Use the serial cost of the item instead of the FIFO layers cost. The layer’s on-hand cost is relieved accordingly.
•
N - Use the FIFO layer cost of the item.
This flag is only accessible if G/L Cost (Company Maintenance > F5) is set to F (FIFO). With FIFO costing, FIFO layers are relieved based on the first-in first-out method and the cost is the layer being relieved. If you have serialized items, FIFO normally ignores the serial cost and updates by layer.
Refer to “Entering and Tracking Serial Numbers” in the Inventory documentation for more information.
Force Transfer To User Location
Determines if the To warehouse in the Transfer Entry screen defaults to the user’s location:
•
Y - Default the user’s location in the To field.
•
N - Require the user to enter a To warehouse.
Force User Location In Inv Adj
Determines if the Warehouse in the Inventory Adjustment Entry screen defaults to the user’s location:
•
Y - Default the user’s location in Warehouse.
•
N - Require the user to enter a Warehouse.
Refer to “Adjusting Inventory Quantities” in the Inventory documentation for more information.
Force Batch Receipt To Today
Determines if today’s date is forced as the receipt date in Batch Stock Receipts:
•
Y - Use today’s date as the receipt date in Batch Stock Receipts.
•
N - Allow any date for the receipt date in Batch Stock Receipts.
New P/N Default To Catalog File?
Determines if you have a choice to add a product to the catalog file or the inventory file when you create product in Product Maintenance:
•
Y - A message displays when you create a new product in Product Maintenance to ask if you want to add the product to the inventory file or the Catalog file. If Product Catalog File in the File Update Parameters screen is set to Y, the Catalog option is the default. If Product Catalog File in the File Update Parameters screen is set to N, the Inventory option is the default.
•
N - No message displays when you create a new product in Product Maintenance.
The inventory count file contains stock items and non-stock items that your warehouse stocks and tracks. The catalog file contains all products that your vendors carry that you store for reference, or products that you no longer sell.
Use Mod10 Check For Upc
No longer applicable.
Update Primary Bin When Incr Onhand
Determines if the primary bin in Product Warehouse Maintenance is updated when the on-hand quantity is increased for the first time:
•
Y - Update the Primary Bin the first time a quantity is received. After that, you must manually change the bin number for the product if necessary.
•
N - Do not update the Primary Bin.
Product License Plate
Determines if you use license plates for lot control:
•
Y - Activates license plates for lot control tracking. Every item in inventory must have a license plate number assigned.
•
N - License plates are not used.
Refer to “Setting Up License Plate Products” in the Stock Receipts Section of the Inventory documentation.
Serial# DS POs Non-Detail
Determines if you can enter serial numbers for serialized products when direct ship purchase orders are received (if detail tracking within Product Maintenance > F5-Add Info is set to N):
•
Y - You can enter serial numbers for serialized products when you receive a direct ship purchase order.
•
N - You cannot enter serial numbers for serialized products on direct ship purchase orders.
Refer to “Entering and Tracking Serial Numbers” in the Inventory documentation for more information.
Country Of Origin Required
Determines if the country of origin is required when you receive products (if the country of origin is not already assigned in Product Maintenance):
•
Y - Require a country of origin when products are received.
•
N - Do not require a country of origin when products are received.
F2-Save
Saves the inventory option parameter information.
