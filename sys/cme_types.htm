Credit Memo Entry Types screen
Credit memo entry types are set up by Prelude and used in Credit Memo Entry to define the type of credit memo. The Credit Memo Entry Types screen allows you to set the default order type that appears in the header screen and to determine which types are valid for your company. Refer to “Credit Memos” in the Order Processing documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Order Type Defaults > Credit Memo Entry Types
Field and Function Key Descriptions
Field/Function Key
Description
Order Type Default
The order type to default in the header screen in Credit Memo Entry.
Valid Order Types
The types that are valid in Credit Memo Entry for your company. The available types in Prelude are:
•
05 - Counter invoice.
•
06 - Confirmation.
F2-Save
Saves the credit memo type information.
F4-Del
Deletes the valid credit memo types.
