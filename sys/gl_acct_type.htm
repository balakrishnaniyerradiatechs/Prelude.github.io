G/L Account# Type Range Maintenance screen
The G/L Account# Type Range screen identifies the range of general ledger numbers to assign to each Account Type. This account type appears in Chart of Accounts Maintenance for each account.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > G/L Account# Type Range
Field and Function Key Descriptions
Field/Function Key
Description
G/L Account Type
The general ledger account type.
A - Asset
The range of general ledger numbers that define asset accounts.
L - Liability
The range of general ledger numbers that define liability accounts.
C - Capitol
The range of general ledger numbers that define capitol accounts.
I - Income/Revenue
The range of general ledger numbers that define income and revenue accounts.
E - Expense
The range of general ledger account numbers that define expense accounts.
F2-Save
Saves the account type information.
F4-Del
Deletes the account type information.
