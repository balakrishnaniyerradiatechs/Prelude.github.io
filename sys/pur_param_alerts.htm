Product Alert Parameters screen
The Product Alert Parameters screen allows you to set a day of the month to evaluate a product’s usage and the percentage of the product’s Forecast that causes an Early Warning alert. If a product’s usage by that day exceeds the defined percent of its Forecast, an Early Warning product alert appears in the Buyer Queue. A product’s Forecast is set up in Product Purchasing Parameters Maintenance. Refer to the Buyer Queue “Product Alerts” in the Purchasing documentation for more information on early warning alerts.
Menu Path: Systems > System Parameters > Maintenance > Purchasing Controls > Purchasing Parameters > F7-Product Alerts
Field and Function Key Descriptions
Field/Function Key
Description
Evaluate Forecast
Day
The day of the month to evaluate a product’s usage.
%
The percentage of the product’s Forecast that causes an Early Warning alert. If a product’s usage by that day exceeds the defined percent of its Forecast, an Early Warning product alert appears in the Buyer Queue. A product’s Forecast is set up in Product Purchasing Parameters Maintenance.
F2-Save
Saves the product alert information.
