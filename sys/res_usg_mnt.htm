Restricted License Info Maintenance screen
The Restricted License Info Maintenance screen allows you to define each state in which the company has a license to sell restricted products to customers.
You must set up both the customer and the product with the same restricted product matrix and usage codes:
•
Products are designated with matrix restrictions in Product Maintenance > F5-Add Info > F6-MSDS/Permits > Restricted Product = M.
Refer to “Assigning MSDS Numbers to Products” in the Inventory documentation.
•
Customers are set up with matrix restricted usage permits in Customer Maintenance > F7-Order > F5-Add Info > Action Bar > Permits > Restrict Use Permit = M.
Refer to the “Customer Permit Information Screen” in the Order Processing documentation.
•
Restricted Usage State Codes are set up in Order Processing > Maintenance > Customer Control Files > Restricted Usage Codes.
Refer to the “Restricted Usage Code Maintenance screen” in the Order Processing documentation.
Menu Path: Systems > System Parameters > Maintenance > Company Maintenance > Action Bar > Restricted Lic Info
Field and Function Key Descriptions
Field/Function Key
Description
Usg Code
The restricted usage state code set up in Restricted Usage Code Maintenance. The usage code must be the same code as the state code on the ship-to address of the customer’s sales order, if the customer is allowed to purchase a product that has matrix sales restrictions.
Permit Number
The permit number for the restricted product in the corresponding state.
Exp Date
The expiration date of the permit.
F2-Save
Saves the restricted usage code information.
