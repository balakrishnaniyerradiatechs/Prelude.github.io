Parker Reporting Information screen
The Parker Reporting Information screen allows you to enter a Parker distribution number for a warehouse. The Parker Hannifin manufacturer requires its distributors to track and report all sales of Parker products. You can set up Parker divisions, marketing codes, and distributor codes, and then assign the codes to product lines or products. Refer to “Parker Reporting Codes” in the Inventory documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > Action Bar > Addl Opts > Parker Reporting Information
Field and Function Key Descriptions
Field/Function Key
Description
Parker Distributor #
The parker distributor number for this warehouse. This number overrides the parker distributor number set up in Inventory > Maintenance > Product Controls Files > Parker Reporting Codes > Parker Distributor Maintenance.
F2-Save
Saves the Parker information.
