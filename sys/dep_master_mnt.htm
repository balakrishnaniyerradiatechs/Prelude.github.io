Department Master Maintenance screen
The Department Master Maintenance screen defines departments set up for your company. Departments are a subset of Locations, used to separate expenses within a warehouse (e.g., accounting, sales, warehouse). Use the Department Address Maintenance screen to add or edit address, division, region, location, and manager information for the department.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Hierarchy Maintenance > F8-Dep List
Field and Function Key Descriptions
Field/Function Key
Description
Dep
The number for each valid department.
Description
The description of the department (e.g., 01, shipping, 01, receiving).
F2-Save
Saves the department information.
F4-Del
Deletes all the departments listed on the screen.
