Region Maintenance screen
If your general ledger hierarchy includes Regions, (General Ledger Controls > Hierarchy Maintenance > F6-Reg List), you can use the Region Maintenance screen to add or edit address and manager information for each region.
You can only access this screen if regions is set to Y in the Hierarchy Level Maintenance screen.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Region Maintenance
