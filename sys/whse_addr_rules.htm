Picking Rules screen
The Picking Rules screen defines the days that this warehouse picks and fills sales orders.
The hierarchy to determine picking days is as follows:
•
Order Processing > Maintenance > Customer Controls Files > Route Codes > F5-Picking Days.
•
Order Processing > Maintenance > Customer Control Files > Ship Via Codes > Action Bar > Addtl Optns > Picking Days.
•
Inventory Controls > Warehouse Maintenance > F5-Add Info > F6-Picking Days.
Refer to “Printing Pick Tickets” in the Order Processing documentation for detailed information on pick ticket rules and the hierarchy used to determine when to print pick tickets.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F5-Add Info > F6-Picking Days
