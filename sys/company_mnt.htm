Company Maintenance
Company Maintenance screen
The Company Maintenance screen allows you to set up address information for a company. From this screen, you can access additional screens to set up system options and activate Prelude modules used by the company. You can have more than one company for your organization.
Menu Path: Systems > System Parameters > Maintenance > Company Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Company Number
The 3-digit number to assign to each company within your organization. Use company number 001 if you have only one company.
Description
The description or name of the company.
Address One
The first line of address for the company.
Address Two
The second line of address for the company.
Address Three
The third line of address for the company.
Country
The country of the company.
City
The city where the company is located.
State
The state where the company is located.
Zip
The zip code.
Remit Address One
The first line of the remittance address to print on the accounts receivable statements.
Remit Address Two
The second line of the remittance address to print on the accounts receivable statements.
Remit Address Three
The third line of the remittance address to print on the accounts receivable statements.
Remit Address Four
The fourth line of the remittance address to print on the accounts receivable statements.
Phone Number
The phone number of the company.
Fax Number
The main fax number of the company.
Federal Tax Id
The company’s federal payroll tax ID number.
State Tax Id
The company’s state payroll tax ID number.
F2-Save
Saves the company address and phone number information.
F4-Del
Deletes the company record.
F5-Add. Info
Opens the Additional Information screen to allow you to set flags for Order Processing, Accounts Payable, Inventory Control, and the General Ledger.
F6-Modules
Opens the Modules screen to allow you to indicate the Prelude Modules available for this company to use.
Action Bar
Restricted Lic Info
Opens the Restricted License Info Maintenance screen to allow you to define the states where this company is allowed to sell restricted products.
GUI Options
Opens the Company GUI Options screen.
Canada Info
Opens the Canada Info Entry screen to allow you to enter tax ID information for Canada, if applicable.
