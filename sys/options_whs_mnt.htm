Option Parameters For Warehousing screen
If you use the Warehousing Module, the Option Parameters for Warehousing screen allows you to set radio frequency options. Other warehousing options are set in Warehouse Maintenance.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Warehousing Parameters
Field and Function Key Descriptions
Field/Function Key
Description
RF Quantities
Determines if the user can scan barcode quantities using RF (radio frequency) units:
•
Y - Bar code quantities can be scanned with RF units.
•
N - Bar code quantities cannot be scanned with RF units.
•
W - Prompt for Worksheet display in the RF and Truck mount screens (e.g., picking, receipts, putaway). If this flag is set to W, you can access the Rf Worksheet Qty Screen flag in Warehousing Maintenance screen.
RF Inv Adj Reason Code
The inventory adjustment reason code to use in RF Inventory Adjustment Entry when a quantity is changed.
F2-Save
Saves the warehousing parameter settings.
