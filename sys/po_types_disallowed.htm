P/O Type Restriction screen
You can use the P/O Type Restriction screen to disallow purchase order types for a user.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance > Action Bar > User Options > P/O Type Restriction
Field and Function Key Descriptions
Field/Function Key
Description
Disallowed P/O Types
The purchase order types that this user cannot use to create a purchase order.
F2-Save
Saves the disallowed P/O type information.
