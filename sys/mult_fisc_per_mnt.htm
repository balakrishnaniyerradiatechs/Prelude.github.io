Multi-Fiscal Period Maint screen
Multi-Fiscal Period Maintenance allows you to define fiscal periods for each fiscal account. You can use this as a cross-reference for each fiscal period/location to indicate which fiscal period to post to in each fiscal account. For example, a transaction posted to the parent account in period 07-05 may post to 07-12 in the fiscal account. Refer to “Using Multiple Fiscal Periods” in the General Ledger documentation.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Multi-Fisc Period Table Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Fiscal Account
The name of the multi fiscal account.
Main Acct
The fiscal period that the parent account should post to.
Fisc Acct
The fiscal period that the fiscal account should post to.
F2-Save
Saves the fiscal period information.
F4-Del
Deletes the multiple fiscal period information.
