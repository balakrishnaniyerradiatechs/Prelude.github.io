Email Document Text Maintenance screen
The Email Document Text Maintenance screen allows you to set up a document code to use when text emailing order acknowledgements or shipping confirmations. Each code represents a text document to use to email an acknowledgement to a customer when a sales order is entered or to email a shipping confirmation once an order ships.
Text Email Documents
Ack Document Option in Customer Maintenance > F7-Order > F5-Add Info must be set to T (text email) for a customer to receive order acknowledgments via text email.
Email Ship Notice in Customer Maintenance > F7-Order > F5-Add Info must be set to Y (text email) for a customer to receive shipping confirmations via text email.
Refer to “Sending Shipping Confirmations” and “Sending Acknowledgments” in the Order Processing documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Miscellaneous Controls > Email Document Text Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Document Code
The 6-digit code to represent the type of text email (e.g., ORDACK for sales order acknowledgements, ORDSHP for shipping acknowledgements).
Language Indicator
The language code to use for this type of text email. Only applicable if the customer has a value in Language Indicator set in Customer Maintenance > F7-Order.
Description
The description of the text email code.
F2-Save
Saves the text email code information.
F4-Delete
Deletes the text email code.
F5-Document Text
Any standard text to send for this type of text email. You can also import data from a sales order, such as the customer name, purchase order number, product detail, ship via, or tracking numbers for a pick ticket release. The following items can be typed in the text itself, however, each line consisting of these items can contain no other text:
•
% CUSNAME% - Customer name.
•
%CPO% - Customer purchase order.
•
%DETAIL% - Product detail, including backordered quantities and freight charges.
•
%SHIP% - Ship via method.
•
%TRACK% - Release tracking numbers.
•
%JOB.DESC% - Job description.
•
%CUSTADDR% - Customer address.
•
%URL% - Link to freight company website for each tracking number in the release.
