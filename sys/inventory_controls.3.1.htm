Inventory Controls
Warehouse Maintenance screen
The Warehouse Maintenance screen allows you to set up warehouse information for each location in your company.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Warehouse
The number to assign to the warehouse. This number must be the same length as Location in the general ledger hierarchy (1 to four characters).
Description
The description of the warehouse (e.g., Dallas, Denver, Atlanta).
Address One
The first line of address for the warehouse.
Address Two
The second line of address for the warehouse.
Address Three
The third line of address for the warehouse.
Country
The country code for the warehouse.
GeoCode
The geo code from Geographic Code Maintenance (Order Processing module). If you enter a geo code, the fields for city, country, state, zip code, and tax jurisdiction will default.
City
The city where the warehouse is located.
State
The state where the warehouse is located.
Zip
The zip code.
County
The country where the warehouse is located.
Region
The region assigned to the warehouse.
Manager
The manager assigned to the warehouse, if applicable.
Phone
The manager’s phone number.
Fax
The fax number for the warehouse.
Email Address
The email address for the warehouse.
Buyer
The default buyer for the warehouse. This person is notified for buyer queue issues if no other buyer is assigned.
Ship Via
The ship-via code to designate the default carrier or shipment method.
Div
The division assigned to the warehouse, if divisions are used. Refer to the Hierarchy Level Maintenance screen.
Reg
The region assigned to the warehouse, if regions are used by your company (refer to the Hierarchy Level Maintenance screen.
F2-Save
Saves the warehouse information.
F4-Del
Deletes the warehouse record.
F5-Add Info
Opens the Additional Information screen to allow you access to additional flag settings for the warehouse.
F6-Printing
Opens the Continuous Printing Setup screen to allow you to designate printers for continuous printing.
F7-RTS Maint
Opens the Radio Terminal System Maint screen to allow you to access radio frequency and warehousing flags and options.
Action Bar
Remit Info
Opens the Remit Information screen to allow you to set up remit-to name and address information for the warehouse.

Rental Information screen.

Transfer Information screen.
•
Via Overrides screen.
•
Warehouse Address SSCE Information screen.
•
Forecasting screen.
•
Reel Optimization screen.
•
TPCx Information screen.
•
Route Info screen.
