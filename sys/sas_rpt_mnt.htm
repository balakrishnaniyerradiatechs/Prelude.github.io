Sales Analysis Maintenance screen
The Sales Analysis Maintenance screen allows you to determine how sales information is stored and tracked for the Sales Analysis reports and for tracking sales rep commission.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > File Update Parameters > F5-Sales Analysis
Field and Function Key Descriptions
Field/Function Key
Description
Key Field/Store By/Length:
For each Key Field, you can determine if sales statistics are stored for sales analysis and commission. You can also set the maximum number of characters to accept for each key field.
Customer Number
Indicates if sales analysis statistics are kept by customer:
•
Y - Maintain sales analysis statistics by customer.
•
N - Do not maintain sales analysis statistics by customer.
Product Number
Indicates if sales analysis statistics are kept by product:
•
Y - Maintain sales analysis statistics by product.
•
N - Do not maintain sales analysis statistics by product.
Warehouse Number
Indicates if sales analysis statistics are kept by warehouse:
•
Y - Maintain sales analysis statistics by warehouse.
•
N - Do not maintain sales analysis statistics by warehouse.
Sales Source
Indicates if sales analysis statistics are kept by sales source (e.g., stock, direct):
•
Y - Maintain sales analysis statistics by sales source.
•
N - Do not maintain sales analysis statistics by sales source.
Ship-To
Indicates if sales analysis statistics are kept separate by customer ship-to:
•
Y - Maintain sales analysis statistics separately by customer ship-to.
•
N - Do not maintain sales analysis statistics separately by customer ship-to.
•
A - Keep all files keyed by ship-to.
Sales Rep1
Indicates if sales analysis information is kept for the Salesrep One assigned to the sales order:
•
Y - Maintain sales analysis for the Salesrep One.
•
N - Do not maintain sales analysis for Salesrep One.
Refer to the Commission Subsystem documentation for more information.
Sales Rep2
Indicates if sales analysis information is kept for the Salesrep Two assigned to the sales order:
•
Y - Maintain sales analysis for the Salesrep Two.
•
N - Do not maintain sales analysis for Salesrep Two.
Refer to the Commission Subsystem documentation for more information.
Months
The number of months of sales analysis information to keep.
Commissions
Indicates if commission information is kept in sales analysis:
•
Y - Capture commission information by salesrep for sales analysis.
•
N - Do not capture commission information by salesrep for sales analysis.
Lost Sales
Indicates if lost sales information is kept for sales analysis:
•
Y - Capture lost sales information for sales analysis.
•
N - Do not capture lost sales information for sales analysis.
Line Count
Indicates if line count information is kept for sales analysis:
•
Y - Capture line count information for sales analysis.
•
N - Do not capture line count information for sales analysis.
Use Converted Hist
Indicates if you want to count the converted order history (orders converted into Prelude) as part of your sales analysis information.
Custom
Indicates if you have a custom report for sales analysis that requires a customer file:
•
Y - Custom reports are set up.
•
N - Custom reports are not set up.
F2-Save
Saves the sales analysis parameter information.
