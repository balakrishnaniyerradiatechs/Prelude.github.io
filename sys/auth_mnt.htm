Credit Authorization Maintenance screen
The Credit Authorization Maintenance screen allows you to set up authorization codes to use for customers that require an authorization code in Sales Order Entry (after the customer number and before the header information). An authorization code refers to any user ID set up as an authorization code.
Refer to “Credit Holds” in the Accounts Receivable documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Accounts Receivable Controls > Authorization Codes
Field and Function Key Descriptions
Field/Function Key
Description
Authorization Code
The user IDs of the employees that are allowed to enter sales orders for customers with a Credit Code of A in Customer Maintenance > F6-A/R.
F2-Save
Saves the authorization code information.
F4-Del
Deletes the authorization codes.
