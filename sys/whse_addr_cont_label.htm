Continuous Label Printing Setup screen
If you print labels, you can determine whether each label printing program prints continuously or by batch.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F6-Printing > F5-Cont Labels Setup
Field and Function Key Descriptions
Field/Function Key
Description0
C
Indicates if the document prints continuously or by batch:
•
Y - This warehouse prints the corresponding document continuously. Enter the Prt (port number) and Form Queue where these documents print.
•
N - This warehouse prints the corresponding document by batch.
•
D - This warehouse prints the corresponding document on more than one continuous printer. Each time the document prints, the user must select a printer or accept the default printer assigned.
Prt
The port number that controls continuous printing for this warehouse/document. Port numbers are set up in Continuous Printer Port Maintenance.
Stock receipts printing flag (unlabeled)
Determines when labels print for stock receipts:
•
U - Print labels after the Batch Update Procedure.
•
L - Print labels after each line is received in Stock Receipts Entry.
Form Queue
The printer number where this document prints. If you entered D for the continuous printing flag, this printer is the default printer. Form Queues are set up in Form Queue Maintenance to indicate where documents may print.
Stk Rec Label Flag
Determines if stock receipts labels print continuously or by batch.
Shipping Label Flag
Determines if shipping labels print continuously or by batch.
W/O Label Flag
Determines if work order labels print continuously or by batch.
Stk Xfer Rec Label Flag
Determines if stock transfer labels print continuously or by batch.
Inv Adj Label Flag
Determines if inventory adjustment labels print continuously or by batch.
Invoice Label Flag
Determines if invoice labels print continuously or by batch.
Sample Labels
Determines if sample labels print continuously or by batch. Available if you use the Book/Sample subsystem.
License Plate Label
Determines if license plate labels print when license plate (lot-controlled) items are received.
SR Wty Tag Labels
Determines if warranty tag labels print continuously or by batch for warranty tag rejections.
RF Conveyer Label
Determines if conveyor labels print continuously or by batch for sales orders, work orders, or transfers.
F2-Save
Saves the label printing information.
