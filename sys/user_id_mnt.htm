User Identification Maintenance screen
The User Identification Maintenance screen allows you to set up a code to identify each user that will access or enter transactions in Prelude. This code stores information for the user, determines what the user has access to, and records any transactions entered by the user.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
User Id Code
The code that identifies a user, up to 4 alphanumeric characters. This code stores information about the user, determines what the user has access to, and records any transactions entered by the user.
Description
The description of the user.
Company Number
The company number that this user is assigned to. Enter a company number (set up in the Company Maintenance screen) or select one of the following:
•
A - All. The user can operate in all companies
•
S - Selective. Opens the Company Selection screen to allow you to enter the companies that this user can operate in.
Division
The division number that this user is assigned to, if you use divisions (refer to the Hierarchy Level Maintenance screen). Enter a division number or select one of the following:
•
A - All. The user can operate in all divisions.
•
S - Selective. Opens the Division Selection screen to allow you to enter the divisions that this user can operate in.
Region
The region number that this user is assigned to, if you use regions (refer to the Hierarchy Level Maintenance screen). Enter a region number or select one of the following:
•
A - All. The user can operate in all regions.
•
S - Selective. Opens the Regions Selection screen to allow you to enter the regions that this user can operate in.
Location
The warehouse or location that this user is assigned to (refer to the Hierarchy Level Maintenance screen):
•
A - All. The user can operate in all locations.
•
S - Selective. Opens the Warehouse Selection screen to allow you to enter the only warehouse locations that this user can operate in.
•
Warehouse number. In the next (unlabeled) field, enter Y to default this warehouse number in the location retrieval screen, but allow the user to change it. Enter N to automatically load this warehouse number and not allow the user to change it.
Department
The department number that this user is assigned to, if you use departments (refer to the Hierarchy Level Maintenance screen). Enter a department number or select one of the following:
•
A - All. The user can operate in all departments.
•
S - Selective. Opens the Departments Selection screen to allow you to enter the departments that this user can operate in.
Cost Center
The cost center number that this user is assigned to, if you use cost centers (refer to the Hierarchy Level Maintenance screen). Enter a cost center number or select one of the following:
•
A - All. The user can operate in all cost centers.
•
S - Selective. Opens the Cost Center Selection screen to allow you to enter the cost center that this user can operate in.
Email Address
The email address for the user.
Desqtop Id
The desqtop Id if your company uses Desqtop (no longer supported by Prelude).
Location Profile
The location profile assigned to this user. Location profiles are set up in the Startup Parameter Maintenance screen to allow you to set up default information for a group of users in one location (e.g, printers, terms codes, tax jurisdictions).
F2-Save
Saves the user ID information.
F4-Del
Deletes the user.
F5-Administrative
Opens the Administrative Settings screen to allow you to assign manager and restriction information for this user.
F6-Add Info
Opens the Additional Information screen to allow you to set flags and restrictions for this user.
F7-Personal
Opens the User Id Personal Information screen to allow you to enter personal information for this user.
Action Bar
User Options
Allows access to the following:
•
User Security Setup (See your systems administrator for more information).
•
Slash Process Security screen.
•
GUI Options screen.
•
Sales Force Automation Salesrep screen.
•
P/O Type Restriction screen.
•
Order Type Restrictions screen.
•
Notes Edit Authorization screen.
•
Warehousing Information screen.
•
PC Applications screen.
Copy
Opens the User Id Copy screen to allow you to copy a user ID. You can only access this screen before you enter a User Id Code in the User Identification Maintenance screen.
