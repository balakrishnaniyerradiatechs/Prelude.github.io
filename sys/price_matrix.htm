Price Matrix Types & Sequence Maintenance screen
Price matrix types are set up by Prelude and are used to define special prices beyond column and/or quantity break prices. The Price Matrix Types & Sequence Maintenance screen allows you to define valid price matrices for your company and the order in which they should be used. Refer to “Price Matrices” in the Order Processing documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Price Matrix Types & Sequence
