Order Processing Controls
Sales Order Entry Types screen
Sales order types are set up by Prelude and used in Sales Order Entry to define the type of sales order and the shipping and backorder conditions for the order. The Sales Order Entry Types screen allows you to set the default sales order type that appears in the Sales Order Header screen and to determine which types are valid for your company. Refer to “Sales Orders Types” in the Order Processing documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Order Type Defaults > Sales Order Entry Types
Field and Function Key Descriptions
Field/Function Key
Description
Order Type Default
The sales order type to default in the Sales Order Header screen.

The sales types that are valid in Sales Order Entry for your company. The available types in Prelude are:

05 - Over the counter.
•
06 - Confirmation.

08 - Confirmation consignment.
•
09 - RGA.
•
10 - Repair.
•
12 - Invoice when complete.
•
13 - Ship backorder complete.
•
14 - Ship line item complete.
•
15 - Show.
•
16 - Order projection.
•
17 - Manufacturer rep.
•
18 - Crib replenishment.
Refer to “Sales Order Types” in the Order Processing documentation for a detailed explanation of each type.
F2-Save
Saves the sales order type information.
F4-Del
Deletes the valid sales order types.
