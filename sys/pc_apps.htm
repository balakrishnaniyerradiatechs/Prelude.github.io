PC Applications screen
The PC Applications screen defines viewing options for reports and inquiries for the user.
Menu Path: Systems > System Parameters > Maintenance > User Identification Maintenance > Action Bar > User Options > Define PC Applications
Field and Function Key Descriptions
Field/Function Key
Description
Spreadsheet App
Defines the spreadsheet application to load when the user selects the L output option in reports and inquiries:
•
E - Microsoft Excel.
•
O - Open Office/Star Office
ADS Conversion
Determines if user uses the ADS Excel Conversion utility:
•
Y - The Excel Conversion utility is in effect.
•
N - The Excel Conversion utility is not used.
Ss Path
The path to the temporary CSV file for terminal servers.
Imaging Path
The path for imaging software.
F2-Save
Saves the PC application information.
