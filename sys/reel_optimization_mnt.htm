Reel Optimization screen
If you have a product that is sold in reels, you are often left with small remaining quantities of the product. To prevent wasting the product, you can let the system determine the reels to cut. Reel optimization allows you to enter the quantity and length of reel cuts on a sales order and then let the system select the optimal lot(s) to fill the order. Refer to “Allocating Optimal Reel Quantities” in the Inventory documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > Action Bar > Addl Opts > Reel Optimization
Field and Function Key Descriptions
Field/Function Key
Description
Sales Order Reel Optimization
Determines if optimal lot allocation is used to allocate lot numbers in Sales Order Entry:
•
Y - Use optimal allocation of a lot/reel quantity on sales orders. The system finds the lot number that contains the majority of the available quantity to fill the lot cut requirement.
If more than one sales order is entered for a lot reel quantity, the system evaluates the lot quantities available to find the best match of each reel quantity for each sales order.
•
N - Reel optimization is not used.
F2-Save
Saves the reel optimization information.
