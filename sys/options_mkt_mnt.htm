Marketing Parameters screen
The Marketing Parameters screen allows you to set flag and option parameters to control how Prelude operates programs in the Marketing module.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > Marketing Parameters
Field and Function Key Descriptions
Field/Function Key
Description
Campaign Priority Code
Determines if campaign priority codes are required for quotes and sales orders:
•
Y - You must enter a campaign code.
•
N - You cannot enter a campaign code.
•
O - You may enter a campaign code but it is not required.
Utilize Campaign Pricing
Determines if campaign pricing is used for campaigns:
•
Y - Use pricing entered through Campaign Maintenance for customers participating in the campaign.
•
N - Use regular pricing, not campaign pricing.
Refer to “Using Campaigns” in the Marketing documentation for information on setting up campaigns.
Contact Name Override
Determines which name appears in Contact in Sales and Quote Order Entry, if you use contacts:
•
Y - Use the contact name as the name on the sales order.
•
N - Use the contact based on the Override Contact Name in Order Processing Option Parameters, which determines if the contact from the customer Ship To should be used or if Contact should be blank.
Default Purchasing Contact
Determines if the Purchasing Contact (Customer Maintenance > F8-Contact) is defaulted in Sales Order and Quote Order Entry:
•
Y - If one purchasing contact exists for the customer, it is the default. If multiple contacts exists, no contact is defaulted.
•
N - Do not default a purchasing contact. You can still enter a Purchasing Contact on the sales order.
Utilize Mail-To
Determines if you want to save mail-to addresses for prospects:
•
Y - Use Mail-To addresses for prospects.
•
N - Do not use Mail-To addresses for prospects.
 
The Mail-to is used in Marketing Activity Entry and in place of a ship-to on quotes and can be converted to ship-to’s.
Print Form Letters Batch
Determines how form letters generate (print, fax, email) from a marketing activity:
•
A - Allows you to choose to generate immediately or by batch.
•
Y - Generate by batch only.
•
N - Generates when you save the activity.
Quote F/U - Activity Inq
Determines if quote follow-ups are included in the marketing activity inquiries:
•
Y - Quote follow-ups are included.
•
N - Quote follow-ups are not included.
 
About Defaults
If you activate defaults for product, product line, major group, document type, and/or document number, that data defaults into marketing activities. For example, if you enter a marketing activity from a sales order, the document type and document number default. If you enter a marketing activity from Product Maintenance, the product number, product line, and major group (if applicable) default.
Activate Product# Default
Indicates if you want to default the product number into a marketing activity:
•
Y - Default the product number.
•
N - Leave the product number blank.
Activate PLine Default
Indicates if you want to default the product line into a marketing activity:
•
Y - Default the product line.
•
N - Leave the product line blank.
Activate MajorGrp Default
Indicates if you want to default the major group into a marketing activity:
•
Y - Default the major group.
•
N - Leave the major group blank.
Activate Document Type
Indicates if you want to default the document type into a marketing activity:
•
Y - Default the document type.
•
N - Leave the document type blank.
Activate Document Number
Indicates if you want to default the document number into a marketing activity:
•
Y - Default the document number.
•
N - Leave the document number blank.
Activate Queue Menu
 
If you want to use a custom menu for your activities, the name of the custom menu. The menu must be named CUSTOM.<INT>, where INT is your company initials (e.g., CUSTOM.ABC). Call Activant support for help with custom menus.
F2-Save
Saves the marketing parameter settings.
