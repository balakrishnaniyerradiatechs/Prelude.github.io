Warehousing Maintenance screen
If you use the Warehousing module, you can use the Warehousing Maintenance screen to customize sales order picking methods for either radio frequency units or pick tickets.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint > F5-Warehousing
Field and Function Key Descriptions
Field/Function Key
Description
Warehousing Flag
Determines if this warehouse uses the warehousing module functions:
•
Y - Activates warehousing for this warehouse.
•
N - This warehouse does not use warehousing. The remaining fields are not accessible.
Validate Radio Frequency Entry
Determines if RF units are required to scan barcodes:
•
Y - RF units are required to scan barcodes (excludes work order processing).
•
N - RF units are not required to scan barcodes.
•
S - Selective validation. Enter the codes for the fields that require scanning verification in the next (unlabeled) field.
Fields that require scanning verification (unlabeled)
 
The code for the fields that require scanning verification:
•
P - Product (excluding asterisk (*) items.
•
F - From bin.
•
T - To bin.
•
Q - Quantity.
•
L - Lot.
Pick/Bulk Bin %
The percent that determines if a work order is picked from the pick bin or the bulk bin.
If a work order requires less than this percent of the bin maximum for the product, then work order picking allocates from the pick bin. If the requirement is greater than this percent of the bin minimum, then it allocates products from the bulk bin. This flag also applies to transfer bin allocation.
Whse Start Time
 
The time the warehouse begins work. This time determines the default date in route processing and pick/pack statistics. Activities before this time are considered as yesterday. Enter as MM:HH.
Rf Worksheet Qty Screen
Determines if you are prompted for the Worksheet Entry screen in the various RF and Truck Mount screens (e.g., picking, receipts, putaway). You can enter the number of cases, quantity per case, and broken cases. This field is only accessible if RF Quantity is set to W in the Option Parameters For Warehousing screen
Trans Bin Move From Bin
Determines if you can enter From Bins in Transaction Bin Move Entry (Warehousing > Maintenance):
•
Y - Allow entry of multiple From bins. Only the quantity from the sales order release is moved to the To bin.
•
N - Do not allow entry of From bins. The field is not accessible.
Sales Order Picking Options
S/O Radio Frequency Picking
Determines if sales orders are picked with radio frequency units:
•
Y - Sales orders are picked with RF units.
•
S - Sales order picking is determined by truck route segments. Refer to the Warehousing Segment Maintenance screen.
•
N - Sales orders are picked with pick tickets.
S/O Picker Assignment Required
Determines if all sales order releases must be assigned to a picker:
•
Y - All sales order releases must have a picker assigned in Sales Order Picker Assignment.
•
N - Pickers can select the next sales order release available.
This flag is only accessible if S/O Radio Frequency Picking is set to Y.
S/O Picking Paperless
Determines if the pick ticket prints or if paperless picking is used:
•
Y - No pick ticket prints (suppress printing).
•
N - The pick ticket prints and the RF sales order picking queue is updated.
S/O Allocation Pickup Ship Via
Determines if bin level commitments are required for pick-up sales orders:
•
Y - Allows bin level commitments for sales orders with ship via set to P for pick-up. This flag is automatically set to Y if S/O Radio Frequency Picking is set to Y.
•
N - Bin level commitments are not used.
Max Lines For Orders
The maximum lines allowed on a sales order before it is sent to a staging bin. Sales orders with more than this number of lines are sent to a staging bin instead of being released as part of the pick ticket routine.
Staging Order Printer
The printer form queue number where final staging pick tickets print when all picking is completed.
Update Pick Label File
Determines if picking information is updated when orders are processed:
•
Y - Picking information is updated when orders are processed. Required if you use wrist RF picking for sales orders.
•
N - Picking information is not updated.
Automatically set to Y if S/O Radio Frequency Picking is set to Y.
Prompt For Dest Bin In PKCE
Determines the default for the destination bin in Pick Ticket Confirmation Entry:
•
Y - Request a destination bin.
•
N - Default PICK as the destination bin.
Skip From Bin
Determines if the RF picker must enter a FROM bin in RF Order Picking:
•
Y - Skip the FROM bin and go directly to the quantity field.
•
N - Require the picker to enter a FROM bin.
Prompt For Route/Ship Via?
Determines if the route and ship via code are requested in RF Order Picking:
•
Y - Request a route and ship via code.
•
N - Do not request a route and ship via code.
If you enter a release number in RF Order Picking, the route and ship via fields are skipped.
Prompt For Dest Bin in RF Pick
Determines if a destination bin is requested in RF Order Picking:
•
Y - Request a destination bin for each release.
•
N - Do not request or use destination bins.
•
S - Request a destination bin only once to use for all releases picked during the session.
Rf Order Hold Area
Determines if you can enter a hold area in RF picking releases:
•
Y - You can enter a hold area which must be set up as a valid location. Prompt For Dest Bin in RF Pick must be set to Y so that you can enter the hold area as the destination bin.
•
N - Hold areas are not used.
Auto Rce After Rf Picking
Determines if releases are automatically confirmed after RF Order Picking and setup for invoicing:
•
Y - Automatically confirm releases after RF Order Picking (including zero or partial pick quantities).
•
N - Do Not Automatically Confirm Releases After RF order picking.
Rce Default Release#
Determines if the next available release number defaults in Release Confirmation Entry:
•
X - Default the next available release number (according to the priority codes set-up for the user) in Release Confirmation Entry. The user ID appears as the packer ID in the header screen.
•
N - Do not default a release number in Release Confirmation Entry.
Rf Picking Shipping Label
Determines if shipping labels print for releases in RF Order Picking:
•
Y - Shipping labels print for releases at the time of RF Order picking if releases are generated by order number.
•
N - Shipping labels do not print for releases.
To Bin Default Method
Determines if the TO bin is blank for the first line on each release in RF Order Picking:
•
P - Previous. Use the last TO bin entered as the first line for the remaining lines on the release.
•
R - Release. Leave the TO bin blank for the first line picked on each release. The remaining lines use the last TO bin entered on the release.
