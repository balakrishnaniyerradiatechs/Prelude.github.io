Warehousing Work Order Maintenance screen
If you use the Warehousing module, you can use the Warehousing Work Order Maintenance screen to set up bin allocation and picking rules for work orders.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > F7-RTS Maint > F5-Warehousing > F7-Work Orders
Field and Function Key Descriptions
Field/Function Key
Description
Work Order Picking Options
WO FIFO Lot Allocation Flag
Determines how work orders use FIFO lot allocation:
•
Y - Use FIFO lot allocation and always allocate the oldest lot first even if another single lot can completely fill a requirement.
•
N - If possible, use the oldest lot that can completely fill an order. If no single lot can fill an order, allocate partial lots using the oldest quantity first.
W/O Bin Sorting Method
Controls how work order bins are sorted:
•
O or blank - Sort starting with the smallest bin on-hand quantity. This option can help clean up the supplying bins with smaller bin on-hand quantities, to release bin space for other products.
•
D - Sort starting with the oldest bin transaction date. This option uses a method similar to a FIFO rotation of inventory.
This is the secondary sort sequence within the allocation priority for non-lot controlled products.
W/O Prompt For RF Picker
Determines if you must enter a user ID in Work Order Picking:
•
Y - You must enter a user ID in Work Order Picking.
•
N - A user ID is not required in Work Order Picking.
W/O Radio Frequency Picking
Determines if work orders are picked with radio frequency units:
•
Y - Work orders are picked with RF units. You can use Work Order Bin Allocation to allocate bins.
•
N - Work orders are picked with pick tickets. Work Order Bin Allocation is not allowed. You must use Work Order Printing to generate the release and allocate bins. Warehousing work orders can be issued in Work Order Entry.
W/O Deletion Bin
The bin to use for deleted work orders or deleted work order line items. Note: The work-in-process bin used in RF Work Order Picking must have Storage Type set to W in Bin Location Maintenance so that work order deletion will occur in Work Order Entry.
If this field is blank, deleted work orders will not move to a bin.
W/O Allocation Priority
The priority sequence for the corresponding Type of allocation method.
Type
The bin type for the corresponding W/O Allocation Priority sequence:
•
B - Bulk bin.
•
D - Dock bin.
•
I - Inspection bin.
•
K - Bulk un-packaged.
•
L - Line stock bin.
•
P - Pick bin.
•
T - Retail bin.
F2-Save
Saves the Work Order picking and bin information.
