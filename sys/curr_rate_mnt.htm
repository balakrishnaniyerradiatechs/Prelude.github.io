Currency Exchange Rate Maint screen
You can use Currency Exchange Rate Maintenance to change the exchange rate for each currency type as often as necessary. Refer to “Using Multiple Currency” in the General Ledger documentation.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Currency Maintenance > Daily Currency Exchange Rate Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Currency Type
The currency type for the country whose currency exchange rate changed.
Effective Date
The date the new exchange rate goes into effect.
Exchange Rate
The currency rate of exchange to use for this Currency Type starting on the Effective Date.
F2-Save
Saves the exchange rate information.
F4-Del
Deletes the exchange rate information.
