Purchasing Warehouse Sequence Maintenance screen
The Purchasing Warehouse Sequence Maintenance screen allows you to set the order to display warehouses for purchasing inquiries. Only warehouses on this list are included on the Recommended P/O Inquiry and the Recommended P/O Report.
If this warehouse sequence is blank, the warehouse sequence from the Warehouse Sequence Maintenance screen is used for purchasing inquiries and reports.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Sequences Maintenance > F5-Purchasing Seq
Field and Function Key Descriptions
Field/Function Key
Description
Whse
The warehouse numbers in the sequence to display in purchasing screens.
Description
The description of the warehouse.
F2-Save
Saves the purchasing warehouse sequence information.
