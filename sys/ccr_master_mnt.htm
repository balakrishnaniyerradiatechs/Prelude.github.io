Cost Center Master Maintenance screen
The Cost Center Master Maintenance screen defines cost centers for your company. Cost centers are a subset of Departments - if Departments are used. Typically used to separate expenses within a warehouse or department. Use the Cost Center Address Maintenance screen to add or edit address, division, region, location, department, and manager information for the Cost Center.
Menu Path: Systems > System Parameters > Maintenance > General Ledger Controls > Hierarchy Maintenance > F9-Ccr List
Field and Function Key Descriptions
Field/Function Key
Description
Ccr
The cost center number for each valid cost center.
Description
The description of the cost center (e.g., shipping, receiving, outside sales, inside sales).
F2-Save
Saves the cost center information.
F4-Del
Deletes all the cost centers listed on the screen.
