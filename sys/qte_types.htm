Quotation Entry Types screen
Quote order types are set up by Prelude and used in Quote Order Entry to define the type of quote. The Quotation Entry Types screen allows you to set the default quote type that appears in the Quotation Header screen and to determine which types are valid for your company. Refer to “Entering a Quote” in the Order Processing documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Order Type Defaults > Quotation Entry Types
Field and Function Key Descriptions
Field/Function Key
Description
Quote Type Default
The order type to default in the Quotation Header screen in Quotation Order Entry.
Expiration Days
The default number of days to use for the Expiration Date in the Quotation Header screen. The quote will expire if not converted this number of days after the Quote Date.
Quote F/U Days
The default number of days to use for the Follow Up Date in the Quotation Header screen. This is the date to be reminded to follow up on the quote if it is not yet converted or deleted. The quotes appear in the Quote Follow Up Inquiry and the Quote Follow Up Report.
Quick Quote Days
The default number of days to use for the Expiration Date in the Quotation Header screen when a quick quote is converted to a quote. The quote will expire if not converted this number of days after the Quote Date.
Quick Quote F/U Days
The default number of days to use for the Follow Up Date in the Quotation Header screen when a quick quote is converted to a quote. This is the date to be reminded to follow up on the quote if it is not yet converted or deleted. The quotes appear in the Quote Follow Up Inquiry and the Quote Follow Up Report.
Valid Quote Types
The types that are valid in Quotation Order Entry for your company. The available types in Prelude are:
•
01 - Regular Quote. A standard quotation to give the customer price and availability information.
•
02 - Planned Quote. When converting the quote into a sales order, use a multiplier to change the quantities. This type is not purged and can be used indefinitely for repeat quotes with different quantities.
•
03 - Repair Type Quote. Repair orders interface with the Work Order module to accumulate material, labor, and other costs associated with the repair.
•
04 - Standing Quote. Allows you to enter information to convert the quote to a sales order on a regular basis for a given time period using the Standing Quote Conversion.
Refer to “Entering a Quote” in the Order Processing documentation for more information.
F2-Save
Saves the quotation type information.
F4-Del
Deletes the valid quotation types
