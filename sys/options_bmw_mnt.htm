BMW/Work Order Parameters screen
The BMW/Work Order Parameters screen allows you to set flag and option parameters to control how Prelude operates programs for bills of material and work orders.
Menu Path: Systems > System Parameters > Maintenance > Option Parameters > BMW/Work Order Parameters
Field and Function Key Descriptions
Field/Function Key
Description
W/O Material Cost
Determines which cost to use on work orders:
•
A - Actual product cost.
•
S - Standard Cost. If you use standard cost as the accounting cost for your company (G/L Cost flag set to S in the Company Maintenance Additional Information screen), you can maintain the cost for a product level or product warehouse.
W/O Labor Cost
Determines which labor cost to use on work orders:
•
A - Actual labor cost.
•
S - Standard cost.
G/L Cost
The accounting cost to use for work orders:
•
A - Average.
•
R - Replacement cost.
•
S - Standard cost.
•
F - FIFO cost.
•
L - LIFO cost.
Roll Up Standard Cost
Determines if maintenance for standard cost for components is rolled into standard cost for the work order:
•
Y - Allow maintenance for standard cost to roll into standard cost of a work order for component items (maintenance only, no G/L updates).
•
N - Do not roll up maintenance cost of components items into standard cost.
W/O Mfg Production
Determines what usage is include in EOQ (economic order quantity) calculations:
•
Y - EOQ calculations are based on sales and transfer usage. This option creates planned work orders based on dependant demand.
•
N - EOQ calculations are based only on work order usage.
Auto Release W/O
Determines if the work order is automatically released in Work Order Entry when all components are available:
•
Y - Work orders automatically release if all components are available.
•
N - Work orders created through Sales Order Entry are not released. You can release work orders created through Work Order Entry for production. If you do not release the work order, it must be released in Work Order Release Inquiry.
Pick Ticket Default Days
Determines when the pick ticket prints:
•
T - Print the pick ticket on the work order entry date.
•
A - Calculate the pick ticket date based on assembly days or an actual number of days.
Display Addl Desc In Woe
Determines if the product’s additional description (if applicable) displays in Work Order Entry:
•
Y - Display the additional description.
•
N - Do not display the additional description.
W/O Close Prompt
Determines when the message appears to close the work order in Work Order Completions:
•
Y - Display the message to close the work order when a work order is partially or totally completed.
•
N - Only display the message to close the work order for total completions.
Released By User Id
Determines if you are asked to enter your user ID when releasing a work order for production:
•
Y - Ask for a user ID.
•
N - Default to the internal user ID if no ID is entered.
Auto. Display Keyword Scr
Determines if the Serial Number / Tags screen appears in Work Order Entry to allow you to add keywords to the bill-of-material:
•
Y - Automatically display the Serial Number / Tags screen in Work Order Entry. When the screen appears, you can choose to add keywords for S - Serial numbers (serial numbers must be required for BOM items) or T - Tag numbers (tag numbers can be used anytime to access a BOM by tag).
•
N - The Serial Number / Tags screen does not appear in Work Order Entry.
Print B/O On Pick Tickets
Indicates if backordered items print on the initial pick ticket:
•
Y - Display backordered items on the first pick ticket only.
•
N - Backordered items do not appear on any pick tickets.
•
A - Display backordered items on all pick ticket releases.
Omit Comp. For Repair S/O
Determines if components are omitted from a work order created through Sales Order Entry as a repair order, since repair orders do not always need all of the components:
•
Y - Omit components for repair work orders created in Sales Order Entry.
•
N - Include components for repair work orders created in Sales Order Entry.
Max. Comp. LDT Days
The maximum number of component lead time commitment days. Inventory will be committed to the work order this number of days before the due date. Works with W/O Fut Commit Safety Days.
W/O Fut Commit Safety Days
The number of days to add to Max. Comp. LDT Days to calculate when inventory on a future work order is committed.
Completion Cost Tolerance %
The tolerance percent difference that is allowed in Work Order Completions. If the current cost of the finished good is different than the work order completion cost by more than this percent, a warning message appears in Work Order Completions. You can either go back to the order to correct any errors or choose to update the order as it is.
Auto Transfer For Repairs
Determines if transfers for a repair are automatically generated when the selling warehouse is different from the repairing warehouse:
•
Y - Automatically generate the transfer.
•
N - Do not automatically generate the transfer. You must generate one if necessary.
Auto Build-To-Order W/O
Determines if the work order is automatically created from within Sales Order Entry:
•
W - Automatically create the work order from Sales Order Entry without any messages.
•
Y - You have the choice to automatically create the work order. A message appears if any items are backordered. No warning appears when work orders are generated in Direct Order Entry, Sales Force Automation, or Electronic Data Interchange.
•
N - Do not automatically generate work orders from Sales Order Entry. You have the choice to generate the work order.
‘*: N/S Price/Cost Prompt WO
Determines if you can enter cost and price on nonstock, asterisk (*) items in Work Order Entry:
•
Y - Prompt for list price and cost.
•
C - Prompt for cost only.
•
N - Prompt for list price only.
Allow New BOM Entry In W/O
Determines if you can create a new bill-of-material in Work Order Entry:
•
Y - You can create a new bill-of-material. When you enter the bill-of-material number and choose to create the bill-of-material, you are taken into Bill-of-Material Maintenance to set up the BOM.
•
N - You cannot create a new bill-of-material from Work Order Entry. You must create the BOM in Bill of Material Maintenance.
Allow SRE Overage In W/O
Determines if an over-received product that is linked to a work order should allocate to the work order:
•
Y - The overage is allocated to the work order and the line item quantity is increased.
•
N - The overage goes into to stock.
Auto Enter Core Credit Line
Determines if a core credit line item is added to a work order after the exchange product:
•
Y - Automatically add the core credit line after the exchange product.
•
N - Do not add the core credit line after the exchange product. You must manually add it.
Allow Comp In W/O Issues
Determines if you can complete a work order in Work Order Issues Entry:
•
Y - You can choose to complete the work order in Work Order Issues Entry. This option eliminates the Work Order Completions step.
•
N - You cannot complete the work order in Work Order Issues Entry. You must use Work Order Completions to complete and close the work order.
Allow Overage In W/O
Determines if overages on a work order are passed on to the linked sales order when the work order is completed:
•
Y - Allocate the overage to increase the quantity of the line item on the linked sales order.
•
N - The overage goes into stock.
Allow W/O Release for Partial
Determines if work orders can be released for a partial quantity:
•
Y - Work orders can be released with a partial or full quantity.
•
N - Work orders must have a full quantity to be released.
Auto W/O Deletion From SOE
Determines if a linked work order is automatically deleted if the line item on the linked sales order is deleted:
•
Y - The linked work order is automatically deleted when the sales order line item is deleted. If the work order is already released for production, a message displays and you cannot delete the item.
•
N - The linked work order is not automatically deleted when the sales order line item is deleted. The work order must be deleted manually.
Allow WO Over To Linked Xfer
Determines if an overage received in Work Order Completions allocates to the linked stock transfer:
•
Y - The over-received line item is automatically allocated to the linked transfer.
•
N - The over-received line item goes into stock.
Allow W/O Comp Without Issues
Determines if you can complete a work order in Work Order Completions that does not have any issues:
•
Y - You can complete Work Orders even if no line items have issues.
•
N - If no line items have issues, you cannot complete the work order. A warning message appears.
Auto Kit Explosion In WOE
Determines if kit component items appear in the line item detail when the kit parent is entered on a work order:
•
Y - Kit components appear (explode) when the kit parent is entered on a work order. Auto BOM Explosion in WOE is not accessible.
•
N - Kit components do not appear (explode) when the kit parent is entered on a work order.
Auto BOM Explosion In WOE
Determines if bill-of-material component items appear in the line item detail when the BOM number is entered on a work order:
•
Y - BOM components appear (explode) when the BOM number is entered on a work order. In this case Auto Kit Explosion in WOE is not accessible.
•
N - BOM components do not appear (explode) when the BOM number is entered on a work order.
When you convert a temporary work order using F7-Auto Convert in the Recommended Work Order Inquiry screen, the components do not automatically explode. The bill-of-material components only explode if this flag is set to Y and you enter the work order directly in the Work Order Entry screen.
Serial Number Priority Code
Determines if the old serial number is used for all components when creating a work order for manually entered items:
•
O - Automatically use the old serial number for all components.
•
N - Do not automatically use the old serial number.
