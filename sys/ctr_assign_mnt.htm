Next Number Assignment Maintenance screen
You can use the Next Number Assignment Maintenance screen to set or reset a starting number for maintenance records (e.g, Vendor Maintenance, Product Maintenance, Customer Maintenance, Prospect Maintenance).
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Starting Product Assignment
Menu Path: Systems > System Parameters > Maintenance > Purchasing Controls > Starting Vendor Assignment
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Starting Customer Assignment
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Starting Prospect Assignment
Field and Function Key Descriptions
Field/Function Key
Description
Number Assignment Type
The name of the file (e.g., PROD#, VEND#, CUST#, PROS#)
Starting Number
The beginning number to use for this file when new records are created.
F2-Save
Saves the number assignment information.
F4-Del
Deletes the number assignment.
