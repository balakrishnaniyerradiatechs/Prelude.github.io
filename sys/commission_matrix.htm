Commission Matrix Types & Sequence Maintenance screen
Commission matrix types are set up by Prelude and are used to calculate sales commission. The Commission Matrix Types & Sequence Maintenance screen allows you to define valid commission matrices for your company and the order in which they should be used.
Refer to the “Commission Subsystem” in the Order Processing documentation for more information on calculating commission rates.
Menu Path: Systems > System Parameters > Maintenance > Order Processing Controls > Commission Matrix Types Sequence
Field and Function Key Descriptions
Field/Function Key
Description
Evaluate WHS Types After DIR Types:
Determines if the system should check for a warehouse type matrix for direct orders:
•
Y - Use a warehouse type matrix if a direct type matrix is not found.
•
N - Do not use a warehouse matrix if a direct matrix is not found.
Comm Matrix Types
The commission matrix types that are valid for your company, in the order they are used to calculate commission. The first matrix in the sequence is used first unless Best Price is turned on or a customer has a contract price. The available types in Prelude are:
•
1 - Customer/Product.
•
2 - Customer/Vendor.
•
3 - Customer/Product Price Group.
•
4 - Customer Price Group/Product.
•
5 - Customer Price Group/Vendor.
•
6 - Customer Price Group/Product Price Group.
•
7 - Warehouse/Vendor.
•
8 - Warehouse/Product Price Group.
•
9 - Customer/Product Line.
•
10 - Customer/Major Group.
•
11 - Territory/Major Group.
•
12 - Customer Ship-To/Product Group.
•
13 - Territory/Product Line.
•
14 - Warehouse/Product Line.
•
15 - Warehouse/Route/Product Line.
•
16 - Product Line/Sales Rep.
•
17 - Product.
•
18 - Customer.
F2-Save
Saves the matrix and sequence information.
F3-Del
Deletes the commission matrices.
