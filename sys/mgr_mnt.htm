Managers & User IDs
Manager Maintenance screen
The Manager Maintenance screen allows you to set up manager codes for each type of hold queue (e.g., admin invoice hold, admin order hold, credit hold, purchase hold, branch hold). Once these manager codes are established, you can assign these codes to users in the User Identification Administrative Settings screen and to each warehouse in the Warehouse Manager Maintenance screen.
Once the manager codes are assigned to users and warehouses, each user listed under Current User Assignment Settings can release orders from this type of hold queue for the warehouses listed under Current Warehouses Assigned.
Menu Path: Systems > System Parameters > Maintenance > Manager Maintenance
Field and Function Key Descriptions
Field/Function Key
Description
Manager
The manager code. You can set up a code for each type of hold queue (e.g., admin invoice hold, admin order hold, credit hold, purchase hold).
Description
The description of the manager code (e.g., Administrative Invoice Manager - whse 10).
Work Queue Type
The work queue from which this manager can release orders:
•
Admin Invoice Hold.
•
Admin Order Hold.
•
Admin Quote Hold.
•
Branch Hold.
•
Inspection Queue.
•
Emergency Cycle Count Hold.
•
Purchase Order Hold.
•
Credit Hold.
Current User Assignment Settings
User
The user ID codes for the users that are assigned to this manager code. Users are assigned to a manager code in the Administrative Settings screen (Identification Maintenance > F5-Administrative).
Description
The description of the user.
Notification Method
The method by which this user is notified that issues exist in their work queue:
•
Prelude Notify - An animated character or pop-up message appears on the user’s screen.
•
Monitor Display - Toolbar alert icons light up.
•
Email - An alert notice is sent to the user’s email account.
Current Warehouses Assigned
Whse
The warehouses for which this manager type can release orders on this type of hold. The manager codes for each warehouse are assigned in the Warehouse Manager Maintenance screen.
Description
The description of the warehouse.
F2-Save
Saves the warehouse manager information.
F4-Del
Deletes the warehouse manager.
