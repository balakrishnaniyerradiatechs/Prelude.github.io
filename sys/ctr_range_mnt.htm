Counter/Range Setup
Counter/Range Maintenance screen
The Counter/Range Maintenance setup screen allows you to set up the range of document numbers to use for each process in Prelude that is assigned a number. The number range for each process is maintained by warehouse. A document number length can range from 6 to 8 digits and can contain a prefix letter to identify the type of document.

Field and Function Key Descriptions
Field/Function Key
Description
Counter/Range Type
The name of the module that requires numbered documents.
Shortest Document Length
Sets the smallest number of characters allowed for the document and accepts a value from 4 to 8. For example if 6 is the shortest document length and you enter the number 1234, it is translated to 001234.
Longest Document Length
The largest number of characters allowed for the document and accepts a value from 6 to 8. The Prefix + range must be at least 6 characters long.
Normally the shortest and longest document length are the same value.
Rollover W/Alphas
Determines if an alpha character is inserted into a document number when the system runs out of numbers and starts over:
•
Y - When the system rolls over to a new numbering sequence, an alpha character is inserted in the document number. This flag only applies if you use prefixes to define document numbers. Example: if 10 is the prefix for warehouse 10, when the cycle counter rolls over to 999999, it rolls back over to the beginning number and assigns document numbers previously used (e.g.,10A00001).
Also, if this flag is set to Y, you can enter any combination of alphanumeric characters for the Order Number as long as it is the correct length.
•
N - An alpha character is not assigned when rolling over, i.e. 10000001.
Whse
All warehouses set up for your company automatically display. If the Counter/Range Type is not used by a warehouse, then the counter numbers for that warehouse is not required.
Description
The description of the warehouse.
Prefix
A set of alphanumeric characters to append to the beginning of the number. A unique prefix is required for each warehouse. Leave blank to use ranges only.
The Prefix + range must be at least 6 characters but not more than the longest document length. For example, if Longest Document Length is 8 characters and the prefix is 3 characters then the range number is 5 digits.
From
The beginning range number to use for each warehouse. If a Prefix is used, the same From and To for each warehouse could be used. If a prefix is not used, then the From and To should be different for each warehouse.
To
The ending range number to use for each warehouse.
Current
The next document number to use for the process. This number is tracked and updated by the system but you can manually change it.
F2-Save
Saves the counter/range information for the module.
F4-Del
Deletes the counter/range information for the module.
