Crib Information Maintenance screen
If you use crib warehouses, you can use the Crib Information Maintenance screen to set up a warehouse as a crib. A crib warehouse is managed as a separate warehouse. Products are maintained, tracked, purchased, and received into the crib warehouse. Refer to “Using Crib Warehouses” in the Inventory documentation for more information.
Menu Path: Systems > System Parameters > Maintenance > Inventory Controls > Warehouse Maintenance > Action Bar > Addl Opts > Crib Information
Field and Function Key Descriptions
Field/Function Key
Description
Crib Flag
Identifies this warehouse as a crib:
•
Y - This warehouse is a crib.
•
N - This warehouse is not a crib.
Crib Type
Indicates if the crib is managed by warehouse or bin location:
•
W - Warehouse.
•
B - Bin. You must set up the customer as a crib bin in Bin Location Maintenance (Bin Type set to C for Crib Warehouse).
Crib Customer Number
The customer that is identified as a crib warehouse. Only accessible if Crib Type is set to W.
Crib Shipto Number
The crib ship-to number associated with the Crib Customer Number.
F2-Save
Saves the warehouse crib information.
